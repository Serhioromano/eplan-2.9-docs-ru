# Диалоговое окно Настройки: Комбинации клавиш

Параметры > Настройки > Пользователь > Управление > Комбинации клавиш.

В этом диалоговом окне можно создавать и обрабатывать сочетания клавиш, которые присваиваются определенным командам меню.

Обзор основных элементов диалогового окна:

### Выбрать команду

В этом списке перечислены все команды меню. Выделите требуемую команду.

### Описание

В этом поле выводится краткое описание выделенной команды меню.

### Присвоенные комбинации клавиш

Здесь выводятся комбинации клавиш, присвоенные выделенной команде меню (при их наличии). Одной команде меню можно присвоить несколько комбинаций клавиш.

### [Создать]

С помощью этой кнопки вызывается диалоговое окно Создать комбинацию клавиш. В нем можно ввести комбинацию клавиш для данной команды меню.

### [Убрать]

С помощью этой кнопки можно удалить комбинацию клавиш, выделенную в поле Присвоенные комбинации клавиш.

### [Сбросить все]

С помощью этой кнопки можно удалить все определенные пользователем комбинации клавиш. После этого будут присвоены стандартные сочетания.

### Диалоговое окно Создать комбинацию клавиш

Параметры > Настройки > Пользователь > Управление > Комбинации клавиш. Нажмите кнопку ++Создать++.

В этом диалоговом окне можно настроить комбинации клавиш для быстрого вызова определенных функций программы.

### Новая комбинация клавиш

Введите здесь требуемую комбинацию клавиш. Клавиши ++Alt++, [Ctrl] и т. д. вводятся просто нажатием соответствующей клавиши.

# Диалоговое окно "Импорт пространства конструкции"

Пространство листа > Пространство конструкции > Импортировать

В этом диалоговом окне вы осуществляете импорт пространств конструкции (файлов STEP из внешних программ 3D-CAD).

!!! note "Замечание:"

    EPLAN поддерживает исключительно импорт файлов STEP, сгенерированных с помощью прикладного протокола AP 203 или AP 214.

Обзор основных элементов диалогового окна:

### Источник

В этом поле отображаются названия импортируемых файлов STEP. Обработка записей в данном поле невозможна.

### Схема

Выбор схемы или возможность обработки или создания схемы. Клавиша [...] открывает диалоговое окно настроек для импорта STEP.

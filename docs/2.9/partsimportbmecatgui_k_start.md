# База данных изделий: Импорт данных изделий по стандарту eCl@ss

eCl@ss представляет собой стандартизированную, межотраслевую систему классификации для товарных групп, товарных признаков и услуг. Его цель — упрощение электронной торговли классифицированной продукцией.

При импорте данных изделий в базу данных изделий в соответствии со стандартом eCl@ss используется формат "BMEcat" на основе XML. В поддерживаемой версии BMEcat 2005.1 среди прочего возможен импорт многоязычных данных.

Для импорта данных в соответствии с классификацией eCl@ss диалоговое окно Импортировать записи данных в поле Тип файла содержит записи для различных поддерживаемых версий eCl@ss.

Импорт осуществляется с помощью схемы, которую вы выбираете в выведенном диалоговом окне. В этой схеме происходит конфигурация присвоения групп и признаков eCl@ss группам продуктов и свойствам изделий базы данных изделий EPLAN.

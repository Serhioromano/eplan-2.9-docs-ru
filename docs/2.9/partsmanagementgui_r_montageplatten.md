# Вкладка Монтажные платы (база данных изделий)

Сервисные программы > Изделие > Управление. В дереве или списке выделено одно изделие главной группы продуктов "Механика", группа продуктов "Электрошкаф".

На этой вкладке определите специфические данные для монтажных плат, которые хотите присвоить электрошкафу, включая позицию платы; это относится к изделиям Главной группы продуктов 'Механика', Группы продуктов 'Электрошкаф'. При этом данные для каждого изделия могут указываться ***многократно***, а именно один раз для каждого варианта.

Обзор основных элементов диалогового окна:

### Номер изделия

Щелкните в этом поле по кнопке ++...++, чтобы выбрать из диалогового окна выбора изделий какое-либо изделие или вариант изделия. Многократный выбор здесь невозможен.

### Вариант

Если в поле Номер изделия выбран вариант изделия, здесь выводится соответствующий номер варианта. При необходимости его можно перезаписать.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

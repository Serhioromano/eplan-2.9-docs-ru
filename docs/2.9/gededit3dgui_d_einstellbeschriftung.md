# Диалоговое окно Настройки: Маркировка

Вы выделили обзор модели. Обработать > Свойства. В диалоговом окне Обзор модели для свойства Обзор модели: Маркировка функционального элемента в поле "Значение" нажмите кнопку ++...++.

В данном диалоговом окне укажите настройки для маркировки функциональных элементов обзора модели.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Таблица

В таблице отображается, какие функциональные элементы из выбранного пространства листа обозначены, какие свойства должны содержаться в маркировке и как должно выглядеть представление маркировки.

* Определение функции: Кнопка ++...++ в ячейке открывает диалоговое окно Определения функций. В этом диалоговом окне выберите определение функции, для которого необходимо создать маркировку.
* Порядок свойств: Кнопка ++...++ в ячейке открывает диалоговое окно Порядок свойств. В данном диалоговом окне можно изменять настройки отображения маркировки (формат, единица измерения / значение, позиция) и для каждого определения функции настроить определенное пользователем представление.
* Из пространства листа: Если флажок в этом столбце установлен, порядок свойств, заданный в виде для определения функции, будет отменен. Вместо него будет использоваться порядок свойств из пространства листа для представления маркировки. Это позволяет использовать созданную в пространстве листа оптимальную наглядность маркировки в полученном из него виде.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

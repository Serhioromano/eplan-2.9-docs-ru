# Диалоговое окно Многопользовательский монитор

Вы открыли проект. Условия многопользовательского режима выполнены. EPLAN связан с сервером монитора. Проект > Многопользовательский монитор.

В данном диалоговом окне в двух таблицах приведены проекты, относящиеся к многопользовательскому режиму.

Доступны следующие элементы диалогового окна:

### Мои проекты, открытые в многопользовательском режиме

В ***левой таблице*** содержится подробная информация по использованию многопользовательского режима. Здесь представлены проекты, которые открыты одновременно вами и другим пользователем. Для данных проектов приведены следующие данные:

* Проект
* Пользователь
* Область
* Версия продукта
* Операция проекта
* Успешно.

Если пользователь запустил для открытого вами проекта длительную операцию (контрольные прогоны, отчеты и т. д.), то индикатор ее выполнения будет отображен в столбце Успешно.

В столбце Пользователь щелкните идентификатор пользователя, представленный в виде гиперссылки, чтобы запустить установленную на вашем компьютере программу электронной почты и отправить этому пользователю электронное сообщение.

При нажатии пиктограммы ![](images/all_plussign_as.png){: .ui-icon }, расположенной перед соответствующим именем проекта, в обеих таблицах будет отображена дополнительная информация о пользователе (имя, телефон, компьютер). Для использования этой функции пользователи, участвующие в многопользовательском режиме, должны указать соответствующие данные пользователя в настройках (Параметры > Настройки > Пользователь > Отображение > Ид. пользователя / адрес).

!!! note "Замечание:"

    * Проекты, которые открыты исключительно вами, не включаются в многопользовательский монитор.
    * Для обмена электронными сообщениями на компьютере должен быть установлен и настроен редактор электронной почты.

### Другие открытые проекты

В ***правой таблице*** диалогового окна перечислены все проекты, которые открыты только другими пользователями. Для данных проектов приведены следующие данные:

* Проект
* Пользователь.

Если проект открыт несколькими другими пользователями, то в столбце Пользователь для соответствующего проекта будет отображено несколько идентификаторов пользователя.

Если вы хотите послать электронное сообщение одному из пользователей, перечисленных в данной таблице, необходимо сначала развернуть следующую информационную строку. Нажмите на имя пользователя, представленное в виде гиперссылки.

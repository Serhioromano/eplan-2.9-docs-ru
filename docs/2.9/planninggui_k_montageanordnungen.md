# Предварительное планирование: Монтажные чертежи

В предварительном планировании монтажный чертеж используется для графического представления объекта планирования и принадлежностей, необходимых для его монтажа. В дополнение к графическому представлению такой монтажный чертеж включает в себя табличный перечень принадлежностей для монтажа, который показывает используемые изделия.

Страницы с монтажными чертежами генерируются в EPLAN с помощью типа отчета "Обзор групп изделий". Такой отчет предоставляет обзорные страницы обо всех используемых в проекте узлах и модульных изделиях.

### Монтажные чертежи в базе данных изделий

Монтажные чертежи для предварительного планирования реализуются в базе данных изделий в виде узлов. Для вывода монтажных чертежей в базе данных изделий должны быть доступны соответствующие узлы и изделия:

* Сами узлы управляются в главной группе продуктов "Механика" в подгруппе продуктов "Монтажный чертеж > Общее".
* Отдельные изделия узла управляются как изделия типа "Отдельная часть" в главной группе продуктов "Механика" в подгруппе продуктов "Монтажный чертеж > Принадлежности".

Если для монтажного чертежа необходимо отобразить графику, то для соответствующего узла необходимо сохранить макрос с желаемым графическим представлением.

### Монтажные чертежи на объектах планирования

Для присвоения монтажного чертежа объекту планирования перейдите на вкладке Изделие диалогового окна 'Свойства' в выбор изделия и выберите там нужное изделие узла.

!!! note "Замечание:"

    * Мы рекомендуем всегда использовать для узлов монтажных чертежей одну и ту же строку изделия (напр., строку 10). Тогда вы сможете легче определить, присвоено ли объекту планирования изделие узла или нет. Кроме того, тем самым вы защищаете присвоенное изделие узла от непреднамеренной перезаписи при дальнейшем выборе изделий.
    * Кроме того, необходимо деактивировать определенную пользователем настройку для выбора изделия Развернуть узел, в противном случае при выборе изделия объекту планирования будет назначено не изделие узла, а номера изделий отдельных элементов узла.

Для идентификации монтажного чертежа на объекте планирования можно также привлечь свойство Монтажный чертеж (ид. 44056). Свойство можно выбрать для отображения в таблице свойств первой вкладки. Это свойство показывает первое изделие узла из группы продуктов "Монтажный чертеж", которое записано на вкладке Изделие.

### Вывод в качестве обзора групп изделий

В отчетах типа "Обзор групп изделий" имеющиеся в проекте изделия модуля и узла по умолчанию выводятся в виде обзорных страниц. Чтобы ограничить вывод узлов для монтажных чертежей, необходимо использовать фильтр для ограничения вывода.

В прилагаемые формы для обзоров групп изделий помещается текст-заполнитель для макросов, который при генерации отчета заменяется графикой макроса, сохраненного в базе данных изделий.

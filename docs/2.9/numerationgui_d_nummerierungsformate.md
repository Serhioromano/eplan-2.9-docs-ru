# Диалоговое окно Форматы нумерации

* Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация (online). В диалоговом окне Настройки: Нумерация (online) рядом с полем Формат нумерации щелкните по кнопке ++...++.
* Сервисные программы > Изменение стандарта. Введите необходимые данные во вкладках Проект, Поворот / инверсия, Основные данные, Символы и Структура, нажимая каждый раз [Далее]. Щелкните на вкладке Нумерация рядом с полем Нумерация устройств по кнопке ++...++.

Это диалоговое окно служит для управления всеми форматами нумерации для нумерации устройств в режиме подключения. Вы можете создать, обработать, скопировать, удалить, импортировать и экспортировать форматы нумерации. Эти форматы нумерации можно выбрать для устройств, размещаемых на схеме соединений через Вставка > Символ. Также можно выбрать форматы нумерации для устройств, вставляемых в схему соединений через макросы или при помощи копирования.

Доступны следующие элементы диалогового окна:

### Схема

В этом поле указано имя, под которым формат нумерации был сохранен в диалоговом окне Новая схема в поле Имя.

Через расположенную рядом [панель инструментов](schemeconfig_m_schemaschaltflaechen.md) можно создавать новые схемы, сохранять изменения в схеме, копировать или удалить схему. Кроме того, схемы можно импортировать и экспортировать.

### Описание

В этом поле указано описание формата нумерации, которое было введено в диалоговом окне Новая схема в поле Описание.

* * *

Вкладка ОУ

### Устройства общего назначения... Механические устройства

В этом поле представлен формат ОУ для соответствующих функций. С помощью кнопки ++...++ откройте диалоговое окно Формат ОУ, в котором можно обработать формат. Если этот флажок установлен, функции будут автоматически пронумерованы при вставке. Если этот флажок снят, функции не будут автоматически нумероваться при вставке.

### Нумеровать вложенные условные обозначения

Если этот флажок установлен, будут автоматически пронумерованы условные обозначения, присвоенные вышестоящему условному обозначению. Если этот флажок снят, эти условные обозначения не будут автоматически нумероваться при вставке.

* * *

Вкладка "Обозначения"

### Клеммы... Выводы устройства сборных шины

Если этот флажок установлен, соответствующие обозначения будут автоматически пронумерованы при вставке. Если этот флажок снят, обозначения не будут автоматически нумероваться при вставке. В поле рядом с этим флажком указан формат для обозначений. С помощью кнопки ++...++ откройте диалоговое окно Формат ОУ, в котором можно обработать формат.

### Адреса ПЛК

Если этот флажок установлен, адреса ПЛК будут автоматически пронумерованы при их вставке. Если этот флажок снят, адреса ПЛК не будут автоматически нумероваться при вставке.

### Идентификаторы сортировки клемм

Эта настройка влияет только на имеющиеся идентификаторы сортировки клемм при вставке макросов и копий.

Если этот флажок установлен, имеющиеся идентификаторы сортировки будут автоматически пронумерованы при их вставке. Для этого определяется самый высокий из имеющихся в клеммнике идентификаторов сортировки, и его значение повышается на единицу. Таким образом, вновь вставленные клеммы всегда помещаются в конец клеммника. Если идентификаторы сортировки имеют несколько клемм, последовательность сохраняется и после увеличения значений. Взаимосвязанные клеммы в результате нумерации не разделяются. Если хотя бы одна клемма в макросе или в копии имеет идентификатор сортировки, то и все другие клеммы, за исключением графически отсортированных клемм, получают идентификатор сортировки.

Если флажок снят, идентификаторы сортировки вставленных клемм не изменяются. В таком случае клеммы располагаются среди уже имеющихся клемм согласно их идентификаторам и обозначениям. Это может привести к разделению прежде взаимосвязанных клемм (например, многоуровневых клемм).

### Не изменять PE-, N и SH-клеммы

Если этот флажок установлен, PE- / PEN-, N- и SH-клеммы не будут автоматически пронумерованы. Обозначение PE- / PEN-, N и SH-клемм остается без изменений.

### Не изменять PE-, N и SH-контакты штекера

Если этот флажок установлен, PE- / PEN-, N- и SH-контакты штекера не будут автоматически пронумерованы. Обозначение PE- / PEN-, N и SH-контактов штекера остается без изменений.

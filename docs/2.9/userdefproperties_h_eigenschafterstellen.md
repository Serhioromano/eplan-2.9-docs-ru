# Создать определенные пользователем свойства

Условия:

Определенные пользователем свойства для ***проекта***:

* Вы открыли проект.
* Вы выбрали пункты меню Данные проекта > Конфигурировать свойства.

Определенные пользователем свойства в ***базе данных изделий***:

* Вы открыли базу данных изделий. Сервисные программы > Изделие > Управление.
* Вы выбрали пункты меню ++Дополнительно++ > Конфигурировать свойства.

1. Щелкните в диалоговом окне Конфигурировать свойства по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
2. В диалоговом окне [Новое свойство](eservicesgui_d_konfigeigenschaften.md) введите уникальное идентифицирующее имя.

!!! info "Для сведения:"

    Идентифицирующее имя затем уже нельзя изменить.

3. В поле Отображаемое имя введите имя, под которым свойство должно отображаться в проекте / базе данных изделий. Возможен многоязычный ввод.
4. В поле Описание при необходимости введите описательный текст, который затем будет отображаться в диалоговых окнах выбора свойств для данного свойства.
5. В поле Присвоение установите, для каких объектов должно быть доступно свойство.

!!! info "Для сведения:"

    Для конфигурации в базе данных изделий в качестве присвоения доступна только запись "Изделие". Эта запись отсутствует при конфигурации для проекта.

6. Установите флажок Также активировать для ссылок изделий, если определенное пользователем свойство для изделия также должно быть доступно как свойство для ссылок изделий.
7. С помощью раскрывающегося списка Тип поля задайте символы и формат, разрешенные для определенного пользователем свойства. Выберите запись "Многоязычный текст", если значения свойств впоследствии нужно перевести.
8. Установите флажок Перевести значение свойства, если значение определенного пользователем многоязычного свойства необходимо учитывать при автоматическом переводе проекта или при автоматическом переводе в базе данных изделий.
9. В поле Помощь ввода укажите, возможен ли свободный ввод для свойства или необходимо выбирать значения из списка выбора.
10. В поле Категория выберите категорию, в которой должно отображаться данное свойство.
11. Если в поле Помощь ввода вы выбрали один из параметров "Список выбора", "Свободный ввод со списком вариантов выбора" или "Саморасширяющийся список выбора", определите в поле Список выбора значений по умолчанию возможные записи в раскрывающемся списке.
12. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Новое определенное пользователем свойство для ***проекта*** сохраняется как специфическое для проекта и доступно в нем на существующем объекте в соответствии с заданным присвоением.

!!! info "Для сведения:"

    Новое определенное пользователем свойство для ***изделия*** сохраняется в базе данных изделий и может присваиваться изделию с помощью вкладки Определенные пользователем свойства.

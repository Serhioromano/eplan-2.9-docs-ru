# Вкладка Исключения при сверлении (экспорт ЧУ для всех машин)

* Параметры > Настройки > Фирма > Механическая обработка > "Формат экспорта". Выберите вкладку Исключения при сверлении.
* Сервисные программы > Данные изготовления > Механическая обработка > "Формат экспорта". Рядом с полем Машина нажмите ++...++. Выберите вкладку Исключения при сверлении.

На этой вкладке задайте определение исключений при сверлении. Для определения нового исключения при сверлении укажите в таблице, какое исходное отверстие следует заменить на другое. На машинах, обладающих подпрограммами со специальными рабочими шагами (например, циркулярным фрезерованием), эти программы также можно использовать в качестве заменяющего отверстия.

Обзор основных элементов диалогового окна:

### Тип сверления

В раскрывающемся списке представлены на выбор доступные типы сверления исходного отверстия:

* Не определено
* Отверстие
* Резьба
* Прямоугольник
* Продольн. отверстие
* Шестиугольник
* Восьмиугольник
* Контур.

### Подтип

Если по умолчанию выбран тип сверления 'Прямоугольник', можно выбрать один из следующих подтипов:

* Углов.
* Со снятой фаской
* Закруглен.

### 1-е измерение / 2-е измерение / 3-е измерение

Размер исходного отверстия в зависимости от выбранного типа сверления.

Тип сверления |  1-е измерение |  2-е измерение |  3-е измерение
---|---|---|---
Сверлить |  Диаметр |  - |  -
Резьба |  Диаметр |  - |  -
Прямоугольный (подтип Многогранный) |  Высота |  Ширина |  -
Прямоугольный (Подтип С фаской) |  Высота |  Ширина |  Ширина фаски
Прямоугольный (Подтип Закругленный) |  Высота |  Ширина |  Радиус закругления
Продольн. отверстие |  Высота |  Ширина |  -
Шестиугольник |  длина ребра |  - |  -
Восьмиугольник |  длина ребра |  - |  -

### Название контура

Имя файла пользовательского контура исходного отверстия.

### Замена типа сверления

В раскрывающемся списке представлены на выбор доступные типы сверления заменяющего отверстия:

* Не определено
* Отверстие
* Резьба
* Прямоугольник
* Продольн. отверстие
* Шестиугольник
* Восьмиугольник
* Контур
* Замена программы ЧУ.

### Замена подтипа

Если по умолчанию выбран тип 'Прямоугольник', можно выбрать один из следующих подтипов:

* Углов.
* Со снятой фаской
* Закруглен.

### Замена 1-го измерения /Замена 2-го измерения / Замена 3-го измерения

Размер заменяющего отверстия в зависимости от выбранного типа сверления.

### Замена названия контура

Имя файла пользовательского контура заменяющего отверстия.

### Замена программы ЧУ

Имя подпрограммы на машине ЧУ, которая должна выполнить соответствующее отверстие (например, циркулярное фрезерование на машинах Steinhauer).

***Резьба, выполненная при помощи циркулярной фрезы,*** фигурирует на машине в качестве программы ЧУ, ее можно вызвать в таком качестве с указанием координат в файле ЧПУ. Отверстие под резьбу уже создано в этой подпрограмме ЧУ. В определении машины на EPLAN Pro Panel можно задавать циркулярное фрезерование с соответствующим номером программы ЧУ.

!!! example "Пример:"

    Для того, чтобы настроить подпрограммы для циркулярного фрезерования (например, M20) на машине Steinhauer, на вкладке Исключения при сверлениинеобходимо установить следующиенастройки:Тип сверления: "Резьба"1-е измерение: (диаметр) "20" мм Замена типа сверления: "Замена программы ЧУ"Замена программы ЧУ: Имя соответствующей подпрограммы машины; например, для M20 следует присвоить программу "1120".

!!! note "Замечание:"

    * EPLAN не может проверить, знает ли машина указанную программу и изготавливает ли она соответствующий выброс. Поэтому эту возможность определения необходимо использовать с особой осторожностью. Необходимо иметь точное представление о машине и ее программах.
    * При помощи данного метода можно заменять не только резьбу, но и другие типы выброса, для которых на машине имеются программы.

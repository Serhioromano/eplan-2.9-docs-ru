# Исключения при сверлении: Принцип

Генерирование отверстия необходимого типа во время машинного изготовления зависит от используемой машины. Так, например, лазер может создать точный шестиугольник, а фреза нет. В данных изделия невозможно установить однозначные настройки, так как неизвестно, какая машина будет применяться для изготовления.

Для каждой машины, вывода DXF и шаблона сверления в настройках можно определить специальные исключения при сверлении. При этом отверстия, которые машина не может выполнить, заменяются другим типом, доступным для машины.

При экспорте данных ЧУ для сверления вместо оригиналов экспортируются альтернативы. После исключения на реальных машинах дополнительно устанавливаются альтернативные способы обработки для отсутствующих инструментов (отсутствует резьбонарезной инструмент, сверление только отверстий под резьбу и т. д.).

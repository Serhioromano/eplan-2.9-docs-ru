# Символы: Конструкция и идентифицирующие свойства

У символов есть графика, выводы устройства и заполнители (с форматированием и позицией) для обозначения устройства, обозначения вывода устройства, описания вывода устройства, перекрестной ссылки и других свойств. При этом логика символа сохраняется в определении функции, символ ссылается на неё.

Идентифицируются следующие свойства символов:

* Номер символа: номер символа определяет позицию символа в пределах библиотеки символов. Этот номер используется только для внутренних, связанных с отчетом потребностей и однозначен для соответствующей библиотеки символов. На символы ссылка идёт через номера символов внутри.
* Имя символа: имя символа, которое должно начинаться с буквенного обозначения ОУ, позволяет осуществить простую идентификацию символа и, кроме этого, может использоваться как критерий поиска при выборе символа.
* Обозначение вывода устройства: обозначение вывода устройства — это непрерывный номер вывода устройства символа. Таким образом может однозначно идентифицироваться каждый вывод устройства. При этом логические данные вывода устройства находятся в определении функции. если, например, создаётся переключающий контакт, все три вывода устройства получают номера "1", "2" и "3". В таблице присвоений определения функции указываются в графе "Переключающий контакт" соответствующие логические данные для функции и выводов устройства Из стандартного символа исходит то, какой вывод устройства является "Корнем" переключающего контакта.
* Описание вывода устройства: описание вывода устройства не предназначено для идентификации символа, т. к. оно содержит только дополнительное описание. Оно может многократно там встретиться.

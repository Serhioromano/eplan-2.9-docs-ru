# Диалоговое окно Цвета соединений / настройки: Цвета соединений

Вы открыли проект.

* Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Цвета соединений.
* Вы открыли диалоговое окно "Свойства" для соединения, точки определения соединения или точки определения потенциала (например, дважды щелкнув по точке определения соединения в графическом редакторе). На вкладке Соединение, Точка определения соединения или Определение потенциала рядом с полем Цвет/номер нажмите на ++...++.

В данном диалоговом окне напротив имен цветов приводятся коды цветов на исходном языке. Данная таблица используется для того, чтобы заполнить свойство Цвет соединения (многоязычный); она также может быть автоматически переведена.
Дополнительно можно задать код цвета маршрутизируемых соединений в пространстве листа.

Обзор основных элементов диалогового окна:

### Код цвета

В этом поле отображаются коды цветов согласно IEC 60757.

### Имя цвета

В этом поле отображаются имена цветов на исходном языке. В диалоговом окне Настройки: Цвета соединений их можно изменять вручную или переводить на другие языки. В диалоговом окне Цвета соединений можно выбрать только имеющиеся цвета соединений.

### Цвет в пространстве листа

Цвет маршрутизируемого соединения в пространстве листа указывается с помощью кода в свойстве соединения Цвет / номер.

В этом поле можно присвоить код цвету маршрутизируемого соединения в пространстве листа. Щелкните по ячейке, а затем по ++...++, откроется диалоговое окно [Выбор цвета](mf_d_farbauswahl.md), где вы сможете выбрать другой цвет.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

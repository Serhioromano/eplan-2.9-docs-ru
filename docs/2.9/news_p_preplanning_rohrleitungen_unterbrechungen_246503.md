# Прерывания трубопроводов для функциональной схемы автоматизации

Трубопроводы на функциональной схеме автоматизации согласно стандарту DIN 28000-4:2014-07 отображаются с прерываниями, когда профиль пересекается с другим символом (например, резервуара) или трубопроводом. Чтобы вставить такие прерывания на функциональную схему автоматизации, теперь в библиотеке символов SPECIAL доступен символ соединения PBR.

![](images/benefit.png)Эффект:

С помощью нового символа для прерываний трубопроводов эти прерывания можно отобразить на функциональной схеме автоматизации в соответствии со стандартами.

В диалоговом окне Выбор символа в предварительном просмотре выделите вариант символа прерывания, который подходит для вертикального или горизонтального участка трубопровода. Для требуемого трубопровода щелкните по позициям, между которыми нужно вставить прерывание. Трубопровод между этими двумя позициями не удалится, а будет только прерван графически.

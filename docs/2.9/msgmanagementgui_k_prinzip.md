# Сообщения контрольного прогона: Номера сообщений, классы и категории

Сообщения контрольного прогона выводятся в диалоговом окне управления сообщениями вместе с номером и отмечаются определенной пиктограммой.

### Номера сообщений, классы

Каждое сообщение контрольного прогона идентифицируется с помощью однозначного номера. Кроме того, номер сообщения информирует о классе и области, для которых выведено данное сообщение. Номер сообщения состоит из шестизначной символьной строки: Первые три цифры обозначают класс, , например, клеммы, штекеры, ПЛК и т. д. Последние три цифры однозначно идентифицируют выведенное сообщение в пределах класса.

!!! example "Пример:"

    В случае номера сообщения "002004" последовательность цифр "002", что это сообщение присвоено классу "Штекеры". Последовательность цифр "004" указывает на ошибку; в данном случае был обнаружен штекер без контактов.

### Категории сообщений

Сообщения контрольного прогона подразделяются на категории в соответствии с серьезностью обнаруженной проблемы. Каждая категория обозначается определенной пиктограммой. Соответствие пиктограмм и категорий сообщений:

![](images/erroricon_error_systemmessages.png){: .ui-icon } Ошибка

![](images/warningicon_systemmessages.png){: .ui-icon } Предупреждение

![](images/infoicon_systemmessages.png){: .ui-icon } Замечание.

Кроме того, можно самостоятельно определять категории сообщений контрольного прогона. Эти настройки сохраняются в виде схемы для определенного проекта. Тем самым гарантируется сохранность настроек для категорий сообщений при передаче проекта.

### Язык отображения

Язык сообщений зависит от настроек и соответствует выбранному языку просмотра. Дополнительный перевод или повторное генерирование сообщений не требуется.

Если, например, изменить язык диалогового окна с немецкого на английский и открыть ранее обработанный проект, все имеющиеся сообщения тоже будут выведены на английском языке.

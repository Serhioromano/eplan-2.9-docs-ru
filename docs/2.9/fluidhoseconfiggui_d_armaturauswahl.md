# Диалоговое окно "Выбрать арматуру"

Открыто диалоговое окно [Обработать ++...++](fluidhoseconfiggui_d_schlauchleitungenbearbeiten.md). Щелкните в поле Арматура слева или Арматура справа по ++...++.

В этом диалоговом окне выберите арматуру, которую необходимо монтировать на шланг слева или справа. Если в настройках вы выбрали по умолчанию настроенную схему DIN 20066_2012-01, вам будут предложены на выбор типы конструкций и серии, представленные в стандарте DIN 20066, с соответствующей графикой предварительного просмотра.

Обзор основных элементов диалогового окна:

### Серия

В раскрывающемся списке можно ограничить отображение типов конструкция по сериям.

### Предв. просмотр

Для каждой серии при наличии выводится окно предварительного просмотра в виде матрицы картинок 2 x 4 с графикой соответствующих типов конструкции.

Чтобы в окне предварительного просмотра появилась графика для данного типа конструкции, в настройках должен быть сохранен соответствующий графический файл типа "*.png" или "*.jpg". По умолчанию сохраненные графические файлы представляют собой анимацию с расширением "*.png". Переместите указатель мыши в окно предварительного просмотра или щелкните по арматуре, запустится анимация изображенной арматуры.

Чтобы выбрать арматуру, щелкните двойным щелчком по окну предварительного просмотра с нужным типом конструкции.

### Кнопки

Если выбранная серия включает более восьми видов арматуры, они отображаются на новых страницах диалогового окна. Кнопки со стрелками позволяют пролистывать страницы выбора арматуры.

Справа рядом с кнопками показано, на какой странице выбора вы находитесь в данный момент и сколько страниц с арматурой всего имеется.

Другие возможности навигации и выбора

Перейти с одной страницы выбора арматуры на другую и выбрать нужную арматуру можно также с помощью клавиатуры. Для этого воспользуйтесь курсором, управляющими клавишами картинок, а также кнопкой "Enter".

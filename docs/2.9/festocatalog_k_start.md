# Каталог продуктов Festo

Если каталог продуктов Festo установлен, данные изделий из этого каталога можно импортировать и использовать в приложении EPLAN. Возможно следующее:

* Изделия из каталога продуктов Festo можно импортировать в собственную базу данных изделий EPLAN.
* Кроме этого, через интерфейс API можно получить прямой доступ к каталогу продуктов Festo и выбрать в нем изделия.
* В процессе импорта происходит присвоение изделия Festo к макросу EPLAN.
* Настройки импорта конфигурируемы, то есть присвоение данных изделий пользователь можно настроить в соответствии со своими потребностями.
* Список устройств расширяется и заполняется данными каталога продуктов Festo.
* Импортированные изделия Festo используются в EPLAN посредством размещения макросов или при работе с устройствами.
* Гиперссылки в данных изделий позволяют обращаться к Web-узлу Festo, на котором содержатся текущая документация и технические данные изделия. Таким образом, при замене номера изделия соответствующая документация и технические данные настраиваются автоматически.

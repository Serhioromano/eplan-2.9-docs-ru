# Устройства в макросе/добавить копии с неизмененным ОУ

При вставке макросов и вырезанных или скопированных устройств можно пронумеровать эти элементы схемы соединений как в режиме подключения, так и в автономном режиме.

Условие:

Вы открыли проект.

### Добавить нумерацию перед вставкой

1. Параметры > Настройки > Проекты > "имя проекта" > Устройство > Нумерация (online)
2. Выберите в групповом поле При вставке макросов и копий опцию Не изменять.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Все устройства, вставляемые с помощью макросов или в виде копий в схему соединений, остаются без изменений. Сохраняется первоначальное ОУ.

### Добавить нумерацию при вставке

Если вы активируете в общих настройках диалоговое окно запроса на нумерацию, каждый раз при вставке будет возможность задавать нумерацию устройств в макросе или копиях заново.

1. Параметры > Настройки > Проекты > "имя проекта" > Устройство > Нумерация (online)
2. Выберите в групповом поле При вставке макросов и копий опцию C запросом.
3. Щелкните по кнопке ++OK++.
4. Чтобы добавить копию из буфера обмена на страницу схемы соединений, выберите пункты меню Обработка > Вставка.
5. Чтобы добавить макрос, выберите пункты меню Вставить > Макрос окна / символа, затем выберите в диалоговом окне макрос и щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Теперь вы можете перемещать макрос/копию при помощи курсора.

6. Переместите курсор в нужное место на странице схемы.
7. Для размещения копии/макроса щелкните страницу.

!!! info "Для сведения:"

    Откроется диалоговое окно "Режим вставки".

8. Выберите параметр Не изменять.
9. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Макрос/копия будут размещены. ОУ останется неизменным.

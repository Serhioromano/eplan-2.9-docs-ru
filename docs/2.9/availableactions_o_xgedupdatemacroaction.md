# Операция: XGedUpdateMacroAction

Эта операция обновляет макросы. С помощью параметра ProjectName можно скопировать весь путь файла проекта. Если заданный проект не открыт, при операции он автоматически открывается и снова закрывается. Если проект не задан, операция выполняется для выделенного в графическом редакторе объекта. В этом случае, чтобы обновить макрос, выбранным объектом должна быть рамка макроса или присвоенный ей объект.

**Параметр** |  **Описание**
---|---
ProjectName  |  Имя проекта с полным путем файла (является опцией). Если заданный проект не открыт, при операции он автоматически открывается и снова закрывается.
AutoAssignLastUsedRecord  |  Автоматически присваивает последний использованный набор значений нескольким заполнителям (не обязательно).

!!! example "Пример:"

    XGedUpdateMacroAction
    /ProjectName:"C:\myFolder\MyProject.elk"

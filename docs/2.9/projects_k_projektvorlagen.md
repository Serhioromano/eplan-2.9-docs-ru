# Шаблоны проектов

Шаблоны проектов — шаблоны, из которых можно создавать новые проекты. При этом все данные шаблона проекта копируются в новый проект.

Тип файла |  Значение
---|---
Шаблон проекта EPLAN (*.ept) |  ***Шаблон проекта / файл *.ept:*** Из файла *.ept можно создавать новые проекты; также можно сохранять проекты EPLAN как файлы *.ept.
Шаблон проекта EPLAN (*.epb) |  ***Шаблон проекта / файл *.epb:*** Далее из файла *.epb можно создать новый проект. Но создать из проекта EPLAN файл *.epb невозможно. Для этого используйте файл *.ept или *.zw9.

Шаблон проекта может содержать следующие настройки проекта и данные проекта.

* ***Все настройки проекта:*** Настройки проекта — это настройки в диалоговом окне Параметры > Настройки > Проекты > "Имя проекта". Сюда относятся также и схемы структуры страниц и устройств.
* ***Все данные проекта:*** Данные проекта, например неразмещенные и размещенные на страницах устройства.
* ***Все страницы:*** Копируются все страницы базового проекта, находящиеся в шаблоне проекта.

!!! tip "Совет:"

    Если необходимо из проекта, содержащего страницы, создать шаблон проекта, не содержащий страницы, скопируйте проект. Удалите страницы из копии проекта и создайте шаблон проекта.

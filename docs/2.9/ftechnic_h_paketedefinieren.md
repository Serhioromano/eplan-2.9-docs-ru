# Вставить группы шланговых соединений, трубопроводы или шлангопроводы

Группы соединений, трубо- или шлангопроводы можно вставлять в схему соединений в качестве устройств.

Условия:

* Вы открыли проект.
* Вы открыли страницу схемы соединений Fluid, на которой находятся параллельно расположенные соединения Fluid.

### Вставить группу соединений

Группа соединений вставляется в схему соединений в качестве ***определения кабеля***.

1. Выберите пункт меню Вставить > Определение кабеля.

!!! info "Для сведения:"

    Символ для определения кабеля прикреплен к курсору.

2. Протяните линию через соединения Fluid, которые должны войти в группу соединений. Для установки начальной и конечной точки линии используйте левую кнопку мыши.

!!! info "Для сведения:"

    В каждой точке пересечения обозначенной линии с линиями автоматического соединения соединений Fluid-техники автоматически размещается точка определения соединения.

!!! info "Для сведения:"

    Открывается диалоговое окно Свойства для группы соединений. Во вкладке Группа соединений / провод в поле Видимое ОУ автоматически записывается обозначение устройства. ОУ соответствует предварительно установленному формату нумерации.

!!! info "Для сведения:"

    Так как линия определения проходит на странице типа "Схема соединений Fluid-техники (I)", во вкладке Данные символа / функции линия отображается в качестве ***линии определения группы соединений***. Кроме того, определению функции группы соединений автоматически присваивается значение ***определения группы соединений***.

3. Дополните во вкладке Группа соединений / провод значениями для еще незаполненных полей Тип, Функциональный текст, Число соединений / шлангов, Длина, Поперечное сечение / диаметр и Единица измерения и определите при необходимости другие свойства.
4. Щелкните по кнопке ++OK++.

#### Изменить символ точек определения соединения

При вставке линии определения группы соединений для точек определения соединения по умолчанию используется символ 311 / CDPNG. Этот символ не содержит графики. Чтобы с помощью графики шлангов отобразить стандартный символ для точек определения соединения согласно DIN ISO 1219 и присвоить правильное определение функции, необходимо изменить символ:

1. Для начала выделите все автоматически размещенные на линии определения группы соединений точки определения соединения и выберите пункт всплывающего меню Свойства.
2. Выведите на передний план вкладку Данные символа / функции.
3. Щелкните в групповом поле Символ (графика) рядом с полем Номер / имя по кнопке ++...++.
4. Выберите в диалоговом окне выбора символа представление в виде списка и введите в поле Прямой ввод значение CDPCP2F5.

!!! info "Для сведения:"

    Символ шланга отображается в режиме предварительного просмотра.

5. Отметьте в режиме предварительного просмотра требуемый вариант символа.
6. Закройте диалоговые окна Выбор символа и Свойства, нажав ++OK++.

!!! info "Для сведения:"

    Теперь все точки определения соединения в группе соединений представлены в качестве шланга.

!!! info "Для сведения:"

    Все точки определения соединения и соединения в группе соединений получают определение функции "Шланг".

#### Скрыть линию определения группы соединений

Так как линия определения группы соединений еще разделяет измененные точки определения соединения с символами шлангов, ее необходимо скрыть.

1. Вызовите диалоговое окно Свойства линии определения группы соединений.
2. Переместите вкладку Формат на передний план.
3. Выберите на уровне иерархии Формат настройку "Да" для свойства Невидимый.

!!! info "Для сведения:"

    Теперь линия определения группы соединений больше не отображается.

### Вставить трубо- и шлангопроводы

Трубо- и шлангопроводы также можно вставлять в схему соединений в качестве устройств. Для этой цели в EPLAN Fluid имеются готовые макросы символов. Для трубопроводов это макросы символов от Pipe1.ems до Pipe5.ems, для шлангопроводов это макрос символа — Hose.ems.

1. Выберите пункт меню Вставить > Макрос окна / символа.
2. Выберите в диалоговом окне Выбрать макрос нужный макрос символа для трубо- или шлангопроводов и нажмите ++Открыть++.

!!! info "Для сведения:"

    Макрос символа прикрепляется к курсору.

3. Разместите символ на нужном соединении между двумя устройствами или символами соединения Fluid.
4. В случае соответствующей предварительной настройки укажите в диалоговом окне Модус вставки, как следует нумеровать вновь вставленное устройство, и закройте диалоговое окно, нажав ++OK++.

!!! info "Для сведения:"

    Вставляется макрос символа. При этом линия автоматического соединения прерывается, и соединение проводится через символ.

5. Вставьте другие макросы символов или прервите процесс вставки, нажав ++Esc++.
6. Задайте в диалоговом окне "Свойства" размещенных трубо- или шлангопроводов обозначение устройства, если оно еще не присвоено автоматически, и определите при необходимости другие свойства.

!!! tip "Совет:"

    Для быстрого и удобного определения и изготовления гидравлических шлангопроводов используйте конфигуратор шлангопроводов [EPLAN Fluid Hose Configurator](fluidhoseconfiggui_k_start.md). Эта программа уже интегрирована как дополнительный модуль, если был установлен EPLAN Fluid. Кроме того, программа доступна также как самостоятельный вариант.

# Вкладка Схема соединений

Сервисные программы > Изделие > Управление. В дереве или списке выделен объект, относящийся к главной группе продуктов Схема соединений.

На этой вкладке определяются стандартные значения для выводов устройств в схеме соединений. На вкладке Технические данные изделий можно использовать сохраненные здесь данные.

Обзор основных элементов диалогового окна:

### Имя

При выборе схемы соединений на вкладке Технические данные изделия это имя и необязательное описание отображаются в диалоговом окне выбора.

Групповое поле Стандартное подключение

### Категория соединения

Из этого раскрывающегося списка выберите значение по умолчанию, которое определяет тип вывода устройства соединений. Нажмите ++...++, чтобы в следующем диалоговом окне задать собственные значения и записать в базу данных. На вкладке Выводы устройства можно определить отдельные выводы устройств и задать значения, отклоняющиеся от них.
На вкладке Схема соединений размещений изделий, использующих эту схему соединений, осуществляется доступ к введенному здесь стандартному значению, если на схеме соединений не определены выводы устройств.

При создании новых схем соединений мы рекомендуем использовать только те версии соединений, которые соответствуют [классификации категории соединения eCl@ss](partsmanagementgui_d_einstanschlussausf.md).

### Размер присоединения

В этом поле можно указать размер винта по умолчанию для винтовых подключений, видов резьбы, ниппелей и штекерных выводов (например, 'M6') или размер накладки (например, '4,8x0,5'). На вкладке Выводы устройства для отдельных выводов устройства можно задать отклоняющиеся от этого значения.

На вкладке Схема соединений размещений изделий, использующих эту схему соединений, осуществляется доступ к введенным здесь значениям, если на схеме соединений не определены выводы устройств.

### Обработка концов проводов (EPLAN Cabinet)

Это свойство используется только в целях совместимости с EPLAN Cabinet. Из раскрывающегося списка выберите требуемый способ обработки концов соединений, например 'снять изоляцию' или 'обжать'.

### Дополнительная длина

Введите здесь значение по умолчанию дополнительной длины расположения вывода устройства. На вкладке Выводы устройства для отдельных выводов устройства можно задать отклоняющиеся от этого значения.

### Направление подсоединения

Выберите из этого раскрывающегося списка значение по умолчанию для направления вывода устройства. На вкладке Выводы устройства для отдельных выводов устройства можно задать отклоняющиеся от этого значения.

# Операция: selectionset

**Параметр** |  **Описание**
---|---
TYPE  |  Вид выполняемой задачи:
PROJECT: Возвращает выбранный проект
PAGES: Возвращает выбранные страницы

!!! note "Замечание:"

    Результат выбора возвращается в вызываемом контексте:

* Если выбран проект:
    Имя параметра в вызываемом контексте = PROJECT. Значение из вызываемого контекста = имя проекта с полным путем файла и расширением имени файла (например, D:\MyProjects\Reactor.elk).
* Если выбраны страницы:
    Имя параметра в вызываемом контексте = PAGES. Значение из вызываемого контекста = все имена страниц разделены точкой с запятой (например, Page1;Page2;Page3).

!!! example "Пример:"

    Возвращает текущие выбранныепроекты:selectionset
    /TYPE:PROJECTРезультаты в вызываемом всплывающем меню: имя параметра = 'PROJECT'Значение = 'C:\Projects\EPLAN\ESS_Sample_Project.elk'Возвращает текущие выбранные страницы:selectionset
    /TYPE:PAGESРезультаты в вызываемом всплывающем меню: имя параметра = 'PAGES'Значение = '=EB3+ET1/1;=EB3+ET1/2;=EB3+ET1/5'

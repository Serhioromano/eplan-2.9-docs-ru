# Импортировать графические 3D-данные

Для создания собственных функциональных элементов и 3D-макросов графические 3D-данные можно импортировать из внешних CAD-систем. Графические данные должны быть доступны в международном формате STEP (***ST***andard for the ***E***xchange of ***P***roduct model data).

!!! note "Замечание:"

    EPLAN поддерживает исключительно импорт файлов STEP, сгенерированных с помощью прикладного протокола AP 203 или AP 214.

При импортировании генерируется новое пространство листа, содержащее имя импортированного файла STEP. После импорта графические данные можно обработать и назначить им функциональную логику.

Условие:

Вы открыли проект. Открыт навигатор пространства листа.

1. Выберите следующие пункты меню: Пространство листа > Импортировать (трехмерная графика).
2. В диалоговом окне Открыть выберите требуемый файл STEP.
3. Щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Трехмерные графические данные будут импортированы в новое пространство листа с описанием из имени файла STEP. Определения функций для отдельных трехмерных объектов будут при этом сгенерированы автоматически.

4. Импортированную графику теперь можно обработать и назначить ей функциональную логику, чтобы использовать ее в качестве функционального элемента шкафа, устройства или 3D-макроса. Выберите для этого пункты меню Обработать > Графика > Повернуть вокруг оси, Обработать > Графика > Объединить и Обработать > Логика устройства > Монтажная поверхность / Точка захвата / Определить точку монтажа / Поверхность установки > Определить.

!!! note "Замечание:"

    Наряду со свойствами изделия Ширина, Высота и Глубина EPLAN Pro Panel дает также возможность наглядно отображать устройства и функциональные элементы при помощи специализированного трехмерного графического макроса.

    Для создания собственных устройств, функциональных элементов и трехмерных графических макросов данные трехмерной модели в формате STEP можно импортировать в EPLAN Pro Panel. Эти данные, в основном предлагаемые производителями компонентов, как правило, основываются на данных конструкции CAD производителя компонентов и часто бывают сложными и подробными.

    Работа со сложными и подробными трехмерными моделями в качестве основы для трехмерного графического макроса EPLAN предполагает высокую нагрузку на оперативную память и может отрицательно сказаться на работе программы при обработке проектов в EPLAN.

    Для получения дополнительной информации и рекомендаций по данной теме обратитесь в службу поддержки EPLAN.

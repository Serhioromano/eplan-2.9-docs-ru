# Поперечное сечение / диаметр соединения для объектов планирования трубопроводов

Аналогично точкам определения трубопроводов для объектов планирования трубопроводов теперь также доступно свойство Поперечное сечение / диаметр соединения (ид. 44085).

![](images/benefit.png)Эффект:

С помощью нового свойства в объектах планирования трубопроводов поперечное сечение / диаметр соединения можно определить уже в предварительном планировании и передать трубопроводам на функциональной схеме автоматизации. Это позволяет избежать избыточного ввода данных.

Если для этого свойства введено значение, то это значение при размещении объекта планирования трубопровода на функциональной схеме автоматизации или во время присвоения переносится на точку определения трубопровода. Измененные значения на объекте планирования (трубопровод) с помощью пункта всплывающего меню Обновить детальное планирование можно перенести на размещенные точки определения трубопровода.

Единица измерения поперечного сечения / диаметра соединения применяется из предварительных настроек технологических соединений для всего проекта (путь меню: Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Свойства; вкладка Технология производственных процессов). Чтобы на точках определения трубопровода отобразить значения вместе с единицей измерения, используйте известное свойство Поперечное сечение соединения / диаметр с единицей измерения (ид. 31007).

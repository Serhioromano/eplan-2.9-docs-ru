# Перенести данные устройства из обновленных изделий в проект

Изделия и другие основные данные постоянно обновляются и становятся доступными, например, через EPLAN Data Portal. Если в прошлом данные изделий, сохраненных в проекте, изменялись в базе данных изделий, то эти данные не переносились на главные функции при синхронизации данных изделий. Чтобы перенести в проект самую актуальную информацию, мы изменили поведение при сравнении синхронизации данных изделий (и схожие функции). Если изделия в проекте старее, чем в базе данных изделий, то в проект теперь включаются не только более новые изделия, но еще и переносятся более новые данные устройства на главные функции, которые используют эти изделия.

![](images/benefit.png)Эффект:

Измененная синхронизация данных изделий теперь позволяет перенести данные устройства из обновленных изделий на главные функции в проекте.

Это изменение поведения влияет на такие функции:

* Синхронизировать текущий проект
* Обновить текущий проект
* Дополнить текущий проект
* Обновить сохраненные изделия при открытии проекта
* Синхронизация данных после изменения изделия

При обновлении или пополнении проекта данные устройства обновляются для ***всех*** используемых изделий, а при синхронизации данных изделий обновляются только данные устройства для ***выбранного*** изделия. При этом соответствующие номера изделий и данные ссылки изделий остаются неизменными.

Для этих функций в списке отмены теперь отмечены два этапа обработки: "Синхронизация данных изделия" и "Обновить данные устройства". Это позволяет при необходимости отменить изменения для схемы соединений.

Выбор изделия и устройства — обновленное изделие

При выборе изделия и устройства для изделия, состояние которого в проекте более старое, чем в базе данных изделий, обновление данных устройства происходит на соответствующей главной функции. Однако, поскольку данные устройства не могут быть перенесены на все варианты использования изделия в проекте (например, если данные шаблона были изменены), в этом случае отображается запрос. Вы можете решить, нужно ли выбрать это обновленное изделие или другое изделие.

Чтобы в таком случае иметь возможность обновлять данные устройства на функциях в проекте, в навигаторе спецификаций доступен новый пункт всплывающего меню Обновить данные устройства.

# Постоянно отображать направление потока трассы трубопровода

Указанные на функциональной схеме автоматизации трубопроводы можно временно выделить цветом посредством пункта меню Трасса трубопровода. При этом также отображается направление потока. В новой версии теперь можно постоянно отображать направление потока трубопроводов, а также распечатывать его. Для этого в библиотеке символов SPECIAL доступны два новых специальных символа (AR1 // 302, AR2 // 303).

![](images/benefit.png)Эффект:

С помощью новых символов для направления потока трассу трубопровода на функциональной схеме автоматизации можно постоянно отображать и распечатывать.

В диалоговом окне Выбор символа выберите один из этих символов и разместите его на линии автоматического соединения соответствующего трубопровода. Чтобы указатель в виде стрелки показывал в правильном направлении, выбранный вариант символа при необходимости нужно привести в соответствие с трассой трубопровода при помощи функций Обновить детальное планирование или Трасса трубопровода.

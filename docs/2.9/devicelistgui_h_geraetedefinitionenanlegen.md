# Создать определения устройств

Определение устройства состоит из набора шаблонов функций. Эти шаблоны задаются в базе данных изделий.

1. Сервисные программы > Изделие > Управление
2. Просмотрите структуру дерева и выделите изделие.
3. На вкладке Шаблоны функции укажите шаблон функции для каждой функции устройства. Для этого щелкните кнопку ![](images/all_new_as.png){: .ui-icon } (Создать).

!!! info "Для сведения:"

    В таблице генерируется новая строка.

4. В первой ячейке откройте при помощи ++...++ выбор функции и выберите в нем определение функции.
5. Щелкните по кнопке ++OK++.
6. В других ячейках вкладки Шаблоны функций введите технические данные функции, где изделию можно присвоить символ в поле Символ.

!!! info "Для сведения:"

    Символ размещается, если отсутствуют шаблоны функций, т. е. если таблица пуста.

!!! info "Для сведения:"

    При наличии шаблонов функций введенные там данные не учитываются, но их можно использовать для фильтрации.

7. Щелкните по кнопке ++Применить++.

!!! info "Для сведения:"

    Определение устройства сохраняется у изделия.

# Импортировать данные устройств и устройства

Данные устройств импортируются из файлов с расширением имени файла TXT, CSV, ASC и из файлов Excel (*.XLS, *.XLSX, *.XLSB, *.XLSM) в открытый или выделенный в навигаторе страниц проект EPLAN.

Условия:

* Вы открыли проект.
* Данные устройств из внешней программы доступны в форме текстового файла или файла Excel.

1. Выберите пункты меню Данные проекта > Устройства > Импортировать.
2. Выберите в диалоговом окне Импортировать данные устройств в поле Тип источника данных, из какого типа файла ("Текстовый" или "Excel") следует импортировать данные устройств.
3. Рядом с полем Источник данных щелкните мышкой по кнопке ++...++.

4. В диалоговом окне [Выбрать источник данных](projectprocessinggui_d_datenquelleauswaehlen.md) выберите файл, из которого должен осуществляться импорт, и задайте необходимые настройки.
5. Щелкните по кнопке ++OK++.
6. В диалоговом окне Импортировать данные устройств рядом с полем Присвоение полей щелкните мышкой по кнопке ++...++.

7. В диалоговом окне Присвоение полей в столбце Свойство укажите, какие свойства EPLAN должны быть присвоены обозначениям внешних полей данных, перечисленным в столбце Внешнее обозначение поля данных.
8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В диалоговом окне Синхронизировать устройства показаны данные устройств из источника данных и их присвоение устройствам в проекте EPLAN. При этом проверяется, имеются ли устройства уже в проекте или еще нет, а также имеют ли устройства в источнике данных и в проекте различные состояния обработки (например, другие значения свойств), а результаты проверки отображаются в столбце Статус. Поля с измененными значениями свойств подкрашены темно-серым цветом.

9. В диалоговом окне Синхронизировать устройства в столбце Действие укажите, что должно происходить с устройствами в зависимости от отображаемого статуса:

Статус |  Действие
---|---
![](images/all_statusnew_as.png){: .ui-icon } |  Создать |  Импортировать или Нет
![](images/all_statusok_as.png){: .ui-icon } |  Идентичны |  Нет
![](images/all_statusdelete_as.png){: .ui-icon } |  Дополнительные |  Удалить или Нет
![](images/all_statusadditional_as.png){: .ui-icon } |  Изменено |  Применить изменение или Нет
10. Для сокращения объема отображения в диалоговом окне Синхронизировать устройства до необходимых устройств можно настроить специфический для раздела фильтр в поле Фильтр.
11. Для отображения импортируемых устройств в соответствии с их текущим статусом выберите статус в раскрывающемся списке в поле Фильтр (статус).
12. Выберите в выделенной строке в диалоговом окне Синхронизировать устройства пункт всплывающего меню Присвоить изделия, чтобы автоматически присвоить импортируемым устройствам имеющийся номер изделия, подходящий для базы данных изделий.
13. Если автоматическое присвоение невозможно, в столбце Номер изделия щелкните мышкой по кнопке ++...++, чтобы открыть выбор изделия и вручную выбрать в нем изделие.
14. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выполняется импорт или выбранные действия.

Статус |  Действие |  Результат
---|---|---
![](images/all_statusnew_as.png){: .ui-icon } |  Создать |  Импортировать |  Устройство импортируется
![](images/all_statusok_as.png){: .ui-icon } |  Идентичны |  Нет |  Устройство остается в проекте
![](images/all_statusdelete_as.png){: .ui-icon } |  Дополнительные |  Удалить |  Устройство удаляется из проекта
Нет |  Устройство остается в проекте
![](images/all_statusadditional_as.png){: .ui-icon } |  Изменено |  Применить изменение |  Применяются измененные значения свойств
Нет |  Измененные значения свойств не применяются

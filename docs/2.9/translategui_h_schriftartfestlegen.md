# Определить виды шрифтов для перевода

Если вы хотите перевести проект на язык, символы которого отсутствуют в стандартных шрифтах вашего компьютера, то следует инсталлировать на ваш компьютер вид шрифта, содержащий эти символы. В настройках EPLAN следует выбрать этот вид шрифта.

Условие:

Вы инсталлировали вид шрифта, содержащий нужные символы.

1. Выберите пункты меню Параметры > Настройки > Фирма > Графическая обработка > Виды шрифта.
2. Из раскрывающегося списка Вид шрифта выберите вид шрифта, содержащий нужные символы.

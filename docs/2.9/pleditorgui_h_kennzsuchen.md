# Поиск и замена идентификаторов

Команда "Найти и заменить" в разделе управления структурными идентификаторами позволяет осуществлять поиск в таблице структурных идентификаторов во вкладке Список по имеющимся данным и заменять их в случае необходимости. При этом поиск осуществляется всегда только в ***одном*** (выделенном) столбце и только в прямом алфавитном порядке. Если выделено все, то поиск осуществляется только в столбце Строка.

!!! note "Замечание:"

    Обратите внимание, что при поиске учитывается только идентифицирующее, а не полное ОУ!

Условие:

Вы выбрали проект, который в структуре проекта использует блоки идентификаторов, например, "Установка" и "Место установки".

1. Данные проекта > Управление структ. идентификаторами
2. В диалоговом окне Управление структурными идентификаторами — <Имя проекта> выберите вкладку Список.
3. Выделите столбцы, в которых должен выполняться поиск, щелкнув по заголовку столбца.
4. Щелкните по ++Дополнительно++ > Поиск.
5. Укажите в диалоговом окне Заменить искомое понятие и текст для замены.
6. Включите флажок Учитывать регистр, если нужно учитывать форму написания текста при поиске и замене.
7. Нажмите кнопку для запуска операции поиска и замены.
8. Если найден недопустимый символ (Настройки в разделе Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Синтаксическая проверка ОУ), примите решение в диалоговом окне Синтаксическая проверка ОУ, как поступить с соответствующим символом.
9. Щелкните по кнопке ++Применить++.
10. Выберите необходимый ответ в запросе.

!!! note "Замечание:"

    * Поиск осуществляется всегда по одному столбцу и только в прямом алфавитном порядке, команда ***Заменить*** ограничена изменяемыми столбцами Обозначение и Описание структуры.
    * По умолчанию значение поля Поиск — это символьная строка текущего выделенного поля, если используется множественный выбор — содержание последнего выбранного поля.

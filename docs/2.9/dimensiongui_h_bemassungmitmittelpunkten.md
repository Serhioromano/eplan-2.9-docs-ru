# Вставить указание размеров посередине между двумя точками

При вставке указаний размеров можно щелчком по двум произвольным точкам, в том числе различным графическим объектам (линиям, многоугольникам и т. д.), определить точку, которая будет располагаться посередине воображаемой соединительной линии между идентифицированными точками. Эта определенная точка служит в качестве начальной и конечной точки указания размера. Рассчитать середину можно при указании размеров для всех фигур за исключением кругового указания размеров. Применять эту функцию целесообразно там, где данная точка отсутствует и ее невозможно определить.

1. Выберите в меню тип указания размеров, например: Вставить > Указание размеров > Отдельное указание размеров.
2. Удерживая клавишу ++Ctrl++, укажите первую точку измерения.

!!! info "Для сведения:"

    При нажатии клавиши ++Ctrl++ и вводе точек измерения происходит определение середины между двумя точками. В графическом редакторе это можно распознать по изменению цвета вспомогательной линии у курсора.

3. Затем щелчком мыши определите конечную точку для расчета середины.

!!! info "Для сведения:"

    Определенная точка будет использоваться в качестве начальной точки для указания размеров.

4. Затем введите вторую точку измерения для указания размеров. Если необходимо определить середину линии / графического объекта, еще раз нажмите клавишу ++Ctrl++ и определите эту точку при помощи двойного щелчка.

!!! info "Для сведения:"

    Указание размеров вставляется между рассчитанной серединой и второй точкой измерения.

5. Завершите операцию, выбрав пункт всплывающего меню Прервать операцию или нажав клавишу ++Esc++.

!!! example "Пример:"

    Фрагмент обзора модели для EPLAN Pro Panel: Отдельное указание размеров между верхним краем электрошкафа и расчетной серединой кругового изображения логического функционального элемента.

# Файл экспорта / маркировки / Внешний файл отчетов

Данные проекта для данных изготовления, маркировок или внешних отчетов выводятся во внешних файлах экспорта, маркировок или отчетов. В файле экспорта / маркировки содержится одна или несколько страниц маркировочных этикеток. Страница содержит нижний и верхний колонтитулs, а также собственно этикетки.

### Верхний колонтитул

Верхний колонтитул находится в начале страницы. Он может быть выведен только один раз или может повторяться на каждой странице.

### Область этикеток

В области этикеток содержится один или несколько рядов этикеток. Ряд этикеток состоит из нескольких этикеток, расположенных рядом друг с другом.

### Нижний колонтитул

Нижний колонтитул находится в конце страницы. Он может быть выведен только один раз или может повторяться на каждой странице.

# Многопользовательский режим

Многопользовательский режим обозначает возможность для нескольких пользователей одновременно обрабатывать проекты на различных рабочих станциях. Все пользователи, принимающие участие в проекте, могут иметь одновременный доступ к данным проекта. Таким образом рабочие процессы могут быть распределены между несколькими пользователями.

В многопользовательском режиме EPLAN у Вас есть следующие возможности:

* Несколько пользователей могут одновременно работать над проектом.
* В управлении проектами Вы можете отобразить, какие другие пользователи обрабатывают выбранный проект.
* если несколько пользователей хотят одновременно обрабатывать одни и те же данные проекта, Вы будете проинформированы EPLAN об этом случае конфликта.
* В случае конфликта Вы можете проинформировать других пользователей, которые обрабатывают те же данные, посредством E-Mail.

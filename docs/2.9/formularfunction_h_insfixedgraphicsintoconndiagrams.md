# Вставить фиксированную графику в схемы подключений

Существуют схемы подключения, на которых, например, на каждую клемму должна представляться одна фиксированная "винтиковая графика" (схема подключения клемм). Или же, в случае схем подключений кабелей, одна фиксированная графика должна показываться для кабельных соединений каждого кабеля. Для этих случаев в распоряжении имеется текст клемм / кабельных соединений / заполнителей 'Присвоение индекса графике'. В данном случае индексом является индекс заполнителя в графике от 1 до 100. В свойствах формы каждому индексу можно присвоить символ. Затем разместите в форме свойство "Присвоение индекса графике [n]", чтобы там всегда размещался именно тот символ, который был присвоен индексу n.

Условия:

* Вы открыли проект.
* В редакторе форм открыта та форма, свойства которой требуется обработать (Сервисные программы > Основные данные > Форма > Открыть).
* Вы вставили текст заполнителя 'Присвоение индекса графике' для соответствующего элемента (клеммы, кабельного соединения, ...).

1. Пометьте форму в навигаторе страниц.
2. Выберите пункт всплывающего меню Свойства.
3. Если свойство "Присвоение индекса графике" в форме не было установлено, в диалоговом окне Свойства формы щелкните ![](images/all_new_as.png){: .ui-icon } (Создать) на панели инструментов над списком свойств, выберите свойство из диалогового окна Выбор свойств и щелкните ++OK++.
4. В диалоговом окне Свойства формы щелкните на ++...++ в столбце Значение свойства 'Присвоение индекса графике'.
5. В диалоговом окне Присвоение индекса графике щелкните ![](images/all_new_as.png){: .ui-icon } (Создать), чтобы генерировать новую строку в таблице присвоений.
6. В поле Имя символа щелкните по кнопке ++...++.
7. В диалоговом окне Выбор символа выберите символ, графику которого необходимо присвоить индексу, и щелкните ++OK++.

!!! info "Для сведения:"

    Соответствующие данные будут применены в полях Библиотека символов и Описание.

8. В поле Вариант символа выберите требуемый вариант из раскрывающегося списка всех имеющихся вариантов текущего символа.
9. Выполните аналогичные действия для всех других присвоений.
10. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Присвоения сохраняются.

В диалоговом окне Свойства формы присвоения выводятся в столбце Значение следующим образом:

<Номер индекса>;<Библиотека символов>;<Номер символа>;<Вариант>|<Номер индекса>;<Библиотека символов>;<Номер символа>;<Вариант>|...

!!! example "Пример:"

    1;SPECIAL;321;0|2;IEC_EPLAN_D;176;1

!!! note "Замечание:"

    Обратите внимание на то, что на просмотр выводятся только символы ***сохраненных*** библиотек символов.

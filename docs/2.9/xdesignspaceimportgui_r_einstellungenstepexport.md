# Вкладка Настройки: Экспорт пространства конструкции

Параметры > Настройки > Пользователь > Интерфейсы > Импорт и экспорт пространства конструкции. Выберите вкладку Экспорт.

На этой вкладке установите настройки для экспорта файлов XML и STEP с пространствами конструкции.

Для каждого изделия, размещенного с помощью EPLAN Pro Panel, при экспорте данных экспортируется файл STEP. Для каждого пространства листа при экспорте данных генерируется файл XML. Файл XML содержит данные всех объектов в пространстве листа, это импортированные из системы CAD объекты, вставленные EPLAN Pro Panel объекты, автоматически генерированное при размещении изделий и вставленные вручную монтажные отверстия.

Файл XML содержит для каждого размещенного изделия в EPLAN Pro Panel ссылку на конкретный файл STEP, который был экспортирован вместе с файлом XML.

Обзор основных элементов диалогового окна:

### Каталог вывода

В этом поле укажите место хранения для экспортируемого файла XML и STEP. В качестве целевых для экспорта данных можно указать любой дисковод и подкаталог. Каталогом по умолчанию является каталог для механической модели $(MD_MECHANICALMODELS).

Нажав ++...++, откройте диалоговое окно Выбрать каталог и перейдите к нужному каталогу или создайте новый.

* * *

Групповое поле файла XML

### Генерировать имена файлов

При настройке имени файла XML существуют следующие возможности:

* Из непрерывного номера
* Из имени пространства листа
* Из свойств

В свойствах доступны свойства пространства листа, свойства проекта и разделитель.

### Трехмерные монтажные данные

Если флажок установлен, трехмерные монтажные данные экспортируются в файл XML.

### Данные ЧУ

Если флажок установлен, данные ЧУ экспортируются в файл XML.

* * *

Групповое поле файла STEP

### Генерировать имена файлов

При настройке имени файла STEP существуют следующие возможности:

* Из непрерывного номера
* Из имени пространства листа
* Из свойств

В свойствах доступны свойства пространства листа, свойства проекта и разделитель.

### Версия формата

Для версий формата имеются следующие возможности:

* STEP AP 203
* STEP AP 214, версия CD
* STEP AP 214, версия DIS
* STEP AP 214, версия IS

### Вспомогательные элементы

Если данный флажок снят, то следующие вспомогательные элементы не экспортируются:

* Сегменты маршрутизации
* Области маршрутизации
* Соединительные отверстия для проводов
* Запретные зоны для сверления
* Запретные зоны для размещения
* Монтажные отверстия

### Маршрутизируемые соединения

Если активирован этот флажок, в ходе экспорта STEP выполняется экспорт маршрутизируемых соединений, имеющихся в пространстве листа. После импорта образованного файла STEP во внешнюю программу CAD в ней могут отображаться маршрутизируемые соединения.

* * *

Групповое поле 'Упрощенное представление'

В данном групповом поле можно настроить упрощенный экспорт функциональных элементов, как при представлении в пространстве листа.

* Клеммники (Определение блока): Если этот флажок установлен, клеммники экспортируются как упрощенный блок.
* 3D-макросы: Если этот флажок установлен, графические макросы, шины и кабельные каналы упрощаются до прямоугольного параллелепипеда.

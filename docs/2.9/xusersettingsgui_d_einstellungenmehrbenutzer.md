# Диалоговое окно Настройки: Ид. пользователя / адрес

Параметры > Настройки > Пользователь > Отображение > Ид. пользователя / адрес

В этом диалоговом окне можно водить данные о пользователе. Вводить "данные пользователя" имеет смысл, например, для многопользовательских лицензий, когда нескольких пользователей могут одновременно обрабатывать проекты на различных рабочих станциях. В случае конфликта между пользователями, например, когда два пользователя одновременно хотят обработать одни и те же данные в EPLAN, то EPLAN обращается к введенным здесь данным и отправляет сообщение с данными о пользователе, который предотвращает обработку данных.

Обзор основных элементов диалогового окна:

### Идентификатор / Имя

Введите в эти поля соответствующие данные о пользователе.

### Имя регистрации

Это поле заполняется EPLAN автоматически: Если управление правами EPLAN ***не*** используется, вносится имя пользователя, под которым вы зарегистрированы в операционной системе Windows. Если управление правами используется, вносится имя, под которым вы зарегистрированы в управлении правами.
В обоих случаях запись нельзя обработать.

### Телефон / Электронная почта

Укажите здесь соответствующий номер с кодом, а также адрес электронной почты.

### Номер клиента

Введите свой номер клиента.

### Для изменений использовать имя регистрации

Если этот флажок активирован, Имя регистрации выдается в автоматически заполненных свойствах, как, например, "Автор" или "Последний обработчик" в проектах, страницах, формах, рамках и т. д.

Если флажок не установлен, Имя регистрации не выдается, вместо этого используются записи полей Идентификатор и Имя. В случае, если поля Идентификатор и Имя пусты, появляется соответствующее сообщение и флажок устанавливается автоматически, т. е. используется имя регистрации.

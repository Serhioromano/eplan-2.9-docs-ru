# Определить свойства соединения посредством точек определения соединения

Точки определения соединений определяют свойства для сквозных соединений, проходящих под ними.

Условие:

Вы открыли проект.В Графическом редакторе открыта страница проекта. Вставлена точка определения соединения.

1. Дважды щелкните по точке определения соединения.
2. В диалоговом окне Свойства ++...++ во вкладке [Точка определения соединения](devicetaggui_r_vdp.md) измените значения предварительно установленных свойств соединения или укажите новые.
3. На вкладке [Отображение](devicetaggui_r_anzeige.md) Вы можете изменить представление свойств на схеме соединений, а на вкладке [Графика соединения](devicetaggui_r_verbindungsgrafik.md) \- графические свойства линий соединения.
4. Во вкладке [Изделие](partselectiongui_r_artikel.md) Вы можете присвоить соединению данные изделия.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Свойства, которые определены, присваиваются только тому соединению, которое пересекает точку определения соединения. (В случае с многократным выбором точек определения соединений Вы можете одновременно изменять свойства нескольких соединений.)

!!! tip "Совет:"

    Во вкладке Точка определения соединения в поле Обозначение соединения можно автоматически определить следующее свободное обозначение соединения. Для этого воспользуйтесь пунктом всплывающего меню Определить новое обозначение или комбинацией клавиш [Ctrl] \+ [N].

!!! note "Замечание:"

    Комплексная обработка свойств соединений возможна только в навигаторе соединений (например, для нескольких видов представления соединения). Комплексная обработка точек определения соединения невозможна — ни для различных видов представления соединения, ни для нескольких точек определения соединения на соединении.

# Диалоговое окно Копировать гиперссылку

Вы открыли страницу в графическом редакторе: Вставить > Графика > Гиперссылка. В диалоговом окне Свойства на вкладке Гиперссылка укажите документ, который нужно связать, и нажмите ++OK++.

В этом диалоговом окне вы указываете, копируется ли документ в каталог проекта или на него предоставляется ссылка из исходного каталога.

Обзор основных элементов диалогового окна:

### Исходный каталог не изменять

В проект вставляется ссылка на документ, и документ отображается из исходного каталога. Изменения в документе будут автоматически обновляться, как только вы откроете страницу. Однако при резервировании данных документы, которые находятся во внешних каталогах и на которые есть ссылки, ***не*** учитываются.

### Копировать

Документ копируется в каталог проекта (то есть в каталог внешних документов проекта), а копируемый документ отображается из каталога проекта. Все файлы в подкаталогах проекта также учитываются при резервировании данных.

### Перезаписать с запросом

Установите этот флажок, чтобы параметр Копировать отображал запрос, если документ уже существует в каталоге проекта. Затем вы можете решить, будет ли он перезаписан или нет.

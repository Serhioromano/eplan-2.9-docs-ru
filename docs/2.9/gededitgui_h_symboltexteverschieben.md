# Переместить тексты свойств

C помощью команды переместить тексты свойств активируются точки ввода текстов свойств условных обозначение и могут становиться видимыми, если щелчкнуть по отдельному условному обозначению. Их можно переместить с помощью перетаскивания мышью.

Кроме того, таким же образом можно вручную переместить ***Образ контакта***.

Условия:

* Вы открыли страницу, форму, рамку или символ.
* Не разрешается присоединять тексты свойств. При необходимости отсоедините тексты (с помощью настройки ![](images/all_undock_as.png){: .ui-icon } (Отсоединить) на вкладке Отображение диалогового окна 'Свойства').

1. Выберите пункты меню Обработать > Текст > Переместить текст свойства.
2. Выделите условные обозначения, тексты свойств которых Вы хотели бы переместить.

!!! info "Для сведения:"

    Точки элементов текстов свойств отображаются как небольшие незаполненные квадраты.

3. Выделите текст свойства и переместите его. Обратите внимание на то, что при этом перемещается случайно установленная указательная линия. (Указательные линии задаются на вкладке Отображение в меню Рамки.)

!!! info "Для сведения:"

    Выделение остается активным до тех пор, пока не будет выделен другой текст свойства или прервана операция.

!!! note "Замечание:"

    Если удерживать нажатой клавишу ++Shift++ при перемещении, текущая сетка будет разделена на "четверти". При этом точку вставки можно перемещать в шагах, которые имеют значение четверти шага сетки.

### Сбросить перемещенные тексты свойств

С помощью опции Сбросить перемещенный текст свойства все перемещенные тексты свойств выделенных символов удаляются. и снова выводятся тексты свойств, сохраненные для символа. (Каждый символ содержит информацию о том, какие тексты свойств по умолчанию отображаются при вставке этого символа.)

1. Выделите требуемые условные обозначения и выберите пункты меню Обработать > Текст > Сбросить перемещенный текст свойства.

!!! info "Для сведения:"

    У всех выделенных условных обозначений будут удалены скопированные тексты свойств и выведены тексты свойств символа, которые отображаются по умолчанию.

# Вставить и обработать тексты заполнители

Тесты-заполнители применяются в формах для того, чтобы определить позиции для размещения полей в одной записи данных. При оценке форм программа автоматически заменяет эти заполнители на соответствующие значения конкретных объектов проекта, для которого выполняется отчет, например, на значения функциональных элементов, страниц, символов и т.д.

Условия:

* Вы открыли проект.
* В редакторе форм открыта форма, в которой необходимо определить тексты заполнители (Сервисные программы > Основные данные > Форма > Открыть > [Открыть]).

1. Вставить > Текст заполнитель
2. В диалоговом окне Свойства (текст заполнитель) выберите параметр Свойство вкладки Размещение и щелкните ++...++.
3. В диалоговом окне Тексты заполнителей — <Тип формы> выберите элемент, для которого необходимо установить текст заполнитель.
4. Выберите это свойство и щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Выбранное свойство копируется в поле Свойство.

5. Перейдите на вкладку Формат и задайте настройки форматирования текста-заполнителя.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Элемент присоединяется к курсору.

7. Позиционируйте текст заполнитель на странице формуляра.

!!! info "Для сведения:"

    Элемент присоединяется к курсору.

8. Позиционируйте текст заполнитель, второй, третий... раз.
9. Для завершения операции выберите пункт всплывающего меню Прервать операцию.

!!! info "Для сведения:"

    Текст заполнитель вставляется на странице формы и при помощи двойного щелчка может быть доработан.

!!! note "Замечание:"

    Если у заполнителя нет блока выравнивания или блок выравнивания есть и одновременно установлен флажок Убрать переходы на новую строку (в свойстве Рамка), то в отчетах переходы на новую строку отфильтровываются.

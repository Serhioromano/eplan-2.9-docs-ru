# Диалоговое окно Настройки: Идентификатор

Параметры > Настройки > Пользователь > Представление > Идентификатор.

В этом диалоговом окне вы устанавливаете, каким образом будут обрабатываться структурные идентификаторы при их создании.

Доступны следующие элементы диалогового окна:

### Сортировать новые идентификаторы по алфавиту

С помощью этой опции вы задаете сортировку новых структурных идентификаторов в алфавитном порядке в соответствующей таблице идентификаторов.

### Вставить новые идентификаторы в конце

Если это опция включена, новые структурные идентификаторы, как правило, добавляются в конец соответствующей таблицы идентификаторов.

!!! tip "Совет:"

    При необходимости в [разделе управления структурными идентификаторами](pleditorgui_d_kennzeichen.md) можно дополнительно задать другую сортировку новых структурных идентификаторов.

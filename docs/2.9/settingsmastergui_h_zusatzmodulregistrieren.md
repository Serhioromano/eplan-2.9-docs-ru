# Зарегистрировать Add-ons вручную / автоматически

### Зарегистрировать Add-on вручную

Зарегистрируйте дополнительный модуль вручную стандартным способом, предварительно сохранив Add-on и соответствующие файлы install.xml в локальном каталоге вашей системы.

1. Выберите следующие пункты меню: Сервисные программы > Add-ons
2. Нажмите в диалоговом окне Add-ons кнопку ![](images/all_new_as.png){: .ui-icon } (Создать) на панели инструментов.
3. Выберите в следующем диалоговом окне соответствующий файл install.xml и нажмите кнопку ++Открыть++.
4. Установите в диалоговом окне Add-ons флажок Зарегистрировано.
5. Подтвердите ввод.

### Настроить автоматическую регистрацию Add-ons

Можно настроить автоматическую установку и регистрацию Add-ons, если ваш администратор сохранил Add-on и соответствующий файл install.xml в каталоге сервера.

1. Выберите следующие пункты меню: Параметры > Настройки > Фирма > Управление > Add-ons
2. Щелкните рядом с полем Путь файла для автоматической регистрации Add-ons по ++...++.
3. В диалоговом окне Выбрать каталог выберите каталог, в котором находится файл install.xml.
4. Подтвердите ввод.

!!! info "Для сведения:"

    При запуске программы EPLAN в следующий раз дополнительный модуль Add-on зарегистрируется автоматически.

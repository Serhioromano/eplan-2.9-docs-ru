# Графическое определение схем сверления

Схемы сверления в проекте макросов можно определять также графически, не используя данные из импортированных файлов ЧУ или DXF\- / DWG. Процедура включает четыре шага:

* черчение линии контура функц. элемента и требуемых отверстий на графической странице,
* присвоение идентифицирующих слоев для генерирования схемы сверления,
* вставка границы схемы сверления,
* автоматическое генерирование схемы сверления.

Условие:

Вы открыли проект макросов.

1. Откройте графическую страницу в масштабе 1:1.
2. Начертите линию контура функц. элемента и требуемые отверстия внутри нее, соблюдая масштаб.
3. Откройте у каждого элемента диалоговое окно "Свойства" и переместите вкладку Формат на передний план. Измените под уровнем иерархии Формат в свойстве Слой слои элементов, предназначенных для схемы сверления, как показано в следующей таблице.

Элемент |  Имя (слой), описание
---|---
Линия контура функционального элемента |  EPLAN802, Графика.Генерирование схемы сверления.Функциональные элементы
Отверстия (все типы сверления) |  EPLAN803, Графика.Генерирование схемы сверления.Отверстия
Отверстия с резьбой |  EPLAN804, Графика.Генерирование схемы сверления.Отверстия с резьбой
Определенные пользователем контуры |  EPLAN805, Графика.Генерирование схемы сверления.Контуры
4. Вставьте границу схемы сверления, которая охватывала бы элементы, предназначенные для схемы сверления. Выберите пункты меню Вставить > Блок / Вывод устройства / Монтажная плата > Граница схемы сверления.

!!! info "Для сведения:"

    Слева внизу на линии контура функц. элемента генерируется нулевая точка.

5. В диалоговом окне "Свойства" границы схемы сверления введите имя, описание и каталог для генерируемых определенных пользователем контуров.
6. Сгенерируйте схемы сверления: Выберите пункты меню Сервисные программы > Генерировать схемы сверления / контуры > Автоматически из проекта макросов.
7. Выберите, должен ли процесс применяться ко всему проекту или только к выделенной границе схемы сверления.

!!! info "Для сведения:"

    Схемы сверления генерируются в базе данных изделий из начерченных объектов, которые присвоены границам схем сверления, в виде сверху.

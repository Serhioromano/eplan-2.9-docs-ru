# Диалоговое окно Открыть контур/Копирование контура

Вы открыли проект.

* Сервисные программы > Основные данные > Контур (...) > Открыть.
* Сервисные программы > Основные данные > Контур (...) > Копировать.

В этом диалоговом окне выберите контур для обработки.

Обзор основных элементов диалогового окна:

### Поиск в

Выберите из раскрывающегося списка требуемые дисковод и каталог, в которых находится открываемый контур.

### Предв. просмотр

Установите этот флажок, чтобы показать в правой области диалогового окна предварительный просмотр графики, выделенной в левом списке контура.

!!! note "Замечание:"

    Многократный выбор контуров здесь ***невозможен***.

### Информационная область

Как только в списке в левой области будет выделен контур, в информационной области под флажком Предв. просмотр показываются (если имеются) следующие свойства:

* Описание (форма, рамка, контур, Профиль)
* Тип контура
* Идентификатор фирмы
* Автор
* Дата создания
* Последний обработчик
* Дата изменения.

### Имя файла

Укажите здесь имя файла или выберите его из раскрывающегося списка.

### Тип файла

В данном поле отображается текущий выбранный тип контура; этот ***не*** может быть изменен.

### Путь

Здесь в поле Поиск в отображается требуемый путь файла (с указанием дисковода). С помощью пункта всплывающего меню Сбросить на стандарт можно автоматически вернуть предварительную настройку пути файла по умолчанию <Дисковод программы>:\<Каталог программы>\Макросы\<Идентификатор фирмы> или настройку для макросов, сделанную в меню Параметры > Настройки > Пользователь > Управление > Каталоги.

Если производится переход в другой каталог, система запоминает новую настройку и предлагает ее при следующем вызове диалогового окна.

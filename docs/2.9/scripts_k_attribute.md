# Атрибуты сценария

В сценарии атрибутами отмечаются выполняемые функции. Следующие атрибуты можно использовать в сценариях:

Атрибут |  Описание
---|---
Start |  При обозначении выполняемой функции используется в сценарии. Когда выполняется сценарий, вызывается эта функция. Эта функция также может иметь параметр.
DeclareAction |  При обозначении выполняемой функции используется в сценарии, если функция сценария в системе должна быть зарегистрирована как операция.
DeclareEventHandler |  Если функция в сценарии должна реагировать на результат в системе, ее необходимо обозначить атрибутом.
DeclareMenu |  При обозначении выполняемой функции используется в сценарии, если функция сценария в системе должна быть вставлена как пункт меню.
DeclareRegister |  Если вставляется новый файл сценария EPLAN, вызывается эта функция.
DeclareUnregister |  Если удаляется новый файл сценария EPLAN, перед удалением эта функция вызывается.

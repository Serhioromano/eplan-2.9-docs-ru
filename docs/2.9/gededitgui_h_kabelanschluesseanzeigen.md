# Отобразить выводы кабеля

Опция Вид > Кабельные соединения упрощает поиск автоматически сгенерированных кабелей, а также соединений, которые учитываются при автоматическом генерировании кабелей. Если этот параметр активирован, выводы функций, при которых активировано свойство Вывод кабеля, обозначается стрелкой. (При автоматическом генерировании кабелей учитываются только те соединения, которые подключены по крайней мере к одному выводу кабеля или определены как кабельное соединение.)

Свойство Вывод кабеля настройте в диалоговом окне [Логическая схема выводов устройства](xfctdefbrowsergui_d_anschlusslogik.md), которое вы вызываете в диалоговом окне свойств во вкладке Данные символа/ функции при помощи кнопки ++Логика++.

Условие:

В Графическом редакторе открыта страница проекта.

1. Выберите пункты меню Вид > Кабельные соединения.

!!! info "Для сведения:"

    Разъемы соединений функций со свойством Вывод кабеля отмечены маленькой стрелкой.

2. Повторно выберите пункты меню Вид > Кабельные соединения.

!!! info "Для сведения:"

    Стрелки для отображения кабельных выводов снова будут скрыты.

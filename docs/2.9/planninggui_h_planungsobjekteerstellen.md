# Создание объектов планирования, технологических контуров, функций ТК, резервуаров и объектов планирования (соединений)

Создание объектов планирования и управление ими осуществляется в навигаторе предварительного планирования. Технологические контуры, функции ТК, резервуары, объекты планирования (соединения) (объекты планирования, трубопроводы; объекты планирования, кабели) создаются так же, как и общие объекты планирования, и управляются параллельно с ними. Какой вид объекта планирования вы создаете, задается через определение сегмента.

!!! note "Замечание:"

    Функции ТК можно создавать только внутри технологических контуров.

При создании технологических контуров и функций ТК заданные в настройках проекта настройки для технологических контуров учитываются в предварительном планировании (через Параметры > Настройки > Проекты > "Имя проекта" > Предварительное планирование > Технологические контуры). В этом диалоговом окне можно, в частности, указать, какие свойства, кроме "Номера", являются идентифицирующими, и в каких слоях установки допустимы идентичные технологические контуры.

1. Выберите следующие пункты меню: Данные проекта > Предварительное планирование > Навигатор
2. Выделите проект, сегмент структуры или уже имеющийся объект планирования и выберите пункт всплывающего меню Новый объект планирования.
3. Выберите в диалоговом окне Выбрать определение сегмента нужное определение сегмента.

!!! info "Для сведения:"

    В зависимости от выделенного уровня иерархии на выбор предлагаются определения сегментов для объектов планирования, технологических контуров, резервуаров, объектов планирования (трубопроводов) или объектов планирования (кабелей). Если вы выделили технологический контур, предлагаются только определения сегментов для функций ТК.

4. Щелкните по кнопке ++OK++.
5. Введите в диалоговом окне Свойства ++...++ нужные данные для нового объекта планирования. Для этого в диалоговом окне "Свойства" сначала вынесите первую вкладку на передний план.

!!! info "Для сведения:"

    В случае с ***технологическими контурами*** и ***функциями ТК*** номер ТК автоматически составляется из введенных данных в групповом поле Обозначение.

6. Для ***объекта планирования (соединения)*** вводятся обозначения сегментов, которые должны представлять источник и цель объекта планирования (соединения). Для объектов планирования (трубопроводов) предусмотрены поля Исходный сегмент и Целевой сегмент, а для объектов планирования (кабелей) — одноименные свойства. В обоих случаях исходный и целевой сегменты также можно выбрать с помощью кнопки ++…++.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Создается объект планирования с соответствующими свойствами.

!!! tip "Совет:"

    При создании объектов планирования (соединений) можно также сначала разблокировать поля Исходный сегмент и Целевой сегмент. Затем во время размещения на странице предварительного планирования источник и цель объекта планирования (соединения) определяются автоматически.

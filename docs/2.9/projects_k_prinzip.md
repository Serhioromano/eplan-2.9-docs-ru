# Проекты: Принцип

В EPLAN схемы соединений и относящиеся к ним документы, такие как списки и обзоры, создаются внутри проектов как страницы. Проект - это база данных, в которой кроме страниц проекта сохранены и все использованные в проекте основные данные (символы, рамки, формы, данные изделий и т. д.)

К проекту также относится файл проекта (*.elk), а также каталог, в котором хранится база данных и специфические для проекта основные данные. Файл проекта представляет собой проект. С помощью этого проекта можно выбрать проект для открытия или распаковки. Он содержит ссылку на соответствующий каталог с данными проекта.

Данные проекта могут иметь различные расширения имени типа файла, например:

*.elk: стандартный проект

*.elp: запакованный проект

*.elr: закрытый проект / исходный проект.

!!! example "Пример:"

    В каталоге..\Проекты\EPLANнаходится файл проектаESS_Sample_Project.elk. В нем содержится ссылка на каталогESS_Sample_Project.edb. В подкаталоге..\Проекты\EPLAN\ESS_Sample_Project.edbнаходятся соответствующиеданные проекта. Если требуется открыть проект с помощью меню Проект > Открыть, вы можете выбрать файлESS_Sample_Project.elkв диалоговом окне выбора файлов.

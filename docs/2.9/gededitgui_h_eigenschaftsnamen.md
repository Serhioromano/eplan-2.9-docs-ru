# Показать имена свойств

После того как вы разместили на странице проекта определенные свойства, по умолчанию на ней отображаются ***Значения*** этих свойств. Чтобы вместо значений сделать видимыми ***имена свойств*** (как, например, ОУ (видимое), Поперечное сечение соединения / диаметр с единицей измерения) и отдельно разместить их, предусмотрена настройка пользователя Показать имена свойств: Если активирована эта настройка, то во всех проектах для всех размещенных свойств будут отображаться имена, а присвоенные значения будут скрыты. Это особенно рационально при обработке свойств макроса на монтажной плате.

1. Выберите следующие пункты меню: Параметры > Настройки > Пользователь > Графическая обработка > 2D

2. Установите флажок Показать имена свойств.

!!! info "Для сведения:"

    Во всех проектах вместо значений свойств будут отображаться имена свойств.

# Отображать внешние цели

Опция Вид > Внешние цели упрощает нахождение внешних целей: Если она активирована, внешние выводы функций обозначаются стрелкой. Это упрощает, например, отслеживание цели в случае с клеммами. Благодаря этому можно узнать, на какой странице спецификации клеммника выведена цель.

Свойство Внутр. / внешн. настройте в диалоговом окне [Логическая схема выводов устройства](xfctdefbrowsergui_d_anschlusslogik.md), которое вызывается в диалоговом окне свойств на вкладке Данные символа / функции с помощью кнопки ++Логика++.

Условие:

В Графическом редакторе открыта страница проекта.

1. Выберите пункты меню Вид > Внешние цели.

!!! info "Для сведения:"

    Внешние выводы функций обозначаются маленькой стрелкой.

2. Повторно выберите пункты меню Вид > Внешние цели.

!!! info "Для сведения:"

    Стрелки для отображения внешних выводов снова скрыты.

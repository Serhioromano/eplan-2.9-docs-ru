# Вкладка Технологические перемычки (экспорт ЧУ Steinhauer)

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт ЧУ Steinhauer. Выберите вкладку Технологические перемычки.
* Сервисные программы > Данные изготовления > Механическая обработка > ЧУ Steinhauer. Рядом с полем Машина нажмите ++...++. Выберите вкладку Технологические перемычки.

Чтобы избежать повреждений фрезерного инструмента при фрезеровке слишком больших прямоугольных поверхностей, эти поверхности можно разделить при помощи технологических перемычек. В результате этого образуются фрезерные области меньшего размера. Возникающие технологические перемычки при монтаже выламываются. На этой вкладке задаются все характеристики для автоматической обработки технологических перемычек.

!!! note "Замечание:"

    * Во избежание искажения контура при позиционировании фрезерного инструмента для контуров с очень малой высотой должны выполняться следующие условия: Технологические перемычки на прямоугольниках выполняются только в том случае, если длина меньшей стороны составляет не менее 20 мм.

    Для прямоугольных отверстий с фаской или закругленных действуют следующие правила.

* Если на этой вкладке указано, что по всей длине стороны (ширине или высоте стороны) должны быть сгенерированы ***три*** технологических перемычки, а имеющейся длины из-за наличия фасок / закруглений хватает только на ***две*** перемычки, то выполняются ***две*** технологических перемычки.
* Если на этой вкладке указано, что по всей длине стороны (ширине или высоте стороны) должны быть сгенерированы ***две*** технологических перемычки, а имеющейся длины из-за наличия фасок/закруглений хватает только на ***одну*** перемычку, то выполняется ***одна*** технологическая перемычка.
* Эти правила применяются в тех случаях, когда разность между длиной стороны и размером фаски / радиусом закругления больше, чем сумма ширины технологической перемычки и удвоенного диаметра фрезы / диаметра отверстия.

Обзор основных элементов диалогового окна:

Групповое поле 'Параметры для неортогональных прямоугольников со снятой фаской'

Используйте стандартные настройки или измените значения для отдельных настроек в соответствии с вашими требованиями.

### Ширина технологической перемычки

По умолчанию ширина технологической перемычки составляет 1 мм.

### Генерировать одну технологическую перемычку, если длина ребра превышает

По умолчанию технологическая перемычка добавляется при фрезеровочной длине свыше 100 мм.

### Генерировать две технологические перемычки, если длина ребра превышает

По умолчанию две технологические перемычки добавляются при фрезеровочной длине свыше 250 мм.

### Генерировать три технологические перемычки, если длина ребра превышает

По умолчанию три технологические перемычки добавляются при фрезеровочной длине свыше 500 мм.

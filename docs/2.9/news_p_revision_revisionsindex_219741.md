# Новое свойство проекта для отображения высшего индекса ревизии

Если управление ревизиями используется с отслеживанием изменений, при закрытии страниц или пространств листа задайте индекс ревизии. Этот индекс ревизии сохраняется на соответствующей странице / в соответствующем пространстве листа в свойстве Индекс ревизии (из отслеживания изменений) (ид. 11071). Посредством нового свойства проекта Высший индекс ревизии (из отслеживания изменений) (ид. 10098) высший индекс ревизии в проекте теперь можно вывести на печать.

![](images/benefit.png)Эффект:

Посредством нового свойства проекта Высший индекс ревизии (из отслеживания изменений) сразу можно увидеть, какой высший индекс ревизии имеет проект.

Свойство Высший индекс ревизии (из отслеживания изменений) предоставляет высший индекс ревизии для всех страниц и пространств листа в проекте, и это свойство можно разместить на странице проекта в виде специального текста и, таким образом, распечатать. Свойство выполняет поиск последнего индекса ревизии, который должен соответствовать формату, установленному в диалоговом окне [Настройки: Отслеживание изменений (нумерация индекса ревизии)](revisionlogmgtgui_d_einstellungenseitenindex.md), и не должен быть дополнительно изменен.

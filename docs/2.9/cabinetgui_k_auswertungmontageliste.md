# Таблицы для сборочного чертежа: Принцип

В EPLAN Pro Panel для трехмерных размещений изделий пространства листа можно сгенерировать отчет ***Таблица для сборочного чертежа***. Таблицы для сборочного чертежа используются для воспроизведения в отчете иерархической структуры чертежа монтажных поверхностей в пространстве листа. При этом отчет выполняется для полностью определенного размещения изделия (монтажная плата, электрошкаф, сборная шина и т. д.).

В данном относящемся к функции отчете анализируются трехмерные размещения изделий, у которых в диалоговом окне "Cвойства" установлено свойство Вывести в таблицу для сборочного чертежа (ид. 36032). При этом данное свойство выбирается у всех трехмерных объектов (кроме монтажных поверхностей) через выбор свойств. Для устройства с установленным свойством в выведенной таблице для сборочного чертежа приведены все устройства непосредственно следующего за ним уровня иерархии. При этом по умолчанию перечисленные объекты сортируются по позиции легенды.

Для генерирования таблицы для сборочного чертежа в диалоговом окне Определить отчет существует тип отчета 'Таблица для сборочного чертежа'. При выводе таблицы для сборочного чертежа можно выполнить выбор вручную. В следующем диалоговом окне перечисляются все трехмерные размещения изделий, для которых установлено свойство Вывести в таблице для сборочного чертежа.

!!! example "Пример:"

    Если свойство Вывести в таблицу для сборочного чертежаустановлено, например, для электрошкафа, в верхнем колонтитуле отчета выводятся данные устройства электрошкафа. В области данных перечисляются все устройства, иерархически расположенные в пространстве листа непосредственно под данным электрошкафом.Для размещенной в данном электрошкафе монтажной платы соответствующие устройства, находящиеся на нижнем уровне иерархии, не выводятся. Если это необходимо выполнить, для монтажной платы также устанавливается свойство Вывести в таблицу для сборочного чертежа. В таком случае выводится вторая таблица для сборочного чертежа с данными монтажной платы и размещенными на ней устройствами.

### Формы таблиц для сборочного чертежа

При генерировании таблицы для сборочного чертежа используется форма, заданная в относящемся к проекту диалоговом окне Настройки: Вывод в страницах для типа формы "Таблица для сборочного чертежа (*.f32)". Для форм данного типа в диалоговом окне Тексты-заполнители — таблица для сборочного чертежа тексты-заполнители выбираются из следующих элементов:

* Данные изделия
* Размещения изделий
* Данные ссылки изделия
* Данные устройства
* Запись данных.

При этом как в верхнем колонтитуле, так и в области данных на выбор предлагаются аналогичные тексты-заполнители.

### Фильтрация в навигаторе пространства листа

Если необходимо узнать, какие устройства предусмотрены для генерирования таблицы для сборочного чертежа, создайте для навигатора пространства листа новый фильтр и выберите для этого фильтра свойство Вывести в таблицу для сборочного чертежа в качестве критерия фильтрации.

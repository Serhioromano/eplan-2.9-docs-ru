# Импорт и экспорт пространства конструкции

В EPLAN Pro Panel Professional можно импортировать геометрические модели из программы 3D-CAD, чтобы сконцентрировать всю важную информацию вокруг чертежа монтажных поверхностей. Геометрическую модель, обогащенную информацией в EPLAN Pro Panel, можно экспортировать назад в программу 3D-CAD, чтобы там получить целостную трехмерную модель со всеми параметрами. Эту технику называют ***Design Space Exchange (DSE)***.

Возможно следующее:

* Импорт данных модели в формате STEP, экспортированных из программ 3D-CAD
* Автоматическая интерпретация геометрической модели
* Реализация необходимого трехмерного чертежа монтажных поверхностей в EPLAN Pro Panel в геометрической модели из программы 3D-CAD; добавление информации об отверстиях, резьбах, монтажных отверстиях стандартной формы или монтажных отверстиях на основе контуров
* Экспорт данных EPLAN, сгенерированных в трехмерном чертеже монтажных поверхностей (несущие шины, кабельные каналы, информация об отверстиях, резьбах, монтажных отверстиях) в форме XML-файла
* Интеграция и интерпретация данных из XML-файла в Autodesk Inventor в оригинальный 3D-CAD-файл
* Автоматическое размещение механических и электрических устройств в параметрической 3D-модели в Autodesk Inventor
* Автоматическая интерпретация важной для ЧУ информации из EPLAN Pro Panel
* Автоматическое преобразование важной для ЧУ информации из EPLAN Pro Panel в компоненты геометрии Inventor, с полным набором параметров (отверстия, резьбы, продольные отверстия, монтажные отверстия и т. д.).

Имеются следующие результаты:

* ***В области электротехники***: Полностью укомплектованный электрошкаф, включая исполнительную документацию для изготовления.
* ***В области механики***: Полная параметрическая трехмерная модель, дополненная информацией из EPLAN в качестве основы для интеграции и документации изготовления корпуса.

# Диалоговое окно Свойства

Вы открыли проект.

* Данные проекта > Устройство / изделие > Список устройств > Всплывающее меню > Создать.
* Данные проекта > Устройство / изделие > Список устройств. Выделите изделия в списке устройств. Всплывающее меню > Свойства.

В данном диалоговом окне вы можете ввести новое подлежащее контролю изделие, а также просмотреть и изменить свойства уже имеющегося изделия.

Обзор основных элементов диалогового окна:

### Номер изделия

Введите здесь вручную номер изделия или, нажав ++...++, откройте [выбор изделия](partselectiongui_d_artikelauswahl.md) и выберите изделие из базы данных изделий.

Номер изделия может иметь длину до 50 символов.

### Вариант

Вариант копируется из базы данных изделий на основании номера изделия.

### ERP-номер

Номер ERP копируется из базы данных изделий на основании номера изделия.

### Номер типа

Номер типа копируется из базы данных изделий на основании номера изделия.

### Номер для заказа

Номер для заказа копируется из базы данных изделий на основании номера изделия.

### Раздел

Раздел перенимается из номера изделия в базе данных изделий и автоматически заполняется первым определением настройки раздела изделия. При необходимости выберите из раскрывающегося списка другой раздел. При размещении устройства из списка устройств в условное обозначение переносятся раздел и подраздел.

### Подраздел

При необходимости здесь можно указать подраздел устройства. Данная информация не считывается из базы данных изделий, но должна вводиться дополнительно.

### Обозначение 1-3

Обозначение копируется из базы данных изделий на основании номера изделия.

### Производитель

Производитель копируется из базы данных изделий на основании номера изделия.

### Макрос

Если изделию присвоен макрос, здесь будет отображено имя этого макроса.

### Разрешенное число штук

Введите здесь число штук текущего изделия, которое может быть задействовано в проекте.

### Разрешенное число штук (итого)

В этом поле для всех изделий с одним номером изделия суммируются все числа штук, введенные в поле Разреш. число штук.

### Установл. число штук

Указывает, как часто изделие встречается в проекте. (К ним причисляются ссылки изделий.) Отклонения относительно разрешенной спецификации допустимы; они отслеживаются при контрольном прогоне.

### Часть установки

Здесь вводится имя той части установки, в которую встраивается устройство (например, "Элемент роликового конвейера 15"). Данная информация не считывается из базы данных изделий, но должна вводиться дополнительно.

### Описание устройства

Введите здесь описание функциональности устройства. Данная информация не считывается из базы данных изделий, но должна вводиться дополнительно. Изделие может встречаться в списке устройств многократно под различными описаниями изделий.

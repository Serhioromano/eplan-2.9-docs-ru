# Диалоговое окно Выбор изделия: Выбрать источник данных

Вы открыли проект. Сервисные программы > Изделие > Управление > [Дополнительно] > Настройки. Установите флажок Выбор изделия с выбором схемы и нажмите кнопку ++ОК++.

* Вы выделили главную функцию в графическом редакторе или навигаторе. Обработать > Свойства. Выберите вкладку Изделие и щелкните в поле Номер изделия по кнопке ++...++.
* Данные проекта > Устройство > Навигатор. Выделите одну запись. Всплывающее меню > Новое устройство.
* Данные проекта > Устройства / Изделие > Навигатор спецификаций. Выделите одну запись. Всплывающее меню > Добавить изделие и Всплывающее меню > Заменить изделие.
* Данные проекта > Устройство / изделие > Список устройств. Выделите одну запись. Всплывающее меню > Добавить изделие.

В этом диалоговом окне выберите схему выбора изделия из источника данных. С ее помощью вы определите, какие источники данных должны использоваться при выборе изделия.

Доступны следующие элементы диалогового окна:

### Источник данных

Выберите из раскрывающегося списка схему, которая определяет пользовательские предварительные настройки для выбора изделия. С помощью кнопки ++...++ откройте диалоговое окно [Настройки: Изделие (пользователь)](partselectiongui_d_datenauswahlbenutzer.md), в котором можно изменить эти предварительные настройки.

### Всегда использовать этот источник данных

Если этот флажок установлен, то выбранный источник данных будет использоваться для всех дальнейших операций, при которых открыт выбор изделия. Диалоговое окно Выбор изделия: Выбрать источник данных более не выводится. Вы можете снова активировать диалоговое окно Выбор изделия: Выбрать источник данных, для чего в меню Сервисные программы > Изделие > Управление > [Дополнительно] > Настройки необходимо снова установить флажок Выбор изделия с выбором схемы.

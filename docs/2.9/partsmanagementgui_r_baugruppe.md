# Вкладка Узел

Сервисные программы > Изделие > Управление. В дереве или списке выделен одно изделие узла.

В этой вкладке выбранному узлу присваиваются позиции, т. е., изделия, которые должны содержаться в нем (т. н. ***позиции узлов***).

Обзор основных элементов диалогового окна:

### Номер позиции / Дополнительный текст

Эти два свойства обусловлены предварительным планированием: при вставке монтажного чертежа в проект они копируются из узла в ссылку изделия.

### Длина

Ввод значений в этот столбец возможен только для изделий с указаниями длины (таких как кабели, соединения, провода / группы соединений, шланги), но невозможен для изделий из главной группы продуктов "Механика".

Для такого изделия, например кабеля, здесь можно ввести часть длины, указанную на самом изделии. Если узел разворачивается при вставке в схему соединений в качестве устройства или при выборе изделия, введенная здесь длина для отдельного изделия переносится в свойство ссылки изделия Подмножество / длина и может выводиться в отчетах. Однако передача работает ***только*** для тех функций, которые из-за определения функции имеют собственное поле длины в дополнение к свойствам ссылки изделия. К ним относятся, например, линии определения кабеля и точки определения соединения.

!!! note "Замечание:"

    * Чтобы узел разворачивался при вставке в качестве устройства или при выборе изделия, в определенных пользователем настройках для выбора изделия и базы данных изделий следует установить флажок Развернуть узел (возможный путь меню для диалогового окна "Настройки": Параметры > Настройки > Пользователь > Управление > Изделие).
    * Если узлы не разворачиваются в схеме соединений, значения длины, указанные в базе данных изделий, все равно можно перечислить в отчетах по изделиям. Для этого в [настройках отчета по изделиям](partslistgui_d_paramartikelauswertung.md) следует установить флажок Развернуть узлы.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Удалить содержимое |  Удаляет содержимое текущих выделенных ячеек.
Перейти к |  Этот пункт меню доступен только для столбца Номер изделия. Переход к изделию, введенному в текущей выделенной строке.

### Разбить на групповую спецификацию изделий

C помощью настройки из этого раскрывающегося списка определяется, следует ли разворачивать узлы в групповой спецификации изделий; значением по умолчанию является запись "Из настроек", т. е. настройка заимствуется из диалогового окна [Настройки: Изделие](partslistgui_d_paramartikelauswertung.md). При выборе другой настройки она перезапишет настройку, заданную в настройках.

### Распределенное размещение узла

С помощью этого флажка можно указать, как должен вести себя узел: Установите флажок, если узел следует разместить отдельно на трехмерном чертеже монтажных поверхностей (например, главный выключатель распределяется на пластине и двери). Для других узлов (например, контакторы со вспомогательными контактами), размещенных вместе и определяющихся в спецификации как узлы, снимите флажок.
Обратите внимание, что распределенное размещение никак не сказывается на отчетах и на компоновке электрошкафа 2D.

!!! note "Замечание:"

    Отдельные элементы узла относятся позже в схеме соединений к ***одному*** устройству.

# Диалоговое окно Формат: Страница / столбец / строка (нумерация с данными ПЛК)

Вы открыли проект. Вы открыли диалоговое окно [Настройки: Нумерация с данными ПЛК](offlinenumerationplcgui_d_einstellungen.md) (например, через Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация с данными ПЛК). В диалоговом окне Настройки: Нумерация с данными ПЛК выбрана пользовательская схема. Вы выделили запись в поле Группы форматов и щелкнули по ![](images/all_edit_as.png){: .ui-icon } (Обработать) или по ![](images/all_new_as.png){: .ui-icon } (Создать).

В диалоговом окне Нумерация с данными ПЛК: Формат в списке Доступные элементы формата выберите элемент формата типа "Страница", "Столбец" или "Строка" и при помощи кнопки со стрелкой перенесите его в список Выбранные элементы формата.

В этом диалоговом окне указывается форматирование для номера страницы, столбца или строки в группе форматов.

Доступны следующие элементы диалогового окна

### От вывода устройства ПЛК

Выберите эту опцию, чтобы для данного формата использовался номер страницы, столбца или строки вывода устройства ПЛК.

### От цели

Выберите эту опцию, чтобы для данного формата использовался номер страницы или столбца целевого функционального элемента.

### Минимальное число разрядов

Здесь указывается минимальное число разрядов для значения свойства. Более длинные значения выводятся полностью. Разрешено использовать только положительные числовые значения. При вводе нуля число разрядов является переменным.

### Заполнитель

Значение заполняется с помощью этого символа, если минимальное число разрядов больше, чем ширина значения.

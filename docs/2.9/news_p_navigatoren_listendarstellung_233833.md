# Новое табличное представление в виде списка для навигаторов

Уже в версии 2.8 представление в виде списка для навигатора макросов было преобразовано в таблицу с частично обрабатываемыми ячейками. В текущей версии это преобразование реализовано также для следующих навигаторов:

* Навигатор пространства листа
* Навигатор устройств
* Навигатор клеммников
* Навигатор штекеров
* Навигатор ПЛК
* Навигатор кабелей
* Навигатор топологии
* Навигатор соединений
* Навигатор потенциалов
* Навигатор трубопроводов
* Навигатор проводного монтажа
* Навигатор точек разрыва
* Навигатор спецификаций.

В новых списках можно напрямую обрабатывать некоторые из отображаемых свойств. Кроме того, возможна непосредственная поблочная обработка свойств.

![](images/benefit.png)Эффект:

В новом представлении в виде списка для навигаторов теперь можно напрямую изменять значения обрабатываемых свойств.

### Использование выбранных данных в табличной обработке

Объекты, выбранные в табличном представлении в виде списка, можно перенести в диалоговое окно табличной обработки с помощью пункта всплывающего меню Табличная обработка. Наряду с функциями и соединениями в этом диалоговом окне вы теперь также можете обрабатывать макросы, точки разрыва, потенциалы, трубопроводы для функциональной схемы автоматизации, а также сегменты предварительного планирования.

![](images/benefit.png)Эффект:

Теперь в табличной обработке наряду с функциями и соединениями можно обрабатывать данные проекта, например макросы, точки разрыва, потенциалы, трубопроводы и сегменты. Перенос выделенных объектов из представления в виде списка в табличную обработку предоставляет лучший обзор выбранных данных и позволяет обрабатывать их целенаправленно.

В диалоговом окне табличной обработки вкладки Функции, Соединения, Точки разрыва и т. д. отображаются, только если выбор содержит также соответствующие типы объектов. В рамках этих расширений диалоговое окно табличной обработки было переименовано:

Старое обозначение |  Новое обозначение
---|---
Обработать данные функции |  Табличная обработка

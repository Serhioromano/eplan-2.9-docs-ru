# Структура проекта

Чтобы наглядно оформить проект, можно структурировать все его объекты:

* При этом предварительно определенные схемы позволяют выбрать структуры обозначений для страниц и устройств (= структура проекта) или выбрать собственные структуры (определенные пользователем) для этих объектов.
* Вы можете присвоить блокам идентификаторов, необходимых для построения структуры проекта, свойства "Идентифицирующ.", "Описывающ." или "Недоступно".
* Вы можете экспортировать структуры проектов из проекта и импортировать в другой проект.
* Разработать расширенное обозначение ссылки со структурой устройства согласно стандарту EN 81346. При этом вы можете использовать маркировку продукции в качестве дополнительного блока идентификаторов.
* Дополнительно возможно также обозначение страниц согласно стандарту DIN EN 61355-1. При таком типе маркировки для структурирования используется т. н. "Идентификатор объекта".

# Вывести монтажные чертежи для объектов планирования

Условия:

* Вы открыли проект, содержащий данные предварительного планирования, и навигатор предварительного планирования.
* В базе данных изделий под главной группой продуктов "Механика" вы создали узел с группой продуктов "Монтажный чертеж". Вы задали макрос для узла на вкладке "Технические данные" с желаемым графическим представлением для монтажного чертежа. Для этого узла в той же главной группе продуктов в разделе "Отдельная часть" для принадлежностей монтажного чертежа доступны изделия.

1. Выделите объект планирования в навигаторе предварительного планирования.
2. Выберите пункт всплывающего меню Свойства.
3. Выберите в диалоговом окне Свойства ++...++ вкладку Изделие.
4. В строке 10 в поле Номер изделия щелкните ++...++.
5. В диалоговом окне Выбор изделия через структуру дерева в разделе "Механика > Монтажный чертеж> Общее" выберите нужное изделие узла.

!!! info "Для сведения:"

    Номера изделий переносятся в диалоговое окно 'Свойства' объекта планирования.

6. Щелкните по кнопке ++OK++.
7. Выберите пункты меню Сервисные программы > Отчеты > Генерировать.
8. Выберите вкладку Отчеты. Щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
9. В следующем диалоговом окне выделите в поле Выбрать тип отчета запись "Обзор групп изделий". Щелкните по кнопке ++OK++.
10. В диалоговом окне Настройки – <Тип отчета> в поле Фильтр установите фильтр для монтажных чертежей. Для этого в следующем диалоговом окне выберите значение "Механика: Монтажный чертеж" для критерия фильтра "Группа продуктов".
11. В следующем диалоговом окне Обзор групп изделий укажите начальную страницу. Для этого в поле Навигатор страниц выберите начальную страницу.

!!! info "Для сведения:"

    Монтажные чертежи, присвоенные объектам планирования, выводятся в виде отчетов.

!!! example "Пример:"

    На рисунке показан монтажный чертеж из образца проектаESS_Sample_Project. Для этого с помощью типа отчета "Обзор групп изделий" для узлаLW 001, который в проекте сохранен на нескольких объектах планирования, создана соответствующая страница отчета.

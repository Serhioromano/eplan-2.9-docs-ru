# Диалоговое окно Настройки — <Тип отчета> (в зависимости от функции)

Вы открыли проект. Сервисные программы > Отчеты > Генерировать. Щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать). Выделите в диалоговом окне Определить отчет относящийся к функции [Тип отчета](formgeneratorgui_k_auswertungstypen.md), как, например, спецификацию клеммника. Щелкните по кнопке ++OK++.

В этом диалоговом окне установите схемы сортировки, фильтрации и разбивки на страницы [для отчетов, относящихся к функции](formgeneratorgui_k_funktionsbezogen.md). У относящихся к функции отчетов по умолчанию в верхнем колонтитуле содержится устройство, как, например, клеммник. В области данных находятся соответствующие функции, как, например, клеммы клеммника. Схемы сортировки и фильтрации сохраняются специфически для пользователя и при генерации страницы отчета сохраняются вместе с отчетом.

Обзор основных элементов диалогового окна:

### Устройство

Эти групповые поля доступны только в том случае, если в диалоговом окне Определить отчет снят флажок Индивидуальный выбор.. Здесь вы зададите критерии для фильтра и сортировки устройства. Для отчета "Спецификация клеммника" является клеммником. Для отчета "Перечень символов" является библиотекой символов.

* Фильтр: В этом поле выберите схему, в которой установлено, какие устройства / библиотеки символов должны анализироваться. С помощью кнопки ++...++ откройте диалоговое окно Фильтр, в котором можно создать или обработать схему. Возможные критерии фильтра, например, блоки идентификаторов структуры проекта.

!!! example "Пример:"

    Вам нужно создать спецификацию клеммника. Определите следующие фильтры:Активн.Отрицат.Критерий Оператор Значение Установка=HF1Все установки клеммника анализируютсяHF1.

* Сортировка: В этом поле выберите схему, в которой установлено, как должны сортироваться отчеты устройства / библиотеки символов. С помощью кнопки ++...++ откройте диалоговое окно Сортировка, в котором можно создать или обработать схему.

!!! example "Пример:"

    Вам нужногенерироватьспецификацию клеммника и выбратьфильтр Установка HF1. Выберите предварительно определенную сортировку Место установки.Все отфильтрованные клеммники анализируются сортируясь по месту установки.

Выбранная схема активируется автоматически. Запись "- Не активировано -" позволяет деактивировать выбранную схему.

### Функции

Здесь вы зададите критерии для фильтра и сортировки функций. Для отчета "Спецификация клеммника" является соответствующими клеммами клеммника. Для отчета "Перечень символов" является символом.

* Фильтр: В этом поле выберите схему, в которой установлено, какие функции / символы должны анализироваться. С помощью кнопки ++...++ откройте диалоговое окно Фильтр, в котором можно создать или обработать схему.

!!! example "Пример:"

    ***Снят флажок Индивидуальный выбор:***Вам нужно генерировать спецификацию клеммника и выбрать в групповом поле Устройстводля клемника существующий фильтр Установка HF1. Определите здесь следующие фильтры:Активн.Отрицат.Критерий Оператор Значение Определение функции=PE- / PEN-клемма, 4 цели Из всех отфильтрованных клеммников анализируются только PE- / PEN-клеммы с 4 целями.***Установлен флажок Индивидуальный выбор:***Вам нужно создать спецификацию клеммника. Определите следующие фильтры:Активн.Отрицат.Критерий Оператор Значение Определение функции=PE- / PEN-клемма, 4 цели Из всех выбранных вручную клеммников анализируются только PE- / PEN-клеммы с 4 целями.

* Сортировка: В этом поле выберите схему, в которой установлено, как должны сортироваться функции / символы. С помощью кнопки ++...++ откройте диалоговое окно Сортировка, в котором можно создать или обработать схему.

!!! example "Пример:"

    Вам нужно создать спецификацию клеммника. Выберите предварительно определенную сортировку Место монтажа. Все отфильтрованные клеммы анализируются сортируясь по месту монтажа.

* Новая страница при изменении значения свойства: В этом поле выберите схему, в которой установлено, когда должна генерироваться страница. С помощью кнопки ++...++ откройте диалоговое окно Новая страница при изменении свойства, в котором можно создать или обработать схему.

Выбранная схема активируется автоматически. Запись "-Не активировано-" позволяет деактивировать выбранную схему.

### Форма (отличается от настроек)

Для того чтобы присвоить выбранному типу отчета форму, которая ***временно*** отличается от заданных параметров в настройках, откройте этот раскрывающийся список и щелкните по "Поиск". В диалоговом окне Выбрать форму настроен каталог, который занесен в настройки как каталог форм. В поле Тип файла автоматически отображается тип формы, подходящий для типа отчетов. Выделите нужную форму и щелкните по ++Открыть++.

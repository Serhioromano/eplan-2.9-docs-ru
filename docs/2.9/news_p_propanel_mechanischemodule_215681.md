# Механические модули для трехмерного чертежа монтажных поверхностей

В базе данных изделий теперь появилась возможность создавать изделия из модулей механики. После выделения главной группы продуктов "Механика" данная операция выполняется, как обычно, через пункты меню Создать > Модуль. В отличие от других модулей, механические модули могут содержать только изделия и не могут содержать других узлов или модулей.

Порядок действий с механическими модулями приблизительно соответствует порядку действий с другими модулями. Для получения дополнительной информации см. раздел "[База данных изделий: Модули](partsmanagementgui_k_module.md)".

Механический модуль для трехмерного чертежа монтажных поверхностей представляет собой стандартный функциональный элемент, который один производитель получает от другого производителя как единое целое (например, цоколь электрошкафа, изготовленный из различных элементов, таких как уголок для цоколя, заглушка для цоколя, кабельная шина и т. д.).

При размещении механических модулей в пространстве листа длина функциональных элементов с переменной длиной определяется из сохраненного макроса и вводится в данные ссылки изделия в свойство Подмножество / длина (ид. 31008). При размещении для механического модуля отображаются точки захвата, чтобы модуль можно было присоединить многократно.

# Определение длины заказа

В групповых спецификациях изделий можно рассчитывать и выводить общую длину всех маршрутизируемых соединений, кабелей, шлангов и т. д. с одинаковым изделием. При этом общая длина рассчитывается из длин сегментов маршрутизации и точек маршрутизации, через которые проходит соединение.

Кроме того, в групповой спецификации изделий можно также рассчитывать и выводить требуемые длины заказа материалов. Таким образом, например, можно рассчитывать длину заказа материалов, необходимых для построения сегментов маршрутизации сети соединенных сегментов. Сегменты маршрутизации могут состоять из таких материалов, как кабельные каналы, кабельные лотки, трубы и т. д. и включать изделие. Общие длины этих материалов и требуемая длина заказа рассчитываются на основе свойств изделий, введенных в сегментах маршрутизации. Длину заказа можно вывести в метрах или числе штук.

!!! example "Пример:"

    Сеть соединенных сегментов включает несколько сегментов маршрутизации, которые представляют собой кабельныеканалы. Этим сегментам маршрутизации присвоены те же изделия кабельных каналов. Поставляемая длина изделия составляет2000 мм, т. е. кабельные каналы поставляются как одна штука длиной 2 м. Если общаядлина трассысегментов маршрутизации составляет10,5 м, то требуемая длина заказа будет равна12 мили6 штукам. Эта длина товара рассчитывается путем ввода формулы в форме для групповой спецификации изделий.

За основу расчета в базе данных изделий для изделий главной группы продуктов "Механика", группы продуктов "Сегмент маршрутизации (топология)" необходимо ввести поставляемую длину (единица измерения — мм). Значение при выборе изделия или устройства переносится в свойство изделия Поставляемая длина (Ид. 22058). Это свойство отображается в диалоговом окне "Свойства" сегмента маршрутизации на вкладке Изделия (категория "Данные изделия").

Использованные длины материалов рассчитываются из свойства ссылки изделия Подмножество / длина (Ид. 31008). Это свойство заполняется значением свойства Топология: Длина трассы (автоматически) (Ид. 20250), как только будет выбрано изделие. В случае внесения изменений эти свойства синхронизируются. Вручную введенная запись на ссылке изделия автоматически копируется во вручную введенную длину трассы.

#### Вывод длины заказа

Для расчета длины заказа в форме групповой спецификации изделий необходимо ввести соответствующую формулу расчета. Для этого вставьте текст заполнитель с опцией Форматированные / рассчитанные свойства. При выборе элементов формата для расчета доступно свойство ссылки изделия Общая длина с единицей измерения, установленной в проекте (Ид. 20513). Это свойство суммирует длины всех функций (напр., связей, сегментов маршрутизации, сборных шин и т. д.) с одинаковым изделием. Единица измерения длины задается в настройках проекта для связей.

!!! example "Пример:"

    Для расчета числа штук общая длина всех функций с одинаковым изделием делится на значение поставляемой длины избазы данных изделий. При необходимости для округления к числу штук можно прибавить резерв.

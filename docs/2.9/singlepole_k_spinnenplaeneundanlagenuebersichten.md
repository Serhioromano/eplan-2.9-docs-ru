# Обзоры установок

Обзоры установок создаются на страницах типа страниц "Однопол. схема соедин.". Они служат для того, чтобы представлять структурированные обзорные планы установки со всеми структурными идентификаторами и соответствующими кабельными соединениями. При этом можно предварительно задать структурные идентификаторы с помощью обозначений местоположения и соединить их с помощью линий определения кабелей.

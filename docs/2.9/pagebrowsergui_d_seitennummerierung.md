# Диалоговое окно Нумеровать страницы

Вы открыли навигатор страниц и открыли (как минимум) один проект.

* В навигаторе страниц выделена одна или несколько страниц. Страница > Нумеровать.
* Вы скопировали и вставили страницы. В диалоговом окне Адаптировать структуру щелкните кнопку ++Нумеровать++.

В этом диалоговом окне можно автоматически задать имена страниц.

Обзор основных элементов диалогового окна:

### Нумерация с учетом структуры

Установите этот флажок для ***проектов DIN***, чтобы выполнить эту нумерацию для каждого структурного идентификатора. Если этот флажок не установлен, выполняется сквозная нумерация страниц без учета структурных идентификаторов.
Эта опция неактивна для полностью пронумерованных проектов.

### Начальный номер

Введите в это поле число, с которого должна начаться нумерация.

### Величина шага

Введите здесь значение интервала номеров страниц.

### Интервал не изменять

В комбинации с указанием начальной страницы этот флажок указывает на то, что интервал выделенных страниц должен использоваться и для целевых страниц. Благодаря этому любое выделенное число страниц можно переместить на определенный интервал. Ввод величины шага в данном случае невозможен.

!!! example "Пример:"

    Предположим, требуется переместить страницы 1, 2, 3 и 8, а в качестве начального номера указано значение "2". После выполнения повторной нумерации в списке отображаются имена страниц с номерами 2, 3, 4 и 9.

### Текст не изменять

Установите этот флажок, если не требуется перезаписывать буквенную часть имени страницы. Эта опция доступна и для подстраниц. Если этот флажок не установлен, текстовая часть будет удалена.

### Подстраницы

Выберите в раскрывающемся списке, каким образом будут вести себя подстраницы при копировании:

* Удерживать: Имеющиеся подстраницы будут скопированы в целевые страницы без изменений.
* Нумеровать: Имеющиеся подстраницы будут заново пронумерованы с использованием начального значения "1" и величины шага "1". При каждой смене главной страницы нумерация подстраниц снова начинается со значения "1".
Нумерация подстраниц выполняется в соответствии с настройкой проекта Символы для подстраниц (Параметры > Настройки > Проекты > "Имя проекта" > Управление > Страницы).
* Преобразовать в главные страницы: Подстраницы преобразуются в главные страницы и заново нумеруются.

# Обработка схем

В EPLAN можно создавать схемы для различных настроек, например, для обозначений устройства или для фильтра. Кроме того, схемы можно удалять, копировать, а также импортировать и экспортировать.

Условие:

Открыто диалоговое окно, в котором можно выбрать схему для настроек.

### Создать схемы

1. Щелкните в соответствующем диалоговом окне по ![](images/all_new_as.png){: .ui-icon } (Создать) рядом с раскрывающимся списком Схема.
2. В поле Имя диалогового окна Новая схема укажите имя для новой схемы.
3. Введите в поле Описание информативное краткое описание для новой схемы.
4. Щелкните по кнопке ++OK++.
5. Произведите необходимые настройки для схемы.
6. Щелкните по кнопке ++OK++.

### Копировать схемы

1. Щелкните по ![](images/all_copy_as.png){: .ui-icon } (Копировать) рядом с раскрывающимся списком Схема.
2. В диалоговом окне Копировать схему: <Имя схемы> в поле Имя введите имя для копии выведенной в строке заголовка диалогового окна текущей схемы.
3. Введите информативный описательный текст для копии схемы.
4. Щелкните по кнопке ++OK++.

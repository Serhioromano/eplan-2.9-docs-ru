# Операция: XMActionDCImport

**Параметр** |  **Описание**
---|---
ProjectLink  |  Имя проекта с полным путем к файлу.
DataConfigurationFile  |  Путь файла для файла конфигурации данных (*.edc).
ProgressTitle  |  Задает название индикатора выполнения (является опцией).

!!! note "Замечание:"

    Эта операция позволяет изменить свойства функций, страниц и / или соединений с помощью импорта. Файл конфигурации данных *.edc можно генерировать, например в меню Сервисные программы > Внешняя обработка свойств > Экспортировать функции.

!!! example "Пример:"

    XMActionDCImport
    /ProjectLink:C:\Projects\EPLAN\test.elk
    /DataConfigurationFile:C:\EPLAN\DataCfgFile.edc

# Генерировать изделия из данных схемы соединений

Из данных, сохраненных в условном обозначении, в EPLAN можно сгенерировать изделие. При этом изделие и типичные для устройства данные (определение функции, обозначение вывода устройства и т. д.) переносятся в базу данных изделий. Одно такое сгенерированное изделие содержит шаблоны функций для всех функций устройства и хранится в проекте как сохраненное изделие.

Условия:

* Вы выделили условное обозначение, которое представляет главную функцию.
* На ОУ записана точно ссылка изделия.

1. Выберите пункт всплывающего меню Генерировать изделие.

!!! info "Для сведения:"

    Открывается диалоговое окно База данных изделий. На основе определения функции на условном обозначении новое изделие определяется в структуру группы продуктов базы данных изделий и предварительно выбирается.

!!! info "Для сведения:"

    Если изделие уже содержится в проекте, отобразится запрос для подтверждения перезаписи. Если изделия еще нет в базе данных изделий, откроется база данных изделий и предварительно будет выбрано новое изделие.

!!! info "Для сведения:"

    Если изделие уже содержится в базе данных изделий, отобразится запрос. Подтвердите запрос, щелкнув [Да], и типовые для устройства данные (напр., шаблонов функций) будут обновлены для изделия, уже содержащегося в базе данных изделий. В таком случае откроется база данных изделий и это изделие будет предварительно выбрано. Щелкните ++Нет++, и содержащееся в проекте изделие будет перезаписано, при этом изделие в базе данных изделий не будет обновлено.

2. При необходимости измените данные нового изделия на вкладках с правой стороны.
3. Щелкните по кнопке ++Закрыть++.

!!! info "Для сведения:"

    Закрывается база данных изделий.

!!! info "Для сведения:"

    Если измененные данные не сохранены с помощью кнопки ++Применить++, последует еще один запрос. Дополнительно выводится сообщение, с помощью которого можно решить, должна ли автоматически проводиться синхронизация данных изделий "База данных изделий -> Проект".

!!! note "Замечание:"

    Изделия, сгенерированные таким образом, содержат на вкладке Технические данные значение "NEW" как запись в поле Группа функций. Это позволяет целенаправленно выполнять фильтрацию по таким изделиям.

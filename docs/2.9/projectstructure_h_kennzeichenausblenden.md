# Задать представление структуры дерева в навигаторах

В представлениях навигаторов в виде дерева (навигатора страниц, навигатора устройств и т. д) по умолчанию отображаются все использованные структурные идентификаторы блоков идентификаторов, для которых в структуре страниц задан статус "Идентифицирующий".

В специфических для проекта настройках можно показать или скрыть структурные идентификаторы одного или нескольких блоков идентификаторов и / или дополнительно отобразить другие свойства (Тип страницы, ОУ: Буквенное обозначение и т. д.) в представлениях в виде дерева. С помощью кнопок со стрелками можно дополнительно изменять последовательность отображенных в дереве структурных идентификаторов.

Следующая операция в качестве примера выполняется для навигатора страниц. Чтобы задать представления в виде дерева в других навигаторах, выполните аналогичные действия.

Условия:

* Вы открыли проект.
* Вы определили структуру проекта.
* Вы открыли Навигатор страниц.

1. Решите, какие структурные идентификаторы блока идентификаторов должны отображаться, а какие нет.
2. Выберите пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Представление > Структура дерева (страницы).
3. Для скрытия структурных идентификаторов блока выделите нужный блок идентификаторов и деактивируйте предшествующий флаг.
4. Чтобы в навигаторе страниц дополнительно отобразить тип страницы, установите флажок Тип страницы.
5. Щелкните по кнопке ++OK++.

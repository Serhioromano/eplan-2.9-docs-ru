# Диалоговое окно Дополнительно обработать данные ревизии

Вы создали ревизию и открыли проект ревизии (расширение имени файла *.ell). Вы произвели изменения внутри проекта ревизии. Страницы закрыты. Сервисные программы > Управление ревизиями > Отслеживание изменений > Дополнительно обработать данные ревизии.

В этом диалоговом окне перечисляются ревизии проекта и измененные страницы ревизий. Здесь можно затем изменить информацию ревизии, например, комментарий, описание или причину изменения.

Обзор основных элементов диалогового окна:

В таблице сначала указываются ревизии проекта. Под ревизиями перечисляются измененные страницы. Таблица содержит следующие столбцы:

* Строка: Показывает непрерывный номер строки.
* Имя ревизии: Показывает имя ревизии. Можно дополнительно изменить имя ревизии. Для отдельных страниц это поле недоступно.
* Область обработки: Показывает имя области обработки, к которой относится ревизия.
* Комментарий: Показывает комментарий, указанный вами при создании ревизии. Можно дополнительно изменить комментарий к ревизии. Для отдельных страниц это поле недоступно.
* Индекс: Показывает индекс ревизий, введенный вами при описании изменения страниц. Можно дополнительно изменить индекс для каждой страницы.
* Описание: Показывает текст описания, введенный вами при описании изменения страниц. Можно дополнительно изменить текст описания для каждой страницы.
* Причина изменения: Показывает причину изменения, введенную вами при описании изменения страниц. Можно дополнительно изменить причину изменения для каждой страницы.
* Страница / пространство листа: Отображает полное имя измененной страницы / измененного пространства листа. При этом имя пространства листа (в отличие от полного имени страницы) отделяются от структурных идентификаторов не разделителем "/", а пробелом и восклицательным знаком (например, =EB3+ET1 (2)). Значение в этом поле изменить невозможно.
* Пользователь: Показывает пользователя, описавшего изменение страницы. Значение в этом поле изменить невозможно.
* Дата: Показывает дату описания изменения страниц. Значение в этом поле изменить невозможно.
* Идентификатор / Имя / Телефон / Эл.почта: Показывает соответствующие данные, как они были заданы через пункты меню Параметры > Настройки > Пользователь > Отображение > Ид. пользователя / адрес. Эти поля изменить невозможно.
* Проверено: Занесите в это поле фамилию обработчика, проверявшего ревизию. Эта запись не переводится.
* Проверено (дата): Занесите в это поле дату и время проверки ревизии. Щелкните в этом поле, а затем по кнопке со стрелкой у правого края, чтобы открыть диалоговое окно для выбора даты, и выберите в нем нужное число.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Удалить последнюю ревизию |  Удаляет последнюю из созданных ревизий (т.е. ту, которая имеет самую свежую дату). При этом удаляется вся информация этой ревизии из свойств страницы. Изменения, выполненные на страницах проекта, сохраняются. Также сохраняются маркеры ревизий.
Объединить ревизии |  Объединяет выделенные ревизии. Для этого открывается диалоговое окно Генерировать ревизию, в котором можно указать имя ревизии (новое или уже существующее) и комментарий. Маркер ревизии на условном обозначении становится активным, а старое имя ревизии заменяется новым. Датой объединенной ревизии становится дата более поздней ревизии.

!!! tip "Совет:"

    Через пункты меню Сервисные программы > Управление ревизиями > Отслеживание изменений > Удалить ревизию можно при необходимости убрать маркеры ревизий. Но при этом удаляются ***все*** маркеры ревизий на выделенных страницах, независимо от соответствующей ревизии.

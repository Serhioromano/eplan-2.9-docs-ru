# Вкладка Страницы

* В навигаторе страниц выделена одна или несколько страниц. Обработать > Копировать. Щелкните в навигаторе страниц по тому месту, в которое необходимо вставить скопированные страницы. Обработать > Вставить. Выберите вкладку Страницы.
* Страница > Макрос страницы > Вставить. Выберите в диалоговом окне Выбрать макрос требуемый макрос страницы и щелкните по кнопке ++Открыть++. Выберите вкладку Страницы.

На этой вкладке в таблице выводятся все структурные идентификаторы страниц и элементов, которые можно изменить.

Обзор основных элементов диалогового окна:

Для источника (то есть исходного проекта или исходного макроса) и целевого проекта в таблице отображаются идентификаторы структуры, указанные в соответствующей структуре проекта, а также номера страниц, номера листов и описания страниц.

### Источник

В полях для структурного идентификатора (например, "=" для установки или "+" для места установки), номера страницы, дополнительного поля номера листа и описания страницы выводятся соответствующие данные исходного проекта или макроса. Здесь их обработать нельзя.

### Цель

Данные исходного проекта или макроса по умолчанию копируются в поля целевого проекта, где их можно изменить: в полях отдельных структурных идентификаторов можно, щелкнув кнопку ++...++, открыть соответствующее диалоговое окно для выбора структурных идентификаторов, имеющихся в целевом проекте. Номер страницы, дополнительного поля номер листа и описание страницы могут быть переписаны вручную.

### Перезаписать

Если этот флажок установлен, уже имеющаяся страница будет перезаписана при вставке страницы или макроса страницы.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

### Автоматическое имя страницы

Установите данный флажок, чтобы наивысший имеющийся номер страницы определялся относительно структуры и для столбца Имя страницы предлагался следующий по порядку наивысший номер / наивысшие номера.

!!! example "Пример:"

    Вы открыли образец проектаESS_Sample_Projectи в навигаторе страниц выделили страницу=GB1+A1&EFS1/1с описанием страниц Электропитание 400/230 В. Затем вставьте односторонний макрос страницы. Сначала в диалоговом окне Адаптировать структуруна вкладке Страницыпредварительно устанавливается имя страницы созданного макроса страницы (например,5). После этого установите флажок Автоматическое имя страницыи имя страницы (поскольку в подструктуре наивысшим имеющимся номером страницы является1) изменится на2.

### [Нумерация]

Открывает диалоговое окно [Нумеровать страницы](pagebrowsergui_d_seitennummerierung.md). В нем можно указать настройки нумерации страниц перед вставкой.

### [Сбросить]

Сбрасывает страницы на состояние исходной структуры. Это может быть полезным тогда, когда необходимо вставить в проект макрос для нескольких страниц, которые должны сохранить исходные структурные обозначения.

### [Дополнительно]

Пункт меню |  Значение
---|---
Обработать структуры |  Открывает диалоговое окно Обработать структуры. Позволяет задавать / обрабатывать главные идентификаторы и до 9 подидентификаторов для отдельных блоков идентификаторов.

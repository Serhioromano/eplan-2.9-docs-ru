# Создать форму

При создании формы открывается диалоговое окно Свойства формы — <Имя формы>, в котором устанавливаются важнейшие свойства новой формы. (Это диалоговое окно может быть вызвано также в последствии для ***открытой*** формы, это позволит Вам в любое время дорабатывать свойства.)

Условие:

Вы открыли проект.

1. Сервисные программы > Основные данные > Форма > Создать
2. В диалоговом окне Создать форму выберите дисковод и каталог, в котором должна быть сохранена новая форма.
3. Введите в поле Имя файла информативное имя для формы.
4. Из раскрывающегося списка Тип файла выберите тип файла (например, содержание, спецификация кабеля и т. д.), который необходимо создать.
5. В поле Путь выберите пункт всплывающего меню Сбросить на стандарт, если требуется вернуться к стандартной настройке дисковода и каталога. (Стандартный каталог является либо специфической предварительной программной настройкой <Дисковод программы>:\<Каталог программы>\Формы\<Идентификатор фирмы>, либо настройкой, установленной через меню Параметры > Настройки > Пользователь > Управление > Каталоги.)
6. Щелкните по кнопке ++Сохранить++.
7. В диалоговом окне Свойства формы — <Имя формы> при необходимости выберите из раскрывающегося списка Тип формы другой тип формы для новой формы.
8. Определите значения для свойств, либо введя или перезаписав запись или же выбрав значение по умолчанию из раскрывающегося списка, или перейдите в специальное диалоговое окно, где можно ввести другие данные.
9. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Новая форма сохраняется согласно вашим данным, причем расширение имени типа файла зависит от выбранного типа формы и выводится в редакторе форм.

При необходимости Вы можете проконтролировать форму во время обработки, чтобы быстро заметить неправильные записи и их устранить.

!!! tip "Совет:"

    Можно ввести имеющиеся формы с помощью функции перетаскивания мышью, если перетянуть выделенную форму из проводника Windows в диалоговое окно навигатора Страницы — <Имя проекта> проекта.

# Рисование блоков ломаными линиями

Обозначения местоположения, черные ящики, блоки ПЛК, рамки макросов и границы схемы сверления могут также обозначаться многоугольниками. Для этого в библиотеке символов SPECIAL появились следующие символы:

* DC2: Черный ящик в виде ломаной линии
* DCF2: Черный ящик Fluid в виде ломаной линии
* PLCC3: Блок ПЛК в виде ломаной линии
* SC2: Обозначение местоположения в виде ломаной линии
* MC2: Рамка макроса в виде ломаной линии
* BB2: Граница схемы сверления в виде ломаной линии.

Условие:

Вы открыли проект и находитесь на странице схемы соединений в Графическом редакторе.

1. Выберите пункт меню Вставить > Символ.
2. Выберите символ для блока как ломаная линия из представления в виде дерева или списка библиотеки символов SPECIAL, например DC2 для черного ящика.
3. Укажите начальную точку ломаной линии и щелкните левой клавишей мыши.
4. Укажите следующие точки ломаной линии и щелкните левой клавишей мыши, чтобы разместить каждую из этих точек.
5. Завершите чертеж блока в пункте всплывающего меню Закрыть ломаную линию.
6. В диалоговом окне Свойства ++...++ введите специфические для блока данные устройства.
7. Щелкните по кнопке ++OK++.

!!! tip "Совет:"

    Быстро выбрать символ для черчения блока в виде ломаной линии можно путем выбора соответствующих пунктов меню для вставки блока (напр., Вставить > Блок / вывод устройства / монтажная плата > Черный ящик) и последующего нажатия клавиши ++Backspace++. В открывшемся диалоговом окне Выбор символа символы уже отфильтрованы: отображаются только символы с одинаковой категорией функции.

!!! note "Замечание:"

    * При включенном захвате сетки (Параметры > Захват сетки) повторно нарисованный блок расположен точно на сетке, на которой он был размещен; если захват сетки отключен, блок можно нарисовать произвольно рядом с сеткой.
    * В отличие от прямоугольных блоков блоки в виде ломанных линий обладают только одним порядком свойств, относящимся к точке вставки. При размещении ломаной линии в качестве точки вставки выбирается угловая точка, расположенная с самого левого края. Если несколько точек размещены друг над другом, то в качестве точки вставки используется самая верхняя точка.
    * Начертить ***рамки макросов*** и ***границы схемы сверления*** можно только в проектах макросов.

# Начертить заблокированные области

На монтажной плате можно указать области, размещение изделий в которых запрещено. Заблокированные области являются самостоятельными объектами, которые представлены в виде прямоугольника.

Условие:

Вы открыли проект.

1. Данные проекта > Устройства / Изделия > Навигатор компоновки электрошкафа 2D
2. Выберите в графическом редакторе ту страницу компоновки электрошкафа, на которой следует начертить заблокированную область.
3. Выберите Всплывающее меню > Заблокированная область.
4. Начертите прямоугольник, представляющий заблокированную область.

!!! info "Для сведения:"

    Выполняется создание заблокированной области.

!!! note "Замечание:"

    Если совершается попытка, установить размещение изделия в заблокированной области, размещение не выполняется и выводится соответствующее сообщение.

# Экспортировать данные проводного монтажа

Условия:

* Вы открыли проект.
* Открыт навигатор проводного монтажа.

1. Выделите одно или несколько определений проводного монтажа в графическом редакторе или в навигаторе проводного монтажа.
2. Выберите следующие пункты меню: Данные проекта > Соединения > Экспортировать > Данные проводного монтажа

!!! info "Для сведения:"

    Открывается диалоговое окно Экспортировать данные проводного монтажа. Выбранный проводной монтаж отображается в поле Источник.

3. В поле Имя файла введите имя, под которым будет сохранен файл экспорта, или выберите его интерактивно с помощью кнопки ++...++.

!!! info "Для сведения:"

    При экспорте в уже имеющийся документ отображается запрос. Потребуется дополнить или перезаписать файл.

4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Генерируется файл с данными проводного монтажа (*.hpdex).

!!! note "Замечание:"

    Если при экспорте данных проводного монтажа установлен флажок Применить ко всему проекту, экспортируется не только весь предварительно определенный проводной монтаж и соответствующие компоненты, но и весь проект со всеми данными. В этом случае в поле Источник не будут перечисляться имена проводного монтажа всего проекта, и поле будет пустым.

# Вкладка Экспорт

Вы открыли проект.

* Проект > Опубликовать. Щелкните в поле Схема диалогового окна Опубликовать на ++...++.
* Параметры > Настройки > Пользователь > Интерфейсы > Опубликовать.

На этой вкладке устанавливаются настройки для экспорта проекта в формат EPDZ. Формат EPDZ доступен для чтения мобильными приложениями на мобильных устройствах.

Обзор основных элементов диалогового окна:

Групповое поле 'Средство экспорта'

### Носитель данных

Если выбрано это средство экспорта, файл EPDZ выводится на носителе данных.

В поле Каталог вывода можно ввести место хранения.

Кнопка ++...++ открывает диалоговое окно Выбрать каталог для выбора любого каталога для экспорта.

### Эл. почта

При выборе этого средства экспорта файл EPDZ отправляется по электронной почте.

!!! note "Замечание:"

    Мобильные устройства не читают разделенные файлы. Поэтому при пересылке файла EPDZ по электронной почте возможность разделить этот файл на несколько небольших частей отсутствует.

Групповое поле "Вывод"

### Исходный язык

Если этот флажок установлен, тексты в файле EPDZ выводятся на настроенном для экспортируемого проекта исходном языке. Если для исходного языка выбран параметр ##_## (Язык диалогового окна), в качестве исходного языка автоматически используется язык диалогового окна, настроенный в данный момент в EPLAN.

### Альтернативный язык

Если этот флажок установлен, можно выбрать, на каком альтернативном языке должны отображаться переведенные тексты файла EPDZ. При этом вы можете выбрать язык из списка настроенных для проекта языков перевода. Тогда при экспорте в EPDZ будет использоваться только этот язык, установленные языки отображения будут игнорироваться.

### Скопировать внешне связанные документы в каталог вывода

Установите этот флажок, чтобы внешние / связанные документы, которые не сохранены в проекте и отсутствуют в стандартном каталоге для документов, при выводе также копировались в целевой каталог файла EPDZ. К внешним связанным документам относятся:

* страницы с типом "Внешний документ";
* гиперссылки, связанные с документами в форме файла;
* ссылки на документы в форме файла в текстах или значениях свойств.

Если этот флажок снят, то в файле EPDZ сохраняются только внутренние связанные документы. Все документы, которые не сохранены в проекте и отсутствуют в стандартном каталоге для документов, игнорируются.

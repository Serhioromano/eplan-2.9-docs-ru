# Диалоговое окно Генерировать функции

Вы открыли проект. Открыт навигатор, в котором отображаются данные устройства. Выделите ОУ. Выберите пункт всплывающего меню Новые функции.

В данном диалоговом окне в соответствующем навигаторе для нового или уже имеющегося ОУ можно сгенерировать несколько функций по определенному образцу нумерации. Эта функция доступна только для определенных устройств (клемм, контактов штекеров, контактов катушек или реле, выводов устройств или ПЛК, а также сборных шин). С помощью этой функции невозможно создать функции без выводов устройства (например, выводы устройства ПЛК с определением функции "Вывод устройства ПЛК, внутр., общ."). В зависимости от выбранного определения функции, образец нумерации определяет обозначения клемм, контактов или выводов устройства.

Обзор основных элементов диалогового окна:

### Полное ОУ

В этом поле показывается полное ОУ, которому присвоены новые функции. Показывается ОУ устройства, которое вы выделили в навигаторе. Вы можете изменить ОУ или ввести новое.

### Образец нумерации

Введите в это поле образец нумерации. Для клемм, контактов штекеров или катушек образец нумерации задает обозначения клемм и контактов штекеров и катушек. Для других устройств образец нумерации определяет обозначения выводов устройства.

### Определение функции

В этом поле находится определение функции, которое применяется для новых функций. Кнопка ++...++ открывает диалоговое окно Определения функций и обеспечивает выбор другого определения функций.

### Число выводов устройства

Значение в этом поле указывает, сколько выводов устройства имеет заново сгенерированная функция. Число выводов устройства зависит от выбранного определения функции.

### Число новых функций

Значение в этом поле указывает, сколько новых функций будет сгенерировано. Число новых функций зависит от образца нумерации и числа выводов устройства, которые указаны в определении функции.

### Многоуровневая клемма

Если этот флажок установлен, то из сгенерированных клемм составляются многоуровневые клеммы.

### Число уровней

Значение в этом поле указывает, сколько уровней имеет каждая сгенерированная многоуровневая клемма.

### Нумеровать с ориентацией на уровни

Если флажок установлен, тогда клеммы нумеруются по уровням. Если флажок не установлен, тогда клеммы нумеруются по клеммам.

# Каналы ПЛК

На одной карте ПЛК — в зависимости от работы с цифровыми или аналоговыми узлами входа или выхода — можно расположить несколько каналов ПЛК. Часто используются узлы ПЛК с 8, 16 или 32 каналами. Каждому каналу ПЛК присвоен однозначный адрес.

Помимо сигнального вывода устройства канал может содержать несколько взаимосвязанных выводов устройства ПЛК. Так, например, некоторые производители размещают на картах ПЛК дополнительные точки подвода питания для каждого канала. Благодаря этому можно, например, 3-проводной бесконтактный переключатель подключить непосредственно к карте ПЛК. В этом случае два вывода устройства ПЛК подают необходимое рабочее напряжение на датчик, а третий вывод устройства ПЛК перехватывает сигнал этого датчика.

Если входы и выходы проектируются в EPLAN с точками подвода питания, присвоение каналов осуществляется графически или посредством обозначения каналов.

Для канала релевантны все данные, имеющие значение и в списках присвоений, например символический адрес или функциональный текст. Эти данные записываются или передаются на тот вывод устройства ПЛК, определением функции которого являются "DE=Дискретный вход", "AE=Аналоговый вход", "DA=Дискретный выход" или "AA=Аналоговый выход".

### Каналы с несколькими выводами устройства В/В

Канал может содержать несколько выводов устройства В/В, только один из которых может быть активен. Все остальные выводы устройства В/В в рамках одного канала должны быть обозначены как деактивированные выводы устройства, можно воспользоваться свойством Деактивированный вывод устройства В/В. Таким образом один и тот же макрос и одно и то же изделие можно использовать для различных видов монтажа.

!!! example "Пример:"

    Макрос содержит один аналоговый канал с двумя выводами устройства В/В, которым приписано свойство Деактивированный вывод устройства В/В. Этот макрос при различных видах монтажа можно использовать для измерения тока и напряжения. После вставки макроса активируйте вывод устройства В/В, соответствующий данному виду монтажа. Для этого необходимо деактивировать свойство Деактивированный вывод устройства В/В.

Деактивированные выводы устройства считаются источником питания. Например, при адресации заполненный адрес заменяется на адрес из канала.

### Аварийные блоки управления

Для аварийных блоков управления часто применяются карты ПЛК, которые могут проверить несколько выводов устройства на совпадение или несовпадение. Программы конфигурации ПЛК часто управляют несколькими каналами с одним и тем же символическим адресом для таких важных с точки зрения безопасности компонентов. Поскольку символический адрес должен быть уникальным в пределах ЦПУ, в пределах канала может быть активен только один вывод устройства В/В. Поэтому пометьте остальные выводы устройства свойством Деактивированный вывод устройства В/В. Таким образом эти выводы устройства при обмене данными ПЛК не будут заполняться ложными данными для программы конфигураций ПЛК.

Для переадресации этого вывода устройства в EPLAN введите то же значение для объединенных выводов устройства в поле Обозначение канала.

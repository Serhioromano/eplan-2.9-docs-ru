# EPLAN Fluid: Структура устройства

В EPLAN под "структурой устройства" понимается объединение всех структур обозначения для страниц и устройств, используемых в проекте.

Понятие "Устройства Fluid" в EPLAN Fluid объединяет следующие типы устройств:

* ***Устройства Fluid общего назначения*** (сюда можно отнести, например, клапаны, фильтры, цилиндры и т.д.)
* ***Распределители и соединители Fluid***.

Для каждого из этих двух типов устройств может быть назначена ***своя независимая*** структура идентификаторов.

Предшествующие знаки и блоки идентификаторов, при помощи которых в EPLAN Fluid можно составить соответствующие стандарту и иерархически структурированные обозначения устройств, перечислены ниже:

Предшествующий знак |  Блок идентификаторов
---|---
== |  Функциональное присвоение
= |  Установка
++ |  Место сборки
+ |  Место установки
|  Номер установки
# |  Определенная пользователем структура
- |  Устройство (состоит из элементов идентификатора "Предш. цифра", "Буквенное обозначение", "Счетчик", "Разделитель" и "Подсчетчик")

Благодаря этим блокам идентификаторов EPLAN Fluid предоставляет множество различных способов структурирования. Например, можно структурировать ОУ согласно стандарту [EN 81436](projectstructure_k_referenzkennzeichnung.md) или ***DIN ISO 1219-2***.

### Обозначение устройства в соответствии со стандартом Fluid-Техники DIN ISO 1219-2

По стандарту DIN ISO 1219-2 предписаны следующие ключи обозначения для функциональных элементов Fluid-техники:

* ***Обозначение установки***
* ***-*** (разделитель)
* ***Ключ среды***
* ***Номер логической ячейки***
* ***.*** (Разделитель)
* ***Номер функц. элемента***

Вместо заданной стандартом составной части идентификатора ***Обозначение установки*** программа предоставляет блок идентификаторов ***Номер установки***. Так как в EPLAN Fluid отсутствуют специальные блоки идентификаторов для требуемых по стандарту структурных элементов "Ключ среды", "Номер логической ячейки" и "Номер функционального элемента", эти элементы ключа обозначения создаются с помощью настройки проекта Fluid: Использовать обозначение раздела в качестве буквенного обозначения (ключ среды). Данная настройка находится в пункте Параметры > Настройки > Проекты > "Имя проекта" > Устройство > ОУ.

Требуемые по стандарту DIN ISO 1219-2 элементы идентификатора "Ключ среды", "Номер логической ячейки" и "Номер функционального элемента" в EPLAN Fluid автоматически присваиваются элементам блока идентификаторов "Устройство", т. е. "обычным" элементам ОУ (см. таблицу ниже).

Структурные элементы в соответствии с DIN ISO 1219-2 |  Составные части идентификатора в EPLAN Fluid
---|---
Обозначение установки |  Соответствует блоку идентификаторов ***Номер установки***. Номер установки вводится без предшествующего знака.
- |  Разделитель (четко определен)
Ключ среды |  Соответствует ***Обозначению раздела***. Присваивается составной части ОУ ***Буквенное обозначение***.
Номер логической ячейки |  Присваивается составной части ОУ ***Счетчик***.
. |  Разделитель (четко определен)
Номер функц. элемента |  Непрерывный номер. Присваивается составной части ОУ ***Подсчетчик***.

!!! example "Пример:"

    P = ключ среды для обозначения раздела "Пневматика"

Присвоение структурных элементов отображается в диалоговом окне Полное ОУ в виде таблицы. Для получения доступа к обозначенным в соответствии с DIN ISO 1219-2 устройствам Fluid следует обратиться к диалоговому окну Свойства ++...++ и рядом с полем Полное ОУ нажать на ++...++.

!!! note "Замечание:"

    * В соответствии с DIN ISO 1219-2 обозначение установки (в EPLAN "Номер установки") можно пропустить, если Fluid-схема состоит только из одной установки.
    * Согласно данному стандарту можно пропустить также ключ среды (в EPLAN "Обозначение раздела"), если внутри установки используется только одна среда. Это требование удовлетворяется в EPLAN Fluid с помощью настройки проекта Fluid: Скрыть буквенное обозначение (ключи среды) в графике (меню Параметры > Настройки > Проекты > "Имя проекта" > Устройство > ОУ). Для этого нужно снять флажок. Обозначение раздела больше не отображается только графически в видимом ОУ Fluid-функции, в видимом и полном ОУ диалогового окна свойств оно сохраняется в данной настройке.

!!! tip "Совет:"

    EPLAN Fluid оказывает поддержку при проектировании согласно стандарту DIN ISO 1219-2. С этой целью после установки программы в распоряжении имеется шаблон проекта FL_1219-2_tpl001.ept. При использовании данного шаблона для устройств Fluid, а также для распределителей и соединителей Fluid автоматически предлагаются правильные настройки для соответствующего стандартам обозначения устройства.

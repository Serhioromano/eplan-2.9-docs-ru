# Вкладка Точка определения изделия

Вы открыли проект.

* В Графическом редакторе выделена точка определения изделия. Обработать > Свойства. В диалоговом окне Свойства ++...++ выберите вкладку Точка определения изделия.
* Вставить > Точка определения изделия. В диалоговом окне Свойства ++...++ выберите вкладку Точка определения изделия.

В этом диалоговом окне вы можете указать изделия, не присвоенные устройству, на графике в схеме соединений. Каждой точке определения изделия (как и любой главной функции) могут быть присвоены максимум 50 изделий.

Чтобы передать изделия, указанные на точке определения изделия, в спецификацию изделий или в групповую спецификацию изделий с сохранением структуры, можно указать на точке определения изделия свойства Видимое ОУ и Полное ОУ. Таким образом изделия присваиваются определенному структурному идентификатору. Тем не менее точки определения изделия не являются устройствами и не перечисляются в навигаторе устройств и в списке обозначений устройств.

Обзор основных элементов диалогового окна:

### Видимое ОУ

Введите здесь ОУ, которое должно отображаться в графическом редакторе у точки определения изделия. Если вы измените одну из структур обозначений в нижней области диалогового окна, то в это поле будет скопирована соответствующая измененная запись.

### Полное ОУ

В этом поле отображается полное ОУ (т. е. структурный идентификатор текущего окружения + видимое ОУ) видимого ОУ / функционального элемента. С помощью кнопки ++...++ откройте диалоговое окно Полное ОУ. Здесь можно обработать структурный идентификатор и, например, выбрать другое место установки.

### Структурный идентификатор

Для всех структурных идентификаторов, используемых в текущем проекте, при помощи кнопки ++...++ можно открыть диалоговое окно <Структурный идентификатор> (<Имя проекта>) и присвоить точке определения изделия другой структурный идентификатор; поля Полное ОУ и Видимое ОУ сразу же автоматически адаптируются, причем поле Видимое ОУ по умолчанию отображает только измененные структурные идентификаторы.

!!! note "Замечание:"

    В зависимости от настроек в меню Параметры > Настройки > Пользователь > Отображение > Идентификатор новые идентификаторы, возникающие на основании данных, заданных в полях Видимое ОУ и Полное ОУ, автоматически сортируются в алфавитном порядке или в конце.

### Раздел

Раздел перенимается из номера изделия в базе данных изделий и автоматически заполняется первым определением настройки раздела изделия. При необходимости выберите из раскрывающегося списка другой раздел.

### Подраздел

При необходимости здесь можно указать подраздел устройства. Данная информация не считывается из базы данных изделий, но должна вводиться дополнительно.

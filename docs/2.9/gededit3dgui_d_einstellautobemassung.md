# Диалоговое окно Настройки: Автоматическое проставление размеров

* Параметры > Настройки > Проекты > "Имя проекта" > Графическая обработка > Автоматическое проставление размеров
* Вы выделили обзор модели. Обработать > Свойства. В диалоговом окне Свойства в свойстве Обзор модели: Схема для автоматического проставления размеров щелкните по кнопке ++...++.

В данном диалоговом окне установите настройки для ***автоматического*** проставления размеров функциональных элементов в обзорах модели, двухмерных отображениях отверстий и развертках шины. Эти настройки ***не применяются*** для размеров, проставленных вручную.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

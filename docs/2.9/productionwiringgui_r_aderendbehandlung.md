# Вкладка Обработка концов проводников

* Параметры > Настройки > Фирма > Сборка проводов > "Машина" > Обработка концов проводников
* Сервисные программы > Данные изготовления > Сборка проводов > "Машина". Щелкните по кнопке ++...++ рядом с полем Схема. Выберите вкладку Обработка концов проводников.

В данной вкладке осуществляется машинное присвоение обработки концов проводов фактической обработке концов проводников.

Обзор основных элементов диалогового окна:

### Обработка концов проводов

В раскрывающемся списке можно выбрать все определения, установленные в проекте и в настройках фирмы для выполнения концов проводов.

### Одиночная обработка концов проводников

В этом столбце обработке концов проводов, расположенной впереди, при помощи наконечников присваивается оснащение концов жил одиночных проводов. Определение выбранных значений для данного элемента диалогового окна дается во вкладке [Машинная команда](productionwiringgui_r_maschinenbefehl.md).

### Двойная обработка концов проводников

В этом столбце обработке концов проводов, расположенной впереди, на машине присваивается обработка концов жил двойных проводов. Определение выбранных значений для данного элемента диалогового окна дается во вкладке [Машинная команда](productionwiringgui_r_maschinenbefehl.md).

### Двойная обработка концов проводников посередине

Настройка в этом столбце использует [Сдвоенный опрессовочный модуль](productionwiringgui_r_einstellungenkomax.md) для машин Komax. Если данный модуль активирован на вкладке Настройки диалогового окна Настройки: Komax, в середине двойной последовательности проводов можно изготовить двойную трубку, при этом начало и конец двойной последовательности просто зачищаются.

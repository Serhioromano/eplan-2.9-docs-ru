# Вкладка Определение функции

Сервисные программы > Изделие > Управление. Вы отметили в дереве или в списке изделие из главной группы продуктов Механика. Выберите вкладку Определение функции.

Изделия из главной группы продуктов Механика вместо вкладки Шаблоны функций имеют вкладку Определение функции. На этой вкладке настроено присвоение определения функции механических функциональных элементов группе продуктов в базе данных изделий.

Обзор основных элементов диалогового окна:

### Определение функции

Настроенное здесь определение функции отображается также в диалоговом окне Свойства (условное обозначение): Размещение изделия (3D).

### Функциональный элемент

Группа продуктов внутри главной группы продуктов Механика, присвоенная определению функции изделия.

Настроенная здесь группа продуктов отображается также в диалоговом окне Свойства (условное обозначение): Размещение изделия (3D).

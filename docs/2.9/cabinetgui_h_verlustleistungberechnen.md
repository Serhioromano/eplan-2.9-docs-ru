# Рассчитать общую мощность потерь на распределительных устройствах

Чтобы правильно рассчитать общую мощность потерь, необходимо подготовить занятые в ней компоненты посредством настройки удельных свойств:

* Рамочные условия определяются как свойства проекта и на электрошкафу.
* Данные изделия должны определяться как свойства для каждого устройства и/или сборной шины. Подходящие значения берутся из входящих в комплект поставки технических паспортов производителя.

Условия:

* Вы открыли проект.
* Навигатор пространства листа открыт, и открыто пространство листа.

1. Занесите в свойства ***проекта*** значения для следующих свойств.

Имя свойства |  Примерное значение
---|---
Тепловой расчет: Напряжение |  230 В
Тепловой расчет: Частота |  50 Гц
Тепловой расчет: Коэффициент одновременности |  0,75
2. В свойства ***электрошкафов*** (= рассчитываемые температурные области) занесите значения рамочных условий расчета мощности потерь.

Имя свойства |  Примерное значение
---|---
Тепловой расчет: Климатическая область |  1
Тепловой расчет: Макс. температура снаружи |  50°
Тепловой расчет: Макс. температура внутри |  65°
Тепловой расчет: Мин. температура снаружи |  0°
Тепловой расчет: Мин. температура внутри |  0°
Тепловой расчет: Ситуация установки |  Одиночный корпус, отдельно стоящий
3. ***Для устройств*** занесите в данные изделия на вкладке Свойства каждого изделия устройства значение мощности потерь.
4. Если ***для устройств*** вы хотите задать значение, отличное от глобального свойства проекта Тепловой расчет: Коэффициент одновременности, занесите в свойства размещения изделий значение для свойства [Тепловой расчет: Коэффициент одновременности (устройство)](cabinetgui_k_verlustleistungprinzip.md).
5. ***Для сборных шин*** занесите в данные каждого изделия шины на вкладке Свойства значения следующих свойств изделия. Некоторые значения заполняются самой программой в зависимости от выбранного материала.

Имя свойства |  Примерное значение
---|---
Поставляемая длина |  900 мм
Поперечное сечение шины |  40 мм²
Материал шины |  Медь, Cu-ETP CW004A
Проводимость (при +20 °C) |  57 МСм/м (= мегасименс на метр)
Температурный коэффициент |  0,004 1/K
6. В ***токопроводящих сборных шинах*** на размещении изделия занесите значение в свойство Тепловой расчет: Рабочий ток, например "800 А".
7. Выделите в навигаторе пространств листа пространство листа, общую мощность потерь которого нужно рассчитать.
8. Выберите пункт меню Сервисные программы > Отчеты > Рассчитать общую мощность потерь.

9. В диалоговом окне Рассчитать мощность потерь выберите анализируемые температурные области.
10. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Общая мощность потерь рассчитывается на основании заданных значений и заносится в свойство проекта Тепловой расчет: Общая мощность потерь для температурной области [n].

!!! note "Замечание:"

    Вводите все данные о мощности потерь в следующей форме: значение и единица измерения в ваттах, например "10Вт".

    При этом обязательно указывайте единицу измерения. Возможно использование пробела, например "10 Вт".

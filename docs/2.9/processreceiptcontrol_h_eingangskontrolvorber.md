# Контроль проекта

При проверке проекта сравниваются друг с другом настройки и свойства проекта для сравнения и проекта с замечаниями. Также выполняются контрольные прогоны для каждого проекта с замечаниями отдельно.

Условия:

* [Выбрать базу данных управления проектами](prjmanagementgui_h_projekteverwalten.md)
* [Считать каталоги и проекты](prjmanagementgui_h_verzeichniseinlesen.md)
* Считанный каталог содержит возвращенный проект с замечаниями.
* Вы имеете доступ к исходному проекту, который вы хотели бы использовать в качестве проекта для сравнения.

1. Проект > Управление
2. Выделите в управлении проектами проект, содержащий обновленное состояние проекта.
3. Выберите ++Дополнительно++ > Проверить проект.
4. Нажмите рядом с полем Выбрать проект для сравнения на кнопку ![](images/all_new_as.png){: .ui-icon }(Создать).
5. Выберите в отобразившемся диалоговом окне исходный проект, который является проектом для сравнения, и нажмите кнопку ++Открыть++.
6. Чтобы выполнить специфическое для проекта сравнение свойств в ходе проверки проекта, активируйте флажок Сравнить свойства и выберите из раскрывшегося ниже списка Схема имеющуюся схему сравнения.

!!! info "Для сведения:"

    С помощью кнопки ++...++ Вы можете создать новую [схему сравнения свойств](processreceiptcontrol_h_vergleichsschema.md).

7. Чтобы выполнить сравнение свойств в ходе проверки проекта, активируйте флажок Сравнить свойства и выберите из раскрывшегося ниже списка Схема имеющуюся схему сравнения.

!!! info "Для сведения:"

    С помощью кнопки ++...++ Вы можете создать новую [ схему сравнения свойств](processreceiptcontrol_h_eigenschaftenvergleich.md).

8. Чтобы выполнить контрольные прогоны в ходе проверки проекта, активируйте флажок Сравнить настройки, выберите из раскрывшегося ниже списка Cхема имеющуюся схему сравнения.

!!! info "Для сведения:"

    С помощью кнопки ++...++ вы можете создать новую [ схему контрольных прогонов для проверки проекта](processreceiptcontrol_h_pruefschema.md).

9. Чтобы сравнить структуры обоих проектов между собой, активируйте флажок Сравнить структуры проектов.
10. Чтобы остановить сравнение проектов при первом изменении, активируйте флажок Остановить сравнение при обнаружении первого различия.
11. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Процесс сравнения проектов завершен; необходимые контрольные прогоны выполнены.

!!! info "Для сведения:"

    В диалоговом окне Проверка проекта: результаты EPLAN автоматически отображает общий результат проверки проекта.

!!! info "Для сведения:"

    С помощью Данные проекта > Сообщения > Управление подробно выводятся результаты.

# Диалоговое окно Автоматический выбор кабеля

Вы открыли проект. Данные проекта > Кабели > Навигатор. Выделите несколько значений, содержащих кабели, и выберите пункт всплывающего меню Автоматический выбор кабеля.

В этом диалоговом окне задаются настройки для автоматического выбора кабеля. Выбор кабеля можно выполнять по желанию пользователя для всего проекта или только для выделенных кабелей.

Обзор основных элементов диалогового окна:

### Настройки

Выберите схему, чтобы определить настройки для автоматического выбора кабелей. С помощью кнопки ++...++ откройте диалоговое окно [Настройки: Автоматический выбор кабеля](cablegui_d_einstellungenautomatischekabelauswahl.md), в котором можно обработать схемы и управлять ими.

### Только автоматически сгенерированные или названные кабели

Если этот флажок установлен, выбор кабеля осуществляется только для автоматически сгенерированных или названных кабелей. Иначе автоматический выбор кабелей будет осуществляться и для кабелей, сгенерированных вручную или кабелей с именами.

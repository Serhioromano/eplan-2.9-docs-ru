# Разместить медную шину

Условия:

* Вы открыли проект. Открыто одно пространство листа.
* В пространстве листа содержатся размещенные компоненты электрошкафа (монтажная плата, профили, системы сборных шин).

1. Выберите пункты меню Вставить > Сборная шина (изогнутая).

!!! info "Для сведения:"

    Отобразится диалоговое окно функционального элемента Сборная шина (изогнутая).

2. Рядом с полем Номер изделия щелкните мышкой по кнопке ++...++.

!!! info "Для сведения:"

    В базе данных изделий откроется структура дерева с подгруппой с изделиями-шинами.

3. Выберите нужное изделие, которое должно использоваться для изогнутой сборной шины, и нажмите кнопку ++ОК++.

!!! info "Для сведения:"

    Номер выбранного изделия будет занесен в диалоговое окно Сборная шина (изогнутая).

!!! info "Для сведения:"

    Значения ширины и глубины, определенные на изделии, используются для установления размеров поперечного сечения шины.

4. Рядом с полем Профиль щелкните мышкой по кнопке ++...++.
5. В диалоговом окне Выбрать профиль выберите нужный чертеж профиля, который должен задавать форму изогнутой сборной шины и нажмите кнопку ++Открыть++.

!!! info "Для сведения:"

    Чертеж профиля будет занесен в диалоговое окно функционального элемента Сборная шина (изогнутая).

6. Введите в поле Радиус изгиба значение для радиуса сгибов.

!!! info "Для сведения:"

    Это значение используется при генерировании графики шины и кладется в основу расчетов в таблице последовательности изгибов. Поэтому радиус изгиба влияет также на выбор инструментов для сгибания на сборочных станках.

7. Выберите один из параметров: Плоский изгиб или Изгиб на ребро в зависимости от того, какую форму должна иметь шина.
8. Если все поля заполнены правильно, нажмите кнопку ++OK++.

!!! info "Для сведения:"

    Медная шина появится рядом с курсором на точке захвата, указанной на чертеже профиля.

9. Перенесите функциональный элемент к монтажной поверхности, на которой его следует разместить.

!!! info "Для сведения:"

    Отображаются трехмерные точки захвата.

!!! info "Для сведения:"

    Монтажная поверхность автоматически активируется.

!!! info "Для сведения:"

    В зависимости от выравнивания монтажной поверхности, шина поворачивается таким образом, чтобы находиться по отношению к ней параллельно.

10. Чтобы ввести смещение для размещения, используйте пункт всплывающего меню Опции размещения.
11. Чтобы повернуть шину на 180°, используйте пункт всплывающего меню Перевернуть положение установки.

!!! info "Для сведения:"

    Таким образом, можно разместить шину на обратной стороне других компонентов, не переключая вид.

12. Чтобы сгенерировать новый жгут при размещении шины, нажмите клавишу ++N++.
13. С помощью щелчка разместите шину на нужном месте.

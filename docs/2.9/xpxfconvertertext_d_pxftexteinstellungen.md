# Диалоговое окно Настройки: Экспорт текстового файла

* Вы открыли проект. Сервисные программы > Внешняя обработка свойств > Настройки ( ++...++). Выберите вкладку Общ.. Выберите из этого раскрывающегося списка Тип файла одну из записей "Текст, разделенный табуляторами", "Файл в формате Юникод с разделением табуляцией (*.txt)" или "XML-файл (*.edc)" и щелкните ++Параметры++.
* Параметры > Настройки > Пользователь > Интерфейсы > Внешняя обработка ( ++...++). Выберите вкладку Общ.. Выберите из этого раскрывающегося списка Тип файла одну из записей "Текст, разделенный табуляторами", "Файл в формате Юникод с разделением табуляцией (*.txt)" или "XML-файл (*.edc)" и щелкните ++Параметры++.

В этом диалоговом окне указываются настройки для внешней обработки страниц, функций или соединений с помощью текстового редактора.

Обзор основных элементов диалогового окна:

### Прикладная программа

Укажите здесь имя (с указанием дисковода и пути) прикладной программы, которую требуется использовать для соответствующего формата, или же нажмите кнопку ++...++ для интерактивного выбора программы.

Выбрав команды Всплывающее меню > Вставить переменную пути, перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути, специфических для EPLAN.

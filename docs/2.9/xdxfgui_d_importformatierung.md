# Диалоговое окно Форматир. импорта

Вы открыли проект. Вы выбрали подходящую схему в настройках (Параметры > Настройки > Пользователь > Интерфейсы > Экспорт и импорт DXF / DWG), а на вкладке Импорт выбрали параметр Корректировка вручную.

* Страница > Импортировать > DXF / DWG. Выберите в диалоговом окне Выбор файла DXF/DWG (как минимум) один файл и щелкните по ++Открыть++. Измените при необходимости в диалоговом окне Импорт DXF/DWG настройки импорта и запустите импорт нажатием на ++OK++. Определите в диалоговом окне Присвоить страницы порядок обработки создаваемых страниц и нажмите ++OK++.
* В Графическом редакторе открыта страница проекта. Вставить > Графика > DXF / DWG. Выберите в диалоговом окне Выбор файла DXF/DWG файл и щелкните по ++Открыть++. Измените при необходимости в диалоговом окне Импорт DXF/DWG настройки импорта и запустите импорт нажатием на ++OK++.
* Сервисные программы > Генерировать макросы > Из файлов DXF / DWG. Выберите в диалоговом окне Выбор файла DXF/DWG (как минимум) один файл и щелкните по ++Открыть++. Измените при необходимости в диалоговом окне Импорт DXF/DWG настройки импорта и запустите импорт нажатием на ++OK++.

В этом диалоговом окне можно дополнительно обработать вручную автоматически определенные границы чертежа.

Обзор основных элементов диалогового окна:

### Масштабирование

Если чертеж больше рамки, то автоматически определяется коэффициент масштабирования. Выберите из раскрывающегося списка вариант масштабирования страницы после импорта:

* Автоматически: страница масштабируется согласно автоматически определенному коэффициенту масштабирования. Коэффициент масштабирования отображается в полях Горизонтальное/вертикальное масштабирование, но его нельзя изменить.
* Вручную: в полях Горизонтальное / вертикальное масштабирование можно изменить определенный автоматически коэффициент масштабирования.
* Нет: коэффициент масштабирования имеет значение "1", и его нельзя изменить.

### Позиция

Выберите из раскрывающегося списка вариант позиционирования страницы после импорта:

* По центру: чертеж выравнивается посередине рамки.
* Внизу слева: нижний левый угол чертежа размещается на точке, которая определяется свойствами рамки "Область отчета, нач. точка X / Y". (Обычно это нулевая точка рамки.)
* Вручную: нижний левый угол чертежа размещается на точке, которую вы определите в полях Горизонтальная / вертикальная позиция.

### Горизонтальное / вертикальное масштабирование

В этих полях выводится определенный автоматически коэффициент масштабирования. При масштабировании вручную введите здесь нужное десятичное значение.

### Горизонтальная / вертикальная позиция

В этих полях выводятся значения координат левого нижнего угла чертежа согласно выбранному позиционированию. При позиционировании вручную введите здесь нужное десятичное значение.

### Ширина / высота

В этих полях выводятся рассчитанные значения ширины и высоты чертежа согласно установленному масштабированию. Если в эти поля вводить данные вручную, то значения в полях Горизонтальное / вертикальное масштабирование изменяются соответственно.

### Границы чертежа

Выберите этот параметр, чтобы дополнительно обработать автоматически полученные границы чертежа, и введите в поля соответствующие значения для границ.

### Соотношение страниц не изменять

Если этот флажок установлен, высота и ширина страниц масштабируются в одинаковом соотношении и никаких деформаций не происходит. При ручном вводе данных в одно из полей в Границах чертежа соответственно изменяются и другие значения.

Если флажок не установлен, Вы можете изменять значения Границ чертежа независимо друг от друга; в этом случае деформации допускаются.

### Графика предварительного просмотра

Если установлен этот флажок, в предварительном графическом просмотре отображается чертеж DXF/DWG в настроенной рамке; таким образом, можно быстро узнать, должны ли значения для масштабирования, позиции и т. д. обрабатываться дальше.

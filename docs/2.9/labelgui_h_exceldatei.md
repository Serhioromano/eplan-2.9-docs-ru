# Определить настройки вывода Excel

Условия:

* Вы открыли проект.
* Задан шаблон Excel.
* На вашем компьютере установлена действующая бесплатная версия Microsoft Office, полученная по лицензии EPLAN.

1. Сервисные программы > Данные изготовления > Экспорт / маркировка
2. В диалоговом окне Экспортировать данные изготовления / Вывести маркировку рядом с полем Настройки щелкните по кнопке ++...++.
3. В диалоговом окне Настройки: Экспорт данных изготовления / маркировка в поле Схема, определенную пользователем.

!!! info "Для сведения:"

    При помощи ![](images/all_new_as.png){: .ui-icon } (Создать) можно [создать новую схему экспорта / маркировки](labelgui_h_beschriftungsschema.md).

4. Выберите на вкладке Файл в поле Тип файла тип файла "Файл Excel (*.xls, *.xlsx, *.xlsm, *.xlsb)".
5. Щелкните рядом с полем Целевой файл по ++...++ и выберите в открывшемся диалоговом окне каталог и файл или задайте в поле Имя файла имя для нового файла.
6. Щелкните рядом с полем Шаблон по кнопке ++...++ и выберите через следующее диалоговое окно [шаблон Excel](labelgui_k_excelexportvorlagen.md).

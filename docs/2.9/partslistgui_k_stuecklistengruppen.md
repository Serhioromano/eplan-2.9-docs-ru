# Группы спецификаций

В EPLAN можно группировать данные изделий проекта по определенным значениям и отображать эти группы в навигаторе спецификаций. Чтобы присвоить группе изделие, выбранное для устройства, на вкладке Изделие диалогового окна "Свойства" в категории "Данные ссылки изделия" доступно свойство Группа спецификаций. Введите здесь требуемое обозначение для группы спецификаций.

### Вид в навигаторе спецификаций

Для отображения групп спецификаций в представлении структуры дерева навигатора спецификаций выберите пункты всплывающего меню Вид > Группирование спецификаций. В виде Группирование спецификаций данные изделий группируются по группе спецификаций. Это означает, что устройства, присвоенные изделия которых имеют одинаковую группу спецификаций, объединяются в один уровень иерархии. В дереве навигатора в этом виде для группы спецификаций отображается пиктограмма ![](images/navigatorgui_partgroup_as.png){: .ui-icon }.

### Отображение и обработка групп спецификаций

Для целенаправленного отображения и обработки групп спецификаций свойство Группа спецификаций можно использовать в качестве критерия фильтра в навигаторе спецификаций и трехмерных монтажных поверхностей и отображать как столбец в виде списка этих навигаторов.

# Панель инструментов схемы

На панели инструментов справа рядом с полем Схема могут быть доступны следующие кнопки:

Кнопка |  Значение
---|---
![](images/all_edit_as.png){: .ui-icon } (Обработать) |  Открывает диалоговое окно, в котором можно обработать выбранную схему.
![](images/all_new_as.png){: .ui-icon } (Создать) |  Открывает диалоговое окно "Новая схема". Позволяет создать схему.
![](images/all_save_as.png){: .ui-icon } (Сохранить) |  Сохраняет схему.
![](images/all_copy_as.png){: .ui-icon } (Копировать) |  Открывает диалоговое окно "Копировать схему". Позволяет копировать имеющуюся схему.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет схему.
![](images/all_export_as.png){: .ui-icon } (Экспортировать) |  Открывает диалоговое окно "Выбрать файл экспорта". Позволяет экспортировать схему. Многократный выбор ***невозможен***.
![](images/all_import_as.png){: .ui-icon } (Импортировать) |  Открывает диалоговое окно "Выбрать файл импорта". Позволяет импортировать схему (схемы). Возможен многократный выбор.

!!! note "Замечание:"

    Как при импорте, так и при экспорте предлагаются однозначные имена файлов по образцу "Операция.Имя схемы.xml", например "CBcr.Стандарт.xml" для экспорта схем "Стандарт" для генерации кабелей ("CBcr" = CABLE create). Это облегчает обзор уже имеющихся схем.

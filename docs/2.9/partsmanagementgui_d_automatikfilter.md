# Диалоговое окно Автом. фильтр

Открыто диалоговое окно Свойства ++...++ для устройства (например, двойным щелчком по устройству в графическом редакторе), в поле Номер изделия вкладки Изделие выполнен щелчок по ++...++ и в диалоговом окне Выбор изделия — <База данных изделий> выполнен щелчок по ++...++ в поле Автоматический фильтр.

В этом диалоговом окне установите, какие свойства выделенного устройства применяются в качестве критериев фильтрации для выбора изделия. При этом сравниваются только значения, определенные в базе данных изделий.

Обзор основных элементов диалогового окна:

### Определение функции:Раздел:Макрос

Установите эти флажки, чтобы предлагались только изделия, имеющие то же определение функции / тот же раздел / тот же макрос, что и выделенное устройство.

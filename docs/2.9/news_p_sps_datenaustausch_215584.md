# Улучшения в диалоговых окнах для обмена данными ПЛК

Возможен обмен данными ПЛК, обработанными в EPLAN, с программами конфигурации ПЛК от различных производителей. При экспорте данных ПЛК выбираются требуемая программа конфигурации ПЛК и формат файла. Этот выбор теперь стал более наглядным.

![](images/benefit.png)Эффект:

Благодаря новой настройке для программы конфигурации ПЛК и формата файла экспорта выбор формата файла стал проще и интуитивно понятнее.

Ранее в диалоговом окне Экспортировать данные ПЛК в поле Формат файла экспорта отображались все доступные комбинации программ конфигурации ПЛК и форматов. Это диалоговое окно можно открыть через путь меню Данные проекта > ПЛК > Экспортировать данные. Для определенных форматов с помощью кнопки ++Параметры++ можно было перейти к последующему диалоговому окну и там задать версию экспорта или тип вывода.

Теперь нужную программу конфигурации ПЛК можно выбрать в новом поле Программа конфигурации ПЛК. В раскрывающемся списке отображаются все доступные комбинации производителей и программ конфигурации ПЛК.

В раскрывающемся списке поля Формат файла экспорта в зависимости от программы конфигурации ПЛК, выбранной в поле Программа конфигурации ПЛК, отображаются поддерживаемые форматы. Выбор различных версий экспорта или типов вывода осуществляется через отдельные записи в списке.

Кнопка ++Параметры++ теперь активна, только если в поле Формат файла экспорта выбран формат AutomationML. В диалоговом окне Настройки: Экспорт AutomationML можно указать степень детализации для файла экспорта в формате AutomationML.

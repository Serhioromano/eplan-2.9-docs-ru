# Диалоговое окно Настройки: Выбор функциональных элементов

Вы открыли проект.В Графическом редакторе открыта страница проекта. Вы вставили обзор модели (Вставить > Графика > Обзор модели) и разместили его. В диалоговом окне Обзор модели выберите вкладку Вид и щелкните по кнопке ++...++ в поле "Значение" свойства Обзор модели: Схема выбора.

В данном диалоговом окне можно создавать и использовать схемы для представления типов функциональных элементов в обзоре модели.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

# Диалоговое окно Выбор символа

Вы открыли проект.

* Сервисные программы > Основные данные > Библиотека символов > Открыть. В диалоговом окне Открыть библиотеку символов выберите библиотеку символов и щелкните на ++Открыть++.
* Вы открыли как минимум одну библиотеку символов. Сервисные программы > Основные данные > Символ > Открыть.
* В Графическом редакторе открыта страница проекта. Вставить > Символ.
* Вы вставили символ (Вставить > Символ) или щелкнули по символу дважды (или выбрали пункт всплывающего меню Свойства; в этом случае можно выделить также несколько символов). Выберите в диалоговом окне Свойства ++...++ вкладку Данные символа / функции и щелкните в поле Номер / имя по ++...++.

В этом диалоговом окне выберите символы для обработки из:

* В редакторе символов (Сервисные программы > Основные данные > ...) можно выбрать несколько символов.
* В Графическом редакторе при обработке схемы соединений можно выбрать только ***один*** символ.

Обзор основных элементов диалогового окна:

### Фильтр

В этом раскрывающемся списке отображаются все доступные фильтры. Выбранный фильтр активируется автоматически и применяется как к дереву, так и к списку. Запись "- Не активировано -" отключает фильтр и приводит к тому, что данные отображаются в неотфильтрованном виде. С помощью кнопки ++...++ откройте диалоговое окно [Фильтр](modaldialogsdb_d_filternnach.md). Здесь можно создать, обработать, удалить, скопировать, экспортировать, импортировать фильтр и управлять им.

Во всплывающем меню раскрывающегося списка Фильтр содержатся следующие записи:

* Выключить: Этот пункт меню доступен, если фильтр установлен: Сбрасывает настройку фильтра до записи "- Не активировано -".
* Активировать <Имя фильтра>: Этот пункт меню доступен, если для настройки фильтра установлено значение "- Не активировано -": Повторно активирует последний активный фильтр.

Таким образом можно быстро переключаться между неотфильтрованным и отфильтрованным в соответствии с требованиями пользователя представлениями.

* * *

### Дерево

В представлении структуры дерева перелистываются символы в иерархической структуре.

Как только вы выделите символ в дереве или в предварительном просмотре выбора символов, в информационной области под структурой дерева и в строке состояния будет отображаться дополнительная информация (напр., соответствующее описание символа).

### Всплывающее меню

Пункт меню |  Значение
---|---
С ориентац. на библиотеку |  Выводит в самом верхнем узле дерева библиотеки символов. Ниже находятся определения функций.
С ориентац. на функцию |  Выводит в самом верхнем узле дерева функциональную категорию (например, "Катушки и контакты", "Сигнальные устройства" и т.д.).
Настройки |  Открывает диалоговое окно Настройки: Библиотеки символов. Здесь можно выбрать выведенные на просмотр библиотеки символов, для этого нужно установить или не устанавливать соответствующий флажок, выбрать новые библиотеки символов для сохранения (Щелчок по ++...++ в поле Библиотеки символов) или удалить ненужные. Кроме того, здесь при необходимости можно определить предшествующий знак для библиотек символов. (Это диалоговое окно можно вызвать через пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Управление > Библиотеки символов.)

* * *

### Список

Представление в виде списка перелистывает символы друг под другом в столбцах, при этом последовательность и число выведенных на просмотр столбцов может настраиваться индивидуально; при этом в любом выведенном на просмотре столбце сортировку можно производить как в восходящем, так и в нисходящем порядке, для чего нужно один раз щелкнуть по заглавию столбца. После этого направление выводится в заглавии столбца в виде графического символа.

### Прямой ввод

Введите в этом поле символьную строку. Курсор немедленно после ввода первого знака перемещается к символу, который начинается с этого знака, и выделяет его. При этом поиск всегда начинается в первом столбце списка. Если первый столбец списка — имя символа, поиск ведется в имени символа; если первый столбец списка — номер символа, поиск ведется в номере символа и т. д. (Соответствующая настройка выполняется в пункте всплывающего меню Конфигурировать представление.)

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Имя с предш. знаком |  Благодаря этому параметру столбец Имя с предш. знаком автоматически отображается на первом месте представления в виде списка; здесь буквенное обозначение библиотеки символов, указанное в Предш. знак (пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Управление > Библиотеки символов), отображается вместе с именем символа (например, I:SL для замыкающего силового контакта библиотеки символов IEC_Symbol\<X\>, предшествующим знаком которой в этом случае является I). Этот предшествующий знак служит для различения библиотек символов, при чем EPLAN автоматически предлагает первые буквы библиотеки символов. Предшествующий знак также может быть двойным; при этом также возможно объединить несколько библиотек символов в представление в виде списка.
Настройки |  Открывает диалоговое окно Настройки: Библиотеки символов. Здесь можно выбрать выведенные на просмотр библиотеки символов, для этого нужно установить или не устанавливать соответствующий флажок, выбрать новые библиотеки символов для сохранения (Щелчок по ++...++ в поле Библиотеки символов) или удалить ненужные. Кроме того, здесь при необходимости можно определить предшествующий знак для библиотек символов. (Это диалоговое окно можно вызвать через пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Управление > Библиотеки символов.)

### Предв. просмотр

Здесь отображается выбранный символ, независимо от структуры представления в виде дерева или списка. Если выбран только один символ, то отображаются все варианты, которые есть в наличии. Если выбрано несколько символов (напр., уровень иерархии в дереве), все они отображаются в предварительном просмотре, но без имеющихся вариантов.

!!! note "Замечание:"

    Если вы активировали предварительный просмотр графики (Вид > Предв. просмотр графики), выделенный символ будет отображен там в увеличенном виде. Это происходит независимо от того, выделен ли символ в списке, в структуре дерева или предварительном просмотре самого выбора символов.

### Вариант

Этот раскрывающийся список доступен, только если вы выбрали несколько символов. Здесь также вы можете выбрать какой вариант должен отображаться в предварительном просмотре. Если выбран только один символ, все варианты этого символа отображаются в предварительном просмотре, а раскрывающийся список неактивен.

С помощью записи "Стандартный вариант" выберите вариант, заданный в настройках проекта для библиотек символов или в свойствах символа в качестве стандартного варианта.

* * *

Следующие настройки доступны, только если было вызвано диалоговое окно Выбор символа из вкладки Данные символа / функции.

### Показать только подходящие для вида представления символы

Если этот флажок установлен, отображаются только те символы, которые соответствуют виду представления текущей функции.

### Отобразить только символы с одинаковой категорией функции

если этот флаг установлен, выводятся на просмотр только символы с одинаковой категорией функции.

### Заменить только символ

Если этот флажок установлен, вам будут предложены символы, которые имеют не больше выводов устройства, чем текущее определение функции. Тогда будет заменен только символ, а определение функции сохранится.

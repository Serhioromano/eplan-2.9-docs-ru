# Вкладка Несущие шины

Сервисные программы > Изделие > Управление. В дереве или списке выделено одно изделие Главн. группа продукт. "механика", Группа продуктов "Сборные шины", Подгруппа продуктов "Адаптер".

На этой вкладке установите специфические данные для несущих шин, т. е. для изделий Главн. группа продукт. "Механика", Группа продуктов "Сборные шины", Подгруппа продуктов "Адаптер". При этом данные для каждого изделия могут указываться ***многократно***, а именно один раз для каждого варианта.

Обзор основных элементов диалогового окна:

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

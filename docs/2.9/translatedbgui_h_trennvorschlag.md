# Ввести предложение по разделению

Разделитель позволяет отделить ключевое слово в определенном месте текста. Если вы вставите в тексте проекта разделитель строки, тексты перевода будут прерываться в том месте, в котором вы установили разделитель.

Условия:

* У вас есть доступ к словарю, в котором содержатся ключевые слова.
* Вы открыли словарь.

1. Выберите следующие пункты меню: Сервисные программы > Перевод > Обработать словарь
2. Выберите вкладку Обработать слова.
3. Выберите в поле Исходный язык язык, на котором ключевое слово хранится в словаре.
4. Введите в поле Поиск существующее ключевое слово и выделите его в списке ключевых слов.
5. Поставьте курсор в поле Ключевое слово в месте текста перевода, в котором вы хотите установить разделитель.

!!! info "Для сведения:"

    Курсор превратится в индикатор вставки.

6. Нажмите комбинацию клавиш [Alt Gr]+[|] (вертикальная черта).

!!! info "Для сведения:"

    EPLAN вставит вертикальную черту.

!!! example "Пример:"

    Тахо|метр

7. Щелкните по кнопке ![](images/all_save_as.png){: .ui-icon } (Сохранить).

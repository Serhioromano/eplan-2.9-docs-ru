# Конфигурировать машину по сборке проводов

В качестве примера интерфейса сборки проводов здесь описывается конфигурация машины для формата 'Общая спецификация проводов'. Конфигурация машин на других интерфейсах происходит аналогичным образом, несмотря на это, она отличается по виду присоединяемых значений в диалоговом окне настройки.

1. Выберите пункты меню Параметры > Настройки > Фирма > Сборка проводов > Общая спецификация проводов.
2. В диалоговом окне Настройки: Общая спецификация проводов выберите схему требуемой машины.
3. При необходимости измените настройки на вкладках или сгенерируйте новое определение машины при помощи кнопки ![](images/all_new_as.png){: .ui-icon } (Создать).
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Настроенные данные машины доступны при экспорте в диалоговом окне экспорта под соответствующим именем машины.

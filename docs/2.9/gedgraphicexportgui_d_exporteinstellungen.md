# Диалоговое окно Настройки: Экспорт графических файлов

* Страница > Экспортировать > Графический файл. Щелкните рядом с полем Схема по ++...++.
* Параметры > Настройки > Фирма > Интерфейсы > Экспорт графических файлов.

В этом диалоговом окне указываются настройки для экспорта графических файлов, после чего настройки сохраняются в виде схемы. Схемы для экспорта графических файлов сохраняются отдельно для каждой фирмы. В этом диалоговом окне можно скопировать и обработать существующую схему или создать новую.

Доступны следующие элементы диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которая определяет настройки.

Через расположенную рядом [панель инструментов](schemeconfig_m_schemaschaltflaechen.md) можно создавать новые схемы, сохранять изменения в схеме, копировать или удалить схему. Кроме того, схемы можно импортировать и экспортировать.

### Описание

Данное поле отображает текст описания, который вы ввели при создании схемы в диалоговом окне Новая схема в поле Описание.

### Целевой каталог

Введите здесь тот каталог, в который должны сохраняться графические файлы. Каждая страница сохраняется в виде отдельного графического файла. С помощью кнопки ++...++ откройте диалоговое окно для выбора каталога.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Тип файла

Выберите требуемый графический формат из раскрывающегося списка. Возможно использование форматов BMP, GIF, JPG, PNG и TIFF.

### Сжатие

В раскрывающемся списке можно выбрать степени сжатия, которые зависят от выбранного типа файла. Выберите требуемую степень сжатия.

Подробные сведения о степени сжатия файлов TIFF см. по адресу: http://partners.adobe.com/public/developer/en/tiff/TIFF6.pdf.

### Глубина цвета

В этом раскрывающемся списке можно выбрать требуемую глубину цвета, которая зависит от выбранного типа файла. Глубина цвета влияет на то, сколько цветов будет использовано в экспортированных графических файлах. Глубина цвета указывается в "битах на пиксель" (например, 1 бит на пиксель соответствует 2 цветам, 4 бита на пиксель соответствуют 16 цветам, 8 битов на пиксель соответствуют 256 цветам и т. д.).

### Ширина

В этом поле указывается требуемая ширина графического файла в пикселах. Соответствующая высота рассчитывается в EPLAN на основании соотношения с размером страницы. Благодаря этой настройке все выведенные при экспорте страницы будут иметь одинаковую ширину независимо от формата (идет ли речь о формате DIN A4 или DIN A0).

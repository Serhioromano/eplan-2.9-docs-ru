# Импорт и экспорт пространства конструкции: Принцип

***Design Space Exchange (DSE)*** предлагает следующие возможности использования.

Программа 3D-CAD предоставляет геометрическое пространство (например, в форме корпуса электрошкафа), куда необходимо установить электротехнические устройства. Эта геометрическая область обозначается как ***Пространство конструкции***. Если это пространство конструкции импортируется в пространство листа EPLAN, импортированная геометрия получает защиту от изменений.

Затем в это монтажное окружение EPLAN Pro Panel помещает трехмерный чертеж монтажных поверхностей с кабельными каналами, несущими шинами, системами сборных шин и устройствами. Параллельная интеграция информации по изготовлению в форме отверстий, резьб или фрагментов является основой для следующего шага процесса: Подготовка этой информации для программы 3D-CAD.

Впоследствии находящиеся сейчас в EPLAN Pro Panel данные по крепежным отверстиям, резьбам или фрагментам передаются в программу 3D-CAD, считываются там и при помощи методов системы CAD интегрируются в исходную трехмерную модель CAD. Это происходит полностью автоматически, при желании, включая также размещенные в EPLAN Pro Panel кабельные каналы, несущие шины и устройства.

Результат: Полная трехмерная параметрическая модель CAD корпуса электрошкафа, включая всю требующуюся для изготовления информацию, как основу для всех следующих шагов процесса, например для изготовления корпуса и механической обработки компонентов.

Одновременно в EPLAN Pro Panel существует целостная проектная документация, включая всю информацию о разводке для сборки кабелей и проводов.

!!! note "Замечание:"

    Обмен данными между EPLAN Pro Panel и программой CAD, которая предоставляет пространство конструкции и интегрирует трехмерную модель, дополненную трехмерным чертежом монтажных поверхностей и данными ЧУ из EPLAN Pro Panel, выполняется с помощью ***дополнительного модуля CAD***. Создание такого дополнительного модуля для программы M-CAD выполняется опытным консультантом M-CAD по поручению клиента и не входит в объем поставки EPLAN для модуля ***EPLAN Pro Panel — Design Space Exchange***.

# Вкладка <Отображаемое имя объекта планирования>

Вы открыли проект.

* Вы сгенерировали объект планирования в навигаторе предварительного планирования или вставили его в Графическом редакторе. Выберите в диалоговом окне Свойства ++...++ вкладку <Отображаемое имя объекта планирования>.
* Вы выделили объект планирования в навигаторе предварительного планирования или в Графическом редакторе. Обработать > Свойства (или Всплывающее меню > Свойства). Выберите вкладку <Отображаемое имя объекта планирования>.

На этой вкладке указываются свойства выделенного объекта планирования.

Если в диалоговом окне свойств записи в полях отличаются от стандартных значений присвоенного шаблона сегмента, они выделяются [желтым цветом фона](planninggui_k_segmentvorlagen.md).

Обзор основных элементов диалогового окна:

### Обозначение

В этом поле введите имя объекта планирования. Указанное здесь обозначение по умолчанию отображается в навигаторе предварительного планирования на объекте планирования.

### Полное обозначение

В этом поле отображается обозначение объекта планирования, включая обозначения вышестоящих объектов планирования и сегментов структуры.

### Описание

Введите здесь описательный текст для объекта планирования. Этот текст по умолчанию отобразится в навигаторе предварительного планирования вместе с обозначением объекта планирования.

### Шаблон сегмента

В этом поле отображается шаблон сегмента, присвоенный сегменту. С помощью кнопки ++...++ откройте диалоговое окно Выбрать шаблон сегмента, чтобы выбрать шаблон сегмента. Отображаются только шаблоны сегментов, подходящие к определению текущего сегмента. При замене шаблона сегмента значения свойств нового шаблона сегмента переносятся на сегмент и отображаются в диалоговом окне Свойства. Значения, измененные вручную, сохраняются.

С помощью кнопки ![](images/all_delete_as.png){: .ui-icon } (Удалить) из сегмента можно удалить ссылку на шаблон сегмента. Определенные в шаблоне сегмента значения свойств удаляются из сегмента. Значения, измененные вручную, сохраняются.

### Техническое описание

Введите здесь обзор технических характеристик для объекта планирования.

### Маркировка продукции

Введите здесь маркировку продукции. Указанное здесь значение по умолчанию отображается в навигаторе предварительного планирования на объекте планирования.

С помощью пункта всплывающего меню Определить новый счетчик можно генерировать новый счетчик для маркировки продукции. При этом уже имеющиеся буквенные обозначения при необходимости сохраняются. Показания счетчика изменяются по нарастающей, когда на одном уровне иерархии в дереве уже существует объект планирования с такими же буквенными обозначениями.

Через пункт всплывающего меню Подклассы буквенных обозначений можно открыть одноименное диалоговое окно и выбрать какой-либо подкласс буквенных обозначений. (подклассы буквенных обозначений сохраняются в настройках проекта, к ним можно перейти через пункты меню "Имя проекта" > Устройство > Нумерация (online)). Подтвердите выбор, чтобы сгенерировать новую маркировку продукции вместе со счетчиком и отбросить уже имеющуюся маркировку продукции (при ее наличии). Если на одном уровне иерархии в дереве уже существует объект планирования с такими же буквенными обозначениями в маркировке продукции, показания счетчика изменяются по нарастающей.

### Маркировка продукции (полная)

В этом поле отображается маркировка продукции, включая маркировку продукции вышестоящих объектов планирования и сегментов структуры.

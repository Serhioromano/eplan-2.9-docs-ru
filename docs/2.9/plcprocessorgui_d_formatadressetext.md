# Диалоговое окно Формат: Адрес / текст

Вы открыли проект. Вы открыли диалоговое окно [Настройки: ПЛК](plcgui_d_allgemeinespseinstellungen.md) (напр., через пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Устройство > ПЛК). В диалоговом окне Настройки: ПЛК рядом с полем Специфические для ПЛК настройки вы щелкнули кнопку ++...++. В диалоговом окне [Настройки: Специфич. для ПЛК](plcprocessorgui_d_spsspezifischeeinstellungen.md) вы выбрали вкладку Формат списка присвоений и рядом с полями Формат строки списка присвоений или Формат адреса списка присвоений щелкнули кнопку ++...++.

В диалоговом окне Формат строки в списке присвоений (или Формат адреса в списке присвоений) выберите в списке Доступные элементы формата любой элемент формата, кроме типа "Разделитель", и нажмите на кнопку ![](images/all_arrowright_as.png){: .ui-icon } (или двойным щелчком выберите из списка Выбранные элементы формата уже определенный ранее элемент).

В этом диалоговом окне задается форматирование для адресов, функциональных текстов, идентификаторов или типов данных в группе форматов.

Обзор основных элементов диалогового окна:

### Длина

В этом поле вы задаете длину элемента формата.

* Неограниченно перемен.: Количество знаков не определено. Вы можете ввести любое количество знаков.
* Ограничено перемен.: Количество знаков свободно выбирается до определенного предела. Максимальное значение задается в поле Число символов.
* Фиксировано: Количество знаков фиксировано и определяется в поле Число символов. Если количество символов меньше заданной величины, элемент формата заполняется пробелами.

### Число символов

Для параметров Ограничено перемен. и Фиксированная. укажите максимально допустимое количество символов.

### Вывести по правому краю

Если вы выбрали параметр Фиксированная с помощью этого флажка Вы можете установить режим выравнивания текста по правому краю.

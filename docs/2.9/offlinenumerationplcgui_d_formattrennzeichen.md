# Диалоговое окно Формат: Разделитель (нумерация с данными ПЛК)

Вы открыли проект. Вы открыли диалоговое окно [Настройки: Нумерация с данными ПЛК](offlinenumerationplcgui_d_einstellungen.md) (например, через Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация с данными ПЛК). В диалоговом окне Настройки: Нумерация с данными ПЛК выбрана пользовательская схема. Вы выделили запись в поле Группы форматов и щелкнули по ![](images/all_edit_as.png){: .ui-icon } (Обработать), или же по ![](images/all_new_as.png){: .ui-icon } (Создать).

В диалоговом окне Нумерация с данными ПЛК: Формат в списке Доступные элементы формата выберите элемент формата типа "Разделитель" и при помощи кнопки со стрелкой перенесите его в список Выбранные элементы формата.

В этом диалоговом окне указывается форматирование разделителя для группы форматов.

Обзор основных элементов диалогового окна:

В качестве разделителя можно использовать следующие символы:

* Пробел
* Другие символы: После выбора этой опции введите в поле ввода требуемую комбинацию символов. Допускаются любые цифры, буквы и специальные символы.

### Число символов

Если в качестве разделителя выбран пробел, здесь можно указать число символов. В случае выбора опции Другие символы это поле становится недоступным и в нем выводится число введенных символов.

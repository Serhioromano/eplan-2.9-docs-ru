# Диалоговое окно Настройки: Основные данные проекта (Изменение стандарта)

Параметры > Настройки > Пользователь > Изменение стандарта > Основные данные проекта.

В этом диалоговом окне перечислены все относящиеся к пользователю установленные ранее присвоения основных данных исходного проекта исходным данным целевого проекта, используемые для всех проектов. Не нужные больше присвоения можно удалить. Если при изменении стандарта используется присвоение, все основные данные исходного проекта заменяются на основные данные целевого проекта.

Обзор основных элементов диалогового окна:

### Панель инструментов

Кнопка |  Значение
---|---
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет выделенные строки. Удаленное присвоение более недоступно ни в одном из проектов.

### Основные данные проекта

Строка содержит присвоение основных данных исходного проекта основным данным целевого проекта. Основными данными, которые могут здесь присваиваться, являются формы, рамки, библиотеки определений функций и библиотеки символов.

* Имя: отображает имя файла основных данных исходного проекта.
* Тип: отображает тип основных данных. Сюда относятся: рамка, тип имени формы / тип отчета, библиотека символов и определения функций.
* Дата изменения: в этом столбце отображается дата последнего изменения основных данных исходного проекта.
* Новое имя: отображает имя файла основных данных целевого проекта.
* Присвоить отдельно: этот столбец доступен только для библиотек символов. Если флажок установлен, отдельные символы присвоены друг другу. Если флажок снят, вся исходная библиотека символов присвоена целевой библиотеке символов.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

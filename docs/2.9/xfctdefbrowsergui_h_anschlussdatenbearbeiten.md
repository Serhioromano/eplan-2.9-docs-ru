# Обработать логическую схему выводов устройства

Условие:

Вы разместили функцию (то есть условное обозначение) на странице схемы соединений. Или вы сгенерировали функцию в одном из навигаторов.

1. Выделите функцию.
2. Выберите пункты меню Обработать > Свойства.
3. Выберите в диалоговом окне Свойства ++...++ вкладку Данные символа / функции, и щелкните по ++Логика++.
4. В диалоговом окне Логическая схема выводов устройства обработайте в таблице свойства для соответствующих выводов устройства ***функции***, как например вид вывода устройства, тип потенциала или число целей.

!!! info "Для сведения:"

    Если записи в полях отличаются от стандартных значений исходного определения функции, они выделяются желтым цветом.

5. Подтвердите ввод.

### Присвоить выводы устройства между функцией и символом

если число выводов устройства для функции и символа одинаковое, тогда выводы устройства будут автоматически правильно присваиваться системой. Также и в случае с символом замыкающего или размыкающего контакта, которые представляют переключающий контакт (функцию), присвоение будет автоматически правильно настраиваться системой. если число выводов устройства в функции больше, чем в символе, Вы должны сами настроить присвоение.

Условие:

Вы разместили функцию (то есть условное обозначение) на странице схемы соединений. Или вы сгенерировали функцию в одном из навигаторов.

1. Выделите функцию.
2. Выберите пункты меню Обработать > Свойства.
3. Выберите в диалоговом окне Свойства ++...++ вкладку Данные символа / функции, и щелкните по ++Логика++.
4. В диалоговом окне Логическая схема выводов устройства присвойте в поле Вывод устройства символа выводам устройства функции выводы устройства символа. Обычно выводу устройства "1" функции присвоен и вывод устройства "1" символа и т.д. Через раскрывающиеся списки это присвоение можно изменить.
5. Подтвердите ввод.

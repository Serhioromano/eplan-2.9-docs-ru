# Диалоговое окно Нумеровать

Вы открыли проект.

* Данные проекта > Устройство / изделие > Навигатор компоновки электрошкафа 2D. Вы выделили в представлении в виде дерева монтажную плату, легенду электрошкафа которой требуется обновить. Всплывающее меню > Обработать позицию легенды > [Нумеровать].
* Вы находитесь в графическом редакторе на странице типа "компоновка электрошкафа" и выделили монтажную плату, легенду электрошкафа которой требуется обновить. Данные проекта > Устройство / изделие > Компоновка электрошкафа 2D > Обработать позицию легенды > [Нумеровать].

В этом диалоговом окне указываются настройки нумерации для позиций легенды.

Обзор основных элементов диалогового окна:

### Начальное значение

Введите здесь начальное значение для нумерации позиций.

### Величина шага

Введите здесь величину шага, на которую соответственно повышается номер.

### Направление нумерации

В этом групповом поле указывается направление нумерации:

* Заданная последовательность: В данном случае используется последовательность позиций легенды, отображаемая в диалоговом окне Компоновка электрошкафа 2D — <Имя проекта>. Выберите эту опцию, если необходимо повторно пронумеровать имеющиеся позиции, но требуется сохранить использовавшуюся ранее последовательность нумерации.
* Горизонтально / вертикально: Здесь нумерация выполняется на основании размещения на монтажной плате, причем для номера легенды решающими являются позиции точки захвата размещений изделий.
* Автоматически (только EPLAN Pro Panel): Присвоение классов функциональных элементов в последовательности сортировки осуществляется через двухступенчатую иерархию. Сначала функциональные элементы сортируются согласно их иерархии и под-иерархии; геометрическая сортировка выполняется только для функциональных элементов в одной иерархии.

!!! note "Замечание:"

    Обратите внимание, что параметры Горизонтально и Вертикально доступны только в навигаторе двухмерной компоновки электрошкафа и недоступны в трехмерном пространстве листа.

### Одинаковые номера для идентичных устройств

Установите этот флажок, если идентичные устройства должны иметь одинаковые позиции легенды. Это особенно важно при работе по ГОСТу.
Если этот флажок не установлен, то позиции легенды назначаются независимо от устройств.

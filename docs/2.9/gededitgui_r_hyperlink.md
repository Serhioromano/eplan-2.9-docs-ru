# Вкладка Гиперссылка

В Графическом редакторе открыта страница проекта.

* Вставить > Графика > Гиперссылка.
* Вы выделили гиперссылку и выбрали пункты меню Обработать > Свойства. В диалоговом окне Свойства выберите вкладку Гиперссылка.

На этой вкладке обрабатываются свойства просмотра гиперссылки.

Доступны следующие элементы диалогового окна:

### Документ

Введите здесь имя файла, на который должна указывать ссылка, или интернет-страницу. Вызовите с помощью кнопки ++...++ диалоговое окно выбора файла и выберите в нем документ, который будет присоединен с помощью гиперссылки. Его имя и полный путь будут выведены в этом поле.

Кроме того, можно ввести URL или имя файла (с путем), использовав функции копирования и вставки. Или укажите адрес электронной почты, например "mailto:info@eplan.de".

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

!!! note "Замечание:"

    Чтобы гиперссылки в Интернете сохраняли свою функциональность в любой среде (например, при экспорте в PDF), они всегда должны начинаться с "http://" или "https://".

### Текст / обозначение

На странице отображается введенный здесь текст. При выборе файла автоматически предлагается имя файла (без расширения файла). Этот текст можно изменить вручную. Сохраненный при этом текст является многоязычным.

### Описание

В этом поле при необходимости можно ввести текст описания гиперссылки.

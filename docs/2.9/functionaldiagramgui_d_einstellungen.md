# Диалоговое окно Настройки: Блок-схемы рабочего процесса

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Блок-схемы рабочего процесса.

В данном диалоговом окне следует задать специальный символ, который будет использоваться для размещения текстов свойств в блок-схемах рабочего процесса. Помимо этого, можно устанавливать настройки для символов или графики макросов, которые размещаются в функциональных диаграммах с ориентацией на символ.

Обзор основных элементов диалогового окна:

Групповое поле 'Символ размещения свойства'

В данном групповом поле можно обрабатывать свойства символа, который используется для размещения текстов свойств в блок-схемах рабочего процесса.

* Библиотека
* Номер / Имя
* Вариант
* Описание.

* * *

### Функциональное размещение с графикой символов

Если данный флажок установлен, то в функциональную диаграмму при размещении функции с видом представления 'Функциональный' дополнительно к текстам свойств вставляется графика символа выбранной в навигаторе устройств функции.

Если флажок снят, то при размещении вставляются только тексты свойств выбранной в навигаторе устройств функции.

На функциональное размещение ***Макросов*** этот флажок никак не влияет. Графика макроса вставляется всякий раз заново.

* * *

Групповое поле 'Целевой размер для графики символа / макроса'

Помимо текстов свойств, в функциональных диаграммах с ориентацией на символ размещаются также символы устройств или символы макросов изделий. Укажите в этом групповом поле максимальный размер графики символа или макроса. Занесите соответствующие значения в поля Ширина и Высота .

При функциональном размещении функции или макроса изделия соответствующая графика в случае необходимости уменьшается. Увеличения не происходит.

Групповое поле 'Позиция для графики символа / макроса'

Укажите в данном групповом поле позицию символа или графики макроса относительно точки вставки символа для размещения свойства. Занесите соответствующие значения в поля X и Y.

# Схемы сверления из машинных данных

Для поддержки автоматизации технологии производители машин чаще всего поставляют вместе со станками обширные библиотеки с информацией о функциональных элементах, изделиях, сверлении, резке и фрезеровке. Эта информация позволяет пользователю, опираясь на машинное ПО, создать формат электрошкафа и соответствующим образом реализовать его. Эти библиотеки, как правило, постоянно расширяются в течение многих лет.

Информацию из этих библиотек можно использовать в EPLAN Pro Panel для дополнения основных данных EPLAN и для упрощения процедуры создания основных данных в целом.

В настоящем модуле расширения EPLAN Pro Panel предлагает функции для импорта в EPLAN информации по сверлению, которая прежде экспортировалась при помощи машинного ПО, и для генерирования из импортированных данных основные данные.

Возможно следующее:

* Из файлов функциональных элементов Rittal — Perforex / Secarex генерируются схемы сверления.
* Из файлов контуров Rittal — Perforex / Secarex генерируются определенные пользователем контуры, которые можно использовать в базе данных изделий для построения схем сверления.
* Графические элементы из файлов DXF / DWG (экспортированных, например, из Steinhauer eCAB или из системы CAD) можно объединить в схемы сверления при помощи границ схем сверления.
* Схемы сверления, созданные путем импорта и графического определения, можно присвоить имеющимся изделиям и благодаря этому использовать для дальнейшего проектирования и машинного изготовления.
* Кроме того, возможно также чисто графическое определение схем сверления путем черчения функциональных элементов, отверстий и контуров.

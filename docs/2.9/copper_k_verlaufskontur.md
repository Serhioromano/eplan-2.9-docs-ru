# Профиль для изогнутых медных шин: Принцип

***Профиль*** – это путь, по которому проходит прямоугольный основной контур, формирующий изогнутую медную шину. Вдоль профиля формируются в основном только прямоугольные контуры; другие формы поперечного сечения присвоить нельзя.

В редакторе контуров создается контур и сохраняется в виде файла Профиль, медь (*.fp1); причем разрешается использовать только линии и полилинии. Чертеж контура должен представлять собой одну сплошную линию; разветвления и перекрестные соединения недопустимы. Для этого можно выполнить проверку профиля. Эта проверка выполняется также перед закрытием чертежа контура.

При размещении изогнутой шины необходимо выбрать чертеж профиля, который затем присваивается прямоугольному поперечному сечению изделия шины.

### Положение нулевой точки на чертеже профиля

Особое значение имеет положение нулевой точки, которое обозначается красным кружком. При размещении на монтажной поверхности в нулевой точке чертежа профиля находится точка захвата шины.

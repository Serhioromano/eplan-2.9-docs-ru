# Создать и обработать опциональные группы

Опциональные группы сохраняются в проекте. Поэтому их следует создавать в том проекте, в котором они позже будут использоваться.

Условия:

* Вы открыли проект.
* Вы открыли навигатор опций проекта: Данные проекта > Опции > Навигатор

1. Выберите пункт всплывающего меню Создать > Опциональная группа.
2. Введите имя опциональной группы в диалоговом окне Опциональная группа в поле Имя.
3. При необходимости введите в поле Описание текст описания.

!!! info "Для сведения:"

    Под описанием понимается многоязычный текст, отображаемый по умолчанию в представлении структуры дерева рядом с именем (отделяется пробелом и заключается в круглые скобки).

4. Щелкните по кнопке ++OK++.

1. Выделите опциональную группу в диалоговом окне Опции — <Имя проекта>.
2. Выберите пункты меню Обработать > Свойства.
3. Измените в диалоговом окне Опциональная группа записи в полях Имя и Описание.
4. Щелкните по кнопке ++OK++.

1. Выделите опциональную группу в диалоговом окне Опции — <Имя проекта>.
2. Выберите пункты меню Обработать > Удалить.
3. Подтвердите запрос, нажав кнопку ++Да++.

!!! info "Для сведения:"

    Опциональная группа и все содержащиеся в ней опции и фрагменты удаляются. При этом объекты включенных опций сохраняются в проекте; объекты выключенных опций удаляются из проекта.

# Создать макросы

В EPLaN существует возможность сохранить определенные выделенные элементы или области на страницах проекта в виде макроса окна или символа. кроме того, в виде макросов страниц можно сохранить одну или несколько страниц, выделенных в навигаторе страниц или открытых в графическом редакторе. При выполнении всех этих операций в соответствующем макросе также сохраняются имеющиеся графические файлы.

### Создать макросы окон и макросы символов

Условие:

Вы открыли проект, находитесь в графической обработке и выделили либо область на странице, которая содержит требуемые элементы, либо отдельные элементы ([Ctrl] \+ щелчок мышью).

1. Выберите пункты меню Обработать > Создать макрос окна / символа. (После выделения элементов на странице пункт меню Создать макрос окна / символа становится доступен также в качестве параметра ***всплывающего меню***).

!!! info "Для сведения:"

    EPLAN открывает диалоговое окно Сохранить как. В поле каталог выводится предварительно настроенный целевой каталог.

2. Введите в поле Имя файла имя макроса. Нажав кнопку ++...++, можно выбрать другое имя или другой целевой каталог.
3. При необходимости выберите в поле Вариант другое обозначение варианта для макроса. Это имеет смысл в том случае, если для макроса требуется создать разные варианты. По умолчанию макрос сохраняется как "Вариант a". Под одним и тем же именем файла можно создать для каждого вида представления до 26 вариантов одного макроса.
4. При необходимости введите в поле Описание информативное описание макроса. Введенный здесь текст отображается в поле комментариев при вставке макросов, облегчая тем самым выбор.
5. Если при вставке макроса необходимо учитывать масштаб страницы, необходимо установить соответствующий флажок Учитывать масштаб страницы.
6. Если Вы хотите самостоятельно определить в макросе точку, к которой прикрепляется курсор во время вставки, выберите в меню кнопки ++Дополнительно++ пункт Определить точку захвата.

!!! info "Для сведения:"

    После этого диалоговое окно Сохранить как кратковременно закрывается.

7. Переместите курсор на требуемую позицию и укажите щелчком мыши новое положение точки захвата.

!!! info "Для сведения:"

    Затем диалоговое окно Сохранить как откроется снова.

8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Макрос сохраняется в указанном каталоге под именем <Имя>.ema (макросы окон) или <Имя>.ems (макросы символов). EPLaN проверяет, не существует ли уже другой вариант с таким же именем. Если такой вариант уже имеется, выводится соответствующий запрос и необходимо принять решение, перезаписывать ли старый вариант макроса или нет.

!!! note "Замечание:"

    * Если на странице ***ничего*** не выделено, можно выбрать область / отдельные элементы уже ***после*** активации этого пункта меню (запрос в строке состояния).
    * Варианты угла и другие варианты макроса не генерируются автоматически, а должны быть созданы вручную на страница проекта, а затем сохранены с тем же именем макроса, но с другим обозначением варианта. Если для макроса созданы различные варианты, при вставке можно выбрать один их них.
    * В соответствии с указанным фрагментом в макрос окна или макрос символа также сохраняется рамка макроса.

### Создать макрос страницы

Если курсор находится на странице проекта в графическом редакторе, в макрос страницы можно записать связанные друг с другом страницы. Помимо этого, в навигаторе страниц в макрос страницы можно сохранить и не связанные друг с другом страницы.

Условие:

Вы открыли Навигатор страниц.

1. Выделите требуемые страницы в навигаторе страниц.
2. Выберите пункты меню Страница > Макрос страницы > Создать. (Макросы страниц также можно создать с помощью пункта всплывающего меню Создать макрос страницы).

!!! info "Для сведения:"

    EPLAN открывает диалоговое окно Сохранить как. В поле каталог выводится предварительно настроенный целевой каталог.

3. Введите в поле Имя файла имя макроса. Нажав кнопку ++...++, можно выбрать другое имя или другой целевой каталог.
4. При необходимости введите в поле Описание информативное описание макроса. Введенный здесь текст отображается в поле комментариев при вставке макросов, облегчая тем самым выбор.
5. При необходимости укажите в поле Число страниц, сколько страниц следует сохранит в макрос. Предварительная настройка в этом поле может быть изменена только в том случае, если в навигаторе страниц выделено не больше одной страницы.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Макрос сохраняется в настроенном каталоге под именем <Имя>.emp. При этом EPLАN проверяет, нет ли уже макроса с указанным именем. Если такой макрос уже имеется, выводится соответствующий запрос и необходимо принять решение, перезаписывать ли старый макрос или нет.

!!! tip "Совет:"

    Можно создать макрос страницы перетаскиванием мышью, выделив требуемые страницы в навигаторе страниц, и, удерживая нажатой клавишу ++Shift++, перетащить страницы в Проводник Windows.

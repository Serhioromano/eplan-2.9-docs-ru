# EPLAN Fluid: Логическая схема выводов устройства функций Fluid-техники

Для функций Fluid-техники следующие свойства важны для определения логической схемы выводов устройства:

* Тип вывода устройства
* Подвод давления / управляющий вывод устройства
* Вид сигнала.

Данное свойство может обрабатываться в диалоговом окне Логическая схема выводов устройства. (Для этого на вкладке Данные символа / функции диалогового окна Свойства ++...++ нажмите кнопку ++Логика++.)

### Тип вывода устройства функций Fluid-техники

В качестве типов вывода устройства при выводах функций Fluid-техники в вашем распоряжении - настройки "Fluid-техники", "внутренний" и "прямой вывод устройства". "Внутренний" и "прямой вывод устройства" предусмотрены специально для распределителей и соединителей, входящих в группы клапанов.

Тип подключения нужен при создании соединений Fluid-техники. Посредством типа вывода устройства "Fluid-техники", устанавливающегося предварительно при создании или помещении функции Fluid-техники, могут составляться и выдаваться специфические таблицы соединений Fluid-техники, например, списки трубопроводов и шлангов. При этом программа дополнительно проверяет раздел соответствующих функций и внесенное в настройки стандартное определение функции для соединения. Если соединение имеет к тому же точку определения соединения, определение функции для создания соединения берется из нее.

### Подвод давления и управляющие подключения функций Fluid-техники

На выводах устройства функций Fluid-техники с помощью свойства подвод давления / управляющий вывод устройства могут устанавливаться различные значения в форме типов линий.

Имеется выбор следующих возможностей:

* Не определено
* Рабочая линия
* Блок управления
* Труб. для слива масла
* Носитель процесса
* Теплоноситель и хладагент
* Запр. трубопр.
* Питающая линия.

Если на самом соединении (через точку определения соединения) показатель свойства подвод давления / управляющий вывод устройства установлен на "неопределенный", тип линии передается автоматически от выводов устройства на соединение.

С помощью контрольного прогона (класс "соединения") можно к тому же установить, подходят ли друг к другу типы линий, заданные на выводах функций. Если имеются противоречащие друг другу данные, например, когда трубопровод для слива масла соединяется с блоком управления, в управлении сообщениями показывается соответствующее сообщение.

Если должны выдаваться таблицы соединений, свойство подвод давления / управляющий вывод устройства может быть задано в качестве критерия сортировки.

### Вид сигнала функций Fluid-техники

Для функций Fluid-техники с помощью свойства вид сигнала можно установить, является ли вывод устройства подводом давления или вакуумным выводом, или же он не определен.

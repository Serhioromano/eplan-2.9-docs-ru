# Диалоговое окно Выбрать источник данных

* Данные проекта > Устройство > Импортировать. Щелкните мышкой в диалоговом окне Импортировать данные устройств рядом с полем Источник данных по кнопке ++...++.
* Данные проекта > Соединения > Импортировать > Маршрутизируемые соединения. Щелкните мышкой в диалоговом окне Импортировать данные устройств рядом с полем Источник данных по кнопке ++...++.

В этом диалоговом окне находятся настройки для управления импортом устройств и маршрутизируемых соединений из импортируемого текстового или Excel-файла. Помимо этого, необходимо установить, начиная с какой строки следует производить импорт.

Обзор основных элементов диалогового окна:

### Файл

В этом поле показано имя файла импортируемых текстовых файлов или файлов Excel. При нажатии [...] открывается диалоговое окно Открыть для выбора импортируемого файла.

### Таблица

Это поле доступно только для файлов Excel. Из раскрывающегося списка выберите имя таблицы внутри файла Excel.

### Число строк верхнего колонтитула

Определяет, начиная с какой строки в импортируемом текстовом файле следует осуществлять импорт. Настраиваемое число строк верхнего колонтитула при импорте игнорируется.

### Обозначение поля данных в строке

Определяет, из какой строки источника данных должны быть взяты обозначения поля данных для столбца "Внешнее поле данных" в таблице присвоений.

### Разделитель

Это поле доступно только для текстовых файлов. Отдельные столбцы в импортируемых текстовых файлах при импорте разделяются при помощи разделителя. В качестве разделителя можно использовать следующие символы:

* Пробел
* Табулятор
* Другие символы: После выбора этого параметра введите в поле ввода под ним требуемую комбинацию символов. Допускаются любые цифры, буквы и специальные символы.

# Вставить динамические символы соединений

Динамическими символами соединений можно управлять в библиотеке символов SPECIAL.slk; динамические символы — это символы без фиксированной графики. К ним относятся:

* Динамические символы для диагонального соединения
* Определения кабеля
* Экранирования
* Точки определения соединения
* Точки определения потенциала
* Выводы потенциала.

### Динамические символы для диагональных соединений

Нередко фазовый сдвиг должен быть сгенерирован через диагональные линии. Для этого есть динамический символ, обозначеный соединениeм через диагональную линию.

Условие:

Вы открыли проект. В Графическом редакторе открыта страница проекта.

1. Выберите пункты меню Вставить > Символ соединения > Диагональ.
2. При необходимости используйте клавишу ++Ctrl++ и переместите мышь, чтобы изменить направление выводов устройства.
3. По очереди разместите начальную и конечную точки диагональной соединительной линии.
4. Выполните аналогичные действия для всех других линий соединения.
5. Завершите операцию, выбрав пункт всплывающего меню Прервать операцию или нажав клавишу ++Esc++.

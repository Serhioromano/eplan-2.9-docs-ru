# Конфигурировать шлангопроводы

Данные шлангопроводов, считанные в диалоговом окне Конфигурировать шлангопроводы, как правило содержат только полное ОУ шлангопроводов. Чтобы генерировать для каждого шлангопровода соответствующие стандарту ***обозначение шлангопровода*** и ***код типа***, необходимо дополнить информацию о шлангопроводе другими свойствами и присвоить значения, т. е. сконфигурировать шлангопровод. Для этого служит диалоговое окно Обработать ++...++. Во время обработки вы возвращаетесь к настроенным в выбранной схеме свойствам и значениям. ***Групповая обработка*** позволяет обрабатывать в этом диалоговом окне данные сразу нескольких шлангопроводов.

Условия:

* У вас установлен конфигуратор шлангопроводов в качестве дополнительного модуля для EPLAN Fluid или в качестве самостоятельной программы.
* Данные шлангопровода считаны: либо непосредственно со страницы схемы соединений открытого проекта (дополнительный модуль), либо с помощью файла обмена (самостоятельная программа). Данные шлангопровода содержат в качестве минимальной информации полное обозначение устройства шлангопровода и отображаются в таблице диалогового окна Конфигурировать шлангопроводы.

1. В Add-on выберите пункт меню Дополнительно > Настройки или в самостоятельной программе — пункт меню Параметры > Настройки.
2. В диалоговом окне Настройки: Конфигуратор шлангопроводов выберите нужную схему, например схему, настроенную по умолчанию DIN 20066, которая позволит генерировать обозначение шлангопровода и код типа в соответствии со стандартом DIN 20066.
3. Щелкните по кнопке ++OK++.
4. В диалоговом окне Конфигурировать шлангопроводы щелкните двойным щелчком строку с полным ОУ.

5. Заполните поля диалогового окна [Обработать ++...++](fluidhoseconfiggui_d_schlauchleitungenbearbeiten.md) необходимыми значениями.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Обработанные данные шлангопроводов вносятся в таблицу диалогового окна Конфигурировать шлангопроводы.

7. В конфигураторе шлангопроводов, установленном в качестве дополнительного модуля, закройте диалоговое окно Конфигурировать шлангопроводы, нажав ++OK++.

!!! info "Для сведения:"

    После обработки данные записываются обратно в проект и сохраняются на выделенных шлангопроводах (это возможно только при условии, что данные шлангопроводов перед обработкой были считаны с открытой страницы схемы соединений проекта, а соответствующие шлангопроводы уже были отмечены на схеме соединений).

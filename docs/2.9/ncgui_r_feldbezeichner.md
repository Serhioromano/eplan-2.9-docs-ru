# Вкладка "Обозначение поля (экспорт Rittal - Perforex BC / Perforex LC / Secarex)"

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт Rittal - Perforex BC / Secarex. Выберите вкладку Обозначение поля.
* Параметры > Настройки > Фирма > Механическая обработка > Экспорт Rittal - Perforex LC / Secarex. Выберите вкладку Обозначение поля.
* Сервисные программы > Данные изготовления > Механическая обработка > Rittal - Perforex BC / Secarex. Рядом с полем Машина нажмите ++...++. Выберите вкладку Обозначение поля.
* Сервисные программы > Данные изготовления > Механическая обработка > Rittal - Perforex LC / Secarex. Рядом с полем Машина нажмите ++...++. Выберите вкладку Обозначение поля.

На этой вкладке можно отдельно конфигурировать обозначения полей для экспорта в программное обеспечение машины Perforex. Для этого в таблице именам свойств в программном обеспечении Perforex присваиваются выбираемые свойства EPLAN. После этого присвоенные обозначения полей Perforex определяются свойствами EPLAN. Например, поле Perforex "Главная программа — Имя" может получить имя текущего проекта EPLAN путем присвоения свойства EPLAN "Имя проекта" (P10000).

Обзор основных элементов диалогового окна:

### Поле

Обозначение поля данных в программном обеспечении Perforex.

### Генерировать из

В раскрывающемся списке можно выбрать, как следует определять обозначения полей в программном обеспечении Perforex:

* Автоматически: обозначения полей определяются по методу предыдущей версии EPLAN 2.7.
* Свойства: обозначения полей определяются посредством свойств EPLAN.

### Форматирование

Свойства EPLAN, которые присваиваются обозначениям полей в программном обеспечении Perforex.

При выделенном столбце Форматирование отображается кнопка ++...++. Если нажать эту кнопку, откроется диалоговое окно Выбрать элементы формата, в котором можно выбрать применяемые свойства.

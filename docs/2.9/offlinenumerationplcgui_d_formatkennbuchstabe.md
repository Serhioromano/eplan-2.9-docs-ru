# Диалоговое окно Формат: Буквенное обозначение

Вы открыли проект. Вы открыли диалоговое окно [Настройки: Нумерация с данными ПЛК](offlinenumerationplcgui_d_einstellungen.md) (например, через Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация с данными ПЛК). В диалоговом окне Настройки: Нумерация с данными ПЛК выбрана пользовательская схема. Вы выделили запись в поле Группы форматов и щелкнули по ![](images/all_edit_as.png){: .ui-icon } (Обработать) или по ![](images/all_new_as.png){: .ui-icon } (Создать).

В диалоговом окне Нумерация с данными ПЛК: Формат в списке Доступные элементы формата выберите элемент формата типа "Буквенное обозначение" и при помощи кнопки со стрелкой перенесите его в список Выбранные элементы формата.

В этом диалоговом окне указывается форматирование буквенных обозначений в группе форматов.

Обзор основных элементов диалогового окна:

### Минимальное число разрядов

Здесь указывается минимальное число разрядов для значения буквенного обозначения. Более длинные значения выводятся полностью. Разрешено использовать только положительные числовые значения. При вводе нуля число разрядов является переменным.

### Заполнитель

Буквенное обозначение заполняется с помощью этого символа, если минимальное число разрядов больше, чем ширина значения.

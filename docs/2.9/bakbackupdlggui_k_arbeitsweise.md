# Резервирование данных проекта

При резервировании проектов следует различать следующие возможности: Дополнительно сохранить, Выгрузить и Архивировать.

В пользовательских настройках (Параметры > Настройки > Пользователь > Резервирование данных > Предварительные настройки > Проекты) можно задавать предварительные настройки для резервирования данных проектов. Предварительные настройки заданы по умолчанию и при резервировании проектов применяются для первого резервируемого проекта. Если резервирование проекта уже выполнено, используемые для резервирования настройки сохраняются вместе с проектом и будут доступны при следующем резервировании данных.

При выборе сразу нескольких проектов всегда используются предварительные настройки, так как в зависимости от существующих проектов настройки для резервирования отдельных проектов могут различаться.

### Дополнительное сохранение

При резервировании данных копия проекта сохраняется на другом носителе. Могут быть выделены несколько проектов, которые будут резервированы последовательно. Выбрать сразу несколько проектов можно при помощи опции "Управление проектами", где есть функция многократного выбора в дереве.

### Выгрузить

Если проект необходимо передать субподрядчикам, заказчикам или просто в другие фирмы для временной обработки, его можно выгрузить.

Копия проекта при этом сохраняется на другом источнике данных, а исходный проект будет открываться только для чтения. Благодаря этому исключается возможность того, что в промежуточный период времени в проект будут внесены изменения, которые окажутся утраченными, когда проект вернется обратно. Все данные выгруженного проекта остаются на исходном жестком диске.

При попытке обработки выгруженного проекта пользователь получает сообщение, что проект выгружен и открыт только для чтения.

### Архивирование

С целью улучшения структуры носителя данных завершенные проекты можно архивировать. Копия проекта при этом сохраняется на другом носителе данных, а исходный проект, за исключением информационного файла, с жесткого диска удаляется.

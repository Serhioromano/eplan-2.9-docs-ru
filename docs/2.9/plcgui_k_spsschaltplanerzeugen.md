# Генерация схемы соединений ПЛК

EPLAN предоставляет возможность автоматически генерировать схему соединений, например, из конфигурации ПЛК. При этом неразмещенные (или неразмещенные в определенных видах представления) функции ПЛК автоматически размещаются, таким образом генерируются схемы соединений ПЛК, включающие различные обзорные страницы. Имеются следующие возможности обработки:

* Можно генерировать страницы схемы соединений с различными видами представления.
* Функциям ПЛК можно назначать макросы как автоматически (из изделия), так и вручную.
* Для переменных из объектов-заполнителей, содержащихся в макросах, вы можете выбрать наборы значений или ввести значения вручную.
* С помощью макросов проводки возможно представление проводки выводов устройства.
* Можно создать конфигурации ПЛК с помощью внешних программ конфигурации ПЛК, а затем импортировать данные в EPLAN и сгенерировать из них схему соединений.
* Объекты, добавленные в конфигурацию ПЛК впоследствии, также могут быть автоматически вставлены в схему соединений. Для этого можно сгенерировать новые страницы для добавленных объектов, заново сгенерировать диапазон страниц или заново сгенерировать полную схему соединений ПЛК.
* Можно импортировать адреса ПЛК из списка присвоений, а затем использовать выбор устройства для генерации неразмещенных функций ПЛК для этих адресов. Эти функции могут быть автоматически размещены при генерации схемы соединения ПЛК.
* Конфигурацию, выполненную в диалоговом окне Генерировать схему соединений ПЛК, можно сохранить, импортировать и экспортировать. Это позволяет прервать проектирование схемы соединений и возобновить его позднее.
* Функции, автоматически размещаемые во время генерации схемы соединений ПЛК, можно удалять и заново размещать.
* В режиме "Общий" можно вызвать диалоговое окно 'Свойства', чтобы обработать в нем свойства устройства.

# Показывать символы соединений с определением цели или как точку

Для символов соединения существует ряд специфических для пользователя настроек, которые касаются ***Начертить*** символов. Кроме того, через меню Параметры > Настройки > Проекты > "Имя проекта" > Графическая обработка > Общее можно применить другие ***относящиеся к проекту*** настройки, влияющие на ***отображение*** символов соединений.

1. Параметры > Настройки > Пользователь > Графическая обработка > Символы соединений
2. В групповом поле Отображать ответвления соединений диалогового окна Настройки: Символы соединения выберите, как должны быть начерчены символы соединения в графическом редакторе.
3. Установите флажок Символы соединения с диалоговым окном запроса, если при черчении символов соединения всегда должно выводиться диалоговое окно запроса. (Если в групповом поле Отображать ответвления соединений выбран параметр С определением цели или как точка, то этот флажок автоматически становится активированным и недоступным.)
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    При черчении символы соединения заполняются соответствующим образом.

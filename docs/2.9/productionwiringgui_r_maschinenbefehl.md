# Вкладка Машинная команда

* Параметры > Настройки > Фирма > Сборка проводов > 'Машина' > Машинная команда
* Сервисные программы > Данные изготовления > Сборка проводов > "Машина". Щелкните по кнопке ++...++ рядом с полем Схема. Выберите вкладку Машинная команда.

В данной вкладке регистрируются машинно ориентированные команды и присваиваются варианты обработки концов проводников.

Обзор основных элементов диалогового окна:

### Обработка концов проводников

Значения из столбца Обработка концов проводников находятся во вкладке Обработка концов проводников в качестве выбираемых значений.

### До поперечного сечения

Столбец До поперечного сечения позволяет определить, до какого поперечного сечения будут использоваться инструкции, введенные соответственно в поле Машинная команда.

### Машинная команда

В этом столбце вводятся инструкции на языке соответствующего программного обеспечения машины, которые должны выполняться при выбранной обработке концов проводников.

!!! example "Пример:"

    AEH1-n= TerminalKey (Использовать оконечную гильзу с подходящим поперечным сечением)$strip$= Снять изоляцию$ultrasonic,xx$= Сварка ультразвуком (xx= Длина участка, свариваемого ультразвуком в мм)

Информацию о машинных командах можно получить у производителей машин.

Длина зачистки под двойную изоляционную трубку на машинах Komax

Если пользователь использует машинную команду Komax $strip$, то открывается доступ к значению из столбца Длина зачистки на вкладке Провода. При использовании двойных изоляционных трубок нужны другие длины зачистки, чем при одиночных изоляционных трубках.

Чтобы можно было управлять изменением длины зачистки при использовании двойных изоляционных трубок, к машинной команде $strip$ добавляют параметры a (длина зачистки) и b (длина сдвига). Здесь следует использовать следующий формат машинной команды $strip,a,b$. Если используется машинная команда $strip,a,b$ то для длины зачистки и длины сдвига используются значения a и b; значения, указанные на вкладке Провода, в данном случае игнорируются.

!!! example "Пример:"

    Обработка концов проводниковдо поперечного сечения Машинная команда Команда strip, двойная трубка6,0$strip,10,5$Результат:Длина зачистки = 10 мм, длина сдвига = 5 мм

Если значение для параметра b не указано ($strip,a$), то в качестве длины сдвига используется значение, равное половине длины зачистки (длина зачистки/2).

Если не указано значение для параметра a ($strip,,b$), то используется значение длины зачистки из столбца Длина зачистки на вкладке Провода.

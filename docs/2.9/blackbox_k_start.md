# Черные ящики

Символы устройств является излюбленным средством для представления устройств, для которых отсутствуют символы в библиотеке символов, предписываемой пользователю заказчиком. Черный ящик используется самыми различными способами:

* Для устройств / узлов, не представленных в библиотеке символов
* Для устройств / узлов, которые представлены в библиотеке символов не в полном объеме, напр., при отсутствующих контактах РЕ / PEN
* Для представления узлов ПЛК
* Для представления сложных устройств, напр., преобразователей частоты; такие устройства разбиваются на несколько страниц схемы соединений и имеют перекрестные ссылки
* Для представления нескольких символов как обозначение ***одного*** устройства, напр., двигатель с тормозом
* Для представления резервных кабельных соединений для кабелей (***без*** черного ящика генерируется сообщение об ошибке 'Кабельное соединение без кабеля')
* Для вкладывания нескольких обозначений устройств, напр., для устройства с несколькими клеммниками:
Устройство: -A1, клеммники: -X1 и -X2
При вкладывании клеммникам присваиваются обозначения устройств -A1-X1 и -A1-X2.
* Для передачи ОУ на клеммы, т. к. ОУ клемм невозможно переместить (в противном случае вместе с ОУ клемм будет перемещено обозначение вывода устройства), например из-за нехватки места на странице
* Для специальных контакторов, которые невозможно отобразить обычными символами, но которые тоже должны отображать образ контакта.

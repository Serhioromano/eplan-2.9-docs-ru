# Диалоговое окно Выбрать монтажную плату

Вы открыли проект. Сервисные программы > Отчеты > Генерировать. Щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать). Выделите в диалоговом окне Определить отчет отчеты "Схема подключения устройства (для диаграммы соединений)", "Схема подключения клемм (для диаграммы соединений)" или "Схема подключения штекера (для диаграммы соединений)". Выберите в раскрывающемся списке Форма вывода запись "Размещение вручную" и установите флажок Индивидуальный выбор. Щелкните по кнопке ++OK++.

!!! note "Замечание:"

    * Если настройки "Размещение вручную" и / или Индивидуальный выбор ***не*** выбраны, отображается соответствующее сообщение. В данном сообщении щелкните по ++OK++, настройки соответственно изменяются.
    * Если проект не содержит монтажных плат, диалоговое окно Выбрать монтажную плату не открывается.

В этом диалоговом окне выберите, для какой монтажной платы генерируется схема соединения. Схема соединения показывает устройство в одинаковой последовательности, как у монтажной платы, при этом одинаковые координаты и размеры не имеют значения.

Обзор основных элементов диалогового окна:

### Монтажная плата

Выберите из раскрывающегося списка монтажную плату, для которой должна быть сгенерирована схема соединения.

### Скрыть уже использованные устройства

Данный флажок доступен, если из раскрывающегося списка Монтажная плата выбрана монтажная плата и существует черный ящик с одинаковым ОУ.
Установите флажок, если устройство, для которого уже сгенерирован отчет, больше не должно предлагаться.

Если флажок снят, для отчета предлагаются все устройства.

# Выбрать устройства

При выборе устройства ему присваивается устройство; оно может содержать одно или несколько изделий. Устройству присваиваются данные из определений функций. Эти данные записываются в базу данных изделий на вкладке Шаблоны функций в таблице определений функций.

Выбор устройства можно вызывать из многих мест:

* из диалогового окна Свойства ++...++ * из навигатора спецификаций.

### Выбор устройств в диалоговом окне Свойства ++...++ Условие

Вы открыли диалоговое окно Свойства ++...++ для условного обозначения и выбрали вкладку Изделие.

1. Щелкните по ++Выбор устройства++.

!!! info "Для сведения:"

    В списке Главное изделие отображаются все изделия из базы данных изделий, которые подходят выбранному устройству и для которых флажок Принадлежности на вкладке Принадлежности не установлен.

!!! info "Для сведения:"

    В списке Выбранные изделия: Функции / шаблоны отображаются функции текущего устройства.

2. Щелкните при необходимости по ++Настройки++ для изменения, напр., критериев выбора устройств.
3. Обработайте в диалоговом окне Настройки: выбор устройства настройки и нажмите ++OK++.
4. Щелкните при необходимости рядом с полем Автом. фильтр по кнопке ++...++ для ввода дополнительных критериев фильтрации.
5. В диалоговом окне Автоматический фильтр: Выбор устройства настройте параметры фильтра для технических значений изделия и щелкните по ++OK++.

!!! info "Для сведения:"

    В списке Главное изделие отображаются соответствующим образом отфильтрованные изделия.

6. Выделите какое-либо изделие в списке Главное изделие.

!!! info "Для сведения:"

    В списке Принадлежности будут отображены детали принадлежностей, подходящие выделенному изделию (если таковые имеются).

!!! info "Для сведения:"

    В списке Выбранные изделия: Функции / шаблоны вы можете увидеть в предварительном просмотре, каким функциям устройства присвоен шаблон функции изделия и остаются ли дополнительные произвольные шаблоны.

7. Выберите необходимое изделие в списке Главное изделие (или предмет оборудования в списке Принадлежности) и щелкните по кнопке ![](images/all_arrowdown_as.png){: .ui-icon } под списком. (Если вы выделили изделие принадлежностей, вы можете одновременно при помощи ![](images/all_arrowdowndouble_as.png){: .ui-icon } вывести принадлежности и соответствующее изделие.

!!! info "Для сведения:"

    Таким образом изделие будет выбрано. Это уточняется в списке Выбранные изделия: Функции / шаблоны с помощью дополнительной пиктограммы (галочки) перед шаблонами функций изделия.

8. Если шаблоны функций присвоены не всем функциям устройства, выберите следующее изделие в одном из списков Главное изделие или Принадлежности.

!!! info "Для сведения:"

    В списке Выбранные изделия: Функции / шаблоны соответствующие шаблоны функций изделий поочередно выделяются цветом. Строки предварительного просмотра изображаются курсивом.

9. Нажмите ![](images/all_arrowdown_as.png){: .ui-icon } под списком, чтобы переместить выбранное изделие в список Выбранные изделия: Функции / шаблоны. При необходимости выбирайте следующие изделия, пока подходящими шаблонами функций изделий не будут перекрыты все функции устройства.
10. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Номера изделий переносятся в диалоговое окно Свойства ++...++.

11. Выйдите из диалогового окна Свойства ++...++ нажатием кнопки ++OK++.

!!! info "Для сведения:"

    Все номера изделий, внесенные в диалоговом окне Свойства ++...++, разыскиваются в базе данных изделий, и соответствующие изделия сохраняются в проект. При этом данные изделий считываются в соответствии с настроенным источником данных. Также сохраняются шаблоны функций изделия.

!!! info "Для сведения:"

    Свойства ***первого*** выбранного изделия записываются на текущее устройство (например, Тип кабеля, Поперечное сечение / диаметр соединения, Обозначение вывода устройства, Технические параметры и т. д.).

!!! info "Для сведения:"

    Данные шаблонов функций сохраненных изделий записываются по текущему устройству (главной функции) и по его вспомогательным функциям.

!!! info "Для сведения:"

    Если устройству уже присвоено изделие, его данные будут без дополнительного запроса перезаписаны данными ***первого*** выбранного изделия. Конфликтное диалоговое окно, как при выборе изделия, при выборе устройства не выводится.

!!! info "Для сведения:"

    Указанные у условного обозначения данные (например, обозначения вывода устройства) не удаляются и не перезаписываются данными из базы данных изделий. При контрольном прогоне сообщается об этих отклонениях.

### Выбор устройств в навигаторе спецификаций

1. Данные проекта > Устройства / Изделия > Навигатор спецификаций
2. Выделите ссылку изделия и выберите Раскрывающееся меню > Выбор устройства.
3. В диалоговом окне Выбор устройства выберите необходимое(ые) изделие(я), как описано выше.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Данные изделия переносятся в диалоговое окно Спецификация — <Имя проекта>, т. е. ссылки изделий обновляются.

!!! warning "Предупреждение:"

    При выборе изделия, имеющего более старый статус в проекте, чем в базе данных изделий, отображается запрос. Вы можете решить, нужно ли выбрать это обновленное изделие или другое изделие. В некоторых случаях (например, при изменении данных шаблона) данные устройства обновленного изделия не могут быть перенесены во все сферы применения изделия в проекте.
    Чтобы в таком случае иметь возможность обновлять данные устройства на функциях в проекте, в навигаторе спецификаций доступен пункт всплывающего меню Обновить данные устройства.

# Выбрать изделия из каталога продуктов Festo

При выборе изделий через интерфейс API можно получить прямой доступ к каталогу продуктов Festo и выбрать изделия из него. В этом случае предварительный импорт изделий во внутреннюю базу данных изделий необязателен.

Условия:

* Вы открыли проект.
* Установлен каталог продуктов Festo.
* В файле конфигурации festo.cfg для импорта определены необходимые настройки.
* Подходящий макрос окна, или макрос символа, или подходящий символ размещены в графическом редакторе.

1. Выделите условное обозначение в графическом редакторе или функцию — в одном из навигаторов и выберите пункт меню Обработать > Свойства.
2. Выберите в диалоговом окне Свойства ++...++ вкладку Изделие.
3. В раскрывающемся списке поля Источн. данных выберите схему "Каталог продуктов Festo".
4. Щелкните в поле Номер изделия по ++...++.

!!! info "Для сведения:"

    Откроется каталог продуктов Festo. Изделия отфильтрованы по свойству Festo "SymbolId", т. е. отображены только изделия с таким значением свойства "SymbolId", которое присвоено соответствующему макросу или символу из файла конфигурации.

5. В каталоге продуктов Festo выберите необходимые изделия и щелкните ++OK++.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Изделия сохраняются в проект.

!!! tip "Совет:"

    Чтобы скопировать импортированные изделия во внутреннюю базу данных изделий, сначала необходимо изменить схему для источника данных. Откройте диалоговое окно свойств для условного обозначения или функции и перейдите на вкладку Изделия. В поле Источн. данных выберите схему "Стандарт". После закрытия диалогового окна свойств при помощи пунктов меню Сервисные программы > Изделие > Синхронизировать текущий проект скопируйте изделие Festo во внутреннюю базу данных изделий.

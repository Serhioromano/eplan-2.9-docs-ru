# Основные данные: Определить настройки форм

Основные данные можно синхронизировать в связи с проектом при открытии проекта, а также при резервировании / восстановлении с пулом основных данных. Специально для форм существуют настройки проекта, которые определяются таким образом:

Условие:

Вы открыли проект.

1. Параметры > Настройки > Проекты > "имя проекта" > Отчеты > Вывод в страницы
2. В диалоговом окне Настройки: Вывод в страницах откройте раскрывающийся список Форма для типа отчета, настройки которого вы хотите обработать.
3. Щелкните по ++Поиск++.
4. Выберите в диалоговом окне Выбрать форму необходимую форму. При этом в поле Тип файла отображаются только подходящие к текущему выбранному типу отчета типы данных.
5. Щелкните по кнопке ++Открыть++.
6. Определите дальнейшие настройки формы.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Форма с новыми/измененными настройками загружается в проект.

# Экспортировать страницы в формате DXF / DWG

При экспорте отдельных страниц проекта или всего проекта в формат DXF или DWG, для каждой выбранной страницы создается файл. При экспорте ***отдельной*** страницы предусмотрена возможность произвольного присвоения имени файла. При экспорте ***нескольких*** страниц невозможна свободная передача имени файла. Файлы DXF/DWG в зависимости от настроек выбранной схемы генерируются или по порядковому номеру, или по имени страницы, или из произвольно конфигурируемых свойств.

Условие:

Вы открыли проект.

1. Выделите в навигаторе страниц те страницы проекта (или весь проект), которые требуется экспортировать.
2. Выберите пункты меню Страница > Экспортировать > DXF / DWG.
3. В поле Схема диалогового окна Экспорт DXF / DWG выберите из раскрывающегося списка подходящую схему. Или щелкните ++...++, чтобы вызвать диалоговое окно Настройки: Экспорт и импорт DXF / DWG. Здесь можно посмотреть настройки стандартной схемы, создать и обработать собственные схемы. В настройках задайте тип файла (DXF или DWG) для экспорта.
4. Выберите в групповом поле Средство экспорта диалогового окна Экспорт DXF/DWG, с помощью какого средства следует выполнить экспорт.
5. При экспорте ***отдельной*** страницы в поле Имя файла введите имя генерируемого файла или выберите его в интерактивном режиме при помощи кнопки ++...++.
6. Установите флажок Применить ко всему проекту, если первым действием вы выбрали отдельные страницы проекта и теперь хотите расширить этот выбор до всего проекта.
7. Щелкните по кнопке ++OK++.

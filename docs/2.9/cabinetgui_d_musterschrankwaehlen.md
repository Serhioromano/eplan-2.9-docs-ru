# Диалоговое окно Выбрать шаблон интерпретации

Вы открыли проект. Проект содержит полностью определенный электрошкаф. Навигатор пространства листов открыт, и одно пространство листов открыто. Пространство листа содержит импортированные трехмерные тела, которые необходимо сохранить в виде электрошкафа. Выделена и выбрана трехмерная геометрическая фигура с прямоугольником.

Обработать > Логика устройства > Автоматическая интерпретация

В этом диалоговом окне можно выбрать электрошкаф из любого пространства листа внутри открытого проекта или из структуры дерева в базе данных изделий, который будет служить образцом для копирования логических элементов. Диалоговое окно содержит такие вкладки:

* Специфический для проекта
* Специфический для изделий.

Обзор основных элементов диалогового окна:

Вкладка Специфический для проекта

На этой вкладке представлено дерево проекта с пространствами листа всех открытых проектов. Это дает возможность использовать логические элементы функциональных элементов из другого (не из активного) проекта, например из проекта макросов. В представлении структуры дерева проекта учитываются возможные настройки фильтра. В дереве можно выбрать одно или несколько пространств листа в качестве объекта шаблона.

### Фильтр

В этом раскрывающемся списке отображаются все доступные фильтры. Выбранный фильтр активируется автоматически и применяется как к дереву, так и к списку. Запись "- Не активировано -" отключает фильтр и приводит к тому, что данные отображаются в неотфильтрованном виде. С помощью кнопки ++...++ откройте диалоговое окно [Фильтр](modaldialogsdb_d_filternnach.md). Здесь можно создать, обработать, удалить, скопировать, экспортировать, импортировать фильтр и управлять им.

Во всплывающем меню раскрывающегося списка Фильтр содержатся следующие записи:

* Выключить: Этот пункт меню доступен, если фильтр установлен: Сбрасывает настройку фильтра до записи "- Не активировано -".
* Активировать <Имя фильтра>: Этот пункт меню доступен, если для настройки фильтра установлено значение "- Не активировано -": Повторно активирует последний активный фильтр.

Таким образом можно быстро переключаться между неотфильтрованным и отфильтрованным в соответствии с требованиями пользователя представлениями.

### Значение: <Свойство>

При помощи [быстрого ввода](modaldialogsdb_k_filter.md) в данном поле для определенного и активированного фильтра можно быстро изменить значение его критерия.

### Дерево

Отображает все имеющиеся в открытом проекте пространства листа, электрошкафы и монтажные платы. Если в представлении в виде дерева выделен электрошкаф, с помощью клавиши ++OK++ можно перенести его логические элементы на ранее выбранные трехмерные объекты.

* * *

Специфический для изделия

На этой вкладке представлено дерево изделий, согласованное с базой данных изделий. Отображаются только изделия, имеющие 3D-макрос. В представлении структуры дерева изделий учитываются возможные настройки фильтра. В дереве можно выбрать одно или несколько изделий в качестве объекта шаблона.

### Список

В списке отображаются изделия, выбранные в качестве объектов шаблонов, с номером изделия и присвоенным графическим макросом.

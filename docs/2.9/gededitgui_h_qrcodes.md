# Вставить QR-коды

QR-код состоит из черных и белых квадратов, расположенных в квадратной сетке. Такой код, например, используется для кодированного отображения веб-адресов, телефонных номеров, учетных данных для доступа к WLAN и т. д. С помощью мобильного устройства делается снимок кода, который интерпретируется посредством приложения, благодаря чему отпадает необходимость в утомительном ручном вводе этих данных.

В графическом редакторе вы можете использовать соответствующий пункт меню, чтобы сгенерировать QR-код из текста, веб-адреса или свойств и отобразить его на странице проекта.

Условие:

В Графическом редакторе открыта страница проекта.

1. Выберите следующие пункты меню: Вставить > Графика > QR-код
2. Задайте первую угловую точку QR-кода.
3. Растяните квадрат QR-кода в любом направлении и задайте конечную точку.
4. В диалоговом окне Свойства (QR-код) выберите вкладку QR-код.
5. В поле Формат блока введите текст или веб-адрес, который необходимо сгенерировать в виде QR-кода и отобразить на странице проекта. Если для QR-кода нужно использовать свойство, с помощью кнопки ++…++ перейдите в следующее диалоговое окно Формат, чтобы выбрать свойство и, при необходимости, дополнительные элементы формата, например разделитель.

!!! info "Для сведения:"

    Если ввести данные в поле Формат блока, QR-код сгенерируется без содержимого.

6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    При перемещении курсора по квадрату QR-кода его содержимое отображается в виде всплывающей подсказки.

!!! note "Замечание:"

    * С помощью вкладки Формат настраивается только начальная точка или размер квадрата QR-кода. Другие настройки формата невозможны.
    * QR-код не обрабатывается при помощи команд Обработать > Поворачивать или Обработать > Отобразить зеркально, поскольку ориентация QR-кода не должна изменяться во время обработки.
    * Для вывода QR-кодов на страницах отчета можно использовать флажок [Отобразить как QR-код](formeditorgui_r_platzierung.md) для текстов-заполнителей в диалоговом окне Свойства (текст-заполнитель) на вкладке Размещение. Это позволяет, например, сохранять гиперссылки на страницы производителя в отчете "Список обозначений устройств" для изделий, используемых в проекте.

!!! tip "Совет:"

    Если QR-код сгенерирован из веб-адреса или адреса электронной почты, то по выбранному QR-коду с помощью пункта всплывающего меню Открыть гиперссылку можно перейти на соответствующую интернет-страницу или запустить программу электронной почты на компьютере.

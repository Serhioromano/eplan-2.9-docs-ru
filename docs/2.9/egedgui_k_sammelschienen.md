# Символы соединения: Сборные шины

Сборные шины EPLAN реализуются при помощи отдельных функций, так называемых "Сборные шины". У них есть обозначение устройства, существуют также главные и (0-n) вспомогательные функции. В отличие от других элементов в схеме соединений нужно учитывать следующие отклонения:

* Для структуры устройства сборных шин имеется собственная настройка (вкладка Структура диалогового окна Свойства проекта), т. к. сборные шины в проектах DIN, как правило, также не имеют установки и места установки.
* Обозначения вывода устройства возможны, но не обязательны, отсутствующие обозначения вывода устройства не опознаются как ошибка.
* Существует передача потенциала с первого вывода главной функции на выводы вспомогательных функций (и на оставшиеся выводы главной функции).
* Существует передача сигнала с первого вывода главного устройства на вывод вспомогательных устройств (и на оставшиеся выводы устройства главного устройства). При этом нужно следующим образом учитывать настройку С разделением сигнала (флажок во вкладке Данные символа / функции диалогового окна Свойства (усл. обозначение)): Если на главной функции установлен флажок С разделением сигнала, то сигнал ***не*** передается на вспомогательную функцию. Если на вспомогательной функции установлен флаг, то этой вспомогательной функции сигнал не передается; это не играет роли для передачи сигнала от главной функции другим вспомогательным функциям.
* Для нумерации соединений сборных шин предусмотрена настройка Группа соединений = "Соединения, подсоединенные к сборным шинам".
* Доступ к изделиям для сборных шин можно получить в поле Главн. группа продукт. = "Электротехника", Группа продуктов = "Пути передачи".

Настройка Категория = "Сборные шины" содержит следующие определения функций:

* Группа "Выводы устройства сборных шины": Определение функции = "Вывод устройства сборной шины"
* Группа "Выводы устройства сборных шины, 2 вывода устройства": Определение функции = "Выводы устройства сборных шин, 2 вывода устройства"
* Группа "Вывод устройства сборной шины, переменная": Определение функции = "Вывод устройства сборной шины, переменная"

Функции по умолчанию являются вспомогательными функциями. Выводы устройства обладают свойствами Тип вывода устройства = "Не определено", Тип потенциала = "Не определено", а настройка Подключение кабеля имеет значение "Нет" (диалоговое окно Логическая схема выводов устройства свойств условного обозначения).

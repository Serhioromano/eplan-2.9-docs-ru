# Создать группы форматов

Условие:

Вы открыли проект.

1. Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Нумерация соединений
2. Выберите в диалоговом окне Настройки: Нумерация соединений определенную пользователем схему из раскрывающегося списка или создайте с помощью команды ![](images/all_new_as.png){: .ui-icon } (Создать) новую схему.
3. Выберите вкладку Обозначение и щелкните на панели инструментов над полем Группы форматов на кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
4. Выберите в диалоговом окне Нумерация соединений: Форматы в поле Группа соединений одну из предварительно определенных групп соединений либо выберите запись "Устройство..." или "Группирование...".
5. Если над записью "Устройство..." или "Группирование..." открылось диалоговое окно Список выбора, выберите в нем одно из имеющихся в проекте ОУ или значение для группирования и нажмите ++OK++.
6. В поле Объем укажите число соединений, которым будет присвоен одинаковый формат обозначений соединений. Выберите запись из раскрывающегося списка.

!!! note "Замечание:"

    Если вы выбрали запись "Сеть", через расширение сети можно увеличить зону воздействия для нумерации.

7. Выделите в поле Доступные элементы формата элемент формата и перенесите его с помощью клавиши со стрелкой вправо в список выбранных элементов формата.

!!! info "Для сведения:"

    В зависимости от элемента формата открывается следующее диалоговое окно. В зависимости от выбранного элемента формата это может быть одно из следующих диалоговых окон: [Формат: Свойства](wirenumberinggui_d_formateigenschaften.md), [Формат: Счетчик](wirenumberinggui_d_formatzaehler.md), [Формат: Страница / Столбец / Строка](wirenumberinggui_d_formatseite.md) или [Формат: Разделитель](wirenumberinggui_d_formattrennzeichen.md).

8. Определите форматирование в следующем диалоговом окне.

!!! info "Для сведения:"

    В поле Предварительный просмотр выводится предварительный просмотр формата обозначения. Отдельные элементы формата выводятся в обозначении соединения в той последовательности, в которой они стоят в списке.

9. При необходимости измените последовательность элементов формата с помощью кнопок со стрелками.
10. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Введенные значения сохраняются в выбранной схеме.

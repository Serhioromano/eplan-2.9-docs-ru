# Устройства в макросе/автоматически нумеровать копии

Макросы — это готовые к использованию фрагменты схем соединений, которые могут содержать устройства/функции. Скопированные или вырезанные элементы схем соединений также могут содержать устройства/функции. Устройства/функции, которые были добавлены таким образом на страницу схемы соединений, могут уже в момент вставки автоматически включаться в нумерацию. При этом для каждого вставленного устройства/каждой функции ***в режиме подключения*** определяется ОУ, которое автоматически присваивается устройству. При этом счетчик ОУ считает по нарастающей.

!!! note "Замечание:"

    Нумеруются группы устройств, которые были выбраны в формате нумерации в режиме подключения.

Условия:

* Вы открыли проект.
* Вы открыли страницу схемы соединений.
* У вас имеется доступ к макросу, или вы скопировали в буфер обмена фрагмент схемы соединений, который содержит устройства/функции.

### Добавить нумерацию перед вставкой

1. Параметры > Настройки > Проекты > "имя проекта" > Устройство > Нумерация (online)
2. Выберите в групповом поле При вставке макросов и копий опцию Нумерация.
3. Если вы хотите создать или обработать формат нумерации в режиме подключения, щелкните в поле Формат нумерации [...].

!!! info "Для сведения:"

    Открывается диалоговое окно Форматы нумерации. Здесь вы можете создать [формат нумерации в режиме подключения](numerationgui_h_onlineformat.md). Щелкните по кнопке ++ОК++.

4. Если вы хотите использовать уже существующий формат нумерации в режиме подключения, выберите в групповом поле При вставке макросов и копий из раскрывающегося списка Формат нумерации формат нумерации в режиме подключения.
5. Щелкните по кнопке ++ОК++.
6. Чтобы добавить копию из буфера обмена на страницу схемы соединений, выберите пункты меню Обработка > Вставка.
7. Чтобы добавить макрос, выберите пункты меню Вставить > Макрос окна / символа, затем выберите в диалоговом окне макрос и щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Копия/макрос появится рядом с курсором.

8. Переместите курсор в нужное место на странице схемы.
9. Для размещения копии/макроса щелкните страницу.

!!! info "Для сведения:"

    Устройство/функция нумеруются автоматически.

10. Выберите пункт меню Обработать > Отменить операцию.

### Добавить нумерацию при вставке

Если вы активируете в общих настройках диалоговое окно запроса на нумерацию, каждый раз при вставке будет возможность задавать нумерацию устройств в макросе или копиях заново.

1. Параметры > Настройки > Проекты > "имя проекта" > Устройство > Нумерация (online)
2. Выберите в групповом поле При вставке макросов и копий опцию C запросом и щелкните ++OK++.
3. Чтобы добавить копию из буфера обмена на страницу схемы соединений, выберите пункты меню Обработка > Вставка.
4. Чтобы добавить макрос, выберите пункты меню Вставить > Макрос окна / символа, затем выберите в диалоговом окне макрос и щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Копия/макрос появится рядом с курсором.

5. Переместите курсор в нужное место на странице схемы.
6. Для размещения копии/макроса щелкните страницу.

!!! info "Для сведения:"

    Откроется диалоговое окно "Режим вставки".

7. Выберите параметр Нумерация.
8. Если вы хотите создать или обработать формат нумерации в режиме подключения, щелкните в поле Формат нумерации [...].

!!! info "Для сведения:"

    Открывается диалоговое окно Форматы нумерации. Здесь вы можете создать [формат нумерации в режиме подключения](numerationgui_h_onlineformat.md). Щелкните по кнопке ++ОК++.

9. Если вы хотите использовать уже существующий формат нумерации в режиме подключения, выберите в диалоговом окне Режим вставки из раскрывающегося списка Формат нумерации формат нумерации в режиме подключения.
10. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Устройство/функция нумеруются автоматически.

11. Выберите пункт меню Обработать > Отменить операцию.

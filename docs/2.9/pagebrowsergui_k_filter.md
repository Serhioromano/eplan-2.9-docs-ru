# Фильтр страниц

В навигаторе страниц можно настроить фильтр для отображения страниц. Использование фильтра особенно полезно при работе с большими проектами, так как он позволяет ограничить отображение до только что обработанных / обрабатываемых странниц. Кроме того, фильтр используется для следующих операций:

* ***Отобразить страницы в навигаторе страниц***
Если пользователь настроил фильтр, в навигаторе страниц будут выведены только выбранные страницы. Если для блока идентификаторов какого-либо слоя (например, "Установка") скрыты все страницы, сам блок идентификаторов также не выводится в дереве.
* ***Смена страниц***
Если требуется обработать только определенные страницы проекта, их можно отфильтровать. После этого во время графической обработки с помощью клавиш [Рисунок вверх] / [Рисунок вниз] можно переходить между выбранными страницами, пропуская другие страницы, находящиеся между ними.
* ***Обработать свойства***
В данном случае также возможно обработать свойства нескольких выделенных страниц и, например, присвоить всем страницам схемы соединений единую рамку.
* ***Печать***
С помощью фильтра можно выводить на печать только определенные страницы (например, все страницы типа "Обзор") или же исключать страницы из вывода на печать.
* ***Экспортировать***
Кроме того, можно экспортировать только определенные страницы или исключать страницы из экспорта.

!!! tip "Совет:"

    Если при печати / экспорте страниц требуется выводить только текущие страницы активного фильтра страниц, необходимо активировать в диалоговом окне Печать параметр Выделение, а в диалоговом окне экспорта флажок Применить ко всему проекту ***удалить***.

Фильтр страниц — это фильтр по полю, который задается (как обычно в EPLAN) с помощью диалогового окна Фильтр и сохраняется в схеме. В качестве критериев фильтра можно использовать различные свойства страницы и / или свойства и определения функций, размещенных на страницах. Это предоставило следующие возможности:

* ***Использовать список результатов поиска для фильтра страниц***
С помощью свойства Содержит результаты в списке результатов поиска (ид. 11045) можно учитывать страницы, содержащиеся в списке результатов поиска, при фильтрации страниц.
* ***Использовать свойства и определения функций для фильтра страниц***
С помощью критериев фильтра ***Функция на странице*** можно фильтровать страницы по размещенным на них функциям. В качестве критериев здесь доступны свойства для ***структурных идентификаторов***, Вида представления, Раздела или Определения функции. В результате применения такого фильтра страниц отображаются только страницы, на которых размещены функции, значения свойств или определения функций которых соответствуют заданным критериям.

!!! example "Пример:"

    ***Фильтр для функций на страницах: Определение функции***Вы открыли проектESS_Sample_Project. В диалоговом окне Фильтрсоздайте схему для PE-клемм и в выборе критериев выберите для нее запись "Функция на странице: Определение функции". В таблице с критериями фильтра задайте следующее:Активно Отрицат.Критерий Оператор Значение Функция на странице / Определение функции=PE-клемма с контактом шины, 2 вывода(ов)Когда вы закроете диалоговое окно Фильтр, нажав кнопку ++OK++, схема фильтра сохранится, а установленный фильтр активируется автоматически. В навигаторе страниц отображаются только страницысхемысоединений, на которых также встречаются PE-клеммы с указанным выше определением функции. В данном примере это, например, страницы "=GB1+A1&EFS1/1", "=GD1+A2&EFS1/1" и т. д.***Фильтр для функций на страницах: Структурный идентификатор***Вы открыли проектESS_Sample_Project. Создайте схему в диалоговом окне Фильтр. Для этого в выборе критериев в уровне иерархии "Функция на странице" выберите свойство Место установки. В таблице с критериями фильтра задайте следующее:Активно Отрицат.Критерий Оператор Значение Функция на странице / Место установки=B4Когда вы закроете диалоговое окно Фильтр, нажав кнопку ++OK++, схема фильтра сохранится, а установленный фильтр активируется автоматически. В навигаторе страниц отображаются только страницы, на которых встречаются функции с заданным структурным идентификатором. В данном примере это страница "=VN01&PFB/1" с описанием "Покрыть заготовку лаком", на которой в качестве функций размещены "Аналоговые датчики" с местом установки "B4".

!!! note "Замечание:"

    Свойство страницы Содержит результаты в списке результатов поиска и критерии для функций на страницах доступны только для выбора критериев фильтра страниц.

Фильтр одновременно активируется в навигаторе страниц для всех открытых проектов посредством выбора схемы фильтра. Поэтому при смене фильтра все страницы проекта могут быть скрыты в навигаторе страниц, так как ни одна из его страниц не удовлетворяет критериям фильтра. Если при активации фильтра генерируются страницы, то они отображаются только если они соответствуют критериям фильтра. При этом не имеет значения, были ли сгенерированы эти страницы с помощью команд Создать, Копировать, Импорт DXF / DWG или с помощью API. Если фильтр в проекте отсутствует, выводятся все страницы.

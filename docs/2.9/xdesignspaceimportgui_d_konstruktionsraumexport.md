# Диалоговое окно "Экспорт пространства конструкции"

Пространство листа > Пространство конструкции > Экспортировать

В этом диалоговом окне выполняется экспорт подготовленных в EPLAN Pro Panel пространств конструкции во внешние программы CAD.

Обзор основных элементов диалогового окна:

### Источник

В этом поле отображается источник экспорта данных. Если указан весь проект, то появляется его имя, а для отдельных пространств листа отображаются полные имена пространств листа. Обработка записей в данном поле невозможна.

### Схема

Выбор схемы или возможность обработки или создания схемы. Кнопка ++...++ открывает диалоговое окно "Настройки" для экспорта STEP.

### Каталог вывода

В этом поле указывается место хранения. Здесь предварительно выбран каталог, заданный в настройках пользователя для экспорта данных.

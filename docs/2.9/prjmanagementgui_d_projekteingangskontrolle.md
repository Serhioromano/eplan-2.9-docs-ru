# Диалоговое окно Проверить проект

Вы выбрали базу данных управления проектами. Вы считали каталоги и проекты. Проект > Управление. Выделите проект ввода, содержащий прогрессивное состояние проекта. Выберите ++Дополнительно++ > Проверить проект.

В этом диалоговом окне определите, какие операции будут производиться во время нормоконтроля EPLAN. В процессе нормоконтроля настройки и свойства проекта для сравнения сравниваются с настройками и свойствами проекта с замечаниями. Вы также выполняете целенаправленные контрольные прогоны по проекту с замечаниями.

Доступны следующие элементы диалогового окна:

### Выбрать проект для сравнения

Выделите в этом поле проект для сравнения. Проект для сравнения — это исходный проект, с которым сравнивается проект с замечаниями. Если поле не содержит подходящий проект для сравнения, щелкните по ![](images/all_new_as.png){: .ui-icon } (Создать) и выберите проект в открывшемся диалоговом окне.

### Сравнить настройки

Если установлен этот флажок, производится сравнение всех специфичных настроек проекта для сравнения и проекта с замечаниями, которые вы выбрали в поле Схема в схеме сравнения для сравнения настроек.

* Схема: Выберите из этого раскрывающегося списка схему сравнения для сравнения настроек. С помощью этой схемы проверьте идентичность специфичных настроек проекта для сравнения и проекта с замечаниями. Через [...] можно создавать новую схему или обрабатывать существующую. Схема сравнения учитывается при сравнении проектов в ходе нормоконтроля, если установлен флажок Сравнить настройки.

!!! tip "Совет:"

    Если необходимо сравнить только настройки проектов, то легче всего это сделать с помощью функции поиска в настройках: просто задайте соответствующее искомое понятие. Таким образом генерируется аналог фильтра и иерархическая структура дерева отображается для всех отрытых проектов на одинаковом уровне иерархии.

### Сравнить свойства

Если установлен этот флажок, производится сравнение всех свойств проекта для сравнения и проекта с замечаниями, которые вы выбрали в поле Схема в схеме сравнения для сравнения свойств.

* Схема: Выберите из этого раскрывающегося списка схему сравнения для сравнения свойств. С помощью этой схемы проверьте идентичность свойств проекта для сравнения и проекта с замечаниями. Через [...] можно создавать новую схему или обрабатывать существующую. Схема сравнения учитывается при сравнении проектов в ходе нормоконтроля, если установлен флажок Сравнить свойства.

### Выполнить контрольные прогоны

Если установлен этот флажок, при сравнении проектов в ходе нормоконтроля выполняются все контрольные прогоны, выбранные в поле Схема в схеме контрольных прогонов.

* Схема: Выберите из этого раскрывающегося списка схему проверки для сравнения проектов при нормоконтроле. Через [...] можно создавать новую схему проверки или обрабатывать существующую. Схема контрольного прогона учитывается при сравнении проектов в ходе нормоконтроля, если установлен флажок Выполнить контрольные прогоны.

### Сравнить структуры проектов

Если флажок установлен, производится сравнение структур проекта для сравнения и проекта с замечаниями. Сравниваются все настройки, определенные через Проект > Свойства на вкладке Структура.

### Сравнить слои

Если флажок установлен, производится сравнение настроек слоев проекта для сравнения и проекта с замечаниями.

* Сообщить о новых слоях: Эта функция доступна, если установлен флажок Сравнить слои. Если флажок установлен, EPLAN регистрирует вновь созданные слои. Новые слои можно создавать, например, с помощью импорта DXF\- / DWG.

### Остановить сравнение при первом отличии

Если флажок установлен, EPLAN останавливает все выбранные сравнения и контрольные прогоны нормоконтроля, как только обнаруживается различие существенных с точки зрения сравнения настроек или свойств в проекте для сравнения и проекте с замечаниями. Откроется диалоговое окно Проверка проекта: Результаты, и EPLAN покажет, какие операции больше не проводились.

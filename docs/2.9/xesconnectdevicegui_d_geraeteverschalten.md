# Диалоговое окно/Вкладка Подключить устройства

Вы открыли проект.

* Данные проекта > Устройствo > Подключить устройства.
* В графическом редакторе уже выделена точка определения соединения. Обработать > Свойства. В диалоговом окне Свойства ++...++ на вкладке Данные символа / функции выбран Вид представления "Однополюсн". Выберите вкладку Подключить устройства.

Диалоговое окно Подключить устройства: В данном диалоговом окне генерируются неразмещенные соединения между выводами устройства. Для этого выберите источник устройства и цель устройства. Вы можете соединять устройства, еще неразмещенные на странице. Вы можете создавать кабели, еще неразмещенные на странице. Новый созданный кабель выводится через меню Данные проекта > Кабели > Навигатор в навигаторе кабеля.

Вкладка Подключить устройства: На данной вкладке существует возможность определения многополюсных соединений в точке определения соединений в ***однополюсном представлении*** схемы соединений. В таком случае в таблице уже отображаются выводы устройств связанного через однополюсное соединение устройства. Произвольный выбор источника и цели невозможен. Также невозможно создание кабелей, так как при этом возникали бы неразмещенные функции.

Обзор основных элементов диалогового окна:

### Источник устройства

В этом поле выберите устройство, которое должно быть началом / источником соединения. Если в навигаторе выделено устройство, имеющее вывод устройства, то устройство автоматически вводится в поле. Выводы выбранного устройства отображаются в находящейся ниже таблице, в столбце Вывод устройства. С помощью кнопки ++...++ можно открыть диалоговое окно Выбрать устройство. Оно показывает все устройства, которые вы можете выбрать как источник для соединения.

Если открыта вкладка Подключить устройства для ***однополюсной точки определения соединения***,то источник устройства однополюсного соединения автоматически вводится в поле. Произвольный выбор устройства невозможен. Выводы выбранного устройства отображаются в находящейся ниже таблице, в столбце Вывод устройства. При этом отображаются только выводы неразмещенных многополюсных функций.

### Функциональный текст

Показывает функциональный текст устройства, которое вы выбрали в поле Источник устройства. Функциональный текст вводится в диалоговом окне Свойства ++...++ в поле Функциональный текст.

### Таблица

Показывает все выводы устройства, относящиеся к устройству, которое вы выбрали в поле Источник устройства. Имеющиеся соединения выводятся в столбце Цвет / номер соединения. На выводах устройства, на которых существует соединение, может быть создано последующее соединение. Представление столбца можно конфигурировать с помощью всплывающего меню. Если щелкнуть по заголовку столбца, можно выполнить сортировку записей.

### Внутр. / внешн.

В раскрывающемся списке выберите отображение только внутренних, только внешних или обеих сторон клеммника, выбранного началом/источником соединения.

Если открыта вкладка Подключить устройства для ***однополюсной точки определения соединения***, то на выбор предлагаются только те стороны клеммника, которые подключенны с помощью однополюсного соединения.

### Выбрано

Показывает число клемм для источника устройства, которые выделены в расположенном выше поле.

### ОУ кабеля

В этом поле выберите кабель. Если в навигаторе устройства или кабеля выделен кабель, тогда он автоматически вводится в поле. Кнопка ++...++ открывает диалоговое окно Выбрать кабель. Диалоговое окно показывает все кабели, которые вы можете выбрать.

Вы создаете новый кабель тем, что вводите ОУ кабеля. С помощью [...] рядом с полем Тип кабеля присвойте кабелю изделие. Если кабелю не присвоено изделие, он доступен с любым числом кабельных соединений.

Если открыта вкладка Подключить устройства для ***однополюсной точки определения соединения***, то отображается кабель, присвоенный однополюсному соединению. Создание нового кабеля невозможно.

### Тип кабеля

Показывает тип кабеля, который введен в поле ОУ кабеля. Кнопка ++...++ доступна, если вы ввели в поле ОУ кабеля новое ОУ кабеля. Открывает диалоговое окно Выбор изделий и позволяет присваивать кабелю изделие.

### Статистика

В этом поле представлена следующая информация о соединениях.

* Свободные кабельные соединения: Показывает число кабельных соединений, которые еще не связаны.
* Свободные PE-жилы: Показывает число PE- / PEN-жил, которые еще не связаны.
* Свободные экранирования: Показывает число экранирований, которые еще не связаны.

### Цель устройства

В этом поле выберите устройство, которое должно быть конечной точкой / целью соединения. С помощью кнопки ++...++ можно открыть диалоговое окно Выбрать устройство. Оно показывает все устройства, которые вы можете выбрать как цель для соединения. Выводы выбранного устройства отображаются в находящейся ниже таблице, в столбце Вывод устройства.

Если открыта вкладка Подключить устройства для ***однополюсной точки определения соединения***, то цель устройства однополюсного соединения автоматически вводится в поле. Произвольный выбор устройства невозможен. Выводы выбранного устройства отображаются в находящейся ниже таблице, в столбце Вывод устройства. При этом отображаются только выводы неразмещенных многополюсных функций.

### Функциональный текст

Показывает функциональный текст устройства, которое вы выбрали в поле Цель устройства. Функциональный текст вводится в диалоговом окне Свойства ++...++ в поле Функциональный текст.

### Таблица

Показывает все выводы устройства, относящиеся к устройству, которое вы выбрали в поле Цель устройства. Имеющиеся соединения выводятся в столбце Цвет / номер соединения. Представление столбца можно конфигурировать с помощью всплывающего меню. Если щелкнуть по заголовку столбца, можно выполнить сортировку записей.

### Внутр. / внешн.

В раскрывающемся списке выберите отображение только внутренних, только внешних или обеих сторон клеммника, выбранного конечной точкой/источником соединения.

Если открыта вкладка Подключить устройства для ***однополюсной точки определения соединения***, то на выбор предлагаются только те стороны клеммника, которые подключенны с помощью однополюсного соединения.

### Выбрано

Показывает для цели устройства число выводов, которые выделены в расположенном выше поле.

### Обработать кабель

Если этот флажок установлен, то при щелчке по кнопке ++Генерировать соединения++ автоматически открывается диалоговое окно Обработать кабели.

### Учитывать мостовые перемычки

Если флажок снят, то в таблице отображаются выводы устройства с типом вывода "Мостовая перемычка". Дополнительно отображаются выводы устройства автоматически сгенерированных мостовых перемычек с любым типом вывода устройства.
Отображение выводов устройств мостовых перемычек означает, что клеммы могут отражаться многократно, так как логическая модель "Клемма, общ." обладает четырьмя выводами устройства. Два таких вывода предусмотрены для мостовых перемычек. Для распознания типа вывода устройства в таком случае необходимо дополнительно отобразить свойство Логическая схема выводов устройства: Тип вывода устройства в таблице (через Всплывающее меню > Конфигурировать представление).

Если флажок не установлен, то выводы устройства с типом вывода устройства "Мостовая перемычка" и выводы устройства с автоматически сгенерированными мостовыми перемычками будут скрыты.

### [Генерировать соединения]

Генерирует соединения между выделенными выводами устройства источника устройства и выделенными выводами устройства цели устройства. Если при прокладке кабельных соединений было выделено неравное число выводов устройства, все соединения генерируются для меньшего количества. Если было выделено больше выводов устройства, чем имеется кабельных соединений, соединяются те выводы устройства, для которых доступны кабельные соединения.

# Маршрутизируемые соединения (пространство листа)

Маршрутизация — это генерирование маршрутизируемых соединений с видом представления "Трехмерный чертеж монтажных поверхностей". Эти соединения прокладываются в пространстве листа. Маршрутизируемое соединение имеет все свойства, которые есть у многополюсного соединения, и дополнительно обладает информацией о своей трассе маршрутизации.

Прокладываются, как правило, соединения, для которых существует размещенное трехмерное размещение изделия. Во время прокладки одного многополюсного соединения генерируется маршрутизируемое соединение. Маршрутизация соединений в однополюсном представлении невозможна, поэтому этот пункт меню недоступен.

В пространстве листа маршрутизируемые соединения отображаются в соответствующем цвете и с указанием внешнего диаметра. Если оба свойства не определены, используется стандартное представление.

При маршрутизации рассчитываются следующие данные:

* Трассы разводки
* Длина проводов
* Обработка концов проводов
* Степень заполнения кабельных каналов.

На основании расчетов маршрутизируемых соединений можно составить следующие отчеты:

* Спецификации проводов для руководств по разводке проводов в электрошкафу.
* Управляющие файлы машин для сборки проводов, которые изготавливают, обжимают и выполняют соответствующую обработку концов проводов определенной длины.

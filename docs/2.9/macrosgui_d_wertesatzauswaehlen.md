# Диалоговое окно Выбрать набор значений

В Графическом редакторе открыта страница проекта. Либо вы открыли пространство листа. Вы выделили объект-заполнитель с наборами значений и выбрали пункт всплывающего меню Присвоить набор значений.

В этом диалоговом окне выберите набор значений объекта-заполнителя. В результате свойства, которым была присвоена переменная, получают значения из набора значений.

Если для переменной на вкладке Значения установлен флажок Изменяемое, для нее можно настроить значение в выбранном наборе значений.

Обзор основных элементов диалогового окна:

### Фильтр

Введите в это поле текст, по которому должна выполняться фильтрация наборов значений. В раскрывающемся списке Набор значений отобразятся те наборы значений, в которых имя набора значений или соответствующее значение содержат введенный текст.

Если необходимо вернуться к исходному выбору, удалите текст из поля ввода. Для этого нажмите кнопку ![](images/all_delete_as.png){: .ui-icon } (Удалить).

### Набор значений

Выберите здесь имя набора значений для объекта-заполнителя.

### Таблица

Ячейки таблицы со значениями можно обрабатывать только для изменяемых переменных.

Таблица содержит следующие столбцы:

* Строка: Показывает непрерывный номер строки.
* Переменная: Здесь отображаются имеющиеся переменные.
* <Набор значений>: Здесь отображаются значения из выбранного набора значений. Здесь можно изменить значения переменных, которые в диалоговом окне 'Свойства' объекта-заполнителя установлены как изменяемые.
В результате создается дополнительный определенный пользователем набор значений, а перед именем первоначально выбранного набора значений устанавливается префикс "Определено пользователем". При повторном выборе исходного набора значений определенный пользователем набор значений снова удаляется.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

### Отобразить переменные для свойств проекта и страницы

Этот флажок доступен, только если в объекте-заполнителе была введена переменная хотя бы для одного свойства проекта или страницы.

Если этот флажок установлен, отобразятся переменные для свойств проекта и / или страницы, а параметры выбранного набора значений для этих переменных перенесутся в проект и / или на соответствующие страницы.

### Увеличение фрагмента

Во время предварительного просмотра отображается страница / пространство листа, куда был вставлен объект-заполнитель. Если этот флажок установлен, область с выбранным объектом-заполнителем и соответствующими объектами отображается с увеличением.

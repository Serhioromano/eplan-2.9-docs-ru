# Диалоговое окно Настройки: Нумерация кабелей

Вы открыли проект.

* Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Кабель (автоматически). Щелкните рядом с полем Нумерация кабелей по ++...++.
* Данные проекта > Кабели > Автоматически генерировать кабели. Щелкните в диалоговом окне Автоматически генерировать кабели в групповом поле Нумерация кабелей рядом с полем Настройки на ++...++.
* Данные проекта > Кабели > Навигатор. Выделите несколько значений, содержащих кабели, и выберите Всплывающее меню > Нумеровать ОУ кабелей. Щелкните в диалоговом окне Нумеровать кабели рядом с полем Настройки по кнопке ++...++.

В этом диалоговом окне задаются настройки для нумерации кабелей.

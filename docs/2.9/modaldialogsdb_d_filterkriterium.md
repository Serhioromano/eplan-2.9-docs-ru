# Диалоговое окно Выбор критериев

Открыто диалоговое окно Фильтр. Щелкните по ![](images/all_new_as.png){: .ui-icon } (Создать) в панели инструментов критериев фильтра или по кнопке ++...++ в поле Критерий.

В этом диалоговом окне можно выбрать свойство, для которого требуется определить критерий фильтра.

Обзор основных элементов диалогового окна:

### Фильтр

Введите в это поле текст, по которому должна выполняться фильтрация свойств / критериев.

Если вы захотите вернуться к прежнему виду выбора критериев, уберите текст из поля ввода. Для этого щелкните ![](images/all_delete_as.png){: .ui-icon } (Удалить).

Критерии:

Доступные критерии отображаются в структуре дерева. В зависимости от того, в каком контексте вы вызвали диалоговое окно Фильтр (в навигаторе данных проекта, при генерировании отчета, при определении областей обработки и т. д.), вам будут предложены на выбор только критерии, имеющие смысл в данном случае.

### (...)

Эта запись заключает в скобки все последующие критерии фильтрации и группирует их. Таким образом можно определить иерархию и важность критериев фильтрации.

# Объединить PDF-комментарии PDF-файлов

Объедините PDF-комментарии, если PDF-файл отправлен для комментирования и получен обратно с комментариями.

Условия:

* Вы являетесь [исполнителем](redlining_k_projektkommentator.md).
* Вы установили и открыли Acrobat Professional 7.0 / Acrobat Standard 7.0.
* [Вывести страницы файла PDF](pdfexportgui_h_seitenexportieren.md)
* [Отправить файл PDF на проверку](redlining_h_permailversenden.md) или [освободить файл PDF для комментирования](redlining_h_freigabe.md)
* От нормоконтролера Вы получили PDF-файл, содержащий PDF-комментарии.

1. Файл > Открыть
2. Выберите в следующем диалоговом окне PDF-файл, содержащий PDF-комментарии, и щелкните ++Открыть++.

!!! info "Для сведения:"

    Открывается диалоговое окно для объединения комментариев.

3. Щелкните по кнопке ++Да++.

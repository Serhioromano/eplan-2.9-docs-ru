# Диалоговое окно Настройки: Группа проектов (присвоение)

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Представление > Группа проектов (присвоение).

В этом диалоговом окне проекту можно присвоить группу проектов. После этого для проекта вступают в силу настройки (цвет в навигаторах данных проекта, подготовка дополнительных навигаторов данных проекта), которые заданы для группы проектов в диалоговом окне Настройки: Группы проектов (определение).

Обзор основных элементов диалогового окна:

### Присвоение групп проектов

Выберите из раскрывающегося списка требуемую группу проектов.

!!! note "Замечание:"

    Две из пяти возможных групп проектов заданы предварительно ("Группа проектов 1: Проекты схем соединений", "Группа проектов 2: Проекты макросов"). Имена этих групп проектов невозможно изменить. Если для групп проектов 3–5 также должно отображаться имя после обозначения "Группа проектов <3–5>:", необходимо задать его в определенных пользователем настройках при определении.

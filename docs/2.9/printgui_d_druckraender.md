# Диалоговое окно Настройки: Границы печати

Вы открыли проект. Страница > Экспортировать > PDF > [Настройки] > Границы печати.

В этом диалоговом окне отображаются и могут быть изменены настройки печати, связанные с экспортом PDF. Выполненные здесь изменения передаются непосредственно в настройки печати (Проект > Печать > [Настройки]).

Обзор основных элементов диалогового окна:

### Слева/ справа / вверху / внизу

Здесь для всех четырех границ страницы устанавливается область, в которой печать не производится. Соответствующие отображаемые здесь единицы берутся из настроек проекта. Ввод может быть в мм и в дюймах.

### Позиция печати

Здесь задается положение PDF-страницы на листе при печати. При этом учитываются указанные границы печати. Выберите из раскрывающегося списка нужное положение.

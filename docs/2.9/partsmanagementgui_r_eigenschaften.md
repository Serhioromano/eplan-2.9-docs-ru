# Вкладка Свойства

Сервисные программы > Изделие > Управление. В дереве или списке выделено одно изделие.

На этой вкладке, в зависимости от выбранной для изделия группы продуктов, специфические для группы продуктов свойства отображаются в таблице и доступны для обработки.

Обзор основных элементов диалогового окна:

### Таблица

Таблица имеет структуру дерева, где большинство самых верхних узлов соответствуют обозначениям прежних вкладок, доступных до версии 2.6. При первом запуске базы данных изделий по умолчанию раскрываются все узлы; в случае изменения, например при сворачивании узла Данные создания / изменения, эта настройка будет действовать для ***всех*** изделий, независимо от группы продуктов.

!!! tip "Совет:"

    Свойства с темно-серым фоном показывают, что с соответствующим изделием что-то не в порядке. Например, если реле было присвоено свойство Штекеры: Вид контактов штекера (при этом группа продуктов была временно изменена), то темно-серый цвет указывает на то, что здесь было присвоено свойство, не предусмотренное для соответствующей группы продуктов. С помощью пункта всплывающего меню Удалить можно убрать такие свойства из таблицы. Этот пункт всплывающего меню доступен только для свойств с темно-серым фоном.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Перейти к |  Доступно только для свойств, которые ссылаются на другое изделие (например, свойство Номер изделия кронштейна сборной шины для изделий группы продуктов "Сборные шины"). Переходит напрямую к указанному изделию внутри базы данных изделий.

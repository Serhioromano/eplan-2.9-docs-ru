# Копировать библиотеки символов

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Сервисные программы > Основные данные > Библиотека символов > Копировать
2. В диалоговом окне Копировать библиотеку символов выберите копируемый файл. Многократный выбор ***невозможен***.
3. Щелкните по кнопке ++Открыть++.
4. В диалоговом окне Создать библиотеку символов из раскрывающегося списка Сохранить в выберите дисковод и каталог, в котором должна быть сохранена копия библиотеки символов.
5. Замените, при необходимости, предлагаемое имя файла оригинала на другое информативное имя.
6. Щелкните по кнопке ++Сохранить++.
7. В диалоговом окне Свойства библиотеки символов – <Имя библиотеки> присвойте (новые) значения нужным свойствам копируемой библиотеки символов.
8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Копия библиотеки символов создается, включается в список открытых библиотек символов и может обрабатываться.

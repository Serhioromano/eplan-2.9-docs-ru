# Выделить цветом условное обозначение и трубопровод со связанными сегментами на функциональной схеме автоматизации

Чтобы на функциональной схеме автоматизации распознать, какие резервуары, технологические контуры, трубопроводы и т. д. связаны с сегментами в предварительном планировании, эти условные обозначения и объекты на функциональной схеме автоматизации можно выделить цветом. Благодаря этому распознаются взаимосвязанные структуры на функциональной схеме и можно раньше определить возможные корректировки.

Условия:

* Вы открыли проект, который содержит данные предварительного планирования.
* Открыта страница типа "Функциональная схема автоматизации".
* На этой странице размещены технологические символы, связанные с линиями автоматического соединения.

1. Выберите следующие пункты меню: Данные проекта > Предварительное планирование > Навигатор
2. Выберите сегмент, подчиненные структуры которого вы хотите сделать видимым в функциональной схеме автоматизации.
3. Выберите пункт всплывающего меню Свойства.
4. В диалоговом окне 'Свойства' сегмента в таблице свойств выберите свойство Цвет для функциональной схемы автоматизации.
5. С помощью кнопки ++...++ откройте диалоговое окно [Выбор цвета](mf_d_farbauswahl.md) и выберите нужный цвет.
6. Выберите следующие пункты меню: Параметры > Настройки > Проекты > "Имя проекта" > Предварительное планирование > Общ.
7. В диалоговом окне Настройки: Общее в групповом поле Окрасить условное обозначение и трубопровод в функц. схеме автоматизации установите флажок С присвоенным сегментом.

!!! info "Для сведения:"

    После закрытия диалогового окна 'Настройки' нажатием кнопки ++OK++ объекты со связанными сегментами отображаются в выбранном цвете.

!!! info "Для сведения:"

    Обозначения и описания сегментов, для которых был установлен цвет с помощью свойства Цвет для функциональной схемы автоматизации, в навигаторе предварительного планирования также отображаются в выбранном цвете.

!!! tip "Совет:"

    Активируйте флажок В трубопроводах (без присвоенного сегмента), чтобы в уже выделенных цветом трубопроводах дополнительно выделить условные обозначения, которые относятся к такому трубопроводу, но которым не присвоен сегмент в предварительном планировании.

!!! note "Замечание:"

    Если флажок С присвоенным сегментом снят, то ни объекты со связанными сегментами, ни дополнительные условные обозначения в трубопроводах без связи с предварительным планированием не выделяются цветом.

# Вкладка Размещение (редакторы)

В Графическом редакторе открыта страница проекта.

* Вставить > Спец. текст > Свойства проекта или Вставить > Спец. текст > Свойства страницы. В следующем диалоговом окне выберите вкладку Размещение.
* Дважды щелкните по специальному тексту (или выберите во всплывающем меню пункт Свойства; в этом случае можно выделить несколько текстов). В следующем диалоговом окне выберите вкладку Размещение.

Вы открыли проект. Сервисные программы > Основные данные > Форма > Открыть > [Открыть].

* Вставить > Текст заполнитель. В следующем диалоговом окне выберите вкладку Размещение.
* Вставить > Спец. текст > Свойства проекта, или Вставить > Спец. текст > Свойства страницы, или Вставить > Спец.текст > Свойства формы. В следующем диалоговом окне выберите вкладку Размещение.
* Дважды щелкните по тексту заполнителю или специальному тексту (или выберите во всплывающем меню пункт Свойства; в этом случае можно выделить несколько текстов). В следующем диалоговом окне выберите вкладку Размещение.

Вы открыли проект. Сервисные программы > Основные данные > Рамка > Открыть > [Открыть].

* Вставить > Спец. текст > Свойства проекта, или Вставить > Спец. текст > Свойства страницы, или Вставить > Спец.текст > Свойства рамки. В следующем диалоговом окне выберите вкладку Размещение.
* Вставить > Спец. текст > Текст столбца или Вставить > Спец. текст > Текст строки.
* Дважды щелкните по специальному тексту (или выберите во всплывающем меню пункт Свойства; в этом случае можно выделить несколько текстов). В следующем диалоговом окне выберите вкладку Размещение.

Вы открыли проект. Сервисные программы > Основные данные > Символ > Открыть > [Открыть]. Вставить > Спец. текст > Свойства проекта или Вставить > Спец. текст > Свойства страницы. В следующем диалоговом окне выберите вкладку Размещение.

В этой вкладке выводится свойство, которое замещается заполнителем / специальным текстом и может быть в последствии доработано.

Обзор основных элементов диалогового окна:

### Свойство

В этом поле выводится текущее свойство, присвоенное тексту-заполнителю / специальному тексту; щелкните по ++...++, чтобы открыть диалоговое окно Тексты заполнителей <Тип формы> или диалоговое окно Выбор свойств и при необходимости заменить свойство другим.

### Форматированные / рассчитанные свойства

В этом поле можно построчно задавать форматирование и расчет: Щелкните кнопку ++...++, чтобы открыть диалоговое окно [Формат: Свойство](eservicesgui_d_formateigenschaft.md) и задать там настройки форматирования для свойств и / или расчетов.

### Операция

В этом поле выводится определенное пользователем расширение, присвоенное заполнителю в ходе выполнения операции; нажмите кнопку ++...++, чтобы открыть диалоговое окно Ввести имя операции и при необходимости заменить одну операцию другой.

### Отобразить как QR-код

Доступно только для параметров Свойство и Форматированные / рассчитанные свойства. Если этот флажок установлен, при генерации отчета на основе определенных значений для выбранного выше свойства генерируются QR-коды, которые отображаются на выведенных страницах отчета.

### Суммирование

В этом раскрывающемся списке выберите тип суммирования. Суммирование возможно для следующих областей:

Суммирование |  Верхний колонтитул |  Заголовок |  Область данных |  Нижний колонтитул области данных |  Нижний колонтитул/произвольно размещенный заполнитель/в статических формах
---|---|---|---|---|---
Промеж. сумма |  Недоступно |  Недоступно |  Недоступно |  суммирует заполнитель заголовка до вывода нижнего колонтитула области данных |  суммирует заполнитель на текущей странице
Промеж. сумма + Перенос |  Недоступно |  Недоступно |  Недоступно |  суммирует заполнитель заголовка до вывода нижнего колонтитула области данных и добавляет перенос вычисления предыдущего вывода нижнего колонтитула области данных. |  суммирует заполнитель на текущей странице и добавляет перенос из вычисления предыдущей страницы
Перенос |  переносит результат вычисления предыдущей страницы |  переносит результат вычисления предыдущих выводов области данных |  Недоступно |  переносит результат вычисления предыдущих выводов области данных |  переносит результат вычисления предыдущей страницы
Общая |  суммирует заполнитель для целого отчета |  Недоступно |  Недоступно |  суммирует заполнитель для целого отчета |  суммирует заполнитель для целого отчета

!!! example "Пример:"

    Выберите дляспецификацииизделий, например, текст заполнитель "Данныессылки изделия/ Общ. количество (число штук)", установите свойству Суммированиезначение "Перенос" и разместите свойство в верхнем колонтитулеформы. В этом случае в верхнем колонтитуле каждой из страниц отчета будет отображаться переходящая сумма общего объема (количества) изделий.Выберите текст заполнитель "Данные ссылки изделия / Общ. количество (число штук)", установите свойству Суммированиезначение "Общая" и разместите свойство в нижнем колонтитуле формы. На последней странице отчета отображается абсолютное общее количество (число штук) изделия.

Если заполнитель с установленным свойством Суммирование помещен в область, где эта настройка недопустима (например, настройка "Промеж. сумма" в верхнем колонтитуле), после закрытия формы появится информационное окно и заполнитель будет возвращен на настройку "Не суммировать".

### Просуммировать свойство в одной строке

Этот флажок можно использовать в случае, если в одной строке объединено несколько объектов. Например, если включено свойство формы Объединять в одну строку по и в нем настроены свойства, для которых в одной строке должны объединяться идентичные данные.

Если флажок установлен, значения свойств объектов, объединенных в одной строке, складываются. В комбинации с одним из параметров поля Суммирование будут выведены суммы из объединенных строк.

!!! note "Замечание:"

    * Если в области данных суммируются свойства в одной строке, необходимо также установить флажок Просуммировать свойство в одной строке на соответствующем тексте-заполнителе, чтобы ***Промеж. сумма*** была выведена правильно.
    * Установленная общая сумма в форме (запись "Общая" в поле Суммирование) выводится в конце отчета как конечная сумма. Однако, если текущая сумма должна отображаться в конце каждой страницы, в ***нижнем колонтитуле*** следует выбрать запись "Промеж. сумма + перенос".

### Заменить идентичный текст

Если этот флажок установлен, то при располагающихся друг под другом одинаковых текстах текст для текста-заполнителя заменяется символьной строкой из настроек (меню Настройки > Проекты > "Имя проекта" > Отчеты > Отображение / вывод). По умолчанию здесь задан символ "=". Учтите, что такая настройка недоступна для содержания.

В случае с текстами-заполнителями для ***функциональных текстов*** настройка Отчеты: Заменить идентичные функциональные тексты (в меню Параметры > Настройки > Проекты > "Имя проекта" > Управление > Совместимость) имеет приоритет перед заданной здесь настройкой.

### Отобразить все соединения

Данный флажок предусмотрен только для текста-заполнителя Таблица кабелей — область данных ... для вывода свойств кабельного соединения в формах типа 'Спецификация клеммника' или 'Спецификация штекера'!

Если флажок установлен, в области данных таблицы кабелей спецификации клеммника или штекера также отображаются отдельные соединения. Отображаются только те отдельные соединения, которые обозначены в точке определения соединения или потенциала.

Если флажок не установлен, в области данных таблицы кабелей отображаются только кабельные соединения кабелей, перечисленных в верхнем колонтитуле.

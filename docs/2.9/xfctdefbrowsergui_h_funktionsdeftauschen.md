# Заменить определения функций

В функции Вы можете менять присвоенные определения функций. При этом предлагаются только те определения функций, которые подходят по числу выводов устройства. Число выводов устройства новых определений функций должно быть равным или больше, чем значение используемого символа. Так, например, возможно для символа замыкающего контакта использовать определение функции "переключающего контакта".

если функция должна иметь меньше выводов устройства, чем используемый символ, Вы должны изменить символ. Новый символ несет с собой новую функцию.

Обмен определений функций возможен также и при помощи "Найти и заменить".

Условие:

Вы разместили функцию (то есть условное обозначение) на странице схемы соединений. Или вы сгенерировали функцию в одном из навигаторов.

1. Выделите функцию.
2. Выберите пункт меню Обработать > Свойства.
3. Выберите в диалоговом окне Свойства ++...++ вкладку Данные символа / функции, и щелкните по полю Определение на ++...++.
4. Перейдите в диалоговое окно Определения функций через дерево и выделите желаемое определение функции.

!!! info "Для сведения:"

    На вкладках справа в диалоговом окне выводятся свойства и данные вывода устройства желаемых определений функций.

5. Щелкните по кнопке ++OK++.
6. В случае необходимости, обработайте в диалоговом окне Логическая схема выводов устройства данные вывода устройства или поменяйте символ.
7. Щелкните по кнопке ++OK++.

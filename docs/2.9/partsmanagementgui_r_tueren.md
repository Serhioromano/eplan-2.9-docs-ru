# Вкладка Двери

Сервисные программы > Изделие > Управление. В дереве или списке выделено одно изделие главной группы продуктов "Механика", группа продуктов "Электрошкаф".

В этой вкладке установите общие специфические для двери данные для изделий главн. группы продукт. "Механика", Группа продуктов "Электрошкаф". При этом данные для каждого изделия могут указываться ***многократно***, а именно один раз для каждого варианта.

# Обзор операций в панелях инструментов

Следующие ***Операции*** можно вставлять в панели инструментов как кнопки:

Операция |  Значение
---|---
Открыть раб. область |  Настраивает интерфейс пользователя EPLAN на определенную рабочую область.
Установить настройку |  Устанавливает определенное значение для настроек пользователя, станции и фирмы.
Импортировать настройки |  Импорт настроек для порядка свойств, диалоговой памяти, конфигурации дерева и списка и внешних программ.
Экспортировать данные изготовления / вывести маркировку |  Выводит данные изготовления / маркировку для проекта.
Установить формат графич. элементов |  Устанавливает на определенные значения одну или несколько настроек форматирования выбранных графических элементов. Если вы еще не выбрали графический элемент, то при нажатии на кнопку курсор изменяется и вы можете выделить нужную область с графическими элементами.
Экспортировать функции |  Экспортирует свойства функций текущего проекта для внешней обработки.
Вставить устройство |  Позволяет разместить заданное устройство без вызова диалогового окна Выбор изделия.
Вставить макрос |  Позволяет вставлять определенный вариант заданного макроса без вызова диалогового окна выбора. Если вариант в макросе отсутствует, то выводится замечание.
Установить свойство проекта |  Устанавливает свойство выбранного проекта на определенное значение.
Установить проектную настройку |  Устанавливает определенное значение для настройки проекта.
Установить свойство усл. обозначения |  Устанавливает свойство выбранного условного обозначения на определенное значение.
Установить формат усл. обознач |  Устанавливает на определенные значения одно или несколько свойств форматирования у всех текстов свойств выбранных условных обозначений. Если вы еще не выбрали условное обозначение, то при нажатии на кнопку курсор изменяется и вы можете выделить нужную область с условными обозначениями.
Экспортировать страницы |  Экспортирует свойства страниц текущего проекта для внешней обработки.
Установить свойство страницы |  Устанавливает свойство выбранной страницы на определенное значение.
Вставить символ |  Позволяет вставить заданный символ без интерактивного выбора в диалоговом окне Выбор символа. Если заполнители "?" в командной строке не заменяются имеющими смысл значениями, то открывается диалоговое окно Выбор символа.
Вставить символ с определением функции |  Позволяет вставлять символы с одинаковым определением функции.
Копировать панели инструментов |  Копирует кнопки одной панели инструментов на другую панель инструментов.
Установ. формат текста |  Устанавливает на определенные значения одну или несколько настроек форматирования выбранных текстовых элементов. Если вы еще не выбрали текстовый элемент, то при нажатии на кнопку курсор изменяется и вы можете выделить нужную область с текстовыми элементами.
Экспортировать соединения |  Экспортирует свойства соединений текущего проекта для внешней обработки.
Установить формат графики соединений (точки определения соединения, точки определения потенциала) |  Устанавливает на определенные значения одну или несколько настроек форматирования выбранных точек определения соединений и / или потенциалов. Если вы еще не выбрали точку определения, то при нажатии на кнопку курсор изменяется и вы можете выделить нужную область с точками определений.

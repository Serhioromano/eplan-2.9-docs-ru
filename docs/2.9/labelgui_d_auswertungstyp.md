# Диалоговое окно Тип отчета

Вы открыли проект. Сервисные программы > Данные изготовления > Экспорт / маркировка. В диалоговом окне Экспортировать данные изготовления / Вывести маркировку рядом с полем Настройки щелкните по кнопке ++...++. В диалоговом окне Настройки: Экспорт данных изготовления / маркировка щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).

В этом диалоговом окне выберите тип отчета для схемы экспортированных данных изготовления / маркировки. Такая схема действительна только для одного типа отчета.

!!! warning "Предупреждение:"

    Тип отчета нельзя изменить после завершения настройки.

Обзор основных элементов диалогового окна:

### Доступные типы отчетов

В этом поле отображаются все доступные типы отчетов. Типы отчеты соответствуют типам форм. Выбранный здесь тип отчета выводится в диалоговом окне Настройки: Экспорт данных изготовления / маркировка в поле Тип отчета.

# Определить настройки отображения символов соединений

Общие относящиеся к проекту настройки для графической обработки охватывают различные области, которые влияют на работу в графическом редакторе.

Условие:

Вы открыли проект.

1. Параметры > Настройки > Проекты > "имя проекта" > Графическая обработка > Общее
2. Выберите в групповом поле Отображать ответвления соединений, как символы соединения должны быть представлены в графическом редакторе.
3. Установите следующие общие настройки путем установки / снятия флажков.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Символы соединения отображаются соответствующим образом.

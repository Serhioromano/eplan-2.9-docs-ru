# Диалоговое окно Экспорт шаблона сверления

Вы открыли проект. В пространстве листа или навигаторе пространства листа выделены объекты, для которых необходим экспорт ЧУ. Сервисные программы > Данные изготовления > Механическая обработка > Шаблон сверления.

В этом диалоговом окне задаются настройки экспорта данных изготовления для механической обработки. Экспортируется файл в формате "PDF" с геометрическими позициями / координатами

* Отверстия
* резьбы,
* контуров.

Файл PDF распечатывается на бумаге и накладывается на функциональный элемент как шаблон. Механическая обработка выполняется вручную путем сверления / фрезерования на обозначенных позициях.

Обзор основных элементов диалогового окна:

### Машина

В комплекте поставки содержится определение машины с названием "Стандарт". Если в настройках фирмы определены другие схемы, можно выбрать из раскрывающегося списка другие определения машины.

Кнопка ++...++ открывает диалоговое окно Настройки: Экспорт шаблона сверления, в котором можно изменить настройки схемы или определить новую схему.

### Каталог вывода

В этом поле выводится место хранения. Здесь по умолчанию выбран каталог, который указан в пользовательских настройках экспорта шаблона сверления.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Целевой файл

Данное поле доступно только в том случае, если выбраны объекты, которые могут быть экспортированы (например, монтажная плата). Здесь вводится название создаваемого PDF-файла или выбирается с помощью [...] в диалоговом окне Сохранить как. При наличии нескольких файлов поле деактивируется и действуют условия создания названий файлов и подкаталогов, определенные в схеме.

### Применить ко всему проекту

Если этот флажок установлен, текущее выделение распространяется на все важные для ЧУ монтажные поверхности проекта.

Если флажок не установлен, экспортируются только важные для ЧУ монтажные поверхности выделенных функциональных элементов.

Если отдельный проект уже выделен (и при этом фильтр не установлен), флажок автоматически включается и становится недоступным. В этом случае настройку больше невозможно изменить.

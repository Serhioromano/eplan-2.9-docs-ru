# Нумерация зон по ГОСТу

ГОСТ удлиняет горизонтальную числовую нумерацию зон справа налево и дополнительную вертикальную алфавитно-числовую нумерацию позиций снизу вверх. Сквозная постраничная нумерация зон "По проектам" также поддерживается: При этом выполняется только вертикальная нумерация зон, а алфавитно-числовые позиции начинаются на каждой странице заново при заданном начальном значении.

Для настройки нумерации зон согласно нормам ГОСТ доступны следующие свойства:

* Обратная нумерация столбцов (ид. 12035): Для проектов по ГОСТу с помощью этого свойства можно настроить нумерацию столбцов справа налево.
* Обратная нумерация строк (ид. 12036): Для проектов по ГОСТу с помощью этого свойства можно настроить нумерацию строк снизу вверх.

При этом обе настройки установлены для свойства Направление создания отчета (ид. 12103) "Вертикально".

Помимо специального свойства Направление поиска ОУ согласно ГОСТу (ид. 12104), свойства Формат нумерации строк (ид. 12009) и Формат нумерации столбцов (ид. 12011) должны быть настроены на значение "Числовое" или "Алфавитно-цифровое".

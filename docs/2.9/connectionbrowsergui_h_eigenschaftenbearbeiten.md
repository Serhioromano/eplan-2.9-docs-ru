# Просмотреть и обработать свойства соединения

Свойства соединения можно просмотреть и обработать в диалоговом окне Соединения — <Имя проекта>.

1. Выберите пункты меню Данные проекта > Соединения > Навигатор.
2. Выделите в диалоговом окне Соединения — <Имя проекта> нужные соединения и выберите пункт всплывающего меню Свойства.
3. В диалоговом окне Свойства ++...++ во вкладке [Соединение](devicetaggui_r_verbindung.md) измените свойства соединений, а во вкладке [Графика соединения](devicetaggui_r_verbindungsgrafik.md) — графические свойства линий соединений.
4. Щелкните по кнопке ++OK++.

!!! note "Замечание:"

    Посредством ***Точки обозначения соединения*** соединениям могут быть присвоены дополнительные свойства. Эти свойства обрабатываются на точке определения соединения.

!!! tip "Совет:"

    Во вкладке Соединение в поле Обозначение соединения можно автоматически определить ближайшее свободное обозначение соединения. Для этого воспользуйтесь пунктом всплывающего меню Определить новое обозначение или комбинацией клавиш [Ctrl] \+ [N].

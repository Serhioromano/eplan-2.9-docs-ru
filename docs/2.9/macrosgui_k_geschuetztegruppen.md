# Макросы: Защищенные группы

Макросы окна и символа при вставке можно генерировать в виде так называемой защищенной группы. Такая группа подобна группе элементов или устройств. Однако защищенную группу невозможно изменить графически или разделить на дальнейшие части. Вставка макросов в виде защищенной группы может быть полезной, например, если вы собираетесь стандартным образом использовать в схеме соединений устройства или предварительно подготовленные фрагменты схемы соединений, но не хотите, чтобы другие пользователи изменяли этот стандарт.

Во время подготовки макросов в ***проекте макросов*** укажите, следует ли генерировать защищенную группу для макроса при его вставке. Для этого на рамке макроса на вкладке Настройки есть флажок При вставке генерировать защищенную группу. Настройка предусмотрена только в проектах макросов, если для рамок макросов установлен метод использования "Определяющий" или "Не задан".

!!! tip "Совет:"

    Если в проекте необходимо удалить это группирование для вставленных макросов, можно использовать сжатие проектов. Для этого в диалоговом окне Настройки: Сжатие на уровне иерархии Убрать данные проекта установите флажок Удалить защищенные группы. При этом группирование удаляется, а вставленные макросы и их компоненты остаются.

!!! note "Замечание:"

    В отличие от групп элементов и устройств группирование в защищенных группах невозможно отменить, отключив параметр Учитывать группирование, или временно выключить нажатием кнопки ++Shift++.

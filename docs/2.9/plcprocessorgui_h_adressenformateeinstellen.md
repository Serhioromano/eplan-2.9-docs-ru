# Настроить форматы адресов

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Параметры > Настройки > Проекты > "Имя проекта" > Устройство > ПЛК
2. В диалоговом окне Настройки: ПЛК щелкните рядом с полем Специфические для ПЛК настройки по кнопке ++...++.
3. В диалоговом окне Настройки: Специфич. для ПЛК выберите вкладку Форматы адресов.
4. Выберите определенную пользователем схему из раскрывающегося списка либо создайте новую.
5. В групповом поле Форматы адресов рядом с полем для требуемого формата адреса щелкните ++...++. Для дискретных и аналоговых вводов и выводов возможно использование разных форматов адресов.
6. В диалоговом окне Формат адреса ПЛК составьте формат адреса, используя элементы формата Счетчик и Разделитель. Выделите в списке Доступные элементы формата элемент Счетчик, и, используя кнопку со стрелкой, скопируйте в список Выбранные элементы формата, расположенный справа.
7. В диалоговом окне Формат: Счетчик задайте формат счетчика. Выберите числовую систему для счетчика, а также начальное и конечное значение для адресации.
8. Установите количество позиций.
9. Если требуется, укажите в поле Значение конфигурации из свойства свойство, значение которого будет задавать начальное значение счетчика (для первого вывода устройства ПЛК).
10. Щелкните по кнопке ++OK++.
11. Выделите в списке Доступные элементы формата элемент Разделитель, и, используя кнопку со стрелкой, скопируйте в список Выбранные элементы формата, расположенный справа.
12. В диалоговом окне Формат: Разделитель укажите, какой знак будет использован в качестве разделителя.
13. Щелкните по кнопке ++OK++.
14. Отформатируете следующий счетчик и следующий разделитель, как описано выше, продолжайте, пока не создадите нужный формат адреса. Каждый счетчик и каждый разделитель можно назначать независимо от уже заданных.
15. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В поле Предварительный просмотр выводятся заданные форматы адресов, где элементы формата изображены с помощью буквенных обозначений.

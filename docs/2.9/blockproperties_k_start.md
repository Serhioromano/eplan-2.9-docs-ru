#  Свойства блока

Свойства блока используются для совместного форматирования и просмотра нескольких прямых и / или косвенных свойств. Свойства блока позволяют:

* Обращаться к косвенным свойствам объекта.
* Вносить свойства блока для любых условных обозначений и просматривать значения свойств в схеме соединений.
* Свойства блока можно выводить в отчеты.
* Свойства блока могут быть предварительно определены для символов и проектов.

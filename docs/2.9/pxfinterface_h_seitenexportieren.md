# Экспорт данных для внешней обработки

Если вы хотите экспортировать данные для внешней обработки, необходимо сначала задать настройки для внешней обработки данных: Параметры > Настройки > Пользователь > Интерфейсы > Внешняя обработка. Затем определите, как и где будут экспортированы выбранные данные. При этом вывод зависит от выбранного ***типа объекта***, который вы указали в схеме.

1. Выберите пункты меню Сервисные программы > Внешняя обработка свойств > Экспортировать данные.
2. В диалоговом окне Внешняя обработка из раскрывающего списка Настройки выберите схему или щелкните ++...++, чтобы задать в диалоговом окне Настройки: Внешняя обработка настройки новой схемы, определенной пользователем.
3. Выберите из раскрывающего списка Язык язык выполнения экспорта.
4. Укажите в поле Целевой файл путь и имя целевого файла или нажмите ++...++, чтобы выбрать путь и каталог в интерактивном режиме.
5. Выберите групповое поле Вид вывода одну из доступных опций для вывода.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выполняется экспорт данных, указатель процесса показывает ход экспорта.

7. В зависимости от специфических для экспорта настроек и настроек вида вывода следует сначала завершить внешнюю обработку, прежде чем продолжить работу в EPLAN.

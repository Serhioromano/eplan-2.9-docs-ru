# Диалоговое окно Содержимое файла резервирования

Вы выполнили резервирование данных (проектов и основных данных) и выделили в диалоговом окне Восстановить проект или <Восстановить> основные данные файл резервирования. [Отображение содержания].

В этом диалоговом окне отображается содержание файла резервирования для выбранного проекта или выбранного файла основных данных.

Доступны следующие элементы диалогового окна:

Вверху списка выводится каталог для резервирования, а также имя текущего файла резервирования.

### Список

В списке перечислены все резервированные отдельные файлы. Если каталог не задан, данные находятся непосредственно в каталоге проектов или основных данных, подкаталоги указываются дополнительно.

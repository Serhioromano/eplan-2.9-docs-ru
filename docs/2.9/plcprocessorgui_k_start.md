# Форматы адресов ПЛК

EPLAN поддерживает вас в процессе распределения адресов ПЛК. Имеются следующие возможности обработки:

* Вы можете установить любой формат для адресов ПЛК.
* Вы можете определять отдельные форматы адресов для цифровых и аналоговых адресов ПЛК.
* Символические адреса могут генерироваться автоматически в соответствии с заданными критериями.
* Вы можете установить любой формат для списков присвоений.

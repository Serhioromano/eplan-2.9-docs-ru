# Обработать группы форматов

Условие:

Вы открыли проект.

1. Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Нумерация соединений
2. В диалоговом окне Настройки: Нумерация соединений выберите вкладку Обозначение.
3. Выделите в поле Группы форматов группу форматов и щелкните на ![](images/all_edit_as.png){: .ui-icon } (Обработать).
4. В диалоговом окне Нумерация соединений: Форматы измените при необходимости в поле Объем количество соединений, которым будет присвоен одинаковый формат обозначений соединений. Выберите запись из раскрывающегося списка.

!!! note "Замечание:"

    Если вы выбрали запись "Сеть", через расширение сети можно увеличить зону воздействия для нумерации.

5. Выделите в поле Выбранные элементы формата запись и щелкните на ![](images/all_edit_as.png){: .ui-icon } (Обработать), чтобы изменить форматирование. Выделите в поле Доступные элементы формата элемент формата и перенесите его с помощью клавиши со стрелкой вправо в список выбранных элементов формата, чтобы определить дополнительный элемент формата.

!!! info "Для сведения:"

    В зависимости от элемента формата открывается следующее диалоговое окно.

6. Задайте форматирование в следующем диалоговом окне (Формат: Свойства, Формат: Счетчик, Формат: Страница / Столбец / Строка или Формат: Разделитель).

!!! info "Для сведения:"

    В поле Предварительный просмотр выводится предварительный просмотр формата обозначения. Отдельные элементы формата выводятся в обозначении соединения в той последовательности, в которой они стоят в списке.

7. При необходимости измените последовательность элементов формата с помощью кнопок со стрелками.
8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Измененные значения сохраняются в выбранной схеме.

### Удалить группы форматов

Условие:

Вы открыли проект.

1. Данные проекта > Соединения > Нумерация > Настройки
2. В диалоговом окне Настройки: Нумерация соединений выберите вкладку Обозначение.
3. Выделите в поле Группы форматов группу форматов и щелкните на ![](images/all_delete_as.png){: .ui-icon } (Удалить).

!!! info "Для сведения:"

    Группа форматов будет удалена.

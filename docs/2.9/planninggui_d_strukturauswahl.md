# Диалоговое окно Выбрать структуру

Вы открыли проект. Открыта страница типа "Функциональная схема автоматизации".

* Технологический контур или функция ТК задаются при помощи выбора символов.
* Фрагмент страницы вставляется как копия из буфера обмена или как макрос. Этот фрагмент содержит условные обозначения, связанные с сегментами из предварительного планирования (технологическими контурами, функциями ТК, резервуарами, объектами планирования, объектами планирования (трубопроводами) и т. д.).

В этом диалоговом окне определяется, в каком месте сегмент присваивается в структуре предварительного планирования и должен ли при этом нумероваться вставленный сегмент.

При вставке макросов или копий в этом диалоговом окне, кроме технологических контуров / функций ТК, также учитываются другие сегменты из предварительного планирования (резервуары, объекты планирования, объекты планирования (трубопроводы) и т. д.). Если в следующем тексте содержится "Технологический контур / функция ТК", это также будет соответствующим образом распространяться для этих сегментов.

Обзор основных элементов диалогового окна:

### Источник

Во вкладке Дерево отображается вышестоящая структура технологических контуров / функций ТК, в которую они встроены в исходном проекте или макросе.

Во вкладке Список в табличной форме отображаются вышестоящие структуры и вставленные технологические контуры / функции ТК. Столбцы этого списка соответствуют конфигурации столбцов в навигаторе предварительного планирования.

Вставка в структуру посредством многократного выбора возможна только при условии, что технологические контуры / функции ТК ***не*** выделяются одновременно с ее вышестоящим узлом.

### Проект

В этом дереве отображается структура целевого проекта, в который вставляются технологические контуры / функции ТК. Отображаются размещенные и неразмещенные сегменты.

### Кнопки со стрелками

Кнопка |  Значение
---|---
![](images/all_arrowright_as.png){: .ui-icon } (Вставить) |  Позволяет вставить технологический контур / функцию ТК, выделенные в поле Источник, ниже сегмента, выделенного в поле Проект. Если в поле Источник выделен вышестоящий сегмент, в целевой проект вставляется вся структура, расположенная ниже выделения. Новые сегменты в поле Проект обозначены пиктограммой ![](images/all_statusnew_as.png){: .ui-icon }.
![](images/all_assign_as.png){: .ui-icon } (Присвоить) |  Сегмент, выделенный в поле Проект, можно присвоить сегменту, выделенному в поле Источник, если их определения базового сегмента совпадают. Таким образом, уже имеющиеся в предварительном планировании технологический контур / функцию ТК можно разместить вместе со вставленной графикой. После этого присвоенный сегмент в поле Проект будет выделен полужирным шрифтом и обозначен пиктограммой статуса ![](images/all_statusok_as.png){: .ui-icon }. Если на таком выделенном полужирным шрифтом сегменте щелкнуть кнопку ![](images/all_arrowleft_as.png){: .ui-icon }, присвоение будет отменено.
![](images/all_arrowleft_as.png){: .ui-icon } (Отменить) |  Позволяет отменить вставку технологического контура / функции ТК. Технологический контур / функция ТК, выделенные в поле Проект, перемещается вместе со своей вышестоящей структурой обратно в поле Источник.

!!! tip "Совет:"

    Вы также можете присвоить структуры с несколькими существующими сегментами друг другу. Для этого выделите в полях Источник и Проект соответствующий узел уже существующего сегмента структуры и нажмите кнопку ![](images/all_assign_as.png){: .ui-icon } (Назначить).

### Всплывающее меню

С помощью всплывающего меню в поле Проект при вставке технологических контуров / функций в функциональную схему автоматизации можно одновременно создавать подобные новые структуры и обрабатывать существующие сегменты.

Пункт меню |  Значение
---|---
Новый сегмент структуры |  Этот пункт меню доступен, только если выделен сегмент структуры. Позволяет создать новый сегмент структуры.
Свойства |  Открывает диалоговое окно Свойства (усл. обозначение): ++...++. Позволяет обрабатывать свойства сегмента.

### Пронумеровать сегменты

Если этот флажок установлен, обозначения устройств и обозначения сегментов нумеруются. При этом настройки для нумерации используются на вкладке Нумерация / доступность диалогового окна [Конфигурировать определения сегмента](planninggui_d_konfigsegmentdef.md).

Если флажок снят, сегменты не нумеруются.

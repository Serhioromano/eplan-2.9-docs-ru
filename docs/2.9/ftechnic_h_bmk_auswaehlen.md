# Выбрать Fluid-ОУ согласно стандарту DIN ISO 1219-2

В случае уже размещенного устройства Fluid программа поддерживает вас при выборе следующего свободного обозначения устройства. Для устройств Fluid, обозначенных согласно DIN ISO 1219-2, при выборе можно задать, следует ли искать следующий свободный номер логической ячейки или номер функционального элемента.

Условия

* Вы открыли проект.
* Проект создан с использованием шаблона `FL_1219-2_tpl001` . Благодаря этому шаблону проекта обозначения устройств Fluid-техники структурируются согласно стандарту DIN ISO 1219-2 по зависимым от раздела ***ключу среды***, ***номеру логической ячейки*** и ***номеру функционального элемента***.
* Создана страница типа "Схема соединений Fluid" и открыта в графическом редакторе.

* На странице схемы соединений размещено как минимум одно устройство Fluid.

### Отобразить структуру идентификаторов для выбора ОУ

1. Выберите следующие пункты меню: Параметры > Настройки > Проекты > "Имя проекта" > Представление > Структура дерева (навигаторы).

2. В диалоговом окне Настройки: Структура проекта (навигаторы) установите флажок ОУ: Предш. цифра, ОУ: Буквенное обозначение, ОУ: Счетчик и ОУ: Подсчетчик.

!!! info "Для сведения:"

    Благодаря этим настройкам отдельные элементы структуры ОУ размещенных функций Fluid-техники в представлении структуры дерева диалогового окна "Выбор ОУ" представлены раздельно и снабжены пиктограммой.

3. Щелкните по кнопке ++OK++.

### Выбрать программно номер переключательной схемы

1. Щелкните двойным щелчком в схеме соединений по устройству Fluid.
2. В диалоговом окне рядом с полем Видимое ОУ щелкните по кнопке ++...++.

!!! info "Для сведения:"

    В открывшемся диалоговом окне Выбор ОУ приводится список всех обозначений устройств всех доступных в проекте функций.

!!! info "Для сведения:"

    ОУ выбранной функции Fluid-техники уже выделено в списке.

3. Чтобы пометить отдельные элементы обозначения ОУ выбранной функции Fluid-техники пиктограммой, выберите вкладку Дерево.
4. Щелкните в открывшемся виде дерева по значению за пиктограммой ![](images/all_countericon_projectstructure.png){: .ui-icon }.
5. Щелкните по кнопке ++Далее++.

!!! info "Для сведения:"

    Выбор ОУ будет закрыт. EPLAN Fluid заносит в поле Видимое ОУ в диалоговом окне Свойства ++...++ следующий свободный номер переключательной схемы.

6. Щелкните по кнопке ++OK++.

### Выбрать программно номер функционального элемента

1. Повторите описанные в предыдущем разделе шаги с 1-го по 3-й.
2. В виде дерева щелкните по требуемому значению за пиктограммой ![](images/navigatorgui_subcounter_as.png){: .ui-icon }.
3. Щелкните по кнопке ++Далее++.

!!! info "Для сведения:"

    Выбор ОУ будет закрыт. EPLAN заносит в поле Видимое ОУ диалогового окна Свойства ++...++ следующий свободный номер функционального элемента.

4. Щелкните по кнопке ++OK++.

* * *

!!! example "Пример:"

    В проекте находятся следующие 3 устройства Fluid:H1.1,H3.7иH4.5В первом случае ОУH3.7присваивается ближайший свободный номер логической ячейки.Новое ОУ будет выглядеть так:H5.7.Во втором случае ОУH3.7присваивается ближайший свободный номер функционального элемента.Новое ОУ будет выглядеть так:H3.8.

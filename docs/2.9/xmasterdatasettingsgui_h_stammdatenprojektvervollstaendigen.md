# Найти и сохранить недостающие основные данные проекта, обновить проекты

Условия:

* Вы открыли проект, основные данные которого должны быть дополнены.
* Вы ***не открыли*** редактор для обработки основных данных (редактор рамок, редактор форм ...).

### Дополнить проект

Путем копирования, напр. страниц, может случиться, что новые ссылки основных данных на формы, рамки и т.д. добавляются в проект. Они необязательно должны быть в целевом проекте, так как автоматическое внедрение основных данных на которые идет ссылка не происходит. Такие проекты можно обновить / дополнить, т. е. используемые, но не сохраненные основные данные копируются в проект из пула основных данных.

1. Выберите пункты меню Сервисные программы > Основные данные > Дополнить текущий проект.

!!! info "Для сведения:"

    Проект ищет недостающие основные данные; в случае, если такие найдены, автоматически сохраняет их.

!!! note "Замечание:"

    Результат действия указывается в системных сообщениях. Таким образом, например, учитывается, когда для отсутствующих основных данных проекта ***нет*** данных в пуле основных данных или если при дополнении происходят ошибки. Соответствующую информацию можно просмотреть в меню Сервисные программы > Системные сообщения.

### Обновить проект

После обработки форм, рамок и / или символов в соответствующем редакторе текущие данные не сохраняются в проекте. Если вы хотите перенести обработанные основные данные в проект, проект можно обновить следующим образом:

1. Выберите пункты меню Сервисные программы > Основные данные > Обновить текущий проект.

!!! info "Для сведения:"

    Проект ищет файлы с основными данными. В случае, если найдены файлы с более актуальными основными данными, чем те, которые представлены в проекте, эти новые файлы автоматически сохраняются, т. е. они перезаписывают старые файлы с основными данными в проекте.

!!! note "Замечание:"

    Результат действия указывается в системных сообщениях. Таким образом, например, регистрируется, что для определенных основных данных проекта ***нет*** данных в пуле основных данных. Соответствующую информацию можно просмотреть в меню Сервисные программы > Системные сообщения.

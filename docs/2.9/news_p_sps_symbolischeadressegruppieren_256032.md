# Группировать символические адреса

Для выводов устройства ПЛК теперь доступно новое свойство Символический адрес: Группа (ид. 20610). Это свойство можно использовать для группировки диапазонов символических адресов в списке присвоений, например, входы, выходы, адреса (отказоуст.) и т. д. Для этого данное свойство можно отобразить в виде столбца в списке присвоений (пункты меню Данные проекта > ПЛК > Адреса / списки присвоения) и ввести в нем произвольно выбираемое обозначение.

Свойство используется при обмене данными ПЛК в формате AutomationML. Это свойство также доступно в качестве элемента формата в диалоговом окне Формат строки в списке присвоений.

![](images/benefit.png)Эффект:

Посредством группирования символических адресов определенные диапазоны списка присвоений, например, адреса (отказоуст.), могут быть сгруппированы при обмене данными ПЛК в формате AutomationML.

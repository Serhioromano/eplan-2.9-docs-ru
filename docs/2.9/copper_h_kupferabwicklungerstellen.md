# Создать развертку шины

Условия:

* Вы открыли проект и находитесь на странице в графическом редакторе.
* Проект содержит пространства листа с расположенным там медными функциональными элементами.
* По ссылке [Параметры > Настройки > Фирма > Механическая обработка > Изгиб шины](cabinetgui_d_einstellungenkupferbiegung.md) вы создали таблицу последовательности изгибов, в которой для каждой толщины материала указаны углы изгиба и радиусы изгиба, встречающиеся на разворачиваемых медных функциональных элементах.

1. Выберите пункты меню Вставить > Графика > Развертка шины.
2. Вводом двух точек поднимите рамку. Рамку можно чертить в любом направлении.

!!! info "Для сведения:"

    Откроется диалоговое окно Развертка.

3. На вкладке Вид укажите специфические данные для развертки.
4. На вкладке Отображение определите настройки отображения для описания развертки, например размер шрифта, выравнивание и ширину / высоту блока выравнивания текста.
5. На вкладке Формат определите под уровнем иерархии Прямоугольник графические свойства рамки развертки, например, начальную точку, конечную точку и ширину.
6. Под уровнем иерархии Формат укажите свойства формата рамки развертки.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Диалоговое окно Развертка закроется, и развертка будет размещена на странице в соответствии с выполненными настройками.

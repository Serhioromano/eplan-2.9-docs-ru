# База данных изделий: Присвоение признаков eCl@ss свойствам изделия EPLAN

Характеристики стандарта eCl@ss могут иметь кардинальность, например число выводов устройства функционального элемента. Таким образом определяется, как часто повторно встречается блок признаков eCl@ss. На момент определения присвоения данных eCl@ss известно только то, что блок может встречаться многократно, но неизвестно, насколько часто он встречается непосредственно при импорте. Результаты могут отличаться для каждого импортированного объекта.

В некоторых случаях необходимо, чтобы характеристики стандарта eCl@ss с кардинальностью можно было присвоить нескольким свойствам изделия EPLAN. Это реализуется следующим образом:

* Для каждого признака eCl@ss, который может многократно встречаться в связи с кардинальностью, пользователь также может выбрать несколько свойств EPLAN.
* При импорте эти характеристики стандарта eCl@ss по очереди присваиваются этим свойствам EPLAN.
* Вы также можете выбрать определенные пользователем свойства и индексированные свойства EPLAN, например атрибуты и произвольные свойства.
* При этом пользователь точно выбирает все используемые индексы, например "Атрибуты с 10 по 20".
* Для присвоения нескольких свойств предусмотрено диалоговое окно со списком для доступных и выбранных свойств, в котором с помощью кнопок можно изменять последовательность выбранных свойств.
* Если импортируемый объект содержит больше характеристик стандарта eCl@ss, чем было определено при присвоении кардинальности, лишние характеристики стандарта eCl@ss не учитываются.

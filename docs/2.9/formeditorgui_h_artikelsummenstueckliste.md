# Работа с групповыми спецификациями изделий

Все групповые спецификации изделий, использованные в проекте, все изделия, приведенные на точке определения изделия, и все размещения изделий без соответствующих изделий. Одинаковые изделия объединяются и суммированно перечисляются. Обратите внимание, что изделия с количеством = 0 также всегда выводятся! Чтобы не выводить такие изделия, необходимо установить соответствующий фильтр.

!!! note "Замечание:"

    При выводе многоязычных свойств обратите внимание на следующее:
    Многоязычные свойства ***без формата*** ***не*** объединяются.
    В том случае, когда многоязычные свойства выводятся в отформатированном виде, они объединяются, если они выведены как ***одноязычные*** (настройка по умолчанию). Если для отформатированных многоязычных свойств выбрана настройка языка Все языки отображения, они не объединяются.

Условия:

* В базе данных изделий на вкладке Цены/прочее в поле Количество/упаковка сохранены действительные значения.
* Открыта форма типа "Групповая спецификация изделий" в редакторе форм (Сервисные программы > Основные данные > Форма > Открыть > [Открыть]).

1. Вставить > Текст заполнитель
2. В диалоговом окне Свойства (текст заполнитель) щелкните кнопку ++...++ в поле Форматированные / рассчитанные свойства вкладки Размещение.
3. В диалоговом окне Формат: Свойство в списке Доступные элементы формата выберите элемент "Вычисление" и переместите его с помощью кнопки ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо) в список Выбранные элементы формата.
4. В диалоговом окне Формат: Вычисление в списке Доступные элементы формата выберите один за другим приведенные в таблице элементы формата и переместите их с помощью кнопки ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо) в список Выбранные элементы формата:

Элемент формата| Значение
---|---
Текст заполнитель| Выберите элемент формата "Текст заполнитель", а в открывшемся диалоговом окне — элемент "Ссылка изделия" со свойством Общее количество (число штук).
/| Выберите элемент формата '/' (делить).
Текст заполнитель| Выберите элемент формата "Текст заполнитель", а в открывшемся диалоговом окне — элемент "Изделие" со свойством Количество/упаковка.
+| Выберите элемент формата '+' (cложение).
Число| Выберите элемент формата 'Число' и введите значение 0,4999. (Ввод данного числа необходим для правильного округления количества упаковки).

!!! example "Пример:"

    В проекте используется 7 ламп. В каждой упаковке содержатся 5 ламп. Вычисление '7/5' дает 1,4; данное значение округляется, что приводит в результате к 1 упаковке. Но это неверно. Если вычисление строится по принципу '7/5+0,4999', получается значение 1,8999, которое округляется до 2, что приводит к правильному результату.

5. Убедитесь, что поле Десятичные разряды настроено на значение 0 и флажок Переменное число десят. разрядов ***не*** установлен.
6. Подтвердите ввод.

!!! info "Для сведения:"

    Поле Свойство диалогового окна Свойства (текст заполнитель) содержит только расчетную формулу в том виде, в котором она отображалась в диалоговых окнах при создании формулы в областях Предварительный просмотр и Символ.

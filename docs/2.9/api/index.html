    What are the capabilities of the EPLAN API?

    The EPLAN API offers you the possibility to add functionality to EPLAN and to integrate it into your company operations. Using the EPLAN API you can program add-ins as well as offline applications. The generation of add-ins allows you to enhance EPLAN by new functions and to directly insert them into the EPLAN user interface. If you use offline applications, you can employ EPLAN functionalities in your own programs. To this end, EPLAN API puts a number of functions at your disposal to allow you to navigate through the EPLAN data model and directly access the objects of an EPLAN project. You can for instance generate, edit, and delete projects, pages, macros, and functions.

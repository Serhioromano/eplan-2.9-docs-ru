# API User Guide

The user guide is a compilation of How-To-Dos, which describe, how you can use the EPLAN API. It shows, how you can manage to write your own API appilcations, like [add-ins](AddIns.md) or [using EPLAN in other applications](EplanInOtherApplications.md).

The user guide contains examples for some typical tasks, you perhaps want to automate in EPLAN using the API. There are examples for [creating or opening EPLAN projects](CreateOpenProject.md), for [creating pages](CreatePages.md) an [placing macros](HE_PlaceMacros.md), etc.

In contrast to the API Reference the user guide does not store detail description of every API class.

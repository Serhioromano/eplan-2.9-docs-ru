# MfRemoveToolbarAction

     Remove a custom toolbar by name
    

   

 

Parameter | Description 

---|--- 

ToolbarName |

   

   

    the name of the toolbar
     

 

Silent |

   

   

    when Silent= Yes, no message is shown, when the toolbar does not exist
     

 

 

Remarks
   

   

    This action needs a mainframe
   

 

**Example**

```
MfRemoveToolbarAction /ToolbarName:"?" /Silent:No
```

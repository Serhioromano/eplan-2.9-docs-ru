# MfExportToolbarAction

     Export one or more user toolbars
    

   

 

Parameter | Description 

---|--- 

File |

   

   

    The xml file including path to export.

     

 

ToolbarName |

   

   

    The name of the first user toolbar to export.

     

 

0 |

   

   

    The name of the next user toolbar to export.

     

 

1 |

   

   

    The name of the next user toolbar to export.

     

 

 

Remarks
   

   

     This action needs a mainframe.
    

   

 

**Example**

```
MfExportToolbarAction /File:"?" /ToolbarName:"?"
```

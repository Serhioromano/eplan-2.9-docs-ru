# XEsUserPropertiesExportAction

     exports user properties to file
     
    

  
Parameter | Description  
---|---  
XMLFile | 
    
    
    the full path of the xml-file
      
  
Project | 
    
    
    full project name
      
  
  
Example
    
    
       XEsUserPropertiesExportAction /XMLFile:c:\my_user.xml  /Project:c:\...\ESS_Sample_Project.elk

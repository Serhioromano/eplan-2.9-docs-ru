# MfImportToolbarAction

     Import a toolbar from a file
     
    

  
Parameter | Description  
---|---  
File | 
    
    
    the xml file inclusive path for import
      
  
Replace | 
    
    
    Existing toolbars with the same name are replaced.
      
  
  
Remarks
    
    
     This action is not done in quiet mode. This action needs a mainframe.
     
    

  
**Example**

```
MfImportToolbarAction /File:"?" /Replace:Yes
```

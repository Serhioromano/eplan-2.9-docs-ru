# UnregisterScript

     Unregister a script.
     
    

  
Parameter | Description  
---|---  
ScriptFile | 
    
    
    Script file name with a complete path.
      
  
  
Remarks
    
    
    Two types of scripts are allowed for use: C#(*.cs) and VB.Net(*.vb)
    

  
**Example**

```
UnregisterScript /ScriptFile:c:\program\EPLAN\Scripts\myScript2.vb
```

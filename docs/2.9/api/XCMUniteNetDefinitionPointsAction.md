# XCMUniteNetDefinitionPointsAction

     Unites net definition points placed on the same net in active project.
     If there are more than one net definition points placed on the same net, these NDP's (i.e. its connections) are merged into
     one single NDP. So after execution of the action, each net has at most one NDP containig all netbased connections of the net.

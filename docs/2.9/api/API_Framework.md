# API Framework

The API Framework provides all necessary functionality to integrate API [add-ins](AddIns.md) into EPLAN, to process [scripts](Scripts.md), and to use base technology of the EPLAN system, like e.g. creating [Actions](Actions.md), writing system messages and managing EPLAN settings. The framework encloses the namespaces `Eplan.EplApi.ApplicationFramework`, `Eplan.EplApi.Base`, `Eplan.EplApi.EServices`, `Eplan.EplApi.Gui`, `Eplan.EplApi.Scripting`, and `Eplan.EplApi.System`.

No electrotechnical functions are made available by the Framework.

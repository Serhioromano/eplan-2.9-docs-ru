# Генерировать легенды электрошкафа

Легенда используется для сбора и просмотра информации, а также для перемещения информации из графики в свободную область. При этом легенда электрошкафа является отчетом, относящимся к функции, т.е. ОУ монтажной платы должно быть однозначным.

Если легенда генерируется на той же странице, на которой находится монтажная плата, то речь идет о ***легенде окна***; легенды окон являются свободно позиционируемыми графическими объектами, имеющими точку вставки и свойства символа, например, обозначение устройства.

Кроме того, легенду можно вывести в виде отдельной формы. В этом случае легенда ***не*** выводится на странице компоновки электрошкафа в графическом редакторе, однако на функциональных элементах все равно имеются номера позиций и обозначения устройств. Этот тип легенды называется ***Легенда страницы***.

### Генерировать легенды страниц

Условие:

Открыт проект, который содержит монтажные платы.

1. Выберите следующие пункты меню: Сервисные программы > Отчеты > Генерировать
2. Если в диалоговом окне Отчеты — <Имя проекта> легенда электрошкафа еще не содержится под уровнем иерархии Страницы, выберите ++Настройки++ > Вывод в страницах.
3. В диалоговом окне Настройки: Вывод в страницах выберите форму для типа отчета Легенда электрошкафа и при необходимости определите другие настройки.
4. Щелкните по кнопке ++OK++.
5. В диалоговом окне Отчеты — <Имя проекта> нажмите кнопку ++Создать++.
6. В диалоговом окне Определить отчет укажите Форма вывода и Исходный проект.
7. Выберите запись Легенда электрошкафа из списка доступных типов отчетов.
8. Установите флаг Индивидуальный выбор, если требуется выбрать монтажные платы, для которых необходимо сгенерировать легенды.
9. Щелкните по кнопке ++OK++.
10. В диалоговом окне Выбрать вручную выберите те монтажные платы, для которых требуется создать легенду.
11. Щелкните по кнопке ++OK++.
12. В диалоговом окне Легенда электрошкафа ++...++ укажите структурные идентификаторы и обозначения страниц, на которых должны быть сгенерированы легенды электрошкафа.
13. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выполняется генерация легенд страниц.

14. В диалоговом окне Отчеты — <Имя проекта> нажмите кнопку ++Закрыть++.

!!! tip "Совет:"

    Перед созданием легенды электрошкафа вы можете указать, что узлы и модули должны быть преобразованы в отдельные изделия. Для этого в диалоговом окне Отчеты — <Имя проекта> под кнопкой [Настройки] выберите пункт меню Изделие. В диалоговом окне Настройки: Изделие установите флажок Развернуть узлы и Разъединить модули.

### Генерировать легенды окон

Легенды окон размещаются непосредственно при генерировании вида отчета Легенда электрошкафа, при этом в диалоговом окне Определить отчет вводятся следующие данные:

Условия:

* Открыт проект, который содержит монтажные платы.
* Вы находитесь в графическом редакторе на странице типа "компоновка электрошкафа".
* Флажок Подавить генерирование легенды электрошкафа (категория "Настройки") во вкладке Монтажная плата ***не*** установлен.

1. Выберите следующие пункты меню: Сервисные программы > Отчеты > Генерировать
2. Если в диалоговом окне Отчеты — <Имя проекта> легенда электрошкафа еще не содержится под уровнем иерархии Встроенные отчеты, выберите ++Настройки++ > Вывод в страницах.
3. В диалоговом окне Настройки: Вывод в страницах выберите форму для типа отчета Легенда электрошкафа и при необходимости определите другие настройки.
4. Щелкните по кнопке ++OK++.
5. Нажмите кнопку ++Создать++.
6. В диалоговом окне Определить отчет выберите из раскрывающегося списка Форма вывода запись Размещение вручную.
7. Выберите запись Легенда электрошкафа из списка доступных типов отчетов.
8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Легенда прикреплена к курсору.

9. Разместите легенду на странице компоновку электрошкафа.

!!! info "Для сведения:"

    Программа проверяет, какие функциональные элементы находятся на странице и после этого создает предварительную спецификацию со всеми данными функциональных изделий, размещенных на монтажной плате.

### Подавить легенды окон

Для каждой монтажной платы по умолчанию создается собственная ***легенда окна***. Чтобы заблокировать эту функцию, выполните следующие действия:

Условия:

* Вы открыли проект.
* Вы находитесь в графическом редакторе на странице типа "компоновка электрошкафа".
* Монтажная плата вставлена или вы дважды щелкнули по монтажной плате (или выбрали пункт Свойства во всплывающем меню; в этом случае можно выделить несколько монтажных плат).

1. Во вкладке Монтажная плата диалогового окна Свойства ++...++ нажмите кнопку ![](images/all_new_as.png){: .ui-icon } (Создать), которая находится в панели инструментов над списком свойств.
2. Выберите в диалоговом окне Выбор свойств свойство Подавить генерирование легенды электрошкафа.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Свойство добавляется в столбец Имя свойства в конце списка.

4. Установите флажок в столбце Значение свойства Подавить генерирование легенды электрошкафа.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Генерирование легенды не выполняется.

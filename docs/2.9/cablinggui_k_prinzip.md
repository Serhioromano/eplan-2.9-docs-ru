# Сети соединенных сегментов (топология): принцип

Как правило, сеть соединенных сегментов можно начертить на странице любого типа. При этом использование типа страницы "Топология" имеет то преимущество, что расположенным там символам автоматически присваивается вид представления "Топология". На страницах типа "Топология" не существует автоматического соединения.

Сеть соединенных сегментов упорядочивается структурой обозначений (для страниц, устройств, и т. д.). Например, можно присвоить каждую сеть соединенных сегментов отдельному месту установки путем соответствующего разделения страниц проекта и использования в случае необходимости обозначений местоположения.

!!! tip "Совет:"

    У вас есть возможность вставить графические файлы (например, импортированный файл DXF / DWG) в качестве фонового изображения. Для этой цели в управлении слоями служит настройка Фон. Элементы в настроенном таким образом слое всегда лежат в самом низу, то есть даже позади линий автоматического соединения. Таким образом, импортированный и помещенный на задний план чертеж в формате DXF / DWG может послужить основой для сети соединенных сегментов в топологии.

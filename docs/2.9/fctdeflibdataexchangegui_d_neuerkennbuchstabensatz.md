# Диалоговое окно Новый набор буквенных обозначений

Вы открыли проект. Сервисные программы > Основные данные > Буквенное обозначение. Выберите пункт всплывающего меню Создать.

В этом диалоговом окне Вы можете установить собственный набор буквенных обозначений.

Обзор основных элементов диалогового окна:

### Обозначение

В этом поле введите имя для нового набора буквенных обозначений. После того, как Вы дали подтверждение, нажав на ++OK++, в диалоговом окне Предложения буквенных обозначений выводится новый столбец. Там Вы можете устанавливать буквенное обозначение для каждого определения функций.

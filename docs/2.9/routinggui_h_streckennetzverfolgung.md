# Выделить сети соединенных сегментов

Сети соединенных сегментов можно временно выделить цветом, чтобы проверить, имеется для маршрутизации соединений в пространстве листа сеть соединенных сегментов без пробелов.

Условие:

Вы открыли проект. Открыто одно пространство листа. Пространство листа содержит одну или несколько сетей соединенных сегментов.

1. Выберите следующие пункты меню: Вид > Соединения > Отслеживание сети соединенных сегментов.
2. Щелкните объект (кабельный канал, сегмент маршрутизации, кривая) с распределенным сегментом маршрутизации сети соединенных сегментов.

!!! info "Для сведения:"

    Выделяются цветом все сегменты маршрутизации сети соединенных сегментов, связанные с сегментом маршрутизации объекта, который вы щелкнули.

3. Щелкните другой сегмент маршрутизации.

!!! info "Для сведения:"

    Выделение первой сети соединенных сегментов снимается, а относящаяся к новому сегменту маршрутизации сеть соединенных сегментов выделяется цветом.

!!! note "Замечание:"

    Фильтры соединений, определенные для сегментов маршрутизации, кабельных каналов и т. д., учитываются при проверке отслеживания сети соединенных сегментов.

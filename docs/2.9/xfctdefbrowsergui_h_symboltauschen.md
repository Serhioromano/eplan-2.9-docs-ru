# Заменить символы

Условие:

Вы разместили функцию (то есть условное обозначение) на странице схемы соединений. Или вы сгенерировали функцию в одном из навигаторов.

1. Выделите функцию.
2. Выберите пункт меню Обработать > Свойства.
3. Выберите в диалоговом окне Свойства ++...++ вкладку Данные символа / функции, и щелкните по полю Номер/Имя на ++...++.
4. Выберите в диалоговом окне "Выбор символа" подходящий символ, и щелкните по ++OK++.
5. Щелкните в диалоговом окне Свойства ++...++ по ++OK++.

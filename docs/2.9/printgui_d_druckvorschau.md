# Диалоговое окно Предварительный просмотр перед печатью

Проект > Печать > [Предв. просм. перед печатью].

В этом диалоговом окне производится предварительный просмотр перед печатью выбранных страниц. Отображение соответствует выводу на принтер, включая положение на странице. Для цветной печати представляются все элементы в цветовой схеме "Цветное на белом"; для черно-белой печати все элементы (за исключением графических файлов) представляются черным на белом фоне.

Доступны следующие элементы диалогового окна:

В строке заголовка диалогового окна отображается имя текущей выведенной страницы.

### Страница печатается

Этот флажок показывает, будет ли выполняться печать страницы. Если в диалоговом окне Печать была выбрана опция Только измененные страницы, то среди страниц, выбранных для печати, могут оказаться такие, которые с момента последнего вывода на печать изменены не были, а потому распечатаны не будут. Для этих страниц данный флажок снят.

### [Назад] / [Далее]

При помощи этих кнопок можно листать страницы, выбранные для печати.

### [Печать]

Сразу запускает процесс печати.

### [Закончить]

Завершается процесс печати и происходит возврат в диалоговое окно Печать.

# Использовать область ввода данных

Область ввода данных предоставляет возможность ввода значений с клавиатуры при использовании различных функций в редакторах (графическом редакторе, 3D-редакторе и т. д.). Форма курсора и требуемая прозрачность настраиваются в [Настройки пользователя](gedviewer_d_einstellkommandozeile.md). Для использования области ввода данных в редакторе, выполните следующее:

Условие:

Вы открыли страницу, форму, рамку или символ.

1. Чтобы активировать область ввода данных, выберите пункты меню Параметры > Область ввода данных.
2. Выберите сначала требуемую операцию (например, Вставить > Графика > Линия).

!!! info "Для сведения:"

    На курсоре мыши отображается небольшое текстовое окно со следующей информацией:

Начальная точка для линии (абсолют.)
Ввод точки: X Y
3. Введите требуемое значение для первой точки линии, например, 250 180. (Это сгенерирует начальную точку линии на координатах X = 250 и Y = 180.)

!!! info "Для сведения:"

    Начальная точка определена. Теперь область ввода данных отображает запрос на задание конечной точки линии:

Конечная точка линии (относит.)
Ввод точки: X Y или угол<длина или угол
4. Задайте конечную точку линии, щелкнув кнопкой мыши или введя значение в область ввода данных, например 45<200, для генерирования линии с углом 45 градусов и длиной 200 мм.
5. Если для конечной точки задан только угол, например, через ввод 145, текстовое окно заново отображается и запрашивает задание абсолютной длины или координат X / Y конечной точки.
6. Завершите операцию, выбрав пункт всплывающего меню Прервать операцию или нажав клавишу ++Esc++.

!!! tip "Совет:"

    * Ввод через область ввода данных можно выполнить в середине диалога и при выключенной опции области ввода данных. Для этого следует вызвать пункт меню Вставить > Графика > Круг, графически разместить пункт меню и ввести значение "1" при выключенной области ввода данных. Область ввода данных откроется, и можно будет дополнить ввод радиуса, например, введя значение "10".
    * Вне зависимости от того, активирована или нет область ввода данных, текстовое окно для области ввода данных можно открыть также клавишей [C]. При таком виде вызова курсор находится сразу в области ввода данных текстового окна. То же самое действует для цифровых клавиш, для клавиш [+] и [-], которые также вызывают область ввода данных.

!!! note "Замечание:"

    Настройка включения и выключения области ввода данных синхронизируется. Т. е. если область ввода данных в настройках активировать, а затем выключить в редакторе, она будет деактивирована в настройках.

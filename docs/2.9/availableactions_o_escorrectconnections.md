# Операция: EsCorrectConnections

Эта операция объединяет графические свойства (цвет, тип линии и т. д.) точек обозначения соединений в одной точке определения потенциала для каждого сигнала, если эти графические свойства одинаковы для всего сигнала.

!!! note "Замечание:"

    Объединение точек обозначения соединения может быть полезным, например, после копирования данных EPLAN 5, во время которого было создано много точек определения соединения с одинаковыми графическими свойствами. Сгенерированные точки определения потенциала автоматически пересчитываются и получают имя сигнала 'коррекция-1', 'коррекция-2' и т. д., а 'пустые' точки обозначения соединения удаляются.

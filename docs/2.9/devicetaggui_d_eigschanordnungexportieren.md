# Диалоговое окно Экспортировать порядок свойств

Вы открыли проект.

* Вы вставили символ в Графическом редакторе или генерировали новую функцию в Навигаторе.
* Вы выделили функцию в графическом редакторе или навигаторе. Обработать > Свойства.

Выберите в диалоговом окне Свойства ++...++ вкладку Отображение. По меньшей мере один определенный пользователем порядок свойств создан и сохранен под каким-либо именем. Нажмите кнопку ![](images/all_export_as.png){: .ui-icon } (Экспортировать) на панели инструментов над раскрывающимся списком.

В данном диалоговом окне экспортируются сохраненные, определенные пользователем порядки свойств определенного варианта символа.

Обзор основных элементов диалогового окна:

### Доступные порядки свойств

В этом поле отображаются сохраненные, определенные пользователем порядки свойств выбранного в диалоговом окне Свойства ++...++ варианта символа. Активируйте флажок для порядков свойств, которые необходимо экспортировать.

### Целевой файл

Укажите в этом поле путь файла и имя файла для файла экспорта. Файл экспорта имеет расширения типа имени файла *.emc.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

!!! note "Замечание:"

    При экспорте определенных пользователем порядков свойств для определенного варианта символа и при экспорте для всего проекта определенных пользователем порядков свойств через меню Проект > Организовать > Экспортировать порядок свойств файлы экспорта по умолчанию сохраняются в одном каталоге ($(EPLAN_DATA)\Xml\<Идентификатор фирмы>) с расширением имени файлов *.emc. Используйте различные имена файлов, чтобы различать сгенерированные файлы.

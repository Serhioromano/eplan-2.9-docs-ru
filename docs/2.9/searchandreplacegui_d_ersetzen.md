# Диалоговое окно Заменить

Вы открыли проект. Поиск > Поиск. Вы ввели искомое понятие и щелкнули по ++OK++. В диалоговом окне Результаты поиска — <Имя проекта> выберите пункт всплывающего меню Заменить.

В этом диалоговом окне укажите, каким значением следует заменить искомое понятие.

Обзор основных элементов диалогового окна:

### Поиск

В этом поле отображается значение первой выделенной записи списка результатов; можно изменить запись вручную.

### Заменить на

Введите в этом поле значение, которым вы хотите заменить значение в поле Поиск. Значение заменяется во всех выделенных записях списка, соответствующих искомому понятию.
Щелкните ++...++, чтобы перейти в диалоговое окно Словарь, а в нем на вкладке Выбрать слова выберите требуемое слово для замены в правильной форме.

При замене текстов интерактивный перевод не осуществляется. Заменяется только выбранное понятие, переводы остаются неизменными.

### Учитывать регистр

Если этот флажок установлен, находятся только те тексты, регистр которых совпадает с регистром поискового признака. Если вы ввели, например, поисковый признак "Двигатель", находятся тексты "Двигатель" и "Автомат защиты двигателя", но не "Двигатель постоянного тока".

### Искать только весь текст

Если этот флажок установлен, используется только полный поисковый признак. Если вы ввели, например, поисковый признак "Двигатель", находится только текст "Двигатель", но не "Автомат защиты двигателя" или "Двигатель постоянного тока".

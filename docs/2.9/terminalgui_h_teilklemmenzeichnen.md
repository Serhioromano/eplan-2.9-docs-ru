# Вставка распределенных клемм

Условия:

* В графическом редакторе открыта страница схемы соединений.
* В настройках проекта (Параметры > Настройки > Проекты > "Имя проекта" > Устройства > Клеммы (распределенные клеммы)) задано определение функции для распределенных клемм.

1. Выберите пункты меню Вставить > Распределенная клемма.
2. Разместите распределенную клемму на странице.

!!! info "Для сведения:"

    Распределенная клемма автоматически получает определение функции, заданное в диалоговом окне Настройки: Клеммы (распределенные клеммы).

3. В диалоговом окне Свойства ++...++ на вкладке Клемма введите данные устройства клеммы. Как минимум должно быть введено ОУ, обозначение и описание вывода устройства можно автоматически задать позже при оптимизации распределенных клемм.

4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    если Вы указали уже существующее ОУ, то клемма будет назначена соответствующему клеммнику. Если вы указали еще не существующее ОУ, то EPLAN создает новый клеммник. Он, однако, еще не имеет своего определения клеммника.

!!! info "Для сведения:"

    Если указано уже существующее обозначение клеммы, то распределенная клемма будет назначена аналогичному клеммнику.

!!! note "Замечание:"

    На вкладке Отображение диалогового окна свойств обозначения выводов устройств клемм по умолчанию приведены в порядке свойств. Это выполняется на вкладке Выводы устройств посредством индексированного свойства Обозначение вывода устройства символа (ID 20028). Чтобы отобразить и распечатать обозначения выводов устройств распределенных клемм в схеме соединений, в управлении слоями для слоя EPLAN430, Размещение свойств.Обозначения выводов клемм следует установить флажок Видимый и Печать.

!!! tip "Совет:"

    С помощью клавиши ++Backspace++ во время вставки можно изменить символ.

### Вставить распределенные клеммы в качестве устройства

Клемму также можно вставить в проект посредством вставки устройства в качестве распределенной клеммы. При этом она автоматически нумеруется.

Настройка проекта Разместить по числу выводов определяет, с какого числа выводов устройства с ***обозначениями выводов устройства*** (1, 2, 3, 4, 6, 8, никогда) клемма разделяется на распределенные клеммы при вставке устройств. Выводы устройств без обозначений выводов устройства при этом исключаются.

Условия:

* В графическом редакторе открыта страница схемы соединений.
* В настройках проекта (Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Клеммы (распределенные клеммы)) с помощью параметра Разместить по числу выводов задано, с какого числа обозначений выводов устройства клеммы при вставке устройств разделяются на распределенные клеммы.
* Вы определили устройства клеммы в базе данных изделий, причем шаблоны функций содержат ***обозначения выводов устройства***.

1. Выберите пункты меню Вставить > Устройство.
2. В диалоговом окне Выбор изделия выберите требуемую клемму.

!!! info "Для сведения:"

    Если выбранная клемма имеет как минимум столько обозначений выводов устройства, сколько определено в настройке проекта Разместить по числу выводов, символ для первой распределенной клеммы находится на курсоре и отображается первое обозначение вывода устройства клеммы.

3. Для изменения направления выводов устройства используйте клавишу ++Tab++.
4. При необходимости 'пролистайте' с помощью клавиши ++N++ имеющиеся функции, пока необходимая функция с обозначением вывода устройства не появится на курсоре. Или с помощью клавиши ++Пробел++ перейдите к следующему устройству клеммы.
5. Разместите распределенную клемму на странице.

!!! info "Для сведения:"

    Первая распределенная клемма автоматически становится главной клеммой.

!!! info "Для сведения:"

    Остальные распределенные клеммы одна за другой вместе с обозначениями выводов устройств и соответствующими описаниями выводов устройств появляются на курсоре. Пустые обозначения выводов устройств пропускаются, и их невозможно вставить.

!!! info "Для сведения:"

    Если в шаблон функции введен символ или макрос, то он размещается.

6. Затем размещаются остальные распределенные клеммы.

!!! tip "Совет:"

    * Используйте символы, у которых используемые для внутренних соединений выводы устройств не обозначены. Таким образом схема соединений остается более наглядной и снижается риск ввести обозначения выводов устройств в неправильные выводы устройств.
    * При вставке распределенных клемм как устройства для каждого обозначения вывода устройства размещается отдельная распределенная клемма. Если необходимо представить несколько выводов устройств клеммы с условным обозначением, то остальные обозначения выводов устройств следует ввести в соответствующую распределенную клемму вручную.

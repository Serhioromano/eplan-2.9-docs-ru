# Обработать свойства форм

Диалоговое окно Свойства формы — <Имя формы> открывается автоматически при создании формы; также впоследствии оно может быть вызвано для ***открытой*** формы.

Условия:

* Вы открыли проект.
* Открыта форма в редакторе форм (Сервисные программы > Основные данные > Форма > Открыть > [Открыть]).

1. Выделите форму, свойства которой необходимо просмотреть или обработать, в диалоговом окне навигатора Страницы — <Имя проекта>.
2. Выберите пункт всплывающего меню Свойства.
3. В диалоговом окне Свойства формы — <Имя формы> при необходимости выберите тип формы из раскрывающегося списка Тип формы.
4. Определите значения для свойств, либо введя или перезаписав запись или же выбрав значение по умолчанию из раскрывающегося списка, или перейдите в специальное диалоговое окно, где можно ввести другие данные.
5. Щелкните по ![](images/all_new_as.png){: .ui-icon } (Создать) в панели инструментов над таблицей Имя свойства / Значение.
6. Выберите в диалоговом окне Выбор свойств новое свойство, которое Вы хотите присвоить форме.
7. Щелкните по кнопке ++OK++.
8. Присвойте свойству значение.
9. Выполните аналогичные действия для всех других свойств.
10. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Форма выводится с измененными / новыми данными в редакторе форм.

!!! note "Замечание:"

    Доступны ***все*** свойства формы, однако их следует выбирать в зависимости от свойства Работа с формой. Так, например, свойства динамических форм (такие как свойство Динамически подобрать высоту строчки) являются нецелесообразными для статических форм.

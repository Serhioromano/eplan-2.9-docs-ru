# Вкладка Настройки (рамка макроса)

В проект макросов вставлена рамка макроса (Вставить > Блок / вывод устройства / монтажная плата > рамка макроса), либо выполнен двойной щелчок по уже имеющейся рамке макроса. В диалоговом окне Свойства ++...++ выберите вкладку Настройки.

На этой вкладке отображаются настройки для рамки макроса. Задайте в ***проекте макросов*** — в зависимости от вида соединения, установленного на вкладке [Рамка макроса](macrosgui_r_makrokasten.md) — различные настройки, которые учитываются при вставке сгенерированного макроса. В ***проекте схемы соединений*** можно задать настройки только для точки захвата

Обзор основных элементов диалогового окна:

### При вставке генерировать защищенную группу

Установите этот флажок, если при вставке макроса необходимо сгенерировать защищенную группу. Такое группирование обеспечивает графическую защиту, т. е. свойства компонентов макроса можно обрабатывать, но сам макрос нельзя изменить графически или разделить на дальнейшие части.

### Вставить также рамку макроса

С помощью этого раскрывающегося списка укажите, следует ли при вставке сгенерированного макроса вставлять также рамку макроса. Эта настройка учитывается и при вставке устройства, если для него сохранен макрос.

Если в этом поле выбрана запись "Из настроек проекта", используется настройка [Вставить также рамки макросов](gedviewer_d_einstellungenprojektallgemein.md) для всего проекта.

### Учитывать обозначения выводов устройства

С помощью этого раскрывающегося списка можно указать, должны ли обозначения выводов устройства быть идентичными при ***присвоении устройств*** наряду с определениями функций. Таким образом присваиваются соответствующие функции из сохраненного макроса, например, при размещении сложных устройств.

Эта настройка также используется при ***обновлении макросов***. Если здесь установлено значение "Да", при обновлении обозначения выводов устройства из файлов макросов переносятся на вставленные макросы в схеме соединений. Если здесь установлено значение "Нет", при обновлении обозначения выводов устройства макросов остаются в схеме соединений.

Если в этом поле выбрана запись "Из настроек пользователя", используется специфическая для пользователя настройка [Учитывать обозначения выводов устройства при присвоении устройств](xessettingsgui_d_betriebsmittelallgemeinbenutzer.md).

!!! note "Замечание:"

    Настройка [Оставить только ОУ](gedviewer_d_einstellungenmakrosaktualisieren.md) для обновления макросов имеет приоритет над настройками, которые учитывают обозначения выводов устройств. Если эта настройка для обновления активирована, все обозначения выводов устройства и другие логические данные, за исключением ОУ, переносятся из файла макроса на функции макроса в схеме соединений, даже если для настройки Учитывать обозначения выводов устройства установлено значение "Нет".

Групповое поле "Точка захвата"

***Точкой захвата*** называется точка, на которой "захватывается" курсор при вставке макроса. Обычно макрос захватывается "вверху, слева" у первой точки вставки.

### Активно

Если этот флажок установлен, показывается стандартная позиция точки захвата, которую можно изменить. Для этого поля Позиция X и Позиция Y деблокируются для обработки, после чего можно перезаписать стандартные значения. После закрытия диалогового окна кнопкой [OK] на положение точки захвата на странице указывает специальный графический символ.

Если этот флажок снят, ввод данных в поля Позиция X и Позиция Y невозможен, а точка захвата не выводится. В случае повторной установки этого флажка точка захвата вновь возвращается на свою стандартную позицию.

### Позиция X / Позиция Y

В случае установленного флажка Активн. здесь указаны стандартные значения для позиции точки захвата. При необходимости эти значения можно обработать.

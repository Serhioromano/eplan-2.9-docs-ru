# Вкладка Объект-заполнитель

В Графическом редакторе открыта страница проекта. Либо вы открыли пространство листа.

* Вставить > Объект-заполнитель. Выделите область с объектами, которые должны принадлежать объекту-заполнителю. В диалоговом окне Свойства ++...++ выберите вкладку Объект-заполнитель.
* Дважды щелкните по символу "Якорь" уже созданного объекта-заполнителя. В диалоговом окне Свойства ++...++ выберите вкладку Объект-заполнитель.
* Данные проекта > Опции > Объекты-заполнители. Выделите в диалоговом окне Объекты-заполнители — <Имя проекта> объект-заполнитель и выберите пункт всплывающего меню Свойства. В диалоговом окне Свойства ++...++ выберите вкладку Объект-заполнитель.

На этой вкладке отображаются выбранные свойства всех объектов, принадлежащих объекту-заполнителю. В дополнение к объектам, выбранным на странице или в пространстве листа (функции, размещения изделий, графические элементы и т. д.), страница или само пространство листа и проект также принадлежат объекту-заполнителю. Свойства этих объектов можно присвоить переменным. Кроме того, есть возможность сохранить наборы значений в объекте-заполнителе.

Обзор основных элементов диалогового окна:

### Имя

Введите здесь имя объекта-заполнителя; в случае обработки объекта-заполнителя (двойной щелчок по графическому символы объекта-заполнителя) в этом поле выводится соответствующее имя.

### Описание

В отличие от ***имени***, которое является идентифицирующим и поэтому не многоязычно, это поле можно использовать для дополнительной информации, которую можно отобразить на иностранном языке. По умолчанию в диалоговом окне Объекты-заполнители - <Имя проекта> отображается имя заполнителя и его описание (в круглых скобках).

### Увеличение фрагмента

Во время предварительного просмотра отображается страница / пространство листа, куда был вставлен объект-заполнитель. Если этот флажок установлен, область с выбранным объектом-заполнителем и соответствующими объектами отображается с увеличением.

Символ "Якорь" объекта-заполнителя на странице при предварительном просмотре выделяется красным цветом. Соответствующие объекты, свойство, переменную или значение которых вы только что выделили на одной из двух вкладок, при предварительном просмотре обозначаются черным цветом.

Вкладка Присвоение

### Таблица

В этой таблице перечислены все объекты объекта-заполнителя, а также выбранные для объектов свойства с текущими значениями. Для нового объекта-заполнителя здесь сначала отображаются только объекты, но не свойства. С помощью кнопки ![](images/all_new_as.png){: .ui-icon } (Создать) для выделенного объекта можно выбрать необходимые свойства и перенести их в таблицу.

Как только Вы будете передвигаться внутри таблицы, предварительный просмотр графики вверху справа обновится. Благодаря этому всегда можно определить, к какому объекту относится выбранное свойство.

Таблица содержит следующие столбцы:

* Свойство: Здесь свойства объекта-заполнителя отображаются в форме структуры дерева. При открытии диалогового окна 'Свойства' дерево в таблице всегда раскрывается до выбранных свойств.
* Текущее значение: Здесь отображается текущее значение соответствующего свойства.
* Переменная: здесь можно ввести переменные для свойств объекта-заполнителя (или также фиксированные значения). Если на вкладке Значения уже имеются переменные, можно с помощью пункта всплывающего меню Выбрать переменную выбрать одну из этих переменных.

!!! tip "Совет:"

    * При наличии графических элементов в объектах-заполнителях также можно получить доступ к некоторым свойствам отображения, которые находятся в диалоговом окне 'Свойства' графических элементов под уровнем иерархии "Формат". Речь идет о свойствах Толщина линии, Цвет, Невидимый и Тип линии. Сконфигурировав набор значений можно, например, сделать прямоугольник невидимым или изменить цвет линии.
    * В объектах-заполнителях путем выбора наборов значений можно менять символы. Для этого на вкладке Присвоение для выбора свойств функций доступно свойство Символ (ид. 20575). Если для этого свойства введена переменная, на вкладке Значения щелчком мыши можно перейти к выбору другого символа для набора значений.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Выбрать переменную |  Только в столбце Переменная: Открывает диалоговое окно Выбрать переменную, в котором можно выбрать одну из заданных переменных, если они имеются.
Перенос переменных |  На вкладке Присвоение в столбце Текущее значение выполняет поиск действующих имен переменных, например <Переменная>, и копирует их в столбец Переменная.
Удалить объект |  Только если выделена по крайней мере одна строка: Удаляет из объекта-заполнителя ***весь*** выделенный объект, включая все его присвоенные свойства, значения и переменные; для удаления нужно подтвердить запрос, нажав кнопку ++Да++.

### Скрыть объекты без выбранных свойств

С помощью этого флажка можно уменьшить количество отображаемых объектов. Если этот флажок установлен, в таблице отображаются только объекты, для которых выбрано свойство.

При открытии диалогового окна 'Свойства' для объектов-заполнителей эта настройка по умолчанию отключена. Установка флажка не сохраняется.

Вкладка Значения

### Таблица

В этой таблице отображаются все переменные и соответствующие наборы значений объекта-заполнителя.

Таблица содержит следующие столбцы:

* Переменная: Здесь отображаются имеющиеся переменные.
* <Набор значений>: Этот столбец доступен только в том случае, если вы создали набор значений на этой вкладке. Созданным таким образом наборам значений по умолчанию присваиваются порядковые номера в качестве имени, которое впоследствии может быть изменено.
Здесь можно ввести требуемые значения для набора значений. Если свойству уже присвоена переменная, в некоторых случаях значения в ячейке можно выбрать с помощью диалогового окна выбора. Это возможно, например, для свойств номеров изделий, определения функции, цветов и структурных идентификаторов.
* Изменяемое: Здесь вы определяете для переменной, можно ли изменять значения при выборе набора значений. Если этот флажок установлен для переменной, в диалоговом окне Выбрать набор значений можно настроить значение переменной.

Переменные или наборы значений (в зависимости от выделенной ячейки) можно добавлять или удалять с помощью кнопок ![](images/all_new_as.png){: .ui-icon } (Создать) и ![](images/all_delete_as.png){: .ui-icon } (Удалить). С помощью кнопок со стрелками (![](images/all_arrowleft_as.png){: .ui-icon } / ![](images/all_arrowright_as.png){: .ui-icon }) можно управлять последовательностью наборов значений.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Новая переменная |  Открывает диалоговое окно Назвать новую переменную, в котором можно сгенерировать новую переменную, при этом EPLaN автоматически проверяет допустимость введенного имени. Новая переменная вставляется под заголовком столбца Переменная.
Удалить неиспользуемые переменные |  Удаляет после соответствующего запроса все переменные, не используемые при присвоении.
Новый набор значений |  Создает новый столбец для создаваемого набора значений, который необходимо ввести в этот столбец.
Удалить набор значений |  Только если выделен набор значений: удаляет после соответствующего запроса текущий выделенный набор значений.
Считать текущие значения |  Только если выделен набор значений: считывает текущие значения (из вкладки Присвоение) в выбранном наборе значений.
Присвоить набор значений |  Только если выделен набор значений: присваивает текущий набор значений. Объекты схемы соединений, свойствам которых во вкладке Присвоение присвоена переменная, содержат соответствующие значения.

# Диалоговое окно Настройки: Предварительная фильтрация изделий

Параметры > Настройки > Пользователь > Управление > Предварительная фильтрация изделий.

В этом диалоговом окне вы можете предварительно отфильтровать базу данных изделий, навигатор основных данных изделий, выбор изделия и выбор устройства на основе определенных свойств.

Обзор основных элементов диалогового окна:

### Фильтр

В этом раскрывающемся списке содержатся все уже определенные схемы фильтра. С помощью кнопки ++...++ перейдите в диалоговое окно [Фильтр](modaldialogsdb_d_filternnach.md), в котором можно создать новую схему и присвоить новые критерии фильтра. При этом в диалоговом окне Выбор критериев представлены такие свойства изделия, как Деталь, снятая с производства, Номер ERP и т. д.

!!! note "Замечание:"

    Если такой фильтр активирован, то в строке заголовка приведенного выше диалогового окна перед именем базы данных изделий появится знак процента "%".

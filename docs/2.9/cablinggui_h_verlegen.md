# Маршрутизировать соединения (топологию)

В результате маршрутизации возникают маршрутизированные соединения между функциями топологии, повторяющие реальное расположение соединений и кабелей в сети соединенных сегментов. Если в функции топологии отсутствуют выводы устройства, они создаются перед маршрутизацией.

Кроме трассы маршрутизации, определяется также и длина соединений. Все полученные сведения могут потом использоваться в качестве инструкции по маршрутизации или для управления сборочными машинами.

Условия:

* Вы открыли проект.
* Вы определили сеть соединенных сегментов.

1. Задайте область, для который хотите сгенерировать маршрутизируемые соединения. Для этого выделите отдельные страницы или проект в навигаторе страниц или область в многополюсной схеме соединений. В качестве альтернативы можно также выделить соединения в навигаторе соединений или функции в навигаторах данных проекта.
2. Выберите пункты меню Данные проекта > Топология > Маршрутизировать.

!!! info "Для сведения:"

    Рассчитывается оптимальная трасса маршрутизации. Отображаются маршрутизируемые соединения.

!!! info "Для сведения:"

    Маршрутизируемые соединения имеют свойства Топология: Трасса маршрутизации, Соединение: Длина, Соединение: Длина до вывода (источник / цель), Размер присоединения (источник / цель) и Обработка концов проводов (источник / цель).

!!! info "Для сведения:"

    Кабели приобретают свойства Топология: Трасса маршрутизации, Длина (в предварительно собранном виде), Кабель: Длина до вывода (источник / цель) и Кабель / Группа соединений: Длина зачистки (источник / цель).

!!! tip "Совет:"

    Если вы хотите узнать, какие кабели или соединения проходят через данный сегмент маршрутизации, выделите его и выберите пункт всплывающего меню Кабели / cоединения. В открывшемся диалоговом окне Кабели / cоединения будут перечислены все соединения, проходящие через данный сегмент маршрутизации. Если через сегмент маршрутизации проходит полный кабель (т.е. все маршрутизируемые соединения кабеля), отдельные соединения не будут перечисляться. Вместо этого отображается ОУ кабеля.

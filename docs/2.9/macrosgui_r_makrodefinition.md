# Вкладка Определение макроса

Вы открыли проект макросов. Вы выделили рамку макроса в графическом редакторе, страницу в навигаторе страниц или пространство листа в навигаторе пространств листа. Обработать > Свойства. Выберите в диалоговом окне Свойства ++...++ вкладку Определение макроса.

Задайте на этой вкладке имена и описание подготовленных макросов в ***проекте макросов***. При автоматическом генерировании макросов из проекта макросов используются заданные здесь настройки для генерируемых файлов макроса.

Если для рамки макросов в проекте макросов установлен метод использования "Ссылающийся" или "Нижестоящий", эта вкладка не отображается. Это также касается ***проекта схемы соединений***.

Если в этой вкладке имя макроса введено в нескольких строках, то рамка макроса, страница или пространство листа будут присвоены нескольким макросам. Затем при автоматическом генерировании в нескольких макросах сохраняется одно и то же содержание (одни и те же рамка макроса, страница или пространство листа).

Обзор основных элементов диалогового окна:

### Таблица

В этой таблице отображаются макросы, которым присваивается соответствующая рамка макроса, страница или пространство листа.

Таблица содержит следующие столбцы:

* Макрос: Имя: Введите здесь имя для макроса, при необходимости указав зону, или выберите его в интерактивном режиме с помощью [...]. Это имя будет использоваться для имени файла генерируемого макроса.
С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.
* Макрос: Описание: Введите здесь информативное описание файла макроса. Текст, введенный в этом поле, становится текстом описания макроса, автоматически созданного из проекта макросов, и выводится в поле комментария просмотр при вставке соответствующего макроса.

!!! note "Замечание:"

    * Значения, введенные в первой строке, служат для определения первого макроса. Как только в первой строке будут введены данные, эти значения синхронизируются со значениями свойств для имени и описания макроса. Таким образом, для рамки макроса в полях Имя и Описание на вкладке Рамка макроса значения синхронизируются с записями в первой строке на вкладке Определение макроса.
    * Указание расширения имени файла в столбце Макрос: Имя требуется только в случае, когда из рамок макросов или пространств листа необходимо сгенерировать макросы символов (*.ems). По умолчанию генерируются двухмерные или трехмерные макросы окна (*.ema), а также макросы страниц (*.emp).

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

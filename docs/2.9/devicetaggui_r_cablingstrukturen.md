# Вкладка Подключенные структуры

Вы открыли проект.

* Вы добавили точку маршрутизации структурного идентификатора в графическом редакторе или сгенерировали ее в навигаторе топологии. Выберите в диалоговом окне Свойства ++...++ вкладку Подключенные структуры.
* Вы выделили точку маршрутизации структурного идентификатора в графическом редакторе или в навигаторе топологии. Обработать > Свойства. Выберите вкладку Подключенные структуры.

В этой вкладке устанавливается фильтр, который будет учитываться при автоматической генерации функций топологии. Фильтр действует как для текущих, так и для подключенных к ним точек маршрутизации структурного идентификатора. Для генерации функций топологии на многополюсной или однополюсной схеме соединений выполняется поиск функций, которые соответствуют структуре обозначений выделенной точки маршрутизации структурного идентификатора или подключенных к ней точек маршрутизации структурного идентификатора, а также отвечает заданным критериям фильтра.

Обзор основных элементов диалогового окна:

### Критерии выбора

В таблице определяются критерии фильтра. Отдельные строки стандартно связаны логическим оператором И.

!!! example "Пример:"

    На странице топологии вставлены триточки маршрутизацииструктурного идентификатора, им присвоеныобозначенияустройств=A1+O1-U1,=A1+O2-U2и=A1+O3-U3. На многополюсной схеме соединений для этого ОУ имеются соответствующие одноименныеобозначения местоположения, которые содержат различныеустройства. Три точки маршрутизации структурного идентификатора соединены между собой сегментами маршрутизации.Теперь, например, необходимомаршрутизироватьсоединениятолько между структурами обозначений=A1+O1-U1и=A1+O2-U2. Для этого откройте диалоговое окно Свойства для точки маршрутизации структурного идентификатора=A1+O1-U1и установите фильтр для автоматически генерируемых функций топологии. В качестве примера выберите критерий Место установкис оператором "=" и значениемO2.Активн.Отрицат.Критерий Оператор Значение Место установки=O2Если затем автоматически сгенерироватьфункциитопологии с помощью пунктов меню Данные проекта > Топология > Генерировать функции топологии > Автоматически, а потом маршрутизировать соединения, появятся толькомаршрутизируемые соединениямежду структурами обозначений=A1+O1-U1и=A1+O2-U2. Отсутствие фильтра также означало бы генерацию маршрутизируемых соединений между структурами обозначений=A1+O1-U1и=A1+O3-U3и между=A1+O2-U2и=A1+O3-U3.

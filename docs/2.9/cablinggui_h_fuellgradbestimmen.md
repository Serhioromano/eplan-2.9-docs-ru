# Определить степень заполнения (топология)

После маршрутизации соединений и кабелей программе становится известно, через какие сегменты маршрутизации проходят эти соединения / кабели. Если поперечное сечение сегментов маршрутизации и внешний диаметр соединений / кабелей известны, программа может оценить, обладают ли сегменты маршрутизации достаточным размером, чтобы вместить в себя все соединения / кабели.

Функция Определить степень заполнения сообщает о наличии свободного места в ручных сегментах маршрутизации. Благодаря ей вы можете узнать, где необходимо изменить маршрутизацию, чтобы провести ее по менее загруженным сегментам, даже если другой путь был бы короче, но привел бы к чрезмерной загруженности.

Условия:

* Вы открыли проект.
* Вы определили сеть соединенных сегментов и поперечные сечения сегментов маршрутизации. Для этого на сегментах маршрутизации, для которых вы хотите рассчитать степень заполнения, необходимо записать поперечное сечение сегмента маршрутизации с указанием единицы измерения в свойство Топология: Поперечное сечение сегмента маршрутизации.
* Соединениям / кабелям, проходящим через сегменты маршрутизации, присвоены изделия с определенным ***внешним диаметром***.
* Вы провели маршрутизацию соединений / кабелей в сети соединенных сегментов.

1. Выберите пункты меню Данные проекта > Топология > Определить степень заполнения.

!!! info "Для сведения:"

    Текущая степень заполнения ручных сегментов маршрутизации будет вычислена и занесена в свойство Топология: Степень заполнения соответствующих сегментов маршрутизации. Степень заполнения рассчитывается для всего проекта.

!!! info "Для сведения:"

    Если для сегмента маршрутизации не указано поперечное сечение, свойство Топология: Степень заполнения удаляется.

!!! tip "Совет:"

    Для отображения степени заполнения на странице схемы соединений добавьте в порядок свойств сегментов маршрутизации свойство Топология: Степень заполнения. Это свойство отображает степень заполнения в процентах.

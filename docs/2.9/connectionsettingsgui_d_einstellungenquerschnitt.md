# Диалоговое окно Поперечные сечения / диаметры / Диалоговое окно Настройки: Поперечные сечения / диаметры

Вы открыли проект.

* Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Поперечные сечения / диаметры.
* Вы открыли диалоговое окно "Свойства" для соединения, точки определения соединения или точки определения потенциала (например, дважды щелкнув по точке определения соединения в графическом редакторе). На вкладке Соединение, Точка определения соединения или Определение потенциала рядом с полем Поперечное сечение/Диаметр нажмите на ++...++.

В этом диалоговом окне определите различные значения по умолчанию для проекта в отношении поперечного сечения/диаметра сечения, включая соответствующую единицу, или выберите предварительно определенное сечение/диаметр.

Обзор основных элементов диалогового окна:

### Поперечное сечение / диаметр:Единица измерения

В этих полях отображаются предварительно определенные поперечные сечения / диаметры, а также относящиеся к ним единицы измерения. В диалоговом окне Настройки: Поперечные сечения / диаметры их можно изменять вручную. В диалоговом окне Поперечные сечения / диаметры можно выбрать только имеющееся поперечное сечение/диаметр, включая соответствующую единицу.

Обзор возможных единиц измерения приводится в соответствующем разделе "[Соединения: Единицы измерения длины и поперечного сечения](connectionbrowsergui_k_einheitenverbindungen.md)".

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

# Указания размеров

Вы можете вводить сведения об указаниях размеров на любых страницах проекта, напр., для компоновки электрошкафа.

В EPLAN есть следующие типы указания размеров:

* Отдельное указание размеров
* Произвольное указание размеров
* Непрерывное указание размеров
* Инкрементальное указание размеров
* Указание размеров исходной линии
* Угловое указание размеров
* Круговое указание размеров.

Указание размеров работает через объектную привязку и не зависит от сетки. Результаты указания размеров можно изменить позже. Префикс и суффикс можно (предварительно) настроить для любой размерной цепи.

Доступны для выбора различные формы представления стрелок и выносных линий, что позволяет настраивать размеры с учетом специфики страны или стандарта.

# Свободная маршрутизация соединений

Функция свободной маршрутизации соединений генерирует электротехнические или гидротехнические маршрутизируемые соединения в форме кривой за пределами сети соединенных сегментов. Такие соединения нужны в случаях, когда использование кабельных каналов невозможно или неприменимо:

* Для маршрутизации проводов больших сечений
* При маршрутизации, например, кабелей Ethernet
* Для определения трасс маршрутизации в корпусах сложной конструкции
* Для гибкого прямого соединения электрических устройств, смонтированных, например, на откидной дверце
* Для соединения гидротехнических устройств со шлангопроводами за пределами сети соединенных сегментов.

Свободно маршрутизированные соединения часто укрепляются или объединяются при помощи принадлежностей для маршрутизации (крепление кабеля / шланга, хомут для кабеля / шланга, защита кабеля / шланга).

Свободно маршрутизируемое соединение генерируется вдоль определенной пространственной кривой между источником и целью. Сеть соединенных сегментов не нужна, ее использование не предусмотрено. После выхода из вывода устройства соединение выравнивается сначала по касательной в направлении вывода устройства на профиле кривой, а затем на цели, снова по касательной, в направлении вывода устройства цели.

Профиль кривой свободно маршрутизируемого соединения можно впоследствии изменить. Для этого можно изменить положение опорных точек или вставить новые опорные точки. После изменения направления кривой соединение маршрутизируется заново.

Условие:

Вы открыли проект. Открыто пространство листа, в котором находится соединение, подлежащее свободной маршрутизации.

!!! note "Замечание:"

    Соединение для свободной маршрутизации можно выбрать до или после вызова функции. Возможности выбора соединений, которые необходимо маршрутизировать свободно, такие же, как и при маршрутизации в сети соединенных сегментов.

    С помощью пункта всплывающего меню Свободная маршрутизация (пространство листа) в навигаторе соединений можно целенаправленно маршрутизировать многополюсные соединения за пределами сети соединенных сегментов.

1. Выберите пункты меню Данные проекта > Соединения > Свободная маршрутизация (пространство листа) или пункт всплывающего меню Свободная маршрутизация (пространство листа) в навигаторе соединений.

!!! info "Для сведения:"

    В строке состояния появится требование выбора соединения для маршрутизации.

2. Укажите две угловые точки прямоугольника, который охватывает маршрутизируемые размещения изделий и соединений.

!!! info "Для сведения:"

    Рассчитывается кривая для профиля маршрутизируемого соединения.

!!! info "Для сведения:"

    Генерируются маршрутизируемые соединения, которым присваиваются указанные свойства.

!!! info "Для сведения:"

    Маршрутизируемые соединения отображаются в 3-хмерном представлении.

!!! example "Пример:"

    Маршрутизируемые соединения, свободно маршрутизированные в пространстве вне сети соединенных сегментов.

3. Чтобы изменить направление свободно маршрутизируемого соединения, выберите в меню пункты Обработать > Графика > Изменить направление кривой и Обработать > Графика > Новая опорная точка.

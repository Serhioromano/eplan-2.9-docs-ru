# Расширенные настройки для нумерации соединений

В настройках для нумерации соединений можно указать, какие соединения следует пронумеровать, как часто должны размещаться обозначения соединений и как они будут форматироваться. Эти настройки сохраняются в схеме. Путь меню к этим настройкам: Параметры > Настройки > Проекты > "Имя проекта" > Управление > Нумерация соединений. При определении формата для обозначения соединений доступны такие расширенные возможности:
– Использовать свойства страницы для нумерации соединений
– Подавить первый счетчик.

### Использовать свойства страницы для нумерации соединений

В диалоговом окне Настройки: Нумерация соединений на вкладке Обозначение можно указать формат для обозначений соединений. Теперь вы можете выбрать свойства страницы исходной и целевой функций для отображения в обозначениях соединений.

![](images/benefit.png)Эффект:

С помощью новых элементов формата "Данные страницы исходной функции" и "Данные страницы целевой функции", которые можно использовать на схемах для нумерации соединений, можно отображать свойства страниц исходной и целевой функций в обозначениях соединений.

Для этого в диалоговом окне [Нумерация соединений: Форматы](wirenumberinggui_d_verbnumformate.md) доступны два новых элемента формата:

* Данные страницы исходной функции: выводит свойства страницы схемы соединений, на которой размещена подключенная исходная функция.
* Данные страницы целевой функции: выводит свойства страницы схемы соединений, на которой размещена подключенная целевая функция.

### Подавить первый счетчик

Если в диалоговом окне Нумерация соединений: Форматы для обработки выбран элемент формата типа "Счетчик" или "Подсчетчик", в диалоговом окне Формат: Счетчик можно задать форматирование для счетчика или подсчетчика.

С помощью флажка Подавить первый подсчетчик ранее можно было подавить только первый подсчетчик на странице, в столбце и т. д. Теперь эта настройка также влияет на счетчик. Соответственно, флажок был переименован:

Старое обозначение |  Новое обозначение
---|---
Подавить первый подсчетчик |  Блокировать первый счетчик / вспомогательный счетчик
Включая следующий разделитель |  Включая предыдущий разделитель

В рамках этого расширения также был переименован флажок Включая следующий разделитель. Установив этот флажок, вместе с первым счетчиком или вспомогательным счетчиком вы можете также подавить предыдущий разделитель.

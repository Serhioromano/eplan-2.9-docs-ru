# Диалоговое окно Настройки: Проверить версию EPLAN

Параметры > Настройки > Фирма > Управление > Проверить версию EPLAN.

В этом диалоговом окне определяется, необходимо ли выполнять проверку наличия обновлений программного обеспечения при запуске EPLAN.

Обзор основных элементов диалогового окна:

### Проверить наличие новой версии EPLAN

Если этот флажок установлен, при запуске EPLAN выполняется проверка наличия последней версии программного обеспечения. В этом случае появится системное сообщение, а в строке состояния отобразится пиктограмма ![](images/modaldialogsgui_update_available.png){: .ui-icon }. Дважды щелкнув по этой пиктограмме, вы откроете EPLAN Download Manager и сможете установить последнюю версию EPLAN.

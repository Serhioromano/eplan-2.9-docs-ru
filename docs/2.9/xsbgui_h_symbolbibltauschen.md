# Поменять библиотеки символов

В настройки, относящиеся к проекту вносятся используемые в проекте библиотеки символов и их можно заменить на другие.

Условия:

* Вы открыли проект, библиотека (библиотеки) символов которого должны быть заменены.
* Используемые в проекте символы и варианты есть в наличии в обеих библиотеках. При этом позиция символов в библиотеке, т.е. номер символа, имеет решающее значение.

1. Параметры > Настройки > Проекты > "имя проекта" > Управление >Библиотеки символов
2. В диалоговом окне Настройки: Библиотеки символов в поле нужной библиотеки символов щелкните ++...++.

!!! info "Для сведения:"

    Открывается диалоговое окно Выбрать библиотеку символов. если библиотеку символов нельзя удалить, отображается соответствующее сообщение.

3. Выберите нужную библиотеку символов.
4. Щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Обе библиотеки символов сравниваются. если обнаруживается разница, которая делает замену невозможной, на просмотр выводится соответствующее сообщение.

5. Если обе библиотеки символов совместимы, в диалоговом окне Поменять библиотеки символов щелкните ++Да++.

!!! info "Для сведения:"

    Имеющаяся в проекте библиотека символов меняется на новую библиотеку символов.

# Удалить определенные пользователем свойства

Условия:

Определенные пользователем свойства для ***проекта***:

* Вы открыли проект.
* Вы выбрали пункты меню Данные проекта > Конфигурировать свойства.

Определенные пользователем свойства в ***базе данных изделий***:

* Вы открыли базу данных изделий. Сервисные программы > Изделие > Управление.
* Вы выбрали пункты меню ++Дополнительно++ > Конфигурировать свойства.

1. Выделите в диалоговом окне Конфигурировать свойства свойство, которое вы хотите удалить.
2. Установите флажок Больше не использовать.
3. Щелкните по кнопке ++Применить++.

!!! info "Для сведения:"

    После подтверждения сообщения свойство больше не будет отображаться в проекте / базе данных изделий.

4. Выберите пункты меню ++Дополнительно++ > Убрать больше не используемые свойства.

!!! info "Для сведения:"

    После подтверждения сообщения определенные пользователем свойства, обозначенные как больше не используемые, будут удалены.

!!! info "Для сведения:"

    Если определенное пользователем свойство удаляется в базе данных изделий и оно также существует в проекте в результате присвоения изделия, для этого свойства во время синхронизации изделий автоматически активируется настройка Больше не использовать.

!!! info "Для сведения:"

    В процессе удаления также будут удалены используемые свойства. Однако эта операция невозможна для определенных пользователем свойств в проекте, которые все еще используются в основных данных. Это записывается в системных сообщениях.

!!! note "Замечание:"

    * В проекте определенные пользователем свойства, которые используются в изделиях или других основных данных, невозможно удалить, пока в столбце Количество использований не отобразится значение "0", даже если параметр "Больше не использовать" включен. Для этого сначала необходимо отменить использование (убрать выбранные изделия, формы и т. д.) и удалить неиспользуемые изделия, формы и т. д. путем сжатия проекта. Только после этого соответствующие свойства можно удалить с помощью пункта меню Убрать больше не используемые свойства или повторного сжатия.
    * При вставке и копировании макросов в проект также вставляются определенные пользователем свойства, которые сохранены в макросе. С помощью специфической для проекта настройки [Не использовать новые конфигурации свойств из макросов](gedviewer_d_einstellungenprojektallgemein.md) можно задать, следует ли использовать эти новые определенные пользователем свойства. Новые определенные пользователем свойства можно также вставить с помощью других основных данных, например форм, рамок и т. д. Эти свойства также учитываются при настройке.

!!! tip "Совет:"

    * Перед удалением определенных пользователем свойств сначала можно проверить, используются ли эти свойства. Для этого под кнопкой [Дополнительно] выберите пункт меню Проверить использование. В столбце Количество использований в представлении в виде списка отобразится текущее количество использований. Не использующиеся свойства получают запись "0".
    С помощью пункта меню Установить для неиспользуемых свойств значение 'Больше не использовать' для неиспользуемых определенных пользователем свойств сначала активируется настройка Больше не использовать, затем эти свойства удаляются через пункт меню Убрать больше не используемые свойства.
    * Чтобы убрать определенные пользователем свойства, больше не используемые в проекте, можно также воспользоваться функцией сжатия проектов. Для этого в диалоговом окне Настройки: Сжатие на уровне иерархии Убрать данные проекта установите флажок Более не используемые пользовательские свойства. Проект будет сжат, при этом будут удалены свойства с установленным флажком "больше не использовать".

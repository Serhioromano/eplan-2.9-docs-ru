# Диалоговое окно Копировать проект

* Вы открыли проект и выделили его в Навигаторе страниц. Проект > Копировать.
* Вы выбрали базу данных управления проектами. Вы считали каталоги и проекты. Проект > Управление. Выделите проект. Выберите пункт всплывающего меню Копировать.

В этом диалоговом окне можно определить, в каком объеме необходимо копировать проект. После копирования скопированный проект автоматически открывается в навигаторе страниц.

!!! note "Замечание:"

    При копировании выгруженного проекта копия проекта не выгружается.

Обзор основных элементов диалогового окна:

### Все с отчетом

При выборе этого параметра копируется весь проект со всеми отчетами.

### Все без отчета

При выборе этого параметра проект копируется без отчетов.

### Только данные заголовка

При выборе этого параметра проект копируется без страниц и без отчетов, но с информацией по слоям.

### Только интерактивные страницы

При выборе этого параметра копируются только страницы схемы соединений. Сюда относятся, например, страницы схемы соединений типа страниц "Внешн. документ", "Графика", "Функциональная схема автоматизации", "Многопол. схема соедин.", "Однополюсная схема соединений", "Схема соединений Fluid", "Компоновка электрошкафа", "Обзор". Отчеты и неразмещенные функции не копируются.

### Исходный проект

В этом поле находится путь и имя проекта, который вы хотите копировать.

### Целевой проект

В этом поле находится путь к целевому каталогу и имя для копии проекта. [...] Открывает диалоговое окно Копировать в. Здесь вы можете выбрать целевой каталог и имя копии проекта.

### Указать дату создания

Если установлен флажок, для копии проекта можно задать дату создания в расположенном ниже поле.

### Указать автора

Эта функция дает возможность задать (другого) автора для копии проекта. Если флажок установлен, в расположенном ниже поле в зависимости от настроек в пунктах Пользователь > Отображение > Ид. пользователя / адрес автоматически вводится ваше имя регистрации или идентификатор пользователя. Если Вы не хотите использовать имя при регистрации или код пользователя, можно ввести другое имя. Если флажок не установлен, отображается имя автора, создавшего оригинальный проект. Если при этом в поле отсутствует запись, для оригинального проекта автор не задается.

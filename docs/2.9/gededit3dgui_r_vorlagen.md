# Вкладка Шаблоны (диалоговое окно Обзоры модели, Развертки шины, Отображения отверстий)

Вы открыли проект.

* Для того чтобы отобразить в проекте имеющиеся обзоры моделей, выберите пункты меню Сервисные программы > Отчеты > Обзоры модели.
* Для того, чтобы отобразить в проекте имеющиеся развертки шины, выберите пункты меню Сервисные программы > Отчеты > Развертки шины.
* Для того чтобы отобразить в проекте имеющиеся двухмерные отображения отверстий, выберите пункты меню Сервисные программы > Отчеты > 2D-отображения отверстий.

Выберите вкладку Шаблоны.

На этой вкладке можно генерировать и обрабатывать шаблоны для автоматической генерации обзоров модели, разверток шины или 2D-отображений отверстий. Шаблоны содержат только данные о форматировании генерируемого обзора модели и о том, для каких основных функциональных элементов должны генерироваться отдельные обзоры. Геометрические данные отсутствуют. Автоматическая генерация страниц происходит на основе шаблонов в диалоговом окне Обзоры модели – <Имя проекта> на вкладке Шаблоны либо после нажатия кнопки ![](images/all_generate_as.png){: .ui-icon } (Генерировать отчет), либо в дереве с помощью пункта всплывающего меню Генерировать отчет.

Обзор основных элементов диалогового окна:

### Имя проекта

В этой структуре дерева отображаются содержащиеся в проекте шаблоны отчетов для обзоров модели, разверток шины или 2D-отображений отверстий.

### Панель инструментов

В панели инструментов, расположенной над структурой дерева, имеются следующие кнопки:

Кнопка |  Значение
---|---
![](images/all_generate_as.png){: .ui-icon } (Генерировать отчет) |  Генерирует обзоры модели, двухмерные отображения отверстий и развертки шины на основе принятых во вкладке Шаблоны настроек.
![](images/all_new_as.png){: .ui-icon } (Создать) |  Создает новый шаблон, имя которого нумеруется по нарастающей, начиная с 1. Это имя впоследствии можно изменить в таблице свойств.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет выделенные шаблоны из списка.
![](images/all_import_as.png){: .ui-icon } (Импортировать) |  Открывает диалоговое окно Импортировать шаблон отчета. Позволяет импортировать шаблон отчета.
![](images/all_export_as.png){: .ui-icon } (Экспортировать) |  Открывает диалоговое окно Экспортировать шаблон отчета. Позволяет экспортировать шаблон отчета.

### Всплывающее меню

Всплывающее меню дерева на вкладке Шаблоны содержит те же пункты меню, что и кнопки на панели инструментов.

Таблица Имя свойства / Значение:

В столбцах этой таблицы отображаются свойства и значения выделенных отчетов, которые можно задать для генерации шаблонов.

Свойство |  Значение
---|---
Имя |  Имя шаблона
Начальная страница блока отчетов |  Начальная страница для всех страниц, автоматически генерируемых с помощью этого шаблона. С помощью кнопки ++...++ можно открыть диалоговое окно [Начальная страница](formgeneratorgui_d_struktur.md).
Настройка фильтра |  Фильтрует основные функциональные элементы, для которых должна автоматически генерироваться страница с обзором модели. С помощью кнопки ++...++ можно открыть диалоговое окно [Фильтр](modaldialogsdb_d_filternnach.md). Если выбранный основной функциональный элемент не содержится в проекте, то для него не будет генерироваться обзор модели. Если ни один основной функциональный элемент не установлен, для каждого пространства листа проекта генерируется обзор модели.
Настройка сортировки |  Здесь задается вид сортировки основных функциональных элементов при генерации обзоров модели. С помощью кнопки ++...++ можно открыть диалоговое окно [Сортировка](formgeneratorgui_d_sortierung.md).
Функции: Настройка фильтра |  Только для обзоров модели! Этим задается, какие функции основного функционального элемента или какие функциональные элементы с иерархической структурой будут представлены в обзоре модели. С помощью кнопки ++...++ можно открыть диалоговое окно [Фильтр](modaldialogsdb_d_filternnach.md).
Раздел |  Заданный раздел присваивается сгенерированной странице.
Ручное описание обзора |  Если флажок Автоматическое описание вида снят, то здесь можно ввести описание обзора.
Автоматическое описание вида |  Если флажок установлен, описание обзора автоматически генерируется из свойства Формат для автоматического описания.
Формат для автоматического описания |  Этим можно задать формат для автоматического описания обзоров и страниц. С помощью кнопки ++...++ можно открыть диалоговое окно [Формат: Свойство](eservicesgui_d_formateigenschaft.md).
Описание страниц вручную |  Если флажок Автоматическое описание страницы снят, то здесь можно ввести описание страниц.
Автоматическое описание страницы |  Если этот флажок установлен, описание страниц автоматически генерируется из свойства Формат для автоматического описания.
Сортировка страниц |  С помощью кнопки ++...++ можно открыть диалоговое окно [Сортировка страниц](formgeneratorgui_d_seitensortierung.md). Значение настройки Сортировка страниц полностью соответствует одноименной специфической для проекта настройке Отчеты > Вывод в страницах.
Макрос |  Заданный здесь макрос используется для вывода обзора в страницах. Кнопка ++...++ открывает диалоговое окно Выбрать макрос. На выбор предлагаются макросы окон, символов и страниц.

!!! note "Замечание:"

    После вставки макросы не сохраняются в проект. В тех случаях, когда проект копируется с шаблонами отчетов или передается другим пользователям, необходимо убедиться, что заданный в шаблоне макрос находится в списке макросов соответствующего пользователя.

# Вкладка Сборная шина

Вы открыли проект.

* Проект содержит как минимум одно пространство листа. Пространство листа содержит сборные шины (изогнутые). Открыт навигатор пространства листа. Выделите сборную шину в навигаторе. Выберите пункт всплывающего меню Свойства. В диалоговом окне Свойства (условное обозначение): Размещение изделия (3D) выберите вкладку Сборная шина.
* Вы выделили сборную шину в трехмерной области отображения при нажатой клавише ++Shift++. Обработать > Свойства. В диалоговом окне Свойства ++...++ выберите вкладку Сборная шина.

На этой вкладке указываются свойства выделенной сборной шины.

Обзор основных элементов диалогового окна:

### Видимое ОУ

Видимое ОУ будет показано в графическом редакторе рядом с условным обозначением. Если вы определили в настройках проекта, что устройство должно автоматически нумероваться при вставке символов, здесь вам будет автоматически предложено ОУ. Вы можете сохранить или переписать вручную это ОУ.

Кроме того, можно использовать [всплывающее меню или кнопку ++...++](devicetaggui_r_feldsichtbaresbmk.md), чтобы cгенерировать или выбрать ОУ.

### Полное ОУ

В этом поле отображается полное ОУ (т. е. структурный идентификатор текущего окружения + видимое ОУ) видимого ОУ / функционального элемента. С помощью кнопки ++...++ откройте диалоговое окно Полное ОУ. Здесь можно обработать структурный идентификатор и, например, выбрать другое место установки.

### Функциональный элемент

В этом поле отображается определение функционального элемента, оно считывается из выбранного определения функции и используется для отображения в навигаторе пространства листа.

### Определение функции

Указанное здесь определение функции для сборной шины присваивается автоматически в зависимости от определения функции соответствующего функционального элемента. Щелкните по ++...++, чтобы открыть диалоговое окно Определения функций и, если нужно, выберите другое определение функции.

### Медный жгут

В данном поле определяется принадлежность медного функц. элемента медному жгуту. Эту принадлежность можно изменить. При нажатии клавиши ++...++ отображается диалоговое окно Выбор 3D-объекта. Здесь перечисляются все входящие в пространство листа медные жгуты. При перемещении выделения на другой медный жгут медный функциональный элемент присваивается этому жгуту. Нажатием клавиши ++OK++ выбор переносится в поле свойств.

# Распознавание многоязычных полей

Для многих свойств и полей ввода в проекте или в базе данных изделий возможен многоязычный ввод. Такие многоязычные тексты могут быть переведены автоматически или вручную. Поля, для которых возможен многоязычный ввод, теперь по умолчанию обозначены в интерфейсе пользователя стоящей впереди пиктограммой.

![](images/benefit.png)Эффект:

По новым пиктограммам сразу видно, в каких полях возможен многоязычный ввод. Это также позволяет распознать не зависящие от языка или уже переведенные тексты.

Пиктограмма |  Значение
---|---
![](images/navigatorgui_globegrey_as.png){: .ui-icon } |  Содержимое поля не зависит от языка (то есть не связано с каким-либо языком) или является пустым.
![](images/navigatorgui_globecolor_as.png){: .ui-icon } |  Содержимое поля зависит от языка. В этом случае активен пункт всплывающего меню Убрать перевод.

Если ввести текст в многоязычное поле, он изначально является независимым от языка. Это означает, что текст не назначен языку и будет отображаться так, как был введен, независимо от выбранного исходного языка. При переводе текст присваивается текущему исходному языку и становится зависящим от языка.

### Возможность настройки для отображения пиктограмм

В настройках пользователя вы можете индивидуально настроить отображение пиктограмм для многоязычных полей. Для этого в диалоговом окне Настройки: Интерфейс пользователя появилось новое групповое поле Отметить многоязычные поля ввода с такими параметрами (путь меню: Параметры > Настройки > Пользователь > Отображение > Интерфейс пользователя):

* Всегда отображать пиктограммы: все многоязычные поля ввода будут обозначены. Обозначение выполняется независимо от того, возможно ли внести запись в поле или данное поле используется только для отображения.
* Отображать пиктограммы только для обрабатываемых полей ввода: обозначаются только те многоязычные поля ввода, в которых возможен ввод.
* Никогда не отображать пиктограммы: многоязычные поля ввода не обозначаются.

По умолчанию обозначаются только обрабатываемые поля ввода. Если для поля даже при активированном параметре Всегда отображать пиктограммы пиктограмма не отображается, это не многоязычное поле.

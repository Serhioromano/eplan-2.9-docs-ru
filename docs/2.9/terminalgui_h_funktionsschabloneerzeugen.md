# Определить строение клеммников

!!! note "Замечание:"

    Данный режим работы поддерживается программой только по причинам совместимости. Управление изделиями рекомендуется осуществлять на главных клеммах.

    Строение клеммника при помощи определения устройства можно задать только в том случае, если клеммник не содержит главных клемм или для него активировано свойство Основные клеммы не используются.

Существует множество клеммников, отличающихся комбинацией типов, количества клемм и принадлежностями. Через [определение устройства](devicelistgui_h_geraetedefinitionenanlegen.md) (т. е. несколько шаблонов функций) можно задать любое ***строение клеммника***, которое позднее не может быть изменено функциями обработки (например, удалением клемм). Шаблоны функций сохраняются в ***Определение клеммника***. То есть, строение клеммника может быть удалено только удалением отдельных шаблонов функций или определения клеммника.

Определение устройства для клеммника можно создать в диалоговом окне Обработать клеммник, для чего определению клеммника необходимо присвоить шаблоны функций. Шаблоны функций вы можете создавать из размещенных или не размещенных функций. Вы можете при этом сразу задавать обозначения клемм, но это не обязательно.

Отдельная клемма идентифицируется в шаблоне функции при помощи обозначения клеммы, определения функции и уровня. если обозначение клеммы отсутствует в шаблоне функции, то он подходит к каждому любому обозначению клеммы в проекте. если на схеме соединений удаляется любая клемма, то она остается в клеммнике и может быть снова использована или размещена.

Определение устройства (т. е. строение клеммника) может быть передано с помощью копирования определения клеммника или макроса.

Условия:

* Вы выделили клеммник или клемму в навигаторе клеммников, навигаторе устройств или в графическом редакторе.
* Клеммник содержит только второстепенные клеммы, либо для определения клеммника активировано свойство Основные клеммы не используются.

### Генерировать шаблоны функций

1. Выберите пункты меню Данные проекта > Клеммники > Обработать.
2. В диалоговом окне Обработать клеммник выделите одну или несколько клемм ***без*** присвоенных шаблонов функций. Найти их можно по пиктограмме в столбце Статус.
3. Выберите пункт всплывающего меню Генерировать шаблоны функций.

!!! info "Для сведения:"

    Пиктограмма для соответствующих клемм изменится в столбце Статус.

4. Подтвердите ввод.

### Удалить шаблоны функций

1. Выберите пункты меню Данные проекта > Клеммники > Обработать.
2. В диалоговом окне Обработать клеммник выделите одну или несколько клемм ***с*** присвоенными шаблонами функций. Найти их можно по пиктограмме в столбце Статус.
3. Выберите пункт всплывающего меню Удалить шаблоны функций.

!!! info "Для сведения:"

    Пиктограмма для соответствующих клемм изменится в столбце Статус.

4. Подтвердите ввод.

# Отобразить, добавить или удалить последовательные принадлежности

В диалоговом окне Обработать клеммник клеммнику можно присвоить последовательную принадлежность, например несущие шины, разделительные пластины, концевые стопоры и т. п. Принадлежности сохраняются как запись изделия на главных клеммах или — если вы не используете главные клеммы — в определении клеммника. Для отображения этих принадлежностей в диалоговом окне необходимо включить параметр Видимые последовательные принадлежности.

Условия:

* В базе данных изделий доступны соответствующие [Принадлежности](terminalgui_k_klemmenzubehoer.md).
* Вы выделили клемму или определение клеммника в навигаторе клеммников, навигаторе устройств или в графическом редакторе. Через пункты меню Данные проекта > Клеммники > Обработать вы открыли диалоговое окно Обработать клеммник.

1. Выберите пункт всплывающего меню Видимые последовательные принадлежности.

!!! info "Для сведения:"

    Последовательные принадлежности будут отображены в диалоговом окне в отдельной строке. В столбце Предварительный просмотр будет отображаться схематический предварительный просмотр принадлежностей.

!!! info "Для сведения:"

    Если принадлежности уже присвоены какой-либо клемме, то клемма и принадлежности отображаются как один блок. Номера строк группируются в одном поле в столбце "Строка".

1. В диалоговом окне Обработать клеммник щелкните по полю с номером строки требуемой клеммы.
2. Выберите пункт всплывающего меню Добавить последовательные принадлежности.
3. Перемещаясь по структуре дерева в диалоговом окне Выбор изделия, выберите необходимые последовательные принадлежности. Возможен многократный выбор.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В диалоговом окне Обработать клеммник последовательная принадлежность будет вставлена под текущей позицией курсора в отдельную строку со штриховкой. Каждую строку можно перемещать по своему усмотрению. Небольшая стрелка в столбце "Строка" указывает направление, в котором принадлежности сортируются перед клеммой или после нее.

5. Выберите пункт всплывающего меню Последовательные принадлежности с очередностью, чтобы вставить последовательные принадлежности перед клеммой или после нее в порядке сортировки.
6. Щелкните по кнопке ++Применить++.

!!! info "Для сведения:"

    Последовательные принадлежности будут присвоены клемме, а изделия заносятся на клемму. Клемма и принадлежности объединяются в один блок, который можно переместить единым целым. Номера строк группируются в одном поле в столбце "Строка".

!!! info "Для сведения:"

    Если для клеммника активировано свойство Основные клеммы не используются, то изделия заносятся в определение клеммника. Строки для последовательных принадлежностей и указатели в виде стрелки для сортировки остаются.

!!! info "Для сведения:"

    При смещении последовательных принадлежностей последовательность изделий на клеммнике записывается в идентификаторе сортировки. Значение заносится в свойство Сортировка изделий на клеммнике, которое видно или на соответствующей главной клемме, или, если вы не используете главные клеммы, в определении клеммника.

!!! tip "Совет:"

    Операцию Добавить последовательные принадлежности можно выполнять и для нескольких выделенных клемм. Последовательные принадлежности, выбранные в следующем диалоговом окне, вносятся на выделенные главные клеммы или на многоуровневые клеммы.

1. В диалоговом окне Обработать клеммник щелкните по соответствующей строке с полем номера строки, чтобы выделить блок, состоящий из клеммы и последовательных принадлежностей.
2. Щелкните по кнопке ![](images/terminalgui_separatemulti_as.png){: .ui-icon } (Разделить).

!!! info "Для сведения:"

    Последовательные принадлежности будут показаны в отдельной строке.

3. В диалоговом окне Обработать клеммник щелкните по соответствующей строке с полем номера строки, чтобы выделить последовательную принадлежность.
4. Выберите пункт всплывающего меню Удалить последовательные принадлежности.

!!! info "Для сведения:"

    Выбранная принадлежность удаляется, т. е. запись изделия убирается из главной клеммы или из определения клеммника.

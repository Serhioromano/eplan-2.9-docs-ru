# Вставить и вывести QR-код

QR-код состоит из черных и белых квадратов, расположенных в квадратной сетке. Такой код, например, используется для кодированного отображения веб-адресов, телефонных номеров, учетных данных для доступа к WLAN и т. д. С помощью мобильного устройства делается снимок кода, который интерпретируется посредством приложения, благодаря чему отпадает необходимость в утомительном ручном вводе этих данных. В графическом редакторе теперь можно использовать новый пункт меню Вставить > Графика > QR-код, чтобы сгенерировать QR-код из текста, веб-адреса или свойств и отобразить его на странице проекта.

![](images/benefit.png)Эффект:

С помощью QR-кода в документации можно сохранять гиперссылки, которые в распечатанном виде считываются мобильными устройствами и открываются на них.

После того как вы двумя щелчками мыши определите положение и размер квадрата для QR-кода, откроется новое диалоговое окно Свойства (QR-код).

На вкладке QR-код в поле Формат блока введите текст или веб-адрес, который необходимо сгенерировать в виде QR-кода и отобразить на странице проекта. Если для QR-кода нужно использовать свойство, с помощью кнопки ++…++ перейдите в следующее диалоговое окно Формат, чтобы выбрать свойство и, при необходимости, дополнительные элементы формата, например разделитель.

### Обработать QR-код

С помощью вкладки Формат настраивается только начальная точка или размер квадрата QR-кода. Другие настройки формата невозможны.

QR-код не обрабатывается при помощи команд Обработать > Поворачивать или Обработать > Отобразить зеркально, поскольку ориентация QR-кода не должна изменяться во время обработки.

### Функция перехода для QR-кода

Если QR-код сгенерирован из веб-адреса или адреса электронной почты, то по выбранному QR-коду с помощью пункта всплывающего меню Открыть гиперссылку можно перейти на соответствующую интернет-страницу или запустить программу электронной почты на компьютере.

Нажатием кнопки ++Ctrl++ можно также активировать функцию перехода. При этом курсор отобразится в виде руки.

### QR-код на страницах отчета

Для вывода QR-кодов на страницах отчетов можно использовать новый флажок [Отобразить как QR-код](formeditorgui_r_platzierung.md) для текстов-заполнителей в диалоговом окне Свойства (текст-заполнитель) на вкладке Размещение. Если этот флажок установлен, при генерации отчета на основе определенных значений для выбранного выше свойства генерируются QR-коды, которые отображаются на выведенных страницах отчета.

Это позволяет, например, сохранять гиперссылки на страницы производителя в отчете "Список обозначений устройств" для изделий, используемых в проекте.

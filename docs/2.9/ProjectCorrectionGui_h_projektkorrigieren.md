# Корректировать проект

Условие:

Вы открыли проект и выделили его в Навигаторе страниц.

### Определить объем корректировки

1. Выберите следующие пункты меню: Проект > Организовать > Корректировать
2. В диалоговом окне Корректировать проект рядом с полем Настройки щелкните по ++...++.
3. В диалоговом окне Настройки: Корректировать щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
4. В диалоговом окне Новая схема в поле Имя введите информативное имя, а в поле Описание – краткий текст описания схемы, после чего нажмите кнопку ++OK++.
5. Чтобы выбрать данные проекта для [корректировочных настроек](ProjectCorrectionGui_k_korrekturdaten.md), в диалоговом окне Настройки: Корректировать в поле Корректируемые данные установите флажок уровня.

!!! info "Для сведения:"

    Если уровень содержит подчиненные уровни, то они автоматически выбираются вместе с вышестоящим уровнем.

6. Щелкните по кнопке ![](images/all_save_as.png){: .ui-icon } (Сохранить).
7. Щелкните по кнопке ++OK++.

### Выбрать схему корректировки и фильтр

Если объем корректировки вы уже сохранили как схему, необходимо только выбрать имеющуюся схему. Чтобы исключить из корректировки уровни структуры проекта, необходимо определить фильтр.

1. В диалоговом окне Корректировать проект в поле Настройки выберите схему для объема корректировки.
2. Для исключения из корректировки данных проекта в поле Фильтр выберите фильтр.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Результат корректировки отображается в диалоговом окне Системные сообщения.

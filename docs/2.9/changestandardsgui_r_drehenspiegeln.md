# Вкладка Поворот / инверсия (Изменение стандарта)

Вы открыли проект. При нескольких открытых проектах: В навигаторе страниц выделен проект. Сервисные программы > Изменение стандарта. Вы определили на вкладке Проект все настройки. [Далее].

Обзор основных элементов диалогового окна:

### Шаблон

В этом поле расположен шаблон для использования в целевом проекте. В меню ++...++ выберите шаблон проекта *.ept.

Если на вкладке Проект установлен флажок Скопировать свойства проекта из исходного проекта, свойства проекта будут скопированы ***не*** из этого шаблона, а из исходного проекта.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Поворот / инверсия

В этом раскрывающемся списке выберите способ вращения и инверсии.

* 0°: элементы схемы соединений не поворачиваются.
* 90°: поворачивает графические элементы (линии, тексты, гиперссылки) на 90° против часовой стрелки. Смещает точку вставки символов на 90° против часовой стрелки. Поворот / зеркальное отображение символов задаются через варианты, выбираемые в схеме присвоения на вкладке Символы. Для специальных символов выполняется автоматическая предварительная настройка подходящей библиотеки символов на вкладке Основные данные. Флажок Присвоить отдельно устанавливается автоматически. На вкладке Символы схема Специальные-IEC-NFPA-IEC автоматически становится доступной. Используйте данную настройку, чтобы перейти от стандарта IEC к стандарту NFPA.
* 270°: поворачивает графические элементы (линии, тексты, гиперссылки) на 270° против часовой стрелки или на 90° по часовой стрелке. Смещает точку вставки символов на 270° против часовой стрелки или на 90° по часовой стрелке. Поворот / зеркальное отображение символов задаются через варианты, выбираемые в схеме присвоения на вкладке Символы. Для специальных символов выполняется автоматическая предварительная настройка подходящей библиотеки символов на вкладке Основные данные. Флажок Присвоить отдельно устанавливается автоматически. На вкладке Символы схема Специальные-IEC-NFPA-IEC автоматически становится доступной. Используйте данную настройку, чтобы перейти от стандарта NFPA к стандарту IEC.

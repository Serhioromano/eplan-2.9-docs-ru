# Диалоговое окно Настройки: Языки вывода PDF

* Параметры > Настройки > Проекты > "Имя проекта" > Управление > Языки вывода PDF.
* Страница > Экспортировать > PDF > [Настройки] > Языки вывода.

В этом диалоговом окне установите, на каких языках выводится PDF.

Обзор основных элементов диалогового окна:

### Языки отображения проекта

При этой настройке установленные для проекта ***языки отображения*** выдаются в PDF-файле.

### Альтернативные языки

При этой настройке выводятся альтернативные, не показываемые языки. Определите нужные языки в лежащем ниже поле Языки.

### Языки

Данный список доступен лишь тогда, когда выбран параметр Альтернативные языки. В этом поле установите, на каких альтернативных языках будет показываться переведенный текст в PDF-файле. При этом вы можете выбрать язык из списка установленных для проекта ***Языков перевода***. При экспорте в PDF будут использоваться только эти языки, установленные языки отображения будут игнорироваться.

!!! note "Замечание:"

    Если в списке Языки при установленном параметре Альтернативные языки нет записей, файл при экспорте в PDF генерируется на языках отображения проекта. "Пустой" список языков может возникать в случае, если язык перевод удален из проекта и этот язык предварительно определен как язык вывода PDF.

### Панель инструментов

Панель инструментов над списком Языки доступна лишь тогда, когда вы выбрали выше параметр Альтернативные языки; здесь доступны следующие кнопки:

Кнопка |  Значение
---|---
![](images/all_new_as.png){: .ui-icon } (Создать) |  Генерирует новую строку в конце списка Языки и открывает раскрывающийся список для выбора одного из возможных альтернативных языков.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удалить выбранные языки из списка.
![](images/all_arrowup_as.png){: .ui-icon } (Переместить вверх) |  Перемещает выделенный язык на одну строку вверх. С помощью данного действия вы определяете языки вывода.
![](images/all_arrowdown_as.png){: .ui-icon } (Переместить вниз) |  Перемещает выделенный язык на одну строку вниз. С помощью данного действия вы определяете языки вывода.

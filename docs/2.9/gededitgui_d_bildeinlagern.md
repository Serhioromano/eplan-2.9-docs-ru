# Диалоговое окно Сохранить графический файл

Вы открыли символ. Вставить > Графика > Графический файл. В диалоговом окне Выбрать графический файл выберите требуемый графический файл и нажмите кнопку ++Открыть++.

В EPLAN можно вставить графические файлы форматов GIF, JPG, BMP, ICO, PNG или TIFF. В этом диалоговом окне указывается, следует сохранить графический файл в библиотеку символов или создать ссылку на исходный каталог.

Доступны следующие элементы диалогового окна:

### Исходный каталог не изменять

В символ вставляется ссылка на графический файл. В случае изменений этот графический файл будет автоматически обновлен при открытии данного символа. Однако файлы, на которые указывают ссылки, не учитываются при резервировании данных.

### Сохранить

Графический файл сохраняется в символе в качестве копии и, таким образом, является составной частью библиотеки символов. Вставленный таким образом графический файл сохраняется после сравнения основных данных, передачи библиотек символов или импорта/экспорта символов. Однако при изменении графического файла он автоматически не обновляется.

# Импорт / экспорт настроек

В диалоговом окне Настройки с помощью команды всплывающего меню в представлении в виде дерева можно импортировать, экспортировать и расширенно экспортировать настройки. Последняя опция позволяет дополнить уже созданный файл экспорта другой областью настройки. Экспортированные настройки сохраняются в файл XML, который можно снова импортировать.

!!! note "Замечание:"

    Обратите внимание на то, что возможность задать дополнительные области настройки для фирменных и специфических для проекта настроек ***недоступна***!

Условие:

Имеется доступ к файлу импорта *.xml.

1. Выберите следующие пункты меню: Параметры > Настройки
2. Выделите в дереве диалогового окна Настройки уровень структуры, в который требуется импортировать настройки.
3. Щелкните кнопку ![](images/all_import_as.png){: .ui-icon } (Импортировать).

!!! info "Для сведения:"

    Если выбрана область настроек "Пользователь" или "Рабочая станция", откроется диалоговое окно Импортировать настройки.

!!! info "Для сведения:"

    Если выбрана одна из областей настроек "Проекты" или "Фирма", откроется диалоговое окно Импортировать настройки: <Путь иерархии>. Здесь можно указать только исходный файл.

4. Для импорта глобальных настроек установите верхний флажок.
5. Для импорта других настроек программы и конфигураций установите в групповом поле Другие области настройки соответствующий флажок требуемой области настройки.
6. Щелкните рядом с полем Исходный файл по кнопке ++...++ и выберите файл импорта *.xml.

!!! info "Для сведения:"

    Настройки импортируются.

1. Выберите следующие пункты меню: Параметры > Настройки
2. Выделите в дереве диалогового окна Настройки область настройки, которую требуется экспортировать.
3. Щелкните ![](images/all_export_as.png){: .ui-icon } (Экспортировать).

!!! info "Для сведения:"

    Если выбрана область настроек "Пользователь" или "Рабочая станция", откроется диалоговое окно Экспортировать настройки.

!!! info "Для сведения:"

    Если выбрана одна из областей настроек "Проекты" или "Фирма", откроется диалоговое окно Экспортировать настройки: <Путь иерархии>. Здесь можно указать целевой файл.

4. Для экспорта глобальных настроек выделенной области следует установить верхний флажок.
5. Для экспорта других настроек программы и конфигураций установите в групповом поле Другие области настройки соответствующий флажок требуемой области настройки.
6. Проверьте в поле Целевой файл путь и имя файла или, нажав кнопку ++...++, укажите другие путь и имя файла.

!!! info "Для сведения:"

    В поле Целевой файл уже указаны путь и имя файла, которые соответствуют области, выделенной в дереве.

!!! example "Пример:"

    В диалоговом окне Настройкивыделена область Пользователь > Отображение > Общее. Программа предлагает имя<Пользователь>+<Отображение>+<Общее>.xml.

7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Настройки будут сохранены в выбранном каталоге в виде XML-файла с указанным именем.

!!! note "Замечание:"

    Если выделить в дереве только основную категорию или подкатегорию, будут экспортированы все нижестоящие области настройки.

Условия:

* Вы открыли проект.
* Имеется доступ к файлу экспорта, в который требуется экспортировать дополнительные настройки.

1. Выберите следующие пункты меню: Параметры > Настройки
2. Выделите в дереве диалогового окна Настройки требуемую основную категорию, подкатегорию или требуемую область настройки, которую следует добавить в существующий файл экспорта.
3. Выберите пункт всплывающего меню Экспортировать (расширить).

!!! info "Для сведения:"

    Если выбрана одна из областей настроек "Пользователь" или "Рабочая станция", откроется диалоговое окно Экспортировать настройки (расшир.).

!!! info "Для сведения:"

    Если выбрана одна из областей настроек "Проекты" или "Фирма", откроется диалоговое окно Экспортировать настройки (расшир.): <Путь иерархии>. Здесь можно указать целевой файл.

4. Для добавления в файл экспорта глобальных настроек выделенной области следует установить верхний флажок.
5. Для добавления в файл экспорта других настроек программы и конфигураций установите в групповом поле Другие области настройки соответствующий флажок требуемой области настройки.
6. Выберите в поле Целевой файл имеющийся файл экспорта.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Настройки будут добавлены в выбранный файл экспорта.

!!! note "Замечание:"

    При расширенном экспорте настроек автоматическое расширение имени файла не производится. Поэтому рекомендуется либо переименовать расширенный файл экспорта, либо сохранить его в отдельном каталоге.

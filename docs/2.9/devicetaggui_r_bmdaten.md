# Вкладки <Категория функции > / <Категория функции> (Устройство)

Вы открыли проект.

* Вы вставили символ в Графическом редакторе или генерировали новую функцию в Навигаторе. Выберите в диалоговом окне Свойства ++...++ вкладку <Категория функции>.
* Вы выделили функцию какого-либо общего устройства в графическом редакторе или навигаторе. Обработать > Свойства. Выберите вкладку <Категория функции>.
* Вы выделили ***Вспомогательную функцию*** какого-либо общего устройства в графическом редакторе или навигаторе. Выберите пункт всплывающего меню Свойства (общие).

На этой вкладке задаются свойства выделенной ***Функции***. На вкладке нанесено обозначение соответствующей категории функции, напр., "Двигатель", "Замыкающий контакт", "Плавкий предохранитель" и т. д.

В режиме обработки ***Свойства (общие)*** вы изменяете свойства для всех представлений функции вместе; в противном случае можно изменять свойства распределенных функций отдельно друг от друга.

Вы открыли проект.

* Вы выделили ***Главную функцию*** какого-либо общего устройства в графическом редакторе или навигаторе. Выберите пункт всплывающего меню Свойства (общие).
* Вы выделили ***Вспомогательную функцию*** какого-либо общего устройства в графическом редакторе или навигаторе. Выберите пункт всплывающего меню Свойства (общие). Выберите вкладку <Категория функции> (Устройство).

В этой вкладке вы изменяете главную функцию и все ее представления в режиме обработки ***Свойства (общие)***, это значит, что вы задаете свойства ***Устройства*** выделенной функции. На вкладке нанесено обозначение соответствующей категории функции, напр., "Двигатель", и дополнение "(Устройство)" для устройства. При этом необязательно доступны все свойства, которые можно изменить в отдельном представлении функции.

Если диалоговое окно "Свойства" открыто для вспомогательной функции, появится вкладка <Категория функции> для выделенной вспомогательной функции, а также дополнительная вкладка <Категория функции> (Устройство) для соответствующей главной функции.

Обзор основных элементов диалогового окна:

### Видимое ОУ

Видимое ОУ будет показано в графическом редакторе рядом с условным обозначением. Если вы определили в настройках проекта, что устройство должно автоматически нумероваться при вставке символов, здесь вам будет автоматически предложено ОУ. Вы можете сохранить или переписать вручную это ОУ.

Кроме того, можно использовать [всплывающее меню или кнопку ++...++](devicetaggui_r_feldsichtbaresbmk.md), чтобы cгенерировать или выбрать ОУ.

### Полное ОУ

В этом поле отображается полное ОУ (т. е. структурный идентификатор текущего окружения + видимое ОУ) видимого ОУ / функционального элемента. С помощью кнопки ++...++ откройте диалоговое окно Полное ОУ. Здесь можно обработать структурный идентификатор и, например, выбрать другое место установки.

### Обозначение вывода устройства

В этом поле отображаются названия обозначений выводов устройств актуальных функций.
Если активирована настройка проекта Копировать обозначение вывода устройства (Параметры > Настройки > Проекты > "Имя проекта" > Управление > Совместимость), на экране дополнительно отображаются названия обозначений выводов устройств функций, копирующих ОУ актуальной функции.

Обозначения выводов устройств для функции или для устройства можно изменить вручную или выбрать запись из раскрывающегося списка. Несколько обозначений выводов устройств разделяются знаком абзаца. При обработке нескольких обозначений выводов устройств вручную необходимо учитывать следующее:

* Отдельные обозначения выводов устройств можно изменить вводом значения на соответствующей позиции между знаками абзаца.
* При перезаписывании всей записи одним отдельным значением это значение будет присвоено первому выводу устройства. Остальные обозначения выводов устройств останутся без изменения.
* При удалении всей записи удаляются все обозначения выводов устройств.

!!! note "Замечание:"

    Содержимое данного поля не сохраняется в отдельном свойстве. Чтобы отобразить обозначения выводов устройств на условном обозначении, можно воспользоваться свойством Обозначения выводов устройства (все) (ID 20038). Это свойство отображает все обозначения выводов устройств текущей функции. Однако это свойство не содержит отображаемые в поле Обозначение вывода устройства обозначения выводов устройств других функций.

### Описание вывода устройства

Введите здесь описание выводов устройств для текущей функции или устройства или выберите эти данные из раскрывающегося списка.

### Технические параметры

Введите здесь технические параметры для функции / устройства.

### Функциональный текст

Укажите здесь функциональный текст для функции / устройства.

Независимо от введенного здесь вручную функционального текста существуют другие (определяемые автоматически) функциональные тексты, которые находятся в таблице:

* Функциональный текст (автоматически): Отображает содержание введенного вручную функционального текста или, при его отсутствии, функциональный текст зоны. Если вы изменяете функциональный текст в режиме обработки ***Свойства (общие)***, его содержание переносится обратно в ту функцию, в которой функциональный текст был первоначально задан. При отображении этого свойства на странице проекта разрывы строк удаляются.
* Функциональный текст (общий): отображает (автоматический) функциональный текст функции или, при его отсутствии, (автоматический) функциональный текст соответствующей главной функции.

### Текст гравировки

Введите здесь текст гравировки для функции / устройства или выберите запись из раскрывающегося списка. В списке отображаются все значения, которые вы вводили в других местах проекта для подобных функций.

### Место монтажа (описыв.)

Введите здесь описываемое место монтажа для функции / устройства или выберите запись из раскрывающегося списка. В списке отображаются все значения, которые вы вводили в других местах проекта для подобных функций.

### Главная функция

Так как общие функции могут связываться перекрестными ссылками, они могут представлять собой как главные, так и вспомогательные функции. Установите / снимите этот флажок, чтобы изменить настройку.

Если флажок установлен, то функция является главной и может содержать сведения об изделии.

Если флажок не установлен, то функция является вспомогательной. В этом случае вкладка Изделие будет скрыта, так как вспомогательные функции ***не*** могут содержать сведений об изделиях.

При использовании имеющегося ОУ, для которого уже задана главная функция, при использовании уже размещенной главной функции, а также при размещении уже размещенной главной функции данное свойство будет автоматически отключено на целевой функции, то есть данный флажок будет снят.

!!! note "Замечание:"

    эта настройка автоматически синхронизируется с одноименным флажком на вкладке Данные символа / функции.

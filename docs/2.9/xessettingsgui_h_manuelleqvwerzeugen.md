# Генерировать ручные перекрестные ссылки

Ручные перекр. ссылки - это простые многострочные тексты, которые отображаются вместе с "нормальной" перекрестной ссылкой на условном обозначении. Система EPLAN позиционирует ручную перекрестную ссылку ***до*** автоматически сгенерированной перекрестной ссылки.

Условие:

Вы открыли проект.

1. Выберите страницу схемы соединений для обработки.
2. Дважды щелкните по условному обозначению, которое Вы бы хотели отобразить ручной перекрестной ссылкой.
3. Щелкните на вкладке <Категория функции> диалогового окна Свойства ++...++ в групповом поле Свойства по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
4. Выберите в диалоговом окне "Выбор свойств" Свойство Перекрестная ссылка вручную.
5. Щелкните по кнопке ++OK++.
6. Задайте в поле Значение Свойства Перекрестная ссылка вручную желаемый текст.
7. Выберите во всплывающем меню > Новая строка, чтобы инициировать новую строку, или выберите во всплывающем меню > Многострочный ввод, чтобы задать в диалоговом окне Многострочный ввод многострочный текст. Новая строка инициируется при помощи ++Ctrl++ +[Ввод].
8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Текст копируется в поле Значение.

9. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Ручная перекрестная ссылка записывается как символ.

# Диалоговое окно Настройки: 3D (Пользователь, Графическая обработка)

Параметры > Настройки > Пользователь > Графическая обработка > 3D.

В данном диалоговом окне задаются личные настройки пользователя для графической обработки в трехмерном виде.

Обзор основных элементов диалогового окна:

Групповое поле "Цветовые настройки"

Все цветовые настройки задаются в диалоговом окне [Выбор цвета](mf_d_farbauswahl.md), которое открывается с помощью кнопки ++...++. Выберите требуемый цвет.

### Фон оттенения 1/Фон оттенения 2

Здесь задаются цвета фона для оттенения в 3-мерном виде.

### Отображение конфликтов

Здесь задается цвет для представления конфликтов между компонентами, возникающих при размещении и изменении в 3-мерном виде.

### Активная монтажная поверхность

Здесь задаются цвета фона для активной монтажной поверхности в 3-мерном виде.

### Активный медный жгут

Здесь задаются цвета фона для активного медного жгута в 3-мерном виде.

* * *

Групповое поле Стандартные размеры сетки

В полях Размер сетки A-Размер сетки E в качестве стандартных значений можно указать пять разных размеров сетки. В пространстве листа эти размеры сетки можно выбрать с помощью соответствующих кнопок ![](images/gededitgui_grida.png) до ![](images/gededitgui_gride.png) в панели инструментов "Вид".

* * *

Групповое поле 'Применить упрощенное представление к'

Здесь указывается, какие функциональные элементы в пространстве листа при применении функции Упрощенное представление отображаются с уменьшенной детализацией графики:

* Клеммники (определение блока): Если этот флажок установлен (стандартная настройка), то клеммники представляются в виде упрощенного блока.
* 3D-макросы: Если этот флажок установлен (стандартная настройка), то графические макросы, шины и кабельные каналы упрощаются до прямоугольного параллелепипеда.

В пространстве листа активируйте заданное здесь представление через пункты меню Вид > Упрощенное представление или в навигаторе пространства листа через Всплывающее меню > Упрощенное представление.

* * *

### Поворачивать при изменении точек наблюдения

При переключении трехмерной точки наблюдения вид поворачивается и принимает новое положение. Если данный флажок не установлен, вид переключается без движения.

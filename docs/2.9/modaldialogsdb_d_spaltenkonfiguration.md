# Диалоговое окно Конфигурировать представление (список)

Вы открыли проект и находитесь в представлении навигатора в виде списка, в представлении диалогового окна Выбор символа в виде списка и т. д. Выберите пункт всплывающего меню Конфигурировать представление.

В этом диалоговом окне можно определять отображение и последовательность соответствующих данных таким образом, чтобы выводилась только релевантная информация.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Конфигурация

В этом поле укажите, какие свойства следует отображать. С помощью кнопки ![](images/all_new_as.png){: .ui-icon } (Создать) можно выбирать другие свойства и копировать их в список. С помощью кнопок со стрелками можно изменять последовательность свойств.

### Всплывающее меню

Пункт меню |  Значение
---|---
Выделить все |  Выделяет все записи в списке.

!!! note "Замечание:"

    Определенные свойства ***всегда*** отображаются. Такие свойства ***невозможно*** удалить из списка в поле Конфигурация. Таким образом, например, свойство Имя проекта отображается в представлении в виде списка всегда в виде столбца, если открыто более одного проекта или, наоборот, не открыт ни один проект.

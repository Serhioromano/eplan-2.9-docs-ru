# Диалоговое окно Настройки: Отображение

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Перекр. ссылки / образы контактов > Отображение.

В этом диалоговом окне можно установить настройку для разных типов страниц, для каких типов страниц должны отображаться перекрестные ссылки и в какой последовательности.

Обзор основных элементов диалогового окна:

### Отображать перекрестные ссылки устройств

Здесь для разных типов страниц выберите те типы страниц, для которых возможны перекрестные ссылки на главной функции, щелкнув в отдельных столбцах по ![](images/all_arrowdropdown_as.png){: .ui-icon } и применив запись из раскрывающегося списка.

!!! example "Пример:"

    При помощинастройки"Компоновка электрошкафа" Вы можетегенерироватьперекрестные ссылки между главными функциями на схеме соединений и размещенными изделиями на***Монтажные платы***. При помощи настройки "Список обозначений устройств" генерируется перекрестная ссылка между главными функциями на схеме соединений и соответствующими устройствами в***Список обозначений устройств***.

### Показывать только первую перекрестную ссылку каждого вида представления

Установите этот флажок, если для отдельных видов представления перекрестные ссылки должны отображаться не в виде списка и нужно показать только перекрестную ссылку первого введенного изделия. Если флажок не установлен, отображаются списки.

### Отображать перекрестные ссылки обзора

В этом групповом поле можно ввести следующие данные:

* От обзора на схему соединений: установите этот флажок, чтобы активировать перекрестные ссылки с обзорных страниц на страницы схем соединений.

* От схемы соединений на обзор: установите этот флажок, чтобы активировать перекрестные ссылки со страниц схем соединений на обзорные страницы.

* У точек подвода питания: установите этот флажок, чтобы активировать перекрестные ссылки также на точках подвода питания.

Используйте эти настройки, чтобы, например, отображать перекрестные ссылки ПЛК между выводами устройства ПЛК на схеме соединений и в обзоре.

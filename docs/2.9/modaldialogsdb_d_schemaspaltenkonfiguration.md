# Диалоговое окно Настройки: Конфигурация столбцов

Сервисные программы > Изделие > Управление. В дереве выделен объект, относящийся к уровню иерархии Схема соединений. Выберите вкладку Выводы устройства и нажмите кнопку ++...++ рядом с полем Схема.

В этом диалоговом окне установите, какие столбцы будут отображаться для данных выводов устройств.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

!!! note "Замечание:"

    Схемы запоминаются отдельно для каждого пользователя. Например, если над одним проектом работают совместно электротехник и Fluid-техник, каждый из них может использовать индивидуальные настройки для конфигурации столбцов. Чтобы передать схему вместе с проектом, схему нужно экспортировать и передать отдельно.

### Конфигурация столбцов

В этом поле вы определяете, какие столбцы будут отображаться в таблице во вкладке Выводы устройства. Отметьте соответствующую опцию. С помощью кнопки со стрелкой вы можете изменять порядок столбцов.

Флажок Обозначение вывода устройства нельзя деактивировать, так как здесь идентифицируется обозначение вывода устройства.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

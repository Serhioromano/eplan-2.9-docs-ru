# Диалоговое окно Настройки: Функциональные признаки приборов

Вы открыли проект.Параметры > Настройки > Проекты > "Имя проекта" > Предварительное планирование > Функциональные признаки приборов.

В этом диалоговом окне вы можете предварительно определить буквенное обозначение для функций обработки технологических контуров.

Обзор основных элементов диалогового окна:

### Буквенное обозначение / описание

В этих столбцах отображаются все предварительно определенные буквенные обозначения и соответствующие описательные тексты функций обработки технологических контуров. Для предварительного определения нового буквенного обозначения щелкните по ![](images/all_new_as.png){: .ui-icon } (Создать) и введите буквенное обозначение во вновь созданную строку. Все определенные здесь буквенные обозначения предлагаются на выбор при создании / обработке технологических контуров / функций ТК для функционального признака прибора с помощью диалогового окна Выбрать функциональный признак прибора.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

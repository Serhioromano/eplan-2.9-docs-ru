# Диалоговое окно Синтакс. проверка ОУ

Вы открыли проект. В настройках для проверки синтаксиса ОУ проверка активирована. Вы ввели в диалоговом окне Свойства ++...++ недопустимый символ в качестве элемента обозначения и нажали [OK].

После ввода свойств устройства EPLAN проверяет правильность синтаксиса введенного обозначения устройства. Если синтаксис неправилен, то появится это диалоговое окно.

!!! note "Замечание:"

    * Допустимые символы Вы можете задать в настройках синтаксиса ОУ.
    * Синтаксическая проверка осуществляется также при вводе обозначений в рамках Управление структ. идентификаторами.

Доступны следующие элементы диалогового окна:

### Сообщение

В этом неизменяемом поле отображаются обозначения и выделяются неверные элементы.

### Описание

Здесь отображается описательный текст к возникшей ошибке.

### [Настройки]

Открывает диалоговое окно Настройки: Синтаксическая проверка ОУ. Здесь Вы можете проверить Ваши настройки синтаксиса и, при необходимости, откорректировать их.

### [<<] / [>>]

При нажатии на одну из этих кнопок будет выделен следующий (или предыдущий) неверный элемент обозначения. При наличии только одного неверного символа кнопки неактивны.

### [Игнорировать]

Игнорирует все ошибки, закрывает окно сообщения и вставляет неподходящее обозначение.

### [Отменить]

Закрытие диалогового окна. Вы вернетесь к диалоговому окну "Свойства" и можете изменить в нем введенные Вами данные.

# Присвоить изделия устройству

Изделия можно присвоить только ***главной функции*** устройства. При этом EPLAN обращается к основным данным, находящимся в базе данных изделий, позволяя выбирать подходящие посредством выбора изделия для соответствующего устройства.

Условия:

* Вы открыли проект.
* Вы открыли диалоговое окно "Свойства" для условного обозначения / функции, например, через меню Обработать > Свойства.
* Установлен флажок Главная функция на вкладке <Категория функции> (либо на вкладке Данные символа/функции).

1. Выберите в диалоговом окне Свойства ++...++ вкладку Изделие.
2. Щелкните в поле Номер изделия по ++...++.
3. В диалоговом окне Выбор изделия выберите нужное изделие. Возможен многократный выбор.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Данные переносятся в соответствующие поля вкладки Изделие, при этом Число шт. / колич. предварительно устанавливается на "1".

5. При необходимости обработайте свойства изделия в правой области диалогового окна.
6. Щелкните по кнопке ++OK++.

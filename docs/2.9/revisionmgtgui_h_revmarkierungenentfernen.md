# Удалить ревизии

Можно удалять маркеры ревизии и данные ревизии, созданные с помощью отслеживания изменений или сравнения свойств проектов, из проекта отдельно друг от друга.

Условие:

Открыт проект, который содержит маркеры ревизии. В навигаторе страниц выделена одна или несколько страниц или проект, либо в навигаторе пространств листа выделено одно или несколько пространств листа или проект.

1. Выберите следующие пункты меню: Сервисные программы > Управление ревизиями > Отслеживание изменений > Удалить ревизию
2. В диалоговом окне Удалить ревизию установите флажок Маркеры ревизий и / или Данные ревизии из отслеживания изменений.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Маркеры ревизии и/или данные ревизии будут удалены из текущего проекта ревизии.

!!! tip "Совет:"

    В графическом редакторе или навигаторе проектных данных с помощью команды всплывающего меню Удалить маркер ревизии можно удалить маркеры ревизии для отдельных выделенных объектов. При этом удаляется также текст маркера ревизии и графический маркер.

!!! note "Замечание:"

    * Во время ревизии для всего проекта ревизия полностью удаляется, если установлен флажок Данные ревизии из отслеживания изменений, а удаление применено ко всему проекту. Никакие последующие изменения в проекте больше не регистрируются. Чтобы во время ревизии областей обработки полностью удалить ревизии, необходимо предварительно отключить области обработки.
    * Если область обработки активна, данные ревизии страниц проекта удалить ***не*** удастся. Чтобы удалить данные ревизии отдельных страниц, необходимо предварительно отключить области обработки.

Условие:

Открыт проект, который содержит маркеры ревизии.

1. Выберите следующие пункты меню: Сервисные программы > Управление ревизиями > Сравнение свойств проектов > Удалить ревизию
2. В диалоговом окне Удалить ревизию установите флажок Маркеры ревизий и / или Данные ревизии сравнения свойств.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Маркеры ревизии и/или данные ревизии будут удалены из текущего проекта.

!!! tip "Совет:"

    Можно также удалить маркеры ревизии для отдельных объектов, удалив в диалоговом окне свойств значения свойств Маркер ревизии (из сравнения свойств) и Маркер изменения ревизии (из сравнения свойств). При этом удаляется также графический маркер.

# Использование имеющихся кабелей, кабельных соединений и экранирований

Использование имеющихся кабелей и кабельных соединений осуществляется посредством выбора ОУ. Соответствующие функции при этом должны подходить друг к другу, в противном случае выбрать ОУ невозможно. Т. е. на линиях определения кабелей можно использовать только линии определения кабелей, а на соединениях - только соединения.

На точках определения соединения или на соединениях для свойства Соединение: Принадлежность необходимо настроить значение "Кабель". В таком случае при использовании имеющегося кабеля открывается диалоговое окно Выбор ОУ для использования имеющихся кабелей, в противном случае — диалоговое окно Использовать имеющиеся соединения.

### Использование имеющихся кабелей или экранирований

При определении кабелей вручную можно выбрать существующее ОУ, сгенерировать новое ОУ или сгенерировать новое определение кабеля для существующего ОУ. При использовании имеющейся неразмещенной линии определения кабеля она удаляется из списка, а все данные этой функции переносятся на линию определения кабеля в схеме соединений.

При выборе экранирующего соединения (т. е. соединения, представляющего экранирование, а не определения кабеля экранирования) происходит перенос ОУ и обозначения экранирования. Экранирующее соединение может быть размещенным или неразмещенным. Оно не удаляется, так как имеющееся соединение используется не в полном объеме.

1. В графическом редакторе выделите линию определения кабеля, экранирование или точку определения соединения.
2. Выберите пункт меню Обработать > Свойства.
3. Выберите в диалоговом окне Свойства ++...++ вкладку Кабель или Экранирование.
4. Или выберите вкладку Точка определения соединения и для свойства Соединение: Принадлежность выберите значение "Кабель" из раскрывающегося списка.
5. Щелкните рядом с полем Видимое ОУ по кнопке ++...++.
6. В диалоговом окне Выбор ОУ выберите одно из имеющихся в проекте ОУ кабеля. Или сгенерируйте – в зависимости от выделения – новое ОУ посредством [Следующее].
7. Подтвердите ввод.

!!! info "Для сведения:"

    ОУ будет перенесено в диалоговое окно Свойства ++...++, а соединение будет присвоено кабелю.

!!! note "Замечание:"

    При генерировании нового ОУ посредством [Следующее] наблюдается следующая характеристика:

* При выделенном ОУ новое ОУ генерируется при помощи следующего свободного счетчика.
* При выделенном определении кабеля для того же ОУ генерируется еще одно определение кабеля.
* При выделенном кабельном соединении для того же ОУ тоже генерируется еще одно определение кабеля.

### Использование имеющихся кабельных соединений

1. В графическом редакторе выделите точку определения соединения.
2. Выберите пункт меню Обработать > Свойства.
3. В диалоговом окне Свойства ++...++ выберите вкладку Точка определения соединения и для свойства Соединение: Принадлежность выберите значение "Кабель" из раскрывающегося списка.
4. Щелкните рядом с полем Видимое ОУ по кнопке ++...++.
5. В диалоговом окне Выбор ОУ выберите соединение.
6. Подтвердите ввод.

!!! info "Для сведения:"

    ОУ будет перенесено в диалоговое окно Свойства ++...++, а соединение будет присвоено кабелю.

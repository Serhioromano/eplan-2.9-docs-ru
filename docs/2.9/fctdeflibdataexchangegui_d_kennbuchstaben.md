# Диалоговое окно Предложения буквенных обозначений

Вы открыли проект. Сервисные программы > Основные данные > Буквенное обозначение.

В этом диалоговом окне вы можете определять собственные наборы буквенных обозначений.

Обзор основных элементов диалогового окна:

### Раздел

Здесь выводится раздел определения функции.

### Категория

Здесь выводится категория определения функции.

### Группа

Здесь выводится группа определения функции.

### Определение функции

Здесь выводится определение функции.

!!! tip "Совет:"

    Щелкните дважды по заголовку одного из столбцов Раздел, Категория, Группа или Определение функции, чтобы выполнить сортировку по данному столбцу в алфавитном восходящем или нисходящем порядке.

### IEC / IEC 61346 / IEC 81346 / NFPA / GB/T 5094

Для данных стандартов прилагаются соответствующие наборы буквенных обозначений и отображаются в этих столбцах. Набор буквенных обозначений IEC не изменяется.

### <Определенный пользователем набор буквенных обозначений>

Если вы создали собственный набор буквенных обозначений, то выводится столбец с соответствующим заголовком. В отдельных ячейках укажите буквенные обозначения отдельных функций, разделяя буквенные обозначения посредством точки с запятой. При вставке функционального элемента в качестве настройки по умолчанию используется первое введенное буквенное обозначение.

### Всплывающее меню

Всплывающее меню содержит следующие приказы:

Команда |  Значение
---|---
Закрепить область |  Закрепляет все строки таблицы, которые находятся над маркировкой, или все столбцы, которые находятся слева от маркировки. В этом случае строки ниже или колонки справа от маркировки можно передвигать с помощью полос прокрутки.
Снять закрепление |  Убирает закрепление, т. е. все строки и столбцы можно передвигать с помощью полос прокрутки.
Подобрать ширину столбца |  Все столбцы так подогнаны к таблице по ширине, что и заголовок, и содержимое столбцов полностью читаемы.
Размер шрифта 100% |  Этот пункт меню доступен, только если размер шрифта в таблице был изменен ([Ctrl] \+ прокручивание колесика мыши). Отображает тексты в таблице с исходным размером шрифта.
Создать |  Открывает диалоговое окно Новый набор буквенных обозначений. Позволяет создать набор буквенных обозначений.
Вырезать |  Удаляет содержимое выделенных полей и копирует его в буфер обмена.
Копировать |  Копирует содержимое выделенных полей в буфер обмена.
Вставить |  Вставляет содержимое буфера обмена в выделенные поля.
Удалить |  Удаляет содержимое текущих выделенных полей.
Импорт |  Импортирует наборы буквенных обозначений из XML-файла.
Экспорт |  Экспортирует выделенные наборы буквенных обозначений в XML-файл.

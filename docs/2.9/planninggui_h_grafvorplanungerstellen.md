# Создание графического предварительного планирования

Сегменты структуры чертятся на странице предварительного планирования как обозначения местоположения, а объекты планирования — как черные ящики. Однако используемые символы отличаются. С объектами планирования (соединениями) можно связывать сегменты, располагающиеся на странице предварительного планирования. Объекты планирования (соединения) рисуются в виде открытых ломаных линий.

Условие:

Открыт проект и страница с типом страницы "Предварительное планирование".

1. Выберите пункты меню Вставить > Блок / вывод устройства / монтажная плата > Обозначение местоположения, чтобы создать сегмент структуры. Или же выберите пункты меню Вставить > Блок / вывод устройства / монтажная плата > Черный ящик, чтобы создать объект планирования.
2. Начертите прямоугольник, который будет представлять блок.

!!! info "Для сведения:"

    Если в проекте содержатся несколько определений сегментов, откроется диалоговое окно Выбрать определение сегмента.

3. Выделите нужное определение сегмента и нажмите ++OK++.
4. В диалоговом окне Свойства ++...++ введите обозначение для сегмента структуры или объекта планирования.
5. Задайте другие настройки на остальных вкладках диалогового окна Свойства ++...++.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Создается сегмент структуры или объект планирования с соответствующими свойствами.

!!! tip "Совет:"

    Сегменты структуры и объекты планирования можно также начертить в виде замкнутых ломаных линий. Для этого в библиотеке символов SPECIAL существуют специальные символы SCSS2 (для сегментов структуры) и DCPLO2 (для объектов планирования). Вставьте такой символ с помощью диалогового окна выбора символа (Вставить > Символ), начертите ломаную линию нужной формы и завершите действие, выбрав пункт всплывающего меню Закрыть ломаную линию.

Условие:

Открыт проект и страница с типом страницы "Предварительное планирование".

1. Выберите следующие пункты меню: Вставить > Символ.
2. Из библиотеки символов SPECIAL выберите символ PLINE для объекта планирования (соединения) в виде ломаной линии.
3. Щелчком мыши в блоках сегмента задается начальная точка ломаной линии.

!!! info "Для сведения:"

    Выбранный сегмент вносится в свойствах объекта планирования (соединения) в виде исходного сегмента.

4. По очереди задайте другие точки ломаной линии.
5. Щелчком мыши в следующем блоке сегмента задается конечная точка ломаной линии.

!!! info "Для сведения:"

    Выбранный сегмент вносится в свойствах объекта планирования (соединения) в виде целевого сегмента.

6. Выберите пункт всплывающего меню Закрыть ломаную линию, чтобы завершить чертеж линии.
7. В диалоговом окне Выбрать определение сегмента выделите требуемое определение сегмента и щелкните ++OK++.
8. В диалоговом окне Свойства ++...++ введите обозначение для объекта планирования (соединения).
9. Задайте другие настройки на остальных вкладках диалогового окна Свойства ++...++.
10. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Создается объект планирования (соединение) с соответствующими свойствами.

Условия:

* Открыт проект и страница с типом страницы "Предварительное планирование".
* Открыт навигатор предварительного планирования.

1. В навигаторе предварительного планирования выделите сегмент структуры или объект планирования.
2. Выберите пункт всплывающего меню Разместить.

!!! info "Для сведения:"

    Символ блока прикрепляется к курсору.

3. Начертите прямоугольник, который будет представлять блок.

!!! info "Для сведения:"

    Свойства сегмента переносятся на блок.

!!! tip "Совет:"

    Графически разместить сегмент можно многократно. Вследствие этого сегмент можно отображать на нескольких страницах предварительного планирования.
    При этом дальнейшие размещения сегмента структуры или объекта планирования невозможно выполнить внутри другого блока, то есть разместить их под другим сегментом. В дереве навигатора предварительного планирования дальнейшие размещения сегмента обозначаются пиктограммой ![](images/navigatorgui_placedsegment_as.png){: .ui-icon }.

Условия:

* Открыт проект и страница с типом страницы "Предварительное планирование".
* Открыт навигатор предварительного планирования.

1. Выделите объект планирования (соединение) в навигаторе предварительного планирования.
2. Выберите пункт всплывающего меню Разместить.

!!! info "Для сведения:"

    Символ ломаной линии прикрепляется к курсору.

3. Задайте начальную точку ломаной линии. При этом щелкните в начерченных блоках сегмента: таким образом этот сегмент переносится как исходный сегмент объекта планирования (соединения).
4. По очереди задайте другие точки ломаной линии. При этом расположите последнюю конечную точку в начерченных блоках сегмента: таким образом этот сегмент переносится как целевой сегмент объекта планирования (соединения).
5. Выберите пункт всплывающего меню Закрыть ломаную линию, чтобы завершить чертеж линии.

!!! info "Для сведения:"

    Свойства объекта планирования (соединения) переносятся на ломаную линию.

!!! tip "Совет:"

    Вы также можете размещать сегменты структуры, объекты планирования и объекты планирования (соединения), просто перетаскивая выделенный сегмент мышью из навигатора предварительного планирования на открытую страницу предварительного планирования в графическом редакторе.

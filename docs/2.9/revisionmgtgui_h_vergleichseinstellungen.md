# Определение настроек сравнения

В диалоговом окне Настройки: Сравнение свойств проектов можно подробно указать, какие свойства следует сравнить. Настройки сохраняются в схеме. Так можно создать разные схемы для различных целей и снова открыть настройки, определенные однажды.

1. Выберите следующие пункты меню: Сервисные программы > Управление ревизиями > Сравнение свойств проектов > Сравнить проекты
2. Щелкните в диалоговом окне Сравнить проекты по ++...++ рядом с полем Настройки для сравнения свойств.
3. В диалоговом окне Настройки: Сравнение свойств проектов выберите из раскрывающегося списка определенную пользователем схему, которую вы хотели бы изменить, или создайте с помощью ![](images/all_new_as.png){: .ui-icon } (Создать) новую [схему](schemeconfig_h_schematabearbeiten.md).
4. Из раскрывающегося списка Категория выберите тип объекта, для которого требуется задать сравниваемые свойства.

!!! info "Для сведения:"

    В зависимости от сделанного выбора в поле Не сравнивать будут отображены все свойства, доступные для сравнения выбранного типа объектов.

5. Выделите в поле Не сравнивать нужные свойства, и переместите их с помощью кнопки со стрелкой в поле Сравнить.
6. Для отмены выбора (если необходимо) переместите второй кнопкой-стрелкой свойства, выделенные в поле Сравнить обратно в поле Не сравнивать.
7. Установите флажок Учитывать удаленные и новые объекты, если удаленные и новые объекты соответствующего типа объектов также должны рассматриваться как изменения.
8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Ваши настройки будут сохранены в выбранной схеме, и диалоговое окно Сравнить проекты откроется снова.

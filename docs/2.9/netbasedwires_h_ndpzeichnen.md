# Вставить точки определения сети

Точка определения сети позволяет указать свойства сети, например имя сети. Кроме того, здесь можно определить сетевые соединения.

Условие:

Вы открыли проект. Вы находитесь на странице в графическом редакторе.

1. Выберите пункты меню Вставить > Точка определения сети.

!!! info "Для сведения:"

    Символ точки определения сети появится рядом с курсором.

2. Щелкните по схеме соединений, чтобы разместить точку определения сети.
3. В диалоговом окне Свойства ++...++ укажите свойства сети на вкладке Определение сети.
4. Определите на вкладке Соединения сети соединения в сети.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Свойства присваиваются соединениям, определенным в точке определения сети.

!!! tip "Совет:"

    * При помощи настройки пользователя (Параметры > Настройки > Пользователь > Графическая обработка > Символы соединения) можно указать, что при вставке точки определения сети свойство Определение целей символов соединений соответствующей сети автоматически отключается.
    * Вы можете вставить одновременно несколько точек определения сети, удерживая нажатой левую клавишу мыши и перетягивая курсор через несколько линий автоматического соединения. В этом случае на каждой пересеченной линии автоматического соединения будет размещена точка определения сети.

# Диалоговое окно Выбор устройства

Вы открыли проект.

* Вы выделили главную функцию в графическом редакторе или навигаторе. Обработать > Свойства. Выберите вкладку Изделие и щелкните по кнопке ++Выбор устройства++.
* Данные проекта > Устройства / Изделия > Навигатор спецификаций. Выделите одну запись. Всплывающее меню > Выбор устройства.

В этом диалоговом окне выбирается устройство, т.е., на основе технических данных одному устройству присваивается один или несколько изделий.

!!! warning "Предупреждение:"

    Устройству присваиваются данные из определений функций. Эти данные записываются в базу данных изделий на вкладке Шаблоны функций в таблице определений функций.

Обзор основных элементов диалогового окна:

### Автом. фильтр

При помощи кнопки ++...++ откройте диалоговое окно Автоматический фильтр: Выбор устройства. Здесь настройте критерии фильтра для изделия, которые предлагаются затем в списке Главное изделие.

### Активн.

Отметьте данную опцию, чтобы использовать настроенные критерии фильтра в списке.

### Главное изделие

В списке отображаются все изделия из базы данных изделий, подходящие к выбранному устройству и не являющиеся принадлежностями (т. е. для которых не установлен флажок Принадлежности на вкладке Принадлежности). Номер изделия, тип записи данных и вариант отображаются всегда; дополнительно можно вывести другие столбцы в зависимости от соответствующего типа функционального элемента (через пункт всплывающего меню Конфигурировать представление).

При повторном выборе устройства последние выбранные изделия автоматически выделяются в списке.

!!! warning "Предупреждение:"

    Для устройств с определениями функций группы "Графика" при выборе устройства отображаются все изделия, имеющие шаблоны функций с определением функции "Графика". Дополнительно отображаются все изделия без шаблонов функций и изделия принадлежностей. Таким способом выбираются, например, изделия главной группы продуктов "Механика".

### Принадлежности

Если в списке Главное изделие выделено изделие, то здесь отображаются соответствующие принадлежности. Данные предметы оборудования должны быть присвоены главному изделию в базе данных изделий, т. е. введены в таблицу на вкладке Принадлежности.

Предмет оборудования можно выбрать только тогда, когда в столбце Выбор введен номер изделия:

* Для ***требуемой ***принадлежности номер изделия автоматически переносится из столбца Номер изделия / имя в столбец Выбор. Таким образом изделие принадлежности можно выбрать сразу.
Для списков принадлежностей номер изделия автоматически не переносится. Щелкните в столбце Выбор по ++...++, чтобы открыть диалоговое окно Выбрать принадлежности. Здесь можно выбрать требуемое изделие из списка принадлежностей.
* В ***опциональные ***принадлежности номер изделия автоматически не переносится. Сначала щелкните столбец Выбор и [...], чтобы перенести номер изделия. После этого можно выбрать принадлежности.
В таком случае для списков принадлежностей откроется диалоговое окно Выбрать принадлежности, в котором можно выбрать требуемые изделия из списка принадлежностей.

!!! warning "Предупреждение:"

    В диалоговом окне Выбрать принадлежности отображаются такие же столбцы, как и в списке Главное изделие.

!!! example "Пример:"

    Главное изделие 001 (контактор + три силовых замыкающих контакта) может комбинироваться с вспомогательным изделием 002 (два размыкающих контакта) или с вспомогательным изделием 003 (два замыкающих контакта).Если в списке Главное изделиевыделеноглавное изделие001, то в списке Принадлежностипредлагаются вспомогательные изделия 002 и 003. Можно также выбрать следующие комбинации:001 (контактор + три силовых замыкающих контакта)001 + 002 (контактор + три силовых замыкающих контакта + два размыкающих контакта)001 + 003 (контактор + три силовых замыкающих контакта + два замыкающих контакта)

### Выбранные изделия: Функции / шаблоны

В этом списке отображаются функции/шаблоны как данные шаблонов функций (например, Определение функции), так и важные данные изделий (например, Номер изделия). Столбцы с данными шаблонов составляются динамически, как и в базе данных изделий. То, какие столбцы будут здесь отображаться, зависит от определения главной функции, от определений других функций устройства и от шаблонов функций выбранного изделия.

Если устройству еще не присвоено изделие, здесь показываются только функции текущего устройства. Восклицательный знак перед функциями указывает на то, что соответствующие функции еще не взяли на себя шаблон функции изделия.

При выделении изделия в поле Главное изделие показывается, какие функции устройства берут на себя функции шаблонов функций изделия и остаются ли свободные шаблоны функции. Это вначале является только предварительным просмотром; изделие еще не присвоено. Строки предварительного просмотра изображаются ***курсивом***.

Если вы перенесли изделие с помощью кнопки со стрелкой в список Выбранные изделия: Функции / шаблоны, изделие выбрано. Это показывается символом "галочка" перед шаблонами функции изделия.

В каждой строке отображается пиктограмма, которая указывает, принимает ли функция устройства функции подходящего шаблона функции изделия:

Пиктограмма |  Значение
---|---
![](images/navigatorgui_function_as.png){: .ui-icon } |  Функция устройства еще не присвоена (т. е. необходимо выбрать другие изделия, чтобы дополнить устройство)
![](images/navigatorgui_functionstate_as.png){: .ui-icon } |  Функция устройства присвоена (функция замещает собой шаблон функции изделия)
![](images/navigatorgui_functiontemplate_as.png){: .ui-icon } |  Шаблон функции изделия еще свободен (т.е., изделие содержит больше функций, чем используется устройством)

Если в списке отображаются несколько изделий, принадлежащие друг другу шаблоны функции выделяются попеременно цветом. Таким образом легче различать отдельные изделия. Если выделена одна строка в списке, тем самым выделено всё изделие. Чтобы это показать, все шаблоны функций, относящиеся к одному и тому же изделию, выделяются ***Полужирный***.

!!! warning "Предупреждение:"

    здесь также отображаются изделия, не имеющие шаблонов функций. Это могут быть, например, изделия принадлежностей или изделия, присвоенные устройству через выбор изделий.

!!! tip "Совет:"

    В конфигурации столбцов списка (с помощью пункта всплывающего меню Конфигурировать представление) все столбцы, в которых отображаются данные шаблонов функций, обобщены в записи Данные шаблона. При помощи этого флажка можно отобразить или скрыть столбцы вместе с данными шаблонов.

### (Удалить)

Удаляет ***все*** шаблоны функций выделенного изделия из списка Выбранные изделия: Функции / шаблоны.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

### [Настройки]

Открывает диалоговое окно Настройки: Выбор устройства. Установите здесь, какие устройства учитываются при выборе устройства.

# Вкладка Атрибуты

Сервисные программы > Изделие > Управление. В дереве или списке выделено одно изделие.

На этой вкладке можно ввести дополнительные атрибуты независимо от типа изделия. Тогда эти атрибуты идентичны для всех вариантов изделия.

Обзор основных элементов диалогового окна:

### Значение

Определите значение атрибутов. В противоположность произвольным свойствам атрибуты не являются многоязычными! Длина ввода ограничена 80 символами.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

!!! note "Замечание:"

    Каждый атрибут может использоваться для выполнения фильтрации с фильтром по полю. Можно указать, например, классы изделий и признаки классификации согласно ETIM и отобразить данные изделия, отфильтрованные согласно этой модели.

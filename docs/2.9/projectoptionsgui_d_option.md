# Диалоговое окно Опция

Вы открыли проект. Данные проекта > Опции > Навигатор.

* Выберите пункты всплывающего меню Создать > Опция.
* Вы выделили опцию. Обработать > Свойства.

В этом диалоговом окне создается новая опция или выполняется обработка уже имеющейся.

Обзор основных элементов диалогового окна:

### Опциональная группа

В этом групповом поле выбирается опциональная группа, которой присваивается опция.

* Имя: Выберите имеющуюся опциональную группу из раскрывающегося списка или введите новое имя. Если оставить это поле пустым (или удалить имеющуюся запись) опция не присваивается никакой опциональной группе.
* Описание: В этом поле отображается текст описания выбранной опциональной группы. При вводе новой опциональной группы это поле доступно для записи, и в него можно ввести текст описания.

### Имя

Введите здесь имя опции или измените имеющееся имя.

### Описание

Введите здесь описывающий текст для опции. Под описанием понимается многоязычный текст, отображаемый по умолчанию в представлении структуры дерева рядом с именем (отделяется пробелом и заключается в круглые скобки).

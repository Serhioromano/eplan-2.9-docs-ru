# Диалоговое окно Объединить соединения: Предв. просмотр результата

Данные проекта > Соединения > Объединить. Вы установили в диалоговом окне Автоматически объединить соединения флажок Предварительный просмотр результата. Щелкните по кнопке ++OK++.

В этом диалоговом окне приводятся все соединения, которые будут изменены. Новые номера объединений проводов можно подтвердить, нажав клавишу ++OK++, или изменить вручную.

Доступны следующие элементы диалогового окна:

Таблица содержит следующие столбцы:

* Строка: Показывает непрерывный номер строки.
* Источник: Источник соединения.
* Цель: Цель соединения.
* Объединенные провода: Номера объединенных проводов, которые перед автоматическим объединением были в соединении.
* Создать объединенные провода: Новый номер объединения проводов, присвоенный при автоматическом объединении. Номер здесь можно изменить вручную, чтобы присвоить соединение другому объединению проводов.

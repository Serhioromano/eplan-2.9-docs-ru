# Диалоговое окно Список соответствия

Вы открыли проект. Проект содержит полностью определенный электрошкаф. Навигатор пространства листов открыт, и одно пространство листов открыто. Пространство листа содержит импортированные трехмерные тела, которые необходимо сохранить в виде электрошкафа. Выделена и выбрана трехмерная геометрическая фигура с прямоугольником.

Обработать > Логика устройства > Автоматическая интерпретация

В этом диалоговом окне отображаются результаты проверки соответствия. Эта проверка активируется в том случае, если в диалоговом окне выбора Выбрать шаблон интерпретации вы выберете несколько различных объектов шаблонов. Определяется, какие шаблоны обладают более точным соответствием / максимальной квотой соответствия определенных признаков с определяемым 3D-объектом.

Обзор основных элементов диалогового окна:

### Список

Список содержит следующие столбцы:

* Полное обозначение функционального элемента: Номер изделия / Номер функционального элемента в зависимости от вкладки, активированной в диалоговом окне Выбрать шаблон интерпретации.
* Определение функции
* Квота: Степень соответствия с определяемым функциональным элементом, указанная в процентах.

Записи в списке сортируются по значению в графе Квота. Объекты шаблонов с максимальным соответствием находятся в верхней части списка. Список может включать в себя макс. 10 записей.

Выделите запись в списке, свойства которой необходимо передать определяемому 3D-объекту. Выбор выделенной записи осуществляется нажатием ++OK++.

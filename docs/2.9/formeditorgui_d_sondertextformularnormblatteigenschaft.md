# Диалоговое окно Свойства (спец. текст): Свойства формы / свойства (спец. текст): Свойства рамки

Вы открыли проект. Сервисные программы > Основные данные > Рамка / Форма > Открыть > [Открыть].

* Вставить > Спец. текст > Свойства формы (в открытых формах) или Вставить > Спец. текст> Свойства рамки (в открытых рамках).
* Вы дважды щелкнули по свойству формы\- / страницы (или выбрали Всплывающее меню > Свойства; в этом случае могут быть выделены также несколько свойств).

В этом диалоговом окне выберите свойство формы- / страницы, которое должно быть добавлено в рамку или форму, и установите его свойства для формы, языка и рамки.
При этом, соответственно, предлагается выбрать только те свойства, которые разрешены для данного типа формы / рамки.

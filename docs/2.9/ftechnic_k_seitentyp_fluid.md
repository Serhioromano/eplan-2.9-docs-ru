# Типы страниц для EPLAN Fluid

### Тип страницы "Схема соединений, Fluid-Техника"

В EPLAN Fluid в вашем распоряжении для проектирования схем соединений имеется специальный интерактивный тип страницы "Схема соединений, Fluid-Техника", обозначенный в навигаторе страниц пиктограммой ![](images/pagebrowsergui_pagefluid_as.png){: .ui-icon }.

При использовании этого типа страниц необходимо учитывать следующие особенности:

* Поворот и отражение этого типа страниц невозможны в целях изменения стандарта.
* Объектам схемы соединений из библиотеки символов электротехники, которые размещаются на странице Fluid-Техники (например, символы клапанов с электромагнитным приводом), автоматически назначается вид представления "Парная перекрестная ссылка". Это также верно для приемных элементов из библиотеки символов Fluid техники, которые имеют в распоряжении электротехнические определения функций.
* Если черные ящики или выводы устройств добавляются на страницу типа " схема соединений Fluid-техники", этим объектам автоматически присваиваются правильные определения функций, например, "Черный ящик (Fluid)" или " вывод устройства (Fluid)".

### Тип страницы "Обзор функций (Fluid-Техника)"

Для упрощенного представления сложных функций Fluid-техники в EPLAN Fluid можно создать специальные страницы обзора. Для этого служит тип страницы "Обзор функций (Fluid-Техника)". Страницы этого типа обозначаются в дереве навигатора страниц пиктограммой ![](images/pagebrowsergui_pagefunctionoverviewfluid_as.png){: .ui-icon }.

В этих обзорах функций для Fluid-техники отображаются все перекрестные ссылки к функциям на схеме соединений Fluid-техники. Кроме того, на страницах этого типа есть автоматическое соединение, соединения и точки разрыва.

Для функций, соединений и макросов, которые вставляются / размещаются на обзорах функций, используется вид представления "Обзор функций (Fluid-Техника)".

* При общей обработке и при генерации отчетов ***функций*** учитывается этот вид представления. В последовательности видов представления для функций этот вид представления находится на последнем месте за видом представления "Функциональный".
* ***Соединения*** на обзорах функций имеют вид представления "Обзор функций (Fluid-Техника)". Такие соединения отображаются в навигаторе соединений, но не имеют обратных эквивалентов на других типах страниц. Поэтому соединения с этим видом представления не учитываются при общей обработке.

### Настройка для раздела

При создании страницы типа "Схема соединений, Fluid-Техника" или "Обзор функций (Fluid-Техника)" в таблице свойств по умолчанию отображается свойство страницы Раздел (ид. 11037).

При помощи этого свойства можно настроить раздел, который следует использовать для страницы Fluid-техники.

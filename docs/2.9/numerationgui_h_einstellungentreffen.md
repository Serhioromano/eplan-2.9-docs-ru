# Исключить группы устройств из автоматической нумерации

При вставке нового устройства в схему соединений можно указать, что для этого устройства ***не*** должна выполняться автоматическая нумерация. Устройствам, исключенным из нумерации, присваивается буквенное обозначение или счетчик. Счетчик не считает по нарастающей.

Условие:

Вы открыли проект.

1. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация (online)
2. В диалоговом окне Настройки: Нумерация (online) рядом с полем Формат нумерации щелкните по кнопке ++...++.
3. Выберите в поле Схема определенную пользователем схему.
4. Выберите вкладку ОУ.
5. Снимите соответствующий флажок для всех групп устройств, которые не должны нумероваться в режиме подключения.
6. Щелкните по кнопке ![](images/all_save_as.png){: .ui-icon } (Сохранить) и по ++OK++.

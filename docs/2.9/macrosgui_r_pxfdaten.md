# Вкладка Данные (Настройки: Внешняя обработка)

* Вы открыли проект. Сервисные программы > Внешняя обработка свойств > Экспортировать данные. Нажмите в диалоговом окне Внешняя обработка рядом с полем Настройки на ++...++. Выберите вкладку Данные.

* Параметры > Настройки > Пользователь > Интерфейсы > Внешняя обработка. Выберите вкладку Данные.

На этой вкладке указываются данные, которые необходимо учитывать при внешней обработке.

Обзор основных элементов диалогового окна:

### Категория

Выберите из этого раскрывающегося списка, какой вид свойств должен быть отображен в списке.

### Фильтр

Введите в это поле текст, по которому должна выполняться фильтрация свойств.

Если необходимо вернуться к прежнему виду свойств, удалите текст из поля ввода. Для этого нажмите кнопку ![](images/all_delete_as.png){: .ui-icon } (Удалить).

### Доступные элементы формата

В зависимости от выбранной категории в списке выводятся возможные элементы формата.

Для экспорта элемента формата необходимо переместить его вправо в список выбранных элементов формата; многократный выбор в данном случае невозможен.

### Выбранные элементы формата

В этом списке в табличной форме представлены все выбранные элементы форма и указанием их обозначения и информации "Возможно изменение" = "Да" / "Нет". С помощью кнопок, расположенных над списком, можно удалять элементы формата или изменять их последовательность.

Кнопка |  Значение
---|---
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет выбранный элемент формата.
![](images/all_arrowup_as.png){: .ui-icon } (Переместить вверх) / ![](images/all_arrowdown_as.png){: .ui-icon } (Переместить вниз) |  Перемещает запись в списке вверх / вниз.

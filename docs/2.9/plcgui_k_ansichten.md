# Виды в навигаторе ПЛК

В представлении структуры дерева диалогового окна навигатора ПЛК — <Имя проекта> можно выбирать разные виды для отображения данных ПЛК.

Во всех видах отображаются все существующие в проекте данные ПЛК, т. е. блоки ПЛК, все выводы устройств ПЛК и шаблоны функций. При этом отображаются как пустые блоки ПЛК, так и блоки ПЛК, содержащие только шаблоны функций.

С помощью пункта всплывающего меню Вид доступны следующие виды:

* С ориентацией на ОУ: Данные ПЛК располагаются согласно структуре проекта по их ОУ.
* С ориентацией на адрес: Данные ПЛК располагаются в пределах структуры проекта по адресам. Т. е. ниже адреса ПЛК отображаются выводы устройства В/В ПЛК. Соответствующие источники питания только тогда отражаются ниже адреса ПЛК, когда при источнике питания внесен соответствующий адрес ПЛК. Дополнительно блоки и выводы устройств ПЛК группируются под соответствующим ЦПУ, если данное свойство внесено в соответствующий блок ПЛК.
* С ориентацией на канал: Данные ПЛК располагаются в пределах структуры проекта по каналам. Т. е. под обозначением канала отображаются выводы устройства В/В и соответствующие источники питания. Пиктограмма ![](images/plcgui_plcchannelwithsafty_as.png){: .ui-icon } для канала отображает, находится ли под ним релевантный для безопасности вывод устройства. Дополнительно установленная пиктограмма ![](images/navigatorgui_nonplaced_as.png){: .ui-icon } отображает, содержит ли канал неразмещенные выводы устройства В/В. Выводы устройств ПЛК, у которых нет ни внесенного вручную, ни автоматически определенного обозначения канала, упорядочиваются согласно ОУ.
* Ориентировать по каркасу: Данные ПЛК отображаются в соответствии с присвоением карт ПЛК каркасам. Каркасы отмечаются стоящей впереди пиктограммой ![](images/navplc_rack_as.png){: .ui-icon }; под ней перечислены карты ПЛК (пиктограмма ![](images/navplc_moduleonrack_as.png){: .ui-icon }) с адресами шины. Последовательность соответствует номерам позиций разъемов. Под картами ПЛК показываются содержащиеся в них функции.
Карты ПЛК, вставленные или интегрированные в первичную станцию, обозначаются пиктограммой ![](images/navplc_moduleonheadmodule_as.png){: .ui-icon }. Карты ПЛК, вставленные рядом с первичной станцией, обозначаются пиктограммой ![](images/navplc_moduleonrack_as.png){: .ui-icon }. Первичная станция одновременно является каркасом и поэтому обозначается стоящей впереди пиктограммой ![](images/navplc_rack_as.png){: .ui-icon }. Под каркасом отображаются сначала карты ПЛК, вставленные в первичную станцию, а затем все остальные карты ПЛК.
Для устройств, находящихся на подразъеме, это отображение возможно за счет того, что за номером разъема, отделенным десятичной точкой, указывается подразъем, напр. 2.1:=EB3+ET1-A99.
* Связано с приводами: данные ПЛК располагаются в пределах структуры проекта по приводам. Приводы обозначается стоящей впереди пиктограммой ![](images/navplc_drive_as.png){: .ui-icon }. Функции отображаются под уровнями иерархии "Проект конфигураций", "Привод" и "Имя станции". Функции, которые не присвоены приводу, также отображаются здесь. Для этого необходимо, чтобы функции были присвоены проекту конфигурации и / или рабочей станции.

Фильтр позволяет ограничить отображение определенными данными.

Для этого щелкните в диалоговом окне ПЛК — <Имя проекта>рядом с полем Фильтрпо ++...++ и в диалоговом окне Фильтрвыполните следующие настройки:Активн.Отрицат.Критерий Оператор Значение Группа функций=Вывод устройства ПЛК, В/В, 1 вывод устройства

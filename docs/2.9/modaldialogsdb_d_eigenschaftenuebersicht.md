# Диалоговое окно Обзор свойств — <Имя проекта>

Вы открыли проект. Вы выделили один или несколько объектов (тексты, условные обозначения, сегменты, графические элементы) в графическом редакторе, пространстве листа или навигаторе. Обработать > Обзор свойств.

В этом диалоговом окне можно обрабатывать свойства одного или нескольких выделенных объектов (тексты, графические элементы, функции, сегменты).

Чтобы отобразить требуемые свойства для выделенных объектов, необходимо выбрать или создать подходящую схему для конфигурации свойств. Для графических элементов, например, можно использовать прилагаемую схему "Отформатировать график". В противном случае для свойств в столбце Значение отображается запись "Нет данных".

Обзор основных элементов диалогового окна:

Выберите для отображения и обработки свойств схему из ***раскрывающегося списка***. Нажмите ++...++, чтобы открыть диалоговое окно для конфигурации свойств. Там вы можете указать для схемы, какие свойства должны отображаться и в какой последовательности.

### Панель инструментов

Кнопка |  Значение
---|---
![](images/all_treeview_as.png){: .ui-icon } (представление в виде дерева) |  Отображает свойства конфигураций свойств, сгруппированные по исходным объектам в представлении в виде дерева.
![](images/all_listview_as.png){: .ui-icon } (представление в виде списка) |  Отображает свойства конфигураций свойств, расположенные друг под другом, в виде списка.

### Фильтр

Введите в это поле текст, по которому должна выполняться фильтрация свойств. При этом выполняется поиск имен отображаемых свойств и их значений.

Если необходимо вернуться к прежнему виду свойств, удалите текст из поля ввода. Для этого нажмите кнопку ![](images/all_delete_as.png){: .ui-icon } (Удалить).

### Свойство / Значение

Отображает для выделенного или выделенных объектов значения определенных свойств. От выбранной схемы зависит, какие значения здесь можно обработать.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

!!! note "Замечание:"

    В диалоговом окне Обзор свойств общая обработка распределенных функций при активированном режиме ***Свойства (общие)*** ***невозможна***.

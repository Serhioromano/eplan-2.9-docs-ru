# Добавить изделие

В отличие от добавления изделий проекта при помощи этого пункта меню присвойте ***текущему*** устройству одно (или несколько) другое(ие) изделие(я).

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Данные проекта > Устройство / изделие > Навигатор спецификаций
2. В диалоговом окне Спецификация — <Имя проекта> выделите устройство (устройства), которому / которым необходимо присвоить другие изделия.
3. Выберите пункт всплывающего меню Добавить изделие.
4. В диалоговом окне Выбор изделия выберите добавляемое изделие(я). Возможен многократный выбор.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Новые изделия присваиваются устройствам.

!!! note "Замечание:"

    Могут быть использованы все функции Выбор изделия, причем учитываются предварительные настройки, специфические для базы данных изделий.

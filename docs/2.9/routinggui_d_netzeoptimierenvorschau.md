# Диалоговое окно Автоматически оптимизировать сети: Предв. просмотр результата

Сгенерированы маршрутизируемые соединения. В навигаторе пространства листа или в пространстве листа выделен компонент, содержащий маршрутизируемые соединения. Данные проекта > Соединения > Автоматически оптимизировать сети. В диалоговом окне Автоматически оптимизировать сети щелкните по ++OK++.

В этом диалоговом окне отображается результат анализа сетей и маршрутизируемых соединений. Данные первого соединения переносятся после оптимизации на все соединения сети.

Доступны следующие элементы диалогового окна:

В таблице отображаются свойства соединений. Таблица содержит следующие столбцы:

* Строка: Показывает непрерывный номер строки.
* Статус: Указывает, является ли соединение оптимизируемым.
"Новое" = соединения, которым в ходе оптимизации присвоена новая трасса маршрутизации.
"Не поддается оптимизации" = неполные или уже оптимально проложенные соединения.
* Источник
* Цель
* Обозначение соединения
* ОУ (полное)
* Цвет/номер соединения
* Поперечное сечение соединения / диаметр с единицей измерения
* Номер изделия [1]: Показывает номер первого изделия, указанного в соединении.

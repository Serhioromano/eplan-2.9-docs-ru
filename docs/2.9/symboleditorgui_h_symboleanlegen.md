# Создать символы

Условие:

Вы открыли проект и как минимум одну библиотеку символов.

1. Выберите следующие пункты меню: Сервисные программы > Основные данные > Символ > Создать
2. если открыто несколько библиотек символов, то в диалоговом окне Выбор библиотеки символов выберите библиотеку символов, в которой должен быть создан символ.
3. Щелкните по кнопке ++OK++.
4. В диалоговом окне Генерировать варианта выберите вариант, который должен быть создан для символа.
5. Щелкните по кнопке ++OK++.
6. В диалоговом окне Свойства символов — <Имя библиотеки> задайте данные создаваемого символа. Номер символа будет при этом заполнен первым свободным номером.
7. Щелкните в поле Определение функции по ++...++, чтобы задать логику символа в диалоговом окне Определения функций.
8. Подтвердите ввод.

!!! info "Для сведения:"

    Символ создается, открывается в отдельном графическом редакторе и может быть непосредственно обработан. При этом следует иметь в виду, что точка вставки может быть только сдвинута, но не удалена.

!!! note "Замечание:"

    * В пустой библиотеке символов в качестве стартового значения предлагается номер символа "0".
    * Не допускаются в качестве номеров символов отрицательные значения и значения, превышающие максимально разрешенный диапазон 32000.

# Диалоговое окно Определить отчет

Вы открыли проект. Сервисные программы > Отчеты > Генерировать. Щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).

В этом диалоговом окне установите, какой тип отчета должен генерироваться, и определите форму вывода и исходный проект.

Обзор основных элементов диалогового окна:

### Форма вывода

В этом поле установите, в какой форме будет генерироваться отчет.

* Страница: Отчет генерируется в форме [страниц отчета](formgeneratorgui_k_auswertungsseiten.md) и выводится в проект.
* Размещение вручную: Этот параметр доступен, если в графическом редакторе открыта страница проекта. Отчет генерируется в форме [встроенного отчета](formgeneratorgui_k_ausgabeformen.md). Вы можете вручную размещать встроенный отчет в текущую открытую страницу проекта. Эта настройка полезна в комбинации с сортировкой страниц "Общая".

!!! note "Замечание:"

    Даже если в меню Параметры > Настройки > Проекты > "Имя проекта" > Отчеты > Вывод в страницах в столбце Сортировка страниц для отчета была выбрана, например, сортировка страниц "Установка и место установки", то разместить вручную можно исключительно общий отчет.

### Исходный проект

В этом поле находится проект, который является источником данных для отчетов и шаблонов. В [...] откройте диалоговое окно Исходный проект, в котором можно выбрать другой проект в качестве источника данных.

### Выбрать тип отчета

Отчеты основаны на [типах отчетов](formgeneratorgui_k_auswertungstypen.md). В этом поле отображаются все доступные типы отчетов. Выделите тип отчета, для которого необходимо сгенерировать отчет.

### Текущая страница

Этот флажок предусмотрен, в зависимости от типа отчета, только для формы вывода "Размещение вручную". Установите его, чтобы в текущем отчете учитывались только данные открытой в данный момент страницы; это имеет смысл, например, для встроенных отчетов.

### Индивидуальный выбор

Эта функциональность доступна только для [отчетов, относящихся к функции](formgeneratorgui_k_funktionsbezogen.md). Если этот флажок установлен, то перед генерацией отчета открывается диалоговое окно Выбрать вручную. В этом диалоговом окне вы можете отдельно выбирать устройства для отчета.

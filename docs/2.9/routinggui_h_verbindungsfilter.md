# Использовать фильтр соединений

У кабельных каналов, сегментов маршрутизации, областей маршрутизации и соединительных отверстий для проводов есть свойство Пространство листа: Фильтр соединений. Фильтр соединений используется при маршрутизации, чтобы указывать, какие соединения можно маршрутизировать по данной трассе. Значениями таких критериев могут быть цветовые обозначения проводов или значения напряжения.

Присвоение значений маршрутизируемым соединениям носит исключительно текстовый характер. Для определения значений используются произвольно вводимые тексты, которые, однако, должны присутствовать также и на изделиях соединений.

### Указать настройки проекта для фильтра соединений

Параметры, которые можно активировать или деактивировать в диалоговом окне Фильтр соединений, указываются в настройках проекта.

1. Чтобы вызвать настройки для фильтра соединений, выберите пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Маршрутизируемые соединения > Общее.
2. В диалоговом окне Настройки: Общее выберите вкладку Фильтр соединений.
3. Нажмите кнопку ++...++.
4. Выберите в диалоговом окне Выбор критериев свойство соединения, для которого следует отображать значения в фильтре соединений, например свойство Цвет/номер соединения, и нажмите ++OK++.
5. Введите в столбце таблицы Значение тексты, описывающие критерий фильтра, например "синий", "красный", "черный", "желтый". Для этого вам предоставлено 15 строк.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Настроенные значения сохранятся и будут отображаться в диалоговом окне Фильтр соединений при его вызове.

### Применение фильтра соединений

Параметры, определенные в фильтре соединений, следует применять к объектам, задействованным в процессе маршрутизации.

1. Отметьте в пространстве листа один или несколько кабельных каналов или сегментов маршрутизации и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно Свойства (усл. обозначение): Размещение изделия (3D).

2. Выберите свойство Фильтр соединений и в столбце Значение щелкните по кнопке ++...++.

!!! info "Для сведения:"

    Откроется диалоговое окно Фильтр соединений. В таблице отображаются значения, которые вы определили в настройках проекта.

3. Установите флажок Активн. над таблицей, чтобы фильтр соединений был включен всегда.

!!! info "Для сведения:"

    Теперь можно установить или снять отдельные значения в таблице.

4. Установите флажок Активно в таблице перед каждым значением, которое должно действовать для этого соединения.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В диалоговом окне Свойства (усл. обозначение): Размещение изделия (3D) в поле Значение свойства Пространство листа: Фильтр соединений перечисляются все значения, которые вы установили в фильтре соединений.

!!! info "Для сведения:"

    При маршрутизации соединения маршрутизируются только через обработанный сегмент маршрутизации, соответствующий установленным значениям.

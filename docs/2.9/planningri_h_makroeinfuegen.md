# Вставить марос окна с данными функциональных схем автоматизации

Для многоразового использования фрагментов страницы, содержащих данные функциональной схемы автоматизации, следует сохранить их в макросе окна. Такой макрос окна, помимо обычных символов из технологии производственных процессов, может содержать условные обозначения, связанные с сегментами из предварительного планирования (технологическими контурами, функциями ТК, резервуарами, объектами планирования, объектами планирования (трубопроводами) и т. д.). При вставке макроса окна на страницу функциональной схемы автоматизации другого проекта учитываются содержащиеся сегменты, которые можно переносить в текущий проект.

Условия:

* Вы открыли проект.
* Открыта страница типа "Функциональная схема автоматизации".

1. Выберите следующие пункты меню: Вставить > Макрос окна / символа.
2. В диалоговом окне Выбрать макрос выберите макрос окна *.ema для вида представления "Функциональная схема автоматизации".

!!! info "Для сведения:"

    Если макрос окна выделен, вид представления отображается в поле Вид представления.

3. Щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Макрос прикрепляется к курсору и может быть размещен в любом месте (один щелчок мышью).

!!! info "Для сведения:"

    Если макрос окна содержит устройства со структурными идентификаторами, которых еще нет в проекте, новые структурные идентификаторы вставляются в соответствии с настройками пользователя для идентификатора (Опция Сортировать новые идентификаторы по алфавиту или Вставить новые идентификаторы в конце) в таблицу идентификаторов.

!!! info "Для сведения:"

    Открывается диалоговое окно Выбрать структуру, позволяющее отсортировать в имеющуюся структуру все технологические контуры / функции ТК / резервуары и т. д. макроса окна.

4. В поле Источник выделите сегмент.
5. В поле Проект выделите сегмент, под которым сегмент должен отсортировываться из источника.
6. Перетащите сегмент из поля Источник в поле Проект при помощи клавиши со стрелкой или мыши.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Если окно макроса содержит ОУ, и в Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация (online) выбран параметр С запросом, на экране отображается диалоговое окно Режим вставки.

8. Для нумерации устройств выберите в диалоговом окне Режим вставки настройку нумерации.
9. Щелкните по кнопке ++OK++.

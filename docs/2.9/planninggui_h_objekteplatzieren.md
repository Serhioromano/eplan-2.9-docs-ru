# Размещение шаблонов функций и адресов ПЛК объектов планирования

Шаблоны функции и адреса ПЛК, присвоенные объекту планирования, можно размещать неограниченное число раз на странице проекта из навигатора предварительного планирования. При этом не имеет значения, размещены ли уже эти объекты или нет и имеют ли они различный вид представления. При размещении необходимо указать вид представления. Сгенерированный вид представления не зависит от типа страницы, на которой размещен объект.

При многократном выборе размещение возможно только в том случае, если выделены однотипные объекты.

Условия:

* Вы открыли проект.
* Открыт навигатор предварительного планирования.
* Вы открыли страницу проекта.

Ниже описывается размещение с видом представления "Многополюсн.". Размещение в других видах представления осуществляется аналогичным образом.

### Разместить шаблоны функций

При размещении объектов планирования размещаются все шаблоны функции, сохраненные на объектах планирования. Эти шаблоны могут быть записаны непосредственно на объекте планирования или могут исходить от сохраненного там изделия.

При размещении отдельных шаблонов функции находятся и размещаются подходящие функции.

1. Выделите объект планирования или отдельные шаблоны функции в навигаторе предварительного планирования.
2. Выберите пункт всплывающего меню Разместить.

!!! info "Для сведения:"

    Функции, найденные для шаблонов, прикрепляются к курсору и могут быть размещены в порядке очереди.

!!! info "Для сведения:"

    Если для шаблона не найдена функция, открывается выбор символа.

3. Разместите функции одну за другой на схеме соединений.

!!! info "Для сведения:"

    Образованные при размещении функции соединяются с объектом планирования, из которого они были размещены.

### Разместить адреса ПЛК

При размещении адресов ПЛК соответствующее определение функции определяется из свойств Тип данных и Направление. Размещаются символы выводов устройства ПЛК, подходящие к определениям функций. При этом используются определения функций с ***одним*** выводом устройства.

1. Выделите один или несколько адресов ПЛК в навигаторе предварительного планирования.
2. Выберите пункт всплывающего меню Разместить.

!!! info "Для сведения:"

    Функции, найденные для адресов ПЛК, прикрепляются к курсору, и могут быть размещены в порядке очереди.

!!! info "Для сведения:"

    Если для адреса ПЛК не найдена функция, открывается выбор символа.

3. Разместите функции одну за другой на схеме соединений.

!!! info "Для сведения:"

    Образованные при размещении функции соединяются с объектом планирования, из которого они были размещены.

!!! tip "Совет:"

    Объекты планирования, шаблоны функций и адреса ПЛК можно разместить с помощью перетаскивания, перетащив выделенный объект с помощью мыши из навигатора на открытую страницу в Графическом редакторе. При этом в зависимости от типа страницы определяется вид представления функции.

### Разместить макросы

При размещении макросов макрос изделия находится в выбранном виде представления и размещается.

1. Выделите объект планирования в навигаторе предварительного планирования.
2. Выберите пункт всплывающего меню Разместить.

!!! info "Для сведения:"

    Откроется диалоговое окно Разместить устройство. Предварительно установлена опция Макрос изделия.

3. Выберите из раскрывающегося списка Вид представления пункт "Многополюсный" и щелкните кнопку ++OK++, чтобы разместить макрос на многополюсной странице схемы соединений.

!!! info "Для сведения:"

    Символ макроса изделия, сохраненного на объекте планирования, прикрепляется к курсору.

4. Разместите макрос на схеме соединений.

!!! info "Для сведения:"

    Образованные при размещении функции соединяются с объектом планирования, из которого они были размещены.

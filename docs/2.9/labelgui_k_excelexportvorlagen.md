# Шаблон Excel для экспорта данных изготовления / маркировки

Для прямой привязки к Excel можно создать шаблоны, с чьей помощью обрабатываются данные в Excel. Пример шаблона Excel:

!!! example "Пример:"

Отдельные столбцы заполняются в соответствии с выбранными на вкладках Верхний колонтитул, Этикетка и Нижний колонтитул элементами формата с данными. При этом действительны следующие присвоения:
#H# означает элементы формата вкладки Верхний колонтитул
### означает элементы формата вкладки Этикетка
#F# означает элементы формата вкладки Нижний колонтитул.

Порядок отображения данных в Excel определяется настройкой в схеме. При этом элементы формата анализируются сверху вниз и слева направо.

Например, если выбран элемент формата для верхнего колонтитула, запись "#H#" должна быть в шаблоне ровно 1 раз (пример на рисунке вверху).

После успешного завершения экспорта все столбцы форматируются по ширине.

!!! note "Замечание:"

    Элемент формата 'Разбивка на странице при смене ОУ' (в ***Нижнем колонтитуле***) вызывает разрыв страницы при каждой смене ОУ (например, во время смены ОУ клеммника при выводе спецификаций клеммника) и записывает данное ОУ на вкладку отдельного рабочего листа.

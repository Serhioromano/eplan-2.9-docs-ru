# Вкладка Медный жгут

Вы открыли проект. Проект содержит как минимум одно пространство листа. Пространство листа содержит один медный жгут. Открыт навигатор пространства листа. Выделите в навигаторе пространства листа или в трехмерной области отображения медный жгут. Выберите пункт всплывающего меню Свойства.

На этой вкладке указываются свойства выделенного медного жгута.

Обзор основных элементов диалогового окна:

### Имя

Указанное здесь имя отображается в навигаторе пространства листа.

### Структурный идентификатор

В данном поле отображается полное ОУ медного жгута (т. е. структурный идентификатор текущего окружения + видимое ОУ); щелкните по кнопке ++...++, чтобы открыть диалоговое окно Структурный идентификатор медного жгута и выбрать, например, другое место установки.

### Описание

В этом поле медному жгуту может быть подчинен описательный текст, который, например, выводится в отчете.

# Создать и обработать клеммы

Клеммы можно создавать либо в графическом редакторе, либо в навигаторе клеммников или устройств. Через графический редактор клеммы можно вставить в схему соединений и изменить их свойства. В навигаторе клеммников или в навигаторе устройств у вас есть возможность предварительно определять клеммы с определенными свойствами, чтобы позднее выполнить размещение клемм.

Вы можете сначала создать клеммы без предварительного определения, а позднее обработать их свойства. Однако, Вы можете также устанавливать свойства клемм сразу при их создании. С помощью определений функций Вы можете создавать клеммы с предварительно определенными свойствами.

Во вкладке [Клемма](devicetaggui_r_klemmen.md) диалогового окна Свойства выполняется обработка свойств клеммы.

### Создание клемм

1. Выберите пункты меню Данные проекта > Клеммники > Навигатор.
2. Отметьте определение клеммника или клеммы под клеммником.
3. Выберите пункт всплывающего меню Генерировать клемму.
4. В диалоговом окне Свойства ++...++ на вкладке Клемма введите необходимые данные для новой клеммы, например обозначение клеммы и обозначение вывода устройства.
5. Снимите флажок Главная клемма, если клемма должна рассматриваться как вспомогательная.
6. Щелкните по кнопке ++OK++.

!!! tip "Совет:"

    * Через всплывающее меню > Новые функции можно [автоматически сгенерировать несколько функций посредством схемы нумерации](navigatorgui_h_mehrerfunktionerzeugen.md). Таким способом можно одновременно создать несколько клемм с одинаковым определением функции.
    * Через Всплывающее меню > Новые клеммы (устройства) можно одновременно создать несколько клемм путем выбора какого-либо изделия. В данном случае клеммы могут иметь различные определения функций, в зависимости от сохраненных на изделии шаблонов функций.

### Создание клемм с предварительно определенными свойствами

1. Выберите пункты меню Данные проекта > Клеммники > Навигатор.
2. Выделите ОУ или определение клеммника, а затем выберите пункт всплывающего меню Создать.
3. В диалоговом окне Определения функций в поле Выбор, перемещаясь по древовидной структуре, отметьте необходимую клемму.
4. Щелкните по кнопке ++OK++.
5. Введите в диалоговом окне Свойства ++...++ на вкладке Клемма нужные данные для новой клеммы.
6. Щелкните по кнопке ++OK++.

!!! tip "Совет:"

    Вы можете копировать и размещать клеммы с помощью перетаскивания мышью, для этого в пределах вашего проекта перетаскивайте отмеченные в навигаторе клеммы на открытую страницу в графическом редакторе.

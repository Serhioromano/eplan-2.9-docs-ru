# Навигатор устройств

В навигаторе устройств в логическом порядке представлены данные проекта:

* Для ускорения поиска данные проекта можно фильтровать.
* Вы можете настроить различные виды для данных проекта.
* Можно определить, размещены функции или нет.
* Можно добавить, обработать, удалить или разместить функции

# Определить свойства блока

Свойства блока можно определить как в функции, так и в символах или в проекте. Далее описывается определение свойств блока в функции; предварительное определение свойств блока в символе или проекте выполняется аналогичным образом. Единственное отличие для проектов состоит в том, что для разных категорий функций существует по одному (индексированному) свойству формата свойства блока: формат (<Категория функции>) [n].

Условия:

* Вы открыли проект.
* Вы открыли диалоговое окно свойств для функции (например, дважды щелкнув условное обозначение в графическом редакторе).

1. Выберите в диалоговом окне Свойства ++...++ вкладку <Категория функции>.
2. Щелкните на панели инструментов над полем Свойства по ![](images/all_new_as.png){: .ui-icon } (Создать).
3. В диалоговом окне Выбор свойства выберите свойство формата и щелкните по ++OK++.

!!! tip "Совет:"

    Одновременно можно выбрать и соответствующее свойство блока для просмотра в графическом редакторе. Выделить несколько свойств одновременно можно, удерживая нажатой клавишу ++Ctrl++.

4. Выберите на вкладке <Категория функции> диалогового окна "Cвойства" свойство Свойство блока: формат [n] и щелкните в столбце Значение по ++...++.
5. В диалоговом окне Формат в списке Доступные элементы формата выберите группу свойств и щелкните по знаку "плюс" для открытия групп свойств следующего уровня.

!!! info "Для сведения:"

    Если вы из объекта хотите выбрать косвенные свойства других объектов, то кнопка ![](images/all_stairup_as.png){: .ui-icon } (Следующ. степень) активна.

6. Чтобы выбрать косвенные свойства, щелкните по ![](images/all_stairup_as.png){: .ui-icon } (Следующая ступень).

!!! info "Для сведения:"

    В поле Доступные элементы формата снова отображается первоначальная группировка свойств – эти свойства являются (косвенными) свойствами дополнительного объекта!

7. Выделите требуемую группу форматов и нажмите кнопку ![](images/all_arrowright_as.png){: .ui-icon } (Переместить вправо).

!!! info "Для сведения:"

    Кнопка ![](images/all_arrowright_as.png){: .ui-icon } (Переместить вправо) активна только в том случае, если можно выбрать и группу форматов.

8. В диалоговом окне Формат: Свойство блока выберите определенное свойство и установите для него форматирование.
9. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В диалоговом окне Формат в поле Выбранные элементы формата выбранные элементы формата отображаются в виде таблицы с их обозначениями, символическим представлением (для предварительного просмотра) и установленными значениями. Последовательность, в которой свойства находятся в списке, определяет их последовательности отображения при выводе свойств блока.

!!! info "Для сведения:"

    При выборе косвенных свойств кнопка ![](images/all_stairdown_as.png){: .ui-icon } (Предыдущая ступень) активна. При помощи этой кнопки можно снова перейти назад до выходного объекта.

10. Для определения других элементов формата повторите шаги 5 - 7.
11. При необходимости при помощи клавиш управления курсором на панели инструментов измените последовательность элементов формата.
12. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Свойство блока определено. В диалоговом окне свойств для соответствующего свойства формата будет заполнено поле Значение с предварительным просмотром установленного формата.

!!! tip "Совет:"

    при помощи функций Копировать и Вставить установленный формат можно перенести и на другие свойства формата.

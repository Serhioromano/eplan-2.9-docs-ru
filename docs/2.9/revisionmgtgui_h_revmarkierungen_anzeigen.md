# Отобразить и скрыть маркеры ревизий

Маркеры ревизии можно показать или скрыть по своему усмотрению. Это не зависит от того, созданы они с помощью отслеживания изменений или сравнения свойств проектов.

* После сравнения свойств проектов маркеры ревизии в проекте отображаются автоматически.
* При генерировании первой ревизии проекта маркеры ревизии включаются автоматически. Если затем отключить отображение маркеров ревизии через пункты меню Вид > Маркеры ревизий, эта настройка сохраняется при генерировании последующих ревизий.

Условие:

Вы создали маркеры ревизии в проекте.

1. Выберите пункты меню Вид > Маркеры ревизий.

!!! info "Для сведения:"

    Маркеры ревизии отображаются в проекте.

2. Выберите пункты меню Вид > Маркеры ревизии повторно.

!!! info "Для сведения:"

    Маркеры ревизии скрываются.

3. При необходимости обновите отображение с помощью пунктов меню Вид > Начертить заново.

!!! note "Замечание:"

    Если параметр Вид > Невидимые элементы активирован, то при скрытии маркеров ревизии ***маркер удаления*** остается видимым. Это объясняется тем, что маркеры удаления (в отличие от стандартных маркеров ревизий) скрываются, а их слои настраиваются невидимыми.

### Поиск изменений ревизии в свойствах

Измененные объекты (в том числе такие, в которых были изменены невидимые свойства) отображаются на странице с помощью графического маркера.

если в ревизии переместить курсор на измененный объект, на просмотр выводится всплывающая подсказка, в которой содержится информация по ревизии.

Можно выполнить поиск изменений в свойствах объектов:

1. Выберите следующие пункты меню: Поиск > Поиск
2. В диалоговом окне Искать введите в поле Поиск текст, который вы ввели для обозначения изменений ревизии в диалоговом окне Сравнить свойства проектов.
3. Установите в групповом поле Поиск в флажок Все свойства всех условных обозначений.
4. Укажите в групповом поле Поиск на, будете ли вы искать во всем проекте или только на текущей странице, и установите флажок Логич. страницы.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Результат поиска отображается в диалоговом окне Результаты поиска.

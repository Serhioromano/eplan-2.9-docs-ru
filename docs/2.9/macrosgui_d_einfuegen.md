# Диалоговое окно Вставить

По пути Параметры > Настройки > Пользователь > Представление > Общ. был снят флаг Использовать только один буфер обмена******. Вы открыли проект. В буфер обмена скопировано как минимум два объекта схемы соединений или две страницы.

* Обработать > Вставить.
* Всплывающее меню > Вставить.

В этом диалоговом окне выводятся все объекты (или страницы), находящиеся в буфере обмена. Выберите объект (или страницу), который необходимо вставить.

!!! note "Замечание:"

    Обратите внимание, что для копирования и вставки объектов и страниц используются ***разные*** буферы обмена! Если курсор находится в навигаторе страниц, при вставке EPLaN открывает диалоговое окно со скопированными страницами. Если же курсор находится на странице проекта, с помощью диалогового окна Вставить можно вставить скопированные объекты схемы соединений.

Доступны следующие элементы диалогового окна:

### Таблица

В этом поле выводятся все объекты, которые были сохранены в буфер обмена с помощью копирования или вырезания.

* Описание: В этом столбце выводится описание, которое было введено в диалоговом окне Буфер обмена.
* Тип: В этом столбце выводится тип объекта.
* Дата / время: В этом столбце указывается, когда объект был сохранен в буфер обмена.
* ![](images/all_delete_as.png){: .ui-icon }: Удаление выделенного объекта из буфера обмена.

### Предв. просмотр

Если этот флаг установлен, то в расположенном ниже поле для объекта, выделенного в таблице, выводится предварительный просмотр.

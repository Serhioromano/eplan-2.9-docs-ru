# Экспортировать списки присвоений

Экспорт списков присвоений всегда связан с ЦПУ. Принадлежность карты ПЛК к ЦПУ задается через [Имя ЦПУ](devicetaggui_r_spsstruktur.md). ЦПУ однозначно идентифицируется путем ввода имени ЦПУ в формате [Проект конфигураций].[Имя станции].[Идентификатор ЦПУ].

!!! note "Замечание:"

    Вы можете экспортировать всякий раз только отдельный список для каждого процессора (ЦПУ). Поэтому в диалоговом окне Адреса / Списки присвоений настройте фильтр таким образом, чтобы отображались только адреса ПЛК ***одного*** ЦПУ. Схема, задающая формат адресов ПЛК, находится на основе ЦПУ, содержащихся в выборе, и здесь ее нельзя изменить. Если ЦПУ не присвоена схема, используется схема, заданная в настройках проекта.

1. Выберите следующие пункты меню: Данные проекта > ПЛК > Адреса / Списки присвоений > [Дополнительно] > Экспортировать список присвоений

!!! info "Для сведения:"

    В диалоговом окне [Экспортировать список присвоений](plcgui_d_exportzuli.md) в поле Специфические для ПЛК настройки отображается схема, задающая формат адресов ПЛК. Над полем дополнительно отображаются проект конфигураций и имя рабочей станции, которым присвоено ЦПУ.

2. В поле Язык выберите соответствующий списку присвоений язык.

3. В поле Имя файла укажите имя, под которым сохраняется экспортированный файл.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Адреса из списка присвоений записываются в файл экспорта. При этом все карты ПЛК учитываются с одинаковым именем ЦПУ.

!!! tip "Совет:"

    При помощи пункта всплывающего меню Создать фильтр для ЦПУ в таблице вы можете автоматически сгенерировать из данных выделенной строки фильтр для отображения адресов ПЛК. При этом в качестве критерия фильтра используются значения свойств Проект конфигураций (косвен.), Имя рабочей станции ПЛК (косвен.) и ЦПУ (косвен.).

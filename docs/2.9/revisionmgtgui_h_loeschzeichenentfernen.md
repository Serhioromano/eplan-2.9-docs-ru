# Удалить маркеры удаления в проектах со сравнением свойств

Маркеры удаления создаются при сравнении свойств проектов, если в диалоговом окне Сравнить свойства проектов установлен флажок Сгенерировать маркер удаления для удаленных объектов.

При сравнении свойств проектов изменения в проекте не регистрируются постоянно. Маркеры ревизий создаются однократно и остаются в этой форме до следующего сравнения. По этой причине можно окончательно удалять из проекта удаленный объект одновременно с удалением маркера ревизии и маркера удаления (например, через пункт всплывающего меню Удалить).

Маркер ревизии и маркер удаления отображаются вновь только после повторного сравнения свойств с неизмененным исходным проектом.

Условие:

Вы открыли проект, содержащий маркеры удаления, и находитесь на странице схем соединений в графическом редакторе.

1. Выделите маркер удаления и выберите пункт всплывающего меню Удалить.

!!! info "Для сведения:"

    Сведения об удаленном объекте будут окончательно удалены из проекта.

!!! tip "Совет:"

    Дополнительно можно удалять все имеющиеся в проекте маркеры ревизий и маркеры удаления, которые были созданы при сравнении свойств проектов. Для этого выберите Сервисные программы > Управление ревизиями > Сравнение свойств проектов > Удалить ревизию.

    При ***сравнении свойств проектов*** маркеры удаления и удаленные страницы выступают в роли маркеров ревизии. Чтобы их удалить, в диалоговом окне Удалить ревизию установите флажок Маркеры ревизий.

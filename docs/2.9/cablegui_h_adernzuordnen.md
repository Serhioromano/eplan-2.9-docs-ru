# Присвоить кабельные соединения вручную

Присвоение кабельных соединений соединениям изделия кабеля можно изменить вручную в диалоговом окне Обработать кабель. Там отображаются имеющиеся в изделии кабеля шаблоны функций, определенные в проекте соединения кабеля, а также их присвоение. Чтобы в диалоговом окне Обработать кабель отображались шаблоны функций, выделенному кабелю должно быть присвоено изделие.

Размещенное или неразмещенное кабельное соединение берет на себя шаблон функции (т. е. изделия кабеля), если оно имеет те же идентифицирующие свойства. При этом в первую очередь рассматривается имя экранирования. Если для шаблона функции с экранированием не найдено подходящего соединения с экранированием, то ему присваивается соединение без экранирования (если остальные идентифицирующиеся свойства совпадают). Т. е. экранирования не обязательно должны отображаться в схеме соединений.

Присвоение кабельных соединений шаблонам функций осуществляется путем перемещения и обмена соединений в правой части диалогового окна. Последовательность шаблонов функций с левой стороны диалогового окна всегда остается неизменной.

### Присвоить соединения свободным шаблонам функций

Соединения можно смещать только на свободные шаблоны функций. Другие соединения при этом не изменяются.

Соединения с типом потенциала РЕ / PEN и SH нельзя переместить на шаблоны функций с другими типами потенциалов. Смещение в другое экранирование тоже невозможно. Соединения без экранирований могут быть, тем не менее, присвоены шаблону функции с экранированием.

1. Выберите пункты меню Данные проекта > Кабель > Навигатор.
2. Выделите в диалоговом окне Кабели — <Имя проекта> определение кабеля, (размещенное) кабельное соединение или экранирование.
3. Выберите пункт всплывающего меню Обработать.

!!! info "Для сведения:"

    В диалоговом окне Обработать кабель справа отображаются все соединения соответствующего кабеля.

!!! info "Для сведения:"

    В левой части диалогового окна отображаются шаблоны функций соответствующего изделия кабеля.

3. В таблице Соединения выделите соединение и переместите его с помощью кнопок управления курсором в позицию со свободным шаблоном функции.

!!! info "Для сведения:"

    Соединение копирует данные шаблона функции.

!!! info "Для сведения:"

    Данные точек определения соединений соответственно обновляются. Имеющиеся данные перезаписываются. Новые данные записываются на точку определения соединения, только если они не записаны на соответствующей линии определения кабеля / экранировании.

!!! tip "Совет:"

    Чтобы открыть диалоговое окно Обработать кабель, вы можете также выделить линию определения кабеля, экранирование или точку обозначения соединения и выбрать пункт меню Данные проекта > Кабели > Обработать.

### Заменить соединения

Соединения можно поменять только в том случае, если это позволяет тип потенциала. Т. е. соединения РЕ / PEN и SH можно поменять только друг с другом, замена с другими соединениями не допускается. Поменять нельзя и соединения, расположенные в разных экранированиях.

1. Выберите пункты меню Данные проекта > Кабель > Навигатор.
2. Выделите в диалоговом окне Кабели — <Имя проекта> определение кабеля, (размещенное) кабельное соединение или экранирование.
3. Выберите пункт всплывающего меню Обработать.

!!! info "Для сведения:"

    В диалоговом окне Обработать кабели отображаются все соединения соответствующего кабеля.

!!! info "Для сведения:"

    В левой части диалогового окна отображаются шаблоны функций соответствующего изделия кабеля.

3. В таблице Соединения выберите два соединения и нажмите ![](images/all_arrowswap_as.png){: .ui-icon } (Поменять).

!!! info "Для сведения:"

    Строки таблицы поменяются местами.

!!! info "Для сведения:"

    На точках определения соединений меняются значения свойств Парный индекс, Цвет / номер и Поперечное сечение / диаметр.

!!! tip "Совет:"

    Чтобы открыть диалоговое окно Обработать кабель, вы можете также выделить линию определения кабеля, экранирование или точку обозначения соединения и выбрать пункт меню Данные проекта > Кабели > Обработать.

!!! note "Замечание:"

    При замене двух экранирований в диалоговом окне Обработать кабель это оказывает косвенное воздействие на экранированные кабельные соединения. При изменении для экранированных кабельных соединений имени их экранирования они больше не подходят для своих шаблонов функций. Полученные изменения сразу же отображаются.

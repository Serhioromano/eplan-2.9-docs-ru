# Нумерация в случае неразмещенного главного устройства.

В случае нумерации в автономном режиме можно использовать формат нумерации, который содержит элементы формата "Столбец" и/или "Строка". Если главная функция размещена, часть размещения ОУ по умолчанию образуется на основании информации размещения соответствующей главной функции.

Если главная функция не размещена, она не содержит информацию о размещении, например:

* ***Клеммники:*** Главная функция "Определение клеммника" не размещена, вспомогательные функции "Клеммы" размещены.
* ***Штекеры:*** Главная функция "Определение штекера" не размещена, вспомогательные функции "Штырь штекера и штекерное гнездо" размещены.

Если в этом случае используется формат нумерации с элементами "Столбец" и/или "Строка", часть размещения ОУ образуется на основании информации о размещении графически первой размещенной клеммы или графически первого размещенного штекера.

# Меню справки

Меню Справка содержит следующие пункты:

Пункт меню |  Значение
---|---
Содержание |  Открывается окно справки с открытой вкладкой Содержание.
Информационный портал EPLAN |  Запускается интернет-страница EPLAN с информацией о продуктах EPLAN (справочные системы, руководства и т. д.). С помощью полнотекстового поиска по всем сохраненным техническим публикациям здесь можно найти требуемые понятия.
Создать запрос на поддержку EPLAN |  Запускает интернет-страницу EPLAN с формой для запроса в службу поддержки.
EPLAN Download Manager |  Запускает EPLAN Download Manager и предоставляет возможность установить компоненты программы для EPLAN.
Программа повышения качества работы, ориентированная на клиента |  Открывает диалоговое окно Программа повышения качества работы, ориентированная на клиента здесь можно принять решение о принятии участия в этой программе.
Информация |  Открывает диалоговое окно Приложение EPLAN, в котором содержится информация по следующим пунктам:

* Продукт
* Объем лицензии
* Авторское право / Торговая марка
* Информация о системе

# Диалоговое окно Выбрать переменную пути

Открыто диалоговое окно, в котором с помощью кнопки ++...++ можно выбрать файл и / или каталог. В соответствующем поле выберите пункт всплывающего меню Вставить переменную пути.

В этом диалоговом окне выберите переменную пути, которая должна быть вставлена в текущее поле. Выбираемые переменные зависят от поля. При этом обязательно обращайте внимание на текущую позицию указателя мыши в соответствующем поле.

Вкладка Переменные программы

На этой вкладке перечислены переменные пути и их текущие значения, которые относятся к текущему приложению EPLAN (каталог программы, версия, настройки каталога и т. д.).

Переменная пути |  Значение
---|---
$(BIN) |  При установке созданного каталога программ; содержит библиотеки программ (*.dll) отдельных модулей.
$(CFG) |  При установке созданного каталога конфигураций; содержит файлы xml отдельных модулей.
$(CFG_COMPANY) |  При установке сгенерированного каталога конфигурации; содержит фирменные настройки.
$(CFG_STATION) |  При установке сгенерированного каталога конфигурации; содержит настройки станции.
$(CFG_USER) |  При установке сгенерированного каталога конфигурации; содержит настройки пользователя.
$(EPLAN) |  Каталог более высокого уровня иерархии, созданный при установке.
$(EPLAN_DATA) |  Вышестоящий каталог lдля основных данных, созданный при установке.
$(LOCALDATE) |  Текущая дата в формате ДДММГГГГ.
$(LOCALTIME) |  Текущее время в формате ЧЧММСС.
$(MD_DOCUMENTS) ... $(MD_XML) |  Каталоги, заданные в пунктах меню Параметры > Настройки > Пользователь > Управление > Каталоги, например, для документов, XML-файлов и т. д.
$(TMP) |  Каталог, используемый операционной системой для временных файлов.

Вкладка Переменные проекта

На этой вкладке отображаются специфические для проекта переменные пути (имя проекта, каталог проекта, свойства проекта и т. д.), а также их текущие значения. Вкладка доступна только там, где целесообразно использовать специфические для проекта переменные пути.

Переменная пути |  Значение
---|---
$(DOC) |  Проектный каталог для документов.
$(IMG) |  Проектный каталог для графики.
$(P) |  Полный каталог проекта для текущего выбранного проекта.
$(PROJECTNAME) |  Имя проекта текущего выбранного проекта без пути к каталогу и без расширения имени файла.
$(PROJECTPATH) |  Полный каталог проекта для текущего выбранного проекта.
$(PROJPROP_\<ID\>) |  Свойство проекта; для обозначения одной из таких переменных пути в имени приводится идентификатор соответствующего свойства.

* * *

Вкладка Переменные окружения

На этой вкладке перечислены переменные окружения и их текущие значения из панели управления, которые можно использовать как переменные пути.

Переменная пути |  Значение
---|---
$(ENVVAR_<Имя переменной>) |  Переменная окружения из панели управления.

* * *

!!! tip "Совет:"

    * Используя переменные пути для текущей даты и времени $(LOCALDATE) и $(LOCALTIME), например, при автоматической обработке, можно выводить файлы в разные автоматически созданные каталоги, не создавая каталоги вручную.
    * Переменные пути для свойств проекта $(PROJPROP_\<ID\>) дают доступ к другим значениям проекта при использовании переменных пути.
    * С помощью этих специфических для проекта переменных пути и окружения $(ENVVAR_<Имя переменной>) можно создавать и использовать собственные переменные пути. К примеру, с помощью EPLAN API при автоматической обработке можно в процессе работы присвоить и удалить определенные значения переменных среды.

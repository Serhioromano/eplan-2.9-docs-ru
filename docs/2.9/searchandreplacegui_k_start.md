# Найти и заменить

Во время проектирования и при сравнении ревизий часто требуется целевой поиск информации в проекте, чтобы дополнительно обработать их. Возможно следующее:

* Можно искать тексты, обозначения устройств, символы, кабели, кабельные соединения, потенциалы, макросы, основные данные, данные заголовка страницы, данные заголовка проекта, свойства (данные форматирования) и т. д. по всему проекту или на определенных страницах.
* При поиске можно использовать заполнители.
* Структурные идентификаторы можно учитывать при поиске.
* При поиске также учитываются невидимые объекты и уровни.
* Результаты поиска записываются в списки результатов. С помощью этих списков возможны следующие действия: Так, например, вы можете открывать диалоговое окно свойств для одного выделенного объекта и обрабатывать его свойства.
* Можно синхронизировать отображения между открытыми навигаторами.

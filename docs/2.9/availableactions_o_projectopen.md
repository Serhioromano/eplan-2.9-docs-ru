# Операция: ProjectOpen

**Параметр** |  **Описание**
---|---
Project  |  Используемый проект.
OpenMode |  Задает режим обработки при открытии проектов (необязательно). Возможные значения:
STANDARD: Открытый таким образом проект допускает любую обработку. В многопользовательском режиме с таким проектом могут одновременно работать несколько пользователей.
READONLY (только чтение): Проект открывается с защитой от записи.
EXCLUSIVE: Проект открывается только для текущего пользователя. В многопользовательском режиме другие пользователи не имеют доступа к открытому таким образом проекту.

!!! example "Пример:"

    ProjectOpen
    /Project:"C:\EPLANmy own ProjectsDEMO.elk"

# Диалоговое окно Формат адреса ПЛК / Диалоговое окно Формат строки / Адрес в списке присвоений

Вы открыли проект. Вы открыли диалоговое окно [Настройки: ПЛК](plcgui_d_allgemeinespseinstellungen.md) (напр., через пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Устройство > ПЛК). В диалоговом окне Настройки: ПЛК щелкните рядом с полем Специфические для ПЛК настройки по кнопке ++...++.

* В диалоговом окне [Настройки: Специфич. для ПЛК](plcprocessorgui_d_spsspezifischeeinstellungen.md) выберите вкладку Форматы адресов. В групповом поле Форматы адресов рядом с полем для требуемого формата адреса щелкните ++...++.
* В диалоговом окне [Настройки: Специфич. для ПЛК](plcprocessorgui_d_spsspezifischeeinstellungen.md) выберите вкладку Формат списка присвоений. Щелкните рядом с полем Формат строки списка присвоений или Формат адреса списка присвоений по кнопке ++...++.

В этом диалоговом окне вы редактируете формат цифровых или аналоговых адресов ПЛК в списке присвоений, строки и адреса списков присвоений. Заголовок диалога может меняться в зависимости от элемента, для которого это диалоговое окно было вызвано, и отображает, какой формат редактируется.

Обзор основных элементов диалогового окна:

### Доступные элементы формата

В этом списке отображаются элементы, которые могут быть частями формата адреса или формата строки списков присвоений.

Для ***дискретных и аналоговых вводов и выводов*** это:

* Счетчик
* Разделитель.

Для ***Строк списков присвоений*** это:

* Адрес
* Функциональный текст: Чтобы учитывать [многосоставные и/ или многоязычные функциональные тексты](plcgui_k_zuordnungslisten.md), можно многократно вставлять этот элемент формата в формат строки списка присвоений.
* Символический адрес: Отображает вручную внесенный символический адрес (если таковой имеется), в противном случае — определенный символический адрес.
* Символический адрес (определен): Отображает символический адрес, определенный посредством отслеживания цели, т. е. подключенное ОУ. С помощью этого элемента формата при экспорте списков присвоения можно экспортировать подключенное ОУ вместе с вручную внесенным символическим адресом. Однако это значение невозможно импортировать снова.
* Символический адрес: Группа: Показывает присвоение символического адреса группе. Эта группа может использоваться для группировки диапазонов символических адресов в списке присвоений, например, входов, выходов, адресов (отказоуст.) и т. д.
* Тип данных
* Разделитель
* Пропущенный текст: Элемент формата "Пропущенный текст" требуется для импорта списков присвоения, в которых встречаются элементы, не имеющие соответствий в программе EPLAN, которые поэтому не могут быть записаны в свойство EPLAN при импорте. Соответствующие элементы строки списка присвоений игнорируются при импорте и недоступны в EPLAN.
При экспорте для каждого элемента формата "Пропущенный текст" в файле экспорта создается пустая запись.
Игнорируемые тексты всегда имеют длину "Неограниченно перемен."

Для ***Адреса списка присвоений*** это:

* Идентификатор
* Байтовый адрес
* Битовый адрес
* Разделитель.

Для форматирования элемента формата необходимо переместить его вправо в список выбранных элементов формата; многократный выбор в данном случае невозможен. Если вы переместите элемент формата, откроется диалоговое окно, в котором можно определить форматирование. Оно отличается в зависимости от элемента формата.

!!! note "Замечание:"

    Обратите внимание, что после элемента формата с переменной длиной должен быть вставлен разделитель для однозначного распознавания начала следующего элемента формата. После элементов формата с фиксированным количеством символов разделитель не требуется.

### Выбранные элементы формата

Этот список показывает, какие элементы формата использовались для построения формата строк списка присвоений. Выбранные элементы формата выводятся вместе с обозначением, символическим представлением (для предварительного просмотра) и заданным значением. С помощью кнопок, расположенных над списком, можно дополнительно изменить форматирование элемента формата, удалить элементы формата, а также изменить их последовательность.

### Предв. просмотр

Отображается формата адреса в режиме предварительного просмотра. Отдельные элементы формата выводятся в адресе ПЛК или в списке присвоений в той последовательности, в которой они стоят в списке. Значение в этом поле изменить невозможно.

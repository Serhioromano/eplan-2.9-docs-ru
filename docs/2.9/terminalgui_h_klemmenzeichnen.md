# Вставить клеммы и определения клеммников

Клеммы и определения клеммников можно определять непосредственно при вставке соответствующего символа. В навигаторе клеммников или в навигаторе устройств вам предоставляется возможность заранее определять клеммы и определения клеммников с определенными свойствами. Созданные таким образом функции вы можете в дальнейшем разместить в схеме соединений. Определения клеммников должны быть размещены, если вы хотите их копировать.

!!! tip "Совет:"

    Если при проектировании установлены требуемые для использования изделия клеммника, то клеммы можно вставлять непосредственно в качестве устройства (меню: Вставить > Устройство). Таким образом, например, можно удобно размещать отдельные символы многоуровневых клемм друг за другом. В качестве помощи для размещения отображаются определение функции и уровень соответствующей клеммы. Клавиша [N] позволяет "пролистать" функции.

### Вставка клемм

1. Выберите пункт меню Вставить > Символ.
2. В диалоговом окне Выбор символа выберите требуемую клемму.
3. Разместите клемму на странице схемы соединений.
4. В диалоговом окне Свойства ++...++ во вкладке Клемма введите данные клеммы, например видимое ОУ.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    если Вы указали уже существующее ОУ, то клемма будет назначена соответствующему клеммнику.

!!! info "Для сведения:"

    Если вы указали еще не существующее ОУ, то EPLAN создает новый клеммник. Он, однако, еще не имеет своего определения клеммника.

!!! tip "Совет:"

    При вставке клеммы рядом с символом на курсоре появится буквенное обозначение и знак вопроса (например, "-X?"), если в направлении поиска для копирования ОУ нет других клемм. Если клемму разместить, то будет сгенерирован новый клеммник.
    Если в направлении поиска для копирования ОУ имеется какая-либо клемма, то буквенное обозначение рядом с символом будет скрыто. При размещении клеммы в этой области ей копируется ОУ имеющейся клеммы, и размещаемая клемма будет присвоена имеющемуся там клеммнику.

    Если в момент вставки удерживать клавишу ++Shift++, соотношение окажется обратным: Если в направлении поиска для копирования ОУ ***нет*** клеммы, она будет присвоена последнему из использовавшихся клеммников. Если в направлении поиска имеется какая-либо клемма, то будет создан новый клеммник.

### Вставка определений клеммников

1. Выберите пункт меню Вставить > Определение клеммника.
2. Разместите определение клеммника на странице схемы соединений.
3. В диалоговом окне Свойства ++...++ во вкладке Определение клеммника введите данные клеммника.
4. Щелкните по кнопке ++OK++.

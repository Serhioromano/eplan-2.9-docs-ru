# Определить кабели и кабельные соединения вручную

Кабели определяются вручную; для этого чертятся линии определений кабелей и точки определений соединений, а затем задаются свойства кабеля. На одно ОУ требуется только по одной линии определения кабеля (но можно установить и несколько); другие соединения можно присвоить кабелю при помощи точек определения соединений.

Чтобы представить кабель в распределенном виде, начертите несколько линий определения кабелей и присвойте им одинаковое ОУ кабеля.

Кабельные соединения можно определить или на точке определения соединения, или на выводе устройства функции. Настройка на точке определения соединения имеет преимущество перед настройкой на выводах устройства.

### Определение кабельных соединений на точке определения соединения

Отдельные соединения можно присвоить кабелю. Для этого разместите точки определения соединений и присвойте им ОУ кабеля:

1. Выберите пункты меню Вставить > Точка определения соединения.
2. Разместите точку определения соединения на соединении.
3. В диалоговом окне Свойства ++...++ на вкладке Точка определения соединения в поле Видимое ОУ введите ОУ кабеля, которому нужно присвоить соединение.
4. В таблице выделите свойство Соединение: Принадлежность и выберите значение "Кабель" из раскрывающегося списка.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Кабельное соединение присваивается кабелю.

### Определение кабельных соединений на выводе устройства

1. Выделите в графическом редакторе любое условное обозначение и выберите пункты меню Обработать > Свойства.
2. Выберите в диалоговом окне Свойства ++...++ вкладку Данные символа / функции, и щелкните по ++Логика++.
3. В диалоговом окне Логическая схема выводов устройства установите флажок Вывод кабеля для нужных выводов устройства.
4. Дополнительно можно изменить свойства Тип вывода устройства, Тип потенциала и другие свойства выводов устройств.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Данные вывода устройства сохраняются на функции.

!!! info "Для сведения:"

    Подключенные к функции соединения получают свойства соответствующего вывода устройства и становятся кабельными соединениями.

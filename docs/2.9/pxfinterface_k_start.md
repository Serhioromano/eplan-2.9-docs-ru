# Внешняя обработка

Так называемая "внешняя обработка" служит для того, чтобы выводить данные из EPLAN, производить их внешнюю обработку и затем опять считывать. Внутри программы эта функциональность реализуется за счет экспорта выделенных данных в XML-совместимый формат. Кроме того, можно менять все графические и электротехнические данные.

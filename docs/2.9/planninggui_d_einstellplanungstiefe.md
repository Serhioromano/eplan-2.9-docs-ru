# Диалоговое окно Настройки: Глубина предварительного планирования

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Предварительное планирование > Глубина предварительного планирования.

В этом диалоговом окне определите контрольные прогоны для различных видов сегментов. Для этого задайте обязательные свойства, т. е. свойства, которые обязательно должны быть заполнены для данных сегментов. Обязательными могут быть как свойства EPLAN, так и свойства, определенные пользователем.

Для обязательных свойств проверяется, внесено ли значение, активировано ли свойство и не выбрана ли в раскрывающемся списке запись "Неопред.".

Для выполнения контрольных прогонов, в схеме контрольных прогонов необходимо задать для нужных контрольных прогонов вид проверки.

Доступны следующие элементы диалогового окна:

В таблице имеются следующие столбцы:

* Строка: непрерывный номер строки.
* Имя: Укажите здесь имя контрольного прогона, например нужную ступень предварительного планирования. Это имя отображается в управлении сообщениями как текст соответствующего сообщения контрольного прогона.
* Определение сегмента: Выберите в раскрывающемся списке, для какого вида сегментов вы хотите задать обязательные свойства. При этом при выборе отдельного определения сегмента проверяются только сегменты одного вида (например, только измерительные контуры), при выборе определения базового сегмента — сегменты определенной категории (например, все технологические контуры, к которым относятся как измерительные контуры, так и исполнительные механизмы), а при выборе записи "Все" — все сегменты.
* Обязательное свойство 1... 10: Для каждого контрольного прогона можно задать до десяти обязательных свойств. Если щелкнуть в ячейке по клавише ++...++, откроется диалоговое окно Выбор свойств и в нем можно будет выбрать нужное свойство.

# Предварительная настройка для изделий на электротехнических соединениях

В настройках проекта для соединений на вкладке Электротехника можно задать предварительные настройки свойств электротехнических соединений для всего проекта. Теперь в новом поле Номер изделия всем отдельным электротехническим соединениям можно присваивать значение по умолчанию для изделия. (Путь меню: Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Свойства.)

![](images/benefit.png)Эффект:

Вам не нужно присваивать изделие каждому соединению вручную. Это экономит время во время проектирования, особенно если вы часто используете одни и те же изделия соединения. В связи с тем, что для всех соединений назначено изделие, при маршрутизации возможен расчет степени заполнения.

Введите здесь номер изделия соединения или, нажав кнопку ++...++, откройте выбор изделия и выберите там изделие. Номер изделия переносится на вкладку Изделие диалогового окна 'Свойства' соединений и используется как значение по умолчанию для новых соединений. Рекомендуется использовать изделие без указания длины и цены. Для вычисления степени заполнения при маршрутизации необходимо ввести на изделии внешний диаметр.

Настройки на вкладке Электротехника теперь влияют только на отдельные соединения (на соединении для свойства Соединение: Принадлежность установлено значение "Отдельное соединение"), но не на кабельные соединения. Значения по умолчанию для кабельных соединений (например, поперечное сечение) заимствуются из изделия кабеля.

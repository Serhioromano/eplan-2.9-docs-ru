# Обработать свойства точек определения контуров

Логика контуров определяется в форме так называемых точек определения контура и доступна только для контуров типа 'Выдавливание контура'. Для этого в редакторе контуров в пунктах меню Вставить > Логика > Логический элемент доступны специальные пункты меню, которые запускают диалог для размещения и нумерации точки определения контуров.

При вставке точки определения контуров нумеруются в зависимости от типа и содержат уникальный счетчик. Точки определения контуров для логических элементов Направляющая рейка, Направляющая рейка с выравниванием, Монтажная поверхность, Монтажная поверхность ЧУ и Размер поля встречаются только попарно и обладают дополнительным подсчетчиком, который в сочетании со счетчиком обеспечивает однозначность. Подсчетчик может принимать только значения "1" или "2" (например, счетчик = 1, подсчетчик = 1.1 и 1.2).

Условия:

* Вы открыли проект.
* Контур, который необходимо обработать, открыт в редакторе контуров (Сервисные программы > Основные данные > Контур (Выдавливание) > Открыть > [Открыть]), добавлена минимум одна точка определения контуров.

1. Дважды щелкните по логическому элементу.
2. В диалоговом окне Точка определения контура: ++...++ выберите вкладку Данные, если необходимо обработать счетчик и / или подсчетчик логического элемента.
3. Выберите вкладку Отображение для обработки настроек отображения логического элемента (например, настройки Размер шрифта, Вид шрифта и т. д.).
4. Щелкните по кнопке ++OK++.

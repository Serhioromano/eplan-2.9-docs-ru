# Диалоговое окно Обработать штекер

Вы открыли проект.

* Вы выделили в Графическом редакторе определение штекера или контакт штекера. Данные проекта > Штекеры > Обработать.
* Данные проекта > Штекеры > Навигатор. В диалоговом окне навигатора Штекер — <Имя проекта> выделите определение штекера или контакт штекера и выберите пункт всплывающего меню Обработать.

В этом диалоговом окне отображаются все настройки и данные выбранного штекера. Эти настройки сохраняются в схеме. Если штекер состоит из нескольких определений (например, при раздельном представлении штырей и гнезд), то в таком случае в этом диалоговом окне обрабатывается не весь штекер, а только выделенное в данный момент ***определение штекера***.

Если штекер содержит контакты штекера, состоящие из гнезда и штыря, то на каждую пару штырь-гнездо отображается по две строки и соединение между ними представляется графически. Через всплывающее меню контакты штекера можно пронумеровать. В дополнение к свойствам контактов штекера также можно отображать свойства подключенных соединений.

!!! note "Замечание:"

    Если диалоговое окно вызывается из выделенного поля ***Штырь***, то в таблице можно обрабатывать поля для штырей. Если диалоговое окно вызывается из выделенного поля ***Гнездо***, то в таблице можно обрабатывать поля для гнезд. Графика в столбце Соединение наглядно иллюстрирует это поведение и показывает, на какой стороне штекера находятся штыри или гнезда.

Обзор основных элементов диалогового окна:

### Схема

Здесь из раскрывающегося списка выберите схему, которая определяет настройки. С помощью кнопки ++...++ откройте диалоговое окно Настройки: Конфигурация столбцов. В нем может быть обработана существующая схема или создана новая схема.

Таблица

В Таблице выводятся столбцы, определенные в выбранной схеме. Здесь показываются лишь определенные данные (поля заливаются серым), остальные могут обрабатываться. Кроме того, можно отобразить следующие столбцы:

* Цель (сторона гнезд) / Цель (сторона штырей): Показывает подсоединенную цель (условное обозначение) на стороне гнезда / штырей штекерного соединения. Указывается полное обозначение устройства и обозначение вывода устройства.
* Кабель (сторона гнезд) / Кабель (сторона штырей): Показывает подсоединенный кабель на стороне гнезда / штырей штекерного соединения. Указывается полное обозначение устройства кабеля, а также цвет / номер соединения.
* Функциональный текст (цель, сторона гнезда) / Функциональный текст (цель, сторона штырей): Показывает функциональный текст подсоединенной цели (условное обозначение) на стороне гнезда / штырей штекерного соединения.
* Размещение (цель, сторона гнезд) / Размещение (цель, сторона штырей): Показывает размещение подсоединенной цели (условное обозначение) на стороне гнезда / штырей штекерного соединения. Указывается полное имя страницы схемы соединений и обозначение столбца.
* Перемычки (сторона гнезд) / Перемычки (сторона штырей): Графически представляет имеющуюся мостовую перемычку на стороне гнезда / штырей штекерного соединения.
* Обозначение контакта штекера (сторона гнезд) / Обозначение контакта штекера (сторона штырей): Показывает обозначение контакта штекера на стороне гнезда / штырей штекерного соединения.
* Тип потенциала выводов устройства (сторона гнезд) / Тип потенциала выводов устройства (сторона штырей): Показывает тип потенциала выводов устройства функции на стороне гнезда / штырей штекерного соединения, если он одинаковый для всех выводов устройства. В противном случае выводится тип потенциала "Не определен".
* Определение функций (сторона гнезд) / Определение функций (сторона штырей): Показывает обозначение контакта штекера на стороне гнезд / штырей штекерного соединения.
* Соединение: Графически представляет соединение (сторона штырей / гнезда штекера).
* Свойства блока [1] - [10]: В этих полях отображаются определенные для контакта штекера свойства блока.
* Сообщения в управлении сообщениями: Если при проектировании или при контрольном прогоне одно из соответствующих контакту штекера сообщений выдается в управлении сообщениями, это отобразится через установленный флажок.
* Номер изделия [1] - [10]: В этих полях отображаются присвоенные контакту штекера изделия. С помощью кнопки ++...++ в поле можно выбрать изделие.
* Цвет / номер соединения (сторона штырей) — Обозначение соединения (сторона гнезд): В этих полях отображаются свойства соединений, которые подключены к стороне гнезда / штырей штекерного соединения.

### Панель инструментов

Используя кнопки со стрелками панели инструментов, вы можете изменять последовательность контактов штекера в рамках штекера. При этом в каждом случае перемещается целая строка.

### [Сортировать]

Позволяет автоматически сортировать контакты текущего штекера. Предоставляются следующие возможности:

* Удалить сортировку: Удаляет идентификатор сортировки контактов штекера.
* Числовой: Сортируются все контакты штекера, обозначение которых начинается с цифры (в возрастающем порядке по величине числа). Все другие контакты штекера остаются на своих позициях.
* Алфавитно-цифров.: Контакты штекера сортируются по их обозначению (сначала в возрастающем порядке по величине числа, затем по буквам).
* По позиции: В зависимости от логики рамки контакты штекера сортируются согласно графической последовательности на схеме соединений. Неразмещенные контакты штекера отсортировываются в начало списка.
* По внешним кабелям: Контакты штекера, на которых (на внешней стороне штекера) подключается общий кабель, проходят друг рядом с другом. Облегчает вывод устройства кабеля, т. к. все кабельные соединения могут быть проложены друг за другом. Такая сортировка может быть проведена только для соединений, которые уже относятся к кабелю. При необходимости сгенерируйте кабель автоматически или определите его вручную.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Новые функции |  Открывает диалоговое окно Генерировать функции, в котором Вы можете генерировать сразу несколько функций одновременно.
Нумерация контактов штекеров |  Открывает диалоговое окно Нумерация контактов штекеров.
Свойства (общие) |  Открывает диалоговое окно Свойства (общие): ++...++. Позволяет обрабатывать свойства устройства.

# Открытие и закрытие страниц

### Открыть страницу

Условие:

Вы открыли проект. Открыто диалоговое окно навигатора Страницы — <Имя проекта>.

1. Выделите страницу в дереве или списке.
2. Выберите пункт меню Страница > Открыть.

!!! info "Для сведения:"

    Страница откроется в графическом редакторе. Имя страницы отображается над графикой как вкладка.

!!! note "Замечание:"

    С помощью настройки интерфейса пользователя Управление графическими редакторами, специфическими для проекта (меню Параметры > Настройки > Пользователь > Представление > Интерфейс пользователя) можно установить, должны ли (при одновременной работе в нескольких проектах) при открытии страницы / пространства листа из каждого следующего проекта всегда открываться отдельный графический редактор / отдельный трехмерный вид или закрываться открытая в данный момент страница проекта при переходе на другую страницу в другом проекте.

#### Открытие страницы в новом окне

1. Выделите страницу в дереве или списке.
2. Выберите пункт всплывающего меню Открыть в новом окне.

!!! info "Для сведения:"

    Страница будет открыта в другом окне графического редактора.

!!! info "Для сведения:"

    Для каждой открытой страницы над графикой отображается вкладка. С помощью щелчка по вкладкам вы можете быстро перемещаться между открытыми страницами.

#### Открытие страниц в многопользовательском режиме

Страницу схемы соединений всегда может обрабатывать только один пользователь Если пользователь находится на какой-либо странице, она блокируется. Второй пользователь может открыть эту странице только для просмотра, о чем информирует соответствующе сообщение.
В этом сообщении содержится имя пользователя в Windows, а также полное имя первого пользователя, благодаря чему второй пользователь информируется о том, кто установил блокировку, и при необходимости может обратиться к этому сотруднику.

### Выбрать следующую страницу / предыдущую страницу

Если открыта страница многостраничного проекта, все страницы проекта можно "пролистать" в графическом редакторе.

!!! tip "Совет:"

    Для пролистывания страниц можно внутри графического редактора также использовать клавиши ++Рисунок вниз++ / [Рисунок вверх] или кнопки ![](images/all_nextpageactive_as.png){: .ui-icon } / ![](images/all_previouspageactive_as.png){: .ui-icon } на панели инструментов Страница. Если курсор находится в диалоговом окне навигатора Страницы - <Имя проекта>, то страницы не перелистываются с помощью клавиш, вместо этого курсор переходит к последнему или первому элементу, видимому в текущем представлении структуры дерева.

### Сохранение страниц

Изменения, выполненные на странице, сохраняются автоматически при переходе на другую страницу или закрытии проекта.

# Диалоговое окно "Присвоения полей / Защитить поля" (база данных изделий)

Сервисные программы > Изделие > Управление. Через меню ++Дополнительно++ > Настройки была открыта база данных изделий. [Дополнительно] > Импортировать. В диалоговом окне Импортировать записи данных щелкните по кнопке ++...++ в поле Присвоения полей.

Вы открыли EPLAN Data Portal и импортируете изделие. В диалоговом окне Импорт изделия в поле Защитить поля нажмите кнопку ++...++.

В этом диалоговом окне создайте и измените схемы, заданные для структуры и произвольных свойств, в каких полях базы данных считываются экспортированные данные. Благодаря этому облегчается обмен данными между клиентами, использующими различные структуры. Укажите также, какие поля или определенные пользователем свойства должны быть "защищенными".

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

* * *

Вкладки "Произвольные свойства/Атрибуты":

!!! note "Замечание:"

    При импорте из EPLAN Data Portal обе вкладки доступны только при условии, что текущая схема содержит присвоения для произвольных свойств / атрибутов.

Обе эти вкладки построены одинаковым образом и содержат следующие элементы диалогового окна:

### Таблица

* Номер, источник: введите здесь номер соответствующего поля исходной базы данных.
* Номер, цель: введите здесь номер поля, в которое необходимо импортировать данные.

* * *

Вкладки "Защитить поля" / "Защитить свойства, определенные пользователем":

На этих вкладках определяются поля и / или определенные пользователем свойства, которые ***не*** должны обновляться при импорте данных изделия. Для этого нажмите кнопку ![](images/all_new_as.png){: .ui-icon } (Создать) и выберите в диалоговом окне Выбор критериев / Выбрать свойство свойство, которое должно игнорироваться при импорте.

### Защитить шаблоны функций

Эта настройка доступна ***только*** на вкладке Защитить поля. Установите этот флажок, если при импорте данных изделия также необходимо защитить шаблоны функций изделий, т. е. шаблоны функций в таком случае вообще ***не будут*** импортироваться.

* * *

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

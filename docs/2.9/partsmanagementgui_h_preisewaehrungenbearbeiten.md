# Обработать цены

Условия:

* Вы открыли базу данных изделий: Сервисные программы > Изделие > Управление
* В настройках указаны валюты и сокращенные обозначения валют (Параметры > Настройки > Пользователь > Управление > Изделие).

1. Выберите вкладку Цены / прочее для выделенных изделий.
2. Введите данные.

!!! example "Пример:"

    Предположим, требуется заказатькабели. Производитель продает кабели по 4 бухты, в каждой по 25 м, за 200,00 евро. Минимальное количество заказа на поставку составит в этом случае 100 м кабеля. Цена единицы — "25", единица измерения — "м", и количество/упаковка — "4". Затем необходимо указать в полях следующие значения:Цена единицы: 25Количественная единица измерения: м Количество/упаковка: 4Закупочная цена/единица цены: 50,00 евро Закупочная цена/упаковка: 200,00 евро

1. Выберите ++Дополнительно++ > Цены для выделенных изделий.
2. Укажите в диалоговом окне Рассчитать цены, какие закупочные цены требуется рассчитать, выбрав соответствующую опцию.

!!! note "Замечание:"

    Для одной валюты всегда можно выбрать только ***одну*** опцию, т. е. можно выполнить расчет либо для опции "Закупочная цена/Единица цены" либо для "Закупочная цена/Упаковка".

!!! example "Пример:"

    Предположим, закупочная цена за упаковку составляет для кабеля 200 евро, а количество/упаковка равно "4". Теперь можно рассчитать другую закупочную цену, выбрав в диалоговом окне Рассчитать ценыопцию Закупочная цена / единица ценыдля валюты 1 и щелкнув по кнопке ++OK++. В этом случае в качестве закупочной цены/единицы цены на вкладке Цены / Прочеедля валюты 1 будет отображаться "50 EUR".

Опции для поля Валюта 2 в диалоговом окне Рассчитать цены ***не*** ведут к пересчету валюты, а производится пересчет закупочной цены. Однако для выполнения этого следует ввести закупочную цену в поле Валюта 2 на вкладке Цены / Прочее.

1. Выберите ++Дополнительно++ > Валюты для выделенных изделий.
2. Укажите направление пересчета в групповом поле Пересчитать диалогового окна Пересчитать валюты.
3. Укажите в групповом поле Выбор, какие цены требуется пересчитать: Установите соответствующие флажки.

!!! note "Замечание:"

    В групповом поле Валюты этого диалогового окна в Параметры > Настройки > Пользователь > Управление > Изделие > [Валюта] выводится указанный обменный курс.

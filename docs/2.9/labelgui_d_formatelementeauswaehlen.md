# Диалоговое окно Выбрать элементы формата

Параметры > Настройки > Фирма > Сборка проводов > Komax или Параметры > Настройки > Фирма > Сборка проводов > Schleuniger. В диалоговом окне Настройки: <Производитель станка по сборке проводов> выберите вкладку Маркировка и щелкните по кнопке ++...++ в столбце Значение.

В этом диалоговом окне можно конфигурировать текст маркировки станков по сборке проводов с помощью элементов формата.

Обзор основных элементов диалогового окна:

### Доступные элементы формата

В данном поле отображаются все элементы формата, которые могут быть составной частью текста маркировки. Чтобы скопировать элемент формата для маркировки в поле Выбранные элементы формата, выделите его и нажмите на ![](images/all_arrowright_as.png){: .ui-icon } (переместить вправо).

### Выбранные элементы формата

В данном поле находятся элементы формата, из которых состоит формат текста маркировки.

### Предв. просмотр

В этом поле объединенный формат выводится для предварительного просмотра.

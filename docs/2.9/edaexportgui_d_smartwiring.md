# Диалоговое окно Экспорт EPLAN Smart Wiring

Вы открыли проект. Выберите пункты меню Сервисные программы > Данные изготовления > EPLAN Smart Wiring.

В этом диалоговом окне устанавливаются настройки экспорта проекта в формат EPDZ для чтения в приложении ***EPLAN Smart Wiring***. Файл EPDZ содержит свойства проекта, свойства страницы и свойства пространства листа, а следовательно, полную структуру проекта, а также данные о соединениях и свойствах кабелей для маршрутизируемых соединений. Файл EPDZ также содержит файл E3D с трехмерными графическими данными для каждого пространства листа.

Обзор основных элементов диалогового окна:

### Источник (страница / трехмерная модель)

В данном поле выводится имя экспортируемого проекта. Обработка записи в данном поле невозможна.

### Файл EPDZ

Имя файла, в который необходимо экспортировать открытый проект. По умолчанию записывается имя проекта с расширением имени файла "epdz".

С помощью кнопки ++...++ также можно свободно задавать имя файла или выбирать имя уже имеющегося файла.

### Каталог вывода

В поле Каталог вывода можно ввести место хранения.

Кнопка ++...++ открывает диалоговое окно Выбрать каталог для выбора любого каталога для экспорта.

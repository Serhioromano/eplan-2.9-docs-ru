# Диалоговое окно Настройки: Специфические для ПЛК

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > ПЛК. В диалоговом окне Настройки: ПЛК щелкните рядом с полем Специфические для ПЛК настройки по кнопке ++...++.

В этом диалоговом окне вы задаете специфические для данного типа ПЛК настройки, такие как форматы для ПЛК-адресов, символические адреса и списки присвоений. Эти настройки сохраняются как схема для проекта.

Процессоры ПЛК можно свободно конфигурировать в EPLAN. Если нужно создать список присвоений, для которого в EPLAN не существует настройки, у Вас есть возможность создать собственную конфигурацию для вашего формата списка присвоений.

Доступны следующие элементы диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которая задает специфические для ПЛК настройки. Предварительная настройка для схемы зависит от записи, выбранной в раскрывающемся списке Специфические для ПЛК настройки (диалоговое окно Настройки: ПЛК).
Через расположенную рядом [панель инструментов](schemeconfig_m_schemaschaltflaechen.md) можно создавать новые схемы, сохранять изменения в схеме, копировать или удалить схему. Кроме того, схемы можно импортировать и экспортировать.

### Описание

Данное поле отображает текст описания, который вы ввели при создании схемы в диалоговом окне Новая схема в поле Описание.

### Вкладки

Для ввода настроек имеются следующие вкладки:

* Адреса: В этой вкладке вы определяете идентификаторы и тип данных для вводов и выводов. Далее определите форматы данных для входов и выходов.
* Форматы адресов: В этой вкладке вы определяете формат адресов ПЛК. Для дискретных и аналоговых вводов и выводов возможно использование разных форматов адресов.
* Формат списка присвоений: В этой вкладке вы определяете формат списка присвоений.
* Формат символического адреса: В этой вкладке вы задаете критерии, по которым будут автоматически генерироваться символические адреса. Кроме того, у вас есть возможность заменять используемые в структурных идентификаторах специальные символы в списках присвоений на другие.

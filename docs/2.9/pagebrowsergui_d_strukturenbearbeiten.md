# Диалоговое окно Обработать структуры

* В навигаторе страниц выделена одна или несколько страниц. Обработать > Копировать. Щелкните в навигаторе страниц по тому месту, в которое необходимо вставить скопированные страницы. Обработать > Вставить. [Дополнительно] > Обработать структуры.
* Страница > Макрос страницы > Вставить. Выберите в диалоговом окне Выбрать макрос требуемый макрос страницы и щелкните по кнопке ++Открыть++. [Дополнительно] > Обработать структуры.

В этом диалоговом окне можно задавать / обрабатывать главные идентификаторы и до 9 подидентификаторов для отдельных блоков идентификаторов.

Обзор основных элементов диалогового окна:

### Таблица

В таблице перечислены блоки идентификаторов, а также полные обозначения исходных функций / страниц. Эти ячейки недоступны для обработки. При этом можно использовать ячейки Главные идентификаторы и Подидентификатор 1 ... Подидентификатор 9, чтобы задать / изменить несколько структур обозначений за один рабочий шаг.

!!! example "Пример:"

    Например, введите требуемыесимволыв ячейку, а затем используйте буфер обмена, чтобы с помощью комбинации клавиш[Ctrl]+[C]и[Ctrl]+[V]скопировать эту структуру в другие ячейки.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

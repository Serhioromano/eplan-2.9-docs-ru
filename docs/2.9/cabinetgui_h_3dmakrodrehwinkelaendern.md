# Изменить угол поворота при вставке 3D-макросов

При вставке 3D-макроса угол размещения макроса в точке захвата можно изменить с шагом в 90°. Таким образом, в сочетании с переменными точками захвата для одного отдельного 3D-макроса доступно до 40 различных положений установки. Эти возможности применимы и к устройствам, изделиям которых присвоен какой-либо 3D-макрос.

Условия:

* Вы открыли проект.
* Навигатор пространства листов открыт, и одно пространство листов открыто.

1. Выберите пункты меню Вставить > Макрос окна / символа и откройте 3D-макрос для размещения.

2. Выберите пункты меню Параметры > Изменить угол вращения.

!!! info "Для сведения:"

    3D-макрос будет повернут на 90° против часовой стрелки.

!!! info "Для сведения:"

    При каждом повторном выборе пунктов меню Параметры > Изменить угол вращения макрос будет повернут на следующие 90°.

3. С помощью клавиши ++A++ можно переключить точку захвата.
4. В измененной точке захвата снова выберите пункты меню Параметры > Изменить угол поворота.

!!! info "Для сведения:"

    Для каждой доступной точки захвата можно указать четыре различных угла поворота. С учетом девяти стандартных точек захвата и одной точки захвата, определенной пользователем, реализовать можно до 40 различных позиций.

!!! tip "Совет:"

    Во время вставки угол поворота можно изменить, используя комбинацию клавиш [Ctrl] \+ [Shift] \+ [R].

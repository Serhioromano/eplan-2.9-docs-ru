# Создать собственные обозначения и описания вывода устройства

Для каждого определения функции можно сохранить десять наборов обозначений и описаний выводов устройства. Затем эти наборы можно использовать из функции.

### Создать обозначения выводов устройства

1. Сервисные программы > Основные данные > Обозначение / описание вывода устройства
2. Выберите вкладку Обозначения выводов устройства.

!!! info "Для сведения:"

    На вкладке Обозначения выводов устройства приведены определения функций с выводами устройств. Для каждого вывода устройства выводится отдельная строка.

3. Введите в поля 1 \- 10 до десяти различных обозначений выводов устройства за один вывод устройства.
4. Щелкните по кнопке ++OK++.

### Создать описания выводов устройства

1. Сервисные программы > Основные данные > Обозначение / описание вывода устройства
2. Выберите вкладку Описания выводов устройства.

!!! info "Для сведения:"

    На вкладке Описания выводов устройства приведены определения функций с выводами устройств. Для каждого вывода устройства выводится отдельная строка.

3. Введите в поля 1 \- 10 до десяти различных описаний выводов устройства за один вывод устройства.
4. Щелкните по кнопке ++OK++.

!!! warning "Предупреждение:"

    После каждого изменения необходимо экспортировать измененные обозначения/описания выводов устройства. Если при обновлении вы получаете новую библиотеку определений функций, необходимо импортировать экспортированные вами обозначения/описания выводов устройств в новую версию программы и дополнить новые определения функций.

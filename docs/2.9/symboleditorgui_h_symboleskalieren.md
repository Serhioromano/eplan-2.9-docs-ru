# Масштабировать символы

Условие:

Вы открыли проект и как минимум одну библиотеку символов.

1. Сервисные программы > Основные данные > Символ > Открыть
2. В диалоговом окне Выбор символа выберите символ/символы, которые вы желаете обработать.
3. Щелкните по кнопке ++OK++.
4. Выберите в редакторе символов пункт меню Обработать > Графика > Масштабировать.
5. Щелкните один раз левой кнопкой мыши в окне обработки, чтобы задать начальную точку для рамки масштабируемых элементов.
6. Двигайте курсор в требуемом направлении, чтобы растянуть рамку для масштабируемых элементов.

!!! info "Для сведения:"

    Линии прямоугольника "перемещаются" вместе с движением мыши.

7. Щелкните второй раз, чтобы определить конечную точку прямоугольника.

!!! info "Для сведения:"

    Выбранные части символа обозначаются цветом.

8. Щелкните по редактору символов.
9. Введите в поле Коэффициент масштабирования диалогового окна Масштабировать требуемое значение.
10. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Символ масштабируется на заданное значение.

# Диалоговое окно Нумерация с данными ПЛК

Вы открыли проект. Вы выделили устройства в графическом редакторе или в навигаторе, или же Вы выделили одну или несколько страниц в диалоговом окне навигатора Страницы — <Имя проекта>. Данные проекта > Устройствo > Нумерация с данными ПЛК.

В этом диалоговом окне указываются настройки для нумерации устройств с данными ПЛК, после чего можно выполнить нумерацию.

Обзор основных элементов диалогового окна:

### Настройки

Выберите из раскрывающегося списка схему, которая определяет настройки нумерации. Или нажмите кнопку ++...++, чтобы в диалоговом окне Настройки: Нумерация с данными ПЛК создать новую схему, обработать уже имеющуюся схему и т. д.

### Использованные группы форматов

В этом поле выводятся группы устройств, которые определены в схеме в качестве объема нумерации.

### Перезаписать

Здесь указывается, какие ОУ / обозначения должны быть перезаписаны во время нумерации:

* Все: Заново вводятся ОУ / обозначения всех устройств, содержащихся в объеме нумерации.
* Только уже "нумерованные с данными ПЛК": Заново нумеруются только те функции, которые уже отмечены как "Нумерация с данными ПЛК".
* Пронумеровать только обозначенные: Заново нумеруются только те функции, которые помечены вопросительным знаком "?" в ОУ или в обозначении.

### Выделить как "Нумерация с данными ПЛК"

Если этот флажок установлен, для функций, которые были пронумерованы, устанавливается свойство "Нумерация с данными ПЛК". В случае последующих прогонов нумерации (нумерация ОУ, нумерация клемм и т. д.) можно будет выбрать, следует ли их тоже нумеровать или же исключить из нумерации.

Если этот флажок снят, функции не помечаются и их невозможно будет отличить от других функций в случае последующих прогонов нумерации.

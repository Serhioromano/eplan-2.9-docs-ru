# Версия AutoCAD при экспорте DXF

Из настроек экспорта данных изготовления ЧУ и меди в формате DXF для механической обработки удалены старые и уже неподдерживаемые версии AutoCAD. В раскрывающемся списке выбора формата в поле Версия AutoCAD теперь доступны следующие версии:

* AutoCAD 2013/2014/2015/2016/2017
* AutoCAD 2018/2019.

!!! note "Замечание:"

    Лицензия на экспорт данных изготовления в формате DXF выдается через два дополнительных модуля расширения, за которые взимается плата. Информацию об этом можно найти в текущей спецификации договорных работ и услуг для EPLAN Pro Panel.

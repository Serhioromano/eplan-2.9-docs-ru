# Импортировать список отсутствующих слов

Экспортированный список отсутствующих слов, содержащий допереведенные тексты, можно снова импортировать в словарь.

Условие:

Вы открыли проект.

1. Сервисные программы > Перевод > Обработать словарь
2. Рядом с полем Словарь щелкните по ++...++ и выберите в диалоговом окне Открыть словарь нужный вам словарь.
3. Выберите ++Дополнительно++ > Импорт.
4. В диалоговом окне Импортировать словарь в поле Тип файла выберите формат, в котором доступен список отсутствующих слов.
5. Выделите список отсутствующих слов и нажмите ++Открыть++.
6. Чтобы задать исходный язык перевода, выделите его в последующем диалоговом окне.

!!! info "Для сведения:"

    Допереведенные тексты языков перевода импортируются в словарь.

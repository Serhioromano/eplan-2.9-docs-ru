# Маршрутизировать соединения

В результате маршрутизации возникают маршрутизированные соединения между 3D-размещениями изделий, повторяющие реальное расположение проводов и кабелей в сети соединенных сегментов. Если при размещении изделия выводы устройства отсутствуют, они создаются перед маршрутизацией.

Кроме трассы маршрутизации, определяется также и длина проводов. Все найденные сведения могут потом использоваться в инструкции по прокладке соединений или для управления машинами по сборке проводов.

Условие:

Вы открыли проект. Открыто одно пространство листа. Создана сеть соединенных сегментов.

1. Выберите пункты меню Данные проекта > Соединения > Маршрутизировать (пространство листа).

!!! info "Для сведения:"

    В строке состояния появится требование выбора соединения для маршрутизации.

2. Укажите две угловые точки прямоугольника, который охватывает маршрутизируемые размещения изделий и соединений.

!!! info "Для сведения:"

    Рассчитывается оптимальная трасса маршрутизации. Маршрутизируемые соединения отображаются в 3-хмерном представлении.

!!! info "Для сведения:"

    Маршрутизируемые соединения имеют свойства: Длина, Трасса маршрутизации, Трасса маршрутизации по умолчанию, Размер присоединения, Направление маршрутизации, Обработка концов проводов и Предписать двойную трубку.

!!! example "Пример:"

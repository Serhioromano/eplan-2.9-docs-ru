# Вкладка Настройки (Komax)

* Параметры > Настройки > Фирма > Сборка проводов > Komax > Настройки
* Сервисные программы > Данные изготовления > Сборка проводов > Komax. Щелкните по кнопке ++...++ рядом с полем Схема. Выберите вкладку Настройки.

В этой вкладке хранится информация о технических возможностях используемой машины. Отдельные настройки задаются заранее индивидуально для машины при помощи подходящих значений, которые можно здесь изменить.

Обзор основных элементов диалогового окна:

### Минимальная длина провода

Минимальная длина, необходимая для заданной обработки провода машиной (стандартное значение: 240 мм). Более короткие провода передаются в машину с данной минимальной длиной.

### Максимальная длина провода

Провода, длина которых превышает указанную здесь величину, машина изготовить не может (стандартное значение: 3000 мм).

### Максимальное поперечное сечение жилы

Максимальное поперечное сечение провода, которое машина может обработать (стандартное значение: 6 мм²).

### Максимальное количество проводов в объединенных проводах

Количество проводов, которые машина может объединить в объединенном проводе (стандартное значение: 50).

### Максимальная сумма поперечного сечения жил

Предельное значение для суммы поперечного сечения всех проводов (стандартное значение: 80 мм²).

### Непрерывный текст по длине провода

Значение длины, начиная с которого печать на провода должна наноситься в виде непрерывного текста. Если значение не указано, печать на все провода наносится в виде непрерывного текста.

### Направление подсоединения сверху / снизу / слева / справа

Тексты, которые необходимо напечатать на проводе для указания отдельных направлений подсоединения.

### Максимальная длина набора печатных символов

Максимальная длина символов, которая может быть у отдельного текста (стандартное значение: 39).

### Объединение в машине

С помощью этого свойства можно определить, как машина должна связывать созданные пучки проводов:

* Одностороннее: Пучки проводов будут соединяться с одной стороны.
* Двустороннее: Пучки проводов будут соединяться с двух сторон.

### Автоматически поворачивать печатные тексты

В формате экспорта Komax предусмотрена возможность при необходимости поворачивать текст.

Если установлен этот флажок, то маркировочные тексты одного из изготавливаемых проводов для идентификации стороны вывода устройства на устройстве в зависимости от того, какой это конец провода, поворачиваются на 180 градусов.

!!! example "Пример:"

    Текст на левом конце провода поворачивается на 180. Потом монтажник сможет понять, что этот конец провода должен быть присоединен к левой стороне устройства.

### Установить сдвоенный опрессовочный модуль

С помощью сдвоенного опрессовочного модуля на различных машинах Komax можно производить провода с двойными изоляционными трубками.

Если этот флажок установлен, то необходимая информация для сдвоенного опрессовочного модуля экспортируется при помощи команды Экспорт Komax.

Поскольку изделие Komax может состоять из одного или двух отрезков провода, требуется различная обработка начального и конечного (при одном отрезке провода), а также начального, среднего и конечного концов (при двух отрезках провода). Если контакт расположен в середине между двумя другими проводами и на вкладке Машинная команда для обработки концов проводников обжатием определена машинная команда ["TerminalKey"](productionwiringgui_r_maschinenbefehl.md) (AEH), то двойная изоляционная трубка будет установлена на средний контакт, а на начальном и конечном проводнике будет только снята изоляция.

Для этого на вкладке Обработка концов проводников предусмотрена настройка [Двойная обработка концов проводников посередине](productionwiringgui_r_aderendbehandlung.md).

### Гибкий бандаж установлен

Эта настройка влияет на число заданий (единиц продукции), генерируемых в файле экспорта для машины Komax.

Если этот флажок установлен, то все процессы, связанные с объединением проводов в пучки, сводятся в одно единственное "главное задание". Таким образом, установленный на машине модуль с гибким бандажом способен выполнить это задание, что позволит ускорить процесс обработки объединенных проводов. Максимальное число проводов в одном пучке / главном задании составляет 99. В случае превышения этого числа генерируется другое главное задание.

Если этот флажок снят, то в файле экспорта для каждого пучка генерируется одно задание.

!!! note "Замечание:"

    Настройка Гибкий бандаж установлен влияет только на число сгенерированных заданий в файле экспорта. EPLAN Pro Panel ***не*** поддерживает расширенные функциональные возможности гибкого бандажа при экспорте данных изготовления. Эти функции необходимо настроить прямо на машине.

### Вид объединенных проводов

Эта настройка определяет, будет ли с помощью станка для сборки проводов Komax создана круглая связка или последовательная связка. Для изготовления последовательной связки в станке Komax должно быть установлено дополнительное оборудование "Гибкий бандаж".

# Диалоговое окно Конфигурировать свойства

Вы открыли проект.

* Вы открыли диалоговое окно, в котором объекту (например, проекту, странице, форме и т.д.) можно присвоить свойства с помощью таблицы. Выберите пункт всплывающего меню Конфигурировать.
* Вы выделили один или несколько объектов (тексты, условные обозначения, сегменты, графические элементы) в графическом редакторе, пространстве листа или навигаторе. Обработать > Обзор свойств. В обзоре свойств в раскрывающемся списке нажмите ++...++.

***Диалоговые окна свойств:*** в этом диалоговом окне выберите свойства и их последовательность, которые должны отображаться по умолчанию на каждом объекте соответствующей категории функции в таблице.

***Обзор свойств:*** в этом диалоговом окне выберите свойства и их последовательность, которые должны отображаться в таблице свойств обзора свойств.

Обзор основных элементов диалогового окна:

!!! note "Замечание:"

    Поля Схема и Описание доступны только для конфигурации свойств обзора свойств и определенных пользователем конфигураций свойств.

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Список

Список содержит выбранные на данный момент свойства. С помощью кнопки ![](images/all_new_as.png){: .ui-icon } (Создать) можно выбирать другие свойства и копировать их в список. С помощью кнопок со стрелками можно изменять последовательность свойств.

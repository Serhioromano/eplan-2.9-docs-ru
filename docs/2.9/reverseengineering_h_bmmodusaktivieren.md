# Активировать режим "Свойства (общие)"

Режим редактирования ***Свойства (общие)*** позволяет обрабатывать общие свойства распределенных функций в одной операции.

Условие:

Вы открыли проект.

1. Выберите меню Параметры.
2. Активируйте пункт меню Свойства (общие).

!!! info "Для сведения:"

    Перед пунктом меню отображается "галочка". При двойном нажатии на функцию свойства функции автоматически открываются в диалоговом окне Свойства (общие): ++...++.

!!! tip "Совет:"

    Если в меню Параметры активирован режим редактирования Свойства (общие), все равно можно изменять свойства распределенных функций отдельно друг от друга. Выделите функцию и выберите Раскрывающееся меню > Свойства.

# Восстановить основные данные

При восстановлении архивов, содержащих только основные данные системы, EPLAN проверяет данные на наличие изменений. При выявлении измененного файла выводится соответствующее сообщение c датами последних изменений файлов. В таком случае необходимо определить, следует ли перезаписывать существующий файл. Основные данные можно восстановить и в другой (новый / пустой) каталог основных данных.

1. Выберите пункты меню Проект > Вернуть > "Основные данные".
2. В диалоговом окне Восстановить <Основные данные> в поле Каталог для резервирования с помощью кнопки ++...++ выберите каталог, в котором располагаются резервированные основные данные.

!!! info "Для сведения:"

    В списке Основные данные: <Основные данные> отображаются резервированные основные данные, содержащиеся в выбранном каталоге.

3. Выделите файлы для восстановления.
4. В поле Целевой каталог с помощью кнопки ++...++ выберите целевой каталог, в который должно быть скопировано резервирование данных.
5. Подтвердите ввод.

!!! info "Для сведения:"

    Данные выгружаемого файла будут скопированы в указанный целевой каталог. При выделении сразу нескольких файлов они будут восстановлены по очереди.

# Блоки отчетов

Блоки отчетов являются взаимосвязанными [страницами отчетов](formgeneratorgui_k_auswertungsseiten.md). Путем [сортировки страниц отчетов](formgeneratorgui_k_seitensortierung.md) укажите, создается отчет в одном или нескольких блоках отчетов. Если вы хотите создать содержание для всего проекта, т.е. для всех уровней структурного идентификатора вместе, отчет создается в блоке отчетов. Если выводится отдельное содержание для каждого уровня структурного идентификатора, отчет создается в нескольких блоках отчетов. Для каждого уровня структурного идентификатора есть блок отчетов. При обновлении отчетов учитываются все страницы отчетов, относящиеся к одному блоку отчетов. Настройка отчета всегда действует на весь блок отчетов.

При генерировании отчетов EPLAN предлагает начальные страницы в структуре страниц. Эти нач. страницы можно копировать или задать другие. Благодаря этому указывается, на каком уровне структуры страниц начинается блок отчетов. Первая страница отчета блока отчетов является одновременно нач. страницей блока отчетов.

Если вы задали начальную страницу, а данная страница уже определена, EPLAN ищет следующую свободную страницу для страницы отчета.

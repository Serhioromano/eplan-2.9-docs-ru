# Генерировать схемы сверления из файлов ЧУ

Импорт информации о сверлении из файлов ЧУ Rittal — Perforex / Secarex охватывает файлы со следующими расширениями имени файла:

* функц. элементы *.PBT
* переменная "Функц. элементы" *.PVB
* Контуры *.PKU.

!!! note "Замечание:"

    * Из файлов функциональных элементов Rittal — Perforex / Secarex (*.PBT или *.PVB) при помощи этого метода можно генерировать только схемы сверления для отдельных функциональных элементов, но не для целых монтажных плат.
    * Из файла контура Rittal — Perforex / Secarex (*.PKU) генерируется определенный пользователем контур. Этому контуру уже присвоена запись данных "ЧУ Rittal — Perforex / Secarex". После импорта создайте в базе данных изделий схему сверления с типом сверления "Определенный пользователем контур", которая будет использовать контур, сгенерированный путем импорта файла PKU.
    * Контур, сгенерированный из файла контура Rittal — Perforex / Secarex, перед созданием схемы сверления следует проверить в редакторе контура на правильность, например замкнутость контура, положение центрового сверла при прямоугольных фрезеровках, правильность записи данных ЧУ, количество элементов закругления при фрезерованных закруглениях.
    * Файлы контура определенных пользователем контуров из импорта ЧУ Rittal — Perforex / Secarex часто соответствуют стандартным типам сверления (прямоугольное, продольное отверстие, шестигранник, восьмигранник). Для упрощения создаваемой схемы сверления такие определенные пользователем контуры можно также заменить на соответствующие типы сверления, для которых, кроме того, не требуется никакой дополнительной записи данных ЧУ.
    * При импорте файлов ЧУ для круглых фрезерных контуров можно создать автоматическую [запись данных ЧУ](ncgui_k_ncdatensatz.md), так как в EPLAN Pro Panel начальная точка для записи данных ЧУ определяется только на прямых сегментах контура. Поэтому круглые монтажные отверстия из файлов ЧУ следует определять в базе данных изделий как отверстия или отверстия с резьбой.

Условия:

* Вы открыли проект.
* Вы сгенерировали при помощи ПО Rittal — Perforex / Secarex файлы ЧУ с расширением имени файла PBT, PVB и PKU.

1. Выберите пункты меню Сервисные программы > Генерировать схемы сверления / контуры > Из файлов ЧУ Rittal — Perforex / Secarex.
2. В диалоговом окне выбора файла выберите импортируемые файлы ЧУ Rittal — Perforex / Secarex и щелкните ++Открыть++.
3. Если вы выбрали для импорта файл PKU, выберите в диалоговом окне Генерировать схемы сверления / контуры каталог, из которого должны генерироваться контуры из файлов ЧУ. Каталогом по умолчанию является каталог для макросов.
4. Выберите схему для станка ЧУ Rittal — Perforex / Secarex, которая должна использоваться при экспорте данных ЧУ Rittal — Perforex / Secarex. Из этих настроек определяется тип сверления и диаметр для отверстий на схемах сверления.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Из файла PBT или PVB в базе данных изделий генерируется схема сверления.

!!! info "Для сведения:"

    Из файла PKU в каталоге макросов генерируется определенный пользователем контур.

6. Создайте в базе данных изделий новую схему сверления с типом сверления "Определенный пользователем контур" и сохраните в ней контур, сгенерированный путем импорта файла PKU.

# Вкладка Определение атрибута

* Параметры > Настройки > Пользователь > Интерфейсы > Экспорт и импорт DXF / DWG. Выберите вкладку Атрибут блока. Выберите запись из списка Доступные свойства, и щелкните по![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо).
* Страница > Экспортировать > DXF / DWG. Щелкните рядом с полем Схема по ++...++. Выберите вкладку Атрибут блока. Выберите запись из списка Доступные свойства, и щелкните по![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо).

Определите в этом диалоговом окне присвоения свойств EPLAN атрибутам AutoCAD.

Доступны следующие элементы диалогового окна:

### Категория

Выберите из раскрывающегося списка категорию, свойства которой должны быть выведены в списке ниже. Отобразятся свойства выбранного в столбце Доступные свойства элемента.

### Обозначение атрибута

Введите обозначение, под которым выбранный элемент должен быть перенесен в AutoCAD.

### Требование ввода

Введите требование ввода для выбранного свойства.

# Перевод описания слоя для определенных пользователем слоев

Слои имеют описание, из которого в диалоговом окне Управление слоями — <Имя проекта> генерируется иерархическая структура представления структуры дерева. До сих пор описание слоя при переводе не учитывалось. Теперь появилась возможность переводить описания ***определенных пользователем*** слоев при переводе проекта.

![](images/benefit.png)Эффект:

С помощью нового свойства Слой: Описание для описания слоя теперь возможен многоязычный ввод. Описание слоя является переводимым и выводится на выбранном языке отображения.

Для этого доступно новое свойство Слой: Описание (ид. 19380). Это свойство можно выбрать в настройках проекта для перевода. Путь меню к настройке проекта: Параметры > Настройки > Проекты > "Имя проекта" > Перевод > Объем. Здесь в поле Раздел выберите запись "Проект" и установите флажок для свойства Слой: Описание.

Имена и описания стандартных слоев EPLAN невозможно изменить и перевести.

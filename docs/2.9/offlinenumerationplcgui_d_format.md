# Диалоговое окно Нумерация с данными ПЛК: Формат

Вы открыли проект. Вы открыли диалоговое окно Настройки: Нумерация с данными ПЛК (например, через Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация с данными ПЛК). В диалоговом окне Настройки: Нумерация с данными ПЛК выберите определенную пользователем схему или создайте такую схему. Выделите запись в поле Группы форматов и щелкните на ![](images/all_edit_as.png){: .ui-icon } (Обработать) или щелкните на ![](images/all_new_as.png){: .ui-icon } (Создать).

В этом диалоговом окне можно изменить формат существующей группы форматов или настроить новый формат.

Обзор основных элементов диалогового окна:

### Группа

Объем нумерации с данными ПЛК настраивается отдельно для каждой группы устройств. В раскрывающемся списке имеются следующие группы устройств:

* Клеммы: Изменяется обозначение клеммы.
* Контакты штекеров: Изменяется обозначение контакта штекера.
* Устройство: Изменяется только ОУ общих функций. ОУ символов устройств, выводов устройств, выводов устройств ПЛК, сборных шин, кабелей, штекеров и клеммников не изменяется.

Если для группы уже настроен формат, эта группа более не выводится в списке.

### Объем

Здесь настраивается объем нумерации для выбранной группы.

* Непосредственно подсоединенная первая цель: Выполняется поиск только первой цели, например контакта штекера на выводе устройства ПЛК.

!!! example "Пример:"

    Для вывода устройства ПЛК E1.0 непосредственной целью является клемма X10; она получает обозначение с данными ПЛК. Вторая клемма, связанная с данным выводом устройства ПЛК, не получает обозначение.

* Сеть: Во внимание принимаются концы всех связанных друг с другом соединений, а также сетевое расширение (клеммы, контакты штекеров...).

!!! example "Пример:"

    Здесь нумеруются все устройства, которые связаны с выводом устройства ПЛК E1.0, то есть обе клеммы и датчик PLC3.

* До первого датчика / исп. элемента: Выполняется поиск первой цели, у которой также заимствуется автоматический символический адрес.

!!! example "Пример:"

    Здесь нумеруются все устройства, которые были найдены по прямому пути к первому датчику, т. е. первая клемма и датчик PLC3.

* До каждого датчика / исп. элемента (сеть): Выполняется поиск во всей сети до первого датчика / исполнительного элемента. От входа выполняется поиск датчиков, а от выхода — поиск исполнительных элементов. Другие, если они являются главными функциями, также нумеруются (с помощью нумерации ОУ). Клеммы и контакты штекера также нумеруются.

!!! example "Пример:"

    На рисунке у условных обозначений в черной рамке активировано свойство "Продолжить нумерацию с данными ПЛК далее послефункции". Находящиеся между ними клеммы также нумеруются. Также в нумерации участвуют другие сети, если они имеются. На втором выводе устройства ПЛК установлено свойство "Предотвратить нумерацию с помощью данных ПЛК".

### Доступныеэлементы формата

В этом списке выводятся элементы, которые могут быть частями формата адреса. Для форматирования элемента формата необходимо переместить его вправо в список выбранных элементов формата; многократный выбор в данном случае невозможен. Если вы переместите элемент формата, откроется диалоговое окно, в котором можно определить форматирование. Оно отличается в зависимости от элемента формата.

Для клемм и контактов штекера возможны следующие элементы формата:

Элемент формата |  Предварительный просмотр |  Значение
---|---|---
Данные вывода устройства ПЛК |  ***T***  |  Выводит свойства вывода устройства ПЛК.
Данные адреса ПЛК |  ***A***  |  Выводит свойства адреса ПЛК.
Разделитель |  |  Разделяет элементы формата друг от друга.

Для устройств общего назначения возможны следующие элементы формата:

Элемент формата |  Предварительный просмотр |  Значение
---|---|---
Буквенное обозначение |  ***I***  |  Выводит буквенное обозначение устройства. (Разрешен только один раз в формате)
Данные вывода устройства ПЛК |  ***T***  |  Выводит свойства вывода устройства ПЛК.
Данные адреса ПЛК |  ***A***  |  Выводит свойства адреса ПЛК.
Страница |  ***S***  |  Здесь указано имя страницы, на которой находится устройство. Нижестоящий счетчик страницы не выводится. (Разрешен только один раз в формате)
Столбец |  ***P***  |  Здесь указан номер столбца, в котором находится устройство. (Разрешен только один раз в формате)
Строка |  ***R***  |  Здесь указан номер строки, в которой находится устройство. (Разрешен только один раз в формате)
Разделитель |  |  Разделяет элементы формата друг от друга.

!!! note "Замечание:"

    Полный адрес ПЛК можно просмотреть в поле "Данные вывода устройства ПЛК"; отдельные части адреса ПЛК можно выбрать в поле "Данные адреса ПЛК". При этом один бит соответствует первому уровню, а байт - второму слою адреса.

### Выбранные элементы формата

В этом списке выводятся элементы формата, настроенные для выбранной группы. Выбранные элементы формата выводятся в табличной форме с их обозначением, символическим представлением (для предварительного просмотра) и настроенным значением. С помощью кнопок, расположенных над списком, можно дополнительно изменить форматирование элемента формата, удалить элементы формата, а также изменить их последовательность.

### Предв. просмотр

Здесь выводится предварительный просмотр формата нумерации. Отдельные элементы формата выводятся в ОУ или обозначении клеммы / контакта штекера в той последовательности, в которой они стоят в списке. Значение в этом поле изменить невозможно.

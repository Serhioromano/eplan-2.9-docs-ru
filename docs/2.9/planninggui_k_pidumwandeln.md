# Копирование данных проектов EPLAN PPE

Проекты EPLAN PPE из более ранних версий EPLAN (версия 2.6 или старше) при открытии во время последующего обновления баз данных проекта автоматически преобразуются в проекты предварительного планирования. (Для этого требуется установка операционной системы Microsoft и приложений Microsoft Office (среди прочих Microsoft Access) полностью в 64-битной версии.)

При этом из сведений, содержащихся в базе данных PPE, генерируются сегменты. Старый проект сохраняется с измененным именем. Если в проекте EPLAN PPE содержатся документы, будет выведено сообщение, в котором можно указать, следует ли копировать эти документы в качестве графических страниц.

При таком автоматическом копировании данных следующие объекты PPE преобразовываются в соответствующие сегменты или объекты предварительного планирования:

Объект PPE |  Сегмент/объект
---|---
Установка -технологическая маркировка -обозначение |  Сегмент структуры -обозначение -описание Структурные идентификаторы определяются по конфигурации в EPLAN PPE.
Технологический контур |  Технологический контур
Функция ТК |  Функция ТК
Элемент ТК -тип устройства -группа устройств |  Объект планирования -описание -определение сегмента
Спецификация |  Шаблон сегмента
Монтажный чертеж |  Узел
Заказы |  \- ничего -
Резервуар |  Резервуар
Свойство |  Определенное пользователем свойство. Группы свойств изображаются в качестве составных частей (дополнительных узлов) идентифицирующего имени свойства.
Внешний документ |  Документ (сохранен на сегменте)
Документ |  Графическая страница. При открытии и преобразовании проектов EPLAN PPE можно указать, следует ли преобразовывать документы из проектов EPLAN PPE в графические страницы или игнорировать.

!!! note "Замечание:"

    После копирования данных проекта EPLAN PPE в навигаторе шаблонов сегментов появляются последующие, предварительно определенные объекты планирования (напр., "Датчик", "Исполнительный элемент", "Задатчик команд" и т. д.) и шаблоны сегментов. В основе этих шаблонов лежат спецификации из EPLAN PPE. Из номера спецификации в PPE при преобразовании образуется идентификатор шаблона сегмента.

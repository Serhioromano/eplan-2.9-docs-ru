# Обработать свойства во всех представлениях главной функции

Главная функция может быть многократно представлена в проекте в распределенном виде как однополюсная функция, многополюсная функция, функция обзора, функция парной перекр. ссылки и / или функция функциональной схемы автоматизации. С помощью функции "Свойства (общие)" можно в одной операции одновременно изменить общие свойства во ***всех*** представлениях функции ******главной функции, как, например, функциональный текст всех представлений катушки.

!!! note "Замечание:"

    Этот принцип работы не действует для свойства Полное ОУ. Если в режиме редактирования ***Свойства (общие)*** изменить Полное ОУ главной функции, изменится ОУ во всех представлениях функции устройства.

Условия:

* Вы открыли проект.
* Открыт Навигатор устройств или Графический редактор.

1. Чтобы обработать свойства главной функции в навигаторе устройств, выделите представление функции главной функции и выберите Всплывающее меню > Свойства (общие).
Чтобы обработать свойства главной функции в графическом редакторе, активируйте в меню Параметры  пункт меню Свойства (общие), щелкните дважды по представлению функции главной функции.

!!! info "Для сведения:"

    Открывается диалоговое окно Свойства (общие): ++...++.

!!! info "Для сведения:"

    Открывается диалоговое окно Свойства (общие): ++...++ На вкладке <Категория функции> (Устройство) нанесено обозначение соответствующей категории функции, например "Катушка " и дополнение "(устройство)" для устройства.

2. Измените требуемое свойство главной функции.
3. Щелкните по кнопке ++ОК++.
4. Чтобы обновить информацию, нажмите клавишу ++F5++.

!!! info "Для сведения:"

    Свойства изменяются во всех других соответствующих представлениях функции выбранной главной функции.

!!! note "Замечание:"

    На вкладках Отображение, Данные символа / функции и Формат не отображаются общие свойства. На этих вкладках можно обрабатывать только выбранное представление функции главной функции. Все другие соответствующие представления функции не изменяются.

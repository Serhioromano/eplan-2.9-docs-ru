# Шаблоны для новых проектов

Шаблоны позволяют создавать проекты на основе имеющихся данных. При формировании проектов с помощью шаблонов всегда требуется шаблон. Шаблоны для новых проектов:

* [Шаблоны проектов](projects_k_projektvorlagen.md)
* [Базовые проекты](projects_k_basisprojekteerweitert.md).

Вы можете самостоятельно создавать шаблоны из проектов, сохранять их в любом каталоге на любом доступном диске, управлять ими.

### Примерные шаблоны

После стандартной установки EPLAN в каталогах ..\Шаблоны\EPLAN и ..\Шаблоны\<Идентификатор фирмы> находятся, например, следующие шаблоны проектов и базовые проекты.

Имя файла |  Значение
---|---
IEC_tpl001.ept |  ***Шаблон проекта:*** Задана структура идентификаторов IEC
Num_tpl001.ept |  ***Шаблон проекта:*** Задана общая нумерация
GOST_tpl001.ept |  ***Шаблон проекта:*** Подходит для создания проектов согласно ГОСТу
IEC_bas001.zw9 |  ***Базовый проект:*** Задана структура идентификаторов IEC. Содержит основные данные, например: библиотеки символов, формы, рамку.
Num_bas001.zw9 |  ***Базовый проект:*** Задана общая нумерация. Содержит основные данные, например: библиотеки символов, формы, рамку.
GOST_bas001.zw9 |  ***Базовый проект:*** Подходит для создания проектов согласно ГОСТу. Содержит основные данные, например: библиотеки символов, формы, рамку.

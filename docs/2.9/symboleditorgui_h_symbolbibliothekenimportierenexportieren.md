# Импортировать / экспортировать библиотеки символов

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Сервисные программы > Основные данные > Библиотека символов > Импорт
2. В диалоговом окне Импортировать библиотеку символов выберите из списка требуемую библиотеку символов (*.esl). Многократный выбор ***невозможен***.
3. Щелкните по кнопке ++Открыть++.
4. В диалоговом окне Создать библиотеку символов из раскрывающегося списка Сохранить в выберите дисковод и каталог, в котором должна быть сохранена импортируемая библиотека символов.
5. Перезапишите имя файла, если это необходимо.
6. Щелкните по кнопке ++Сохранить++.

!!! info "Для сведения:"

    Библиотека символов импортируется и доступна для дальнейшей обработки в текущем проекте.

!!! tip "Совет:"

    Вы можете импортировать библиотеки символов с помощью перетаскивания мышью, перетащив выделенную библиотеку символов из Windows Explorer в диалог навигатора Страницы — <Имя проекта> вашего проекта.

1. Выберите следующие пункты меню: Сервисные программы > Основные данные > Библиотека символов > Экспорт
2. В диалоговом окне Экспортировать библиотеку символов выберите из списка экспортируемую библиотеку символов (*.slk). Многократный выбор ***невозможен***.
3. Щелкните по кнопке ++Открыть++.
4. В диалоговом окне Создать файл экспорта из раскрывающегося списка Сохранить в выберите дисковод и каталог, в котором должна быть сохранена экспортируемая библиотека символов.
5. Перезапишите имя файла, если это необходимо.
6. Щелкните по кнопке ++Сохранить++.

!!! info "Для сведения:"

    Файл экспорта создается под указанным именем на указанном дисководе и в указанном каталоге.

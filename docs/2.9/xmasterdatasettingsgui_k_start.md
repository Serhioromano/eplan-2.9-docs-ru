# Основные данные

Управление основными данными является особенно важным в отношениях с поставщиками и при обмене данными. По этой причине возможно группировать все основные данные по заказчикам и т.п., после чего просто и ясно переключаться.

Под понятием "Основные данные" здесь понимаются все "вспомогательные данные" вне проекта, которые создаются для того, чтобы их использовать в рамках проекта.

В EPLAN к основным данным относятся следующие данные:

* Символы
* Рамки
* Формы для отчетов
* Макросы
* Шаблоны проектов и базовые проекты
* База данных иностранных языков
* Данные изделий включая определения устройств и т.д.
* Определения функций
* Графические файлы
* Файлы DXF / DWG
* Специфическая для EPLAN база данных прав.

Этими данными можно управлять в определенных каталогах, которые могут быть предварительно установлены в меню Параметры > Настройки > Пользователь > Управление > Каталоги.

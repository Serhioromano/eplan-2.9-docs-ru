# Обработать контуры

Чтобы копировать контур со всем его содержимым, сохраните его в другом месте хранения и/или под другим именем.

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Сервисные программы > Основные данные > Контур (...) > Копировать
2. В диалоговом окне Копирование контура выберите дисковод и каталог копируемого контура.
3. Выберите копируемый файл контура. (Многократный выбор здесь ***невозможен***.)
4. Щелкните по кнопке ++Открыть++.
5. В диалоговом окне Создание контура установите место хранения для копии контура.
6. Введите в поле Имя файла информативное имя для нового контура.
7. Щелкните по кнопке ++Сохранить++.

!!! info "Для сведения:"

    Контур копируется, и копия открывается для обработки.

Контуры можно поворачивать под любым углом вокруг точки вращения.

Условие:

Вы открыли контур или создали новый контур.

Поворачивать контур без копии

1. Выберите следующие пункты меню: Обработать > Поворачивать
2. Выберите объекты, которые нужно повернуть, заключив их в прямоугольник.
3. Установите точку, вокруг которой нужно выполнить поворот. Эта точка может быть на контуре или вне контура, например на вспомогательной линии.
4. Задайте угол поворота.

!!! info "Для сведения:"

    Контур поворачивается вокруг указанной точки вращения под заданным углом вращения.

Поворачивать контур с копией

1. Выберите следующие пункты меню: Обработать > Поворачивать
2. Выберите объекты, которые нужно повернуть, заключив их в прямоугольник.
3. Щелчком мыши установите точку, вокруг которой нужно выполнить поворот. Эта точка может быть на контуре или вне контура, например на вспомогательной линии. Отображается предварительный просмотр ожидаемого результата.
4. Задайте угол поворота. Подтвердите ввод, нажав комбинацию клавиш [Ctrl] \+ [Ввод].
Или, удерживая нажатой клавишу ++Ctrl++, установите вторую точку угла поворота щелчком мыши.

!!! info "Для сведения:"

    Копия контура вращается вокруг указанной точки вращения под заданным углом вращения.

Контуры можно отобразить зеркально вокруг задаваемой оси с копией или без нее.

Используйте только доступные для поиска точки на существующих геометрических элементах. Вы также можете использовать вспомогательную линию, которую следует удалить после зеркального отображения.

Условие:

Вы открыли контур или создали новый контур.

Отобразить зеркально без копии

1. Выберите следующие пункты меню: Обработать > Отобразить зеркально
2. Выберите объекты для зеркального отображения, заключив их в прямоугольник.
3. Задайте начальную точку оси зеркального отображения. Отображается предварительный просмотр ожидаемого результата.
4. Задайте конечную точку оси зеркального отображения.

!!! info "Для сведения:"

    Контур зеркально отображается на указанной оси.

Отобразить зеркально с копией

1. Выберите следующие пункты меню: Обработать > Отобразить зеркально
2. Выберите объекты для зеркального отображения, заключив их в прямоугольник.
3. Задайте начальную точку оси зеркального отображения. Отображается предварительный просмотр ожидаемого результата.
4. Нажмите клавишу ++Ctrl++ и задайте конечную точку оси зеркального отображения.

!!! info "Для сведения:"

    Копия контура зеркально отображается на указанной оси.

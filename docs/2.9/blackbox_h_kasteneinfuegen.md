# Вставить черные ящики

Черный ящик представляет собой физический функциональный элемент, состоящий из нескольких графических элементов; все они расположены внутри прямоугольника черного ящика (напр., изображение блока питания). При создании черного ящика программа присваивает ему собственное ОУ.

Условие:

Вы открыли проект и находитесь на странице схемы соединений в Графическом редакторе.

1. Вставить > Блок / Вывод устройства / Монтажная плата > Черный ящик
2. Нарисуйте прямоугольник, который будет представлять блок.
3. В диалоговом окне черного ящика Свойства введите данные устройства.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Черный ящик будет сохранен в проекте со всеми свойствами.

!!! note "Замечание:"

    При включенном захвате сетки (Параметры > Захват сетки) повторно нарисованный черный ящик расположен точно на сетке, на которой он был размещен; если захват сетки отключен, блок можно нарисовать произвольно рядом с сеткой.

!!! tip "Совет:"

    Черный ящик можно отобразить при помощи символов DC2 или DCF2, известных также при черчении ломаных линий. Для этого в диалоговом окне Выбор символа выберите символ и задайте начальную точку блока, щелкнув левой клавишей мыши в схеме соединений. После задания всех остальных пунктов завершите действие через Всплывающее меню Закрыть ломаную линию.

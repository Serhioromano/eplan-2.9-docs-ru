# Вкладка Определенные пользователем свойства

Сервисные программы > Изделие > Управление. В дереве или списке выделено одно изделие.

На этой вкладке вы присваиваете определенные пользователем свойства изделию, которое вы ранее создали и настроили в [диалоговом окне конфигурации](eservicesgui_d_konfigeigenschaften.md) для базы данных изделий. Эти свойства одинаковы для всех вариантов изделия.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать. Или щелкните по кнопке ++...++, чтобы в диалоговом окне Конфигурировать свойства обработать или создать схему.

### [Присвоить схему]

Эта кнопка используется для ввода определенных пользователем свойств, сохраненных в схеме, на вкладке. Если свойства уже назначены, то определенные пользователем свойства прикрепляются из схемы. Если свойство уже выбрано и существует в схеме, его значения во время присвоения не изменяются.

Таблица

В таблице отображаются определенные пользователем свойства, присвоенные изделию.

С помощью кнопки ![](images/all_new_as.png){: .ui-icon } (Создать) для изделия можно свободно выбирать определенные пользователем свойства или присваивать изделию предварительно заданный набор определенных пользователем свойств с помощью кнопки ++Присвоить схему++.

Если впоследствии выбрать изделие или устройство, эти определенные пользователем свойства со значениями будут перенесены в соответствующую главную функцию.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

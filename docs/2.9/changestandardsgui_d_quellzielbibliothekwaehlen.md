# Диалоговое окно Выбрать исходную / целевую библиотеку символов (Изменение стандарта)

Вы открыли проект. При нескольких открытых проектах: В навигаторе страниц выделен проект. Сервисные программы > Изменение стандарта. Вы определили на вкладке Проект все настройки. [Далее]. Вы определили на вкладке Поворот / инверсия все настройки. [Далее]. Вы отметили опцию на вкладке Основные данные в строке библиотеки символов в столбце Присвоить отдельно. [Далее]. Щелкните на вкладке Символы по ++Схема++. Выберите ++Дополнительно++ > Выбрать исходную и целевую библиотеку символов.

Обзор основных элементов диалогового окна:

### Основные данные проекта

Отображает все исходные и целевые библиотеки символов, для которых на вкладке Основные данные установлен флажок Присвоить отдельно.

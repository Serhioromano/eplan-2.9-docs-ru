# Изменение точки наблюдения 3D пространства листа с помощью комбинаций клавиш

Теперь для переключения точки наблюдения 3D можно также использовать комбинации клавиш. Для пунктов меню под путем меню Вид > Точка наблюдения 3D по умолчанию доступны следующие комбинации клавиш:

Команда |  Комбинация клавиш
---|---
Точка наблюдения 3D вверху |  [Ctrl] \+ [Alt] \+ [T]
Точка наблюдения 3D снизу |  [Ctrl] \+ [Alt] + [D]
Точка наблюдения 3D слева |  [Ctrl] \+ [Alt] + [L]
Точка наблюдения 3D справа |  [Ctrl] \+ [Alt] \+ [R]
Точка наблюдения 3D спереди |  [Ctrl] \+ [Alt] + [F]
Точка наблюдения 3D сзади |  [Ctrl] \+ [Alt] \+ [B]
Точка наблюдения 3D, ЮЗ изометрия |  [Ctrl] \+ [Alt] \+ [1]
Точка наблюдения 3D, ЮВ изометрия |  [Ctrl] \+ [Alt] \+ [2]
Точка наблюдения 3D, СВ изометрия |  [Ctrl] \+ [Alt] \+ [3]
Точка наблюдения 3D, СЗ изометрия |  [Ctrl] \+ [Alt] \+ [4]

Вы также можете назначить эти комбинации клавиш для функциональных клавиш мыши Spacemouse.

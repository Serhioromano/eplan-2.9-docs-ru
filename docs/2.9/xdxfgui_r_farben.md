# Вкладка Цвета

* Параметры > Настройки > Пользователь > Интерфейсы > Экспорт и импорт DXF / DWG. Выберите вкладку Цвета.
* Страница > Экспортировать > DXF/DWG. Щелкните рядом с полем Схема на ++...++. Выберите вкладку Цвета.

На этой вкладке определяется присвоение между специфическими для САD и EPLAN цветами в таблице.

Обзор основных элементов диалогового окна:

### EPLAN

В этом поле нажатием ++...++ можно отобразить все цвета в EPLAN и выбрать цвета, которые необходимо обработать. При этом, выбирать можно только те цвета, которые должны быть преобразованы.

### САПР

Здесь указывается соответствующий цвет САD.

!!! example "Пример:"

    Выберите, например, цвет EPLAN "Черная" и присвойте ему цвет САD "250".

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

!!! note "Замечание:"

    Цвета, которые не приведены в этой таблице, переносятся 1:1.

# Диалоговое окно Генерировать клеммы (устройства)

Вы открыли проект. Данные проекта > Клеммники > Навигатор. Выделите ОУ. Выберите пункт всплывающего меню Новые клеммы (устройства).

В данном диалоговом окне для нового или уже имеющегося ОУ можно сгенерировать несколько клемм по определенному образцу нумерации. Клеммы генерируются из шаблонов функций выбранного изделия, при этом образец нумерации определяет обозначения клемм. Каждое устройство имеет по одной главной клемме: этой главной клеммой становится первая сгенерированная клемма, которой присваивается выбранное изделие.

Обзор основных элементов диалогового окна:

### Полное ОУ

В этом поле показывается полное ОУ, которому присвоены новые функции. Показывается ОУ устройства, которое вы выделили в навигаторе. Вы можете изменить ОУ или ввести новое.

### Образец нумерации

Введите в это поле образец нумерации. Образец нумерации определяет обозначения клеммы.

### Номер изделия

В этом поле выводится номер изделия. Кнопка ++...++ открывает диалоговое окно Выбор изделия и обеспечивает выбор изделия клеммы. Изделие должно обладать минимум одним шаблоном функций клеммы, в обратном случае генерировать клеммы невозможно.

### Вариант изделия

В этом поле отображается вариант изделия выбранного изделия.

### Число новых функций

Значение в этом поле указывает, сколько новых функций будет сгенерировано. Число новых функций зависит от образца нумерации.

### Нумеровать с ориентацией на уровни

Если флажок установлен, тогда клеммы нумеруются по уровням. Если флажок не установлен, тогда клеммы нумеруются по клеммам.

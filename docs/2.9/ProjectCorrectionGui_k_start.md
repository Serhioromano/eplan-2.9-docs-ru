# Корректировка проекта

В ходе проектирования часто генерируются данные проекта, которые в конце проектирования требуют корректировки. Поэтому для определенных данных (клемм, штекеров и т. д.) можно запустить корректировочные прогоны, в ходе которых автоматически корректируются обнаруженные ошибки. При этом учитываются только многополюсные функции.

* Вы можете задать, какие данные проекта должны быть откорректированы.
* Данные проекта корректируются для всего проекта. Вы можете определить фильтр для исключения корректировки данных проекта или для корректировки только выбранных уровней иерархии.
* Настройки объема корректировки сохраняются в схеме. Благодаря этому настройки могут использоваться повторно.

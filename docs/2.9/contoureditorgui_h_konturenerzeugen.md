# Создание контуров

При создании контура открывается диалоговое окно Свойства контура - <Название контура>, в котором устанавливаются самые значительные свойства нового контура. (Это диалоговое окно может быть вызвано также впоследствии для ***открытого*** контура, это позволит вам в любое время дорабатывать свойства.)

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Сервисные программы > Основные данные > Контур (...) > Создать
2. В диалоговом окне Создание контура выберите дисковод и каталог, в котором должен быть сохранен новый контур.
3. Введите в поле Имя файла информативное имя для контура.

!!! info "Для сведения:"

    Раскрывающийся список Тип файла при этом задан типом контура ("Выдавить контур" или "Генерирование записи данных контура ЧУ"), который выбирается в соответствующем пункте меню. Это свойство затем ***не***возможно изменить.

4. В поле Путь выберите пункт всплывающего меню Сбросить на стандарт, если требуется вернуться к стандартной настройке дисковода и каталога. (Стандартный каталог является либо специфической предварительной программной настройкой <Дисковод программы>:\<Каталог программы>\Макросы\<Идентификатор фирмы>, либо настройкой, установленной для макросов через пункты меню Параметры > Настройки > Пользователь > Управление > Каталоги.)
5. Щелкните по кнопке ++Сохранить++.
6. В диалоговом окне Свойства контура - <Название контура> выберите из раскрывающегося списка Категория, какие свойства должны быть выведены в списке ниже.
7. Определите значения для свойств, либо введя или перезаписав запись или же выбрав значение по умолчанию из раскрывающегося списка, или перейдите в специальное диалоговое окно, где можно ввести другие данные.
8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Новый контур сохраняется согласно вашим записям, причем расширение имени типа файла зависит от выбранного типа контура и выводится в редакторе контуров.

!!! info "Для сведения:"

    Нулевая точка системы координат выделяется с помощью красного круга. При необходимости нулевую точку можно передвинуть.

При необходимости вы можете проверить контур во время обработки, чтобы быстро заметить неправильные записи и их устранить.

!!! tip "Совет:"

    Вы можете ввести имеющиеся контуры с помощью перетаскивания мыши, если перетяните выделенный контур из проводника Windows в диалоговое окно навигатора Страницы - <Имя проекта> вашего проекта.

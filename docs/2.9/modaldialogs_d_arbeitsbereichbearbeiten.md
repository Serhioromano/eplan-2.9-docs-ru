# Диалоговое окно Обработать раб. область

Конфигурация интерфейса пользователя для рабочей области была изменена.

* Вид > Рабочая область. В диалоговом окне Рабочая область щелкните ![](images/all_edit_as.png){: .ui-icon } (Обработать).
* Параметры > Настройки > Пользователь > Отображение > Рабочая область. В диалоговом окне Настройки: Рабочая область щелкните ![](images/all_edit_as.png){: .ui-icon } (Обработать).

В этом диалоговом окне можно обработать рабочую область.

Обзор основных элементов диалогового окна:

### Шаблон для " ++...++ "

В этом списке перечисляются все уже имеющиеся схемы рабочих областей, причем для каждой схемы выводятся свойства Имя и Описание.

Здесь выберите запись Текущая конфигурация, если интерфейс пользователя текущей рабочей области был изменен, и требуется заново определить текущую схему.

Если в качестве шаблона для текущей рабочей области должна использоваться другая схема, выберите здесь соответствующее имя. В таком случае все настройки текущей рабочей области перезаписываются.

# Разместить функциональные элементы без схемы соединений

С учетом различных методов работы конструкторов и инженеров также возможно выполнить работу по компоновке электрошкафа еще до создания схемы соединений.

Условия:

* Вы открыли проект.
* Открыто диалоговое окно Компоновка электрошкафа 2D - <Имя проекта> (Данные проекта > Устройство / Изделие > Навигатор компоновки электрошкафа 2D).
* Вы находитесь в графическом редакторе на странице типа "компоновка электрошкафа".
* Вы вставили монтажную плату.
* В диалоговом окне Настройки: Компоновка электрошкафа 2D (Всплывающее меню > Настройки) ***не*** выбрана ни одна опция Ввод вручную ++...++.

1. Всплывающее меню > Новое устройство
2. В диалоговом окне Выбор изделия выберите требуемое изделие.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Символ изделия масштабируется до указанного размера (применимы только значения более 0,0) и прикрепляется к курсору.

4. Разместите изделие на монтажной плате.

!!! info "Для сведения:"

    Открывается диалоговое окно Свойства ++...++: Размещение изделия.

5. На вкладке Размещение изделия изделию присваивается обозначение устройства.
6. При необходимости обработайте другие свойства размещения изделия.
7. Щелкните по кнопке ++ОК++.
8. Выполните аналогичные действия для всех других изделий, которые требуется разместить.

При установке изделия выполняется проверка тока, имеет ли текущая страница тип "Компоновка электрошкафа" и находится ли размещение изделия в пределах монтажной платы. Если одно из этих условий нее выполнено, размещение изделия не выполняется и выводится соответствующее сообщение.

При генерировании размещения изделия определение функции выводится из основных данных изделий (вкладка Технические данные в диалоговом окне База данных изделий) следующим образом:

Группа продуктов |  Подгруппа продуктов |  Определение функции
---|---|---
Клеммы |  Несущая шина |  Размещение изделия, клеммник
Клеммы |  (все равно) |  Размещение изделия, клемма
Штекеры |  (все равно) |  Размещение изделия, штекер, штырь / гнездо
ПЛК |  (все равно) |  Размещение изделия, карта ПЛК
(Все группы продуктов кроме клемм / штекеров / ПЛК) |  (все равно) |  Размещение изделия, стандартный функциональный элемент

!!! note "Замечание:"

    Если в диалоговом окне Настройки: Компоновка электрошкафа 2D (Всплывающее меню > Настройки) выбрать опцию Ввод вручную ++...++, перед размещением устройства необходимо также указать его размеры в диалоговом окне Удлинение и интервал.

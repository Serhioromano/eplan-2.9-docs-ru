# Диалоговое окно Рабочая область / Настройки: Рабочая область

* Вид > Рабочая область.
* Параметры > Настройки > Пользователь > Отображение > Рабочая область.

В этом диалоговом окне можно управлять схемами для рабочих областей и обрабатывать их. В такой схеме сохраняется конфигурация интерфейса пользователя, которая была настроена в соответствии с личными предпочтениями или рабочими задачами.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

Помимо этого здесь находится кнопка ![](images/all_edit_as.png){: .ui-icon } (Обработать). С помощью этой кнопки можно перейти в диалоговое окно Обработать раб. область, в котором можно заново сохранить текущую рабочую область при изменении интерфейса пользователя или перезаписать ее с помощью содержимого другой рабочей области.

!!! note "Замечание:"

    При удалении текущей настроенной ***и определенной пользователем*** рабочей области с помощью кнопки ![](images/all_delete_as.png){: .ui-icon } (Удалить), EPLAN переключается на "стандартную" рабочую область.

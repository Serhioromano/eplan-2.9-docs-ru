# Диалоговое окно Настройки: Счетчик области сброса

Вы открыли проект. Вы открыли диалоговое окно [Настройки: Нумерация соединений](wirenumberinggui_d_verbnumeinstellungen.md) (например, через пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Нумерация соединений). В диалоговом окне Настройки: Нумерация соединений вы выбрали определенную пользователем схему и перешли во вкладку Обозначение. Вы выделили запись в поле Группы форматов и щелкнули по ![](images/all_edit_as.png){: .ui-icon } (Обработать), или же по ![](images/all_new_as.png){: .ui-icon } (Создать).

В диалоговом окне Нумерация соединений: Форматы в списке Доступные элементы формата: выбран элемент формата типа "Счетчик" и с помощью кнопки со стрелкой перенесен в список Выбранные элементы формата, или выбран уже имеющийся элемент формата типа "Счетчик" и нажата кнопка ![](images/all_edit_as.png){: .ui-icon } (Обработать).

В диалоговом окне Формат: Счетчик установлен флажок Сброс на одну структуру / страницу. Щелкните кнопку ++Настройки++.

В данном диалоговом окне задайте область для счетчика, в пределах которой счетчик будет считать по нарастающей. При изменении одного из заданных критериев счетчик будет сброшен на начальное значение.

При обозначении соединений можно, например, задать, что обозначения соединений в пределах такой области сброса должны быть уникальными.

Обзор основных элементов диалогового окна:

### Функциональное присвоение ... Определено пользователем

Если в поле установлен флажок, то счетчик сбрасывается на начальное значение, как только структурный идентификатор изменяется для соответствующего блока идентификаторов.

### Подидентификатор

Если в поле установлен флажок, то счетчик сбрасывается на начальное значение, как только подидентификатор изменяется для соответствующего блока идентификаторов.

!!! note "Замечание:"

    Здесь ***невозможно*** обработать знаки и разделители между отдельными структурными идентификаторами; эти знаки и разделители обусловлены настройками диалогового окна Расширенные структуры проекта, которое вы можете открыть, нажав на вкладке Структура свойств проекта кнопку ++Другие++.

### Страница

Если установлен флажок, то счетчик сбрасывается на начальное значение при изменении страницы.

### Подстраница

Если установлен флажок, то счетчик сбрасывается на начальное значение при изменении подчиненной страницы.

### От источника

Если выбран данный параметр, заданные выше критерии используются при исходных функциях соединений. Т.е. область изменяется, если изменяется критерий у источника соединения.

### От цели

Если выбран данный параметр, заданные выше критерии используются при целевых функциях соединений. Т.е. область изменяется, если изменяется критерий у цели соединения.

# Диалоговое окно Конфигурация дерева — Главные узлы

* Сервисные программы > Изделие > Управление > [Дополнительно] > Настройки.
* Параметры > Настройки > Пользователь > Управление > Изделия.

Щелкните в поле Конфигурация дерева по ++...++. Щелкните в диалоговом окне Конфигурация дерева по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать) на панели инструментов над списком главных узлов, или нажмите ++...++ в строке уже определенных главных узлов.

В этом диалоговом окне укажите, как должны конфигурироваться отдельные главные узлы представления в виде дерева базы данных изделий.

Обзор основных элементов диалогового окна:

### Тип записи данных

Выберите из этого раскрывающегося списка, какой тип данных, относящихся к изделиям (например, клиенты, схемы сверления и т. д.), следует использовать для главного узла представления в виде дерева. Соответствующий тип записи данных, для которого был создан главный узел, отображается в диалоговом окне Конфигурация дерева в столбце Тип записи данных.

### Имя

Введите здесь имя главного узла, как он должен отображаться в представлении в виде дерева базы данных изделий.

### Фильтр

Это поле показывает фильтр, который задается с помощью кнопки ++...++ в диалоговом окне Фильтр.

### Свойства

Этот список содержит свойства, присвоенные текущему узлу. Свойства зависят от типа записи данных, выбранного в поле Тип записи данных.

# Создать шаблоны проектов

[Шаблоны проекта](projects_k_projektvorlagen.md) можно создавать с помощью меню проектов или управления проектами. Можно использовать шаблон проекта для создания проекта. При создании шаблона из имеющегося проекта сохраняются все специфические для проекта настройки и страницы в шаблоне проекта.

Условие:

Вы открыли проект.

Меню проектов: Проект > Организовать > Создать шаблон проекта.

Управление проектами: [Дополнительно] > Создать шаблон проекта.

1. В диалоговом окне Создать шаблон проекта укажите место хранения и имя для шаблона проекта.

!!! info "Для сведения:"

    В качестве места хранения EPLAN предлагает каталог шаблонов, указанный в настройках пользователя для основных данных в диалоговом окне Настройки: Каталоги.

2. Щелкните по кнопке ++Сохранить++.

!!! info "Для сведения:"

    Будет создан шаблон проекта *.ept.

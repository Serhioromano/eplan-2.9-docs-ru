# Диалоговое окно Присвоить страницы

Вы открыли проект. Страница > Импортировать > DXF / DWG. Выберите в диалоговом окне Выбор файла DXF/DWG (как минимум) один файл и щелкните по ++Открыть++. При необходимости измените в диалоговом окне Импорт DXF/DWG настройки импорта и начните импорт нажатием ++OK++.

В этом диалоговом окне задается порядок обработки новых страниц проекта.

Обзор основных элементов диалогового окна:

### Цель

Данные для импорта зависят от структуры страниц целевого проекта. Доступны следующие столбцы:

* Перезаписать: установите этот флажок, чтобы определить, будет ли перезаписываться уже имеющаяся страница при импорте файла DXF / DWG.
* Установка / место установки / ...: щелкните ++...++ в полях установки, места установки и т. д., чтобы выбрать другую установку / другое место установки и т. д. из списка имеющихся в данном проекте.
* Имя страницы: в этом поле можно вручную перезаписать предварительную настройку.
* Доп. поле номера листа: Номер листа является дополнительным свойством страницы для имени страницы. При необходимости укажите здесь дополнительную информацию для импортированной страницы.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

### [Нумерация]

Открывает диалоговое окно [Нумеровать страницы](pagebrowsergui_d_seitennummerierung.md), в котором можно автоматически задать имена страниц.

!!! note "Замечание:"

    Если в меню Параметры > Настройки > Пользователь > Интерфейсы > Экспорт и импорт DXF / DWG на вкладке Импорт выбран параметр Корректировка вручную, перед началом самой операции импорта откроется диалоговое окно Форматир. импорта, где вы сможете дополнительно обработать автоматически полученные границы чертежа.

# Диалоговое окно Настройки: Точки разрыва

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Перекр. ссылки / образы контактов> Точки разрыва.

Для перекрестных ссылок на точках разрыва могут быть установлены специальные настройки, которые воздействуют на актуальный проект.

Доступны следующие элементы диалогового окна:

### Отображение

В этом групповом поле можно ввести следующие данные:

* По строкам/по столбцам: Выберите желаемую опцию для представления перекрестных ссылок на точках разрыва.

* Число на строку / столбец: Задайте здесь, сколько перекрестных ссылок должно отображаться на строку или на столбец. если их количество больше, чем предварительная настройка "1", то перекрестные ссылки точек разрыва отображаются в матрице.

!!! example "Пример:"

    Представление По строкам,Число на строку / столбецзадается равным 3:0 1 23 4 56Если представление изменяется на По столбцам, отображение перекрестных ссылок выглядит следующим образом:0 3 61 42 5

### Отобразить перекрестную ссылку

Если этот флажок активирован, отображаются перекрестные ссылки на точках разрыва.

### Отображать в перекр. ссылке только имена страниц

В системе EPLAN на определенной исходной точке разрыва может выводиться обзор всех одноименных точек разрыва проекта схемы соединений. У вас есть возможность подавлять в обзоре при помощи этих настроек отображение данных о столбцах и строках: если этот флажок активирован, перекрестная ссылка будет отображаться только с именем страницы (без данных о столбцах и строках).

!!! example "Пример:"

    Нормальная перекрестная ссылка7.8/L-; если активирован флажок Отображать в перекр. ссылке только имена страниц, то она представлена как7/L-.

!!! note "Замечание:"

    Установить или снять этот флажок можно только в том случае, если флажок настройки Отобразить перекрестную ссылку ***установлен***.

### Полностью отображать в перекрестной ссылке структурные идентификаторы

Если этот флажок установлен, дополнительно к информации о страницах и / или зоне (строка / столбец) в перекрестной ссылке отображается полная структура обозначений точки разрыва с перекрестной ссылкой.

!!! example "Пример:"

    В проекте используютсяблоки идентификаторов"Установка" и "Место установки". Структурный идентификатор для блока идентификаторов "Установка" имеет вид:AA1,структурный идентификатордля блока идентификаторов "Место установки" имеет вид:BC1.В проекте на страницесхемысоединений с полным именем страницы=AA1+BC1/1в столбце 8 находится точка разрыва-L1. Одноименный обратный эквивалент точки разрыва находится на странице схемы соединений с полным именем страницы=AA1+BC1/2в столбце 5. Между обеими точками разрыва строится перекрестная ссылка:Если флажок Полностью отображать в перекрестной ссылке структурные идентификаторыустановлен, на точке разрыва на странице=AA1+BC1/1в перекрестной ссылке отображается запись=AA1+BC1/2.5. На обратном эквиваленте на странице проекта=AA1+BC1/2перекрестная ссылка, соответственно, имеет вид=AA1+BC1/1.8. Если снять флажок, в обеих перекрестных ссылках будет отображаться только информация о странице и / или зоне без структурных идентификаторов.

!!! note "Замечание:"

    Установить или снять этот флажок можно только в том случае, если флажок настройки Отобразить перекрестную ссылку ***также установлен***.

### Отображение цели на парных перекрестных ссылках

если этот флажок активирован, то отображается цель на парных перекрестных ссылках, причем в основном всегда сначала отображается (первая) цель, а затем перекрестная ссылка (например, L1/ X1:1 /7.04).

### Отображение цели на перекр. ссылках типа звезда

если этот флажок активирован, отображается цель на перекр. ссылках типа звезда.

!!! example "Пример:"

    Точка разрыва как источник типа звезда с перекрестной ссылкой типа "звезда" (за перекрестной ссылкой следуют данные соответствующего целевого выводаустройства)

### Представить цель под перекр. ссылкой

Если этот флажок активирован, отображается цель под перекрестной ссылкой.

### Разделитель между именем и перекр. ссылкой

На точках разрыва страница и столбец могут отображаться как целевая стрелка. Задайте здесь символ, который должен разделять обозначение точек разрыва от обозначения строки и столбца перекрестной ссылки.

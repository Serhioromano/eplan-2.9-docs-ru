# Вставить файлы DXF / DWG в графическом редакторе

Условие:

В Графическом редакторе открыта страница проекта.

1. Вставить > Графика > DXF / DWG
2. В диалоговом окне Выбор файла DXF / DWG выберите необходимый файл и щелкните ++Открыть++. (Многократный выбор здесь ***невозможен***.)
3. В поле Схема диалогового окна Импорт DXF/DWG выберите из раскрывающегося списка подходящую схему. Или щелкните ++...++, чтобы вызвать диалоговое окно Настройки: Экспорт и импорт DXF / DWG. Здесь можно посмотреть настройки стандартной схемы, создать и обработать собственные схемы.
4. Щелкните в диалоговом окне Импорт DXF / DWG на ++OK++.
5. Если в меню Параметры > Настройки > Пользователь > Интерфейсы > Экспорт и импорт DXF / DWG на вкладке Импорт выбран параметр Корректировка вручную, откроется диалоговое окно Форматир. импорта, и вы сможете дополнительно обработать автоматически полученные границы чертежа.
6. Щелкните по кнопке ++OK++.
7. С помощью мыши разместите график на текущей странице.

!!! note "Замечание:"

    Вставку файлов DXF / DWG на страницах можно также выполнить в редакторе символов, форм или рамок. Выберите следующие команды меню: Вставить > Графика > DXF / DWG.

!!! tip "Совет:"

    Вы можете вставить файлы DXF / DWG с помощью функции перетаскивания мышью Drag & Drop, перетащив выделенный файл DXF / DWG из Windows Explorer в графический редактор вашего проекта.

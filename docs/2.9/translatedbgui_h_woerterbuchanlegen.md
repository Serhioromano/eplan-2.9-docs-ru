# Создать словарь

Вы можете создать и использовать несколько разных словарей. Выберите для своих проектов соответствующий словарь.

1. Выберите следующие пункты меню: Сервисные программы > Перевод > Обработать словарь
2. На вкладке Управление выберите параметр Access, чтобы потом выбрать файл базы данных *.mdb с помощью кнопки ++...++ в интерактивном режиме.
Или выберите параметр Сервер SQL и одну из 15 последних использованных баз данных SLQ из раскрывающегося списка, при этом последние использованные базы данных находятся в верхней позиции.
3. Чтобы генерировать новую базу данных для словаря, щелкните ![](images/all_new_as.png){: .ui-icon } (Создать) рядом с полями Access / Сервер SQL.
4. Щелкните по кнопке ++Сохранить++.

### Добавить язык словаря

1. На вкладке Управление выберите с помощью кнопки ++...++ словарь, в который необходимо добавить язык словаря.
2. Рядом с полем Языки словаря щелкните ![](images/all_new_as.png){: .ui-icon } (Создать).

!!! info "Для сведения:"

    EPLAN покажет все языки, которые можно добавить в словарь.

3. Последовательно выберите все языки, которые имеются в словаре.

# Вкладка Обработать слова / Выбрать слова

Сервисные программы > Перевод > Обработать словарь. Выберите вкладку Обработать слова.

Вы открыли диалоговое окно "Свойства" для условного обозначения, текста, страницы проекта и т. д. В многоязычном поле выберите пункт всплывающего меню Многоязычный ввод. В диалоговом окне Многоязычный ввод щелкните по кнопке ++Словарь++.

На этой вкладке вы можете включать в словарь новые ключевые слова с текстами их перевода и изменять уже имеющиеся слова и их переводы. Словарь различает прописные/строчные буквы ключевых слов.

Обзор основных элементов диалогового окна:

### Полнотекст. фильтр

В данном поле введите текст для поиска и щелкните по ![](images/all_search_as.png){: .ui-icon } (Поиск). После этого в списке ключевых слов отображаются только записи, содержащие искомый текст.
Нажмите кнопку ![](images/all_delete_as.png){: .ui-icon } (Удалить), чтобы удалить искомое понятие и вернуться к отображению всех ключевых слов.

!!! note "Замечание:"

    ***Не*** используйте заполнители (такие как "*", "?" или "_"), так как при данном виде поиска они ***не*** учитываются. Эти ограничения зависят от поисковых механизмов используемых баз данных. Кроме того, при вводе не учитывается регистр, а также не поддерживаются операции И и ИЛИ.

### Исходный язык

Выберите один из языков словаря в качестве исходного языка ключевого слова, которое вы хотите изменить. Под исходным языком понимается язык, на котором ключевое слово сохранено в словаре.

### Пoиск

В этом поле вы можете осуществлять поиск всех ключевых слов, которые уже имеются в словаре. Дополнительно используя полнотекстовый фильтр, в отфильтрованном списке можно выполнять поиск по ключевым словам.
Чтобы найти нужное ключевое слово, введите его в поле ввода. Чтобы отобразить все тексты переводов для слова, щелкните на соответствующем слове в списке ключевых слов.

### Ключевое слово

В этом поле указаны все тексты переводов выбранного ключевого слова. Для каждого языка словаря отображается отдельная строка. Ключевое слово должно иметь перевод на исходном языке, так как оно сохранено в словаре под этой записью. В последней строке можно ввести комментарий. Комментарий необходим для тех слов, которые сохранены в словаре дважды, но под разными значениями. Комментарий отображается вместе с предлагаемыми вариантами перевода слова в диалоговом окне Найденные слова.

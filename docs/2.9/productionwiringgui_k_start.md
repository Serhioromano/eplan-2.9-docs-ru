# Генерировать данные изготовления для машин по сборке проводов

EPLAN Pro Panel позволяет выводить данные изготовления для машин по сборке проводов от различных производителей для дальнейшей передачи производителю услуг:

* Rittal — Wire Terminal WT
* Komax
* CadCabel
* Metzner (Triathlon Kabelcenter)
* Schleuniger (Easy Production Server)
* Steinhauer (Personal Wiring Assistant)

Каждый из этих форматов должен быть лицензирован как отдельный интерфейс.

Кроме того, можно сконфигурировать и вывести 'Общую спецификацию проводов'. Экспортированный файл может использоваться сборочными машинами, интерпретирующими данный формат, для автоматического обмера в длину, маркировки и сборки концов жил кабелей. В спецификации проводов содержится вся информация о соединениях между устройствами, их номера изделия, положение и длина:

* Номер провода.
* Стартовое устройство с однозначным ОУ.
* Вывод устройства данного устройства.
* Целевое устройство с однозначным ОУ.
* Вывод устройства данного устройства.
* Номер изделия провода.
* Длина провода.
* Стартовое направление из вывода устройства первого устройства.
* Конечное направление к выводу устройства второго устройства
* Номер объединенных проводов.

Во всех случаях экспорта машин из EPLAN Pro Panel в качестве заключительного сообщения отображается диалоговое окно, в котором перечисляются конкретные данные об экспортируемых компонентах или указываются причины, почему компоненты не могут быть экспортированы. В диалоговом окне отображается следующая информация:

* Имя экспортированного формата
* Имя использованной схемы и настроенного пути вывода
* Количество экспортированных компонентов
* Альтернативные варианты экспорта
* Перечень неэкспортированных компонентов с указанием причины, по которой они не были экспортированы.
* Указание на возможно выводившиеся системные сообщения.

!!! note "Замечание:"

    Для подготовки экспортированных с помощью EPLAN данных изготовления для машины по сборке проводов ***Rittal Wire Terminal WT*** требуется шаблон импорта, который можно создать с помощью программного обеспечения ***Wire Cockpit***.

# Заменить символы в объектах-заполнителях посредством наборов значений

Теперь в объектах-заполнителях можно заменять символы путем выбора наборов значений. Для этого на вкладке Присвоение для выбора свойств функций доступно новое свойство Символ (ид. 20575). Если для этого свойства введена переменная, на вкладке Значения щелчком мыши можно перейти к выбору другого символа для набора значений.

![](images/benefit.png)Эффект:

Благодаря тому, что символы можно менять при помощи наборов значений, требуется меньше вариантов макросов.

# Вставить вилки соединений и линии кабелей в схемы подключений

Условия:

* Вы открыли проект.
* В редакторе форм открыта форма ***Схема подключения***, свойства которой требуется обработать (Сервисные программы > Основные данные > Форма > Открыть).

### Добавить соединительные вилки

Чтобы сделать представление соединений наглядным, соединения лежащих рядом функциональных элементов могут объединяться и представляться графически в форме двусторонней "вилки", при этом на каждой стороне вилки столько "зубцов", сколько было объединено отдельных соединений. Для этого имеется свойство Графика соединения (ID 13048), которое может вставляться в форму в качестве заполнителя для внутренних и внешних кабелей. Индекс заполнителей дает всякий раз степень схемы подключения. Растяжение вилки определяется блоком выравнивания заполнителя; а высота зубцов определяется размером шрифта заполнителя. Выравнивание вилки задается автоматически выравниванием формы.

1. Вставить > Текст заполнитель
2. На вкладке Размещение диалогового окна Свойства (текст заполнитель) выберите параметр Свойство и щелкните ++...++.
3. В диалоговом окне Тексты заполнителей — <Тип формы> в поле Элемент дважды щелкните по одному из элементов "Кабель внутр. 'степень (1–3)'" или "Кабель внешн. 'степень (1–3)'".
4. Выберите запись.
5. Выберите свойство Графика соединения и щелкните ++ОК++.
6. Задайте представление текста-заполнителя в форме на вкладке Формат диалогового окна Свойства (текст заполнитель).
7. Щелкните по кнопке ++OK++.
8. Разместите текст-заполнитель с помощью мыши.

### Добавить линии кабелей

Для графического представления линий кабелей на схеме подключения имеется свойство Графика кабеля (ID 13107), которое может вставляться в форму для внутренних и наружных кабелей, при этом индекс заполнителей дает степени схемы подключения. Расширение кабельной линии устанавливается блоком выравнивания заполнителя. При этом если высота его равна "0", линия кабеля чертится перпендикулярно соединению, если же высота блока выравнивания не равна "0", линия кабеля чертится под углом. Несколько лежащих рядом соединений могут объединяться, если относятся к одному и тому же кабелю, при этом длина линии кабеля не меняется.

1. Выполните действия, описанные выше вплоть до шага 5.
2. Выберите свойство Графика кабеля и щелкните ++ОК++.
3. Задайте представление текста-заполнителя в форме на вкладке Формат диалогового окна Свойства (текст заполнитель).
4. Щелкните по кнопке ++OK++.
5. Разместите текст-заполнитель с помощью мыши.

!!! note "Замечание:"

    В формах с несколькими столбцами и строками одновременно это свойство использоваться не может.

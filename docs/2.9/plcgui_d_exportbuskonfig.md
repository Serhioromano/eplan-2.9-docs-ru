# Диалоговое окно Экспортировать данные ПЛК

Вы открыли проект. Данные проекта > ПЛК > Экспортировать данные.

В этом диалоговом окне экспортируются все данные ПЛК проекта (т.е. данные всех действительных блоков и выводов устройства ПЛК) в файл XML. Ими можно, например, обмениваться с различными программами конфигурации ПЛК.

Данные различных размещений одной функции при этом обощаются и при импорте снова распределяются на размещения. Для блоков ПЛК экспортируются только данные главной функции.

Свободные шаблоны функции не экспортируются. Записи В/В из списков присвоений также не экспортируются.

Обзор основных элементов диалогового окна:

### Проекты конфигураций

В раскрывающемся списке показываются все проекты конфигураций. Выберите нужный проект. Экспорт учитывает все карты со всеми их выводами устройств, которые либо на блоке ПЛК, либо на своих выводах кабеля сети / шины содержат соответствующее значение в поле Проект конфигураций.

### Язык

В этом раскрывающемся списке представлены все языки, определенные в настройках перевода проекта. Выберите язык, на котором экспортируются данные ПЛК. Данные всегда выдаются на одном языке.

При [обмене данными ПЛК в формате AutomationML](plcgui_k_amlbusdatenaustausch.md) экспортируются все языки. Установленный здесь язык является основным языком в файле обмена.

### Программа конфигурации ПЛК

В раскрывающемся списке отображаются все доступные комбинации производителей и программ конфигурации ПЛК. Выберите нужную запись.

### Формат файла экспорта

В раскрывающемся списке отображаются поддерживаемые форматы программы конфигурации ПЛК, выбранной в поле Программа конфигурации ПЛК. Выберите требуемый формат.

### Имя файла / Путь файла

В этом поле введите имя, под которым сохраняется файл экспорта. Через [...] откройте диалоговое окно выбора файла и там можно изменить в любой каталог.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

Если в поле Программа конфигурации ПЛК выбрана запись Siemens SIMATIC STEP 7 5.6, то следует указать не имя файла, а путь к файлу. С помощью кнопки ++...++ откройте диалоговое окно Выбрать каталог, чтобы выбрать каталог. В указанном каталоге при экспорте будет создано по одному файлу на рабочую станцию, причем в качестве имени файла будет использовано имя соответствующей рабочей станции. (В одном проекте конфигураций может содержаться несколько рабочих станций, имена рабочих станций при этом указываются как свойства блока ПЛК.) Если в указанном каталоге уже имеется одноименный файл, появится запрос, необходимо ли перезаписать этот файл.

### [Параметры]

Эта кнопка активна, если в поле Формат файла экспорта выбран формат AutomationML. В [следующем диалоговом окне](plcgui_d_optionsaml.md) укажите степень детализации для файла экспорта в формате AutomationML.

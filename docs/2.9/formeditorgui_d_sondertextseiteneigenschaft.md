# Диалоговое окно Свойства (спец. текст): Свойства страницы

Вы открыли проект. Сервисные программы > Основные данные > Рамка / Форма > Открыть > [Открыть].

* Вставить > Спец. текст > Свойства страницы.
* Вы дважды щелкнули по свойству страницы (или выбрали Всплывающее меню > Свойства; в этом случае могут быть выделены также несколько свойств страницы).

В этом диалоговом окне выберите свойство страницы, которое должно быть добавлено в форму или рамку, и установите его свойства для формы, языка и рамки. Эти специальные тексты заполняются в соответствии с введенными данными программы.

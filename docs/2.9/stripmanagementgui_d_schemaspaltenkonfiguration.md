# Диалоговое окно Настройки: Конфигурация столбцов (клеммы)

Вы открыли проект. Вы открыли диалоговое окно [Обработать клеммник](stripmanagementgui_d_klemmenleistebearbeiten.md). Щелкните по ++...++ рядом с полем Схема.

В этом диалоговом окне установите, какие столбцы будут отображаться для данных клемм.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

!!! note "Замечание:"

    Схемы запоминаются отдельно для каждого пользователя. Чтобы передать схему вместе с проектом, схему нужно экспортировать и передать отдельно.

### Конфигурация столбцов

В этом поле вы определяете, какие столбцы будут отображаться в таблице диалогового окна Обработать клеммник. Отметьте соответствующую опцию. С помощью кнопки со стрелкой вы можете изменять порядок столбцов.

!!! tip "Совет:"

    Установите флажок Предв. просмотр, чтобы отобразить схематический предварительный просмотр клемм. Кроме того, это облегчает задачу распознавания различных типов клемм, типов потенциалов и многоуровневых клемм.

# Операция: export3D

**Параметр** |  **Описание**
---|---
TYPE  |  Вид задачи, которую должна выполнить операция:
STEPINSTALLATIONSPACES: Экспортировать пространство листа в формате STEP.
PROJECTNAME  |  Имя проекта с полным путем файла (является опцией).
Если не задано, то выбранный проект используется, когда операция вызывается через интерфейс пользователя (напр., через сценарий или панель инструментов). При вызове из командной строки Windows следует определить PROJECTNAME, или сначала следует использовать ProjectAction. В противном случае отобразится системное сообщение.
INSTALLATIONSPACENAME  |  Имя экспортируемого пространства листа. Этот параметр оказывает влияние, только когда параметр TYPE имеет следующее значение: STEPINSTALLATIONSPACES.
INSTALLATIONSPACENAMEn  |  Имена экспортируемых пространств листа (является опцией) При этом n — это номер, например /INSTALLATIONSPACENAME1:LS1 /INSTALLATIONSPACENAME1:LS4 /INSTALLATIONSPACENAME1:LS7 и т. д. Этот параметр оказывает влияние, только когда параметр TYPE имеет следующее значение: STEPINSTALLATIONSPACES.
STRUCTURE  |  Структурный идентификатор экспортируемого пространства листа (является опцией). Этот параметр оказывает влияние, только когда параметр TYPE имеет следующее значение: STEPINSTALLATIONSPACES.
STRUCTUREn  |  Структурный идентификатор экспортируемых пространств листа (является опцией). При этом n — это номер, например /INSTALLATIONSPACENAME1:LS1 /STRUCTURE1:=EB3+ET1 /INSTALLATIONSPACENAME2:LS2 /STRUCTURE2:=EB3+ET2 и т. д. Этот параметр оказывает влияние, только когда параметр TYPE имеет следующее значение: STEPINSTALLATIONSPACES.
DESTINATIONPATH  |  Целевой каталог.
Для STEPINSTALLATIONSPACES: каталог, в котором сохраняются файлы STEP. Если этого каталога еще нет, то он создается.
EXPORTSCHEME  |  Схема для экспорта STEP пространств листа (является опцией). Если данный параметр отсутствует или пуст (""), то используется последняя использованная схема экспорта STEP. Этот параметр оказывает влияние, только когда параметр TYPE имеет следующее значение: STEPINSTALLATIONSPACES.
SEPARATEFILES  |  Для TYPE = STEPINSTALLATIONSPACES: если значение равно 1, несколько пространств листа экспортируются в отдельные файлы. Если значение равно 0, генерируются отдельные файлы, имена которых генерируются на основе свойства Имя пространства листа.

!!! note "Замечание:"

    Если указанное имя пространства листа неоднозначно в проекте, а параметр STRUCTURE не задан, выводятся все пространства листа с одинаковым именем.

!!! example "Пример:"

    export3d
    /TYPE:STEPINSTALLATIONSPACES
    /PROJECTNAME:C:\Projects\EPLAN\DEMO2_D.elk
    /INSTALLATIONSPACENAME1:BR1 /STRUCTURE1:=EB3+ET1 /INSTALLATIONSPACENAME2:BR2 /STRUCTURE2:=EB3+ET2
    /DESTINATIONPATH:C:\temp\step_export

# Вкладка Свойства устройства / Свойства изделия

* Страница > Экспортировать > PDF. Щелкните в поле Схема диалогового окна Экспорт PDF[...]. Выберите вкладку Свойства устройства или Свойства изделия.
* Параметры > Настройки > Пользователь > Интерфейсы > Экспорт PDF. Выберите вкладку Свойства устройства или Свойства изделия.

Во вкладке Свойства устройства и Свойства изделия указываются, какие свойства в какой последовательности выводятся страницы PDF с данными устройств.

Обзор основных элементов диалогового окна:

### Таблица

В столбце Свойство показаны свойства, которые по умолчанию выводятся для PDF-файла. В поле Имя отображаются предварительно определенные имена свойств. Эти многоязычные тексты также можно обработать.

Свойства, которые в диалоговом окне "Свойства" "пустые" и имеют значение "0" либо устройство вовсе его не имеет (например, Свойство Тип кабеля на плавком предохранителе), в PDF ***не*** перечисляются.

### Развернуть узел/развернуть модуль

Только для ***Свойства изделия***!

При установке одного из этих флажков разворачиваются узлы / модули, и устройству присваиваются номера изделий отдельных составных частей узла / модуля.
При снятии флажка устройству присваивается номер изделия узла/модуля.

По умолчанию флажок снят. Потом при создании отчетов можно решить, разворачиваются ли узлы/модули (например, должны ли появиться в спецификации номера изделия и заказа узла/модуля или отдельного изделия).

### До степени

Если установлен флажок Развернуть узлы и/или Разъединить модули, укажите до какой глубины вложения должны разворачиваться узлы.

# Диалоговое окно Экспортировать настройки ++...++: <Путь иерархии>

Параметры > Настройки. Выделите одну из областей настроек "Проект" или "Фирма" и нажмите кнопку ![](images/all_export_as.png){: .ui-icon } (Экспортировать).

В этом диалоговом окне определяется, в какой целевой файл будут экспортированы настройки выбранной области настройки.

Обзор основных элементов диалогового окна:

### Целевой файл

В этом поле выводится предложение пути и имени файла экспорта. По умолчанию файл экспорта получает имя, которое соответствует выделенной области в дереве диалогового окна Настройки. Имя файла состоит из следующих элементов: <Основная категория>+<Подкатегория>+<Область настройки>.xml. Нажав кнопку ++...++, можно указать другой путь или имя файла экспорта.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

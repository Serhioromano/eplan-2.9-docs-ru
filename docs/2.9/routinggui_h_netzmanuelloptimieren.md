# Оптимизировать сеть вручную

В многополюсном планировании сети можно соединить:

* через функции, для которых включено свойство Соединенный с сетью,
* через передачу потенциала
* или через перемычку.

Логика передачи по сети через оптимизацию сети в ручном режиме может использоваться и для размещения изделий. Генерируются новые цепочки жил. Фиксированные соединения (перемычки, прямые соединения, внутренние соединения) остаются; их нельзя оптимизировать.

Условие:

Сгенерированы маршрутизируемые соединения. В навигаторе пространства листа или в пространстве листа выделена ***одна*** полная сеть.

1. Выберите пункты меню Данные проекта > Соединения > Оптимизировать сеть вручную.

!!! info "Для сведения:"

    Все подлежащие объединению размещения изделий помечаются красным цветом.

!!! info "Для сведения:"

    Неразделенные размещения изделий представлены прозрачными.

!!! info "Для сведения:"

    Все невидимые источники или цели отображаются.

2. Последовательно щелкните на всех красных размещениях изделий, которые нужно соединить друг с другом.

!!! info "Для сведения:"

    Все размещения изделий, имеющие только один вывод устройства в сети и два соединения, выделяются зеленым цветом.

!!! info "Для сведения:"

    Размещения изделий, имеющие один вывод устройства в сети и одно соединение, выделяются желтым цветом.

!!! info "Для сведения:"

    Если размещение изделия имеет в сети несколько выводов устройств, то при щелчке на нем выводится список выводов устройств, в котором можно выбрать один вывод устройства.

!!! info "Для сведения:"

    Соединения генерируются в той же последовательности, в которой вы щелкаете мышью.

3. Чтобы определить новую цепочку, нажмите клавишу ++N++.

!!! info "Для сведения:"

    Размещения изделий с перемычками, выделенные красным цветом, а также находящиеся в начале и в конце размещения, имеющие только одно подключенное соединение, выделяются желтым цветом. Их можно использовать еще раз для того, чтобы замкнуть кольцо или начать новую цепочку.

!!! info "Для сведения:"

    После щелчка на последнем выделенном красным цветом размещении изделия выполняется проверка, является ли сеть полностью взаимосвязанной.

!!! info "Для сведения:"

    Отображается диалоговое окно Оптимизировать сеть вручную, в котором можно указать, следует ли замкнуть последнюю цепочку в кольцо и нужно ли маршрутизировать новые соединения.

4. В диалоговом окне Оптимизировать сеть вручную нажмите кнопку ++ОК++.

!!! info "Для сведения:"

    Сгенерированные соединения будут маршрутизированы по новой трассе маршрутизации.

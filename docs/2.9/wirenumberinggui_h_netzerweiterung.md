# Определить сетевое расширение для нумерации соединений

Сеть в EPLAN является подмножеством сигнала. Обычно сеть заканчивается клеммами, контактами штекера или выводами устройств функций. Через сетевое расширение сеть в пределах сигнала, например, можно распространить через клеммы и контакты штекеров. Для этого в диалоговом окне Сетевое расширение требуется указать, какие функции ***не*** должны рассматриваться как границы сети. Эта настройка влияет на обозначение, если в качестве объема выбрана "сеть". Сетевое расширение для нумерации соединений не зависит от свойства Соединенный с сетью соответствующей функции.

Условие:

Вы открыли проект.

1. Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Нумерация соединений
2. Выберите в диалоговом окне Настройки: Нумерация соединений определенную пользователем схему из раскрывающегося списка или создайте с помощью команды ![](images/all_new_as.png){: .ui-icon } (Создать) новую схему.
3. Выберите вкладку Обозначение и щелкните на ++Сетевое расширение++.
4. Выделите в диалоговом окне Сетевое расширение необходимые функции в списке Выбор и перенесите их с помощью клавиши со стрелкой в список Сетевое расширение через следующие объекты.

!!! info "Для сведения:"

    В этом случае сеть не заканчивается на выбранных функциях, а продолжается за ними.

!!! info "Для сведения:"

    Сетевое расширение действительно для всех групп форматов, которые имеют объем "сеть".

5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Настройка сохраняется в выбранной схеме.

# Диалоговое окно Запись данных контура ЧУ

Вы открыли проект. В редакторе контуров открыт контур типа 'Генерирование записи данных контура ЧУ' (например, через пункты меню Сервисные программы > Основные данные > Контур (данные ЧУ) > Открыть). При этом чертеж контура содержит только единичный непересекающийся закрытый контур. Сервисные программы > Основные данные > Контур (данные ЧУ) > Генерировать запись данных > Rittal - Perforex BC / Secarex или ... > Rittal - Perforex LC / Secarex или ... > ЧУ Steinhauer.

В диалоговом окне для чертежа контура задайте релевантные машинно зависимые данные. Данные сохраняются на контуре.

!!! note "Замечание:"

    Обратите внимание на то, что для одного контура может существовать несколько машинных конфигураций. На машину разрешается только одна запись данных ЧУ.

Обзор основных элементов диалогового окна:

### Машина

Выберите из этого раскрывающегося списка машину, для которой необходимо создать запись данных ЧУ.

### Описание

Данное поле можно использовать для дополнительных указаний для записи данных.

### Фреза

(Только для Rittal - Perforex BC / Secarex и ЧУ Steinhauer)

Выберите из этого раскрывающегося списка фрезерный инструмент.

* * *

Групповое поле 'Способ пуска'

Через параметры данного группового поля задайте длину пути:

* Касательная (дуга): Выберите данный параметр, если длина пути является касательной, т. е. должна проходить вдоль оси поворота.
* Вертикальная (прямая): Выберите данный параметр, если длина пути является вертикальной, т. е. должна проходить сверху или снизу, и в поле Интервал задайте интервал.

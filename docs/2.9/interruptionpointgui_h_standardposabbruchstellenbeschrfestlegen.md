# Определить стандартные позиции для описания точек разрыва

Условие:

Вы открыли проект. В Графическом редакторе открыта страница проекта.

1. Откройте диалоговое окно Свойства для точки разрыва, например, двойным щелчком по точке разрыва.
2. Переместите вкладку Отображение диалогового окна Свойства ++...++ на передний план.
3. При необходимости щелкните в области свойств по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
4. Выберите в диалоговом окне Выбор свойств Свойство"Описание точки разрыва".
5. Щелкните по кнопке ++OK++.
6. Выберите на вкладке Отображение одну из имеющихся стандартных позиций, изменив настройки Присоединение по умолчанию, Выравнивание и Угол.
7. Во вкладке Данные символа выберите другой вариант из раскрывающегося списка Вариант символа.
8. Вернитесь на вкладку Отображение.

!!! info "Для сведения:"

    Значения стандартных позиций в указанных выше полях изменяются в соответствии с новым вариантом.

9. Щелкните ![](images/all_save_as.png){: .ui-icon } (Сохранить) сверху поля Порядок свойств, чтобы сохранить определенные пользователем настройки.
10. В диалоговом окне Сохранить порядок свойств описательное название для порядка свойств.
11. Если необходимо использовать заново установленный порядок свойств как стандарт, установите флажок Использовать как стандарт и щелкните ++OK++.
12. В диалоговом окне Свойства щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Описание точки разрыва выводится в соответствии с указанными настройками отображения.

!!! info "Для сведения:"

    При следующей вставке точки разрыва в качестве настройки по умолчанию используется последний выбранный символ со всеми настройками, в том числе и теми, которые относятся к позиции текста.

!!! note "Замечание:"

    Если на вкладке Отображение ввести другие значения вручную (например, для позиции X/Y) или переместить описание точки разрыва с помощью мыши, стандартные значения не будут выводиться, а значение в поле Порядок свойств изменится на "Определено пользователем". Настройки можно сохранить, как описано выше. При повторном выборе одного из стандартных порядков свойств снова автоматически устанавливаются выбранные стандартные значения.

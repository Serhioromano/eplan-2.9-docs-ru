# Операция: RegisterScript

**Параметр** |  **Описание**
---|---
ScriptFile  |  Имя файла сценария с полным путем файла. Можно использовать сценарии двух типов: C# (*.cs) и VB.NET (*.vb).
RegisterAgain  |  Позволяет повторно регистрировать сценарий (необязательно, 0 = нет, 1 = да).
ShowDecider  |  Отображает запрос о повторной регистрации сценария (необязательно, 0 = нет, 1 = да). Этот параметр пропускается, если параметр "RegisterAgain" имеет значение "1".

!!! example "Пример:"

    RegisterScript
    /ScriptFile:"C:\myScript.cs"

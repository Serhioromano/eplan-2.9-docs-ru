# Обработать проект автоматически

При использовании автоматизированных операций в проекте можно выбрать, следует ли их применять ко всему проекту или к отдельным страницам, выбранным в навигаторе страниц.

Условия:

* Вы открыли проект.
* Создана схема, содержащая последовательность автоматических операций.

1. Выберите пункт меню Страница > Навигатор.
2. Выделите в навигаторе страниц одну или несколько страниц проекта или проект целиком.
3. Выберите пункты меню Сервисные программы > Автоматизированная обработка.
4. В диалоговом окне Выполнить: Автоматизированная обработка в поле Схема выберите схему с автоматизированными операциями.

!!! info "Для сведения:"

    Сохраненная в схеме последовательность действий отображается в списке Операции.

5. Установите флажок Применить ко всему проекту для автоматической обработки всего проекта.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Начнется выполнение операций.

# Предварительно определить сокращенные буквенные обозначения для функций обработки

Для функций обработки можно предварительно определить несколько буквенных обозначений. При создании технологических контуров можно получить доступ к этим предопределенным буквенным обозначениям. Если никакие буквенные обозначения не определены предварительно, при создании технологических контуров буквенное обозначение можно ввести напрямую.

Условие:

Вы открыли проект.

1. Параметры > Настройки > Проекты > "Имя проекта" > Предварительное планирование > Функциональные признаки приборов
2. В правой части диалогового окна Настройки: Функциональные признаки приборов нажмите кнопку ![](images/all_new_as.png){: .ui-icon } (Создать).

!!! info "Для сведения:"

    В столбцах Букв. обозначение и Описание будет вставлена новая строка.

3. В столбце Букв. обозначение введите букву, а в столбце Описание – описательный текст.
4. Щелкните по кнопке ++OK++.

# Экспортировать распределение адресного пространства

В диалоговом окне Адреса / списки присвоения (пункты меню Данные проекта > ПЛК > Адреса / списки присвоения) осуществляется управление всеми входами и выходами блоков управления ПЛК, которые используются в проекте. С помощью нового пункта меню Экспортировать распределение адресного пространства под кнопкой [Дополнительно] распределение адресного пространства ЦПУ теперь можно экспортировать в файл в виде списка. Для этого диапазон адресов на картах ПЛК необходимо указать путем указания начального адреса и соответствующей длины данных.

![](images/benefit.png)Эффект:

Экспортируя загрузку адресного пространства, вы получаете обзор диапазонов адресов всех карт ПЛК, которые присвоены одному и тому же ЦПУ.

Вы можете экспортировать только один отдельный список для каждого процессора (ЦПУ). Поэтому в диалоговом окне Адреса / списки присвоений настройте фильтр так, чтобы отображались только адреса ПЛК ***одного*** ЦПУ. С помощью пунктов меню ++Дополнительно++ > Экспортировать распределение адресного пространства открывается одноименное диалоговое окно экспорта. Здесь вводится имя файла, под которым будет сохраняется список загрузки адресного пространства. Список можно сохранить как файл CSV или TXT.

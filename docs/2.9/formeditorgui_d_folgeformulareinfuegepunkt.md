# Диалоговое окно Точка вставки следующей формы

Вы открыли проект.

* Сервисные программы > Основные данные > Форма > Открыть . В диалоговом окне Открыть форму выберите форму и щелкните по ++Открыть++. Вставить > Точка вставки следующей формы.
* Сервисные программы > Основные данные > Форма > Создать. В диалоговом окне Создать форму выберите подходящий тип файла, задайте имя файла и щелкните по ++Сохранить++. В диалоговом окне Свойства формы <Имя формы> щелкните по ++OK++. Вставить > Точка вставки следующей формы.

В этом диалоговом окне задайте тип отчета и форму следующей формы. Дополнительно можно задать критерии фильтра и сортировки для объема и сортировки отчета.

Обзор основных элементов диалогового окна:

### Тип отчета

В этом раскрывающемся списке выберите тип отчета для следующей формы. В зависимости от типа формы открытой главной формы здесь предлагаются на выбор различные записи. Для большинства типов форм доступна "Групповая спецификация изделий" и "Список изделий".

Если Вы измените здесь типа отчета, выполненные настройки для свойств Форма, Фильтр, Сортировка и Новая страница снова снимаются.

* * *

Таблица Свойство / Значение:

В этой таблице выполните другие настройки для следующей формы.

### Форма

Здесь определите форму следующей формы, т. е. выберите вторую форму для отчета. В столбце Значение вы можете выбрать подходящую форму из раскрывающегося списка или при помощи записи "Поиск" перейти в диалоговое окно выбора форм. Если, например, вы генерируете схему кабельных соединений, вы можете выдавать данные кабеля в главной форме, а соответствующие изделия — в следующей форме, которая записана в данном поле.

### Фильтр

При помощи кнопки ++...++ в столбце Значение перейдите в диалоговое окно Фильтр.

### Сортировка

При помощи кнопки ++...++ в столбце Значение перейдите в диалоговое окно Сортировка.

!!! note "Замечание:"

    Для фильтра и сортировки вы найдете те же схемы и критерии фильтра, что и при генерировании отчета соответствующего типа. В типах отчетов, относящихся к функции (таких как спецификация клеммника, спецификация штекера и т. д.), речь идет о критериях фильтрации и сортировки для функций.

### Новая страница

Если установлен этот флажок, сначала генерируется отчет из главной формы, а потом, начиная со следующей страницы, отчет последующей формы.

Если флажок не установлен, второй отчет вставляется на первой странице непосредственно на точке вставки последующей формы.

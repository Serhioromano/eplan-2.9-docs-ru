# Диалоговое окно Настройки: Общие (Проекты, Устройства)

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Общее.

В данном диалоговом окне задайте общие относящиеся к проекту настройки для устройств, например как должна выполняться синхронизация устройств.

Обзор основных элементов диалогового окна:

Групповое поле синхронизация

### Пустые свойства перезаписывают заполненные

При синхронизации взаимосвязанных функций свойства одной функции копируются для других функций. Вообщем, значения свойств исходной функции копируются в свойства целевой функции. Этот флажок предназначен для свойств исходных функций, которые ***не*** имеют записей. если он активирован, целевая функция принимает пустые записи. Существующие записи целевой функции удаляются. если флажок деактивирован, целевая функция не принимает пустые записи. Существующие записи сохраняются.

### Заполненные свойства перезаписывают пустые

Этот флажок предназначен для свойств исходных функций, которые имеют записи. если он активирован, целевая функция без записей принимает запись исходной функции. если он деактивирован, целевая функция без записей не принимает запись исходной функции. Целевая функция остается без записи.

### Учитывать макрос при генерировании устройств

Эта настройка указывает, должны ли при генерации нового устройства в навигаторах (через пункт всплывающего меню Новое устройство) или в графическом редакторе (через пункты меню Вставить > Устройство) функции из записанного на изделии макроса, которые не размещаются сразу, создаваться как неразмещенные функции. По умолчанию флажок ***снят***.

В ***навигаторах*** различаются следующие случаи:

Флажки |  Условие (изделие) |  Результат
---|---|---
активировано |  Изделие содержит шаблоны функции, и задан макрос. |  Все функции из макроса генерируются как неразмещенные функции, если они относятся к этому устройству. Если в одном из вариантов макроса функция дублируется, то другие функции из варианта макроса и из всех следующих вариантов не учитываются.
деактивировано |  Изделие содержит шаблоны функции, и задан макрос. |  Из ***первой*** главной функции, которая записана в шаблонах функции, создается одна неразмещенная функциия. Остальные шаблоны остаются свободными.
активировано |  Изделие ***не*** содержит шаблонов функции, и задан макрос. |  Все функции из макроса генерируются как неразмещенные функции, если они относятся к этому устройству. Если в одном из вариантов макроса функция дублируется, то другие функции из варианта макроса и из всех следующих вариантов не учитываются.
деактивировано |  Изделие ***не*** содержит шаблонов функции, и задан макрос. |  Только ***первая*** главная функция из макроса создается как неразмещенная функция.

В ***Графическом редакторе*** эта настройка действует в следующем случае:

Изделие содержит шаблоны функции, и задан файл макроса. При этом функции, подходящие к шаблонам, распределяются по нескольким вариантам макроса. Однако через пункты меню Вставить > Устройство вставляется только первый (подходящий к типу страницы) вариант макроса.

Флажки |  Результат
---|---
активировано |  Функции из других вариантов макроса, относящиеся к устройству, создаются как неразмещенные функции. Если в одном из вариантов макроса функция дублируется, то другие функции из варианта макроса и из всех следующих вариантов не учитываются.
деактивировано |  Остальные шаблоны, для которых не найдена подходящая функция в первом варианте макроса, остаются свободными.

### Выводить изделия в соответствии с их позицией

Если этот флажок установлен, то данные изделий и ссылок изделий выводятся в соответствии с их позицией, независимо от того, имеются между записями изделий пустые строки или нет. Однако эта настройка ***не*** влияет на изделия типа 'Присвоение: источник / цель' и на принадлежности клеммы.

Если этот флажок снят, изделия выводятся независимо от их позиции.

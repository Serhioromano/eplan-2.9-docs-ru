# Выбрать кабель

Кабели можно выбирать автоматически или вручную.

На каждом кабеле при помощи свойства Разрешить выбор кабеля только вручную можно задать, что этот кабель может выбираться только вручную. При автоматическом выборе кабеля такие кабели не учитываются.

### Выбрать кабели вручную

При выборе кабеля вручную выберите изделие из базы данных изделий. Порядок действия такой же, как и при выборе любого другого изделия. Предусмотрены следующие возможности:

* [Выбрать изделие](partselectiongui_h_artikelauswahl.md) в диалоговом окне Свойства ++...++ * [Выбрать изделие в навигаторах или из списка устройств](partselectiongui_h_artikelauswahlnavigator.md)

### Выбрать кабели автоматически

Автоматический выбор кабеля только заменяет главное изделие (первое изделие группы продуктов "Кабели") подходящим изделием из базы данных изделий.

В настройках проекта можно определить список предварительного выбора типов кабеля (изделий-кабелей) для автоматического выбора кабелей. При автоматическом выборе кабеля используются только кабели из данного списка.

Условия:

* Вы выделили определения кабеля в Графическом редакторе или в диалоговом окне навигатора Кабели — <Имя проекта>.
* Вы создали схему с настройками для автоматического выбора кабеля ([Диалоговое окно Настройки: Автоматический выбор кабеля](cablegui_d_einstellungenautomatischekabelauswahl.md)).

1. Выберите пункты меню Данные проекта> Кабели > Автоматический выбор кабеля.
2. В диалоговом окне [Автоматический выбор кабеля](cablegui_d_automatischekabelauswahl.md) установите параметры для автоматического выбора кабелей.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Подходящий кабель выбирается из базы данных изделий согласно настройкам, которые учитывают свойства автоматического выбора кабеля. При этом сравниваются, например, подаваемое на кабель напряжение, напряжение на кабельных соединениях (определяется через потенциалы), имеющееся количество соединений и число резервных кабельных соединений. При отсутствии указаний для производителя берется первый найденный кабель.

!!! note "Замечание:"

    * Заданное в поле РЕ-жилы число РЕ- / PEN-жил (0, 1 или 2) учитывается при определении числа соединений. Если в кабеле нет РЕ- / PEN-жил или в базе данных изделий в соответствующем диапазоне напряжения нет кабеля без РЕ- / PEN-жил, берется кабель с РЕ- / PEN-жилой. Жила PE / PEN суммируется с необходимым числом соединений.
    * Для отдельных кабелей уже могут быть заданы любые значения. Напр., на кабеле в схеме соединений может быть внесен тип кабеля, которого нет в настройках для выбора кабелей. После этого выбирается кабель, соответствующий записанному типу кабеля и имеющий подходящее число соединений. Недостающее поперечное сечение / диаметр можно определить в зависимости от напряжения по заданным значениям. При этом учитываются такие же значения, как и при выборе кабеля вручную.

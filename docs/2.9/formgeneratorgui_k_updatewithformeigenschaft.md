# Отчеты с отличающейся формой

При генерировании отчетов по умолчанию используется форма, занесенная в диалоговое окно Настройки: Вывод в страницах. Вы можете также непосредственно при генерировании выбрать форму, которая отличается от значения, заданного в настройках.

После того как при генерировании отчета будет определен тип отчета, открывается диалоговое окно Настройки – <Тип отчета> с различными настройками для соответствующего отчета. В этом диалоговом окне есть раскрывающийся список Форма (отличается от настроек). В этом списке на выбор предлагаются все формы, сохраненные в проекте. Часто сначала на экране отображается только форма, внесенная в настройки. Для присвоения другой формы щелкните в раскрывающемся списке по записи "Поиск" и выберите в открывшемся диалоговом окне Выбрать форму нужную форму.

### Блоки отчетов с отличающейся формой

Если при генерировании нового отчета выбрана другая форма, форма заносится в диалоговом окне Отчеты в свойство Форма блока отчетов. Расположенное ниже свойство Обновление с этой формой активировано.

### Обработать блоки отчетов

Свойства Обновление с этой формой и Форма можно обрабатывать во всех блоках отчетов. Путем активизации свойства Обновление с этой формой разблокируется свойство Форма, и вы можете выбрать другую форму для уже имеющегося отчета.

### Обновить отчеты

Если для блока отчетов установлен флажок Обновление с этой формой, то при обновлении отчетов используется форма, внесенная в свойство Форма.

### Генерировать шаблон отчета

Если для блока отчетов выбрана другая форма, эта форма при генерировании шаблона заносится в свойство Форма на шаблоне.

### Составление отчетов шаблонов

При составлении отчетов шаблонов происходит переписывание форм, занесенных в соответствующие блоки отчетов! При этом несущественно, внесены ли формы для шаблонов в шаблоны или они берутся из настроек.

### Составление отчетов проектов

Активированное свойство Обновление с этой формой приводит при ***составлении отчетов проекта*** к следующим действиям:

* Встроенные отчеты:
Встроенные отчеты только обновляются. Форма, внесенная для соответствующего блока отчетов, сохраняется.
* Страницы отчетов:
При составлении отчетов проекта выполняется оценка имеющихся в проекта шаблонов. При этом страницы отчетов с отличающимися формами сохраняются только в том случае, если для соответствующего типа отчетов нет шаблонов или имеются отчеты, в которых сохранена другая форма.

!!! note "Замечание:"

    На некоторых основных функциях (определение кабеля, определение клеммника, определение штекера и т.п.) можно сохранять формы для отчета. Эти формы обладают преимуществом перед всеми остальными настройками, в т. ч. перед формами, занесенными в шаблоны.

# Диалоговое окно Настройки: Изгиб шины (фирма)

Параметры > Настройки > Фирма > Механическая обработка > Изгиб шины.

В этом диалоговом окне можно установить схемы с настройками для механического сгибания медных деталей. Каждой толщине материала присвоена таблица последовательности изгибов, которая содержит поправочные коэффициенты угла изгиба для каждого предусмотренного радиуса изгиба. Эти данные учитываются при развертке медных деталей.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Толщина материала

В поле Толщина материала изначально стоит только значение 10,00 мм. Для этого значения имеется минимальная таблица последовательности изгибов. Она состоит изначально из колонки для радиуса изгиба (1,00 мм) и строчки для угла изгиба. Туда уже внесен коэффициент поправки 0,00 мм.

Могут быть заданы другие толщины материала. Для каждой новой толщины материала автоматически задается новая таблица последовательности изгибов, которая создается пользователем и должна быть заполнена данными. При выборе толщины материала отображается соответствующая таблица последовательности изгибов.

### Таблица последовательности изгибов

Таблица последовательности изгибов содержит для каждой толщины материала матрицу коэффициентов поправки для радиуса изгиба, которые действительны для отдельных углов изгиба. Эти коэффициенты поправки выравнивают изменения длины медных деталей, которые возникают в связи с неравномерном удлинением и сжатием материала при изгибе.

Задача пользователя состоит в том, чтобы создать для каждого требуемого радиуса изгиба новую колонку и для каждого требуемого угла изгиба новую строчку. Таким образом, созданную матрицу следует заполнить коэффициентами поправки. Требуемые функции находятся во всплывающем меню таблицы последовательности изгибов.

!!! tip "Совет:"

    Определить углы и радиусы изгиба, встречающиеся на разворачиваемых медных функциональных элементах, можно из свойств медных функц. элементов. Если при нажатой клавише ++Shift++ дважды щелкнуть в пространстве листа по сборной шине, отобразится диалоговое окно "Свойства". На вкладке Места изгибов приводятся все места изгибов с указанием угла и радиуса изгиба, а также информация о том, содержатся ли эти данные уже в таблице последовательности изгибов. Перенесите недостающие значения в таблицу последовательности изгибов путем копирования.

!!! note "Замечание:"

    После изменения таблицы последовательности изгибов медные детали рассчитываются заново для экспорта. Отображение медных деталей в пространстве листа, в обзоре моделей и развертке шины после этого не адаптируется автоматически к результату экспорта. Поэтому необходимо обновить отображение.

* Пространство листа: Всплывающее меню > Обновить размещение изделия
* Обзоры модели: Сервисные программы > Отчеты > Обзор модели > Обновить
* Развертки шины: Сервисные программы > Отчеты > Развертки шины > Обновить

### Всплывающее меню

Пункт меню |  Значение
---|---
Новый радиус изгиба |  Создает новою колонку с радиусом изгиба. Отображается диалоговое окно Новый радиус изгиба, в котором надо задать желаемый радиус изгиба. Разрешены положительные значения выше 0.
Изменить радиус изгиба |  Позволяет изменить значение радиуса изгиба. Отображается диалоговое окно Изменить радиус изгиба, в котором надо задать новый радиус изгиба.
Удалить радиус изгиба |  Убирает всю колонку с выбранным радиусом изгиба.
Новый угол изгиба |  Создает новою строчку с углом изгиба. Отображается диалоговое окно Новый угол изгиба, в котором надо задать желаемый угол изгиба. Разрешены положительные значения больше 0. С новым углом изгиба автоматически генерируются записи в колонках существующих радиусов изгиба, которые уже предварительно заполнены значением 0,00 мм. Здесь можно установить нужные коэффициенты поправки.
Изменить угол изгиба |  Позволяет изменить значение угла изгиба. Отображается диалоговое окно Изменить угол изгиба, в котором надо задать новый угол изгиба.
Удалить угол изгиба |  Убирает всю строчку с выбранным углом изгиба и соответствующими радиусами изгиба.

# Обработать список устройств

Условие:

Вы открыли страницу проекта.

### Создать записи

Есть две возможности добавить изделие в список устройств: добавить изделие через выбор изделий или создать новое изделие прямо в списке устройств.

1. Выберите следующие пункты меню: Данные проекта > Устройство / изделие > Список устройств
2. Выберите пункт всплывающего меню Добавить изделие.
3. В диалоговом окне Выбор изделия выберите требуемое изделие.
4. Щелкните по кнопке ++OK++.
5. В диалоговом окне Свойства сохраните специфические для изделия данные.

!!! info "Для сведения:"

    Изделие вставляется с разрешенным числом штук "1" в список устройств.

6. Выберите пункт всплывающего меню Создать.
7. В диалоговом окне Свойства ++...++ укажите номер изделия. С помощью кнопки ++...++ Вы можете открыть выбор изделий и выбрать там имеющееся изделие.
8. В поле Разреш. число штук введите число штук этого изделия, разрешенное для задействования в проекте.
9. В полях Часть установки и Описание устройства введите имя части установки, в которую встраивается устройство (например, "Элемент роликового конвейера 15"), и описание функции для изделия. Список устройств можно сортировать по этим описаниям, нажимая на заголовки столбцов.
10. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Изделие вставляется в список устройств с указанным разрешенным числом штук.

### Обработать записи

1. Выделите требуемое изделие в списке устройств.
2. Выберите пункт всплывающего меню Свойства.
3. При необходимости в диалоговом окне Свойства ++...++ измените данные, введенные в полях Номер изделия, Разреш. число штук, а также Часть установки и Описание устройства.

!!! info "Для сведения:"

    Все остальные поля заполняются автоматически и не изменяются.

4. Щелкните по кнопке ++OK++.

### Удалить записи

1. Выделите одно или несколько изделий в списке устройств.
2. Выберите пункт всплывающего меню Удалить.

!!! info "Для сведения:"

    Выделенные изделия удаляются из списка устройств. Это не влияет на размещение соответствующего изделия. Это значит только то, что контроль за соответствующими изделиями больше не ведется.

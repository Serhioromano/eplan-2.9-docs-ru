# Диалоговое окно Управление правами

Сервисные программы > Управление правами

В этом диалоговом окне осуществляется управление пользователями и группами пользователей. Для этого необходимо определить права на обработку, которые присвоены группе пользователей. Сведения о пользователях, паролях и принадлежности к группам, а также о назначении прав группам пользователей сохраняются в центральном файле в базе данных прав.

Если на базу данных прав (через операционную систему) установлена система защиты от записи, в строке заголовка диалогового окна Управление правами отображается дополнение "С защитой от записи". При внесении изменений в защищенную от записи базу данных прав они не сохранятся, а отобразится системное сообщение.

!!! note "Замечание:"

    Обратите внимание на то, что администрирование управления правами не является многопользовательской функцией. Управление правами должно выполняться одновременно только ***одним*** администратором EPLAN; в противном случае будут сохранены только изменения, внесенные тем администратором EPLAN, который в конце сеанса закрыл управления правами.

Обзор основных элементов диалогового окна:

В левом поле отображаются следующие папки:

* Пользователь: Если выделить эту папку, справа будут показаны все имеющиеся пользователи. Вы можете изменять их или создавать новых пользователей. Кроме того, можно импортировать пользователей и группы Windows из Active Directory. Откройте папку, чтобы выделить отдельных пользователей и определить их принадлежность к группам пользователей.
* Группы: Если выделить эту папку, справа будут показаны все имеющиеся группы пользователей. Вы можете изменять их или создавать новые группы. Откройте папку, чтобы выделить отдельные группы и определить их права на обработку.
* Рабочий режим: Здесь определяется, будет ли EPLAN использоваться с включенной функцией управления правами или без нее.

### [Импортировать]

Позволяет импортировать базу данных прав из предыдущих версий EPLAN в управление правами. В следующем диалоговом окне выберите нужный файл базы данных. Возможные типы файлов: "База данных прав в формате EPLAN (*.erm)" и "База данных прав в формате Access (*.mdb)". До EPLAN версии 2.2 сведения о пользователях и группах пользователей хранились в базе данных Access; начиная с версии 2.3 используется внутренний формат данных EPLAN. Импорт баз данных Access (*.mdb) в 64-битную систему EPLAN невозможен.

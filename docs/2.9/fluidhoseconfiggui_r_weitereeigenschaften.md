# Вкладка "Другие свойства"

У вас установлен конфигуратор шлангопроводов в качестве дополнительного модуля для EPLAN Fluid. Вы запустили EPLAN Fluid и конфигуратор шлангопроводов.

* Выберите в EPLAN Fluid пункты меню Параметры > Настройки > Пользователь > Add-ons > Конфигуратор шлангопроводов и щелкните в следующем диалоговом окне по вкладке Другие свойства.
* Или выберите в диалоговом окне Конфигурировать шлангопроводы конфигуратора шлангопроводов пункты меню Дополнительно > Настройки и щелкните в следующем диалоговом окне по вкладке Другие свойства.

Или у вас установлен конфигуратор шлангопроводов в качестве самостоятельной программы, и программа запущена.

* Выберите в конфигураторе шлангопроводов пункты меню Параметры > Настройки > Пользователь > Add-ons > Конфигуратор шлангопроводов и щелкните в следующем диалоговом окне по вкладке Другие свойства.

На этой вкладке можно определить до 200 дополнительных свойств и присвоить каждому свойству определенное значение. С помощью этих свойств можно расширить нужный для заказа шлангопроводов код типа и адаптировать его в индивидуальном порядке к данным разных производителей. Свойства определяются как так называемые "произвольные свойства", т. е. их имена и значения вы задаете сами в зависимости от своих потребностей.

Например, некоторые производители предлагают шлангопроводы с защитой шлангов. В этом случае в информации по заказу, кроме кода типа согласно DIN 20066, нужно ввести данные по защите шлангов, т. е. указать тип и длину защиты шлангов.

!!! note "Замечание:"

    Определенные на этой вкладке произвольные свойства могут быть реализованы в коде типа, только если формат кода типа на вкладке Общее также был соответствующим образом расширен.

Обзор основных элементов диалогового окна:

### Свойство (список)

В этом списке определяются свободные свойства.

Используйте кнопки панели инструментов над списком, чтобы создавать новые записи, обрабатывать имеющиеся и/или удалять более ненужные записи.

### Таблица

В этой таблице каждому свойству присваивается соответствующее значение.

Используйте кнопки панели инструментов над таблицей для создания новых записей и/или удаления ненужных записей.

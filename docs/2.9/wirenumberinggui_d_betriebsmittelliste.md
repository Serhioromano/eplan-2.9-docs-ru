# Диалоговое окно Список выбора

Вы открыли проект. Вы открыли диалоговое окно [Настройки: Нумерация соединений](wirenumberinggui_d_verbnumeinstellungen.md) (например, через пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Нумерация соединений). В диалоговом окне Настройки: Нумерация соединений выберите определенную пользователем схему, а также вкладку Обозначение. Щелкните над полем Группы форматов на кнопке ![](images/all_new_as.png){: .ui-icon } (Создать). В диалоговом окне Нумерация соединений: Форматы выберите из раскрывающегося списка Группа соединений запись "Устройство..." или "Группирование...".

В этом диалоговом окне укажите одно из имеющихся в проекте ОУ или значение группировки. Группа соединений будет определена через все те соединения, которые подключены к соответствующим функциям или имеют значение, указанное для свойства Группировка.

Обзор основных элементов диалогового окна:

При вызове диалогового окна через "Устройства..." имеющиеся в проекте устройства отображаются в виде списка с полным видимым ОУ.

При вызове диалогового окна через "Группировка..." в виде списка отображаются значения, присутствующие в свойстве Группировка.

### Значение

Если вы выделили определенную запись в списке, здесь отобразится полное ОУ или значение группировки.

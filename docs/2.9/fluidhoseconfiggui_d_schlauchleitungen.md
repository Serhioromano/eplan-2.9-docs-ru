# Диалоговое окно "Конфигурировать шлангопроводы"

* У вас установлен конфигуратор шлангопроводов в качестве дополнительного модуля для EPLAN Fluid. Вы открыли проект в EPLAN Fluid и выделили шлангопроводы на странице схемы соединений Fluid. Данные проекта > Соединения > Конфигурировать шлангопроводы.
* У вас установлен конфигуратор шлангопроводов в качестве самостоятельной программы. Запустите конфигуратор шлангопроводов, дважды щелкнув по символу программы на рабочем столе Windows. Щелкните на панели инструментов программы по ![](images/toolbar_fluidhoseconfig.png) (Конфигурировать шлангопроводы). Затем щелкните по ![](images/toolbar_openfile.png) (Открыть) и выберите необходимый файл обмена.

В этом диалоговом окне вы получите табличный обзор считанных и обработанных данных шлангопроводов.

Возможно следующее:

* С помощью этого диалогового окна можно создать новые шлангопроводы. Далее можно копировать технические данные уже конфигурированного шлангопровода и присваивать их вновь созданному шлангопроводу.
* Можно выделить сразу несколько строк таблицы и тем самым конфигурировать в диалоговом окне обработки сразу несколько шлангопроводов.
* Кроме того, можно импортировать данные шлангопровода из других приложений, например из MS Excel, и снова сохранить измененные данные в файле обмена или экспортировать их.
* Кроме того, с помощью диалогового окна можно вызвать настройки конфигуратора шлангопроводов.

Обзор основных элементов диалогового окна:

### Таблица

В таблице отображаются данные конфигурации шлангопроводов, которые состоят из технической, а также специфической для обозначений и изделий информации, включая код типа. Через всплывающее меню или двойным щелчком мыши по строке таблицы можно обрабатывать отображающиеся данные шлангопровода. Кроме того, в таблице можно изменять данные сразу нескольких выделенных шлангопроводов (групповая обработка).

* * *

### [Дополнительно]

Эта кнопка доступна только в конфигураторе шлангопроводов, установленном в качестве дополнительного модуля.

Пункт меню |  Значение
---|---
Импортировать |  Открывает диалоговое окно Открыть для импорта данных шлангопровода, которые были сохранены в файле обмена. Задайте в этом диалоговом окне имя и место хранения открываемого файла обмена. Можно выбрать только разделенные табулятором текстовые файлы Юникод (*.txt).

Экспортировать |  Открывает диалоговое окно Сохранить как для экспорта данных шлангопровода в файл обмена, который может считываться конфигуратором шлангопровода как самостоятельной программой или другим приложением. В диалоговом окне вы задаете имя и место хранения обработанных данных шлангопроводов при сохранении. Данные сохраняются в разделенном табулятором текстовом файле Юникод (*.txt).

Настройки |  Открывает диалоговое окно Настройки: Конфигуратор шлангопроводов, в котором можно задать свойства и значения для конфигурации шлангопроводов и сохранить их в виде схемы.

### Всплывающее меню

Пункт меню |  Значение
---|---
Выделить все |  Выделяет все записи в таблице.

Подобрать ширину столбца |  Все столбцы так подогнаны к таблице по ширине, что и заголовок, и содержимое столбцов полностью читаемы.

Копировать |  Копирует все шлангопроводы в буфер обмена, чтобы иметь доступ к данным также из других приложений.
Вставить технические данные |  Вставляет технические данные ранее скопированного шланга в новую строку.

!!! note "Замечание:"

    * Если выбранная строка уже содержит данные шлангопровода, эта информация, кроме полного ОУ, переписывается без запроса.
    * Этот пункт меню доступен только тогда, когда через пункт меню Копировать ранее был скопирован ровно один шлангопровод.

Обработать |  Открывает диалоговое окно Обработать, в котором можно конфигурировать один или несколько выделенных шлангопроводов.
Создать |  Создает новую строку таблицы для шлангопровода, которая позволит вам конфигурировать новый шлангопровод.
Обновить |  Повторно рассчитывает код типа и обозначение шлангопровода для всех выделенных шлангопроводов. Всегда используйте этот пункт меню, если для одного или нескольких шлангопроводов на схеме были произведены изменения свойств, имеющих значение для кода типа, например длина шлангопровода, или имеющих значение для обозначения, например значение рабочего давления.
Удалить |  Удаляет выделенные шлангопроводы без запроса.

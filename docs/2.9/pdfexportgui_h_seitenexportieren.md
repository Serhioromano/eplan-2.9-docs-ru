# Вывести страницы в PDF-файл

При экспорте в PDF независимо от выполненного выделения (отдельные страницы проекта или весь проект) всегда создается только один файл PDF.

!!! note "Замечание:"

    При экспорте больших страниц учитывайте, что экспорт в оригинальном размере может вызвать проблемы. Поэтому сначала следует разбить импортируемые блоки DXF, чтобы можно было масштабировать размер страницы. (Это выполняется с помощью пунктов меню Обработать > Другое > Разбить блок.)

Условие:

Вы открыли проект.

1. Выделите в навигаторе страниц те страницы проекта (или весь проект), которые требуется экспортировать.
2. Выберите пункты меню Страница > Экспортировать > PDF.

!!! info "Для сведения:"

    EPLAN открывает диалоговое окно Экспорт PDF. Выбранные объекты отобразятся в поле Источник (страница / трехмерная модель).

3. Из раскрывающегося списка Схема выберите схему или нажмите ++...++, чтобы в диалоговом окне Настройки: Экспорт PDF создать или изменить схему.
4. В поле PDF-файл вводится каталог и имя для генерируемого PDF-файла или выбирается интерактивно с помощью кнопки ++...++. В качестве предварительных настроек здесь используется каталог документации DOC и имя выбранного проекта (...\<Проект.ebb>\DOC\<Проект>.PDF).
5. Укажите в групповом поле Средство экспорта, каким средством следует воспользоваться при выводе.
6. Укажите в групповом поле Вывод, как должен осуществляться вывод.
7. Нажмите ++Настройки++ для обзора настроек языков вывода и полей для печати, если необходимо, измените их.
8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    EPLAN создаст PDF-файл.

!!! note "Замечание:"

    Для возможности просмотра PDF-файла на вашем компьютере должна быть установлена программа Acrobat Reader компании Adobe (версия не ниже 10.0).

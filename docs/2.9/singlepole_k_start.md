# Однополюсное представление

Однополюсное представление схем соединения разрешает упрощенное проектирование и наглядную документацию проектов, в особенности при работе со стандартизованными модулями.

Однополюсное представление дает следующие возможности:

* Можно чертить упрощенные принципиальные схемы, перечни установок и структуры сети.
* При высокой плотности разводки Вы можете с помощью специальных символов вывода жгута создавать наглядные схемы токопрохождения.
* Вы можете изменить вид представления символов.
* Вы можете синхронизировать функции между двумя разными видами представления.
* В однополюсном представлении можно определить многополюсные соединения. Многополюсный отчет по выводам устройств и соединений можно задать при исключительно однополюсном представлении.

# Обработать свойства устройств

Условие:

Вы открыли диалоговое окно "Свойства" для условного обозначения / функции, например, через меню Обработать > Свойства.

На вкладке <Категория функции> диалогового окна Свойства ++...++ указываются свойства выделенной функции. На вкладке нанесено обозначение соответствующей категории функции, напр. "Двигатель", "Замыкающий контакт", "Плавкий предохранитель" и т.д. Важнейшие свойства можно ввести в полях в верхней области вкладки, напр., видимое или полное ОУ, обозначение вывода устройства, функционального текста и т.д. В таблице группового поля Свойства можно задать дополнительные свойства.

!!! tip "Совет:"

    * Вы можете также открыть диалоговое окно свойств, дважды щелкнув по условному обозначению в графическом редакторе или по функции в навигаторе. Либо выберите пункт всплывающего меню Свойства; в этом случае могут быть выделены несколько условных обозначений или функций.
    * В таблице свойств через команды всплывающего меню Вставить свойства / значения можно вставить одно или несколько свойств, скопированных с другого объекта вместе с их значениями. Если свойство уже имеется, вставляются только значения. Если свойство еще не отображалось в таблице, после вставки оно появится вместе со скопированным значением.

### Задать конфигурацию свойств

При конфигурации свойств для таблицы свойств между предварительно определенными группами свойств (категория "Данные", "Настройки" и т. д.) и самостоятельно составленными группами (категория "Определено пользователем") существуют различия. В дальнейшем описывается конфигурация предварительно определенных групп.

Условие:

Вы открыли диалоговое окно "Свойства" для условного обозначения / функции. Вкладка <Категория функции> находится на переднем плане.

1. В списке Категория выберите одну из предварительно выбранных категорий ("Данные", "Настройки" и т. д.).
2. В таблице Свойства выберите пункт всплывающего меню Конфигурировать.

!!! info "Для сведения:"

    В диалоговом окне Конфигурировать свойства отображаются все свойства, отображаемые в данный момент в таблице. Отображение зависит от выбранной категории.

3. Щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать), чтобы добавить свойство конфигурации свойств.
4. В диалоговом окне Выбор свойств выберите требуемое свойство.
5. Используя кнопки со стрелками, задайте последовательность, в которой будут отображаться свойства.
6. Установите флажок перед свойствами, которые должны отображаться для всех подобных функций в диалоговом окне Свойства ++...++.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Специфические для пользователя настройки сохраняются.

!!! info "Для сведения:"

    Свойства, для которых установлен флажок, ***Всегда*** отображаются при следующем вызове диалогового окна, даже если они пусты.

!!! info "Для сведения:"

    Свойства, для которых флажок не установлен, отображаются тогда, когда они заполнены. При этом действительна заданная последовательность.

!!! note "Замечание:"

    * Выбранные свойства отображаются только для подобных условных обозначений в диалоговом окне "Свойства". Вы можете задать для устройств общего назначения, клемм, вывода устройства ПЛК и т.д. тот или иной выбор свойств, которые будут отображаться в таблице.
    * Если изменяемые вручную свойства заполнены, то их невозможно убрать ни в таблице Свойства, ни в диалоговом окне Конфигурировать свойства. Добавление новых свойств происходит автоматически. Такие свойства отсортировываются в конец таблицы.
    * С помощью категории "Определено пользователем" можно сохранить конфигурацию самостоятельно составленных свойств в качестве схемы. При этом в таблице отображаются только свойства, также заданные в определенной пользователем конфигурации свойств. Вручную изменяемые заполненные свойства не будут отображаться в таблице.

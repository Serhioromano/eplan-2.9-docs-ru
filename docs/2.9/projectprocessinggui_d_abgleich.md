# Диалоговое окно "Синхронизировать устройства" / Диалоговое окно "Синхронизировать маршрутизируемые соединения"

Вы открыли проект.

* Данные проекта > Устройство > Импортировать. В диалоговом окне [Импортировать данные устройств](projectprocessinggui_d_bmdatenimport.md) вы выполнили необходимые настройки для источника данных и присвоения полей. Щелкните по кнопке ++OK++.
* Данные проекта > Соединения > Импортировать > Маршрутизируемые соединения. В диалоговом окне [Импортировать данные соединения](projectprocessinggui_d_verbindungsdatenimport.md) вы выполнили необходимые настройки для источника данных и присвоения полей. Щелкните по кнопке ++OK++.

В этом диалоговом окне в табличной форме отображаются данные устройств / данные соединений из источника данных и из проекта EPLAN. В области таблицы Источник данных отображаются данные устройств / данные соединений из источника данных, в области таблицы Проект EPLAN — данные из проекта EPLAN. Объем отображения можно ограничить, установив фильтры. Для обработки пользователя доступны только оба столбца Действие и Номер изделия в области таблицы Проект EPLAN.

Обзор основных элементов диалогового окна:

### Фильтр

С помощью кнопки ++...++ откройте диалоговое окно Фильтр. Настройте здесь критерии фильтра для отображаемых устройств и соединений.

### Область таблицы "Источник данных"

* Строка: Показывает порядковый номер строки.
* Номер изделия: Номер изделия устройства / соединения в источнике данных; здесь в качестве заголовка столбца отображается имя присвоенного свойства EPLAN.
* Количество, уровень и т. д.: В качестве заголовка столбца отображаются другие свойства устройства / соединения в источнике данных, если таковые имеются. Объем отображаемых столбцов можно расширить или сократить при помощи пункта всплывающего меню Конфигурировать представление.

### Область таблицы "Проект EPLAN"

* Статус: В этом столбце отображается, имеются ли устройства / соединения уже в проекте или еще нет, а также имеют ли устройства / соединения в источнике данных и в проекте различные состояния обработки (например, другие значения свойств).

* Операция: Укажите здесь, какое действие должно быть выполнено в зависимости от отображаемого статуса по отношению к устройствам / маршрутизируемым соединениям:
Статус |  Значение |  Возможное действие
---|---|---
![](images/all_statusnew_as.png){: .ui-icon } |  Создать: Устройство / маршрутизируемое соединение в источнике данных пока отсутствует в проекте. |  Импортировать или Нет
![](images/all_statusok_as.png){: .ui-icon } |  Идентичный: Устройство / маршрутизируемое соединение в источнике данных уже содержится в проекте и не импортируется еще раз. Все значения свойств идентичны.

![](images/all_statusadditional_as.png){: .ui-icon } |  Изменено: Устройство / маршрутизируемое соединение содержится в источнике данных и в проекте. Одно или несколько значений свойств отличаются, например номер изделия.

!!! note "Замечание:"

    Поля с измененными значениями свойств подкрашены темно-серым цветом. Всплывающая подсказка для такого поля показывает соответствующее значение в проекте.

* Номер изделия: Номер изделия устройства / маршрутизируемого соединения в проекте EPLAN.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Присвоить изделие |  Если номера изделия из источника данных отсутствуют в базе данных изделий EPLAN, то они отображаются в диалоговом окне жирным шрифтом. Этот пункт всплывающего меню ищет в базе данных изделий другие совпадающие свойства (напр., номер для заказа, номер типа) импортируемых устройств / маршрутизируемых соединений и присваивает им, кроме того, номер изделия, подходящий для базы данных. Если автоматическое присвоение невозможно, в столбце Номер изделия щелкните мышкой по кнопке ++...++, чтобы открыть выбор изделия и вручную выбрать в нем изделие.

### Фильтр (статус)

Сокращает отображение импортируемых устройств в зависимости от их статуса:

* Все (настройка по умолчанию)
* Изменено
* Идентичны
* Создать
* Дополнительные.

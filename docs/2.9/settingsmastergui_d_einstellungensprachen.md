# Диалоговое окно Настройки: Языки

Параметры > Настройки > Пользователь > Отображение > Языки

В этом диалоговом окне задаются настройки для языков интерфейса пользователя.

Обзор основных элементов диалогового окна:

### Язык диалогового окна

В этом раскрывающемся списке предлагаются разные языки для интерфейса пользователя на выбор.

!!! note "Замечание:"

    * Помните, что в списке Язык диалогового окна дополнительные языки можно выбрать, только если они тоже установлены во время инсталляции.
    * После выбора другого языка диалогового окна требуется перезапустить EPLAN, прежде чем новый язык появится в интерфейсе пользователя.

### Альтернативный язык

С помощью этого раскрывающегося списка EPLAN предлагает разные языки в качестве альтернативы. Альтернативный язык можно использовать, когда язык проекта не доступен в качестве языка словаря.

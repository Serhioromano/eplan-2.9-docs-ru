# Изменить размер заблокированных областей, соединительных отверстий для проводов и областей маршрутизации

До настоящего времени изменение размеров в запретных зонах для сверления и размещения, соединительных отверстий для проводов и областей маршрутизации были возможны только путем удаления и повторного размещения. Теперь размер размещаемых объектов можно изменить следующим образом:

* ***Изменение свойств формата в диалоговом окне 'Свойства'***: на вкладке Формат в диалоговом окне 'Свойства' введите измененные значения в поля Ширина и Высота. Изменение производится относительно нижней левой угловой точки.
* ***Графическое изменение (для заблокированных областей и соединительных отверстий для проводов)***: выделите объект. На угловых точках и в центре боковых линий, как и для 2D-прямоугольников, отображаются указатели в виде стрелок в направлении, в котором можно растянуть или сжать объект. При перемещении вдоль указателя на соседних объектах 3D отображаются точки захвата, которые можно использовать для точного позиционирования.
* ***Графическое изменение (только для областей маршрутизации)***: выберите пункты меню Обработать > Графика > Изменить длину и щелкните по объекту, который нужно изменить. В четырех угловых точках передней поверхности при наведении курсора отобразится трехмерная система координат. Выберите размер, который нужно изменить, нажав нужную ось. Выбранная ось будет выделена зеленым цветом.
После установки направления изменения растяните трехмерный объект в требуемом направлении. Задать новое значение размера можно несколькими способами, например щелчком мыши, с помощью точек захвата, проекции или растра и т. д.

![](images/benefit.png)Эффект:

Теперь изменения размеров для заблокированных областей, соединительных отверстий для проводов и областей маршрутизации можно вносить прямо на размещенных объектах. Больше не нужно использовать обходной путь в виде удаления и повторного размещения.

# Вкладка Контакт штекера

Вы открыли проект.

* В Графическом редакторе вы выделили контакт штекера. Обработать > Свойства. В диалоговом окне Свойства ++...++ выберите вкладку Контакт штекера.
* Вы выделили контакт штекера в навигаторе штекеров или в навигаторе устройств. Пункт всплывающего меню Свойства или Свойства (общие). В диалоговом окне Свойства ++...++ выберите вкладку Контакт штекера.
* В навигаторе штекеров при помощи пункта всплывающего меню Генерировать контакт штекера генерируется контакт штекера.

В этой вкладке вы можете задать и обработать свойства контакта штекера. Контакт штекера может представлять собой или штырь, или гнездо, или комбинацию штыря и гнезда. Неизменяемые свойства недоступны для выбора.

В режиме обработки ***Свойства (общие)*** совместно обрабатываются свойства для всех представлений контактов штекера; в этом случае отображаются дополнительные вкладки Определение штекера (устройство) и Изделие (устройство) для соответствующей главной функции.

Обзор основных элементов диалогового окна:

### Видимое ОУ

Видимое ОУ будет показано в графическом редакторе рядом с условным обозначением. Если вы определили в настройках проекта, что устройство должно автоматически нумероваться при вставке символов, здесь вам будет автоматически предложено ОУ. Вы можете сохранить или переписать вручную это ОУ.

Кроме того, можно использовать [всплывающее меню или кнопку ++...++](devicetaggui_r_feldsichtbaresbmk.md), чтобы cгенерировать или выбрать ОУ.

### Полное ОУ

В этом поле отображается полное ОУ (т. е. структурный идентификатор текущего окружения + видимое ОУ) видимого ОУ / функционального элемента. С помощью кнопки ++...++ откройте диалоговое окно Полное ОУ. Здесь можно обработать структурный идентификатор и, например, выбрать другое место установки.

### Обозначение

Это поле показывает обозначение контакта штекера. На основании обозначения контакта штекера определяется позиция контакта штекера в штекере. Введенное здесь значение сохраняется в свойстве Обозначение клеммы / контакта штекера.

!!! tip "Совет:"

    Если в поле Видимое ОУ ввести двоеточие, все символы после двоеточия автоматически перенесутся в поле Обозначение.

### Описание

Введите здесь описание контакта штекера.

### Обозначение вывода устройства

В этом поле отображаются обозначения выводов устройства текущей функции, а также обозначения выводов устройств тех функций, в которые копируется ОУ текущей функции.

Обозначения выводов устройств для функции или для устройства можно изменить вручную или выбрать запись из раскрывающегося списка. Несколько обозначений выводов устройств разделяются знаком абзаца. При обработке нескольких обозначений выводов устройств вручную необходимо учитывать следующее:

* Отдельные обозначения выводов устройств можно изменить вводом значения на соответствующей позиции между знаками абзаца.
* При перезаписывании всей записи одним отдельным значением это значение будет присвоено первому выводу устройства. Остальные обозначения выводов устройств останутся без изменения.
* При удалении всей записи удаляются все обозначения выводов устройств.

!!! note "Замечание:"

    * Содержимое данного поля не сохраняется в отдельном свойстве. Чтобы отобразить обозначения выводов устройств на условном обозначении, можно воспользоваться свойством Обозначения выводов устройства (все) (ID 20038). Это свойство отображает все обозначения выводов устройств текущей функции. Однако это свойство не содержит отображаемые в поле Обозначение вывода устройства обозначения выводов устройств других функций.
    * При помощи опции Отображать обозначения выводов устройства в повернутом виде в настройках проекта (Параметры > Настройки > Проекты > "Имя проекта" > Графическая обработка > Общее) вы можете задать, чтобы свойства Обозначение клеммы / контакта штекера и Обозначение вывода устройства отображались в повернутом виде. Эта настройка учитывается только для порядка свойств "Стандартное значение по умолчанию".

### Описание вывода устройства

Введите здесь описание выводов устройств для текущей функции или устройства или выберите эти данные из раскрывающегося списка.

### Присвоение к стороне штыря / гнезда

При раздельном управлении штырями штекера и гнездами укажите в этом поле, какой стороне присваивается контакт штекера - стороне штырей штекера или стороне гнезд штекера. В раскрывающемся списке доступны следующие опции:

* Автоматически: Присвоение происходит посредством ОУ и определения функции: Штырь присваивается определению штекера для штырей, а гнездо — определению штекера для гнезд, если контакт штекера и определение штекера имеют одинаковое ОУ.
* Сторона штырей: Контакт штекера (штырь или гнездо) присваивается стороне штырей штекера.
* Сторона гнезд: Контакт штекера (штырь или гнездо) присваивается стороне гнезд штекера.

### Функциональный текст

Укажите здесь функциональный текст для функции / устройства.

Независимо от введенного здесь вручную функционального текста существуют другие (определяемые автоматически) функциональные тексты, которые находятся в таблице:

* Функциональный текст (автоматически): Отображает содержание введенного вручную функционального текста или, при его отсутствии, функциональный текст зоны. Если вы изменяете функциональный текст в режиме обработки ***Свойства (общие)***, его содержание переносится обратно в ту функцию, в которой функциональный текст был первоначально задан. При отображении этого свойства на странице проекта разрывы строк удаляются.
* Функциональный текст (общий): отображает (автоматический) функциональный текст функции или, при его отсутствии, (автоматический) функциональный текст соответствующей главной функции.

### Мостовая перемычка

Здесь можно указать позицию мостовой перемычки. В раскрывающемся списке доступны следующие опции:

* Без перемычки
* Вручную, начало перемычки
* Вручную, конец перемычки
* Вручную, центр перемычки.

### Определение функции

С помощью кнопки ++...++ откройте диалоговое окно Определения функций. Определения функций там уже отфильтрованы, поэтому выводятся только контакты штекера. Выберите нужный контакт штекера.

Это поле синхронизируется с полем Определение, находящимся на вкладке Данные символа / функции.

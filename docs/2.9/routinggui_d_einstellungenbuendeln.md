# Диалоговое окно Настройки: Объединить

Сгенерированы маршрутизируемые соединения. В навигаторе соединений выделены маршрутизируемые соединения. Выберите пункты меню Данные проекта > Соединения > Объединить. В диалоговом окне Объединить маршрутизируемые соединения щелкните в таблице в Схема объединенных проводов в ячейке по кнопке ++...++.

В этом диалоговом окне задаются подробные настройки для выбранной схемы объединенных проводов.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Фильтр

Выберите из раскрывающегося списка схему для фильтрации маршрутизируемых соединений. Нажатием на кнопку ++...++ можно отобразить диалоговое окно Фильтр, в котором можно создать новую схему фильтрации и обработать имеющийся фильтр.

### Сортировка

Выберите из раскрывающегося списка схему для сортировки маршрутизируемых соединений. Нажатием на кнопку ++...++ можно отобразить диалоговое окно Сортировка, в котором можно создать новую схему сортировки и обработать имеющуюся схему.

### Новые объединенные провода с измененным значением свойства

Выберите из раскрывающегося списка схему с критериями объединения, при изменении которых необходимо создать новые объединенные провода. Нажатием на кнопку ++...++ можно отобразить диалоговое окно Новые объединенные провода с измененным значением свойства, в котором можно создать новую схему с критериями объединения и обработать имеющуюся схему.

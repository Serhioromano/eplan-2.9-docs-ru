# Диалоговое окно Настройки — <Тип отчета>

Вы открыли проект. Сервисные программы > Отчеты > Генерировать. Щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать). Выделите в диалоговом окне Определить отчет [Тип отчета](formgeneratorgui_k_auswertungstypen.md) для перечня отчетов, как, например, список обозначений устройств. Щелкните по кнопке ++OK++.

В это диалоговое окно установите схемы сортировки, фильтрации и разбивки на страницы [Перечни отчетов](formgeneratorgui_k_auswertungsuebersicht.md). Схемы сортировки, фильтрации и разбивки на страницы сохраняются специфически для пользователя и при генерации страниц отчета сохраняются вместе с отчетом.

Обзор основных элементов диалогового окна:

### Фильтр

В этом поле выберите схему, в которой установлено, какие данные проекта или изделий должны выводиться. С помощью кнопки ++...++ откройте диалоговое окно Фильтр, в котором можно создать или обработать схему.

!!! example "Пример:"

    Вам нужно сгенерировать содержание. Определите следующие фильтры:Активн.Отрицат.Критерий Оператор Значение Установка=FB3Место установки=FT1EPLAN генерирует содержание для тех страниц проекта, которые находятся в уровне структуры=FB3+ET3.

### Сортировка

В этом поле выберите схему, в которой установлено, какие данные проекта или изделий должны сортироваться в отчетах. С помощью кнопки ++...++ откройте диалоговое окно Сортировка, в котором можно создать или обработать схему.

### Новая страница при изменении значения свойства

В этом поле выберите схему, в которой установлено, когда должна генерироваться страница. В схеме сохранено свойство, как, например, "Установка". Всегда, когда меняется значение данного свойства, вставляется разбивка на страницы и создается новая страница. С помощью кнопки ++...++ откройте диалоговое окно Новая страница при изменении свойства, в котором можно создать или обработать схему.

Выбранная схема активируется автоматически. Запись "-Не активировано-" позволяет деактивировать выбранную схему.

### Форма (отличается от настроек)

Для того чтобы присвоить выбранному типу отчета форму, которая ***временно*** отличается от заданных параметров в настройках, откройте этот раскрывающийся список и щелкните по "Поиск". В диалоговом окне Выбрать форму настроен каталог, который занесен в настройки как каталог форм. В поле Тип файла автоматически отображается тип формы, подходящий для типа отчетов. Выделите нужную форму и щелкните по ++Открыть++.

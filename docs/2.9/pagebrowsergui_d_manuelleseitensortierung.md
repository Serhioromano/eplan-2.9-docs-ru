# Диалоговое окно Сортировка страниц вручную

Вы открыли навигатор страниц и открыли (как минимум) один проект. Во вкладке Список выберите пункт всплывающего меню Сортировка страниц вручную.

В этом диалоговом окне задайте последовательность, в которой выполняется сортировка страниц для таких операций, как перелистывание, печать, вывод перекрестных ссылок и т. д. Чтобы эта последовательность соблюдалась, необходимо установить флажок Сортировка страниц вручную в настройках проекта (Параметры > Настройки > Проекты > "Имя проекта" > Управление > Страницы). Сортировка страниц вручную влияет только на представление в виде списка, представление в виде дерева сохраняется без изменений.

Если новые страницы вставляются в проект с помощью функций Создать, Копировать, Импортировать и т. д., они добавляются в конец списка и должны при необходимости заново сортироваться.

Обзор основных элементов диалогового окна:

В таблице страницы открытых проектов отображаются с указанием свойств, выбранных в диалоговом окне Конфигурировать представление. При этом учитывается любой фильтр страниц, который может быть настроен в этом случае.

С помощью кнопки-стрелки вы можете изменять порядок страниц внутри списка. При этом в каждом случае перемещается целая строка.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Стандартная сортировка |  Восстановите стандартные настройки для вручную измененной последовательности.

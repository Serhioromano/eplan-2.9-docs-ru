# Вставить кривые

Кривые — это вставленные вручную трассы маршрутизации, вдоль которых можно маршрутизировать электротехнические и гидротехнические соединения.

* Определение кривой осуществляется через начальную точку и минимум две следующие друг за другом опорные точки. По завершении размещения кривой размещенная последней опорная точка считается конечной точкой.
* Отдельные опорные точки кривой помимо координат расположения X, Y и Z содержат информацию о наклоне кривой и коэффициенте изгиба.
* Графически кривые, как и сегменты маршрутизации, обозначаются синими линиями с точками захвата в начальных, конечных и опорных точках.
* При вставке кривых можно использовать точки захвата на имеющихся сегментах маршрутизации. Но можно также свободно определять кривые при помощи курсора или опций размещения.
* Монтажные поверхности, которые при размещении точек кривой находятся под курсором, активируются автоматически. Когда монтажная поверхность активирована, все точки кривой размещаются на монтажной поверхности.

!!! tip "Совет:"

    При вставке кривых и изменении направления кривых можно использовать ортогональные функции для размещения всех значений в одной плоскости. Нажмите ++<++, чтобы активировать ортогональную функцию и переместить опорную точку по оси Х. При каждом следующем нажатии клавиши ++<++ активируются ось Y, ось Z, плоскость XY, плоскость XZ и плоскость YZ в указанной последовательности.

    С помощью клавиши ++Tab++ при изменении направления кривой можно изменить угол наклона кривой в опорной точке. С помощью клавиш [Pos1]/[Ende] изменяется коэффициент изгиба в опорной точке.

### Вставка кривой

1. Выберите пункты меню Вставить > Кривая.

!!! info "Для сведения:"

    В строке состояния отображается требование "Начальная точка кривой".

2. Переместите курсор к одному из существующих концов сегмента.

!!! info "Для сведения:"

    Начальная, средняя и конечная точка имеющегося сегмента маршрутизации отображаются в виде точек захвата.

!!! info "Для сведения:"

    При размещении точек можно использовать опции размещения, чтобы указать смещение в направлении X, Y, Z.

3. Разместите начальную точку, щелкнув по захваченной точке.

!!! info "Для сведения:"

    В строке состояния отображается требование "Следующая опорная точка кривой".

!!! info "Для сведения:"

    Кривую в районе курсора можно свободно перемещать, вращать и вытягивать во всех направлениях.

!!! info "Для сведения:"

    При свободном размещении в пространстве описанный параллелепипед показывает пространство, занимаемое профилем кривой между последней введенной точкой и текущим положением курсора. При размещении на плоскости для этого используется прямоугольник.

4. Разместите первую опорную точку кривой.

!!! info "Для сведения:"

    В строке состояния отображается требование "Следующая опорная точка кривой, для завершения нажать клавишу пробела".

5. Разместите вторую опорную точку кривой.
6. Разместите остальные опорные точки или завершите размещение с помощью клавиши пробела.

!!! info "Для сведения:"

    Кривая отображается в пространстве листа.

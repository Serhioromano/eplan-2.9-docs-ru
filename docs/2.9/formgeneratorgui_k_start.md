# Отчеты

Отчеты представляют собой запрос данных проекта. Они дают возможность целенаправленно выводить данные проекта. Они генерируются автоматически и выводятся непосредственно на страницах отчетов, или же во внешний файл, например для маркировки этикеток функциональных элементов. Отчеты можно также размещать в качестве встроенного отчета непосредственно на существующей странице проекта, например, для легенд электрошкафа. Отчеты подразделяются на [типы отчетов](formgeneratorgui_k_auswertungstypen.md).

!!! note "Замечание:"

    Если проект не содержит требуемые для отчета объекты, или при установке фильтров не отображаются подходящие объекты, страница отчета генерироваться не будет.

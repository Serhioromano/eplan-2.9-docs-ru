# Назначить сегменты / объекты

В навигаторе предварительного планирования можно назначить различные сегменты и связанные с ними подчиненные объекты (шаблоны функций, адреса ПЛК) функциям, размещенным на схеме соединений. При присвоении объектов планирования, шаблонов функций и адресов ПЛК данные могут также передаваться на условное обозначение. Сегменты структуры могут быть назначены только обозначениям местоположения на страницах проекта.

Кроме того, сегменты можно назначить 3D-размещениям изделий, вставленным в пространство листа.

Присвоение связывает функции или 3D-размещение изделий с сегментом, из которого они получили присвоение.

Защищенные функции в принципе не могут быть целью присвоения.

### Присвоить объекты планирования

Присвоение описывается здесь на примере объектов планирования; технологические контуры, шаблоны функций и адреса ПЛК также могут быть назначены аналогичным образом.

Условия:

* Вы открыли проект.
* Открыт навигатор предварительного планирования.
* В Графическом редакторе открыта страница проекта.

1. Выделите объект планирования в навигаторе предварительного планирования.
2. Выберите пункт всплывающего меню Присвоить.

!!! info "Для сведения:"

    Обозначение выделенного объекта планирования зависит от курсора.

3. Наведите курсор на условное обозначение, чтобы изменился цвет условного обозначения.
4. Дважды щелкните по условному обозначению.

!!! info "Для сведения:"

    Данные объекта планирования переносятся на условное обозначение.

!!! info "Для сведения:"

    Целевые функции связываются с объектом планирования, от которого они получили присвоение.

На один объект планирования приходится только одна главная функция. Если цель является главной функцией, изделия, сохраненные на объекте планирования, переносятся при этом на целевую функцию.

!!! note "Замечание:"

    * При присвоении ***шаблонов функций*** возможен многократный выбор. Данные шаблона функции при присвоении переносятся на условное обозначение, а функция условного обозначения накладывается на шаблон функции.
    * ***Выводу устройства ПЛК*** при назначении присваивается адрес ПЛК, символический адрес, функциональный текст и тип данных.

!!! tip "Совет:"

    Если на схеме соединений вы хотите обработать назначенный сегмент из предварительного планирования для определенной функции, выберите для выделенной функции пункт всплывающего меню Свойства (предварительное планирование). Откроется диалоговое окно "Свойства" присвоенного сегмента, и вы сможете сразу обрабатывать свойства из предварительного планирования, минуя навигатор предварительного планирования.

### Назначить сегменты 3D-размещениям изделий

Условия:

* Вы открыли проект.
* Открыт навигатор предварительного планирования.
* Вы открыли пространство листа.
* Вы разместили один электрошкаф или монтажную плату с 3D-размещениями изделий.

1. Выделите сегмент в навигаторе предварительного планирования.
2. Выберите пункт всплывающего меню Присвоить.
3. Наведите курсор на 3D-размещение изделия, чтобы изменился цвет функционального элемента.
4. Щелкните по 3D-размещению изделия.

!!! info "Для сведения:"

    Данные сегмента не переносятся на 3D-размещение изделия.

!!! info "Для сведения:"

    3D-размещения изделий связываются с сегментом, из которого они получили присвоение.

!!! tip "Совет:"

    Это присвоение можно использовать, например, для связывания в проекте макросов функциональных элементов, содержащихся в 3D-макросе, с сегментами в предварительном планировании. Готовый 3D-макрос при соответствующем присвоении содержит целые подструктуры. Если такой 3D-макрос вставлен в проект схемы соединений, то содержащиеся в макросе подструктуры размещаются в структуре предварительного планирования проекта.

# Определить нумерацию устройств (Изменение стандарта)

Пользовательские и предварительно определенные схемы для нумерации устройств применяются из шаблона, который был указан при создании целевого проекта.

Условия:

* Вы открыли проект.
* При нескольких открытых проектах: В навигаторе страниц выделен проект.
* Вы настроили целевой проект.
* Вы определили шаблон и поворот / инверсию.
* Вы заменили необходимые рамки, формы, библиотеки символов и символы.
* Вы определили структуру проекта.

1. Сервисные программы > Изменение стандарта > Вкладка "Проект" > [Далее] > Вкладка "Поворот / инверсия" > [Далее] > Вкладка Основные данные > [Далее] > Вкладка "Структура" > [Далее]
2. На вкладке Нумеровать установите флажок Повторно нумеровать устройствo.
3. Выберите из раскрывающегося списка Нумерация устройствo формат нумерации.
4. Нажмите кнопку ++Завершить++.

!!! info "Для сведения:"

    Целевой проект будет отображен в навигаторе страниц.

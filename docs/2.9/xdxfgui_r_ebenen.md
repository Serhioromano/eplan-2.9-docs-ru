# Вкладка Слои

* Параметры > Настройки > Пользователь > Интерфейсы > Экспорт и импорт DXF / DWG. Выберите вкладку Слои.
* Страница > Экспортировать > DXF/DWG. Щелкните рядом с полем Схема на ++...++. Выберите вкладку Слои.

Чтобы однозначно распознать логические элементы страницы, в файле DXF / DWG должны использоваться различные слои. Для этого присвойте в таблице на этой вкладке заданным стандартным и проектным слоям EPLAN использованные в AutoCAD имена уровней.

Обзор основных элементов диалогового окна:

### Имя

Имя слоя EPLAN из раздела Управление слоями (Данные проекта > Управление слоями), например EPLAN100. Щелкните по стрелке в поле, чтобы открыть раскрывающийся список всех слоев EPLAN и выбрать требуемый слой.

### Описание

Соответствующее описание слоя EPLAN, также из раздела Управление слоями (Данные проекта > Управление слоями), например Символьная графика.Общее.
Обратите внимание на то, что описание в стандартном слое изменить невозможно! (Стандартные слои — это все слои с именем EPLANxxx.)

### Имя CAD

Предварительная установка специфического для САПР имени слоя генерируется из имени слоя EPLAN и может быть здесь изменена.

### Видимый / Печать / Заблокировано

Свойства слоя, определенные в пунктах меню Данные проекта > Управление слоями. Изменение свойства на этой вкладке ***не*** влияет на открытый проект.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

### [Добавить]

Щелкните на этой кнопке, чтобы автоматически занести все примененные в текущем проекте слои в таблицу и обработать присвоения. если присвоения слоев уже определены, они ***не*** перезаписываются этой функцией.

!!! note "Замечание:"

    Слои, которые не содержатся в этом списке, переносятся 1:1, по возможности также переносятся все свойства слоев (например, толщина линии, размер шрифта и т. д.).

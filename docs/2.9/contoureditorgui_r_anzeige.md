# Вкладка Отображение (Точка определения контура)

Вы открыли проект. Открыт контур типа 'Выдавить контур' и добавлен минимум один логический элемент (Вставить > Логика > 'Логический элемент'). Выделите логический элемент. Обработать > Свойства. В диалоговом окне Точка определения контура: <Логический элемент> выберите вкладку Отображение.

На этой вкладке указываются настройки отображения для соответствующих свойств точки определения контура. При этом размещенные по отдельности свойства могут быть присоединены друг к другу и образовывать, таким образом, общий блок; это позволяет Вам перемещать все (требуемые) тексты одновременно и предотвращает перекрытие многострочных текстов другими текстами.

Обзор основных элементов диалогового окна:

### Порядок свойств

В раскрывающемся списке предлагаются на выбор различные предварительно определенные конфигурации порядка. Эти конфигурации сохранены со своими именами в точке определения контура.

Выберите из раскрывающегося списка требуемый порядок свойств; этот порядок свойств задает размещенные свойства и настройки их отображения. При предварительно заданных конфигурациях расположения для отображения выбираются уже определенные свойства, и их значения задаются в правой части диалогового окна. Выберите "Определ. пользователем", если вы желаете задать свой порядок.

Если в списке внутри панели инструментов выделено какое-либо свойство, то в таблице, расположенной в правой части диалогового окна, можно обработать соответствующие свойства отображения.

### Свойство/Присвоение

Если в списке выделено какое-либо свойство, то в таблице, расположенной в правой части диалогового окна, можно обработать соответствующие свойства отображения. Здесь доступны уровни иерархии Формат, Рамка текста, Блок выравнивания, Значение / единица измерения и Позиция, открыть которые можно, щелкнув по знаку ![](images/all_plussign_as.png){: .ui-icon }.

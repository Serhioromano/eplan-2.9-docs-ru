# Обработать свойства во всех представлениях вспомогательной функции

Вспомогательная функция может быть многократно представлена в проекте в распределенном виде как однополюсная функция, многополюсная функция, функция обзора, функция парной перекр. ссылки и / или функция функциональной схемы автоматизации. С помощью функции "Свойства (общие)" можно в одной операции одновременно изменить общие свойства во ***всех*** представлениях функции ******вспомогательной функции, как, например, функциональный текст всех представлений замыкающего контакта.

Условия:

* Вы открыли проект.
* Открыт Навигатор устройств или Графический редактор.

1. Чтобы обработать свойства вспомогательной функции в навигаторе устройств, выделите представление функции вспомогательной функции и выберите Всплывающее меню > Свойства (общие).
Чтобы обработать свойства вспомогательной функции в графическом редакторе, активируйте в меню Параметры пункт меню Свойства (общие), щелкните дважды по представлению вспомогательной функции.

!!! info "Для сведения:"

    Открывается диалоговое окно Свойства (общие): ++...++ На вкладке <категория функции> указано обозначение соответствующего определения функции, например, "Замыкающий контакт" или "Плавкий предохранитель".

2. Измените требуемое свойство вспомогательной функции.
3. Щелкните по кнопке ++ОК++.
4. Чтобы обновить информацию, нажмите клавишу ++F5++.

!!! info "Для сведения:"

    Свойства изменяются во всех других соответствующих представлениях функции выбранной вспомогательной функции.

!!! note "Замечание:"

    В режиме редактирования ***Свойства (общие)*** невозможно обрабатывать Видимое ОУ, так как видимое ОУ отдельных представлений функций может различаться. Поле в этом случае недоступно. Однако если вы изменяете Полное ОУ, видимое ОУ автоматически определяется. На вкладке <Категория функции> можно изменить Полное ОУ для всех представлений функций вспомогательной функции. Тогда эти представления функции больше не относятся к устройству. Или на вкладке <Категория функции> (Устройство) можно изменить Полное ОУ для всего устройства.

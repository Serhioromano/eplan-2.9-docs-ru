# Определить формат и объем нумерации

Условие:

Вы открыли проект.

1. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация с данными ПЛК
2. Выберите в диалоговом окне Настройки: Нумерация с данными ПЛК определенную пользователем схему из раскрывающегося списка или создайте с помощью команды ![](images/all_new_as.png){: .ui-icon } (Создать) новую схему.
3. Нажмите в диалоговом окне Настройки: Нумерация с данными ПЛК над полем Группы форматов кнопку ![](images/all_new_as.png){: .ui-icon } (Создать).
4. В диалоговом окне Нумерация с данными ПЛК: Формат выберите из раскрывающегося списка Группа ту группу устройств, для которой требуется настроить формат. Возможен выбор клемм, контактов штекера и устройств.
5. Выберите объем нумерации для выбранной группы из раскрывающегося списка Объем.

!!! note "Замечание:"

    Если вы выбрали запись "Сеть", через расширение сети можно увеличить зону воздействия для нумерации, указав [сетевое расширения](offlinenumerationplcgui_h_netzerweiterung.md).

6. Выделите элемент формата в поле Доступные элементы формата и перенесите его с помощью ![](images/all_arrowright_as.png){: .ui-icon } вправо в список выбранных элементов формата.

!!! info "Для сведения:"

    В зависимости от элемента формата открывается следующее диалоговое окно. В зависимости от выбранного элемента формата это может быть одно из следующих диалоговых окон: [Формат: Данные ПЛК](offlinenumerationplcgui_d_formateigenschaften.md), [Формат: Буквенное обозначение](offlinenumerationplcgui_d_formatkennbuchstabe.md), [Формат: Страница / столбец / строка](offlinenumerationplcgui_d_formatseite.md) или [Формат: Разделитель](offlinenumerationplcgui_d_formattrennzeichen.md).

7. Определите форматирование в следующем диалоговом окне.

!!! info "Для сведения:"

    В поле Предв. просмотр отображается предварительный просмотр формата нумерации. Отдельные элементы формата выводятся в обозначении соединения в той последовательности, в которой они стоят в списке.

8. При необходимости измените последовательность элементов формата с помощью кнопок со стрелками.
9. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Введенные значения сохраняются в выбранной схеме.

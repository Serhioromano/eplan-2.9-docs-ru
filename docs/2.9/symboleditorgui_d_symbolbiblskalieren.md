# Диалоговое окно Копировать и масштабировать библиотеку символов

Вы открыли проект и как минимум одну библиотеку символов. Сервисные программы > Основные данные > Библиотека символов > Масштабировать.

В этом диалоговом окне указывается, какую библиотеку символов необходимо сначала копировать, а затем масштабировать, а также задаются свойства генерируемой библиотеки.

Обзор основных элементов диалогового окна:

### Библиотека символов (источник)

В этом поле отображается масштабируемая библиотека символов EPLAN.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Размер сетки (источник)

В этом поле отображается размер сетки библиотеки символов, указанной в верхнем поле. Эту запись ***нельзя*** изменить.

### Библиотека символов (цель)

Укажите в этом поле имя создаваемой библиотеки символов. Или щелкните мышью по ++...++, чтобы открыть диалоговое окно Открыть и выбрать уже имеющуюся библиотеку символов.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Размер сетки (цель)

Укажите здесь размер сетки (в мм) для вновь генерируемой библиотеки символов.

### Свободные тексты и размещения свойств

В данном групповом поле задайте параметры масштабирования для свободных текстов и размещения свойств:

* Не масштабировать (1:1): если выбран этот параметр, свободные тексты и размещения свойств не масштабируются, а остаются в том виде, в каком были в источнике. В этом случае все остальные поля группового поля недоступны.
* Масштабировать: выберите этот параметр, если необходимо масштабировать свободные тексты и размещения свойств в библиотеке целевых символов. Укажите в этом поле Коэффициент, на который следует масштабировать тексты и размещения свойств. Допускаются только целые числа больше 0.
* Округлить до стандартных размеров: установите данный флажок, если необходимо округлить масштабированные тексты и размещения свойств в библиотеке целевых символов до стандартных размеров, и укажите, используя нижеследующие параметры, в каких единицах должно выполняться округление: в мм или дюймах.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

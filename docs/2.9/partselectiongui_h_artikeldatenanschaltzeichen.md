# Сохранить данные ссылки изделия на функции

Данные ссылки изделия — это сохраненные на функции данные, соотносящиеся к изделию, записанному на этой функции.

На вкладке Изделие диалогового окна Свойства ++...++ вы можете присвоить функции данные ссылки изделия. Таблица в пределах вкладки служит как для отображения данных, задаваемых изделием, так и для ввода справочных данных изделий.

Условия:

* Вы дважды щелкнули по условному обозначению и выбрали в диалоговом окне Свойства ++...++ вкладку Изделие.
* Вы вручную ввели номер изделия или выполнили выбор изделия или устройства, а затем перенесли данные изделия при помощи ++Копировать++ на вкладку Изделие.

1. Выберите из раскрывающегося списка Категория запись "Данные ссылки изделия".

!!! info "Для сведения:"

    В таблице отображаются соответствующие данные из изделия. Если вы вручную ввели номер изделия, не определенный в базе данных изделий, поля будут пустыми.

2. Измените имеющиеся значения или запишите новые.

!!! info "Для сведения:"

    Значения данных ссылки изделия, содержащие отличающееся от свойства изделия значение, представляются в ячейке с желтым цветом фона.

!!! warning "Предупреждение:"

    Если вы удалите запись в поле и оставите поле пустым, в это поле будет снова занесено соответствующее значение из изделия, если вы щелкните по кнопке ++Копировать++ или ++OK++.

3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Данные ссылки изделия сохраняются на функции.

!!! tip "Совет:"

    Если необходимо полностью удалить внесенные вручную данные ссылки изделия, выделите все значения в таблице (за исключением значения для варианта изделия) и удалите их. Затем щелкните мышкой по кнопке ++Копировать++, чтобы заполнить поля значениями из изделия.

### Удаление по функции

Во вкладке Изделие диалогового окна Свойства ++...++ вы можете удалить присвоенные функции данные изделия.

Условие:

Вы дважды щелкнули по условному обозначению и выбрали в диалоговом окне Свойства ++...++ вкладку Изделие.

1. Выделите необходимые номера изделия в списке и щедкните по ![](images/all_delete_as.png){: .ui-icon }.

!!! info "Для сведения:"

    Соответствующие изделия удаляются на функции, но изделия еще сохраняются в проекте.

!!! info "Для сведения:"

    Внесенные данные ссылки изделия полностью удаляются.

!!! tip "Совет:"

    Для того чтобы удалить из проекта сохраненные изделия, которые больше не используются, вы можете воспользоваться сжатием проектов. Для этого в диалоговом окне Настройки: Сжатие на уровне иерархии Убрать данные проекта установите флажок Неиспользованные изделия.

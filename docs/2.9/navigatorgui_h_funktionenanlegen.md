# Создать неразмещенные устройства в навигаторе устройств

В навигаторе устройств посредством выбора определения функции можно создать функцию с новым устройством или функцию для существующего устройства. Новая функция не размещается в схеме соединений.

Неразмещенные устройства можно удалить только в навигаторах данных проекта (например, в навигаторе устройств).

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Данные проекта > Устройства > Навигатор
2. Выделите устройство и выберите пункт всплывающего меню Создать.
3. Выделите определение функции в диалоговом окне Определения функций в поле Выбор.

4. Щелкните по кнопке ++OK++.
5. Выберите в диалоговом окне Свойства ++...++ вкладку <Категория функции>.

!!! info "Для сведения:"

    В поле Видимое ОУ выводится видимое обозначение выделенного устройства.

6. Для создания функции с ***новым*** устройством введите новое видимое ОУ. Чтобы создать функцию для ***имеющегося*** выделенного устройства, оставьте ОУ без изменений.
7. Щелкните по кнопке ++OK++.
8. Для обновления вида нажмите клавишу ++F5++.

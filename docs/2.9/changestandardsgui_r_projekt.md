# Вкладка Проект (Изменение стандарта)

Вы открыли проект. При нескольких открытых проектах: В навигаторе страниц выделен проект. Сервисные программы > Изменение стандарта.

На вкладке настраивается целевой проект. Для адаптации проекта к другому стандарту создается целевой проект.

Обзор основных элементов диалогового окна:

### Имя исходного проекта

В этом поле указан проект для адаптации к другому стандарту. Этот исходный проект при изменении стандарта не изменяется.

### Имя целевого проекта

Укажите в этом поле имя для целевого проекта без расширения имени файла.

### Место хранения

В этом поле расположен каталог для сохранения нового проекта. В качестве места хранения EPLAN предлагает каталог, который устанавливается в поле Проекты через меню Параметры > Настройки > Пользователь > Основные данные > Каталоги.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Скопировать свойства проекта из исходного проекта

Если этот флажок установлен, свойства копируются из свойств исходного проекта; если флажок снят, свойства применяются из шаблона. Определенные пользователем свойства всегда копируются из исходного проекта.

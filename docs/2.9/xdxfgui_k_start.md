# Интерфейс DXF/DWG

EPLAN поддерживает импорт и экспорт данных в формате обмена чертежными данными DXF (***D***rawing e***X***change ***F***ormat) и чертежном формате DWG (DWG = ***D***ra***W***in***G***) компании Autodesk. Эти распространенные форматы позволяют обмениваться данными между различными программами CAD. Оба формата были предложены вместе с выпуском программы AutoCAD и в настоящее время являются отраслевыми стандартами.

С помощью экспорта данных в формате DXF / DWG проектную документацию можно направлять заказчикам, которые не используют EPLAN. Логика EPLAN при этом не переносится. EPLAN позволяет экспортировать страницы проекта или проекты целиком.

Реализованные в EPLAN функции импорта данных в формате DXF / DWG позволяют извлекать конструкторские данные из внешних систем и переносить их в проекты EPLAN. Эту операцию можно выполнить тремя способами:

* Импорт файлов DXF / DWG на уровне проекта в виде графической страницы
* Вставка чертежей DXF / DWG на страницах проекта
* Создание макроса окна EPLAN из файлов DXF / DWG.

!!! note "Замечание:"

    EPLAN поддерживает формат DXF / DWG различных версий формата AutoCAD, которые указаны или можно задать в меню Параметры > Настройки > Пользователь > Интерфейсы > Экспорт и импорт DXF / DWG на вкладке Экспорт. если вы используете файлы DXF / DWG с элементами, которые не являются частью версии формата AutoCAD, EPLAN не будет воспринимать эти элементы.

# Диалоговое окно Экспортировать список присвоений

Вы открыли проект. Данные проекта > ПЛК > Адреса / списки присвоения > [Дополнительно] > Экспортировать список присвоений.

В этом диалоговом окне экспортируются списки присвоений, которые были вновь созданы или обработаны.

!!! note "Замечание:"

    Вы можете экспортировать всякий раз только отдельный список для каждого процессора (ЦПУ). Поэтому в диалоговом окне Адреса / Списки присвоений настройте фильтр таким образом, чтобы отображались только адреса ПЛК ***одного*** ЦПУ.

Обзор основных элементов диалогового окна:

### Специфические для ПЛК настройки

В этом поле отображается схема, задающая формат адресов ПЛК. Схема находится на основе ЦПУ, содержащихся в выборе, и здесь ее нельзя изменить. Если ЦПУ не присвоена схема, используется схема, заданная в настройках проекта. Над полем дополнительно отображаются проект конфигураций и имя рабочей станции, которым присвоено ЦПУ.

### Язык

В этом раскрывающемся списке представлены все языки, определенные в настройках перевода проекта. Выберите язык, на котором экспортируется список присвоений. С помощью записи "Все языки отображения" список присвоений экспортируются на всех языках отображения, выбранных в настройках перевода. Подробную информацию см. в разделе "[Экспорт и импорт многоязычных функциональных текстов в списке присвоений](plcgui_k_zuordnungslisten.md)".

!!! note "Замечание:"

    Чтобы экспортировать список присвоений в языки Юникод (например, китайский и русский), в диалоговом окне Настройки: Специфические для ПЛК на вкладке Формат списка присвоений необходимо выбрать запись "UNICODE" из раскрывающегося списка Набор символов.

### Имя файла

В этом поле введите имя, под которым сохраняется список присвоений. Через [...] откройте диалоговое окно выбора файла и там можно выбрать список присвоений, который необходимо переписать или изменить его имя.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

!!! note "Замечание:"

    Предложенные в диалоговом окне выбора файла типы файла зависят от схемы, выбранной в диалоговом окне Настройки: ПЛК.

# Вкладка Свойства

* Вы открыли проект. Страница > Навигатор. Выделите проект. Выберите пункт всплывающего меню Свойства. Выберите вкладку Свойства.
* Вы выбрали базу данных управления проектами. Вы считали каталоги и проекты. Проект > Управление. Выделите проект. Выберите вкладку Свойства.

На этой вкладке показаны текущие свойства выбранного проекта.

Обзор основных элементов диалогового окна:

### Категория

В зависимости от установленной категории, в этом раскрывающемся списке производится фильтрация вида свойств в столбце Имя свойства. Фильтруется также вид свойств, которые вы можете выбрать при помощи ![](images/all_new_as.png){: .ui-icon } (Создать) в диалоговом окне Выбор свойств.

### Панель инструментов

Кнопка |  Значение
---|---
![](images/all_new_as.png){: .ui-icon } (Создать) |  Открывает диалоговое окно Выбор свойств. Позволяет выбрать свойство.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет один или несколько выделенных свойств.

### Имя свойства

В этом столбце показаны свойства выделенного проекта.

### Значение

В этом столбце отображаются и могут быть обработаны значения свойств проекта. В управлении проектами сначала необходимо активировать режим обработки с помощью кнопки ++Обработать++.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

# Диалоговое окно Регистрация в EPLAN

Вы установили рабочий режим "Регистрация". Вы запускаете EPLAN.

В этом диалоговом окне вы входите с помощью своего имени пользователя и пароля. При запуске EPLAN автоматически открывает это диалоговое окно.

Обзор основных элементов диалогового окна:

### Имя

Введите здесь имя пользователя.

### Пароль

Введите здесь пароль. Для защиты введенных данных пароль отображается не открытым текстом, а в виде символов *.

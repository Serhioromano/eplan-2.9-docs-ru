# Диалоговое окно "Частичные проекты"

Вы открыли проект и выделили его в Навигаторе страниц. В проекте установлены области обработки. Проект > Частичные проекты.

!!! note "Замечание:"

    Чтобы иметь возможность работать с частичными проектами, нужно открыть проект в режиме обработки "Эксклюзивный". При выборе пунктов меню Проект > Частичные проекты осуществляется внутренняя проверка того, был ли проект открыт в эксклюзивном режиме. Если это не так, проект закрывается и открывается автоматически в эксклюзивном режиме. Если это невозможно, появится сообщение о том, что эксклюзивный доступ невозможен, и действие прерывается.

В этом диалоговом окне отображаются определенные областями обработки возможные частичные проекты выбранного открытого в эксклюзивном режиме проекта.

Обзор основных элементов диалогового окна:

### Проект

В этом поле указано имя проекта, для которого вы хотите определить частичные проекты. Запись нельзя обработать, и поле неактивно.

### Частичные проекты

В этой таблице показана информация о частичных проектах, а именно: статус, номер частичного проекта, имя и описание схемы соответствующей области обработки.

В столбце Каталог с помощью кнопки ++...++ можно выбрать другой целевой каталог, при этом данная информация будет сохранена в исходном проекте. В качестве предварительной настройки здесь указан заданный каталог проектов.

Перед выгрузкой установите флажок Только расширить, тогда объекты (страницы, пространства листа, сегменты предварительного планирования и т. д.) сгенерированного частичного проекта получат защиту от изменений. Для эти объектов автоматически активируется свойство Защита от изменений (ид. 3014). Благодаря этому частичный проект можно только расширить. Это означает, что, хотя в данный частичный проект возможна вставка новых страниц, пространства листа, сегментов предварительного планирования и т. д., нельзя удалить или изменить уже имеющиеся объекты.

Частичные проекты пронумерованы в соответствии с очередностью областей обработки и дополнены впереди нулями (например ESS_Sample_Project#003). Если область обработки была добавлена позже, соответствующий частичный проект получает следующий по порядку свободный номер.

Статус частичных проектов поясняется следующими пиктограммами:

Пиктограмма |  Значение
---|---
![](images/subprojects_overlapwarning_as.png){: .ui-icon } |  Область обработки накладывается — части этой области содержатся и в других областях обработки. Выгрузка невозможна.
![](images/subprojects_stored_as.png){: .ui-icon } |  Частичный проект загружен.
![](images/subprojects_filedoff_as.png){: .ui-icon } |  Частичный проект выгружен.
![](images/subprojects_backupfiledoff_as.png){: .ui-icon } |  Частичный проект не является обрабатываемым проектом (*.elk), его нельзя загрузить. Такая пиктограмма отображается, когда, например, частичный проект дополнительно выгружается также посредством резервирования данных.
![](images/revisionmgtgui_redstamp_as.png){: .ui-icon } |  Невозможно выгрузить частичный проект. Относящаяся к нему область обработки находится в процессе ревизии.

### Всплывающее меню

В таблице можно вызвать всплывающее меню, в котором содержатся следующие пункты:

Пункт меню |  Значение
---|---
Закрепить область |  Закрепляет все строки таблицы, которые находятся над маркировкой, или все столбцы, которые находятся слева от маркировки. В этом случае строки ниже или колонки справа от маркировки можно передвигать с помощью полос прокрутки.
Снять закрепление |  Убирает закрепление, т. е. все строки и столбцы можно передвигать с помощью полос прокрутки.
Подобрать ширину столбца |  Все столбцы так подогнаны к таблице по ширине, что и заголовок, и содержимое столбцов полностью читаемы.

### [Выгрузить]

С помощью этой кнопки можно сгенерировать частичные проекты, т. е. выгрузить из проекта. Наложенные области обработки нельзя выгрузить. Если вы выделили такую область обработки, клавиша [Выгрузить] становится недоступной.

### [Сохранить]

С помощью этой кнопки можно снова вставить в проект выгруженные частичные проекты. Возможен многократный выбор выгруженных частичных проектов.

### [Обновить]

С помощью этой кнопки можно обновить главный проект, т. е. "сохранить" новые состояния частичных проектов без их повторной выгрузки.

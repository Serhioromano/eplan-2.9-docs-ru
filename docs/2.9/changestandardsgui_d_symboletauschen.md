# Диалоговое окно Настройки: Присвоение символов (Изменение стандарта)

Вы открыли проект. При нескольких открытых проектах: Вы выделили проект в Навигаторе страниц. Сервисные программы > Замена стандарта. Вы определили на вкладке Проект все настройки. [Далее]. Вы определили на вкладке Поворот / инверсия все настройки. [Далее]. Вы отметили опцию на вкладке Основные данные в строке библиотеки символов в столбце Присвоить отдельно. [Далее]. Щелкните на вкладке Символы по ++Схема++.

В этом диалоговом окне можно создать и осуществлять управление схемой присвоения для присвоения символов библиотеки исходных символов символам библиотеки целевых символов. В схеме присвоения при помощи варианта определяется, будет ли символ повернут или сохранен.

Доступны следующие элементы диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую вы хотите обработать. Вы можете обрабатывать только пользовательские схемы. Через расположенную рядом [панель инструментов](schemeconfig_m_schemaschaltflaechen.md) можно создавать новые схемы, сохранять изменения в схеме, копировать или удалить схему. Кроме того, схемы можно импортировать и экспортировать.

### Описание

Данное поле отображает тексты описаний, которые вы ввели при создании схемы в диалоговом окне Новая схема в поле Описание.

### Присвоить символы

Кнопка |  Значение
---|---
![](images/all_new_as.png){: .ui-icon } (Создать) |  Создает новую строку присвоения. Позволяет по отдельности выбирать и присваивать символы библиотеки исходных и целевых символов, которая была определена по пути [Дополнительно] > Выбрать исходную и целевую библиотеку символов.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет выделенную строку присвоения. Строка должна быть выделена в столбце Строка.

### Источник

Отображает символы библиотеки исходных символов.

* Имя символа. Отображает имена символов исходного символа. [...] открывает установленную библиотеку исходных символов в диалоговом окне Выбор символа. Позволяет отобразить свойства выбранного символа или выбрать другой символ.
* Номер символа. Отображает номера символов исходного символа.
* Вариант. Отображает используемые в исходном проекте варианты исходного символа.
* Описание. Отображает описание символов исходного символа.

### Цель

Отображает символы библиотеки целевых символов.

* Имя символа. Отображает имена символов целевого символа. [...] открывает установленную библиотеку целевых символов в диалоговом окне Выбор символа. Позволяет отобразить свойства выбранного символа или выбрать другой символ.
* Номер символа. Отображает номера символов целевого символа.
* Вариант. Отображает вариант целевого символа. Вариант определяет поворот и инверсию целевого символа.
* Описание. Отображает описание символов целевого символа.

### Статус

Проверяется присвоение символов. В данном столбце отображается статус проверки. Статус "OK", когда определения функций символов соответствуют друг другу, а число выводов устройств исходных символов меньше или равно числу выводов устройства целевых символов.

### [Дополнительно]

Пункт меню |  Значение
---|---
Выбрать исходную и целевую библиотеку символов |  Открывает диалоговое окно Выбрать исходную библиотеку символов. Позволяет задать исходную библиотеку символов для схемы присвоения символов. После этого открывается диалоговое окно Выбрать целевую библиотеку символов. Позволяет задать целевую библиотеку символов для схемы присвоения символов.
Автоматически добавить исходные символы |  Добавляет в строки все символы исходной библиотеки символов, указанной по пути [Дополнительно] > Выбрать исходную и целевую библиотеку символов.
Предложить подходящие целевые символы |  Автоматически присваивает символы целевой библиотеки символов, установленной по пути [Дополнительно] > Выбрать исходную и целевую библиотеку символов, символам исходной библиотеки символов.

# Определенные пользователем свойства

EPLAN позволяет создавать дополнительные определенные пользователем свойства. Эти свойства расширяют свойства, присутствующие по умолчанию, и служат для ввода и отображения дополнительной информации.

Возможно следующее:

* Свойства идентифицируются по свободно выбираемым однозначным идентифицирующим именам.
* Для отображаемого имени, под которым отображается свойство, возможен многоязычный ввод. Переведенные отображаемые имена свойств, определенных пользователем, отображаются на установленном языке диалогового окна.
* Вы можете установить, на каком объекте (например, в проекте, на странице, в функции, в шаблоне сегмента) должно быть доступно свойство. В отличие от конфигурации в проекте, созданные в базе данных изделий и определенные пользователем свойства могут быть назначены только типу объекта "Изделие".
* Вы можете свободно конфигурировать свойства и указывать, какой вид ввода возможен, какие значения по умолчанию имеются в списке выбора и в какой категории отображается свойство.
* В предварительном планировании вы можете сохранить определенные пользователем свойства в шаблоне сегмента и задать для них значения заранее. Эти предварительно определенные свойства по умолчанию доступны при создании новых сегментов структуры и объектов планирования.

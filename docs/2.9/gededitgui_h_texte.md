# Вставить текст

Условие:

Вы открыли страницу, форму, рамку или символ.

1. Вставить > Графика > Текст
2. Введите требуемый текст на вкладке [Текст](gededitgui_r_text.md) диалогового окна Свойства.

!!! info "Для сведения:"

    При открытии диалогового окна для ввода/обработки текста, в строке состояния отображается текущий язык ввода.

!!! tip "Совет:"

    С помощью всплывающего меню можно выполнить дальнейшую обработку текста. Таким образом, например, можно вставлять специальные символы и новые строки. Переходы на новую строку можно вставить с помощью комбинации клавиш [Ctrl] \+ [Ввод].

3. На вкладке Формат задайте такие свойства формата, как размер, цвет и направление, а также свойства рамок, например, размещается ли текст в пределах блока выравнивания и как он должен подгоняться под блок выравнивания.
4. Щелкните по кнопке ++OK++.
5. Разместите элемент на странице.

!!! tip "Совет:"

    Чтобы преобразовать текст в функциональный текст зоны, активируйте флажок функциональный текст зоны.

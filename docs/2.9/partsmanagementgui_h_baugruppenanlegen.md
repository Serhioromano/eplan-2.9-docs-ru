# Работа с узлами

Узлом является коллекция изделий, которые относятся к ***одному*** устройству (напр., кнопочный переключатель с замыкающим контактом, соответствующим кронштейном и нажимной кнопкой). Узел имеет собственный номер изделия и может и сам содержать узлы. изделия, которые содержатся в узлах, имеют также информацию о вариантах изделия. Именно при выборе устройства или для отчетов в этом случае учитываются и технические данные вариантов.

### Присвоение позиций узла

Условия:

* Вы открыли базу данных изделий: Сервисные программы > Изделие > Управление
* Выбран какой-либо узел и выделена вкладка Узел.

1. Щелкните по ![](images/all_new_as.png){: .ui-icon } (Создать), чтобы создать строку в списке позиций узла.
2. Выберите требуемое изделие из структуры дерева или списка в выборе изделий.
Многократный выбор здесь невозможен, т. е. всегда копируется самое верхнее изделие из выделенных.
3. Щелкните по кнопке ++OK++.
4. Повторите эти шаги для всех других присвоений.
5. Если вы хотите, чтобы узел был размещен в трехмерном чертеже монтажных поверхностей в распределенном виде, установите флажок Распределенное размещение узла.
6. Щелкните по ++Применить++, чтобы записать введенные данные и изменения в базу данных, не выходя из базы данных изделий.

!!! note "Замечание:"

    Предварительной настройкой для новой строки является текущая выбранная настройка, т. е. ее данные по числу и номеру изделия будут скопированы. Если ни одна запись не выделена, будет создана пустая строка (с предварительной настройкой Количество = "1").

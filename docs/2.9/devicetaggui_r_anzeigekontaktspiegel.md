# Свойства отображения: Образ контакта

Вы открыли проект. Вы открыли диалоговое окно, в котором можно обрабатывать свойства отображения размещенных свойств, например диалоговое окно "Свойства" для условного обозначения. Для порядка свойств вы выбрали вкладку Образ контакта. Представление образа контакта уже отображается, или вы нажимаете ![](images/all_new_as.png){: .ui-icon } (Создать), чтобы выбрать выравнивание образа контакта.

* [Вкладка Отображение (Устройство)](devicetaggui_r_anzeige.md)
* [Диалоговое окно Размещение свойства](symboleditorgui_d_eigenschaftsplatzierung.md)

Для образа контакта с выбранным выравниванием в таблице Свойство / присвоение сначала отображаются настройки образа контакта (предварительно установленные). Настройки образа контакта, за исключением позиции, предварительно задаются как специфические для проекта (Параметры > Настройки > Проекты > "Имя проекта" > Перекр. ссылки / образы контактов), однако их можно задать отдельно для каждого устройства. Настройка условного обозначения имеет преимущество перед настройкой проекта. Здесь доступны уровни иерархии Позиция и Настройка, которые открываются щелчком по символу ![](images/all_plussign_as.png){: .ui-icon }.

Обзор основных элементов диалогового окна:

### Выровнять автоматически

Снимите этот флажок, если требуется самостоятельно задать позицию образа контакта путем ввода соответствующих значений.

### Абсцисса / ордината

Задайте в этих полях значения для позиции образа контакта; исходной точкой является точка вставки устройства, для которого вы определили образ контакта.

### Определено пользователем

Установите этот флажок, если вы не желаете копировать настройки из настроек проекта; снимите флажок, если Вам (снова) потребуется использовать специфические для проекта предварительные настройки. (В этом случае варианты ввода на этом уровне иерархии деактивированы, т. е. выделены серым цветом.)

### Отображать собственную перекрестную ссылку

Установите этот флажок, если перекрестная ссылка устройства также должна отображаться в образе контакта.

### Отображать как образ символов

Установите этот флажок, если у устройства должно отображаться представление образа символов. В случае этого вида представления выводятся все условные обозначения, по умолчанию подавляется только главная функция.

### Стандарт

Установите этот флажок, если требуется задать значение по умолчанию для представления образов контакта на условных обозначениях или для представления образов контакта в зоне условного обозначения. В этом случае все настройки, находящиеся между этим флажком и настройкой Внедренная форма, включаются для активации / выбора.

### Расположить вертикально

Установите этот флажок, если образ контакта должен быть расположен вертикально.

!!! example "Пример:"

    На левом рисунке показано представление при неустановленном флажке, в случае правого рисунка флажок уже установлен. (В обоих случаях также активирована настройка Отображать в форме таблицы.)(В обоих рисунках буквы S и Ö обозначают замыкающие и размыкающие контакты, причем эта настройка производится в поле Таблица образов контактов: Маркировкав меню Параметры > Настройки > Проекты > "Имя проекта" > Перекр. ссылки / образы контактов > Общее.)

### Представить перекрестную ссылку вертикально

Установите этот флажок, если порядок представления текстов перекрестных ссылок должен быть вертикальным.

### Подчеркнуть перекрестную ссылку размыкающего и переключающего контакта

Установите этот флажок, если перекрестные ссылки размыкающих и переключающих контактов должны выделяться подчеркиванием.

### Уменьшить перекр. ссылку и обозначения выводов устройства

Установите этот флажок, если нужно уменьшить тексты перекрестных ссылок и обозначения выводов устройства; благодаря этому в представлении может разместиться больше информации.

### Отображать символы

Установите этот флажок, если кроме текстов перекрестных ссылок в образе контакта должны отображаться и символы контактов. Следующие флажки дают дополнительные возможности влияния на представление символов в образе контакта.

### Отображать перекрестные ссылки слева

Установите этот флажок, если перекрестная ссылка должна отображаться слева от символа.

### Скрыть левое / правое обозначение вывода устройства

Установите этот флажок, если левое / правое обозначение вывода устройства не должно отображаться; это также может улучшить обзорность представления.

### Отображать в форме таблицы

Установите этот флажок, если число страниц и перекрестные ссылки зоны должны быть внесены в таблицу, причем на порядок влияет настройка Расположить вертикально; при этом заголовки таблиц для размыкающих и замыкающих контактов берутся из настройки Таблица образов контактов: Маркировка (в меню Параметры > Настройки > Проекты > "Имя проекта" > Перекр. ссылки / образы контактов > Общее).

### Табличный крест с переменной длиной

Установите этот флажок, если табличный крест должен автоматически быть адаптирован к максимальной длине записей образа контакта; в таком случае настройка Таблица образов контактов: Ширина / высота (в меню Параметры > Настройки > Проекты > "Имя проекта" > Перекр. ссылки / образы контактов > Общее) ***не*** учитывается.

### Поворот

Образ контакта устройства находится обычно в той же зоне, что и соответствующее устройство. Если этого не нужно, выберите из этого раскрывающегося списка угол, на который должен поворачиваться образ контакта.

### Отображение 1-го / 2-го изделия

Выберите из раскрывающегося списка, должен ли отображаться соответствующий номер изделия главной функции / вспомогательного блока, и если да, то где. (При этом данная настройка всегда используется по выравниванию образа контакта, а номера изделий выводятся в многострочном представлении по центру)

### Отображение 1-го / 2-го типа изделия

Выберите из раскрывающегося списка, должен ли отображаться соответствующий тип изделия, и если да, то где. (При этом данная настройка всегда используется по выравниванию образа контакта, а типы изделий отображаются в многострочном представлении по центру)

### Внедренная форма

Установите этот флажок, если требуется создать образ контакта при помощи внедренных форм. Все настройки, находящиеся между этим флажком и настройкой Стандарт, выделяются серым цветом и больше не доступны для активации.

### Форма

Выберите из этого раскрывающегося списка внедренную форму для создания образов контактов на основе формы.

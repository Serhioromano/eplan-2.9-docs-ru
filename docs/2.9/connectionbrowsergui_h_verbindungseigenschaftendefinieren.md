# Предварительно определить свойства соединения

Соединения получают свои свойства один раз посредством предварительного назначения в проекте и посредством свойств потенциала. Все другие свойства указываются посредством точек определения соединений.

В настройках проекта вы можете предварительно задать для соединений определенные свойства, которые присваиваются каждому соединению при генерировании.

Установленные у проекта свойства наследует потенциал. Свойства от потенциалов в свою очередь наследуют сигналы, а затем - соединения. На отдельных стадиях вы можете изменять данные вручную и определять другие свойства.

1. Выберите пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Свойства.
2. Выберите требуемую вкладку, например [Электротехника](connectionsettingsgui_r_projekteinstellungenelektrotechnik.md), [Гидравлика](connectionsettingsgui_r_projekteinstellungenhydraulik.md) или [Технология производственных процессов](connectionsettingsgui_r_projekteinstellungenhydraulik.md).
3. Укажите в отдельных полях значения по умолчанию для свойств соединения.
4. Подтвердите ввод.

!!! info "Для сведения:"

    Выполняется сохранение значений по умолчанию. При генерировании соединения ему автоматически присваиваются соответствующие значения по умолчанию.

!!! tip "Совет:"

    в больших проектах генерирование соединений может занять больше времени. Если в распоряжении имеется недостаточный запас времени, нажмите в запросе кнопку ++Нет++. Соединения можно будет сгенерировать позднее, вызвав действие Данные проекта > Соединения > Обновить, удерживая нажатыми клавиши ++Shift++ +[Ctrl].

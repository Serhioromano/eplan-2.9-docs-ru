# Перенести изделие

Для переноса с помощью функции Перенести изделие системы линейно присоединенных шкафов Rittal TS8 на систему линейно присоединенных шкафов Rittal VX25 в имеющихся проектах EPLAN Pro Panel применяются условия и ограничения, описанные в разделе "[Перенос изделий электрошкафа](cabinetgui_k_artikelmigrieren.md)".

!!! note "Замечание:"

    Перед использованием функции Перенести изделие сначала прочитайте Заявление об отказе от ответственности EPLAN.

Условия:

* Вы открыли минимум один проект.
* Открыт навигатор пространства листа.

В зависимости от выделения в навигаторе пространства листа выполняется анализ проекта, полного содержания пространства листа или отдельного электрошкафа. Список изделий Rittal TS8, которые можно переносить автоматически, находится [здесь](cabinetgui_k_artikelmigrieren.md).

1. Выделите проект, пространство листа или отдельный электрошкаф в навигаторе пространства листа.
2. Выберите пункты меню Обработать > Перенести изделие или пункт всплывающего меню Перенести изделие.

!!! info "Для сведения:"

    После этого выполняется анализ выбранного изделия, чтобы установить, является ли именем производителя "Rittal", и представляют ли номера заказов изделий изделия TS8.

Если электрошкаф невозможно перенести полностью из-за отсутствия изделий для замены, отображается диалоговое окно Перенести изделие. В этом диалоговом окне можно определить дальнейший порядок действий.

1. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Изделия переносятся по возможности.

2. Нажмите кнопку ++Экспортировать++.

!!! info "Для сведения:"

    Требуемые, но отсутствующие изделия Rittal VX25 для замены, экспортируются в файл Parts.csv. Этот файл используется для автоматического импорта этого изделия в корзину EPLAN Data Portal.

Доступные для переноса принадлежности, определенные после автоматического переноса с помощью [контрольного прогона 026104](msg/messages_p_026104.md), можно перенести вручную позже, когда появится изделие для замены.

Список изделий, которые можно переносить вручную, находится [здесь](cabinetgui_k_artikelmigrieren.md).

Условия:

* Перенос электрошкафа выполнен.
* Контрольный прогон 026104 выполнен.

1. Выделите изделие, определенное с помощью контрольного прогона, в навигаторе пространства листа или в пространстве листа.
2. Выберите пункты меню Обработать > Перенести изделие или пункт всплывающего меню Перенести изделие.

!!! info "Для сведения:"

    Выбранное изделие заменяется изделием VX25.

!!! info "Для сведения:"

    Изделие для замены находится рядом с курсором.

3. Разместите изделие на подходящем месте в перенесенном электрошкафу.

!!! tip "Совет:"

    В диалоговом окне "Настройки" в пунктах меню Параметры > Настройки > Пользователь > Устройство > Перенос изделия можно задать, какие вспомогательные элементы следует перенести при переносе изделия. По умолчанию вспомогательные элементы не переносятся.

После переноса электрошкафов TS8 в пределах построения в линию из-за ширины электрошкафов VX25, которая больше на 2 мм, возникают пересечения, которые контрольный прогон 026011 квалифицирует как наложение. Такие наложения позже можно откорректировать.

Условия:

* Перенос построения электрошкафов в линию выполнен.
* Контрольный прогон 026011 выполнен.

1. Выберите пункты меню Обработать > Корректировать интервал электрошкафов.
2. Очертите все электрошкафы VX25 построения в линию прямоугольником.

!!! info "Для сведения:"

    Электрошкафы VX25 сместятся на 2 мм по ширине.

!!! info "Для сведения:"

    Наложение корректируется.

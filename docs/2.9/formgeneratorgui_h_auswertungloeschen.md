# Удалить страницы отчета

При удалении страниц отчетов всегда удаляются страницы отчетов, принадлежащие [блоку отчетов](formgeneratorgui_k_auswertungsbloecke.md).

Условие:

Вы открыли проект.

1. Сервисные программы > Отчеты > Генерировать
2. Выберите в диалоговом окне Отчеты — <Имя проекта> вкладку Отчеты.
3. Выделите один блок отчетов.
4. Выберите пункт всплывающего меню Удалить блок отчетов.

!!! info "Для сведения:"

    Удаляются все страницы отчетов, принадлежащие этому блоку отчетов.

# Вкладка <Технологический контур> / Вкладка <Функция ТК>

Вы открыли проект.

* Вы создали в навигаторе предварительного планирования или вставили в Графическом редакторе объект планирования с определением сегмента "Измерительный контур", "Исполнительный механизм", "Функция измерения" или "Функция потребителя". Выберите в диалоговом окне Свойства ++...++ вкладку <Технологический контур> или <Функция ТК>.
* Вы выделили технологический контур или функцию ТК в навигаторе предварительного планирования или в Графическом редакторе. Обработать > Свойства (или Всплывающее меню > Свойства). Выберите вкладку <Технологический контур> или <Функция ТК>.

На этой вкладке задаются свойства выделенного технологического контура или функции ТК.

Если в диалоговом окне свойств записи в полях отличаются от стандартных значений присвоенного шаблона сегмента, они выделяются [желтым цветом фона](planninggui_k_segmentvorlagen.md).

Обзор основных элементов диалогового окна:

Групповое поле Обозначение

В этих полях можно задать все свойства для номера ТК нового технологического контура / функции ТК. Номер ТК автоматически складывается из этих свойств и отображается в поле Полное обозначение.

### Техническое оборудование

В этом поле вы по выбору вводите буквенное обозначение для технического оборудования.

### Измеряемая величина

В этом поле введите буквенное обозначение для измеряемой величины. При помощи кнопки ++...++ можно выбрать заранее определенное буквенное обозначение для измеряемой величины.

### Функциональный признак прибора

В этом поле введите буквенное обозначение для функционального признака прибора. При помощи кнопки ++...++ можно выбрать предопределенное буквенное обозначение для функционального признака прибора. (Буквенные обозначения для функциональных признаков приборов можно заранее определить в диалоговом окне [Настройки: Функциональные признаки приборов](planninggui_d_einstellverarbeitfunktion.md).)

### Номер

В этом поле введите счетчик.

### Полное обозначение

В этом поле отображается номер ТК, включая обозначения вышестоящих сегментов.

* * *

### Описание

Введите здесь описательный текст для объекта планирования. Этот текст по умолчанию отобразится в навигаторе предварительного планирования вместе с обозначением объекта планирования.

### Шаблон сегмента

В этом поле отображается шаблон сегмента, присвоенный сегменту. С помощью кнопки ++...++ откройте диалоговое окно Выбрать шаблон сегмента, чтобы выбрать шаблон сегмента. Отображаются только шаблоны сегментов, подходящие к определению текущего сегмента. При замене шаблона сегмента значения свойств нового шаблона сегмента переносятся на сегмент и отображаются в диалоговом окне Свойства. Значения, измененные вручную, сохраняются.

С помощью кнопки ![](images/all_delete_as.png){: .ui-icon } (Удалить) из сегмента можно удалить ссылку на шаблон сегмента. Определенные в шаблоне сегмента значения свойств удаляются из сегмента. Значения, измененные вручную, сохраняются.

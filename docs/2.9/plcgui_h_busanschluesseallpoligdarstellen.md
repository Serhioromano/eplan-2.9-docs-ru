# Многополюсное представление выводов шины

Для обмена данными ПЛК в формате AutomationML структура сети для обмена определяется по соединенным однополюсным или многополюсным выводам шины; для всех других форматов учитывается только однополюсное представление.

Многополюсное представление шины не требуется, подробные многополюсные представления стандартных коммуникационных шин можно выполнить с помощью схемы кабельных соединений.

В базе данных изделий выводы кабеля сети / шины (кратко: выводы шины) и выводы устройства ПЛК карты ПЛК представляются в виде шаблонов функций. Вывод шины представляет собой штекер и изображается только в виде ***одного*** шаблона функции. Если необходимо разместить выводы шины однополюсно, а остальные выводы устройства ПЛК многополюсно, следует использовать макросы изделия, содержащие как однополюсные, так и многополюсные варианты.

!!! example "Пример:"

    Слева отображается однополюсный вариант макроса изделия, а справа — многополюсный.

Чтобы шаблоны функций правильно перекрывались в навигаторе ПЛК, требуется, чтобы обозначение штекера (и, при необходимости, имя интерфейса шины) или обозначение вывода устройства было внесено на однополюсном выводе шины.

При необходимости однополюсный вывод шины можно дополнительно разместить с видом представления "Многополюсный".

### Многополюсное преставление всех контактов вывода шины

Если требуется, чтобы коммуникационная шина была представлена многополюсно со всеми отдельными контактами вывода шины, можно увеличить количество выводов на выводе шины и разместить для заданного количества выводов подходящий символ.

Условия:

* В Графическом редакторе открыта страница проекта.
* Открыт навигатор ПЛК.

1. Выберите следующие пункты меню: Вставить > Блок / вывод устройства / монтажная плата > Вывод шины
2. Разместите вывод шины на странице проекта.
3. На вкладке Данные символа / функции диалогового окна "Свойства" убедитесь, что вывод шины имеет вид представления "Однополюсный".
4. В поле Выводы устройства введите количество выводов устройства, т. е. количество контактов вывода кабеля сети / шины.
5. Щелкните по кнопке ++OK++.
6. Выделите вывод шины в навигаторе ПЛК и во всплывающем меню выберите пункт Разместить.
7. Нажмите клавишу ++Backspace++, и выберите символ для вывода шины с соответствующим количеством выводов.
8. Подтвердите ввод.
9. Разместите символ на странице проекта. При необходимости измените вид представления на "Многополюсный".

!!! info "Для сведения:"

    В навигаторе ПЛК отображается многополюсный и однополюсный выводы шины, данные которых синхронизируются, и которые таким образом вместе перекрывают шаблон функции.

!!! note "Замечание:"

    Многополюсный вывод шины можно размещать со всеми его контактами в символе только один раз. Только таким образом можно обеспечить общую обработку и правильное перекрытие шаблонов функций.

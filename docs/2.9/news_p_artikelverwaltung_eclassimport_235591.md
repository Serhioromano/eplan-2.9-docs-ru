# Расширения для импорта eCl@ss

Начиная с версии 2.6, платформа EPLAN поддерживает импорт данных изделий в базу данных изделий в соответствии с eCl@ss версии 9.0. В EPLAN 2.9 теперь поддерживается еще и eCl@ss версии 10.0.1. В обеих версиях для импорта используется формат "BMEcat" на основе XML.

Таким образом в диалоговом окне Импортировать записи данных для поля Тип файла дополнительно доступна новая запись "eCl@ss 10.0.1 (BMEcat 2005.1)". В рамках расширений, перечисленных здесь, был удален лишний тип файла "BMEcat (1.2)".

### Фильтр для выбора признака при импорте eCl@ss

В настройках импорта eCl@ss теперь доступна опция фильтрации для присвоения признака при выборе признаков. Для этого в диалоговое окно Выбор признака было встроено поле Фильтр признака. (Это диалоговое окно выбора можно открыть, например, щелкнув по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать) на вкладке Присвоение признака.)

В данном поле введите текст для поиска и щелкните по ![](images/all_search_as.png){: .ui-icon } (Поиск). При этом выполняется поиск имен признаков и их описания. Затем в списке признаков отображаются только записи, содержащие искомый текст.

### Определенные пользователем свойства при присвоении признака

При присвоении признака для выбранного признака eCl@ss теперь вместо свойства изделия EPLAN можно также присвоить определенное пользователем свойство. Для этого на вкладке Присвоение признака доступен новый столбец Определенное пользователем свойство. Если свойство изделия уже назначено, определенное пользователем свойство не может быть назначено, и наоборот. Присвоение одного или нескольких определенных пользователем свойств, зависит от кардинальности признака eCl@ss.

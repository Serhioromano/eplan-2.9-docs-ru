# Вкладка Определение трубопровода

Вы открыли проект.

* В графическом редакторе выделена точка определения трубопровода или вывод трубопровода. Обработать > Свойства. В диалоговом окне Свойства ++...++ выберите вкладку Определение трубопровода.
* Вставить > Точка определения трубопровода или Вставить > Вывод трубопровода. В диалоговом окне Свойства ++...++ выберите вкладку Определение трубопровода.

На этой вкладке задаются свойства точки определения трубопровода или вывода трубопровода.

Обзор основных элементов диалогового окна:

###

### Имя трубопровода

Введите здесь имя для трубопровода.

### Класс трубы:Вещество

Укажите здесь имя для класса трубы / вещества. С помощью [...] откройте диалоговое окно выбора, в котором для выбора доступны классы трубы или вещества, управляемые в диалоговом окне Шаблоны сегментов - <Имя проекта>.

### Состояние (вещество)

Введите здесь состояние для выбранного вещества или выберите запись из раскрывающегося списка. В списке отображаются все значения, которые уже вводились для вещества в других местах проекта.

### Поперечное сечение / диаметр:Единица измерения сечения / диаметра

Поперечное сечение или диаметр соединения копируется из базы данных изделий и могут быть использованы в качестве критерия отбора при выборе устройств. Щелкните по ++...++, чтобы открыть диалоговое окно Поперечные сечения / диаметры и выбрать сечение / диаметр из списка.

Если в поле Поперечное сечение / диаметр вы выбрали предварительно определенное поперечное сечение / диаметр, то в поле Единица измерения сечения / диаметра автоматически появится соответствующая единица. С помощью раскрывающегося списка возможен выбор другой единицы измерения. Обзор возможных единиц измерения приводится в соответствующем разделе "[Соединения: Единицы измерения длины и поперечного сечения](connectionbrowsergui_k_einheitenverbindungen.md)".

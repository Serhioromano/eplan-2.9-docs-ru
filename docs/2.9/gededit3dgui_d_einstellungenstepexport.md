# Диалоговое окно Настройки: Экспорт STEP / VRML

Параметры > Настройки > Пользователь > Интерфейсы > Экспорт STEP / VRML.

В этом диалоговом окне указываются настройки для экспорта файлов STEP или VRML, после чего настройки сохраняются в виде схемы. Схемы для экспорта файлов STEP / VRML сохраняются отдельно для каждого пользователя. В этом диалоговом окне можно скопировать и обработать существующую схему или создать новую.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

* * *

Групповое поле 'Средство экспорта'

В данном групповом поле задайте, на каком средстве экспорта должен выполняться экспорт:

* Носитель данных: При выборе этого параметра файл STEP / VRML выводится на носитель данных. Каждое пространство листа сохраняется как отдельный файл STEP.
В поле Каталог вывода укажите место хранения. В качестве целевых для экспорта данных можно указать любой дисковод и подкаталог.
Нажав ++...++, откройте диалоговое окно Выбрать каталог и перейдите к нужному каталогу или создайте новый.
С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.
* Электронная почта: При выборе этой опции файл STEP / VRML высылается по электронной почте. После запуска функции файл запаковывается посредством внутреннего архиватора, после чего запускается настроенный редактор электронной почты.
Из раскрывающегося списка Ограничение размера выберите максимальный размер для отдельных файлов, отправляемых по электронной почте. Вы можете установить значение от 1 до 10 Mб с шагом 1 Mб или выбрать запись "Неогранич.". EPLAN автоматически создаст то количество файлов для отправки, которое необходимо для отправки всех частей. Всем частям будет сопутствовать следующий текст: "<Имя файла> n/m". При этом "n" — это номер отдельного файла для отправки по электронной почте, а "m" — общее количество файлов.

!!! note "Замечание:"

    Для обмена электронными сообщениями на вашем компьютере должен быть установлен и настроен клиент электронной почты, поддерживающий MAPI. По этой же причине не может применяться почтовое приложение Windows 8®.

* * *

### Генерировать имена файлов

При настройках имени файла существуют следующие возможности:

* Из непрерывного номера
* Из имени пространства листа
* Из свойств

В свойствах доступны свойства пространства листа, свойства проекта и разделитель.

### Версия формата: (STEP)

Для версий формата имеются следующие возможности:

* STEP AP 203
* STEP AP 214, версия CD
* STEP AP 214, версия DIS
* STEP AP 214, версия IS

### Только внешн. представление электрошкафа: (STEP)

Если установлен данный флажок (стандартная настройка), то электрошкаф экспортируется со всеми функциональными элементами, которые видны снаружи при закрытом электрошкафе. Это правило также действует для произвольно размещенной монтажной платы, которая не является составной частью электрошкафа.

Экспортируются все размещенные изделия с ниже расположенными функциональными элементами, которые в иерархии находятся под электрошкафом или монтажной платой.

### Отдельный экспорт для каждого электрошкафа: (STEP)

Если установлен этот флажок (стандартная настройка), то для электрошкафа или свободно стоящей монтажной платы генерируется отдельный файл STEP. Если для генерирования имени файла используется настройка Из имени пространства листа, то имя электрошкафа зависит от имени файла.

### Экспортировать вспомогательные элементы: (STEP)

Если данный флажок снят (стандартная настройка), то следующие вспомогательные элементы не экспортируются:

* Сегменты маршрутизации
* Области маршрутизации
* Соединительные отверстия для проводов
* Запретные зоны для сверления
* Запретные зоны для размещения
* Монтажные отверстия

### Экспортировать маршрутизируемые соединения: (STEP)

Если активирован этот флажок, в ходе экспорта STEP выполняется экспорт маршрутизируемых соединений, имеющихся в пространстве листа. После импорта образованного файла STEP во внешнюю программу CAD в ней могут отображаться маршрутизируемые соединения.

* * *

Групповое поле Упрощенное представление (STEP)

В данном групповом поле можно настроить упрощенный экспорт функциональных элементов, как при представлении в пространстве листа.

* Клеммники (Определение блока): Если этот флажок установлен (стандартная настройка), то клеммники экспортируются как упрощенный блок.
* 3D-макросы: Если этот флажок установлен (стандартная настройка), то графические макросы, шины и кабельные каналы упрощаются до прямоугольного параллелепипеда.

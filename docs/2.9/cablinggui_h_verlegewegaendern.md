# Изменить маршрутизацию (топология)

Трассу маршрутизации, определенную при маршрутизации маршрутизируемого соединения, можно изменять при изменении маршрутизации.

* Маршрутизируемые соединения можно перекладывать с одного сегмента маршрутизации на другой.
* Немаршрутизированные соединения можно прокладывать по сегменту маршрутизации, в результате чего из них получаются маршрутизированные соединения.

При ручном изменении маршрутизации заполняется свойство соединения Топология: Трасса маршрутизации — по умолчанию. После этого при маршрутизации этих соединений фильтры соединений, настроенные для этого сегмента маршрутизации, не учитываются.

Для изменения маршрутизации используйте средние точки сегментов маршрутизации или графику непроложенных маршрутизируемых соединений (тонкие оранжевые линии). При помощи перетягивания такой средней точки / такой линии на другой сегмент маршрутизации можно изменять трассу маршрутизации соединений. Если через сегмент маршрутизации проходят несколько соединений, на экран выводится диалоговое окно выбора, в котором вы указываете, какое из этих соединений необходимо перенести на новый сегмент маршрутизации. Затем на выбранных соединениях данный сегмент маршрутизации заносится в свойство Топология: Трасса маршрутизации — по умолчанию заносятся сегменты маршрутизации.

1. Щелкните мышью по сегменту маршрутизации.

!!! info "Для сведения:"

    На сегменте маршрутизации будут видны точки захвата.

2. Перетяните среднюю точку сегмента маршрутизации на другой участок маршрутизации.

!!! info "Для сведения:"

    Отобразится диалоговое окно Соединения, в котором перечислены все соединения, проходящие через сегмент маршрутизации источника. Отображенные соединения уже выделены в диалоговом окне.

3. Установите флажок в столбце Изменить маршрутизацию у тех соединений, которые вы хотите заново маршрутизировать в выбранном сегменте маршрутизации цели.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выбранные соединения будут маршрутизированы заново.

!!! tip "Совет:"

    Если вы хотите снова учитывать фильтр соединений при маршрутизации, можно вручную удалить для соответствующих соединений значение свойства Топология: Трасса маршрутизации — по умолчанию. Или убрать соединение с указанного здесь сегмента маршрутизации, в этом случае сегмент маршрутизации будет удален из свойства.

# Диалоговое окно Управление правами: Пользователь

Сервисные программы > Управление правами > Пользователь

В этом диалоговом окне можно создавать и изменять пользователей.

Обзор основных элементов диалогового окна:

Если выделить в левом поле папку Пользователи, справа откроется список всех зарегистрированных пользователей. [Пиктограммы](uradministration_d_rechte.md) используются для того, чтобы различать пользователей, созданных в EPLAN, и пользователей, импортированных из Windows.

### [Изменить пароль]

Открывает диалоговое окно Изменить пароль. Здесь вы можете изменить пароль, при помощи которого соответствующий пользователь регистрируется в системе EPLAN. ***Невозможно*** изменить пароль для предопределенного пользователя "GUEST" или для пользователей, импортированных из Windows.

Для каждого отдельного пользователя доступны следующие элементы диалогового окна:

Если в левом поле развернуть папку Пользователи и выделить пункт ***Отдельный пользователь***, в списке справа будут показаны группы пользователей. Галочка перед соответствующей группой пользователей показывает, что выделенный пользователь относится к данной группе и, соответственно, обладает правами, присвоенными этой группе. Любой пользователь может относится к любому количеству групп.

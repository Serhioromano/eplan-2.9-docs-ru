# Документировать библиотеки символов

Документация библиотеки символов может создаваться в оригинальном проекте или, по желанию, в отдельном проекте. В таком проекте Вы можете, например, добавлять рамки или пополнять текстовую информацию, однако обрабатывать символы Вы ***не*** можете!

Для создания перечня символов, т.е. оглавления символов в библиотеке символов, действуйте следующим образом:

Условия:

* Вы открыли проект, но редактор символов ***не*** активирован.
* С помощью пунктов меню Параметры > Настройки > Проекты > "Имя проекта" > Отчеты > Вывод в страницах > Перечень символов вы выбрали форму и задали настройки для генерируемых страниц.

1. Сервисные программы > Отчеты > Генерировать
2. Если во вкладке Отчеты в диалоговом окне Отчеты ‑ <Имя проекта> не указан перечень символов, нажмите на ++Создать++.
3. Выберите в диалоговом окне Определить отчет тип отчета Перечень символов.
4. При необходимости измените настройки в полях Форма вывода и Исходный проект.
5. Активируйте флажок Индивидуальный выбор, если не все библиотеки символов должны документироваться автоматически.
6. Щелкните по кнопке ++OK++.
7. Если установлен флажок Индивидуальный выбор, то в диалоговом окне Выбрать вручную необходимо указать, какие библиотеки символов должны учитываться.
8. Щелкните по кнопке ++OK++.
9. При необходимости в диалоговом окне Настройки — перечень символов задайте схемы сортировки, фильтрации и разбивки страниц.
10. Щелкните по кнопке ++OK++.
11. В диалоговом окне Перечень символов задайте настройки для создаваемых страниц.
12. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Создается перечень символов, причем продолжительность этого процесса зависит от размера выбранных библиотек символов.

!!! info "Для сведения:"

    В список отчетов добавляется уровень древовидной структуры Перечень символов и под ним — список документированных библиотек символов.

13. Закройте диалоговое окно Отчеты — <Имя проекта>.

!!! info "Для сведения:"

    Сгенерированные страницы отчета вставляются в заданный целевой проект.

!!! note "Замечание:"

    При помощи свойства формы Только использованные символы (ид. 13114) можно ограничить вывод перечня символами, которые используются в проекте.

# Копирование данных проектов EPLAN PPE: Группы устройств

Из групп приборов элементов ТК при копировании проектов PPE автоматически генерируются следующие ***определения сегментов***:

* Датчик
* Исполнительный элемент
* Задатчики
* Преобразователи
* Функциональное устройство
* Распределительное устройство
* Анализатор / сигнальное устройство
* Автоматизация
* Функция ПО
* Защитное устройство
* Отопит. / осветит. устройство
* Прочие устройства
* Аппараты
* Арматура
* Машина.

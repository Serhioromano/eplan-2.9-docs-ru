# Диалоговое окно Выбор значения

Открыто диалоговое окно Фильтр. Щелкните по ++...++ в поле Фильтр.

Выберите в этом диалоговом окне значение для сравнения, с которым следует сравнить значение выбранного свойства. Возможно использование [заполнителей](searchandreplacegui_k_platzhalter.md) типа "*" и "?". Ввод нескольких значений на одной строке (с разделением точкой с запятой) расценивается как операция ИЛИ.

Обзор основных элементов диалогового окна:

Список содержит значения, доступные для данного критерия. Если в качестве критерия выбран, например, структурный идентификатор, диалоговое окно выбора содержит имеющиеся в проекте значения этого структурного идентификатора.

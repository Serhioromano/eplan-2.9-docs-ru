# Вкладка Формат списка присвоений

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > ПЛК. В диалоговом окне Настройки: ПЛК щелкните рядом с полем Специфические для ПЛК настройки по кнопке ++...++. В диалоговом окне Настройки: Специфич. для ПЛК выберите вкладку Формат списка присвоений.

В этой вкладке вы определяете формат списка присвоений.

Обзор основных элементов диалогового окна:

### Набор символов

Выберите набор символов, в котором Вы хотите сохранить списки присовений. Доступные форматы — ANSI, ASCII и UNICODE.

### Разрешить умлауты

Включите этот флажок, если хотите использовать умлауты в списках присвоений.

Если флажок выключен, умлауты заменяются при написании в списках присвоений.

### Имя файла

Укажите путь и имя файла списка присвоений вручную либо нажмите на ++...++, чтобы открыть диалоговое окно для выбора файла.

Выбрав команды Всплывающее меню > Вставить переменную пути, перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути, специфических для EPLAN.

### Строка верхн. колонт.

Здесь вы можете ввести текст, который появится в верхнем колонтитуле списка присвоений.

### Строка нижн. колонт.

Здесь вы можете ввести текст, который появится в нижнем колонтитуле списка присвоений.

### Формат строки списка присвоений

С помощью [...] откройте диалоговое окно Формат строки списка присвоений. Здесь можно изменить формат цифровых и аналоговых адресов ПЛК в строках списков присвоений.

### Формат адреса списка присвоений

С помощью [...] откройте диалоговое окно Формат адреса списка присвоений. Здесь можно изменить формат цифровых и аналоговых адресов ПЛК в адресах списков присвоений.

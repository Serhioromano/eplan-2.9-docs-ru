# Всплывающие подсказки к значениям проекта при импорте данных

Если при импорте данных предварительного планирования, устройств, соединения и проводного монтажа между значениями в источнике данных и значениями в проекте возникнут различия, эти поля в соответствующем диалоговом окне синхронизации (например, Синхронизировать данные предварительного планирования) будут выделены темно-серым цветом. Для лучшего распознавания различий между источником данных и проектом теперь для таких полей во всплывающих подсказках отображаются соответствующие значения свойств в проекте.

![](images/benefit.png)Эффект:

Благодаря отображению отличающихся значений проекта во всплывающих подсказках вам не придется постоянно прокручивать назад и вперед в разделенных на две части диалоговых окнах синхронизации, чтобы сравнить значения. Нет необходимости отображать столбцы для проекта, чтобы просмотреть и сравнить значения проекта.

# Диалоговое окно Формат для автоматического описания страниц

Вы открыли проект. Сервисные программы > Основные данные > Форма > Открыть > [Открыть]. Вы открыли диалоговое окно Свойства формы — <Имя формы> (Всплывающее меню > Свойства на форме в диалоговом окне навигатора Страницы — <Имя проекта>) и в поле Значение свойства Формат для автоматического описания страниц щелкнули кнопку ++...++.

В этом диалоговом окне установите, какая информация страниц (имя страницы, описание страницы, установка и т.д.) должна быть автоматически задана при генерации отчетов для описания страниц.

Обзор основных элементов диалогового окна:

### Доступные элементы формата

В этом столбце перечислены все возможные элементы формы. К ним относятся:

* Тип страницы / тип формы: Автоматически копирует тип страницы / тип формы в описание страниц.
* Свойства формы: Позволяет копировать в описание страниц определенные свойства форм из диалогового окна Выбор свойств.
* Имя объекта заголовка: Автоматически копирует объект заголовка в описание страниц.
* Имя первого объекта данных: Автоматически копирует первый объект данных в описание страниц. (В спецификации клеммника первый объект данных списка отчетов 1 мог бы быть например клеммой номер 1, и, если отображаются 30 клемм на страницу, клемма номер 31 — в списке отчетов 2.)
* Имя последнего объекта данных: Копирует последний объект данных в описание страниц. (Для верхней спецификации клеммника это бы выглядело так — клемма номер 30 на странице отчетов 1 и клемма номер 60 на странице отчетов 2.)
* Разделитель заголовков: открывает диалоговое окно Формат: разделитель, в котором для заголовков может быть установлен отдельный разделитель или разделительный текст. Этот разделитель можно определить только ***один*** раз и автоматически вставить в ***конце*** списка Выбранные элементы формата; эту позицию нельзя изменить.

### Выбранные элементы формата

Здесь из списка Доступные элементы формата выводятся скопированные элементы. При этом отображение включает в себя следующие столбцы:

* Элемент формата: Имя элемента формата идентично имени из списка Доступные элементы формата
* Символ: Символ элемента формата, используемый в поле Предв. просмотр
* Настройка: Число знаков / длина записи / ид. свойства.

### Предв. просмотр

В этой строке выводится предварительный просмотр элементов списка Выбранные элементы формы, при этом используются применяемые в столбце Настройка записи. Чтобы отличить элемент формата Разделитель от элемента формата Разделитель заголовков, для последнего используется символ "<" и ">", например <[SP]> ("space" = пробел).

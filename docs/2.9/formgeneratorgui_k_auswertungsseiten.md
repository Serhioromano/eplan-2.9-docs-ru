# Страницы отчетов

Страницы отчетов представляют собой результат отчетов на отдельных страницах. Отчеты подразделяются на [типы отчетов](formgeneratorgui_k_auswertungstypen.md). Благодаря этому анализируются группы одинаковой информации. Чтобы на страницах отчетов отобразить структурированные и отформатированные данные проекта, типам отчетов присваиваются формы. Формы содержат заполнители, которые заполняются данными проекта во время генерации отчета.

При генерировании отчета можно выбрать, генерировать страницы отчета или [встроенный отчет](formgeneratorgui_k_ausgabeformen.md). Данные проекта выводятся на страницах отчета, если при генерировании отчетов как форм вывода выбрать "Страницу". EPLAN сортирует сгенерированные страницы в структуру имеющихся страниц проекта. Страницу отчета можно определить по типу страницы, например "Спецификация клеммника (A)". Посредством [сортировки страниц отчетов](formgeneratorgui_k_seitensortierung.md) указывается, генерируется отчет в одном или нескольких [блоках отчетов](formgeneratorgui_k_auswertungsbloecke.md).

# Операция: MfImportToolbarAction

**Параметр** |  **Описание**
---|---
File  |  Импортируемый XML-файл, включая путь файла.
Replace  |  Имеющиеся одноименные панели инструментов заменяются.

!!! note "Замечание:"

    Данная операция использует главное окно.

!!! example "Пример:"

    MfImportToolbarAction
    /File:"?"
    /Replace:Yes

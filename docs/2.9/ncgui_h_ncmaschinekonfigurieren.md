# Конфигурирование машины ЧУ

Вместо всех интерфейсов ЧУ для конфигурации машины здесь применяется формат "Rittal - Perforex BC / Secarex". Конфигурация машин на других интерфейсах происходит аналогичным образом, несмотря на это, она отличается по виду присоединяемых в диалоговом окне Настройки значений.

1. Выберите пункты меню Параметры > Настройки > Фирма > Механическая обработка > Экспорт Rittal - Perforex BC / Secarex.
2. В диалоговом окне Настройки: Экспорт Rittal - Perforex BC / Secarex выберите требуемую машину. При необходимости измените настройки на вкладках или сгенерируйте новое определение машины при помощи кнопки ![](images/all_new_as.png){: .ui-icon } (Создать).
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Настроенные машинные данные доступны при экспорте ЧУ в диалоговом окне экспорта под соответствующими названиями машин.

# Обзор свойств изделий EPLAN

В файле конфигурации festo.cfg можно задать присвоение свойств изделий Festo свойствам изделий EPLAN. На вкладках базы данных изделий EPLAN можно ввести эти свойства.

В справке [База данных изделий: Импорт и экспорт CSV для изделий](partsmanagementgui_k_konfigurationcsvartikel.md) найдите отсортированные по обозначениям вкладок свойства изделий. А в справке "Свойства EPLAN" (под обзорами) можно отобразить внутренние свойства EPLAN для класса объекта "Изделие (вложенное)" отсортированными в алфавитном порядке.

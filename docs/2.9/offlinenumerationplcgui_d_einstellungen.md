# Диалоговое окно Настройки: Нумерация с данными ПЛК

Вы открыли проект.

* Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация с данными ПЛК
* Вы выделили устройства в графическом редакторе или в навигаторе, или же Вы выделили одну или несколько страниц в диалоговом окне навигатора Страницы — <Имя проекта>. Данные проекта > Устройствo > Нумерация с данными ПЛК. В диалоговом окне Нумерация с данными ПЛК щелкните рядом с полем Настройки по кнопке ++...++.

В этом диалоговом окне указываются настройки для нумерации устройств с данными ПЛК.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Группы форматов

Здесь выводятся группы устройств, для которых был настроен формат. С помощью кнопок, находящихся над списком, можно генерировать, дополнительно изменять или удалять группы форматов. Последовательность групп не влияет на нумерацию (в отличие от нумерации соединений).

### [Сетевое расширение]

Открывает диалоговое окно [Сетевое расширение](offlinenumerationplcgui_d_netzerweiterung.md). В этом диалоговом окне можно расширить определение сети. Эта настройка влияет на обозначение, если в качестве объема выбрана "сеть".

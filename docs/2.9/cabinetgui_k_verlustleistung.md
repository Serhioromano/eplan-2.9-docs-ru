# Расчет общей мощности потерь на распределительных устройствах

В EPLAN Pro Panel есть функция, позволяющая определить общую мощность потерь на распределительных устройствах. Общая мощность потерь состоит из мощности потерь устройств и мощности потерь систем сборных шин. Результаты этого расчета можно передать в программы для расчета нагревания (например, ***Rittal Therm***), что позволит определить решения в области кондиционирования (выбор подходящего холодильного аппарата).

Полученные значения являются ориентировочными и призваны лишь облегчить вам выбор подходящего решения в области кондиционирования для вашего конкретного случая применения.

Решение в области кондиционирования, выбранное на основании полученных значений общей мощности потерь, должно быть тщательно проверено производителем комбинации коммутационных устройств (в дальнейшем называемой электрошкафом), к примеру, посредством соответствующих процедур и опытов, на предмет пригодности в рассматриваемом случае.

Особое значение для подбора параметров имеет правильность предоставленных производителем компонентов и/или сохраненных пользователем в EPLAN данных изделий, касающихся мощности потерь отдельных устройств, а также заданные пользователем рамочные условия для проектирования, которые могут привести к значительным отклонениям результатов для общей мощности потерь.

По этой причине EPLAN Software & Service не гарантирует правильность полученной общей мощности потерь и выбранного на ее основе решения в области кондиционирования для конкретного случая применения, а также не несет ответственности за ущерб, возникший в результате игнорирования данного замечания.

!!! note "Замечание:"

    Использование online-версии Rittal Therm возможно только для зарегистрированных пользователей. При регистрации укажите следующие данные:

    Пользователь:

    ваш адрес электронной почты

    Пароль:

* 4 цифры (в начале стоит 1, за ней следуют любые 3 цифры из диапазона 0–9)
* 2 буквы из диапазона G–K
* дефис
* 1 буква из диапазона E–N
* Символы пишутся подряд, без пробелов, буквы заглавные.

    ***Пример:*** 1234GK-M

# Вкладка Типы линий

* Параметры > Настройки > Пользователь > Интерфейсы > Экспорт и импорт DXF / DWG. Выберите вкладку Типы линий.
* Страница > Экспортировать > DXF/DWG. Щелкните рядом с полем Схема на ++...++. Выберите вкладку Типы линий.

На этой вкладке определяется присвоение между специфическими для САПР и EPLAN типами линий в таблице.

!!! note "Замечание:"

    Для правильного отображения толщины линий необходимо установить отображение линий в AutoCAD.

Обзор основных элементов диалогового окна:

### EPLAN

Нажмите на стрелку в этом поле, чтобы отобразить все типы линий EPLAN и выбрать те, которые необходимо обработать.

### САПР

Здесь указывается соответствующий тип шрифта САD.

!!! example "Пример:"

    Выберите, например, тип линий EPLAN "Сплошная" и присвойте ему тип линий САD "CONTINUOUS".

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

# Диалоговое окно Монтажное отверстие <Тип>

Вы открыли проект. Навигатор пространства листов открыт, и одно пространство листов открыто. Пространство листа содержит монтажную плату и / или конструктивные функциональные элементы. Вставить > Монтажное отверстие > <Тип>.

В этом диалоговом окне можно выбирать или вводить параметры для всех доступных типов монтажного отверстия. Единицы измерения копируются из настроек проекта или в случае величин в дюймах переводятся в миллиметры.

Обзор основных элементов диалогового окна:

Отверстие, отверстие с резьбой

Элемент диалогового окна |  Значение
---|---
Диаметр:
Номинальный диаметр: |  Значение диаметра отверстия. В раскрывающемся списке перечислены все диаметры отверстий, использованные в проекте, включая последний.

Прямоугольник

Элемент диалогового окна |  Значение
---|---
Тип: |  В раскрывающемся списке на выбор предлагаются типы 'Угловой', 'Закругленный' и 'Со снятой фаской'.
Ширина: |  Ширина прямоугольника
Высота: |  Высота прямоугольника
Радиус / размер фаски: |  Это поле ввода доступно, если в поле Тип выбрана запись 'Закруглен.' или 'Со снятой фаской'. Здесь, в зависимости от типа прямоугольника, можно выбрать значение для радиуса закругленного прямоугольника или размера фаски.

Продольн. отверстие

Элемент диалогового окна |  Значение
---|---
Длина: |  Длина продольного отверстия. Введенное значение для длины должно быть больше, чем значение для высоты.
Высота: |  Высота продольного отверстия.

Шестиугольник, восьмиугольник

Элемент диалогового окна |  Значение
---|---
Длина ребра:  |  Длина ребра шестиугольника / восьмиугольника.

Определенный пользователем контур

Элемент диалогового окна |  Значение
---|---
Контур:  |  Имя файла контура, определенного пользователем *.fc1. С помощью кнопки ++...++ можно выбрать файл из каталога контуров, определенного в фирменных настройках.

Выбрать схему сверления

Элемент диалогового окна |  Значение
---|---
Список: |  В этом диалоговом окне выбирается схема сверления, определенная в базе данных изделий, для монтажного отверстия типа 'из схемы сверления'. При этом все необходимые отверстия можно вставить в функциональный элемент за один рабочий шаг. Этот список содержит имя и описание схемы сверления, определенной в базе данных изделий. Чтобы вставить схему сверления, выделите запись и щелкните по кнопке ++OK++. После размещения отдельные монтажные отверстия группируются в одну схему сверления. После этого элементы схемы сверления можно обрабатывать как при отдельном размещении.

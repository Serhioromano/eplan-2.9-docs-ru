# Вкладка Основные данные (Изменение стандарта)

Вы открыли проект. При нескольких открытых проектах: Вы выделили проект в навигаторе страниц. Сервисные программы > Изменение стандарта. Вы определили на вкладке Проект все настройки. [Далее]. Вы определили на вкладке Поворот / инверсия все настройки. [Далее].

На вкладке перечислены все основные данные, которые используются в проекте и которые можно заменить. К основным данным относятся формы, рамки, библиотеки определений функций и библиотеки символов.

Доступны следующие элементы диалогового окна:

### Основные данные проекта

Строка показывает присвоение основных данных исходной библиотеки основным данным целевой библиотеки.

* Имя: В этом столбце расположены все основные данные, используемые в проекте. Отображается имя файла основных данных.
* Тип: В этом столбце расположено описание типа основных данных. Типами основных данных, которые можно заменить в этом диалоговом окне, являются библиотеки символов и библиотеки определений функций, формы и рамки.
* Дата изменения: В данном столбце находится дата последнего изменения основных данных исходного проекта.
* Новое имя: Посредством этого столбца осуществляется замена основных данных. Щелкните в поле и выберите с помощью кнопки ++...++ основные данные для целевого проекта. Если поля пустые, основные данные копируются из исходного проекта в целевой проект.
* Присвоить отдельно: Этот флажок доступен только для библиотек символов. Если этот флажок установлен, символы библиотеки исходных и целевых символов можно по отдельности присвоить друг другу. Если флажок снят, присваиваться будет вся библиотека символов.

# Автоматически генерировать кабели

При автоматическом генерировании кабелей свободные кабельные соединения автоматически присваиваются имеющемуся подходящему кабелю. Если подходящий кабель не найден, EPLAN генерирует новое ОУ кабеля. Для соответствующего соединения автоматически размещается точка определения соединения, куда заносится ОУ кабеля.

После генерирования кабели автоматически нумеруются и проводится выбор кабелей. Будет ли эта операция проводиться только для повторно сгенерированных кабелей или для всех кабелей, зависит от соответствующих настроек. Таким образом, вы получите уже обозначенные и снабженные изделиями кабели.

Условия:

* В графическом редакторе или навигаторе вы выделили соединения или объекты, содержащие соединения.
* На соединениях для свойства Соединение: Принадлежность настроено значение "Кабель". (Настроить эту функцию можно или в диалоговом окне cвойств соединения или при помощи точки определения соединения.) Или хотя бы один вывод устройства источника подключенной функции имеет свойство Вывод кабеля (настраивается в логической схеме).
* Сгенерированы обновленные соединения (Данные проекта > Соединения > Обновить). В завершение обновите вид, выбрав команды Вид > Начертить заново.

1. Выберите пункты меню Данные проекта > Кабели > Автоматически генерировать кабели.
2. В диалоговом окне [Автоматически генерировать кабели](cablegui_d_kabelerzeugen.md) задайте настройки, в соответствии с которыми генерируются кабели. В групповых полях Генерация кабеля, Нумерация кабелей и Автоматический выбор кабеля выберите соответствующие схемы.
3. Установите флажок Только автом. сгенерированные или названные кабели, чтобы выбрать только автоматически сгенерированные кабели. В противном случае будут учтены и те кабели, которые были созданы вручную.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В схеме соединений будут сгенерированы линии определения кабелей и точки определения соединений. На графически первом соединении кабеля будет размещено только по одной линии определения кабеля и одной точке определения соединений. На всех других соединениях будут размещены точки определения соединений с одинаковым ОУ.

!!! info "Для сведения:"

    Если вследствие автоматического прогона неназванная линия определения кабеля получает имя, оно обозначается как 'назначенное автоматически' (Свойство Автоматически задать имя). При следующем автоматическом прогоне автоматически заданные имена удаляются и заменяются новым именем; имена, назначенные вручную, сохраняются.

!!! note "Замечание:"

    * При автоматическом генерировании кабелей учитываются только те соединения, у которых для свойства Соединение: Принадлежность установлено значение "Кабель" и отсутствует Полное ОУ. Это могут быть даже кабельные соединения, входящие в уже существующий кабель. В этом случае размещаются точки определения соединений, которые уже содержат правильное ОУ.
    * Экранирования не генерируются автоматически.
    * Для связей, в которых хотя бы одна цель состоит из графической или внешней функции, автоматическое генерирование кабеля невозможно.
    * Линии определения кабелей размещаются на отдельном уровне, а точки определения соединений - нет.

### Изменение автоматически сгенерированных кабелей на кабели, сгенерированные вручную

На линиях определения кабелей и точках определения соединений, начерченных автоматически, устанавливается свойство Сгенерировано автоматически. При каждом повторном генерировании кабеля обозначенные таким образом объекты будут удаляться в первую очередь. Далее будут повторно сгенерированы соответствующие соединения и начерчены новые линии определения кабелей и точки определения соединений.

Кабели и точки определения соединений, сгенерированные автоматически, можно преобразовать на сгенерированные вручную.

1. Выделите кабели в графическом редакторе или выделите страницы или проект в навигаторе страниц.
2. Выберите пункты меню Данные проекта > Кабели > Автоматически > Ручной.

!!! info "Для сведения:"

    На выделенных кабелях свойство Сгенерировано автоматически снимается. Теперь соответствующие кабели будут рассматриваться как кабели, сгенерированные вручную, и не будут изменяться при повторных автоматических обработках.

!!! tip "Совет:"

    Для того, чтобы изменить на сгенерированные вручную ***отдельные*** кабели, откройте диалоговое окно Свойства ++...++ (напр., двойным щелчком по кабелю). На вкладке Кабели снимите флажок свойства Сгенерировано автоматически.

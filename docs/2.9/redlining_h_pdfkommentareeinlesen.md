# Импортировать PDF-комментарии в проект EPLAN

Проект EPLAN, который был экспортирован в PDF-файл и для которого в программе Acrobat Reader 8.0 можно просматривать PDF-комментарии, можно импортировать в проект EPLAN и в системе EPLAN прочитать PDF-комментарии. При импортировании соответствующий проект EPLAN должен находиться в каталоге, в котором он был во время вывода PDF.

Условия:

* Вы являетесь [исполнителем](redlining_k_projektkommentator.md).
* [Вывести страницы в PDF-файл](pdfexportgui_h_seitenexportieren.md)
* [Отправить файл PDF на проверку](redlining_h_permailversenden.md) или [освободить файл PDF для комментирования](redlining_h_freigabe.md)
* От нормоконтролера Вы получили PDF-файл, содержащий PDF-комментарии.
* [Объединить PDF-комментарии PDF-файлов](redlining_h_zusammenfuehren.md)

1. Проект > Открыть
2. Выберите проект EPLAN, который был экспортирован в PDF-файл для комментирования, и нажмите кнопку ++Открыть++.
3. Выберите пункты меню Страница > Импортировать > PDF-комментарии.
4. В диалоговом окне выберите каталог, в котором находится соответствующий PDF-файл с PDF-комментариями, и нажмите кнопку ++Открыть++.

!!! info "Для сведения:"

    PDF-комментарии импортируются в проект EPLAN.

!!! info "Для сведения:"

    Теперь можно [считатьимпортированные в EPLAN PDF-комментарии](redlining_h_projektkommentarebearbeiten.md).

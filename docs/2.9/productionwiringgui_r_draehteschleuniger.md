# Вкладка Провода (Schleuniger)

* Параметры > Настройки > Фирма > Сборка проводов > Schleuniger > Провода
* Сервисные программы > Данные изготовления > Сборка проводов > Schleuniger. Щелкните по кнопке ++...++ рядом с полем Схема. Выберите вкладку Провода.

В этой вкладке определяются основные данные по проводам для обработки проводов на машине 'Schleuniger'.

Обзор основных элементов диалогового окна:

Кнопка |  Значение
---|---

![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет выделенные строки.

### Цвет

Цвет провода.

### Поперечное сечение

Поперечное сечение провода.

### Номер изделия

Номер изделия для изделия проводов.

### Номер изделия Schleuniger

Для работы на машине 'Schleuniger' каждый провод должен иметь маркировку в виде заданного 'Schleuniger' номера изделия. В этом поле таблицы проводам в проекте присваиваются соответствующие номера изделий Schleuniger. Информацию о необходимых номерах изделий можно получить у производителя.

### Интервал стороны 1 / интервал стороны 2

Интервал, через который наносится маркировка от обоих концов провода.

### Интервал непрерывного текста

Интервал между отдельными текстовыми блоками.

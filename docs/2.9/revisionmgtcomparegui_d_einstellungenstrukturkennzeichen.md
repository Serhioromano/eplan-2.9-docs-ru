# Диалоговое окно Настройки структурного идентификатора

Вы открыли проект. Вы создали исходный проект. Сервисные программы > Управление ревизиями > Сравнение свойств проектов > Сравнить проекты. В диалоговом окне Сравнить свойства проектов рядом с полем Настройка структурного идентификатора щелкните по ++...++.

В данном диалоговом окне установите, какой структурный идентификатор в исходном проекте заменен на структурный идентификатор в текущем проекте, и сохраните эти настройки в собственной схеме. При сравнении свойств проектов введенные структурные идентификаторы больше не считаются различными.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Таблица Проект/исходный проект

В панели инструментов, расположенной над таблицей, имеются следующие кнопки:

Кнопка |  Значение
---|---
![](images/all_new_as.png){: .ui-icon } (Создать) |  Открывает диалоговое окно Выбор критериев для выбора блока идентификаторов. Позволяет задать структурный идентификатор в таблице на строку.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет текущую выделенную строку(и). Возможен многократный выбор.

* Блок идентификаторов: Здесь отображается блок идентификаторов. Щелкните в ячейке [...], чтобы открыть диалоговое окно Выбор критериев и, если нужно, выберите другой блок идентификаторов.
* Обозначение (проект): Укажите, какой структурный идентификатор переименован в текущем проекте. Щелкните в ячейке [...], чтобы в следующем диалоговом окне выбрать структурный идентификатор из текущего проекта.
* Обозначение (исходный проект): Укажите, какое обозначение имел переименованный структурный идентификатор в исходном проекте. Щелкните в ячейке [...], чтобы в следующем диалоговом окне выбрать структурный идентификатор из исходного проекта.
* Статус: При вводе переименованного структурного идентификатора проверяется, присвоен ли структурному идентификатору из текущего проекта структурный идентификатор из исходного проекта. Если это так, отображается зеленая пиктограмма ![](images/all_statusok_as.png){: .ui-icon }. В строках, где нет структурного идентификатора, отображается желтая пиктограмма ![](images/all_statusnotok_as.png){: .ui-icon }.

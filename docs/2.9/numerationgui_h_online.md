# Автоматическая нумерация новых устройств

Новые устройства/функции, которые вы добавляете на страницы схемы соединений через Вставка > Символ или создаете в навигаторе без размещения, могут автоматически нумероваться уже в момент вставки или создания. При этом для каждого нового устройства/каждой функции в зависимости от выбранного формата нумерации ***online*** определяется ОУ, которое автоматически присваивается устройству. Если формат нумерации в режиме подключения содержит счетчик, счетчик ОУ нумеруется по возрастающей.

!!! note "Замечание:"

    Нумеруются группы устройств, которые вы выбрали в формате нумерации в режиме подключения.

Условия:

* Вы открыли проект.
* Вы открыли страницу схемы соединений.

1. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация (online)
2. Выберите в групповом поле При вставке символов опцию Нумерация.
3. Если вы хотите создать или обработать формат нумерации в режиме подключения, щелкните рядом с полем Формат нумерации [...].

!!! info "Для сведения:"

    Открывается диалоговое окно Форматы нумерации. Здесь вы можете создать [формат нумерации в режиме подключения](numerationgui_h_onlineformat.md). Щелкните по кнопке ++ОК++.

4. Если вы хотите использовать уже существующий формат нумерации в режиме подключения, выберите в групповом поле При вставке символов из раскрывающегося списка Формат нумерации формат нумерации в режиме подключения.
5. Щелкните по кнопке ++ОК++.

### Вставлять и нумеровать устройства/функции через символ

1. Выберите пункт меню Вставить > Символ.
2. Выберите символ в диалоговом окне Выбор символа и нажмите ++OK++.

!!! info "Для сведения:"

    Символ появится рядом с курсором.

3. Переместите курсор в нужное место на странице схемы.
4. Для размещения символа щелкните по странице.

!!! info "Для сведения:"

    Диалоговое окно Свойства (усл. обозначение): ++...++.

!!! info "Для сведения:"

    В поле Видимое ОУ будет предложено видимое обозначение для нового устройства/новой функции в зависимости от выбранного формата нумерации в режиме подключения.

5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Символ будет размещен, и ОУ отображено на схеме соединений.

### Создавать и нумеровать устройства/функции без размещения

1. Выберите пункты меню Данные проекта > Устройство > Навигатор.
2. Выберите вкладку Дерево.
3. Выделите слой проекта или структурный идентификатор.
4. Выберите во всплывающем меню > "Создать".
5. Отметьте в диалоговом окне Определения функций в поле Выбор определение функции и щелкните ++OK++.

!!! info "Для сведения:"

    Диалоговое окно Свойства (усл. обозначение): ++...++.

!!! info "Для сведения:"

    Если вы уже выделили слой проекта, в поле Видимое ОУ будет предложено видимое обозначение для нового устройства/новой функции без обозначения структуры в зависимости от выбранного формата нумерации в режиме подключения.

!!! info "Для сведения:"

    Если вы уже выделили обозначение структуры, в поле Видимое ОУ будет предложено видимое обозначение для нового устройства/новой функции с обозначением структуры в зависимости от выбранного формата нумерации в режиме подключения.

6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Устройство/функция будут созданы в навигаторе.

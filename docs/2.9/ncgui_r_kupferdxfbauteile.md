# Вкладка Функциональные элементы (Экспорт DXF, медь)

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт DXF, медь. Выберите вкладку Функциональные элементы.
* Сервисные программы > Данные изготовления > Механическая обработка > DXF, медь. Рядом с полем Машина нажмите ++...++. Выберите вкладку Функциональные элементы.

На этой вкладке можно настроить область видимости, слои и цвета контуров функциональных элементов и экспортированной информации для сгибания.

Обзор основных элементов диалогового окна:

Групповое поле 'Формат контура функц. элемента'

### Генерировать контур функц. элемента

Контуры трехмерных функциональных элементов показывают их пространственное ограничение дополнительно к основным контурам и путям выдавливания. Если флажок установлен, генерируются контуры функциональных элементов в файле DXF. Их можно вывести в файле DXF на отдельных слоях и присвоить им цвета.

Если флажок не установлен, контуры функциональных элементов не генерируются. Это позволяет сократить количество данных.

### Слой

Стандартное имя слоя, на котором установлены контуры функциональных элементов в файле DXF.

### Цвет

Стандартный цвет, который присваивается контурам функциональных элементов в файле DXF.

Групповое поле Формат для информации для сгибания

### Экспортировать линии изгиба

Если флажок установлен, линии изгибов генерируются из развертки в файле DXF.

### Слой

Стандартное имя слоя, на котором установлены линии изгиба в файле DXF.

### Цвет

Стандартный цвет, который присваивается линиям изгибов в файле DXF.

### Экспортировать концы изгиба

Если флажок установлен, концы изгибов генерируются из развертки в файле DXF.

### Слой

Стандартное имя слоя, на котором установлены концы изгибов в файле DXF.

### Цвет

Стандартный цвет, который присваивается концам изгибов в файле DXF.

### Экспортировать примечания для сгибания

Если флажок установлен, примечания для сгибания генерируются из развертки как текст в файле DXF.

### Слой

Стандартное имя слоя, на котором установлены примечания для сгибания в файле DXF.

### Цвет

Стандартный цвет, который присваивается примечаниям для сгибания в файле DXF.

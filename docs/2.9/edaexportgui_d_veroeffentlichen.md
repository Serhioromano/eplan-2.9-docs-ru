# Диалоговое окно Опубликовать

Вы открыли проект. Выберите пункты меню Проект > Опубликовать.

В этом диалоговом окне устанавливаются настройки для экспорта проекта в формат EPDZ. Формат EPDZ можно прочитать при помощи приложения ***EPLAN View*** через Dropbox на мобильных устройствах (например, iPad). Файл EPDZ содержит свойства проекта, свойства страницы и свойства пространства листа, а следовательно, полную структуру проекта. Кроме того, файл EPDZ содержит для каждой страницы проекта файл SVG с 2D векторной графикой, а для каждого пространства листа — файл E3D с трехмерными графическими данными.

!!! note "Замечание:"

    Экспорт проекта в файл EPDZ [для использования в ***EPLAN Smart Wiring***](productionwiringgui_h_smartwiringdatenexportieren.md) можно найти в пунктах меню Сервисные программы > Данные изготовления > EPLAN Smart Wiring.

Обзор основных элементов диалогового окна:

### Источник (страница / трехмерная модель)

В данном поле выводится имя экспортируемого проекта. Обработка записи в данном поле невозможна.

### Схема

Выберите из раскрывающегося списка схему, которая содержит настройки для экспорта EPDZ.

С помощью кнопки ++...++ можно открыть диалоговое окно Настройки: Опубликовать. В этом диалоговом окне указываются настройки для экспорта файлов EPDZ, после чего настройки сохраняются в виде схемы.

### Файл EPDZ

Имя файла, в который необходимо экспортировать открытый проект. По умолчанию записывается имя проекта с расширением имени файла "epdz".

С помощью кнопки ++...++ также можно свободно задавать имя файла или выбирать имя уже имеющегося файла.

* * *

Групповое поле 'Средство экспорта'

### Носитель данных

Если выбрано это средство экспорта, файл EPDZ выводится на носителе данных. При публикации на носителе данных файл EPDZ можно сохранить на устройстве USB или записать непосредственно на CD / DVD.

В поле Каталог вывода можно ввести место хранения.

Кнопка ++...++ открывает диалоговое окно Выбрать каталог для выбора любого каталога для экспорта.

### Эл. почта

При выборе этого средства экспорта файл EPDZ отправляется по электронной почте. При публикации в виде сообщения электронной почты файл EPDZ можно отправить непосредственно на мобильное устройство.

!!! note "Замечание:"

    Мобильные устройства не читают разделенные файлы. Поэтому при пересылке файла EPDZ по электронной почте возможность разделить этот файл на несколько небольших частей отсутствует.

# Автоматическая генерация обзоров

В проектах с механическими компонентами в пространствах листа можно автоматически генерировать страницы с обзорами модели, 2D-отображениями отверстий и развертками шины. Основу этих автоматических отчетов составляют шаблоны отчетов с заданными настройками для генерируемых страниц. Эти шаблоны создаются и анализируются в диалоговом окне Обзоры модели – <Имя проекта> на вкладке Шаблоны.

С помощью макросов, которые создаются индивидуально и присваиваются шаблонам отчетов, можно установить, как будут отображаться обзоры. Через настройки фильтра в шаблонах создается конфигурация, позволяющая установить, из каких основных функциональных элементов (электрошкафов, монтажных плат и т. д.) в пространствах листа генерируются обзоры.

Автоматическую генерацию страниц с обзорами на основе шаблонов можно использовать для стандартизированного представления обзоров. Данный автоматизированный отчет позволяет сократить время, необходимое для создания страниц вручную. Это обеспечивает существенную экономию времени при разработке технической документации.

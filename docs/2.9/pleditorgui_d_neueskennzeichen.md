# Диалоговое окно Новый идентификатор

Вы открыли проект, который в структуре проекта использует блоки идентификаторов, например, "Установка" и "Место установки". Данные проекта > Управление структ. идентификаторами. Выберите вкладку Дерево, выделите структурный идентификатор, под которым вы хотите сгенерировать новый идентификатор, и щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).

В этом диалоговом окне можно сгенерировать новый структурный идентификатор внутри блока идентификаторов и определить его свойства.

Обзор основных элементов диалогового окна:

### Обозначение

Укажите здесь обозначение, под которым должно осуществляться управление структурным идентификатором.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

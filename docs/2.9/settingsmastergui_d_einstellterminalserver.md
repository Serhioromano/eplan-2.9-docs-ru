# Диалоговое окно Настройки: Общие (Рабочая станция, Представление)

Параметры > Настройки > Рабочая станция > Представление > Общее.

При работе с EPLAN через терминальный сервер или удаленный рабочий стол в данном диалоговом окне задаются настройки для терминального сервера. Это позволяет ускорить обновление отображения при изменениях интерфейса.

Обзор основных элементов диалогового окна:

### Оптимизация для сервера терминалов

Если этот флажок установлен, фон Bitmap больше не отображается при запуске EPLAN. Кроме того, ***не*** отображаются всплывающие подсказки к символам в графическом редакторе.

!!! note "Замечание:"

    Обратите внимание, что отображение и отключение фоновой графики активируются только после перезапуска программы.

Групповое поле Графический редактор

Если вы установили флажок Оптимизация для сервера терминалов, вы можете установить дополнительные настройки оптимизации в этом групповом поле.

### Кэширование графики

Если этот флажок установлен, данные Bitmap кэшируются. Представление графики при этом будет более плавным, но обновление вида замедлится.

Если флажок снят (по умолчанию), режим отрисовки терминального сервера оптимизируется, чтобы минимизировать объем передачи между терминалом и сервером. В этом случае, однако, в представлении возможно мигание.

### Скрыть указатель мыши

Если этот флажок установлен, указатель мыши скрывается в графическом редакторе.

# Справочная система

Справочная система EPLAN содержит концептуальную информацию, указания по действиям и описание диалоговых окон по большинству функций программы. Окно онлайн-справки откроется через интернет-соединение. Справочная система отображается в интернет-браузере, установленном на вашем компьютере по умолчанию. Справочную систему можно использовать в зависимости от контекста с помощью [F1] или вне зависимости от контекста.

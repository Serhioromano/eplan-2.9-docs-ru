# Произвести настройки для внешней обработки

Прежде чем экспортировать данные для внешней обработки, необходимо сначала ввести соответствующие настройки:

1. Откройте проект, для которого вы хотите ввести настройки для внешней обработки.
2. Выберите пункты меню Параметры > Настройки > Пользователь > Интерфейсы > Внешняя обработка.
3. В диалоговом окне Настройки: Внешняя обработка выберите из раскрывающего списка Схема схему или щелкните по ++...++, чтобы ввести в выведенном диалоговом окне настройки новой схемы, определенной пользователем.
4. Выберите требуемый тип файла, созданный при экспорте, из раскрывающегося списка "Тип файла".
5. Нажмите ++Параметры++, чтобы указать настройки экспорта для выбранного типа файла, и щелкните по ++OK++.
6. Укажите в поле "Целевой файл" диск, каталог и имя создаваемого целевого файла или щелкните по ++...++, чтобы выбрать диск и каталог в интерактивном режиме.
7. Если в раскрывающемся списке Тип файла выбрана запись "Файл Excel (*.xls, *.xlsx, *.xlsm, *.xlsb)", то в поле Шаблон можно ввести путь файла и имя файла шаблона или выбрать его, нажав на ++...++, в этом случае при экспорте этот шаблон определяет формат данных.
8. Выберите вкладку Данные.
9. Выберите в списке "Доступные элементы формата" объект, который следует учитывать при экспорте; многократный выбор здесь невозможен.
10. Переместите объект с помощью кнопки ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо) в списке "Выбранные элементы формата".
11. Введите таким же образом остальные элементы форматы, которые следует учитывать при экспорте.
12. Определите последовательность выбранных элементов формата во время использования кнопок панели инструментов.
13. Щелкните по кнопке ++ОК++.

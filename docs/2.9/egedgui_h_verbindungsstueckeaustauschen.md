# Заменить символы соединений

Символы соединения могут быть заменены при помощи "Перечерчивания".

Условия:

* Вы открыли проект.В Графическом редакторе открыта страница проекта.
* Страница уже содержит символы соединения.
* В настройках пользователя (Параметры > Настройки > Пользователь > Графическая обработка > Символы соединения) установлен флажок Символы соединений с диалоговым окном запроса.

1. Выберите пункт меню Вставить > Символ соединения.
2. Выберите из дополнительного меню символ соединения, который Вы хотите вставить.
3. Для тройников, перекрестных соединений и перемычек выберите из дополнительного меню желаемый специфический для цели вариант.
4. Если необходимо символ соединения начертить в виде точки соединения, установите флажок Начертить как точку.
5. Щелкните по кнопке ++OK++.
6. При необходимости нажмите клавишу ++Ctrl++ и переместите мышь, чтобы просмотреть имеющиеся варианты.
7. Щелкните левой кнопкой мыши по точке вставки уже имеющегося символа соединения, чтобы на этом месте разместить новый символ соединения.
8. Выполните аналогичные действия для всех других символов соединения.
9. Завершите операцию, выбрав пункт всплывающего меню Прервать операцию или нажав клавишу ++Esc++.

!!! info "Для сведения:"

    Исходный символ соединения, например угол, заменяется новым, например, тройником. Для точек разрыва это действует со следующим ограничением: Точка разрыва может перечертить угол (тройник, ...), но не наоборот.

!!! note "Замечание:"

    Как только Вы дважды щелкните левой кнопкой мыши по символу соединения, открывается специфическое диалоговое окно, в котором перечислены варианты выбранного символа соединения. Путем выбора варианта может быть изменен текущий вариант; но это не действует для углов.

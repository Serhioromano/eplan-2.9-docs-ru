# Определить и изменить поверхность установки

Для корректного размещения на других трехмерных объектах 3D-макросам требуются сведения о том, с использованием какой поверхности они размещаются и в каком направлении они должны быть при необходимости повернуты. Эти сведения задаются в 3D-макросах через определение ***поверхности установки***. Расположение поверхности установки определяет и глубину установки, с которой трехмерный объект будет расположен на монтажной поверхности. На поверхности установки автоматически генерируется девять точек захвата, которые могут быть использованы при размещении. Кроме того, можно определить еще одну точку захвата при помощи пунктов меню Обработать > Логика устройства > Точка захвата. Указанная точка захвата при размещении всегда проецируется на поверхность установки.

Условия:

* Вы открыли проект схемы соединений или проект макросов.
* Навигатор пространства листов открыт, и одно пространство листов открыто.
* Пространство листа содержит импортированные трехмерные тела, которые необходимо сохранить в виде макроса.

1. Выберите пункты меню Обработать > Логика устройства > Поверхность установки > Определить.
2. Переместите курсор в область трехмерного тела.

!!! info "Для сведения:"

    Расположенная под курсором поверхность будет автоматически выделена цветом.

3. После выявления требуемой поверхности щелкните по ней мышью.

!!! info "Для сведения:"

    Выбранная поверхность будет определена как поверхность установки. Это приведет к отображению прозрачной поверхности, выходящей за пределы выделенной области. В угловых и средних точках граничных ребер генерируются точки захвата.

!!! note "Замечание:"

    Поверхность, которую необходимо определить как поверхность установки, должна быть ограничена точками, лежащими геометрически в одном слое. В зависимости от качества импортированных трехмерных данных могут появляться поверхности, которые не соответствуют данному условию. Такие поверхности нельзя определить как поверхности установки.

!!! example "Пример:"

    На рисунке изображен 3D-макрос с поверхностью установки под цокольной плитой. Точки захвата расположены в углах и в средних точках ребер.

Функция Перевернуть поверхность установки переворачивает поверхность установки в горизонтальном направлении на 180°.

1. Выберите пункты меню Обработать > Логика устройства > Поверхность установки > Перевернуть.

!!! info "Для сведения:"

    Поверхность установки и трехмерное тело вращаются на 180°.

!!! example "Пример:"

    3D-макрос с перевернутой поверхностью установки

Функция Переместить поверхность установки изменяет положение поверхности установки параллельно оси Z. Таким образом можно изменить глубину установки 3D-макроса.

1. Выберите пункты меню Обработать > Логика устройства > Поверхность установки > Переместить.
2. Введите значение перемещения.

!!! info "Для сведения:"

    При вводе положительного числа или числа без предшествующего знака поверхность установки перемещается вверх.

!!! info "Для сведения:"

    При вводе отрицательного числа с предшествующим знаком '–' поверхность установки перемещается вниз.

!!! example "Пример:"

    На рисунке изображен 3D-макрос с поверхностью установки, передвинутой на 100 мм.

Функция Повернуть поверхность установки поворачивает поверхность установки вокруг центра в вертикальном направлении. Таким образом можно добиться повернутого размещения 3D-макроса.

1. Выберите пункты меню Обработать > Логика устройства > Поверхность установки > Поворачивать.
2. Введите угол отверстия.

!!! info "Для сведения:"

    При вводе положительного числа или числа без предшествующего знака трехмерное тело на поверхности установки будет повернуто в направлении против часовой стрелки.

!!! info "Для сведения:"

    При вводе отрицательного числа с предшествующим знаком '-' трехмерное тело на поверхности установки будет повернуто в направлении по часовой стрелке.

!!! example "Пример:"

    На рисунке изображен 3D-макрос с поверхностью установки, повернутой на -15°. Точки захвата на нижней поверхности остаются без изменений.

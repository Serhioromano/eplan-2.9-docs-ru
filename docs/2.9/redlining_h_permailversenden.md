# Отправить файл PDF на проверку

Чтобы иметь возможность комментировать проект EPLAN, выведенный в файле PDF в Acrobat Reader, его необходимо освободить с помощью программы Acrobat Professional. Для этого файл PDF можно отправить третьей стороне с помощью Acrobat через электронную почту. Файл PDF автоматически откроется для комментирования.

Условия:

* Вы являетесь [исполнителем](redlining_k_projektkommentator.md).
* Вы установили и открыли Acrobat Professional 7.0 / Acrobat Standard 7.0.
* [Вывести страницы файла PDF](pdfexportgui_h_seitenexportieren.md)

1. Файл > Отправить на проверку
2. Выберите пункт меню, с помощью которого файл отправляется на проверку через электронную почту.
3. Выберите в следующем диалоговом окне PDF-файл, содержащий страницы EPLAN и щелкните ++Далее++.
4. Введите в следующем диалоговом окне адрес эл. почты получателя и нажмите кнопку отправки сообщения.

!!! info "Для сведения:"

    Файл PDF отправлен. Получатель может сейчас [создать PDF-комментарии в файле PDF](redlining_h_pdfkommentare.md) в Acrobat Reader.

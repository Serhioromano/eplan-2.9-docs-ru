# Суммировать длину кабелей

Используемую в проекте длину кабелей можно автоматически суммировать для каждого типа кабелей; таким образом определяется количество необходимых кабельных барабанов. Аналогичные типы кабеля обобщаются в ***одно*** покупное кабельное изделие, а длина отдельных кабелей суммируется. При этом определяющим является не совпадение изделий, а совпадение физических свойств кабелей. Для этой цели анализируется [полный тип кабеля](cablegui_d_kabelmengenberechnung.md).

Суммирование длины кабелей может быть выполнено только для всего проекта.

Условия:

* На кабелях указана длина кабеля.
* В навигаторе страниц выделен проект.

1. Выберите пункты меню Данные проекта > Кабели > Суммировать длину кабелей.

!!! info "Для сведения:"

    В диалоговом окне Суммировать длину кабелей отображаются типы кабелей и соответствующая рассчитанная длина. Кроме того, отображается длина максимальн длинного отдельного кабеля. Дополнительно отображается изделие, к которому присоединен самый длинный отдельный кабель и его длина. Можно просмотреть и требуемое количество изделий.

2. При необходимости отображаемое значение можно изменить в поле Требуемое число. (Это может понадобиться при возникновении проблем сочетаемости: для трех отдельных отрезков длиной 57 м требуется три барабана по 100 м, а не два барабана.)
3. Если нужно выбрать новое изделие кабеля, выберите пункт всплывающего меню Выбор изделия.
4. Для удаления изделия и исключения его из последующего отчета выберите пункт всплывающего меню Удалить.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Старые изделия проекта кабеля будут удалены и внесены новые.

!!! info "Для сведения:"

    На отдельных кабелях количество изделий обнуляется.

!!! note "Замечание:"

    В диалоговом окне навигатора Спецификация - <Имя проекта> можно просмотреть, изменить и удалить покупные изделия. Однако изменения перезаписываются при следующем суммировании длины кабелей.

    Чтобы отобразить покупные изделия, в диалоговом окне Настройки: Изделие (вызывается из навигатора спецификаций через меню Всплывающее меню > Настройки) установите флажок Покупные кабельные изделия. Чтобы был отображен соответствующий столбец, в конфигурации столбцов для представления навигатора спецификаций в виде списка (вызывается через меню Всплывающее меню > Конфигурировать представление) должен быть установлен флажок Покупные изделия.

### Исключение кабелей из суммирования длины кабелей

Существует возможность исключить из суммирования длины кабелей предварительно изготовленные кабели.

1. Дважды щелкните по линии определения кабеля.
2. В диалоговом окне Свойства ++...++ выберите вкладку Кабель.
3. Если свойство Не учитывать кабель при суммировании длины кабелей еще не отображается в таблице, вставьте его туда посредством ![](images/all_new_as.png){: .ui-icon } (Создать).
4. Установите флажок свойства Не учитывать кабель при суммировании длины кабелей.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Если количество главного изделия кабеля равно нулю, отображается сообщение, так как это значение могло быть сгенерировано при суммировании длины кабелей.

# Экспорт трехмерных графических данных в формат VRML

Экспорт VRML записывает геометрию трехмерных тел пространства листа в файл в формате VRML. Экспортированные файлы в браузерах отображаются как трехмерные модели, которые можно поворачивать во всех направлениях и масштабировать.

При каждом экспорте VRML параллельно сгенерированному файлу VRML генерируется файл с расширением имени файла *.wri. Ко всем функциональным элементам, имеющим номер изделия или ОУ, прикрепляется номер в форме 'ID000001'. При этом число увеличивается по нарастающей от 1 до n.

Настройки для экспорта VRML задаются в меню Параметры > Настройки > Пользователь > Интерфейсы > Экспорт VRML.

Условие:

Вы открыли проект. Открыт навигатор пространства листа.

1. Выделите в навигаторе пространства листа пространство листа, содержание которого нужно экспортировать.
2. Выберите следующие пункты меню: Пространство листа > Экспортировать > VRML
3. В диалоговом окне Экспорт VRML в поле Имя файла введите имя генерируемого файла VRML или используйте кнопку ++...++, чтобы задать каталог и имя файла.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Трехмерные графические данные экспортируются в файл VRML и файл WRI с ранее заданным именем.

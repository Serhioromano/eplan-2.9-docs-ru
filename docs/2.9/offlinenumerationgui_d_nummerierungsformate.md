# Диалоговое окно Настройки: Нумеровать (автономно)

* Вы открыли проект. Страница > Навигатор. Выделены одна или несколько страниц. Данные проекта > Устройство > Нумеровать > [Настройки] > Нумерация.
* Вы открыли проект. Данные проекта > Устройство > Навигатор. Выделено одно или несколько устройств. Данные проекта > Устройство > Нумеровать > [Настройки] > Нумерация.
* Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Устройства > Нумеровать (автономно).

В этом диалоговом окне можно создать или обработать формат нумерации устройств в автономном режиме.

!!! note "Замечание:"

    Форматы нумерации, создаваемые или обрабатываемые в этом диалоговом окне, отличаются от форматов нумерации, используемых при нумерации в режиме подключения.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Доступные элементы формата

В этом поле представлены все доступные элементы формата. Эти элементы формата либо являются частью формата или указывают, каким образом должна выполняться нумерация. Чтобы использовать элемент формата для формата, его необходимо скопировать в поле Выбранные элементы формата, использовав кнопку ![](images/all_arrowright_as.png){: .ui-icon } (Переместить вправо).

Элемент формата |  Предварительный просмотр |  Значение
---|---|---
Страница |  ***S***  |  Здесь указано имя страницы, на которой находится устройство.
Столбец |  ***P***  |  Здесь указан номер столбца, в котором находится устройство.
Буквенное обозначение |  ***I***  |  Выводит буквенное обозначение устройства.
Непрерывный счетчик по строкам |  ***CL***  |  Устройства нумеруются с помощью сквозной и построчной нумерации. Продолжается сквозная нумерация по всем страницам.
Непрерывный счетчик по столбцам |  ***CC***  |  Устройства нумеруются с помощью сквозной нумерации и нумерации по столбцам. Продолжается сквозная нумерация по всем страницам.
Страница, непрерывный счетчик по строкам |  ***SL***  |  Устройства нумеруются с помощью сквозной и построчной нумерации. Счетчик начинается на каждой странице заново с начального значения.
Страница, непрерывный счетчик по столбцам |  ***Sc***  |  Устройства нумеруются с помощью сквозной нумерации и нумерации по столбцам. Счетчик начинается на каждой странице заново с начального значения.
Разделитель |  |  Разделяет элементы формата друг от друга.
Строка |  ***R***  |  Здесь указан номер строки, в которой находится устройство.

!!! note "Замечание:"

    Обычно нумерация происходит в соответствии с сортировкой страниц. 3D-размещения изделий, которые расположены только в пространстве листа и не имеют эквивалента на страницах в 2D (например: кабельный канал или монтажная плата), могут быть правильно пронумерованы только в том случае, если формат нумерации не учитывает сортировку страниц (например, "стандарт"). Поле формата нумерации Выбранные элементы формата, который применяется для нумерации таких устройств, не должно содержать элемент формата "Страница".

### Выбранные элементы формата

В этом поле выводятся все элементы формата, из которых состоит текущий формат. Текущий формат выводится в поле Схема.

### Предш. цифру не изменять

Эта функция относится к тем ОУ, которые имеют предшествующую цифру. Если этот флажок установлен, предшествующая цифра в ОУ не убирается. Если этот флажок установлен, предшествующая цифра в ОУ убирается. С помощью этой функциональности можно, например, определить, оставить ли номер схемы в ОУ для устройств Fluid.

### Счетчик не изменять

Если установлен этот флажок, оставляется весь счетчик с буквами и цифрами, но без подсчетчиков.

### Буквенные элементы счетчика не изменять

Если установлен этот флажок, оставляется буквенная часть счетчика.

!!! example "Пример:"

    ОУ устройства содержит счетчик***T3***. Таким образом, счетчик состоит из буквы и цифры. После нумерации остается буква***T***.

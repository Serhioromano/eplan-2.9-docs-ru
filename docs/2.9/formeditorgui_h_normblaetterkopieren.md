# Копировать рамки

Чтобы копировать рамку со всем ее содержимым, сохраните ее в другом месте / или под другим именем.

Условие:

Вы открыли проект.

1. Сервисные программы > Основные данные > Рамка > Копировать
2. В диалоговом окне Копировать рамку выберите дисковод и каталог копируемой рамки.
3. Выберите требуемый файл рамки. (Многократный выбор здесь ***невозможен***.)
4. Щелкните по кнопке ++Открыть++.
5. В диалоговом окне Создать рамку установите место хранения для копии рамки.
6. Введите в поле Имя файла информативное имя для рамки.
7. Щелкните по кнопке ++Сохранить++.

!!! info "Для сведения:"

    Рамка копируется, и копия открывается для обработки.

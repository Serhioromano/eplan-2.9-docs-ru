# Размещения отчета на странице вручную

Размещение отчетов вручную позволяет разместить данные проекта как [встроенные отчеты](formgeneratorgui_k_ausgabeformen.md) на имеющихся страницах проекта.

!!! note "Замечание:"

    Для встроенного отчета можно выбрать форму вывода "Размещение вручную", только если открыта нужная страница проекта в графическом редакторе.

Условия:

* Вы открыли проект.
* Требуемому типу отчета присвоена форма.
* Вы открыли страницу, где хотите разместить отчет, в графическом редакторе.

1. Сервисные программы > Отчеты > Генерировать
2. Выберите в диалоговом окне Отчеты — <Имя проекта> вкладку Отчеты и щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
3. Выберите из раскрывающегося списка Форма вывода диалогового окна Определить отчет форму вывода "Размещение вручную".
4. Выделите в поле Тип отчета тип отчета и щелкните по ++OK++.

!!! info "Для сведения:"

    Если открывается диалоговое окно Настройки — <Тип отчета>, дополнительно установите фильтрацию и сортировку и щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Отчет появится рядом с курсором мыши.

5. Установите курсор мыши на то место страницы, где должен быть размещен отчет.
6. Щелкните по левой кнопке мыши.

!!! info "Для сведения:"

    Отчет вставляется в страницу.

!!! info "Для сведения:"

    Отчет отображается в Сервисные программы > Отчеты > Генерировать в перечне отчетов как "Встроенный отчет".

# Разрешите комментирование в PDF-файле

Чтобы иметь возможность комментировать проект EPLAN, выведенный в файле PDF, его необходимо освободить с помощью программы Acrobat.

Условия:

* Вы являетесь [исполнителем](redlining_k_projektkommentator.md).
* Вы установили и открыли Acrobat Professional 7.0 / Acrobat Standard 7.0.
* [Вывести страницы файла PDF](pdfexportgui_h_seitenexportieren.md)

1. Файл > Открыть
2. Выберите в следующем диалоговом окне файл PDF, содержащий страницы EPLAN, и нажмите ++Открыть++.

!!! info "Для сведения:"

    Файл PDF откроется в Acrobat .

3. Выберите пункт меню Комментарий.
4. Выберите пункт меню, который активирует файл для комментирования в Adobe Reader.
5. Выберите в диалоговом окне целевой каталог и нажмите кнопку ++Сохранить++.

!!! info "Для сведения:"

    В Acrobat Reader сейчас можно [Создать PDF-комментарий в файле PDF](redlining_h_pdfkommentare.md).

# Диалоговое окно Новые объединенные провода с измененным значением свойства

Сгенерированы маршрутизируемые соединения. В навигаторе соединений выделены маршрутизируемые соединения. Выберите пункты меню Данные проекта > Соединения > Объединить. В диалоговом окне Объединить маршрутизируемые соединения щелкните в таблице в Схема объединенных проводов в ячейке по кнопке ++...++. В диалоговом окне Настройки: Объединить в поле Новые объединенные провода с измененным значением свойства щелкните по кнопке ++...++.

В этом диалоговом окне определите схемы с порядком свойств. Каждый раз в случае изменения значения одного из свойств генерируются новые объединенные провода с новым номером объединенных проводов. Построение и принцип работы данного диалогового окна соответствуют параметрам диалогового окна Сортировка (Объединить).

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Категория / свойство

В этом списке отображаются критерии, в соответствии с которыми генерируются объединенные провода в случае изменения значения одного из этих свойств. Для того чтобы задать правила для генерирования новой группы объединенных проводов, можно комбинировать друг с другом несколько критериев.

При нажатии кнопки ![](images/all_new_as.png){: .ui-icon } (Создать) отображается диалоговое окно Выбор критериев, в котором вы можете выбрать свойства из областей Источник, Цель и Соединение, при изменении которых необходимо выполнить сортировку соединений в новую группу объединенных проводов.

### По позиции вывода устройства

Для устройств, выводы которых расположены на двух или более уровнях выводов устройств друг над другом, в процессе производства разводка выполняется снизу вверх. Для этого нужно, чтобы провода на каждом уровне выводов устройств можно было объединить в одну группу объединенных проводов. Для выполнения этого требования можно воспользоваться следующими флажками в качестве настройки сортировки и объединения:

* Источник X, Цель X
* Источник Y, Цель Y
* Источник Z, Цель Z.

Эти настройки учитывают указания позиции X, позиции Y и позиции Z в глобальной и локальной схеме соединений для размещения изделия. Если поставлены эти флажки, сортировка проводов от выводов устройств с одинаковым указанием позиции и к ним выполняется в соответствии с общей группой объединенных проводов.

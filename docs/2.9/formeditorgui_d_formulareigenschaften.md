# Диалоговое окно Свойства формы — <Имя формы>

Вы открыли проект.

* Сервисные программы > Основные данные > Форма > Создать > [Сохранить].
* Сервисные программы > Основные данные > Форма > Открыть. Выберите пункт всплывающего меню Свойства на форме в диалоговом окне навигатора Страницы — <Имя проекта>.

В этом диалоговом окне установите свойства вновь созданной и открытой формы.

Обзор основных элементов диалогового окна:

### Тип формы

Выберите желаемый тип формы из раскрывающегося списка (например, "Схема подключения клемм", "Содержание", "Список заказов" и т. д.).
Это свойство определяет тип формы. Каждому типу формы присвоен определенный ид. В программе этот идентификатор используется для генерирования списка доступных свойств (текстов заполнителей), которые могут быть использованы при обработке формы. Таким образом, каждая форма имеет определенный список доступных полей. Кроме того, при помощи ид. типа формы генерируется список форм, которые в диалоговом окне Отчеты доступны для определенного документа. Если Вы к примеру хотите сгенерировать список обозначений устройств, то на выбор предоставляются только имеющиеся формы, которые отмечены как формы списков обозначений устройств.

### Имя свойства / Значение

В зависимости от выбранной категории выводятся возможные свойства; некоторым из них, например свойствам Идентификатор фирмы и Дата изменения, уже присвоено значение, которое невозможно изменить. Для других свойств, например для свойств Описание и Версия, невозможно присвоить значение или же изменить уже имеющееся значение.

!!! note "Замечание:"

    Доступны ***все*** свойства формы, однако их следует выбирать в зависимости от свойства Работа с формой. Так, например, свойства динамических форм (такие как свойство Динамически подобрать высоту строчки) являются нецелесообразными для статических форм.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

# Диалоговое окно Синхронизировать данные проекта — <Имя проекта>

* Вы открыли проект.
* Вы выделили проект в навигаторе страниц. Проект > Организовать > Синхронизировать.

В этом диалоговом окне задается, какие данные проекта должны быть обновлены и пополнены в целевом проекте.

Проект включает в себя не только страницы и основные данные, но и настройки, шаблоны отчетов и другие данные, такие как слои, определенные пользователем свойства, порядок свойств и т. д. С помощью этого диалогового окна синхронизации можно избирательно переносить такие настройки и данные проекта из исходного проекта. При этом данные копируются в целевой проект.

Обзор основных элементов диалогового окна:

### Целевой проект

В этом списке отображаются данные целевого проекта.

### Тип

С помощью этого раскрывающегося списка диалоговое окно синхронизации данных проекта можно настроить так, чтобы здесь отображались только данные определенного типа.

Список для ***целевого проекта*** содержит такие записи:

* Статус: Сведения о статусе, например "Более новый" или "Идентичный"
* Имя: имя файла элемента данных
* Тип: тип данных проекта (слой, свойство, порядок свойств и т. д.)
* Дата изменения: дата последнего изменения элемента данных

### Исходный проект

Щелкните по ++...++, чтобы в последующем диалоговом окне выбрать проект, который будет использоваться в качестве исходного проекта для различных данных проекта. При следующем открытии диалогового окна синхронизации предварительно устанавливается последний выбранный исходный проект.

Список данных проекта для ***исходного проекта*** содержит такие записи:

* Статус: сведения о статусе, например "Более старый" или "Не совпадает"
* Имя: имя файла элемента данных
* Тип: тип данных проекта (слой, свойство, порядок свойств и т. д.)
* Дата изменения: дата последнего изменения элемента данных

Записи в двух списках для целевого и исходного проекта могут быть отсортированы по столбцам в алфавитно-цифровом виде. Щелкните, например, по столбцу Тип, чтобы все записи были показаны отсортированными по типу данных проекта.

### Кнопка со стрелкой

Используйте кнопку со стрелкой для копирования отдельных данных проекта в целевой проект. Возможен многократный выбор.

### Фильтр

С помощью этого раскрывающегося списка диалоговое окно синхронизации данных проекта можно настроить так, чтобы здесь отображались только определенные данные. Выбранный фильтр активируется автоматически. Запись "-Не активировано-" деактивирует фильтр и приводит к неотфильтрованному отображению данных.

* Отобразить различия: Выберите этот фильтр, чтобы отображать только те данные проекта, которые имеются как в исходном, так и в целевом проекте и имеют статус "Более старый", "Новый" или "Не совпадает".
* Отобразить более старые данные: выберите этот фильтр, чтобы в списке Целевой проект отображались только те данные проекта, которые старее, чем соответствующие данные исходного проекта.

### [Обновить]

С помощью пунктов меню под этой кнопкой можно обновить и пополнить все данные проекта, относящиеся к целевому проекту.

Пункт меню |  Значение
---|---
Обновить проект |  Заменяет все устаревшие и отличающиеся друг от друга данные проекта в целевом проекте данными исходного проекта.
Дополнить проект |  Добавляет к проектным данным целевого проекта данные, которые ранее были доступны только в исходном проекте.

!!! note "Замечание:"

    Если во время синхронизации выбрать данные типа "Настройка" и перенести их в целевой проект, действие Отменить будет недоступно. Но если выбраны другие типы данных, настройку можно будет отменить.

# Диалоговое окно Настройки: Диалоговые окна свойств

Параметры > Настройки > Пользователь > Графическая обработка > Диалоговые окна свойств

Укажите в этом диалоговом окне, будет ли отображаться диалоговое окно свойств при вставке определенных объектов.

Обзор основных элементов диалогового окна:

### При вставке распределителей

Если установлен этот флажок, при вставке распределителей или соединителей выводится диалоговое окно Свойства ++...++. В нем можно сразу же ввести свойства устройства для распределителя или соединителя.

### При вставке экранирования

Если установлен этот флаг, при вставке экранирований выводится диалоговое окно Свойства ++...++. В нем можно сразу же ввести свойства устройства для экранирования.

### При вставке точек маршрутизации и сегментов маршрутизации (топология)

Если установлен этот флаг, при вставке точек маршрутизации и сегментов маршрутизации в топологии появляется диалоговое окно Свойства ++...++. В нем можно сразу же ввести свойства устройства для точек маршрутизации или сегментов маршрутизации.

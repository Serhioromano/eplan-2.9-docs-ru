# Обработать символы

Условие:

Вы открыли проект и как минимум одну библиотеку символов.

1. Выберите следующие пункты меню: Сервисные программы > Основные данные > Символ > Открыть
2. В диалоговом окне Выбор символа выберите символы, которые следует обработать.
3. Щелкните для более быстрого выбора при необходимости по ++...++ в поле Фильтр, и выберите критерии фильтра, или определите новые.
4. Или введите в поле Прямой ввод***Представления в виде списка*** символьную строку.

!!! info "Для сведения:"

    Курсор немедленно после ввода первого знака перемещается к символу, который начинается с этого знака, и выделяет его.

5. Выберите в ***Представлении в виде дерева*** пункт всплывающего меню С ориентацией на функцию, если иерархическое представление символов должно производиться не по библиотеке символов, а по функциональной категории (например "Катушки и контакты", "Сигнальные устройства" и т. д.).
6. Выберите в ***Представлении в виде списка*** пункт всплывающего меню Конфигурировать представление, если требуется изменить отображение и / или последовательность свойств символов в списке.
7. Щелкните по кнопке ++OK++.
8. Обработайте графику символа, т. е. добавьте выводы устройства и дополните графической и функциональной информацией.

!!! info "Для сведения:"

    Каждый открытый символ отображается в отдельном графическом редакторе, причем всегда учитывается первый имеющийся вариант символа. Обмен данными между несколькими открытыми символами может при этом выполняться через буфер обмена.

!!! info "Для сведения:"

    Предварительные настройки для свойств линий (тип, толщина, цвет...), а также специфичные для отображения свойства обозначения вывода устройства и описания вывода устройства берутся из соответствующих слоев (Данные проекта > Управление слоями > Графика символа / Данные проекта > Управление слоями > Размещение свойства).

!!! note "Замечание:"

    * Используемая сетка запоминается в свойствах библиотеки символов и, следовательно, действительна для всех символов данной библиотеки символов.
    * Точка вставки символа может быть только сдвинута, но не удалена, так как она обозначает место, на котором символ будет прикреплен к курсору при размещении. Кроме того, точка вставки является определяющей для позиции символа и, тем самым, также для перекрестной ссылки.

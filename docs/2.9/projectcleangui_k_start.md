# Сжатие проекта

В ходе проектирования часто генерируются данные проекта, которые в конце проектирования больше в проекте не используются. Вы можете удалить эти данные из проекта сжатием проекта.

* Вы можете уменьшить размер проекта, удалив не используемые данные проекта.
* Вы можете задать, какие данные проекта должны быть удалены.
* Данные проекта сжимаются для всего проекта. Вы можете определить фильтр для исключения сжатия данных проекта или для сжатия только выбранных уровней иерархии.
* Результаты сжатия представляются в виде системного сообщения.

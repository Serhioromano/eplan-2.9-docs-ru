# Вкладка Соединения сети

Вы открыли проект.

* В графическом редакторе выделена точка определения сети. Обработать > Свойства. В диалоговом окне Свойства ++...++ выберите вкладку Соединения сети.
* Вставить > Точка определения сети. В диалоговом окне Свойства ++...++ выберите вкладку Соединения сети.

Укажите на этой вкладке соединения сети. Сеть состоит из выводов устройства, непосредственно связанных друг с другом, и соединениями между этими выводами.

Обзор основных элементов диалогового окна:

Таблица Выводы устройств

В данной таблице отображаются все выводы устройства сети со следующими данными:

* Вывод устройства: вывод устройства, который является источником соединения
* Вывод устройства символов: уникальный номер вывода устройства символа. Выводы устройства различаются по данным номерам, если не заданы обозначения вывода устройства.
* Идентификатор сетевых соединений: однозначный идентификатор вывода устройства
* Число целей: Число целей каждого вывода устройства.

Если в настройках проекта (Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Общее) установлен флажок Управление выводами мостовых перемычек в точке определения сети, то здесь также отображаются ***выводы устройства символов*** с типом вывода устройства "Мостовая перемычка". Дополнительные ***Выводы устройства функции*** для мостовых перемычек не отображаются.

Если в настройках проекта установлен флажок Управление всеми соединениями сети в точке определения сети, то для напрямую связанных сетевых функций, цели не определяются автоматически. В этом случае соединения не генерируются автоматически, но должны задаваться вручную.

!!! note "Замечание:"

    * Сетевые соединения невозможно использовать вместе с оптимизацией перемычек распределенных клемм. Обе функции генерируют различные соединения, несовместимые друг с другом.
    * Если между сетевыми функциями находится символ соединения с определением цели (например, [Тройник](egedgui_d_tstueck.md)), то в каждом случае соединения определяются автоматически.

### Всплывающее меню

Пункт меню |  Значение
---|---
Создать шлейф |  Позволяет создавать новые шлейфы. После этого каждый вывод устройства, по которому вы щелкните, будет присоединен к шлейфу.

Свойства |  Позволяет обработать идентификатор вывода устройства (Свойство Идентификатор сетевых соединений) для выделенного в данный момент вывода устройства.

Таблица Соединения

В этой таблице шлейфы отображаются как список выводов устройства. При этом по умолчанию отображается вывод устройства и идентификатор сетевых соединений. Дополнительно можно отобразить свойства соединений. Каждый вывод устройства в шлейфе имеет соединение с выводом устройства, стоящим выше или ниже в списке. Это означает, что выводы устройства, стоящие в середине шлейфа имеют каждый по два соединения, а выводы устройства в начале и в конце шлейфа имеют по одному соединению.

!!! tip "Совет:"

    Последовательность источника и цели для соединения, сохраненного в точке определения сети, можно изменить, активируя в диалоговом окне свойств данного соединения свойство Поменять источник и цель.

Пункт меню |  Значение
---|---
Остальные соединения |  Открывает диалоговое окно Остальные соединения, которое отображает другие соединения сети, не сохраненные в точке определения сети.

# Создание и обработка опций

Опции сохраняются в проекте. Поэтому их следует создавать в том проекте, в котором они позже будут использоваться.

Условия:

* Вы открыли проект.
* Вы открыли навигатор опций проекта: Данные проекта > Опции > Навигатор

1. Выберите пункт всплывающего меню Создать > Опция.
2. Выберите в диалоговом окне Опция из раскрывающегося списка Имя ту опциональную группу, которой должна быть присвоена опция.
3. Кроме того, новую опциональную группу можно ввести вручную.

!!! info "Для сведения:"

    Относящееся к опциональной группе поле Описание становится доступными для записи.

!!! info "Для сведения:"

    Если оставить поле Опциональная группа пустым (или удалить имеющуюся запись), опция не присваивается никакой опциональной группе.

4. При необходимости введите в поле Описание текст описания для опциональной группы.
5. Введите имя опции в поле Имя.
6. При необходимости введите в поле Описание текст описания.

!!! info "Для сведения:"

    Под описанием понимается многоязычный текст, отображаемый по умолчанию в представлении структуры дерева рядом с именем (отделяется пробелом и заключается в круглые скобки).

7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Если было введено новое имя для опциональной группы, то она будет сгенерирована.

!!! info "Для сведения:"

    Новая опция включается автоматически. Уже имеющиеся опции в той же опциональной группе выключаются.

1. Выделите опцию в диалоговом окне Опции — <Имя проекта>.
2. Выберите пункты меню Обработать > Свойства.
3. Измените в диалоговом окне Опция записи в полях Имя и Описание.
4. Щелкните по кнопке ++OK++.

1. Выделите опцию в диалоговом окне Опции — <Имя проекта>.
2. Выберите пункты меню Обработать > Удалить.
3. Подтвердите запрос, нажав кнопку ++Да++.

!!! info "Для сведения:"

    Опция и все содержащиеся в ней фрагменты удаляются. Если опция была включена, содержащиеся в ней объекты сохраняются в проекте; объекты выключенной опции удаляются из проекта.

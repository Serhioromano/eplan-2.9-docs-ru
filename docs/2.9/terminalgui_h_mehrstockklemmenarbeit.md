# Определить многоуровневые клеммы

### Программно определить многоуровневые клеммы

Условия:

* Вы выделили клемму или определение клеммника в навигаторе клеммников, навигаторе устройств или в графическом редакторе.
* Через пункты меню Данные проекта > Клеммники > Обработать вы открыли диалоговое окно Обработать клеммник.

#### Генерировать отдельные многоуровневые клеммы

1. В диалоговом окне Обработать клеммник выделите необходимые клеммы.
2. Нажмите кнопку ![](images/terminalgui_createmulti_as.png){: .ui-icon } (Генерировать многоуровневую клемму).

!!! info "Для сведения:"

    ***Все выделенные*** клеммы объединяются в ***одну*** многоуровневую клемму.

#### Генерировать несколько многоуровневых клемм

1. В диалоговом окне Обработать клеммник выделите необходимые клеммы.
2. Нажмите кнопку ![](images/terminalgui_createmulti2_as.png){: .ui-icon } (Генерировать многоуровневые клеммы).
3. В следующем диалоговом окне укажите количество уровней.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выделенные клеммы объединяются в многоуровневые клеммы в соответствии с заданным количеством уровней.

!!! example "Пример:"

    В диалоговом окне Обработать клеммниквыделены восемь клемм.Вариант 1: Вы нажимаете кнопку. Генерируется многоуровневая клемма с восемью уровнями.Вариант 2: Вы нажимаете кнопку. В следующем диалоговом окне для количества уровней вы вводите значение3. Генерируются две многоуровневые клеммы, имеющие по три уровня. Две клеммы остаются и не объединяются.

Каждой сгенерированной многоуровневой клемме уровни присваиваются автоматически в убывающем порядке. Клемма с самым высоким уровнем становится главной клеммой, другие клеммы становятся второстепенными клеммами.

Если главная клемма в сгенерированной многоуровневой клемме перед объединением уже была главной клеммой с присвоенным ей изделием, шаблоны функции изделия перекрываются клеммами. Присвоение отдельных уровней складывается из шаблонов функции.

!!! tip "Совет:"

    * С помощью кнопки ![](images/terminalgui_separatemulti_as.png){: .ui-icon } (Разделить) можно удалить одну или несколько выделенных клемм из многоуровневой клеммы.
    * Выделенные клеммы также можно объединять в многоуровневую клемму в графическом редакторе или навигаторе клеммников. Для этого выберите пункты меню Данные проекта > Клеммники > Генерировать многоуровневую клемму. Первая клемма в графическом отображении (в порядке, установленном на схеме соединений или в навигаторе клеммников) становится главной клеммой.

### Вручную определить многоуровневые клеммы

Условие:

Вы выделили одну клемму в навигаторе клеммников, навигаторе устройств или в графическом редакторе.

1. Выберите пункты меню Обработать > Свойства.
2. Выберите в диалоговом окне Свойства ++...++ вкладку Клемма.
3. Задайте в поле Уровень желаемое значение.

!!! info "Для сведения:"

    Значение > 0 определяет клемму как многоуровневую; значение "0" определяет клемму как "простую" клемму. Уровни можно сортировать в восходящем или нисходящем порядке. При сортировке в восходящем порядке самому низкому уровню в корпусе клеммы присваивается номер уровня "1".

4. Введите значение в свойство Клемма: Положение устройства. Все клеммы, относящиеся к одной и той же многоуровневой клемме, также должны иметь одинаковое значение для положения устройства.
5. С помощью свойства Идентификатор сортировки (клемма / контакт штекера) задайте последовательность внутри многоуровневой клеммы.
6. Снимите флажок Главная клемма, если клемма должна рассматриваться как вспомогательная. (Главной клеммой может быть клемма либо с самым нижним, либо с самым верхним уровнем.)
7. Щелкните по кнопке ++OK++.
8. Повторите описанные выше шаги для других клемм, которые являются частью многоуровневой клеммы.

# Области обработки определить как администратора проекта

При использовании управления правами EPLAN области обработки может централизованно распределять администратор проекта.

!!! note "Замечание:"

    * Имена пользователей, используемые при разделении областей обработки, должны быть созданы в управлении правами заранее.
    * В EPLAN имена пользователей всегда пишутся прописными буквами. Поэтому имена пользователей, указанные в диалоговом окне Определить области обработки, преобразуются и сохраняются в верхнем регистре.

Условие:

Вы открыли проект.

1. Выберите пункты меню Проект > Области обработки > Определить как администратора проекта.

!!! info "Для сведения:"

    В диалоговом окне Области обработки определить как администратора проекта отображаются схемы, которые уже были созданы для разделения областей обработки.

2. Если схема отсутствует или вы хотите обработать схему, нажмите кнопку ![](images/all_edit_as.png){: .ui-icon } (обработать) над таблицей.
3. В диалоговом окне [Фильтр](modaldialogsdb_d_filternnach.md) создайте новую схему и задайте критерии для разделения областей обработки. Или обработайте критерии для имеющейся схемы.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Ваши настройки будут сохранены в выбранной схеме, а вы вернетесь в диалоговое окно Области обработки определить как администратора проекта.

!!! info "Для сведения:"

    В столбце Все обработчики для каждой области обработки можно указать одного или нескольких обработчиков.

5. Для этого щелкните мышкой в строке необходимой области обработки по ячейке таблицы Все обработчики, а затем по кнопке ++...++.

!!! info "Для сведения:"

    В диалоговом окне Выбрать обработчика отображаются пользователи, определенные в управлении правами.

6. В столбце Присвоить установите флажок для обработчиков, которым хотите присвоить соответствующую область обработки. Или снимите флажок с обработчиков, у которых необходимо забрать область обработки.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выбранные обработчики отображаются в столбце Все обработчики. Если одной области обработки присвоены несколько обработчиков, то они разделяются запятой.

8. Щелкните по кнопке ++OK++.

!!! tip "Совет:"

    * В столбец Все обработчики можно вводить имена пользователей вручную. Не забывайте о том, что вводимое имя пользователя должно существовать в управлении правами.
    * В столбце Все обработчики можно путем копирования и вставки переносить содержимое одной ячейки в другие.

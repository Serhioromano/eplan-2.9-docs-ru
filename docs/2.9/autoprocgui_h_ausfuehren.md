# Обработать несколько проектов автоматически

В управлении проектами предусмотрена возможность автоматического применения операций сразу к нескольким проектам.

Условия:

* В диалоговом окне Управление проектами определены каталоги и проекты.
* Создана схема, содержащая последовательность автоматических операций.

1. Проект > Управление
2. Для выбора сразу нескольких проектов удерживайте в нажатом положении клавишу ++Ctrl++ и отметьте нужные проекты в дереве управления проектами.
3. Выберите пункт всплывающего меню Автоматизированная обработка.
4. В диалоговом окне Выполнить: Автоматизированная обработка в поле Схема выберите схему с автоматизированными операциями.

!!! info "Для сведения:"

    Сохраненная в схеме последовательность действий отображается в списке Операции.

5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Проекты будут автоматически обработаны.

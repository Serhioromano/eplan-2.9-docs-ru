# Генерировать новые заголовки на страницах формы

При помощи свойства формы Создать заголовки для первых х свойств сортировки (ид. 13060) создайте ссылку на сортировку соответствующих отчетов. Т.е., в форме устанавливается, должны ли использоваться свойства сортировки для генерации новых заголовков.

Для этого свойства доступны следующие значения:

1. "0": Новые заголовки не генерируются.
2. "1": Первое свойство сортировки используется для нового заголовка.
3. "1": Первое и второе свойства сортировки используются для нового заголовка.
4. "1": Первое, второе и третье свойства сортировки используются для нового заголовка.
5. и т.д.

Условия:

* Вы открыли проект.
* В редакторе форм открыта та форма, свойства которой требуется обработать (Сервисные программы > Основные данные > Форма > Открыть > [Открыть]).
* Открыто диалоговое окно Свойства формы — <Имя формы> (Всплывающее меню > Свойства на форме в диалоговом окне навигатора Страницы — <Имя проекта>).

1. В диалоговом окне Свойства формы — <Имя формы> щелкните ![](images/all_new_as.png){: .ui-icon } (Создать) над списком Имя свойства / значение.
2. В диалоговом окне Выбор свойств выберите свойство Создать заголовки для первых х свойств сортировки.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Свойство копируется в список.

4. Укажите в столбце Значение количество свойств сортировки, которые надо учитывать.
5. При необходимости обработайте другие свойства формы.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Установленные свойства копируются для формы, и диалоговое окно Свойства формы — <Имя формы> закрывается.

!!! example "Пример:"

    Для вывода списка обозначений устройств в диалоговом окне Сортировкавыбрано Схема "Установка +место установки+ ОУ ". В форме для списка обозначений устройств определено свойство Создать заголовки для первых свойств сортировкисо значением "1".При генерировании страниц отчета для списка обозначений устройств для каждой новой установки генерируется новый заголовок, в котором указано имяустройства. Данные между заголовками далее сортируются по месту установки и ОУ.

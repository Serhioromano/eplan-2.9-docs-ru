# Определить шаблон и поворот / инверсию (изменение стандарта)

Используя шаблон, обозначения и названия можно подстроить под целевой стандарт. Настройка поворота / инверсии в сочетании с выбранными библиотеками символов задает расположение элементов схемы соединений на схеме.

Условия:

* Вы открыли проект.
* При нескольких открытых проектах: В навигаторе страниц выделен проект.
* Вы имеете шаблон, содержащий специфические для страны или стандарта настройки.
* Вы настроили целевой проект.

1. Сервисные программы > Изменение стандарта
2. Для определения шаблона целевого проекта на вкладке Поворот / инверсия с помощью кнопки ++...++ выберите шаблон проекта *.ept.
3. Из раскрывающегося списка Поворот / инверсия выберите способ вращения и инверсии.
4. Щелкните по ++Далее++.

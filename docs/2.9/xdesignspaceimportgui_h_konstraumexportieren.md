# Экспортировать пространство конструкции

Ниже описывается третий рабочий шаг при использовании ***EPLAN Design Space Exchange***.

* Экспорт расширенного за счет чертежа монтажных поверхностей и данных ЧУ пространства конструкции в программу 3D-CAD.

Настройки для экспорта пространств конструкции выполняются в Параметры > Настройки > Пользователь > Интерфейсы > Импорт и экспорт пространства конструкции.

Условие:

Вы открыли проект. Открыт навигатор пространства листа. В импортированном прежде пространстве конструкции создан чертеж монтажных поверхностей.

1. Выделите в навигаторе пространства листа пространство конструкции, которое нужно экспортировать.
2. Выберите Пространство листа > Пространство конструкции > Экспортировать.
3. Выберите в диалоговом окне Экспорт пространства конструкции схему с настройками для экспорта выбранного пространства конструкции или с помощью кнопки ++...++ откройте диалоговое окно "Настройки", чтобы изменить настройки.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Для каждого изделия, размещенного с помощью EPLAN Pro Panel, при экспорте данных экспортируется файл STEP.

!!! info "Для сведения:"

    Для каждого пространства листа при экспорте данных генерируется файл XML с данными всех объектов в пространстве листа (объекты, импортированные из системы CAD, объекты, добавленные с помощью EPLAN Pro Panel, монтажные отверстия, сгенерированные автоматически при размещении изделия или вставленные вручную).

5. Импортируйте сгенерированные файлы STEP с помощью [дополнительного модуля CAD](xdesignspaceimportgui_k_prinzip.md) обратно в трехмерную модель в программу 3D-CAD-

# Диалоговое окно Обработать маркеры ревизий

Вы создали ревизию и открыли проект ревизии (расширение имени файла *.ell). Выделен измененный объект. Всплывающее меню > Обработать маркеры ревизий.

В этом диалоговом окне укажите настройки для автоматически созданного текста маркировки ревизии или сделайте это изменение вручную.

Доступны следующие элементы диалогового окна:

### Формат текста маркировки

В этом поле отображается настроенный формат для автоматического текста маркировки. Он складывается из различных свойств, например, имени ревизии, обработчика, даты изменения. Через [...] откройте диалоговое окно Формат, в котором можно определять свойства и их форматирование.

### Текст маркировки

В этом поле отображается автоматически созданный текст маркировки, т. е. значения свойств, определенных выше. Вы можете вручную дополнить или переписать этот текст.

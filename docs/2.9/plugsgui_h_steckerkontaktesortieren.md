# Сортировать контакты штекеров

В диалоговом окне Обработать штекер вы можете изменить порядок штекерных контактов в пределах одного штекера.

Условие:

Вы выделили определение штекера или контакт штекера в графическом редакторе, в навигаторе штекеров или навигаторе устройств.

1. Выберите пункты меню Данные проекта > Штекеры > Обработать.
2. В таблице диалогового окна Обработать штекер щелкните по номеру строки требуемого контакта штекера, чтобы отметить строку.
3. Удерживая нажатой клавишу ++Ctrl++, отметьте все остальные контакты штекера, которые вы хотите передвинуть.
4. Переместите выделенные контакты с помощью кнопок с изображением стрелки в нужное положение в штекере

!!! tip "Совет:"

    С помощью кнопки ++Сортировать++ вы можете автоматически отсортировать все контакты штекера по различным критериям.

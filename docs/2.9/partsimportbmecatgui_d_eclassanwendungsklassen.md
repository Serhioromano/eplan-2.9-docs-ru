# Диалоговое окно Выбор класса применения

Сервисные программы > Изделие > Управление > [Дополнительно] > Импортировать. В поле Тип файла выбрана запись для версии eCl@ss. Щелкните кнопку ++...++ рядом с полем Тип файла, выберите вкладку Присвоение класса и щелкните ![](images/all_new_as.png){: .ui-icon } (Создать).

В этом диалоговом окне представляется полный проводной монтаж eCl@ss для сегментов eCl@ss 19 ("Информационная, коммуникационная и мультимедийная техники"), 23 ("Деталь установки, средство крепления, покрытие"), 27 ("Электротехника, автоматизированная техника и технологический контур") и 37 ("Трубопроводная техника").

Обзор основных элементов диалогового окна:

### Список

Список содержит все классы для соответствующих сегментов eCl@ss в структурированном дереве. Здесь можно произвольно выбрать сегменты, главные группы и т. д. для присвоения свойствам EPLAN.

!!! note "Замечание:"

    Характеристики стандарта eCl@ss, обозначенные как "Deprecated", для присвоения не предлагаются.

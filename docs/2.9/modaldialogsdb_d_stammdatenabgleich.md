# Диалоговое окно Синхронизация основных данных — основные данные системы

* Дополнительно к имеющейся версии установлена новая версия программы EPLAN, и эта программа запускается в первый раз.
* Установлен дополнительный модуль с основными данными, и программа запускается в первый раз.
* Сервисные программы > Основные данные > Синхронизировать основные данные системы.

В данном диалоговом окне указывается, должны ли обновляться основные данные в каталоге идентификатора клиента и каким образом такое обновление должно происходить. При первом запуске установленного обновления программы или установленного нового Add-ons данные копируются автоматически, если они еще не содержатся в целевом месте назначения. Если при этом требуется перезапись, то на экран выводится сообщение с приглашением решить, следует ли выполнить синхронизацию напрямую:

* Выбор значения [Определ. пользователем] приведет к считыванию основных данных и открытию диалогового окна Синхронизация основных данных — основные данные системы. При помощи данного диалогового окна можно целенаправленно скопировать отдельные основные данные источника EPLAN.
* При выборе [Да] выполняется ***автоматическая*** синхронизация основных данных системы. Более старые основные данные системы при этом перезаписываются, отсутствующие –добавляются.
* При выборе [Нет] работу можно продолжить без синхронизации.

Обзор основных элементов диалогового окна:

### Каталоги

Данный раскрывающийся список позволяет выбрать различные каталоги для синхронизации основных данных. Чтобы обеспечить эту возможность, различные настройки каталогов необходимо предварительно сохранить в виде схемы. Это может пригодиться при использовании различных каталогов основных данных для различных фирм / клиентов. Щелкните по кнопке ++...++, чтобы открыть диалоговое окно Настройки: Каталоги и создать новую схему для каталогов.

### Источник

Выберите из этого раскрывающегося источник для синхронизации. Список при этом содержит данные всех установленных версий программы, например, такие записи как "EPLAN Electric P8 Data" или "EPLAN Fluid Data".

### Основные данные системы

Список основных данных системы в фирменных каталогах содержит следующие записи:

* Статус: Сведения о статусе, например "Более новый" или "Идентичный"
* Имя: Имя файла объекта основных данных, без расширения имени файла
* Тип: Тип основных данных (библиотека символов, форма для титульной страницы, рамка и т. д.)
* Дата изменения: Дата последнего изменения объекта основных данных.

Записи сортируются по столбцам в алфавитно-числовом порядке, при этом сортировка по умолчанию производится по записи Имя. Щелкните столбец Тип, в результате все записи отображаются отсортированными по типу основных данных.

### Основные данные EPLAN

Список основных данных источника EPLAN содержит следующие записи.

* Статус: Сведения о статусе, например "Не сохранено" или "Идентичный"
* Имя: Имя файла объекта основных данных, без расширения имени файла
* Тип: Тип основных данных (библиотека символов, форма для титульной страницы, рамка и т. д.)
* Дата изменения: Дата последнего изменения объекта основных данных.

Записи сортируются по столбцам в алфавитно-числовом порядке, при этом сортировка по умолчанию производится по записи Имя. Щелкните столбец Тип, в результате все записи отображаются отсортированными по типу основных данных.

### Кнопка со стрелкой

Кнопка со стрелкой позволяет скопировать отдельные основные данные EPLAN в основные данные системы. Возможен многократный выбор.

### Фильтр

При помощи данного раскрывающегося списка диалоговое окно для синхронизации основных данных можно настроить так, чтобы здесь отображались только определенные данные. Выбранный фильтр активируется автоматически. Запись "-Не активировано-" деактивирует фильтр и приводит к неотфильтрованному отображению данных.

* Отобразить различия: Выберите этот фильтр, чтобы отобразить только те основные данные, которые содержатся как в основных данных EPLAN выбранного источника, так и в основных данных системы выбранного каталога, и которые имеют статус "Более старые" или "Более новые".
* Основные данные системы (прежние): Выберите этот фильтр, чтобы в списке Основные данные системы отобразить только более ранние основные данные системы по сравнению с соответствующими основными данными EPLAN.
* Основные данные EPLAN (прежние): Выберите этот фильтр, чтобы в списке Основные данные EPLAN отобразить только более ранние основные данные EPLAN по сравнению с соответствующими основными данными системы.
* Дополнения: Выберите этот фильтр, чтобы в списке Основные данные EPLAN отобразить только основные данные, которые пока отсутствуют в основных данных системы (статус "Только в основных данных EPLAN").
* Основные данные системы (прежние) и дополнения: Выберите этот фильтр, чтобы отобразить более ранние основные данные системы по сравнению с соответствующими основными данными EPLAN. Дополнительно будут отображены основные данные EPLAN, которые еще не были включены в основные данные системы (статус "Только в основных данных EPLAN").

### [Обновить]

Пункты меню, расположенные под данной кнопкой, позволяют выполнить глобальное обновление основных данных.

Пункт меню |  Значение
---|---
Основные данные системы (прежние) |  Заменяет все устаревшие основные данные системы более новыми основными данными EPLAN.
Дополнения |  Дополняет основные данные системы данными, которые до настоящего момента входили только в основные данные EPLAN.
Основные данные системы (прежние) и дополнения |  Заменяет все устаревшие основные данные системы более новыми основными данными EPLAN и дополняет данными, еще не включенными в основные данные системы.

!!! note "Замечание:"

    * Если при синхронизации новая версия объектов основных данных заменяется на более старую (идентификация в записи в колонке Дата изменения), выводится соответствующее указание, и вы можете прервать обновление.
    * При сравнении основных данных допускаются имена файлов с прописными и строчными буквами, но обрабатываются они одинаково. То есть имя файла ABC.XYZ идентично с abc.xyz.
    * Чтобы использовать обновленные основные данные системы в каком-либо конкретном проекте, требуется выполнить синхронизацию основных данных данного проекта с основными данными системы.

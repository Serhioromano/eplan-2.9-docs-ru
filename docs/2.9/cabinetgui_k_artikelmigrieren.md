# Перенос изделий электрошкафа

Функция Перенести изделие помогает переносить имеющиеся изделия из системы линейно присоединенных шкафов Rittal TS8 на систему линейно присоединенных шкафов Rittal VX25 для проектов на базе системы Rittal TS8.

!!! note "Замечание:"

    Перед использованием функции Перенести изделие сначала прочитайте Заявление об отказе от ответственности EPLAN.

При этом применяются следующие условия и ограничения:

* Пункты меню Перенести изделие и Корректировать интервал электрошкафов недоступны в отдельно поставляемом дополнительном модуле EPLAN Pro Panel.
* Перенос возможен только для системы Rittal TS8.
* Все изделия должны быть размещены с помощью 3D-макроса.
* Специальные изделия, которых нет в каталоге Rittal и EPLAN Data Portal, не переносятся.
* Размещения изделий с автоматической и ручной защитой не переносятся. Поэтому перед переносом необходимо снять всю защиту.
* Электрошкафы, созданные с помощью конфигуратора RICS, не переносятся, поскольку подлежат автоматической защите.
* Электрошкафы, которые созданы во внешних системах и интегрированы посредством импорта STEP, перенести невозможно.
* Необходимые изделия для замены VX25 должны содержаться в используемой базе данных изделий. Недостающие изделия для замены определяются и могут экспортироваться в файл CSV. Их можно получить через корзину EPLAN Data Portal. Рекомендуется получать недостающие изделия для замены только на EPLAN Data Portal.
* [Контрольный прогон 026104](msg/messages_p_026104.md) распознает переносимые изделия Rittal TS8, которые еще не были перенесены.
* [Контрольный прогон 026102](msg/messages_p_026102.md) распознает непереносимые изделия Rittal TS8. Здесь идет речь о деталях, снятых с производства.
* [Контрольный прогон 026103](msg/messages_p_026103.md) распознает размещения изделий, монтажная поверхность которых не существует в перенесенном электрошкафу. Эти объекты отображаются в навигаторе пространства листа на уровне иерархии "Размещ. неверно". Для корректировки неправильные размещения изделий необходимо разместить на имеющихся монтажных поверхностях вручную.
* Все размещения изделий копируются в электрошкаф на одноименную монтажную поверхность без геометрических изменений и с той же позицией X / Y по отношению к нулевой точке.

* Изделиям Rittal TS8, остающимся в программе в измененной форме (например, с продольными отверстиями вместо круглых), можно присвоить новый 3D-макрос с помощью пункта меню Обновить размещение изделия.
* Размещения изделий, для которых в целевом электрошкафу не найдена одноименная монтажная поверхность, отображаются в навигаторе пространства листа на уровне иерархии "Размещ. неверно". Для этих размещений изделий требуется повторное размещение вручную.

### Автоматический перенос

В зависимости от выделения в навигаторе пространства листа выполняется анализ проекта, полного содержания пространства листа или отдельного электрошкафа и перенос на уровень изделия. Следующие изделия можно переносить автоматически:

* Электрошкаф
* Цоколь
* Двери
* Стенка, например боковая стенка, разделительная пластина
* Кожух с дверью
* Кожух разъединителя TS
* Панель крыши
* Верхний модуль.

!!! note "Замечание:"

    С цоколями / верхними модулями связана такая особенность: перенос генерирует полностью закрытый цоколь с накладками.

### Перенос вручную

Размещения изделий, которые не переносятся автоматически, можно перенести вручную, например:

* Части оборудования линейно присоединенных шкафов
* Поворотная рама
* Шины
* Шасси
* Плата для последов. сборки
* Секционные двери
* Панель основания
* Запорная система
* Профиль адаптера
* Кабельный канал
* Шарнир.

### Необходимый контроль и дополнительная обработка после переноса

После переноса необходимо тщательно проверить его результаты. Выполните на перенесенном электрошкафу следующие обновления и проверки:

* Проверьте правильность позиции всех размещений изделий
* Повторно пронумеруйте ОУ
* Повторно пронумеруйте позицию легенды
* Обновите обзоры (обзоры модели, 2D-отображения отверстий, развертки шины)
* Обновите отчеты (спецификации, легенду электрошкафа)
* Проверьте и откорректируйте сеть соединенных сегментов
* Маршрутизируйте соединения.

### Корректировка интервала электрошкафов при построении в линию

Поскольку электрошкаф VX25 на 2 мм шире электрошкафа TS8, после переноса выстроенных в линию электрошкафов перенесенные электрошкафы VX25 пересекаются. Контрольный прогон P026011 квалифицирует это как наложение.

Для корректировки интервала в пределах построения в линию выберите пункты меню Обработать > Корректировать интервал электрошкафов.

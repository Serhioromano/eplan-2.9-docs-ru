# Вкладка Фильтр соединений

Вы открыли проект. Параметры > Настройки > Проекты > 'Имя проекта' > Маршрутизируемые соединения > Общее. Выберите вкладку Фильтр соединений.

В этой вкладке можно настроить фильтр для создания сетей соединенных сегментов. Этот фильтр используется для сети соединенных сегментов в пространстве листа и в топологии.

Определенные здесь критерии фильтра в ***пространстве листа*** затем предоставляются на выбор в проекте в каналах, сегментах маршрутизации и областях маршрутизации в свойстве Пространство листа: Фильтр соединений (ид. 36026). Фильтр соединений применяется при маршрутизации для указания того, какие соединения разрешено прокладывать по данной трассе маршрутизации (канал, ручной сегмент маршрутизации, область маршрутизации, соединительное отверстие для проводов). Фильтры соединений учитываются, только если не заполнено свойство Пространство листа: Трасса маршрутизации - по умолчанию (ид. 31094).

Определенные здесь критерии фильтра в ***топологии*** затем предоставляются на выбор в проекте на сегментах маршрутизации в свойстве Топология: Фильтр соединений (ид. 20247). Фильтр соединений применяется при маршрутизации для указания того, какие соединения разрешено прокладывать по данной трассе маршрутизации (сегменты маршрутизации и точки маршрутизации).

Обзор основных элементов диалогового окна:

### Выбранное свойство соединения

С помощью [...] откройте диалоговое окно Выбор критериев, в котором вы можете выбрать одно из предлагаемых здесь свойств соединения.

Для выбранного свойства соединения в таблице можно указать до 15 критериев.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

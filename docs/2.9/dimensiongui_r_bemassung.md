# Вкладка Указания размеров

Открыта страница в графическом редакторе. Выделено указание размеров, и выбран пункт меню Обработать > Свойства. В диалоговом окне Свойства выберите вкладку Указание размеров.

На этой вкладке обработайте свойства просмотра и, при необходимости, значения указаний размеров.

Обзор основных элементов диалогового окна:

Групповое поле "Отображение"

### Сегменты

В этом поле отображаются отдельные компоненты для следующих величин: ***Непрерывные указания размеров***, ***Инкрементальные указания размеров*** и ***Указания размеров исходной линии***. Для всех других указаний размеров это поле не отображается. Выделите сегменты, свойства которых вы желаете просмотреть и изменить. (Вы можете выделить несколько сегментов, удерживая нажатой кнопку ++Ctrl++ и щелкая по соответствующим сегментам.)

### Префикс

Префикс ставится перед числовой мерой. Выберите запись из раскрывающегося списка или введите значение вручную. Разрешены любые символы.

### Размерное число

По умолчанию это поле защищено от записи и служит для отображения автоматически рассчитываемой числовой меры. Снимите флажок Автоматич., если хотите изменить числовую меру вручную. Таким образом можно определять размеры для немасштабных длин.

!!! note "Замечание:"

    Если при введенных ***вручную*** числовых мерах изменяется свойство Единица измерения или свойство Точность, в зависимости от настройки введенное значение может быть утеряно. Если вы ввели с точностью до "0,000" размерное число "1,234 мм", то оно примет вид "0,001 м", если изменить единицу измерения на "м". При изменении единицы измерения обратно на "мм", вы получите размерное число "1,000 мм".

### Суффикс

Суффикс добавляется в конце числовой меры. Выберите запись из раскрывающегося списка или введите значение вручную. Разрешены любые символы.

* * *

Групповое поле "Размерное число"

### Интервал до линии с размерами

Введите здесь требуемый интервал размерного числа до линии с размерами. По умолчанию это значение копируется из настроек проекта.

### Центрировать размерное число

Если установлен этот флажок, то числовая мера отображается в центре указания размеров или соответствующего сегмента. Настройка также влияет на угол и выравнивание размерного числа; они представлены серым цветом на вкладке Формат, и их невозможно изменить при активированной настройке Центрир. размерное число.

!!! note "Замечание:"

    Если размерное число смещается вручную при помощи мыши, одновременно с этим на вкладке Указание размеров снимается флажок Центрир. размерное число и удаляется значение для поля Интервал до линии с размерами. Как только этот флажок будет снова установлен и снова будет указан первоначальный интервал, соответствующее размерное число перемещается обратно.

### Единица измерения

Выберите единицу указания размеров из раскрывающегося списка. По умолчанию это значение копируется из настроек проекта.

### Отображать единицы измерения

Если этот флажок установлен, эта единица выводится вместе с размерным числом на линии с размерами.

Если флажок не установлен, выводится только размерное число.

### Отобразить две единицы измерения

Если данный флажок установлен, то размерное число отображается как в метрических единицах измерения (мм, см и т. д.), так и в дюймах.

Размерное число со второй единицей измерения (не заданное в поле Единица измерения) отображается в квадратных скобках непосредственно после первого размерного числа с первой единицей измерения (например, 88 мм [3,46"]). Отображение размерного числа во второй единице измерения выполняется с той же точностью, что и первого размерного числа; отображается максимум два разряда после запятой.

Если флажок не установлен, то размерное число отображается только вместе с единицей измерения, указанной в поле Единица измерения.

!!! note "Замечание:"

    Для углового указания размеров отобразить две единицы измерения невозможно. Соответственно в таком случае флажок Отобразить две единицы измерения в диалоговом окне "Свойства" недоступен.

### Точность

Выберите из этого раскрывающегося требуемое число десятичных разрядов для расчета размерного числа. По умолчанию это значение копируется из настроек проекта.

### Подавить нуль

Если этот флажок установлен, нули в десятичных разрядах не выводятся.

### Отображение размерного числа как дробное значение

Если этот флажок установлен, то десятичные числа "0,5", "0,25", "0,125", "0,0625" или кратные им числа отображаются как дробные значения. Возможными дробными значениями также являются "1/2", "1/4", "1/8", "1/16" и кратные им числа. Другие десятичные числа не отображаются как дробные значения.

### Горизонтальное размерное число

Если этот флажок установлен, то размерное число (независимо от направления указания размера) всегда отображается горизонтально.

Если этот флажок не установлен, то размерное число отображается соответственно направлению указания размера.

### Размерное число по центру линии с размерами

Если этот флажок установлен, то размерное число размещается по центру линии с размерами, разрывая ее. Размерное число всегда отображается горизонтально и выравнивается по центру; в этом случае флажок Горизонтальное размерное число и Центрировать размерное число отображается неактивным. Настройка также влияет на угол и выравнивание размерного числа; эти значения представлены серым цветом на вкладке Формат, и их нельзя изменить.

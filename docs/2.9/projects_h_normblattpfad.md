# Определить рамку и нумерацию зон

Настройки для стандартной рамки и для нумерации зон, устройств и ПЛК переносятся при создании проекта из выбранного шаблона (шаблон проекта или базовый проект). Если вы не хотите сохранить настройки из шаблона, их можно адаптировать.

Условие:

Вы открыли проект.

### Установка стандартной рамки

1. Выберите Параметры > Настройки > Проекты > "Имя проекта" > Управление > Страницы.
2. В диалоговом окне Настройки: Страницы выберите запись "Поиск" из раскрывающегося списка Стандартная рамка.
3. В диалоговом окне Выбрать рамку перейдите в каталог Рамки\EPLAN.
4. Выделите файл.
5. Щелкните по кнопке ++Открыть++.

!!! note "Замечание:"

    Выбранная рамка никоим образом не влияет на нумерацию зон.

### Установка нумерации зон

1. Выберите Параметры > Настройки > Проекты > "Имя проекта" > Управление > Страницы.
2. В диалоговом окне Настройки: Страницы в групповом поле Нумерация зон выбран один из доступных параметров.
3. Чтобы использовать измененную настройку для нумерации зон по проекту, после закрытия этого диалогового окна необходимо задать ее вручную с помощью Сервисные программы > Обновить обозначения зон.

### Выбор формата для нумерации устройств

1. Выберите пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Устройствo > Нумерация (онлайн).
2. В диалоговом окне Настройки: Нумерация (online) в поле Формат нумерации выберите определенный формат.
3. Для настройки или создания формата щелкните по ++...++.

!!! info "Для сведения:"

    В диалоговом окне Форматы нумерации можно [создать формат нумерации](numerationgui_h_onlineformat.md).

4. Щелкните по кнопке ++OK++.

### Выбор формата для нумерации ПЛК

1. Выберите пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Устройство > ПЛК.
2. В диалоговом окне Настройки: ПЛК в поле Специфические для ПЛК настройки выберите схему из раскрывающегося списка.
3. Для адаптации или создания схемы щелкните мышкой по кнопке ++...++.

!!! info "Для сведения:"

    В диалоговом окне Настройки: Специфические для ПЛК можно задать специфические для ПЛК настройки нумерации.

4. Щелкните по кнопке ++OK++.

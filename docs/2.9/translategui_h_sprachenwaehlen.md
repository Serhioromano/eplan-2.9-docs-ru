# Определить языки перевода и языки отображения

Языки перевода и отображения позволяют определить, на какие языки переводится проект и какие языки видны.

Условие:

Вы открыли проект.

### Определение языков перевода

Языки перевода – это языки, на которые осуществляется перевод текстов. Все переведенные тексты сохраняются в проекте и могут быть удалены из проекта с помощью функции Убрать перевод из проекта.

1. Выберите следующие пункты меню: Параметры > Настройки > Проекты > "Имя проекта" > Перевод > Общее
2. В групповом поле Перевод щелкните рядом с полем Языки по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
3. Выберите из раскрывающегося списка все языки, на которые необходимо перевести проект.

### Определить исходный язык

Определить Посредством исходного языка также задается, на каком языке в диалоговых окнах отображются переводимые тексты проекта.

1. Выберите запись из раскрывающегося списка Исходный язык.

### Выбрать и позиционировать языки отображения

Языки отображения – это языки перевода, на которых отображаются тексты проекта. Позиция языков отображения определяет, в какой последовательности выводятся языки отображения в переведенных проектах.

1. В групповом поле Отображение щелкните рядом с полем Языки по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
2. Выберите из раскрывающегося списка все языки, на которых будут отображаться страницы проекта.

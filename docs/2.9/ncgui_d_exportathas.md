# Диалоговое окно Экспорт Rittal — Athex

Вы открыли проект. В пространстве листа или в навигаторе пространства листа выделены объекты, которые необходимо экспортировать в систему автоматического оснащения Athex. Сервисные программы > Данные изготовления > Автоматическое оснащение Rittal — Athex.

В этом диалоговом окне задаются настройки экспорта данных для управления системой автоматического оснащения Rittal — Athex для клеммников и последовательных принадлежностей клеммы.

Обзор основных элементов диалогового окна:

### Машина

В комплекте поставки содержится определение машины с названием "Стандарт". Если в настройках фирмы определены другие схемы, можно выбрать из раскрывающегося списка другие определения машины.

Кнопка ++...++ открывает диалоговое окно Настройки: Rittal — Athex, в котором можно изменить настройки схемы или определить новую схему.

### Каталог вывода

В этом поле выводится место хранения. Здесь по умолчанию выбран каталог, который указан в пользовательских настройках для экспорта Автоматического оснащения Rittal — Athex.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Целевой файл

Данное поле доступно только в том случае, если выбраны объекты, которые можно экспортировать (например несущая шина). Здесь можно ввести или выбрать с помощью кнопки ++...++ в диалоговом окне Сохранить как имя генерируемого XML-файла. При наличии нескольких файлов поле деактивируется и действуют условия создания названий файлов и подкаталогов, определенные в схеме.

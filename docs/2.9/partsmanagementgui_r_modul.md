# Вкладка Модуль

Сервисные программы > Изделие > Управление. В дереве или списке выделено одно изделие модуля.

В этой вкладке выбранному модулю присваиваются позиции, т. е. изделия, которые должны содержаться в нем (т. н. ***позиции модулей***).

Обзор основных элементов диалогового окна:

### ОУ

Введите здесь нижестоящее обозначение устройства, которое относится к этому модулю.

### Идентификатор ОУ

Введите здесь идентификатор ОУ для шаблона функции устройства, который относится к этому модулю. Возможен ввод от 1 до 4.

Чтобы модули также учитывались при выборе устройства, необходимо определить идентификатор ОУ как на главных, так и на вспомогательных функциях.

!!! note "Замечание:"

    * Устройство должно быть частью макроса, присвоенного модулю. Указанное в строке изделие присваивается этому устройству при вставке модуля.
    * Для идентификации устройства, находящегося в присвоенном макросе, используется либо нижестоящее ОУ устройства, либо указанный на устройстве идентификатор ОУ. Соответственно, на строку (т. е. для каждой позиции модуля) возможен ввод либо в столбец ОУ, либо в столбец Идентификатор ОУ. При сохранении выполняется соответствующая проверка. Если ***оба поля*** не заполнены, выводится сообщение.

### Длина

Ввод значений в этот столбец возможен только для изделий с указаниями длины (таких как кабели, соединения, провода / группы соединений, шланги), но невозможен для изделий из главной группы продуктов "Механика".

Для такого изделия, например кабеля, здесь можно ввести часть длины, указанную на самом изделии. Если узел разворачивается при вставке в схему соединений в качестве устройства или при выборе изделия, введенная здесь длина для отдельного изделия переносится в свойство ссылки изделия Подмножество / длина и может выводиться в отчетах. Однако передача работает ***только*** для тех функций, которые из-за определения функции имеют собственное поле длины в дополнение к свойствам ссылки изделия. К ним относятся, например, линии определения кабеля и точки определения соединения.

!!! note "Замечание:"

    Если в отчетах по изделиям (например, в спецификации изделий) необходимо вывести длину отдельных изделий модулей, используемых в схеме соединений, следует развернуть модули для этих отчетов. Для этого в [настройках отчета по изделиям](partslistgui_d_paramartikelauswertung.md) следует установить флажок Развернуть модули.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Удалить содержимое |  Удаляет содержимое текущих выделенных ячеек.
Перейти к |  Этот пункт меню доступен только для столбца Номер изделия. Переход к изделию, введенному в текущей выделенной строке.

### Разбить на групповую спецификацию изделий

C помощью настройки из этого раскрывающегося списка определяется, следует ли разворачивать модули в групповой спецификации изделий; значением по умолчанию является запись "Из настроек", т. е. настройка заимствуется из диалогового окна [Настройки: Изделие](partslistgui_d_paramartikelauswertung.md). При выборе другой настройки она перезапишет настройку, заданную в настройках.

### Распределенное размещение модуля

С помощью настройки из этого раскрывающегося списка задайте размещение модуля в трехмерном чертеже монтажных поверхностей. В соответствии с выбранным здесь параметром соответствующее изделие модуля и / или его элементы (т. е. изделия, которые содержатся в модуле) отображаются в трехмерном навигаторе монтажных поверхностей, если они присвоены устройству, и могут быть размещены в пространстве листа.

Обратите внимание, что эта настройка никак не сказывается на отчетах и на действиях в двумерной компоновке электрошкафа.

!!! note "Замечание:"

    Механические модули можно разместить только как единое целое. В связи с этим настройка Распределенное размещение модуля для модулей из главной группы продуктов "Механика" предустановлена на параметр "Разместить модульные изделия и элементы", и эту настройку нельзя изменить.

# Определить предварительный выбор изделия

Предварительный список изделий представляет собой список изделий, который Вы составляете и сохраняете под определенным именем . Он может применяться как фильтр при выборе устройств и изделий. Предварительный выбор изделий сохраняется для конкретного проекта.

Условие:

Вы дважды щелкнули по условному обозначению и вывели на передний план в диалоговом окне Свойства ++...++ вкладку Изделие.

1. [Настройки] > Выбор устройства
2. В диалоговом окне Настройки: выбор устройства щелкните рядом с полем Фильтр по ++...++.
3. Выберите из раскрывающегося списка определенную пользователем схему, которую вы хотели бы изменить, или создайте через![](images/all_new_as.png){: .ui-icon } (Создать) новую [схему](schemeconfig_h_schematabearbeiten.md).

!!! info "Для сведения:"

    Если предварительный выбор изделий уже содержит изделия, то они показываются в списке.

4. Через![](images/all_new_as.png){: .ui-icon } (Создать) над списком изделий вы открываете диалоговое окно Выбор изделия.
5. Пролистайте структуру дерева и выделите изделия, которые необходимо записать в предварительный выбор изделий.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Изделия записываются в список.

7. Щелкните в диалоговом окне Предварительный выбор изделий по ++OK++.

!!! info "Для сведения:"

    Схема для предварительного выбора изделий сохраняется и показывается в диалоговом окне Настройки: выбор устройств в раскрывающемся списке Список выбора.

8. Закройте диалоговое окно Настройки: выбор устройств кнопкой [OK].

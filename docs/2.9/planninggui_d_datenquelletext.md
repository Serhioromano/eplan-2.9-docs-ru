# Диалоговое окно Выбрать источник данных (предварительное планирование, импорт текстовых файлов)

Вы открыли проект и навигатор предварительного планирования. Данные проекта > Предварительное планирование > Импортировать. В диалоговом окне Импортировать данные предварительного планирования — <Имя проекта> выберите в поле Тип источника данных запись "Текст", а затем щелкните в поле Источник данных на ++...++.

В этом диалоговом окне выберите источник данных для импорта текстового файла.

Обзор основных элементов диалогового окна:

### Текстовый файл

В этом поле выберите текстовый файл: щелкните по ++...++, чтобы в открывшемся диалоговом окне перемещаться по структуре каталогов для выбора требуемого файла.

### Имя столбца в заголовке таблицы

В текстовом файле обязательно должны указываться имена столбцов, чтобы данные могли присваиваться свойствам EPLAN. Поэтому это поле для текстовых файлов всегда активно и выделяется серым цветом.

### Разделитель

Укажите здесь, какие разделители должны использоваться в текстовом файле для разделения столбцов. Доступны следующие значения:

* Пробел
* Табулятор
* Другой символ: После выбора этого параметра введите в поле ввода под ним нужный символ. Допускаются любые цифры, буквы и специальные символы.

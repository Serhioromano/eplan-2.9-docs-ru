# Медный жгут: Принцип

Медный функциональный элемент – это всегда составная часть медного жгута. Медный жгут тем самым является группой размещенных функциональных элементов-шин.

При размещении функционального элемента-шины он записывается в навигаторе пространства листа в ступени иерархии "медные жгуты" с названием жгута и номером. При каждом жгуте записаны отдельные сборные шины, из которых состоит жгут. Жгут, сгенерированный первым, автоматически активируется. Каждый последующий размещенный функциональный элемент-шина присваивается активному жгуту.

Во время размещения медного функционального элемента имеющиеся медные жгуты можно активировать автоматически или вручную, чтобы присвоить функциональный элемент нужному жгуту. Если ни один жгут не активирован, то при размещении функционального элемента генерируется новый жгут. Нажатием клавиши ++N++ во время размещения также можно генерировать новый жгут.

Присвоение медного функц. элемента медному жгуту сохраняется в свойстве Медный жгут. Это свойство можно изменить в диалоговом окне Свойства (усл. обозначение): Размещение изделия (3D). При нажатии клавиши ++...++ отображается диалоговое окно Выбор 3D-объекта. Выбрав другой, уже существующий жгут, можно присвоить медный функциональный элемент этому жгуту. При нажатии кнопки ++Создать++ генерируется новый жгут, который изначально активирован.

Отдельное отображение компонентов электрошкафов также возможно и для медных жгутов. Для этого в навигаторе пространства листа выберите пункты меню Всплывающее меню > Отобразить > Медь.

Если вы выделите медный функциональный элемент в трехмерной области отображения, например, чтобы отобразить свойства, то сначала будет выбран медный жгут вышестоящего уровня. Чтобы выбрать отдельную шину в жгуте, удерживайте нажатой клавишу ++Shift++ при выделении.

Выделенный в навигаторе или в трехмерной области отображения медный жгут можно удалить нажатием клавиши ++Delete++ или через меню Обработать > Удалить. Отдельные медные функц. элементы из медного жгута будут удалены, если при выборе удаляемого элемента в трехмерной области отображения будет нажата клавиша [Shift].

# Вкладка "Арматура"

У вас установлен конфигуратор шлангопроводов в качестве дополнительного модуля для EPLAN Fluid. Вы запустили EPLAN Fluid и конфигуратор шлангопроводов.

* Выберите в EPLAN Fluid пункты меню Параметры > Настройки > Пользователь > Add-ons > Конфигуратор шлангопроводов и щелкните в следующем диалоговом окне по вкладке Арматура.
* Или выберите в диалоговом окне Конфигурировать шлангопроводы конфигуратора шлангопроводов пункты меню Дополнительно > Настройки и щелкните в следующем диалоговом окне по вкладке Арматура.

Или у вас установлен конфигуратор шлангопроводов в качестве самостоятельной программы, и программа запущена.

* Выберите в меню конфигуратора шлангопроводов пункты Параметры > Настройки > Пользователь > Add-ons > Конфигуратор шлангопроводов и щелкните в следующем диалоговом окне по вкладке Арматура.

В этой вкладке можно управлять данными арматуры для шлангопроводов. При конфигурировании шлангопроводов записанные значения предлагаются на выбор в диалоговом окне Обработать ++...++.

Обзор основных элементов диалогового окна:

### Таблица

В этой таблице можно определять типы конструкций, серии, номера изделий и графические файлы арматур.

При использовании конфигуратора шлангопроводов в качестве Add-on к EPLAN Fluid с помощью кнопки ++...++, предусмотренной в табличных ячейках столбца ***Номер изделия***, можно перейти в базу данных изделий и выбрать там нужное изделие.

В столбце ***Графический файл*** для каждой арматуры можно сохранить путь к графическому файлу. Графический файл отображается в окне предварительного просмотра при выборе арматуры.
С помощью доступной в ячейках таблицы кнопки ++...++ можно задать графический файл через окно выбора. Возможные типы графических файлов — *. gif и *.jpg.

Используйте кнопки панели инструментов над таблицей для создания новых записей и/или удаления ненужных записей.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

!!! note "Замечание:"

    Если выбрана стандартная схема DIN 20066_2012-01, таблица заполняется значениями по умолчанию из стандарта DIN 20066. На выбор предлагаются следующие типы конструкции и серии:

* Типы конструкции: D, N, N45, N90, R, R45, R90, E, P, P45, P90, S, S45 и S90
* Серии: "L" (для "метрически легкий") и "S" (для "метрически тяжелый").

# Вкладка Общее (Экспорт шаблона сверления)

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт шаблона сверления. Выберите вкладку Общ..
* Сервисные программы > Данные изготовления > Механическая обработка > Шаблон сверления. Рядом с полем Машина нажмите ++...++. Выберите вкладку Общ..

В этой вкладке вы определяете общие настройки для экспорта шаблонов сверления.

Обзор основных элементов диалогового окна:

Групповое поле 'Края'

В этом групповом поле укажите минимальное расстояние от геометрии схемы сверления до края поверхности сверления в полях Вверху / Слева / Справа / Внизу.

* * *

### Каталог вывода

Шаблон сверления будет создан в виде PDF-файла. В этом поле укажите место хранения. В качестве целевых для экспорта данных можно указать любой дисковод и подкаталог.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Генерировать имена файлов

В этом раскрывающемся списке определяется стандартный порядок присвоения имен файлам экспорта PDF.

* Из непрерывного номера: При выборе этого параметра EPLAN по умолчанию присваивает файлам PDF имена по образцу <непрерывный_номер>.PDF. Если уже имеются PDF-файлы, то EPLAN начинает нумерацию со следующего доступного числа.
* Из обозначения функц. элемента: При выборе этого параметра EPLAN по умолчанию присваивает файлам PDF имена по образцу <Обозначение функц. элемента>.PDF.
* Из свойств: Если выбран этот параметр, имя файла составляется из свойств пространства листа, проекта и устройства, а также свободно определяемого разделителя. Используя кнопку ++...++, откройте диалоговое окно Формат: Свойство для конфигурации имени файла.

### Генерировать подкаталоги

В этом раскрывающемся списке определяется стандартный порядок создания подкаталогов при экспорте файлов PDF.

* Нет: Если выбран данный параметр, при экспорте подкаталоги не генерируются.
* Из структуры проекта: Если выбран данный параметр, то при экспорте автоматически генерируются подкаталоги, отражающие структуру проекта.
* Из свойств: Если выбран этот параметр, имя подкаталога составляется из свойств пространства листа, проекта и устройства, а также свободно определяемого разделителя. Используя кнопку ++...++, откройте диалоговое окно Формат: Свойство для конфигурации имени файла. Используйте здесь, например, элемент формата "Разделитель каталога", чтобы сгенерировать структуру каталога с несколькими расположенными друг под другом подкаталогами.

# Обновить соединения

Свойства соединений определены в различных местах: в настройках проекта, на выводах устройств потенциалов, точках определения потенциалов и точках определения соединений. При изменениях данные соединений не обновляются автоматически сей же час. При необходимости обновите их вручную.

!!! note "Замечание:"

    Статус обновления соединений отображается в правом краю Строки состояния: Если в ней стоит символ "#", значит, проект содержит необновленные соединения. Если дополнительно отображается символ "*", значит, необновленные соединения присутствуют на открытой странице.

По умолчанию в настройках пользователя (Параметры > Настройки > Пользователь > Представление > Общее) поставлен флажок Обновить соединения при смене страницы. Таким образом соединения автоматически обновляются во время обработки в графическом редакторе. Соединения обновляются в пределах обработанной страницы после каждого шага обработки. При смене страницы соединения обновляются как на открытых страницах, так и на тех страницах, где соединения проводятся через точки разрыва.

### Обновить соединения вручную

Управление поведением при обновлении соединений вручную происходит через настройки пользователя: По умолчанию в настройках пользователя (Параметры > Настройки > Пользователь > Представление > Общее) установлен флажок Обновить соединения для всего проекта (вручную). Благодаря этому обновляются соединения во всем проекте, независимо от текущего выбора.

Если этот флажок снят, соединения обновляются только для текущего выбора. Если выделены одна или несколько страниц, обновляются только соединения на этих страницах. Если выделены функции в графическом редакторе (или одном из навигаторов), происходит обновление соединений только для тех страниц, на которых находятся функции соответствующего устройства, а также неразмещенных соединений устройства.

Условие:

Вы открыли страницу в графическом редакторе либо выделили страницы или проект в навигаторе страниц. Вы выделили функции в графическом редакторе или одном из навигаторов.

1. Выберите следующие пункты меню: Данные проекта > Соединения > Обновить

!!! info "Для сведения:"

    EPLAN обновляет соединения в зависимости от пользовательской настройки, либо во всем проекте, либо в текущем выборе. Это значит, что при необходимости генерируются новые соединения и обновляются изменения имеющихся соединений.

!!! tip "Совет:"

    Чтобы заново сгенерировать все соединения проекта, при вызове действия Обновить удерживайте нажатой клавишу ++Shift++. При этом применяются изменения свойств соединений, определенных в настройках проекта. В любом случае при обработке крупных проектов это может занять больше времени.

### Удалить соединения

Соединения удаляются автоматически, если вы переместите условное обозначение в схеме соединений, а его выводы устройств не найдут себе больше ни одного эквивалента. Возможные имеющиеся точки определения соединений останутся.

Неразмещенные соединения можно удалить в диалоговом окне Соединения — <Имя проекта>:

1. Выберите следующие пункты меню: Данные проекта > Соединения > Навигатор
2. Выделите в диалоговом окне Соединения — <Имя проекта> неразмещенные соединения, которые вы хотите удалить.
3. Выберите пункт всплывающего меню Удалить.

!!! info "Для сведения:"

    После запроса на подтверждение соответствующие соединения будут удалены.

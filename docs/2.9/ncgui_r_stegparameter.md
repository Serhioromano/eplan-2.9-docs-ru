# Вкладка Технологические перемычки (экспорт Rittal - Perforex BC / Secarex)

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт Rittal - Perforex BC / Secarex. Выберите вкладку Технологические перемычки.
* Сервисные программы > Данные изготовления > Механическая обработка > Rittal - Perforex BC / Secarex. Рядом с полем Машина нажмите ++...++. Выберите вкладку Технологические перемычки.

Чтобы избежать повреждений фрезерного инструмента при фрезеровке слишком больших прямоугольных поверхностей, эти поверхности можно разделить при помощи технологических перемычек. В результате этого образуются фрезерные области меньшего размера. Возникающие технологические перемычки при монтаже выламываются. На этой вкладке задаются все характеристики для автоматической обработки технологических перемычек.

!!! note "Замечание:"

    * Во избежание искажения контура при позиционировании фрезерного инструмента для контуров с очень малой высотой должны выполняться следующие условия: Технологические перемычки на прямоугольниках выполняются только в том случае, если длина меньшей стороны составляет не менее 20 мм.

    Для прямоугольных отверстий с фаской или закругленных действуют следующие правила.

* Если на этой вкладке указано, что по всей длине стороны (ширине или высоте стороны) должны быть сгенерированы ***три*** технологических перемычки, а имеющейся длины из-за наличия фасок / закруглений хватает только на ***две*** перемычки, то выполняются ***две*** технологических перемычки.
* Если на этой вкладке указано, что по всей длине стороны (ширине или высоте стороны) должны быть сгенерированы ***две*** технологических перемычки, а имеющейся длины из-за наличия фасок/закруглений хватает только на ***одну*** перемычку, то выполняется ***одна*** технологическая перемычка.
* Эти правила применяются в тех случаях, когда разность между длиной стороны и размером фаски / радиусом закругления больше, чем сумма ширины технологической перемычки и удвоенного диаметра фрезы / диаметра отверстия.

    Доработка:

* Чтобы не допустить неотфрезованных фрагментов на материале, вносятся следующие поправки: Если на этой вкладке указано, что на обеих параллельных сторонах должно быть сгенерировано по ***одной*** технологической перемычке, а на остальных сторонах перемычки не предусмотрены, то фактически генерируются ***две*** технологические перемычки.

Обзор основных элементов диалогового окна:

Групповое поле 'Настройки для стандартных контуров'

Используйте стандартные настройки или измените значения для отдельных настроек в соответствии с вашими требованиями.

### Ширина технологической перемычки

По умолчанию ширина технологической перемычки составляет 1 мм.

### Генерировать одну технологическую перемычку, если длина ребра превышает

По умолчанию технологическая перемычка добавляется при фрезеровочной длине свыше 100 мм.

### Генерировать две технологические перемычки, если длина ребра превышает

По умолчанию две технологические перемычки добавляются при фрезеровочной длине свыше 250 мм.

### Генерировать три технологические перемычки, если длина ребра превышает

По умолчанию три технологические перемычки добавляются при фрезеровочной длине свыше 500 мм.

### Дополнительно обработать технологические перемычки

Из-за того что технологические перемычки при работе на машине Perforex после изготовления удаляются клещами вручную, возникают шероховатые поверхности, которые подлежат ручной обработке (например, шлифовке). Машина Perforex может автоматически выполнять такую дополнительную обработку технологических перемычек.

Дополнительная обработка технологических перемычек возможна только для прямоугольных контуров; технологические перемычки в пользовательских контурах автоматически обработать нельзя.

Если установлен этот флажок, для всех полей, на которых обрабатываются контуры с технологическими перемычками, на машину посылаются две схемы сверления.

Первая схема сверления содержит данные по обработке с технологическими перемычками, причем контуры для дополнительной обработки уменьшены на введенное значение интенсивности. На второй схеме сверления передается дальнейший ход сверления для контуров с технологическими перемычками. При этом речь идет о точном внешнем контуре без технологических перемычек. Дополнительная обработка проводится в тех местах, где после первого захода осталась технологическая перемычка. Между местами технологических перемычек фреза выходит из детали и быстро перемещается к следующей технологической перемычке, не выполняя фрезеровку.

Вторую схему сверления можно также использовать для дополнительной обработки после отламывания технологических перемычек вручную для тонкой шлифовки мест слома.

### Интенсивность дополнительной обработки

Если флажок Дополнительно обработать технологические перемычки установлен, укажите здесь величину, на которую вырезается контур технологической перемычки после отламывания. По умолчанию установлено 0,00 мм.

### Суффикс для программы дополнительной обработки

Указанная здесь последовательность символов добавляется при генерировании имени программы дополнительной обработки Perforex и имени контура дополнительной обработки Perforex. По умолчанию используется настройка _NA.

Если в программном обеспечении Perforex дополнительно установлен суффикс, убедитесь, что обе настройки совпадают.

* * *

Оба следующих флажка доступны, если на вкладке Общ. для ПО машины Perforex задан формат "Windows от 1.9x", активирована настройка Дополнительно обработать технологические перемычки, а в поле Интенсивность дополнительной обработки внесено значение 0 мм.

### Функция M50

Если этот флажок установлен, используется функция M50 машины Perforex. Эта функция автоматически обрабатывает остатки технологических перемычек в прямоугольных фрезерных контурах в центре каждого ребра без остановки на проверку или смену фрезерного инструмента. Для этого точки обработки подводятся в режиме повышенного быстродействия по диагонали за пределами фрезерного контура.

!!! note "Замечание:"

    Для использования функции M50 значение настройки Z высокое на вкладке Инструменты должно составлять не менее 6 мм.

### Не генерировать программу дополнительной обработки

Если этот флажок установлен, в ПО Perforex не будет генерироваться дополнительная программа ЧУ для дополнительной обработки. Дополнительная обработка интегрируется в программу фрезерования.

Для дополнительной обработки технологических перемычек не нужно генерировать программу дополнительной обработки, если на действующем оборудовании установлено программное обеспечение Perforex начиная с версии 1.9.2.74 и выше.

# Вкладка Изготовление

Сервисные программы > Изделие > Управление. В дереве, списке или комбинированном представлении выделено одно изделие.

Укажите на этой вкладке производственно-техническую информацию, которая не зависит от выбранного типа изделия и группы продуктов (отдельная часть, кабели и т. д.). Эти данные можно указать ***один раз*** для каждого изделия.
Обращайте внимание на то, чтобы схема сверления ***не*** была изделием, а только служила для изделия дополнительным источником информации относительно конструкции. Каждое изделие может содержать сколько угодно таких сведений о схеме сверления.

Обзор основных элементов диалогового окна:

### Схема сверления

Здесь щелкните по ++...++ и выберите в диалоговом окне Выбрать схему сверления схему или схемы расположения обрабатываемых отверстий, которые должны быть назначены изделию.

Устройства с различными вариантами монтажа для каждого варианта должны иметь заданную подходящую схему сверления. При размещении такого изделия на экране отображается диалоговое окно, в котором вы можете выбрать используемую схему сверления. Если диалоговое окно выбора Выбрать схему сверления больше не нужно отображать, можно деактивировать его при помощи флажка Больше не отображать это диалоговое окно. В этом случае всегда используется первая схема сверления, записанная на изделии.

Схема сверления, выбранная при размещении, регистрируется в параметрах размещения изделия на вкладке Изделие в категории Данные ссылки изделия в поле Используемая схема сверления. Здесь можно дополнительно изменить схему сверления. Для этого в раскрывающемся списке на выбор предлагаются все схемы сверления, присвоенные данному изделию.

### Смещение в направлении X / Смещение в направлении Y

Укажите в этих полях для каждой схемы сверления смещение в направлениях X и Y. Таким образом, можно использовать идентичные схемы сверления для изделий разной величины.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

# Определить формат нумерации в режиме подключения

Задайте через формат нумерации в режиме подключения автоматический процесс нумерации для нового устройства/функции. К примеру, она применима к устройствам/функциям, которые вы разместили в схеме соединений посредством Вставить > Символ или создали в навигаторе устройств.

Таким же образом задайте через формат нумерации в режиме подключения процесс автоматической нумерации для устройств/функций, которые вы вставили в схему соединений посредством макроса и/или скопировали из буфера обмена.

Для каждого вставленного устройства в зависимости от формата нумерации ***online*** определяется ОУ, которое автоматически присваивается устройству. Для обоих способов вы можете указать одинаковые или различные форматы нумерации в режиме подключения.

Условие:

Вы открыли проект.

1. Параметры > Настройки > Проекты > "имя проекта" > Устройство > Нумерация (online)
2. Нажмите в групповом поле При вставке символов или При вставке макросов и копий рядом с полем Формат нумерации кнопку ++...++.
3. Для настройки формата нумерации нажмите в диалоговом окне Форматы нумерации рядом с полем Схема кнопку ![](images/all_new_as.png){: .ui-icon } (Создать).
4. Укажите в поле Имя какое-нибудь выразительное имя, а в поле Описание дайте краткий сопроводительный текст, затем нажмите ++OK++.
5. Выберите вкладку ОУ.

!!! info "Для сведения:"

    На этой вкладке отображаются все устройства, для которых активирована нумерация в режиме подключения и для которых вы можете задавать форматы ОУ независимо друг от друга. К ним относятся: "Устройства общего назначения", "Клеммники", "Штекеры", "Черные ящики", "ПЛК", "Кабели", "Устройства Fluid", "Распределители / соединители проводов Fluid".

6. Например, чтобы выбрать для нумерации в режиме подключения все устройства группы устройств "Устройства общего назначения", установите флажок Устройства общ. назначения.
7. Чтобы установить формат ОУ для всех устройств группы устройств "Устройства общего назначения", нажмите у поля Устройства общего назначения кнопку ++...++.

!!! info "Для сведения:"

    Откроется диалоговое окно Формат ОУ. По умолчанию формат ОУ уже содержит элементы формата "буквенное обозначение" и "счетчик". Эти элементы формата отображаются в поле Выбранные элементы формата.

8. Чтобы выбрать следующие элементы формата для формата ОУ, выделите в поле Доступные элементы формата один элемент формата и нажмите кнопку ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо).
9. Укажите в диалоговом окне Формат в поле Мин. число разрядов, которое должно иметь значение ОУ, а также задайте в поле Заполнитель тот знак, которым должны быть заполнены недостающие позиции ОУ, и нажмите ++OK++.

!!! info "Для сведения:"

    Элемент формата копируется в поле Выбранные элементы формата.

10. Для обработки ранее выбранных элементов выделите в поле Доступные элементы формата какую-нибудь запись и нажмите кнопку ![](images/all_edit_as.png){: .ui-icon } (Обработать), задайте желаемые настройки в диалоговом окне Формат и нажмите ++OK++.
11. Чтобы задать последовательность элементов формата в ОУ, выделите один элемент формата и нажмите кнопку ![](images/all_arrowup_as.png){: .ui-icon } (Переместить вверх) или ![](images/all_arrowdown_as.png){: .ui-icon } (Переместить вниз).

!!! info "Для сведения:"

    Результат выводится в поле Предв. просмотр.

12. Выберите и обработайте один за другим все нужные элементы формата. При этом необходимо следовать правилам [формата нумерации в режиме подключения](numerationgui_k_format.md).
13. Щелкните по кнопке ++ОК++.
14. Чтобы задать формат ОУ для обозначений вывода устройства, активируйте вкладку Обозначения.

!!! info "Для сведения:"

    На этой вкладке отображаются все группы устройств для выводов, для которых активируется нумерация в режиме подключения и/или для которых вы можете задавать форматы ОУ для обозначения вывода устройств независимо друг от друга. К ним относятся: "Клеммы", "Контакты-штекеры", "Выводы устройства", "Выводы устройства ПЛК".

15. Например, чтобы выбрать все клеммы для нумерации в режиме подключения, установите флажок Клеммы.
16. Например, чтобы задать формат ОУ для обозначений выводов клемм, щелкните у поля Клеммы [...], укажите в диалоговом окне Формат ОУ элементы формата и нажмите ++OK++.
17. Щелкните по кнопке ![](images/all_save_as.png){: .ui-icon } (Сохранить) и по ++OK++.

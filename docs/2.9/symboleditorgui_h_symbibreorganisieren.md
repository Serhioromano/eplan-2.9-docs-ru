# Реорганизовать библиотеки символов

При реорганизации библиотеки символов удаленные данные окончательно удаляются из базы данных. При этом база данных уменьшается.

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Сервисные программы > Основные данные > Библиотека символов > Реорганизовать
2. В диалоговом окне Реорганизовать библиотеку символов выберите библиотеку, которую следует реорганизовать, и нажмите кнопку ++Открыть++.

!!! info "Для сведения:"

    Библиотека символов сжимается и реорганизуется.

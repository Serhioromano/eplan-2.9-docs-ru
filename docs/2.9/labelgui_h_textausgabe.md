# Определить настройки вывода *.txt

С помощью настроек для *.txt-вывода укажите, что файл экспорта / маркировки / внешний файл отчетов генерируются в формате txt.

Условие:

Вы открыли проект.

1. Сервисные программы > Данные изготовления > Экспорт / маркировка
2. В диалоговом окне Экспортировать данные изготовления / Вывести маркировку рядом с полем Настройки щелкните по кнопке ++...++.
3. В диалоговом окне Настройки: Экспорт данных изготовления / маркировка в поле Схема, определенную пользователем.

!!! info "Для сведения:"

    При помощи ![](images/all_new_as.png){: .ui-icon } (Создать) можно [создать новую схему экспорта / маркировки](labelgui_h_beschriftungsschema.md).

4. Выберите на вкладке Файл в поле Тип файла тип файла "Текст-файл (*.txt)".
5. Щелкните рядом с полем Целевой файл по ++...++ и выберите в открывшемся диалоговом окне каталог и файл или задайте в поле Имя файла имя.
6. Нажмите ++Параметры++.
7. Для генерирования текстового файла для приложения Windows выберите в диалоговом окне Настройки: Экспорт текста в поле Набор символов набор символов ANSI.
8. При необходимости установите флажок Разрешить умляуты.
9. Щелкните рядом с полем Прикладная программа по кнопке ++...++ и выберите через следующее диалоговое окно файл приложения *.exe того редактора, в котором должен быть открыт файл.

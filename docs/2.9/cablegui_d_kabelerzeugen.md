# Диалоговое окно Автоматически генерировать кабели

Вы открыли проект.

* Вы выделили условные обозначения в Графическом редакторе.
* Либо вы выделили кабели или кабельные соединения в диалоговом окне навигатора Кабели — <Имя проекта>.
* Либо вы выделили соединения в диалоговом окне навигатора Соединения — <Имя проекта>.
* Либо вы выделили страницы или проект в диалоговом окне навигатора Страницы — <Имя проекта>.

Данные проекта > Кабели > Автоматически генерировать кабели.

В этом диалоговом окне задаются настройки, используемые для автоматического генерирования кабелей. Далее нужно запустить генерирование кабелей.

Обзор основных элементов диалогового окна:

Групповое поле Генерация кабелей

### Настройки

Выбранная здесь схема определяет, где должны быть размещены свойства кабелей и какие кабельные соединения должны учитываться при генерации кабелей. С помощью кнопки ++...++ откройте диалоговое окно [Настройки: Генерация кабеля](cablegui_d_einstellungenkabelerzeugen.md), в котором можно обработать схемы и управлять ими.

* * *

Групповое поле Нумерация кабелей

### Настройки

Выбранная здесь схема определяет, в какой последовательности выполняется нумерация кабелей, какие свойства будут отображаться в ОУ кабеля и каким образом будет осуществляться их форматирование. С помощью кнопки ++...++ откройте диалоговое окно [Настройки: Нумерация кабелей](cablegui_d_einstellungenkabelnummerierung.md), в котором можно обработать схемы и управлять ими.

### Начальное значение

Введите здесь начальное значение для счетчика ОУ.

### Величина шага

Введите здесь значение, на которое должны каждый раз увеличиваться показания счетчика ОУ.

### Имеющиеся ОУ кабеля не изменять

Установите этот флажок, если вы хотите дополнительно пронумеровать заново сгенерированный или скопированный кабель. ОУ уже пронумерованного кабеля сохраняется. Нумеруются только кабели со знаком "?" в буквенном обозначении.

Если флажок не установлен, все ОУ кабелей нумеруются заново.

* * *

Групповое поле Автоматический выбор кабеля

### Настройки

Выбранная здесь схема определяет критерии для автоматического выбора кабеля. С помощью кнопки ++...++ откройте диалоговое окно [Настройки: Автоматический выбор кабеля](cablegui_d_einstellungenautomatischekabelauswahl.md), в котором можно обработать схемы и управлять ими.

### Только автоматически сгенерированные или названные кабели

Если этот флажок установлен, выбор кабеля осуществляется только для автоматически сгенерированных кабелей. Сгенерированные вручную кабели не учитываются.

# EPLAN Fluid: Нумерация ОУ

В EPLAN Fluid предусмотрено несколько способов нумерации обозначений устройств Fluid-техники:

* [Нумерация (online)](numerationgui_k_start.md)
* [Нумеровать (автономно)](offlinenumerationgui_k_start.md)
* Нумерация посредством диалогового окна [Выбор ОУ](modaldialogsdb_d_bmkauswahl.md).

Если для устройств Fluid используется обозначение согласно DIN ISO 1219-2, то при ***вставке символов*** номер функционального элемента по умолчанию будет увеличиваться по нарастающей. В окне Вставка макросов или копий можно определить, как должна работать программа при нумерации ОУ Fluid-техники. Вы можете задать подсчет по возрастанию либо номеров схемы, либо номеров функциональных элементов.

При выборе ОУ из обозначенных согласно DIN ISO 1219-2 устройств Fluid-техники можно программно выбрать следующий свободный номер логической ячейки или функционального элемента в видимом обозначении устройства.

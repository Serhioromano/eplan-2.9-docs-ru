# Выполнить или загрузить сценарии

Простые сценарии, функции которых должны быть выполнены однократно, выполняются через одноименный пункт меню. Для обозначения соответствующей функции используется атрибут [Start]. Необходимо загрузить сценарии, функции которых должны быть постоянно доступны для системы. Здесь для обозначения используются другие атрибуты.

### Выполнить сценарии

Условие:

Простой сценарий создан (через атрибут [Start]) на языке программирования C# или Visual Basic.Net.

1. Выберите пункты меню Сервисные программы > Отчеты > Выполнить.

!!! info "Для сведения:"

    Откроется диалоговое окно Выбор файла сценария.

2. Выберите сценарий через Тип файла CSharp (*.cs) или VB.NET (*.vb).
3. Щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Выбирается сценарий.

Если при выполнении сценария возникает ошибка, то она записывается в журнал Системные сообщения.

### Загрузить сценарии

Условие:

Комплексный сценарий создан (например, через собственные операции) на языке программирования C# или Visual Basic.Net. В комплексных сценариях используются [DeclareEventHandler], [DeclareAction] и / или ++DeclareMenu++.

1. Выберите пункты меню Сервисные программы > Сценарии > Загрузить.

!!! info "Для сведения:"

    Откроется диалоговое окно Выбор файла сценария.

2. Выберите сценарий через Тип файла CSharp (*.cs) или VB.NET (*.vb).
3. Щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Загружается сценарий.

Если при загрузке сценария возникает ошибка, то она записывается в журнал Системные сообщения. Если сценарий один раз загружен, при последующем запуске программы он загружается автоматически.

### Выгрузить сценарии

Условие:

Сценарий загружен.

1. Выберите пункты меню Сервисные программы > Сценарии > Выгрузить.

!!! info "Для сведения:"

    Открывается диалоговое окно Выгрузить сценарий.

2. Выберите в списке зарегистрированных сценариев соответствующий сценарий.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Отмена сценария.

При последующем запуске программы выгруженный сценарий больше не регистрируется автоматически в EPLAN.

# Следующие формы

Для определенных видов отчетов (например, список обозначений устройств, схема кабельных соединений, перечень клеммников и т.д.) возможно указание следующей формы. Она выводится во время оценки после текущей формы. Последующая форма может быть вставлена либо на той же странице, что и текущая форма, либо на следующей странице; это определяется с помощью точки вставки последующей формы.

### Точка вставки следующей формы

Точка вставки последующей формы может быть размещена в другой форме, чтобы разместить на этом месте другой отчет. Благодаря этому можно генерировать блоки отчетов, состоящих из двух разных форм. Все страницы, сгенерированные для одного имени формы, образуют вместе блок отчетов.

В форму можно также вставить несколько точек вставки следующей формы. Последовательность отчета определяется графикой.

Форма и тип отчета следующей формы будут записаны на точке вставки следующей формы. Дополнительно можно задать критерии фильтра и сортировки для объема и сортировки отчета.

Все созданные страницы копируют тип отчета / тип страницы из главной формы (например из схемы кабельных соединений.) Последующий отчет только лишь описывается на странице (как встроенный отчет). В последующей форме вероятно содержащиеся точки вставки последующей формы игнорируются.

### Типы отчетов

В зависимости от типа формы открытой главной формы имеются различные типы отчетов для следующих форм. Для большинства типов форм доступна "Групповая спецификация изделий" и "Спецификация изделий". "Тот же тип формы" в таблице означает, что, например, для типа отчета "Спецификация кабеля" возможна также следующая форма типа отчета "Спецификация кабеля".

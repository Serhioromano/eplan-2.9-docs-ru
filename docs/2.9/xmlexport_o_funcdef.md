# Импорт и экспорт XML: Обзор определения функций

Для импорта или экспорта данных об изделии в EPLAN, XML-данные должны содержать тэг \<functiontemplate\>. У этого тэга есть следующие атрибуты: "functiondefcategory" (определение функции Категория), "functiondefgroup" (определение функции Группа) и "functiondefid" (определение функции Определение функции). Три эти атрибута вместе определяют функцию изделия.

Указанные ниже внутренние значения ссылаются на приведенные выше атрибуты. Порядок такой: ***Категория.Группа.Определение_функции***.

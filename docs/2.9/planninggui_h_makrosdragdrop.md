# Создать подробное планирование путем перетаскивания мышью

Если на объектах планирования сохранены макросы, из них можно создать подробное планирование. Для этого необходимо разместить макросы на страницах проекта. Быстрый и простой способ разместить макросы — это перетащить мышью объекты планирования из дерева навигатора предварительного планирования в навигатор страниц.

Условия:

* Вы открыли проект.
* Вы создали объекты планирования и сохранили на них макросы.
* Вы открыли навигатор предварительного планирования и навигатор страниц.

### Разместить макрос путем перетаскивания мышью на новой странице

Условие:

Настройка проекта Разместить на одной странице несколько макросов отключена.

1. Выделите объект планирования в дереве навигатора предварительного планирования.
2. Перетащите объект планирования в навигатор страниц.

!!! info "Для сведения:"

    Если на объекте планирования сохранен макрос окна с несколькими видами представления и / или вариантами, откроется диалоговое окно выбора. С помощью диалогового окна можно задать вид представления и вариант размещаемого макроса.

!!! info "Для сведения:"

    Генерируется страница. Имя и структурные идентификаторы страницы автоматически получаются из структурных идентификаторов объекта планирования.

!!! info "Для сведения:"

    Макрос, сохраненный на объекте планирования, размещается на этой странице.

!!! info "Для сведения:"

    Автоматически сгенерированной странице схемы соединений назначается свойство Сгенерировано автоматически со значением "Предварительное планирование".

!!! info "Для сведения:"

    Страница автоматически заносится в список присвоенных страниц на объекте планирования (вкладка Документы / страницы в диалоговом окне "Свойства").

При многократном выборе объектов планирования по умолчанию для каждого макроса генерируется новая страница проекта.

!!! tip "Совет:"

    * Можно генерировать страницы схемы детального планирования также из представления в виде списка навигатора предварительного планирования. Для этого выделите требуемые сегменты с указанным макросом в списке и выберите пункт всплывающего меню Генерировать новые страницы из макроса.
    * С помощью активированного фильтра можно выполнить быстрый многократный выбор сегментов в списке навигатора предварительного планирования и таким образом целенаправленно сгенерировать несколько страниц схем соединений через всплывающее меню. В представлении структуры дерева быстрый и простой многократный выбор сегментов невозможен, т. к. здесь при активированном фильтре также отображаются вышестоящие сегменты.

### Разместить макрос путем перетаскивания мышью на имеющейся странице

Условия:

* Настройка проекта Разместить на одной странице несколько макросов активирована.
* На странице, на которую перетягивается объект планирования, еще есть место для макроса.

1. Выделите объект планирования в дереве навигатора предварительного планирования.
2. Перетащите объект планирования на имеющуюся страницу в навигаторе страниц.

!!! info "Для сведения:"

    Если на объекте планирования сохранен макрос окна с несколькими видами представления и / или вариантами, откроется диалоговое окно выбора. С помощью диалогового окна можно задать вид представления и вариант размещаемого макроса.

!!! info "Для сведения:"

    Макрос, сохраненный на объекте планирования, размещается на этой странице.

!!! info "Для сведения:"

    Страница автоматически заносится в список присвоенных страниц на объекте планирования (вкладка Документы / страницы в диалоговом окне "Свойства").

При многократном выборе объектов планирования несколько макросов автоматически располагаются на уже имеющейся странице, пока там есть место. После этого генерируется новая страница и там размещаются следующие макросы. При перетаскивании объектов планирования на узел в навигаторе страниц одновременно генерируется новая страница, и на ней размещаются несколько макросов, пока там есть место.

Возможность размещения нескольких макросов на странице друг за другом или друг под другом зависит от настройки проекта Разместить макросы друг под другом. По умолчанию эта настройка отключена, а макросы размещаются друг за другом.

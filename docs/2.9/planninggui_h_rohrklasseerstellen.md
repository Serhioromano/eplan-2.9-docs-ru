# Создать и обработать классы трубы и вещества

Создавать классы трубы и вещества и управлять ими можно в навигаторе шаблонов сегментов.

На вкладке <Отображаемое имя класса трубы> или <Отображаемое имя вещества> диалогового окна "Свойства" обработайте свойства класса трубы или вещества.

### Создать классы трубы

1. Выберите следующие пункты меню: Данные проекта > Предварительное планирование > Навигатор шаблонов сегментов
2. Выделите "определение сегмента" для класса трубы (напр., "Класс трубы, общая информация") и выберите пункт всплывающего меню Новый класс трубы.
3. В диалоговом окне Свойства ++...++ на вкладке <Отображаемое имя класса трубы> введите идентифицирующее имя и, при желании, описание для нового класса трубы.
4. При необходимости присвойте классу трубы дополнительные свойства. Чтобы в выборе свойств для таблицы Свойства были доступны определенные пользователем свойства, необходимо присвоить выбранному "определению сегмента" для класса трубы требуемые свойства в диалоговом окне Конфигурировать определения сегмента.
5. При необходимости запишите на вкладке Изделие изделие. Затем при выборе изделия выбирать на сегментах можно только изделия, присвоенные классу трубы.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Будет создан класс трубы с соответствующими свойствами.

Для создания веществ с помощью пункта всплывающего меню Новое вещество выполните аналогичные действия. В этом случае присвоение изделий невозможно.

В проекте для выбора доступны созданные классы трубы и вещества на сегментах предварительного планирования, а также на точках определения трубопровода и выводах трубопровода на функциональной схеме автоматизации.

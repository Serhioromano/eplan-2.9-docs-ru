# Вкладка Группа соединений / провод

Вы открыли проект.

* Вы выделили в Графическом редакторе линию определения группы соединений или символ определения группы соединений, трубопроводов или шлангопроводов. Обработать > Свойства. Выберите в диалоговом окне Свойства ++...++ вкладку Группа соединений / провод.
* Определение группы соединений выделено в навигаторе устройства. Пункт всплывающего меню Свойства или Свойства (общие).Выберите в диалоговом окне Свойства ++...++ вкладку Группа соединений / провод.

На этой вкладке обрабатываются свойства группы соединений, трубо- или шлангопровода.

В режиме обработки ***Свойства (общие)*** вы изменяете свойства сразу для всех представлений функции, в этом случае во вкладке дополнительно отображается "(Устройство)".

Обзор основных элементов диалогового окна:

### Видимое ОУ

Видимое ОУ будет показано в графическом редакторе рядом с условным обозначением. Если вы определили в настройках проекта, что устройство должно автоматически нумероваться при вставке символов, здесь вам будет автоматически предложено ОУ. Вы можете сохранить или переписать вручную это ОУ.

Кроме того, можно использовать [всплывающее меню или кнопку ++...++](devicetaggui_r_feldsichtbaresbmk.md), чтобы cгенерировать или выбрать ОУ.

### Полное ОУ

В этом поле отображается полное ОУ (т. е. структурный идентификатор текущего окружения + видимое ОУ) видимого ОУ / функционального элемента. С помощью кнопки ++...++ откройте диалоговое окно Полное ОУ. Здесь можно обработать структурный идентификатор и, например, выбрать другое место установки.

### Тип

В этом поле можно ввести тип группы соединений или проводов. С помощью кнопки ++...++ откройте диалоговое окно Выбор изделия. В нем вы можете выбрать изделие и присвоить его технические данные текущей группе соединений или текущему проводу.

Данные об изделиях будут обновлены (как при выборе нового изделия). В отличие от обычного выбора изделий диалоговое окно конфликта, в котором отображаются старые и новые данные, здесь не появляется, а вместо этого старые данные перезаписываются.

### Функциональный текст

Укажите здесь функциональный текст для функции / устройства.

Независимо от введенного здесь вручную функционального текста существуют другие (определяемые автоматически) функциональные тексты, которые находятся в таблице:

* Функциональный текст (автоматически): Отображает содержание введенного вручную функционального текста или, при его отсутствии, функциональный текст зоны. Если вы изменяете функциональный текст в режиме обработки ***Свойства (общие)***, его содержание переносится обратно в ту функцию, в которой функциональный текст был первоначально задан. При отображении этого свойства на странице проекта разрывы строк удаляются.
* Функциональный текст (общий): отображает (автоматический) функциональный текст функции или, при его отсутствии, (автоматический) функциональный текст соответствующей главной функции.

### Число соединений / шлангов

Введите сюда количество соединений и/или шлангов, которые содержатся в группе соединений. При выборе изделий это значение копируется из базы данных изделий и может быть использовано в качестве критерия фильтра при выборе устройств. В случае отдельных проводов, например шлангопроводов, оставляйте поле незаполненным.

### Длина

Укажите здесь длину группы соединений или проводов (включая единицу измерения). Если вы не указали единицу, будет использована единица, определенная в настройках проекта. Обзор возможных единиц измерения приводится в соответствующем разделе "[Соединения: Единицы измерения длины и поперечного сечения](connectionbrowsergui_k_einheitenverbindungen.md)".

!!! note "Замечание:"

    Рядом со свойством ***Длина*** существует другое свойство с именем ***Длина шлангопровода***, которое отображается в таблице свойств. Это свойство заполняется данными, когда вы копируете данные шлангопроводов, содержащие длину шлангопровода, из конфигуратора шлангопроводов EPLAN.

### Поперечное сечение / диаметр:Единица измерения

Поперечное сечение или диаметр соединения копируется из базы данных изделий и могут быть использованы в качестве критерия отбора при выборе устройств. Щелкните по ++...++, чтобы открыть диалоговое окно [Поперечные сечения / диаметры](connectionsettingsgui_d_einstellungenquerschnitt.md) и выбрать сечение / диаметр, включая соответствующую единицу, из предложенного списка.

Если в поле Поперечное сечение / диаметр вы выбрали предварительно определенное поперечное сечение / диаметр, то в поле Единица измерения автоматически появится соответствующая единица. С помощью раскрывающегося списка возможен выбор другой единицы измерения. Обзор возможных единиц измерения приводится в соответствующем разделе "[Соединения: Единицы измерения длины и поперечного сечения](connectionbrowsergui_k_einheitenverbindungen.md)".

### Главная функция

установите этот флажок, если текущая функция является главной.
При использовании имеющегося ОУ, для которого уже задана главная функция, при использовании уже размещенной главной функции, а также при размещении уже размещенной главной функции данное свойство будет автоматически отключено на целевой функции, то есть данный флажок будет снят.

!!! note "Замечание:"

    эта настройка автоматически синхронизируется с одноименным флажком на вкладке Данные символа / функции.

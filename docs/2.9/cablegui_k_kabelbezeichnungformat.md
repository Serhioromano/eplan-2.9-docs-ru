# Позиция и форматирование свойств кабелей

Свойства кабелей и экранирований (например, ОУ) можно так же как и для других устройств, просматривать по символу в схеме соединений. При этом можно установить четыре разных угла.

Для поля Порядок свойств поставляются определенные конфигурации, из которых (на вкладке Отображение диалогового окна свойств) можно выбрать нужное расположение. Эти конфигурации расположения воздействуют на размещение ОУ и на все присоединенные к нему другие тексты свойств.

При выборе такой предварительно определенной конфигурации расположения расположение текстов свойств сохраняется и после изменения удлинения линии определения кабеля или экранирования. Положение текстов свойств каждый раз рассчитывается заново.

От положений текстов свойств зависит и выравнивание текста. Т. е. при "Слева, Вверху" текст вытягивается слева и вверх.

Позиционирование и форматирование свойств можно настроить и произвольно (определенный пользователем порядок свойств). В этом случае позиция также всегда задается относительно точки вставки. При смещении точки вставки линии определения кабеля (или экранирования) смещаются и отображаемые свойства.

!!! example "Пример:"

    Начерчено экранирование и открыто диалоговое окно "Cвойства". На вкладке Отображениев поле Порядок свойствпредлагаются различные предварительно определенные конфигурации. Выберите конфигурацию Обозначение экранирования, справа сверху, 0°. Обозначение экранирования будет выведено справа сверху рядом с экранированием.

### Альтернативное поперечное сечение / диаметр

Для кабелей и соединений можно задать альтернативное поперечное сечение / диаметр, а также соответствующую альтернативную единицу. Эти свойства независимы от базы данных изделий, их следует заполняться вручную, отчет также должен генерироваться вручную. С помощью этих свойств поперечное сечение / диаметр соединения можно указать одновременно в различных единицах (например, AWG и мм²). Свойства доступны в диалоговом окне "Cвойства" для кабелей и соединений и могут использовать в отчетах.

Для кабелей:

* Кабель / Группа соединений: Альтернативное поперечное сечение / диаметр
* Кабель / Группа соединений: Единица альтерн. поперечного сечения / диаметра
* Кабель / Группа соединений: Альтернат. поперечное сечение/диаметр с единицей измерения.

Для соединений:

* Альтернат. поперечное сечение / диаметр соединения
* Альтернат. поперечное сечение соединения / диаметр с единицей измерения
* Единица измерения альтернат. поперечное сечение/диаметр соединения.

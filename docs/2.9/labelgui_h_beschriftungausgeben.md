# Вывести данные изготовления / маркировку

Условия:

* Вы открыли проект.
* Схема экспорта / маркировки создана.
* Заданы настройки вывода *.txt.
* Определены верхний колонтитул, область этикетки и нижний колонтитул.

1. Страница > Навигатор
2. Выделите проект, страницы, функции или слои структурного идентификатора.
3. Выберите пункты меню Сервисные программы > Данные изготовления > Экспорт / маркировка.
4. Выберите схему из раскрывающегося списка Настройки.

!!! info "Для сведения:"

    В поле Тип отчета выводится тот тип отчета, на котором основана схема.

5. Выберите из раскрывающегося списка Язык тот язык, на котором должен быть выведен текст.
6. Щелкните рядом с полем Целевой файл кнопку ++...++ и в открывшемся диалоговом окне укажите место хранения и имя файла. Щелкните по кнопке ++Открыть++.
7. Укажите в поле Повторений на этикетку, сколько раз текст каждой этикетки должен быть выведен в файле.
8. Укажите в поле Повторений на полный вывод, сколько раз весь текст должен быть выведен в файле.
9. Для открытия файла во внешнем приложении выберите опцию Экспортировать и запустить приложение.
10. Щелкните по кнопке ++OK++.

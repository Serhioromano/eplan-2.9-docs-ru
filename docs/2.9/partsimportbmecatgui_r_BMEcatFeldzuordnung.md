# Вкладка Присвоение поля BMEcat

Сервисные программы > Изделие > Управление > [Дополнительно] > Импортировать. В поле Тип файла выбрана запись для версии eCl@ss, а в поле Имя файла выбран XML-файл с наборами данных изделий и нажата кнопка ++OK++. В диалоговом окне Импорт eCl@ss нажмите кнопку ++...++ рядом с полем Настройки и выберите вкладку Присвоение поля BMEcat.

Здесь свойствам изделия базы данных изделий можно присвоить несколько фиксированных полей BMEcat, не имеющих признака eCl@ss или отдельно сохраненных в поле BMEcat. Определенные пользователем свойства и произвольные свойства недоступны для выбора, т. к. отсутствует (многоязычное) имя признака eCl@ss. Для импорта таких данных предусмотрена специальная обработка нескольких фиксированных полей BMEcat.

Обзор основных элементов диалогового окна:

### Поле BMEcat

Содержимое этих полей импортируется из файла BMEcat и ***не подлежит*** изменениям.

### Свойство изделия

Щелкните по кнопке ++...++ в этих полях и в диалоговом окне Выбор свойств выберите свойство, которое необходимо присвоить полю BMEcat.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

# Диалоговое окно Синхронизировать данные проводного монтажа

Открыты проект и навигатор проводного монтажа. Данные проекта > Соединения > Импортировать > Данные проводного монтажа. В диалоговом окне Импортировать данные проводного монтажа выбран файл (*.hpdex) с данными проводного монтажа. Щелкните по кнопке ++Открыть++.

В этом диалоговом окне данные проводного монтажа (функции, соединения), импортируемые из источника данных, отображаются в таблице и сравниваются с уже существующими в проекте данными проводного монтажа. Объем отображения можно ограничить, установив фильтры. В зависимости от отображаемых сведений о статусе определяется операция, которую необходимо выполнить с данными проводного монтажа.

Обзор основных элементов диалогового окна:

На вкладке Функции сравниваются функции из источника данных и из проекта.

На вкладке Соединения сравниваются соединения из источника данных и из проекта.

### Фильтр

В этом раскрывающемся списке отображаются все доступные фильтры. Выбранный фильтр активируется автоматически. Запись "- Не активировано -" отключает фильтр и приводит к тому, что данные отображаются в неотфильтрованном виде. С помощью кнопки ++...++ откройте диалоговое окно [Фильтр](modaldialogsdb_d_filternnach.md). Здесь можно создать, обработать, удалить, скопировать, экспортировать, импортировать фильтр и управлять им.

Во всплывающем меню раскрывающегося списка Фильтр содержатся следующие записи:

* Выключить: Этот пункт меню доступен, если фильтр установлен: Сбрасывает настройку фильтра до записи "- Не активировано -".
* Активировать <Имя фильтра>: Этот пункт меню доступен, если для настройки фильтра установлено значение "- Не активировано -": Повторно активирует последний активный фильтр.

Таким образом можно быстро переключаться между неотфильтрованным и отфильтрованным в соответствии с требованиями пользователя представлениями.

### Значение

При помощи [быстрого ввода](modaldialogsdb_k_filter.md) в данном поле для определенного и активированного фильтра можно быстро изменить значение его критерия.

!!! note "Замечание:"

    Обратите внимание на то, что активация фильтра ***не*** влияет на импорт. Всегда импортируется полный файл согласно заданным действиям. Поэтому перед импортом рекомендуется отключить все фильтры и проверить заданные в таблице операции.

### Таблица

В таблице отображается содержимое выбранного файла импорта. Заголовки столбцов соответствуют присвоенным свойствам EPLAN. В ***левой области таблицы*** рядом с фиксированными столбцами Статус и Операция отображаются данные проводного монтажа из источника данных, в правой области — данные из проекта EPLAN. Всегда выводятся следующие столбцы:

* Строка: Показывает порядковый номер строки.
* Статус: В этом столбце с помощью пиктограммы отображается, имеются ли данные проводного монтажа в проекте или нет, удалялись ли они в источнике данных, а также имеют ли эти данные в источнике данных и в проекте различные состояния обработки (например, другие значения свойств).
* Операция: Укажите здесь, какая операция должна быть выполнена с данными проводного монтажа в зависимости от отображаемого статуса.

Статус| Значение| Возможное действие
---|---|---
![](images/all_statusnew_as.png){: .ui-icon }| Создать: Функция / соединение в источнике данных еще отсутствует в проекте.| Импортировать или Нет
![](images/all_statusok_as.png){: .ui-icon }| Идентичный: Функция / соединение в источнике данных уже содержится в проекте и не будет импортироваться еще раз. Все значения свойств идентичны.| Нет
![](images/all_statusadditional_as.png){: .ui-icon }| Изменено: Функция / соединение содержится в источнике данных и в проекте. Одно или несколько значений свойств отличаются, например номер изделия.

!!! note "Замечание:"

    * Поля с измененными значениями свойств подкрашены темно-серым цветом. Всплывающая подсказка для такого поля показывает соответствующее значение в проекте.
    * Если функция / соединение обозначена как "Выделено для удаления", станет также доступной операция Удалить. Эта операция позволяет удалить функцию / соединение в проекте.

| Изменить или Нет
![](images/erroricon_error_systemmessages.png){: .ui-icon }| Недействительно: Невозможно присвоить функцию / соединение (напр., из-за отсутствия источника и цели соединения).| Нет

В ***правой области таблицы*** для данных из проекта EPLAN рядом со столбцами Статус и Операция отображаются следующие столбцы:

* Имя проводного монтажа: Отображает идентифицирующее имя проводного монтажа.
* ОУ (полное): Отображает полное ОУ функции / соединения.
* Выделено для удаления: Отображает, отмечена ли функция / соединение для удаления.

Объем отображаемых здесь столбцов можно расширить или сократить при помощи пункта всплывающего меню Конфигурировать представление.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

### Фильтр (статус)

Здесь можно дополнительно к установленному выше фильтру фильтровать импортируемые данные проводного монтажа по сведениям об их статусе. Доступны следующие значения:

* Все: Показываются все данные проводного монтажа из файла импорта.
* Измененные: Показываются имеющие отличия данные проводного монтажа, которые содержатся как в источнике данных, так и в проекте.
* Удаленные: Показываются данные проводного монтажа, которые содержатся только в проекте и были удалены в источнике данных.
* Идентичные: Показываются идентичные данные проводного монтажа, которые содержатся как в источнике данных, так и в проекте.
* Новые: Показываются только те данные проводного монтажа, которые еще отсутствуют в проекте.

### Отображать только релевантные столбцы

С помощью этого флажка можно значительно уменьшить число отображенных столбцов. Если соответствующий флажок установлен, в таблице для функций / соединений будут отображаться только самые важные столбцы (ОУ (полное), Статус, Операция, столбцы с измененными данными и т. д.).

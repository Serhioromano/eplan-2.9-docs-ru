# ПЛК

EPLAN оказывает поддержку при проектировании блоков управления ПЛК и систем шин и управлении ими. При этом возможен обмен данными с различными программами конфигурации ПЛК. Имеются следующие возможности обработки:

* Всю информацию ПЛК проекта можно показывать и обрабатывать в диалоговом окне.
* Информацию ПЛК можно представлять на страницах обзора.
* Данные ПЛК можно обрабатывать как на странице схемы соединений, так и на странице обзора, в навигаторе ПЛК или в навигаторе устройств.
* Можно адресовать новые выводы устройства ПЛК.
* Можно раздельно управлять несколькими блоками управления ПЛК.
* С помощью функции экспорта и импорта можно обмениваться данными ПЛК (списками присвоений) с различными программами конфигурации ПЛК.
* Данными конфигурации блоков управления ПЛК систем шин можно обмениваться с различными программами конфигурации ПЛК.

# Диалоговое окно Вывод: Список удаленных соединений

Сервисные программы > Данные изготовления > Сборка проводов > Список удаленных соединений

В этом диалоговом окне задаются настройки для вывода списка удаленных соединений.

Список удаленных соединений — это список всех ранее созданных соединений, которые однако были удалены в ходе обработки проекта. Следовательно, эти соединения когда-то содержались в проекте, затем были экспортированы и после этого удалены из проекта. Генерируется список, разделенный табуляторами.

Условие:

Должен был проведен экспорт соединений.

Обзор основных элементов диалогового окна:

### Целевой файл

Поле ввода для имени генерируемого файла со списком удаленных соединений. При нажатии кнопки ++...++ открывается диалоговое окно Открыть, в котором можно выбрать уже имеющийся файл типа Файл в формате Юникод с разд. табуляцией (*.txt) или ввести новое имя.

### Только удаленные соединения

Возможно также, что после экспорта в машину соединения не были удалены, но изменилась их длина. Если этот флажок установлен, в списке удаленных соединений выводятся только действительно удаленные соединения. Соединения, у которых изменилась длина, подавляются.

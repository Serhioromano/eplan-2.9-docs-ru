# Диалоговое окно Формат (online-нумерация)

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация (online). В диалоговом окне Настройки: Нумерация (online) рядом с полем Формат нумерации щелкните по кнопке ++...++. Выберите в диалоговом окне Форматы нумерации определенную пользователем схему. Щелкните на вкладке ОУ или Обозначения по кнопке ++...++. Дважды щелкните по элементу формата в диалоговом окне Формат ОУ.

В этом диалоговом окне можно обработать элементы формата, являющиеся частью формата нумерации.

Обзор основных элементов диалогового окна:

### Минимальное число разрядов

Значение в этом поле указывает минимальное число разрядов при выводе.

### Заполнитель

Отсутствующие места заполняются этим символом.

### Начальное значение

(Только для элементов формата типа "Счетчик")

Здесь вводится значение, с которого должна начинаться нумерация устройств. Разрешенные значения: 0-X.

### Использовать алфавитно-числов. счетчик

(Только для элементов формата типа "Счетчик")

Если установлен этот флажок, применяется алфавитно-числовой счетчик. Если флажок снят, применяется числовой счетчик.

### Алфавитно-числовой счетчик

(Только для элементов формата типа "Счетчик")

При использовании алфавитно-числового счетчика здесь необходимо определить символы, которые будут использоваться для счетчика. Разрешается использовать только буквы и цифры.

Алфавитно-числовые счетчики сортируются по символам слева направа (как в столбцах в Excel). Если, например, предварительно заданы символы "AKS", то имеет место следующий счет: A, K, S, AA, AK, AS, KA, ..., AAA, AAK, AAS, AKA ..., AAAA, AAAK, ...

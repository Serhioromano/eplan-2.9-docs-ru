# Операция: exportProductionWiring

**Параметр** |  **Описание**
---|---
MachineType  |  Целочисленное значение, необходимо указать. Доступны следующие значения:

* 1 = Komax
* 2 = CadCabel
* 3 = Schleuniger
* 4 = Steinhauer PWA
* 5 = Metzner
* 7 = Общая спецификация проводов

MachineName  |  Символьная строка, необходимо указать. Имя машины должно соответствовать имени, заданному в настройках.
PROJECTNAME  |  Имя проекта с полным путем файла (является опцией).
Если не задано, то выбранный проект используется, когда операция вызывается через интерфейс пользователя (напр., через сценарий или панель инструментов). При вызове из командной строки Windows следует определить PROJECTNAME, или сначала следует использовать ProjectAction. В противном случае отобразится системное сообщение.
USESELECTION  |  Устанавливает, необходимо ли учитывать текущий выбор в интерфейсе пользователя при выводе данных провода (необязательно, 0 = Нет, 1 = Да).
Значение по умолчанию: 0. Доступно для использования, только когда не используется параметр PROJECTNAME.
TargetDirectory  |  Целевой каталог (является опцией). Если параметр задан, перезаписывается заданный в настройках путь файла.
FileName  |  Имя файла (является опцией). Если параметр задан, перезаписывается заданное в настройках имя файла.
LANGUAGE  |  Язык (например, en_US), необязательно.
MultipleConnections  |  Если значение = 1, повторно экспортируются уже экспортированные провода. Если значение = 0, выводятся только те провода, которые еще не экспортировались. Предварительная настройка: 0.

!!! example "Пример:"

    ExportProductionWiring
    /MACHINETYPE:5
    /MACHINENAME:Metzner
    /TARGETDIRECTORY:D:export
    /MultipleConnections:1
    /UseSelection:0

# Импортировать пространство конструкции

***EPLAN Design Space Exchange*** используется в 3 следующих рабочих этапа.

* Импорт пространств конструкции в качестве файлов STEP из программ 3D-CAD.
* Создание чертежа монтажных поверхностей в EPLAN Pro Panel.
* Экспорт расширенного за счет чертежа монтажных поверхностей и данных ЧУ пространства конструкции в программу 3D-CAD.

Далее описывается первый рабочий этап — импорт файлов STEP.

Настройки для импорта STEP выполняются в Параметры > Настройки > Пользователь > Интерфейсы > Импорт и экспорт пространства конструкции.

Условие:

Вы открыли проект типа "Проект схемы соединений". Открыт навигатор пространства листа.

1. Выберите Пространство листа > Пространство конструкции > Импортировать.
2. Выберите в диалоговом окне Открыть файл STEP, генерированный программой 3D-CAD.
3. Выберите в диалоговом окне Импорт пространства конструкции схему с настройками для импорта выбранного исходного файла или откройте с помощью кнопки ++...++ диалоговое окно настроек, чтобы изменить настройки.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Данные 3D-графики импортируются из файла STEP в отдельное пространство листа. Импортированная геометрия защищена от изменений.

5. Создайте в импортированном пространстве конструкции чертеж монтажных поверхностей с функциональностью EPLAN Pro Panel.

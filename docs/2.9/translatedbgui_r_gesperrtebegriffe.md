# Вкладка Автокоррекция

Сервисные программы > Перевод > Обработать словарь. Выберите вкладку Автокоррекция.

На этой вкладке вы можете вести список слов, которые не должны использоваться на страницах проекта и будут автоматически заменяться на другие. Таким образом обеспечивается терминологическое единобразие в рамках проекта.

Для каждого слова, которое не должно использоваться, необходимо ввести текст замены. При переводе нежелательные слова будут заменяться введенными вами текстами замены. Автоматически будут заменяться только слова исходного языка. Нежелательные слова на других языках отображения заменяться не будут. В настройках перевода во вкладке Пользователь должен быть установлен флажок Автокоррекция.

Обзор основных элементов диалогового окна:

### Пoиск

Используя это поле, можно выполнять поиск всех ключевых слов, которые уже включены в список ключевых слов для автокоррекции. Чтобы найти нужное ключевое слово, введите его в поле ввода. Чтобы отобразить текстзамены для ключевого слова, выделите соответствующее ключевое слово в списке ключевых слов. Ключевое слово отображается в поле Заменить. Текст замены отображается в поле На.

### Заменить

В этом поле изменяется ключевое слово, которое будет заменяться другим. Используя панель инструментов, можно создать, сохранить, скопировать или удалить ключевое слово для автокоррекции.

# Раздельное представление клемм

Выводы устройств клемм можно представлять раздельно с помощью распределенных клемм. Это относится как к клеммам потенциала, так и к общим клеммам. Таким образом можно, например, вставить выводы устройств клемм потенциала прямо туда, где они нужны в проекте. Это облегчает читаемость схемы соединений, так как выводы устройств клемм больше не требуется искать через точки разрыва.

* Для любой распределенной клеммы можно представить один или несколько выводов устройств клемм. Число выводов устройств определяется посредством выбора определений функций. Количество изображенных выводов устройства (от 0 до n) любой функции клеммы с условным обозначением может быть любым и зависит от заданных обозначений выводов устройства.
* Клеммные цели анализируются только на выводах устройств, имеющих обозначение выводов устройств. Поэтому имеет значение, у какого вывода устройства какое обозначение введено. Выводы мостовых перемычек и выводы устройств внутренних перемычек не требуют обозначения выводов устройств.
* Распределенные клеммы относятся к той же функции клемм, если они имеют одинаковое ***определение функции***, одинаковое идентифицирующее ОУ, одинаковое ***обозначение клеммы*** и одинаковый ***номер уровня***. Распределенные клеммы для функций клемм накладываются на все такие же шаблоны функций.
* При необходимости распределенные клеммы дополнительно различаются посредством свойства Индекс распределенной клеммы. Это необходимо, если нужно одинаково обозначить несколько отдельно представленных клемм (например, если клеммы внутри одного сигнала должны иметь одинаковое обозначение). Если распределенные клеммы имеют одинаковое обозначение, они должны различаться по свойству Индекс распределенной клеммы. Распределенные клеммы с одинаковым индексом распределенных клемм распознаются как взаимосвязанные. Значение для свойства Индекс распределенной клеммы может быть введено вручную или задано автоматически при оптимизации распределенных клемм.
* В диалоговом окне Обработать клеммник родственные распределенные клеммы объединяются и отображаются как ***одна клемма***.
* В навигаторах распределенные клеммы обозначаются специальной пиктограммой (![](images/navigatorgui_partialterminal_as.png){: .ui-icon }).
* При копировании и вставке распределенных клемм свободные шаблоны функций не ищутся и имеющиеся не используются, а распределенные клеммы автоматически нумеруются в соответствии с настройками Online-режима нумерации.

### Общая обработка

Данные, введенные на распределенных клеммах той же функции клемм, объединяются. При раздельном представлении многоуровневых клемм выполняется обобщение данных по каждому уровню. Если общие свойства распределенных клемм содержат различные данные, то свойства при общей обработке и составлении отчетов учитываются в следующей последовательности:

* Учитываются данные распределенных клемм с наименьшим обозначением выводов устройств.
* Распределенные клеммы сначала объединяются по виду представления. Затем объединяются различные виды представлений функций клемм (в соответствии с [последовательностью видов представлений при общей обработке](reverseengineering_k_reihenfolge.md)).

#### Обратная запись свойств выводов устройства

Если обозначения и / или описания выводов устройств при общей обработке изменяются, то новые значения перезаписываются на всех распределенных клеммах. Свойства выводов устройств при этом перезаписываются на исходные места, т. е. обновляются уже заполненные поля, но в пустые поля значения не вводятся.

### Раздельное представление клемм потенциала

Клеммы потенциала используются, чтобы применить потенциал. В большинстве случаев речь идет о многоуровневых клеммах, уровни которых связаны друг с другом и которые можно произвольно присвоить какому-либо потенциалу. Уровни потенциальных клемм не проектируются в EPLAN: потенциальная клемма состоит из одной функции клеммы или шаблона функции с n выводами устройства и уровнем "0".

Если необходимо раздельно представить соединенные друг с другом выводы устройства клемм потенциала, следует использовать распределенные клеммы. Потенциальная клемма при этом изображается через следующие одна над другой распределенные клеммы с одинаковым обозначением клемм и различными обозначениями выводов устройств.

!!! example "Пример:"

    Потенциальная клеммаX1:1, все выводы устройств которой соединены один под другим, должна быть представлена раздельно. Для этого в схему соединений вставляются восемь распределенных клемм:X1:1:a[0],X1:1:b[0],X1:1:c[0],X1:1:d[0],X1:1:e[0],X1:1:f[0],X1:1:g[0]иX1:1:h[0]. ([0]обозначает уровень; он всегда равен 0, так как существует только один шаблон с функцией клеммы с восемью выводами устройств и информацией об уровне. На выводе устройства информация об уровне отсутствует.)Для простоты предположим, что логическая последовательность соответствует графической последовательности. В графическом редакторе раздельное представление с обозначенными соединениями будет выглядеть следующим образом:Если установлена настройка проекта Автоматически связать родственные распределенные клеммы, то при обновлении соединений распределенные клеммы соединяются друг под другом, даже если между ними нет обозначенных соединений. (Автоматически сгенерированныесоединенияздесь представлены дугами зеленого цвета.)

### Отчет по распределенным клеммам

В отчетах распределенные клеммы одной функции клемм объединяются и выводятся свойства объединенных клемм.

### Перекрестные ссылки между распределенными клеммами

Чтобы легче находить родственные распределенные клеммы, можно отобразить перекрестные ссылки между этими распределенными клеммами с помощью свойства Перекрестная ссылка распределенной клеммы (Ид. 20252).

Перекрестная ссылка распределенной клеммы ссылается от первой распределенной клеммы на все остальные распределенные клеммы той же функции клеммы. Первая распределенная клемма — это главная клемма либо, если главная клемма отсутствует, распределенная клемма с наименьшим не пустым обозначением вывода устройства. На остальных распределенных клеммах эта перекрестная ссылка указывает на первую распределенную клемму.

Перекрестная ссылка распределенной клеммы по умолчанию отображается на отдельном слое EPLAN444, Размещение свойства.Перекрестная ссылка распределенной клеммы и может быть скрыта или показана с помощью соответствующих настроек слоя.

### Клеммы с переменным числом выводов устройства

С помощью определения функции "Клемма, перем." можно определить клеммы с произвольным числом выводов устройства. Таким способом можно проектировать, например, ***клеммы распределителя***. Если эти клеммы необходимо представить как распределенные клеммы, выводы устройства распределенных клемм должны быть связаны между собой. Это происходит автоматически, если активирована настройка Автоматически связать родственные распределенные клеммы (в диалоговом окне [Настройки: Распределенные клеммы](terminalgui_d_projekteinstellungenklemmen.md)).

Чтобы равномерно перенести потенциал на все выводы устройства, в логической схеме выводов устройства клеммы необходимо заполнить свойство Передать потенциал в для первого и последнего выводов устройства. Здесь для первого вывода устройства вводится значение последнего вывода устройства, а для последнего — первого.

!!! example "Пример:"

    Определена клемма с 20 выводами устройства. Для этого на вкладке Данные символа / функциидиалогового окна свойств выбрано определение функции "Клемма, перем.", а в поле Выводы устройствауказано значение20. В диалоговом окне Логическая схема выводов устройствав свойство Передать потенциал вдля первого вывода устройства введите значение20, а для последнего —1.

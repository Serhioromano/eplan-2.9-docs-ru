# Операция: SpExportSubProject

**Параметр** |  **Описание**
---|---
SPNAME  |  Имя частичного проекта.
PROJECTNAME  |  Имя проекта с полным путем файла (является опцией).
Если не задано, то выбранный проект используется, когда операция вызывается через интерфейс пользователя (напр., через сценарий или панель инструментов). При вызове из командной строки Windows следует определить PROJECTNAME или сначала следует использовать ProjectAction. В противном случае отобразится системное сообщение.
DBID  |  Идентификатор исходного проекта (необязательно).
Без ввода используется параметр PROJECTNAME.
SPNR  |  Номер частичного проекта.
AREA  |  Имя области обработки.
AREAPATH  |  Настройка пути файла для областей обработки.
EXPANDONLY  |  Только расширить частичный проект. 0 = Нет, 1 = Да. В случае выбора "Да" объекты (страницы, пространство листа, сегменты предварительного планирования и т. д.) сгенерированного частичного проекта получают защиту от изменений и могут быть только расширены (удаление или изменение невозможно).

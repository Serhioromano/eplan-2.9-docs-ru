# Автоматическое указание информации о странице

Свойство формы Формат для автоматического описания страниц делает возможным задать информацию страниц (имя страницы, описание страницы, установка и др.) для автоматически распределяемого при создании отчета описания страниц.

Условия:

* Вы открыли проект.
* В редакторе форм открыта та форма, свойства которой требуется обработать (Сервисные программы > Основные данные > Форма > Открыть > [Открыть]).
* Открыто диалоговое окно Свойства формы — <Имя формы> (Всплывающее меню > Свойства на форме в диалоговом окне навигатора Страницы — <Имя проекта>).

1. В диалоговом окне Свойства формы — <Имя формы> щелкните ++...++ в столбце Значение свойства Формат для автоматического описания страниц.
2. В диалоговом окне Формат для автоматического описания страниц выберите из списка Доступные элементы формата элемент, который должен использоваться при автоматическом описании страниц. (Многократный выбор здесь ***невозможен***.)
3. Щелкните по ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо).

!!! info "Для сведения:"

    Элемент формата копируется в список Выбранные элементы формата.

4. Выполните аналогичные действия для всех последующих элементов формата.
5. Щелкните по ![](images/all_arrowup_as.png){: .ui-icon } (Переместить вверх) / ![](images/all_arrowdown_as.png){: .ui-icon } (Переместить вниз), чтобы переместить выделенный формат элемента, если возможно, вверх или вниз. (Выбор сразу нескольких объектов здесь ***невозможен***, а элемент формата типа Разделитель заголовков***не может*** перемещаться, так как он всегда автоматически вставляется в конец списка.)
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выполняется сохранение элементов формата.

В диалоговом окне Свойства формы — <Имя формы> элементы формата выводятся в столбце Значение следующим образом:

<Номер доступного элемента формата>;<Число / Длина / Ид. свойства>|<Номер доступного элемента формата>;<Число / Длина / Ид. свойства>|...,
где номера присвоены следующим образом:

* Тип страницы / формата: 1
* Свойства формы: 9
* Имя объекта заголовка: 2
* Другие свойства объекта заголовка: 10
* Имя первого объекта данных: 3
* Имя последнего объекта данных: 4
* Разделитель: 5
* Разделитель заголовков: 11\.

!!! example "Пример:"

    2;1|9;<18000,0>|5;4для:Имя объекта заголовка, число / длина:1Свойства формы, свойство Имя (форма, рамка, контур, профиль)(ид.18000, индекс 0)Разделитель, число / длина:4

# Многопользовательский монитор: Способ действий

Информация, предоставляемая многопользовательским монитором, позволит уменьшить проблемы в работе с использованием многопользовательского режима. При этом вы можете получать информацию в диалоговом окне до начала или во время выполнения операции.

Чтобы открыть многопользовательский монитор, выберите Проект > Многопользовательский монитор. Ход выполнения текущих запущенных операций будет отображен в столбце Успешно посредством индикатора выполнения с указанием процента.

### До выполнения операции многопользовательский монитор предоставит следующую информацию

Если вы хотите запустить длительную операцию в многопользовательском режиме для открытого проекта (например, контрольный прогон для всего проекта, оценку всего проекта и т. д.). Прежде всего монитор предоставит вам информацию о том, запускалась ли длительная операция для текущего проекта другим пользователем.

Если не запускалась, вы можете запустить требуемую операцию.

При выполнении другой длительной операции по текущему проекту вы можете:

* Подтвердить более продолжительное время выполнения и запустить свою операцию.
* Отложить выполнение своей операции на более позднее время.
* Связаться с другим пользователем для достижения общей договоренности. Для этого можно, например, щелкнуть идентификатор другого пользователя в столбце Пользователь и отправить электронное сообщение.

### Во время выполнения операции многопользовательский монитор предоставит следующую информацию

Вы запускаете длительную операцию. Поскольку выполнение операции требует больше времени, чем обычно, откройте многопользовательский монитор. Здесь вы получите информацию о том, вызвана ли задержка использованием многопользовательского режима.

Если да, то вам доступны следующие возможности:

* Подтвердить более продолжительное время выполнения и подождать завершения выполнения вашей операции.
* Остановить операцию, нажав ++Отменить++. Затем вы можете (как упоминалось ранее) связаться с другим пользователем для достижения общей договоренности.

# Создание схемы сравнения свойств

В схеме сравнения свойств устанавливается, какие свойства сравниваются друг с другом при сравнении проектов в ходе проверки проекта.

Условия:

* [Выбрать базу данных управления проектами](prjmanagementgui_h_projekteverwalten.md)
* [Считать каталоги с проектами](prjmanagementgui_h_verzeichniseinlesen.md)

1. Проект > Управление
2. Выделите проект.
3. Выберите ++Дополнительно++ > Проверить проект.
4. Рядом с флажком Сравнить свойства щелкните ++...++.
5. Щелкните рядом с полем Схема кнопку ![](images/all_new_as.png){: .ui-icon }(Создать).
6. В диалоговом окне Новая схема в поле Имя введите информативное имя для новой схемы и нажмите кнопку ++OK++.

!!! info "Для сведения:"

    Во вкладках отображаются свойства групп данных проекта "Устройство", "Соединения", "Графика", "Страницы", "Потенциалы" и "Проект".

7. Выберите вкладку какой-либо группы данных проекта.
8. Чтобы выбрать свойства для сравнения свойств, выделите одно или несколько свойств в поле Не сравнивать и нажмите на ![](images/all_arrowright_as.png){: .ui-icon }(Переместить вправо).
9. Щелкните по кнопке ++Сохранить++ и по кнопке ++OK++.

# Диалоговое окно Создать проект

* Проект > Создать.
* Вы выбрали базу данных управления проектами. Вы считали каталоги и проекты. Проект > Управление. Выделите в структуре дерева каталог. Выберите пункт всплывающего меню Создать.

В этом диалоговом окне с помощью шаблона проекта/базового проекта нового проекта.

Обзор основных элементов диалогового окна:

### Имя проекта

Введите в этом поле имя проекта без расширения имени файла. Имя проекта может содержать специальные символы и умлауты.

### Место хранения

В этом поле находится каталог, в котором вы хотите сохранить новый проект. С помощью кнопки ++...++ откройте диалоговое окно выбора каталога. В данном диалоговом окне вы можете выбрать другой каталог в качестве места хранения для проекта. Вы можете использовать любой каталог на любом доступном дисководе.

### Шаблон

Отображается шаблон, который используется для нового проекта. В качестве шаблона можно выбрать шаблоны проекта или базовые проекты. С помощью кнопки ++...++ откройте диалоговое окно Выбрать базовый проект / шаблон проекта. В этом диалоговом окне вы можете выбрать шаблон.

### Указать дату создания

Если установлен флажок, для нового проекта можно задать другую дату создания в расположенном ниже поле. В этом случае сначала вносится текущая дата. При необходимости данную дату можно изменить.

Если флажок не установлен, используется дата создания выбранного шаблона.

### Указать автора

Если флажок установлен, в расположенном ниже поле в зависимости от настроек в пунктах Пользователь > Отображение > Ид. пользователя / адрес автоматически вводится ваше имя регистрации или идентификатор пользователя. Если Вы не хотите использовать имя при регистрации или код пользователя, можно ввести другое имя.

Если флажок не установлен, отображается автор, создавший шаблон. Если при этом в поле отсутствует запись, для шаблона автор не задается.

!!! tip "Совет:"

    При помощи приведенной выше функции — флажков Указать дату создания и Указать автора — для нового проекта можно ввести имя текущего автора и текущую дату создания и изменить их при необходимости.

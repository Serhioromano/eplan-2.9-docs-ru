# Работать с сетками

Для упрощения работы при размещении элементов имеется возможность использования сетки. При это точки вставки и точки элементов позиционируются по точкам сетки. При работе в трехмерном пространстве листа используется линейный растр. В этом случае точки вставки лежат на точках дуг окружностей линий.

Шаг используемой сетки (т.е. интервал между отдельными точками сетки) сохраняется как свойство страницы и выводится в строке состояния. Из раскрывающегося списка можно выбрать один из шагов сетки по умолчанию или ввести собственное значение. В пространствах листа размер сетки не является свойством страницы.

В пользовательских настройках можно сохранить пять различных шагов сетки; их можно выбрать в графическом редакторе с помощью кнопок ![](images/gededitgui_grida.png) и ![](images/gededitgui_gride.png) на панели инструментов "Вид". Настройка возможна для 2D и 3D раздельно.

Условие:

Вы открыли страницу, форму, рамку или символ.

### Отобразить сетку

1. Чтобы включить или выключить сетку, выберите пункты меню Вид > Сетка.

!!! example "Пример:"

    Фрагментсхемы соединений: Сетка выключена Фрагментсхемысоединений: Сетка включена

### Использовать захват растра

Использование захвата сетки не зависит от ее отображения.

1. Чтобы включить или выключить захват сетки, выберите пункты меню Параметры > Захват сетки.

!!! info "Для сведения:"

    Если захват сетки активирован, в пределах точек сетки выполняются все следующие операции.

!!! info "Для сведения:"

    Наличие текста "ВКЛ." или "ВЫКЛ." перед значением шага сетки в строке состояния указывает, включен ли захват сетки.

### Выровнять по сетке

Эта функция используется для дополнительного выравнивания по настройкам сетки тех объектов, которые были начерчена без использования сетки. Если включить захват сетки позже, то, возможно, нельзя будет получить доступ к ранее начерченным объектам, так как они не находятся на точке сетки.

1. Выделите эти объекты и выберите пункты меню Обработать > Другое > Выровнять по сетке.

!!! info "Для сведения:"

    Выбранные объекты выравниваются заново (т. е. перемещаются) таким образом, чтобы все их важные точки (например, начальная точка линии, конечная точка дуги и т. д.) находились на сетке.

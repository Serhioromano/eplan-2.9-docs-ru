# Настройки

Чтобы подстроить программу под ваши требования и потребности, можно выполнить различные настройки:

* Можно указать свойства проекта, например, графическое представление электротехнических функций или заданных форматов для обозначений устройств.
* Можно индивидуально настроить рабочую среду и тем самым подстроить функциональность программы согласно принципу работы соответствующего пользователя.
* Можно указать настройки, относящиеся к станции, такие как путь файла, для управления системными сообщениями.
* Можно выполнить настройки с учетом потребностей предприятия, чтобы указать заданные значения и условия компании для всех используемых рабочих станций.

# Точки разрыва

С помощью точки разрыва проводится соединение, сеть, потенциал или сигнал, при этом в качестве имени точки разрыва может использоваться имя сигнала. При этом не проводится различие между точками разрыва для источников и для целей; источник и цель определяются автоматически.

Присоединения к точке разрыва могут подходить с обеих сторон. Если это не требуется, одно из соединений можно разорвать с помощью точки прерывания. Таким образом, наличие точки разрыва с открытым соединение ***не*** является ошибкой.

Кроме того, имеется таблица имен точек разрыва, в которой можно выбрать имя для вновь сгенерированной точки разрыва (диалоговое окно Использ. имеющ. точку разрыва).

Так же как и все символы соединений, управление точками разрыва осуществляется в библиотеке символов SPECIAL.slk; они влияют на прохождение соединение и не имеют определений функций.

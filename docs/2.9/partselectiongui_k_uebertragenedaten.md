# Обзор перенесенных данных изделия

При помощи выбора изделия устройству присваивается одно или несколько изделий. При этом перечисленные ниже данные ***первого*** изделия, входящего в выбор, переносятся на функцию / условное обозначение.

Далее переносятся следующие данные:

* Все свойства "Главного шаблона". Главный шаблон — это первый шаблон функции на вкладке Шаблоны функций базы данных изделий, в определении функции которого установлено "Главная функция".
* Присвоение изделий, настроенное в рамке макроса (если на изделии записан макрос и там сконфигурированы присвоения изделия).
* При выборе изделия значение из поля Технические параметры на вкладке Технические данные переносится, только если для этого свойства не было задано значение в "Главном шаблоне" или если нет доступных шаблонов функций.

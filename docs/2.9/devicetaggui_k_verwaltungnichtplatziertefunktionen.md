# Управление неразмещенных вспомогательных функций на главной функции

Для главных функций доступно свойство Неразмещенные вспомогательные функции управляются в главной функции. Если данное свойство установлено у главной функции, присвоенные неразмещенные вспомогательные функции управляются вместе с главной функцией. Т. е. при копировании и вставке главной функции неразмещенные вспомогательные функции копируются, вставляются и автоматически переименовываются аналогично характеристикам шаблонов функций. Если главная функция удалена, то удаляются все присвоенные неразмещенные вспомогательные функции. У вспомогательной функции данное свойство отображает значение соответствующей главной функции.
Таким образом можно, например, сохранить все функции устройства в макросе или копию, даже если в схеме соединений размещена только главная функция.

Свойство может использоваться как отдельно, так и в комбинации с защитой устройства.

### Характеристики при использовании и присвоении имеющегося

Характеристики неразмещенных вспомогательных функций при использовании и присвоении имеющихся зависят от того, установлено ли свойство Неразмещенные вспомогательные функции управляются в главной функции на исходной или целевой функции.

Активированное свойство Неразмещенные вспомогательные функции управляются в главной функции |  Характеристики при использовании и присвоении имеющегося
---|---
на исходной или целевой функции |  Исходная функция имеет при себе неразмещенные вспомогательные функции. Неразмещенные вспомогательные функции цели удаляются вместе с главной функцией.
только у исходной функции |  Исходная функция имеет при себе неразмещенные вспомогательные функции. Неразмещенные вспомогательные функции цели остаются.
только у целевой функции |  Исходная функция ***не*** имеет при себе неразмещенных вспомогательных функций. Неразмещенные вспомогательные функции цели удаляются вместе с главной функцией.

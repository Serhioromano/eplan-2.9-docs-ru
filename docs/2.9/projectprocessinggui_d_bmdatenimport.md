# Диалоговое окно "Импортировать данные устройств"

Данные проекта > Устройство > Импортировать

В этом диалоговом окне выполняется импорт данных из сторонних программ (другие системы CAE, системы ERP, Excel и т. д.) в проект EPLAN для генерирования на основе этих данных полной компоновки электрошкафа, включая маршрутизируемые соединения. Импортируются списки обозначений устройств / клемм с расширениями имен файлов TXT и CSV, а также текстовые файлы с расширением имени файла ASC.

Обзор основных элементов диалогового окна:

### Тип источника данных

В раскрывающемся списке предлагаются следующие типы файлов:

* Текст (*.txt, файл Юникод с разделением запятой (*.csv), файл ASCII (*.asc))
* файл Excel (*.xls, *.xlsx, *.xlsb, *.xlsm).

### Источник данных

В этом поле показано имя файла импортируемых списков обозначений устройств.

### Присвоения полей

Здесь вы выбираете схему импорта для импортируемых данных. По умолчанию установлена схема с названием "Стандарт". С помощью кнопки ++...++ откройте диалоговое окно Присвоение полей. Это диалоговое окно делает возможным присвоение внешнего поля данных из источника данных свойству EPLAN.

### Всплывающее меню

Внутри поля Источник данных можно вызвать всплывающее меню, которое имеет следующие пункты:

* Пункты всплывающего меню для обработки значений

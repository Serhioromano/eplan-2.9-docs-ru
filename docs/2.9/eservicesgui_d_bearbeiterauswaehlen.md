# Диалоговое окно Выбрать обработчик

Вы используете управление правами EPLAN и имеете право распределять области обработки в качестве администратора проекта. Вы открыли проект. Проект > Области обработки > Определить как администратор проекта. Для этого щелкните мышкой в строке необходимой области обработки по ячейке таблицы Все обработчики, а затем по кнопке ++...++.

В этом диалоговом окне можно присвоить одной области обработки несколько обработчиков.

Обзор основных элементов диалогового окна:

### Группа пользователей

В этом раскрывающемся списке отображаются все группы пользователей, которым в управлении присвоено право пользователя. При выборе записи в таблице Обработчики отображаются только пользователи, входящие в соответствующую группу.

### Таблица 'Обработчики'

В таблице отображаются свойства Имя и Описание пользователей, определенных в управлении правами. Дополнительно отображаются пользователи, которые были введены вручную, но не определены в управлении правами.

В столбце Присвоить установите флажок для обработчиков, которым разрешено редактировать текущую область обработки. Или снимите флажок напротив тех обработчиков, которым хотите запретить вносить изменения в текущую область обработки.

# Вкладка Точка захвата / вкладка Монтажная сетка

Вы открыли проект. Проект содержит как минимум одно пространство листа. Пространство листа содержит 3D-объекты. На 3D-объекте определена точка захвата или монтажная сетка. Выделите точку захвата или монтажную сетку и выберите пункты меню Всплывающее меню > Свойства. Или дважды щелкните на точке захвата или монтажной сетке.

На этой вкладке задаются свойства точки захвата / монтажной сетки.

Обзор основных элементов диалогового окна:

### Имя

Введите здесь имя точки захвата / монтажной сетки.

### Описание

Введите здесь описывающий текст для точки захвата / монтажной сетки.

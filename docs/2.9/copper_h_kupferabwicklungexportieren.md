# Экспортировать развертку шины

Для передачи данных по изготовлению на гибочные, сверлильные и фрезерные станки используется развертка медного функционального элемента. При этом учитывается также информация для изгиба на ребро. Чтобы иметь возможность разрезать в том числе и неизогнутые медные шины, неизогнутые медные шины также экспортируются с определением функции "Сборная шина".

Имеется два формата экспорта:

* Экспорт DXF, медь: Графический экспорт развертки в формате DXF
* Экспорт ЧУ, медь: Экспорт развертки в формате XML.

Для примера далее описан экспорт ЧУ меди.

Условия:

* Открыт проект, содержащий пространства листа с размещенными медными функциональными элементами.
* Таблица последовательности изгибов в настройках фирмы содержит свежие данные.
* Для медных функциональных элементов были сгенерированы развертки.

1. Выделите медный функциональный элемент, с которого сгенерированы развертка, в навигаторе пространств листа или в трехмерной области отображения.
2. Выберите пункты меню Сервисные программы > Данные изготовления > Механическая обработка > ЧУ, медь.
3. Выберите в диалоговом окне Экспорт ЧУ, медь определение станка, каталог вывода, целевой файл и материал.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Целевой файл будет сгенерирован в указанном каталоге. Теперь он может быть передан на станок.

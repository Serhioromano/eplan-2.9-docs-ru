# Вкладка Статистика

* Вы открыли проект. Страница > Навигатор. Выделите проект. Выберите пункт всплывающего меню Свойства. Выберите вкладку Статистика.
* Вы выбрали базу данных управления проектами. Вы считали каталоги и проекты. Проект > Управление. Выделите проект. Выберите вкладку Статистика.

На этой вкладке показывается, сколько страниц разных типов содержится в проекте.

Обзор основных элементов диалогового окна:

Таблица

### Тип страницы

Показывает все [типы страниц](pagebrowsergui_k_seitentypen.md), доступные в EPLAN.

### Число

Показывает количество страниц всех типов, содержащихся в проекте.

### Последнее изменение

Здеь указывается дата изменения количества страниц определенного типа.

### Необновляемые страницы

Показывает для каждого типа страниц количество [замороженных страниц отчета](formgeneratorgui_k_eingefrorseiten.md). В навигаторе страниц вы можете фильтровать замороженные страницы отчета.

* * *

### Индивидуально созданные страницы

Показывает количесиво страниц, которые вы создали в проекте.

### Сгенерированные страницы отчета

Показывает количество страниц, сгенерированных при составлении отчетов и находящихся в проекте.

### Общее число страниц

Показаны все данные, которые содержатся в проекте.

### Количество пространств листа

Показывает количество пространств листа, которые имеются в проекте.

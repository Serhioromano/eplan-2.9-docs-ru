# Включить автокоррекцию для текстов проекта

Функция "Автокоррекция" позволяет автоматически заменять тексты проекта текстами замены до начала перевода. Эта настройка специфична для пользователя. Укажите в словаре, какие ключевые слова на какой текст замены следует заменять автоматически.

Условие:

Выбран словарь, содержащий ключевые слова для автокоррекции.

1. Параметры > Настройки > Пользователь > Перевод > Словарь
2. Установите флажок Автокоррекция и щелкните по кнопке ++OK++.
3. Чтобы проверить автоматическую замену, откройте страницу схемы соединений, и выберите пункты меню Вставить > Графика > Текст.
4. Введите текст, который содержится в списке ключевых слов для автокоррекции, и выберите пункт Всплывающее меню > Перевести.

!!! info "Для сведения:"

    Ключевое слово автоматически заменяется и одновременно переводится, если в наличии имеется текст перевода.

# Выбор объектов для групповой обработки

Для выбора объектов для групповой обработки можно воспользоваться несколькими методами.

### Непосредственный выбор объектов

С помощью прямого выбора щелчком мыши для групповой обработки можно выбрать такие объекты, как проекты, страницы, данные проекта и условные обозначения.

### Выбор объектов одного типа

Условия:

* Вы открыли проект. В Графическом редакторе открыта страница проекта.
* На странице выделено несколько объектов.

1. Переместите курсор на требуемый объект.

!!! info "Для сведения:"

    Объект выделяется цветом.

2. Выберите пункт всплывающего меню Выделить объекты одного типа.

!!! info "Для сведения:"

    Все объекты выбранного типа выделяются цветом на текущей открытой странице, также для них выводятся точки вставки.

!!! example "Пример:"

    Страницасхемы соединенийсодержиткабели-W20,-W21и-W22. Для групповой обработки этих кабелей можно использовать опцию обработки объектов одного типа.Для этого переместите перекрестие на линиюопределения кабеля-W20. Кабель-W20выделяется цветом.Затем выберите пункт всплывающего меню Выделить объекты одного типа. Кабели-W20,-W21и-W22выделяются цветом, также выводятся их точки вставки. Теперь с помощью групповой обработки можно изменитьсвойствадля всех этих кабелей сразу.

### Выбор объектов в области

Помимо этого EPLAN позволяет выбирать объекты для групповой обработки в любой области страницы. Для этого в графическом редакторе с помощью мыши при нажатой левой клавише возьмите в рамку требуемые объекты.

#### Выбор объектов одного типа в области

В области, выделенной рамкой. также возможно выбрать объекты одного типа. Для это также воспользуйтесь пунктом всплывающего меню Выделить объекты одного типа.

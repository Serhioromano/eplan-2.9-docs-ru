# Синхронизировать данные проекта

Данные, хранящиеся в проекте, включают в себя такие данные, как слои, определенные пользователем свойства, порядок свойств и т. д., а также настройки проекта, схемы контрольных прогонов и шаблоны отчетов. Синхронизацию данных проекта можно использовать для простой переноса таких данных из исходного проекта (например, базового проекта) в другой.

Условие:

Вы открыли проект, данные которого необходимо синхронизировать.

### Синхронизировать отдельные данные проекта

1. Выберите следующие пункты меню: Проект > Организовать > Синхронизировать

!!! info "Для сведения:"

    В диалоговом окне Синхронизация проекта — <Имя проекта> выводятся исходный и целевой проект в виде двух списков. В списке Целевой проект слева отображаются данные проекта, который в данный момент открыт и выделен в навигаторе страниц, в который копируются данные проекта.

2. В поле Исходный проект выберите проект, из которого необходимо перенести данные.

!!! info "Для сведения:"

    При следующем открытии диалогового окна синхронизации предварительно устанавливается последний выбранный исходный проект.

3. В списке Исходный проект щелкните по заголовкам столбцов, чтобы отсортировать данные в алфавитно-цифровом виде по выбранному столбцу в порядке возрастания или убывания. Щелкните, например, по столбцу Тип, чтобы все записи были показаны отсортированными по типу данных проекта.
4. Выберите запись под списком Целевой проект в раскрывающемся списке Тип, чтобы уменьшить количество отображаемых и выбираемых данных проекта.
5. В списке Исходный проект выберите данные проекта, которые нужно перенести в целевой проект. Возможен многократный выбор.
6. Щелкните ![](images/all_arrowleft_as.png){: .ui-icon } (Копировать влево).
7. Щелкните по кнопке ++Закрыть++.

### Синхронизировать все данные проекта

1. Выберите следующие пункты меню: Проект > Организовать > Синхронизировать

!!! info "Для сведения:"

    В диалоговом окне Синхронизация проекта — <Имя проекта> выводятся исходный и целевой проект в виде двух списков. В списке Целевой проект слева отображаются данные проекта, который в данный момент открыт и выделен в навигаторе страниц, в который копируются данные проекта.

2. В поле Исходный проект выберите проект, из которого необходимо перенести данные.

!!! info "Для сведения:"

    При следующем открытии диалогового окна синхронизации предварительно устанавливается последний выбранный исходный проект.

3. Выберите ++Обновить++ > Обновить проект, чтобы заменить все устаревшие или отличающиеся данные целевого проекта данными исходного проекта.
4. Выберите ++Обновить++ > Дополнить проект, чтобы пополнить данные целевого проекта данными из исходного проекта.
5. Щелкните по кнопке ++Закрыть++.

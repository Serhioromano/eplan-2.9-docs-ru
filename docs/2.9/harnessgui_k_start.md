# Проводной монтаж

EPLAN оказывает поддержку при проектировании проводного монтажа и его компонентов, а также при управлении ими. С помощью функции импорта в EPLAN Harness proD можно импортировать созданный проводной монтаж и его компоненты в проекты EPLAN и там обрабатывать и анализировать их. Проводной монтаж и его компоненты можно снова экспортировать для повторной обработки в EPLAN Harness proD.

Возможно следующее:

* Созданный с помощью EPLAN Harness proD проводной монтаж можно импортировать в проект на платформе EPLAN.
* Проводной монтаж можно генерировать с предварительно заданными свойствами или вставить в качестве символа на схему соединений.
* Проводному монтажу можно присваивать другие функции, соединения или изделия, а также убирать из него имеющиеся присвоения.
* Изделия, внесенные на определения проводного монтажа и его компоненты, можно выводить в отчетах или использовать для экспорта данных изготовления / маркировки.
* Проводной монтаж можно экспортировать для последующей обработки в EPLAN Harness proD.

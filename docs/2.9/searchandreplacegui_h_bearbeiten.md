# Обработать записи для поиска

Условие:

Выполнен поиск, и открыто диалоговое окно Результаты поиска — <Имя проекта>.

1. Выделите запись в списке результатов поиска и выберите пункт всплывающего меню Свойства.
Возможен многократный выбор. В этом случае отображаются только свойства, общие для всех объектов.
2. Выполните в диалоговом окне Свойства нужные изменения свойств объекта.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Диалоговое окно Свойства закроется, и на экране снова появится список результатов поиска.

Обработка объектов схем соединений, данных проекта и страницы

Данные страницы, проекта и объекты схем соединений можно обрабатывать только по отдельности, не смешивая их.
Если вы сделали смешанный выбор, появится соответствующее сообщение, и вам нужно будет изменить выбор.

Многопользовательский режим

В многопользовательском режиме при вызове команды меню Свойства блокируются все объекты, которые должны быть обработаны. Если блокировка невозможна для всех объектов, эти объекты тоже нельзя обработать, и отображается соответствующее указание. В этом случае измените выбор.

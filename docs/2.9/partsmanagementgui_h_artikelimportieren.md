# Импортировать данные изделий

Условия:

* Вы открыли базу данных изделий: Сервисные программы > Изделие > Управление
* Через меню ++Дополнительно++ > Настройки в диалоговом окне База данных изделий была открыта база данных изделий.

1. Выберите ++Дополнительно++ > Импортировать.
2. Выберите требуемый тип файла из раскрывающегося списка Тип файла диалогового окна Импортировать записи данных.

!!! info "Для сведения:"

    В зависимости от установленного здесь формата импорта кнопка ++...++ становится активной, и в последующем диалоговом окне можно указать более подробную обработку при импорте (например, относительно файла конфигурации).

3. Введите в поле Имя файла имена для файла импорта или щелкните ++...++, чтобы выбрать файлы в интерактивном режиме (возможен многократный выбор).

!!! info "Для сведения:"

    Расширение имени файла добавляется автоматически в зависимости от типа файла, выбранного в поле Тип файла (например, .csv для всех типов файла 'CSV для ++...++ ').

4. Выберите из этого раскрывающегося списка Присвоения полей схему, которая должна использоваться для присвоения полей. Или щелкните по ++...++, чтобы в диалоговом окне Присвоение полей обработать имеющуюся схему или создать новую.
5. С помощью опций определите действия, которые следует выполнить при импорте с новыми и имеющимися записями данных.
6. Щелкните по кнопке ++OK++.

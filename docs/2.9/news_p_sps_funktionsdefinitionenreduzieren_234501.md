# Упрощение для выводов шины

В версии 2.9 до сих пор используются выводы шины с определениями функций "Кабельный вывод сети / шины, общее" и "Кабельный вывод сети / шины, вывод переключателя". Другие определения функций для выводов шины все еще поддерживаются в целях совместимости, но уже не могут быть вставлены в новые проекты.

![](images/benefit.png)Эффект:

Благодаря сокращенным определениям функций проектирование выводов шины становится проще и нагляднее. Все системы шин теперь проектируются с общими кабельными выводами сети / шины. Для пассивных устройств (например, клемм, контактов штекеров, пассивных переключателей, шинных повторителей) используются выводы переключателя.

Определения функций, которые с версии 2.9 стали ненужными, были обозначены дополнением "больше не использовать":

* "Кабельный вывод сети / шины, вход (больше не использовать)"
* "Кабельный вывод сети / шины, выход (больше не использовать)"
* "Кабельный вывод сети / шины, источник (больше не использовать)"
* "Кабельный вывод сети / шины, конец (больше не использовать)"

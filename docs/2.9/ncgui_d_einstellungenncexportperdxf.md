# Диалоговое окно Настройки: Экспорт ЧУ DXF / Диалоговое окно Настройки: Экспорт DXF, медь

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт ЧУ DXF
* Параметры > Настройки > Фирма > Механическая обработка > Экспорт DXF, медь
* Сервисные программы > Данные изготовления > Механическая обработка > ЧУ DXF. Рядом с полем Машина нажмите ++...++.
* Сервисные программы > Данные изготовления > Механическая обработка > DXF, медь. Рядом с полем Машина нажмите ++...++.

В этом диалоговом окне установите настройки для экспорта ЧУ и меди в формате DXF.

Обзор основных элементов диалогового окна:

### Машина

В комплекте поставки содержится соответствующее определение машины с названием "Стандарт". Если в качестве схемы сгенерирована другая машина, можно выбрать из раскрывающегося списка другие определения машины и работать с ними.

Через расположенную рядом [панель инструментов](schemeconfig_m_schemaschaltflaechen.md) можно создавать новые схемы, сохранять изменения в схеме, копировать или удалить схему. Кроме того, схемы можно импортировать и экспортировать.

### Описание

В этом поле дается описание, которое было введено в диалоговом окне Новая схема в поле Описание для машины.

# Создать PDF-комментарии в PDF-файле

Проект EPLAN, экспортированный в PDF-файл, можно открыть в приложении Acrobat Reader 8.0 и в целях проведения экспертизы предусмотреть возможность создания в нем PDF-комментариев. Для этого отправитель должен либо отправить PDF-файл на проверку в приложении Acrobat, либо разрешить в нем комментарии.

Условия:

* Вы [нормоконтролер](redlining_k_projektkommentator.md).
* У вас установлено приложение Acrobat Reader 8.0.
* От [исполнителя](redlining_k_projektkommentator.md) вы получили PDF-файл.

1. Файл > Открыть
2. В появившемся диалоговом окне выберите PDF-файл, который необходимо прокомментировать, и нажмите кнопку ++Открыть++.

!!! info "Для сведения:"

    PDF-файл откроется в приложении Acrobat Reader.

3. Выберите пункты меню Просмотр > Панели инструментов > Комментарии и пометки.

!!! info "Для сведения:"

    Появится панель инструментов.

4. Чтобы на странице PDF разместить примечание, выберите на панели инструментов инструмент выделения графики и мышью "нарисуйте", например, прямоугольник.
5. Чтобы в изображенное поле примечания ввести текст комментария, дважды щелкните это поле.
6. Введите текст комментария в поле комментариев.

!!! info "Для сведения:"

    После импортирования в проект EPLAN текст комментария можно будет обработать в системе EPLAN.

7. Чтобы к полю примечаний вставить ответ, в поле комментария выберите пункты меню Параметры > Ответить.
8. Введите текст в поле ответа.

!!! info "Для сведения:"

    После импортирования в проект EPLAN текст ответа будет отображаться в системе EPLAN, но изменить его будет невозможно.

9. Чтобы задать статус PDF-комментария, в поле комментария выберите пункты меню Параметры > Установить статус и выберите нужный статус.
10. Чтобы закрыть поле комментария и ответа, нажмите в поле комментария на символ в правом верхнем углу.

!!! info "Для сведения:"

    Если навести курсор на изображенное примечание, будет показана всплывающая подсказка с текстом комментария и количеством ответов.

11. Выберите пункты меню Файл> Сохранить.
12. Чтобы PDF-файл с комментариями отправить по электронной почте, выберите на панели инструментов команду Отправить комментарии.
13. В диалоговом окне введите адрес электронной почты и нажмите кнопку ++Отправить++.

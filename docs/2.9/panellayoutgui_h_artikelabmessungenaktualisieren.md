# Обновить размеры изделий

Размеры изделий можно изменить позже, из-за чего может потребоваться их обновление.

Условие:

Вы открыли проект.

1. Данные проекта > Устройство / изделие > Навигатор компоновки электрошкафа 2D
2. Откройте в графическом редакторе ту страницу компоновки электрошкафа, для которой требуется выполнить обновление.
3. Выберите пункты Всплывающее меню > Обновить размеры изделий.
4. Щелкните по кнопке ++Да++, чтобы применить размеры изделий.
5. Щелкните по кнопке ++Да++ для каждого последующего изделия.

!!! info "Для сведения:"

    Текущие размеры элементов переносятся на размещения изделий. При этом программа проверяет каждое размещение изделия на текущие размеры изделия, а в случае обнаружения отличающегося значения отображает сообщение, в котором требуется подтвердить или отклонить обновление.

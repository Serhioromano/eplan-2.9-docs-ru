# Выбрать вариант символа

Символ может иметь до восьми вариантов с именами вариантов от А до Н и вариант образа контакта, при чем все варианты символа имеют общие свойства. То есть все варианты ведут себя одинаково и они отражают одинаковую в каждом случае функцию. То есть у них всегда одинаковое количество выводов устройства и они отличаются только графическим расположением выводов устройства.

Перед размещением символа вы можете выбрать варианты символа поворотом курсора. При этом предварительная настройка для варианта вывода берется из свойства символа "Стандартный вариант", которое можно предварительно задать для каждой библиотеки символов в диалоговом окне Настройки: Библиотеки символов (меню Параметры > Настройки > Проекты > "Имя проекта" > Управление > Библиотеки символов).

Условие:

Вы открыли проект. В Графическом редакторе открыта страница проекта.

1. Выберите следующие пункты меню: Вставить > Символ
2. В диалоговом окне Выбор символа выберите нужный символ.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Диалоговое окно закрывается, и символ висит на курсоре, причем в качестве предварительной настройки используется первый вариант (стандартный вариант).

4. Переместите курсор в нужное место в графическом редакторе.
5. Нажмите ++Ctrl++ и переместите мышь, чтобы переключаться между всеми вариантами.
Или нажмите на ++Tab++, чтобы провернуть символ через все варианты по часовой стрелке.
6. Разместите нужный вариант символа на странице схемы соединений (простым щелчком).
7. В диалоговом окне Свойства ++...++ введите нужные данные для устройства.
8. Щелкните по кнопке ++OK++.
9. Аналогично переходите ко всем остальным действиям по вводу выбранного символа.
10. Завершите операцию, выбрав пункт всплывающего меню Прервать операцию или нажав клавишу ++Esc++.

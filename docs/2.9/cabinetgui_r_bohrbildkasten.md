# Вкладка "Граница схемы сверления"

Вставлена граница схемы сверления через меню Вставить > Блок / вывод устройства / монтажная плата > Граница схемы сверления, или выполнен двойной щелчок по уже имеющейся границе схемы сверления.

На этой вкладке задаются свойства границы схемы сверления. что возможно только в ***Проект макросов***. В ***проекте схемы соединений*** большинство этих полей используются только для отображения данных и не могут быть обработаны.

Обзор основных элементов диалогового окна:

Групповое поле "Общее"

### Имя

Это поле можно обрабатывать только в ***Проект макросов***! Введите здесь имя границы схемы сверления, при необходимости указав зону. Это имя будет использоваться для имени файла генерируемой схемы сверления. В соответствии с этим схему сверления нельзя создать без введенного имени.

В ***проекте схемы соединений*** это поле заполняется автоматически и не может быть обработано.

### Описание

Это поле можно обрабатывать только в ***Проект макросов***! Текст, введенный в этом поле, становится текстом описания схемы сверления, автоматически генерированной из проекта макросов, и выводится в поле комментария при вставке соответствующей схемы сверления.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

### Подкаталог для контура (данные ЧУ)

Место хранения для файлов *.fc1 (данные контура ЧУ), генерируемых автоматически при генерировании схемы сверления. Каталогом по умолчанию является каталог для макросов ($MD_MACROS).

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Присвоение объектов вручную

Обычно все объекты, находящиеся внутри границы схемы сверления, относятся к схеме сверления. Это присвоение объектов можно изменить в ***Проект макросов***.

Установите этот флажок, если, например, требуется также присвоить объекты, находящиеся вне рамки. Затем необходимо выделить все объекты, которые должны относиться к схеме сверления, и выбрать во всплывающем меню границы схемы сверления функцию Присвоить объекты границе схемы сверления. Если снова снять этот флажок, в схеме сверления восстанавливается первоначальное присвоение объектов.

В ***проекте схемы соединений*** эту настройку невозможно изменить!

Групповое поле "Нулевая точка"

### Позиция X / Позиция Y

В случае установленного флажка Активн. здесь указаны стандартные значения для позиции точки захвата. При необходимости эти значения можно обработать.

### Активно

Стандартная позиция нулевой точки — слева внизу на рамке границы схемы сверления.

Если этот флажок установлен, показывается стандартная позиция нулевой точки, которую можно изменить. Для этого поля Позиция X и Позиция Y деблокируются для обработки, после чего можно перезаписать стандартные значения. После закрытия диалогового окна кнопкой [OK] на положение нулевой точки на странице указывает специальный графический символ.

Если этот флажок снят, ввод данных в поля Позиция X и Позиция Y невозможен, а нулевая точка не выводится. В случае повторной установки этого флажка нулевая точка вновь возвращается на свою стандартную позицию.

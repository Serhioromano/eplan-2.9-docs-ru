# Диалоговое окно Подклассы буквенных обозначений

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация (online). Рядом с полем Подклассы буквенных обозначений щелкните ++...++.

В этом диалоговом окне для данного проекта можно установить специфические для него буквенные обозначения с подклассами, включая текстовое описание подкласса. Описания при этом задаются как многоязычные тексты. Сохраненные подклассы можно выбрать через всплывающее меню из соответствующего списка выбора в поле Видимое ОУ диалогового окна свойств условных обозначений.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую вы хотите обработать. Вы можете обрабатывать только пользовательские схемы.

Через расположенную рядом [панель инструментов](schemeconfig_m_schemaschaltflaechen.md) можно создавать новые схемы, сохранять изменения в схеме, копировать или удалить схему. Кроме того, схемы можно импортировать и экспортировать.

### Описание

Данное поле отображает тексты описаний, которые вы ввели при создании схемы в диалоговом окне Новая схема в поле Описание.

### Панель инструментов

На панели инструментов имеются следующие кнопки:

Кнопка |  Значение
---|---
![](images/all_new_as.png){: .ui-icon } (Создать) |  Добавляет новую строку в конец таблицы.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет текущую строку в таблице.

### Подкласс / Задача подкласса

Укажите в столбце Подкласс соответствующее обозначение подкласса; эта строка символов копируется в поле Видимое ОУ диалогового окна свойств условных обозначений, когда в нем открывается список выбора. В Задача подкласса введите поясняющий текст. Он может быть переведен.

Внутри столбца сортировку можно производить как в восходящем, так и в нисходящем порядке, для чего нужно два раза щелкнуть заголовок столбца. После этого направление выводится в заглавии столбца в виде графического символа.

### Всплывающее меню

Пункт меню |  Значение
---|---
Закрепить область |  Закрепляет все строки таблицы, которые находятся над маркировкой, или все столбцы, которые находятся слева от маркировки. В этом случае строки ниже или колонки справа от маркировки можно передвигать с помощью полос прокрутки.
Снять закрепление |  Убирает закрепление, т. е. все строки и столбцы можно передвигать с помощью полос прокрутки.
Подобрать ширину столбца |  Все столбцы так подогнаны к таблице по ширине, что и заголовок, и содержимое столбцов полностью читаемы.
Размер шрифта 100% |  Этот пункт меню доступен, только если размер шрифта в таблице был изменен ([Ctrl] \+ прокручивание колесика мыши). Отображает тексты в таблице с исходным размером шрифта.
Конфигурировать представление |  Открывает диалоговое окно Конфигурировать представление, в котором можно указать последовательность и число отображаемых столбцов.

При двойном щелчке на ячейке для обработки дополнительно доступны следующие пункты меню:

Пункт меню |  Значение
---|---
Специальный символ |  Открывает диалоговое окно Специальный символ, в котором Вы можете выбрать требуемый символ и вставить его на место текущей позиции курсора с помощью [OK].
Новая строка |  Вставляет новую строку на место текущей позиции курсора.
Многострочный ввод |  Открывает диалоговое окно Многострочный ввод, в котором можно ввести требуемые строки, при этом новые строки добавляются с помощью [Ctrl]+[Ввод].
Перевести |  Открывает диалоговое окно Найденные слова, если в словаре имеются переводы для введенного понятия; выберите подходящий текст перевода. Если в словаре ***нет*** переводов, открывается диалоговое окно Перевести, после чего можно ввести новый текст перевода.
Убрать переводы |  Убирает все тексты переводов.
Обозначить текст, не требующий перевода |  Обозначает выделенный текст как "Перевод не требуется". Тексты, выделенные таким образом, исключаются из перевода и вывода в файле пропущенных слов.
Удалить обозначение |  Удаляет обозначение "Перевод не требуется" на выделенном тексте, и соответствующий текст становится частью перевода.
Многоязычный ввод |  Открывает диалоговое окно Многоязычный ввод, в котором можно ввести требуемые тексты переводов для вывода.

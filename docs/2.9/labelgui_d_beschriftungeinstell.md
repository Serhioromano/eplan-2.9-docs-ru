# Диалоговое окно Настройки: Экспорт данных изготовления / маркировка

* Вы открыли проект. Сервисные программы > Данные изготовления > Экспорт / маркировка. В диалоговом окне Экспортировать данные изготовления / Вывести маркировку рядом с полем Настройки щелкните по кнопке ++...++.
* Параметры > Настройки > Пользователь > Интерфейсы > Экспорт данных изготовления / маркировка.

В этом диалоговом окне можно управлять схемами экспорта данных изготовления / маркировки и обрабатывать их. На этой схеме заданы соответствующие настройки.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Тип отчета

Схема для экспорта / маркировки всегда создается вместе с типом отчета. В этом поле выводится тип отчета, который был выбран для текущей схемы.

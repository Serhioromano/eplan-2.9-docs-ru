# Дублировать элементы

С помощью команды Дублировать можно очень просто одновременно вставить несколько копий. Функция дублирования работает только в пределах одной страницы, она не может быть использована для всех страниц сразу.

Условие:

Вы открыли страницу, форму, рамку или символ.

1. Выделите дублируемые элементы и выберите пункты меню Обработать > Дублировать.
2. Разместите первую копию.
3. Укажите в диалоговом окне Дублировать общее число генерируемых элементов.

!!! info "Для сведения:"

    EPLAN автоматически генерирует указанное число копий и размещает их, используя интервал, выбранный для первой копии.

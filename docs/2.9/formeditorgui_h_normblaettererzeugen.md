# Создать рамку

При создании рамки открывается диалоговое окно Свойства рамки — <Имя рамки>, в котором устанавливаются важнейшие свойства новой рамки. (Это диалоговое окно может быть вызвано также в последствии для ***открытой*** рамки, это позволит Вам в любое время дорабатывать свойства.)

Условие:

Вы открыли проект.

1. Сервисные программы > Основные данные > Рамка > Создать
2. В диалоговом окне Создать рамку выберите дисковод и каталог, в котором следует сохранить новую рамку.
3. Введите в поле Имя файла наглядное имя для рамки.
4. В поле Путь выберите пункт всплывающего меню Сбросить на стандарт, если требуется вернуться к стандартной настройке дисковода и каталога. (Стандартный каталог является либо специфической предварительной программной настройкой <Дисковод программы>: \<Каталог программы>\Рамки\<Идентификатор фирмы>, либо настройкой, установленной через пункты меню Параметры > Настройки > Пользователь > Управление > Каталоги.)
5. Щелкните по кнопке ++Сохранить++.
6. В диалоговом окне Свойства рамки — <Имя рамки> из раскрывающегося списка Категория выберите свойства, которые должны отображаться в списке ниже.
7. Определите значения свойств, введя или перезаписав запись или выбрав значение по умолчанию из раскрывающегося списка.
8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Новая рамка сохраняется в соответствии с вашими данными и отображается в редакторе рамок.

!!! info "Для сведения:"

    Нулевая точка рамки обозначается красным кругом, как в редакторе контура и редакторе символов. Позиция нулевой точки учитывается при масштабировании, но ее нельзя перемещать вручную.

# Экспортировать собственные наборы буквенных обозначений, обозначения и описания выводов устройства

Если созданы собственные наборы буквенных обозначений, обозначений выводов устройства и описаний выводов устройства, то они экспортируются в XML-файл. Этот файл можно использовать для резервирования данных или для обмена данными.

### Экспортировать наборы буквенных обозначений

1. Сервисные программы > Основные данные > Буквенное обозначение
2. Выделите столбцы с наборами буквенных обозначений, которые Вы хотите экспортировать.
3. Выберите пункт всплывающего меню Экспорт.
4. В диалоговом окне Экспорт перейдите в требуемый каталог и введите имя файла.
5. Щелкните по кнопке ++Сохранить++.

!!! info "Для сведения:"

    ***Выделенные*** наборы буквенных обозначений сохраняются в XML-файл.

### Экспортировать обозначения и описания вывода устройства

1. Сервисные программы > Основные данные > Обозначение / описание вывода устройства
2. Выберите вкладку Обозначения выводов устройства или Описания выводов устройства.
3. Выберите пункт всплывающего меню Экспорт.
4. В диалоговом окне Экспорт перейдите в требуемый каталог и введите имя файла.
5. Щелкните по кнопке ++Сохранить++.

!!! info "Для сведения:"

    ***Все*** наборы обозначений и описаний выводов устройства сохраняются в XML-файле.

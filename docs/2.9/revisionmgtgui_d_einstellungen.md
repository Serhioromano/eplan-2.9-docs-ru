# Диалоговое окно Настройки: Ревизия (сравнение свойств проектов)

Вы открыли проект. Проект не защищен от записи. Параметры > Настройки > Проекты > "Имя проекта" > Управление > Ревизия (Сравнение свойств проектов).

В этом новом диалоговом окне установите настройки для сравнения свойств проектов.

Доступны следующие элементы диалогового окна:

### Разместить на каждой странице маркер ревизий для соединений

Если этот флажок установлен, при изменении соединения, проходящего через несколько страниц, это соединение выделяется на каждой странице.

Если этот флажок не установлен, соединение выделяется только на первой странице.

### Максимальное число отображенных результатов

В этом поле задайте максимальное число сообщений, которые отображаются в диалоговом окне Результаты сравнения свойств — <Имя проекта>. Эта настройка необходима для ускорения отображения диалогового окна и не влияет на выполнение сравнения свойств. Независимо от заданного здесь числа, все объекты сравниваются, а все результаты записываются в список результатов ревизии. (Этот список сохраняется в базе данных текущего проекта.)

# Выбрать изделие

При помощи выбора изделия устройству присваивается одно или несколько изделий. При многократном выборе изделий возможно одновременное присвоение нескольких изделий. При этом функции / условному обозначению присваиваются данные ***первого*** изделия, содержащегося в выборе.

Выбор изделия можно вызывать из многих мест:

* из диалогового окна Свойства ++...++ * из навигатора устройств
* из навигатора спецификаций
* из Список устройств.

### Выбор изделия в диалоговом окне "Свойства" ++...++ Условие

Вы открыли диалоговое окно Свойства ++...++ для главной функции и выбрали вкладку Изделие.

1. Щелкните в поле Номер изделия по ++...++.
2. В диалоговом окне Выбор изделия можно просмотреть структуру дерева и выбрать требуемое(ые) изделие(я).
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Номера изделий переносятся в диалоговое окно Свойства ++...++.

4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Все номера изделий, внесенные в диалоговом окне Свойства ++...++, разыскиваются в базе данных изделий, и соответствующие изделия сохраняются в проект. При этом данные изделий считываются в соответствии с настроенным источником данных. Также сохраняются шаблоны функций изделия.

!!! info "Для сведения:"

    Свойства ***первого*** изделия, содержащегося в выборе, записываются на текущем устройстве (например, Тип кабеля, Поперечное сечение / диаметр соединения и т. д.).

!!! info "Для сведения:"

    При последующих изменениях изделия в базе данных изделий данные функции / условного обозначения в проекте остаются неизменными.

!!! info "Для сведения:"

    Если устройству уже присвоено изделие или технические свойства (например, Технические параметры, Обозначение вывода устройства, Кабель / Группа соединений: Тип и т. д.), отображается диалоговое окно [Конфликт](partselectiongui_d_konflikte.md). Диалоговое окно содержит предыдущие данные устройства и новые свойства, которые записываются по выбранному изделию.

5. При необходимости измените свойства и закройте диалоговое окно Конфликт нажатием кнопки ++OK++.

!!! info "Для сведения:"

    Новые данные переносятся на функцию / условное обозначение.

!!! info "Для сведения:"

    При выборе изделия значение из поля Технические параметры на вкладке Технические данные переносится, только если для этого свойства не было задано значение в "Главном шаблоне" или если нет доступных шаблонов функций. Главный шаблон — это первый шаблон функции на вкладке Шаблоны функций базы данных изделий, в определении функции которого установлено "Главная функция".

!!! warning "Предупреждение:"

    Если необходимо хотя бы одному символу присвоить несколько номеров изделий, следует обратить внимание на то, чтобы при каждом новом присвоении выделялась пустая строка, в противном случае выделенный номер изделия будет перезаписан новым.

!!! warning "Предупреждение:"

    * Технические свойства (технические параметры, обозначения выводов устройства и т. д.) первого выбранного изделия можно перенести только при выборе изделия для ***первой строки*** на вкладке Изделие. При повторном выборе изделия для другого присвоенного изделия устройства также отображается диалоговое окно Конфликт, если данные ссылки уже присвоенного изделия (например, номер заказа, производитель и т. д.) отличаются от данных нового изделия. В этом случае в диалоговом окне конфликта представлены только отличающиеся данные ссылки изделия. С помощью номера индекса, отображаемого в столбце Свойства, можно узнать, для какого изделия было открыто конфликтное диалоговое окно.
    * При выборе изделия, которое появляется в проекте в более старом состоянии, чем в базе данных изделий, отображается запрос. Вы можете решить, нужно ли выбрать это обновленное изделие или другое изделие. В некоторых случаях (например, при изменении данных шаблона) данные устройства обновленного изделия не могут быть перенесены во все сферы применения изделия в проекте.
    Чтобы в таком случае иметь возможность обновлять данные устройства на функциях в проекте, в навигаторе спецификаций доступен пункт всплывающего меню Обновить данные устройства.

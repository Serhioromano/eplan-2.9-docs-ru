# Диалоговое окно Выбрать раздел

Вы открыли проект. В базе данных изделий (Сервисные программы > Изделие > Управление) существует как минимум одно изделие, которому присвоено более одного раздела. Вставить > Устройство. В диалоговом окне Выбор изделия выберите одно из изделий, которым присвоено более одного раздела, щелкните мышкой по кнопке ++OK++ и разместите устройство.

В этом диалоговом окне выберите раздел из списка разделов, сохраненных для изделия, который должен быть у устройства.

Обзор основных элементов диалогового окна:

### Раздел

В этом списке предлагаются на выбор все присвоенные изделию разделы Fluid-техники, а также дополнительно раздел "Fluid (не определен)".
Щелкните мышкой по зависимой кнопке раздела, который нужно присвоить функциям устройства.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

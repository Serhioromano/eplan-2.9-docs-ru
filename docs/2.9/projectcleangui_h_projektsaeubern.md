# Сжать проект

При сжатии EPLAN проверяет, какие данные будут использоваться в проекте и согласованы ли данные проекта. Все данные проекта, которые не будут использоваться и не входят в сжимаемые данные, удаляются. Настройки для объема сжатия сохраняются в схеме. Благодаря этому настройки могут использоваться повторно.

Если объем сжатия вы уже сохранили как схему, необходимо только выбрать имеющуюся схему. Чтобы исключить из сжатия уровни структуры проекта, необходимо определить фильтр.

### Определить объем сжатия

Условие:

Вы открыли проект и выделили его в Навигаторе страниц.

1. Выберите пункты меню Проект > Организовать > Сжать.
2. В диалоговом окне Сжать проект рядом с полем Настройки щелкните по ++...++.
3. В диалоговом окне Настройки: Сжатие выберите существующую схему для обработки или создайте новую.
4. В поле Сжимаемые данные установите флажок для сжимаемых данных.

!!! info "Для сведения:"

    Если флажок установлен для узла, все нижестоящие опции выбираются автоматически.

5. Щелкните по кнопке ![](images/all_save_as.png){: .ui-icon } (Сохранить).
6. Щелкните по кнопке ++OK++.

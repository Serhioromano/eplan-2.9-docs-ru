# Вкладка Форматы адресов

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > ПЛК. В диалоговом окне Настройки: ПЛК щелкните рядом с полем Специфические для ПЛК настройки по кнопке ++...++. В диалоговом окне Настройки: Специфич. для ПЛК выберите вкладку Форматы адресов.

В этой вкладке вы определяете формат адресов ПЛК. Для дискретных и аналоговых вводов и выводов возможно использование разных форматов адресов.

Обзор основных элементов диалогового окна:

Групповое поле Форматы адресов

В отдельных полях можно задавать форматы адресов отдельно для дискретных и аналоговых входов и выходов. С помощью кнопки ++...++ вы открываете диалоговое окно Формат адреса ПЛК и там задаете формат.

### Раздельная область адресов для входов и выходов

Этот флажок может быть установлен, только если форматы адресов входов и выходов совпадают. При этом форматы дискретных и аналоговых выводов устройства ПЛК могут отличаться.

Если этот флажок установлен, то для вводов и выводов ПЛК используются отдельные диапазоны адресов. Каждый вывод устройства ПЛК должна иметь однозначное обозначение, то есть, если, например, существует адрес "A001.0", такой адрес "E001.0" ***нельзя*** больше использовать.

Если этот флажок снят, для входов и выходов ПЛК используется общий диапазон адресов. В этом случае адреса отличаются дополнительными префиксами, то есть в ПЛК могут встречаться адреса с одинаковыми байтами и битами, например, "A001.0" и "E001.0".

!!! tip "Совет:"

    При снятом флажке для карт ПЛК, которые имеют как входы, так и выходы, с помощью свойства Диапазон адресов 2 можно определить отдельный диапазон адресов для выходов. Свойство вводится на вкладке "Блок ПЛК" в базе данных изделий.

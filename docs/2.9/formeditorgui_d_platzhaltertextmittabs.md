# Диалоговое окно Свойства (текст заполнитель)

Вы открыли проект. Сервисные программы > Основные данные > Форма > Открыть > [Открыть].

* Вставить > Текст заполнитель.
* Вы дважды щелкнули по тексту-заполнителю (или выбрали во всплывающем меню > Свойства ; в этом случае может быть выделено несколько текстов-заполнителей).

В этом диалоговом окне выберите специфический для обработки текст-заполнитель, который должен быть добавлен в форму, и установите его свойства для формы, языка и рамки. При оценке формы программа автоматически заменяет эти заполнители на соответствующие значения конкретных объектов проекта, для которого выполняется отчет, например, на значения функциональных элементов, страниц, символов и т.д.

!!! note "Замечание:"

    Если у заполнителя нет блока выравнивания или блок выравнивания есть и одновременно установлен флажок Удалить переходы на новую строку (в свойстве Рамки), то в отчетах переходы на новую строку отфильтровываются.

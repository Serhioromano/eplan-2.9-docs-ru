# Импортировать данные предварительного планирования

Для создания предварительного планирования у вас есть возможность импортировать списки с данными предварительного планирования, созданные во внешних приложениях. Источниками данных могут быть файлы Excel и текстовые файлы.

Условия:

* Вы открыли проект.
* Вы подготовили источник данных для импорта. Чтобы подготовить таблицу Excel для импорта данных, определите область, содержащую импортируемые данные. Чтобы импортировать текстовый файл, определите разделители и заголовки столбцов.

Далее описывается импорт на примере типа источника данных "Excel".

1. Выберите пункты меню Данные проекта > Предварительное планирование > Импортировать.
2. В диалоговом окне [Импортировать данные предварительного планирования — <Имя проекта>](planninggui_d_importvorplanung.md) выберите в поле Тип источника данных запись "Excel".
3. Щелкните рядом с полем Источн. данных по кнопке ++...++.
4. Выберите в диалоговом окне [Выбрать источник данных](planninggui_d_datenquelleexcel.md) нужный файл Excel для импорта.

!!! info "Для сведения:"

    В поле Таблица перечислены все рабочие листы и названные области, находящиеся в файле Excel.

5. Выберите в поле Таблица имя области, определенной при подготовке файла Excel для импорта.
6. Если файл Excel содержит заголовки столбцов, установите флажок Имя столбца в заголовке таблицы.
7. Щелкните по кнопке ++OK++.
8. В поле Присвоения полей определите схему для присвоения внешних полей свойствам EPLAN. Если схема для присвоения полей уже была создана, ее можно выбрать из раскрывающегося списка. Для создания или изменения схемы щелкните по кнопке ++...++.

!!! info "Для сведения:"

    В диалоговом окне Присвоения полей имена столбцов из файла Excel отображаются в столбце Внешнее поле. Если файл не содержит имен столбцов, то поля нумеруются по порядку.

9. В таблице диалогового окна [Присвоения полей](planninggui_d_feldzuordnung.md) определите присвоение внешних полей свойствам EPLAN и сохраните данное присвоение в виде схемы.
10. Щелкните по кнопке ++OK++.
11. Определите с помощью флажка в нижней области диалогового окна Импортировать данные предварительного планирования — <Имя проекта>, должны ли перезаписываться данные существующего объекта планирования и должны ли создаваться новые сегменты при импорте. В зависимости от этих настроек в диалоговом окне Синхронизировать данные предварительного планирования предварительно устанавливается значение в столбце Действие.
12. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В диалоговом окне [Синхронизировать данные предварительного планирования](planninggui_d_importabgleich.md) отображаются импортируемые данные из файла Excel.

13. Определите в столбце Действие, что необходимо сделать с соответствующими сегментами: импортировать, изменить или не импортировать.
14. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Данные будут импортированы в соответствии с настройками в открытый проект.

!!! info "Для сведения:"

    Автоматически сгенерированным сегментам в навигаторе предварительного планирования назначается свойство Сгенерировано автоматически со значением "Импорт".

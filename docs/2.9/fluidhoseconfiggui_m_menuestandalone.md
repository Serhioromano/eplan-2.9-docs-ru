# Пункты главного меню конфигуратора шлангопроводов в качестве самостоятельной программы (самостоятельного варианта)

В главном меню конфигуратора шлангопроводов в качестве самостоятельной программы (самостоятельного варианта) доступны следующие меню:

Меню Файл

Пункт меню |  Символ |  Значение
---|---|---
Создать |  ![](images/toolbar_newfile.png) |  Открывает диалоговое окно Новый файл, если открыто диалоговое окно Конфигурировать шлангопроводы ***и*** создана хотя бы одна строка с данными шлангопроводов. Подтвердите запрос о сохранении изменений, нажав кнопку ++Да++, после чего откроется диалоговое окно Сохранить как, в котором можно указать требуемое имя и место хранения данных шлангопроводов при сохранении. Данные сохраняются в разделенном табулятором текстовом файле Юникод (*.txt), который можно открыть также с помощью других приложений (файл обмена). Если в диалоговом окне Создать нажать [Нет], созданные данные шлангопроводов будут утеряны.
Открыть |  ![](images/toolbar_openfile.png) |  Открывает диалоговое окно Открыть для импорта данных шлангопровода, которые были сохранены в файле обмена. Задайте в этом диалоговом окне имя и место хранения открываемого файла обмена. Можно выбрать только разделенные табулятором текстовые файлы Юникод (*.txt).
Сохранить |  ![](images/toolbar_save.png) |  Сохраняет обработанные данные шлангопроводов в ранее выбранном вами файле обмена с помощью пункта меню Сохранить как.
Сохранить как |  ![](images/toolbar_saveas.png) |  Открывает диалоговое окно Сохранить как для экспорта данных шлангопровода в файл обмена, который может считываться другим приложением. В диалоговом окне вы задаете имя и место хранения обработанных данных шлангопроводов при сохранении. Данные сохраняются в разделенном табулятором текстовом файле Юникод (*.txt).
Конфигурировать шлангопроводы |  ![](images/toolbar_fluidhoseconfig.png) |  Открывает диалоговое окно Конфигурировать шлангопроводы, в котором отображаются считанные и обработанные данные шлангопроводов.
Завершить |  |  Завершает работу программы.

Меню Настройки

Пункт меню |  Символ |  Значение
---|---|---
Параметры |  ![](images/toolbar_settings.png) |  Открывает диалоговое окно Настройки.  Здесь можно задавать свойства и значения для конфигурации шлангопроводов (категории Пользователь > Add-ons > Конфигуратор шлангопроводов) и сохранять их как схему. Можно также выбрать язык диалогового окна (категории Пользователь > Представление > Языки).

Меню справки

Пункт меню |  Значение
---|---
Содержание |  Открывается окно справки с открытой вкладкой Содержание.
Программа повышения качества работы, ориентированная на клиента |  Открывает диалоговое окно Программа повышения качества работы, ориентированная на клиента. Здесь можно принять решение о принятии участия в этой программе.
Информация |  Открывает диалоговое окно Приложение EPLAN, в котором содержится информация по следующим пунктам:

* Продукт
* Объем лицензии
* Авторское право / Торговая марка
* Информация о системе

# Диалоговое окно Обработать ++...++ Вы открыли диалоговое окно [Конфигурировать шлангопроводы](fluidhoseconfiggui_d_schlauchleitungen.md).

* Дважды щелкните по нужной строке таблицы с данными шлангопровода.
* Или выделите в таблице несколько строк с данными шлангопровода и выберите пункт всплывающего меню Обработать.

В этом диалоговом окне конфигурируйте выделенные в таблице диалогового окна Конфигурировать шлангопроводы шлангопроводы, заполнив их свойства и добавив значения. Предлагаемые на выбор в раскрывающемся списке значения копируются из настроек выбранной схемы. Как результат конфигурации генерируется обозначение шлангопровода и код типа.

В диалоговом окне возможна групповая обработка. Т. е. эта функция позволяет одновременно обрабатывать выбранные данные конфигурации сразу нескольких шлангопроводов. Если обрабатываемые шлангопроводы имеют разные значения для одного и того же свойства, это помечается записью "< ++...++ >".

Обзор основных элементов диалогового окна:

Групповое поле "Шланг"

### Рабочее давление

Под предлагаемыми к выбору значениями понимается максимально допустимое динамическое рабочее давление шлангопровода в "бар".

### Номинальный диаметр

Предлагаемые на выбор значения отображаются вместе с выбранными в настройках единицами измерения. Если единица измерения не указана, записи автоматически соотносятся с единицей измерения "мм".

### Тип шланга – Стандарт

Так как шланги являются стандартизированными функциональными элементами, дополнительно к типу шланга отображается еще и обозначение стандарта. Тип шланга зависит от выбранного максимально допустимого динамического рабочего давления и от номинального диаметра шланга.

### Длина шлангопровода

Эта информация необходима для вывода кода типа. Есть возможность вводить значение с единицами измерения и без. Все введенные значения автоматически пересчитываются в единицы измерения, указанные в настройках на вкладке Общее в поле Единица измерения длины шлангопровода.

Длина шлангопровода обычно берется как расстояние между уплотняющими головками или — при изогнутой арматуре — как расстояние между центрами арматуры.

Групповое поле "Прочие данные"

### Производитель

Щелкните кнопку ++...++ и в открывшемся диалоговом окне выберите требуемого производителя шлангопровода. Эта информация нужна для вывода обозначения шлангопровода.

### Дата производства

При помощи кнопок со стрелками и / или календаря укажите дату производства шлангопровода. Дата отображается в поле в формате <ММ / ГГГГ>. Дата производства является частью обозначения шлангопровода (см. пояснение к полю Идентификатор).

### Единица измерения давления

Выбранная единица измерения рабочего давления выводится в обозначении шлангопровода. Выберите единицу измерения из двух отображаемых: ***бар*** или ***МПа***.

Так как при вводе рабочего давления в настройках за основу берется единица измерения "бар", при использовании единицы измерения "МПа" конфигуратор шлангопроводов соответственно пересчитывает значение рабочего давления для отображения.

### Идентификатор

В этом поле отображается ***обозначение шлангопровода***.

Последовательность составных частей обозначения шлангопровода соответствует значениям по умолчанию по стандарту DIN 20066. Слева направо составные части состоят из значений следующих свойств:

* Производитель (только краткое имя);
* Рабочее давление (с указанием единицы измерения);
* Дата производства (Отображается в формате <ГГММ> без косой черты. Например, введенное значение даты "1608" означает, что шлангопровод произведен в августе 2016 года.)

Каждое значение отделяется косой чертой с пробелом с каждой стороны.

### Арматура слева / арматура справа

Щелкните рядом с полем кнопку ++...++ и из открывшегося диалогового окна Выбрать арматуру выберите нужную ***серию*** и ***тип конструкции*** арматуры, которая монтируется на шланг ***слева*** или ***справа***. Если в настройках для выбранной арматуры имеется путь к графическому файлу, в окне предварительного просмотра под соответствующим полем отображается картинка арматуры.

### Угол кручения

Укажите в этом поле угол кручения для монтируемых на концах шланга арматур без единиц измерения. Графика под полем объясняет, как определяется угол.

### Код типа

В этом поле отображается код типа сконфигурированного шлангопровода, который необходимо ввести для правильного заказа шлангопровода.

Составные части кода типа и последовательность их вывода можно задать в настройках.

Если вы используете установленную по умолчанию в настройках схему DIN 20066_2012-01, генерируется код типа, который соответствует значениям стандарта DIN 20066. В этом случае код типа состоит из значений следующих свойств, которые отображаются слева направо следующим образом:

* Тип шланга (зависит от рабочего давления и номинального диаметра шланга).
* Номинальный диаметр (без указания единицы измерения)*
* Тип конструкции арматуры, слева*
* Тип конструкции арматуры, справа*
* Длина шланга (без указания единицы измерения).
* Угол кручения (релевантен только для изогнутой арматуры, без единицы измерения).

Значения свойств, обозначенных "*", выводятся в коде типа без разделителя. Все остальные значения отделяются друг от друга дефисом.

### [Другие]

Некоторые производители шлангов предлагают расширенные опции для заказа шлангопроводов. Например, шлангопроводы можно заказывать с защитой шланга. Чтобы правильно отразить зависящие от производителя данные заказа, в конфигураторе шлангопроводов вам теперь дается возможность добавить в код типа произвольные свойства с помощью данной кнопки.

Нажмите на клавишу и выберите в диалоговом окне Свойства (код типа) имя и значение свободного свойства.

!!! note "Замечание:"

    Кнопка активируется только при выполнении следующих предпосылок в настройках конфигуратора шлангопровода:

* На вкладке Прочие свойства определены и указаны со значением одно или несколько свободных свойств.
* На вкладке Общее в формат кода типа добавлены соответствующие элементы формата (произвольные свойства и разделители).

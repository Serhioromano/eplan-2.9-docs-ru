# Диалоговое окно Генерировать соединения фазовых шин

Открыто одно пространство листа. Открыт навигатор соединений. Генерировать Данные проекта > Соединения > Соединения фазовых шин.

В этом диалоговом окне перечислены все соединения с видами представления "Многополюсный" и "Трехмерный чертеж монтажных поверхностей", источником и целью которых соответственно является один из функциональных элементов, выбранных для генерирования соединения фазовых шин. Это могут быть, например, соединения, с определением функции "Жила / Провод".

Дополнительно отображается ОУ соединения, с помощью которого фазная шина используется для группировки соединений.

Генерировать и удалить соединения фазных шин, а также присвоить изделия можно при помощи пунктов всплывающего меню.

Доступны следующие элементы диалогового окна:

Таблица содержит следующие столбцы:

### ОУ (идентифицирующее)

ОУ соединения.

### Определение функции

Для соединений, не являющихся соединениями фазных шин, в этом столбце отображается определение функции "Жила / Провод". После преобразования в соединение фазных шин отображаемое определение функции изменится на "Соединение фазной шины". При удалении соединения фазных шин запись в этом столбце возвращается снова на "Жила / Провод".

### Источник

Источник соединения.

### Цель

Цель соединения.

* * *

### Всплывающее меню

Пункт меню |  Значение
---|---
Генерировать соединение фазных шин |  Этот пункт всплывающего меню преобразует соединение в выделенной строке в соединение фазных шин с определением функции "Соединение фазных шин". Все свойства данного соединения будут удалены. Если соединение обладает видом представления "Трехмерный чертеж монтажных поверхностей", из него образуется немаршрутизируемое соединение. После этого удалите сформированные немаршрутизируемые соединения в навигаторе соединений.
Удалить соединение фазных шин |  С помощью этого пункта всплывающего меню уже определенное соединение фазных шин снова отменяется. Определение функции изменяется обратно на "Жила / Провод". Вследствие этого удаляются все свойства, записанные на соединении фазных шин. Для соединения с определением функции "Жила / Провод" необходимо заново выполнить маршрутизацию в пространстве листа.
Присвоить фазную шину |  Открывает диалоговое окно Присвоить фазную шину, в котором соединению фазных шин можно присвоить изделие. Кроме того, можно ввести или выбрать ОУ для соединения фазных шин.
Выделить все |  Выделяет все записи в таблице.
Подобрать ширину столбца |  Все столбцы так подогнаны к таблице по ширине, что и заголовок, и содержимое столбцов полностью читаемы.
Конфигурировать представление |  Открывает диалоговое окно Конфигурировать представление, в котором можно задать, какие свойства должны отображаться в таблице. Таким образом вы получите возможность отобразить в таблице дополнительные свойства соединения.

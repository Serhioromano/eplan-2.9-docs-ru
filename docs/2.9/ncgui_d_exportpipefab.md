# Диалоговое окно Экспорт изгибания труб

Вы открыли проект. В пространстве листа или в навигаторе соединений выделены свободно маршрутизируемые трубопроводы, которые необходимо экспортировать в систему изгибания труб. Сервисные программы > Данные изготовления > Механическая обработка > Изгибание труб.

В этом диалоговом окне задаются настройки экспорта данных для системы управления машинами по изгибанию.

Обзор основных элементов диалогового окна:

### Машина

В комплекте поставки содержится определение машины с названием "Стандарт". Если в настройках фирмы определены другие схемы, можно выбрать из раскрывающегося списка другие определения машины.

Кнопка ++...++ открывает диалоговое окно Настройки: Экспорт изгибания труб, в котором можно изменить настройки схемы или определить новую схему.

### Каталог вывода

В этом поле выводится место хранения. Здесь по умолчанию выбран каталог, который указан в настройках пользователя экспорта изгибаний труб.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Целевой файл

При экспорте изгибания труб образуются несколько файлов. Поле отключено и действуют условия генерирования имен файлов, определенные в схеме.

# Нумерация клемм / контактов штекера с ориентацией на страницы альтернативно выравниванию рамки

При нумерации клемм или контактов штекера в поле Последовательность можно установить требуемую последовательность нумерации. Путь меню: Данные проекта > Клеммники > Нумеровать клеммы. Нумерация клемм / контактов штекера по страницам ранее была возможна через настройку "C ориент. на страницы" в соответствии с направлением создания отчета для рамки. Например, при работе с проектами по стандарту IEC нумерация осуществляется по столбцам, а с проектами по стандарту NFPA — по строкам. С помощью новой настройки "С ориентацией на страницы (альтернативно выравниванию рамки)" теперь можно задать, чтобы клеммы / контакты штекера нумеровались по страницам, при этом последовательность является альтернативной направлению создания отчета для рамки. Например, при работе с проектами по стандарту IEC нумерация осуществляется по строкам, а с проектами по стандарту NFPA — по столбцам.

![](images/benefit.png)Эффект:

С помощью новой настройки последовательности нумерации клеммы / контакты штекера при нумерации с ориентацией на страницы теперь могут нумероваться по столбцам или строкам (на выбор).

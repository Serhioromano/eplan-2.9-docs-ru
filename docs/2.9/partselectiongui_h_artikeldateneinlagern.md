# Синхронизировать сохраненные изделия с базой данных изделий

Сохраненные свойства изделия являются неотъемлемой частью проекта, тогда как к несохраненным свойствам обращаются только через Базу данных изделий.

Конфигурация экономических полей данных для выбора изделия действительна также и для выбора устройств, т. е. существует общая конфигурация.

### Обновить изделия в проекте

Сервисные программы > Изделие > Обновить текущий проект

!!! info "Для сведения:"

    Сохраненные в проекте изделия обновляются данными из внутренней базы данных изделий. Кроме того, более новые данные устройства переносятся на главные функции, использующие эти изделия.

!!! info "Для сведения:"

    Возможные сообщения записываются в управление сообщениями.

### Синхронизировать изделие

1. Выберите следующие пункты меню: 'Сервисные программы' > 'Изделие' > 'Синхронизировать текущий проект'

!!! info "Для сведения:"

    Открывается диалоговое окно [Синхронизация изделий — <Имя проекта>](partsextensiongui_d_artikelabgleich.md).

2. Выберите из раскрывающегося списка Фильтр требуемый фильтр. Вы можете либо отобразить изделия, которые различаются в проекте и / или основных данных системы, либо отобразить только старые основные данные проекта или только старые основные данные системы.
3. В списке Сохраненные изделия выделите изделия, которые необходимо перенести в основные данные системы, и нажмите кнопку со стрелкой.

!!! info "Для сведения:"

    Сохраненные в проекте изделия переносятся во внутреннюю базу данных изделий. При этом шаблоны функции сохраненных изделий сохраняются на изделиях внутренней базы данных.

4. Выделите в списке Изделие системы изделия, которые необходимо перенести в проект, и нажмите кнопку со стрелкой.

!!! info "Для сведения:"

    Выбранные изделия сохраняются в проекте. Кроме того, более новые данные устройства переносятся на главные функции, использующие эти изделия.

!!! warning "Предупреждение:"

    При переносе сохраненных изделий в основные данные системы синхронизируются только изделия, еще не содержащиеся в базе данных изделий. Уже имеющиеся изделия не обновляются.

!!! tip "Совет:"

    Меню ++Дополнительно++ > Обновить текущий проект позволяет заменять все устаревшие сохраненные изделия новыми изделиями системы.
    Меню ++Дополнительно++ > Дополнить текущую систему позволяет дополнять основные данные системы всеми сохраненными в проекте изделиями, которые пока отсутствуют в системе.

# Диалоговое окно Настройки: Символы соединения

Параметры > Настройки > Пользователь > Графическая обработка > Символы соединения.

Укажите в этом диалоговом окне настройки для ***черчения*** символов соединений.

Обзор основных элементов диалогового окна:

### Отображать ответвления соединений

В этом групповом поле указывается, каким образом будут начерчены символы соединений при вставке в схему соединений.

* С определением цели: При вставке символа соединения настройка Определение целей по умолчанию предварительно заполнена. Выполняется черчение "обычных" символов соединений, по которым видна последовательность целей соединения.
* Как точка: При вставке символа соединения настройка Начертить как точку по умолчанию предварительно заполнена. Ответвления или перекрестные соединения обозначаются как точка соединения.
* С определением цели или как точка: При вставке символа соединения всегда выводится диалоговое окно запроса. Здесь необходимо выбрать вариант угла символа соединения и решить, должен ли он быть начерчен с определением цели или как точка. В качестве настройки по умолчанию используется опция, выбранная при предыдущем вызове этого диалогового окна.

!!! note "Замечание:"

    Укажите в настройках проекта (Параметры > Настройки > Проекты > "Имя проекта" > Графическая обработка > Общее ), как в схеме соединений должны быть ***представлены*** символы соединений.

### Символы соединений с диалоговым окном запроса

Если этот флаг установлен, при вставке символов соединений всегда выводится диалоговое окно запроса. Здесь необходимо выбрать вариант угла символа соединения и решить, должен ли он быть начерчен с определением цели или как точка. Если в групповом поле Отображать ответвления соединений выбрана опция С определением цели или как точка, этот флаг устанавливается автоматически и становится недоступным.

Если этот флаг снят, символы соединений чертятся автоматически в соответствии с установленной настройкой. Варианты углов можно выбрать, удерживая нажатой клавишу ++Ctrl++ и повернув символ с помощью мыши.

При настройке С определением цели или как точка всегда выводится диалоговое окно запроса.

### Выключить свойство 'Определение целей' при вставке точки определения сети

Если этот флажок установлен, при вставке точек определения сети свойство Определение целей отключено.

### Перенести обозначения соединений на периметр

Если этот флажок установлен, при изменении обозначения соединения ***вручную*** (в навигаторе соединений или на точке определения соединения) отображается диалоговое окно Перенести обозначения соединений. Здесь вам дается возможность автоматически перенести измененное обозначение соединения на другие соединения. Обозначение соединения можно перенести на все соединения потенциала, сигнала или сети.

Если флажок снят, измененные обозначения соединений не переносятся автоматически на другие соединения.

!!! note "Замечание:"

    * При переносе обозначений соединений учитывается фильтр, который определен для нумерации соединений. Этот фильтр устанавливается в диалоговом окне Настройки: Нумерация соединений на вкладке Фильтр (перейти в него можно, например, по пути: Данные проекта > Соединения > Нумерация > Настройки).
    * Настройка Перенести обозначения соединений на периметр не действует, если обозначения соединений изменяются через нумерацию соединений.

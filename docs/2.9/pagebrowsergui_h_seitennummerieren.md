# Нумеровать и переименовать страницы

С помощью команды Нумеровать можно заново пронумеровать страницы. Благодаря этому можно ***переименовать*** страницы проекта, тем самым ***переместить*** их.

В случае автоматического указания имен страниц EPLAN предлагает следующее свободное имя страницы в целевом проекте. Для подстраниц подсчет выполняется по нарастающей. В случае проектов DIN предложение имени страницы относится к группе. Предложенные (начальные) имена страниц можно оставить или изменить. При нумерации нескольких страниц можно изменить настройку величины шага.

Условие:

Откройте проект и выделите одну или несколько страниц в диалоговом окне навигатора Страницы — <Имя проекта>.

1. Страница > Нумерация
2. Введите новый начальный номер и величину шага (а при необходимости и другие настройки нумерации страниц) в диалоговом окне [Нумеровать страницы](pagebrowsergui_d_seitennummerierung.md).
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Нумерация страниц выполняется в том случае, если при этом не требуется перезаписать имеющиеся страницы.

!!! info "Для сведения:"

    Если из-за настроек начального номера и величины шага должны быть перезаписаны имена страниц, открывается диалоговое окно [Нумеровать страницы: Предварительный просмотр результата](pagebrowsergui_d_seitennummerierungsvorschau.md).

4. Измените соответствующие свойства страницы.Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выполняется новая нумерация страниц.

### Переименовать отдельную страницу

Условие:

Проект открыт, и страница в древовидном представлении в диалоговом окне навигатора Страницы — <Имя проекта> выделена.

1. Страница > Переименовать
2. Введите имя страницы и нажмите клавишу ++ВВОД++.

!!! info "Для сведения:"

    Страницы переименовываются, при этом описание страниц сохраняется.

!!! tip "Совет:"

    * В представлении в виде списка навигатора страниц некоторые поля (Описание страниц, Номер чертежа и т. д.) можно обрабатывать прямо в таблице.
    * В представлении в виде дерева навигатора страниц страницы можно копировать и перемещать с помощью перетаскивания мышью. Для копирования удерживайте клавишу ++Ctrl++ нажатой.

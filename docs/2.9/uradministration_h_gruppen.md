# Управление группами пользователей

Управление правами позволяет создавать группы пользователей и присваивать им определенные права. При необходимости созданные группы пользователей можно будет изменить или удалить.

!!! note "Замечание:"

    Невозможно обработать или удалить предварительно определенные в EPLAN группы пользователей "Администраторы" и "Гости".

### Создать группу пользователей

1. Выберите следующие пункты меню: Сервисные программы > Управление правами
2. Выделите слева в диалоговом окне Управление правами папку Группы и нажмите ![](images/all_new_as.png){: .ui-icon } (Создать).
3. Введите в диалоговом окне Группа имя и краткое описание возможностей обработки, которыми будет обладать новая группа пользователей. В отличие от имен пользователей, в именах групп различаются прописные и заглавные буквы, т. е. группа "QS" и группа "Qs" — это разные группы.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Группа создана, но ей еще не присвоены права на обработку.

5. Присвойте группе права на обработку, следуя инструкциям в разделе "Присвоение группам прав".

### Копировать группу пользователей

1. Выберите следующие пункты меню: Сервисные программы > Управление правами
2. Выделите слева в диалоговом окне Управление правами папку Группы.
3. Выделите в правом поле группу пользователей и щелкните по ![](images/all_copy_as.png){: .ui-icon } (Копировать).
4. Введите в диалоговом окне Группа имя и краткое описание возможностей обработки, которыми будет обладать новая группа пользователей. В отличие от имен пользователей, в именах групп различаются прописные и заглавные буквы, т. е. группа "QS" и группа "Qs" — это разные группы.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Группа будет создана, и ей присвоены права выделенной прежде группы пользователей.

6. При необходимости измените права на обработку, следуя инструкциям в разделе "Присвоение группам прав".

### Обработать группу пользователей

1. Выберите следующие пункты меню: Сервисные программы > Управление правами
2. Выделите слева в диалоговом окне Управление правами папку Группы.
3. Выделите в правом поле (самоопределяемое) группу пользователей и нажмите на ![](images/all_edit_as.png){: .ui-icon } (Обработать).
4. Измените в диалоговом окне Группа имя или описание.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Диалоговое окно закрывается, а измененные данные выводятся на просмотр в диалоговом окне Управление правами.

### Удалить группу пользователей

1. Выберите следующие пункты меню: Сервисные программы > Управление правами
2. Выделите слева в диалоговом окне Управление правами папку Группы.
3. Выделите в правом поле (самоопределяемую) группу пользователей и нажмите на ![](images/all_delete_as.png){: .ui-icon } (Удалить).

!!! info "Для сведения:"

    После подтверждения предупреждающего сообщения имя группы будет удалено.

### Присвоение группам прав

1. Выберите следующие пункты меню: Сервисные программы > Управление правами
2. В диалоговом окне Управление правами щелкните рядом с папкой Группы знак плюс, чтобы открыть папку, и выделите группу пользователей.

!!! info "Для сведения:"

    Справа отображаются возможные права, сгруппированные по тематическим областям.

3. Щелкните на знаке плюса перед тематической областью, чтобы развернуть список прав.

!!! info "Для сведения:"

    Права подразделяются на права изменения пунктов меню, права доступа к диалоговым окнам и права на осуществление API-действий. Права доступа к диалоговым окнам отмечаются дополнением "Диалоговое окно". Права на API-действия отмечаются дополнением "Интерфейс".

4. Установите флажок перед правами, которыми должна обладать данная группа пользователей и удалите флажок перед правами, которыми она обладать не должна.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Активированные права присваиваются данной группе пользователей.

!!! info "Для сведения:"

    Если для какой-либо тематической области активированы не все права, флажок тематической области будет отображаться серым цветом.

!!! tip "Совет:"

    Вы можете присвоить или аннулировать сразу все права в одной тематической области, устанавливая или убирая флажок перед этой тематической областью.

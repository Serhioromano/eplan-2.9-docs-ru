# Обработать изделие спецификации

Условия:

* Вы открыли проект.
* Вы открыли навигатор спецификаций: Данные проекта > Устройство / изделие > Навигатор спецификаций

1. В диалоговом окне Спецификация — <Имя проекта> выберите вкладку Список.
2. Выберите пункт всплывающего меню Конфигурировать представление.
3. В диалоговом окне Конфигурировать представление установите флажок Размещение и щелкните по ++OK++.

!!! info "Для сведения:"

    Представление в виде списка увеличивается на ширину столбца Размещение.

4. Выберите пункт всплывающего меню Обработать изделие для обрабатываемых неразмещенных изделий, т. е. для изделия, которое ***не*** имеет записи в столбце Размещение. Возможен многократный выбор.
5. Щелкните в поле Номер изделия диалогового окна Обработать изделие по ++...++.
6. В диалоговом окне Выбор изделия выберите при необходимости другое изделие. Многократный выбор ***невозможен***. Если выделены несколько изделий, перенесено будет только самое верхнее изделие из выделенных.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Номер изделия переносится в поле Номер изделия.

8. При необходимости обработайте другие свойства.
9. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Данные изделия переносятся на устройство.

Понятие "Размещение изделия" обозначает размещенное изделие на странице компоновки электрошкафа. На число штук изделия возможно одно размещение. Виды размещения изделия распознаются по записи "Компоновка электрошкафа" в столбце Вид представления представления в виде списка навигатора спецификаций.

1. В диалоговом окне Спецификация — <Имя проекта> выберите вкладку Список.
2. Выберите пункт всплывающего меню Конфигурировать представление.
3. В диалоговом окне Конфигурировать представление установите флажки Размещение и Вид представления и щелкните по ++OK++.

!!! info "Для сведения:"

    Представление в виде списка увеличивается на ширину столбцов Размещение и Вид представления.

4. Выполните двойной щелчок по обрабатываемому изделию, которое уже было размещено и имеет в столбце Вид представления запись "Компоновка электрошкафа".
Или выберите пункт всплывающего меню Свойства (в этом случае возможен многократный выбор).
5. Установите в диалоговом окне Свойства ++...++: Размещение изделия необходимые настройки для вкладок Размещение изделия, Отображение и Формат.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Данные изделия переносятся на устройство.

Покупными изделиями являются изделия, которые не присвоены специальному устройству. В представлении в виде списка навигатора спецификаций покупное изделие распознается по тому, что оно ***не имеет*** записи в столбце Обозначение устройства или указывается в столбце Покупные изделия как таковое.

1. В диалоговом окне Спецификация — <Имя проекта> выберите вкладку Список.
2. Выберите пункт всплывающего меню Конфигурировать представление.
3. В диалоговом окне Конфигурировать представление установите флажок Изделие проекта и щелкните по ++OK++.

!!! info "Для сведения:"

    Представление в виде списка увеличивается на ширину столбца Изделие проекта.

4. Выполните двойной щелчок по изделиям проекта, которые необходимо обработать, и присвойте им соответствующие данные в диалоговом окне Обработать изделие.
5. Щелкните по кнопке ++OK++.

1. В диалоговом окне Спецификация — <Имя проекта> выберите вкладку Список.
2. Выберите пункт всплывающего меню Конфигурировать представление.
3. В диалоговом окне Конфигурировать представление установите флажок Размещение и щелкните по ++OK++.

!!! info "Для сведения:"

    Представление в виде списка увеличивается на ширину столбца Размещение.

4. Выберите пункт всплывающего меню Разместить устройство для обрабатываемого неразмещенного изделия, т. е. для изделия, которое ***не*** имеет записи в столбце Разместить устройства. Многократный выбор ***невозможен***.
5. Непосредственно перед размещением нажмите клавишу ++Backspace++.

!!! info "Для сведения:"

    Откроется диалоговое окно Вставить устройство.

6. Выберите опцию Макрос изделия, если в данных изделия указаны как минимум один графический макрос и сведения о ширине и высоте; в этом случае других запросов не отображается, и изделие можно разместить в графике, как описано в следующем шаге.
Если выбрана опция Выбор символа, выберите символ, который необходимо присвоить изделию, из диалогового окна Выбор символа и щелкните по ++OK++.
Если выбрана опция Отдельные функции, выберите функцию, которую необходимо присвоить изделию, из диалогового окна Определения функций и нажмите ++OK++.
Если выбрана опция Выбор макроса, выберите макрос, который необходимо присвоить изделию, из диалогового окна Выбрать макрос и щелкните по ++Открыть++.
7. Разместите изделие в графике.

Замена изделия осуществляется при замене имеющихся изделий спецификации на другие.

1. В диалоговом окне Спецификация — <Имя проекта> выделите заменяемые изделия.
2. Выберите пункт всплывающего меню Заменить изделие.
3. В диалоговом окне Выбор изделия выберите сменное изделие. Многократный выбор ***невозможен***.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Изделия, присвоенные ранее устройству, заменяются на вновь выбранное изделие. При этом количество изделий сохраняется. Узлы также сохраняются.

!!! example "Пример:"

    Устройство со ссылкой изделия -> Результат после операции заменыK1, 0815заменено на0915->K1, 0915K1, 0815; K2, 0816заменены на0915->K1, 0915; K2, 0915

1. В диалоговом окне Спецификация — <Имя проекта> выделите удаляемые изделия.
2. Выберите пункт всплывающего меню Удалить.
3. Выберите необходимый ответ в запросе.

!!! info "Для сведения:"

    В зависимости от ответа на запрос выделенные ссылки изделия удаляются. Соответствующая позиция ссылки изделия в устройстве после этой операции пуста и не заполняется.

# Вкладка "Номинальный диаметр"

У вас установлен конфигуратор шлангопроводов в качестве дополнительного модуля для EPLAN Fluid. Вы запустили EPLAN Fluid и конфигуратор шлангопроводов.

* Выберите в EPLAN Fluid пункты меню Параметры > Настройки > Пользователь > Add-ons > Конфигуратор шлангопроводов и щелкните в следующем диалоговом окне по вкладке Номинальный диаметр.
* Или выберите в диалоговом окне Конфигурировать шлангопроводы конфигуратора шлангопроводов пункты меню Дополнительно > Настройки и щелкните в следующем диалоговом окне по вкладке Номинальный диаметр.

Или у вас установлен конфигуратор шлангопроводов в качестве самостоятельной программы, и программа запущена.

* Выберите в конфигураторе шлангопроводов пункты меню Параметры > Настройки > Пользователь > Add-ons > Конфигуратор шлангопроводов и щелкните в следующем диалоговом окне по вкладке Номинальный диаметр.

На этой вкладке определите значения по умолчанию для номинального диаметра используемых шлангов. Сохраненные в таблице значения предлагаются вам на выбор при конфигурировании шлангопроводов в диалоговом окне Обработать ++...++.

### Список

Значения номинальных диаметров всегда указываются в "мм". Поэтому их можно вводить без указания единицы измерения.

Используйте кнопки панели инструментов над списком для создания новых записей, удаления ненужных и/или обработки имеющихся.

!!! note "Замечание:"

    Если выбрана стандартная схема DIN 20066_2012-01, список заполняется значениями по умолчанию из стандарта DIN 20066, которые присвоены стандартному номинальному диаметру шланга "d1".

# Диалоговое окно Список устройств

Вы открыли проект. Данные проекта > Устройство / изделие > Список устройств.

В этом диалоговом окне обрабатывается список устройств. Список устройств контролирует все задействованные в проекте изделия. Диалоговое окно при открытии автоматически обновляется; обновить вручную всегда можно при помощи ++F5++.

Список устройств по умолчанию сортируется по части установки и описанию изделия. Возможна сортировка при помощи заголовков столбцов.

Обзор основных элементов диалогового окна:

### Фильтр

В этом раскрывающемся списке отображаются все доступные фильтры. Выбранный фильтр активируется автоматически. Запись "- Не активировано -" отключает фильтр и приводит к тому, что данные отображаются в неотфильтрованном виде. С помощью кнопки ++...++ откройте диалоговое окно [Фильтр](modaldialogsdb_d_filternnach.md). Здесь можно создать, обработать, удалить, скопировать, экспортировать, импортировать фильтр и управлять им.

Во всплывающем меню раскрывающегося списка Фильтр содержатся следующие записи:

* Выключить: Этот пункт меню доступен, если фильтр установлен: Сбрасывает настройку фильтра до записи "- Не активировано -".
* Активировать <Имя фильтра>: Этот пункт меню доступен, если для настройки фильтра установлено значение "- Не активировано -": Повторно активирует последний активный фильтр.

Таким образом можно быстро переключаться между неотфильтрованным и отфильтрованным в соответствии с требованиями пользователя представлениями.

### Значение: <Свойство>

При помощи [быстрого ввода](modaldialogsdb_k_filter.md) в данном поле для определенного и активированного фильтра можно быстро изменить значение его критерия.

### Список

В списке устройств могут отображаться следующие свойства (настраивается через Всплывающее меню > Конфигурировать представление):

Свойство |  Содержание
---|---
Имя проекта |  Имя проекта отображается только в том случае, если открыто несколько проектов.
Резерв  |  Содержит значение разности между значениями свойств Разреш. число штук (итого) и Установленное число штук. Изделия, резерв которых < 0, отмечаются в начале строки символом.
Номер изделия  |  Идентифицирует изделие и может быть введен вручную или скопирован из базы данных изделий (или же из импортированного списка).
Часть установки  |  Имя той части установки, в которую встроено устройство (напр., "Элемент роликового конвейера 15"). Не считывается из базы данных изделий, но должно вводится дополнительно.
Описание устройства |  Описание функциональности. Не считывается из базы данных изделий, но должно вводится дополнительно.
Разреш. число штук |  Указывает, как часто изделие может задействоваться в проекте.
Разреш. число штук (итого) |  Содержит сумму всех разрешенных чисел штук, введенных для соответствующего изделия в список устройств.
Установл. число штук |  Указывает, как часто изделие встречается в проекте. Отклонения относительно разрешенной спецификации допустимы; они отслеживаются при контрольном прогоне.
Номер типа  |  Определяется на основании базы данных изделий согласно номеру изделия.
Номер для заказа  |  Определяется на основании базы данных изделий согласно номеру изделия.
Вариант |  Определяется на основании базы данных изделий согласно номеру изделия.
Обозначение 1 - 3 |  Определяется на основании базы данных изделий согласно номеру изделия.
Производитель |  Определяется на основании базы данных изделий согласно номеру изделия.
Макрос |  Указывает имя макроса, если изделию присвоен макрос.
Раздел |  Показывает первую установленную настройку для раздела, если изделию присвоены разделы.
Подраздел |  Показывает подраздел для введенного текста, если изделию в диалоговом окне 'Свойства' присвоен подраздел.

### Всплывающее меню

Пункт меню |  Значение
---|---
Выделить все |  Выделяет все изделия в списке.
Подобрать ширину столбца |  Все столбцы так подогнаны к таблице по ширине, что и заголовок, и содержимое столбцов полностью читаемы.
Создать |  Генерирует изделие (через диалоговое окно Свойства). Новое изделие отсортировывается в список согласно номеру изделия.
Добавить изделие |  Открывает диалоговое окно [Выбор изделия](partselectiongui_d_artikelauswahl.md). Вы можете добавить одно или несколько изделий в список; они добавляются в конец списка. Возможны многократно встречающиеся изделий.
Корзина Festo |  Предлагает возможность дополнить список устройств изделиями из каталога продуктов Festo.
Удалить |  Удаляет выделенные изделия. После этого контроль за соответствующими изделиями больше не ведется.
Вставить устройство |  Размещает выделенное изделие как устройство. Для этого сначала открывает диалоговое окно [Вставить устройство](egedgui_d_geraeteinfuegen.md), в котором можно задать параметры вставки.
Экспортировать |  Экспортирует все изделия в файл, независимо от выбора. При этом на основании расширения файла соответствующего типа определяется вид экспорта (XML, текст, CSV, ...).
Импортировать |  Импортирует изделия из файла. При этом на основании расширения файла соответствующего типа определяется вид импорта (XML, текст, CSV, ...). Из файла считываются поля Номер изделия, Разрешенное число и Описание устройства. Записи добавляются в конец существующего списка устройств.
Перейти к (графика) |  Отображает в графическом редакторе первый размещенный объект, который содержит ссылку на выделенное изделие.
Вставить в список результатов поиска |  Заносит все объекты проекта, содержащие ссылку на выделенные изделия, в список результатов поиска.
Конфигурировать представление |  Открывает диалоговое окно Конфигурировать представление, в котором можно задать, какие свойства должны отображаться в списке.
Свойства |  Открывает диалоговое окно Свойства для выделенных изделий списка. Отображаются и могут быть обработаны все выделенные изделия (групповая обработка). В поле Номер изделия можно запустить выбор изделия, нажав ++...++. В этом случае в рамках выбора изделия допускается выбор только одного отдельного изделия. При изменении номера изделия свойства автоматически обновляются, как только Вы перейдете в другое поле.

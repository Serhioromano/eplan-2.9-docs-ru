# Диалоговое окно Переименовать проект

Вы выделили проект в Навигаторе страниц. Проект > Переименовать.

В этом диалоговом окне вы можете существующему проекту присвоить другое имя.

Обзор основных элементов диалогового окна:

### Старое имя проекта

В этом поле находится имя существующего проекта.

### Новое имя проекта

Введите в этом поле новое имя проекта.

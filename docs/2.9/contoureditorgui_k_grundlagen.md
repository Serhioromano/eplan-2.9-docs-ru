# Редактор контура: Основные положения

Как только в редакторе контуров будет открыт контур, который необходимо обработать, можно начинать обрабатывать его элементы.

В качестве первого шага обработки начертите двухмерную геометрию нового контура. Для этого доступны все разрешенные для контуров графические элементы и функции обработки.

Должны быть выполнены следующие условия:

* Черчение выполняется в масштабе 1:1.
* Использование графических элементов, таких, как линии, ломаные линии, прямоугольники, круги и дуги окружностей.
* Обратите внимание на то, что элементы, составляющие контур, должны быть закрыты во всех местах перехода.

Дальнейший способ действий зависит от типа контура: У контуров типа ***Выдавить контур*** логика определяется в форме так называемых "точек определения контуров". Для этого в редакторе контуров в пунктах меню Вставить > Логика доступны следующие пункты меню, которые запускают специальный диалог для размещения и нумерации точки определения контура:

* Направляющие рейки: Размещаются с любой частотой
* Направляющие рейки с выравниванием: Размещаются с любой частотой
* Монтажные поверхности: Размещаемые попарно по часовой стрелке (внешний контур) / против часовой стрелки (внутренний контур) на прямом сегменте контура
* Монтажные поверхности ЧУ: Эти элементы могут быть размещены на прямом сегменте контура попарно по часовой стрелке (внешний контур) / против часовой стрелки (внутренний контур)
* Размер поля: Размещаемые попарно по часовой стрелке относительно внешнего контура.

Для контуров типа ***Генерирование записи данных контура ЧУ*** существует специальное диалоговое окно — [Запись данных контура ЧУ](contoureditorgui_d_ncdatensatz.md), в данном диалоговом окне для чертежа контура задайте релевантные машинно-зависимые данные. Данные сохраняются на контуре.

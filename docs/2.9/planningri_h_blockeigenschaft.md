# Определение свойств блока для технологического контура / функции ТК

Свойства блока позволяют управлять данными форматов нескольких свойств в одном свойстве. Каждому полю Свойство блока [n] соответствует свойство Свойство блока: Формат [n]. Полю Свойство блока [1] соответствует свойство Свойство блока: Формат [1]. В свойстве Свойство блока: Формат [1] определите данные формата нескольких свойств. В поле Свойство блока [1] отображаются на основе этих форматных данных конкретные данные.

Условия:

* Вы открыли проект, который содержит данные предварительного планирования.
* В Графическом редакторе открыта страница проекта.
* Вы еще не определили свойство Свойство блока: Формат [1].

1. Выделите технологический контур или функцию ТК.
2. Выберите следующие пункты меню: Обработать > Свойства
3. В диалоговом окне Свойства ++...++: Технологический контур или Свойства ++...++: Функция ТК выберите вкладку <Технологический контур> / <Функция ТК>.
4. Нажмите над таблицей Свойства кнопку ![](images/all_new_as.png){: .ui-icon } (Создать).
5. В диалоговом окне Выбор свойств выделите свойство Свойство блока: Формат [1] и щелкните ++OK++.

!!! info "Для сведения:"

    Свойство отображается в столбце Имя свойства.

6. Щелкните соседний столбец Значение и кнопку ++...++.
7. Выделите в диалоговом окне Формат в поле Доступные элементы форматов свойство Технологический контур или Функция ТК и щелкните по ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо).

!!! info "Для сведения:"

    В диалоговом окне Формат: Свойство блока отображаются все свойства, доступные для технологических контуров или функций ТК.

8. Выделите запись в списке под полем Категория.
9. Чтобы определить для текста свойства количество местоположений и знаки заполнителей для пустых мест, введите в полях Мин. число разрядов и Заполнитель необходимые значения.
10. Для определения того, когда свойство будет удалено, установите условия в групповом поле Удалить свойство при.
11. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Данные формата свойства отображаются в поле Выбранные элементы формата.

12. Таким же образом определите другие данные формата.
13. После определения формата свойства блока в диалоговом окне Формат щелкните ++OK++.

!!! info "Для сведения:"

    Данные формата свойства Свойство блока: Формат [1] отображаются в столбце Значение.

14. Щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
15. Выделите в диалоговом окне Выбор свойства в группе Свойство блока свойство Свойство блока [1] и щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Рядом со Свойством блока [1] отображаются конкретные значения свойства. Если свойство не обновляется, необходимо закрыть и снова открыть диалоговое окно.

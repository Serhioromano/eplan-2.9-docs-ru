# Обзор свойств

Так называемый ***Обзор свойств*** работает как навигатор и может присоединяться к различным позициям в главном окне EPLAN. Присоединяемое диалоговое окно обзора свойств позволяет быстро обрабатывать свойства различных объектов без необходимости открывать и закрывать несколько диалоговых окон 'Свойства' одно за другим. Это присоединяемое диалоговое окно можно сконфигурировать так, чтобы отображались только те свойства, которые важны для вас.

В обзоре свойств можно обрабатывать свойства отдельного объекта или свойства нескольких выделенных объектов одновременно. Поскольку объекты (тексты, графические элементы, условные обозначения, сегменты) иногда имеют одинаковые свойства, свойства в представлении структуры дерева сгруппированы в соответствии с их исходными объектами.

### Форматировать несколько графических элементов одновременно

Обзор свойств можно использовать, например, для совместного выделения нескольких графических элементов (прямоугольников, линий и т. д.) и их одновременного форматирования. Схема "Отформатировать график" доступна в обзоре свойств для форматирования графических элементов. При совместном форматировании можно изменять такие свойства, как Толщина линии, Цвет, Тип линии и т. д.

### Обработать независимые от функций свойства условных обозначений

Обзор свойств также позволяет использовать специальную форму групповой обработки. При этом отображаются и обрабатываются общие для всех различных условных обозначений свойства. Это называется [функционально-независимой групповой обработкой](multipleselection_k_prinzip.md). Для этого можно использовать схему "Независимые от функций свойства" в обзоре свойств.

# Вкладка Настройки (объект-заполнитель)

Обзор основных элементов диалогового окна:

В Графическом редакторе открыта страница проекта. Либо вы открыли пространство листа.

* Вставить > Объект-заполнитель. Выделите область с объектами, которые должны принадлежать объекту-заполнителю. В диалоговом окне Свойства ++...++ выберите вкладку Настройки.
* Дважды щелкните по символу "Якорь" уже созданного объекта-заполнителя. В диалоговом окне Свойства ++...++ выберите вкладку Настройки.
* Данные проекта > Опции > Объекты-заполнители. Выделите в диалоговом окне Объекты-заполнители — <Имя проекта> объект-заполнитель и выберите пункт всплывающего меню Свойства. В диалоговом окне Свойства ++...++ выберите вкладку Настройки.

На этой вкладке задаются настройки для использования объектов-заполнителей в EPLAN Cogineer. В EPLAN Cogineer объекты-заполнители используются в макросах для изменения данных при генерировании схем соединений или данных на сгенерированных страницах схем соединений.

Обзор основных элементов диалогового окна:

### Использовать объект-заполнитель в EPLAN Cogineer

Если этот флажок установлен, объект-заполнитель становится доступным после добавления макроса в EPLAN Cogineer и учитывается при генерации схемы соединения в EPLAN Cogineer. В этом случае значения, присвоенные в EPLAN Cogineer, отображаются на объекте-заполнителе на сгенерированной странице схемы соединений.

Если флажок не установлен, в EPLAN Cogineer объект-заполнитель недоступен. В этом случае объект-заполнитель отображается на сгенерированной странице схемы соединения без изменений. На странице схемы соединений можно в дальнейшем присваивать набор значений.

### Удалить объект-заполнитель после генерирования

Этот флажок доступен, только если была активирована настройка Использовать объект-заполнитель в EPLAN Cogineer.

Если этот флажок установлен, объект-заполнитель будет удален после генерации схемы соединений с помощью EPLAN Cogineer из сгенерированной схемы соединений.

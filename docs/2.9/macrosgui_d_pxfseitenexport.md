# Диалоговое окно «Внешняя обработка»

Вы открыли проект. Установлены настройки для внешней обработки данных (Параметры > Настройки > Пользователь > Интерфейсы > Внешняя обработка). Сервисные программы > Внешняя обработка свойств > Экспортировать данные.

В этом диалоговом окне можно указать, как и куда должны быть экспортированы выбранные данные. Вывод зависит от типа объекта, который был задан при создании схемы и после этого уже не подлежит изменению.

Обзор основных элементов диалогового окна:

### Настройки

В этом раскрывающемся списке отображаются все пользовательские и предварительно определенные схемы для внешней обработки. Для обработки или создания схемы нажмите кнопку ++...++.

### Тип объекта

Схема для внешней обработки всегда создается вместе с типом объекта. В этом поле отображается тип объекта, выбранный при создании текущей схемы. Тип объекта затем уже ***нельзя*** изменить!

### Язык

Выберите из раскрывающегося списка, на каком языке должен выполняться экспорт:

* Все языки отображения: В случае этой настройки многоязычные данные всегда выводятся многократно (в зависимости от числа языков, установленных в проекте). При импорте заменяются только импортированные языки, другие имеющиеся языки не изменяются. В этом случае затем требуется запустить соответствующий прогон перевода.
* Имеющиеся в проекте отдельные языки.

### Целевой файл

Введите в это поле путь и имя целевого файла или щелкните по ++...++, чтобы выполнить интерактивный выбор пути и каталога.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

!!! note "Замечание:"

    Обратите внимание на то, что при экспорте ***страниц*** переменные пути для "Полный каталог проекта" и "Имя проекта" пусты, поэтому их можно постоянно изменять в зависимости от выбранного целевого файла.

### Вид вывода

Выберите в этом групповом поле одну из доступных опций вывода:

* Экспортировать: Выберите эту опцию, если соответствующие данные требуется только экспортировать, например, чтобы обработать их на другом рабочем месте.
* Экспортировать и запустить приложение: Выберите эту опцию, если после экспорта следует сразу запустить соответствующее приложение для внешней обработки, например, если данные необходимо обработать на другом рабочем месте. Однако перед этим требуется просмотреть их и при необходимости переработать.
* Обработать во внешнем приложении и считать обратно: Выберите эту опцию, чтобы обработать данные во внешнем приложении и сразу же считать их обратно в проект при закрытии этого приложения.

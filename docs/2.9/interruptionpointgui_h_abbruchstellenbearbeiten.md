# Обработать точки разрыва

Условие:

Вы открыли проект. В Графическом редакторе открыта страница проекта.

1. Откройте диалоговое окно Свойства для точки разрыва, например, двойным щелчком по точке разрыва.
2. На вкладке [Точка разрыва](devicetaggui_r_abbruchstelle.md) диалогового окна Свойства ++...++ в поле Видимое ОУ введите обозначение устройства, которое должно отображаться в графическом редакторе у точки разрыва; или щелкните по кнопке ++...++, чтобы открыть диалоговое окно Использ. имеющ. точку разрыва и выбрать в нем имеющееся имя точки разрыва.
3. При необходимости щелкните в поле Полное ОУ по кнопке ++...++, чтобы открыть диалоговое окно Полное ОУ и, например, выбрать другое место установки.
4. При необходимости введите другие свойства во вкладке Точка разрыва и в других вкладках.
5. Щелкните по кнопке ++OK++.

!!! tip "Совет:"

    Нижестоящее ОУ не вносится по умолчанию в точки разрыва. Однако это можно сделать путем изменения структуры всего проекта в свойствах проекта и активации [расширенного обозначения ссылки](projectstructure_k_referenzkennzeichnung.md). Для этого во вкладке [Структура](prjmanagementgui_r_struktur.md) установите флажок Расширенное обозначение ссылки. После этого в диалоговом окне Полное ОУ поля обработки для ОУ представлены в виде таблицы. Здесь следует ввести ОУ и нижестоящее ОУ в отдельные ячейки.

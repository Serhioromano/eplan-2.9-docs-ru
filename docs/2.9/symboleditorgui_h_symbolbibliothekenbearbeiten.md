# Обработать библиотеки символов

!!! warning "Предупреждение:"

    Некоторые библиотеки символов, такие как, например, SPECIAL.slk, заблокированы и недоступны для обработки. При попытке открыть такую библиотеку символов появится сообщение и процесс будет прерван.

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Сервисные программы > Основные данные > Библиотека символов > Открыть
2. В диалоговом окне Открыть библиотеку символов выберите из списка требуемую библиотеку символов.Возможен многократный выбор.
3. Щелкните по кнопке ++Открыть++.
4. В следующем диалоговом окне Выбор символа отметьте открываемые символы.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Каждый открытый символ отображается в отдельном редакторе символов и может там обрабатываться.

!!! note "Замечание:"

    Через настройку проекта (Параметры > Настройки > Проекты > "Имя проекта" > Управление > Библиотеки символов) проекту можно присвоить любое количество библиотек символов. Затем выбранные библиотеки символов сохраняются в проекте.

!!! tip "Совет:"

    Открыв библиотеку символов и выбрав в диалоговом окне Выбор символа символ, который следует открыть, вы можете 'пролистать' в редакторе символов все символы библиотеки символов. Воспользуйтесь для этого пунктами меню Страница > Следующая страница и Страница > Предыдущая страница или нажмите для пролистывания кнопки ++Рисунок вниз++ и [Рисунок вверх]. При этом отобразится только вариант, который вы выделили в предварительном просмотре в диалоговом окне Выбор символа. Если у символа отсутствует настроенный вариант, то редактор символов отображает пустую страницу с текстом 'Вариант не существует'.

# Диалоговое окно Конфигурировать представление (дерево)

Вы открыли проект и находитесь в представлении структуры дерева определенных навигаторов. Выберите пункт всплывающего меню Конфигурировать представление.

В этом диалоговом окне можно определять отображение и последовательность соответствующих данных таким образом, чтобы выводилась только релевантная информация.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Конфигурация

В столбце Объект перечисляются объекты (проекты, страницы, устройства, функции, структурные идентификаторы и т. д.), которые можно отобразить в соответствующем дереве.

В столбце Формат блока отображается, предусмотрено ли форматирование для объекта. С помощью форматирования определяется, какие свойства будут отображаться для объекта. С помощью кнопки ++…++ можно выбирать другие элементы формата (свойства, разделители) и задавать их последовательность.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

!!! note "Замечание:"

    Определенные свойства ***всегда*** отображаются. Например, свойство Имя проекта также отображается в представлении структуры дерева, если для столбца Формат блока не задан формат.

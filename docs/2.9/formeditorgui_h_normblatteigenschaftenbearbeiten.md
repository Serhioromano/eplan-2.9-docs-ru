# Обработать свойства рамок

Диалоговое окно Свойства рамки — <Имя рамки> открывается автоматически при создании рамки; также впоследствии оно может быть вызвано для ***открытой*** рамки.

Условия:

* Вы открыли проект.
* Открыта рамка в редакторе рамок (Сервисные программы > Основные данные > Рамка > Открыть > [Открыть]).

1. В диалоговом окне навигатора Страницы — <Имя проекта> выделите рамку, свойства которой необходимо просмотреть или обработать.
2. Выберите пункт всплывающего меню Свойства.
3. В диалоговом окне Свойства рамки — <Имя рамки> из раскрывающегося списка Категория выберите свойства, которые должны быть выведены в списке ниже.
4. Определите значения для свойств, либо введя или перезаписав запись или же выбрав значение по умолчанию из раскрывающегося списка.
5. Щелкните по ![](images/all_new_as.png){: .ui-icon } (Создать) в панели инструментов над таблицей Имя свойства / Значение.
6. Выберите в диалоговом окне Выбор свойств новое свойство, которое Вы хотите присвоить рамке.
7. Щелкните по кнопке ++OK++.
8. Присвойте свойству значение.
9. Выполните аналогичные действия для всех других свойств.
10. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Рамка выводится с измененными / новыми данными в редакторе рамок.

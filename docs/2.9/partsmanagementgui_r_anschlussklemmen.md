# Вкладка Выводы устройства

Сервисные программы > Изделие > Управление. В дереве выделен объект, относящийся к уровню иерархии Схема соединений.

В этой вкладке определяется расположение (т. е. группа) выводов устройства. Задайте пространственное расположение и специфические данные выводов устройства, относящихся к одной схеме соединений. На вкладке Технические данные изделий можно использовать сохраненные здесь данные.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую вы хотели бы использовать для отображения данных вывода устройства. Или щелкните по кнопке ++...++, чтобы в диалоговом окне [Настройки: Конфигурация столбцов](modaldialogsdb_d_schemaspaltenkonfiguration.md) обработать схему или создать новую.

* * *

Таблица

По умолчанию в таблице отображаются перечисленные далее данные выводов устройств. Выбрав другую схему, можно повлиять на это отображение.

### Обозначение штекера:Имя интерфейса шины

Введите здесь ОУ штекера, с которым связан вывод устройства. Обозначение штекера поддерживает идентификацию выводов устройства ПЛК или выводов устройства. Для выводов шины для идентификации также используется имя интерфейса шины.

### Уровень

Если вывод устройства представляет собой многоуровневую клемму, запишите сюда уровень.

### Внутр. / внешн. индекс

Указывает для вывода устройства на схеме соединений, какой по счету внутренний или внешний вывод устройства он представляет.

### Позиция Х

В полях от Позиция X до Позиция Z задайте значения координат выводов устройства.

!!! note "Замечание:"

    При указании выводов устройства следует следить за тем, чтобы они находились в рамках геометрических размеров устройства. Если выводы устройства выходят за рамки устройства, соединения не будут маршрутизированы.

### Направление подсоединения

В этом поле задается направление, в котором маршрутизируемое соединение после выхода из вывода устройства предпочтительно должно искать вход в сеть соединенных сегментов.

### Вектор X

В полях от Вектор X до Вектор Z задайте значения для вектора вывода устройства, в направлении которого маршрутизируемое соединение выходит из вывода устройства. Значения задают преимущество, с которым вектор указывает в направлении соответствующей оси.

!!! example "Пример:"

    Оси X, Y и Z расположены под углом 90° друг к другу.Значение вектора 1 означает: параллельная ориентация по отношению к соответствующей оси.Значение вектора 0 означает: ось не влияет на выравнивание вектора.Значение вектора между 0 и 1 смещает влияние оси в диапазоне между 0° и 90° (0,5 = 45°, 0,25 = 22,5°). Также возможны отрицательные значения.Если у всех осей одно и то же значение вектора, вектор выравнивается точно в точке пересечения осей под углом 45° ко всем осям.Вектор Ввод значенийX = 1, Y = 0, Z = 0определяет вектор параллельно оси Х.X = 0,5, Y = 0, Z = 1поворачивает вектор, параллельный оси Z, на 45° в направлении оси X.X = -0,5, Y = 0, Z = 1поворачивает вектор, параллельный оси Z, на 45° по оси X в отрицательном направлении.X = 0, Y = 0,5, Z = 0,5определяет пространственный вектор между осями Y и Z.

### Дополнительная длина

Здесь для отдельных выводов устройства можно задать значение, отличающееся от значения по умолчанию.

### Категория соединения

Из этого раскрывающегося списка выберите значение по умолчанию, которое определяет тип вывода устройства соединений. Соответствующие записи заносятся в диалоговое окно [Настройки: Категория соединения](partsmanagementgui_d_einstanschlussausf.md); в это диалоговое окно можно попасть через пункты меню Параметры > Настройки > Фирма > Управление > Категория соединения.

При создании новых схем соединений мы рекомендуем использовать только те версии соединений, которые соответствуют [[классификации категории соединения eCl@ss](partsmanagementgui_d_einstanschlussausf.md)](partsmanagementgui_d_einstanschlussausf.md).

### Размер присоединения

В этом поле для винтовых подключений, видов резьбы, ниппелей и штекерных выводов можно указать размер винта (например, 'M6') или накладки (например, '4,8x0,5').

### Мин. поперечное сечение:Макс. поперечное сечение

Укажите здесь значения для наименьшего и наибольшего поперечного сечения провода, которые можно подключить к выводу устройства.

### Предписать двойную трубку

При помощи флажка укажите, следует ли использовать для вывода устройства двойные трубки, если к этому выводу устройства ведут два соединения.

### Мин. AWG:Макс. AWG

Укажите здесь значения для минимального и максимального поперечного сечения AWG, которые можно подключить к выводу устройства.

### Размер наконечника

Инструмент для завинчивания (например, шуруповерт) состоит из привода и рабочей части. Привод — это компонент, который приводит в движение инструмент, например рукоять или вал двигателя. Рабочая часть — это компонент, который приводится в действие, например наконечник шуруповерта. Размер наконечника описывает габариты рабочей части (наконечника), например "3 x 0,5" = ширина 3 мм и толщина 0,5 мм. С помощью стандартной аббревиатуры можно указать тип исполнения рабочей части, например "PZ 1" — для крестообразного шлица Pozidriv или "TX 5" — для резьбовой головки Torx.

### Мин. момент затяжки:Макс. момент затяжки

Указывается в ньютон-метрах; описывает силу, с которой, например, затягивается винт, т. е. силу, которая передается с привода на рабочую часть.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

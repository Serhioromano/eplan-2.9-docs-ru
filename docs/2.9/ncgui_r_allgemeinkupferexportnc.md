# Вкладка Общее (экспорт ЧУ, медь)

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт ЧУ, медь. Выберите вкладку Общ..
* Сервисные программы > Данные изготовления > Механическая обработка > ЧУ, медь. Рядом с полем Машина нажмите ++...++. Выберите вкладку Общ..

На этой вкладке установите общие настройки для экспорта ЧУ медных функциональных элементов.

Обзор основных элементов диалогового окна:

### Каталог вывода

Экспорт ЧУ меди будет сгенерирован в виде файла XML. В этом поле укажите место хранения. В качестве целевых для экспорта данных можно указать любой дисковод и подкаталог.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Генерировать имена файлов

В этом раскрывающемся списке определяется стандартный порядок присвоения имен файлам экспорта ЧУ меди.

* Из непрерывного номера: при выборе этого параметра EPLAN по умолчанию присваивает файлам XML имена по образцу <непрерывный_номер>.XML. Если уже имеются файлы XML, EPLAN начнет нумерацию со следующего доступного числа.
* Из обозначения функц. элемента: При выборе этого параметра EPLAN по умолчанию присваивает файлам XML имена по образцу <Обозначение функц. элемента>.XML.
* Из свойств: Если выбран этот параметр, имя файла составляется из свойств проекта, а также свободно определяемого разделителя. Используя кнопку ++...++, откройте диалоговое окно Формат: Свойство для конфигурации имени файла.

### Генерировать подкаталоги

В этом раскрывающемся списке определяется стандартный порядок сгенерирован подкаталогов при экспорте ЧУ меди.

* Нет: Если выбран данный параметр, при экспорте подкаталоги не генерируются.
* Из структуры проекта: Если выбран данный параметр, то при экспорте автоматически генерируются подкаталоги, отражающие структуру проекта.
* Из свойств: Если выбран этот параметр, имя подкаталога составляется из свойств проекта, а также свободно определяемого разделителя. Используя кнопку ++...++, откройте диалоговое окно Формат: Свойство для конфигурации имени файла. Используйте здесь, например, элемент формата "Разделитель каталога", чтобы сгенерировать структуру каталога с несколькими расположенными друг под другом подкаталогами.

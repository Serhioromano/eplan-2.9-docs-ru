# Обновить данные объектов-заполнителей на сегментах

При изменении данных объектов-заполнителей в макросах, сохраненных на объектах планирования или технологических контурах на вкладке Макрос, ранее необходимо было открывать по отдельности каждый сегмент с этим присвоенным макросом и обновлять данные. Чтобы обновление можно было выполнять для нескольких сегментов одновременно, теперь в ***навигаторе предварительного планирования*** появился новый пункт всплывающего меню Обновить данные заполнителя.

Чтобы обновление данных объектов-заполнителей из макросов можно было выполнять для нескольких шаблонов сегментов одновременно, в навигатор шаблонов сегментов был также добавлен пункт всплывающего меню Обновить данные заполнителя.

Если страница схемы соединений уже сгенерирована с помощью макроса и обновленные данные должны быть перенесены из сегмента в детальное планирование, необходимо обновить детальное планирование.

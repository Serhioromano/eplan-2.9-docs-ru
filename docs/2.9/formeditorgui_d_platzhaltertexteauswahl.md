# Диалоговое окно Тексты заполнителей — <Тип формы>

Вы открыли проект. Сервисные программы > Основные данные > Форма > Открыть > [Открыть].

* Вставить > Текст заполнитель. В поле Свойство вкладки Размещение нажмите ++...++.
* Вы дважды щелкнули по тексту-заполнителю (или выбрали пункт всплывающего меню Свойства; в этом случае можно выделить даже несколько текстов заполнителей), а в диалоговом окне Свойства (текст заполнитель) во вкладке Размещение в поле Свойства щелкнули кнопку ++...++.

В этом диалоговом окне выберите специфический для формы текст-заполнитель, в который должна быть вставлена форма. При оценке формы программа автоматически заменяет эти заполнители на соответствующие значения конкретных объектов проекта, для которого выполняется отчет, например, на значения функциональных элементов, страниц, символов и т.д.

Обзор основных элементов диалогового окна:

### Элемент

В зависимости от текущего типа формы здесь показываются все элементы, для которых может быть определен заполнитель. Выделите запись, чтобы в правой области вывести соответствующий список свойств.

!!! example "Пример:"

    Выделите к примеру элемент Свойства страницы(Тип формы 'Содержание'), таким образом в правом списке выводятся подчиненныесвойства Установка,Номер страницы,Доп. поле страницыи др.

!!! note "Замечание:"

    Для некоторых элементов, например Изделие (источник) возможен индекс, в этом случае рядом с элементом отображается знак 'плюс'. Щелкните по этому символу, тогда раскроется иерархический список возможных индексированных свойств, и вы сможете выбрать необходимое. Обратите внимание на то, что данный индекс не обязательно соответствует индексу на вкладке Изделие диалогового окна свойств. Если, например, в строке 1 и строке 20 введено изделие, в отчете получаются данные второго изделия через индекс 2 (а не через индекс 20).

### Категория

Ограничьте отображение подчиненных свойств, выбрав запись из этого раскрывающегося списка.

!!! example "Пример:"

    Выделите, к примеру, элемент Свойства страницы(Тип формы 'Содержание') и выберите категорию 'Ревизия', таким образом в правом списке выводятся только специфические для обработки подчиненные свойства Автор,Дата создания,Последний обработчики др.

### Фильтр

Введите в это поле текст, по которому должна выполняться фильтрация свойств.

Если необходимо вернуться к прежнему виду свойств, удалите текст из поля ввода. Для этого нажмите кнопку ![](images/all_delete_as.png){: .ui-icon } (Удалить).

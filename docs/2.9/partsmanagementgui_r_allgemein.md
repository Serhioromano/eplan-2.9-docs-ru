# Вкладка Общ. (база данных изделий)

Сервисные программы > Изделие > Управление. В дереве или списке выделено одно изделие.

На этой вкладке указывается общая информация, которая не зависит от выбранного типа изделия (отдельная деталь, кабели и т. д.). Эти данные можно указать ***один раз*** для каждого изделия.

Доступны следующие элементы диалогового окна:

### Главная группа продуктов / Группа продуктов / Подгруппа продуктов

В этих полях можно выбрать группу из фиксированных заданных записей. В зависимости от настройки, выбранной в разделе [Дополнительно] > Настройки для конфигурации дерева, при изменении группы продуктов выполняется соответствующая пересортировка изделий в дереве.

### Раздел/подраздел

Щелкните в данном поле по кнопке ++...++, чтобы в последующем диалоговом окне при помощи установки флажков задать дополнительные разделы, в которых должно содержаться изделие.

### Номер изделия

Номер изделия идентифицируется для изделия, т. е. номера изделия должны встречаться только раз на каждый уровень иерархии. В случае вариантов изделия идентифицируется комбинация номера изделия и варианта. При сохранении изделия или варианта это проверяется автоматически.
Номера изделий могут включать до 255 символов.

При указании номеров изделий следует с большой тщательностью соблюдать организацию номеров.

!!! example "Пример:"

    Номер_группы.Производитель.Прочая_информация.Порядковый_номер; обозначение при вводе:KM22.KLM.0750.0001Производитель.Номер_группы.Порядковый_номер; Обозначение при вводе:SIE.3TB.0001

### Деталь, снятая с производства

Установите этот флажок, если необходимо отметить изделие как 'более не используемое'. Такие изделия можно потом вывести или отфильтровать, используя конфигурацию представления в виде списка в навигаторах изделий и спецификаций и фильтр на основе поля.

### ERP-номер

Укажите номер изделия внешней системы ERP. Как и номер изделия, данный номер должен быть уникальным, что проверяется при сохранении изделия или варианта.
Номер ERP может включать до 50 символов.

### Номер типа

В отличие от номера изделия номер типа не является однозначным, особенно это касается кабелей. В данном случае введенные данные должны по возможности соответствовать данным из каталога производителя.

### Обозначение 1 / Обозначение 2 / Обозначение 3

В эти поля можно вносить дополнительные данные по изделию, например, также согласно данным в каталоге производителя. Для таких данных не проводится отдельный анализ, однако они могут быть выведены в спецификации.

### Производитель / поставщик

В этих полях с помощью кнопки ++...++ можно выбрать соответствующее краткое и подробное имя. Кроме того, возможен прямой ввод краткого имени, однако поле для подробного имени обработать нельзя.

Если в этих полях вводится краткое имя, которое еще не существует, автоматически создается соответствующая пустая запись данных "клиент / поставщик", ее данные необходимо дополнить.

### Номер для заказа

Внесите здесь номер заказа изделия, по возможности также согласно данным в каталоге производителя.

### Описание

Также это поле можно применять для дополнительных данных по изделию и можно, например, всегда использовать для внутренней информации.

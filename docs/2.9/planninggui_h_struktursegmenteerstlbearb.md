# Создание и обработка сегментов структуры

Создавать сегменты структуры и управлять ими можно в навигаторе предварительного планирования.

Работать со свойствами сегмента структуры можно во вкладках <Отображаемое имя сегмента структуры> и Документы / страницы диалогового окна "Свойства".

!!! note "Замечание:"

    Сегменты структуры нельзя создавать ниже объектов планирования, а также перемещать туда.

1. Выберите следующие пункты меню: Данные проекта > Предварительное планирование > Навигатор
2. Выделите проект или уже имеющийся сегмент структуры и выберите пункт всплывающего меню Новый сегмент структуры.

!!! info "Для сведения:"

    Если в проекте имеется несколько определений сегментов, откроется диалоговое окно Выбрать определение сегмента.

3. Выделите нужное определение сегмента и нажмите ++OK++.
4. Введите в диалоговом окне Свойства ++...++ на вкладке <Отображаемое имя сегмента структуры> нужные данные для нового сегмента структуры.
5. При необходимости присвойте сегменту структуры на вкладке Документы / страницы внешние документы и/или страницы проекта.
6. Щелкните по кнопке ++OK++.

# Диалоговое окно Настройки: Переводимые страницы

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Перевод > Переводимые страницы.

В этом диалоговом окне можно определить тип страниц которые будут переводиться. Эта настройка сохраняется для определенного проекта.

Обзор основных элементов диалогового окна:

### Типы страниц

В этом поле показаны все имеющиеся в EPLAN типы страниц. если перед одним из типов страниц установить флажок, все страницы этого типа будут переводиться.

# Диалоговое окно Закрыть проект / Диалоговое окно Закрыть область обработки

Вы создали ревизию и открыли проект ревизии (расширение имени файла *.ell). Вы выполнили изменения на одной или нескольких страницах. Вы выделили проект или страницы проекта в навигаторе страниц. Сервисные программы > Управление ревизиями > Закрыть проект / область.

В этом диалоговом окне можно описать изменения, выполненные в проекте ревизии. Для этого можно указать для всех измененных страниц проекта ревизии общие данные ревизии (индекс ревизии, описание и причину изменения). Введенные тексты анализируются для вывода на печать и/или графическую страницу обзора ревизий.

Если область обработки выбрана и активирована, то изменения описываются здесь в пределах активной области обработки.

Если вы хотите затем обработать введенные здесь тексты, можно изменить их в диалоговом окне [Дополнительно обработать данные ревизии](revisionlogmgtgui_d_revisionsdatennachbearbeiten.md).

Обзор основных элементов диалогового окна:

### Индекс ревизии

Это поле предусмотрено для текста, с помощью которого можно пометить изменения страницы. Индекс ревизии может состоять из любых символов. С помощью кнопки ++...++ откройте диалоговое окно [Выбрать индекс ревизии](revisionlogmgtgui_d_indexauswaehlen.md). Там можно выбрать уже введенный индекс ревизии или внести изменения при необходимости.

### Описание

В этом поле можно прокомментировать выполненные изменения. В качестве предложения автоматически вводится комментарий к ревизии.

### Причина изменения

Это поле предусмотрено для ввода причин изменений.

### Пользователь

В этом поле автоматически вводится пользователь, который описал изменение. Запись изменить невозможно.

### Область обработки

В этом поле отображается текущая активная область обработки. Запись изменить невозможно.

### Дата

В этом поле автоматически вводится дата создания описания. Запись изменить невозможно.

### Сгенерировать отчет при закрытии проекта

Если этот флажок установлен, то при закрытии проекта обновляются все страницы отчета проекта. После того как проект закрыт таким образом, все страницы отчета обновляются и для всех страниц удаляется обозначение "План". Страницы отчета имеют тот же индекс ревизии, который задается при закрытии.

Если этот флажок не установлен, то страницы отчета не обновляются.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

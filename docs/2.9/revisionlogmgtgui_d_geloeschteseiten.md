# Диалоговое окно Удаленные страницы

Вы создали ревизию и открыли проект ревизии (расширение имени файла *.ell). В настройках проекта вы активировали настройку При удалении сгенерировать маркер удаления и перечислить удаленные страницы. Сервисные программы > Управление ревизиями > Отслеживание изменений > Удаленные страницы.

В этом диалоговом окне отображаются страницы проекта ревизии. Некоторые из отображенных здесь свойств представлены только для информации и не могут быть изменены. В полях Указатель, Описание и Причина изменения можно описать выполненные изменения.

Доступны следующие элементы диалогового окна:

Удаленные страницы отображаются в таблице после ввода следующих данных:

* Строка: Непрерывный номер строки.
* Страница: Полное имя удаленной страницы.
* Область обработки: Показывает имя области обработки, к которой относится страница.
* Индекс: Это поле предусмотрено для текста, с помощью которого можно пометить изменение страницы. Индекс может состоять из любых значков.
* Описание: В этом поле можно прокомментировать выполненные изменения страницы. В качестве предложения автоматически вводится "Страница удалена".
* Причина изменения: Это поле предусмотрено для ввода причин изменений.
* Ревизия проекта: Имя ревизии проекта, в котором содержалась страница на момент удаления.
* Пользователь: Имя пользователя, удалившего страницу.
* Дата: Время, когда была удалена страница. Указать дату и время.

# Принадлежность функций приводам

Системы привода состоят из различных компонентов, таких как двигатели, преобразователи, датчики и т. д. Такие компоненты, принадлежащие приводу, теперь можно сгруппировать в EPLAN. Для этого на функциях в схеме соединений доступно новое индексированное свойство Привод [1–10] (ид. 20576). С помощью этого свойства одной функции можно присвоить до десяти приводов.

![](images/benefit.png)Эффект:

С помощью нового свойства Привод можно сгруппировать компоненты привода. В новом связанном с приводом виде в навигаторе ПЛК отображается это группирование, и вы можете обрабатывать компоненты привода в наглядном представлении.

Для функций, присвоенных приводу, в новом свойстве Привод: Тип устройства (ид. 20577) можно точнее определить тип устройства, например "Синхронный двигатель", "Преобразователь", "Датчик угла поворота" и т. д. Свойство отображается в данных ссылки изделия. Значения, вводимые для типа устройства, обычно указываются производителем устройства.

С помощью одноименного свойства изделия Привод: Тип устройства (ид. 40349) тип устройства можно сохранить на изделии. При выборе изделия тип устройства переносится на главную функцию.

### Связанный с приводом вид в навигаторе ПЛК

Для представления приводов со связанными функциями представление структуры дерева навигатора ПЛК было расширено дополнительным видом Связано с приводами.

В этом виде данные ПЛК располагаются в пределах структуры проекта по приводам. Приводы обозначается стоящей впереди пиктограммой ![](images/navplc_drive_as.png){: .ui-icon }. Функции отображаются под уровнями иерархии "Проект конфигураций", "Привод" и "Имя станции". В этом виде также отображаются функции, которым в диалоговом окне 'Свойства' назначен проект конфигураций и / или имя станции, но не привод.

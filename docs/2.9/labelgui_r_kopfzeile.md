# Вкладки Верхний колонтитул / Нижний колонтитул

* Вы открыли проект. Сервисные программы > Данные изготовления > Экспорт / маркировка. В диалоговом окне Экспортировать данные изготовления / Вывести маркировку рядом с полем Настройки щелкните по кнопке ++...++. В диалоговом окне Настройки: Маркировка выберите вкладку Верхний колонтитул или вкладку Нижний колонтитул.
* Параметры > Настройки > Пользователь > Интерфейсы > Экспорт данных изготовления / маркировка. В диалоговом окне Настройки: Экспорт данных изготовления / маркировка выберите вкладку Верхний колонтитул или вкладку Нижний колонтитул.

На этих вкладках указывается форма и вид верхнего и нижнего колонтитула страницы экспорта / маркировки.

Обзор основных элементов диалогового окна:

### Доступные элементы формата

В этом поле представлены доступные элементы формата. Доступные элементы формата зависят от выбранного типа отчета

### Выбранные элементы формата

В этом поле находятся элементы формата, из которых состоит формат верхнего или нижнего колонтитула.

* Элемент формата: В этом столбце выводится имя элемента формата.
* Символ: В этом столбце выводится сокращение элемента формата. Сокращения выбранных элементов формата представлены в поле Предв. просмотр.
* Настройки: В этом столбце выводятся сокращения и значения настроек элемента формата.

### Повторить верхний колонтитул на каждой странице / Повторить нижний колонтитул на каждой странице

Если этот флажок установлен, верхний / нижний колонтитул выводится на каждой странице файла экспорта / маркировки. Если этот флажок не установлен, соответствующая область выводится только на первой странице.

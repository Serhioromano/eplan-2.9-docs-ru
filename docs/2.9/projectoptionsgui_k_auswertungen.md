# Отчеты для опций и объектов-заполнителей

Опции и объекты-заполнители можно выводить в отчетах и при экспорте данных изготовления / маркировки.

В ***перечне опций*** выводятся все опции, которые имеются в проекте. Установив соответствующий критерий фильтра, можно вывести только активные опции.

Опции имеют следующие свойства:

* Опциональная группа: Имя
* Опциональная группа: Описание
* Опция: Имя
* Опция: Описание
* Активная опция (Да/Нет)
* Фрагмент опции: Имя (индексировано)
* Фрагмент опции: Описание (индексировано)
* Имя объекта-заполнителя (индексировано)
* Объект-заполнитель: Последний выбранный набор значений (индексировано)
* Объект-заполнитель: Размещение (индексировано).

В ***перечне объектов-заполнителей*** выводятся все объекты-заполнители, которые имеются в проекте. С помощью фильтра можно просмотреть объекты-заполнители, которые относятся к одному фрагменту.

Объекты-заполнители имеют следующие свойства:

* Имя объекта-заполнителя
* Последний выбранный набор значений
* Размещение.

Если был сгенерирован отчет для опций или объектов-заполнителей, после чего опции и объекты-заполнители были убраны из проекта (например, при сжатии), эти отчеты больше не удастся обновить.

Отчеты не будут автоматически обновляться, если опции или объекты-заполнители более не содержатся в проекте.

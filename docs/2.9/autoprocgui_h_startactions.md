# Создать схемы для автоматизированной обработки

Управление автоматизированной обработкой проектов в EPLAN осуществляется при помощи технологии схем. При этом все автоматически исполняемые операции сохраняются в пользовательской схеме. На основе этой схемы EPLAN автоматически создает файл сценария (*.cs), который по умолчанию сохраняется в каталог <Дисковод программы>:\<Каталог программы>\Сценарии\<Идентификатор фирмы>.

Условие:

Вы открыли проект. В навигаторе страниц выделен проект или страницы проекта.

### Составление схемы

1. Выберите следующие пункты меню: Сервисные программы > Автоматизированная обработка
2. В диалоговом окне Выполнить: Автоматизированная обработка щелкните по кнопке ++...++ рядом с полем Схема.
3. В диалоговом окне Настройки: Автоматизированная обработка рядом с полем Схема щелкните ![](images/all_new_as.png){: .ui-icon } (Создать).
4. В диалоговом окне Новая схема в поле Имя введите описательное название схемы. В поле Описание можно ввести дополнительные сведения.
5. Щелкните по кнопке ![](images/all_save_as.png){: .ui-icon } (Сохранить).

### Выбрать операции

1. В диалоговом окне Выполнить: Автоматизированная обработка щелкните по кнопке ++...++ рядом с полем Схема.
2. В диалоговом окне Настройки: Автоматизированная обработка из раскрывающегося списка Схема выберите схему для редактирования.
3. Выделите в поле Доступные операции операцию и нажмите кнопку ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо).
4. В открывшемся диалоговом окне EPLAN настройте параметры для выбранной операции и нажмите кнопку ++OK++.

!!! info "Для сведения:"

    Операция копируется в поле Выбранные операции.

5. Аналогичным образом выберите другие операции для автоматизации.
6. Чтобы определить последовательность операций, выберите в поле Выбранные операции операцию и нажимайте на ![](images/all_arrowup_as.png){: .ui-icon } (Переместить вверх) или ![](images/all_arrowdown_as.png){: .ui-icon } (Переместить вниз) до тех пор, пока запись не будет установлена в нужную позицию.
7. Щелкните по кнопке ![](images/all_save_as.png){: .ui-icon } (Сохранить).

!!! info "Для сведения:"

    Схема сохранится, и автоматически будет создан файл сценария (*.cs).

8. Щелкните по кнопке ++OK++.

# Экспортировать сборку проводов

В качестве примера интерфейса сборки проводов здесь описывается экспорт для формата 'Общая спецификация проводов'. Экспорт других интерфейсов выполняется подобным образом и отличается только видом значений, которые вводятся в диалоговом окне экспорта, и форматом генерируемых файлов.

Условия:

* Вы открыли проект.
* В настройках фирмы определена машина нужного типа.
* Навигатор пространства листов открыт, и одно пространство листов открыто.
* Открыт навигатор соединений.
* Сгенерированы маршрутизируемые соединения.
* Выделены соединения в навигаторе соединений, или выбрана экспортируемая область в пространстве листа или в навигаторе пространства листов.

1. Выберите пункты меню Сервисные программы > Данные изготовления > Сборка проводов > Общая спецификация проводов.
2. В диалоговом окне Сборка проводов: Общая спецификация проводов выберите схему нужной машины, язык и каталог вывода или примените предварительные настройки.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В настроенном каталоге вывода файл спецификации проводов генерируется с заранее выбранным именем. При экспорте специфических для машины интерфейсов генерируются, в зависимости от производителя, несколько заданных имен файлов и расширений имен файлов. Эти данные передаются в программное обеспечение машины по сборке проводов.

!!! note "Замечание:"

    Учтите, что спецификация проводов сортируется с ориентацией на текст по номеру объединенных проводов, т. е. вывод выглядит следующим образом: "1, 10, 2, ...". Чтобы избежать этого, добавьте ведущие нули.

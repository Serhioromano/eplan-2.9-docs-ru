# Диалоговое окно Выполнить контр. прогон

Вы открыли проект. В навигаторе страниц выделена одна или несколько страниц или проект, или же в графическом редакторе открыта страница. Данные проекта > Сообщения > Выполнить контр. прогон.

В этом диалоговом окне выберите схему для проверки данных проекта. Схема указывает, какие виды проверок проводились и как классифицированы сообщения.

Доступны следующие элементы диалогового окна:

### Настройки

Выберите схему из раскрывающегося списка. С помощью кнопки ++...++ откройте диалоговое окно [Настройки: сообщения и контрольные прогоны](xesinspectiongui_d_einstellungenmeldungen.md) и определите собственную схему отображения сообщений о контрольном прогоне.

### Применить ко всему проекту

Установите этот флажок, чтобы проверить весь проект.
Если был выделен проект, этот параметр используется по умолчанию и не может быть отключен.

### Проверять только выполненные сообщения

Если этот флажок установлен, при следующем контрольном прогоне в автономном режиме будут проверяться только те сообщения, для которых в управлении сообщениями установлен флажок Выполнено. Все невыполненные сообщения останутся в базе данных сообщений без изменения.

В этом случае все другие настройки контрольного прогона или выбор, сделанный в навигаторах, не учитываются. Поэтому поле Настройки и флажок Применить ко всему проекту недоступны.

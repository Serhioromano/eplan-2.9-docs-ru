# Диалоговое окно Результаты сравнения свойств — <Имя проекта>

Вы выполнили сравнение свойств проектов и открыли проект. Сервисные программы > Управление ревизиями > Сравнение свойств проектов > Отобразить результаты сравнения свойств.

В этом диалоговом окне отображаются результаты сравнения свойств проектов. Данные считываются из списка результатов ревизии, в котором хранятся результаты сравнения свойств. (Этот список сохраняется в базе данных текущего проекта.) Чтобы ускорить обзор результатов, отображаются не все записи из списка результатов ревизии, а лишь столько записей, сколько указано в настройках проекта в Максимальное число отображенных результатов.

Обзор основных элементов диалогового окна:

В таблице, в частности, отображаются следующие столбцы (с возможностью настройки через Всплывающее меню > Конфигурировать представление):

* Тип объекта: Отображает тип объекта, например функцию, графический файл и т. д.
* ОУ / Имя: Отображает полное ОУ функции или имя объекта. В соединениях отображаются источник и цель, в проектах –имя проекта, включая путь к файлу; в графических файлах –имя файла, включая путь к файлу; в гиперссылках –обозначение.
* Страница объекта: Отображает страницу, на которой обнаружены отличия.
* Вид изменения: Отображает, был ли объект изменен, удален или заново добавлен.
* Свойство: Отображает измененное свойство.
* Старое значение: Отображает значение свойства в исходном проекте.
* Новое значение: Отображает значение свойства в текущем проекте.
* Номер изделия [1]: Показывает первое изделие, введенное .на функции.
* Имя потенциала: В соединениях отображает имя потенциала, к которому подключено соединение.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Выделить все |  Выделяет все записи в списке.
Копировать |  Копирует содержимое выделенных полей в буфер обмена.
Сбросить да старых значений |  Сбрасывает значение выбранного свойства до значения этого свойства в исходном проекте. Возможен многократный выбор. Сбрасывать можно только доступные для записи свойства (например Функциональный текст, Текст гравировки, Примечание). Автоматически определенные свойства (например Функциональный текст (автоматически)) сбрасывать ***нельзя***, и в этом случае отображается замечание.
Изменения можно увидеть только на объекте в проекте, представление в диалоговом окне не изменяется. Маркер ревизии объектов также остается неизменным. Для обновления маркеров ревизии следует удалить их и выполнить новое сравнение проектов.
Перейти к проекту (графика) |  Переходит к соответствующему размещенному объекту в текущем проекте.
Перейти к исходному проекту (графика) |  Переходит к соответствующему размещенному объекту в исходном проекте.
Синхронизировать выбор |  Выделяет соответствующий объект в открытых навигаторах.

!!! tip "Совет:"

    Для определенных объектов (функций, соединений, потенциалов, технологических контуров, функций ТК), которые в графическом редакторе снабжены маркером ревизий, возможна синхронизация с помощью диалогового окна Результаты сравнения свойств — <Имя проекта>. Выделите требуемый объект и выберите всплывающее меню > Синхронизировать выбор. После этого выделяется соответствующая строка в списке результатов.

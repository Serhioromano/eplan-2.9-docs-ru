# Диалоговое окно Изменить свойства ссылки блока

Открыта страница в графическом редакторе. Несколько элементов объединено в блок, и этот блок выделен. Обработать > Свойства (или Всплывающее меню > Свойства).

В этом диалоговом окне отображаются свойства ссылки блока.

Доступны следующие элементы диалогового окна:

### Число объектов

В этом поле отображается количество элементов, объединенных в один блок.

### Число ссылок

В этом поле отображается количество размещенных в проекте ссылок на этот блок. (В настоящее время допускается только одна ссылка.)

### Имя блока, на который идет ссылка

Здесь указывается имя блока. При экспорте DXF-/DWG-файлов это имя сохраняется в экспортируемом файле.

### [Разбить блок]

Разбивает блок на составные части. После этого отдельные элементы могут быть обработаны по отдельности.

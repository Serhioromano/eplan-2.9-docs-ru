# Заменить настройки

Можно передавать ***проектные настройки*** в другие проекты. Для этого необходимо сохранить настройки исходного проекта вместе с проектом в качестве шаблона и затем создать из этого шаблона новый проект.

### Сохранение проекта в качестве шаблона

Условия:

* Вы открыли меню управления проектами: Проект > Управление
* В структуре дерева управления проектами находится считанный каталог с проектом.

1. Выделите в диалоговом окне Управление проектом исходный проект.
2. Выберите ++Дополнительно++ > Создать шаблон проекта.
3. Выберите место хранения и имя для шаблона проекта.
4. Щелкните по кнопке ++Сохранить++.

!!! info "Для сведения:"

    Будет создан шаблон проекта.

### Создание проекта из шаблона

1. Выберите пункты меню Проект > Создать.
2. В диалоговом окне Создать проект задайте имя для нового проекта.
3. Нажмите на ++...++ рядом с пунктом Шаблон.
4. Выберите в диалоговом окне Выбрать шаблон проекта / базовый проект нужный шаблон и щелкните по кнопке ++Открыть++.
5. Выберите место хранения для проекта.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Будет создан проект на основе шаблона.

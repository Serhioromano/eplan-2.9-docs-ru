# Диалоговое окно Выполнить: Автоматизированная обработка

Вы открыли проект. В навигаторе страниц выделен проект или страницы проекта. Сервисные программы > Автоматизированная обработка.

В этом диалоговом окне выберите с помощью схемы последовательность действий для автоматического выполнения.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать. При нажатии кнопки ++...++ открывается диалоговое окно Настройки: Автоматизированная обработка.

### Операции

В списке отображаются операции, сохраненные в выбранной схеме.

# Вставить обзоры модели

Обзоры модели являются стандартизированными обзорами 2D трехмерного чертежа монтажных поверхностей, например оснащенных монтажных поверхностей.

Условие:

Вы открыли проект.В Графическом редакторе открыта страница проекта.

1. Выберите пункты меню Вставить > Графика > Обзор модели.
2. Укажите первый угол области для обзора модели на текущей позиции курсора, нажав клавишу ++Ввод++.
3. Начертите рамку с помощью клавиш со стрелками или с помощью мыши. Рамку можно чертить в любом направлении.
4. Завершите выбор области, нажав клавишу ++Ввод++.
5. В диалоговом окне Обзор модели на вкладке Вид укажите специфические для обзора модели данные.
6. На вкладке Отображение определите настройки отображения для описания обзора модели, например размер шрифта, выравнивание и ширину / высоту блока выравнивания.
7. На вкладке Формат определите графические свойства рамки обзора модели, например начальную точку, конечную точку и ширину, а также другие свойства формата.
8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Диалоговое окно Обзор модели закроется, и обзор модели будет размещен на странице в соответствии с выполненными настройками.

# Операция: MfRemoveToolbarAction

**Параметр** |  **Описание**
---|---
ToolbarName  |  Имя панели инструментов.
Silent  |  Если значение равно 1, то при отсутствии существующей панели инструментов сообщения не отображаются.

!!! note "Замечание:"

    Данная операция использует главное окно.

!!! example "Пример:"

    MfRemoveToolbarAction
    /ToolbarName:"?"
    /Silent:No

# Операция: GenerateMacros

**Параметр** |  **Описание**
---|---
PROJECTNAME |  Имя проекта с полным путем файла (является опцией).
Если не задано, то выбранный проект используется, когда операция вызывается через интерфейс пользователя (напр., через сценарий или панель инструментов). При вызове из командной строки Windows следует определить PROJECTNAME или сначала следует использовать ProjectAction. В противном случае отобразится системное сообщение.
PAGENAMEn |  Имена учитываемых страниц (является опцией). При этом n — это номер.
INSTALLATIONSPACENAMEn |  Имена учитываемых пространств листа (является опцией). При этом n — это номер.
STRUCTUREn |  Структурные идентификаторы учитываемых пространств листа (является опцией). При этом n — это номер.
WINDOWMACRODIR |  Каталог для макросов окна (является опцией).
PAGEMACRODIR |  Каталог для макросов страниц (является опцией).
FILTERSCHEME |  Схема фильтра (необязательно).
Значение по умолчанию: последняя использованная схема фильтра. При передаче пустой символьной строки используется последняя использованная схема.
OVERWRITE |  Режим перезаписывания (является опцией).

!!! note "Замечание:"

    Учитываются только те проекты, в которых свойство Вид проекта установлено на "Проект макросов".

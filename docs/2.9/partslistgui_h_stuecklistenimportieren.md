# Импортировать спецификации

Данные изделия спецификации можно импортировать в текущий проект. Обратите внимание на то, что экспорт и импорт можно осуществлять только в проекте!

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Данные проекта > Устройство / изделие > Навигатор спецификаций
2. В диалоговом окне Спецификация — <Имя проекта> выберите пункт всплывающего меню Импортировать.
3. В диалоговом окне Импортировать спецификацию из раскрывающегося списка Поиск в выберите дисковод и каталог, в котором находится импортируемый файл.
4. Выберите из раскрывающегося списка Тип файла одну из записей для формата импортируемого файла.
5. Выберите требуемый файл импорта из списка.
6. Щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Записи данных импортируются. Если при импорте возникают ошибки, отображается соответствующее замечание, и подробную информацию можно просмотреть в управлении сообщениями (Данные проекта > Сообщения > Управление).

Если данные экспортированы и затем обработаны, то при импорте записываются только определенные поля файла экспорта. Обзор полей приведен в разделе "[Спецификации: Поля в файле экспорта](partslistgui_k_felderexportdatei.md)".

!!! note "Замечание:"

    При импорте уже присвоенной ОУ ссылки изделия обозначение устройства должно уже быть доступным. Если этого не происходит, то создается ссылка изделия для ***изделия проекта***.

# Вкладка Разделитель

Вы открыли проект. Сервисные программы > Данные изготовления > Экспорт / маркировка. В диалоговом окне Экспортировать данные изготовления / Вывести маркировку рядом с полем Настройки щелкните по кнопке ++...++. В диалоговом окне Настройки: Экспорт данных изготовления / маркировка выберите одну из вкладок Верхний колонтитул, Нижний колонтитул или Этикетка, выделите элемент формата "Разделитель" и нажмите на ![](images/all_arrowright_as.png){: .ui-icon } (переместить вправо).

В этом диалоговом окне указывается, какой разделитель должен быть установлен между текстами со свойствами.

Обзор основных элементов диалогового окна:

### Пробел

Если выбрана эта опция, после или перед текстом выводится пробел.

### Табуляция

Если выбрана эта опция, после или перед текстом выводится табулятор.

### Новая строка

Вставка перехода на новую строку. Обеспечивает вывод многострочного текста.

### Другие символы

Введите в этом поле один или несколько символов, которые следует вывести перед или после текста. Пробелы вводятся нажатием на клавишу пробела. Число введенных символов автоматически выводится в поле Число символов.

### Число символов

В этом поле выводится число символов, введенных в поле Другие символы. Особенно большое значение это имеет в том случае, если введено несколько пробелов.

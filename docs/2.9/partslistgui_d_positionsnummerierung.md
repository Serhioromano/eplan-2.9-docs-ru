# Диалоговое окно Нумеровать позиции

Вы открыли проект. Данные проекта > Устройство / изделие > Навигатор спецификаций. В диалоговом окне навигатора Спецификация - <Имя проекта> выделены требуемые записи данных и в дереве или в списке выбран пункт всплывающего меню Задать номер позиций.

В этом диалоговом окне можно заново пронумеровать позиции, выделенные в спецификации. При этом номера позиций являются чистым признаком структуризации в пределах спецификации.

Обзор основных элементов диалогового окна:

### Сортировка

Выберите имеющуюся схему сортировки из раскрывающегося списка или откройте, нажав кнопку ++...++, диалоговое окно [Сортировка](formgeneratorgui_d_sortierung.md). Там можно обработать имеющиеся схемы или создать новые.

Выбранная схема активируется автоматически. Запись "-Не активировано-" позволяет деактивировать выбранную схему.

### Новая нумерация при измененном свойстве

Выберите доступную схему из раскрывающегося списка или откройте с помощью [...] диалоговое окно Новая нумерация при измененном свойстве. Там можно обработать имеющиеся схемы или создать новые. С помощью такой схемы номера позиций присваиваются с учетом структуры или страниц. Для этого в качестве критериев можно, например, выбрать свойства Установка или Имя страницы. Если значение заданных в схеме свойств изменяется (например, структурный идентификатор для установки), то нумерация снова начинается с заданного в диалоговом окне начального значения.

Выбранная схема активируется автоматически. Запись "-Не активировано-" позволяет деактивировать выбранную схему.

### Начальное значение

Введите здесь начальный номер для нумерации..

### Величина шага

Введите здесь величину шага, на которую соответственно повышается номер.

### Ширина поля

Установите в этом поле количество предусмотренных разрядов для номера позиции.

### Одинаковые номера для идентичных устройств

Установите этот флажок, если устройства с идентичными номерами изделия должны иметь одинаковые номера позиций.
Если этот флажок не установлен, то номера позиций назначаются независимо от устройств.

Обратите внимание на то, что устройства ***без присвоенного номера изделия*** обрабатываются как разные устройства, то есть всегда получают новый номер позиции.

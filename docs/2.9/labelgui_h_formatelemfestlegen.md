# Определить верхний колонтитул, область этикетки и нижний колонтитул

С помощью элементов формата указывается, какие данные необходимо вывести в качестве текстов в области этикетки, верхнем и нижнем колонтитулах страницы экспорта / маркировки, и как они должны быть отформатированы.

Условие:

Вы открыли проект.

1. Сервисные программы > Данные изготовления > Экспорт / маркировка
2. В диалоговом окне Экспортировать данные изготовления / Вывести маркировку рядом с полем Настройки щелкните по кнопке ++...++.
3. В диалоговом окне Настройки: Экспорт данных изготовления / маркировка в поле Схема, определенную пользователем или создайте схему.

### Выбрать элементы формата

Элементы формата для области этикетки, нижнего и верхнего колонтитула выбираются таким же образом. Доступность определенного элемента формата зависит от выбранного типа отчета. Элементы формата "Новая строка" и "Разбивка на странице при смене ОУ" доступны только для нижнего колонтитула.

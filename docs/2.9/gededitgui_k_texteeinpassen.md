# Блок выравнивания для текстов

Блоки выравнивания применяются в графическом редакторе и редакторе форм для подгона текстов или текстов заполнителей под имеющееся свободное место на странице или в форме. Для влияния на отображение текстов в блоке выравнивания в диалоговом окне "Свойства" текстов / текстов заполнителей на вкладке Формат имеются различные возможности настройки. Например, таким образом можно подогнать текст по ширине и высоте блока выравнивания с помощью настроек Ширина текста фиксирована и Высота текста фиксирована.

Однако может случиться так, что выводимые тексты в сгенерированных отчетах оказываются слишком длинными для своих блоков выравнивания или не вмещаются в блоки после перевода на различные отображаемые языки. В таких случаях текст часто обрезался или становился нечитаемым из-за наложения языков.

Для корректировки таких текстов вы можете подгонять тексты в блоке выравнивания. Теперь при соответствующей подготовке текстов заполнителей в формах при генерировании отчетов подгон проводится автоматически.

Для масштабирования текста таким образом, чтобы он вмещался в блок выравнивания без обрезки, можно воспользоваться свойством отображения Разрешен подгон текста. Этот флажок ***позволяет выполнять*** масштабирование текстов. Сам подгон выполняется в ходе последующего действия.

### Условия

Для того чтобы подогнать текст в блоке выравнивания, в качестве условия необходимо включить следующие четыре настройки (в диалоговом окне "Свойства" текста на вкладке Формат на уровне иерархии Блок выравнивания):

* Активировать блок выравнивания
* ширина текста фиксирована
* высота текста фиксирована
* разрешен подгон текста.

При соблюдении требуемых условий подгон текстов в блоке выравнивания выполняется следующим образом:

* ***Автоматически при генерировании отчетов:***
При генерировании отчетов выведенные тексты заполнители подгоняются автоматически.
* ***Вручную с помощью пункта меню:***
В графическом (и других) редакторе подгон текстов можно выполнить вручную через пункты меню Обработать > Текст > Подогнать тексты в блоке выравнивания.

### Ограничения при подгоне

* В редакторе символов и рамок невозможно выполнить подгон текстов в блоке выравнивания. В данных редакторах нет пункта меню Подогнать тексты в блоке выравнивания.
* Для текстов, объединенных с другими графическими элементами в единый блок подгон в блоке выравнивания также невозможен.
* Подгон размещенных текстов свойств в блоке выравнивания можно выполнить, только если соответствующая функция размещена. Для неразмещенных функций пункт меню Подогнать тексты в блоке выравнивания не действует.

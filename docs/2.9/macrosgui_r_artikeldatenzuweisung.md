# Диалоговое окно / вкладка Присвоение данных изделия

Вы открыли проект. Выделены требуемые элементы или страницы, которые необходимо сохранить в виде макроса. Обработать > Создать макрос окна / символа или Страница > Макрос страницы > Создать. В диалоговом окне Сохранить как выберите ++Дополнительно++ > Присвоить данные изделия.

В проект макросов вставлена рамка макроса (Вставить > Блок / вывод устройства / монтажная плата > рамка макроса), либо выполнен двойной щелчок по уже имеющейся рамке макроса. Выберите вкладку Присвоение данных изделия.

При вставке устройств (изделий) с сохраненными макросами свойства изделия копируются из базы данных изделий EPLAN на любые свойства объектов, которые вставляются из макроса в проект. В этом диалоговом окне / на этой вкладке можно обрабатывать свойства объектов, вставленных из макроса, причем также учитываются значения из свободных свойств изделия.

Существует возможность занести свойство в поле Номер изделия. Если изделие выбрано, оно сохраняется. Данные такого изделия еще раз не переносятся на функцию.

При использовании изделия это присвоение используется при переносе данных изделий на функции в макросе. Это автоматически происходит при операции "Вставить устройство", а также при выборе устройства и изделия.

!!! note "Замечание:"

    Если в макросе происходит одновременное присвоение данных изделия и значений объекта-заполнителя аналогичному свойству объекта, присвоение данных изделия "имеет преимущество" при вставке устройства с соответствующим указанным макросом.

Обзор основных элементов диалогового окна:

### Категория

Выберите из раскрывающегося списка вид свойств, который должен быть отображен в таблице внизу.

### Таблица

В этой таблице отображены все свойства с текущими значениями, которые относятся к объектам макроса. Отображение свойств в таблице зависит от выбранной Категория.

Таблица содержит следующие столбцы:

* Свойство: Здесь свойства объекта отображаются в форме структуры дерева.
Разверните дерево, щелкнув по символу ![](images/all_plussign_as.png){: .ui-icon }; в первую очередь отобразятся имена объектов (например, вид символа), а на самом нижнем уровне отобразятся свойства объекта.
* Текущее значение: Здесь отображается текущее значение соответствующего свойства.
* Присвоение: Сохраните новое / измененное значение для свойства.
Щелкните по ++...++, чтобы открыть диалоговое окно Формат и выберите из списка доступных элементов формы необходимое.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Вырезать / копировать / вставить |  Только в столбце Присвоение: Соответствует таким же функциями Windows для обработки записей данных; при вставке EPLAN автоматически проверяет, подходит ли новая запись к соответствующему свойству и при необходимости прерывает процесс, выводя необходимое сообщение.
Удалить |  Только в столбце Присвоение: Удаляет текущую запись.
Удалить объект |  Только если выделена по крайней мере одна строка: Удаляет из объекта-заполнителя ***весь*** выделенный объект, включая все его присвоенные свойства; для удаления нужно подтвердить запрос, нажав кнопку ++Да++.

### Показать только заполненные

С помощью этого флажка можно значительно уменьшить число отображенных свойств. Если этот флажок установлен, в таблице отображаются только те свойства, при которых столбец Присвоение содержит запись.

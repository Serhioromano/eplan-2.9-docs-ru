# Поворачивание и зеркальное отображение элементов

Условие:

Вы открыли страницу, форму, рамку или символ.

### Поворачивание элементов

1. Выделите соответствующие элементы и выберите пункты меню Обработать > Поворачивать.
2. Укажите исходную точку для поворота.
3. Затем с помощью мыши поверните выделенные элементы вокруг исходной точки и щелкните левой кнопкой мыши по странице, чтобы завершить поворот.

!!! info "Для сведения:"

    Выделенные элементы поворачиваются вокруг заданного угла поворота.

!!! warning "Предупреждение:"

    Если выделенная группа элементов содержит символ или выбранный элемент является символом, возможен только поворот с шагом 90°.

### Зеркально отобразить элементы

1. Выделите требуемые элементы и выберите пункты меню Обработать > Отобразить зеркально.
2. Задайте ось отражения, определив начальную и конечную точки с помощью мыши.

!!! info "Для сведения:"

    Выделенные элементы зеркально отображаются относительно оси зеркального отображения.

!!! warning "Предупреждение:"

    Если выделенная группа элементов содержит символ или выбранный элемент является символом, возможно зеркальное отображение только вдоль горизонтальной или вертикальной оси зеркального отображения.

!!! tip "Совет:"

    * Параметры точки вращения, угла поворота или оси отражения можно указать в области ввода данных.
    * При повороте / зеркальном отображении нажмите клавишу ++Ctrl++, чтобы вставить повернутую / отраженную копию выбранных элементов. При этом исходные элементы не будут изменены.
    При вводе с помощью ***области ввода данных*** достаточно во время подтверждения последнего ввода удерживать нажатой клавишу ++Ctrl++.
    * ***Макросы ***поворачиваются и зеркально отображаются на основе вариантов макросов. Если поворачивание и зеркальное отображение макросов должны осуществляться только на основе ***символов***, то при выполнении данной операции нажмите клавишу ++Shift++. При использовании комбинации клавиш [Ctrl] \+ [Shift] операция выполняется на основе символов с одновременным сохранением исходного макроса.

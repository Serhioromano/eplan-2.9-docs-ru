# Экспортировать трубопровод

Следующим шагом после свободной маршрутизации и адаптации трубопровода является передача этого трубопровода на изготовление и сгибание с помощью трубогибочной машины. Для передачи трубопровода в программу трубогибочной машины EPLAN генерирует по одному XML-файлу с необходимыми данными на каждый трубопровод.

С помощью программы трубогибочной машины рассчитываются длина резки и данные по сгибанию трубы. Затем данные изготовления для изгибания труб переносятся из программы в трубогибочную машину.

Условие:

Открыт проект, содержащий пространства листа со свободно размещенными трубопроводами.

1. В навигаторе соединений или в пространстве листа выделите один или несколько трубопроводов, из которых следует сгенерировать данные изготовления.
2. Выберите пункты меню Сервисные программы > Данные изготовления > Механическая обработка > Изгибание труб.
3. В диалоговом окне Экспорт изгибания труб выберите определение станка, каталог вывода и целевой файл.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Целевой файл будет сгенерирован в указанном каталоге. Теперь он может быть передан на станок.

# Операция: SetProjectLanguage

**Параметр** |  **Описание**
---|---
PROJECTNAME  |  Полное имя проекта. Проект должен быть открыт.
При вызове командной строки Windows сначала следует использовать ProjectAction.
DISPLAY  |  Язык отображения (например, en_US). При указании нескольких языков отдельные коды языков отделяются точкой с запятой (';').
VARIABLE  |  Одноязычный (перемен.) язык отображения.
SOURCE  |  Исходный язык.

!!! note "Замечание:"

    * SetProjectLanguage ***не*** переводит тексты проектов, а только изменяет настройки отображения. Операция настраивает языки, на которых переведенный текст проекта отображается на его страницах.
    * В защищенных от записи проектах язык отображения изменяется при помощи операции SetProjectLanguage только временно, эта настройка не сохраняется в проекте.

!!! example "Пример:"

    SetProjectLanguage
    /DISPLAY:en_US;de_DE
    /VARIABLE:en_US

# Операции EPLAN

В EPLAN функциональности программы, вызывающие видимые результаты для пользователя, осуществляются посредством так называемых "Операций" (actions). Так, например, все позиции меню и кнопки панелей инструментов связаны с той или иной операцией.

Некоторую часть этих операций можно четко распознать и таким образом, к примеру, задать автоматическое осуществление функциональностей программы. В данной справке приведены все доступные операции, а также их возможные параметры.

Операции EPLAN предлагают следующие возможности:

* Вы можете вставить операции в виде кнопок при настройке панелей инструментов.
* Некоторые из этих операций можно использовать для запуска EPLAN посредством вызова командной строки.
* Кроме того, операции можно использовать при создании и выполнении сценариев.
* В автоматически создаваемых сценариях функциональности Автоматизированная обработка также применяются операции EPLAN.

!!! note "Замечание:"

    При использовании Операций EPLAN в сценарии или вызове командной строки следует учитывать регистр. Написание имени операции и параметров должно соответствовать приведенному здесь написанию.

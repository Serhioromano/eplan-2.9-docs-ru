# Создать конфигурации для представления в виде дерева

Чтобы представление в виде дерева, которое отображает данные в иерархической структуре, настроить в соответствии с вашими требованиями, можно сконфигурировать структуру дерева. Это выполняется с помощью настроек пользователя, которые также можно вызвать из базы данных изделий.

Конфигурация задается через так называемые "Главные узлы", которые затем отображаются как высшая иерархия в структуре дерева. При этом можно использовать все поля, имеющиеся в базе данных изделий, в качестве структурных узлов для группировки. Таким образом можно подготовить данные изделия для разных клиентов и различных видов проектирования.

Условие:

Вы открыли диалоговое окно База данных изделий (Сервисные программы > Изделие > Управление).

1. [Дополнительно] > Настройки
2. В следующем диалоговом окне в поле Конфигурация дерева щелкните по ++...++.
3. В следующем диалоговом окне выберите схему из раскрывающегося списка Схема либо создайте новую.
4. Щелкните на панели инструментов у поля Главные узлы по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
5. Задайте в диалоговом окне Конфигурация дерева — Главные узлы в поле Имя имя главного узла, которое должно отражаться в представлении в виде дерева базы данных изделий.
6. При необходимости щелкните ++...++ в поле Фильтр, чтобы в последующем диалоговом окне Фильтр задать фильтр.
7. Щелкните на панели инструментов у поля Свойства по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
8. Выберите из списка Свойства нужное (например "Производитель"), и нажмите ++OK++.
9. Повторите эти шаги, чтобы задать другие критерии / создать другие главные узлы.
10. Щелкайте по кнопке ++OK++ до тех пор, пока не вернетесь к диалоговому окну База данных изделий.

!!! info "Для сведения:"

    Выполняется адаптация представления структуры дерева.

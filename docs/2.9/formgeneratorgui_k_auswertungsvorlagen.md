# Шаблоны отчетов

Шаблоны отчетов позволяют сохранять и повторно использовать настройки для отчетов. Отчеты могут быть сгенерированы на базе шаблонов отчетов. Все настройки, которые уже были установлены в шаблонах отчетов, применяются для этих отчетов. Шаблоны отчетов экспортируются и импортируются как файлы *.xml.

При генерации существующего отчета из шаблона отчета текущий отчет будет удален, а взамен создан новый отчет.

!!! tip "Совет:"

    Вы можете создать для одного типа отчета ***несколько шаблонов отчета***. Если имеющийся шаблон отчета (например, с именем 0001) уже существует, то при создании остальных шаблонов данного типа отображается запрос, должны ли перезаписываться прежние шаблоны. При выборе [Нет] создаются два одноименных шаблона, и их имена затем необходимо изменить.

### Условные отчеты

Посредством шаблонов отчетов вы можете генерировать отчеты независимо от свойств проекта. Для этого определите фильтр проекта и назначьте для него шаблон отчета. При генерации из шаблона отчета генерируется только один отчет, если текущий проект удовлетворяет условиям фильтра проекта.

Данный шаблон отчета получит имяTitel_EPLAN_Demo.Для второго проектаEPLAN_Testсоздайте в базовом проекте шаблон отчета для титульного/первичного листа с формой B. Данный шаблон отчета получит имяTitel_EPLAN_Test.Определите в фильтре проекта свойство проекта как условие фильтра и присвойте фильтр проекта шаблону отчетаTitel_EPLAN_Demo.Создать проекты Создайте из базового проектапроектыEPLAN_DemoиEPLAN_Test.Чтобы различать проекты при генерации, назначьте свойство проекта первому проектуEPLAN_Demo, которое вы указали в фильтре проекта в качестве условий фильтра.Генерация отчета по проектам При генерации проектаEPLAN_Demoна основе шаблона отчета EPLAN использует шаблон отчетаTitel_EPLAN_Demoи генерирует титульный лист изформыA.При генерации проектаEPLAN_TestEPLAN использует шаблон отчетаTitel_EPLAN_Testи генерирует титульный лист из формы B.

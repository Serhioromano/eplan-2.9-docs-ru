# Вкладка Инструменты (экспорт ЧУ для всех машин)

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт Rittal - Perforex BC / Secarex. Выберите вкладку Инструменты.
* Сервисные программы > Данные изготовления > Механическая обработка > Экспорт Rittal - Perforex BC / Secarex. Рядом с полем Машина нажмите ++...++. Выберите вкладку Инструменты.

На этой вкладке определяется загрузка инструментов машины. При этом в используемой машине уже настроена загрузка по умолчанию.

Обзор основных элементов диалогового окна:

### Активно

Этот флажок служит для активирования необходимой фрезы. С помощью него можно определить несколько фрез и указать, какую фрезу следует использовать для экспорта. У всех других типов инструментов флажок не установлен.

### Номер

С помощью номера инструмент присваивается оснащению машины. Вводить можно только целые положительные числа.

### Тип сверления

Из раскрывающегося списка можно выбрать тип инструмента 'Сверло', 'Резьбонарезной инструмент' или 'Фреза'.

### Диаметр

Диаметр инструмента, например сверла. Вводить можно только положительные числа. Разрешается вводить десятичные дроби.

!!! note "Замечание:"

    Во избежание разрушения контура при позиционировании фрезерного инструмента для контуров с очень малой высотой должны выполняться следующие условия:

* Продольные отверстия экспортируются только в том случае, если высота не менее чем на 0,2 мм превышает больший из двух диаметров фрезы / центрового сверла.
* Прямоугольные отверстия экспортируются только в том случае, если высота не менее чем на 2 мм превышает диаметр фрезы / центрового сверла.
* Отверстия типа "шестиугольник" и "восьмиугольник" экспортируются только в том случае, если длина ребра не менее чем на 2 мм превышает диаметр фрезы / центрового сверла.
* Закругления или фаски прямоугольных отверстий при экспорте ЧУ Rittal — Perforex / Secarex не экспортируются, если радиус фрезы больше радиуса закругления или ширины фаски.

### Z высокое

При неровной поверхности детали фрезерный инструмент подвергается опасности при перемещении от контура к контуру. Эта настройка регулирует, насколько высоко инструмент должен подниматься над деталью при свободном перемещении.

Z высокое описывает значение, на которое фрезер поднимается над деталью, то есть высоту, на которой движется острие фрезы, когда инструмент не срезает стружку (например, при технологической перемычке). Вводить можно только положительные числа.

### Z глубокое 1 / 2

Поскольку фрезерный инструмент не движется по детали постоянно на одной и той же глубине (что привело бы к его быстрой поломке), а совершает волнообразные движения вдоль фрезерного контура, эта настройка регулирует глубину проникновения в материал при работе.

Z глубокое 1 и Z глубокое 2 описывают глубину проникновения фрезы (под поверхность детали). Фреза варьирует между этими двумя величинами, что обеспечивает более равномерное стачивание. Возможен ввод отрицательных величин.

Производитель рекомендует здесь значения -4 и -7.

### Номер центрового сверла

Заданный на машине номер инструмента центрового сверла. В схеме "Стандарт" для этого инструмента определена фреза. Вводить можно только целые положительные числа.

### Материалы

Обозначение материала, которое присвоено машине на вкладке Общ. номера материала. Это назначение относится только к фрезе.

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт Rittal - Perforex LC / Secarex. Выберите вкладку Инструменты.
* Сервисные программы > Данные изготовления > Механическая обработка > Rittal - Perforex LC / Secarex. Рядом с полем Машина нажмите ++...++. Выберите вкладку Инструменты.

На этой вкладке определяется загрузка инструментов машины. Резьбонарезной инструмент можно отрегулировать, используя предоставляемые возможности.

Обзор основных элементов диалогового окна:

### Номер

С помощью номера инструмент присваивается оснащению машины. Вводить можно только целые положительные числа.

### Тип сверления

Из раскрывающегося списка можно выбрать тип инструмента "Не определено" или "Резьбонарезной инструмент".

### Диаметр

Диаметр резьбонарезного инструмента. Вводить можно только положительные числа. Разрешается вводить десятичные дроби.

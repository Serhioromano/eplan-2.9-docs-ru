# Новый пункт меню для экспорта данных изготовления

Для экспорта данных проекта в EPLAN Smart Wiring на платформе EPLAN под путем меню Сервисные программы > Данные изготовления теперь доступен новый отдельный пункт меню EPLAN Smart Wiring. В открывшемся диалоговом окне содержатся только релевантные параметры настройки для имени и каталога вывода файла EPDZ.

### Дополнение данных экспорта свойствами кабелей

Данные экспорта будут по-прежнему выводиться в формате EPDZ. Для предоставления возможности маршрутизации кабелей в электрошкафах в приложении EPLAN Smart Wiring теперь дополнительно экспортируются значения таких свойств кабелей:

* ОУ (полное) (ид. 200006)
* Кабель / Группа соединений: Длина зачистки источника (ид. 20081)
* Кабель / Группа соединений: Длина зачистки цели (ид. 20082)
* Кабель / Группа соединений: Длина (полностью) (ид. 20257)
* Кабель / Группа соединений: Тип (ид. 20040)
* Кабель / Группа соединений: Количество соединений (ид. 20042)
* Место установки: Описание (ид. 1250)
* Примечание (ид. 20045).

!!! note "Замечание:"

    Лицензия на этот новый интерфейс предоставляется через дополнительный модуль расширения, за который взимается плата. Информацию об этом можно найти в текущей спецификации договорных работ и услуг для EPLAN Pro Panel.

# Экспортировать спецификации

Можно экспортировать ***все*** данные изделия спецификации независимо от настроек или активных фильтров, т. е. выдаются все изделия со всеми полями данных.

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Данные проекта > Устройство / изделие > Навигатор спецификаций
2. В диалоговом окне Спецификация — <Имя проекта> выберите пункт всплывающего меню Экспортировать.
3. В диалоговом окне Экспортировать спецификацию из раскрывающегося списка Сохранить в выберите дисковод и каталог, в котором должен создаваться файл экспорта.
4. Введите в поле Имя файла имя создаваемого файла экспорта.
5. Выберите из раскрывающегося списка Тип файла одну из записей для формата создаваемого файла экспорта.
6. Щелкните по кнопке ++Сохранить++.

!!! info "Для сведения:"

    Спецификация экспортируется под указанным именем в установленный каталог.

Экспортированные таким образом данные можно обрабатывать и снова экспортировать. Однако при импорте записываются только определенные поля файла экспорта. Обзор полей приведен в разделе "[Спецификации: Поля в файле экспорта](partslistgui_k_felderexportdatei.md)".

Обратите внимание на то, что экспорт и импорт можно осуществлять только в проекте!

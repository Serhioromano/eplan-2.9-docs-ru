# Определить графику в зависимости от определений функций свойство в формах

Свойство формы Присвоение определения функции графике требуется, чтобы различная графика могла размещаться в зависимости от конкретных значений определения функции, например, символы для клемм в зависимости от типа клемм.

Условия:

* Вы открыли проект.
* В редакторе форм открыта та форма, свойства которой требуется обработать (Сервисные программы > Основные данные > Форма > Открыть > [Открыть]).
* Открыто диалоговое окно Свойства формы — <Имя формы> (Всплывающее меню > Свойства на форме в диалоговом окне навигатора Страницы — <Имя проекта>).

1. В диалоговом окне Свойства формы — <Имя формы> щелкните кнопку ++...++ в столбце Значение свойства Присвоение определения функции графике.
2. В диалоговом окне Присвоение определения функции графике щелкните ![](images/all_new_as.png){: .ui-icon } (Создать), чтобы создать новую строку в таблице присвоений.
3. В поле Определение функции щелкните по кнопке ++...++.
4. В диалоговом окне Определения функции выберите определение функции, которому Вы хотите присвоить графику.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Соответствующие данные будут скопированы в поля Категория, Группа и Описание.

6. В поле Имя символа щелкните по кнопке ++...++.
7. Выберите в диалоговом окне Выбор символов символ, графику которого вы хотите присвоить определению функции.
8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Соответствующие данные будут применены в полях Библиотека символов и Описание.

9. В поле Вариант символа выберите требуемый вариант из раскрывающегося списка всех имеющихся вариантов текущего символа.
10. Выполните аналогичные действия для всех других присвоений.
11. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Присвоения сохраняются и могут быть вставлены с помощью заполнителей в форме.

В диалоговом окне Свойства формы — <Имя формы> присвоения выводятся в столбце Значение следующим образом:

<Категория>;<Группа>;<Определение функции>; <Библиотека символов>;<Номер символа>;<Вариант>|
<Категория>;<Группа>;<Определение функции>; <Библиотека символов>;<Номер символа>;<Вариант>|...

!!! example "Пример:"

    200;1;1;SPECIAL;30;0|100;99;1;IEC_EPLAN_D;211;1

!!! note "Замечание:"

    Обратите внимание на то, что на просмотр выводятся только символы ***сохраненных*** библиотек символов.

# Обработать основные персональные данные

!!! warning "Предупреждение:"

    Персональные основные данные можно просматривать и обрабатывать ***только*** в ***структуре дерева***, список показывает только данные изделий, но не производителей, поставщиков и клиентов.

### Создать основные персональные данные

Условия:

* Вы открыли базу данных изделий: Сервисные программы > Изделие > Управление
* В ***дереве*** выделен уровень иерархии 'Клиенты' или 'Производители / поставщики'.

1. Выберите пункт всплывающего меню Создать.

!!! info "Для сведения:"

    EPLAN автоматически генерирует новую запись данных с кратким именем Создать_<порядковый_номер>.

2. На вкладке Адрес в правой части диалогового окна следует ввести данные клиента или производителя / поставщика.

!!! note "Замечание:"

    Краткие имена клиентов и производителей / поставщиков должны быть однозначными. Если случайно введено имя, которое уже существует, EPLAN укажет на это при нажатии кнопок [Применить] или ++Закрыть++. Запись данных будет сохранена только после того, как будет указано новое имя.

### Ввод персональных основных данных

Условия:

* Вы открыли базу данных изделий: Сервисные программы > Изделие > Управление
* Выбраны требуемые клиенты или производители / поставщики.

1. Чтобы ввести данные для выделенных записей данных, используйте вкладку Адрес в правой части диалогового окна.

!!! info "Для сведения:"

    При выборе нескольких записей данных отображаются только те данные, которые являются идентичными или не определены (т. е. "Пусто"); эти данные можно глобально вводить или изменять для всех выделенных записей данных.

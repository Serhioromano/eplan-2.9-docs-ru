# Создание пользователем конфигураций свойств

Везде, где для объекта свойства выведены в табличной форме (cвойства проекта, страницы, устройства и т. д.), существует возможность создания собственной конфигурации свойств. Таким образом, свойства из различных групп могут быть сведены вместе и сохранены в определенной пользователем конфигурации свойств.

Условие:

Для объекта открыто диалоговое окно свойств, например диалоговое окно Свойства проекта.

1. Выберите из раскрывающегося списка Категория запись "Определено пользователем".
2. Выберите пункт всплывающего меню Конфигурировать.
3. Щелкните ![](images/all_new_as.png){: .ui-icon } (Создать) рядом с раскрывающимся списком Схема диалогового окна Конфигурировать свойства.
4. Создайте новую схему и щелкните ++OK++.
5. Обработайте свойства, добавьте остальные свойства или удалите ненужные свойства.
6. Для изменения последовательности отображения свойств используйте кнопки со стрелками.
7. Щелкните ![](images/all_save_as.png){: .ui-icon } (Сохранить) для сохранения выбранных свойств под названиями схем. Или щелкните ![](images/all_export_as.png){: .ui-icon } (Экспортировать) для экспорта конфигурации свойств и обеспечения доступа для других пользователей.
8. Щелкните по кнопке ++OK++.

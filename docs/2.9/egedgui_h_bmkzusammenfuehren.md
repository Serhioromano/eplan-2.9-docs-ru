# Объединить ОУ

В EPLAN усл. обозначения с различными обозначениями устройства сводятся к одному новому устройству путем присвоения условным обозначениям единого ОУ.

Условие:

Вы открыли проект. В Графическом редакторе открыта страница проекта.

1. Выделите условные обозначения (***как минимум*** два), которые должны быть объеденены.
2. Выберите пункты меню Обработать > Обзор свойств.
3. Выберите в диалоговом окне Обзор свойств из раскрывающегося списка запись "Независимые от функций свойства".
4. Нажмите в таблице свойств для свойства Имя (полное) в столбце Значение, а затем — кнопку ++...++.
5. В диалоговом окне Полное ОУ введите ОУ, которое должно быть одновременно присвоено всем выделенным условным обозначениям.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    ОУ присваивается всем выделенным условным обозначениям. На условных обозначениях автоматически отображаются перекрестные ссылки оборудования.

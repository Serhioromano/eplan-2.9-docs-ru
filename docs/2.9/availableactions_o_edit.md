# Операция: edit

**Параметр** |  **Описание**
---|---
PROJECTNAME  |  Имя проекта с полным путем файла (является опцией).
Если не задано, то выбранный проект используется, когда операция вызывается через интерфейс пользователя (напр., через сценарий или панель инструментов). При вызове из командной строки Windows следует определить PROJECTNAME или сначала следует использовать ProjectAction. В противном случае отобразится системное сообщение.
PAGENAME  |  Имя открываемой страницы (необязательно). Требуется, если параметрами были указаны X, Y.
DEVICENAME  |  Имя функционального элемента (необязательно).
X  |  Абсцисса (необязательно). Параметр действителен только тогда, когда указан параметр PAGENAME.
Y  |  Ордината (необязательно). Параметр действителен только тогда, когда указан параметр PAGENAME.
INSTALLATIONSPACE  |  Имя открываемого пространства листа (является опцией). Не может использоваться вместе с PAGENAME.
DEVICENAME3D  |  Имя функционального элемента в пространстве листа (является опцией).

!!! example "Пример:"

    Открыть проект:edit
    /PROJECTNAME:C:\Projects\EPLAN\ESS_Sample_Project.elkОткрыть страницу:edit
    /PROJECTNAME:C:\Projects\EPLAN\ESS_Sample_Project.elk
    /PAGENAME:=CA1+EAA/1Открыть страницу с обозначением устройства:edit
    /PROJECTNAME:C:\Projects\EPLAN\ESS_Sample_Project.elk
    /DEVICENAME:=EB3+ET1-F1Открыть страницу и установить курсор на позицию X, Y:edit
    /PROJECTNAME:C:\Projects\EPLAN\ESS_Sample_Project.elk
    /PAGENAME:=EB3+ET1/2
    /X:200
    /Y:100

# Диалоговое окно Графический выбор макроса

В настройках пользователя (Параметры > Настройки > Пользователь > Графическая обработка > Макросы) установлен флажок Использовать графический выбор макроса.

* Вставить > Макрос окна / символа.
* Страница > Макрос страницы > Вставить.

В этом диалоговом окне в определенном каталоге отображаются все подкаталоги и, независимо от выбора, макросы, содержащиеся в определенном подкаталоге, с информацией о файлах, а также графическим предварительным просмотром.

Обзор основных элементов диалогового окна:

### Путь файла

В этом поле отображается текущий путь файла для макросов. По умолчанию это путь <Дисковод>:\<Каталог программ>\Макросы\<Ид. клиента>, созданный при установке. Однако можно в ручном режиме ввести другой каталог или выбрать его, щелкнув мышью по соответствующему изображению каталога.

При нажатии кнопки ![](images/all_nexthigherlevel_as.png) (На один слой выше) или соответствующей картинки в области Каталог вы переходите в структуре каталога на один уровень выше.

!!! tip "Совет:"

    Если поставить в этом поле ввода для пути к каталогу обратную косую черту (\\) после названия каталога, а затем букву (или цифру), то в каталоге с макросами первый файл макроса с этой буквой (или цифрой) будет выделен в области Выбор файлов.

### Каталог

В этой области в избранном каталоге отображаются все имеющиеся подкаталоги в виде рисунков. По умолчанию используется логотип EPLAN и соответствующее имя подкаталога отображается в виде текста.

В определенной записи каталога выберите пункт всплывающего меню Определение рисунка, тогда в настроенном стандартном каталоге для рисунков откроется диалоговое окно выбора, с помощью которого можно будет выбрать для каталога другой графический файл (*.jpg, *.png и т. д.).

В определенной записи каталога выберите пункт всплывающего меню Обработка текста, в этом случае открывается диалоговое окно Ввод. Здесь можно ввести многоязычный текст для выбранного каталога.

Заданные здесь для каталога значения сохраняются в специфических для фирмы настройках, и их можно в любое время изменить через пункты меню графического выбора макроса или через диалоговое окно Настройки (Параметры > Настройки > Фирма > Представление > Графический выбор макроса).

### Выбор файла

Здесь представлены в виде таблицы найденные в выбранном каталоге макросы с именем, типом, датой и размером. В эту область интегрирован предварительный просмотр.

Чтобы узнать, какой вид представления имеет макрос и имеется ли несколько вариантов, щелкните по символу "+" перед именем макроса, а на уровнях иерархии под ним — по видам представления ("многополюсный", "однополюсный" и т. д.).

Для выбора макроса щелкните мышкой по желаемому варианту, а затем нажмите ++OK++. При двойном щелчке по имени макроса для вставки выбирается первый из имеющихся вариантов макроса, и диалоговое окно выбора закрывается.

### Всплывающее меню

Пункт меню |  Значение
---|---
Рисунки |  Позволяет отображать рисунки каталогов в размерах "мелкий", "средний" или "большой".
Фильтр: <Тип файла макроса> |  Фильтрует отображение выбора файла в соответствии с выбранным типом файла (*.ema, *.ems, *.emp или *.fc2).
Сбросить на стандарт |  Автоматически сбрасывает выбранный каталог на предварительно настроенный каталог макросов.

!!! tip "Совет:"

    Наряду с известными типами файлов макросов можно также фильтровать отображение выбора файлов по контурам типа "Выдавливание контура" (*.fc2). Это относится ко вкладке Технические данные в базе данных изделий. При помощи поля Макрос в ней можно присвоить устройству чертеж контура. При выборе чертежа контура можно также использовать графический выбор макроса.

При повторном вызове графического выбора макроса снова открывается диалоговое окно с последним выбранным макросом и последними настройками.

!!! note "Замечание:"

    В отличие от обычно используемого выбора макросов в диалоговом окне Графический выбор макроса нельзя выполнять функции, касающиеся каталогов или файлов. Так, например, нельзя создавать, переименовывать или удалять каталоги или файлы.

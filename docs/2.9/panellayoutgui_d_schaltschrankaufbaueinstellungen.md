# Диалоговое окно Настройки: Компоновка электрошкафа 2D

* Вы открыли проект. Данные проекта > Устройство / изделие > Навигатор компоновки электрошкафа 2D > Всплывающее меню > Настройки.
* Параметры > Настройки > Пользователь > Графическая обработка > Компоновка электрошкафа 2D.

В этом диалоговом окне для размещения изделий на монтажной плате указываются предварительные настройки, которые затем будут влиять на обработку проекта в диалоговом окне Компоновка электрошкафа 2D - <Имя проекта>.

Обзор основных элементов диалогового окна:

### Направление оснащения

В этом групповом поле указывается, должно ли оснащение монтажной платы выполняться в горизонтальном или вертикальном направлении: выберите соответствующую опцию. Обратите внимание на то, что направление оснащения Вертикально при многократном размещении направлено сверху вниз.

### Размещение изделия

В этом групповом поле можно ввести следующие данные:

* Угол поворота: Выберите из этого раскрывающегося списка, на какой угол требуется повернуть изделие при размещении.
* Точка захвата: Выберите из этого раскрывающегося списка, на какой позиции изделия должна находиться точка захвата во время размещения.
Если вы активировали отображение точек вставки символов и текста в меню Вид > Точки вставки, а точка захвата отклоняется от точки вставки, она будет отображаться дополнительно.
* Преимущ. из макроса: Если в групповом поле Копирование выбран один из параметров Макрос ... из осн. данных изделия, этот флажок позволит установить способ использования точки захвата: если он установлен и имеется макрос, будет использоваться точка захвата из макроса.

### Копирование

В этом групповом поле можно ввести следующие данные:

* Размеры из основных данных изделия: Выберите эту опцию, если необходимо скопировать размеры изделий из основных данных изделий.
* Макрос и размеры из осн. данных изделия: Выберите эту опцию, если необходимо скопировать размеры макросов и изделий из основных данных изделий.
* Макрос из осн. данных изделия: Выберите эту опцию, если установленный в изделии макрос необходимо разместить с его оригинальными размерами. Тогда установленные в изделии размеры игнорируются.
* Ввод вручную: Выберите эту опцию, если при размещении требуется вручную ввести размеры в диалоговом окне Удлинение и интервал.
* Ввод вручную, макрос из основных данных изделия: выберите этот параметр, если при размещении требуется вручную ввести размеры в диалоговом окне Удлинение и интервал, предварительная настройка для размеров должна быть обусловлена критериями основных данных изделия.

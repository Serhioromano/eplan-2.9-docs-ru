# Сгенерировать неразмещенные соединения

Неразмещенные соединения – соединения между выводами устройства, еще не размещенные на странице. Есть возможность сгенерировать неразмещенные соединения между выводами устройства. При генерировании соединения можно сгенерировать жильное / проволочное соединение, использовать имеющиеся кабели или создать новые кабели.

### Сгенерировать неразмещенное жильное / проволочное соединение

Условие:

Создано минимум 2 устройства с выводами устройств.

1. Данные проекта > Устройства > Навигатор
2. Выделите устройство с выводом, которое должно стать источником соединения.
3. Выберите пункты меню Данные проекта > Устройствo > Подключить устройства.

!!! info "Для сведения:"

    Открывается диалоговое окно Подключить устройства.

!!! info "Для сведения:"

    Выделенное устройство автоматически отображается в поле Источник устройства. Выводы выделенного устройства отображаются в столбце Вывод устройства.

4. Щелкните рядом с полем Цель устройства по ++...++, выберите в диалоговом окне устройство, которое должно стать целью соединения, и щелкните ++OK++.

!!! info "Для сведения:"

    Выбранное устройство автоматически отображается в поле Цель устройства. Выводы выбранного устройства отображаются в столбце Вывод устройства.

5. Выделите в левой и правой областях диалогового окна равное количество выводов устройства.

!!! info "Для сведения:"

    Если выделено неравное число выводов устройства, тогда все соединения создаются для меньшего количества. Если было выделено больше выводов устройства, чем имеется кабельных соединений, соединяются те выводы устройства, для которых доступны кабельные соединения.

6. Чтобы сгенерировать жильные / проволочные соединения, нажмите ++Генерировать соединения++.

!!! info "Для сведения:"

    Генерируются соединения.

### Сгенерировать неразмещенное кабельное соединение

Условие:

Создано минимум 2 устройства с выводами устройств.

1. Данные проекта > Устройства > Навигатор
2. Выделите устройство с выводом, которое должно стать источником соединения.
3. Выберите пункты меню Данные проекта > Устройствo > Подключить устройства.

!!! info "Для сведения:"

    Открывается диалоговое окно Подключить устройства.

!!! info "Для сведения:"

    Выделенное устройство автоматически отображается в поле Источник устройства. Выводы выделенного устройства отображаются в столбце Вывод устройства.

4. Щелкните рядом с полем Цель устройства по ++...++, выберите в диалоговом окне устройство, которое должно стать целью соединения, и щелкните ++OK++.

!!! info "Для сведения:"

    Выбранное устройство автоматически отображается в поле Цель устройства. Выводы выбранного устройства отображаются в столбце Вывод устройства.

5. Выделите в левой и правой областях диалогового окна равное количество выводов устройства.

!!! info "Для сведения:"

    Если выделено неравное число выводов устройства, тогда все соединения создаются для меньшего количества. Если было выделено больше выводов устройства, чем имеется кабельных соединений, соединяются те выводы устройства, для которых доступны кабельные соединения.

6. Чтобы для соединения использовать имеющийся кабель, щелкните рядом с полем ОУ кабеля по ++...++, выберите кабель в диалоговом окне и нажмите ++OK++.
7. Чтобы создать новый кабель для соединения, щелкните рядом с полем  Тип кабеля по ++...++, выберите в диалоговом окне типа кабеля подходящее изделие и нажмите ++OK++.
8. Введите в поле Тип кабеля ОУ для кабеля.
9. Чтобы открыть кабель после генерирования соединения в диалоговом окне Обработать кабели, установите флажок Обработать кабели.
10. Щелкните по ++Генерировать соединения++.

!!! info "Для сведения:"

    Генерируются соединения.

!!! info "Для сведения:"

    Кабель автоматически отображается в диалоговом окне Обработать кабели.

!!! info "Для сведения:"

    Когда кабель создан, новый кабель отображается в навигаторе кабелей по пути Данные проекта > Кабели > Навигатор.

# Вкладка Общее

У вас установлен конфигуратор шлангопроводов в качестве дополнительного модуля для EPLAN Fluid. Вы запустили EPLAN Fluid и конфигуратор шлангопроводов.

* Выберите в EPLAN Fluid пункты меню Параметры > Настройки > Пользователь > Add-ons > Конфигуратор шлангопроводов и щелкните в следующем диалоговом окне по вкладке Общее.
* Или выберите в диалоговом окне Конфигурировать шлангопроводы конфигуратора шлангопроводов пункты меню Дополнительно > Настройки и щелкните в следующем диалоговом окне по вкладке Общее.

Или у вас установлен конфигуратор шлангопроводов в качестве самостоятельной программы, и программа запущена.

* Выберите в конфигураторе шлангопроводов пункты меню Параметры > Настройки > Пользователь > Add-ons > Конфигуратор шлангопроводов и щелкните в следующем диалоговом окне по вкладке Общее.

На этой вкладке можно задать общие настройки для конфигуратора шлангопроводов.

Обзор основных элементов диалогового окна:

### Префикс

Введите в этом поле любой знак или последовательность знаков.

Каждый созданный в конфигураторе шлангопровод получает временное ОУ, состоящее из заданного префикса и порядкового номера. Префикс служит для идентификации заново создаваемых шлангопроводов, в которых еще нет окончательного ОУ.

!!! note "Замечание:"

    При выборе стандартной схемы DIN 20066_2012-01 в этом поле в качестве префикса по умолчанию стоит запись "H".

!!! example "Пример:"

    Вы создаете два новых шлангопровода в конфигураторе. В качестве префикса вы выбрали значение по умолчаниюH. Первый заново созданный шлангопровод получает временное ОУH1, второй шлангопровод — временное ОУH2.

### Шаблон изделия

Для каждого сгенерированного кода типа можно создать на выбор изделие проекта или ссылку изделия.

С помощью поля Шаблон изделия можно оказать следующее влияние на его поведение: Если в поле Шаблон изделия не вносится никакое значение, создается ***ссылка изделия***. Если вы, напротив, заполнили поле значением, создается ***изделие проекта***.

Если вы используете конфигуратор шлангопроводов в качестве дополнительного модуля к EPLAN Fluid, через имеющуюся в ячейках таблицы клавишу ++...++ вы можете получить доступ к выбору изделий и выбрать там нужный шаблон.

!!! note "Замечание:"

    При выборе стандартной схемы DIN 20066_2012-01 в этом поле по умолчанию стоит запись "Fluid-Hose-1".

### Формат кода типа

С помощью этого поля вы задаете формат отображения для кода типа шлангопроводов и определяете состав частей кода типа. Генерированный код типа выводится в диалоговом окне Обработать ++...++.

При помощи кнопки ++...++ открывается диалоговое окно [Формат: Свойство](fluidhoseconfiggui_d_formattypencode.md), в котором можно выбрать элементы формата кода типа в форме свойств и разделителей и расположить их в нужном порядке. По умолчанию в этом диалоговом окне предлагается формат отображения для генерирования кода типа согласно стандарту DIN 20066. При выборе других комбинаций элементов формата и/или путем добавления произвольных свойств можно также определить отклонение кода типа от стандарта DIN. Так, например, можно заказать шлангопроводы в соответствии со специальными данными производителя.

### Единица измерения длины шлангопровода

Выберите из раскрывающегося списка нужную единицу измерения для отображения длины шлангопровода. Единица измерения ***не*** отображается в сгенерированном коде типа.

!!! note "Замечание:"

    При выборе стандартной схемы DIN 20066_2012-01 в этом поле по умолчанию стоит запись "мм".

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

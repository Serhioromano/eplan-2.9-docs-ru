# Наборы буквенных обозначений

В EPLAN можно создавать различные наборы буквенных обозначений для разных стандартов, стран и клиентов. Буквенные обозначения могут состоять из нескольких символов, например KT или KTP. Буквенные обозначения экспортируются и импортируются. При экспорте буквенные обозначения сохраняются со спецификой определения функции, а при импорте они снова присваиваются определениям функции.

В настройках проекта (Устройство > Нумерация (online)) установите, какой набор буквенных обозначений необходимо использовать для проекта (например, DIN). Каждому определению функции присвоено определенное буквенное обозначение. При создании функции соответствующее буквенное обозначение копируется из определения функции в обозначение устройства, но может изменяться вручную. Буквенные обозначения вводятся в функцию всегда фиксированными и не описываются.

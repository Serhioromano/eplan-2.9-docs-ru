# Обновление баз данных проекта

В результате различных дополнений и оптимизаций базы данных проекта обновлены до версии 2.9. Эти изменения и оптимизации привели к тому, что **новыепроекты** принципиально создаются с новыми базами данных проектов.

Как это повлияет на старые проекты

Чтобы обрабатывать **старые проекты** в новой версии EPLAN, а также просто просматривать их, необходимо ***обязательно*** выполнить обновление баз данных этих проектов. Обновление старых проектов осуществляется при первом открытии актуальной версии EPLAN. Система спрашивает, следует ли копировать проект в текущую версию.

В случае подтверждения клавишей [Да] проект сначала обновляется, а затем открывается. При этом в каталоге проекта дополнительно создается резервная копия вашего проекта с еще не обновленными базами данных проекта. Этот файл резервирования *.zw1 можно, как обычно, восстановить путем резервирования данных.

Если старый проект ***не обновить***, он ***не откроется*** в версии 2.9.

!!! warning "Предупреждение:"

    Новые проекты, а также старые обновленные проекты больше ***нельзя*** открывать в старой версии EPLAN.

!!! tip "Совет:"

    В случае обновления программы вы можете обновить сразу несколько старых проектов в управлении проектами. Для этого выберите пункт всплывающего меню Автоматизированная обработка для выделенных проектов. Для обновления баз данных проектов в диалоговом окне Выполнить: Автоматизированная обработка доступна новая схема "Копирование проекта".

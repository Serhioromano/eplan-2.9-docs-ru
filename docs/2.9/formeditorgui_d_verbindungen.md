# Диалоговое окно Соединения

Вы открыли проект. Сервисные программы > Основные данные > Форма > Открыть > [Открыть] для формы типа "Схема кабельных соединений". Вы открыли диалоговое окно Свойства формы — <Имя формы> (Всплывающее меню > Свойства на форме в диалоговом окне навигатора Страницы — <Имя проекта>) и в поле Значение свойства Соединения щелкнули кнопку ++...++.

В этом диалоговом окне Вы управляете присвоением соединений к выводам устройств в формах для отображения спецификации кабеля.

Обзор основных элементов диалогового окна:

### Источник (обозначение) / Цель (обозначение)

В этой таблице сохраняются обозначения источников и целей соединения.

### Соединение (цвет / номер)

Здесь Вы можете сохранить цвет или номер соединения.

Таким образом, присвоение соединения двум выводам устройства будет представлено в виде трех "записей"; несколько присвоений соединений разделяются с помощью "обратного" слеша.
("Запись" — математическое понятие, обозначающее упорядоченный набор объектов, элементы которых расположены в заданной последовательности.)

!!! example "Пример:"

    Запись1;2:красный|2;3;|;3;4;синийозначает для кабеля:Красная жила типа кабеля соединяет вывод устройства1с выводом устройства2. Вторая жила типа кабеля соединяет вывод устройства2с выводом устройства3, а синяя жила типа кабеля соединяет вывод устройства3с выводом устройства4.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

# Создание схемы сравнения настроек

В схеме сравнения настроек устанавливается, какие свойства сравниваются друг с другом при сравнении проектов в ходе проверки проекта. С помощью этой схемы EPLAN может сравнить все настройки, которые устанавливаются для соответствующего проекта в диалоговом окне Параметры > Настройки > Проекты > "Имя проекта".

Условия:

* [Выбрать данных управления проектами](prjmanagementgui_h_projekteverwalten.md)
* [Считать каталоги и проекты](prjmanagementgui_h_verzeichniseinlesen.md)

1. Проект > Управление
2. Выделите проект.
3. Выберите ++Дополнительно++ > Импорт.
4. Щелкните рядом с флажком Сравнить настройки по кнопке ++...++.
5. Щелкните рядом с полем Схема по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
6. введите в диалоговом окне Новая схема в поле Имя введите информативное имя для новой схемы, и щелкните по ++OK++.
7. Чтобы выбрать настройку для сравнения настроек, активируйте в поле Сравниваемые настройки флажок перед настройкой.
8. Чтобы выбрать настройку для сравнения настроек, активируйте в поле Сравниваемые настройки флажок перед настройкой.
9. Щелкните по кнопке ++Сохранить++ и по ++OK++.

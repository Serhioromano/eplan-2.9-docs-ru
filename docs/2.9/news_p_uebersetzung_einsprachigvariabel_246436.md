# Расширенный выбор языка для одноязычных текстов с переменной

В специфических для проекта настройках перевода в поле Одноязычн. (перемен.) указывается язык отображения для переведенных текстов проекта, которым присвоено свойство "Одноязычн. (перемен.)". В раскрывающемся списке этого поля теперь доступны новые записи для языка отображения, например, "Язык отображения 1", "Язык отображения 2" и т. д. (Путь меню для диалогового окна "Настройки": Параметры > Настройки > Проекты > "Имя проекта" > Перевод > Общее.)

![](images/benefit.png)Эффект:

Благодаря новым записям в выборе языка ("Язык отображения 1", "Язык отображения 2" и т. д.) одноязычные тексты с переменной всегда могут выводиться на текущем языке отображения, даже если язык изменяется в настройках перевода.

При выборе такой записи тексты свойства отображаются на языке, указанном в качестве первого, второго и т. д. языка отображения. В зависимости от количества языков отображения здесь можно внести до пяти записей.

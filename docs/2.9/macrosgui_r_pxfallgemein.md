# Вкладка Общ. (Настройки: Внешняя обработка)

* Вы открыли проект. Сервисные программы > Внешняя обработка свойств > Экспортировать данные. Нажмите в диалоговом окне Внешняя обработка рядом с полем Настройки по ++...++. Выберите вкладку Общ..

* Параметры > Настройки > Пользователь > Интерфейсы > Внешняя обработка. Выберите вкладку Общ..

На этой вкладке можно задать общие настройки для внешней обработки.

Доступны следующие элементы диалогового окна:

### Тип файла

Выберите из этого раскрывающегося списка требуемый тип файла, который необходимо сгенерировать при экспорте. В зависимости от выполненного выбора можно открыть диалоговое окно для указания соответствующих настроек, нажав на кнопку ++Параметры++.

### Целевой файл

Введите в это поле дисковод, каталог и имя генерируемого целевого файла или щелкните по ++...++, чтобы выполнить интерактивный выбор дисковода и каталога.

Через пункт всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути, специфических для EPLAN.

### Шаблон

Если в раскрывающемся списке Тип файла выбрана запись "Файл Excel (*.xls, *.xlsx, *.xlsm, *.xlsb)", то в этом поле можно ввести путь и имя файла шаблона или выбрать его, нажав кнопку ++...++; в этом случае при экспорте этот шаблон определяет формат данных.

Через пункт всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути, специфических для EPLAN.

# Сортировать клеммы

В диалоговом окне Обработать клеммник вы можете изменить последовательность клемм внутри клеммника.

Условие:

Вы выделили клемму или определение клеммника в навигаторе клеммников, навигаторе устройств или в графическом редакторе.

1. Выберите пункт меню Данные проекта > Клеммники > Обработать.
2. В таблице диалогового окна Обработать клеммник щелкните по номеру строки требуемой клеммы, чтобы отметить строку.
3. Удерживая нажатой клавишу ++Ctrl++, отметьте все остальные клеммы, которые вы хотите передвинуть.
4. Передвиньте помеченные клеммы с помощью клавиш-стрелок на желаемую позицию в клеммнике.

!!! info "Для сведения:"

    Клеммам присваивается положение устройства, которое сохраняется в виде числового значения в свойстве Клемма: Положение устройства.

!!! info "Для сведения:"

    Внутри группы клемм (то есть взаимосвязанных клемм с одинаковой позицией) клеммам дополнительно присваивается идентификатор сортировки, который сохраняется в виде числового значения в свойстве Идентификатор сортировки (клемма / контакт штекера).

5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Клеммы сортируются согласно последовательности положения устройств на клеммнике. Если несколько клемм имеют одинаковое положение устройства, то они сортируются согласно последовательности идентификаторов сортировки. Если несколько клемм имеют одинаковый идентификатор сортировки или идентификатор пуст, то они сортируются согласно последовательности обозначения клеммы.

!!! tip "Совет:"

    * Кнопка ++Сортировать++ дает Вам возможность автоматической сортировки всех клемм клеммника по определенным критериям.
    * Отдельные клеммы можно сортировать простым перетаскиванием мышью. Для этого выделите номер строчки нужной клеммы и с помощью функции Drag & Drop перетащите его в нужное место. Также возможен многократный выбор клемм.

!!! note "Замечание:"

    Многоуровневые клеммы и другие группы клемм при сортировке не изменяются. Кроме того, все клеммы одной самостоятельной области получают одинаковое положение устройства. Если клеммы необходимо отсортировать без сохранения многоуровневых клемм, нужно сначала в диалоговом окне Обработать клеммник разделить многоуровневые клеммы с помощью пункта всплывающего меню Разделить.

# Генерировать исходный проект

Условия:

* Вы выделили проект в Навигаторе страниц.
* Вы обладаете полномочиями на генерирование исходных проектов.

1. Выберите пункты меню Сервисные программы > Управление ревизиями > Сравнение свойств проектов > Сгенерировать исходный проект.
2. В диалоговом окне [Сгенерировать исходный проект](revisionmgtgui_d_vergleichsprojekterzeugen.md) введите в соответствующих полях имя и краткое описание (необязательно) исходного проекта.
3. Установите флажок Сгенерировать временный исходный проект, если необходимо сгенерировать временный исходный проект.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В зависимости от того, установлен флажок Сгенерировать временный исходный проект или снят, будет сгенерирован "стандартный" исходный проект с расширением имени файла *.elr или временный исходный проект с расширением имени файла *.elt. Временный исходный проект можно удалить в управлении проектами даже в том случае, если вы ***не*** обладаете полномочием на удаление проектов с расширением имени файла elr.

### Отображать закрытые проекты/проекты для сравнения

Для отображения закрытых проектов или проектов для сравнения в навигаторе страниц выполните следующие действия:

1. Выберите пункт меню Проект > Открыть.
2. В диалоговом окне Открыть проект выберите в поле Тип файла запись 'Закрытые EPLAN-проекты (*.elr)'.
3. Выберите нужный проект (если необходимо, смените соответствующий каталог) и щелкните по ++Открыть++.

!!! info "Для сведения:"

    Выбранный проект отобразится в навигаторе страниц.

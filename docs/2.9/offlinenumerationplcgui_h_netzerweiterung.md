# Определить сетевое расширение для нумерации с данными ПЛК

Обычно сеть заканчивается на клеммах, контактах штекера или выводах функций; с помощью сетевого расширения можно распространить сеть, например за пределы клемм и контактов штекеров. Для этого в диалоговом окне Сетевое расширение требуется указать, какие функции ***не*** должны рассматриваться как границы сети. Эта настройка влияет на обозначение, если в качестве объема выбрана "сеть". Сетевое расширение для нумерации с данными ПЛК не зависит от свойства Соединенный с сетью соответствующей функции.

Условие:

Вы открыли проект.

1. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация с данными ПЛК
2. Выберите в диалоговом окне Настройки: Нумерация с данными ПЛК определенную пользователем схему из раскрывающегося списка или создайте с помощью команды ![](images/all_new_as.png){: .ui-icon } (Создать) новую схему.
3. Щелкните по кнопке ++Сетевое расширение++.
4. Перенесите в диалоговом окне Сетевое расширение требуемые функции с помощью кнопки со стрелкой из списка Выбор в список Сетевое расширение через следующие объекты.

!!! info "Для сведения:"

    В этом случае сеть не заканчивается на выбранных функциях, а продолжается за ними.

!!! info "Для сведения:"

    Сетевое расширение действительно для всех групп форматов, которые имеют объем "сеть".

5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Настройка сохраняется в выбранной схеме.

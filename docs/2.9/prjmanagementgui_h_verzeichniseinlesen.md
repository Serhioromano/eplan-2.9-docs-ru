# Считать каталоги и проекты

Чтобы иметь возможность доступа к существующему проекту при помощи управления проектами, необходимо сосчитать каталоги с проектами в управление проектами. В управлении проектами можно считать только каталоги, содержащие проекты EPLAN.

Условие:

В каталогах, которые вы хотите считать, должны быть проекты.

1. Выберите следующие пункты меню: Проект > Управление
2. В диалоговом окне Управление проектами выберите пункт меню ++Дополнительно++ > Считать каталог.
3. Выберите в диалоговом окне Выбрать каталог уровень каталога в качестве стандартного каталога и щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    EPLAN ищет во всех подчиненных каталогах проекты и считывает их со структурой каталогов в управление проектами.

# Диалоговое окно Показать пользователей

Вы выбрали базу данных управления проектами. Вы считали каталоги и проекты. Проект > Управление. Выделите проект. Выберите ++Дополнительно++ > Показать пользователей.

В этом диалоговом окне отображаются все пользователи, обрабатывающие выделенный проект.

Обзор основных элементов диалогового окна:

### Сейчас проект обрабатывается следующим пользователем

В этом поле перечисляются все пользователи, обрабатывающие выделенный проект. Данные в столбцах Имя регистр., Имя, Телефон, Эл. почта и Имя компьютера указаны в настройках соответствующего пользователя. В столбце Режим отображается режим обработки.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Копировать |  Копирует выделенные строки. Также можно копировать целые строки и, например, вставлять их в MS Excel.
Отправить по эл. почте |  Открывает диалоговое окно Выбрать профиль. Позволяет послать сообщение электронной почты другому пользователю.

### [Обновить]

Для обновления вида нажмите эту кнопку.

# Интерфейс DXF/DWG: Основные положения

Как формат чертежей для общих задач САПР, прежде всего, в области механики, формат DXF/DWG имеет достаточно ограниченные возможности. Так, например, он имеет адаптированную к выполнению этих задач систему координат. Но он не может передавать логику, которая отличает одну схему соединений от другой. Поэтому интерфейс DXF/DWG в EPLAN предназначен прежде всего для импорта и экспорта общей графики; информацию о логике страниц схем соединений перенести нельзя.

Поэтому внутренний формат EPLAN существенно отличается от файловой структуры DXF или DWG. Так, ОУ в EPLAN – это не просто текст из определенного набора букв; это, в том числе, подробные данные, которые могут использоваться при создании, например, списка перекрестных ссылок. А символ – это не просто сочетание линий, он несет в себе информацию, которая лежит в основе перекрестной ссылки. При переносе данных эта информация утрачивается!

Поведение символов, рамок и стандартных форм можно предварительно определить (Параметры > Настройки > Пользователь > Интерфейсы > Экспорт и импорт DXF / DWG, вкладка Определение блока) так, чтобы эти элементы интерпретировались в дальнейшем как блоки AutoCAD или просто графические элементы. В случае их интерпретации как блоков AutoCAD специфические для EPLAN свойства объектов переносятся как свойства блоков.

Для импорта и экспорта DXF/DWG в EPLAN имеются стандартные настройки. В настройках есть возможность создавать схемы с собственными настройками.

При экспорте DXF/DWG отдельных страниц или целого проекта для каждой страницы создается файл DXF или DWG. Рисунки, вставленные в страницы проекта, в файлах DXF / DWG не загружаются, а отображаются в виде ссылок. Чтобы продемонстрировать изображения, содержащиеся в экспортированных файлах, другому пользователю, необходимо предоставить соответствующие файлы изображений. Экспорт включенных в проект документов из внешних приложений невозможен.

Импорт в формате DXF/DWG возможен только для двухмерных чертежей. При импорте файлов DXF / DWG на уровне проекта для каждого выбранного файла создается страница типа "Графика". При этом можно импортировать также изображения, ссылки на которых содержатся в файлах DXF/DWG (при условии, что EPLAN поддерживает тип файла изображения, а у вас есть файлы самих изображений).

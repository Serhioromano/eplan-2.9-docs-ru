# Вставить контакты штекеров и определения штекеров

Контакты и определения штекеров можно определить непосредственно при вставке соответствующего символа в схему соединений. В навигаторе штекеров или навигаторе устройств для контактов и определения штекеров можно выполнить предварительную настройку отдельных свойств. Созданные таким образом функции вы можете в дальнейшем разместить в схеме соединений. Определения штекеров должны быть размещены, если их требуется скопировать.

!!! tip "Совет:"

    Если при проектировании установлены требуемые для использования изделия штекера, то штекеры можно вставлять непосредственно в качестве устройства (меню: Вставить > Устройство). Таким образом можно удобно размещать определение штекера и его соответствующие контакты друг за другом. В качестве помощи для размещения отображается определение функции соответствующего контакта штекера. Клавиша [N] позволяет "пролистать" функции.

### Вставить контакты штекеров

1. Выберите пункт меню Вставить > Символ.
2. В диалоговом окне Выбор символа выберите требуемый контакт штекера.
3. Разместите контакт штекера на странице схемы соединений.
4. В диалоговом окне Свойства ++...++ на вкладке Контакт штекера укажите данные устройства контакта штекера, например видимое ОУ.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Если вы указываете уже существующее ОУ, то контакт штекера будет назначен соответствующему штекеру.

!!! info "Для сведения:"

    Если вы указали несуществующее ОУ, EPLAN сгенерирует новый штекер. Однако такой штекер не будет иметь определения.

!!! tip "Совет:"

    При вставке контакта штекера рядом с символом на курсоре появится буквенное обозначение и знак вопроса (например, "-X?"), если в направлении поиска для копирования ОУ нет никакого другого контакта штекера. Если разместить контакт штекера, то будет сгенерирован новый штекер.
    Если в направлении поиска для копирования ОУ имеется какой-либо контакт штекера, то буквенное обозначение рядом с символом будет скрыто. При размещении там контакта штекера он скопирует ОУ из имеющегося контакта штекера и таким образом будет присвоен имеющемуся штекеру.

    Если в момент вставки удерживать клавишу ++Shift++, соотношение окажется обратным: Если в направлении поиска для копирования ОУ ***нет*** контакта штекера, контакт штекера будет присвоен последнему из использовавшихся штекеров. Если в направлении поиска имеется контакт штекера, создается новый штекер.

### Вставить определения штекера

1. Выберите пункт меню Вставить > Определение штекера.
2. Разместите определение штекера на странице схемы соединений.
3. В диалоговом окне Свойства ++...++ во вкладке Определение штекера укажите данные устройства штекера.
4. Щелкните по кнопке ++OK++.

# Диалоговое окно Экспорт графических файлов

Выделен проект или как минимум одна страница проекта. Страница > Экспортировать > Графический файл.

В этом диалоговом окне начинается собственно процесс экспорта. При экспорте отдельных страниц проекта или всего проекта в формат графического файла для каждой выбранной страницы генерируется отдельный файл.

Обзор основных элементов диалогового окна:

### Источник

В этом поле выводится на просмотр источник экспорта. В случае с целым проектом это имя проекта, для отдельных страниц отображаются полные имена страниц (включая свойства страниц Функциональное присвоение, Установка и т. д.). Здесь ***нельзя*** обрабатывать записи.

### Схема

Выберите из раскрывающегося списка схему, которая определяет настройки для экспорта графических файлов. С помощью кнопки ++...++ откройте соответствующее диалоговое окно настроек для экспорта. В нем можно задать настройки и сохранить их как схему, или [обработать имеющиеся схемы](schemeconfig_h_schematabearbeiten.md).

### Целевой каталог

Здесь вводится целевой каталог для генерируемых файлов или выбирается интерактивно с помощью кнопки ++...++. В качестве предварительной настройки здесь указан каталог, заданный в выбранной схеме в качестве целевого.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Имя файла

При экспорте ***отдельной*** страницы существует возможность ввода любого имени файла. Через [...] также можно свободно задавать имя файла или выбирать имя уже имеющегося файла. Выбранный при этом каталог переносится в поле Целевой каталог.

Если никакое имя файла не указано, то файл экспорта будет назван по имени страницы (например, =EB3+ET1/1.bmp). Чтобы предотвратить смешивание отдельных файлов различных проектов, в указанном каталоге генерируется подкаталог с именем проекта, в который затем экспортируются файлы.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

!!! note "Замечание:"

    При экспорте ***нескольких*** страниц произвольное указание имени файла невозможно, графические файлы генерируются по именам страниц.

### Черно-белый вывод

Если этот флажок установлен, все элементы на страницах проекта (за исключением графических файлов) выводятся черным цветом на белом фоне. Графические файлы всегда выводятся в оригинальных цветах.

!!! note "Замечание:"

    Если имеющиеся на экспортированных страницах "цветные" графические файлы нужно распечатать в оттенках серого цвета, следует настроить драйвер цветного принтера на черно-белый цвет или воспользоваться черно-белым принтером.

Если флажок не установлен, все элементы выводятся в цветах, которые соответствуют цветовой схеме "Цветное на белом".

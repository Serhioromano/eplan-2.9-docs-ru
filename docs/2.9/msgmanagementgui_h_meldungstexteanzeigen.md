# Отобразить тексты сообщений в графическом редакторе

Тексты сообщений из сообщений управления сообщениями можно также отобразить непосредственно для условного обозначения в графическом редакторе. При соответствующей конфигурации контрольного прогона, таким образом, ошибки при проектировании сразу видны при их возникновении (т. е. "online") на схеме соединений, и их можно непосредственно корректировать.

Условие:

* Открыта страница в графическом редакторе.
* Контрольные прогоны сконфигурированы таким образом, что проверка проводится как ***online***, так и ***в автономном режиме*** (Вид проверки "Online / автономно").

1. Выберите пункты меню Данные проекта > Управление слоями.
2. Установите для уровня EPLAN520 ("Размещение свойств.Текст сообщения") флажок Видимый.
3. Щелкните по кнопке ++OK++.

4. Выделите в графическом редакторе нужное условное обозначение.
5. Выберите пункты меню Обработать > Свойства.

6. Выберите в диалоговом окне свойств вкладку Отображение.
7. Щелкните по ![](images/all_new_as.png){: .ui-icon } (Нов.) в панели инструментов сверху списка свойств.
8. Выберите в диалоговом окне Выбор свойств свойство "Текст сообщения".
9. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В диалоговом окне Свойства в раскрывающемся списке Порядок свойств сейчас отображается запись "Определено пользователем".

!!! info "Для сведения:"

    В список свойств отображения добавляется новое свойство "Текст сообщения".

10. Измените настройки отображения этих новых свойств в соответствии с вашими представлениями.
11. Нажмите ![](images/all_save_as.png){: .ui-icon } (Сохранить) на панели инструментов над раскрывающимся списком Порядок свойств, если это определенное пользователем расположение нужно также использовать для других условных обозначений с аналогичными вариантами символов.
12. Укажите в диалоговом окне Сохранить порядок свойств наглядное название для порядка свойств и установите флажок Использовать как стандарт.
13. Подтвердите ввод.

!!! info "Для сведения:"

    При неверном вводе условного обозначения или неверной операции текст сообщения отображается online в графическом редакторе.

!!! note "Замечание:"

    Если Вы откорректировали ошибку, соответствующий текст сообщения при условном обозначении ***не*** удаляется автоматически, а должен удаляться аналогично обработанному сообщению в диалоге управления сообщениями при контрольном прогоне (т. е. "в автономном режиме").

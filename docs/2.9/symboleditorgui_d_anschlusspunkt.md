# Диалоговое окно Вывод устройства

Вы открыли проект.

* Сервисные программы > Основные данные > Символ > Открыть. Вы открыли библиотеку символов, выбрали обрабатываемый символ и в редакторе символов добавили к нему вывод устройства: Вставить > Вывод устройства "Направление. Или Вы сделали двойной щелчок по уже существующему выводу устройства.
* Сервисные программы > Основные данные > Символ > Создать. Вы выбрали создаваемый вариант символа, задали свойства Номер символа и Имя символа и добавили вывод устройства в редакторе символов: Вставить > Вывод устройства "Направление".

В этом диалоговом окне задайте или обработайте свойства вывода устройства.

Обзор основных элементов диалогового окна:

### Обозначение вывода устройства символа / Описание вывода устройства символа

Выберите из этого списка одну или несколько записей для размещения свойств выводов устройства (возможны многократный выбор и групповая обработка):

* Обозначение / описание вывода устройства не повернуто
* Обозначение / описание вывода устройства повернуто (актуально для новых DIN).

!!! note "Замечание:"

    Отображение и переключение выполняются через пункты меню Вид > Неповернутые обозначения выводов устройства / Вид > Повернутые обозначения выводов устройства.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

### Свойство/Присвоение

Если выделено какое-либо свойство, его свойства отображения можно обработать в таблице в правой части диалогового окна. Здесь доступны уровни иерархии Формат, Рамка текста, Блок выравнивания, Значение / единица измерения и Позиция, открыть которые можно, щелкнув по знаку ![](images/all_plussign_as.png){: .ui-icon }. Для свойств отображения доступны возможности настройки, описанные в следующем разделе:

* [Свойства отображения: Размещенное свойство](devicetaggui_r_anzeigeeigenschaften.md)

!!! tip "Совет:"

    Через пункты меню Вид > Свойства как числа могут отображаться тексты свойств (обозначение свойств, функциональный текст и т. п.), чтобы сделать представление более обзорным. Утвержденное свойство выводится в отчет в виде файла PDF также при распечатке.

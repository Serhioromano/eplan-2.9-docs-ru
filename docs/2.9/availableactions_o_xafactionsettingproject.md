# Операция: XAfActionSettingProject

**Параметр** |  **Описание**
---|---
Project  |  Имя целевого проекта (необязательно). Если проект не указан, используется текущий выбранный проект.
set |  Имя настройки проекта, которая подлежит настройке.
index |  Индекс настройки проекта (не обязательно). Если индекс не указан, используется 0.
value |  Новое значение настройки проекта.

!!! example "Пример:"

    XAfActionSettingProject
    /set:MacrosGui.InsertMacrobox
    /value:1В этом примере включается специфическая для проекта настройка для графической обработки Вставить также рамки макросов.

# Диалоговое окно Настройки: Add-ons

Параметры > Настройки > Фирма > Управление > Add-ons

В этом диалоговом окне задайте путь к каталогу для автоматической регистрации Add-ons.

Обзор основных элементов диалогового окна:

### Путь файла для автоматической регистрации Add-ons

Задайте в этом поле путь к каталогу на сервере, в котором находится файл install.xml для автоматической регистрации Add-ons. Стандартным способом сохраните Add-ons и соответствующий файл install.xml в центральном каталоге на сервере или попросите администратора сделать это. При запуске EPLAN осуществляется поиск на каталоге этого сервера. Add-ons и более поздние версии Add-ons регистрируются автоматически.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

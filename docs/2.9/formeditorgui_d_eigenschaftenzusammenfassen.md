# Диалоговое окно Свойства для объединения

Вы открыли проект. Сервисные программы > Основные данные > Форма > Открыть > [Открыть]. Вы открыли диалоговое окно Свойства формы — <Имя формы> (Всплывающее меню > Свойства на форме в диалоговом окне навигатора Страницы — <Имя проекта>) и в поле Значение свойства Объединять в одну строку по щелкнули кнопку ++...++.

В данном диалоговом окне укажите, для каких свойств в строке должны быть объединены идентичные данные. Например, можно объединить содержание данных на тип страницы. При этом по умолчанию в качестве разделителя используется точка с запятой, а для объединенных значений символьная строка '...'; через свойства Разделитель нескольких значений и Разделитель комплексных значений можно задать другие разделители.

!!! example "Пример:"

    Для форм типа 'Содержание' выбрано свойство для объединения типа страниц. При этом выводится, например, следующий текст в качестве свойства Имя страницы (полное):=EB3+EBS/1;=EB3+ET1/1...=EB3+ET1/3;=EB3+ET1/5...=EB3+ET1/8; =EB3+ET2/1...=EB3+ET2/3;=EB3+ET3/1...=EB3+ET3/3;=EB3+ET4/1...=EB3+ET4/3

Если вы установили свойство Создать заголовки для первых х свойств сортировки, установленные свойства сортировки при объединении автоматически обрабатываются таким образом, что затем объединяются только под одним заголовком.

!!! note "Замечание:"

    Если для спецификации изделий была установлена настройка Объединить изделия устройства (например, через меню Параметры > Настройки > Проекты > "Имя проекта" > Отчеты > Изделие), и объединение выполнялось по свойству Номер изделия, то изделия все равно будут выводиться по отдельности. Обратите внимание, что при этом определяется объединение по свойству формы Объединять в одну строку по (Ид. 13111) и свойство может быть установлено произвольно, т. е. не ограничивается только номером изделия.

Объединяются также многоязычные тексты, если они ***полностью идентичны***. При этом числа в многоязычных текстах не учитываются.

Обзор основных элементов диалогового окна:

### Список

Список отображает свойства, выбранные для объединения.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

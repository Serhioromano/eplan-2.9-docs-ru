# Определить структуру проекта

Структура проекта состоит из структуры страниц и устройств. Структура устройства состоит из дополнительных отдельных структур, например, для параметров "Устройства общего назначения", "Клеммники", "Кабели", "Черные ящики" и т. д.

Каждая структура задается отдельно при помощи схем идентификаторов. В этих схемах предусмотрено использование разных блоков идентификаторов, которые окружаются соответствующими структурными идентификаторами. В EPLAN содержатся предварительно определенные схемы обозначения. Кроме того, можно создавать определенные пользователем схемы идентификаторов для собственных структур проекта.

Структура страниц и устройств при создании проекта автоматически задается посредством выбранного шаблона проекта или базового проекта. В диалоговом окне свойств проекта можно также впоследствии изменить структуру страниц и устройств. При этом повторно изменять структуру страниц в проекте со страницами впоследствии можно только при определенных [условиях](projectstructure_k_hinterg.md).

Условие:

Вы создали проект и выделили его в навигаторе страниц.

1. Выберите пункты меню Проект > Свойства и переместите вкладку Структура на передний план.
2. Выберите из раскрывающегося списка Страницы одну из доступных схем идентификаторов или нажмите кнопку ++...++, чтобы через диалоговое окно Структура страницы установить определенную пользователем структуру страницы.
3. Выберите из раскрывающегося списка для типа оборудования, например, Устройства общего назначения, одну из доступных схем идентификаторов или нажмите кнопку ++...++, чтобы через диалоговое окно Структура оборудования установить определенную пользователем структуру оборудования.
4. Если вы хотите использовать подидентификаторы, перейдите при помощи кнопки ++Другие++ в диалоговое окно Расширенные структуры проекта и выберите там Разделитель.
5. Щелкните по кнопке ++OK++.

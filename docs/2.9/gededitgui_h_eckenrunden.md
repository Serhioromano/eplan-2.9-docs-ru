# Закругление/скашивание углов (фаски)

В EPLAN закруглить/скосить (фаски) можно для следующих элементов:

* Угол, образованный двумя лучами, выходящими из одной точки
* Угол, образованный кривой и ломаной линиями
* Угол, образованный двумя ломаными
* Углы ломаной (незамкнутой и замкнутой)
* Прямоугольники.

Условие:

Вы открыли страницу, форму, рамку или символ.

### Закругленные углы

С помощью функции Закругленные углы можно закруглить закрытые углы менее 180°. Углы между открытыми линиями закруглить невозможно, т. е. с помощью закругления линии нельзя соединить.

1. Выберите следующие пункты меню: Обработать > Графика > Закругленные углы
2. Щелкните по углу, который необходимо закруглить.
3. Определите радиус закругления, передвинув мышь, и, как только будет достигнуто требуемое значение, щелкните левой кнопкой мыши.

!!! info "Для сведения:"

    Угол будет соответственно закруглен.

4. Щелкните по следующему углу, который должен быть закруглен с тем же радиусом.

!!! info "Для сведения:"

    Угол закругляется автоматически с ранее определенным радиусом закругления.

5. Завершите операцию, выбрав пункт всплывающего меню Прервать операцию или нажав клавишу ++Esc++.

!!! example "Пример:"

    Угол прямоугольника закруглен с радиусом закругления 16 мм.

### Скосить углы (фаски)

С помощью функции Фаски углы различных элементов можно скосить симметрично. Эта функция применяется для закрытых углов менее 180°. При этом исходные линии укорачиваются симметрично на одну величину (так называемую ширину фаски), которая отмеряется от вершины угла.

1. Выберите следующие пункты меню: Обработать > Графика > Фаски
2. Щелкните по углу, который необходимо скосить.
3. Определите ширину фаски, передвинув мышь, и, как только будет достигнуто требуемое значение, щелкните левой кнопкой мыши.

!!! info "Для сведения:"

    Угол будет соответственно скошен.

4. Щелкните по следующему углу, который должен быть скошен с той же шириной фаски.

!!! info "Для сведения:"

    Угол закругляется автоматически с ранее определенной шириной фаски.

5. Завершите операцию, выбрав пункт всплывающего меню Прервать операцию или нажав клавишу ++Esc++.

!!! example "Пример:"

    Угол прямоугольника скошен с шириной фаски 16 мм.

!!! note "Замечание:"

    Кроме того, у сгруппированных элементов можно закруглять или скашивать углы отдельных элементов.

!!! tip "Совет:"

    При закруглении и скашивании можно также использовать область ввода данных и там задать радиус закругления или ширину фаски.

# Групповая обработка

В ходе обработки проекта EPLAN часто требуется изменить свойства нескольких объектов за один рабочий шаг. Например, Вы копируете имеющийся проект и, изменяя обозначения устройств, адаптируете его к новым значениям и задачам. Или же требуется переименовать определенное число страниц, заново пронумеровать их или синхронизировать тексты с различным содержимым. При этом существует опасность, что некоторые объекты будут пропущены и останутся не измененными .

EPLAN позволяет одновременно изменять свойства нескольких объектов. Этот вид обработки называется "групповой обработкой".

* С помощью этой функции можно обрабатывать свойства нескольких открытых проектов за один рабочий шаг.
* За один раз можно изменить свойства выбранных страницы текущего открытого проекта.
* Кроме того, групповая обработка позволяет быстро синхронизировать свойства условных обозначений на текущей странице схему соединений.
* С помощью так называемых навигаторов, например, навигатора устройств, можно одновременно обрабатывать свойства функций или данные проекта для всего проекта.

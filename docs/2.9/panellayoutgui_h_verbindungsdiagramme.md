# Генерировать диаграммы соединений

На диаграмме соединений устройства размещены в той же последовательности, что и на монтажной плате, при этом реальные координаты и размеры не имеют значения. Диаграммы соединений предназначены, скорее, для вывода сведений о подключенных кабельных соединениях и целях. Они реализуются следующим образом:

* Для каждой цели чертится линия. Если вывод устройства имеет несколько целей, то интервал до следующего вывода устройства может увеличиваться.
* Можно выводить символ для каждой функции. При этом символы могут отличаться от выводов устройств.

Условия:

* Вы открыли проект.
* Вы создали монтажные платы и вставили размещения изделий.
* Вы разместили черный ящик и присвоили ему ОУ монтажной платы. В качестве Вид представления выбран "Обзор". При этом черный ящик может размещаться непосредственно рядом с монтажной платой или на странице обзора, но ни в коем случае не на странице отчета.

1. Сервисные программы > Отчеты > Генерировать
2. Щелкните в диалоговом окне Отчеты- <Имя проекта> по ++Нов.++.
3. Выберите из раскрывающегося списка Форма вывода диалогового окна Определить отчет форму вывода "Размещение вручную".
4. В поле Выбрать тип отчета выберите один из отчетов: "Схема подключения устройства (для диаграммы соединений)", "Схема подключения клемм (для диаграммы соединений)" или "Схема подключения штекера (для диаграммы соединений)".
5. Установите флажок Индивидуальный выбор.
6. Щелкните по кнопке ++ОК++.
7. В диалоговом окне Выбрать монтажную плату выберите монтажную плату, для которой должна быть сгенерирована схема соединения.
8. При активированном флажке Скрыть уже использованные устройства устройства, для которых этот отчет уже выполнен, предлагаться не будут.
9. Щелкните по кнопке ++ОК++.
10. В диалоговом окне Выбрать вручную выберите устройство, для которого необходимо сгенерировать отчет. Возможен многократный выбор.

!!! info "Для сведения:"

    Если выбран тип отчета "Схема подключения клемм", здесь будут предложены только клеммники. Если выбрана "Схема подключения штекера", здесь будут предложены только штекерные колодки.

11. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Отчет будет создан и прикреплен к курсору.

12. Разместите отчет на соответствующих позициях в черном ящике.

# Просмотреть и обработать сообщения контрольного прогона

В диалоговом окне Управление сообщениями — <имя проекта> выводятся сообщения ***текущего выделенного*** проекта. Имя проекта указано в строке заголовка диалогового окна. Если диалоговое окно открыто, а в навигаторе страниц Вы выделяете другой проект, содержимое диалогового окна обновляется.

Разные категории сообщения обозначены соответствующими пиктограммами. Тем самым сразу же видны критичные сообщения, даже если они не отсортированы или отфильтрованы в таблице.

### Просмотр сообщений контрольного прогона

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Данные проекта > Сообщения > Управление.

2. Для сортировки сообщений дважды щелкните заголовки столбцов в диалоговом окне Управление сообщениями — <Имя проекта>. Щелкнув кнопкой мыши несколько раз, можно изменить сортировку в нисходящем порядке на сортировку в восходящем порядке.

!!! note "Замечание:"

    Нажав клавишу ++F1++ к каждому сообщению можно вызвать подробную справку с предложениями по исправлению ошибки.

### Фильтрация сообщений контрольного прогона

Условия:

* Вы открыли проект.
* Открыто диалоговое окно управления сообщениями.

1. В диалоговом окне рядом с полем Фильтр щелкните ++...++.
2. В диалоговом окне Фильтр: Сообщения в таблице указываются классы, для которых выводятся сообщения, например клеммы, штекеры, кабели, и т. д. Для этого в столбце Выбор необходимо установить соответствующий флажок.
3. С помощью флажков, распложенных под таблицей, можно настроить те области, по которым должны выводиться сообщения (Электротехника, Fluid-Техника и Общее).
4. Укажите категории (ошибка, предупреждение, замечание), которые должны выводиться во время работы. Отметьте соответствующую опцию.
5. Установите флажок Выполненные сообщения, если необходимо вывести и те сообщения, которые уже помечены как обработанные.
6. Щелкните по кнопке ++OK++.
7. Для просмотра только тех сообщений, которые соответствуют фильтру, установите в диалоговом окне управления сообщениями флажок Активно.

!!! info "Для сведения:"

    Последние настроенные критерии фильтра сохраняются после закрытия диалогового окна и активируются при его повторном открытии.

!!! note "Замечание:"

    В случае открытия проекта или смены проекта после вызова управления проектами всегда выводится текущая версия сообщений для всего проекта. Просмотр можно ограничить с помощью различных фильтров.

### Обработка сообщений контрольного прогона

Условия:

* Открыто диалоговое окно управления сообщениями.
* В диалоговом окне отображаются сообщения.

1. Выделите сообщение и выберите пункт всплывающего меню Перейти к (графика). (Или дважды щелкните сообщение.)

!!! info "Для сведения:"

    Некорректный объект выделяется в графическом редакторе, а курсор перемещается на точку вставки этого объекта.

2. Дважды щелкните по выделенному объекту в графическом редакторе.

3. В диалоговом окне Свойства измените данные, ставшие причиной ошибки, и щелкните по кнопке ++OK++.
4. Запустите новый контрольный прогон, выбрав пункты меню Данные проекта > Сообщения > Выполнить контр. прогон, чтобы проверить, устранена ли ошибка.

!!! info "Для сведения:"

    Сообщения в диалоговом окне Управление сообщениями — <имя проекта> обновляются.

!!! note "Замечание:"

    * Для обработанных сообщений можно установить флаг Выполнено и скрыть их, используя фильтр по этому свойству.
    * Чтобы убрать обработанные сообщения из диалогового окна управления сообщениями, перед запуском контрольного прогона установите флажок Проверять только выполненные сообщения.

# Диалоговое окно Настройки: Библиотеки символов

Вы открыли проект.

* Параметры > Настройки > Проекты > "имя проекта" > Управление >Библиотеки символов
* Вы открыли страницу схемы соединений. Вставить > Символ > Всплывающее меню > Настройки.
* Через меню Сервисные программы > Основные данные > ... была открыта библиотека символов или символ. Для библиотек символов необходимо еще в диалоговом окне Открыть библиотеку символов выбрать библиотеку символов и щелкнуть на ++Открыть++. Всплывающее меню > Настройки.

В этом диалоговом окне возможно добавить библиотеки символов к текущему проекту, т.е. сохранить их в проекте или снова удалить сохраненные библиотеки. Выбор символа (Вставить > Символ) показывает всегда только символы ***сохраненной*** библиотеки символов, при чем они идентифицируются в проекте через их имена (без пути).

На библиотеки символов идет ссылка следующим образом:

* Все библиотеки символов, которые определяются в меню Параметры > Настройки > Проекты > "Имя проекта" > Управление > Библиотеки символов
* Все библиотеки символов, которые открываются в меню Сервисные программы > Основные данные > Библиотека символов > Открыть или генерируются через меню Сервисные программы > Основные данные > Библиотека символов > Создать
* Библиотеки символов, содержащиеся в свойствах формы Присвоение свойства / значения графике, Присвоение определения функции графике и Присвоение индекса графике.

!!! note "Замечание:"

    Если в диалоговом окне Синхронизация основных данных <Имя проекта> (меню Сервисные программы > Основные данные > Синхронизировать текущий проект) запускается обновление библиотеки символов, происходит точная проверка отдельных символов, перед тем как библиотека символов будет заменена. Если находятся несовместимые данные, процесс прерывается с выводом соответствующего сообщения.

Обзор основных элементов диалогового окна:

### Синхронизировать

Если этот флажок установлен, то при открытии проекта соответствующая библиотека символов автоматически синхронизируется с пулом основных данных, если в диалоговом окне Настройки: Общее (проекты, управление) (в меню Параметры > Настройки > Проекты > "Имя проекта" > Управление > Общее) установлен флажок Синхронизировать основные данные проекта при открытии.

### Предшествующий знак

Задайте здесь предшествующий знак, который должен использоваться для различения библиотек символов. EPLAN автоматически предлагает первую букву библиотеки символов.

В ***представлении в виде списка*** диалогового окна Выбор символа с помощью пункта всплывающего меню Имя с предш. знаком можно задать, чтобы отображался дополнительный столбец, содержащий определенный здесь предшествующий знак для библиотеки символов и собственное имя символа, отделенное двоеточием.

!!! example "Пример:"

    если у вас есть для библиотеки символовIEC_Symbolпредшествующий знакIто, напр. замыкающий силовой контакт этой библиотеки отображается какI:SL.

### Библиотеки символов

Щелкните в этом столбце и потом на ++...++ чтобы выбрать новую библиотеку символов, которая заменит текущую (если первоначальная библиотека символов и новая выбранная совместимы), или если вы нажали на ++...++ в ***пустом*** столбце, чтобы сохранить новую дополнительную библиотеку символов.

### Стандартный вариант

Щелкните по этому столбцу, чтобы открыть раскрывающийся список и выбрать пункт. Выбранный здесь пункт будет скопирован как заданное по умолчанию значение для стандартного варианта символов соответствующей библиотеки символов. Для отдельных символов этот стандартный вариант можно изменить в диалоговом окне Свойства символа — < Имя символа>.

### (Удалить)

Щелкните на эту иконку в панели инструментов выше таблицы, чтобы удалить ***неиспользуемую*** библиотеку символов из проекта. EPLAN проверяет, используются ли символы этой библиотеки символов в проекте и вставляет в этом случае соответствующее примечание.

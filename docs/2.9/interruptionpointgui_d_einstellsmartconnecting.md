# Диалоговое окно Настройки: Интеллектуальное соединение

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Интеллектуальное соединение. Выберите вкладку Электротехника, Fluid-техника или Технология производственных процессов.

Это диалоговое окно служит для настройки точек разрыва, которые автоматически вставляются при вырезании или вставке элементов схемы соединений при включении опции Интеллектуальное соединение.

Доступны следующие элементы диалогового окна:

Диалоговое окно содержит такие вкладки:

* Электротехника
* Fluid-Техника
* Технол. произв. процессов.

По умолчанию для различных разделов: "Электротехника", "Fluid-техника" и "Технология производственных процессов" настроены разные символы точек разрыва. Благодаря этому в зависимости от типа страницы, на которой вставляются вырезанные элементы схемы соединений, используются разные символы точек разрыва.

Нак каждой из вкладок доступны следующие поля:

### Символ точек разрыва

Выберите из этого раскрывающегося списка схему для символа точек разрыва. EPLAN предлагает для этого различные схемы с разными точками разрыва. С помощью кнопки ++...++ откройте диалоговое окно [Настройки: Точки разрыва для интеллектуального соединения](interruptionpointgui_d_einstellsmartabbruchstellen.md) и получите возможность генерировать или обрабатывать схему точек разрыва.

* * *

Групповое поле ОУ точек разрыва

В этом групповом поле определите настройки для присвоения видимого ОУ точек разрыва, автоматически генерируемых при выполнении интеллектуального соединения.

!!! note "Замечание:"

    * ОУ автоматически генерируемых точек разрыва отображается с предшествующим знаком, если для точек разрыва подключена настройка Установить предш. знак (Путь: Параметры > Настройки > Проекты > "Имя проекта" > Устройство > ОУ).
    * Настройки в групповом поле ОУ точек разрыва используются также для функции Определить новое ОУ. Эта функция находится в диалоговом окне "Свойства" для точек разрыва во всплывающем меню поля Видимое ОУ.

### Мин. число разрядов

Значение в этом поле указывает на минимальное используемое число разрядов. Более длинные значения выводятся полностью. Разрешено использовать только положительные числовые значения. При вводе нуля число разрядов является переменным.

### Заполнитель

Здесь можно указать символ заполнителя. Значение заполняется с помощью этого символа, если минимальное число разрядов больше, чем ширина значения. При переменном числе разрядов (Запись "0" в поле Мин. число разрядов) ввод заполнителя невозможен.

### Использовать алфавитно-числов. счетчик

Если установлен этот флажок, применяется алфавитно-числовой счетчик. Если флажок снят, применяется числовой счетчик.

### Алфавитно-числовой счетчик

При использовании алфавитно-числового счетчика здесь необходимо определить символы, которые будут использоваться для счетчика. Разрешается использовать только буквы и цифры.

Алфавитно-числовые счетчики сортируются по символам слева направа (как в столбцах в Excel). Если, например, Вы предварительно задали символы "AKS", то имеет место следующий счет: A, K, S, AA, AK, AS, KA, ..., AAA, AAK, AAS, AKA ..., AAAA, AAAK, ...

# Диалоговое окно Порядок свойств

Вы открыли проект. Вы выделили обзор модели. Обработать > Свойства. Вы щелкнули по ++...++ в поле Маркировка функционального элемента диалогового окна Обзор модели. Щелкните в столбце Порядок свойств по какой-нибудь ячейке, а затем нажмите ++...++.

В этом диалоговом окне указываются настройки отображения для соответствующих свойств функционального элемента, размещенного в пространстве листа. При этом размещенные по отдельности свойства могут быть присоединены друг к другу и образовывать, таким образом, общий блок; это позволяет Вам перемещать все (требуемые) тексты одновременно и предотвращает перекрытие многострочных текстов другими текстами.

Обзор основных элементов диалогового окна:

### Порядок свойств

В раскрывающемся списке предлагаются на выбор различные предварительно определенные конфигурации порядка.

Выберите из раскрывающегося списка нужный порядок свойств; в этом случае свойства, выбранные в данной конфигурации для отображения, будут выведены в поле под раскрывающимся списком. Настройки порядков свойств влияют на размещение и отображение свойств.

если выделено какое-либо свойство, его свойства отображения можно обработать в правой части диалогового окна.

### Список размещенных свойств

Этот список отображает уже размещенные свойства функционального элемента. если выделено какое-либо свойство, его свойства отображения можно обработать в правой части диалогового окна.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Добавить |  Открывает диалоговое окно Выбор свойств, в котором можно выбрать другие свойства и скопировать их в список. (Вставленные таким образом свойства добавляются в конце списка).
Удалить |  Выделенные свойства убираются из списка. При этом значения свойств не удаляются. Свойства, которым присвоены значения, в автоматическом режиме снова отображаются в таблице при следующем открытии диалогового окна.
Переместить вверх / переместить вниз |  Перемещает выделенное свойство, если это возможно, вверх или вниз. Многократный выбор ***невозможен***..
Присоединить / Отсоединить |  Присоединяет или отсоединяет выделенное свойство, если это возможно. Многократный выбор ***невозможен***.

### Свойство/Присвоение

Если выделено какое-либо свойство, его свойства отображения можно обработать в таблице в правой части диалогового окна. Здесь доступны уровни иерархии Формат, Рамка текста, Блок выравнивания, Значение / единица измерения и Позиция, открыть которые можно, щелкнув по знаку ![](images/all_plussign_as.png){: .ui-icon }.

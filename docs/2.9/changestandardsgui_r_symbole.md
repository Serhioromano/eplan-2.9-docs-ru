# Вкладка Символы (Изменение стандарта)

Вы открыли проект. При нескольких открытых проектах: Вы выделили проект в Навигаторе страниц. Сервисные программы > Замена стандарта. Вы определили на вкладке Проект все настройки. [Далее]. Вы определили на вкладке Поворот / инверсия все настройки. [Далее]. Вы отметили опцию на вкладке Основные данные в строке библиотеки символов в столбце Присвоить отдельно. [Далее].

На этой вкладке можно выбрать схему присвоения для присвоения символов библиотеки исходных символов символам библиотеки целевых символов.

Доступны следующие элементы диалогового окна:

### Присвоить символы

Строка содержит присвоение символов исходной библиотеки символов символам целевой библиотеки символов. Отображаются только те пары присвоения библиотеки символов, для которых на вкладке Основные данные установлен флажок Присвоить отдельно.

* Библиотека символов, источник: Показывает имя файла библиотеки символов исходного проекта без расширения.
* Библиотека символов, цель: Показывает имя файла библиотеки символов целевого проекта без расширения.
* Схема: В этом столбце можно выбрать схему. В схеме определено, какие символы будут присваиваться друг другу. Создать или изменить схему можно посредством [Схема].
* Статус: Если в столбце Схема выбрана схема, будет проведена проверка присвоения символов. В данном столбце отображается статус проверки. Статус "OK", когда определения функций символов соответствуют друг другу, а число выводов устройств исходных символов меньше или равно числу выводов устройства целевых символов.

### [Схема]

Открывает диалоговое окно Настройки: Присвоение символа. Позволяет отдельно присваивать символы библиотеки исходных символов символам библиотеки целевых символов.

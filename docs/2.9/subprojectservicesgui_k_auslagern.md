# Выгрузка частичных проектов

Для выгрузки частичных проектов выделите в диалоговом окне Частичные проекты нужные, не накладывающиеся друг на друга области обработки и нажмите кнопку ++Выгрузить++.

При этом произойдет следующее:

* В диалоговых окнах для областей обработки частичный проект записывается как пользователь область обработки (его можно узнать по номеру частичного проекта). Область обработки блокируется для других пользователей, и установка флажка в столбце Моя область обработки также становится невозможна.
* При генерировании частичного проекта используется функция Копировать проект с параметром Только данные заголовка. Это обеспечивает "наследование" всех свойств проекта, настроек и основных данных на частичный проект. Кроме того, в частичный проект копируются все страницы, пространства листа и сегменты предварительного планирования соответствующей области обработки.
Чтобы сохранить присвоение к оригинальному проекту, в частичном проекте сохраняется имя оригинального проекта в свойстве проекта Частичный проект от (ид. 25100). Дополнительно в свойствах проекта сохраняется номер частичного проекта.
* Все функции и точки разрыва в частичном проекте, имеющие перекрестную ссылку на объект вне этого частичного проекта, автоматически получают свойство Перекрестная ссылка (внешняя) (ид. 25102). Посредством этого свойства сохраняются перекрестные ссылки на обратные эквиваленты, находящиеся в оригинальном проекте. Кроме того, все объекты с перекрестной ссылкой на оригинальный проект получают защиту от изменений через свойство Защита от изменений (ид. 3014).
* В оригинальном проекте соответствующие страницы, пространства листа, сегменты предварительного планирования и структурные идентификаторы блокируются для изменений. Для эти объектов автоматически активируется свойство Защита от изменений (ид. 3014). Номер выгруженного частичного проекта отображается на страницах, пространствах листа и сегментах предварительного планирования оригинального проекта в свойстве Номер частичного проекта (ид. 25101).
* Сгенерированные частичные проекты отображаются в навигаторе страниц. Они содержат имя оригинального проекта, к которому добавляется трехзначный номер в квадратных скобках. При этом речь идет о номере частичного проекта (например ESS_Sample_Project#003).

После выгрузки частичный проект можно или обрабатывать непосредственно, или перенаправить внешним обработчикам посредством резервирования данных.

# Диалоговое окно Выбор изделия

Вы открыли проект.

* Вы выделили главную функцию в графическом редакторе или навигаторе. Обработать > Свойства. Выберите вкладку Изделие и щелкните в поле Номер изделия по кнопке ++...++.
* Данные проекта > Устройство > Навигатор. Выделите одну запись. Всплывающее меню > Новое устройство.
* Данные проекта > Устройства / Изделие > Навигатор спецификаций. Выделите одну запись. Всплывающее меню > Добавить изделие и Всплывающее меню > Заменить изделие.
* Данные проекта > Устройство / Изделие > Список устройств. Выделите одну запись. Всплывающее меню > Добавить изделие.

В этом диалоговом окне выбирается изделие на основе его экономических данных. При многократном выборе изделий возможно одновременное присвоение нескольких изделий. При этом функции / условному обозначению присваиваются данные первого изделия, содержащегося в выборе.

Обзор основных элементов диалогового окна:

### Фильтр по полю

С помощью фильтра по полю можно задать критерии фильтра для определенных полей в записи данных.

В этом раскрывающемся списке отображаются все доступные фильтры. Выбранный фильтр активируется автоматически. Запись "- Не активировано -" отключает фильтр и приводит к тому, что данные отображаются в неотфильтрованном виде. С помощью кнопки ++...++ откройте диалоговое окно [Фильтр](modaldialogsdb_d_filternnach.md). Здесь можно создать, обработать, удалить, скопировать, экспортировать, импортировать фильтр и управлять им.

Во всплывающем меню раскрывающегося списка Фильтр содержатся следующие записи:

* Выключить: Этот пункт меню доступен, если фильтр установлен: Сбрасывает настройку фильтра до записи "- Не активировано -".
* Активировать <Имя фильтра>: Этот пункт меню доступен, если для настройки фильтра установлено значение "- Не активировано -": Повторно активирует последний активный фильтр.

Таким образом можно быстро переключаться между неотфильтрованным и отфильтрованным в соответствии с требованиями пользователя представлениями.

### Полнотекстовый фильтр

В данном поле введите текст для поиска и щелкните по ![](images/all_search_as.png){: .ui-icon } (Поиск). Далее выполняется поиск текста во всех полях записи данных, при этом в ***представлении структуры дерева*** также учитываются главные узлы "Список принадлежностей", "Размещение принадлежностей", "Схема сверления", "Схема соединений".
При выполнении поиска в первый раз требуется подтвердить генерирование индекса поиска нажатием кнопки ++ОК++; или же индекс поиска можно обновить в любое время в пункте меню ++Дополнительно++ > Обновить индекс поиска.
Чтобы снова удалить искомое понятие и вернуться к отображению всех изделий, щелкните по кнопке ![](images/all_delete_as.png){: .ui-icon } (Удалить).

Правила ***синтаксиса искомых понятий*** находятся [здесь](partsmanagementgui_d_artverw.md).

### Автом. фильтр

При помощи ++...++ откройте диалоговое окно [Автом. фильтр](partsmanagementgui_d_automatikfilter.md) и там можно установить определенные свойства выделенного условного обозначения в качестве критериев фильтрации.

Для использования этих настроек фильтра установите флажок Активн..

В структуре дерева или в списке показываются подходящие изделия из базы данных изделий согласно настройкам фильтра. Если при этом активировано диалоговое окно Предварительный просмотр графики и изделию присвоено фото продукта, то оно отображается.

В зависимости от выделения в структуре дерева или в списке с правой стороны в диалоговом окне показываются соответствующие вкладки из базы данных изделий, в которых видны свойства выделенных элементов.

### Дерево

Под этими полями фильтра можно выводить на экран типы изделий, а также клиентов и производителей / поставщиков в иерархической структуре дерева, причем в качестве последнего уровня иерархии записываются отдельные номера изделия или при наличии варианты изделия.

Отображение для каждого изделия можно свободно конфигурировать через [Дополнительно] > Настройки.

### Список

Кроме дерева, данные могут отображаться в виде списка, причем объем отображаемых данных настраивается через всплывающее меню > Конфигурировать представление. При этом в списке представлены ***все*** изделия, но в нем нет производителей, поставщиков и клиентов, кроме того, выводятся только те свойства, которые имеют ***все*** изделия (т. е., например, номер изделия, группа продуктов и т.д.).

### Комбинация

Эта вкладка содержит комбинированное представление дерева и списка изделий: В верхней части вкладки на представлении структуры дерева изделий можно сделать предварительный выбор. Если вы выделите на дереве узел, например группу продуктов "Кабели", то в расположенном ниже списке вы увидите соответствующие изделия.

В зависимости от выбора в дереве, списке или комбинированном представлении справа в диалоговом окне отображаются соответствующие вкладки. Таким образом, при выборе клиента выводится подробный вид данных для этого клиента, а при выборе кабелей отображаются вкладки, специфические для кабелей.

### Информация

На этой вкладке отображается информация, например имя и тип, используемой в данный момент базы данных изделий, количества содержащихся изделий, количества производителей / поставщиков, схем сверления, списков принадлежностей и т. д. Содержимое вкладки автоматически обновляется при открытии, при этом в любой момент также можно выполнить обновление вручную с помощью клавиши ++F5++.

### Всплывающее меню

Команда |  Значение
---|---
Список с предварительным выбором (только дерево) |  Уменьшает число отображаемых элементов, чтобы ускорить ориентирование в представлении в виде списка. Если этот параметр активирован, представление в виде списка вызывается с автоматическим фильтром (предварительный выбор), причем этот фильтр содержит только что выбранные элементы.
Выделить все (только список) |  Выделяет все записи в списке.
Подобрать ширину столбца (только список) |  Все столбцы так подогнаны к таблице по ширине, что и заголовок, и содержимое столбцов полностью читаемы.
Конфигурировать представление |  Открывает диалоговое окно Конфигурировать представление, в котором можно задать, какие свойства должны отображаться в представлении в виде списка и в представлении структуры дерева.
Открыть гиперссылку |  Дает возможность открыть внешние документы или графический файл, если они были присвоены изделию в базе данных изделий.

### [Дополнительно]

В меню ++Дополнительно++ имеется следующий пункт:

Пункт меню |  Значение
---|---
Настройки |  Открывает диалоговое окно Настройки: Изделие (пользователь), в котором можно, например, конфигурировать представление структуры дерева без прерывания производственного процесса.

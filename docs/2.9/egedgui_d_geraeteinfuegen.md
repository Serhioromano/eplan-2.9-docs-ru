# Диалоговое окно Вставить устройство

Вы открыли проект. Выберите пункты меню Вставить > Устройство или пункт всплывающего меню Вставить устройство из навигатора. Выберите подходящее изделие и щелкните по кнопке ++ОК++.

В этом диалоговом окне указываются параметры вставки для устройства.

Обзор основных элементов диалогового окна:

### Макрос изделия

Если изделию в базе данных изделий присвоен один макрос, его можно вставить, воспользовавшись этой опцией.
В противном случае эта опция будет недоступна.

### Отдельные функции

Если изделию в базе данных изделий присвоены шаблоны функции, их можно вставить, воспользовавшись этой опцией.
Нажмите клавишу ++N++, чтобы "перелистать" функции и выбрать нужную. Или используйте во время размещения вспомогательной функции клавишу ++Пробел++, чтобы прервать размещение вспомогательной функции и заново начать размещение главной функции.

### Выбор макроса

Выберите эту опцию, если после подтверждения и нажатия [OK] необходимо открыть диалоговое окно Выбрать макрос, чтобы выбрать макрос для устройства (другой). Сохраненные в изделии макросы и символы при этом не учитываются.

### Выбор символа

Выберите эту опцию, если после подтверждения и нажатия [OK] необходимо открыть диалоговое окно Выбор символа, чтобы выбрать символ для ***устройства*** (другой). Выбор символа при этом отфильтрован в соответствии с заданным видом представления и всеми символами с видом представления "Не определено".

### Вид представления

Выбрав одну из опций Макрос изделия, Отдельные функции или Выбор символа, можно воспользоваться этим раскрывающимся списком, чтобы присвоить устройству вид представления (другой).

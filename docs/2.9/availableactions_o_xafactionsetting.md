# Операция: XAfActionSetting

**Параметр** |  **Описание**
---|---
set |  Имя настройки, которую необходимо установить.
index |  Индекс настройки (не обязательно). Если индекс не указан, используется 0.
value |  Новое значение настройки.

!!! example "Пример:"

    XAfActionSetting
    /set:USER.SYSTEM.GUI.SHOW_PROPERTY_NR
    /value:1В этом примере устанавливается специфическая для пользователя настройка интерфейса Отображать идентифицирующие номера.

# Новые комбинации клавиш

В интерфейсе пользователя платформы EPLAN теперь доступны по умолчанию следующие комбинации клавиш:

Команда |  Комбинация клавиш
---|---
Генерировать отчеты |  [Ctrl] \+ [E]
Генерировать макросы автоматически из проекта макросов |  [Ctrl] \+ [K]
Использование имеющихся выводов устройства ПЛК поблочно |  [Ctrl] \+ [L]
Адресация ПЛК |  [Ctrl] \+ [O]
Функция поиска: Синхронизировать выбор |  [Ctrl] \+ [S]
Обновить отчеты |  [Ctrl] \+ [<]
Выполнить контр. прогон |  [Ctrl] \+ [Shift] \+ [P]
Генерация отчета по проекту |  [Ctrl] \+ [Shift] \+ [<]
Обработать раб. область |  [Ctrl] \+ [Alt] \+ [A]

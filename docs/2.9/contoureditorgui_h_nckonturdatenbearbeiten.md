# Генерирование и обработка записей данных контура ЧУ

Условия:

* Вы открыли проект.

* В редакторе контуров открыт контур типа 'Генерирование записи данных контура ЧУ' (например, через пункты меню Сервисные программы > Основные данные > Контур (данные ЧУ) > Открыть). При этом чертеж контура содержит только единичный непересекающийся закрытый контур.

1. Выберите пункты меню Сервисные программы > Основные данные > Контур (данные ЧУ) > Генерировать запись данных > Rittal - Perforex BC / Secarex (или ... > Rittal - Perforex LC / Secarex или ... > ЧУ Steinhauer).
2. В диалоговом окне Запись данных контура ЧУ ++...++ задайте специфические для записи данных свойства: Производитель, Машина, Способ пуска и т. д.
3. Щелкните по кнопке ++OK++.
4. Выберите на контуре начальную точку. Для правильного подвода инструмента к контуру начальная точка должна лежать на прямой, а не в углу.

!!! info "Для сведения:"

    Траектория фрезы обозначается как графическое представление программы ЧУ на контуре. Каждая конечная точка шага программы (длина по прямой или дуге окружности) отображается через окружность. Траектория фрезы обозначается параллельными линиями, дугами или круговыми сегментами.

!!! info "Для сведения:"

    Генерируется запись данных.

!!! note "Замечание:"

    Программа не позволяет разместить начальную точку слишком близко к дугам окружностей, закруглению или вершине прямоугольника. В этом случае ввод игнорируется. В строке состояния появляется сообщение о неверной позиции и предложение определить другую начальную точку.

5. Повторите данные шаги, чтобы сгенерировать записи данных ЧУ. Обратите внимание, что на машину допускается только ***одна*** запись ЧУ.
6. Выберите пункты меню Сервисные программы > Основные данные > Контур (данные ЧУ) > Обработать запись данных ЧУ > Генерировать отверстие и ... > Генерировать технологическую перемычку, чтобы сгенерировать отверстия и технологическуюперемычку.
7. Чтобы сделать видимыми в открытом контуре сохраненные записи данных ЧУ, используйте функции Вид > Следующая запись данных ЧУ и Вид > Предыдущая запись данных ЧУ.

!!! note "Замечание:"

    Чтобы минимизировать опасность повреждения заготовок и инструментов из-за неправильного экспорта ЧУ, ***перед*** замыканием контура ЧУ следует провести проверку заданной записи данных контура ЧУ. Так, например, нужно следить за тем, чтобы диаметр фрезы не был слишком большим при работе с закруглениями и углами.

    Во многих случаях бывает надежнее и проще использовать стандартные контуры (круг, прямоугольник с фаской / закругленный, продольное отверстие, шестигранник, восьмигранник).

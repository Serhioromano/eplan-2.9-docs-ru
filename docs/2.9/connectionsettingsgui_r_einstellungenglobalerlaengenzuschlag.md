# Вкладка Общая дополнительная длина

Вы открыли проект. Параметры > Настройки > Проекты > 'Имя проекта' > Маршрутизируемые соединения > Общее. Выберите вкладку Общая дополнительная длина.

На этой вкладке определяется общая дополнительная длина маршрутизируемых соединений. Эта длина прибавляется при расчете длины к каждому соединению.

Обзор основных элементов диалогового окна:

Возможны отдельные настройки для соединений различных разделов:

* Абсолютно: Здесь можно ввести фиксированное значение длины в заданных для проекта единицах измерения (мм, дюйм). Значение может иметь два знака после запятой.
* В процентах: Дополнительная длина в процентах от длины соединения. Значение должно быть целым и лежать в диапазоне от 0 до 100.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Сбросить на стандарт |  Восстанавливает во всех полях стандартные настройки.

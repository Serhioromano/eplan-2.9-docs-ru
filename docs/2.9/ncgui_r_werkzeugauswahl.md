# Вкладка Выбор инструмента (экспорт ЧУ для всех машин)

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт ЧУ (...). Выберите вкладку Выбор инструмента.
* Сервисные программы > Данные изготовления > Механическая обработка > (...) ЧУ. Рядом с полем Машина нажмите ++...++. Выберите вкладку Выбор инструмента.

На этой вкладке настраиваются параметры для экспорта при отсутствии подходящего инструмента в машине.

Отдельно определяется образ действий при отсутствии сверла и резьбонарезного инструмента.

Кроме того, здесь указывается диаметр круглого фрагмента, при превышении которого применяется не сверление, а фрезерование.

Обзор основных элементов диалогового окна:

Групповое поле Отверстия (не с Rittal - Perforex LC)

### Использовать другое сверло (если подходящий инструмент отсутствует)

Если на машине нет сверла требуемого диаметра, можно использовать другое сверло.

Если этот флажок установлен, при отсутствии нужного инструмента используется другой инструмент. Через раскрывающийся список Для выбранного сверла укажите, следует ли взять для работы следующее меньшее или большее по диаметру сверло.

### Фрезеровать большие отверстия

Если машина может сверлить и фрезеровать, можно указать диаметр отверстия, для которого отсутствует подходящий инструмент для сверления, и следует перейти к другой технике.

Если установлен этот флажок, отверстия, диаметр которых превосходит указанную величину, будут фрезероваться. В поле Диаметр — предельное значение введите диаметр отверстия, начиная с которого следует перейти к использованию альтернативной техники.

* * *

Групповое поле Резьба (когда отсутствует подходящий инструмент)

Если для изготовления резьбовых отверстий на машине не был задан резьбонарезной инструмент, подходящий для нужного диаметра резьбы, существуют два возможных варианта возможной реакции машины.

### Сверлить только отверстия под резьбу

При этом необходимо полностью отказаться от резьбовых отверстий. Если активирована данная опция, при помощи соответствующего инструмента сверлится только отверстие под резьбу, сама резьба нарезается позднее вручную.

### Использовать другой резьбонарезной инструмент

В качестве альтернативы отказу от нарезания резьбы можно использовать другой резьбонарезной инструмент.

Если активирована данная опция, при отсутствии нужного инструмента используется другой инструмент. Через раскрывающийся список Для выбранного резьбонарезного инструмента укажите, следует ли взять для работы следующий меньший или больший по диаметру резьбонарезной инструмент.

# Внешние отчеты

При помощи функции "Экспорт / маркировка" возможно генерировать внешние отчеты. При этом данные проекта анализируются, как при экспорте / маркировке, и выводятся в файле отчетов.

!!! warning "Предупреждение:"

    Обязательным условием для создания в EPLAN файлов в формате Microsoft Office является наличие установленных на компьютере следующих разрешенных EPLAN программных продуктов.

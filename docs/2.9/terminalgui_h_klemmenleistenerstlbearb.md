# Создать и обработать клеммники

Клеммники управляются с помощью определения клеммника, а также их можно создавать в навигаторе клеммников или устройств или вставляя в схему соединений.

Если необходимо создать клеммники с предварительно заданными свойствами, нужно сначала создать определение клеммника. При этом можно определить строение клеммников.

Чтобы создать новый клеммник (без указания дополнительных свойств), просто укажите при вставке клеммы ОУ клеммника в диалоговом окне Свойства ++...++. При этом создается клеммник без предварительного определения, который не имеет определения клеммника. Такой клеммник без предварительного определения виден в отчетах, однако Вы не можете назначать ему никаких свойств (это можно сделать только через определение клеммника).

Вы можете сначала создать клеммники без предварительного определения, а позднее обработать их свойства. Однако, Вы можете также устанавливать свойства клеммников сразу при их создании. С помощью определений функций Вы можете создавать клеммник с предварительно определенными свойствами.

Во вкладке [Определение клеммника](devicetaggui_r_klemmenleisten.md) диалогового окна Свойства выполняется обработка свойств клеммника.

### Создание клеммников

1. Выберите пункты меню Данные проекта > Клеммники > Навигатор.
2. Выберите пункт всплывающего меню Генерировать определение клеммника.
3. В диалоговом окне Свойства ++...++ во вкладке Определение клеммника введите требуемые данные для нового клеммника.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Создается определение клеммника с соответствующими свойствами. если вы не ввели данные, создается клеммник без предварительного определения. Оно могут быть определены позднее.

### Создание клеммников с предварительно определенными свойствами

1. Выберите пункты меню Данные проекта > Клеммники > Навигатор.
2. Выделите ОУ и выберите пункт всплывающего меню Создать.
3. В диалоговом окне Определения функций выберите в виде дерева самый нижний уровень "Определение клеммника".
4. Щелкните по кнопке ++OK++.
5. В диалоговом окне Свойства ++...++ во вкладке Определение клеммника введите требуемые данные для нового клеммника.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Под отмеченным ОУ создается определение клеммника с соответствующими свойствами. если вы не ввели данные, создается клеммник без предварительного определения. Оно могут быть определены позднее.

!!! tip "Совет:"

    Клеммники можно копировать и размещать при помощи функции перетаскивания, перемещая внутри проекта клеммники, выделенные в навигаторе, на открытую страницу схемы соединений, открытую в графическом редакторе.

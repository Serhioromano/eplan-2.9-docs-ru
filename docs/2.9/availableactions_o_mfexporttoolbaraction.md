# Операция: MfExportToolbarAction

**Параметр** |  **Описание**
---|---
File  |  Экспортируемый XML-файл, включая путь файла.
ToolbarName  |  Имя первой экспортируемой пользовательской панели инструментов.
0  |  Имя следующей экспортируемой пользовательской панели инструментов.
1  |  Имя следующей экспортируемой пользовательской панели инструментов.

!!! note "Замечание:"

    Данная операция использует главное окно.

!!! example "Пример:"

    MfExportToolbarAction
    /File:"?"
    /ToolbarName:"?"

# Предварительное определение свойств блока

Свойства блока могут быть предварительно определены для символов и проектов.

Если функции присваивается свойство блока [n], а формат не внесен ни для самой функции, ни для символа, программа будет использовать соответствующее свойство формата, введенное в проекте для этой категории функции.

Внесите формат, который будет применяться для функции.

### Предварительное определение для символа

Для символов существует индексированное свойство Свойство блока: Формат [n]. Здесь можно задать предварительный формат, который будет использоваться в проекте при вставке символа.

Если функции присваивается свойство блока [n], а формат функции не внесен, программа по символу будет проверять, внесен ли формат с соответствующим индексом в свойство Свойство блока: Формат [n], и будет использовать этот формат.

Внесите формат функции, который будет применяться вместо формата, установленного для символа.

### Предварительное определение для проекта

Для проектов существуют индексированные форматы для свойств блока. Для каждой из различных категорий функций предусмотрено индексированное свойство, например Свойство блока: Формат (точка разрыва), Свойство блока: Формат (клемма), Свойство блока: Формат (технологический контур) и т. д.

!!! tip "Совет:"

    Сначала установите формат определенной категории функции для конкретной функции в проекте. Например, чтобы задать свойство формата Свойство блока: Формат (клемма), определите на клемме свойство Свойство блока: Формат. Для функции в диалоговом окне Формат: Свойство блока вместе со свойствами отображаются имеющиеся значения. Это упрощает поиск и выбор необходимых свойств. Затем скопируйте определенный таким образом формат в свойство формата для проекта и при необходимости удалите формат из функции.

### Предварительное определение текстов замены

В свойстве проекта Свойство блока: текст замены с помощью индекса можно выбрать до десяти таблиц с текстами замены. Эти таблицы используются для замены значений ***свойств блока***: во время редактирования свойств блока в диалоговом окне [Формат: Свойство блока](eservicesgui_d_formatblockeigenschaft.md) можно выбрать таблицу, используемую для замены текста свойства. Значения выбранного свойства сравниваются с текстами, сохраненными в таблице, и при необходимости заменяются.

В свойстве формы Форматированные свойства: текст замены с помощью индекса можно дополнительно выбрать до десяти таблиц с текстами замены. Эти таблицы используются в отчетах форм, содержащих тексты заполнителей с отформатированными свойствами, для замены данных свойств: при обработке текстов заполнителей в диалоговом окне Формат: Свойство блока можно выбрать таблицу, используемую для замены текста свойства. Значения выбранного свойства сравниваются с текстами, сохраненными в таблице, и при необходимости заменяются.
При составлении отчета форм в первую очередь используются таблицы, настроенные для свойства формы. Если выбранная таблица не определена в свойстве формы, вместо нее используется таблица с таким же именем из свойства проекта Свойство блока: текст замены.

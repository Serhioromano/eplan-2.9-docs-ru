# Поиск данных проекта

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Поиск > Поиск
2. В диалоговом окне [Поиск](searchandreplacegui_d_suchen.md) в поле Фильтр для функций при необходимости выберите фильтр из раскрывающегося списка. Поиск по функциям можно ограничить определенным видом представления или определением функции.
3. Введите искомое понятие в поле Поиск.
При поиске можно использовать [заполнители](searchandreplacegui_k_platzhalter.md) (такие, как * или ?).
Можно искать несколько взаимосвязанных понятий. Но поиск нескольких несвязанных понятий производить нельзя.
4. Определите с помощью флажка, должен ли при поиске учитываться регистр или только весь текст (то есть в точности записанный в поле ввода текст).
5. В групповом поле Поиск в определите, в каких данных проекта программа должна производить поиск искомого понятия. Установите соответствующий флажок.
6. В групповом поле Поиск на введите область поиска. Установите флажок для типов страниц, в которых следует искать искомое понятие. Для исключения определенных типов страниц из поиска, снимите соответствующие флажки.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    EPLAN начнет поиск, откроет диалоговое окно Результаты поиска — <Имя проекта> и покажет найденные записи.

### Отображение результатов поиска

1. Выберите пункты меню Поиск > Отобразить результаты.

!!! info "Для сведения:"

    Откроется диалоговое окно Результаты поиска — <Имя проекта> и отобразятся результаты поиска активного проекта.

!!! info "Для сведения:"

    Если диалоговое окно Результаты поиска — <Имя проекта> уже открыто, но не находится на переднем плане, оно будет заполнено данными активного проекта и выведено на передний план.

!!! info "Для сведения:"

    Если диалоговое окно Результаты поиска — <Имя проекта> открыто и находится на переднем плане, оно будет закрыто с помощью этой команды меню.

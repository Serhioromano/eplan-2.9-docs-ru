# Закрыть библиотеки символов

Условие:

Вы открыли проект и как минимум одну библиотеку символов.

1. Сервисные программы > Основные данные > Библиотека символов > Закрыть
2. если открыто несколько библиотек символов, то пометьте закрываемую (-ые) в диалоговом окне Выбор библиотеки символов и щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    если открыта только ***одна*** библиотека символов, то она будет закрыта без дополнительных диалогов.

!!! info "Для сведения:"

    если в символ были внесены изменения, которые могут быть потеряны при закрытии библиотеки, то следует запрос, надо ли сначала сохранить соответствующие страницы.

!!! note "Замечание:"

    если для символа будет прерван запрос на закрытие / сохранение, то закрытие библиотеки символов будет также прервано.

# Использовать предварительный просмотр соединений

Предварительный просмотр соединений является вспомогательным средством для размещения устройств из трехмерного навигатора монтажных поверхностей. Голубая линия показывает, с каким уже размещенным устройством размещаемое устройство электрически соединено в плане проекта. Предварительный просмотр соединений облегчает группировку связанных между собой устройств и помогает избежать чрезмерно длинных разводок.

Условия:

* Вы открыли проект.
* Навигатор пространства листов открыт, и одно пространство листов открыто.
* Пространство листа содержит размещения изделия.
* Трехмерный навигатор монтажных поверхностей открыт.

1. Выделите устройство в трехмерном навигаторе монтажных поверхностей.
2. Выберите пункт всплывающего меню Разместить.

!!! info "Для сведения:"

    Размещаемое изделие перемещается вместе с курсором в предварительном просмотре.

3. Выберите пункты меню Вид > Соединения > Предварительный просмотр.

!!! info "Для сведения:"

    Между размещаемым изделием и устройством, с которым оно соединено при помощи электрических проводов, отобразится голубая линия.

4. Разместите изделие на подходящем месте.

!!! example "Пример:"

    Выбор Результат Размещаемое устройство на курсоре связано с двумя размещенными устройствами Удобное место размещения с относительно короткой разводкой к соединяемым устройствам.

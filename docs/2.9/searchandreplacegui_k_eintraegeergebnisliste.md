# Записи в списках результатов

Из навигаторов или на условном обозначении в графическом редакторе с помощью пунктов меню Перейти к (перекрестная ссылка) или Перейти к (все виды представлений) меню Поиск или с помощью всплывающего меню можно вводить данные в список результатов Перейти к. Для этого диалоговое окно Результаты поиска не должно быть открытым. Таким образом в список "Перейти к" можно также вводить тексты.

Вносится следующая информация:

Объект |  Значение
---|---
Тексты |  На первом языке отображения проекта
Функции |  (Видимое) ОУ
Точки разрыва |  (Видимое) ОУ (имя)
Точки обозначения соединения |  Обозначение соединения
Точки определения потенциала |  Имя потенциала
Тексты-заполнители отчетов |  Тип страниц

Следующие объекты ***не*** вносятся в список результатов:

* Обозначения местоположения
* Уголки, тройники, мосты, перекрестные соединения, ...
* Графические элементы (линии, круги, прямоугольники, ...)
* Соединения
* Монтажные платы
* Рамки макросов
* Объекты-заполнители.

Запись в списке результатов включает в себя следующую информацию:

* Индекс (он указывает графическую последовательность в схеме соединений. Неразмещенные функции добавляются в конец списка. Индекс не отображается в диалоговом окне.)
* Вид объекта (напр., катушка)
* Имя объекта плюс первое обозначение вывода устройства (напр., -K1:A1)
* Свойство (напр., функциональный текст)
* Значение (напр., главный привод).

### Последовательность результатов поиска

Результаты сначала отображаются согласно их графической последовательности в схеме соединений. Согласно языку, на котором был найден результат, заносится строка. Щелкнув по заголовкам столбцов, можно сортировать записи по нарастанию или убыванию.

* Объекты на странице 1
* Страница 1
* Объекты на странице 2
* Страница 2
* ...
* Неразмещенные устройства
* Проект.

В строке состояния диалогового окна отображаются полное имя страницы, положение на странице, тип страницы, а также язык первой выделенной записи поиска.

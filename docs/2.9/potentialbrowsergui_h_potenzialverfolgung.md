# Выделить потенциалы, сигналы или сеть

Потенциалы и сигналы можно временно выделять цветом, что позволяет быстро просматривать их расширение. Текущий выбранный потенциал / сигнал / сеть выделяются заданным в настройках пользователя цветом.

Ниже описан процесс выделения ***потенциалов***; сигналы и сети можно подобным образом выделять при помощи пункта меню Вид > Отслеживание сигнала или Вид > Отслеживание сети.

Условие:

В настройках пользователя (Параметры > Настройки > Пользователь > Графическая обработка > 2D в раскрывающемся списке Отслеживание потенциала) определен цвет для выделения потенциала / сигнала / сети.

1. Выберите следующие пункты меню: Вид > Отслеживание потенциалов
2. Щелкните по символу соединения или по линии соединения.

!!! info "Для сведения:"

    EPLAN найдет соответствующий потенциал и полностью выделит его цветом (т.е. также и за пределами страницы).

3. Щелкните по другому соединению.

!!! info "Для сведения:"

    Выделение цветом будет снято с предыдущего потенциала, будет выделен потенциал, связанный с этим соединением.

4. Выберите во второй раз пункты меню Вид > Отслеживание потенциалов.

!!! info "Для сведения:"

    Выделение потенциала цветом снимается.

!!! note "Замечание:"

    * Потенциалы, сигналы и сети выделяются цветом тогда, когда соединения являются актуальными. Следите за тем, чтобы данные потенциала записывались на соединения только при обновлении соединений ***для всего проекта***. При смене страницы обновляются только данные соединений.
    * После завершения отслеживание потенциала, сигнала или сети посредством кнопки ++Esc++ или всплывающего меню > Прервать операцию, для повторного выделения цветом потенциала / сигнала / сети необходимо отключить и вновь подключить соединение (например, посредством Вид > Отслеживание потенциалов).

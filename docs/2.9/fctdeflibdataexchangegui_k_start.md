# Собственные обозначения выводов устройства и условные обозначения

При обработке основных данных у Вас есть возможность для каждого определения функции определять собственные обозначения выводов устройства и буквенные обозначения.

* При помощи собственных наборов вы можете учитывать различные стандарты, страны и клиентов.
* Для каждого определения функции можно сохранить десять наборов обозначений и описаний выводов устройства. Затем эти наборы можно использовать из функции.
* Определенные Вами наборы буквенных обозначений, обозначений выводов устройства и описаний выводов устройства экспортируются в XML-файл. Этот файл можно использовать для резервирования данных или для обмена данными.
* Наборы буквенных обозначений, обозначений выводов устройства и описаний выводов устройства можно импортировать из XML-файла, а затем использовать их для проектирования в EPLAN.

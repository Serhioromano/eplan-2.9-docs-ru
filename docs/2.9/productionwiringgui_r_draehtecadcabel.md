# Вкладка Провода (CadCabel)

* Параметры > Настройки > Фирма > Сборка проводов > CadCabel > Провода
* Сервисные программы > Данные изготовления > Сборка проводов > CadCabel. Щелкните по кнопке ++...++ рядом с полем Схема. Выберите вкладку Провода.

На этой вкладке устанавливаются основные данные проводов для обработки проводов на машине CadCabel.

Обзор основных элементов диалогового окна:

### Панель инструментов

Кнопка |  Значение
---|---
![](images/all_update_as.png){: .ui-icon } (Обновить) |  Заново считывает провода и изделия проводов из проекта.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет выделенные строки.

### Цвет

Цвет провода.

### Поперечное сечение

Поперечное сечение провода.

### Номер изделия

Номер изделия для изделия проводов.

### Тип провода CadCabel

В заказе CadCabel каждый провод должен соответствовать типу провода, указанному CadCabel. В этом поле таблицы назначаются соответствующие типы проводов CadCabel для проводов в проекте. CadCabel предоставляет электронную таблицу Excel, из которой можно считать соответствующие типы проводов.

### Код кабеля CadCabel

В заказе CadCabel каждый провод должен быть обозначен кодом кабеля, указанным CadCabel. В этом поле таблицы назначается соответствующий код изделия CadCabel для проводов в проекте. CadCabel предоставляет электронную таблицу Excel, из которой можно считать соответствующие коды кабелей.

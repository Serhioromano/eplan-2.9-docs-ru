#  Перекрестные ссылки

Распределение выбранной рамки страницы на строки и столбцы является основой для образования перекрестных ссылок между условными обозначениями в проекте. При этом строка соответствует горизонтальному разделу, столбец - вертикальному разделу в рамке. Порядок нумерации разделов может быть как алфавитным, так и числовым. Дополнительно возможно определить разную нумерацию для горизонтального или вертикального распределения разделов (числовое / алфавитное).

Сама перекрестная ссылка позволяет с уверенностью находить определенное условное обозначение / определенную информацию среди большого количества страниц. По этой причине она содержит как минимум искомое обозначение страницы. Дополнительно она может содержать данные о столбце для ориентации в пределах страницы и данные о строке для дальнейшей ориентации.

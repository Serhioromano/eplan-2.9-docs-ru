# Вставить сборные шины

Символ для сборных шин содержится в библиотеке символов SPECIAL.slk; у него есть четыре варианта, при этом позиция текста управляется также, как и в выводах устройств.

Условие:

Вы открыли проект. В Графическом редакторе открыта страница проекта.

1. Выберите пункты меню Вставить > Блок / вывод устройства / монтажная плата > Вывод устройства сборной шины.
2. При необходимости нажмите клавишу ++Ctrl++ и переместите мышь, чтобы просмотреть имеющиеся варианты.
3. Разместите символ на странице.
4. Установите свойства вывода устройства сборной шины в диалоговом окне Свойства ++...++.
5. Щелкните по кнопке ++OK++.
6. Выполните аналогичные действия для всех других выводов устройства сборной шины.
7. Завершите операцию, выбрав пункт всплывающего меню Прервать операцию или нажав клавишу ++Esc++.

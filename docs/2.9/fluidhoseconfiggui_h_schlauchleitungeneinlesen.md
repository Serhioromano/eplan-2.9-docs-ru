# Считать данные шлангопроводов

Прежде чем конфигурировать шлангопроводы, необходимо считать данные шлангопроводов, которые, как правило, состоят только из полных ОУ шлангопроводов, из EPLAN Fluid в конфигуратор шлангопроводов.

#### Считать данные шлангопроводов непосредственно со схемы соединений Fluid-техники

Условия:

* У вас установлен конфигуратор шлангопроводов в качестве дополнительного модуля для EPLAN Fluid.
* Вы запустили EPLAN Fluid и открыли проект.
* В этом проекте созданы одна или несколько страниц схемы соединений Fluid-техники, которые содержат шлангопроводы, вставленные с помощью макроса символа Hose.ems.
* Шлангопроводам в качестве минимальной информации присвоено ОУ.

1. Выделите в проекте шлангопроводы, которые необходимо конфигурировать.
2. Выберите следующие пункты меню: Данные проекта > Соединения > Конфигурировать шлангопроводы

!!! info "Для сведения:"

    Откроется диалоговое окно Конфигурировать шлангопроводы.

!!! info "Для сведения:"

    Все выделенные в проекте шлангопроводы автоматически отобразятся с полным ОУ в таблице данного диалогового окна и будут предложены для обработки.

#### Импортировать данные шлангопроводов для файла обмена

Условия:

* У вас установлен и запущен конфигуратор шлангопроводов в качестве дополнительного модуля для EPLAN Fluid.
* Например, вы получили из конструкторского отдела файл обмена в виде разделенного табуляторами текстового файла Юникод с неполными данными шлангопроводов.

1. В диалоговом окне Конфигурировать шлангопроводы выберите пункты меню Дополнительно > Импортировать.
2. В следующем диалоговом окне выберите место хранения и имя файла обмена и щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Импортированные данные шлангопроводов отображаются в диалоговом окне Конфигурировать шлангопроводы, и теперь их можно обрабатывать.

Условия:

* У вас установлен конфигуратор шлангопроводов в качестве самостоятельной программы.
* Например, вы получили из конструкторского отдела файл обмена в виде разделенного табуляторами текстового файла Юникод с неполными данными шлангопроводов.

1. Запустите конфигуратор шлангопроводов, дважды щелкнув на рабочем столе Windows по символу программы со шлангопроводом.
2. Выберите пункты меню Файл > Конфигурировать шлангопроводы.

!!! info "Для сведения:"

    Откроется диалоговое окно Конфигурировать шлангопроводы.

3. Выберите пункты меню Файл > Открыть.
4. В следующем диалоговом окне выберите место хранения и имя файла обмена и щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Импортированные данные шлангопроводов отображаются в диалоговом окне Конфигурировать шлангопроводы, и теперь их можно обрабатывать.

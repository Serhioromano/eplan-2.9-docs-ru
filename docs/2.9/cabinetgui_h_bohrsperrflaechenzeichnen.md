# Начертить и изменить запретные зоны для сверления

Запретные зоны для сверления закрывают затронутые области от определения координат сверления. При этом различают два случая:

* ***Размещение монтажных отверстий или функциональных элементов со схемами сверления по заранее определенной запретной зоне для сверления***: Размещение монтажных отверстий невозможно. Размещение функциональных элементов со схемами сверления хотя и возможно, однако схемы сверления не будут видны в отображении отверстий и не экспортируются как данные ЧУ.
* ***Размещение запретной зоны для сверления поверх уже имеющихся монтажных отверстий или функциональных элементов со схемами сверления***: Имеющиеся отверстия со схем сверления, лежащие на этой поверхности, не будут видны в отображении отверстий и не экспортируются. Имеющиеся монтажные отверстия будут отображаться по-прежнему и будут экспортироваться как данные ЧУ, несмотря на размещенную поверх запретную зону для сверления. Если такие монтажные отверстия не должны входить в экспорт ЧУ, их следует удалять вручную. Для определения таких монтажных отверстий можно воспользоваться [Контрольным прогоном 026013](msg/messages_p_026013.md).

Запретные зоны для сверления размещаются таким же образом, как и запретные зоны для размещения. В отличие от запретных зон для размещения, запретные зоны для сверления проникают сквозь весь функциональный элемент и становятся видны и активны с противоположной стороны.

Запретные зоны для сверления отображаются в навигаторе пространства листа для каждой монтажной поверхности на уровне иерархии Заблокированные области.

Условие:

Вы открыли проект. Открыто одно пространство листа. Вы разместили один электрошкаф или монтажную плату.

1. Выберите пункт меню Вставить > Заблокированная область > Запретная зона для сверления.
2. Переместите курсор на область, которую требуется заблокировать, чтобы автоматически ее активировать; либо активируйте монтажную поверхность непосредственно в навигаторе пространства листов.

!!! info "Для сведения:"

    Автоматически или напрямую активированная монтажная область выделяется цветом. Ввод теперь возможен только на этой поверхности.

3. Начертите прямоугольник, представляющий запретную зону для сверления.

### Изменить размер запретных зон для сверления

Размер запретных зон для сверления можно изменить двумя способами:

* ***Изменение свойств формата в диалоговом окне 'Свойства'***: на вкладке Формат в диалоговом окне 'Свойства' введите измененные значения в поля Ширина и Высота. Изменение производится относительно нижней левой угловой точки.
* ***Графическое изменение***: выделите объект. На угловых точках и в центре боковых линий отображаются указатели в виде стрелок в направлении, в котором можно растянуть или сжать объект. При перемещении вдоль указателя на соседних объектах 3D отображаются точки захвата, которые можно использовать для точного позиционирования.

# Настроить целевой проект (изменение стандарта)

При изменении стандарта происходит преобразование исходного проекта. Преобразованный исходный проект сохраняется в целевом проекте. Исходный проект сохраняется без изменений.

!!! note "Замечание:"

    Обратите внимание, что в целевой проект копируются только те объекты, которые находятся внутри области отчета рамки (т. е. области, установленной через строки и столбцы).

Условия:

* Вы открыли проект.
* При нескольких открытых проектах: Вы выделили проект в Навигаторе страниц.
* Вы имеете шаблон, содержащий специфические для страны или стандарта настройки.

1. Сервисные программы > Изменение стандарта
2. На вкладке Проект в поле Имя проекта Целевой проект задайте информативное имя проекта без расширения имени файла.
3. Чтобы выбрать место хранения, щелкните рядом с полем место хранения по ++...++.
4. Выберите в диалоговом окне Выбрать каталог каталог для сохранения нового проекта и щелкните по кнопке ++OK++.
5. Нажмите на кнопку ++Далее++.

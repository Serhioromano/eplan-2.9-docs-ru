# Операция: ImportPrePlanningData

**Параметр** |  **Описание**
---|---
PROJECTNAME  |  Имя проекта с полным путем файла (является опцией).
Если не задано, то выбранный проект используется, когда операция вызывается через интерфейс пользователя (напр., через сценарий или панель инструментов). При вызове из командной строки Windows следует определить PROJECTNAME или сначала следует использовать ProjectAction. В противном случае отобразится системное сообщение.
FILENAME  |  Полный путь и имя импортируемого файла.
SCHEMENAME  |  Имя схемы, определяющей присвоение внешних полей данных свойствам EPLAN; обязательно указывается.
TABLENAME  |  Если импорт выполняется из файла Excel, то в этом параметре указывается имя таблицы или области данных импортируемого источника данных.
DELIMITER  |  Если импорт выполняется из текстового файла, то в этом параметре определяется разделитель, который применяется в текстовом файле для разделения столбцов.
HEADER  |  Вывод обозначений столбцов (необязательно), имеет значение только для импорта из Excel. 1: Обозначения столбцов таблицы данных выводятся в столбце "Внешнее поле" таблицы присвоений. 0: Обозначения столбцов не выводятся. Значение по умолчанию: 0.
TARGETNAME  |  Имя объекта (свойство "Обозначение (полностью)", Ид. 44009), под которым необходимо вставить импортированные данные. Если данный параметр пуст, данные вставляются под именем проекта.
SKIPERRORS  |  Если значение равно 1, импорт не прерывается, когда возникает ошибка или выводятся сообщения (необязательно). Значение по умолчанию: 1.
OVERWRITE  |  Если значение равно 1, данные предварительного планирования с одинаковым именем перезаписываются при импорте (необязательно). Значение по умолчанию: 1.
UPDATEONLY  |  Если значение равно 1, обновляются только данные имеющихся сегментов структуры и данные планирования (необязательно). Значение по умолчанию: 0.

!!! example "Пример:"

    Импортировать данные предварительного планирования из Excel:ImportPrePlanningData
    /PROJECTNAME:C:\Projects\ESS_Sample_Project.elk
    /FILENAME:C:\Projects\EPLAN\preplanning.xls
    /SCHEMENAME:config_scheme
    /Header:1
    /UPDATEONLY:1

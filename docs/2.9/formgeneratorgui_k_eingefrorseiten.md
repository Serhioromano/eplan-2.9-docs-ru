# Необновляемые страницы отчета

Страницы отчетов можно зафиксировать. В этом случае обновление соответствующих страниц больше не выполняется. Блок отчетов удаляется. Необновляемые страницы отчета остаются в навигаторе страниц. [Встроенные отчеты](formgeneratorgui_k_ausgabeformen.md) не фиксируются.

Обновляемые страницы отчета далее обновляются.

Пиктограмма в навигаторе страниц |  Значение
---|---
![](images/pagebrowsergui_pageautomatic_as.png){: .ui-icon } |  Страница отчета
![](images/pagebrowsergui_pageautomaticfrozen_as.png){: .ui-icon } |  Необновляемая страница отчета

!!! tip "Совет:"

    Чтобы генерировать новые страницы отчета вам сначала нужно удалить вручную существующие страницы и затем создать новые. Если вы не удалите страницы отчета, то будут созданы дополнительные страницы отчета.

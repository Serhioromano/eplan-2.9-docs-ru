# Выбор изделия / устройства

Изделия можно присваивать какой-либо функции при помощи выбора изделия или устройства. Выбор изделия позволяет выполнять выбор изделия на основе ***коммерческих*** данных, выбор устройства осуществляется на основе ***технических*** признаков.

Технические признаки обычно получают из внутренней базы данных изделий, тогда как экономические данные можно получать также из внешней базы данных. Также технические данные можно дополнять из внешней базы данных. Можно также работать только с внутренней базой данных изделий или дополнительно подключать внешнюю базу данных.

### Выбор устройства

Выбор устройства ***всегда*** выполняется через собственную базу данных изделий EPLAN. Тем самым во время выбора устройства используются только данные, управляемые в пределах внутренней базы данных изделий.

При подсоединении к выбору устройства данные изделий можно при необходимости дополнять из внешней базы данных. Для этого имеются следующие источники данных:

* Интерфейс API
* База данных Access
* Сервер SQL.

Источник данных для дополнения данных изделий может настраиваться для выбора изделия отдельно от источника данных.

Дополнение данных изделий через внутреннее присоединение также необходимо после ввода макроса, чтобы заполнить данными изделия, используемые после этого только в качестве ссылки.

### Выбор изделий

Выбор изделий можно выполнять выборочно через собственную базу данных изделий или внешнюю базу данных. Для этого имеются следующие источники данных:

* Интерфейс API
* База данных Access
* Сервер SQL.

Источник данных для выбора изделий может настраиваться независимо от источника данных для дополнения данных после выбора устройства.

При подсоединении к выбору изделий данные изделий можно еще дополнять техническими данными через внутреннее присоединение.

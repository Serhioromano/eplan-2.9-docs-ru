# Вкладка Определение потенциала

Вы открыли проект.

* В Графическом редакторе выделена точка определения потенциала. Обработать > Свойства. В диалоговом окне Свойства ++...++ выберите вкладку Определение потенциала.
* Вставить > Точка определения потенциала. В диалоговом окне Свойства ++...++ выберите вкладку Определение потенциала.

На этой вкладке задаются свойства точки определения потенциала.

Обзор основных элементов диалогового окна:

### Имя потенциала:Имя сигнала

Укажите здесь имя для потенциала / сигнала. С помощью кнопки ++...++ откройте диалоговое окно Выбрать потенциал / сигнал. Здесь выводятся для выбора уже имеющиеся в проекте имена потенциалов, а также соответствующие имена сигналов.

Если поле Имя потенциала пусто, то сигнал, введенный в поле Имя сигнала, сам является потенциалом. Если заполнены оба поля, то это сигнал. Если в поле Имя потенциала введен потенциал, которого нет или который является сигналом, то появляется сообщение об ошибке и применяются настройки проекта.

### Поперечное сечение / диаметр:Единица измерения сечения / диаметра

Поперечное сечение или диаметр соединения копируется из базы данных изделий и могут быть использованы в качестве критерия отбора при выборе устройств. Щелкните по ++...++, чтобы открыть диалоговое окно Поперечные сечения / диаметры и выбрать сечение / диаметр, включая соответствующую единицу, из предложенного списка.

Если в поле Поперечное сечение / диаметр вы выбрали предварительно определенное поперечное сечение / диаметр, то в поле Единица измерения сечения / диаметра автоматически появится соответствующая единица. С помощью раскрывающегося списка возможен выбор другой единицы измерения. Обзор возможных единиц измерения приводится в соответствующем разделе "[Соединения: Единицы измерения длины и поперечного сечения](connectionbrowsergui_k_einheitenverbindungen.md)".

### Цвет / номер

Введите здесь цвет или номер соединения, это свойство идентифицирует это соединение. Щелкните ++...++, чтобы открыть диалоговое окно Цвета соединений и выбрать цвет из списка.

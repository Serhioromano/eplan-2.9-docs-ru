# EPLAN Fluid: Особенности работы с макросами в смешанных проектах

При помощи макросов можно сохранять любые фрагменты схемы соединений для последующего использования, таким образом существенно ускоряя процесс проектирования схемы соединений.

Нередко в смешанных проектах требуется сохранение документации по машинам или установкам, в которой содержатся как электротехнические страницы схемы соединений, так и страницы Fluid-техники. Работа с макросами целесообразна и в межпредметном проектировании. Уже во время создания макросов в рамках ***одного и того же*** файла макроса можно использовать фрагменты обоих типов схем соединений. (Это возможно при работе только с макросами окон и макросами символов.)

Для этого необходимо, чтобы макросы различались между собой свойством ***Вид представления***. Поэтому в пределах одного файла макроса фрагмент схемы соединений Fluid-техники должен быть сохранен с видом представления "Многопол. Fluid", а соответствующий фрагмент электрической схемы соединений – с видом представления "Многополюс.".

При вставке макроса окна или макроса символа на одну страницу схемы соединений вид представления позволяет программе автоматически определить тип страницы для верного размещения соответствующего фрагмента.

Этот же принцип действует при размещении изделий "перетаскиванием" из списка устройств или навигатора спецификаций, которым присвоен файл макроса с фрагментами схемы соединений обоих типов.

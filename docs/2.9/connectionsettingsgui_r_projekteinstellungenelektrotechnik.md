# Вкладка Электротехника

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Свойства. Выберите вкладку Электротехника.

На данной вкладке определите предварительные настройки для свойств электротехнических соединений всего проекта. Данные свойства по умолчанию присваиваются всем соединениям проекта. Однако они могут быть перезаписаны отдельными соединениями.

!!! note "Замечание:"

    Настройки на этой вкладке действуют только для отдельных соединений (на соединении для свойства Соединение: Принадлежность установлено значение "Отдельное соединение"), но не для кабельных соединений. Значения по умолчанию для кабельных соединений (например, поперечное сечение) берутся из изделия кабеля.

Обзор основных элементов диалогового окна:

Групповое поле "Поперечное сечение/диаметр"

### Значение:Единица измерения

Введите здесь значение по умолчанию для поперечного сечения или диаметра соединений, а также для соответствующей единицы измерения.

Обзор возможных единиц измерения приводится в соответствующем разделе "[Соединения: Единицы измерения длины и поперечного сечения](connectionbrowsergui_k_einheitenverbindungen.md)".

### Альтернативное значение:Альтернативная единица

Здесь можно задать альтернативное поперечное сечение / диаметр, а также соответствующую альтернативную единицу измерения.

Альтернативные свойства независимы от базы данных изделий, должны заполняться вручную, отчет также должен генерироваться вручную. С помощью этих свойств поперечное сечение / диаметр соединения можно указать одновременно в различных единицах (например, AWG и мм²). Свойства доступны в диалоговом окне свойств для соединений и могут использоваться в отчетах.

* * *

### Длина

Введите здесь значение по умолчанию в метрах для длины соединения.

### Единица измерения длины

Раскрывающийся список предлагает на выбор возможные единицы измерения длины соединения. Обзор возможных единиц измерения приводится в соответствующем разделе "[Соединения: Единицы измерения длины и поперечного сечения](connectionbrowsergui_k_einheitenverbindungen.md)".

### Обозначение типа

Укажите здесь обозначение типа изделия соединения (например, номер типа провода). Значение переносится в свойство Соединение: Обозначение типа и служит в качестве значения по умолчанию для новых соединений.
При выборе изделия значение из базы данных изделий, заданное на вкладке Свойства под узлом Данные соединения перемещается в свойство и перезаписывает значение по умолчанию.

### Обработка концов проводов, источник / цель

Введите здесь, чем должно заканчиваться соединение, напр., втулкой для жилы или кабельным наконечником. Это свойство требуется при переносе на сборочные машины.

### Сечение наконечника, источник / цель

Если соединение заканчивается втулкой для жилы, то укажите здесь поперечное сечение втулки. Это свойство требуется при переносе на сборочные машины.

### Длина зачистки, источник / цель

Введите здесь длину, на которой удаляется изоляция. Это свойство требуется при переносе на сборочные машины.

### Номер изделия

Введите здесь номер изделия соединения или, нажав кнопку ++...++, откройте выбор изделия и выберите там изделие. Номер изделия переносится на вкладку Изделие диалогового окна 'Свойства' соединений и используется как значение по умолчанию для новых соединений. Рекомендуется использовать изделие без указания длины и цены. Для вычисления степени заполнения при маршрутизации необходимо ввести на изделии внешний диаметр.

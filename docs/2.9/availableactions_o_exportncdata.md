# Операция: ExportNCData

**Параметр** |  **Описание**
---|---
MachineType  |  Целочисленное значение, необходимо указать. Доступны следующие значения:

* 1 = ЧУ Steinhauer
* 2 = Rittal - Perforex BC
* 3 = ЧУ DXF
* 4 = Шаблон сверления
* 5 = Rittal - Perforex LC
* 6 = ЧУ, медь
* 7 = DXF, медь
* 8 = Изгибание труб

MachineName  |  Символьная строка, необходимо указать. Имя машины должно соответствовать имени, заданному в настройках.
PROJECTNAME  |  Имя проекта с полным путем файла (является опцией).
Если не задано, то выбранный проект используется, когда операция вызывается через интерфейс пользователя (напр., через сценарий или панель инструментов). При вызове из командной строки Windows следует определить PROJECTNAME, или сначала следует использовать ProjectAction. В противном случае отобразится системное сообщение.
USESELECTION  |  Устанавливает, необходимо ли учитывать текущий выбор в интерфейсе пользователя при выводе данных ЧУ (необязательно, 0 = Нет, 1 = Да).
Значение по умолчанию: 0. Доступно для использования, только когда не используется параметр PROJECTNAME.
TargetDirectory  |  Целевой каталог (является опцией). Если параметр задан, перезаписывается заданный в настройках каталог вывода.
MaterialName  |  Доступно только для машин Rittal - Perforex и ЧУ, медь. Если параметр задан, соответствующий материал заимствуется из настроек машины и используется для экспорта.

!!! example "Пример:"

    ExportNCData
    /MACHINETYPE:2
    /MACHINENAME:Standard
    /TARGETDIRECTORY:D:export
    /MATERIALNAME:Steel

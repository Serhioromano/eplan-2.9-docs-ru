# Свойства страницы

В навигаторе страниц и в графическом редакторе можно просмотреть свойства выделенной или текущей открытой страницы. К важнейшим свойствам страницы относятся следующие:

### Имя страницы

Имя страницы содержит полное имя с возможными разделителями и обозначениями подстраниц, однако без структурных идентификаторов. Имена страниц могут состоять из любых комбинаций цифр и букв. Во всех диалоговых окнах всегда выводится полное имя страницы.

Используя цифры и буквы, помимо имени страницы можно указать и ***подстраницу***. Это имеет особенно целесообразно в том случае, если из-за изменений в имеющейся схеме или списке требуется ввести дополнительные страницы. Имя подстраницы всегда отделяется точкой от имени главной страницы. При наличии нескольких точек разделителем является первая точка слева. Для имен подстраниц используется четыре разряда.

### Имя страницы (полное)

Полное имя страницы содержит имя страницы и все структурные идентификаторы (идентифицирующие и описывающие) в соответствии со структурой проекта.

### Имя страницы (идентифиц.)

Идентифицирующее имя страницы содержит имя страницы и идентифицирующие структурные идентификаторы в соответствии со структурой проекта.

### Тип страниц

В EPLAN используются различные [типы страниц](pagebrowsergui_k_seitentypen.md). Содержимым этих типов страниц могут быть, например, требуемые формы для вывода информации схемы соединений или же сама схема соединений.

### Описание страниц

Для каждой страницы можно ввести описание, например, "Силовая электрическая цепь, насос 1" или "Управляющая электрическая цепь, транспортер 3". Описание страниц является (многоязычным) специальным текстом и может содержать переходы на новую строку. Его можно использовать в разных местах, а также распечатать.

### Настройки проекта "Символы для подстраниц"

Для создания новых страниц можно определить символы, которые будут использоваться при выводе предлагаемого обозначения подстраниц.
Имеются следующие возможности:

Нумерация подстраниц |  Пример
---|---
Без подстраниц |  1 2 3 4 5 A B C 11 12 AA BB
Строчные буквы |  1 1.a 1.b 1.c 2 2.a 2.b A.a A.b 100.c 100.d 100.e
Прописные буквы |  1 1.A 1.B 1.C 2 2.A 2.B A.A A.B 100.C 100.D 100.E
Числа |  1 1.1 1.2 1.3 2

По умолчанию указана запись "Строчные буквы". Изменение этой настройки в существующем проекте ***не*** ведет к изменению последовательности страниц, т. е. логика перекрестных ссылок и т. д. не изменяется.

### Настройка проекта "Минимальная длина имени страницы"

Здесь настраивается представление имен страниц. Эта запись указывает на форматирование имен страниц при их выводе. Страницы выводятся с ведущими нулями в соответствии с указанным здесь числом. В случае ввода записи "0" ведущие нули не выводятся. Эта запись указана по умолчанию.

!!! note "Замечание:"

    Если указать здесь число, не равное нулю, а позже вручную дополнить ведущие нули в имени страницы, то введенные вручную записи подавляются.

### Доп. поле номера листа

Номер листа — это свойство страницы, дополняющее имя страницы. Он может использоваться только один раз для каждой страницы с одинаковым структурным идентификатором. Номера листов могут состоять из любых комбинаций цифр и букв. Их можно вывести в представлении в виде списка.

Для автоматического определения свойства при отчетах необходимо ввести начальный номер. Тогда нумерация листов продолжится на следующих страницах отчета одинакового типа, начиная с начального номера.

!!! note "Замечание:"

    Так как Доп. поле номера листа можно ввести вручную и автоматически, возможно возникновение пропусков и двойных номеров листа. Для отображения порядкового номера страницы, например в свойствах страницы, в графическом редакторе или на страницах отчета используется свойство страницы Счетчик числа страниц (Ид. 25020). Это свойство можно вставить в редакторах как специальный текст.

### Масштаб

Для любой страницы независимо от ее типа можно указать масштаб. Масштаб требуется в том случае, если используется указание размеров, или для точного масштабирования при вставке.

### Сетка

При создании страницы EPLAN автоматически указывает стандартное разрешение для сетки. Это поле является "полем ввода с единицами". С помощью раскрывающегося списка можно изменить эти значения, значение по умолчанию и выбираемая стандартная сетка являются изменяемыми. При поставке значения по умолчанию для логических страниц установлены на "4 мм", а для графических страниц - на "1 мм".

### Выравнивание страниц / формат бумаги

Это свойство настраивается с помощью присвоенной рамки и выводится только в ней. Стандартный форматы бумаги (длина и ширина) отличаются в зависимости от региона. Кроме того, в EPLAN можно использовать форматы бумаги, определенные пользователем.
Формат бумаги можно изменить и позже, присвоив другую рамку. Если большой формат заменяется на меньший, например DIN A2 на DIN A4, некоторые части схемы соединений, графики или форм могут находиться за пределами видимой области. В этом случае требуется выполнить ручную настройку.

### Рамка

Рамки могут использоваться на всех типах страниц. В настройках проекта и страницы указывается, какая рамка применяется для страницы. При этом настройка, выполненная для страницы, перезаписывает настройку, выполненную для проекта. Рамка сохраняется в проект, если она указана в качестве свойства страницы.

### Форма

Формы могут использоваться для всех типов страниц. Для этого в свойстве страницы сохраняется имя формы, которое также сохраняется и в проекте. Формы можно выбрать с помощью диалогового окна выбора файла (с предварительным просмотром и отображением текста описания) или же указать вручную.

### Автоматическая дата изменения

Для каждой страницы автоматически записывается дата последнего изменения. Это свойство невозможно изменить вручную. Поле Автоматическая дата изменения обновляется в случае любого изменения в графическом редакторе и в свойствах страницы, а также при генерировании и обновлении страниц отчетов.

### Последний обработчик

Для каждой страницы автоматически записывается имя последнего пользователя, обработавшего страницу. Это свойство невозможно изменить вручную.

### Ручная дата изменения

Для каждой страницы можно вручную ввести дату. Представление даты можно конфигурировать в настройках проекта (Параметры > Настройки > Проекты > "Имя проекта" > Представление > Дата / время / числа).

!!! example "Пример:"

    19.07.1991; 19.Jun 1991; 19-07-91; 19/07/1991

### Определение числа / имен страниц на свойство

Используя пункты меню Страница > Число / имена страниц на свойство, можно определить число и имена страниц, имеющих одинаковое значение определенного свойства. Кроме того, можно определить, о какой странице с одинаковым значением этого свойства идет речь.

Свойство, которое используется в качестве критерия для сравнения, определяется в настройках проекта с помощью настройки Число / имена страниц на свойство. Результат записывается в свойства страницы Количество страниц в свойстве, Имена страниц в свойстве и Счетчик страниц в свойстве.

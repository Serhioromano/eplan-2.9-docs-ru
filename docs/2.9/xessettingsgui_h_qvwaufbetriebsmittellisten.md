# Генерировать перекрестные ссылки на списках обозначений устройств

На всех главных и вспомогательных функциях на схеме соединений может быть генерирована дополнительная перекрестная ссылка, которая отображает список обозначений устройств, в котором содержится соответствующее устройство. Перекрестная ссылка может быть обозначена при помощи выделения, особого символа, чтобы отличаться от других перекрестных ссылок.

Условия:

* Вы открыли проект.
* Вы создали многополюсную страницу схемы соединений.
* В меню Параметры > Настройки > Проекты > "Имя проекта" > Перекр. ссылки / образы контактов > Отображать в групповом поле Просмотреть перекр. ссылки устройств вы указали, что перекрестные ссылки должны отображаться между многополюсной страницей схемы соединений и страницей типа"Список обозначений устройств".

1. Откройте многополюсную страницу схемы соединений в Графическом редакторе и добавьте символы, например, трансформатор, предохранитель и сигнальная лампа.
2. Задайте в диалоговом окне Свойства ++...++ на вкладке <Категория функции>, соответственно, Видимое ОУ для условного обозначения, например "-T1" для трансформатора, "-F1" для предохранителя и "-H1" для сигнальной лампы.
3. Установите, если он еще установлен, для каждого условного обозначения флажок Главная функция.
4. Выберите для каждого условного обозначения вкладку Изделие и присвойте через выбор изделия каждому условному обозначению соответствующий номер изделия.
5. Щелкните по кнопке ++OK++.
6. Выберите пункты меню Сервисные программы > Отчеты > Генерировать.
7. Щелкните в диалоговом окне Отчеты — <Имя проекта> по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
8. Выделите в диалоговом окне Определить отчет в списке Выбрать тип отчета запись "Список обозначений устройств" и щелкните по кнопке ++OK++.
9. Щелкните в диалоговом окне Настройки — Список обозначений устройств по ++OK++.
10. Введите в диалоговом окне Список обозначений устройств ++...++ соответствующие структурные идентификаторы для присоединения списка обозначений устройств к проекту, определите номер страницы и щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Создается список обозначений устройств.

!!! info "Для сведения:"

    На главные функции условных обозначений на схеме соединений теперь выводится информация о перекрестных ссылках, которые отображают соотвествующие условные обозначения в списке обозначений устройств. И наоборот в списке обозначений устройств выводится информация о перекрестных ссылках, которые отображают главные функции условных обозначений на схеме соединений.

11. Закройте диалоговое окно Отчеты- <Имя проекта>.
12. Откройте схему соединений и список обозначений устройств в отдельных редакторах, чтобы можно было просмотреть образование перекрестных ссылок.

!!! example "Пример:"

    Трансформатор T1, предохранитель F1 и сигнальная лампа H1 на схеме соединений (левая часть рисунка) указывают на соответствующие условные обозначения в списке обозначений устройств (правая часть рисунка). И наоборот — перекрестные ссылки, выведенные в списке обозначений устройств, указывают на соответствующие условные обозначения на схеме соединений.

### Отобразить перекрестные ссылки спискa обозначений устройств вспомогательных функций

если необходимо отобразить перекрестные ссылки оборудования вспомогательных функций распределено представленных устройств на схеме соединений для вспомогательных функций, выполните следующие действия:

1. Выберите Параметры > Настройки > Проекты > "Имя проекта" > Перекр. ссылки / Образ контакта > Общее и установите флажок Вспомогательные функции также указывают на список обозначений устройств.
2. Щелкните по кнопке ++OK++.
3. Обновите при необходимости представление схемы соединений через пункты меню Вид > Начертить заново.

### Выделите перекрестные ссылки спискa обозначений устройств

Чтобы указать перекрестные ссылки устройств в схеме соединений при помощи выделения, выполните такие действия:

1. Выберите Параметры > Настройки > Проекты > "Имя проекта" > Перекр. ссылки / Образ контакта > Общее и задайте в групповом поле Выделение для любой символ в поле Список обозначений устройств.
2. Щелкните по кнопке ++OK++.
3. Обновите при необходимости представление схемы соединений через пункты меню Вид > Начертить заново.

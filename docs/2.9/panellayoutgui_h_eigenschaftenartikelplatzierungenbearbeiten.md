# Обработать и отобразить свойства размещений изделий

При ***отображении*** свойств размещений изделий необходимо обратить внимание на следующее: Отдельные размещенные свойства могут быть присоединены друг к другу, образовывая таким образом общий блок; это позволяет перемещать все (требуемые) тексты одновременно и предотвращает перекрытие многострочных текстов другими текстами.

Условия:

* Вы открыли проект.
* Вы находитесь в графическом редакторе на странице типа "компоновка электрошкафа".
* Вставлено размещение изделия или Вы дважды щелкнули по размещению изделия (или выбрали Всплывающее меню > Свойства; в этом случае можно выделить несколько размещений изделий).

1. Щелкните на вкладке Размещение изделия по кнопке ++...++ в поле Видимое ОУ.
2. Выберите в диалоговом окне Выбор ОУ то ОУ, которое должно быть выведено в графическом редакторе на размещении изделия.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    ОУ копируется в поле Видимое ОУ.

4. Щелкните по кнопке ++...++ в поле Изделие, чтобы присвоить изделие размещению изделия.
5. Выберите требуемое изделие в диалоговом окне Выбор изделия..
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Номер изделия копируется в поле Изделие.

7. Обработайте свойства размещения изделия, введя в поле Значение символьную строку, выбрав значение из раскрывающегося списка и т.д.
8. Щелкните по ![](images/all_new_as.png){: .ui-icon } (Создать) в панели инструментов сверху списка свойств.
9. Выберите в диалоговом окне Выбор свойств другие свойства, которым требуется присвоить значение. (Возможен многократный выбор.)
10. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Свойства добавляется в столбец Имя свойства в конце списка.

11. Присвойте требуемые значения новым свойствам.
12. Выделите свойства, которые Вам (более) не нужны. (Возможен многократный выбор.)
13. Щелкните по ![](images/all_delete_as.png){: .ui-icon } (Удалить) в панели инструментов сверху списка свойств.

!!! info "Для сведения:"

    Свойства будут убраны из списка, т. е. будут скопированы без присвоенных им ранее значений в диалоговое окно Выбор свойств, после чего их можно снова присваивать.

14. Перейдите на вкладку Отображение.
15. Выберите из раскрывающегося списка Порядок свойств запись Определенный пользователем, чтобы определить необходимую Вам форму представления.
16. Выделите в списке свойств то свойство, для которого требуется указать настройки отображения.

!!! info "Для сведения:"

    Данные в отдельных полях копируются из стандартного значения по умолчанию.

17. Укажите настройки для отображения в полях.
18. Перейдите на вкладку Формат.
19. Укажите настройки формата размещения изделия, выбрав записи из раскрывающихся списков, установив / сняв флаги и т.д.
20. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выполняется обновление отображения свойств.

!!! note "Замечание:"

    Каждое устройство может содержать несколько ссылок изделий в разном количестве. Однако одно размещение изделия может быть размещено только с количеством "1". Поэтому может случиться так, что несколько размещений изделий с одним и тем же обозначением устройства будут установлены на одну монтажную плату.

# Создать идентификаторы при вставке новых страниц, ОУ и макросов

Если посредством создания новых страниц или устройств или посредством добавления страниц и макросов в проект добавляются новые структурные идентификаторы, в таблицу идентификаторов вводятся новые структурные идентификаторы в соответствии с настройками пользователя для идентификаторов (параметр Сортировать новые идентификаторы по алфавиту или Вставить новые идентификаторы в конце). В управлении структурными идентификаторами можно впоследствии получить доступ к данным идентификаторов выбранных блоков идентификаторов и, например, изменить их классификацию.

!!! note "Замечание:"

    Обратите внимание, при создании нового идентификатора используется настройка проекта Преобразование прописных букв (в разделе Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Общее).

### Создать идентификатор при обработке страниц

При создании новой страницы или при переименовании / перемещении уже существующей Вы можете указать новый структурный идентификатор, которого еще нет в базе данных. Как новый структурный идентификатор стандартно сортируется в списке (в алфавитном порядке или в конце списка), зависит от определенных настроек идентификатора. Имеющиеся структурные идентификаторы здесь изменены быть не могут.

Условие:

Вы открыли проект, который в структуре проекта использует блоки идентификаторов, например, "Установка" и "Место установки".

1. Выберите пункты меню Страница > Создать.
2. Если вы хотите увидеть доступные структурные идентификаторы, щелкните по кнопке ++...++ рядом с полем Полное имя страницы диалогового окна Новая страница.
3. Щелкните в диалоговом окне Полное имя страницы рядом с соответствующим полем для блока идентификаторов кнопку ++...++.

!!! info "Для сведения:"

    В диалоговом окне <Блоки идентификаторов> перечислены все имеющиеся структурные идентификаторы. При этом на вкладке Список отображается содержание обоих полей Полный структурный идентификатор и Описание структуры.

4. Щелкните по кнопке ++Отменить++, чтобы закрыть список.
5. Задайте в диалоговом окне Полное имя страницы новый структурный идентификатор в одном из полей для блоков идентификаторов.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В диалоговом окне Новая страница соответственно обновится полное название страницы.

### Создать идентификатор при обработке устройств

При создании новых или при переименовании / перемещении уже существующих устройств, кабелей и т. д., Вы можете указать новый структурный идентификатор, которого еще нет в базе данных. Как новый структурный идентификатор стандартно сортируется в списке (в алфавитном порядке или в конце списка), зависит от определенных настроек идентификатора. Имеющиеся структурные идентификаторы здесь изменены быть не могут.

Условия:

* Вы открыли проект, который в структуре проекта использует блоки идентификаторов, например, "Установка" и "Место установки".
* Вы открыли страницу схемы соединений.

1. Выберите пункт меню Вставить > Символ.
2. В диалоговом окне Выбор символа выберите требуемый символ и щелкните по кнопке ++ОК++.
3. С помощью мыши разместите символ.
4. Щелкните в поле Полное ОУ в компонентно-зависимом диалоговом окне Свойства ++...++ на ++...++.
5. Чтобы увидеть доступные структурные идентификаторы, нажмите кнопку ++...++ рядом с соответствующим полем для блока идентификаторов.

!!! info "Для сведения:"

    В диалоговом окне <Блоки идентификаторов> перечислены все имеющиеся структурные идентификаторы.

6. Щелкните по кнопке ++Отменить++.
7. Задайте в диалоговом окне Полное ОУ новый структурный идентификатор в одном из полей для блоков идентификаторов.

### Создать идентификатор при обработке макроса

Даже при добавлении макроса с устройством, имеющим блоки идентификаторов, Вы можете указать новый структурный идентификатор, не существующий в базе данных. Как новый структурный идентификатор стандартно сортируется в списке (в алфавитном порядке или в конце списка), зависит от определенных настроек идентификатора. Имеющиеся структурные идентификаторы здесь изменены быть не могут.

Условие:

Вы открыли проект, который в структуре проекта использует блоки идентификаторов, например, "Установка" и "Место установки".

1. Выберите пункты меню Страница > Макрос страницы > Вставить.
2. В следующем диалоговом окне выберите вставляемый макрос страницы и щелкните по кнопке ++Открыть++.
3. В диалоговом окне Адаптировать структуру определите способ обработки названий страниц для целевого макроса.
4. Чтобы увидеть имеющиеся идентификаторы, нажмите кнопку ++...++ внутри соответствующего поля для блока идентификаторов цели.

!!! info "Для сведения:"

    В диалоговом окне <Блоки идентификаторов> перечислены все имеющиеся идентификаторы.

5. Щелкните по кнопке ++Отменить++.
6. Задайте в диалоговом окне Адаптировать структуру новый структурный идентификатор в одном из полей для блока идентификаторов цели (например, "Установка" или "Место установки").

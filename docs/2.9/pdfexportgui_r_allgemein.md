# Вкладка Общее (пользователь, Экспорт PDF)

* Страница > Экспортировать > PDF. Щелкните в поле Схема диалогового окна Экспорт PDF[...]. Выберите вкладку Общ..
* Параметры > Настройки > Пользователь > Интерфейсы > Экспорт PDF. Выберите вкладку Общее.

В этой вкладке установите общие фирменные настройки для экспорта PDF-файлов.

Обзор основных элементов диалогового окна:

### Использовать масштабирование

Установите этот флажок, если в PDF при переходах на условное обозначение должно использоваться окно масштабирования. Условное обозначение затем выводится по центру окна масштабирования. При установке флажка нижерасположенное поле ввода Масштабирование разблокируется.

При снятии флажка выводится вся страница при выполнении переходов.

### Масштабирование

Если установлен флажок  Использовать масштабирование, внесите здесь требуемое масштабирование в "мм". При этом введенное здесь значение соответствует высоте фрагмента изображения, который должен появиться на экране после перехода в увеличенном виде. Меньшое значение (напр., 20 мм) приводит соответственно к сильному увеличению изображения соответствующего условного обозначения на экране.

### Упрощенная функция перехода

При установке флажка сохраненный файл PDF имеет только "простые" функции перехода.
При снятии флажка в PDF на всех условных обозначениях появляются "трехходовые" переходы.

Благодаря применению настройки Упрощенная функция перехода можно, кроме того, также еще уменьшать величину файла PDF.

### Быстрое веб-отображение

При установке этого флажка файл PDF оптимизируется для отображения в интернете. В таком случае первые страницы файла PDF уже показываются тогда, когда остальная часть файла еще загружается.

При снятии флажка файл будет выводиться только после его полной загрузки.

### С защитой от записи

Установите этот флажок, если при экспорте необходимо создать файл PDF с защитой от записи. Один из PDF-документов, созданных при помощи этого параметра, нельзя больше изменять при помощи программы Adobe Acrobat.

### Скопировать внешне связанные документы в каталог вывода

Установите этот флажок, чтобы посторонние/связанные документы, которые не сохранены в проекте и отсутствуют в стандартном каталоге для документов, при выводе также копировались в целевой каталог.

Если флажок снят, все документы, которые не сохранены в проекте ***и*** отсутствуют в стандартном каталоге для документов, будут проигнорированы.

### Формат архивирования PDF/A

Чтобы обеспечить доступность документации на длительное время, существует возможность архивирования файлов PDF. Установите этот флажок, чтобы сохранять данные в формате PDF/A-1b. Обратите внимание, что модели и связанные документы в этом формате ***не*** учитываются.

В PDF-файлах формата PDF/A обозначить цели перехода посредством красной рамки невозможно.

### Дерево страниц по

В этом групповом поле вы можете установить представление в виде дерева для выводимого PDF-документа.

* Структурный идентификатор: Если этот параметр выбран, страницы на дереве страниц PDF-файла упорядочиваются по структурным идентификаторам.
* Тип страницы: Если этот параметр выбран, страницы на дереве страниц PDF-файла упорядочиваются по типам страниц.

### Тексты в PDF-файле

В этой таблице выводятся свойства, которые встречаются в PDF-файле в качестве вкладок и других устойчивых понятий (например, типы страниц, названия отображаемых областей). Предварительно определенные имена данных свойств могут быть обработаны в столбце Имя.

При экспорте файлов PDF тексты затем выводятся на настроенном языке вывода PDF (Параметры > Настройки > Проекты > "Имя проекта" > Управление > Языки вывода PDF). Если текст не может быть найден на данном языке, то он выводится на первом языке из списка имеющихся.

# Обозначения выводов устройства и описания

В EPLAN можно создавать несколько наборов обозначений выводов устройства и описаний. Они экспортируются и импортируются. При экспорте обозначения и описания выводов устройства сохраняются со спецификой определения функции, а при импорте они снова присваиваются определениям функции.

Логическая информация содержится в том же файле, что и сами определения функций. Они задаются программным обеспечением EPLAN Software & Service и не подлежат изменению. Все же для определенных логических настроек Вы можете создавать соответствующие макросы символов.

Обозначения выводов устройства

Для каждого вывода устройства определения функции Вы можете определять до десяти предложений для обозначения выводов устройства. Эти данные сохраняются по отдельным определениям функций. Обозначение вывода устройства является идентифицирующим признаком вывода устройства.

При создании функции обозначения выводов устройства заполняются обозначением выводов устройства "1". В диалоговом окне [Логическая схема выводов устройства](xfctdefbrowsergui_d_anschlusslogik.md) Вы можете изменять обозначения выводов устройства. Выберите запись из раскрывающегося списка или введите собственное обозначение вывода устройства. Обозначения выводов устройства вводятся в функцию всегда фиксированными и не описываются.

!!! example "Пример:"

    Вы создаете функцию с определением функции "Замыкающий контакт". По умолчанию обозначения выводов устройства заполнены 1, 2. Предварительно определенные Вами наборы обозначений выводов устройства выводятся в раскрывающемся списке, например, 13, 14, или 23, 24, или 33, 34. Из них выберите один набор, например, 23, 24. Список с наборами обозначений выводов устройства сохранен на определении функции, тогда как выбранные обозначения выводов устройства (в этом случае 23, 24) сохраняются на функцию.

Описания вывода устройства

Для каждого вывода устройства определения функции Вы можете определять до десяти предложений для описания выводов устройства. Эти данные сохраняются по отдельным определениям функций. Описание вывода устройства содержит дополнительные, неидентифицирующие данные.

### Функции с переменным числом выводов устройства

Существуют определения функции для установленного числа выводов устройства и такие же для переменного числа выводов устройства. Для них введено число выводов устройства "n". Последний вывод устройства обрабатывается отдельно, если был введен номер вывода устройства "z".

Если Вы выбрали определение функции, у которого число выводов устройства является "n", тогда число выводов устройства функции зависит от используемого символа. Если в определении функции имеется вывод устройства "z", тогда его данные (логика, ...) присваиваются последнему выводу устройства функции. Все остальные выводы устройства содержат данные из вывода устройства "n". Для неразмещенных функций (без символа), например в навигаторе клеммников, Вы должны сами вводить число выводов устройства.

Переменные функции применяются для тех случаев, для которых нет другой подходящей функции. Логическую информацию, такую как передача потенциала и отслеживания цели, Вы должны сами вводить на функцию. Они не могут быть предварительно определены, т.к. число выводов устройства не было известно. В каждой категории имеются переменные функции, так что Вы всегда сможете найти желаемую оценку условного обозначения.

### Определения функции для однополюсного представления

Существуют специальные определения функции для однополюсного представления. Также для переменных определений функции существуют однополюсные определения функции.

Для клемм и штекеров не имеется специальных однополюсных определений функции. Здесь используются стандартные клеммы и штекеры, которые затем соответствующе анализируются.

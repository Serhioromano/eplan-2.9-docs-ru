# Диалоговое окно Настройки: Изгиб шины (пользователь)

Параметры > Настройки > Пользователь > Графическая обработка > Изгиб шины.

В этом диалоговом окне можно установить поведение при автоматическом унаследовании отверстий в медных деталях.

Обзор основных элементов диалогового окна:

### Автоматически несколько монтажных отверстий

При оснащении медных функц. элементов можно автоматически передавать ссылки на монтажные отверстия с одного медного функц. элемента на другой. Если медный функциональный элемент размещается на монтажной поверхности другого и в области перекрытия находится монтажное отверстие, то для монтажного отверстия автоматически производится ссылка на новый размещенный медный функциональный элемент.

Унаследование монтажных отверстий из схемы сверления управляется свойством Допускается несколько монтажных отверстий размещения изделия. Это свойство устанавливается только при размещении любого размещения изделия на медный функциональный элемент.

Если установлен флажок Автоматически несколько монтажных отверстий, при размещении включается это свойство и выполняется привязка.

### Автоматическое обновление многократных пробоев

На унаследование монтажных отверстий могут воздействовать разные операции, которые изменяют медные функциональные элементы. Возможно их неправильное отображение. Если этот флажок установлен, то обновление производится после каждого изменения монтажных отверстий автоматически.

!!! note "Замечание:"

    Использование настройки Автоматическое обновление многократных пробоев может привести к снижению рабочей скорости в проектах с многочисленными медными функциональными элементами и многочисленными унаследованными пробоями. В таком случае рекомендуется отключить эту настройку и при необходимости провести обновление вручную через меню Обработать > Графика > Обновление многократных пробоев.

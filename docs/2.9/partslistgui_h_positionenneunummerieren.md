# Нумеровать позиции заново

Условие:

Вы открыли проект. В проекте содержатся устройства, которым присвоены изделия.

1. Выберите пункты меню Данные проекта > Устройства / изделия > Навигатор спецификаций.
2. В диалоговом окне Спецификация — <Имя проекта> выделите нумеруемые позиции.
3. Выберите пункт всплывающего меню Задать номер позиций.
4. В диалоговом окне [Нумеровать позиции](partslistgui_d_positionsnummerierung.md) задайте критерии для указания номеров позиций.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Если флажок Предварительный просмотр результата ***не*** установлен, новая нумерация копируется непосредственно в спецификацию. Иначе открывается диалоговое окно Нумеровать позиции: Предв. просмотр результата.

6. Обработайте в диалоговом окне Нумеровать позиции: Предв. просмотр результата необходимые номера позиций в столбце Цель.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Новая нумерация копируется в спецификацию.

!!! note "Замечание:"

    Если для одной функции присвоены несколько изделий, свойство Список номеров позиций (ид. 20346) отображает номера позиций всех изделий через запятую. С помощью этого свойства номера позиций изделий можно просматривать на функциях в графическом редакторе или выводить их в отчетах.

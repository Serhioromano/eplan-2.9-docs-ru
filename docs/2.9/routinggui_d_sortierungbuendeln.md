# Диалоговое окно Сортировка (Объединить)

Сгенерированы маршрутизируемые соединения. В навигаторе соединений выделены маршрутизируемые соединения. Выберите пункты меню Данные проекта > Соединения > Объединить. В диалоговом окне Объединить маршрутизируемые соединения щелкните в таблице в Схема объединенных проводов в ячейке по кнопке ++...++. В диалоговом окне Настройки: Объединить в поле Сортировка щелкните по кнопке ++...++.

В данном диалоговом окне определите схемы с критериями, по которым выполняется сортировка маршрутизируемых соединений в группы.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Категория / свойство

В этом списке отображаются критерии, по которым генерируются объединенные провода. Всем соединениям в одной группе присваивается одинаковый номер группы объединенных проводов. Для того, чтобы задать правила для генерирования группы объединенных проводов, можно комбинировать друг с другом несколько критериев. В этом случае в группу объединенных проводов объединяются те провода, для которых совпадают все критерии. Так, при комбинировании источника и цели учитываются оба критерия.

При нажатии кнопки ![](images/all_new_as.png){: .ui-icon } (Создать) отображается диалоговое окно Выбор критериев, в котором вы можете выбрать свойства из областей Источник, Цель и Соединение, по которым должна выполняться сортировка соединений в новую группу объединенных проводов.

В следующей таблице приведены примеры возможных критериев сортировки:

Критерий |  Значение
---|---
Предшествующий знак группирования для обозначения функц. элемента (автоматич.) |  За счет использования данного свойства в качестве критерия сортировки объединение проводов выполняется отдельно для каждого электрошкафа или каждой монтажной платы. Сначала происходит объединение первого электрошкафа из первого пространства листа, затем второго электрошкафа из первого пространства листа, далее остальных электрошкафов первого пространства листа, потом всех электрошкафов второго пространства листа и т. д. Если в пространствах листа присутствуют только монтажные платы, они используются как электрошкаф и включаются в анализ в такой же последовательности. Если электрошкафы и монтажные платы присутствуют в пространствах листа, сначала анализируются электрошкафы, а потом монтажные платы. Программа графически нумерует объединения проводов и сортирует в следующей последовательности:

* Сортировка по целям соединений.
* Соединения сортируются по первой цели каждого соединения. При наличии нескольких одинаковых позиций решающей является графическая позиция последней цели соединения.
* После этого соединения сортируются в объединении проводов по порядку.
* Объединения проводов нумеруются по порядку их возникновения.

ОУ, источник |  Все провода с одинаковым ОУ источника объединяются в одну группу проводов.
ОУ, цель |  Все провода с одинаковым ОУ цели объединяются в одну группу проводов.
Сеть |  Все соединения одной сети получают одинаковые номера объединенных проводов. Под "Сетью" понимают все выводы устройств, связанных друг с другом посредством электрических соединений. Использование одной этой настройки на автоматических группах объединенных проводов не приводит к оптимизации разводки. Невозможно поменять источник и цель соединения.
Цвет / номер |  Все провода с одинаковым свойством соединения Номер цвета объединяются в одну группу проводов.
Поперечное сечение / диаметр |  Все провода с одинаковым свойством соединения Поперечное сечение или Диаметр объединяются в одну группу проводов.
Группа объединенных проводов |  Все провода с одинаковым свойством соединения Группа объединенных проводов объединяются в одну группу проводов.
Соединение: обозначение типа |  Все провода с одинаковым свойством соединения Обозначение типа объединяются в одну группу проводов.
Номер изделия (главное изделие 1) |  Все провода с одинаковым свойством соединения Номер изделия объединяются в одну группу проводов.
Монтажная поверхность |  Все провода на монтажной поверхности, на которых расположены подключенные функции, объединяются в одну группу проводов.
Обозначение вывода устройства |  Все провода с одинаковой маркировкой выводов устройств на функциях, связанных при помощи соединения, объединяются в одну группу проводов.
Описание вывода устройства |  Все провода с одинаковым описанием выводов устройств на функциях, связанных при помощи соединения, объединяются в одну группу проводов.
Описание клемм / контактов штекера |  Все провода с одинаковым описанием клеммы / контакта штекера на функциях, связанных при помощи соединения, объединяются в одну группу проводов.

### По позиции вывода устройства

Для устройств, выводы которых расположены на двух или более уровнях выводов устройств друг над другом, в процессе производства разводка выполняется снизу вверх. Для этого нужно, чтобы провода на каждом уровне выводов устройств можно было объединить в одну группу объединенных проводов. Для выполнения этого требования можно воспользоваться следующими флажками в качестве настройки сортировки и объединения:

* Источник X, Цель X
* Источник Y, Цель Y
* Источник Z, Цель Z.

Эти настройки учитывают указания позиции X, позиции Y и позиции Z в глобальной и локальной схеме соединений для размещения изделия. Если поставлены эти флажки, сортировка проводов от выводов устройств с одинаковым указанием позиции и к ним выполняется в соответствии с общей группой объединенных проводов.

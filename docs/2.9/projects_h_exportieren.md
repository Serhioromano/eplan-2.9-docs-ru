# Экспорт проектов

Проекты можно экспортировать с помощью навигатора страниц или управления проектами. Экспортируемые проекты имеют формат, совместимый с XML, и затем (во внешнем приложении, например редакторе) их можно обрабатывать и снова импортировать обратно в EPLAN.

Условие:

Вы выделили проект. При экспорте с помощью навигатора страниц достаточно выделить страницу проекта в диалоговом окне Страницы — <имя проекта> или открыть страницу проекта в графическом редакторе.

Навигатор страниц: Проект > Организовать > Экспортировать.

Управление проектами: [Организовать] > Экспортировать.

1. Введите в диалоговом окне Экспортировать проект в место хранения экспортируемого проекта. Через пункт всплывающего меню Создать > Папка можно при необходимости создать новый каталог.
2. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выполняется экспорт проекта, указатель процесса показывает ход экспорта. В целевом каталоге сохраняется экспортированный проект *.epj в формате XML.

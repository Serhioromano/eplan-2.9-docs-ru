# Диалоговое окно Ввод координат

Открыта страница в графическом редакторе. Параметры > Ввод координат.

В этом диалоговом окне вводятся целевые координаты для установки курсора. Ввод выполняется относительно исходной точки использованной системы координат.

Обзор основных элементов диалогового окна:

### Текущая позиция курсора

При вызове диалогового окна здесь выводится текущая позиция курсора. Занесите целевые координаты в поля Xи Y.

### Система координат

В этом групповом поле по умолчанию через тип страницы выделяется присвоенная система координат. Систему координат можно выбрать временно, что влияет на положение исходной точки (начало координат).

* Графика: система координат графики начинается внизу слева. Значения координат задаются с единицей измерения "мм".
* Электротехника: система координат электротехники начинается в верхнем левом углу поверхности для черчения. Координатные значения выводятся в шагах сетки.
* Fluid-техника: система координат Fluid-техники начинается внизу слева. Координатные значения выводятся в шагах сетки.
* Технология производственных процессов: система координат технологии производственных процессов начинается внизу слева. Координатные значения выводятся в шагах сетки.

# Вкладка Слова не переводить

Сервисные программы > Перевод > Обработать словарь. Выберите вкладку Не переводить слова.

На этой вкладке вы можете указать слова, которые не требуется переводить, т. е., остаются без перевода во всех языках. При переводе эти слова будут игнорироваться. Настройки действительны только при пословном переводе.

Обзор основных элементов диалогового окна:

### Пoиск

Используя это поле, вы можете выполнять поиск всех слов, которые уже включены в список слов, не требующих перевода. Чтобы найти нужное слово, введите его в поле ввода.

### Слово, которое не переводится

В этом поле можно изменить слово, которое не требуется переводить.

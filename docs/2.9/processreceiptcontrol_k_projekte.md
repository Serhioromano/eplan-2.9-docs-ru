# Проект для сравнения и проект с замечаниями

В процессе [проверки проекта](processreceiptcontrol_k_start.md) сравниваются между собой два проекта: проект сравнения и проект с замечаниями.

###  Проект сравнения

Проект сравнения — это исходный проект или проект, который сгенерирован из исходного проекта без изменений. В нем содержится исходное состояние проекта, с этим проектом и будет сравниваться проект с замечаниями. В качестве проекта сравнения могут использоваться следующие проекты:

* стандартные проекты EPLAN (*.elk);
* обрабатываемые проекты EPLAN (*.elk, *.ell);
* выгруженные проекты EPLAN (*.els);
* Закрытые проекты EPLAN/Исходные проекты EPLAN (*.elr)
* временные проекты EPLAN для сравнения (*.elt);
* Исходные проекты EPLAN с отслеживанием изменений(*.ell).

### Проект с замечаниями

Проект с замечаниями — это проект, который вы хотите проверить. В нем содержится продвинутое состояние проекта Проверьте, удовлетворяет ли ваш собственный проект установленным стандартам или отвечает проект вашим требованиям после прохождения третьей стадии обработки.

# Диалоговое окно Настройки: Выбор изделия (проект)

Вы открыли проект.

* Параметры > Настройки > Проекты > "Имя проекта" > Управление > Выбор изделия.
* Вы выделили главную функцию в графическом редакторе или навигаторе. Обработать > Свойства. Выберите вкладку Изделие. [Настройки] > Выбор изделия (проект).

В этом диалоговом окне для всего проекта устанавливается, какой источник данных используется для выбора изделия.

Обзор основных элементов диалогового окна:

### Выбор изделия

Выберите одну из следующих опций:

* Определено пользователем: Для выбора изделия используются специфические для пользователя настройки, установленные в меню Параметры > Настройки > Пользователи > Управление > Выбор изделия.
* Проект: При выборе изделий предлагаются только такие изделия, которые сохранены в проекте. Они сохраняются в базе данных покупных изделий, которая генерируется из сохраненных изделий. После изменения данных изделий необходимо снова генерировать базу данных покупных изделий через [Обновление].
Учтите, что для этого изменения требуется установка приложений Microsoft Office полностью в 64-битной версии!

### [Обновить]

Щелкните по этой кнопке для повторного генерирования базы данных покупных изделий из сохраненных изделий.

### Предварительный выбор изделий

Выберите имеющуюся схему (т. е. список изделий) из раскрывающегося списка. Затем при выборе изделий предлагаются только те изделия, которые имеются в выбранном списке.

С помощью [...] откройте диалоговое окно Предварительный выбор изделий, где можно создать новый список изделий и сохранить его как схему или изменить имеющийся список.

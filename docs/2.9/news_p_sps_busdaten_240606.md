# Улучшения в диалоговом окне "Свойства" для выводов шины

В диалоговом окне "Свойства" выводов шины на вкладке Данные шины задаются свойства для вывода шины внутри системы шин. Эта вкладка стала более наглядной, и были реализованы различные улучшения.
– Поля ввода для логической сети перемещены в таблицу свойств
– Отображение дополнительной информации про систему шин
– Автоматический перенос проекта конфигураций из блока ПЛК.

### Поля ввода для логической сети перемещены в таблицу свойств

Поскольку указание логической сети требуется только для определенных систем шин, из вкладки были удалены соответствующие поля ввода. Вместо этого теперь в таблице свойств доступны свойства Логическая сеть: Имя (ид. 20414) и Логическая сеть: Вывод шины является главным (ид. 20310).

### Отображение дополнительной информации про систему шин

Под полем Система шин в информационной области отображается дополнительная информация про выбранную систему шин, которая имеет отношение к действиям при обмене данными ПЛК. При этом отображаются комбинации следующих записей:

* Неправильное соединение с конкретным портом: эти системы шин можно экспортировать с подробным соединением портов.
* Физическая сеть: эти системы шин формируют физическую сеть.
* Логическая сеть: для этих систем шин также экспортируется логическая сеть, указанная в свойстве Логическая сеть: Имя.

![](images/benefit.png)Эффект:

Дополнительно отображаемая информация позволяет определить при выборе системы шин, какие данные должны быть экспортированы во время обмена данными ПЛК и какие поля нужно заполнить дополнительно (например, Физическая сеть: Имя или Логическая сеть: Имя).

### Автоматический перенос проекта конфигураций из блока ПЛК

Для выводов шин теперь нет необходимости вводить проект конфигураций. Если поле Проект конфигураций пустое, проект конфигураций будет автоматически определен связанным блоком ПЛК (точнее, главной функцией). Для отображения на вкладке Данные шины доступно новое свойство Проект конфигурации (автоматически, на выводах шин) (ид. 20581). Данное свойство выводит введенный вручную проект конфигураций на выводе шины или, если этот проект конфигураций пуст, выводит проект конфигураций соответствующего блока (главная функция) ПЛК.

![](images/benefit.png)Эффект:

Автоматический перенос проекта конфигураций из блока ПЛК облегчает проектирование при копировании и вставке макроса. После копирования и вставки блока ПЛК с выводами шины больше нет необходимости изменять проект конфигураций на выводах шин после изменения проекта конфигураций на блоке ПЛК.

С помощью корректировки проекта можно удалить проект конфигураций, введенный для выводов шины, если он соответствует проекту конфигураций связанного блока ПЛК.

# Использование прямой обработки

С помощью прямой обработки можно включить режим обработки, который позволят напрямую изменять отдельные тексты. При этом могут обрабатываться как тексты свойств, так и графические тексты. Если включен режим обработки, то это выводится на панель состояния с помощью рамки обработки и над пиктограммой ![](images/inplaceeditinggui_icon_as.png){: .ui-icon }.

!!! note "Замечание:"

    Если включен режим Прямая обработка, с помощью двойного щелчка мышью по объекту с текстом, как правило, открывается не диалоговое окно свойств, а рамка обработки.

Условие:

Вы открыли страницу, форму, рамку или символ.

1. Параметры > Прямая обработка
2. Щелкните по тексту, который требуется обработать.

!!! info "Для сведения:"

    Текст выводится в рамке обработки.

3. Обработайте текст. Для дополнения, например, специальных текстов или новых строк используйте всплывающее меню (вызывается щелчком правой кнопки мыши в рамке обработки).
4. Нажмите клавишу ++Enter++ для сохранения изменений и закрытия поле обработки; нажмите клавишу ++Esc++ для отклонения изменений. При щелчке мышью за пределами рамки обработки измененный текст также сохраняется.
5. Если больше не требуется использовать прямую обработку, снова выберите пункты меню Параметры > Прямая обработка.

!!! tip "Совет:"

    * Если выполняется прямая обработка размещенных свойств условного обозначения, нажав клавишу ++Tab++, можно перейти от одного текста свойства к другому. При переходе выполняется сохранение измененного тексты и автоматически в рамке обработки открывается следующий текст свойства.
    * Прямую обработку также можно активировать без включенного режима обработки. Если вы выделили текст/условное обозначение или курсор находится точно на тексте, просто нажмите клавишу ++F2++ и обработайте отображенный текст. С помощью [Ввод] сохраняются изменения и рамка обработки снова закрывается. Работа системы аналогична MS Excel.
    * Дополнительно можно также активировать прямую обработку с помощью мыши. Это выполняется аналогично при переименовании имен файлов в Windows Explorer. Выделите в первую очередь текст или условное обозначение на странице, и пусть курсор остается на обрабатываемом тексте. При открытии рамки обработки удержите левую клавишу мыши короткое время, а затем отпустите.

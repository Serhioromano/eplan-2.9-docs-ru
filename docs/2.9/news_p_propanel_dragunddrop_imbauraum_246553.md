# Изменить иерархическую структуру в навигаторе пространства листа путем перетаскивания мышью

При создании основных данных в ***проекте макросов*** теперь можно изменить иерархическую структуру функциональных элементов в навигаторе пространства листа с помощью перетаскивания мышью. Это особенно актуально для электрошкафов и сопутствующих компонентов шкафа, входящих в стандартный объем поставки. Таким образом в иерархии функциональных компонентов можно, например, вместо электрошкафа присвоить дверь вертикальному перфорированному профилю справа по переднему краю.

![](images/benefit.png)Эффект:

При подготовке 3D-макросов иерархию функциональных элементов в электрошкафах теперь можно индивидуально настроить путем перетаскивания в навигаторе пространства листа.

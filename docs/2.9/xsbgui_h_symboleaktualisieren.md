# Обновить символы

Библиотека символов, сохраненная в проекте может быть изменена в любое время также вне этого проекта. В этом случае сохраненная в проекте старая библиотека символов меняется на измененную. При этом меняется только графика, функция остается неизменной.

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Сервисные программы > Основные данные > Синхронизировать текущий проект
2. В диалоговом окне Синхронизация основных данных — <Имя проекта> в меню Основные данные системы выберите библиотеки символов, которые вы изменили вне проекта. Возможен многократный выбор.
3. Щелкните ![](images/all_arrowleft_as.png){: .ui-icon } (Копировать влево).

!!! info "Для сведения:"

    EPLAN сравнивает библиотеки между собой и выводит на экран сообщение с результатом.

4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Основные данные проекта обновляются, т. е. переписываются основными данными системы.

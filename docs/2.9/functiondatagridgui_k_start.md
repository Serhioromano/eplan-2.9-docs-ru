# Табличная обработка

Табличная обработка позволяет совместно обрабатывать различные объекты на разных страницах проекта в одном диалоговом окне. Возможно следующее:

* ОУ выделенного устройства представляются наглядно в форме таблицы и могут обрабатываться поблочно — например, переименовываться и нумероваться.
* Обозначения устройства можно разбивать на столбцы, число и названия которых вы задаете самостоятельно. Таким образом, можно, например, обрабатывать структурные поля в Fluid-Технике.
* Свойства функций, соединений, макросов, точек разрыва, потенциалов, трубопроводов и сегментов из предварительного планирования отображаются в виде таблицы и доступны для поблочной обработки. Отдельные вкладки для этих разных объектов отображаются, только если выбор содержит несколько типов объектов.
* Данные объектов можно копировать и вставлять в Excel, обрабатывать там и снова вставлять в EPLAN.

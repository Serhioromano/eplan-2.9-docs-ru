# Управление основными данными

Открытие отдельных файлов основных данных системы для обработки или сохранения в проекте осуществляется через стандартное диалоговое окно, в котором откроется предварительно определенный каталог. Этот каталог настраивается для отдельных типов основных данных в меню Параметры > Настройки > Пользователи > Управление > Каталоги.

Собственно сохранение и управление зависят, в таком случае, от соответствующего типа основных данных, при этом различают управление следующими объектами:

* [Данные изделия](xmasterdatasettingsgui_h_artikeldatenverwalten.md)
* Формы
* Рамки
* [Макросы](xmasterdatasettingsgui_h_makrosverwalten.md)
* Библиотеки символов.

если настроенной формы / рамки и т.д. нет в пуле основных данных, ее можно найти и, в случае необходимости, автоматически сохранить при помощи функции Дополнить проект. если отсутствующие основные данные не могут быть дополнительно загружены из пула основных данных, появляется соответствующее сообщение.

# Резервуары и выводы резервуара на функциональной схеме автоматизации

На функциональной схеме автоматизации могут быть размещены резервуары и выводы резервуара. Для этого в программе предусмотрены пункты меню Вставить > Блок / вывод устройства / монтажная плата > Резервуар и ... > Вывод резервуара.

### Резервуары

Резервуар может быть представлен на функциональной схеме автоматизации в виде фиксированного символа без возможности масштабирования или в виде специального обозначения. Символы резервуаров находятся в библиотеке символов Special.slk.

В открывшемся диалоговом окне Выбор символа символы уже отфильтрованы: отображаются только символы с одинаковой категорией функции.

### Выводы резервуара

Резервуары могут быть дополнены выводами (фланцами). Если вывода резервуара размещены в месте размещения резервуара, они считаются принадлежащими ему в том случае, если точка вставки находится внутри окружающего прямоугольника. Для выводов резервуара на логической схеме выводов устройства по умолчанию задается тип вывода устройства "Технология производственных процессов".

# Обработать выводы устройств

Условие:

Вы открыли проект.

1. Сервисные программы > Основные данные > Символ > Открыть
2. Выберите пункты меню Сервисные программы > Нумеровать точки выводов устройства.
3. Задайте в групповом поле Последовательность, будут ли выводы устройства согласно схеме нумероваться "сверху вниз и затем слева направо" или "слева направо и затем сверху вниз".
4. Щелкните по кнопке ++OK++.
5. Выберите пункты меню Вставить > Вывод устройства "Направление".
6. Разместите вывод устройства.
Или, перед размещением, нажмите клавишу ++Ctrl++ и двигайте мышь для переключения направления вывода устройства.
Или нажмите клавишу ++Tab++, для вращения вывода устройства через все направления против часовой стрелки.
7. Введите в поле Номер вывода устройства диалогового окна Вывод устройства  номер вывода устройства (или используйте предустановку "следующий свободный номер").
8. Выберите из раскрывающегося списка желаемое направление вывода устройства.
9. Выберите в левой области списка одну или несколько записей для наборов свойств (обозначение и описание вывода устройства).
10. Задайте настройки для соответствующих текстов в полях правой области окна диалога.
11. Щелкните по кнопке ++OK++.

!!! warning "Предупреждение:"

    Несколько выводов устройства одного символа не должны располагаться друг над другом.

Данные выводы устройства принимаются для символа согласно своим настройкам. При этом графическое изображение вывода устройства однозначно описывает направление вывода устройства, однако оно не является составной частью готового символа.

!!! note "Замечание:"

    * Через пункты меню Вид > Свойства как числа могут отображаться тексты свойств (обозначение свойств, функциональный текст и т. п.), чтобы сделать представление более обзорным. Если вы распечатаете текущую страницу с этой настройкой или экспортируете в качестве файла PDF, на распечатке/в экспортированном файле PDF также будут выданы числа вместо текстов свойств.
    * Отображение и переключение наборов свойств Обозначение / описание вывода устройства выполняются через пункт меню Вид > Неповерн. обозначения вывода устройства или Вид > Поверн. обозначения вывода устройства.

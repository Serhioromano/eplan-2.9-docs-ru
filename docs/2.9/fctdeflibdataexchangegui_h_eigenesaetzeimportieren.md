# Импортировать собственные наборы буквенных обозначений, обозначения и описания выводов устройства

Наборы буквенных обозначений, обозначений выводов устройства и описаний выводов устройства можно импортировать из XML-файла, а затем использовать их для проектирования в EPLAN.

### Импортировать наборы буквенных обозначений

1. Сервисные программы > Основные данные > Буквенное обозначение
2. Выберите пункт всплывающего меню Импорт.
3. В диалоговом окне Импорт перейдите в требуемый каталог и введите имя файла.
4. Щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Наборы буквенных обозначений из XML-файла копируются в диалоговое окно.

### Импортировать обозначения и описания выводов устройства

1. Сервисные программы > Основные данные > Обозначение / описание вывода устройства
2. Выберите пункт всплывающего меню Импорт.
3. В диалоговом окне Импорт перейдите в требуемый каталог и введите имя файла.
4. Щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Все наборы обозначений и описаний выводов устройства копируются из XML-файла в диалоговое окно.

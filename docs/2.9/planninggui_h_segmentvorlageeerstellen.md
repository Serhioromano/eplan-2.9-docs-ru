# Создать и обработать шаблоны сегментов

Создавать шаблоны сегментов и управлять ими можно в навигаторе шаблонов сегментов.

### Создать шаблоны сегментов

1. Данные проекта > Предварительное планирование > Навигатор шаблонов сегментов
2. Выделите определение сегмента и выберите пункт всплывающего меню Новый шаблон сегмента.

!!! info "Для сведения:"

    Открывается диалоговое окно Свойства ++...++.

3. Введите в диалоговом окне Свойства ++...++ на вкладке Шаблон сегмента идентифицирующее имя и, при желании, описание для нового шаблона сегмента.
4. При необходимости присвойте шаблону сегмента дополнительные свойства на других вкладках. То, какие вкладки отображаются, зависит от определения сегмента.
5. При необходимости запишите на вкладке Изделие изделие. Шаблоны функции изделия позднее можно будет использовать для создания подробного планирования.
6. При необходимости запишите на вкладке Макрос макрос. Этот макрос позднее можно будет использовать при создании подробного планирования, как описано в разделе [Создать подробное планирование путем перетаскивания мышью](planninggui_h_makrosdragdrop.md).
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Будет создан шаблон сегмента с соответствующими свойствами.

!!! tip "Совет:"

    В представлении структуры дерева навигатора шаблонов сегментов можно переместить шаблоны сегментов с помощью перетаскивания в другое место внутри определения основного сегмента, если эти шаблоны еще не использованы в проекте. При этом шаблону сегмента автоматически присваивается определение сегмента, под которое он был перемещен. При перетаскивании удерживайте нажатой кнопку ++Ctrl++, так происходит копирование выделенных шаблонов сегментов.

### Создать шаблон сегмента из имеющегося сегмента

Условия:

* Вы открыли проект, который уже содержит сегменты.
* Вы открыли навигатор предварительного планирования и навигатор шаблонов сегментов.

1. Выделите сегмент в навигаторе предварительного планирования.
2. Перетащите сегмент в навигатор шаблонов сегментов.

!!! info "Для сведения:"

    Будет создан шаблон сегмента со свойствами выделенного сегмента.

!!! note "Замечание:"

    Шаблон сегмента не присваивается сегменту, из которого он был генерирован. При необходимости это присвоение нужно выполнить вручную.

### Обработать свойства шаблонов сегментов

1. Данные проекта > Предварительное планирование > Навигатор шаблонов сегментов
2. Выделите шаблон сегмента и выберите пункт всплывающего меню Свойства.
3. Обработайте в диалоговом окне Свойства ++...++ на вкладке Шаблон сегмента свойства отображения (Идентифицирующее имя и Описание) выбранного шаблона сегмента.
4. При необходимости присвойте шаблону сегмента дополнительные свойства на других вкладках или измените имеющиеся свойства. То, какие вкладки отображаются, зависит от определения сегмента.
5. Щелкните по кнопке ++OK++.

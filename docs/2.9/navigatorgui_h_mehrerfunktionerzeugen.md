# Автоматически генерировать функции с помощью схемы нумерации

В EPLAN можно автоматически генерировать несколько неразмещенных функций с помощью схемы нумерации. Эти функции могут использоваться для следующего:

* Клеммы
* Контакты штекеров
* Контакты катушки контактора, катушки реле (замыкающий, переключающий, размыкающий контакт)
* Выводы устройств и ПЛК
* Сборные шины.

В зависимости от выбранного определения функции, образец нумерации определяет количество новых функций и обозначения клемм, штекеров или выводов устройства. Образец нумерации 1a,1b,2a-8b позволяет создать, к примеру, 16 функций с обозначениями 1a,1b,2a,2b,3a,3b,4a,4b,5a,5b,6a,6b,7a,7b,8a,8b.

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Данные проекта > Устройство > Навигатор
2. Выделите имеющееся ОУ.
3. Выберите пункт всплывающего меню Новые функции.

!!! info "Для сведения:"

    Полное ОУ заносится в диалоговое окно Генерировать функции. Если нужно сгенерировать новое ОУ, введите в диалоговом окне Генерирование функций в поле Полное ОУ новое полное ОУ.

4. Рядом с полем Определение функции щелкните ++...++ и в открывшемся диалоговом окне выберите определение функции.

!!! info "Для сведения:"

    В поле Число выводов устройства автоматически заносится количество выводов устройства выбранного определения функции.

5. В поле Образец нумерации укажите обозначения выводов устройства для создания функций.

!!! info "Для сведения:"

    В поле Число новых функций, в зависимости от образца нумерации, автоматически отображается количество новых созданных функций.

6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Функции создаются в навигаторе устройств.

!!! tip "Совет:"

    Таким же образом можно создавать функции в навигаторе ПЛК, в навигаторе клеммников и в навигаторе штекеров.

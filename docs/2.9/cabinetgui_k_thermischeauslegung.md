#  Thermal Design Integration

***Thermal Design Integration***  является вспомогательным решением при тепловом расчете распределительных устройств:

* Виртуальное планирование и определение размеров энергоэффективных решений в области кондиционирования
* Подтверждение чертежей монтажных поверхностей.

***Thermal Design Integration*** предоставляет функциональные возможности, расширенные данные продуктов, своды правил и программные средства принятия решений, призванные оказать помощь планировщику, в том числе при утверждении результатов планирования:

* Расчет общей мощности потерь всех устройств
* Визуализация распределения тепловыделения для предотвращения появления особо горячих мест
* Расчет распределения тепловыделения для оптимального расположения компонентов в электрошкафу
* Интерфейс к Rittal Therm для расчета и определения размеров компонента системы кондиционирования
* Опционально: Использование расширенных данных изделий Rittal для решений в области кондиционирования через EPLAN Data Portal
* Опционально: Представление области с оптимальным кондиционированием для каждого компонента системы кондиционирования, в которой компоненты электрошкафа расположены в предпочтительном порядке
* Опционально: Зарезервированные зоны для воздушного потока для каждого компонента системы кондиционирования для оптимизации циркуляции холодного воздуха и защиты компонентов от перегрева или переохлаждения
* Опционально: Направление движения воздуха компонентов системы кондиционирования.

Благодаря комбинации таких средств планирования и принятия решений возможно выполнение следующих задач:

* Простое планирование и энергоэффективный тепловой расчет для решений в области кондиционирования
* Следование правилам выполнения чертежа монтажных поверхностей компонентов
* Стандартное проектирование согласно стандарту IEC 61439 "Обеспечение бесперебойной эксплуатации" машин и установок.

!!! note "Замечание:"

    Функции Направление движения воздуха, Области оптимального кондиционирования и Зарезервированные зоны для воздушного потока можно использовать только с расширенными данными изделий, которые можно получить у производителя Rittal через EPLAN Data Portal.

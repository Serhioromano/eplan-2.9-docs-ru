# Вывести отчет в другой проект

Отчет по данным так называемого "исходного проекта" находится в текущем открытом проекте, "проекте отчета". На вкладке Шаблоны проекта отчета укажите исходный проект, из которого взяты данные.
При последующем отчете исходный проект на момент составления отчета открыт для чтения на заднем плане, данные считываются, и в проекте отчета генерируются отчеты.

!!! note "Замечание:"

    * При генерировании отчетов в EPLAN, как правило, используются основные данные (формы, библиотеки символов и т. д.) и настройки текущего открытого проекта, т. е. проекта отчета.
    * Перед генерированием таблицы соединений в стандартном проекте соединения обновляются автоматически. При выводе таблицы соединений в проект отчета соединения ***не обновляются автоматически***! Если в проекте отчета должна выводиться также текущая таблица соединений исходного проекта, то необходимо предварительно обновить соединения в соответствующем исходном проекте.

Условия:

* Вы открыли и выделили проект.
* Требуемому типу отчета присвоена форма.
* В диалоговом окне Настройки: Вывод в страницах заданы все необходимые настройки.

1. Выберите следующие пункты меню: Сервисные программы > Отчеты > Генерировать
2. Щелкните в диалоговом окне Отчеты — <Имя проекта> по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
3. В поле Форма вывода диалогового окна Определить отчет выберите форму вывода "Страница".
4. Нажмите рядом с полем Исходный проект кнопку ++...++.
5. Чтобы выбрать другой проект в качестве исходного проекта, выделите проект и нажмите ++Открыть++.
6. Выделите в диалоговом окне Определить отчет тип отчета и нажмите ++OK++.
7. Щелкните в диалоговом окне Настройки — <Тип отчета> по кнопке ++OK++.
8. Задайте в следующем диалоговом окне начальную страницу, щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Отчет будет создан.

1. Выберите в диалоговом окне Отчеты — <Имя проекта> вкладку Шаблоны.
2. В иерархической структуре дерева перейдите к шаблонам, которым необходимо присвоить исходный проект.
3. В правой части диалогового окна щелкните в столбце Значение свойства Исходный проект по ++...++.
4. Выделите проект, который должен быть присвоен в качестве исходного проекта, и щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Проект скопируется в столбец Значение свойства Исходный проект и будет присвоен шаблонам.

1. Выберите в диалоговом окне Отчеты — <Имя проекта> вкладку Шаблоны.
2. Выделите проект, который следует установить для исходного проекта, и выберите пункт всплывающего меню Исходный проект > Выбрать исходный проект.
3. Выделите проект, который должен быть присвоен в качестве исходного проекта, и щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Проект скопируется в столбец Значение свойства Исходный проект для всех шаблонов и будет им присвоен.

1. Выберите в диалоговом окне Отчеты — <Имя проекта> вкладку Шаблоны.
2. Выделите в дереве вкладки Шаблоны определенный уровень структуры дерева (имя проекта, тип отчета, например, "Спецификация изделий" и т. д.) и выберите пункты всплывающего меню Исходный проект > Удалить записанный исходный проект.

!!! info "Для сведения:"

    Проект будет удален из столбца Значение свойства Исходный проект для всех шаблонов выбранного уровня структуры дерева.

# Вкладка Технологические перемычки (экспорт Rittal - Perforex LC / Secarex)

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт Rittal - Perforex LC / Secarex. Выберите вкладку Технологические перемычки.
* Сервисные программы > Данные изготовления > Механическая обработка > Rittal - Perforex LC / Secarex. Рядом с полем Машина нажмите ++...++. Выберите вкладку Технологические перемычки.

Чтобы избежать повреждений из-за падения больших фрезерованных прямоугольных поверхностей, эти поверхности можно разделить при помощи технологических перемычек. В результате этого образуются фрезерные области меньшего размера. Возникающие технологические перемычки при монтаже выламываются. На этой вкладке задаются все характеристики для генерирования технологических перемычек.

Обзор основных элементов диалогового окна:

Групповое поле 'Настройки для стандартных контуров'

Используйте стандартные настройки или измените значения для отдельных настроек в соответствии с вашими требованиями.

### Ширина технологической перемычки

По умолчанию ширина технологической перемычки составляет 0,1 мм.

### Высота технологической перемычки

По умолчанию высота технологической перемычки составляет 2 мм.

### Генерировать одну технологическую перемычку, если длина ребра превышает

По умолчанию технологическая перемычка добавляется при фрезеровочной длине свыше 100 мм.

### Генерировать две технологические перемычки, если длина ребра превышает

По умолчанию две технологические перемычки добавляются при фрезеровочной длине свыше 250 мм.

### Генерировать три технологические перемычки, если длина ребра превышает

По умолчанию три технологические перемычки добавляются при фрезеровочной длине свыше 500 мм.

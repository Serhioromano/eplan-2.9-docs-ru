# Диалоговое окно Экспорт DXF, медь

Вы открыли проект. В пространстве листа или в навигаторе пространства листа выделены медные функц. элементы, для которых необходим экспорт ЧУ. Сервисные программы > Данные изготовления > Механическая обработка > DXF, медь.

В этом диалоговом окне устанавливаются настройки для графического экспорта развертки медных функц. элементов в формате 'DXF'. Передаются следующие данные:

* Внешние измерения медной шины в развернутом состоянии, представленные в виде прямоугольника (ширина х общая длина в развертке)
* Линии изгиба
* Концы изгиба
* Примечания для сгибания
* Все механические монтажные отверстия.

Обзор основных элементов диалогового окна:

### Машина

В комплекте поставки содержится определение машины с названием "Стандарт". Если в настройках фирмы определены другие схемы, можно выбрать из раскрывающегося списка другие определения машины.

Кнопка ++...++ открывает диалоговое окно Настройки: Экспорт DXF, медь, в котором можно изменить настройки схемы или определить новую схему.

### Каталог вывода

В этом поле выводится место хранения. Здесь по умолчанию выбран каталог, который определен в пунктах меню Настройки > Фирма > Механическая обработка для экспорта DXF, медь.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Целевой файл

Это поле активно только тогда, когда выбор содержит медные функц. элементы, для которых существует развертка. Здесь вводится название генерируемого файла DXF или выбирается с помощью клавиши ++...++ в диалоговом окне Открыть файл типа DXF, медь (*.dxf). При наличии нескольких файлов поле деактивируется и действуют условия создания названий файлов и подкаталогов, определенные в схеме.

### Применить ко всему проекту

Если этот флажок установлен, текущее выделение распространяется на все развернутые медные функц. элементы проекта.

Если флажок снят, экспортируются только развертки выделенных функциональных элементов.

Если отдельный проект уже выделен (и при этом фильтр не установлен), флажок автоматически включается и становится недоступным. В этом случае настройку больше невозможно изменить.

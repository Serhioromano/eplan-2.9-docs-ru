# Вкладка "Комментарий"

Вы открыли проект, который содержит импортированные возможности PPE-комментарии.

* Страница > Навигатор. Выделите страницу и выберите пункты меню Страница > Открыть. Щелкните дважды по комментарию на странице.
* Страница > Навигатор комментариев. Выделите комментарий. Всплывающее меню > Перейти к (графика). Щелкните дважды комментарий на странице или выделите комментарий и выберите всплывающее меню > Свойства.

На этой вкладке отображаются свойства текущего комментария, как, например, текст комментария и статус комментария.

Доступны следующие элементы диалогового окна:

### Автор

Показывает имя пользователя в Windows, который создал PDF-комментарий.

### Дата создания

Показывает, когда PDF-комментарий был создан автором.

### Дата изменения

Показывает, когда PDF-комментарий был изменен автором в последний раз.

### Текст комментария

Отображает текст комментария, который автор внес в файл PDF в поле комментария.

### Статус

Отображает статус комментария, который автор задал в файле PDF в поле комментария Параметры > Статус. Вы можете изменить этот статус.

### Ответ(ы)

Отображает ответные комментарии, которые автор ввел в файл PDF в одном или нескольких полях ответов поля комментария с помощью Параметры > Ответы.

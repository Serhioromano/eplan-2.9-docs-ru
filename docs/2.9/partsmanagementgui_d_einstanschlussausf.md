# Диалоговое окно Настройки: Категория соединения

Параметры > Настройки > Фирма > Управление > Категория соединения.

В этом диалоговом поле задаются категории подключения, которые по умолчанию предлагаются и используются в базе данных изделий при управлении схемами соединений.

Обзор основных элементов диалогового окна:

### Таблица

В столбце Номер выводится порядковый номер, задающий последовательность, в которой отображаются категории подключения в раскрывающемся списке в базе данных изделий. В разделе Категория соединения (стандарт) отображаются предварительно определенные имена этих данных. Обозначенные серым цветом записи заданы EPLAN и недоступны для обработки; все остальные обозначения для категорий соединения можно присваивать и при необходимости снова удалять.

Перечисленные ниже категории соединений соответствуют классификации eCl@ass и должны использоваться при создании новых схем соединений:

* Винтовое подключение
* Пружинная клемма
* Вставное соединение
* Сквозное соединение
* Подключение со съемом изоляции
* Штепсельное соединение
* Соединение с клеммой штепсельного типа
* Болтовое соединение
* Соединение с помощью плоского штекера
* Беспаечный контакт
* Ленточный вывод
* Шинное соединение
* Лепестковый вывод для припайки
* Соединение, полученное методом накрутки

Перечисленные ниже категории соединений все еще доступны по причинам совместимости, но больше не должны использоваться в будущем при создании новых схем соединений:

* Одиночное подключение с винтовым зажимом
* Двойное подключение с винтовым зажимом
* Подключение с пружинным зажимом
* Винтовое подключение (кольцевой или U-образный кабельный наконечник)
* Штекерное подключение (кабельный наконечник)
* Штекерное подключение
* Подключение пайкой
* Разъемное подключение (Fluid)
* Ниппель
* Накидная гайка
* Штуцерное соединение с врезным кольцом

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

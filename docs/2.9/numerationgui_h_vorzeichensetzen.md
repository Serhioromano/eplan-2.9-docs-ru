# Установить предшествующий знак в видимом ОУ

Видимое ОУ устройств может выводиться с предшествующим знаком, например -M1, или без предшествующего знака, например M1. Можно раздельно установить эти настройки для всех групп устройств. Группы устройств, например "Устройства общего назначения", "Клеммники", "Штекеры", "Черные ящики".

Условие:

Вы открыли проект.

1. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > ОУ
2. Выберите в диалоговом окне Настройки: ОУ в групповом поле Установить предш. знак флажок группы устройств, которые должны быть представлены видимым ОУ с предшествующим знаком.
3. Щелкните по кнопке ++ОК++.

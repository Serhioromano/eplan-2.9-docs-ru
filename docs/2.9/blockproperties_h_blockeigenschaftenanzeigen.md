# Просмотреть свойства блока

Свойства блока можно просмотреть в схеме соединений и вывести в формы.

### Просмотр свойств блока в схеме соединений

Условия:

* Вы открыли проект.
* Вы открыли диалоговое окно свойств для функции (например, дважды щелкнув условное обозначение в графическом редакторе).

1. Выберите в диалоговом окне Свойства ++...++ вкладку <Категория функции>.
2. Щелкните на панели инструментов над полем Свойства по ![](images/all_new_as.png){: .ui-icon } (Создать).
3. В диалоговом окне Выбор свойства выберите свойство блока с уже определенным свойством формата и щелкните по ++OK++.

!!! tip "Совет:"

    Выделить несколько свойств одновременно можно, удерживая нажатой клавишу ++Ctrl++.

4. Перейдите ко вкладке Отображение и при необходимости измените настройки для просмотра свойства блока.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Свойство блока будет отображаться согласно настройкам в схеме соединений и содержать значения свойств, определенных в соответствующем свойстве формата.

!!! note "Замечание:"

    Чтобы свойства блока отображались в проекте макросов на страницах проекта, необходимо установить соответствующий режим отображения в диалоговом окне [Настройки: Представление в проектах макросов](settingsmastergui_d_darstellungmakroprojekten.md).

### Вывод свойств блока в формы

Условия:

* Вы открыли проект.
* В редакторе форм открыта форма.
* Для функций, анализирующихся в форме, уже определены свойства блока (определены соответствующие свойства блока для символа или для проекта).

1. Вставить > Текст заполнитель
2. В диалоговом окне Свойства - Текст заполнитель на вкладке Размещение щелкните по ++...++.
3. В диалоговом окне Тексты заполнителей — <Тип формы> выберите нужное свойство блока (Свойство блока [n]) и нажмите ++OK++.
4. Щелкните в диалоговом окне Свойства - Текст заполнитель по ++OK++.
5. Разместите свойство в форме.
6. Чтобы при необходимости разместить в форме другие заполнители, повторите приведенные выше шаги.

!!! info "Для сведения:"

    Если форма используется для отчета, заполнители для свойств блока будут заполняться значениями свойств, определенных в соответствующих свойствах формата.

# Операция: restore

**Параметр** |  **Описание**
---|---
TYPE |  Вид выполняемой задачи:
PROJECT: Восстановление всего проекта
MASTERDATA: Восстановление основных данных
PROJECTNAME |  Имя проекта с полным путем к файлу. Действительно только для TYPE = PROJECT.
ARCHIVENAME |  Имя архива. Имя восстанавливаемого архива.
DESTINATIONPATH |  Целевой каталог. Путь файла, по которому следует восстанавливать проекты или данные.
Действительно только для TYPE = MASTERDATA.
UNPACKPROJECT |  Указывает, следует ли распаковывать предварительно запакованный проект (необязательно, Да = 1, Нет = 0).
Значение данного маркера следует устанавливать на 1 только в том случае, если нужно распаковывать предварительно запакованные проекты.
Значение по умолчанию = 0.
Действительно только для TYPE = PROJECT.
MODE  |  Указывает, как следует восстанавливать файлы, содержащиеся в резервировании данных (необязательно, стандарт (восстановление без ProjectInfo.xml) = 0, ProjectInfo.xml при восстановлении с учетом = 1).
Значение по умолчанию = 0.
Действительно только для TYPE = PROJECT.

!!! example "Пример:"

    Восстановить проект:restore
    /TYPE:PROJECT
    /ARCHIVENAME:U:\temp\my_prj.zw1
    /PROJECTNAME:U:\temp\w3_proj.elk
    /UNPACKPROJECT:0
    /MODE:1Восстановить основные данные (рамки и т.д.) в целевом каталоге:restore
    /TYPE:MASTERDATA
    /ARCHIVENAME:U:\temp\SSHmy_MasterData.zw2
    /DESTINATIONPATH:U:\temp

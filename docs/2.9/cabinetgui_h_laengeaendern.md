# Изменение длины объектов в пространстве листа

Длину функциональных элементов с переменной длиной, размещенных в пространстве листа, можно изменять позднее. Возможно изменение длины только для отдельного объекта. Выбор нескольких объектов невозможен. Для задания новой длины можно использовать различные формы ввода:

* Свободный ввод точки посредством щелчка клавиши мыши
* Захват точки проекции в другом объекте
* Ввод положительного или отрицательного значения в область ввода данных.

Условия:

* Вы открыли проект.
* Навигатор пространства листа открыт, и открыто пространство листа.
* Пространство листа содержит как минимум один регулируемый по длине функциональный элемент.

1. Выберите пункты меню Обработать > Графика > Изменить длину.
2. Щелкните по тому краю изменяемого объекта, к которому следует применить изменение.

!!! info "Для сведения:"

    Объект будет отображен на текущей позиции курсора. Рядом с курсором на выделенном крае появится точка захвата красного цвета. Выделенный щелчком мыши конец объекта можно перемещать курсором в обоих направлениях. 3D-точки захвата монтажной платы отображаются синим цветом; точки захвата других компонентов появляются в пространстве листа при затрагивании их курсором.

1. Переместите курсор на точку, до которой необходимо удлинить или укоротить объект.

!!! info "Для сведения:"

    Представление объекта следует за курсором.

2. Определите новую конечную точку объекта.

1. Переместите курсор в область рядом с одной из обозначенных точек захвата 3D. Точки захвата на других подобных объектах появляются при наведении на них курсора. Без отображения точки захвата будут найдены также ребра монтажной платы.

!!! info "Для сведения:"

    Курсор будет зафиксирован в найденной точке захвата или на ребре. Вокруг красной точки захвата с курсором появится красный квадрат.

2. Определите новую конечную точку объекта.

1. Щелкните по тому краю изменяемого объекта, к которому следует применить изменение.
2. Введите в область ввода данных значение, на которое длину объекта следует увеличить или уменьшить. Значения без предшествующего знака либо со знаком "+" означают удлинение; значения с предшествующим знаком "-" означают уменьшение длины.
3. Подтвердите ввод значений.

!!! example "Пример:"

    Изменение длины посредством различных форм ввода и результат Форма ввода Ввод Результат Щелчок мыши Точка проекции Область ввода данных

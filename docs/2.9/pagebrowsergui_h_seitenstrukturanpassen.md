# Адаптировать структуру страниц

При копировании страницы или вставке макроса можно изменить структурные идентификаторы, номера страниц и описания страниц перед вставкой страницы.

Далее описан процесс копирования страниц. При вставке макросов страниц таким же образом можно адаптировать и структуру страниц.

Условие:

Выделите одну или несколько страниц в диалоговом окне навигатора Страницы — <Имя проекта>.

1. Выберите пункты меню Обработать > Копировать.
2. Щелкните в диалоговом окне навигатора Страницы — <Имя проекта> по тому месту, куда требуется вставить скопированные страницы, и выберите пункты меню Обработать > Вставить.
3. В диалоговом окне Адаптировать структуру на вкладке Страницы измените номер страницы, описание страницы и / или структурные идентификаторы. Номер страницы можно ввести вручную или выполнить новую нумерацию в [Нумеровать]. При помощи флажка Автоматическое имя страницы можно также определять следующие по порядку наивысшие номера страниц (относительно структуры).
4. Нажмите на ++Сбросить++, если исходные структурные идентификаторы необходимо оставить без изменения (например, при вставке макросов нескольких страниц).
5. Для изменения структурных идентификаторов щелкните по соответствующему полю в таблице. Можно ввести (новые) идентификаторы вручную или вызвать с помощью кнопки ++...++ диалоговое окно выбора. В диалоговом окне выбора содержатся все идентификаторы, имеющиеся в целевом проекте.
6. Чтобы изменять структурные идентификаторы в большом количестве (например, переименовать главные идентификаторы нескольких страниц или ОУ), выберите пункт [Дополнительно] > Обработать структуры; затем в открывшемся окне Обработать структуры можно задать / обработать главные идентификаторы и до 9 подидентификаторов для отдельных блоков идентификаторов.
7. Установите флажок Перезаписать, если требуется перезаписать одноименные страницы в целевом проекте.
8. Выберите вкладку ОУ, если скопированные ОУ содержат структурные идентификаторы.
9. При необходимости измените видимое ОУ на вкладке ОУ.
10. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Если изменения не выполнялись, то в целевом проекте будут созданы структурные идентификаторы исходных страниц.

!!! info "Для сведения:"

    Если в целевом проекте содержатся страницы с такими же именами, которые могут быть перезаписаны, выводится соответствующий запрос. Щелкните по кнопке ++Да++, чтобы перезаписать страницы в целевом проекте. Щелкните по кнопке ++Нет++ и вернитесь к диалоговому окну Адаптировать структуру, где можно изменить введенные данные.

!!! info "Для сведения:"

    Если в пункте меню Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация (online) Вы выбрали опцию С запросом, будет выведено диалоговое окно Режим вставки. Здесь можно определить, каким образом должна быть выполнена нумерация ОУ на скопированных страницах.

11. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Страницы вставляются в целевой проект, а структурные идентификаторы и ОУ изменяются в соответствии с введенными данными.

### Поведение структурных идентификаторов при копировании

При копировании страниц проекта в другой проект с другой (более плоской) структурой убираются все структурные идентификаторы, которые в целевом проекте определены как "Недоступно". Структурные идентификаторы, настроенные в целевом проекте как "Описывающ.", не изменяются.

!!! example "Пример:"

    При копировании страниц из проекта с блоками идентификаторов "Установка" и "Место установки" в проект с общей нумерацией эти структурные идентификаторы удаляются из структуры проекта.При копировании из проекта со структурными идентификаторами "Установка" и "Место установки" в проект, в котором используется только блок идентификаторов "Установка" (без описывающего места установки),место установкиудаляется из свойств страницы.

Если из проекта со структурными идентификаторами (например, с блоками идентификаторов "Установка" и "Место установки") выполняется копирование страниц в проект со структурой страниц, в которой используется схема идентификации "установка идентифицирующая" и "место установки описывающее", можно пронумеровать эти страницы заново. Без этой нумерации имена страниц могут быть неоднозначными, так как идентифицирующая часть убирается.

# Групповое поле Свойства

Открыто диалоговое окно Свойства ++...++. Первая вкладка находится на переднем плане.

В этом групповом поле указываются дополнительные свойства объекта.

Обзор основных элементов диалогового окна:

### Категория

В зависимости от выбранной категории в диалоговом окне Конфигурировать свойства заданные свойства отображаются в таблице.

!!! example "Пример:"

    В категории "Устройство" среди прочих доступны следующие свойства:ОУ (идентифицирующее):символьная строка, защищенная от записи Установка (Главный ид.):символьная строка, защищенная от записи В категории "Настройки" среди прочих доступны следующие свойства:Отображение перекрестных ссылок:раскрывающийся список Отображение перекрестных ссылок: Количество строчек / колонок:целочисленное значение Отображение перекрестных ссылок: Представление:раскрывающийся список

### Панель инструментов

Кнопка |  Значение
---|---
![](images/all_new_as.png){: .ui-icon } (Создать) |  Открывает диалоговое окно Выбор свойств, в котором можно выбрать другие свойства и скопировать их в список. Вставленные таким образом свойства добавляются в конце списка.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Выделенные свойства убираются из списка. При этом значения свойств не удаляются. Свойства, которым присвоены значения, в автоматическом режиме снова отображаются в таблице при следующем открытии диалогового окна.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

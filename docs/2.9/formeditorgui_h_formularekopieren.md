# Копировать формы

Чтобы копировать форму со всем ее содержимым, сохраните ее в другом месте или под другим именем.

Условие:

Вы открыли проект.

1. Сервисные программы > Основные данные > Форма > Копировать
2. В диалоговом окне Копировать форму выберите дисковод и каталог копируемой формы.
3. Выберите копируемый файл формы. (Многократный выбор здесь ***невозможен***.)
4. Щелкните по кнопке ++Открыть++.
5. В диалоговом окне Создать форму установите место хранения для копии формы.
6. Введите в поле Имя файла информативное имя для новой формы.
7. Щелкните по кнопке ++Сохранить++.

!!! info "Для сведения:"

    Форма копируется, и копия открывается для обработки.

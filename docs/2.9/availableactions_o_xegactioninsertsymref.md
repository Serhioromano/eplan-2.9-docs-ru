# Операция: XEGActionInsertSymRef

**Параметр** |  **Описание**
---|---
SymbolLibName  |  Имя библиотеки символов, в которой содержится символ
SymbolId |  Номер символа, который следует вставить
VariantId  |  Номер варианта, если у символа есть варианты
FctDefTag  |  Идентификатор определения функции для поиска вставляемого символа
Placementmode  |  Идентификатор вида представления для поиска вставляемого символа. Строка, которая определяет вид представление символа. Возможные значения:

* all: Многополюсный
* single: Однополюсный
* paircrossreference: Парная перекрестная ссылка
* overview: Обзор
* graphics: Графика
* processandinstdiagram: Функциональная схема автоматизации
* cabling: Топология
* functional: Функциональный.

SymbolType  |  Идентификатор типа символа для поиска вставляемого символа
CustomSymbols  |  Имя специфической для проекта настройки для загрузки вставляемого символа клиента.

!!! note "Замечание:"

    Данная операция использует главное окно. Если указанный символ не найден, откроется диалоговое окно Выбор символа.

!!! example "Пример:"

    XEGActionInsertSymRef
    /SymbolLibName:IEC_symbol
    /SymbolId:55
    /VariantId:0

# Вкладка Маркеры удаления

Вы открыли проект, содержащий маркеры удаления, и находитесь на странице схем соединений в графическом редакторе. Вы дважды щелкнули по маркеру удаления (или выбрали всплывающее меню Свойства; в этом случае могут быть выделены также несколько маркеров удаления). В диалоговом окне Свойства ++...++ выберите вкладку Маркер удаления.

На этой вкладке отображаются свойства удаленного объекта. По умолчанию маркер удаления размещается на слое EPLAN576, графика. Удаленные объекты.

Обзор основных элементов диалогового окна:

### Удаленный объект

В этом поле указывается имя удаленного объекта в момент удаления, например ОУ в случае функции или имя страницы в случае страницы.

### Тип

В этом поле отображается тип удаленного объекта, например Функция, Обозначение местоположения, Страница и т. д.

### Имя пользователя

В этом поле отображается имя пользователя, который удалил объект.

### Дата удаления

В этом поле отображается момент (дата и время), когда был удален объект.

!!! note "Замечание:"

    При помощи специальных свойств можно охватить объекты, которые были размещены на том же месте, что и маркер удаления и оказались удалены вместе с ним. Свойства других удаленных объектов можно просмотреть через свойства Удаленный объект (доп.): Имя, Удаленный объект (доп.): Тип, Имя пользователя (доп.) и Дата удаления (доп.). Отдельные объекты можно выбрать при помощи индекса, то есть свойства второго удаленного объекта отображаются с индексом [1], свойства третьего удаленного объекта - с индексом [2] и т. д. Объект, удаленный первым, имеет наибольшее значение индекса.

!!! example "Пример:"

    На одном и том же месте в схеме соединений были последовательно размещены, а затем вновь удалены три условныхобозначения. На маркере удаления могут отображаться свойства всех трех объектов: Имя объекта, удаленного последним, отображается в поле Удаленный объект. Свойство Удаленный объект (доп.): Имявыводит с индексом [1] имя второго удаленного объекта и с индексом [2] — имя объекта, удаленного первым.

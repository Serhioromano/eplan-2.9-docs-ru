# Отобразить интервалы установки

В базе данных изделий в свойстве изделия на вкладке Монтажные данные для интервала установки можно указать ширину, высоту и глубину. Данный интервал установки обеспечивает соблюдение допустимой тепловой нагрузки на изделия, расположенные рядом друг с другом или друг над другом.

В целях проверки при размещении можно отобразить заданные интервалы установки. Они представляются в виде прозрачных окружающих объектов. Точку захвата размещаемого изделия в опциях размещения можно указать со ссылкой на ребро функционального элемента (стандартная настройка) или на интервалы установки.

Условия:

* Вы открыли проект.
* Навигатор пространства листов открыт, и одно пространство листов открыто.
* Пространство листа содержит размещения изделия.

1. Выберите пункты меню Вид > Интервалы установки.

!!! info "Для сведения:"

    Вокруг размещенных изделий, для которых задан интервал установки, будут отображены прозрачные объекты.

!!! example "Пример:"

    Выбор Результат Интервалы установки отображены Интервалы установки скрыты

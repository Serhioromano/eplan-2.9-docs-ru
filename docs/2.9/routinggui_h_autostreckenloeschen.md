# Удалить автоматические сегменты маршрутизации

При создании сети соединенных сегментов автоматические сегменты маршрутизации генерируются внутри кабельных каналов, на концах и в качестве перемычек в зазорах между кабельными каналами. Эти автоматические сегменты маршрутизации нельзя выделить для обработки. Если вы хотите, чтобы трасса маршрутизации определялась исключительно сегментами маршрутизации, размещенными вручную, автоматические сегменты маршрутизации можно удалить из сети соединенных сегментов перед началом маршрутизации.

Условие:

Вы открыли проект. Открыто одно пространство листа. Вы рассчитали сеть соединенных сегментов в открытом пространстве листа.

1. Выберите пункты меню Данные проекта > Соединения > Удалить автоматические сегменты маршрутизации.

!!! info "Для сведения:"

    Все автоматические сегменты маршрутизации кабельных каналов будут удалены из открытого пространства листа.

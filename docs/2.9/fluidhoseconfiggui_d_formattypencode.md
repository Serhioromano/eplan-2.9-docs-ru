# Диалоговое окно Формат: свойство

У вас установлен конфигуратор шлангопроводов в качестве дополнительного модуля для EPLAN Fluid. Вы запустили EPLAN Fluid и конфигуратор шлангопроводов.

* Выберите в EPLAN Fluid пункты меню Параметры > Настройки > Пользователь > Add-ons > Конфигуратор шлангопроводов. Переместите вкладку Общее на передний план и щелкните рядом с полем Формат кода типа по кнопке ++...++.
* Или выберите в диалоговом окне Конфигурировать шлангопроводы конфигуратора шлангопроводов пункты меню Дополнительно > Настройки. Переместите вкладку Общее на передний план и щелкните рядом с полем Формат кода типа по кнопке ++...++.

Или у вас установлен конфигуратор шлангопроводов в качестве самостоятельной программы, и программа запущена.

* Выберите в меню конфигуратора шлангопроводов пункты Параметры > Настройки > Пользователь > Add-ons. Переместите вкладку Общее на передний план и щелкните рядом с полем Формат кода типа по кнопке ++...++.

На этой вкладке укажите формат отображения для кода типа шлангопроводов.

Обзор основных элементов диалогового окна:

### Доступные элементы формата (список)

В данном списке отображаются все доступные элементы формата, которые могут быть составной частью формата отображения кода типа. Чтобы использовать элемент формата для формата отображения, его необходимо скопировать с помощью кнопки ![](images/all_arrowright_as.png){: .ui-icon } (Переместить вправо) в поле Выбранные элементы формата. При этом откроется диалоговое окно Выбор свойств, в котором нужно выбрать свойство.

В распоряжении имеются следующие элементы формата и свойства:

Доступные элементы формата |  Имя выбираемого свойства |  Значение
---|---|---
Свойства |  Кабель / Группа соединений: Поперечное сечение/диаметр |  Показывает выбранный номинальный диаметр шланга (без единицы измерения).
Кабель / Группа соединений: Тип |  Показывает выбранный тип шланга.
Длина шлангопровода |  Показывает выбранную длину шлангопровода (без единицы измерения).
Вид изделия [1]-[50]: Для арматур доступны виды изделия [1] и [2]. |  ***Вид изделия [1]*** / ***Вид изделия [2]*** указывает выбранный тип конструкции арматуры, монтируемой на шланг слева / справа.
Угол кручения |  Показывает выбранный угол кручения использованной арматуры (без единицы измерения).
Разделитель |  Разделитель |  Разделяет элементы формата друг от друга.

### Выбранные элементы формата (таблица)

В этой таблице отображаются все элементы формата, из которых состоит текущий код типа.

Используйте кнопки панели инструментов над таблицей, чтобы обрабатывать элементы формата, менять их последовательность для вывода в коде типа и / или удалять элементы формата, которые больше не требуются.

!!! note "Замечание:"

    При выборе стандартной схемы DIN 20066_2012-01 элементы формата в этом диалоговом окне уже выбраны таким образом, что генерируется код типа, соответствующий стандарту DIN 20066.

### Предв. просмотр

В этом поле отображается последовательность выбранных элементов формата в коде типа. Отдельные элементы формата при этом представлены заполнителями свойств.

# Разместить подсоединенные функции

Подсоединенные функции – все функции, подсоединенные с помощью соединений к устройствам. При размещении устройства с помощью функции "Подсоединенные функции" можно автоматически разместить все соединения и функции, подсоединенные к уже размещенной функции.

Условия:

* Создано минимум 2 неразмещенных устройства с выводами устройств.
* Между устройствами [сгенерированы неразмещенные соединения](planningconnections_h_createconnections.md).
* Вы открыли страницу.

1. Данные проекта > Устройства > Навигатор
2. Выделите устройство с подсоединенными через соединения функциями.
3. Переместите устройство на страницу путем перетаскивания мышью.
4. Выделите размещенное устройство.
5. Выберите пункты меню Вставить > Подсоединенные функции.

!!! info "Для сведения:"

    Размещаются все прямо подсоединенные неразмещенные соединения и функции.

6. Чтобы обновить информацию, нажмите клавишу ++F5++.

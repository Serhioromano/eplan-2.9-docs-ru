# Преобразовать излишние главные функции

В ходе проектирования могут появляться устройства с несколькими главными функциями. Для получения логически правильного проекта одно устройство может содержать максимум одну главную функцию. При помощи корректировочного прогона все излишние главные функции проекта можно преобразовать во вспомогательные функции для всего проекта. При этом можно определять, какие функции остаются главными: однополюсные, многополюсные или функции обзора.

Кроме того, излишние главные клеммы корректируются. Если одна и та же клемма несколько раз встречается в разных видах представления, то только одна из них может быть главной. Излишние главные клеммы будут преобразованы во вспомогательные.

Условия:

Вы открыли проект.

1. Выберите следующие пункты меню: Сервисные программы > Синхронизировать > Функции > Корректировать главные функции
2. В диалоговом окне Корректировать главные функции в групповом поле Преимущ. сохраняемая главная функция выберите требуемый вид представления. Например, для сохранения многополюсных главных функций выберите параметр Многополюсный.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Все излишние главные функции будут преобразованы во вспомогательные.

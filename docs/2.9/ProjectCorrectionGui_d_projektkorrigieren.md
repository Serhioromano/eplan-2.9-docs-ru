# Диалоговое окно Корректировать проект

Проект > Организовать > Корректировать

В этом диалоговом окне могут быть указаны схема и фильтр для объема корректировки. Последние настройки сохраняются в связи с проектом.

Обзор основных элементов диалогового окна:

### Настройки

В этом поле выберите схему, содержащую необходимые корректировочные настройки. С помощью кнопки ++...++ откройте диалоговое окно Настройки: Корректировать. Здесь может быть обработана существующая схема или создана новая схема.

### Фильтр

При помощи фильтра данные проекта могут целенаправленно корректироваться или исключаться из корректировки.

В этом раскрывающемся списке отображаются все доступные фильтры. Выбранный фильтр активируется автоматически. Запись "- Не активировано -" отключает фильтр и приводит к тому, что данные отображаются в неотфильтрованном виде. С помощью кнопки ++...++ откройте диалоговое окно [Фильтр](modaldialogsdb_d_filternnach.md). Здесь можно создать, обработать, удалить, скопировать, экспортировать, импортировать фильтр и управлять им.

Во всплывающем меню раскрывающегося списка Фильтр содержатся следующие записи:

* Выключить: Этот пункт меню доступен, если фильтр установлен: Сбрасывает настройку фильтра до записи "- Не активировано -".
* Активировать <Имя фильтра>: Этот пункт меню доступен, если для настройки фильтра установлено значение "- Не активировано -": Повторно активирует последний активный фильтр.

Таким образом можно быстро переключаться между неотфильтрованным и отфильтрованным в соответствии с требованиями пользователя представлениями.

!!! example "Пример:"

    Критериями фильтрации могут быть:Структурный идентификатор Буквенныеобозначения Определения функций.

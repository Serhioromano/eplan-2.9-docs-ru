# Запаковка и распаковка проектов

Запаковывать проекты можно или в навигаторе страниц, или в управлении проектами. Запакованный проект открывается как любой другой проект.

Условие:

Вы открыли проект.

1. Выделите требуемый проект в навигаторе страниц.
2. Выберите пункт меню Проект > Запаковать.
3. Подтвердите ввод.

!!! info "Для сведения:"

    Запакованный проект сохраняется в каталоге проектов с расширением имени файла *.elp и закрывается в EPLAN.

Условия:

* Вы настроили управление проектами.
* В представлении структуры дерева управления проектами находится считанный каталог, как минимум, с одним проектом.

1. Выберите следующие пункты меню: Проект > Управление
2. Выделите в представлении структуры дерева диалогового окна Управление проектами требуемые проекты.
3. Выберите пункт всплывающего меню Запаковать.
4. Подтвердите ввод.

!!! info "Для сведения:"

    Запакованные проекты сохраняются в каталоге проектов с расширением имени файла *.elp. Проекты, открытые во время процесса запаковывания, будут закрыты.

1. Выберите следующие пункты меню: Проект > Открыть
2. В диалоговом окне Открыть проект в поле Тип файла выберите запись " Запакованный проект EPLAN (*.elp)".
3. Откройте каталог, в котором хранится проект, и выберите нужный проект.
4. Щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Проект будет распакован и открыт; при этом вы увидите указатель процесса и сможете, при необходимости, прервать процесс. Если открыт навигатор страниц, проект отображается в Навигаторе страниц.

!!! tip "Совет:"

    Запакованный проект можно также открыть, дважды щелкнув в каталоге проекта по проекту с расширением имени файла *.elp. Проект будет распакован и открыт автоматически.

Условие:

Имеются запакованные проекты.

1. Выберите пункты меню Проект > Распаковать.

!!! info "Для сведения:"

    В диалоговом окне Открыть проект предварительно установлен Тип файла "Запакованный проект EPLAN (*.elp)", т. е. отображаются только запакованные проекты.

2. В диалоговом окне Открыть проект выберите нужный проект; при необходимости откройте другой каталог.
3. Щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Проект будет распакован, но не откроется.

!!! tip "Совет:"

    В управлении проектами распакуйте выбранный запакованный проект через меню ++Организовать++ > Распаковать.

# Вкладка Настройки (общая спецификация проводов)

* Параметры > Настройки > Фирма > Сборка проводов > Общая спецификация проводов > Настройки
* Сервисные программы > Данные изготовления > Сборка проводов > Общая спецификация проводов. Щелкните по кнопке ++...++ рядом с полем Схема. Выберите вкладку Настройки.

В этой вкладке хранится информация о технических возможностях используемой машины. Отдельные настройки задаются заранее индивидуально для машины при помощи подходящих значений, которые можно здесь изменить.

Обзор основных элементов диалогового окна:

### Минимальная длина провода

Минимальная длина, необходимая для заданной обработки провода машиной (стандартное значение: 1 мм). Более короткие провода передаются в машину с данной минимальной длиной.

### Максимальная длина провода

Провода, длина которых превышает указанную здесь величину, машина изготовить не может (стандартное значение: 150 000 мм).

### Максимальное поперечное сечение жилы

Максимальное поперечное сечение провода, которое машина может обработать (стандартное значение: 50 мм²).

### Максимальное количество проводов в объединенных проводах

Количество проводов, которые машина может объединить в объединенном проводе (стандартное значение: 10 000).

### Максимальная сумма поперечного сечения жил

Предельное значение для суммы поперечного сечения всех проводов (стандартное значение: 10 000 мм²).

### Непрерывный текст по длине провода

Значение длины, начиная с которого печать на провода должна наноситься в виде непрерывного текста. Если значение не указано, печать на все провода наносится в виде непрерывного текста.

### Направление подсоединения сверху / снизу / слева / справа

Тексты, которые необходимо напечатать на проводе для указания отдельных направлений подсоединения.

### Пересчитать поперечное сечение соединения не в мм²

При экспорте данных изготовления Общая спецификация проводов соединения, поперечное сечение которых имеет единицу измерения "AWG", экспортируются вместе с фактическим значением, напр. "18AWG". Включите эту настройку, чтобы не допустить автоматического перевода значения AWG в мм².

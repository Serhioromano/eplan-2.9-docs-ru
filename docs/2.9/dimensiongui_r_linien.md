# Вкладка Линии

Открыта страница в графическом редакторе. Выделено указание размеров, и выбран пункт меню Обработать > Свойства. В диалоговом окне Свойства выберите вкладку Линии.

На этой вкладке можно обрабатывать свойства отображения линий с размерами и выносных линий.

Обзор основных элементов диалогового окна:

Групповое поле "Выносная линия"

### Удлинение

Укажите здесь, насколько выносные линии могут выходить за линию с размерами.

### Интервал 1-й выносн. линии / Интервал 2-й выносн. линии

В этом поле можно изменять интервал между точкой измерения и началом выносной линии. При положительном значении выносные линии укорачиваются, при отрицательном — удлиняются.

### Подавить первую / Подавить вторую

Если установлен этот флажок, то первая / вторая выносная линия не отображается.

* * *

### Ограничение линии с размерами

Выберите из раскрывающегося списка вид представления концов линий с размерами, например, со стрелками.

### Подавить линию с размерами

Если установлен этот флажок, то отображается только числовая мера, но линия с размерами не отображается.

### Линия с размерами прерывается размерным числом

Если установлен этот флажок, то линия с размерами прерывается, когда ее пересекает размерное число.

Если этот флажок не установлен, то линия с размерами (независимо от положения размерного числа) никогда не прерывается.

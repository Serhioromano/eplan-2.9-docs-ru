# Диалоговое окно Выбрать вручную

Вы открыли проект. Сервисные программы > Отчеты > Генерировать. Щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать). Выделите в диалоговом окне Определить отчет относящийся к функции [Тип отчета](formgeneratorgui_k_auswertungstypen.md). Установите флажок Индивидуальный выбор. Щелкните по кнопке ++OK++.

В этом диалоговом окне выберите, для каких устройств, изделий или библиотек символов генерируется отчет.

Обзор основных элементов диалогового окна:

### Доступные

В этом поле перечислены все устройства, изделия или библиотеки символов, для которых выбранный тип отчета может создать отчет. Отображение типов записей зависит от выбранного типа отчета. Чтобы определить, какие устройства, изделия или библиотеки символов должны обрабатываться, выделите одну или несколько записей и нажмите на ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо).

### Выбранные

Для всех записей, находящихся в этом поле, генерируется отчет.

### Категория

Выберите из этого списка категорию, чтобы уменьшить вид свойств в поле Имя свойства / Значение.

### Имя свойства / Значение

Показывает свойства и значения устройств или библиотек символов, которые выделены в поле Доступные.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

### [Сортировать]

Если нажать эту кнопку, устройства в поле Выбранные сортируются в алфавитно-цифровой последовательности.

# Автоматически генерировать страницы с обзорами

В проектах с механическими компонентами в пространствах листа можно автоматически генерировать страницы с обзорами модели, 2D-отображениями отверстий и развертками шины.

Условие:

Вы открыли проект. В Графическом редакторе открыта страница проекта.

1. Создайте обзор модели, 2D-отображение отверстий или развертку шины.
2. Сохраните обзор как макрос с помощью пунктов меню Страница > Макрос страницы > Создать.
3. Выберите пункты меню Сервисные программы > Отчеты > Обзоры модели.
4. Во вкладке Шаблоны щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать), чтобы создать новый шаблон.
5. Внесите в свойства шаблона подходящие значения, как описано в разделе [Вкладка Шаблоны (диалоговое окно Обзоры модели, Развертки шины, Отображения отверстий)](gededit3dgui_r_vorlagen.md).
6. С помощью настроек фильтра определите, из каких основных функциональных элементов генерируются обзоры, и какие функциональные элементы с иерархической структурой представляются в обзоре.
7. Внесите в свойство Макрос обзор, сохраненный как макрос.
8. Щелкните по кнопке ![](images/all_generate_as.png){: .ui-icon } (Генерировать отчет) или выберите пункт всплывающего меню Генерация отчета, чтобы сгенерировать автоматические отчеты.

!!! info "Для сведения:"

    Страницы с отчетами генерируются на основе настроек, заданных в шаблоне, и с учетом выбранного макроса.

!!! note "Замечание:"

    При использовании функции [Составление отчетов проектов](formgeneratorgui_h_projektauswerten.md) через пункты меню Сервисные программы > Отчеты > Генерация отчета по проекту автоматически создаются страницы с обзорами модели, 2D-отображениями отверстий и развертками шины, если в диалоговых окнах доступны соответствующие шаблоны для обзоров. Если эти обзоры уже включены в проект, они обновляются.

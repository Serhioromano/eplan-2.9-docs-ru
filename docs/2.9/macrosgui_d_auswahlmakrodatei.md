# Диалоговое окно Выбрать макрос

Вы открыли проект. В Графическом редакторе открыта страница проекта. В наличии имеются макросы требуемого типа.

* Вставить > Макрос окна / символа.
* Страница > Макрос страницы > Вставить

Выберите требуемый макрос в этом диалоговом окне. Учтите, что для ***макросов страницы*** невозможен многократный выбор!

Обзор основных элементов диалогового окна:

### Предв. просмотр

Установите этот флажок, чтобы вывести в правой части диалогового окна просмотр графики выделенного в левом списке макроса. При вставке макросов страниц с помощью функции предварительного просмотра можно просмотреть до 12 страниц.

Под предварительным просмотром графики находится ***поле комментария***, которое недоступно для обработки. В этом поле выводятся следующие данные для выделенного макроса:

* ***Версия***: только для макросов из проектов макросов
* ***Исходный проект***: проект, на основе которого был создан макрос
* ***Описание***: текст, сохраненный в качестве описания при создании макроса (может состоять из нескольких строк); или свойство Макрос: Описание (ид. 11057) при выборе автоматически сгенерированных макросов страницы. Если в макросе страницы сохранены разные описания, то отображается только описание первой страницы макроса страницы.
* ***Источник / Ссылка***: только для макросов окна, созданных на основе файлов DXF / DWG. При этом запись "DXF" указывает на макросы из файлов DXF, а запись "DWG" - на макросы из файлов DWG.
* ***Количество страниц***: только для макросов страниц.

### Тип файла

В этом поле выводится на просмотр выбранный тип файла. При вставке макросов окон или символов доступны оба типа файлов "Макрос окна (*.ema)" и "Макрос символа (*.ems)", а также запись "Макрос окна, макрос символа (*.ema, *.ems)". При вставке макросов страниц в раскрывающемся списке можно выбрать любой тип макроса. Благодаря этому макросы окно и символов также можно вставлять как новые страницы.

### Вид представления

Недоступен для макросов страниц! Вид представления макроса служит для сортировки, т. е. она помогает в управлении вашими макросами. Она не влияет на функции внутри макроса, сохраняя соответствующий вид их представления. Таким образом, все макросы изделия могут содержаться в файле. Для каждого вида представления макроса можно создать до 26 вариантов. Вид представления предварительно настроен согласно выделенным функциям или согласно типу страниц. Выберите требуемую запись из раскрывающегося списка:

* Многополюсный: Для макросов, которые размещаются на многополюсных страницах схемы соединений.
* Многопол. Fluid: для макросов, которые размещаются на страницах схемы соединений для Fluid-Техники.
* Обзор: Для макросов, которые размещаются на страницах обзора.
* Парная перекрестная ссылка: Для макросов, которые используются для реализации парной перекрестной ссылки.
* Однополюсный: Для макросов, которые размещаются на однополюсных страницах схемы соединений.
* Функциональная схема автоматизации: Для макросов, которые размещаются на страницах функциональной схемы автоматизации.
* Графика: Для макросов, которые содержат только графические условные обозначения. Графические условные обозначения не учитывается ни при прогонах отчетов, ни при контрольных прогонах, ни при образовании перекрестных ссылок, и ***не*** регистрируются как цель.
* Размещение изделия: Для макросов, которые размещаются на монтажных платах.

!!! note "Замечание:"

    Если функция имеет другие виды представлений, где общие свойства содержат различные данные, в зависимости от вида представления выносится решение о том, какие свойства будут учитываться при составлении отчета. Для этого в EPLAN существует установленная [последовательность использования видов представлений при общей обработке и генерации отчетов](reverseengineering_k_reihenfolge.md).

### Вариант

Недоступен для макросов страниц! Если у макроса имеется несколько вариантов, требуемый вариант можно выбрать с помощью этого раскрывающегося списка. Выбранный вариант также выводится и в предварительном просмотре.

### Путь

Здесь в поле Поиск в выводится требуемый путь (с указанием дисковода). Через Всплывающее меню > Сбросить на стандарт можно автоматически установить путь в настройках установленного каталога макросов.

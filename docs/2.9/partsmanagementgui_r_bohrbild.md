# Вкладка Монтажные отверстия

Сервисные программы > Изделие > Управление. В дереве выделен объект, относящийся к уровню иерархии Схема сверления.

На этой вкладке задайте пространственное расположение и специфические данные монтажных отверстий, которые относятся к одной схеме сверления.

Обзор основных элементов диалогового окна:

### Название контура

Если в качестве типа сверления вы выбрали 'Определенный пользователем контур', здесь вы можете задать обозначение контура.

### Позиция X / Позиция Y

Позиции X / Y являются координатами, выходящими из центра монтажного отверстия или из нулевой точки определенного пользователем контура по отношению к нулевой точке функционального элемента (внизу, слева).

### 1-е измерение / 2-е измерение / 3-е измерение

Введите в этих полях требуемые размеры для определенных типов отверстия.

Тип сверления |  1-е измерение |  2-е измерение |  3-е измерение
---|---|---|---
Сверлить |  Диаметр |  - |  -
Резьба |  Диаметр |  - |  -
Прямоугольный (подтип Многогранный) |  Высота |  Ширина |  -
Прямоугольный (Подтип С фаской) |  Высота |  Ширина |  Ширина фаски
Прямоугольный (Подтип Закругленный) |  Высота |  Ширина |  Радиус закругления
Продольн. отверстие |  Высота |  Ширина |  -
Шестиугольник |  длина ребра |  - |  -
Восьмиугольник |  длина ребра |  - |  -

### Интервал повтора

Введите в этом поле расстояние, на котором должны находиться все другие отверстия по отношению друг к другу.

### Конечный интервал

Введите в этом поле минимальное расстояние от последнего отверстия до правого края объекта, то есть, ни одно отверстие не должно располагаться к краю объекта ближе введенного значения. Например, для кабельных каналов этим значением вы можете установить, насколько отверстие будет удалено от края канала, чтобы не было проблем при привинчивании.

### Просверл. кажд. n-е отверст.

В этом поле указывается, какие отверстия следует просверлить; если, например, ввести "4", будет просверлено только каждое четвертое отверстие.

!!! note "Замечание:"

    Помните, что свойства Интервал повтора, Конечный интервал и Просверлить каждое n-е отверстие оцениваются только для функциональных элементов, варьируемых по длине. Таким образом, использование этих свойств невозможно для функциональных элементов, ширина, высота и глубина которых жестко определена в свойствах изделия или в 3D-макросе.

### Всегда создавать

Если флажок установлен, механическая обработка, указанная в поле Тип сверления (отверстие, резьба, прямоугольное и т. д.), проводится всегда, даже если соответствующее изделие не размещено непосредственно на монтажной поверхности, релевантной для ЧУ. Это происходит, когда изделие размещено на несущей шине. Система автоматически ищет в структуре ближайшую монтажную поверхность, которую можно создать, и выдает данные отверстия на этой монтажной поверхности.

Если флажок снят (стандартные настройки), отверстие сверлится только тогда, когда оно размещено на монтажной поверхности, релевантной для ЧУ.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

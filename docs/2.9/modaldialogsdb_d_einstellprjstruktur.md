# Диалоговое окно Настройки: Структура дерева (...)

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Представление > Структура дерева (...)

В этих диалоговых окнах можно задать отображение и последовательность данных для структуры дерева в зависимости от устройств или страниц таким образом, чтобы в ***представлении структуры дерева*** навигаторов отображалась только релевантная информация.

!!! note "Замечание:"

    * Эти настройки не влияют на представления в виде списка навигаторов. Они по-прежнему показывают логическую последовательность страниц и данных проекта.
    * Обратите внимание, что данные настройки игнорируются при ***импорте*** и ***экспорте*** настроек проекта!

Обзор основных элементов диалогового окна:

### Список

В списке перечислены все свойства, который могут быть выведены для просмотра, рядом со свойствами находятся флажки; для просмотра требуемых свойств необходимо установить соответствующие флажки, а чтобы не выводить излишние свойства, следует снять соответствующие флажки.

!!! note "Замечание:"

    Обратите внимание на то, что в зависимости от отображаемой информации некоторые столбцы ***всегда*** выводятся на просмотр, т. е. эти флажки снять невозможно.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

### Также отображать описывающие идентификаторы

Установите этот флажок, когда в древовидном представлении навигаторов должен отобразиться идентифицирующий и описательный структурный идентификатор. Если этот флажок не установлен, в дереве отображаются только идентифицирующие структуры проекта.

### Развернутое представление подструктур

Если этот флажок установлен, то в древовидном представлении в навигаторах подструктуры структурных идентификаторов будут отображены как отдельные узлы дерева.

!!! example "Пример:"

    Вы установили этот флажок и создали новый структурный идентификатор для блока идентификаторов "Установка" с именем Новый.2. В структуре дерева создается уровень Новый, а в нем – уровень2. Если этот флажок снят, создается только один уровень,Новый.2.

### Использовать последовательность представления также для вывода страниц

Этот флажок доступен только в диалоговом окне Настройки: Структура дерева (страницы).

Если он установлен, то заданная в этом диалоговом окне последовательность для структуры дерева навигатора страниц также будет учитываться в следующих функциях:

* Пролистать страницы в графическом редакторе
* Печать страниц
* Экспортировать страницы как PDF-файл
* Экспортировать страницы в формате DXF / DWG
* Вывести страницы в свойствах Следующая страница / Предыдущая страница.

Если флажок снят, то последовательность вывода страниц соответствует представлению в виде списка.

!!! note "Замечание:"

    При выводе страниц (и других функциях) по умолчанию установлена логическая последовательность страниц в ***представлении в виде списка*** навигатора страниц.

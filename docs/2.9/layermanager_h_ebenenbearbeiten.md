# Работа со слоями

В EPLAN всегда имеются стандартные слои; эти слои невозможно удалить. Имена и описания этих стандартных слоев также защищены от записи. Имена стандартных слоев всегда начинают с записи "EPLAN". Так, например, слой для общей графики символов называется EPLAN300. Другие свойства стандартных слоев, например цвет, тип линии и т. д., можно изменять в соответствии с требованиями проекта. Кроме того, имеется возможность создания собственных слоев, имена и свойства которых пользователь указывает самостоятельно.

!!! note "Замечание:"

    Перед изменением настройки управления слоями для проекта следует сначала экспортировать файл конфигурации. Благодаря этому выполняется резервирование стандартных настроек, и стандартные значения для слоев можно будет позднее восстановить импортированием.

Условия:

* Вы открыли проект.
* Открыто диалоговое окно управления слоями: Данные проекта > Управление слоями

1. В диалоговом окне Управление слоями — <Имя проекта> выделите узел (или слой под узлом), для которого требуется создать слой, и щелкните ![](images/all_new_as.png){: .ui-icon } (Создать).

!!! info "Для сведения:"

    В таблице генерируется новая строка. В поле Слой автоматически вводится стандартная запись НОВЫЙ_СЛОЙ_1. Для слоя используются свойства выделенного слоя. (Если выделен узел, используются свойства первого слоя, находящегося под этим узлом.) При создании других слоев они получают имена НОВЫЙ_СЛОЙ_2, НОВЫЙ_СЛОЙ_3 и т. д.

2. При необходимости измените указанное по умолчанию имя слоя, а также запись в поле Описание. Если изменить описание слоя, EPLAN автоматически перемещает его в соответствующий узел.

!!! note "Замечание:"

    В именах слоев не разрешается использовать специальные символы <>^"?:*|'`.=/ .

3. Укажите в таблице свойства слоя, например цвет, тип линии и т. д.
4. Подтвердите ввод.

!!! info "Для сведения:"

    Слой размещается на дереве в соответствии с введенным описанием. Точки в описании слоя расцениваются как подструктура.

!!! note "Замечание:"

    Слои, использующиеся в проекте, и стандартные слои EPLAN удалить невозможно.

1. В диалоговом окне Управление слоями — <Имя проекта> выделите нужные слои и щелкните ![](images/all_delete_as.png){: .ui-icon } (Удалить).
2. Подтвердите ввод.

В диалоговом окне управления слоями можно искать все объекты, которым присвоен определенный слой в проекте или в его подразделе. Таким образом можно, например, находить объекты с пользовательскими слоями, отменять их присвоение и удалять пользовательские слои, когда они больше не используются.

1. Выберите пункты меню Данные проекта > Управление слоями.
2. В диалоговом окне Управление слоями — <Имя проекта> выделите слой, который нужно найти. Многократный выбор ***невозможен***.
3. Выделите в навигаторе страниц проект, в котором нужно выполнить поиск. Если вы не хотите выполнять поиск по всему проекту, вы можете ограничить область поиска. Для этого выделите, например, страницу, пространство листа или несколько объектов на странице или в пространстве листа.
4. В диалоговом окне Управление слоями — <Имя проекта> щелкните по кнопке ![](images/all_search_as.png){: .ui-icon } (Поиск).

!!! info "Для сведения:"

    В диалоговом окне Результаты поиска — <Имя проекта> отображаются все объекты, которым присвоен искомый слой.

После этого вы можете, например, открыть диалоговое окно "Свойства" найденного объекта через пункт всплывающего меню Свойства и вручную изменить присвоение искомого слоя. Или в диалоговом окне управления слоями нажмите кнопку ![](images/all_assignlayer_as.png){: .ui-icon } (Изменить присвоение слоя) и таким образом настройте присвоенные слои для одного или нескольких найденных объектов.

!!! note "Замечание:"

    Поиск предоставляет результаты, только если в управлении слоями и в навигаторе страниц выделен тот же проект.

Функция "Изменить присвоение слоя" позволяет присваивать объектам, которым присвоен определенный слой (слой источника), другой слой (целевой слой).

1. Выделите объекты, которым вы хотите присвоить другой слой. Это могут быть объекты в списке результатов поиска. Вы также можете выделить отдельные страницы / пространства листа или объекты на страницах / в пространствах листа. Это актуально, например, если вы хотите заменить все слои источника на странице целевым слоем.
2. В диалоговом окне Управление слоями — <Имя проекта> щелкните по кнопке ![](images/all_assignlayer_as.png){: .ui-icon } (Изменить присвоение слоя).
3. В поле Слой источника выберите слой, который вы хотите заменить.
4. В поле Целевой слой выберите слой, который нужно присвоить выбранным объектам.
5. Повторите шаги 1–4, если вы хотите изменить присвоение слоя для других объектов или изменить слои источника.

Определенные пользователем слои источника, которые больше не присвоены объекту, затем можно удалить.

Имя и описание каждого слоя (за исключением стандартных слоев EPLAN) можно менять любым образом. Не разрешается использовать совпадающие имена слоев.

!!! warning "Предупреждение:"

    Если попытаться изменить имя слоя на уже имеющееся имя, это оба этих слоя будут объединены. При этом будут утеряны настройки обработанного слоя.

1. В диалоговом окне Управление слоями — <Имя проекта> выделите требуемый слой и измените имя слоя в поле Слой.
2. Подтвердите ввод.

!!! info "Для сведения:"

    Все объекты, находящиеся на обрабатываемом слое, будут перенесены на новый слой, а обрабатываемый слой будет удален.

С помощью экспорта и импорта файла конфигурации слоев (*.elc) можно сохранить настройки слоев текущего проекта в качестве стандартных, перенести их в другой проект или передать другому пользователю.

1. В диалоговом окне Управление слоями — <Имя проекта> щелкните ![](images/all_export_as.png){: .ui-icon } (Экспортировать).

!!! info "Для сведения:"

    В поле Имя файла диалогового окна экспорта в качестве имени экспортируемого файла конфигурации по умолчанию указано имя текущего проекта.

2. При необходимости измените целевой каталог и имя экспортируемого файла и щелкните ++Сохранить++.

1. В диалоговом окне Управление слоями — <Имя проекта> щелкните ![](images/all_import_as.png){: .ui-icon } (Импортировать).
2. При необходимости укажите каталог, в котором находится требуемый файл конфигурации (*.elc). Выберите файл и щелкните ++Открыть++.

!!! info "Для сведения:"

    Если файл конфигурации не содержит определенные пользователем слои, которые отсутствуют в текущем проекте, эти слои будут созданы в проекте. Определенные пользователем слои, которые содержатся как в файле конфигурации, так и в проекте, будут адаптированы в соответствии с настройками, имеющимися в файле конфигурации.

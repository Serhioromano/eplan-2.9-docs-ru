# Заменить рамки, формы и библиотеки определения функций (изменение стандарта)

Для адаптации проекта к другому стандарту можно заменить существующие рамки и формы на другие.

Условия:

* Вы открыли проект.
* При нескольких открытых проектах: Вы выделили проект в Навигаторе страниц.
* Вы настроили целевой проект.
* Вы определили шаблон и поворот / инверсию.

1. Сервисные программы > Изменение стандарта > Вкладка Проект > [Далее] > Вкладка Поворот / инверсия > [Далее]
2. На вкладке Основные данные в столбце Имя выберите форму, рамку или библиотеку определения функций, которую необходимо заменить.
3. Щелкните внутри той же строки в столбце Создать имя по ++...++.
4. В выведенном диалоговом окне выберите форму, рамку или библиотеку определения функций, которые будут использоваться для целевого проекта, и щелкните по ++Открыть++.

!!! info "Для сведения:"

    Выбранный файл заносится в столбец Создать имя.

!!! info "Для сведения:"

    Далее можно заменить [Библиотеки символов и Заменить символы](changestandardsgui_h_symboletauschen.md).

# Сортировка страниц отчетов

Управление страницами проекта происходит через структуру страниц проекта в уровнях структурного идентификатора. При генерации отчетов EPLAN анализирует данные проекта и отображает их в [страницах отчетов](formgeneratorgui_k_auswertungsseiten.md). Эти страницы сортируются в структуру страницы проекта.

Путем сортировки страниц указывается, для каких уровней структурного идентификатора генерируются [Блоки отчетов](formgeneratorgui_k_auswertungsbloecke.md). При генерации, к примеру, содержания сортировка страниц генерирует "Общее" перекрывающееся содержание для общей документации схемы соединений, в то время как сортировка страниц "Установка" соответственно генерирует собственное содержание для каждой установки. Например, имеются следующие сортировки страниц:

* Можно генерировать перекрывающийся блок отчетов для всего проекта.
* Можно генерировать для каждого уровня структурных идентификаторов блока идентификаторов собственный блок отчетов.
* Можно генерировать для каждого уровня структурных идентификаторов нескольких блоков идентификаторов собственный блок отчетов.

!!! example "Пример:"

    В меню Параметры > Настройки > Проекты > "Имя проекта" > Отчеты > Вывод в страницахдля типа отчета "Содержание" задана сортировка страниц Установка + Место установки. При такой настройке EPLAN генерирует следующиеблоки отчетов:В каждом случае собственное содержание для каждого уровня структурных идентификаторов блока идентификаторов Установка. Это происходит всегда при изменении установки.В каждом случае собственное содержание для каждого уровня структурных идентификаторов блока идентификаторов Место установки. Это происходит всегда при изменении места установки.

* Вы можете генерировать перекрывающийся блок отчетов для всего проекта и дополнительно для каждого уровня структурных идентификаторов одного или нескольких блоков идентификаторов.

!!! example "Пример:"

    В меню Параметры > Настройки > Проекты > "Имя проекта" > Отчеты > Вывод в страницахдля типа отчета "Содержание" задана сортировка страниц Общая + Установка + Место установки. При такой настройке EPLAN генерирует следующиеблокиотчетов:Перекрывающееся содержание для всего проекта.В каждом случае собственное содержание для каждого уровня структурных идентификаторов блока идентификаторов Установкаи Место установки. Это происходит всегда при изменении установки и места установки.

* Вы также можете генерировать собственный блок отчетов при изменении условных обозначений-свойств "Группы вывода".

!!! note "Замечание:"

    То, какие виды сортировки страниц доступны, зависит от типа отчетов. Для типа отчетов "Перечень форм"доступна, к примеру, только сортировка страниц "Общая".

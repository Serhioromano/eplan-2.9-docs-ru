# EPLAN Fluid: Разделы

В EPLAN Fluid можно генерировать отчеты, фильтруемые и сортируемые по так называемым ***разделам***. В EPLAN Fluid имеются следующие разделы:

* Электротехника
* Гидравлика
* Пневматика
* Охлаждение
* Смазка
* Смазочно-охлаждающая жидкость
* Газовая техника
* Fluid (не определен).

Например, заданием раздела можно выводить отдельные списки соединений и/или спецификации для гидравлических и пневматических схем.

Вы устанавливайте раздел либо при создании новой функции в навигаторе, либо при вставке функции в схему соединений. Настройка производится в диалоговом окне Свойства ++...++ соответствующей функции. Для этого таблица Свойства на вкладке <Категория функций> может расширяться раскрывающимся списком Раздел.

EPLAN Fluid автоматически берет последнюю выбранную настройку раздела для предварительной настройки вновь создаваемой или вновь размещенного устройства.

Если вы присваиваете размещенному устройству новое определение, выбранное из области общих функций, то, какой раздел будет предустановлен для функции, определяется типом страницы. В этом случае на странице тип" схема соединений Fluid-техники" автоматически выбираются специфические разделы Fluid-техники, в то время, как на странице типа "многополюсная схема соединений" задается раздел "электротехника".

!!! note "Замечание:"

    Если Вы устанавливаете в Fluid схему соединений электромагнитное управление клапанами (магнитными катушками) из входящей в поставку библиотеки символов Fluid техники, посредством определения функции автоматически загружается раздел "Электротехника" с корректными предварительными настройками.

Раздел может устанавливаться также для покупных изделий и монтажных плат.

Напротив, раздел ***не может настраиваться*** для ***точек определения соединения***. Это свойство берется автоматически из подключаемых функций, то есть, из целей соединений.

### Подразделы

В дополнение к установкам раздела в EPLAN Fluid можно определять для функций так называемые "подразделы". Например, к функциям раздела "охлаждение" можно присоединить подразделы "водяное охлаждение" и "масляное охлаждение". Определение подразделов позволяет использовать в отчетах весьма детализованные критерии сортировки и фильтрации.

Подразделы также устанавливаются в диалоговом окне Свойства ++...++. Для этого таблица Свойства на вкладке <Категория функций> может расширяться раскрывающимся списком Подраздел. Обозначения для подраздела выбираются произвольно.

### Вставка устройств с неопределенным разделом Fluid-техники

В базе данных изделий можно присвоить изделию более одного раздела. Если вы вставляете изделие с таким множественным присвоением в качестве устройства в схему соединений (например, через пункты меню Вставить > Устройство), открывается диалоговое окно Выбрать раздел. В этом диалоговом окне вам предлагаются на выбор все присвоенные изделию разделы Fluid-техники, а также дополнительно раздел "Fluid (не определен)". Вы должны выбрать один из этих разделов. Этот раздел присваивается функциям устройства.

Вы можете, например, в навигаторе устройств организовать фильтрацию по разделу "Fluid (не определен)", а затем присвоить соответствующим функциям однозначный раздел.

Кроме того, в табличной обработке (пункты меню Обработать > В табличном виде) у вас есть возможность в ходе работы присвоить функциям устройств с разделом "Fluid (не определен)" определенный раздел.

# Диалоговое окно Настройки: Дата / время / числа

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Отображение > Дата/время/числа.

Наряду с возможностью копировать формат для даты и времени из настроек операционной системы в данном диалоговом окне есть возможность настраивать формат, специфический для проекта. Например, это имеет смысл в том случае, если работа над проектом ведется в рамках страны; тогда в этом случае как раз можно использовать единый формат данных.

Доступны следующие элементы диалогового окна:

### Формат операционной системы

Выберите этот параметр, если формат для даты и времени следует брать из настроек операционной системы. В находящемся ниже поле Предварительный просмотр выводится предварительный просмотр для формата.

### Специфический для проекта формат

Выберите этот параметр, если формат для даты и времени должен отличаться от формата операционной системы. В этом случае задайте детальные установки в находящихся ниже полях, причем в поле Предварительный просмотр будет выводиться соответствующий обновленный предварительный просмотр для формата. Доступны следующие возможности настроек:

* Формат даты: Выберите из раскрывающегося списка нужный формат даты, например "ДДММГГГГ" для представления "05.01.2007" или позицию "ДМГГ" для представления "5.1.07".
* Первый разделитель даты / Второй разделитель даты: Выберите из этих раскрывающихся списков один из возможных разделителей ".", "/" или "-".
* Формат времени: Выберите из раскрывающегося списка нужный формат времени, например, "ЧЧММСС" для представления "07:05:51" или вариант "ЧММ" для представления "7:05". При этом малое "h" задает время в 12-часовом формате, т.е. "1:05" для "13:05" и "1:05".
* Разделитель времени: Выберите из данного раскрывающегося списка один из возможных разделителей "<пусто>" или ":".
* Символ для первой половины / Символ для второй половины дня: Выберите из этого раскрывающегося списка один из возможных символов для первой и второй половины дня, например "AM" и "PM".

### В значениях выводить единицы измерения через пробел

Установите этот флажок, если у чисел, которые должны вводиться в единицах измерения "кг", перед единицей измерения должен стоять пробел. Если флажок не установлен, между числом и единицей не вставляется пробел.

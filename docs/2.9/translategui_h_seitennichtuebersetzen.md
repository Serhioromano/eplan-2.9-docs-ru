# Переводить типы страниц везде / не переводить

Чтобы не переводить все страницы проекта, просто исключите из перевода соответствующие типы страниц. В этом случае все страницы выбранного типа будут оставлены без перевода. Эти настройки сохраняются в связи с проектом. Тип выделенной страницы отображается в диалоговом окне навигатора Страницы — <Имя проекта> с помощью пункта всплывающего меню Свойства в поле Тип страницы.

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Параметры > Настройки > Проекты > "Имя проекта" > Перевод > Переводимые страницы
2. Чтобы выбрать ***определенный*** тип страниц, которые необходимо перевести, установите флажок перед соответствующим типом страниц.
3. Чтобы выбрать ***несколько*** типов страниц, выделите необходимые типы страниц, удерживая клавишу ++Ctrl++ или ++Shift++, и установите флажок перед одним из них.
4. Если страницы определенных типов переводить не требуется, снимите флажок перед выделенными типами страниц.

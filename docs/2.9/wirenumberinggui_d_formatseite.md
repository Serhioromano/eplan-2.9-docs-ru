# Диалоговое окно Формат: Страница / Столбец / Строка (нумерация соединений)

Вы открыли проект. Вы открыли диалоговое окно [Настройки: Нумерация соединений](wirenumberinggui_d_verbnumeinstellungen.md) (например, через пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Нумерация соединений). В диалоговом окне Настройки: Нумерация соединений вы выбрали определенную пользователем схему и перешли во вкладку Обозначение. Вы выделили запись в поле Группы форматов и щелкнули по ![](images/all_edit_as.png){: .ui-icon } (Обработать) или по ![](images/all_new_as.png){: .ui-icon } (Создать).

В диалоговом окне Нумерация соединений: Форматы в списке Доступные элементы формата выберите элемент формата типа "Страница", "Столбец" или "Строка" и при помощи кнопки со стрелкой перенесите его в список Выбранные элементы формата.

В этом диалоговом окне указывается форматирование для номера страницы, столбца или строки в группе форматов.

Обзор основных элементов диалогового окна:

### От источника

Выберите эту опцию, чтобы для данного формата использовался номер страницы, столбца или строки исходного функционального элемента.

### От цели

Выберите эту опцию, чтобы для данного формата использовался номер страницы или столбца целевого функционального элемента.

### Минимальное число разрядов

Здесь указывается минимальное число разрядов для значения свойства. Более длинные значения выводятся полностью. Разрешено использовать только положительные числовые значения. При вводе нуля число разрядов является переменным.

### Заполнитель

Значение заполняется с помощью этого символа, если минимальное число разрядов больше, чем ширина значения. При переменном числе разрядов (Запись "0" в поле Мин. число разрядов) ввод заполнителя невозможен.

### Отобразить значения отд. соединений

Если этот флажок установлен, для выбранного свойства на просмотр выводится значение определенного соединения.

Если флажок не установлен, для свойства на просмотр выводится значение первого соединения, которое определяет объем. Это значение в этом случае равно для всего объема (например, сети).

# Нет отображения при значении "0" на схемах автоматизации

В графическом представлении схем автоматизации для автоматизации зданий порядки свойств используются для отображения количества используемых или необходимых входов / выходов ПЛК на технологический контур, по пути или построчно. При представлении в соответствии со стандартами значение "0" не отображается. Для этого на вкладке Отображение диалогового окна "Свойства" теперь доступно новое свойство отображения Ноль как знак пробела.

![](images/benefit.png)Эффект:

При вставке макросов или копий в функциональную схему автоматизации уже существующие в проекте сегменты могут присваиваться быстрее, что позволяет избежать утомительного выбора сегментов по отдельности.

Это свойство отображения под уровнем иерархии Значение / единица измерения можно активировать только для свойств с числовыми значениями. Если этот флажок установлен, значение "0" заменяется пробелом.

Кроме того, это свойство отображения доступно для текстов, например текстов-заполнителей, на вкладке Формат.

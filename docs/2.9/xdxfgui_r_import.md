# Вкладка Импорт

* Параметры > Настройки > Пользователь > Интерфейсы > Экспорт и импорт DXF / DWG. Выберите вкладку Импорт.
* Страница > Экспортировать > DXF/DWG. Щелкните рядом с полем Схема на ++...++. Выберите вкладку Импорт.

Определите на этой вкладке специальные настройки импорта DXF / DWG.

Обзор основных элементов диалогового окна:

### Нач. каталог

Укажите в этом поле начальный каталог импорта или выберите его в интерактивном режиме с помощью кнопки ++...++.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Присвоение к страницам EPLAN

Сортировка файлов DXF / DWG будет выполняться в последовательности, в которой файлы были перенесены из окна выбора файлов. Вы можете изменить эту последовательность, выбрав параметры Числовой или Алфавитно-цифров.:

* Цифровой: файлы будут сортироваться в числовом порядке по именам файлов. Все файлы, имена которых созданы не в числовом формате, будут включены в числовую сортировку в алфавитно-цифровой последовательности.
* Алфавитно-цифров.: файлы DXF / DWG будут сортироваться в алфавитно-цифровом порядке (ASCII) по именам файлов.

### Генерир. страницы

Введите в этом групповом поле следующие данные:

* Нач. страница: укажите в этом поле номер страницы, под которым импортированная страница будет создана в навигаторе страниц проекта: начиная с 1 и заканчивая 99999.
* Величина шага: если вы импортируете несколько файлов, здесь можно задать величину шага для нумерации страниц.

### Игнорировать замороженные слои

С помощью этого флажка можно управлять процессом импорта, если в импортируемом файле DXF / DWG содержатся слои, для которых в AutoCAD установлена настройка "Замороженный". В AutoCAD элементы на таких слоях (Layer) невидимы и недоступны для обработки.

Если флажок установлен, элементы на таких слоях не импортируются.

Если флажок снят, элементы импортируются на замороженных слоях, но относящиеся к ним слои по умолчанию будут невидимыми в EPLAN и заблокированными для обработки.

Групповое поле Масштабирование

В этом групповом поле можно выполнить следующие настройки размера чертежа при импорте:

### Диалоговое окно масштабирования

Выберите этот параметр, чтобы дополнительно обработать автоматически полученные границы чертежа. В этом случае становится доступным групповое поле Определение границ чертежа, где необходимо указать, на основе чего по умолчанию будут определяться границы чертежа. Предлагаются следующие варианты:

* Из штампа чертежа: границы чертежа определяются заголовком чертежа.
* Из координат чертежа: проверяются координаты всего чертежа, и в качестве границ чертежа автоматически принимаются самые маленькие и самые большие значения координат.
* Из самого большого элемента чертежа: границы чертежа определяются заголовком чертежа. Затем проверяются координаты всего чертежа, и автоматически принимаются значения самого большого элемента чертежа.

### Фиксированное масштабирование

Выберите этот параметр, если автоматическая корректировка должна быть выполнена на основе заданных в следующих полях значений:

* Горизонтальное / вертикальное масштабирование: в этих полях определяются горизонтальный и вертикальный коэффициент масштабирования для чертежа. Введите здесь нужное десятичное значение, предварительно установленное значение — "1".
* Горизонтальная / вертикальная позиция: в этих полях определяются координаты левого нижнего угла чертежа. Введите здесь нужное десятичное значение, предварительно установленное значение — "0,00".

### Подогнать масштаб страницы

Установите этот флажок, если необходимо импортировать схемы DXF / DWG с гораздо большим удлинением (схемы цехов, схемы местностей и т. д.). Чтобы для таких чертежей генерировалась правильная толщина линий с указанием размеров, импорт DXF / DWG настраивается следующим образом:

* Чертежи импортируются в масштабе 1:1 без возможности его изменения.
* После импорта содержимое чертежей в рамке EPLAN выравнивается по центру.
* Масштаб сгенерированной страницы EPLAN автоматически устанавливается по размеру чертежа.

!!! note "Замечание:"

    * На схеме импорта DXF "Стандарт" настройка Подогнать масштаб страницы активирована по умолчанию.
    * ***Не*** активируйте настройку Подогнать масштаб страницы, когда вставляете файл DXF / DWG через меню Вставить > Графика > DXF / DWG на странице в графическом редакторе. При импорте на существующей странице автоматическая подгонка масштаба страницы невозможна.
    * На странице, сгенерированной с помощью этой настройки (т. е. экспортированной из EPLAN и повторно импортированной), размеры текстов, расположенных на слоях с установленной [настройкой "Масштабируемый"](layermanager_d_ebenenverwaltung.md), также изменяются в соответствии с масштабом страницы и, как следствие, становятся слишком большими. Т. к. процесс адаптации слоев линий и текста к масштабу страницы в таком случае потребует больше времени и действий, не следует повторно экспортировать в DXF / DWG, а затем еще раз импортировать страницы, сгенерированные с помощью настройки Подогнать масштаб страницы.

# Диалоговое окно Экспорт ЧУ, медь

Вы открыли проект. В пространстве листа или в навигаторе пространства листа выделены медные функц. элементы, для которых необходим экспорт ЧУ. Сервисные программы > Данные изготовления > Механическая обработка > ЧУ, медь.

В этом диалоговом окне задаются настройки экспорта данных изготовления для механической обработки медных функциональных элементов. Файл передачи на машины имеет формат "XML". Передаются следующие данные:

* Указания по версии ПО
* Информация проекта
* Материал
* Измерения шин
* Изгибы
* Перфорация.

Обзор основных элементов диалогового окна:

### Машина

В комплекте поставки содержится определение машины с названием "Стандарт". Если в настройках фирмы определены другие схемы, можно выбрать из раскрывающегося списка другие определения машины.

Кнопка ++...++ открывает диалоговое окно Настройки: Экспорт ЧУ, медь, в котором можно изменить настройки схемы или определить новую схему.

### Каталог вывода

В этом поле выводится место хранения. Здесь по умолчанию выбран каталог, который определен в пунктах меню Настройки > Фирма > Механическая обработка для экспорта ЧУ, медь.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Целевой файл

Это поле активно только тогда, когда выбор содержит медные функц. элементы, которые можно экспортировать. Здесь вводится название генерируемого файла ЧУ или выбирается с помощью клавиши ++...++ в диалоговом окне Открыть файл типа ЧУ, медь (*.xml) . При наличии нескольких файлов поле деактивируется и действуют условия создания названий файлов и подкаталогов, определенные в схеме.

### Материал

Данные материала для обработки на машине. Здесь можно ввести любой короткий текст; поле может также остаться пустым. Стандартное значение: "Cu".

### Применить ко всему проекту

Если этот флажок установлен, текущее выделение распространяется на все медные функц. элементы проекта.

Если флажок снят, экспортируются только выделенные медные функц. элементы.

Если отдельный проект уже выделен (и при этом фильтр не установлен), флажок автоматически включается и становится недоступным. В этом случае настройку больше невозможно изменить.

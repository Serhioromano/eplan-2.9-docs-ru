# Вкладка Файл

* Вы открыли проект. Сервисные программы > Данные изготовления > Экспорт / маркировка. В диалоговом окне Экспортировать данные изготовления / Вывести маркировку рядом с полем Настройки щелкните по кнопке ++...++. В диалоговом окне Настройки: Экспорт данных изготовления / маркировка выберите вкладку Файл.
* Параметры > Настройки > Пользователь > Интерфейсы > Экспорт данных изготовления / маркировка. В диалоговом окне Настройки: Экспорт данных изготовления / маркировка выберите вкладку Файл.

На этой вкладке указываются настройки для форматов файлов, в которых выводится экспорт / маркировка.

Обзор основных элементов диалогового окна:

### Тип файла

В этом поле указывается тип файла, в котором выводятся данные.

* Текстовый файл (*.txt): Данные выводятся в качестве файла txt.
* Файл Excel (*.xls, *.xlsx, *.xlsm, *.xlsb): Данные выводятся в виде файла Excel в следующем формате:
*.xls: Формат для Microsoft Excel 97 - 2003
*.xlsx: Формат для Microsoft Excel 2007 / 2010 без макросов
*.xlsm: Формат для Microsoft Excel 2007 / 2010 с макросами
*.xlsb: бинарный формат для Microsoft Excel 2007 / 2010.
В поле Шаблон можно указать форму Excel.
* XML-файл (*.xml): Данные выводятся в качестве файла XML.

### Целевой файл

Введите здесь путь к файлу для места хранения и имя файла экспорта / маркировки.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Шаблон

При выборе в поле Тип файла файла Excel как файла экспорта / маркировки задайте в данном поле [форму Excel как шаблон](labelgui_k_excelexportvorlagen.md). Данные будут выведены в эту форму.

### [Параметры]

В зависимости от выбранного типа файла с помощью этой кнопки можно открыть диалоговое окно Настройки: Текст. экспорт или диалоговое окно Настройки: Экспорт Excel.

# Копировать проект (Управление проектами)

Вы можете создать проект посредством копирования и обновления существующего проекта.

Условие:

В структуре дерева управления проектами находится считанный каталог с проектом.

1. Выберите следующие пункты меню: Проект > Управление
2. Выделите в структуре дерева имя проекта.
3. Выберите пункт всплывающего меню Копировать.
4. В диалоговом окне [Копировать проект](prjmanagementgui_d_projektkopieren.md), выберите, в каком объеме должен копироваться проект.
5. Нажмите рядом с полем Целевой проект кнопку ++...++ и укажите в диалоговом окне Копировать в место хранения и имя копии проекта.
6. Подтвердите ввод.

!!! info "Для сведения:"

    EPLAN копирует файл проекта *.elk и соответствующий каталог проекта *.edb в целевой каталог.

!!! info "Для сведения:"

    Скопированный проект автоматически считывается в управление проектами.

!!! info "Для сведения:"

    Скопированный (последний) проект автоматически открывается.

!!! note "Замечание:"

    Вы можете также выбрать и скопировать ***несколько проектов***. При этом несколько раз одно за другим открываются диалоговые окна Копировать проект.

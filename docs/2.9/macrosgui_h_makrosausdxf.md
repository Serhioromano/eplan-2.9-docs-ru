# Генерировать макросы из файлов DXF / DWG

1. Сервисные программы > Генерировать макросы > Из файлов DXF / DWG
2. Выберите в диалоговом окне Выбор файла DXF- / DWG требуемый макрос / макросы.
3. Щелкните по кнопке ++Открыть++.
4. Выберите требуемую схемы в диалоговом окне Импорт DXF / DWG и укажите целевой каталог.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выбранные файлы сохраняются в указанный каталог (по умолчанию используется путь <Дисковод_EPLAN>:\<Каталог программы>\Макросы\<Идентификатор фирмы>) в качестве макросов окон с именем <исходное_имя>.ema, после чего они могут использоваться в EPLAN как "обычные" макросы окон. При вставке соответствующих макросов окон в поле комментария диалогового окна выбора выводится запись "DXF" для макросов из файлов DXF и запись "DWG" для макросов из файлов DWG.

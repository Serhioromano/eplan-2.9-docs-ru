# Диалоговое окно Формат: Данные ПЛК

Вы открыли проект. Вы открыли диалоговое окно [Настройки: Нумерация с данными ПЛК](offlinenumerationplcgui_d_einstellungen.md) (например, через Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация с данными ПЛК). В диалоговом окне Настройки: Нумерация с данными ПЛК выбрана пользовательская схема. Вы выделили запись в поле Группы форматов и щелкнули по ![](images/all_edit_as.png){: .ui-icon } (Обработать) или по ![](images/all_new_as.png){: .ui-icon } (Создать).

В диалоговом окне Нумерация с данными ПЛК: Формат в списке Доступные элементы формата выберите элемент формата типа "Данные" и при помощи кнопки со стрелкой перенесите его в список Выбранные элементы формата.

В этом диалоговом окне указывается форматирование для свойства в группе форматов.

Доступны следующие элементы диалогового окна

### Категория

Выберите из раскрывающегося списка, какие свойства должны быть выведены в списке ниже. Затем выводятся все свойства выбранного типа, которым еще не было присвоено форматирование.

### Минимальное число разрядов

Здесь указывается минимальное число разрядов для значения свойства. Более длинные значения выводятся полностью. Разрешено использовать только положительные числовые значения. При вводе нуля число разрядов является переменным.

### Заполнитель

Значение заполняется с помощью этого символа, если минимальное число разрядов больше, чем ширина значения.

### Подавить пустые свойства

Установите этот флажок, чтобы не выводить свойства, которые не установлены. Кроме того, с помощью флажков Включая заданный разделитель и Включая следующий разделитель можно подавить разделители перед свойством и после него.

### Подавить свойства, равные нулю

Установите этот флажок, чтобы не выводить числовые свойства, значение которых равно нулю. Кроме того, с помощью флажков Включая заданный разделитель и Включая следующий разделитель можно подавить разделители перед свойством и после него.

### Язык

Для многоязычных свойство с помощью раскрывающегося списка можно указать язык вывода.

# Диалоговое окно Шаблоны сегментов — <Имя проекта>

Данные проекта > Предварительное планирование > Навигатор шаблонов сегментов.

В этом диалоговом окне в представлении в виде дерева или списка по вашему выбору можно отобразить и обработать определенные в проекте шаблоны сегментов и шаблоны классов трубы и веществ.

Обзор основных элементов диалогового окна:

На вкладке Дерево шаблоны сегментов отображаются в соответствии с их иерархической структурой. Самый высокий уровень иерархии образует проект, под ним шаблоны сегментов проекта разделены на следующие уровни:

* ***Определения основных сегментов*** (напр., "Сегмент структуры", "Объект планирования", "Технологический контур" и т. д.).
* ***Определения сегментов*** (напр., "Общий сегмент структуры", "Общий объект планирования", "Измерительный контур")
* ***Шаблоны сегмента***: На самом низком уровне иерархии отображены определенные пользователем шаблоны сегментов. Шаблоны сегментов обозначены стоящей впереди пиктограммой ![](images/all_segtemplate_as.png){: .ui-icon }. В ином случае для определений сегментов и основных сегментов отображаются такие же пиктограммы, как и в навигаторе предварительного планирования.

В дополнение к шаблонам сегментов в дереве под уровнем иерархии ***Другие шаблоны*** (пиктограмма ![](images/navigatorgui_othertemplates_as.png){: .ui-icon }) отображаются содержащиеся в проекте шаблоны классов трубы (пиктограмма ![](images/navigatorgui_pipingtemplate_as.png){: .ui-icon }) и веществ (пиктограмма ![](images/navigatorgui_substancetemplate_as.png){: .ui-icon }).

На вкладке Список обычно отображены имя шаблона сегмента, его описание, имя определения сегмента, обозначение и описание сегмента, а также структурный идентификатор для блоков идентификаторов "Установка" и "Место установки". В столбцах Шаблон сегмента: Имя (идентифиц.) и Шаблон сегмента: Описание также отображаются имена и описания "других шаблонов" (классов трубы, веществ).

Дополнительно можно вывести структурные идентификаторы для других блоков идентификаторов, а также другие свойства сегментов структуры и объектов планирования (например, свойства Техническое описание, Расход энергии или определенные пользователем свойства). В представлении в виде списка значения свойств доступны для непосредственной обработки.

!!! tip "Совет:"

    Для обновления вида в навигаторе шаблонов сегментов нажмите клавишу ++F5++.

### Фильтр

В этом раскрывающемся списке отображаются все доступные фильтры. Выбранный фильтр активируется автоматически и применяется как к дереву, так и к списку. Запись "- Не активировано -" отключает фильтр и приводит к тому, что данные отображаются в неотфильтрованном виде. С помощью кнопки ++...++ откройте диалоговое окно [Фильтр](modaldialogsdb_d_filternnach.md). Здесь можно создать, обработать, удалить, скопировать, экспортировать, импортировать фильтр и управлять им.

Во всплывающем меню раскрывающегося списка Фильтр содержатся следующие записи:

* Выключить: Этот пункт меню доступен, если фильтр установлен: Сбрасывает настройку фильтра до записи "- Не активировано -".
* Активировать <Имя фильтра>: Этот пункт меню доступен, если для настройки фильтра установлено значение "- Не активировано -": Повторно активирует последний активный фильтр.

Таким образом можно быстро переключаться между неотфильтрованным и отфильтрованным в соответствии с требованиями пользователя представлениями.

### Значение: <Свойство>

При помощи [быстрого ввода](modaldialogsdb_k_filter.md) в данном поле для определенного и активированного фильтра можно быстро изменить значение его критерия.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Новый шаблон сегмента |  Этот пункт меню доступен, если выделено определение сегмента. Позволяет создать шаблон сегмента.
Новый класс трубы |  Этот пункт меню доступен, если "Определение сегмента" выделено для классов трубы. Позволяет создать класс трубы.
Новое вещество |  Этот пункт меню доступен, если "Определение сегмента" выделено для веществ. Позволяет создать вещество.
Копировать / Вставить |  Соответствует знакомым функциям Windows для обработки записей данных. Позволяет копировать и вставлять шаблоны сегментов внутри навигатора шаблонов сегментов. Скопированные шаблоны сегментов автоматически нумеруются. При копировании определений сегментов или определений основных сегментов копируется также и полная подчиненная структура.
Дублировать |  Дублирует выделенный шаблон сегмента. Идентифицирующее имя при этом автоматически получает более высокий номер.
Экспортировать |  Позволяет экспортировать шаблоны сегментов в файл формата XML.
Импортировать |  Позволяет импортировать шаблоны сегментов из файла формата XML.
Удалить |  Удаляет все выделенные шаблоны сегментов или — в представлении структуры дерева — все шаблоны сегментов, расположенные ниже выделенного уровня структуры дерева. (Возможен многократный выбор шаблонов сегментов или уровней структуры дерева.) Удалить можно только шаблоны сегментов, не использующиеся в проекте.
Переименовать (только дерево) |  С помощью этого пункта меню можно изменять идентифицирующее имя выделенного шаблона сегмента.
|

!!! note "Замечание:"

    В случае с шаблонами для классов трубы и веществ пункты всплывающего меню Переименовать, Обновить сегменты, Присвоить и Изменить определение сегмента недоступны!

Обновить сегменты |  Обновляет данные адресов ПЛК и / или изделия на сегментах (объектах планирования, технологических контурах и т. д.), которым были присвоены выделенные шаблоны сегментов. В соответствии с изменениями на вкладках ПЛК и / или Изделие шаблона сегмента это необходимо, чтобы отобразить текущие данные шаблона сегмента также на сегментах, а затем в списке присвоений ПЛК и / или в отчетах (напр., спецификациях изделий). Обновление возможно, только если адреса ПЛК и / или изделия не присваиваются сегментам вручную!
Присвоить |  Этот пункт меню доступен в том случае, если открыта страница проекта (страница предварительного планирования, функциональная схема автоматизации). Присваивает выделенный шаблон сегмента выбранному сегменту на странице проекта. Определенные в шаблоне сегмента значения свойств переносятся на сегмент. Значения, ранее вручную измененные в сегменте, сохраняются.
Обновить данные заполнителя |  Обновляет данные объектов-заполнителей для выделенных шаблонов сегментов. Измененные данные заполнителя переносятся из уже присвоенных макросов на шаблоны сегментов для технологических контуров, объектов планирования и т. д. Это означает, что нет необходимости открывать шаблоны сегментов с сохраненными макросами по отдельности и обновлять данные.
Изменить определение сегмента |  Открывает диалоговое окно Выбрать определение сегмента и позволяет присвоить шаблону сегмента другое определение сегмента (если шаблон сегмента не используется в проекте).
Список с предварительным выбором (только дерево) |  Уменьшает число отображаемых элементов, чтобы ускорить ориентирование в представлении в виде списка. Если этот параметр активирован, представление в виде списка вызывается с автоматическим фильтром (предварительный выбор), причем этот фильтр содержит только что выбранные элементы.
Выбрать в дереве (только список) |  Показывает выделенный объект во вкладке Дерево.
Свойства |  Открывает диалоговое окно Свойства (усл. обозначение): ++...++. Позволяет обрабатывать свойства сегмента или функции.
Свойства (общие) |  Открывает диалоговое окно Свойства (общие): ++...++. Позволяет обрабатывать свойства устройства.

# Импортировать данные соединений и маршрутизируемые соединения

Данные соединений импортируются из файлов с расширением имени файла TXT, CSV, ASC и из файлов Excel (*.XLS, *.XLSX, *.XLSB, *.XLSM) в открытый или выделенный в навигаторе страниц проект EPLAN.

Условия:

* Вы открыли проект.
* Данные соединений из внешней программы доступны в форме текстового файла или файла Excel.

1. Выберите пункты меню Данные проекта > Соединения > Импортировать > Маршрутизируемые соединения.
2. В диалоговом окне Импортировать данные соединений в поле Тип источника данных выберите, из какого типа файла ("Текстовый" или "Excel") следует импортировать данные соединений.
3. Рядом с полем Источник данных щелкните мышкой по кнопке ++...++.

4. В диалоговом окне [Выбрать источник данных](projectprocessinggui_d_datenquelleauswaehlen.md) выберите файл, из которого должен осуществляться импорт, и задайте необходимые настройки.
5. Щелкните по кнопке ++OK++.
6. В диалоговом окне Импортировать данные соединений в полях Обозначение для внутр. / внешн. введите текст, который должен обозначать позицию соответствующего вывода провода клемм.
7. В диалоговом окне Импортировать данные соединений рядом с полем Присвоение полей щелкните мышкой по кнопке ++...++.
8. В диалоговом окне Присвоение полей в столбце Свойство укажите, какие свойства EPLAN должны быть присвоены обозначениям внешних полей данных, перечисленным в столбце Внешнее обозначение поля данных.
9. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В диалоговом окне Синхронизировать маршрутизируемые соединения показаны данные соединений из источника данных и их присвоение маршрутизируемым соединениям в проекте EPLAN. При этом проверяется, содержатся ли соединения уже в проекте или нет, и результаты проверки выводятся в столбце Статус.

10. В диалоговом окне Синхронизировать маршрутизируемые соединения в столбце Операция укажите, что должно происходить с соединениями в зависимости от отображаемого статуса:

Статус |  Действие
---|---
![](images/all_statusnew_as.png){: .ui-icon } |  Создать |  Импортировать или Нет
![](images/all_statusok_as.png){: .ui-icon } |  Идентичны |  Нет
![](images/all_statusdelete_as.png){: .ui-icon } |  Дополнительные |  Удалить или Нет
![](images/all_statusadditional_as.png){: .ui-icon } |  Изменено |  Применить изменение или Нет
![](images/erroricon_error_systemmessages.png){: .ui-icon } |  Недействительные |  Нет
11. Для сокращения объема отображения в диалоговом окне Синхронизировать маршрутизируемые соединения до свойств EPLAN одного маршрутизируемого соединения можно настроить специфический для раздела фильтр в поле Фильтр.
12. Выберите в выделенной строке в диалоговом окне Синхронизировать маршрутизируемые соединения пункт всплывающего меню Присвоить изделие, чтобы автоматически присвоить импортируемым устройствам имеющийся номер изделия, подходящий для базы данных изделий.
13. Если автоматическое присвоение невозможно, в столбце Номер изделия щелкните мышкой по кнопке ++...++, чтобы открыть выбор изделия и вручную выбрать в нем изделие.
14. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выполняется импорт или выбранные действия.

Статус |  Действие |  Результат
---|---|---
![](images/all_statusnew_as.png){: .ui-icon } |  Создать |  Импортировать |  Маршрутизируемое соединение импортируется
![](images/all_statusok_as.png){: .ui-icon } |  Идентичны |  Нет |  Маршрутизируемое соединение остается в проекте
![](images/all_statusdelete_as.png){: .ui-icon } |  Дополнительные |  Удалить |  Маршрутизируемое соединение удаляется из проекта
Нет |  Маршрутизируемое соединение остается в проекте
![](images/all_statusadditional_as.png){: .ui-icon } |  Изменено |  Применить изменение |  Применяются измененные значения свойств
Нет |  Измененные значения свойств не применяются
![](images/erroricon_error_systemmessages.png){: .ui-icon } |  Недействительные |  Нет |  Маршрутизируемое соединение не импортируется

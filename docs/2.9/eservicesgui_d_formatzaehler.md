# Диалоговое окно Формат: Счетчики (форматы адресов ПЛК)

Вы открыли проект. Вы открыли диалоговое окно [Настройки: ПЛК](plcgui_d_allgemeinespseinstellungen.md) (напр., через пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Устройство > ПЛК). В диалоговом окне Настройки: ПЛК рядом с полем Специфические для ПЛК настройки вы щелкнули кнопку ++...++. В диалоговом окне [Настройки: Специфич. для ПЛК](plcprocessorgui_d_spsspezifischeeinstellungen.md) выбрана вкладка Форматы адресов, а рядом с полем для требуемого формата адреса нажата кнопка ++...++.

В диалоговом окне Формат адреса ПЛК выберите в списке Доступные элементы формата элемент формата типа 'Счетчик' и щелкните по ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо) (или щелкните два раза в списке Выбранные элементы формата по уже определенному счетчику).

В этом диалоговом окне установите форматирование для счетчика в группе форматов.

Обзор основных элементов диалогового окна:

### Система счисления

В этом поле установите систему счисления, в которой отображается счетчик. Доступны следующие значения:

* Восьмеричный: 0,1,2,3....7, 10,11....17, 20....
* Десятичный: 0,1,2,3....9, 10,11....19, 20....
* Шестнадцатеричный: 0,1,2,3....9, a,b,c,d,e,f, 10,11....19, 1a,1b,1c,1d,1e,1f, 20....

### Нач. значение

Начальное и конечное значение вместе задают допустимый диапазон значений счетчика. Начальное значение должно быть положительным и соответствовать выбранной системе счисления.

### Конечное значение

Конечное значение должно быть больше начального значения и также соответствовать выбранной системе счисления.

Если в процессе адресации достигается конечное значение, то автоматически происходит перенос в предшествующий счетчик (т.е. в находящийся слева), и текущий счетчик начинает отсчет снова с начального значения. Первый счетчик (расположенный дальше всех слева) будет непрерывно повышаться, иногда выходя за пределы конечного значения.

### Число разрядов

Число разрядов факультативно и должно быть положительным; оно служит для графического представления. Если отображаемое значение содержит меньше цифр, чем установленное число разрядов, тогда цифры заполняются нулями.

### Значение конфигурации из свойства

В раскрывающемся списке предлагаются различные свойства ПЛК. При этом речь идет о свойствах, которые можно показать на выводе устройства ПЛК и там отобразить соответствующий блок ПЛК. Значения конфигурации можно изменять только на блоках ПЛК!

При адресации значение конфигурации, т. е. значение выбранного свойства, применяется как начальное значение для счетчика (для первого вывода устройства ПЛК). Показатель счетчика непрерывно повышается в соответствии с данным значением. Если в процессе адресации найден вывод устройства, при котором изменяется значение конфигурации, то счетчик принимает это значение и отсчитывается начиная с него.

Если достигнуто конечное значение, то отсчет идет дальше, исходя из Нач. значение.

!!! note "Замечание:"

    При адресации могут использоваться только ***числовые*** значения конфигурации. Если выбранное свойство содержит буквенные или алфавитно-цифровые значения, выводится сообщение.

Доступны следующие свойства:

* Без значения конфигурации: Начальное значение для адресации не определяется из свойства, используется заданный в диалоговом окне Новая адресация выводов устройства ПЛК начальный адрес.
* Имя станции: Имя станции ПЛК соответствующего блока ПЛК.
* Карта ПЛК вставлена в каркас: Имя каркаса, в который вставлена соответствующая карта ПЛК.
* На позиции / разъем: Разъем / позиция соответствующей карты ПЛК.
* Начальный адрес карты ПЛК: Введенный в блоке ПЛК начальный адрес карты ПЛК. Для программируемых контроллеров в формате Siemens задайте, например, в качестве начального значения наименьший байтовый адрес карты ПЛК.
* Обозначение канала (автоматич.): Обозначение канала точки подвода питания ПЛК. Показывает или содержание обозначения канала, внесенного вручную, или содержание канала, определенного из соседнего вывода устройства В/В.
* ЦПУ: Имя: Имя центрального обрабатывающего устройства (процессора) блока управления ПЛК.

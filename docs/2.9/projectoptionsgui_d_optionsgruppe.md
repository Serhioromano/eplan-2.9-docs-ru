# Диалоговое окно Опциональная группа

Вы открыли проект. Данные проекта > Опции > Навигатор.

* Выберите пункты всплывающего меню Создать > Опциональная группа.
* Вы выделили опциональную группу. Обработать > Свойства.

В этом диалоговом окне создается новая опциональная группа или выполняется обработка уже имеющейся.

Обзор основных элементов диалогового окна:

### Имя

Введите здесь имя опциональной группы или измените имеющееся имя.

### Описание

Введите здесь описывающий текст для опциональной группы. Под описанием понимается многоязычный текст, отображаемый по умолчанию в представлении структуры дерева рядом с именем (отделяется пробелом и заключается в круглые скобки).

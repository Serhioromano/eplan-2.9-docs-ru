# Диалоговое окно Найденные слова

Открыта страница в графическом редакторе. Или открыт проект и редактор (редактор форм, редактор символов и т. д.).

* Вставить > Графика > Текст / Вставить > Функциональный текст зоны. Выберите вкладку Текст. Введите многозначное слово. Выберите пункт всплывающего меню Перевести.
* Выделите текст или функциональный текст зоны и выберите пункты меню Обработать > Свойства или дважды щелкните по тексту или функциональному тексту зоны. Выберите вкладку Текст. Введите многозначное слово. Выберите пункт всплывающего меню Перевести.

В этом диалоговом окне будут показаны варианты перевода указанного вами слова. Для него EPLAN ищет в словаре подходящие ключевые слова.

Обзор основных элементов диалогового окна:

### Контекстуальная информация

Доступна, только если при ***автоматическом*** переводе будет обнаружено многозначное слово. Для этого необходимо установить флажок Индивидуал. выбор при многозначных значениях. Для перевода ***данных проекта*** эта настройка находится в специфических для проекта настройках перевода; для перевода ***данных изделий*** одноименная определенная пользователем настройка находится на вкладке Вне зависимости от проекта.
В этой области диалогового окна отображается информация с контекстом для соответствующего выделенного слова. При этом строка Контекст содержит тип текста (напр., "Произв. текст") или категорию функции, ОУ, а также свойство, в котором всплывает слово; в строке ниже отображается полный текст, который содержит слово.

!!! tip "Совет:"

    Если текст уже переводился, для поля Полный текст с помощью пункта всплывающего меню Многоязычный ввод можно выполнить переход в одноименное диалоговое окно и посмотреть, какой перевод уже использовался в проекте / в базе данных изделий. Эту информацию можно продолжать использовать при выборе предлагаемых вариантов перевода слова. Во время этой операции ввод данных в диалоговом окне ***невозможен***.

### Варианты перевода

Если вы переводите во время ввода, EPLAN будет предлагать различные варианты перевода для введенного слова, если оно сохранено в словаре под несколькими значениями, ***а также*** различные варианты перевода для слов, которые начинаются с введенного слова.

!!! example "Пример:"

    ***Многозначное слово***В качестве исходного языка выбранen_US. Вы вводите слово "Connector". Слово дважды фигурирует в словаре как ключевое слово, но переводится на язык переводаfr_FRпо-разному.en_USfr_FRКомментарийConnectorConnecteurОпределение функцииConnectorSymbole de connexionСимвол***Слова, начинающиеся со слова, вводимого пользователем***Вы вводите слово "Coil". EPLAN предлагает из словаря все слова, начинающиеся со слова "Coil".en_USfr_FRCoilBobinecoil currentcourant de bobinecoil voltagetension de bobineCoil, ACBobine, courant alternatif

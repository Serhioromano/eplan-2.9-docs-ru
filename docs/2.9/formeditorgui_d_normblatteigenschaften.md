# Диалоговое окно Свойства рамки — <Имя рамки>

Вы открыли проект.

* Сервисные программы > Основные данные > Рамка > Создать > [Сохранить].
* Сервисные программы > Основные данные > Рамка > Открыть. Выберите пункт всплывающего меню Свойства на рамке в диалоговом окне навигатора Страницы — <Имя проекта>.

В этом диалоговом окне установите свойства вновь созданной и открытой рамки.

Обзор основных элементов диалогового окна:

### Имя свойства / Значение

В зависимости от выбранной категории выводятся возможные свойства; некоторым из них, например свойствам Идентификатор фирмы и Дата изменения, уже присвоено значение, которое невозможно изменить. Для других свойств, например для свойств Описание и Версия, невозможно присвоить значение или же изменить уже имеющееся значение.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

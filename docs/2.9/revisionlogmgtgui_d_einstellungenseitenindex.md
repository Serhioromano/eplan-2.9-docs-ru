# Диалоговое окно Настройки: Отслеживание изменений (нумерация индекса ревизии)

Параметры > Настройки > Фирма > Управление > Отслеживание изменений (нумерация индекса ревизии).

В этом диалоговом окне определите схему нумерации для индекса ревизии. Если закрыть одну или несколько страниц, то на основе этой схемы нумерации определяется предлагаемый новый индекс ревизии.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Доступные элементы формата

В этом списке выводятся элементы, которые могут быть частями формата нумерации:

* Разделитель
* Счетчик.

### Выбранные элементы формата

Этот список показывает, какие элементы формата использовались для построения формата нумерации. С помощью кнопок, расположенных над списком, можно дополнительно изменить форматирование элемента формата, удалить элементы формата, а также изменить их последовательность.

### Предв. просмотр

Здесь выводится предварительный просмотр формата нумерации. Отдельные элементы формата выводятся в индексе ревизии в той последовательности, в которой они стоят в списке. Значение в этом поле изменить невозможно.

### Показать предупреждение, если индекс ревизии не соответствует схеме

Если этот флажок установлен, то при закрытии страниц в случае, если введенное значение индекса ревизии не соответствует предложенному, выводится предупреждение. Если подтвердить данное предупреждение, нажав ++OK++, то используется введенное значение и нумерация индексов прерывается. При следующем закрытии нумерация индексов ревизии начинается заново.

Если флажок не установлен, то введенный индекс ревизии не проверяется.

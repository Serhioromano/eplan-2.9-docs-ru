# Диалоговое окно Экспорт DXF/DWG

Выделен проект или как минимум одна страница проекта. Страница > Экспортировать > DXF/DWG.

В этом диалоговом окне начинается собственно процесс экспорта. При экспорте отдельных страниц проекта или всего проекта в формат DXF или DWG для каждой выбранной страницы создается файл.

Обзор основных элементов диалогового окна:

### Источник

В этом поле выводится на просмотр источник экспорта. В случае с целым проектом это имя проекта, для отдельных страниц отображаются полные имена страниц (включая свойства страниц Функциональное присвоение, Установка и т. д.). Здесь ***нельзя*** обрабатывать записи.

### Схема

Выберите из раскрывающегося списка схему, которая определяет настройки для экспорта файлов. С помощью кнопки ++...++ откройте соответствующее диалоговое окно настроек для экспорта. Там можно определить настройки и сохранить их как схему или [обработать имеющиеся схемы](schemeconfig_h_schematabearbeiten.md).

### Средство экспорта

В данном групповом поле задайте, на каком средстве экспорта должен выполняться экспорт:

* Носитель данных: При выборе этого параметра файл DXF/DWG выводится на носитель данных.
В поле Каталог вывода укажите место хранения. В качестве целевых для экспорта данных можно указать любой дисковод и подкаталог.
* Электронная почта: При выборе этого параметра файл DXF / DWG высылается по электронной почте. После запуска функции файл запаковывается посредством внутреннего архиватора, после чего запускается настроенный редактор электронной почты.
Из раскрывающегося списка Ограничение размера выберите максимальный размер для отдельных файлов, отправляемых по электронной почте. Вы можете установить значение от 1 до 10 Mб с шагом 1 Mб или выбрать запись "Неогранич.". EPLAN автоматически создаст то количество файлов для отправки, которое необходимо для отправки всех частей. Всем частям будет сопутствовать следующий текст: "<Имя файла> n/m". При этом "n" — это номер отдельного файла для отправки по электронной почте, а "m" — общее количество файлов.

!!! note "Замечание:"

    Для обмена электронными сообщениями на вашем компьютере должен быть установлен и настроен клиент электронной почты, поддерживающий MAPI. По этой же причине не может применяться почтовое приложение Windows 8®.

### Имя файла

При экспорте ***отдельной*** страницы существует возможность ввода любого имени файла. Через [...] также можно свободно задавать имя файла или выбирать имя уже имеющегося файла.

Если специальное имя файла не введено, экспортированному файлу присваивается имя в зависимости от настройки Генерировать имена файлов.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

!!! note "Замечание:"

    При экспорте ***нескольких*** страниц невозможна свободная передача имени файла. Файлы DXF/DWG генерируются в заданном целевом каталоге с непрерывной нумерацией или из имен страниц, или из свойств; это зависит от настроек выбранной схемы.

# Диалоговое окно Нумерация с данными ПЛК: Предварительный просмотр результата

Вы выделили устройства в графическом редакторе или навигаторе, или же вы выделили одну или несколько страниц либо проект в диалоговом окне навигатора Страницы — <Имя проекта>. Данные проекта > Устройствo > Нумерация с данными ПЛК. Установите в диалоговом окне Нумерация с данными ПЛК флажок Предварительный просмотр результата. Щелкните по кнопке ++ОК++.

В этом диалоговом окне выводится предварительный просмотр результата нумерации. Обозначения клемм и контактов штекеров и ОУ устройств общего назначения отображаются на двух отдельных вкладках.

Обзор основных элементов диалогового окна:

### Вкладка Обозначения клеммы и контакта штекера / вкладка Устройство

На вкладке Обозначения клеммы / контакта штекера выводятся обозначения клемм и контактов штекеров перед и после нумерации.

На вкладке Устройство выводятся все пронумерованные главные и вспомогательные функции, которые содержит введенное видимое ОУ. Функции, которые не содержат видимое ОУ, а копируют его, не выводятся. На этой вкладке можно вручную изменить нумерацию устройств. Если присваиваете вспомогательной функции другое ОУ, то можно изменить перекрестную ссылку над ней.

В таблице содержатся строки для каждой пронумерованной функции. Всегда выводятся следующие столбцы:

* Ошибка: Если во время нумерации с данными ПЛК происходит ошибка, ячейка в этом столбце выделяется красным цветом. Если поле выделено, в поле Описание сообщения выводится статус ошибки.
* Имя (идентифиц.): Здесь выводится идентифицирующее имя функции.
* Обозначение клеммы / контакта штекера: Показывает обозначение перед нумерацией (только для клемм и контактов штекера).
* Обозначения выводов устройства (все): Здесь выводятся обозначения выводов устройства функции (только для устройств общего назначения).
* Новое обозначение / Новое ОУ: Просмотр обозначения / ОУ после нумерации. В этом столбце можно изменить обозначение / ОУ.
* Функциональный текст (автоматически): Здесь выводится функциональный текст.
* Размещение: Указывает, на какой странице размещена функция.

Другие столбцы можно добавить, используя во всплывающем меню пункт Конфигурировать представление.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Использовать исходное обозначение / Использовать исходное ОУ |  С помощью этой опции клемме или контакту штекера после нумерации можно снова присвоить исходное обозначение / исходный идентификатор.

### Описание сообщения

В этом месте выводится сообщение для той выделенной строки, в которой произошла ошибка. Вы можете игнорировать эту ошибку или устранить ее.

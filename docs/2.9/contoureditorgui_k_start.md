# Редактор контура

Подобно рамкам и формам редактор контуров открывается через пункт меню Сервисные программы > Основные данные. При этом функциональность редактора контуров зависит от типа создаваемого контура ('Выдавливание контура' или 'Генерирование записи данных контура ЧУ'), а следовательно, и от выбранного типа файла (*.fc2 или *.fc1):

* В редакторе контуров для 'создания записи данных ЧУ' используются двухмерные контуры, из которых можно создавать записи данных ЧУ (*.fc1).
* В редакторе контуров для 'выдавливаний' используются двухмерные контуры, из которых позже с помощью выдавливания можно смоделировать трехмерные тела (*.fc2).

!!! note "Замечание:"

    Обратите внимание на то, что последующее изменение свойства Тип контура ***невозможно***.

Созданный/обозначенный контур сохраняется как отдельный самостоятельный файл в формате основных данных соответствующего типа в специальном каталоге <Дисковод программы>: <Каталог программы>\Макросы\<Идентификатор фирмы>.
Помимо сохранения самого чертежа контура в отдельных случаях необходимо сохранить дополнительные данные, например машинную информацию для программ ЧУ или логику.

В общих чертежах контуров, создаваемых для заказанных клиентом несущих шин и т. д., может содержаться несколько непересекающихся контуров. В отличие от них чертеж контура, из которого будут создаваться записи данных ЧУ, может содержать только ***одну единственную замкнутую*** линию контура.

# Диалоговое окно Настройки: Автоматизированная обработка

* Вы открыли проект. Сервисные программы > Автоматизированная обработка. В диалоговом окне Выполнить: Автоматизированная обработка щелкните по кнопке ++...++ рядом с полем Схема.
* Проект > Управление. Вы считали в диалоговом окне Управление проектами каталоги и выделили, как минимум, один проект. Всплывающее меню > Автоматизированная обработка. В диалоговом окне Выполнить: Автоматизированная обработка щелкните по кнопке ++...++ рядом с полем Схема.

В этом диалоговом окне определяется последовательность действий, которые выполняются автоматически. Эта последовательность сохраняется в виде схемы для каждого пользователя, которую можно копировать, экспортировать и импортировать.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Описание

Данное поле отображает текст описания, который вы ввели при создании схемы в диалоговом окне Новая схема в поле Описание.

### Панель инструментов

Справа на панели инструментов рядом с полем Схема, помимо стандартных кнопок для схем, доступны следующие клавиши:

Кнопка |  Значение
---|---
![](images/all_importscript_as.png){: .ui-icon } (импорт сценария) |  Открывает диалоговое окно Импортировать сценарий из предыдущей версии. Позволяет импортировать файл сценария, который был создан в предыдущей версии EPLAN (версия 2.1 или более ранняя версия). При этом из файла сценария создается схема. Имя и описание схемы определяются во время импорта в диалоговом окне Новая схема.

### Доступные операции

В этом поле перечислены все операции, которые можно выбрать для автоматизированной обработки. Чтобы скопировать операцию в схему, при помощи кнопки ![](images/all_arrowright_as.png){: .ui-icon } (переместить направо) переместите ее в поле Выбранные операции.

### Выбранные операции

В этом поле отображаются все операции, содержащиеся в выбранной схеме.

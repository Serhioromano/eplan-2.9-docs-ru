# Генерировать объекты-заполнители

Условие:

В Графическом редакторе открыта страница проекта. Либо вы открыли пространство листа.

1. Выберите следующие пункты меню: Вставить > Объект-заполнитель
2. Выделите область с объектами, которые должны принадлежать объекту-заполнителю.
3. В диалоговом окне Свойства ++...++: Объект-заполнитель выберите вкладку Объект-заполнитель.
4. Введите имя объекта-заполнителя в поле Имя.

!!! info "Для сведения:"

    Для нового объекта-заполнителя в таблице на вкладке Присвоение сначала отображаются только объекты (функции, графические элементы, страница / пространство листа и т. д.), но не свойства.

5. Выделите на вкладке Присвоение в столбце Свойство объект из объекта-заполнителя и нажмите кнопку ![](images/all_new_as.png){: .ui-icon } (Создать), чтобы выбрать для этого объекта необходимые свойства и перенести их в таблицу.
6. Установите флажок Скрыть объекты без выбранных свойств, если в таблице следует отображать только объекты, для которых выбрано свойство.
7. Перейдите на вкладку Значения.
8. Выберите пункт всплывающего меню Новая переменная.
9. Введите в диалоговом окне Назвать новую переменную имя новой переменной, при этом EPLaN автоматически проверяет, допустимо ли введенное имя.

!!! info "Для сведения:"

    Новая переменная вставляется под заголовком столбца Переменная.

10. Выберите пункт всплывающего меню Новый набор значений.
11. Введите в заголовке заново созданного столбца имя набора значений, а в находящейся ниже ячейке таблицы — значение переменной.
12. Установите для переменной флажок в столбце Изменяемое, чтобы значения этой переменной можно было изменить при выборе набора значений.
13. Повторите эти шаги для всех генерируемых переменных и наборов значений.
14. Перейдите на вкладку Присвоение.
15. Выбрав свойство, щелкните в ячейке столбца Переменная правой кнопкой мыши, а затем выберите пункт всплывающего меню Выбрать переменную.
16. Выберите в следующем диалоговом окне требуемую переменную и щелкните по кнопке ++Ok++.

!!! info "Для сведения:"

    Переменная вставляется в форме <введенное_имя> в столбец Переменная.

17. Повторите это присвоение переменных для всех требуемых свойств.
18. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Новый объект-заполнитель автоматически размещается слева над ранее выделенной областью. На странице в графическом редакторе представлен объект-заполнитель со специальным символом "Якорь".

!!! info "Для сведения:"

    Новый объект-заполнитель в пространстве листа отображается не в виде символа "Якорь" в графике, а как пиктограмма якоря в структуре дерева навигатора пространства листа.

!!! note "Замечание:"

    * Групповая обработка ***нескольких*** объектов-заполнителей возможна, если все выбранные объекты-заполнители имеют одинаковые имена переменных и наборов значений. В таком случае в открывшемся диалоговом окне "Свойства" для объектов-заполнителей поля Имя и Описание, а также переменные и наборы значений на вкладке Значения можно обрабатывать совместно. Созданный с помощью кнопки ![](images/all_new_as.png){: .ui-icon } (Создать) набор значений или новая переменная добавляется для всех выбранных объектов-заполнителей.
    Если вышеуказанные условия не выполнены, то при многократном выборе объектов-заполнителей можно изменить ***только*** имя и описание.
    * Для того чтобы убрать все объекты-заполнители в проекте, в диалоговом окне Сжать проект имеется соответствующая настройка сжатия. Установите для этого флажок Объекты-заполнители (после записи Убрать размещенные устройства) и выполните сжатие.
    * Отображением символов "Якорь" на странице и в графическом редакторе можно управлять в диалоговом окне Управление слоями при помощи настройки Видимый для слоя EPLAN322, Символьная графика.Макрос.Объекты-заполнители.

!!! tip "Совет:"

    * Новую переменную можно определить прямо на вкладке Присвоение. Введите имя в нужной ячейке в таблице Переменная. Новая переменная всегда доступна как новая ячейка вкладки Значения для определения наборов значений.
    * Если в диалоговом окне Свойства ++...++: Объект-заполнитель присвоить переменную определенным свойствам, на вкладке Значения можно выбрать значение для этой переменной с помощью раскрывающегося списка или диалогового окна выбора с предварительно определенными значениями. Для этого щелкните соответствующую ячейку таблицы и откройте раскрывающийся список или вызовите соответствующее диалоговое окно выбора, нажав кнопку ++...++. Диалоговые окна для выбора значений доступны, например, в свойствах Номер изделия [1] – [50], Определение функции: Категория/Группа/Определение, Цвет / номер соединения, в свойствах для структурных идентификаторов и т. д.

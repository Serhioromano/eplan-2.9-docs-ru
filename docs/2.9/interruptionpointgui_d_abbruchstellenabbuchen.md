# Диалоговое окно Использ. имеющ. точку разрыва

Вы открыли проект.

* Вы находитесь на странице схемы соединений в графическом редакторе. Вставить > Символ соединения > Точка разрыва. Разместите точку разрыва и щелкните в диалоговом окне Свойства ++...++ на вкладке Точка разрыва в поле Видимое ОУ по кнопке ++...++.
* Данные проекта > Соединения > Навигатор точек разрыва. В диалоговом окне Точки разрыва — <Имя проекта> вы дважды щелкнули по точке разрыва или выделили точку разрыва и выбрали пункт всплывающего меню Свойства. Щелкните в диалоговом окне Свойства ++...++ на вкладке Точка разрыва в поле Видимое ОУ по кнопке ++...++.

Видимое имя очки разрыва обозначает связанность точек разрыва: Все точки разрыва с одинаковым именем связаны друг с другом. При этом это имя может состоять из имени потенциала (например, "L1", "L+", "PE", "PEN", "N" и т. д.) или содержать данные о принципе действия (например, "Not", "+15V", имя сигнала), или просто состоять из буквенно-цифровых символов.

Точки разрыва настраиваются с помощью имени для конкретного проекта, т.е. каждый проект имеет список точек разрыва, которые помечаются именами. Поэтому при вставке точек разрыва в схему соединений новой точке разрыва можно присвоить уже имеющееся имя.

Доступны следующие элементы диалогового окна:

### Список

В представлении в виде списка выводятся использованные в проекте блоки идентификаторов (т.е. "Установка", "Место установки" и т.д.) и обозначения устройств, а также, при наличии - описание источника точки разрыва.

### Дерево

В представлении в виде дерева в иерархической структуре выводится информация "Блок идентификаторов" и "Имя точки разрыва", а ниже выводятся данные об источнике и цели.

!!! note "Замечание:"

    Если при использовании имени точки разрыва оно выделено, то используется только оно, а все другие данные игнорируются.
    Если, напротив, выбрана определенная функция, то данные вкладки Данные символа/функции и Отображение остаются неизменными, но используются все доступные для записи свойства выбранной функции, за исключением свойств, относящихся к отображению, в категории "Настройки", а также полей Идентификатор сортировки и Источник типа "звезда".

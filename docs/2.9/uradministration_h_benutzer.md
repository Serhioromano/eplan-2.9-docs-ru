# Управление пользователями

Управление правами позволяет создавать пользователей и распределять их по группам. При необходимости созданные пользовательские профили можно будет изменить или удалить.

!!! note "Замечание:"

    Невозможно изменить или удалить предварительно определенных в EPLAN пользователей "ADMINISTRATOR" и "GUEST".

Условие:

Открыто управление правами: Сервисные программы > Управление правами

1. Выделите слева в диалоговом окне Управление правами папку Пользователь и нажмите ![](images/all_new_as.png){: .ui-icon } (Создать).
2. В открывшемся диалоговом окне Пользователь введите имя и краткое описание нового пользователя.

!!! note "Замечание:"

    В EPLAN имя пользователя управляется всегда в верхнем регистре (как и в Windows). Это значит, что в EPLAN пользователи "Дмитрий Иванов" и "ДМИТРИЙ ИВАНОВ" — это одно и то же лицо с теми же правами.

3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Пользователь отображается в диалоговом окне Управление правами, но ему еще не присвоены права на обработку, т. к. он не принадлежит ни к одной из групп пользователей.

4. Присвойте пользователю одну или несколько групп, следуя инструкциям, приведенным далее в разделе "Обработка пользователя".

!!! info "Для сведения:"

    Пользователю присваиваются соответствующие права на обработку.

5. При необходимости присвойте пользователю пароль, как описано в "Ввести или изменить пароли".

1. Выделите слева в диалоговом окне Управление правами папку Пользователь и щелкните ![](images/all_import_as.png){: .ui-icon } (Импортировать пользователей и группы из системы Windows).
2. В диалоговом окне Выбрать пользователей или группы выберите требуемого пользователя Windows или группу Active Directory.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Пользователь отображается в диалоговом окне Управление правами, но ему еще не присвоены права на обработку, т. к. он не принадлежит ни к одной из групп пользователей.

!!! info "Для сведения:"

    Импортированные из системы Windows группы Active Directory отображаются и управляются как пользователь.

4. Присвойте пользователю системы Windows или группе пользователей Active Directory одну или несколько групп EPLAN, следуя инструкциям, приведенным далее в разделе "Обработка пользователя".

!!! info "Для сведения:"

    Пользователю присваиваются соответствующие права на обработку. В случае с группой пользователей Active Directory все члены этой группы получают соответствующие права на обработку. Не нужно создавать и обрабатывать отдельного пользователя для каждого члена группы.

!!! note "Замечание:"

    * Для пользователей, импортированных из системы Windows, регистрация осуществляется с помощью пароля Windows. Изменить пароль здесь невозможно. Имя также не может быть изменено.
    * Если импортированный из системы Windows пользователь уже существует как пользователь, созданный в EPLAN, то после импорта таких пользователей будет два. В таком случае используется только пользователь системы Windows, а ненужного пользователя EPLAN следует удалить.

1. Выделите слева в диалоговом окне Управление правами папку Пользователи.
2. Выделите в правом поле (самоопределяемое) имя пользователя и нажмите на ![](images/all_edit_as.png){: .ui-icon } (Обработать).
3. Измените в диалоговом окне Пользователь имя и/или описание и нажмите ++OK++.

!!! info "Для сведения:"

    Диалоговое окно закрывается, а измененные данные выводятся на просмотр в диалоговом окне Управление правами.

1. Выделите слева в диалоговом окне Управление правами папку Пользователи.
2. Выделите в правом поле (самоопределяемое) имя пользователя и нажмите на ![](images/all_delete_as.png){: .ui-icon } (Удалить).

!!! info "Для сведения:"

    После подтверждения предупреждающего сообщения имя пользователя будет удалено.

1. В диалоговом окне Управление правами щелкните рядом с папкой Пользователи на знак плюса, чтобы открыть папку, и выделите имя пользователя.

!!! info "Для сведения:"

    Справа в диалоговом окне будут показаны имеющиеся группы пользователей.

2. Установите флажок перед группами, к которым необходимо отнести выделенного пользователя.

!!! info "Для сведения:"

    Пользователю будут присвоены права на обработку соответствующих групп. Присвоение будет сохранено после того, как вы закроете диалоговое окно нажатием кнопки ++OK++.

1. Выделите слева в диалоговом окне Управление правами папку Пользователи.
2. Выделите в правом поле пользователя и нажмите ++Изменить пароль++.
3. Введите в диалоговом окне Изменить пароль (новый) пароль в поле Новый пароль. Во избежание опечаток при вводе пароль необходимо повторно ввести в поле Подтвердить новый пароль.

!!! info "Для сведения:"

    Для защиты введенных данных пароль отображается не открытым текстом, а в виде символов *.

4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Пользователю присваивается введенный пароль, и в будущем он должен ввести его в диалоговое окно Регистрация в EPLAN.

# Диалоговое окно Отображения отверстий — <Имя проекта>

Вы открыли проект.

Для того, чтобы отобразить в проекте имеющиеся двухмерные отображения отверстий, выберите пункты меню Сервисные программы > Отчеты > 2D-отображения отверстий.

В этом диалоговом окне через вкладку Отчеты вы осуществляете управление всеми двухмерными отображениями отверстий в открытом проекте.

Обзор основных элементов диалогового окна:

### Отображения отверстий

В этой структуре дерева в качестве самого высокого слоя структуры отображается тип отчета "Отображения отверстий". Второй слой структуры отображает содержащиеся в проекте пространства листа. Под каждым узлом пространства листа отображаются двухмерные отображения отверстий, присвоенные этому пространству листа, в том числе страницы, на которых размещается отчет.

### Всплывающее меню

Пункт меню |  Значение
---|---
Обновить |  Изменения объектов в пространстве листа не отображаются в отчете автоматически. Функциональность Обновить сравнивает двухмерное отображение отверстия с текущими данными проекта в пространстве листа и исправляет изображение в случае изменения объектов.
Удалить отображение отверстия |  Удаляет все двухмерные отображения отверстий под выделенным узлом.
Открыть страницу (только отображения отверстий узла) |  Открывает текущую выделенную страницу с двухмерным отображением отверстий в графическом редакторе. Диалоговое окно Отчеты — <Имя проекта> закроется.

### Имя свойства / Значение

В этих столбцах отображаются свойства и значения выделенных двухмерных отображения отверстий / разверток шины.

### Обзор модели: Фиксированный

Если поставлен флажок данного свойства, соответствующее двухмерное отображение отверстия блокируется от обновления. Изображение остается без изменений, даже если были изменены объекты в пространстве листа. При помощи этой настройки можно зафиксировать различные состояния обработки в двухмерных отображениях отверстия.

### Обзор модели: ширина ребер

С помощью этого свойства можно задавать фиксированное значение для ширины линий ребер в обзоре модели, 2D-отображении отверстий и развертке шины, например 0,5 мм. При масштабировании и изменении масштаба ширина линий изменяется соответственно. При экспорте PDF линии ребер выводятся с заданной шириной.

### [Обновить]

Изменения объектов в пространстве листа не отображаются в отчете автоматически. Кнопка Обновить сравнивает двухмерное отображение отверстия с текущими данными проекта в пространстве листа и исправляет изображение в случае изменения объектов.

# Копировать страницы между различными проектами

Страницы можно копировать как в пределах одного проекта, так и между разными проектами. При этом можно копировать отдельные страницы или целые блоки страниц.

Условие:

Вы выделили как минимум одну страницу в диалоговом окне навигатора Страница — <Имя проекта>.

1. Выберите пункты меню Страница > Копировать из / в.

!!! info "Для сведения:"

    В диалоговом окне Копировать страницы выводятся исходный и целевой проект в виде двух деревьев. В правом поле Текущий проект выведен открытый в навигаторе страниц текущий и ***выделенный*** проект, в который будет выполнено копирование страниц. Сначала в обоих полях выводится текущий проект, так чаще всего выполняется копирование в пределах одного проекта.

2. В левом поле Выбранный проект выберите исходный проект, из которого требуется выполнить копирование. Это может быть уже открытый проект или еще не открытый проект. Нажав кнопку ++...++, можно открыть диалоговое окно для выбора проекта; щелкните в нем по кнопке ![](images/all_fileopen_as.png){: .ui-icon } (Открыть), чтобы выбрать любой проект в любом каталоге.
3. При необходимости настройте [фильтр](modaldialogsdb_d_filternnach.md) для текущего проекта, чтобы вывести только определенные страницы проекта.
4. Выделите в левом поле Выбранный проект страницы, которые необходимо скопировать. Также можно выделить уровень структуры, тем самым выделив все страницы внутри этого уровня структуры.

!!! info "Для сведения:"

    В таблице Свойства выводятся свойства (первой) выделенной страницы.

5. Скопируйте выделенные страницы с помощью кнопки со стрелкой в целевой проект в правое поле Текущий проект.

!!! info "Для сведения:"

    В диалоговом окне Адаптировать структуру на вкладке [Страницы](pagebrowsergui_r_seitenstruktur.md) перечислены номера страниц, описания страниц и структурные идентификаторы исходной и целевой страниц.

Номера страниц, описания страниц и структурные идентификаторы можно оставить или изменить. Если такие страницы уже имеются в целевом проекте, автоматически устанавливается флажок Перезаписать. Снимите его, если данные страницы не требуется перезаписывать, и измените свойства страниц.
6. Нажав кнопку ++Нумеровать++, можно открыть диалоговое окно [Нумеровать страницы](pagebrowsergui_d_seitennummerierung.md) и заново пронумеровать скопированные страницы.
7. Если скопированные ОУ содержат структурные идентификаторы, можно изменять на вкладке [ОУ](pagebrowsergui_r_ortkaestenstruktur.md) видимые ОУ.
8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Если в целевом проекте содержатся страницы с такими же именами, которые могут быть перезаписаны, выводится соответствующий запрос. Щелкните по кнопке ++Да++, чтобы перезаписать страницы в целевом проекте. Щелкните по кнопке ++Нет++ и вернитесь к диалоговому окну Адаптировать структуру, где можно изменить введенные данные.

!!! info "Для сведения:"

    Если в пункте меню Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация (online) Вы выбрали опцию С запросом, будет выведено диалоговое окно Режим вставки. Здесь можно определить, каким образом должна быть выполнена нумерация ОУ на скопированных страницах.

9. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Страницы вставляются в целевой проект, а структурные идентификаторы и ОУ изменяются в соответствии с введенными данными.

!!! tip "Совет:"

    В навигаторе страниц страницы можно копировать и перемещать с помощью перетаскивания мышью. Для копирования удерживайте клавишу ++Ctrl++ нажатой.

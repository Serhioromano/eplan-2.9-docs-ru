# Экспорт данных изготовления / маркировка

Для оптического обозначения и идентификации устройств и соединений по месту нахождения установки необходимо экспортировать данные изготовления и выполнить их маркировку. То есть на устройства наклеиваются, например, этикетки и таблички. Для маркировки этикеток и табличек используется информация непосредственно из EPLAN:

* Вся идентифицирующая и описывающая информация функциональных элементов и соединений может быть подготовлена для использования в качестве экспорта данных изготовления / маркировки.
* Вы можете сохранять настройки для экспорта данных изготовления / вывода маркировки в пользовательских схемах, чтобы использовать их многократно.
* В комплект поставки входят фиксированные схемы, которые можно использовать в соответствии с индивидуальными требованиями.
* При выводе можно выбрать требуемый язык.
* Для вывода можно использовать файлы в формате *.txt и Excel. В каждой схеме можно указать шаблон Excel, который сразу после вывода запускает Excel. При этом новый файл сразу загружается в Excel. Таким образом можно подготавливать формы в Excel, которые соответствуют определенному выводу.

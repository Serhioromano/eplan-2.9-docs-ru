# Фильтры и предварительный выбор изделия

Для рационального использования фильтров необходимо вначале соответствующим образом подготовить данные в базе данных изделий. Фильтры действуют только на те данные, которые внесены в полях базы данных изделий, но все же не на данные из определений функции.

При выборе изделий можно указывать различные фильтры, их можно также комбинировать друг с другом:

* ***Фильтр по полю:*** С помощью фильтра по полю можно выполнить поиск только в определенных полях записей данных (например для поля Номер типа). При использовании этого фильтра проверяется, совпадает ли все содержимое поле с искомым текстом.
* ***Полнотекст. фильтр:*** Через этот фильтр можно ввести любой искомый текст, поиск которого осуществляется во всех полях базы данных изделий.
Обратите внимание, что полнотекстовый фильтр в отображении ***Представление в виде дерева*** также будет использоваться на главных узлах "Список принадлежностей", "Размещение принадлежностей", "Схема сверления", "Схема соединений". На вкладках Список и Комбинация, наоборот, будут учитываться только данные изделия.
* ***Автоматический фильтр:*** Определенные свойства выделенного устройства применяются в качестве критериев фильтрации.

Настройки фильтра сохраняются относительно к пользователю.

### Предварительный выбор изделий

Дополнительно к фильтрам вы можете создать и применить предварительный выбор изделий, также список изделий. При выборе изделий затем предлагаются только изделия из этого списка. Можно создавать различные списки и соответственно сохранять под каким-либо именем, так что их всегда можно применить.

Списки предварительного выбора изделия сохраняются для конкретного проекта. При помощи функции экспорта и импорта можно все же их также использовать в других проектах.

#### Выбор кабеля

При выборе кабеля имеется дополнительный фильтр для свойств жилы кабеля. Кроме того, можно определить специальный предварительный список изделий для изделия кабеля.

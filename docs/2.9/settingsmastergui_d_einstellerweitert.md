# Диалоговое окно Экспортировать настройки (расшир.)

Параметры > Настройки. Выделите одну из областей настроек "Пользователь" или "Рабочая станция" и выберите во всплывающем меню пункт Экспортировать (расширить).

В этом диалоговом окне выполняется дополнение уже созданного файла экспорта другими областями настройки.

Обзор основных элементов диалогового окна:

### <Путь иерархии>

Текст рядом с флажком указывает на то, какая область настройки выделена в диалоговом окне Настройки в представлении в виде дерева. Если этот флажок установлен, выделенная область настройки и все нижестоящие области настройки экспортируются в имеющийся файл экспорта.

### Другие области настройки

Здесь указывается, какие другие настройки программы и конфигурации также следует экспортировать. Если этот влаг установлен, то выполняется их экспорт. Доступность флажков зависит от того, какая область настройки выделена в диалоговом окне Настройки в представлении в виде дерева.

!!! note "Замечание:"

    Эти дополнительные настройки доступны только для областей настроек "Пользователь" и "Рабочая станция"; поэтому для настроек проекта и фирмы описанные далее флажки недоступны и не могут быть установлены.

* Конфигурация свойств: Экспортирует конфигурации свойств устройств, страниц и т. д., заданные соответствующим образом в диалоговом окне Свойства ++...++ через пункт всплывающего меню Конфигурировать.
* Диалоговая память: Экспортирует из диалоговых окон, например, размер, положение на экране, разрешение экрана, статус "Свернуто" / "Развернуто" и настройки ширины столбцов.

!!! note "Замечание:"

    При импорте / экспорте настроек для диалоговой памяти рекомендуется установить одинаковое разрешение экрана на разных рабочих станциях. В случае разного разрешения экрана диалоговые окна могут выйти за пределы видимой области экрана.

* Конфигурации дерева и списка: Экспортирует все конфигурации для представлений навигаторов в виде дерева и в виде списка, которые были заданы через пункт всплывающего меню Конфигурировать представление.
* Внешние программы: Экспортирует список для быстрого доступа к внешним программам, которые были заданы в пунктах меню Параметры > Внешние программы.
* Фильтр: Экспортирует все ***определенные пользователем*** настройки фильтра.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

### Целевой файл

В этом поле указывается файл экспорта, который необходимо дополнить другими областями настройки. По умолчанию файл экспорта получает имя, которое соответствует выделенной области в дереве диалогового окна Настройки. Имя файла состоит из следующих элементов: <Основная категория>+<Подкатегория>+<Область настройки>.xml. Нажав кнопку ++...++, можно указать другой путь или имя файла экспорта.

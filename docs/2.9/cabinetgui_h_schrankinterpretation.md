# Представить и перенести логику 3D-макросов

При помощи функции Автоматическая интерпретация перенесите имеющуюся логику любого 3D-макроса (электрошкаф, компоненты шкафа, механические элементы, контактор и т. д.) на другой заново определяемый 3D-макрос такого же или подобного типа. Таким образом можно, например, значительно упростить и ускорить рабочие шаги по созданию собственной серии электрошкафов.

Условия:

* Вы открыли проект.
* Проект содержит полностью определенный электрошкаф.
* Навигатор пространства листов открыт, и одно пространство листов открыто.
* Пространство листа содержит импортированные трехмерные тела, которые необходимо сохранить в виде электрошкафа.
* Импортированные трехмерные тела при помощи функций Обработать > Логика устройства > Поверхность установки > Определить и Обработать > Логика устройства > Поверхность установки > Поворачивать были приведены в верное, выровненное по оси X / Y, положение установки.

1. Выберите пункты меню Обработать > Логика устройства > Автоматическая интерпретация.

2. Опишите прямоугольник вокруг трехмерных объектов для нового электрошкафа.
3. Выберите в представлении в виде дерева диалогового окна Выбрать шаблон интерпретации пространство листа и в нем электрошкаф, свойства которого необходимо перенести на трехмерную геометрию.
4. Если выбрать несколько различных объектов шаблонов и щелкнуть по кнопке ++OK++, будет запущена проверка соответствия, в ходе которой будет определено, какие функциональные элементы наиболее подходят в качестве объекта шаблонов.

!!! info "Для сведения:"

    Результаты отображаются на экране с указанием степени соответствия в процентах в диалоговом окне Список соответствия.

5. В списке соответствия выберите самый подходящий функциональный элемент. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выбранный функциональный элемент копируется в диалоговое окно Выбрать шаблон интерпретации.

6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Трехмерная геометрия в навигаторе объединена в узле 'Электрошкаф'.

!!! info "Для сведения:"

    Определение функционального элемента модели электрошкафа переносится на новый электрошкаф.

!!! info "Для сведения:"

    На всех определениях функционального элемента генерируются логические элементы модели электрошкафа (точки захвата, монтажные поверхности, точки монтажа).

7. Создайте [трехмерный макрос](cabinetgui_h_3dmakroerstellen.md) из нового электрошкафа, который можно присвоить соответствующему изделию в базе данных изделий.

!!! tip "Совет:"

    Сначала можно выбрать объекты, а затем запустить функцию Автоматическая интерпретация.

# Функции ПЛК для Fluid-техники

EPLAN Fluid предоставляет новые определения функций, такие как "Блок ПЛК (Fluid-техника)" или "Вывод устройства ПЛК (Fluid-техника)", с помощью которых можно реализовать функциональные элементы ПЛК для Fluid-техники.

![](images/benefit.png)Эффект:

С помощью новых определений функций функции ПЛК можно проектировать отдельно по разделам. Устройства ПЛК, спроектированные во Fluid-технике, могут быть заменены на программы конфигурации ПЛК, например Siemens TIA Portal. Это обеспечивает бесперебойный обмен между разделами даже в смешанных проектах.

При вставке на схему соединений блока или вывода устройства ПЛК программа проверяет, на странице какого типа это происходит. Если речь при этом идет о странице типа "Схема соединений, Fluid-техника", автоматически устанавливается соответствующее определение функции и вы вставляете "Блок ПЛК (Fluid-техника)" или "Вывод устройства ПЛК (Fluid-техника)".

Функции ПЛК для Fluid-техники работают так же, как и соответствующие функции для электротехники.

* Функции ПЛК для Fluid-техники отображаются в диалоговых окнах ПЛК и отчетах и учитываются при обмене данными ПЛК.
* Соответствующие адреса ПЛК отображаются в списке присвоений и доступны для обработки.
* Прилагаемые фильтры в навигаторе ПЛК учитывают функции ПЛК как для электротехники, так и для Fluid-техники.
* Функции ПЛК для Fluid-техники также учитываются при выполнении контрольных прогонов класса "ПЛК / шина".

#### Нумерация

Нумерация функций ПЛК Fluid-техники выполняется в соответствии с настройками устройства ПЛК и, следовательно, может отличаться от нумерации устройств Fluid-техники. Благодаря этому в смешанных проектах, которые содержат страницы схем соединений как Fluid-техники, так и электротехники, компоненты ПЛК нумеруются одинаково.

#### Перенос ОУ

При переносе ОУ функции ПЛК для Fluid-техники работают так же, как и все другие устройства Fluid. Если [перенос ОУ в устройствах Fluid](prjmanagementgui_r_vererbung.md) не доступен, функции ПЛК для Fluid-техники не перенимают обозначение ОУ и обозначение штекера. В этом случае ОУ и обозначение штекера необходимо вводить по отдельности на всех выводах устройства ПЛК. При необходимости используйте для этого табличную обработку.

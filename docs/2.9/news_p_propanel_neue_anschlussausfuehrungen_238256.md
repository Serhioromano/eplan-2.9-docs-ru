# Новые категории соединений для схем соединений

В диалоговом окне Настройки: Категория соединения задаются категории соединений, которые по умолчанию предлагаются и используются в базе данных изделий при управлении схемами соединений. В этом диалоговом окне теперь доступны новые предварительно определенные категории соединений в соответствии с классификацией eCl@ss. При создании новых схем соединений мы рекомендуем использовать только новые категории соединений.

В это диалоговое окно можно зайти через путь меню Параметры > Настройки > Фирма > Управление > Категория соединения.

Следующие категории соединений являются новыми и соответствуют классификации ***eCl@ass***:

* Винтовое подключение
* Пружинная клемма
* Вставное соединение
* Сквозное соединение
* Штепсельное соединение
* Соединение с клеммой штепсельного типа
* Болтовое соединение
* Соединение с помощью плоского штекера
* Беспаечный контакт
* Ленточный вывод
* Шинное соединение
* Лепестковый вывод для припайки
* Соединение, полученное методом накрутки

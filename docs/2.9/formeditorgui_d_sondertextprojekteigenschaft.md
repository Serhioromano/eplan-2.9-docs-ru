# Диалоговое окно Свойства (спец. текст): Свойства проекта

Вы открыли проект. Сервисные программы > Основные данные > Рамка / Форма > Открыть > [Открыть].

* Вставить > Спец. текст > Свойства проекта.
* Вы дважды щелкнули по свойству проекта (или выбрали Всплывающее меню > Свойства; в этом случае могут быть выделены также несколько свойств проекта).

В этом диалоговом окне выберите свойство проекта, которое должно быть добавлено в форму или рамку, и установите его свойства для формы, языка и рамки. Эти специальные тексты заполняются в соответствии с введенными данными программы.

# Дополнение данных ПЛК перемещено в корректировку проекта

Под пунктами меню Данные проекта > ПЛК > Дополнить данные раньше были доступны несколько действий для синхронизации данных ПЛК и подготовки проекта для обмена данными ПЛК. Этот пункт меню был удален, а вместо него в новой версии доступны новые действия для корректировки проекта. Таким образом, данные ПЛК теперь всегда будут дополняться для всего проекта.

![](images/benefit.png)Эффект:

Настройки для дополнения данных ПЛК теперь доступны при корректировке проекта (аналогично корректировочным настройкам для клемм), их можно там сохранять на схеме и постоянно использовать. Так как корректировка всегда выполняется для всего проекта, наличие неполных или противоречивых данных исключено.

В диалоговом окне Настройки: Корректировать можно указать, какие данные проекта нужно исправить с помощью корректировочного прогона автоматически. Путь меню к этому диалоговому окну: Параметры > Настройки > Проекты > "Имя проекта" > Управление > Корректировать. Здесь доступен новый уровень иерархии ПЛК с такими настройками:

* Записать обратно обозначения выводов устройства
Выбор отдельных ЦПУ теперь невозможен. Диалоговое окно Записать обратно обозначения выводов устройства было удалено.
* Выполнить обратную запись обозначений вывода устройства и перенести на подключенные контакты штекера / клеммы
* Установить типы данных
* Синхронизировать имена интерфейсов шины
* Синхронизировать имена физических сетей
* Удалить лишний проект конфигураций на выводах шины
С помощью этой новой операции можно удалить проект конфигураций, введенный для выводов шины, если он соответствует проекту конфигураций связанного блока ПЛК. Тогда проект конфигураций вывода шины будет определен связанным блоком ПЛК и автоматически обновится в случае изменений.

Операция Записать обратно проект конфигураций на выводах шины уже не требуется, поскольку проект конфигураций на выводах шины теперь определяется связанным блоком ПЛК автоматически. Соответствующий пункт меню был удален без замены.

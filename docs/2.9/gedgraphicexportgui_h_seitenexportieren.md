# Вывести страницы в виде графических файлов

Страницы проекта можно вывести в следующих графических форматах: BMP, GIF, JPG, PNG и TIFF.

При экспорте ***отдельной*** страницы предусмотрена возможность произвольного присвоения имени файла. При экспорте ***нескольких*** страниц невозможна свободная передача имени файла. В этом случае графические файлы в качестве имени используют полное имя страницы.

Условие:

Вы открыли проект.

1. Страница > Навигатор
2. Выделите в диалоговом окне Страницы — <Имя проекта> те страницы проекта, которые требуется экспортировать (или весь проект).
3. Выберите пункты меню Страница > Экспортировать > Графический файл.

!!! info "Для сведения:"

    EPLAN открывает диалоговое окно Экспорт графических файлов. Выбранные файлы отобразятся в поле Источник.

4. Выберите в поле Схема требуемую схему для экспорта графических файлов.
5. При необходимости измените указанный целевой каталог и укажите, следует ли экспортировать все страницы проекта или только выделенные страницы.
6. При экспорте ***отдельной*** страницы в поле Имя файла введите имя генерируемого файла или выберите его в интерактивном режиме при помощи кнопки ++...++.
7. Установите флаг Черно-белый вывод, чтобы все элементы на странице проекта были выведены черным цветом на белом фоне.
8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Каждая страница сохраняется в указанном целевом каталоге в виде отдельного графического файла.

!!! info "Для сведения:"

    Если выделен проект, в целевом каталоге генерируется подкаталог с именем проекта и графические файлы сохраняются в этом подкаталоге.

!!! tip "Совет:"

    В графическом редакторе можно экспортировать и текущую открытую страницу, использовав путь Страница > Экспортировать > Графический файл.

!!! example "Пример:"

    В диалоговом окне Страницы — <Имя проекта>выделен образец проектаESS_Sample_Project, страницы которого необходимо экспортировать в виде файлов формата TIFF.В целевом каталоге генерируется подкаталогESS_Sample_Project. В этом каталоге в виде файлов формата TIFF сохраняются все страницы проекта.

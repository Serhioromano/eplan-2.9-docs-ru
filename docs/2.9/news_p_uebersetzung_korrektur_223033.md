# Корректировка не зависящих от языка текстов в базе данных изделий

В настройках пользователя для перевода в диалоговом окне Настройки: База данных изделий можно указать, какие многоязычные поля в базе данных изделий подлежат переводу. Теперь здесь появилась возможность преобразовывать тексты, которые исключены из перевода, но для которых текст перевода был введен ошибочно, обратно в тексты, не зависящие от языка. Таким образом, эти тексты по-прежнему будут отображаться при изменении исходного языка. Для этого в настройки пользователя для перевода была добавлена кнопка ++Корректировка++, известная из настроек проекта. Путь меню для этой настройки: Параметры > Настройки > Пользователь > Перевод > База данных изделий.

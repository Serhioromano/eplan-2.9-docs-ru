# Встроенные отчеты

Встроенный отчет является таким отчетом, который вы вручную размещаете в имеющуюся страницу проекта (I). Не создаются отдельные [страницы отчета](formgeneratorgui_k_auswertungsseiten.md). Данные проекта можно вывести в качестве встроенного отчета, если при создании отчетов как формы вывода выбрано "Размещение вручную". Отчет вы можете размещать в любом месте на странице проекта. При обновлении отчетов актуализируются и встроенные отчеты.

!!! note "Замечание:"

    * При копировании и вставке устройств и соответствующих относящихся к функции встроенных отчетов сохраняется ссылка между устройствами и скопированным относящимся к функции встроенным отчетом. То же самое действует и при создании соответствующих макросов с относящимися к функции встроенными отчетами. Встроенный отчет обновляется непосредственно при вставке. Благодаря этому сразу распознается, к какому устройству он относится.
    * Для встроенного отчета можно выбрать форму вывода "Размещение вручную", только если открыта нужная страница проекта в графическом редакторе.

# Диалоговое окно Результаты измерений

Пространство листа > Измерить.

В данном диалоговом окне отображаются результаты, полученные при помощи функции Измерить.

Обзор основных элементов диалогового окна:

### Результат "Длина"

показывает наименьшее расстояние между двумя измеренными элементами.

### Результат dx/dy/dz

показывает расстояние до обоих измеренных элементов при измерении по осям X, Y и Z.

### 1-й элемент X/Y/Z

показывает расстояние до первого измеренного элемента при измерении по осям X, Y и Z.

Данное значение отображается только тогда, когда для измерения выбраны точки.

### 2-й элемент X/Y/Z

показывает расстояние до второго измеренного элемента при измерении по осям X, Y и Z.

Данное значение отображается только тогда, когда для измерения выбраны точки.

### Параметры: Параллельно монтажной поверхности

координаты и расстояния вычисляются относительно нулевой точки монтажной поверхности, на которой размещены оба измеряемых элемента. Расстояния по оси Z при этом игнорируются и приравниваются к 0. Если данный флажок снят, то отображаются и расстояния по оси Z.

Если у элементов нет общей монтажной поверхности, данная функция деактивирована. В данном случае координаты и расстояния измеряются относительно нулевой точки пространства листа.

Если этот параметр отключен, все значения находятся относительно нулевой точки единой системы координат.

!!! example "Пример:"

    Измерить Результат Измеренные точки Параллельно монтажной поверхности: ВКЛ.Параллельно монтажной поверхности: ВЫКЛ.

# Форма для схемы кабельных соединений

Форма для схемы кабельных соединений содержит таблицу с данными кабелями. Внутри формы можно начертить детальное представление используемого стандартного кабеля, включая монтаж по концам и экранирования. Кроме того, через открывшуюся точку вставки следующей формы можно вставить групповую спецификацию изделий, она может располагаться на той же странице или на следующей. Для каждого стандартного кабеля вы должны создать и ввести новую форму.

Схема кабельных соединений может быть создана также при помощи динамической формы. При этом на каждый тип кабеля создается по отдельному отчету.

!!! note "Замечание:"

    При создании схем кабельных соединений все кабели, для которых записана одинаковая форма схемы кабельных соединений, выдаются вместе в ***одном*** блоке отчета. Возможная сортировка учитывается только внутри блока отчета.

### Данные для формы кабельных соединений

На странице кабельных соединений выводятся свойства графически первого кабеля как данные заголовка. Кабели копируются в их последовательности в схеме соединений; неразмещенные кабели, рассортированные по ОУ, вставляются ниже. Следующие данные отдельных кабелей выводятся в форме таблицы:

* Все данные, которые выводятся также в перечне кабелей (например: ОУ кабеля, источник, цель, число кабельных соединений и т. д.)
* Изделие кабеля (Подгруппа продуктов "Изделие, источник")
* Изделие кабеля (Подгруппа продуктов "Изделие, цель")
* Изделие кабеля (все другие подгруппы продуктов).

### Данные для групповой спецификации изделий

Групповая спецификация изделий выводится как последующая форма. Здесь выводятся изделия всех кабелей, выводимых в схемах кабельных соединений с соответственно суммированными числами штук. При этом доступны все заполнители из "стандартной" групповой спецификации изделий.

Дополнительно существует следующий заполнитель, ***Сумма длин кабелей***. Сумма длин кабелей суммирует все длины кабелей для одного изделия кабеля, которые должны учитываться в блоке отчетов и в которые введено это изделие кабеля. Каждый кабель учитывается только один раз.

Длина выводится в стандартной единице для длин кабелей.

Система распознает изделие кабеля лишь потому, что группа продуктов 'Кабель' и подгруппа продуктов 'Не определено' установлены на изделии.

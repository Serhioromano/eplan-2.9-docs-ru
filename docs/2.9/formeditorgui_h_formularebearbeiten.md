# Обработать формы

Как только Вы открыли форму, которую хотите обработать, в редакторе форм, Вы можете начать обрабатывать ее элементы.

Условия:

* Вы открыли проект.
* В редакторе форм открыта та форма, которую требуется обработать (Сервисные программы > Основные данные > Форма > Открыть > [Открыть]).

### Обработать статические элементы

Статические элементы не влияют на данные, которые генерирует программа при оценке логики схемы соединений. Например, это рамки — такие графические объекты, как прямоугольники, линии, вставленные побитовые изображения и т. д. Статические элементы обрабатываются точно так же, как соответствующие объекты на страницах схемы соединений. Т.е. их можно заполнять, перемещать, перетаскивать на другую позицию и присваивать им определенные свойства, например тип линии, выравнивание текста и т.д.

1. Выберите пункт меню Вставить > Графика и один из последующих пунктов меню, чтобы вставить (другие) графические элементы, например прямоугольники, линии и т. д.
2. Для сохранения текстовой информации выберите пункты меню Вставить > Графика > Текст.
3. Выберите пункты меню Вставить > Графика > Графический файл, чтобы в диалоговом окне Выбрать графический файл выбрать графику и, например, вставить ее как логотип в нижний колонтитул формы.
4. Дважды щелкните по уже заполненному элементу, чтобы открыть соответствующее диалоговое окно Свойства и просмотреть или изменить свойства.

!!! tip "Совет:"

    Текст можно вставить перетаскиванием мышью, перетащив выделенный текст из приложения Office в открытую форму.

### Вставить тексты-заполнители

Вставить > Текст заполнитель

Тесты-заполнители применяются в форме для того, чтобы определить позиции для размещения полей в одной записи данных. Они анализируются по каждому объекту и автоматически размножаются, при этом позиция устанавливается при помощи фиксированной подачи в направлении X и Y. Подачу вы устанавливаете в свойствах формы Ширина столбца и Высота строки. Тексты-заполнители заполняются только при прогоне отчета и не активируются в режиме подключения.

!!! note "Замечание:"

    Если у заполнителя нет блока выравнивания или блок выравнивания есть и одновременно установлен флажок Убрать переходы на новую строку (в свойстве Рамки), то в отчетах переходы на новую строку отфильтровываются.

### Вставить специальные тексты

Вставить > Спец. текст

Специальные тексты для форм являются специальными заполнителями для свойства проекта и страниц. Они автоматически заполняются согласно введенным данным соответствующими свойствами.

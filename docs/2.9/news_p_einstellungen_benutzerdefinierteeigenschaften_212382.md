# Новая настройка для новых определенных пользователем свойств из макросов

До сих пор новым определенным пользователем свойствам, которые, например, добавлялись в проект путем вставки и копирования макросов, автоматически назначалась настройка Больше не использовать. Данные свойства не отображались в связанных объектах и удалялись при повторном сжатии проекта. В новой версии теперь можно решать, будут ли новые определенные пользователем свойства использоваться в текущем проекте или нет.

При этом в специфических для проекта настройках графической обработки теперь доступен новый флажок Не использовать новые конфигурации свойств из макросов. (Путь меню: Параметры > Настройки > Проекты > "Имя проекта" > Графическая обработка > Общее.)

По умолчанию этот флажок снят. В этом случае новые определенные пользователем свойства остаются доступными в проекте и отображаются в связанных объектах.

Если для новых определенных пользователем свойств нужно использовать предыдущие параметры, установите этот флажок.

![](images/benefit.png)Эффект:

С помощью новой настройки Не использовать новые конфигурации свойств из макросов теперь можно установить, будут ли использоваться новые определенные пользователем свойства из макросов или других основных данных. Если используется настройка по умолчанию, новые определенные пользователем свойства будут видны непосредственно в проекте, что облегчит их поиск.

!!! note "Замечание:"

    Новые определенные пользователем свойства можно также вставить с помощью других основных данных, например форм, рамок и т. д. Эти свойства также учитываются при настройке.

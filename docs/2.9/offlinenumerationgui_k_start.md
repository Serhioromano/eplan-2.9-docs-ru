# Нумеровать (автономно)

EPLAN позволяет выполнить повторную нумерацию нескольких устройств. Пользователь может настроить объем устройств, которые требуется пронумеровать, и выбрать формат нумерации. На основании этих настроек EPLAN нумерует устройства ***в автономном режиме***:

* Можно выполнить нумерацию для всех устройств всего проекта или для выбранных устройств, находящихся на выбранных страницах.
* Нумерация может быть выполнена для размещенных и неразмещенных устройств.
* Можно использовать предварительно определенные или собственные форматы нумерации.
* После выполнения нумерации устройств можно дополнительно вручную настроить нумерацию для отдельных устройств.
* Если во время нумерации устройств обнаруживаются логические несоответствия, выводятся соответствующие сообщения.

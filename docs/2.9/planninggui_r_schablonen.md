# Вкладка Шаблоны

Вы открыли проект.

* Вы сгенерировали объект планирования в навигаторе предварительного планирования или вставили его в Графическом редакторе. Выберите в диалоговом окне Свойства ++...++ вкладку Шаблоны.
* Вы выделили объект планирования в навигаторе предварительного планирования или в Графическом редакторе. Обработать > Свойства (или Всплывающее меню > Свойства). Выберите вкладку Шаблоны.

На этой вкладке объекту планирования присваиваются шаблоны функции. Кроме того, объект планирования может получить другие шаблоны функций через изделие, которое присвоено объекту планирования. Поскольку каждый объект планирования может иметь только одну главную функцию, рекомендуется присваивать шаблоны функций исключительно либо через изделие, либо через вкладку.

Доступны следующие элементы диалогового окна:

### Выбор устройства (шаблоны функций)

В таблице показаны присвоенные объекту планирования шаблоны функций. В самом верху таблицы показаны шаблоны функций, сохраненные на изделиях, которые присвоены объекту планирования. Эти шаблоны функций и их порядок изменить нельзя. Этот порядок задается, исходя из порядка изделий и шаблонов функций в каждом изделии.
Если изделие в объекте планирования было удалено или изменено, то принадлежащие ему шаблоны функций также будут удалены или изменены.

Ниже показаны шаблоны функций, которые сохранены непосредственно на объекте планирования. Если в столбце Определение функции щелкнуть в ячейке по кнопке ++...++, откроется диалоговое окно Определения функций, в котором можно выбрать нужное определение функции.

!!! tip "Совет:"

    Если вы планируете принять решение по конкретному изделию только при подробном планировании, внесите в объект планирования только шаблоны функций, а не само изделие.

### Панель инструментов

Кнопка |  Значение
---|---
![](images/all_search_as.png){: .ui-icon } (Поиск) |  В открывшемся диалоговом окне Выбор изделия можно скопировать шаблоны функций из изделия на объект планирования. При этом изделие не присваивается объекту планирования.
![](images/all_new_as.png){: .ui-icon } (Создать) |  Создает новую строку. Новая строка добавляется в конец списка.
Предварительной настройкой для новой строки является текущая выделенная, т. е. копируются указанные в ней данные. Если ни одна запись не выделена, будет создана пустая строка.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет строку, в ячейке которой находится курсор, со всем содержимым, т. е. включая данные; при необходимости выполняется новая нумерация позиций. Возможен многократный выбор. Обратите внимание на то, что измененные данные записываются в базу данных только после того, как выполнен щелчок по кнопке ++Закрыть++ или ++Применить++.
![](images/all_arrowup_as.png){: .ui-icon } / ![](images/all_arrowdown_as.png){: .ui-icon } (Переместить вверх / вниз) |  Перемещает выделенную строку(и) на соответственно одну запись вверх или вниз, если это возможно.

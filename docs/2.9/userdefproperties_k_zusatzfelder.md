# Дополнительные поля и произвольные свойства

Кроме концепции свойств, определенных пользователем, в EPLAN есть дополнительные возможности для ввода данных, для которых по умолчанию нет полей ввода:

### Дополнительные поля

Для проектов, страниц и функций имеются четко определенные дополнительные поля. Эти свойства отображаются на поверхности с неизменяемым именем Дополнительное поле [n] (где [n] указывает индекс) и идентифицируются через четко заданный номер свойства (ид.). С помощью индекса можно различать до 1 000 дополнительных полей.

* Дополнительное поле для проектов: ид. 10901
* Дополнительное поле для страниц: ид. 11901
* Дополнительное поле для функций: ид. 20901

В этих дополнительных полях можно записывать любые данные. Заданное вами значение дополнительных полей необходимо запомнить или записать.

### Произвольные свойства в базе данных изделий

В базе данных изделий для каждого изделия можно задать до 1 000 т. н. "произвольных свойств". Эти свойства определяются в форме [Описание] [Значение] [Единица измерения] и отображаются в интерфейсе пользователя как Произв. свойство: Описание [n], Произв. свойство: Значение [n] и Произвольн. свойства: Единица измерения [n], где [n] обозначает индекс. Через одно и то же значение индекса свойству присваивается его значение и единица измерения.

Определяющим для свойств является заданное программой значение ид., а не обозначение. Т. е. поля можно использовать для различных изделий с различным значением.

!!! note "Замечание:"

    В базе данных изделий для ввода и отображения дополнительной информации мы рекомендуем использовать ***определенные пользователем свойства***. При конфигурации они предоставляют больше возможностей по настройке, чем произвольные свойства, и их характеристики аналогичны свойствам, определенным пользователем для проекта. Произвольные свойства доступны только для совместимости.

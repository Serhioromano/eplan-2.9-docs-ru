# Заменить источник и цель для объектов планирования соединения

С помощью нового пункта меню Заменить источник и цель под меню Данные проекта > Соединения теперь можно заменить источник и цель объектов планирования соединения. Для этого в навигаторе предварительного планирования нужно выбрать соответствующие сегменты.

В результате для объектов планирования трубопровода будут заменены записи в полях Исходный сегмент и Целевой сегмент, а для объектов планирования кабеля — записи в одноименных свойствах. Графика на страницах предварительного планирования будет скорректирована соответствующим образом.

Для точки определения трубопровода, которая в дереве навигатора размещена под объектом планирования трубопровода, с помощью функции Заменить источник и цель будет также скорректировано ***направление потока в трубопроводе*** на функциональной схеме автоматизации, если источник и цель на связанном объекте планирования трубопровода были указаны в предварительном планировании. Кроме того, исходные или целевые функции трубопровода на функциональной схеме автоматизации должны быть связаны с объектами планирования, которые соответствуют исходным или целевым сегментам на объекте планирования трубопровода.

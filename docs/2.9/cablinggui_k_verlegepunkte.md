# Точки маршрутизации топологии

Точки маршрутизации — это особые места в сети соединенных сегментов, такие как повороты, проломы в стенах, смещение, выпуск в пути, отверстия электрошкафа и т. д. Точки маршрутизации не обязательно состоят из какого-либо материала.

Точка маршрутизации служит для входа соединения к цели, для разветвления на несколько сегментов или для соединения сегментов маршрутизации. Сегменты маршрутизации всегда заканчиваются на точке маршрутизации, т.е. точка маршрутизации прерывает сегменты маршрутизации.

Точка маршрутизации может иметь не более одного вывода устройства топологии, по умолчанию используется точка вставки как вывод устройства.

### Точки маршрутизации структурного идентификатора

Точки маршрутизации структурного идентификатора — это точки маршрутизации с определением функции "Точка маршрутизации структурного идентификатора топологии". Точка маршрутизации структурного идентификатора объединяет в себе определенную структуру обозначений и подключенные функции топологии. В одной точке маршрутизации структурного идентификатора можно генерировать функции топологии вручную и автоматически.

С помощью точек маршрутизации структурного идентификатора можно прокладывать соединения между различными структурами обозначений, не размещая подключенные функции топологии. Таким образом можно, например, прокладывать кабель между различными электрошкафами, не распознавая все подключенные устройства или не размещая их по отдельности.

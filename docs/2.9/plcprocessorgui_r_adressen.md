# Вкладка Адреса (форматы адресов ПЛК)

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > ПЛК. В диалоговом окне Настройки: ПЛК щелкните рядом с полем Специфические для ПЛК настройки по кнопке ++...++. В диалоговом окне Настройки: Специфич. для ПЛК выберите вкладку Адреса.

В этой вкладке вы определяете идентификаторы и тип данных для вводов и выводов. Далее задается вид сигнала для входов и выходов. После этого EPLAN ищет в схеме соединений данные, соответствующие заданным идентификаторам и типам данных, и строит список присвоений в соответствии с найденными установками.

Обзор основных элементов диалогового окна:

### Адресация

Вводы и выводы могут адресоваться следующим образом:

* побитно
* побайтно (8 бит)
* словами (2 байта или 16 бит)
* двойными словами (4 байта или 32 бита)
* длинными словами (8 байт или 64 бита).

Для каждого типа адресации можно определить отдельные настройки.

### Идентификатор

Здесь вы назначаете идентификатор для типа данных. Идентификатор может состоять из нескольких буквенных и/или цифровых обозначений. Благодаря идентификатору EPLAN идентифицирует вводы и выводы на схеме соединений. Кроме того, идентификатор используется для фильтрации записей при импорте списков присвоений.

!!! example "Пример:"

    Присвоение идентификаторов:Идентификатор состоит Ввод ПЛКВывод ПЛКБуквенныеобозначенияE 3.1A 4.5Цифровые обозначения1 3.12 4.5

### Тип данных

Здесь вы присваиваете виду адресации определенный тип данных. Ввод осуществляется вручную либо выбором значений из раскрывающегося списка. Имеют смысл такие значения:

* BOOL
* BYTE
* WORD
* DWORD
* LWORD.

Кроме того, можно вводить данные INT, DINT, LINT, они соответствуют WORD, DWORD, LWORD и при обмене данными ПЛК поддерживаются в ***формате AutomationML***.

### Вид сигнала

Выберите из раскрывающегося списка вид ввода-вывода - дискретный или аналоговый.

### Величина шага

Укажите шаг приращения для нумерации адресов ПЛК.

# Диалоговое окно Формат: Вычисление (экспорт данных изготовления / маркировка)

Вы открыли проект. Сервисные программы > Данные изготовления > Экспорт / маркировка. В диалоговом окне Экспортировать данные изготовления / Вывести маркировку рядом с полем Настройки щелкните по кнопке ++...++. В диалоговом окне Настройки: Экспорт данных изготовления / маркировка выберите вкладку Этикетка, выделите элемент формата "Вычисление" и щелкните по кнопке ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо).

В этом диалоговом окне при помощи элементов формата можно создать формулу вычисления для данных проекта и данных изделий, чтобы вывести ее в файле.

### Выбранные элементы формата

В данном поле находятся элементы формата, из которых состоит формат расчетной формулы.

### Мин. число разрядов

Значение в этом поле выдает результат, каково минимальное используемое число разрядов. Это значение относится только к разрядам до запятой. Если результат перед запятой содержит меньше цифр, чем указанное мин. число разрядов, то излишние позиции заполняются символом, указанным в поле Заполнитель.

### Заполнитель

Отсутствующие места заполняются этим символом. То, сколько разрядов заполняется, зависит от значения, введенного в поле Мин. число разрядов.

!!! example "Пример:"

    Если в поле Мин. число разрядоввведено значение4, а в поле Заполнитель— значение0, числа выводятся следующим образом:0001.

### Десятичные разряды

Значение в этом поле выдает результат, сколько отображается разрядов после запятой. Если результат после запятой содержит больше цифр, чем разрешенных десятичных разрядов, то цифры округляются до разрешенного числа десятичных разрядов. Если результат после запятой содержит меньше цифр, чем указанное мин. число разрядов, то излишние десятичные разряды заполняются символом, указанным в поле Заполнитель.

### Переменное число десят. разрядов

Если этот флажок установлен, десятичные разряды никогда не заполняются символом - заполнителем. Если результат после запятой содержит меньше цифр, чем разрешенных десятичных разрядов, то результат не изменяется. Если результат после запятой содержит больше цифр, чем разрешенных десятичных разрядов, то цифры округляются до разрешенного числа десятичных разрядов.

### Длина

В этом групповом поле указывается длина поля для результата расчетной формулы.

* Неограниченно перемен.: Если выбрана эта опция, длина поля является неограниченной.
* Ограничено перемен.: Если выбрана эта опция, длина поля является ограниченной. Если текст выделенного свойства не входит в поле, то он укорачивается до длины, указанной в поле Число символов.
* Фиксированная: Если выбрана эта опция, длина поля является фиксированной. Длина поля вводится в поле Число символов.
* Число символов: Это поле доступно в том случае, если выбрана опция Фиксированная или опция Ограничено перемен. Число в этом поле указывает на длину поля.
* Вывести по правому краю: Это поле доступно, если выбран параметр Фиксированная. Если установлен этот флажок, текст выделенного свойства выводится по правому краю.

### Разделитель для индекса 0

Эта функция относится только к элементам формата, являющимися вышестоящими элементами-заполнителями. Вышестоящий элемент-заполнитель обозначается как индекс 0, как, например, "Изделие клеммы". Он содержит нижестоящие элементы-заполнители с индекса [1] до индекса [10], как, например, "Изделие клеммы [1]" до "Изделие клеммы [10]". Вышестоящий элемент-заполнитель "Изделие клеммы" выводит изделия клеммы всего клеммника. В этом поле отображается разделитель, заданный при помощи ++...++ в диалоговом окне Формат: Разделитель. Этот разделитель вставляется между данными проекта.

# Обновить базу данных изделий

Если вы работаете с несколькими версиями EPLAN (например, при переходе к версии 2.9) и в версии 2.9 настроена "старая" база данных изделий, при открытии базы данных изделий поступит запрос на ее обновление.

Если нажать [Нет], база данных изделий не обновляется. База данных изделий откроется, но она будет доступна только для чтения. При этом изделия и устройства доступны для выбора.

!!! note "Замечание:"

    * При работе с несколькими версиями EPLAN для редактирования и обслуживания базы данных изделий необходимо всегда использовать последнюю версию EPLAN.
    * Новые и обновленные старые базы данных изделий можно открывать и обрабатывать в версиях EPLAN 2.5, 2.6, 2.7 и 2.8. В более ранних версиях EPLAN (версия 2.4 или старше) эти базы данных изделий можно открыть, например, для выбора изделий или устройств, но не для их обработки.

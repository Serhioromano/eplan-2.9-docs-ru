# Сети соединенных сегментов (топология)

Под топологией в EPLAN понимается область внутри двухмерной обработки схемы соединений, в которой планируются сети соединенных сегментов и маршрутизируются соединения. При этом можно маршрутизировать любые соединения, кабели, шланги и трубы. Укажите для этих соединений трассы маршрутизации внутри топологии, по которым должны проходить соединения.

Доступны следующие действия:

* Сеть соединенных сегментов можно создавать независимо от других представлений в проекте. Структуризация сети соединенных сегментов осуществляется через структуру идентификаторов (для страниц, устройств и т. д.).
* Сеть соединенных сегментов можно проектировать как размещенную или неразмещенную, по вашему усмотрению. Для размещения служит тип страницы "Топология".
* У функций и соединений есть дополнительный вид представления "Топология". Соединения с таким видом представления учитываются при синхронизации.
* Через фильтр соединений можно управлять тем, какие соединения допускается прокладывать по трассе маршрутизации.
* При маршрутизации определяется длина проложенного кабеля или соединения.
* На основе сети соединенных сегментов и сгенерированных маршрутизируемых соединений можно генерировать ответы, отображающие, например, трассы маршрутизации, сегменты маршрутизации или проходящие через сегменты маршрутизации кабели и соединения.

# Диалоговое окно Настройки: Интерфейс пользователя

Параметры > Настройки > Пользователь > Отображение > Интерфейс пользователя

В этом диалоговом окне можно задать разные настройки интерфейса.

Обзор основных элементов диалогового окна:

### Число последних открытых проектов

Последние открытые проекты перечислены в главном меню, в меню Проект. Щелкните по имени проекта, откроется соответствующий проект; при этом не требуется использовать пункт меню Открыть. Введите в этом поле, сколько записей должен иметь список проекта. Максимальное количество — 10.

### Мин. ширина частей в предв. просмотре

Введите в это поле значение ширины (в пикселях) для "частей" при выборе символа и в графическом предварительном просмотре навигатора страниц. Эта ширина также является и минимальной высотой, благодаря чему части имеют по возможности квадратную форму. Система определяет ширины поля и рассчитывает на основе веденной здесь минимальной ширины, сколько частей может быть отображено рядом друг с другом.

### Открыть последние проекты

если отмечен этот флажок, EPLAN автоматически запустит последние открытые проекты при следующем открытии программы.

Если флажок не установлен, запуск EPLAN осуществляется при следующем вызове программы без открытия проекта.

### Открыть последнюю страницу

если этот флажок не отмечен, EPLAN при следующем пуске автоматически отобразит последнюю открытую страницу в графическом редакторе. Настройка действует, только если вы отметили флажок Открыть последний проект.

если флажок не отмечен, EPLAN запускается, не отображая последнюю открытую страницу.

### Управление графическими редакторами, специфическими для проекта

Если этот флажок установлен, при работе одновременно в нескольких проектах для каждого проекта используется отдельный графический редактор / отдельный трехмерный вид.

Если флажок снят, открытая в данный момент страница проекта закроется при переходе на страницу в другом проекте.

### Отображать идентифицирующие номера

Если установлен этот флажок, отображаются следующие идентифицирующие номера:

* В списке ***Свойства*** в диалоговых окнах свойств также отображаются номера свойств. Номер указывается в перед соответствующим свойством в фигурных скобках (напр., "\<20026\> Определение функции").
* Для ***Вида представления*** функций в диалоговом окне свойств отображается однозначный номер.
* При появлении ***Системного сообщения*** однозначный номер сообщения отображается в строке заголовка диалогового окна сообщений.

Если флажок не установлен, идентифицирующие номера для свойств, системных сообщений и вида представления не отображаются.

### После имени

Если установлен флажок Отображать идентифицирующие номера, можно указать, должны ли отображаться номера после текста свойства или вида представления. Если этот флажок установлен, свойство отображается в вышеупомянутoм примере следующим образом: "Определение функции \<20026\>".

### Активировать подавленные сообщения

В EPLAN можно подавить менее важные сообщения и диалоговые окна. Для этого в соответствующих диалоговых окнах имеется флажок Больше не отображать это диалоговое окно. если вы подавили диалоговые окна, вы можете здесь снова активировать их:

если флажок отмечен, подавление диалоговых окон отменяется.

если флажок отмечен, сообщения и диалоговые окна, подавленные однажды, и дальше остаются подавленными.

### Цветовые настройки страниц / пространств листа с защитой от записи

При помощи группового поля можно задавать отдельные цветовые настройки для графического редактора и трехмерного вида в защищенных от записи страницах / пространствах листа. Это позволяет легко отличать такие страницы и пространства листа от обрабатываемых страниц и пространств листа.

Страницы и / или пространства листа защищены от записи, если для них настроена автоматическая или ручная [защита от изменений](devicetaggui_k_aenderungsschutz.md) либо проект был открыт в режиме "С защитой от записи".

!!! note "Замечание:"

    * Для обозначения защищенных от записи проектов в представлениях навигаторов в виде дерева и в виде списка используется серый цвет фона.
    * Активность автоматической или ручной защиты от изменений для страниц / пространств листа отображается в диалоговом окне Конфигурировать защиту (путь меню: Обработать > Другое > Конфигурировать защиту).

### Схема (2D)

Выберите здесь цветовую схему графического редактора для защищенных от записи страниц / пространств листа. Имена цветовых схем указывают на цвет фона экрана.

### Фон оттенения 1 (3D) / Фон оттенения 2 (3D)

Здесь для защищенных от записи страниц / пространств листа задаются цвета фона для оттенения в трехмерном виде: с помощью кнопки ++...++ откройте диалоговое окно Выбор цвета. Выберите требуемый цвет.

Групповое поле "Отметить многоязычные поля ввода"

С помощью настроек в этом групповом поле можно задать, должны ли многоязычные поля ввода в интерфейсе пользователя обозначаться стоящей впереди пиктограммой.

* Всегда отображать пиктограммы: все многоязычные поля ввода будут обозначены. Обозначение выполняется независимо от того, возможно ли внести запись в поле или данное поле используется только для отображения.
* Отображать пиктограммы только для обрабатываемых полей ввода: обозначаются только те многоязычные поля ввода, в которых возможен ввод.
* Никогда не отображать пиктограммы: многоязычные поля ввода не обозначаются.

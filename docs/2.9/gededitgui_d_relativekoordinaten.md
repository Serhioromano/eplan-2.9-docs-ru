# Диалоговое окно Относительный ввод координат

Открыта страница в графическом редакторе. Параметры > Относительный ввод координат.

В этом диалоговом окне вводятся целевые координаты для установки курсора. Ввод выполняется относительно текущей позиции курсора. Для этого предусмотрены две возможности. В поле Интервал указываются целевые координаты относительно текущей позиции курсора. Или целевые координаты указываются с помощью угла и длины линии перемещения.

!!! note "Замечание:"

    Обратите внимание, что результаты перехода к целевым координатам не точные, а ориентированы на сетку, если вводится длина, и сетка включена.

Доступны следующие элементы диалогового окна:

### Интервал(<система координат>)

При вызове диалогового окна здесь выводятся установленная система координат и текущая позиция курсора. Введите в поля X и Y значения для перемещения конечной точки в направлении оси X и Y от координат начальной точки. Возможен ввод положительных и отрицательных значений. При этом автоматически изменяются значения в полях Угол и Длина.

### Угол

Введите значение угла, чтобы определить направление перемещения курсора.

### Длина

Укажите длину, на которую должен быть перемещен курсор в направлении введенного угла. При этом автоматически изменяются значения X и Y в поле Интервал.

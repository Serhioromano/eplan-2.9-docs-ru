# Повернутые страницы / Неповернутые страницы

При адаптации проекта от одного стандарта к другому страницы по умолчанию поворачиваются. Некоторые страницы необязательно поворачивать. В следующей таблице показано, какие страницы не поворачиваются.

Страницы, которые не поворачиваются |  Тип страниц
---|---
Графические страницы |  Тип страниц "Графика (I)"
Страницы монтажных плат |  Тип страниц "Компоновка электрошкафа (I)"
Fluid-страницы |  Тип страниц "Схема соединений Fluid-техники (I)"
Страницы ТС |  Тип страниц "Функциональная схема автоматизации (I)"
Обзоры модели |  Тип страницы "Обзоры модели (I)"
Все страницы отчетов |

# Сохранить логическую схему выводов устройства при изменении определения функции

В диалоговом окне Логическая схема выводов устройства обработайте логическую модель функции. Если здесь были внесены определенные пользователем изменения для одного или нескольких выводов устройства, то эти настройки будут удалены, как только на вкладке Данные символа / функции диалогового окна 'Свойства' будет выбрано другое определение функции. Чтобы предотвратить это в будущем, в диалоговом окне Определения функций доступен новый флажок Не изменять логическую схему выводов устройства функции. Если этот флажок активирован, при выборе другого определения функции логическая схема выводов устройства функции не будет изменена. Таким образом, определенные пользователем изменения в диалоговом окне Логическая схема выводов устройства сохраняются.

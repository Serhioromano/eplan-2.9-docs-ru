# Вкладка "Группы шлангов"

У вас установлен конфигуратор шлангопроводов в качестве дополнительного модуля для EPLAN Fluid. Вы запустили EPLAN Fluid и конфигуратор шлангопроводов.

* Выберите в EPLAN Fluid пункты меню Опции > Настройки > Пользователь > Add-ons > Конфигуратор шлангопроводов и щелкните в следующем диалоговом окне по вкладке Группы шлангов.
* Или выберите в диалоговом окне Конфигурировать шлангопроводы конфигуратора шлангопроводов пункты меню Дополнительно > Настройки и щелкните в следующем диалоговом окне по вкладке Группы шлангов.

Или у вас установлен конфигуратор шлангопроводов в качестве самостоятельной программы, и программа запущена.

* Выберите в конфигураторе шлангопроводов пункты меню Параметры > Настройки > Пользователь > Add-ons > Конфигуратор шлангопроводов и щелкните в следующем диалоговом окне по вкладке Группы шлангов.

На этой вкладке определяются данные по умолчанию для групп шлангов. При этом вы объединяете разные типы шлангов в группы и сохраняете для выбранных типов шлангов соответствующую информацию по стандартам для шлангов и изделиям. Сохраненные в таблице указания на стандарты для типов шлангов предлагаются на выбор во время конфигурации шлангопроводов в диалоговом окне Обработать ++...++.

Обзор основных элементов диалогового окна:

### Номер группы (список)

Задайте группы шлангов, присваивая номер каждой группе шлангов.

Используйте кнопки панели инструментов над списком, чтобы создавать новые записи, обрабатывать имеющиеся и/или удалять более ненужные записи.

!!! note "Замечание:"

    Любой номер группы можно использовать только ***один раз***. Если имеющийся номер группы задан еще раз, вы получите соответствующее сообщение.

### Таблица

В этой таблице соответствующей группе шлангов присваиваются требуемые типы шланга, стандарты и номера изделий.

При использовании конфигуратора шлангопроводов в качестве дополнительного модуля к EPLAN Fluid с помощью кнопки ++...++, предусмотренной в табличных ячейках столбца ***Изделие***, можно перейти в базу данных изделий и выбрать там нужное изделие.

Используйте кнопки панели инструментов над таблицей для создания новых записей и/или удаления ненужных записей.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

!!! note "Замечание:"

    При выборе стандартной схемы DIN 20066_2012-01 настройки на этой вкладке и на вкладке [Параметры шлангов](fluidhoseconfiggui_r_schlauchzuordnungen.md) действуют таким образом, что в диалоговом окне Обработать ++...++ для каждого выбранного рабочего давления и номинального диаметра шланга на выбор предлагаются типы шлангов, соответствующие стандарту DIN 20066.

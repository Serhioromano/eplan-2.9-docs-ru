# Экспорт ЧУ: Принцип

Как правило, каждый экспорт ЧУ ориентируется на выбор функциональных элементов. Этот выбор может осуществляться в пространстве листа или в навигаторе пространства листа. При этом вместе с выбранным объектом всегда автоматически выбираются и экспортируются все объекты, расположенные под ним в иерархии.

Экспортируются только выбранные функциональные элементы, релевантные для ЧУ. Если в навигаторе пространства листа выделены только отдельные монтажные поверхности, то экспортируется только тот функциональный элемент, к которому они относятся.

Выбор экспортируемых объектов можно автоматически распространить на весь проект.

### Функциональные элементы, релевантные для ЧУ

Не на каждой монтажной поверхности проводятся механические работы. Кабельный канал с расположением отверстий для сверления, как правило, генерирует на монтажной плате механические работы. Однако, если такой же кабельный канал размещен, например, на шасси, отверстия не возникают.

Возникнут отверстия или нет, зависит не от устанавливаемого функционального элемента (напр., кабельного канала), а прежде всего от функционального элемента (а следовательно, и монтажной поверхности), на котором функциональный элемент установлен.

Поэтому каждое трехмерное размещение изделия получает свойство Механическая обработка. Пользователь не может изменить это свойство вручную.

Свойство Механическая обработка активируется при следующих определениях функции:

* Панель основания, общее
* Фланцевая плата
* Боковая стенка для верхнего модуля
* Крышка, общее
* Монтажная плата, общее
* Область ввода кабеля, боковая стенка
* Заглушка для цоколя
* Дверца, общее
* Пластина модуля
* Разделительная пластина, общее
* Стена, общее
* Сборные шины, общее
* Шкаф, общее.

При всех других определениях функции свойство по умолчанию деактивировано.

### Поле для обработки ЧУ

Поле в экспорте ЧУ — это (частичная) поверхность, которая обрабатывается станком ЧУ. Каждый функциональный элемент, релевантный для ЧУ (то есть с активированным свойством Механическая обработка), при экспорте в автоматы ЧУ или при создании шаблона сверления учитывается в качестве поля. Естественно, это правило действует только тогда, когда он содержится в выборе для экспорта, и это поле в проекте оснащено, а значит, подлежит механической обработке.

### Размер поля в 3D-макросах

При создании 3D-макросов с поверхностями, релевантными для ЧУ, необходимо, кроме монтажной поверхности, определить также и [Размер поля](cabinetgui_d_navigator.md), который, как правило, соответствует описанному прямоугольнику монтажной поверхности. Поверхности, релевантные для ЧУ, без размера поля не экспортируются. Контрольный прогон ([P026030](msg/messages_p_026030.md)) проверяет, все ли монтажные поверхности, релевантные для ЧУ, имеют определенный размер поля.

### Контроль за креплением

Контроль за креплением проверяет, все ли функциональные элементы, которые размещены на поверхностях, релевантных для ЧУ, имеют схему сверления. Проверка выполняется в ходе контрольного прогона. Если какому-нибудь размещению изделия на монтажной поверхности, релевантной для ЧУ, не присвоено действительной схемы сверления, появится сообщение контрольного прогона.

### Особенности экспорта ЧУ для машин Steinhauer и Rittal - Perforex BC / Secarex

!!! note "Замечание:"

    Во избежание разрушения контура при позиционировании фрезерного инструмента для контуров с очень малой высотой должны выполняться следующие условия:

* Продольные отверстия экспортируются только в том случае, если высота не менее чем на 0,2 мм превышает больший из двух диаметров фрезы / центрового сверла.
* Прямоугольные отверстия экспортируются только в том случае, если высота не менее чем на 2 мм превышает диаметр фрезы / центрового сверла.
* Отверстия типа "шестиугольник" и "восьмиугольник" экспортируются только в том случае, если длина ребра не менее чем на 2 мм превышает диаметр фрезы / центрового сверла.
* Закругления или фаски прямоугольных отверстий при экспорте ЧУ Rittal — Perforex / Secarex не экспортируются, если радиус фрезы больше радиуса закругления или ширины фаски.

# Открыть проект (Управление проектами)

В рамках управления проектами вы можете открыть простые и запакованные проекты. Уже открытые проекты обозначаются выделенным полужирным шрифтом именем проекта.

Условия:

* Вы настроили управление проектами.
* В представлении структуры дерева управления проектами находится считанный каталог, как минимум, с одним проектом.

### Открыть проекты и закрыть управление проектами

Этот способ действий позволяет одновременно открыть один или несколько проектов и обработать непосредственно в навигаторе страниц. Для этого после открытия проектов управление проектами автоматически закрывается.

1. Выберите следующие пункты меню: Проект > Управление
2. В диалоговом окне Управление проектами выделите в структуре дерева необходимый проект.
3. Щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Управление проектами закрывается.

!!! info "Для сведения:"

    В данном случае EPLAN проверяет, был ли выход из базы данных проекта выполнен "ненадлежащим образом", например из-за перебоя в сети или отключения электросети. Если это так, проект / база данных снова закрывается.

Если проект в порядке, он открывается в диалоговом окне Страницы — <Имя проекта>.

!!! info "Для сведения:"

    Имя проекта выделяется в управлении проектами полужирным шрифтом.

### Открыть проекты и не закрыть управление проектами

Этот способ действий позволяет одновременно открыть один или несколько проектов и обработать непосредственно в управлении проектами.

1. Выберите следующие пункты меню: Проект > Управление
2. Выделите в представлении структуры дерева диалогового окна Управление проектами один или несколько проектов.
3. Выберите ++Организовать++ > Открыть.

!!! info "Для сведения:"

    В данном случае EPLAN проверяет, был ли выход из баз данных проекта выполнен "ненадлежащим образом", например из-за перебоя в сети или отключения электросети. Если это так, проекты / базы данных снова закрываются.

В противном случае выделенные проекты / проекты, которые в порядке, открываются в диалоговом окне Страницы — <Имя проекта>.

!!! info "Для сведения:"

    Имена открытых проектов выделяются в управлении проектами полужирным шрифтом.

!!! tip "Совет:"

    * Вы можете открыть проекты с помощью перетаскивания мющью, для этого перетащите выделенный файл проекта *.elk из Windows-Explorer в навигатор страниц.
    * Проекты, считанные вами в управлении проектами, также можно открыть вне управления проектами. При повторном открытии управления проектами данные проекты автоматически открываются в управлении проектами.

# Создать проект (Управление проектами)

Проекты можно создавать с помощью управления проектами или непосредственно. При этом проект создается на основе шаблона проекта или базового проекта. Если вы сами еще не создали шаблон проекта или базовый проект, можно использовать [Примерные шаблоны](projects_k_basisprojekt.md) EPLAN. Ниже описывается, как создать проект с помощью управления проектами.

Условия:

* Вы имеете доступ к [Шаблоны проекта](projects_k_projektvorlagen.md) или [Базовые проекты](projects_k_basisprojekteerweitert.md).
* Вы создали базу данных управления проектами.

1. Выберите следующие пункты меню: Проект > Управление
2. Выберите ++Организовать++ > Создать.
3. В диалоговом окне [Создать проект](prjmanagementgui_d_projekterstellen.md) укажите имя проекта и место хранения и выберите нужный для проекта шаблон.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В целевом каталоге создается новый проект *.elk.

!!! info "Для сведения:"

    Новый проект автоматически считывается в управление проектами.

5. Проверьте все записи в диалоговом окне Свойства проекта: <Имя проекта> и скорректируйте их при необходимости.

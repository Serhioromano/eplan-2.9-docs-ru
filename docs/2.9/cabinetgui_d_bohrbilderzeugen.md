# Диалоговое окно Генерировать схемы сверления / контуры

Вы открыли проект макросов. Выберите пункты меню Сервисные программы > Генерировать схемы сверления / контуры > Из файлов ЧУ Rittal — Perforex / Secarex. В диалоговом окне выбора файла выберите импортируемые файлы ЧУ Rittal — Perforex / Secarex и щелкните Открыть.

В этом диалоговом окне настройте каталог, в котором необходимо сохранять определенные пользователем контуры, сгенерированные вместе со схемой сверления.

Далее выберите схему с определением станка Rittal — Perforex / Secarex, из которой будет использоваться дополнительная информация для генерируемой схемы сверления.

Обзор основных элементов диалогового окна:

### Машина

Возможность выбора для схемы с определением станка Rittal — Perforex / Secarex в соответствии с настройками, заданными в настройках экспорта ЧУ Rittal — Perforex / Secarex. Из настроек в этой схеме определяют тип сверления и диаметр отверстий в генерируемой схеме сверления.

### Каталог для контуров (данные ЧУ)

Место хранения для файлов *.fc1 (данные контура ЧУ), генерируемых автоматически при генерировании схемы сверления. Каталогом по умолчанию является каталог для макросов ($MD_MACROS).

# Основные данные: Схемы

В этом разделе содержится информация о наиболее важных изменениях в схемах.

Для фильтра в ***навигаторе страниц*** были созданы такие новые схемы:

* Автоматические страницы
* Интерактивные страницы
* Установки
* Вид документа
* Место установки

С помощью этих новых схем фильтрации можно фильтровать страницы проекта по типу страницы (интерактивные или автоматически генерируемые страницы отчетов) или по структурным идентификаторам для блоков идентификаторов "Установка", "Место установки" или "Вид документа".

ПЛК

Схемы для определения ***форматов адресов*** ПЛК в диалоговом окне Настройки: Специфич. для ПЛК переделаны, а ненужные схемы удалены.

Для адресации ПЛК в формате для семейства продуктов Rockwell ControlLogix была создана такая новая схема:

* Allen-Bradley (ControlLogix)

В рамках этого изменения шаблон проекта и базовый проект "Inch Template" были преобразованы в новую схему Allen-Bradley (ControlLogix).

Схемы для форматов адресов, перечисленные ниже, были удалены. Это схемы для систем, аппаратное обеспечение которых больше недоступно, или схемы для программ конфигурации ПЛК, которые больше не могут выполняться на текущем программном обеспечении.

* Allen Bradley PLC-5
* BOSCH
* ESS PC WORX
* PL7
* PS40
* SIMATIC S5 (E/A)

Кроме того, схема Rexroth переименована на Bosch Rexroth.

Предварительное планирование

Для ***экспорта*** данных изготовления / вывода маркировки в диалоговом окне Экспортировать данные изготовления / вывести маркировку создана такая схема:

* Технологические контуры VDI 3814

EPLAN Pro Panel

Для фильтра в навигаторе пространства листа была создана такая новая схема:

* Инструмент для монтажных работ

С помощью этой новой схемы фильтрации в проекте можно отобразить используемые инструменты для монтажных работ (точки монтажа, сборочные линии и т. д.).

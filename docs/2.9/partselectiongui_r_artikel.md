# Вкладка Изделие

Вы открыли проект.

* Вы вставили символ в Графическом редакторе или генерировали новую функцию в Навигаторе. В диалоговом окне Свойства ++...++ на вкладке <Категория функции > или на вкладке Данные символа / функции установлен флажок Главная функция. Выберите вкладку Изделие.
* Вы выделили устройство общего назначения в Графическом редакторе или Навигаторе. Обработать > Свойства. В диалоговом окне Свойства ++...++ на вкладке <Категория функции> или на вкладке Данные символа / функции установлен флажок Главная функция. Выберите вкладку Изделие.

На этой вкладке сохраните или измените специфические для изделия данные для устройства. Изделия могут всегда сохраняться только на вкладке ***Главная функция***, поэтому укажите здесь изделия для всех функций устройства.

Обзор основных элементов диалогового окна:

В таблице в левой области диалогового кона перечисляются изделия, присвоенные устройству. Щелкните мышкой в поле Номер изделия по кнопке ++...++, чтобы открыть диалоговое окно Выбор изделия. Выбор здесь определен автоматическим фильтром (свойство Определение функции), который активируется путем установки флажка Активн. рядом с полем Автоматический фильтр. Отображение выбранных изделий ограничивается изделиями, шаблон функции которых соответствует автоматическому фильтру.
Значение в столбце Число штук / количество при этом будет равняться "1"; но его можно переписать.

!!! note "Замечание:"

    Если в диалоговом окне Настройки: Выбор изделия (например, через Параметры > Настройки > Пользователи > Управление > Выбор изделия) установлен флажок Разрешено изменение при выборе, то изделия из базы данных изделий, имеющиеся при выборе изделия, могут быть изменены и новые изделия могут быть созданы.
    Если флажок не установлен, то изделия можно только выбирать, но нельзя создавать или обрабатывать.

### Панель инструментов

С помощью кнопок со стрелками на панели инструментов можно изменять позицию строки одного или нескольких выделенных изделий. Это влияет, например, на последовательность, в которой изделия выводятся в отчетах. При этом изделия можно перемещать только в свободные строки. При перемещении изделие всегда перемещается полностью со всеми данными ссылки изделия. Кнопки активны только в том случае, если перемещение выделенного изделия возможно.

При ***удалении*** следует обращать внимание на следующее: данные изделия убираются из проекта. Данные, введенные вручную, при этом безвозвратно теряются, а данные, вставленные через выбор изделия, можно ввести снова — через новую операцию выбора изделия.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Открыть гиперссылку |  Дает возможность открыть внешние документы или графический файл, если они были присвоены изделию в базе данных изделий.
Присвоить данные |  Дает возможность переместить данные указанного ниже изделия в функцию.
Конфигурировать представление |  Открывает диалоговое окно Конфигурировать представление, в котором можно задать, какие свойства изделия и ссылки изделия должны отображаться в таблице в левой области. Свойства, выбранные для представления, больше не отображаются в таблице свойств в правой области (данные изделия и данные ссылки изделия).

### Категория

Выберите из раскрывающегося списка категорию, свойства которой должны быть введены в нижеуказанную таблицу:

* Данные изделия: Сохраненные в проекте данные из базы данных изделий проекта, напр. номер изделия и номер типа. Эти значения невозможно изменить здесь, и поэтому они будут выделены серым цветом.
* Данные ссылки изделия: Данные, скопированные на устройство. Большинство из них обрабатываются вручную, например срок службы. Значения данных ссылки изделия, содержащие отличающееся от свойства изделия значение, представляются в ячейке с желтым цветом фона. Это касается как таблицы свойств левой области, так и правой.
Учтите, что при повторном выборе изделия данные ссылок будут удалены, если они идентичны данным изделия.

В категории Данные ссылки изделия при помощи свойства Присвоение изделия определяется, является ли присвоенное изделие главным изделием или частью принадлежностей:

* Главное изделие: Обозначает действительное главное изделие, напр. кабель. Это значение предварительно настривается для каждого изделия по умолчанию.
* Принадлежности: Обозначает часть принадлежностей.
* Изделие Источник / Изделие Цель: Обозначает изделия, которые присваиваются кабелю по источнику и цели, напр. кабельные наконечники или штекеры.

Это свойство имеется также в отчетах и учитывается при автоматическом выборе кабеля.

В категории Данные ссылки изделия с помощью свойства Изделие является составной частью модуля задайте настройку, позволяющую рассматривать определенное изделие как изделие модуля. При выдаче спецификации изделий такие изделия подавляются.

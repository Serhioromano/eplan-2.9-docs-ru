# Обзор общих параметров командной строки EPLAN

Система EPLAN использует следующие общие параметры командной строки:

Параметр  |  описание
---|---
/NoLoadWorkspace |  Не загружена или не восстановлена рабочая область.
/NoSplash |  При запуске системы заставка не появляется.
/Language:<Значение> |  Запускает EPLAN на языке, указанном как язык диалогового окна (напр., при /Language:en_US на английском языке). При этом язык диалогового окна, установленный в настройках, не меняется.  EPLAN может запускаться на указанном языке только в том случае, если этот язык был установлен во время инсталляции и если имеется лицензия для этого языка.
/Auto |  Завершает работу EPLAN после выполнения командной строки.
/Quiet |  Диалоговые окна не отображаются, когда выполняется командная строка.
/Frame:<Значение> |  Определяет главное окно EPLAN, например:

* /Frame:0 => Главное окно EPLAN не отображается.
* /Frame:1 => Главное окно EPLAN отображается в исходном размере и исходной позиции.
* /Frame:2 => Главное окно EPLAN сворачивается.
* /Frame:3 => Главное окно EPLAN разворачивается.

/Setup:<Значение> |  Все настройки возвращаются к стандартным значениям, имеющимся после инсталляции:

* /Setup:0 => Настройки пользователя, рабочей станции и фирменные настройки сбрасываются до стандартных значений установки, а базы данных резервируются.
* /Setup:USER => Настройки пользователя сбрасываются до стандартных значений установки, а базы данных резервируются.
* /Setup:STATION => Настройки станции сбрасываются до стандартных значений установки, а базы данных резервируются.
* /Setup:COMPANY => Фирменные настройки сбрасываются до стандартных значений установки, а базы данных резервируются.

/PathsScheme |  Возвращает настройки каталога к заранее определенной схеме (например, /PathsScheme:MyDirectories). Если схема не задана, используется стандартная схема.
/Вариант |  Указывать только в том случае, если EPLAN запускается через файл EPLAN.exe компонента программы "EPLAN Platform"! Показывает используемый лицензионный вариант программы (например, "Electric P8", "Fluid", "ProPanel" и т. д.).
/User:<Значение> |  <Значение> указывает, какие настройки пользователя должны использоваться при запуске. Если установлено управление правами, этот параметр также задает соответствующее имя пользователя.
/Password:<Значение> |  Если установлено управление правами, этот параметр задает пароль для "/User".
/Station:<Значение> |  <Значение> указывает, какие настройки станции должны использоваться при запуске.
/Company:<Значение> |  <Значение> указывает, какие фирменные настройки должны использоваться при запуске.
/SystemConfiguration:<Значение> |  <Значение> указывает имя схемы конфигурации системы.
/NoRemoting:<Значение> |  Если <Значение> = "1", удаленный сервер не запустится. Предварительная настройка — "0" означает, что удаленный сервер запустится.
/License:<Значение> |  <Значение> указывает, какой файл лицензии по умолчанию *.lis (включая указание пути и расширения имени файла) должен использоваться.

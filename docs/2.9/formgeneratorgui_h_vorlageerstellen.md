# Генерировать шаблон отчета

В [шаблонах отчетов](formgeneratorgui_k_auswertungsvorlagen.md) можно сохранить настройки для отчетов. Эти настройки можно использовать при генерации новых отчетов. Шаблоны отчетов могут быть созданы "с чистого листа" или на базе существующего отчета.

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Сервисные программы > Отчеты > Генерировать
2. Выберите в диалоговом окне Отчеты — <Имя проекта> вкладку Шаблоны.
3. Щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
4. Выберите в диалоговом окне Определить отчет тип отчета и щелкните по ++OK++.
5. Выберите в диалоговом окне Настройки — <Тип отчета> схемы для фильтрации, сортировки и разбивки на страницы и щелкните по ++OK++.
6. Установите в диалоговом окне <Тип отчета> Структурные идентификаторы и имя начальной страницы отчета.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    EPLAN создаст шаблон отчета и автоматически присвоит ему имя 0001. Если уже сгенерирован шаблон типа отчета, открывается запрос на перезаписывание прежнего шаблона. При выборе [Нет] создаются два одноименных отчета, и их имена затем необходимо изменить.

Условие:

Вы сгенерировали отчет.

1. Выберите в диалоговом окне Отчеты — <Имя проекта> вкладку Отчеты.
2. Выделите отчет и выберите Всплывающее меню > Создать шаблон отчета.

!!! info "Для сведения:"

    Будет создан шаблон отчета и внесен во вкладку Шаблоны.

3. Для проверки или адаптации настроек шаблона отчета выберите вкладку Шаблоны.
4. Выделите шаблон, и адаптируйте настройки в правой области столбца Значение.

Следующие инструкции помогут генерировать [условные отчеты](formgeneratorgui_k_auswertungsvorlagen.md).

Условие:

Вы создали шаблон отчета.

1. Выберите в диалоговом окне Отчеты — <Имя проекта> вкладку Шаблоны.
2. Выделите шаблоны отчетов.
3. Щелкните столбец Значение в строке Фильтр проекта и нажмите на кнопку ++...++.
4. Определите в следующем диалоговом окне фильтр проекта с условиями фильтра и нажмите кнопку ++OК++.

!!! info "Для сведения:"

    Настройте все условия фильтра в соответствии с текущим проектом, чтобы автоматически ***установить*** флажок в поле Шаблон активен. В таком случае, согласно инструкциям "Анализировать" и "Генерация отчета по проекту", из шаблона отчета будет создан отчет.

!!! info "Для сведения:"

    Настройте не все условия фильтра в соответствии с текущим проектом, чтобы автоматически ***снять*** флажок в поле Шаблон активен. В этом случае согласно инструкциям "Анализировать" и "Анализировать проект" из шаблона отчета не будет создан отчет.

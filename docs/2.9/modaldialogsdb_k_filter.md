# Фильтр

Фильтры создают по определенным правилам подмножества из множества. Они ограничивают, например, количество просматриваемых элементов в списке или дереве. Для этого необходимо определить критерии фильтра. Один раз определенный фильтр может сохраняться как схема и затем использоваться повторно или дорабатываться. Все сохраненные фильтры вызываются через раскрывающийся список. Как только выбирается фильтр, данные фильтруются на основании настроенных критериев и выводятся в диалоговом окне.

### Быстрый ввод

В различных диалоговых окнах (напр., навигаторах данных проекта, управлении проектами) можно фильтровать отображенные данные проекта при помощи быстрого ввода. Для этого в диалоговых окнах под фильтром предусмотрено поле Значение: <Свойство>. При помощи быстрого ввода в данном поле для определенного и активированного фильтра можно быстро изменить значение его критерия.

Для этого в диалоговом окне Фильтр, открыть которое можно при помощи кнопки ++...++ в навигаторе и которое позволяет задать критерии фильтра, имеется дополнительный столбец Быстрый ввод. Флажок быстрого ввода можно установить только для ***одного*** критерия фильтра. Если этот флажок установлен, поле Значение: <Свойство> деблокируется для навигатора, как только будет выбрана соответствующая схема фильтра. Будет показано не только свойство, выбранное в качестве критерия фильтрации, но и выбранный оператор.

В зависимости от свойства, выбранного в качестве критерия (например, структурный идентификатор), в поле Значение: <Свойство> можно щелкнуть по кнопке ++...++ и в следующем диалоговом окне выбрать для критерия фильтра другое значение. При вводе в это поле нескольких значений их следует разделять точкой с запятой. Данная запись будет интерпретирована как использование операции "ИЛИ".

!!! note "Замечание:"

    Если в диалоговом окне Фильтр ни для какого критерия фильтра быстрый ввод не активирован, то в соответствующем навигаторе быстрый ввод будет отключен. Поле Значение: <Свойство> будет недоступно.

!!! example "Пример:"

    В навигаторе устройств создана схема фильтра для образца проектаESS_Sample_Project. При этом в качестве критерия выбрано свойство Место установки(ID 1220) со значениемA1. Кроме того, для этого критерия установлен флажок Быстрый ввод. После этого в навигаторе будут отображаться толькоустройствас местом установкиA1.В поле Значение: Место установкивведитеместо установкиA2и подтвердите ввод. Сразу после этого в навигаторе будут отображаться только устройства с заданным структурным идентификатором. Если в этот момент перейти в диалоговое окно Фильтр, то можно увидеть, что значениеA2было скопировано для критерия фильтра Место установки.

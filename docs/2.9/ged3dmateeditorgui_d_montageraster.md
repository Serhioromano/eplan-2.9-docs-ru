# Диалоговое окно Монтажная сетка

Вы открыли проект. Проект содержит как минимум одно пространство листа. Пространство листа содержит трехмерный объект. Обработать > Логика устройства > Монтажная сетка.

В этом диалоговом окне вы можете настроить свойства монтажной сетки. Монтажные сетки присваиваются отдельным поверхностям трехмерного размещения изделия и при размещении других компонентов отображаются на этих поверхностях. Таким образом, размещение возможно теперь только в точках пересечения линий сетки. Монтажная сетка позволяет увидеть, например, на профилях электрошкафа образцы отверстий, специфических для производителя и типа, внутри которых при монтаже будет выполняться резьбовое соединение.

Точки сетки, определенные в диалоговом окне, всегда привязаны к нулевой точке площади, которая отображается в системе координат.

Обзор основных элементов диалогового окна:

Групповое поле "Направление X"

### Интервал сетки

Задайте интервал между двумя линиями сетки в направлении X.

### Количество линий сетки

Число повторений интервала сетки, заданного посредством линий сетки в направлении X.

Сетка распространяется до конца монтажной поверхности, если в раскрывающемся списке выбрано значение "Без ограничений".

Число линий сетки в направлении X и Y не должно одновременно равняться 1.

* * *

Групповое поле "Направление Y"

### Интервал сетки

Задайте интервал между двумя линиями сетки в направлении Y.

### Количество линий сетки

Число повторений интервала сетки, заданного посредством линий сетки в направлении Y.

Сетка распространяется до конца монтажной поверхности, если в раскрывающемся списке выбрано значение "Без ограничений".

Число линий сетки в направлении X и Y не должно одновременно равняться 1.

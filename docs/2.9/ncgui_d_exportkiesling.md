# Диалоговое окно Экспорт Rittal - Perforex BC / Secarex

Вы открыли проект. В пространстве листа или навигаторе пространства листа выделены объекты, для которых необходим экспорт ЧУ. Сервисные программы > Данные изготовления > Механическая обработка > Rittal - Perforex BC / Secarex.

В этом диалоговом окне устанавливаются настройки для экспорта данных ЧУ в формате "Rittal - Perforex BC / Secarex".

Обзор основных элементов диалогового окна:

### Машина

В комплекте поставки содержится определение машины с названием "Стандарт". Если в настройках фирмы определены другие схемы, можно выбрать из раскрывающегося списка другие определения машины.

С помощью кнопки ++...++ открывается диалоговое окно Настройки: Экспорт Rittal - Perforex BC / Secarex, в котором можно изменить настройки схемы или определить новую схему.

### Материал

В раскрывающемся списке выберите тип текущего обрабатываемого материала. Этот материал в схеме определен как [Номер материала](ncgui_r_allgemeinexportkiesling.md).

### Каталог вывода

В этом поле выводится место хранения. Здесь по умолчанию выбран каталог, который указан в пользовательских настройках экспорта Rittal - Perforex BC / Secarex.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Целевой файл

Данное поле доступно только в том случае, если выбраны объекты, которые могут быть экспортированы (например, монтажная плата). Здесь можно ввести название генерируемого файла ЧУ или выбрать с помощью [...] в диалоговом окне Сохранить как. При наличии нескольких файлов поле деактивируется и действуют условия создания названий файлов и подкаталогов, определенные в схеме.

### Применить ко всему проекту

Если этот флажок установлен, текущее выделение распространяется на все важные для ЧУ монтажные поверхности проекта.

Если флажок не установлен, экспортируются только важные для ЧУ монтажные поверхности выделенных функциональных элементов.

Если отдельный проект уже выделен (и при этом фильтр не установлен), флажок автоматически включается и становится недоступным. В этом случае настройку больше невозможно изменить.

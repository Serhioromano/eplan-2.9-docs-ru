# Обработать свойства страницы

В диалоговом окне навигатора Страницы — <Имя проекта> можно просмотреть или обработать свойства страниц.

Условие:

Вы открыли проект.

1. Выделите необходимую страницу в дереве или списке. Возможен многократный выбор.
2. Выберите пункты меню Обработать > Свойства.
3. Измените требуемые свойства в диалоговом окне [Свойства страницы](pagebrowsergui_d_seiteneigenschaften.md).
4. Щелкните по кнопке ++OK++.

!!! note "Замечание:"

    В случае изменения идентифицирующих свойств страницы, например свойства Установка, все ОУ страницы также переименовываются с использованием этой установки.

!!! tip "Совет:"

    В представлении в виде списка навигатора страниц некоторые поля (Описание страниц, Номер чертежа и т. д.) можно обрабатывать прямо в таблице.

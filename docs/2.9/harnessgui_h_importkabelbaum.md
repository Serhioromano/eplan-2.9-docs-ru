# Импортировать данные проводного монтажа

Для проектирования проводного монтажа на платформе EPLAN можно импортировать данные проводного монтажа из EPLAN Harness proD.

Во время импорта выполняется синхронизация с уже имеющимися в проекте данными проводного монтажа (функциями и соединениями). При этом функции и соединения, содержащиеся только в проекте, а не в файле импорта, можно удалить из проекта. Если данные проводного монтажа отсутствуют в проекте, они импортируются в проект как неразмещенные функции и соединения.

Условия:

* Вы открыли проект.
* Данные проводного монтажа доступны в виде файла в формате hpdex.

1. Выберите следующие пункты меню: Данные проекта > Соединения > Импортировать > Данные проводного монтажа
2. В диалоговом окне Импорт данных проводного монтажа выберите для импорта требуемый файл с данными проводного монтажа (*.hpdex).
3. Щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    В диалоговом окне [Синхронизировать данные проводного монтажа](macrosgui_d_importabgleich.md) отображаются импортируемые и уже содержащиеся в проекте данные проводного монтажа.

!!! info "Для сведения:"

    В столбце Статус отображается, содержатся или не содержатся в проекте функции / соединения, а также имеют ли функции / соединения различные состояния обработки (например, другие значения свойств) в источнике данных и в проекте. Поля с измененными значениями свойств подкрашены темно-серым цветом.

4. В столбце Операция определите, что необходимо сделать с соответствующими функциями / соединениями: импортировать, изменить или не импортировать.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Данные будут импортированы в соответствии с настройками в открытый проект.

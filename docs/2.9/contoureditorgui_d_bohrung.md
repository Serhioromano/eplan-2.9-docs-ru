# Диалоговое окно Отверстие

Вы открыли проект. В редакторе контуров открыт контур типа "Генерирование записи данных контура ЧУ" (например, через пункты меню Сервисные программы > Основные данные > Контур (данные ЧУ) > Открыть), и для данного контура сгенерирована запись данных ЧУ. Сервисные программы > Основные данные > Контур (данные ЧУ) > Обработать запись данных > Генерировать отверстие.

В этом диалоговом окне указываются свойства отверстия.

Обзор основных элементов диалогового окна:

### Диаметр

Из раскрывающегося списка выберите определенный для машины диаметр отверстия. Ручной ввод здесь ***невозможен***.

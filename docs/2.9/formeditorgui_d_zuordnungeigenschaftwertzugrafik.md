# Диалоговое окно Присвоение "свойства / значений к графике"

Вы открыли проект.

* Сервисные программы > Основные данные > Форма > Открыть > [Открыть]. Вы открыли диалоговое окно Свойства формы — <Имя формы> (Всплывающее меню > Свойства на форме в диалоговом окне навигатора Страницы — <Имя проекта>) и в поле Значение свойства Присвоение свойства / значения графике щелкнули кнопку ++...++.
* Сервисные программы > Основные данные > Рамка > Открыть > [Открыть]. Вы открыли диалоговое окно Свойства рамки — <Имя рамки> (Всплывающее меню > Свойства на рамке в диалоговом окне навигатора Страницы — <Имя проекта>) и в поле Значение свойства Присвоение свойства / значения графике щелкнули кнопку ++...++.

В этом диалоговом окне укажите, для какого свойства формы или рамки, которое имеет определенное значение, должна быть выведена определенная графика в области данных. Для этого доступно десять свойств.

Если после определения присвоения в данном диалоговом окне в форме/рамке разместить свойство Присвоение свойства/значения графике [n], то символ будет размещен в зависимости от значения выбранного свойства. Для форм в клеммах, например, предусмотрены свойства Графическое представление в отчетах [1] и Графическое представление в отчетах [2], но могут быть использованы и любые другие свойства. Или свойство страницы Пользователь, доп. поле 1 можно использовать для отображения электронной подписи в рамке.

Обзор основных элементов диалогового окна:

### Библиотека символов

При выборе символа в этом поле выводится библиотека символов.

### Имя символа

В это поле вводится символ, графика которого должна быть присвоена свойству с указанным значением: Щелкните кнопку ++...++, чтобы открыть диалоговое окно Выбор символа и выбрать символ. Многократный выбор здесь ***невозможен***.

!!! note "Замечание:"

    Обратите внимание на то, что на просмотр выводятся только символы ***сохраненных*** библиотек символов. Если эта форма/рамка будет использоваться в проекте, в котором отсутствует или не сохранена соответствующая библиотека символов, то в отчете эти условные обозначения будут отсутствовать!

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

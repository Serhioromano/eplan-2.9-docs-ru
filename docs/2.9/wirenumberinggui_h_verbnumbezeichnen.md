# Обозначить соединения

Обозначение соединений происходит через точки определения соединения. Соединение можно обозначить только в том случае, если выбраны источник и цель соединения. (Если в графическом редакторе открыта страница, вы можете вручную выделить соединения, в ином случае они выбираются через страницы или проект.)

При дополнительной нумерации соединений учитываются объем и подобъем. Для содержащихся в выборе соединений автоматически определяется соответствующий объем и обозначение соединений копируется из объема согласно настройкам.

Условие:

Вы выделили соединения в графическом редакторе или в диалоговом окне навигатора Соединения — <Имя проекта>. Или вы выделили одну или несколько страниц либо проект в диалоговом окне навигатора Страницы — <Имя проекта>.

1. Выберите следующие пункты меню: Данные проекта > Соединения > Нумерация > Обозначить

!!! info "Для сведения:"

    После вызова пункта меню сначала автоматически генерируются новые соединения.

2. В диалоговом окне Обозначить соединения в поле Настройки выберите из раскрывающегося списка схему, которая будет определять настройки нумерации соединений. С помощью кнопки ++...++ откройте диалоговое окно [Настройки: Нумерация соединений](wirenumberinggui_d_verbnumeinstellungen.md). Здесь вы можете определять настройки для нумерации и сохранять их в схеме.
3. В таблице определите для каждой отображенной группы соединений начальное значение и величину шага нумерации. Если начальное значение и величина шага для всех счетчиков группы одинаковы, установите флажок Группа.
4. Установите флажок Выделить как "установленные вручную", чтобы присвоить свойство установленные вручную всем точкам определения соединений, для которых при нумерации будет определено обозначение.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Если установлен флажок Предварительный просмотр результата, сначала отображается диалоговое окно Обозначить соединения: Предварительный просмотр результата. В нем можно проверить новые обозначения соединений и при необходимости изменить их вручную.

!!! info "Для сведения:"

    Обозначения соединений выводятся на просмотр на выбранных для нумерации соединениях, если там уже размещены точки определения соединений.

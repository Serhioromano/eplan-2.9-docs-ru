# Вставить сегменты маршрутизации (топология)

Сегменты маршрутизации — это вставленные в ручном или автоматическом режиме маршруты, вдоль которых можно маршрутизировать соединения.

Условие:

В графическом редакторе уже открыта страница топологии.

1. Выберите пункты меню: Вставить > Топология > Сегмент маршрутизации.
2. Подвигайте курсор рядом с имеющейся функцией топологии, точкой маршрутизации или точкой разрыва топологии.

!!! info "Для сведения:"

    Имеющаяся точка будет выделена.

3. Разместите начальную точку, щелкнув мышью по выделенному объекту. Или разместите начальную точку произвольно, щелкнув мышью по свободной плоскости.
4. Укажите следующие точки сегмента маршрутизации и щелкните левой клавишей мыши, чтобы разместить каждую из этих точек.
5. Разместите конечную точку, щелкнув по другой функции топологии, точке маршрутизации или точке разрыва топологии.

!!! info "Для сведения:"

    Откроется диалоговое окно Свойства ++...++ (если в настройках пользователя для графической обработки в категории Диалоговые окна свойств установлен флажок При вставке точек маршрутизации и сегментов маршрутизации (топология)).

6. На вкладке [Сегмент маршрутизации](devicetaggui_r_cablingstrecke.md) введите данные для данного сегмента маршрутизации.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Свойства сегмента маршрутизации отображаются при нечетном числе участков на среднем участке, а при четном числе участков — на участке перед серединой.

8. Завершите операцию, выбрав пункт всплывающего меню Прервать операцию или нажав клавишу ++Esc++.

!!! tip "Совет:"

    * Через пункт всплывающего меню Завершить сегмент маршрутизации или при помощи клавиши ++Пробел++ можно свободно разместить конечную точку сегмента маршрутизации.
    * В навигаторе топологии можно генерировать неразмещенные сегменты через пункт всплывающего меню Генерировать сегмент маршрутизации.

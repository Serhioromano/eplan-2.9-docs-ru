# Выбрать элементы

Выбор элемента выполняется щелчком клавиши мыши по нем; в случае графических элементов с заливкой щелчок можно выполнить и внутри заливки.

Условие:

Вы открыли страницу, форму, рамку или символ.

### Выбрать несколько элементов по отдельности

1. Нажмите клавишу ++Ctrl++ во время выбора отдельных элементов.

!!! info "Для сведения:"

    Выделенные элементы будут представлены в другом цвете.

!!! tip "Совет:"

    Нажав ++Ctrl++ \+ [пробел], можно выделить несколько отдельных элементов с помощью клавиатуры.

### Выбрать область с элементами

1. Если необходимо выбрать несколько элементов, которые находятся рядом друг с другом, начертите мышью рамку вокруг этой группы.

!!! info "Для сведения:"

    Будут выделены все элементы, находящиеся внутри этой рамки.

!!! note "Замечание:"

    Чтобы выделить только те элементы, которые полностью находятся в рамке, перемещайте мышь слева направо. И наоборот, переместите мышь справа налево, чтобы выделить и "задетые" элементы, которые лишь частично находятся в рамке.

2. Для отмены выбора элемента в выделенной группе нажмите ++Ctrl++ и одновременно щелкните по элементу.

### Выбрать область с помощью клавиатуры

1. Обработать > Выделить > Область
2. Укажите первый угол для выбора области, нажав клавишу ++Ввод++.
3. Начертите рамку с помощью клавиш со стрелками. Рамку можно чертить в любом направлении.
4. С помощью клавиши ++Tab++ курсор можно перемещать между нижним правым углом и верхним левым углом, чтобы растянуть область в любом направлении.
5. Завершите выбор области, нажав клавишу ++Ввод++.

!!! info "Для сведения:"

    Будут выделены все элементы, находящиеся внутри этой области выбора.

### Выбрать все элементы

1. Выберите пункты меню Обработать > Выделить > Все, чтобы выбрать все элементы текущей страницы, текущей формы и т. п.

### Отменить выбор элемента

1. Для отмены выделения элементов нажмите клавишу ++Esc++.

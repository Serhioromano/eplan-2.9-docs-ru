# Диалоговое окно Внешние программы

Параметры > Внешние программы.

В этом диалоговом окне определяются настройки для внешних программ, которые можно вызывать из EPLAN.

Доступны следующие элементы диалогового окна:

### Содержимое меню

В этом списке содержатся тексты меню, настроенные для соответствующей внешней программы в поле Текст меню.

### Текст меню

Введите здесь тот текст, который должен выводится для соответствующей программы в меню Параметры. В меню перед текстов автоматически вставляется цифра, использующаяся в качестве "горячей клавиши" (т.е. она имеет подчеркивание и позволяет выполнить вызов с помощью клавиатуры).

!!! example "Пример:"

    Например, если ввести "Excel", в меню выводится запись "1Excel". Таким образом, нажав клавишу "1" , можно вызвать программу Excel.

### Команда

Настройте здесь последовательность вызова для соответствующей программы; щелкните по кнопке ++...++, чтобы просмотреть дисководы и каталоги и скопировать путь для вызова.

### Аргументы

Введите в этом поле те аргументы, которые должны передаваться при вызове соответствующей программы. Например, если программа должна при вызове открыть определенный файл, укажите здесь имя файла.

### Начальный каталог

Настройте здесь, в каком каталоге должна быть запущена внешняя программа (он не должен соответствовать каталогу, указанному в поле Команда); щелкните по кнопке ++...++, чтобы просмотреть дисководы и каталоги и выбрать каталог в интерактивном режиме.

### Панель инструментов

В окне Панель инструментов рядом с настройками для внешних программ находятся следующие кнопки:

Кнопка | Значение
---|---
[Добавить] | Вставляет новую запись в список Содержимое меню и в строку Текст меню; после этого запись можно обработать.
[Убрать] | Убирает текущую выделенную запись из списка Содержимое меню. Многократный выбор здесь невозможен.
[Переместить вверх] / [Переместить вниз] | Позволяет перемещать вверх или вниз текущую выделенную запись в списке Содержимое меню, влияя тем самым на последовательность отображения внешних программ в меню Параметры.

# Диалоговое окно Формат: Свойства

Вы открыли проект. Вы открыли диалоговое окно [Настройки: Нумерация соединений](wirenumberinggui_d_verbnumeinstellungen.md) (например, через пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Нумерация соединений). В диалоговом окне Настройки: Нумерация соединений вы выбрали определенную пользователем схему и перешли во вкладку Обозначение. Вы выделили запись в поле Группы форматов и щелкнули ![](images/all_edit_as.png){: .ui-icon } (Обработать) или ![](images/all_new_as.png){: .ui-icon } (Создать).

В диалоговом окне Нумерация соединений: Форматы в списке Доступные элементы формата выберите элемент формата типа "Данные" и перенесите его при помощи кнопки со стрелкой в список Выбранные элементы формата.

В этом диалоговом окне указывается форматирование для свойства в группе форматов.

Обзор основных элементов диалогового окна:

### Категория

Выберите из раскрывающегося списка свойства, которые будут отображены в списке ниже, например "Все", "Данные" или "Настройки". В списке выводятся все свойства выбранного типа, которым еще не было присвоено форматирование.

### Фильтр

Введите в это поле текст, по которому должна выполняться фильтрация свойств.

Если необходимо вернуться к прежнему виду свойств, удалите текст из поля ввода. Для этого нажмите кнопку ![](images/all_delete_as.png){: .ui-icon } (Удалить).

### Мин. число разрядов

Здесь указывается минимальное число разрядов для значения свойства. Более длинные значения выводятся полностью. Разрешено использовать только положительные числовые значения. При вводе нуля число разрядов является переменным.

### Заполнитель

Значение заполняется с помощью этого символа, если минимальное число разрядов больше, чем ширина значения. При переменном числе разрядов (Запись "0" в поле Мин. число разрядов) ввод заполнителя невозможен.

### Подавить пустые свойства

Установите этот флажок, чтобы не выводить свойства, которые не установлены. Кроме того, с помощью флажков Включая заданный разделитель и Включая следующий разделитель можно подавить разделители перед свойством и после него.

### Подавить свойства, равные нулю

Установите этот флажок, чтобы не выводить числовые свойства, значение которых равно нулю. Кроме того, с помощью флажков Включая заданный разделитель и Включая следующий разделитель можно подавить разделители перед свойством и после него.

### Язык

Для многоязычных свойство с помощью раскрывающегося списка можно указать язык вывода.

### Отобразить значения отд. соединений

Если этот флажок установлен, для выбранного свойства на просмотр выводится значение определенного соединения.

Если флажок не установлен, для свойства на просмотр выводится значение первого соединения, которое определяет объем. Это значение в этом случае равно для всего объема (например, сети).

# Вкладка Тексты заполнители

Вы открыли проект. Вы открыли в редакторе форм динамическую форму, которую хотите обработать (Сервисные программы > Основные данные > Форма > Открыть > [Открыть]). Вставить > Динамическая область > ++...++. Воспользуйтесь мышкой, чтобы начертить прямоугольник, представляющий динамическую область, и дважды щелкните по новой заданной области.

На этой вкладке в виде таблицы выводятся тексты-заполнители, присвоенные соответствующей области. Во вновь вставленной динамической области можно таким образом задать для отчета новые тексты-заполнители и настройки для позиции этих заполнителей.

Обзор основных элементов диалогового окна:

### Позиция

В этом раскрывающемся списке задается распределение текстов-заполнителей в динамической области. Тексты-заполнители без активированного блока выравнивания сохраняют свое положение всегда, независимо от выбранной настройки. Предусмотрен выбор следующих опций:

* Сохранить положение: Положения всех текстов заполнителей сохраняются.
* Подогнать тексты заполнители: Тексты заполнители подгоняются под область. В этом случае для самого нижнего текста-заполнителя с активированным блоком выравнивания ячейка в столбце Ширина становится неактивной. Ширина блока выравнивания для этого текста-заполнителя получается из ширины области за вычетом ширины для блоков выравнивания других подогнанных текстов-заполнителей.
* Равномерно распределить тексты заполнители: Тексты заполнители равномерно распределяются по области. В этом случае ячейки в столбце Ширина для текстов-заполнителей с активированным блоком выравнивания становятся неактивны. Значения ширины соответствующих блоков выравнивания рассчитываются таким образом, чтобы они подходили к настроенной сетке формы.

### Таблица

В таблице можно выполнить настройки для важнейших свойств отображения текстов-заполнителей (Активировать блок выравнивания, Начертить рамку позиции, Ширина блока выравнивания).

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

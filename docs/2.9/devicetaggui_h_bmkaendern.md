# Изменить ОУ функций

Вы можете изменить ОУ всех функций и представления устройства одним действием. Кроме того, можно изменить ОУ только у определенных представлений одной функции (напр., у однополюсного) или только у выбранной функции.

Условия:

* Вы открыли проект.
* Открыт Навигатор устройств или Графический редактор.

Если устройство содержит, например, главные функции, вспомогательные функции и различные представления функций, то Вы можете одновременно изменить ОУ для всех функций и представлений.

1. Активируйте режим "Свойства (общие)" через меню Параметры > Свойства (общие).
2. Откройте диалоговое окно "Свойства" для функции устройства, например, через меню Обработать > Свойства.
3. В диалоговом окне Свойства (общие): ++...++ выберите вкладку <Категория функции> (ОУ) главной функции.

!!! info "Для сведения:"

    Вкладка отображает свойства ***устройства***.

4. Щелкните рядом с полем Полное ОУ по ++...++.
5. В диалоговом окне Полное ОУ измените ОУ.
6. Подтвердите ввод.

!!! info "Для сведения:"

    ОУ изменяется у всех функций и представлений устройства.

!!! info "Для сведения:"

    Если при изменении в рамках устройства также изменяются и функции, не идентичные выделенной функции, то появляется диалоговое окно сообщения Сохранить изменение ОУ. Щелкните по кнопке ++Да++.

!!! tip "Совет:"

    * ОУ всех представлений функции устройства можно изменить и ***напрямую***. В навигаторе устройств выделите какое-либо устройство и выберите пункт всплывающего меню Переименовать. Напрямую введите новое ОУ.
    * С помощью пункта всплывающего меню Свойства (общие) можно производить изменения в рамках устройства для выделенной функции без предварительной активации режима "Свойства (общие)".

Если устройство содержит, например, главные функции, вспомогательные функции и различные представления функций, то вы можете одновременно изменить ОУ для ***всех*** представлений ***одной*** функции.

1. Активируйте режим "Свойства (общие)" через меню Параметры > Свойства (общие).
2. Откройте диалоговое окно "Свойства" для функции, например, через меню Обработать > Свойства.
3. В диалоговом окне Свойства (общие): ++...++ выберите вкладку <Категория функции> выбранной функции.

!!! info "Для сведения:"

    Вкладка отображает свойства ***Функция***.

4. Щелкните рядом с полем Полное ОУ по ++...++.
5. В диалоговом окне Полное ОУ измените ОУ.
6. Подтвердите ввод.

!!! info "Для сведения:"

    ОУ изменяется ***у всех представлений*** выбранной функции.

Если устройство содержит, например, главные функции, вспомогательные функции и различные представления функций, то Вы можете изменить ОУ исключительно для выбранной функции.

1. Отключите режим "Свойства (общие)" через меню Параметры > Свойства (общие).
2. Откройте диалоговое окно "Свойства" для функции, например, через меню Обработать > Свойства.
3. В диалоговом окне Свойства ++...++ активируйте вкладку <Категория функции> выбранной функции.

!!! info "Для сведения:"

    Вкладка отображает свойства ***выбранной функции***.

4. Щелкните рядом с полем Полное ОУ по ++...++.
5. В диалоговом окне Полное ОУ измените ОУ.
6. Подтвердите ввод.

!!! info "Для сведения:"

    ОУ у ***выбранной функции*** изменяется.

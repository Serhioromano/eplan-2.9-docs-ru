# Вставить графику форм

Определенные заполнители во время прогона отчетов могут быть заменены не текстом, а графическими символами, при этом присвоение соответствующей графике определяется в диалоговом окне Свойства формы — <Имя формы>. Имеются следующие тексты-заполнители:

* Присвоение индекса графике [1-100]: Заменяет текст заполнитель символом, который указан в диалоговом окне Присвоение индекса графике для соответствующего индекса.
В данном случае индексом является индекс заполнителя в графике от 1 до 100. В свойствах формы каждому индексу можно присвоить символ. Затем разместите в форме свойство Присвоение индекса графике [n], чтобы там всегда размещался именно тот символ, который был присвоен индексу n.
* Присвоение определения функции графике: Заменяет текст заполнитель символом, который был указан в диалоговом окне Присвоение определения функции графике для соответствующего определения функции.
Если после этого разместить свойство Присвоение определения функции графике в форме, то символ будет размещен в зависимости от определения выводимой функции.
* Присвоение свойства / значения графике [1-10]: Заменяет текст заполнитель символом, который был указан в диалоговом окне Присвоение свойства / значения графике для соответствующего свойства.
Если после этого разместить свойство Присвоение свойства / значения графике [n] в форме, то символ будет размещен в зависимости от значения выбранного свойства выводимой функции. Для клемм, например, предусмотрены свойства Графическое представление в отчетах [1] и Графическое представление в отчетах [2], но могут быть использованы и любые другие свойства.

Кроме того, имеются также специальные заполнители для созданной графики, которые используются в схемах клемм, в спецификации клемм и в схемах подключения клемм в качестве графики перемычек. Сюда относятся, например, заполнители "Графика для мостовых, вставных и проволочных перемычек", "Графика для внутр. пров. перемычек", "Графика для мостовых перемычек" и др.

Условия:

* Вы открыли проект.
* В редакторе форм открыта подходящая форма, в которой необходимо заменить тексты заполнители графикой (Сервисные программы > Основные данные > Форма > Открыть > [Открыть]).
* Установлены присвоения Индекса графике, Определения функции графике и Свойства / значения графике в диалоговом окне Свойства формы — <Имя формы>.

1. Вставить > Текст заполнитель
2. В диалоговом окне Свойства (текст заполнитель) щелкните кнопку ++...++ в поле Свойство вкладки Размещение.
3. В диалоговом окне Тексты заполнителей — <Тип формы> выберите элемент (например, свойства клеммы), для которого имеются тексты заполнителей для графики.
4. Выберите требуемый вид присвоения.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Свойство копируется в поле Свойство во вкладке Размещение.

6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Элемент присоединяется к курсору.

7. Позиционируйте текст заполнитель на странице формуляра.

!!! info "Для сведения:"

    Элемент присоединяется к курсору.

8. Позиционируйте текст заполнитель, второй, третий... раз.
9. Для завершения операции выберите пункт всплывающего меню Прервать операцию.

!!! info "Для сведения:"

    Текст-заполнитель размещается на странице формы и заменяется при соответствующем отчете на выбранную в свойствах формы графику.

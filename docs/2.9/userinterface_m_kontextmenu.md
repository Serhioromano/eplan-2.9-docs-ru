# Пункты всплывающего меню

Во многих диалоговых окнах всплывающее меню доступно в таблицах и полях. То, какие пункты меню отображаются и / или могут быть активированы, зависит, с одной стороны, от типа поля выбранного поля или выбранной ячейки таблицы, а с другой стороны, от того, выделена ли ячейка таблицы (одним щелчком по клавише мыши) или выбрана для обработки значений (двойным щелчком).

Перечисленные ниже пункты меню оказывают влияние на представление таблицы.

Пункт меню |  Значение
---|---
Закрепить область |  Закрепляет все строки таблицы, которые находятся над маркировкой, или все столбцы, которые находятся слева от маркировки. Строки ниже или столбцы справа от выделения можно передвигать с помощью полос прокрутки.
Снять закрепление |  Убирает закрепление, т. е. все строки и столбцы можно передвигать с помощью полос прокрутки.
Подобрать ширину столбца |  Все столбцы так подогнаны к таблице по ширине, что и заголовок, и содержимое столбцов полностью читаемы.
Размер шрифта 100% |  Этот пункт меню доступен, только если размер шрифта в таблице был изменен ([Ctrl] \+ прокручивание колесика мыши). Отображает тексты в таблице с исходным размером шрифта.
Развернуть все |  Этот пункт меню доступен, только если в столбце таблицы отображаются представления структуры дерева. Разворачивает все древовидные представления в столбце таблицы одновременно.
Свернуть все |  Этот пункт меню доступен, только если в столбце таблицы отображаются представления структуры дерева. Сворачивает все древовидные представления в столбце таблицы одновременно.
Конфигурировать |  Этот пункт меню доступен только в диалоговых окнах "Свойства". Открывает диалоговое окно Конфигурировать свойства. В этом диалоговом окне выберите те свойства, которые должны стандартно выводиться в таблице для каждого объекта соответствующей категории.
Конфигурировать представление |  Это пункт меню доступен во многих диалоговых окнах с табличным представлением либо представлением в виде списка или структуры дерева. Открывает диалоговое окно Конфигурировать представление, в котором можно задать, какие свойства должны отображаться в таблице, дереве или списке.

Для ввода и обработки значений в полях и таблицах используются перечисленные ниже пункты меню. Кроме того, известные функции Windows, такие как вырезать, копировать, вставить, удалить и т. д., доступны во многих местах — как для выделенных, так и для выбранных полей / ячеек таблицы.

Пункты меню для обработки выбранных полей / ячеек таблицы:

Пункт меню |  Значение
---|---
Вставить свойства / значения |  Только в таблицах свойств (имя свойства / значение). Позволяет вставить одно или несколько свойств, скопированных из другого объекта, вместе с соответствующими значениями. Если свойство уже имеется, вставляются только значения. Если свойство раньше еще не отображалось в таблице, после вставки оно появится вместе со скопированным значением.
Специальный символ |  Открывает диалоговое окно Специальный символ, в котором вы можете выбрать требуемый символ и вставить его на место текущей позиции курсора с помощью [OK].
Новая строка |  Вставляет новую строку на место текущей позиции курсора.
Многострочный ввод |  Открывает диалоговое окно Многострочный ввод, в котором можно ввести требуемые строки, при этом новые строки добавляются с помощью [Ctrl]+[Ввод].
Обновить список выбора |  Только в полях, содержащих список выбора. Добавляет отсутствующие записи и удаляет неиспользуемые из списка выбора: Для определенных свойств занесенные здесь записи сохраняются в списке выбора и могут быть использованы для других (аналогичных) объектов.

Перечисленные ниже пункты меню используются для ввода и обработки многоязычных текстов в полях и таблицах.

Пункт меню |  Значение
---|---
Перевести |  Открывает диалоговое окно Найденные слова, если в словаре уже есть переводы для введенного термина; выберите соответствующий текст перевода. Если в словаре ***нет*** переводов, открывается диалоговое окно Перевести, после чего можно ввести новый текст перевода.
Убрать переводы |  Удаляет все тексты перевода для выделенных текстов. Выделенные тексты становятся независимыми от языка.
Обозначить текст, не требующий перевода |  Обозначает выделенный текст как "Перевод не требуется". Тексты, выделенные таким образом, исключаются из перевода и вывода в файле пропущенных слов.
Удалить обозначение |  Удаляет пометку "Не переводить" для выделенных текстов, так что они снова включаются в перевод.
Многоязычный ввод |  Открывает диалоговое окно Многоязычный ввод, в котором можно ввести требуемые тексты перевода для исходного текста.

# Формы для схем подключения

Схема подключения представляет целевую информацию о выводе устройства. Выводом устройства в этом смысле является, например, клемма по отношению к клеммнику, контакт штекера по отношению к штекеру и т. д. Схемы подключения могут быть представлены в многоступенчатом виде, т. е. поиск целей ведется в этом случае также через клеммы или другие передающие устройства, напр. переключатели.

Для клемм / контактов штекеров цели находятся на основе внутренних и внешних выводов устройства. Для других устройств цели находятся на основе свойства Отслеживание цели (ПЛК) по. Это свойство определяется в логической схеме выводов устройств.

Отслеживание цели в схемах подключения прерывается, если устройство находится повторно. После этого устройство больше не выводится. За счет этого, например, в клеммниках с несколькими PE- / PEN-клеммами, не связанными перемычкой, эти PE- / PEN-клеммы не выводятся несколько раз.

Схемы подключения могут иметь сколько угодно ступеней. Однако тексты заполнители имеются только для степеней 1-3. Для всех следующих ступеней тексты заполнители первой ступени удваиваются соответствующее количество раз.

### Свойства для форм схем подключения

Для схем подключения имеется ряд свойств формы, при помощи которых задается структура схем подключения. Ниже перечислены важнейшие из них:

* Схема подключений: Число степеней, внутр. / Схема подключений: Число степеней, внеш. (ид. 13077 / 13079): В схемах подключений с переменным числом степеней задает, сколько степеней предусмотрено для представления внутренних / внешних целей. Дополнительно при помощи свойств Схема подключений: Интервал между степенями, внутр. / Схема подключений: Интервал между степенями, внеш. (ид. 13078 / 13080) можно задать расстояние между двумя степенями для представления внутренних / внешних целей. Все степени представляются с одинаковым интервалом.
* Схема подключений: Отображать все цели (ид. 13084): При активации этого свойства на схеме подключения выводятся ***все*** цели, вплоть до первой степени. Для всех остальных степеней выводятся только ***первые*** цели. На схеме подключения цели размещаются в строке и объединяются, только если выводы устройств относятся к тому же устройству и в графике находятся рядом. При наличии нескольких целей графика соединения отображается в виде косой линии.
Если свойство деактивировано, отображается только первая цель.
Если нужно, чтобы цели отображались не в сгруппированном виде, а в форме таблицы, воспользуйтесь свойствами Схема подключений: Внутренние цели в табличной форме / Схема подключений: Внешние цели в табличной форме (ид. 13086 / 13087).
* Схема подключений: Выводить только сопряженные выводы устройства (ид. 13085): Если это свойство активировано, на схемах подключения на целях представлены только связанные выводы в графике. Таким способом освобождается место на схемах подключения. Данное свойство влияет только на цели, а не на анализируемые устройства, кабели, клеммники и штекеры. Данные функции также приводятся на схемах подключения. В выведенной схеме подключения клемм, например для двигателя, являющегося целью клеммника, выводятся только присоединенные на клеммнике выводы устройств функции.
* Схема подключений: Преследование цели по всем устройствам (ид. 13117): Если это свойство активировано, цели в схемах подключений отслеживаются через любые устройства. Для поиска остальных целей используется свойство Отслеживание цели (ПЛК) по в логической схеме выводов устройств. Кроме того, свойство Схема подключений: Выводить только сопряженные выводы устройства (ид. 13085) выводится в отчетах как активированное.
Последней внешней и внутренней целью является первое обнаруженное устройство, для которого не установлены выводы устройства для передачи в свойстве Отслеживание цели (ПЛК) в.
Если свойство деактивировано, поиск целей будет выполняться через клеммы и контакты штекера. На 'нормальных' устройствах поиск заканчивается.
* Схема подключений: Выводить только подключенные выводы устройства (ид. 13121): Это свойство доступно только для ***схем подключения устройства***; если оно активировано, выводятся только подключенные выводы оцененного устройства.
* Схема подключений: Выводить на целях только подключенные выводы устройства (ид. 13085): Это свойство доступно для схем подключения устройств, клемм, кабелей и штекеров; если оно активировано, выводиться будут только подключенные выводы устройства отображаемых целей.

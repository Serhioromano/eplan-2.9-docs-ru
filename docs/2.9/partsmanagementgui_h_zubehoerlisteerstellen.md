# Создать список принадлежностей и присвоить его главному изделию

Чтобы обеспечить простое управление частями принадлежностей, в базе данных изделий имеются так называемые "Списки принадлежностей". Они позволяют выбрать изделие из списка возможных альтернативных вариантов. При этом списки принадлежностей создают особый уровень иерархии в представлении структуры дерева базы данных изделий и отображаются там со своими именами.

После создания списков принадлежностей их можно присвоить главному изделию какого-либо типа изделий в базе данных изделий.

Условие:

Вы открыли базу данных изделий: Сервисные программы > Изделие > Управление

1. В ***дереве*** выделите уровень иерархии Список принадлежностей.
2. Выберите пункт всплывающего меню Создать.

!!! info "Для сведения:"

    EPLAN автоматически генерирует новый список принадлежностей с именем Создать_<порядковый_номер>.

3. Перейдите справа в диалоговом окне во вкладку Список принадлежностей.
4. В поле Имя измените автоматически заданную запись.

!!! info "Для сведения:"

    Свойства Дата создания и Дата изменения (вкладка Свойства) EPLAN автоматически заполняет соответствующими данными, которые не могут быть изменены.

5. Переместите вкладку Изделие на передний план.
6. Щелкните по ![](images/all_new_as.png){: .ui-icon } (Создать), чтобы создать строку в списке принадлежностей.
7. Щелкните в поле Номер изделия по ++...++.

!!! info "Для сведения:"

    Откроется диалоговое окно Выбор изделия. С помощью предварительной настройки "Принадлежности" фильтра по полю в виде дерева здесь отображаются только предварительно заданные детали принадлежностей.

8. Выделите нужный предмет оборудования в дереве и щелкните по кнопке ++ОК++.
9. Повторите выбор, если необходимо добавить другой предмет оборудования.
10. Закройте базу данных изделий и сохраните измененные данные.
11. Синхронизируйте данные изделия.

1. Выделите в дереве или списке нужное главное изделие.
2. Переместите вкладку Принадлежности на передний план.
3. Щелкните по ![](images/all_new_as.png){: .ui-icon } (Создать), чтобы создать строку в списке предметов оборудования.
4. Щелкните в поле Номер изделия / Имя по ++...++.
5. Выделите в структуре дерева заново созданный список принадлежностей на уровне иерархии Список принадлежностей.
6. Щелкните по кнопке ++OK++.
7. Закройте базу данных изделий и сохраните измененные данные.
8. Синхронизируйте данные изделия.

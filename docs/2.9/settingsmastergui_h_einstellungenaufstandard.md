# Сброс настроек на стандартные значения

В диалоговом окне настроек можно сбросить отдельные настройки с помощью команды во всплывающем меню в базовое состояние, определенное EPLAN. При этом можно сбросить отдельные поля или — посредством выбора в дереве — полные области настроек на значения по умолчанию.

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Параметры > Настройки
2. В диалоговом окне Настройки просмотрите структуру дерева до необходимой подкатегории.
3. Чтобы сбросить требуемую область настроек на стандартные значения, выделите ее в дереве.
4. Чтобы сбросить отдельное поле на стандартные значения, щелкните в правой части диалогового окна настроек по нужному полю или флажку.
5. Выберите пункт всплывающего меню Сбросить на стандарт.

!!! info "Для сведения:"

    Выбранные настройки (то есть поле или область) сбрасываются на значения по умолчанию.

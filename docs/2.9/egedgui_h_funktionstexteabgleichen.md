# Синхронизировать функциональные тексты

Синхронизация функциональных текстов используется для синхронизации разных текстов размещенных и неразмещенных функций обозначения устройства, например, с целью унификации различных видов написания.

Условия:

* Вы открыли проект. В Графическом редакторе открыта страница проекта.
* Выделено условное обозначение.

1. Выберите пункт всплывающего меню Синхронизировать функциональные тексты.
2. В диалоговом окне Синхронизировать функциональные тексты <ОУ> выберите определение функции, функциональный текст которого должен быть перенесен на все другие определения функций устройства.
3. При необходимости введите функциональный текст в столбце Функциональный текст или измените имеющийся функциональный текст.
4. Выберите пункт всплывающего меню Перенос функционального текста в строке определения функции, функциональный текст которого необходимо перенести на все другие определения функций.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выбранный функциональный текст копируется в свойства Функциональный текст и Функциональный текст (автоматически)***всех*** определений функции, при этом также копируются форматирование и перевод с иностранных языков. Уже существующие функциональные тексты при этом переписываются.

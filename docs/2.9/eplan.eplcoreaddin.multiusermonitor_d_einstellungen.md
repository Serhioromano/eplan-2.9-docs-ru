# Диалоговое окно Настройки: Многопользовательский монитор

Параметры > Настройки > Пользователь > Управление > Многопользовательский монитор

В данном диалоговом окне укажите, на каком компьютере программа Monitorserver настроена как служба.

!!! note "Замечание:"

    Со всеми вопросами по данным настройкам обратитесь к системному администратору.

Доступны следующие элементы диалогового окна:

### Имя сервера

В данном поле введите имя компьютера, на котором установлено приложение сервера для многопользовательского монитора.

### Порт

В данном поле введите номер порта приложения сервера.

### Активн.

Установите данный флажок, чтобы установить соединение с сервером.

### Статус

В этом поле отображается состояние соединения.

### [Проверить]

Чтобы проверить настройки, нажмите эту кнопку. При наличии соединения с компьютером сервера в поле Статус появится соответствующее сообщение.

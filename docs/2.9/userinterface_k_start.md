# Интерфейс пользователя

Интерфейс пользователя EPLAN позволяет выполнять следующие действия:

* Вызывать все функции программы, используя клавиатуру и мышь.
* По своему усмотрению изменять сочетания клавиш.
* Настраивать панели инструментов.
* Фильтровать записи данных.
* Определять, создавать и изменять схемы для фильтров и других настроек.
* Настраивать EPLAN для работы на нескольких мониторах.
* Определять последовательность отображения и сортировки данных.

# Использовать фильтр соединения (топология)

Сегменты маршрутизации имеют свойство Топология: Фильтр соединений. Фильтр соединений используется при маршрутизации, чтобы указывать, какие соединения можно маршрутизировать по данной трассе. Значениями таких критериев могут быть цветовые обозначения проводов, поперечных сечений или значения напряжения.

Присвоение значений маршрутизируемым соединениям носит исключительно текстовый характер. Для определения значений используются произвольно вводимые тексты, которые, однако, должны присутствовать также и на соединениях.

### Указать настройки проекта для фильтра соединений

Параметры, которые можно активировать или деактивировать в диалоговом окне Фильтр соединений, указываются в настройках проекта.

1. Выберите пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Маршрутизируемые соединения.
2. Выберите в диалоговом окне Настройки: Маршрутизируемые соединения вкладку Фильтр соединений.
3. Нажмите кнопку ++...++.
4. Выберите в диалоговом окне Выбор критериев свойство соединения, для которого следует отображать значения в фильтре соединений, например Цвет / номер соединения, и нажмите кнопку ++OK++.
5. Введите в столбце таблицы Значение тексты, описывающие критерий фильтра, например 'синий', 'красный', 'черный', 'желтый'. Для этого вам предоставлено 15 строк.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Настроенные значения сохранятся и будут отображаться в диалоговом окне Фильтр соединений при его вызове.

### Применение фильтра соединений

Используйте значения, определенные в фильтре соединений, на сегментах маршрутизации.

1. Выделите на странице топологии один или несколько сегментов маршрутизации и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Отобразится диалоговое окно Свойства (усл. обозначение): Сегмент маршрутизации.

2. Выберите свойство Топология: Фильтр соединений и в столбце Значение щелкните по кнопке ++...++.

!!! info "Для сведения:"

    Откроется диалоговое окно Фильтр соединений. В таблице отображаются значения, которые вы определили в настройках проекта.

3. Установите флажок Активн. над таблицей, чтобы фильтр соединений был включен всегда.

!!! info "Для сведения:"

    Теперь можно установить или снять отдельные значения в таблице.

4. Установите или снимите флажок Активн. в таблице перед каждым значением, которое учитывается для данного соединения.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В диалоговом окне Свойства (усл. обозначение): Сегмент маршрутизации в поле Значение свойства Фильтр соединений перечисляются все значения, которые вы активировали в фильтре соединений.

!!! info "Для сведения:"

    При маршрутизации соединения маршрутизируются только через обработанный сегмент маршрутизации, соответствующий установленным критериям фильтра.

# Опции

EPLAN поддерживает возможность создания исходного проекта, который содержит все возможные опции механизма, что, например, целесообразно для серийного производства. Эти опции являются вариантами механизма, например он может поставляться с отсасывающим устройством или без него. Кроме того, можно выполнить и изменение размеров. Благодаря простому подключению и отключению опций имеется возможность создания проекта для конкретного заказа.

Имеются следующие возможности обработки:

* Вы можете определить части установки и по выбору скрыть или показать их. Скрытые области не включаются в отчеты.
* Для опций можно указывать имена и описания.
* Фрагменты могут содержать объекты-заполнители, в которых сохраняются различные наборы значений.

# Исходные точки: Принцип

Исходные точки — это элементы логики устройства, которые используются при работе с принадлежностями электрошкафа. Под исходными точками понимаются строго определенные точки монтажа в пространстве листа. При помощи исходных точек можно устанавливать функциональные элементы шкафа (например, боковые стенки, цоколь) на строго определенном в управлении принадлежностями месте. После этого перемещать функциональные элементы уже нельзя.

Различают автоматически создаваемые исходные точки и исходные точки, определяемые пользователем. Автоматические исходные точки делают возможным размещение электрошкафов и соответствующих принадлежностей из управления принадлежностями без создания для этого подробных макросов. Также имеется возможность сохранять пользовательские исходные точки в макросе и использовать их в качестве свойства размещения принадлежностей.

### Автоматические исходные точки

Автоматические исходные точки генерируются в электрошкафах, созданных непосредственно без макроса из базы данных изделий (группа продуктов "Электрошкаф", подгруппы "Отдельная часть" и "Шкаф"). Автоматически создаваемые исходные точки используются только для автоматического размещения принадлежности и не учитываются при размещении вручную.

Автоматические исходные точки генерируются на размещениях следующих изделий:

Перфорированные профили горизонтальные, пол

* Перфорированный профиль внизу: слева по переднему краю
* Перфорированный профиль внизу: слева по заднему краю
* Перфорированный профиль внизу: справа по переднему краю
* Перфорированный профиль внизу: справа по заднему краю.

Перфорированные профили горизонтальные, крыша

* Перфорированный профиль вверху: слева по переднему краю
* Перфорированный профиль вверху: слева по заднему краю
* Перфорированный профиль вверху: справа по переднему краю
* Перфорированный профиль вверху: справа по заднему краю.

Корпус

* Корпус снаружи: вверху слева по переднему краю
* Корпус снаружи: вверху слева по заднему краю
* Корпус снаружи: вверху справа по переднему краю
* Корпус снаружи: вверху справа по заднему краю
* Корпус снаружи: внизу слева по переднему краю
* Корпус снаружи: внизу слева по заднему краю
* Корпус снаружи: внизу справа по переднему краю
* Корпус снаружи: внизу справа по заднему краю
* Корпус внутри: вверху слева по переднему краю
* Корпус внутри: вверху слева по заднему краю
* Корпус внутри: вверху справа по переднему краю
* Корпус внутри: вверху справа по заднему краю
* Корпус внутри: внизу слева по переднему краю
* Корпус внутри: внизу слева по заднему краю
* Корпус внутри: внизу справа по переднему краю
* Корпус внутри: внизу справа по заднему краю.

### Исходные точки, определяемые пользователем

Исходные точки, определяемые пользователем, — это точки монтажа с предварительно определенными свойствами, вручную размещаемые пользователем в макросе. Их можно добавить с помощью пунктов меню Обработать > Логика устройства > Исходные точки.

Как и при определении точек монтажа, в пространстве листа выбирается точка, на которой будет определена исходная точка. Для этого можно использовать опции размещения. В открывшемся диалоговом окне "Свойства" предлагаются на выбор все исходные точки, в том числе автоматические.

### Схема расположения точек вставки

Для электрошкафов, импортируемых в пространство листа в качестве данных STEP, все исходные точки необходимо создавать вручную. Однако исходные точки для серий электрошкафа Rittal (например, 'TS8') всегда находятся на одних и тех же местах. При создании электрошкафов серий Rittal 'TS8', 'AE' и 'CM' имеется возможность автоматически разместить исходные точки, соответствующие серии электрошкафа. С помощью пунктов меню Обработать > Логика устройства > Перенести схему расположения точек вставки можно автоматически сгенерировать исходные точки, соответствующие выбранной серии электрошкафа.

# Вкладка Общее (экспорт ЧУ Steinhauer)

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт ЧУ Steinhauer. Выберите вкладку Общ..
* Сервисные программы > Данные изготовления > Механическая обработка > ЧУ Steinhauer. Рядом с полем Машина нажмите ++...++. Выберите вкладку Общ..

На этой вкладке установите общие настройки для экспорта ЧУ для машин Steinhauer.

Обзор основных элементов диалогового окна:

### Формат

Управление машинами Steinhauer может осуществляться в следующих форматах:

* ЧПУ с десятичной точкой
* ЧПУ с десятичной запятой.

### Каталог вывода

Выберите каталог, в котором следует разместить экспортируемые файлы. Рабочее место в мастерской возле машины должно иметь доступ к этому каталогу.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Генерировать имена файлов

В этом раскрывающемся списке определяется стандартный порядок присвоения имен файлам экспорта ЧПУ.

* Из непрерывного номера: При выборе этого параметра EPLAN по умолчанию присваивает файлам ЧПУ имена по образцу <непрерывный_номер>.ЧПУ. Если уже имеются файлы ЧПУ, EPLAN начнет нумерацию со следующего доступного числа.
* Из обозначения функц. элемента: При выборе этого параметра EPLAN по умолчанию присваивает файлам ЧПУ имена по образцу <Обозначение функц. элемента>.ЧПУ.
* Из свойств: Если выбран этот параметр, имя файла составляется из свойств пространства листа, проекта и устройства, а также свободно определяемого разделителя. Используя кнопку ++...++, откройте диалоговое окно Формат: Свойство для конфигурации имени файла.

### Генерировать подкаталоги

В этом раскрывающемся списке определяется стандартный порядок создания подкаталогов при экспорте файлов ЧПУ.

* Нет: Если выбран данный параметр, при экспорте подкаталоги не генерируются.
* Из структуры проекта: Если выбран данный параметр, то при экспорте автоматически генерируются подкаталоги, отражающие структуру проекта.
* Из свойств: Если выбран этот параметр, имя подкаталога составляется из свойств пространства листа, проекта и устройства, а также свободно определяемого разделителя. При нажатии кнопки ++...++ открывается диалоговое окно Формат: Свойство для конфигурации имени каталога. Воспользуйтесь, например, элементом формата "Разделитель каталога", чтобы создать структуру каталога с несколькими подкаталогами, расположенными друг под другом.

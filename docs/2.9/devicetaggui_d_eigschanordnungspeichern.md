# Диалоговое окно Сохранить порядок свойств

Вы открыли проект.

* Вы вставили символ в Графическом редакторе или генерировали новую функцию в Навигаторе.
* Вы выделили функцию в графическом редакторе или навигаторе. Обработать > Свойства.

Выберите в диалоговом окне Свойства ++...++ вкладку Отображение. В списке Порядок свойств выберите запись "Опред. пользователем", обработайте порядки свойств и щелкните по кнопке ![](images/all_save_as.png){: .ui-icon } (Сохранить) на панели инструментов, расположенной над раскрывающимся списком.

При помощи этого диалогового окна сохраните связанный с пользователем порядок свойств под независимым от языка именем.

Обзор основных элементов диалогового окна:

### Порядок свойств

Укажите в этом поле описательное название для порядка свойств. Сразу после сохранения пользовательского порядка свойств можно выбрать имеющееся имя из раскрывающегося списка и перезаписать старый порядок свойств.

!!! note "Замечание:"

    В случае перезаписи определенного пользователем порядка свойств измененные настройки отображения будут перенесены на все условные обозначения, использующие данный порядок свойств.

### Использовать как стандарт

Этот флажок доступен только для тех символов и функций, которые ***не являются точками обозначения соединения***.

Установите этот флажок, если сохраненный порядок должен использоваться по умолчанию со всеми дальнейшими применениями этого варианта символа. При этом все порядки, сохраненные для данного варианта символа, будут игнорироваться.

### Использовать как стандарт для соединений / Использовать как стандарт для кабельных соединений

Этот флажок доступен только для тех символов/ функций, которые ***являются точками обозначения соединения***!

Установите флажок Использовать как стандарт для соединений, если сохраненный порядок должен использоваться по умолчанию со всеми дальнейшими применениями этого варианта символа для "обычных" точек определения соединений. Порядок свойств используется также для точек определения соединений, возникающих при нумерации соединений.

Установите флажок Использовать как стандарт для кабельных соединений, если сохраненный порядок должен использоваться по умолчанию со всеми дальнейшими применениями этого варианта символа для точек обозначения соединения, которые дополнительно размещаются при вычерчивании кабелей/экранирований.

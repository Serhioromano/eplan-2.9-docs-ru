# Создать схему присвоения для символов (изменение стандарта)

В схеме присвоения символы целевой библиотеки символов присваиваются символам исходной библиотеки символов. При изменении стандарта символы исходной библиотеки символов заменяются на символы целевой библиотеки символов.

Условия:

* Вы открыли проект.
* При нескольких открытых проектах: В навигаторе страниц выделен проект.
* Вы настроили целевой проект.
* Вы заменили необходимые формы и рамки.
* На вкладке Основные данные выбрана целевая библиотека символов и установлен флажок Присвоить отдельно.

### Составление схемы

1. Сервисные программы > Изменение стандарта > Вкладка "Проект" > [Далее] > Вкладка "Поворот / инверсия" > [Далее] > Вкладка "Основные данные" > [Далее]
2. Щелкните на вкладке Символы по ++Схема++.
3. В диалоговом окне Настройки: присвоение символов в поле Схема выберите схему, которую нужно изменить, или щелкните ![](images/all_new_as.png){: .ui-icon } (Создать) и создайте схему.

### Выбрать исходную и целевую библиотеку символов

1. Выберите ++Дополнительно++ > Выбрать исходную и целевую библиотеку символов.
2. В диалоговом окне Выбрать исходную библиотеку символов выберите библиотеку символов, для которой нужно создать присвоение, и щелкните по ++ОК++.
3. В диалоговом окне Выбрать целевую библиотеку символов выберите библиотеку символов, для которой нужно создать присвоение, и щелкните по ++ОК++.

#### Автоматическая запись и присвоение исходных и целевых символов

1. Выберите ++Дополнительно++ > Автоматически добавить исходные символы.

!!! info "Для сведения:"

    Все символы выбранной библиотеки исходных символов будут добавлены в таблицу Присвоить символы.

2. Выберите ++Дополнительно++ > Предложить подходящие целевые символы.

!!! info "Для сведения:"

    Все символы выбранной библиотеки целевых символов будут добавлены в таблицу Присвоить символы и при этом присвоены исходным символам.

3. Проверьте присвоение символов.

!!! info "Для сведения:"

    Если EPLAN находит подходящие символы для [Замены символов](changestandardsgui_k_symbolaustausch.md), отображается состояние 'В порядке' ![](images/all_statusok_as.png){: .ui-icon }.

4. Щелкните по кнопке ![](images/all_save_as.png){: .ui-icon } (Сохранить) и по ++OK++.

!!! info "Для сведения:"

    Схема будет занесена на вкладку Символы.

5. Щелкните по ++Далее++.

!!! info "Для сведения:"

    [Определить Структура проекта](changestandardsgui_h_projektstrukturanpassen.md).

#### Добавление и присвоение исходных и целевых символов вручную

Условие:

Выбрана библиотека исходных и целевых символов.

1. В диалоговом окне Настройки: присвоение символов рядом с полем Присвоить символы щелкните ![](images/all_new_as.png){: .ui-icon } (Создать).

!!! info "Для сведения:"

    В таблицу Присвоить символы вставляется пустая строка.

2. Щелкните в области Источник в столбце Имя символа.

!!! info "Для сведения:"

    Библиотека исходных символов откроется в диалоговом окне Выбор символа.

3. Выберите символ и щелкните по ++OK++.
4. В диалоговом окне Настройки: присвоение символов в области Цель щелкните в столбце Имя символа.

!!! info "Для сведения:"

    Библиотека целевых символов откроется в диалоговом окне Выбор символа.

5. Выберите в диалоговом окне [Замена символов](changestandardsgui_k_symbolaustausch.md) подходящий символ и щелкните по кнопке ++ОК++.

!!! info "Для сведения:"

    Проверяется присвоение символов.

6. Аналогичным образом присвойте друг другу другие символы.
7. Щелкните по кнопке ![](images/all_save_as.png){: .ui-icon } (Сохранить) и по ++OK++.

!!! info "Для сведения:"

    Схема будет занесена на вкладку Символы.

8. Щелкните по ++Далее++.

!!! info "Для сведения:"

    [Определить Структура проекта](changestandardsgui_h_projektstrukturanpassen.md).

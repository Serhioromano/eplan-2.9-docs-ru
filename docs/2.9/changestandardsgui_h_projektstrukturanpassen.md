# Определить структуру проекта (изменение стандарта)

Пользовательские и предварительно определенные схемы для структуры страниц и устройств применяются из шаблона, который был указан при создании целевого проекта.

Условия:

* Вы открыли проект.
* При нескольких открытых проектах: В навигаторе страниц выделен проект.
* Вы настроили целевой проект.
* Вы определили шаблон и поворот / инверсию.
* Вы заменили необходимые рамки, формы, библиотеки символов и символы.

1. Сервисные программы > Изменение стандарта > Вкладка "Проект" > [Далее] > Вкладка "Поворот / инверсия" > [Далее] > Вкладка "Основные данные" > [Далее]
2. Для изменения структуры страниц выберите на вкладке Структура из раскрывающегося списка Страницы схему идентификаторов для структуры страниц.
3. Для изменения разделителя нажмите ++Другие++ и введите в диалоговом окне Расширенные структуры проекта нужные разделители.
4. Для изменения схемы идентификаторов для устройств выберите из соответствующего раскрывающегося списка схему идентификаторов.
5. Нажмите кнопку ++Далее++.

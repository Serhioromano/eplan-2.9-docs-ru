# Определить многополюсные соединения в однополюсной точке определения соединений

В диалоговом окне "Свойства" однополюсной точки определения соединений определите, между какими выводами устройств должны быть многополюсные соединения. При этом можно генерировать перемычки (соединения между выводами устройств того же устройства) и добавить изоляцию. Кроме того, для определенных многополюсных соединений можно задать данные соединений.

Условия:

* Многополюсные функции устройства (например, контакты штекеров) должны создаваться в проекте как неразмещенные функции.
* Однополюсная функция устройства (например, штекера) является главной функцией. Для этой функции должно быть активировано свойство Неразмещенные вспомогательные функции управляются в главной функции.

* В однополюсном представлении сгенерированы однополюсные соединения.
* Вставлена точка определения соединений и присвоена виду представления "Однополюсн".

1. Выделите точку определения соединения и выберите пункт меню Обработать > Свойства (или щелкните по точке определения соединений).

2. Выберите в диалоговом окне Свойства ++...++ вкладку Подключить устройства.

!!! info "Для сведения:"

    В поле Источник устройства автоматически отображается устройство, являющееся источником многополюсного соединения. Выводы устройства отображаются в столбце Вывод устройства. При этом отображаются только выводы устройств неразмещенных многополюсных функций.

!!! info "Для сведения:"

    В поле Цель устройства автоматически отображается устройство, являющееся целью многополюсного соединения. Выводы устройства отображаются в столбце Вывод устройства. При этом отображаются только выводы неразмещенных многополюсных функций.

3. Выделите в левой и правой областях диалогового окна равное количество выводов устройства.

!!! info "Для сведения:"

    Если выделено неравное число выводов устройства, тогда все соединения создаются для меньшего количества. Если было выделено больше выводов устройства, чем имеется кабельных соединений, соединяются те выводы устройства, для которых доступны кабельные соединения.

4. Щелкните по ++Генерировать соединения++.

!!! info "Для сведения:"

    Генерируются многополюсные соединения. Если кабель присвоен однополюсному соединению, генерируются соединения кабелей, в противном случае генерируется жильное / проволочное соединение.

5. Обработайте при необходимости соединения. Для этого выделите требуемое соединение и выберите Всплывающее меню > Обработать.
6. Введите в диалоговом окне Свойства (усл. обозначение): соединение требуемые данные соединения (например, цвет соединения и обозначение соединения) и щелкните ++OK++.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Многополюсные соединения и данные соединений сохраняются в точке определения соединений.

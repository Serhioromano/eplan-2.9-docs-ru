# Диалоговое окно Свойства (<Графический элемент>)

Вы открыли страницу, форму, рамку или символ. Вы выделили графический элемент, выбрали пункты меню Обработать > Свойства или дважды щелкнули по графическому файлу.

В этом диалоговом окне можно обрабатывать свойства отображения графических элементов, например линий, кругов, прямоугольников и т. д.

Обзор основных элементов диалогового окна:

### Свойство/Присвоение

В таблице существуют уровни иерархии <Графический элемент> и Формат, которые открываются после щелчка по символу ![](images/all_plussign_as.png){: .ui-icon }.

### Невидимый

В раскрывающемся списке выберите запись "Да", если элемент не должен отображаться на странице схемы соединений. Однако он имеется в проекте. В настройках пользователя определяется, требуется ли выводить и / или распечатывать невидимые элементы (другого цвета).

### Тип линии

Выберите из раскрывающегося списка требуемый тип линии.

!!! note "Замечание:"

    * Есть определенные типы линий, которые содержат дополнительную графику. Эти типы линий предусмотрены в основном для использования в функциональной схеме автоматизации. Если для графического элемента, состоящего из дуг или кривых, установлен такой тип линии, тогда этот тип линии игнорируется, и линия отображается как сплошная. При этом несущественно, задается ли тип линии непосредственно через поле Тип линии или через выбранный слой. Все другие настройки для линии (толщина линии, цвет и т.д.) остаются прежними.
    * Для дополнительной графики (окружности, кресты и т.д.) используются стандартные размеры. В случае значительных изменений (например, длины образца или толщины линии) это может привести к тому, что эти дополнительные виды графики больше невозможно будет распознать. Кроме того, двойные линии отображаются как отдельные линии только при очень маленькой толщине.

### Выступ

Этот флажок доступен только для ***линий***. Установите его, чтобы задать выступ для начальной и конечной точек (например, при изображении средних линий). Согласно DIN он будет рассчитан на основе толщины линии и изображен длиной в "12 d", (при этом d = диаметр штриха (толщина линии)).

!!! note "Замечание:"

    Точки захвата и указание размеров линии при вставке выступа остаются неизменными.

# Предварительное планирование: Отчеты

Для предварительного планирования существуют следующие типы отчетов:

* ***Предварительное планирование: Обзор сегментов структуры:*** Обзор всех сегментов структуры и необходимых затрат (время, энергия и расходы) по подчиненным объектам планирования.
* ***Предварительное планирование: План сегмента структуры:*** Для каждого сегмента структуры создается отчет, при этом можно вывести различные свойства сегмента структуры (Техническое описание, Расход энергии и т. д.). Дополнительно выводятся непосредственно подчиненные сегменты области данных.
* ***Предварительное планирование: Обзор объектов планирования:*** Обзор всех объектов планирования и необходимых затрат (время, энергия, и расходы), а также заложенных для этого документов, изделий и адресов ПЛК. С помощью обзора объектов планирования также можно выводить (при использовании подходящих форм) списки функций для автоматизации зданий, а также отчеты — для объектов планирования (трубопроводов), включая относящиеся к ним свойства (вещество, класс трубы, исходный и целевой сегменты и т. д.).
* ***Предварительное планирование: План объекта планирования:*** Соответствует таблице параметров. Для каждого объекта планирования выводится список изделий, адресов ПЛК и т. д., которые записаны на объекте планирования. Дополнительно выводятся непосредственно подчиненные сегменты области данных. С помощью этого отчета также можно выводить (при использовании подходящей формы) отчеты для объектов планирования (трубопроводов), включая относящиеся к ним свойства (вещество, класс трубы, исходный и целевой сегменты и т. д.).
* ***Предварительное планирование: Обзор шаблонов сегментов:*** Обзор всех шаблонов сегментов.
* ***Предварительное планирование: Схема шаблонов сегментов:*** На шаблон сегмента генерируется один отчет, при этом выводятся различные свойства шаблона сегмента, например изделия, внесенные в шаблон сегмента. Дополнительно выводятся свойства сегментов, которые используют шаблон сегмента.
* ***Предварительное планирование: Обзор категории трубы:*** Перечисляет все классы трубы проекта, управляемые в навигаторе шаблонов сегментов. С помощью обзора категории трубы также выводятся отчеты для класса трубы (при использовании соответствующей формы). При этом для каждого класса трубы выводится список свойств, которые указываются на классе трубы.
* ***Предварительное планирование: Обзор среды:*** Перечисляет все вещества проекта, управляемые в навигаторе шаблонов сегментов. С помощью обзора среды также выводятся отчеты для вещества (при использовании соответствующей формы). При этом для каждого вещества выводится список свойств, которые указываются на веществе.
* ***Обзор групп изделий:*** Обзор всех узлов и модулей, примененных в проекте. Свойство Тип записи данных позволяет видеть различия между узлами и модулями. С помощью этого отчета можно выводить [монтажные чертежи](planninggui_k_montageanordnungen.md) для объектов планирования.

Изделия, занесенные ***только*** в предварительном планировании на объектах планирования, могут также учитываться в значениях изделий (спецификация изделий, групповая спецификация изделий и т.п.), а также в спецификации. Для этого в диалоговом окне [Настройки: Изделие](partslistgui_d_paramartikelauswertung.md) необходимо установить флажок Изделие предварительного планирования.

В других отчетах (список устройств, спецификация клеммника, Схема ПЛК и т.п.) вы можете выводить объекты планирования, связанные с главными функциями. Для такого вывода в соответствующих формах имеются элементы-заполнители для объектов планирования.

# Символы

Символы нужны, чтобы начертить схему соединений простым способом. Они обладают большим количеством информации, такой как графические элементы, точки вывода устройства, присвоение к группе символов, логика ... При этом связь между символом и функцией может выглядеть следующим образом:

* Символ не отображает функцию, например угол, тройник.
* Символ отображает функцию, например замыкающий контакт, размыкающий контакт.
* Символ представляет много функций, напр. автомат защиты двигателя, тройной предохранитель.
* Символ представляет часть функции, напр. вывод устройства, переменный контактный крючок.

Для управления символами имеются библиотеки символов. При этом в библиотеке символов могут содержаться сколько угодно символов. При обработке символа открывается и записывается не вся библиотека символов, но всегда только изменяемый символ. Это сокращает проблемы, которые могут возникнуть, если несколько пользователей одновременно обрабатывают одну библиотеку символов.

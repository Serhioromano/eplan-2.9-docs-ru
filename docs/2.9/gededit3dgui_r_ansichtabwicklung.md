# Вкладка Вид (развертка)

* Вы открыли проект, содержащий пространства листов и размещенные в них сборные шины (гнутые), и находитесь на странице графического редактора. Вставлена и размещена развертка шины (Вставить > Графика > Развертка шины). Выберите в диалоговом окне Развертка вкладку Вид.
* Вы открыли проект, содержащий пространства листов и размещенные в них сборные шины (гнутые), и находитесь на странице графического редактора. Вы дважды щелкнули по развертке. Выберите в диалоговом окне Развертка вкладку Вид.

На этой вкладке обрабатываются свойства разверток.

Обзор основных элементов диалогового окна:

### Имя вида

Каждая развертка идентифицируется через уникальное имя, сохраненное в соответствующем настроенном языке. Наличие нескольких разверток с одинаковыми именами невозможно.

### Описание

Описание представляет собой любой многоязычный текст, подробно описывающий вид.

### Пространство листа

Развертка может быть создана только для одного пространства листа. В этом раскрывающемся списке перечислены все созданные в проекте пространства листов. При первом открытии диалогового окна в нем отображается имя первого пространства листа проекта.

### Основной функциональный элемент

Укажите здесь, какой медный функц. элемент должен быть представлен в развертке. Таким образом можно дополнительно ограничить отображаемое содержимое пространства листа. Щелкните по кнопке ++...++, чтобы открыть диалоговое окно [Выбор 3D-объекта](gededit3dgui_d_3dobjektauswahl.md) и выбрать медный жгут или сборную шину, которые должны отображаться в развертке.

В двухмерных отображениях отверстий и в обзорах моделей вы можете также выбрать в качестве базисного функционального элемента компоненты, расположенные глубже в иерархии функциональных элементов, например монтажные поверхности. В случае с развертками шины это невозможно.

### Настройка масштаба

Следующий способ можно настроить для вычисления масштаба.

* Автоматически: Масштаб вычисляется таким образом, чтобы развертка в любом случае подходила для выделенной области.
* Подогнать: Масштаб вычисляется таким образом, чтобы развертка как можно больше заполняла выделенную область.
* Задано вручную: При выборе данной настройки разблокируется поле Масштаб, и можно установить любой масштаб.

### Масштаб

Данное поле активно, если в поле Настройка масштаба выбран параметр 'Задано вручную'; введите здесь масштаб в формате '1:x' или выберите значение из раскрывающегося списка.

[Групповое поле "Свойства"](devicetaggui_r_gruppenfeldeigenschaften.md)

Для разверток шины предлагаются следующие свойства:

* Маркировка функциональных элементов: Здесь определяется схема, согласно которой маркируются функциональные элементы в развертке шины. Щелкните по кнопке ++...++, чтобы открыть диалоговое окно [Настройки: Маркировка](gededit3dgui_d_einstellbeschriftung.md) и выбрать имеющуюся схему или создать новую.
* Поворот: При выборе значения из этого раскрывающегося списка содержание развертки шины поворачивается на 90°, 180° или 270° против часовой стрелки. Это может быть полезно для оптимального использования вертикальных разверток шины на страницах альбомного формата. При этом рамка развертки шины не поворачивается. Тексты развертки шины, повернутой на 90°, выводятся повернутыми под таким же углом. Исключение составляют тексты свойств, читаемые справа. Чтобы они оставались читабельными, поворот выполняется на -90°.
* Автоматическое проставление размеров: Здесь определяется схема, согласно которой в развертке шины автоматически проставляются размеры функциональных элементов. Щелкните по кнопке ++...++, чтобы открыть диалоговое окно [Настройки: Автоматическое проставление размеров](gededit3dgui_d_einstellautobemassung.md) и выбрать имеющуюся схему или создать новую.
* Отобразить схему сверления: Если этот флажок установлен, отверстия, имеющиеся на функциональном элементе из меди, отображаются в развертке.
* Отобразить линии изгиба: Если этот флажок установлен, на развертке в качестве сплошных линий отображаются линии изгибов отдельных мест изгиба.
* Отобразить концы изгиба: Если этот флажок установлен, на развертке в качестве пунктирных линий отображаются концы изгибов отдельных мест изгиба.
* Представить примечание для сгибания: Если этот флажок установлен, на линиях сгиба в качестве текстов отображаются данные по направлению изгиба, углу изгиба и радиусу изгиба.
* Обзор модели: ширина ребер: С помощью этого свойства можно задавать фиксированное значение для ширины линий ребер в обзоре модели, 2D-отображении отверстий и развертке шины, например 0,5 мм. При масштабировании и изменении масштаба ширина линий изменяется соответственно. При экспорте PDF линии ребер выводятся с заданной шириной.

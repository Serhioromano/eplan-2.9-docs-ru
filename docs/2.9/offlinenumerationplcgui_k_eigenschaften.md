# Свойства для нумерации с данными ПЛК

У каждой функции имеется два свойства для нумерации с данными ПЛК

* Нумерация с данными ПЛК
* Продолжить нумерацию с данными ПЛК далее после функции.

Свойство Нумерация с данными ПЛК указывает, было ли изменено ОУ (или обозначение клеммы / контакта штекера) во время нумерации с данными ПЛК. Оно автоматически устанавливается во время нумерации, но его можно присвоить и вручную.

В этом случае при нумерации ОУ, клемм или контактов штекеров можно будет выбрать, следует ли нумеровать функции с этим свойством или же исключить из нумерации.

!!! example "Пример:"

    Выполняется нумерация с данными ПЛК для клемм какого-либо клеммника. При этом нумеруются пять из пяти клемм, так как они непосредственно соединены с выводомустройства ПЛК. После этого пронумеруйте все пять клемм с помощью обычнойопциидля нумерации клемм.Если при этом также выполняется нумерация клемм, уже пронумерованных с данными ПЛК, все клеммы нумеруются заново, аобозначения клемм, введенные при нумерации с данными ПЛК, удаляются.Если же требуетсясохранитьобозначенияклемм, введенные при нумерации с данными ПЛК, из стандартной нумерации клемм необходимо исключить те функции, которые были изменены во время нумерации с данными ПЛК.

Если у функции достигнут конец нумерации (в соответствии с настроенным объемом), а вам требуется пронумеровать в том числе и функции, подсоединенные к этой функции, укажите это с помощью свойства Продолжить нумерацию с данными ПЛК далее после функции. Эта настройка относится к каждой группе и каждому объему. Теперь после этой функции действителен тот же объем, что и перед ней.

Кроме того, для выводов устройства ПЛК имеется свойство Предотвратить нумерацию с помощью данных ПЛК. Если для вывода устройства ПЛК установлено это свойство, то функции, которые соединены с этим выводом устройства ПЛК, не нумеруются. Эта настройка относится к каждой группе и каждому объему.

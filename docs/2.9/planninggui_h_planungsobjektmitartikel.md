# Создание объектов планирования в качестве устройств

1. Выберите следующие пункты меню: Данные проекта > Предварительное планирование > Навигатор
2. Выделите проект, сегмент структуры или уже имеющийся объект планирования и выберите пункт всплывающего меню Новое устройство (объект планирования).
3. В диалоговом окне Выбор изделия выберите изделие.
4. Выберите в диалоговом окне Выбрать определение сегмента нужное определение сегмента. Это диалоговое окно откроется, если выделен сегмент структуры и в проекте содержится несколько определений сегментов для объектов планирования.
5. Введите в диалоговом окне Свойства ++...++ нужные данные для нового объекта планирования.

!!! info "Для сведения:"

    Во вкладке Изделие отображается номер изделия, а во вкладке Шаблоны — шаблоны функций изделия.

6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Создается объект планирования с соответствующими свойствами.

### Создание объектов планирования путем перетаскивания изделий мышью

Условия:

* Вы открыли навигатор основных данных изделий или список устройств.
* Открыт навигатор предварительного планирования.

1. Выделите изделие в навигаторе основных данных изделий или в списке устройств.
2. Перетащите изделие с помощью мышки в навигатор предварительного планирования и отпустите его на сегменте структуры или объекте планирования.
3. Выберите в диалоговом окне Выбрать определение сегмента нужное определение сегмента. Это диалоговое окно откроется, если выделен сегмент структуры и в проекте содержится несколько определений сегментов для объектов планирования.

!!! info "Для сведения:"

    Открывается диалоговое окно Свойства ++...++ и номер изделия переносится на вкладку Изделие.

!!! info "Для сведения:"

    Шаблоны функции изделия отображаются на вкладке Шаблоны.

4. Введите в диалоговом окне Свойства ++...++ нужные данные для нового объекта планирования.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Ниже выделенного сегмента структуры или объекта планирования будет создан объект планирования с соответствующими свойствами.

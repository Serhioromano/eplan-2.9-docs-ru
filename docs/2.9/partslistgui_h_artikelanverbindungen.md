# Учитывать изделия у соединений

Изделия, которые указываются в точках определения соединения и оттуда записываются на соединения при создании таблицы соединений, могут также учитываться в диалоговом окне Спецификация — <Имя проекта>.

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Данные проекта > Устройство / изделие > Навигатор спецификаций
2. В диалоговом окне Спецификация — <Имя проекта> выберите пункт всплывающего меню Настройки.
3. В диалоговом окне Настройки: Изделие установите флажок Изделие соединения в групповом поле Учитывать изделие.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Изделия соединения выводятся в спецификации.

!!! note "Замечание:"

    * Если изделие применяется в точке определения соединения, через которую проходит больше одного соединения, это изделие записывается на каждое из этих соединений. Число изделий получают здесь только после анализа соединения.
    * Если, наоборот, соединение проходит через более, чем одну точку определения соединения и каждая из этих точек определения соединения имеет одинаковое определение изделия, то изделие появляется затем только один раз.
    * Поэтому для изделий на соединениях необходимо создавать список соединений (Сервисные программы > Отчеты > Генерировать > Таблица соединений), чтобы определить фактическое число штук.

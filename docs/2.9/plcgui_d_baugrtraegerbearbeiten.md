# Диалоговое окно Обработать каркас

Вы открыли проект. Данные проекта > ПЛК > Навигатор. В виде, ориентированном по каркасу, выделен каркас. Выберите пункт всплывающего меню Обработать.

В данном диалоговом окне задайте последовательность карт ПЛК в каркасе. Обычно карты ПЛК сортируются соответственно значению свойства На позиции / разъем. Как только последовательность карт ПЛК в этом диалоговом окне изменяется, карты ПЛК получают идентификатор сортировки (свойство Идентификатор сортировки (позиция в каркасе)) и сортируются соответственно этому идентификатору. Это целесообразно в первую очередь тогда, когда каркас содержит карты ПЛК, для которых введены одинаковые значения в свойстве На позиции / разъем или для которых значения не указаны.

Обзор основных элементов диалогового окна:

В таблице имеются следующие столбцы:

* Строка: Непрерывный номер строки. Для каждой карты ПЛК каркаса отображается одна строка.
* На позиции / разъем: указывает, в каком разъеме / на какой позиции вставлена эта карта ПЛК на каркасе. Здесь можно изменять значение.
* ОУ (идентифицирующее): Отображает идентифицирующее ОУ карты ПЛК.
* Обозначение типа ПЛК: Отображает номер или обозначение типа рабочей станции ПЛК.
* Имя карты ПЛК: Отображает имя карты ПЛК. Здесь можно изменять значение.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Нумеровать "На позиции / разъем" |  Предоставляет возможность автоматически присваивать значения свойства На позиции / разъем выделенным картам ПЛК или всему каркасу. В последующем диалоговом окне вводится начальное значение и величина шага для нумерации.

!!! tip "Совет:"

    Отсортируйте карты ПЛК в нужной последовательности с помощью кнопок со стрелками, а затем используйте пункт всплывающего меню Нумеровать "На позиции / разъем", чтобы заново пронумеровать значения свойства На позиции / разъем.

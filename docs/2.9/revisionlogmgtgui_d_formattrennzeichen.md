# Диалоговое окно Формат: Разделитель (нумерация индекса ревизии)

Параметры > Настройки > Фирма > Управление > Отслеживание изменений (нумерация индекса ревизии). В диалоговом окне Настройки: Отслеживание изменений (нумерация индекса ревизии) выберите в списке Доступные элементы формата элемент формата типа "Разделитель" и нажмите на ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо) (или дважды щелкните в списке Выбранные элементы формата на уже определенном разделителе).

В этом диалоговом окне указывается форматирование разделителя для группы форматов.

Обзор основных элементов диалогового окна:

### В качестве разделителя можно использовать следующие символы

* Пробел
* Другие символы: После выбора этой опции введите в поле ввода требуемую комбинацию символов. Разрешается использовать любые цифры, буквы и специальные символы за исключением символов #, < и >. Однако их можно ввести, использовав перед ними косую черту (\\).

### Число символов

Если в качестве разделителя выбран пробел, здесь можно указать число символов. В случае выбора опции Другие символы это поле становится недоступным и в нем выводится число введенных символов.

# Особенности при переносе ОУ

Главным образом, функция может переносить ОУ от следующих других функций:

Функция |  Общая функция |  Клемма |  Контакт штекера |  Черный ящик |  Вывод устройства |  Определение кабеля / Экранирование
---|---|---|---|---|---|---
Общая функция |  X |  X |  X |  X |  X |
Клемма |  X |  X |  X |  X |  X |
Контакт штекера |  X |  X |  X |  X |  X |
Черный ящик |  X |  X |  X |  X |  X |
Вывод устройства |  X |  X |  X |  X |  X |
Определение кабеля / Экранирование |  |  |  |  |  |

Как при переносе ОУ, так и при передаче обозначений выводов устройства ***не*** учитываются следующие функции:

* Обозначения местоположения
* Определения клеммников
* Определения штекеров
* Кабельные соединения
* Точки обозначения соединения
* Точки разрыва
* Символы соединения (углы, тройники и т.д.).

#### Черные ящики

Функции в черном ящике копируют ОУ из блока, но не из функций вне блока. Функции за пределами блоков могут копировать ОУ функций, находящихся в пределах блоков.

Клеммы и контакты штекеров могут также копировать ОУ черного ящика и символов в черном ящике.

Если в рамках черного ящика размещаются символы без собственного ОУ, то они копируют ОУ блока, даже если на символах не производилось записей. Т.е., у функционального элемента без ОУ даже в черном ящике будет ОУ функционального элемента, находящегося в направлении поиска. Если в блоке нет функционального элемента на той же высоте, то он наследует ОУ из блока. Если два черных ящика с символами находятся на одинаковой высоте, то символы во втором черном ящике не связываются с символами, размещенными в первом черном ящике.

#### Обозначения местоположения

Функции в обозначении местоположения копируют ОУ из блока, но не из функций вне блока. Функции за пределами блоков могут копировать ОУ функций, находящихся в пределах блоков.

#### Выводы устройства ПЛК

Направление поиска в случае с выводами устройства ПЛК служит не только для переноса ОУ, но и для переноса обозначения контакта штекера с другого вывода устройства ПЛК.

В случае со страницами перечня поиск по умолчанию всегда производится снизу вверх независимо от выравнивания рамки. Посредством переключения направления поиска непосредственно на выводе устройства ПЛК можно инициировать поиск налево.

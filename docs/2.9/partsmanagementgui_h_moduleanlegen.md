# Создать модули

Модули на платформе EPLAN предоставляют одну из опций, которая позволяет отображать сложные устройства. Модули обладают теми же свойствами, что и изделия, однако дополнительно содержат списки подизделий (т. н. позиций модулей). Эти подизделия идентифицируются с помощью нижестоящего ОУ или "идентификатора ОУ".

Условия:

* Вы открыли базу данных изделий: Сервисные программы > Изделие > Управление
* Создан подходящий макрос для модуля.
* В дереве выделен уровень иерархии ***Модули*** или в списке выделено изделие модулей.
* Или в дереве выделено свойство Главн. группа продукт. "Электротехника", "Fluid-техника" или "Механика" (при условии, что в дереве ***отсутствует*** уровень иерархии ***модулей***).

1. Выберите пункт всплывающего меню Создать.

!!! info "Для сведения:"

    EPLAN автоматически создаст новый модуль с номером изделия Создать_<порядковый_номер>.

2. На вкладках в правой части диалогового окна следует ввести данные модуля.
3. Выберите вкладку  Технические данные. Для модулей из главной группы продуктов "Механика" выберите вкладку Монтажные данные.
4. Щелкните ++...++ в поле Макрос / Графический макрос и выберите созданный ранее макрос для модуля. При необходимости можно включить предварительный просмотр макроса, установив флажок в одноименном поле. (Отображение предварительного просмотра выполняется как в диалоговом окне выбора, так и в окне Предварительный просмотр графики, если оно активировано.)
5. Подтвердите ввод.

Условия:

* Вы открыли базу данных изделий: Сервисные программы > Изделие > Управление
* Выбран какой-либо модуль и выделена вкладка Модуль.

1. Щелкните по ![](images/all_new_as.png){: .ui-icon } (Создать), чтобы создать строку в списке позиций модуля.
2. Щелкните в поле Номер изделия по ++...++.
3. Выберите требуемое изделие из структуры дерева или списка в выборе изделий.
Многократный выбор здесь невозможен, т. е. всегда копируется самое верхнее изделие из выделенных.
4. Щелкните по кнопке ++OK++.
5. В ячейку ОУ введите нижестоящее обозначение устройства, которое относится к модулю.

!!! note "Замечание:"

    * Устройство должно быть частью макроса, присвоенного модулю. Указанное в строке изделие присваивается этому устройству при вставке модуля.
    * Для идентификации устройства, находящегося в присвоенном макросе, используется либо нижестоящее ОУ устройства, либо указанный на устройстве идентификатор ОУ. Соответственно, на строку (т. е. для каждой позиции модуля) возможен ввод либо в столбец ОУ, либо в столбец Идентификатор ОУ.

6. Щелкните по ++Применить++, чтобы записать введенные данные и изменения в базу данных, не выходя из базы данных изделий.
7. Выберите пункт меню ++Дополнительно++ > Суммировать шаблоны функций.

!!! info "Для сведения:"

    Шаблоны функции, находящиеся на подизделиях модулей, суммируются и указываются для всех модулей во вкладке Шаблоны функций.

!!! note "Замечание:"

    Предварительной настройкой для новой строки является текущая выбранная настройка, т. е. ее данные по количеству, номеру изделия и ОУ будут скопированы. Если ни одна запись не выделена, будет создана пустая строка (с предварительной настройкой Количество = "1").

Условие:

В Графическом редакторе открыта страница проекта.

1. Выберите следующие пункты меню: Вставить > Устройство
2. Выберите в диалоговом окне База данных изделий созданный ранее макрос для модуля и нажмите ++OK++.
3. Щелчком мыши разместите модуль на странице схемы соединений.

!!! info "Для сведения:"

    Изделие модуля заносится в черный ящик, и устройства модуля содержат перечисленные в модуле подизделия.

Модули из главной группы продуктов "Механика" также вставляются через пункты меню Вставить > Устройство в открытом пространстве листа. При размещении на трехмерном чертеже монтажных поверхностей отображаются точки захвата, чтобы модуль можно было присоединить многократно.

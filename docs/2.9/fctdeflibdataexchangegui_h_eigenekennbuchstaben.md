# Создать собственные наборы буквенных обозначений

В EPLAN вы можете создавать собственные наборы буквенных обозначений и, тем самым, учитывать разные стандарты, страны и клиентов.

1. Сервисные программы > Основные данные > Буквенное обозначение
2. Выберите пункт всплывающего меню Создать.
3. В диалоговом окне Новый набор буквенных обозначений введите обозначение для набора буквенных обозначений и щелкните по ++OK++.

!!! info "Для сведения:"

    Выводится новый столбец с соответствующим заголовком.

4. Введите в отдельных ячейках буквенные обозначения для отдельных функций. Буквенные обозначения могут также состоять из нескольких символов.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выполняется сохранение введенных буквенных обозначений.

!!! warning "Предупреждение:"

    После каждого внесенного изменения необходимо экспортировать новый набор буквенных обозначений. Если при обновлении вы получаете новую библиотеку определений функций, необходимо импортировать экспортированные вами наборы буквенных обозначений в новую версию программы и дополнить новые определения функций.

### Выбрать наборы буквенных обозначений

!!! warning "Предупреждение:"

    Перед выбором набора буквенных значений обязательно нужно провести синхронизацию проекта (Сервисные программы > Основные данные), чтобы созданный набор буквенных значений попал в проект и предлагался для выбора в списках.

1. Параметры > Настройки > Проекты > "имя проекта" > Устройство > Нумерация (online)
2. Выберите в поле Текущий набор буквенных обозначений желаемый набор буквенных обозначений из раскрывающегося списка.
3. В поле Подклассы буквенных обозначений выберите необходимый подкласс из раскрывающегося списка.
4. Либо щелкните по кнопке ++...++ в поле Подклассы буквенных обозначений и в диалоговом окне Подклассы буквенных обозначений создайте новый класс.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    При создании функции используется соответствующее буквенное обозначение/соответствующий подкласс.

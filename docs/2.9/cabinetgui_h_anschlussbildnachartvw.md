# Перенос локальной схемы соединений в базу данных изделий

Локальные данные выводов устройств для размещения изделия можно использовать в базе данных изделий, чтобы определять схемы соединений в качестве значений по умолчанию для других размещений изделий. Для этого локальную схему соединений, которая, например, была расширена за счет графически заданных выводов устройств, можно перенести непосредственно в базу данных изделий.

1. Выделите в пространстве листа функциональный элемент со схемой соединений.
2. Выберите пункт всплывающего меню Генерировать схему соединений.

!!! info "Для сведения:"

    В базе данных изделий внутри уровня иерархии "Схема соединений" генерируется новая схема соединений с автоматически присвоенным именем Новый_<непрерывный номер> (например, Новый_1).

!!! info "Для сведения:"

    Открывается диалоговое окно База данных изделий.

3. В открывшейся базе данных изделий измените автоматически сгенерированное имя схемы соединений.
4. При необходимости вы можете присвоить изделию схему соединения с новым именем.

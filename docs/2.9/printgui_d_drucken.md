# Диалоговое окно Печать

Проект > Печать .

В этом диалоговом окне выберите принтер и задайте страницы, которые должны распечатываться.

Доступны следующие элементы диалогового окна:

### Принтер

В раскрывающемся списке Имя отображаются все принтеры, установленные на вашем компьютере. Из этого списка выберите нужный принтер. В полях Статус, Тип, Местопол. и Комментарий отображается информация по настройкам печати Windows, установленным для этого принтера.

### [Свойства]

При нажатии этой кнопки открывается диалоговое окно Windows для настроек печати. В этом окне вы можете выбрать настройки печати. Дополнительные сведения вы можете получить из справки для соответствующего принтера.

### Изменить вывод для печати в файл

Этот флажок устанавливается, если вы хотите распечатать выбранные страницы в файл. EPLAN создает файл *.PRN, который можно распечатать позже или использовать для создания файла PDF.

### Диапазон страниц

В этом групповом поле установите, какие страницы вы хотите распечатать. Если диалоговое окно Печать вызвано из графического редактора, вы можете распечатать либо находящуюся в настоящее время на переднем плане страницу, либо все страницы проекта. При вызове из навигатора страниц вы можете распечатать выделенные страницы или все страницы проекта.

* Текущая страница: Выполняется печать страницы, находящейся на переднем плане в графическом редакторе.
* Выделение: Выполняется печать страниц, выделенных в навигаторе страниц. Если в навигаторе страниц выделен проект и одновременно включен фильтр страниц, выполняется печать только отфильтрованных страниц проекта.
* Весь проект: Выполняется печать всех страниц. При этом включенный фильтр страниц не учитывется.

### Экземпляры

В этом групповом поле установите количество распечаток и способ их сортировки.

* Число: Здесь вы устанавливаете, сколько экземпляров выбранных страниц должно быть распечатано.
* Сортировать: Этот флажок устанавливается только тогда, когда необходимо распечатать несколько экземпляров. Если он установлен, наборы выбранных страниц печатаются друг за другом. Если флажок не установлен, друг за другом печатаются все одноименные страницы. Небольшое изображение в диалоговом окне показывает, как нужно действовать.
* Обратный порядок печати: Если этот флажок установлен, страницы печатаются в обратном порядке. Если флажок сброшен, страницы печатаются в той последовательности, в которой они следуют в проекте. Небольшое изображение в диалоговом окне показывает, как нужно действовать.

### Только измененные страницы

Установите этот флажок, чтобы распечатать все страницы, которые были отмечены как "измененные" со времени последней печати, т. е. все страницы, в которых свойство "Дата изменения (автоматически)" имеет более позднюю дату, чем дата последней печати.

### [Настройки]

Откройте с помощью этой кнопки диалоговое окно [Настройки: Печать](printgui_d_druckoptionen.md). Там вы можете установить масштабирование, границы печати и положение на странице, а также настройку для цветной или черно-белой печати.

### [Предв. просмотр]

При нажатии этой кнопки открывается диалоговое окно [Предв. просмотр](printgui_d_druckvorschau.md).

# Медная шина: Добавить новое место изгиба

Размещенной медной шине можно добавить дополнительные места изгиба, которые не определены в профиле. Таким образом можно увеличить число мест изгиба у уже размещенных и измененных функциональных элементов — шин, и при этом не придется чертить новый профиль и размещать новый функциональный элемент — шину.

Новое место изгиба можно вставить на любом прямом сегменте сборной шины.

* Чтобы вставить плоский изгиб, необходимо выполнить все настройки в изометрической 3D-точке наблюдения на верхней или нижней поверхностях сегмента шины.
* Чтобы вставить изгиб на ребро, необходимо выполнить все настройки на боковых поверхностях сегмента шины.
* На одну сборную шину можно вставить одновременно плоский изгиб и изгиб на ребро, в этом случае функциональный элемент будет определен в пространстве изогнутым вокруг разных осей.

Условия:

* Вы открыли проект. Открыто одно пространство листа.
* Пространство листа содержит размещенные сборные шины (изогнутые).

1. Обработать > Графика > Новое место изгиба.

!!! info "Для сведения:"

    Рядом с курсором отображается параллелепипед. Этот параллелепипед можно установить на любой монтажной поверхности прямых сегментов медной шины и на точках захвата. Как только параллелепипед коснется монтажной поверхности, на ней отобразится голубая линия. При размещении можно использовать опции размещения.

2. Щелкните по тому концу медного функц. элемента, который должен остаться без изменений.
3. Щелкните по тому месту, куда вы хотите вставить новое место изгиба.
4. При помощи клавиатуры введите угол изгиба.
5. Введите радиус изгиба с клавиатуры и нажмите кнопку ++Ввод++.

!!! info "Для сведения:"

    Будет вставлено новое место изгиба.

!!! info "Для сведения:"

    Данная функция остается активной, вы можете добавить еще несколько мест изгиба на тот же функциональный элемент.

# Диалоговое окно Настройки: Выбор устройства

Вы открыли проект.

* Параметры > Настройки > Проекты > "Имя проекта" > Управление > Выбор устройства.
* Вы выделили главную функцию в графическом редакторе или навигаторе. Обработать > Свойства. Выберите вкладку Изделие. [Настройки] > Выбор устройства (или ++Выбор устройства++ > [Настройки]).

В этом диалоговом окне устанавливаются настройки для выбора устройства.

Выбор устройства обычно предлагает только те изделия, которые имеют "подходящий" или "увеличенный" набор необходимых шаблонов функции. Проверку подходящих функций можно отключить, выключив все критерии выбора. В этом поле отображены все доступные изделия.

Обзор основных элементов диалогового окна:

### Список выбора

Выберите имеющуюся схему (т. е. список изделий) из раскрывающегося списка. Затем при выборе устройства предлагаются только изделия, содержащиеся в выбранном списке.

Через [...] откройте диалоговое окно Предварительный выбор изделий и там можно создать новый список изделий и сохранить как схему или изменить имеющийся список.

### Автоматический выбор устройства

Если этот флажок установлен, при выборе устройства данные внесенного в функцию изделия сравниваются с критериями, настроенными для выбора устройств. Диалоговое окно Выбор устройства открывается только тогда, когда необходим повторный выбор устройства (если была обнаружена ошибка, например переполнение) или в функцию не внесено изделие. Если изделие соответствует настроенным для выбора устройства критериям, диалоговое окно Выбор изделия не открывается. При выборе устройства для нескольких устройств пропускаются те устройства, которым уже присвоено подходящее изделие.

Если подходящее изделие уже перенесено на функцию, но технические данные изделия отсутствуют (например, так как номер изделия внесен вручную или через внешнюю обработку), то данные изделия автоматически дополняются при выборе устройства. При этом переносятся данные сохраненных в изделии шаблонов функций.

Если флажок снят, то диалоговое окно Выбор устройства открывается для всех выбранных устройств, независимо от того, присвоено ли подходящее изделие. Автоматическое дополнение технических данных существующего изделия в данном случае невозможно.

### Учитывать только осн. функцию

Установите этот флажок для предложения всех изделия, которые подходят выбранной главной функции, независимо от того, какие имеются вспомогательные функции. Эта настройка может быть целесообразной, напр., для устройств модульной конструкции.

При снятии флажка предлагаются только те устройства, которые обладают всеми функциями: как главной, так и вспомогательными. Т. е. при выборе устройства доступны только те изделия, которые (с принадлежностями или без них) способны компенсировать все функции устройства в виде шаблонов функций. Таким образом, не предлагается изделие, с помощью которого невозможно будет выполнить функции устройства. При этом не имеет значения требуемое количество принадлежностей.

### Свернуть свободные функции

Установите этот флажок для предложения только устройства, которые в крайнем случае имеют указанное число на свободных (т. е., не использованных на устройстве) функциях. Число устанавливается в поле Число свободных функций.

### Отображение, даже если подходит только одно изделие

Устанавливайте этот флажок для вывода диалогового окна Выбор устройства в каждом случае.
При снятии флажка сразу же копируется изделие, если к устройству подходит только номер изделия (т. е. тип устройства).

### Перезаписать старые изделия с помощью текущего выбора

Если этот флажок установлен, то выбор главного изделия осуществляется в диалоговом окне Выбор устройства одним щелчком. Изделие, содержащееся в списке Выбранные изделия: Функции / шаблоны, удаляется и заменяется на выделенное главное изделие. Это также действительно для выбора принадлежности. Как только для принадлежности введен номер изделия, принадлежность можно выбирать только щелчком. После применения выбора изделия на условном обозначении заменяются.
При снятии флажка к уже выбранным изделиям добавляются новые.

### Поменять символ при изменяющейся функции

Если этот флажок установлен, то при выборе устройства меняется символ, когда у изделия другое определение функции. При этом используется тот же символ, что и при размещении соответствующей функции. Если в шаблон функции изделия занесен один символ, то используется этот символ.

Критерии выбора

### Использовать имеющиеся данные функции

Установите флажок для применения имеющихся данных функций в качестве критериев выбора. В следующих вкладках можно более подробно определить критерии для отдельных групп продуктов, установив там соответствующие флажки.
При снятии флажка при выборе игнорируются имеющиеся данные функций. Флажки в последующих вкладках затем изображаются как неактивные.

Установите на отдельных вкладках необходимые флажки для группы продуктов, чтобы соответствующие свойства функций учитывались в качестве критериев выбора. После этого будут предлагаться только устройства, которые соответствуют значениям этих свойств. После выбора устройства отдельным функциям присваиваются соответствующие шаблоны функций изделий в соответствии с определенными здесь критериями выбора. При этом в функциях остаются только значения ***включенных*** в этом окне свойств. Значения ***отключенных*** свойств перезаписываются значениями из шаблонов функций.

!!! note "Замечание:"

    * Если здесь во вкладке Кабель / Группа соединений установить флажок Тип потенциала, то при выборе устройства будут учитываться только потенциалы типа "N", "PE" / "PEN" и "SH".
    * Если здесь на вкладке Технология производственных процессов установить флажок Класс трубы, а для устройства на функциональной схеме автоматизации выбрать класс трубы, для устройств в выборе устройств будут учитываться только изделия, указанные в классе трубы. Это же условие действует в выборе устройства для соединений, если на вкладке Соединение установлен флажок Класс трубы.

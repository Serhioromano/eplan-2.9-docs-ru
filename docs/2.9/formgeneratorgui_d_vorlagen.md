# Вклада Шаблоны

Вы открыли проект. Сервисные программы > Отчеты > Генерировать. Выберите вкладку Шаблоны.

В этой вкладке вы создаете, обрабатываете шаблонами отчетов для отчетов и управляете ими. Отчеты могут быть сгенерированы на базе такого шаблона отчетов.

Обзор основных элементов диалогового окна:

### <Имя проекта>

В этой структуре дерева в качестве самого высокого слоя структуры отображаются имена текущих проектов. Если исходный проект внесен в шаблон, то шаблон будет представлен под именем этого проекта (пиктограмма: ![](images/formgeneratorgui_sourceproject_as.png){: .ui-icon }).
Под именами проектов отображаются все имеющиеся в проекте [Шаблоны отчетов](formgeneratorgui_k_auswertungsvorlagen.md). Все шаблоны отчетов рассортированы по типу отчетов. Можно самостоятельно создать шаблоны отчетов.

### Всплывающее меню

Пункт меню |  Значение
---|---
Создать |  Открывает диалоговое окно Определить отчет. Обеспечивает создание шаблона отчетов и определение настроек для него.
Генерация отчета |  Создает отчеты на основе выделенных шаблонов отчетов. Даже если выделен только один шаблон отчета, анализируются все шаблоны отчетов данного типа отчета, иначе они будут взаимно перезаписаны.
Удалить |  Удаляет один или несколько шаблонов отчетов.
Экспортировать |  Открывает диалоговое окно Экспортировать шаблоны отчетов. Позволяет экспортировать шаблоны отчетов.
Импортировать |  Открывает диалоговое окно Импортировать шаблоны отчетов. Позволяет импортировать шаблоны отчетов.
Исходный проект > Выбрать исходный проект |  Открывает диалоговое окно Исходный проект. Позволяет задать исходный проект для выбранных шаблонов отчетов.
Исходный проект > Удалить записанный исходный проект |  Позволяет убрать запись 'Исходный проект' для всех выбранных шаблонов отчетов.
Генерировать проект отчета |  Открывает диалоговое окно Генерировать проект отчета. Позволяет сгенерировать проект отчета для всех выбранных шаблонов отчетов.

!!! tip "Совет:"

    Некоторые функции всплывающего меню доступны также в виде кнопок на панели инструментов, которая находится над структурой дерева в этой вкладке.

### Свойство / Значение

Если вы выделяете запись в дереве, то в этой таблице выводятся на просмотр и обрабатываются установленные свойства шаблона. Некоторые свойства доступны как на вкладке Шаблоны, так и на вкладке Отчеты.

Свойство |  Значение
---|---
Имя |  Доступно только для шаблонов отчетов. Показывает имя шаблона отчетов. При создании имя присваивается автоматически. Позволяет ввести другое имя для шаблона отчетов.
Начальная страница блока отчетов |  Показывает начальную страницу блоков отчетов, заданную в диалоговом окне <Тип отчета> для блока отчетов, к которому относится выделенная страница отчетов / встроенный отчет.
Доп. поле номера листа |  Выводит запись, которую вы задали в диалоговом окне <Тип отчета> в поле Доп. поле номера листа. Обеспечивает ввод новой записи или изменение существующей записи.
Исходный проект |  Отображает имя исходного проекта, которое вы указали в диалоговом окне Определить отчет в поле Исходный проект. Позволяет нажать поле и с помощью кнопки ++...++ выбрать (другой) проект в диалоговом окне выбора проекта.
Настройка фильтра |  Показывает в [перечнях отчетов](formgeneratorgui_k_auswertungsuebersicht.md) схему фильтра, которая была активирована при создании отчета в диалоговом окне Настройки. Отображает в [отчетах, относящихся к функции,](formgeneratorgui_k_funktionsbezogen.md) схему фильтра, которая была активирована при создании отчета в диалоговом окне Настройки для устройства. Позволяет щелкнуть на поле и с помощью кнопки ++...++ определить схему фильтра.
Настройка сортировки |  Показывает в [перечнях отчетов](formgeneratorgui_k_auswertungsuebersicht.md) схему сортировки, которая была активирована при создании отчета в диалоговом окне Настройки. Отображает в [отчетах, относящихся к функции,](formgeneratorgui_k_funktionsbezogen.md) схему сортировки, которая была активирована при создании отчета в диалоговом окне Настройки для устройства. Позволяет щелкнуть на поле и с помощью кнопки ++...++ определить схему сортировки.
Новая страница при изменении свойства |  Показывает в [перечнях отчетов](formgeneratorgui_k_auswertungsuebersicht.md) схему разбивки на страницы, которая была активирована при создании отчета в диалоговом окне Настройки. Отображает в [отчетах, относящихся к функции,](formgeneratorgui_k_funktionsbezogen.md) схему разбивки на страницы, которая была активирована при создании отчета в диалоговом окне Настройки для устройства. Позволяет щелкнуть на поле и с помощью кнопки ++...++ определить схему разбивки на страницы.
Функции: Настройка фильтра |  Отображает в [отчетах, относящихся к функции,](formgeneratorgui_k_funktionsbezogen.md) схему фильтра, которая была активирована при создании отчета в диалоговом окне Настройки для функций. Позволяет щелкнуть на поле и с помощью кнопки ++...++ определить схему фильтра.
Функции: Настройка сортировки |  Отображает в [отчетах, относящихся к функции,](formgeneratorgui_k_funktionsbezogen.md) схему сортировки, которая была активирована при создании отчета в диалоговом окне Настройки для функций. Позволяет щелкнуть на поле и с помощью кнопки ++...++ определить схему сортировки.
Функции: Новая страница при изменении свойства |  Отображает в [отчетах, относящихся к функции,](formgeneratorgui_k_funktionsbezogen.md) схему разбивки на страницы, которая была активирована при создании отчета в диалоговом окне Настройки для функций. Позволяет щелкнуть на поле и с помощью кнопки ++...++ определить схему разбивки на страницы.
Автор |  Доступно только для отчетов. Отображает имя пользователя, сгенерировавшего отчет.
Дата создания |  Доступно только для отчетов. Выводит время создания отчета.
Описание страниц вручную |  Если флажок Автоматическое описание страницы (диалоговое окно <Тип отчета>) не установлен, то текст для описания страницы отчета можно ввести вручную или можно адаптировать уже существующий текст.
Автоматическое описание страницы |  Если флажок установлен, то текст для описания страниц отчета автоматически генерируется из свойства формы Формат для автоматического описания страниц.
Устройства |  Доступно только для отчетов. Показывает при [отчетах, относящихся к функции](formgeneratorgui_k_funktionsbezogen.md) проанализированное устройство или, в непосредственной связи с функцией Объединить, все проанализорованные устройства блока отчетов. Показывает в [перечнях отчетов](formgeneratorgui_k_auswertungsuebersicht.md) устройств первое и последнее проанализорованное устройство блока отчетов.
Сортировка страниц |  Для отчетов: Показывает сортировку страниц, установленную в меню ++Настройки++ > Вывод в страницах для данного типа отчета. Для шаблонов отчетов: Если в этом поле выбрана запись "Из настроек", используется сортировка страниц, присвоенная типу отчета в меню ++Настройки++ > Вывод в страницах. В этом поле независимо от настроек можно задать [сортировку страниц отчетов](formgeneratorgui_k_seitensortierung.md). Нажмите кнопку ++...++ и выберите в следующем диалоговом окне требуемую сортировку страниц. Будет применена выбранная сортировка страниц. Если вы удаляете запись о клавише ++Del++, то будет снова вставлена запись "Из настроек".
Форма |  Для отчетов: показывает использованную форму. Для шаблонов отчетов: Если запись в этом поле пуста, то выделенному шаблону отчета присваивается форма, выбранная в меню ++Настройки++ > Вывод в страницах для данного типа отчета. В этом поле вы можете независимо от настроек присвоить другую форму выделенному шаблону. При составлении отчета формы, занесенные в соответствующий блок отчетов, перезаписываются.
Обновление с этой формой |  Доступно только для отчетов. При установленном флажке в случае обновления будет применена заданная форма из поля, расположенного ниже. Таким образом, больше не используется форма, присвоенная типу отчета в [Настройки] > Вывод в страницах.
Доп. поле |  Доступно только для отчетов. Это поле содержит запись только для спецификации штекера. В качестве дополнительной информации выводится определение функции проанализированных устройств, как, например, "Штыри", "Гнезда" или "Штыри и гнезда".
Частичный вывод |  Доступно только для шаблонов отчетов. Если запись в этом поле пуста, то частичному выводу выделенного шаблона отчета присваивается форма, выбранная в разделе [Настройки] > Вывод в страницах для типа отчета в столбце Частичный вывод. В этом поле вы можете независимо от настроек присвоить другую форму частичному выводу выделенного шаблона.
Следующая форма |  Доступно только для шаблонов отчетов. С помощью раскрывающегося списка в этом поле для конкретных шаблонов отчетов можно выбрать, какая форма должна использоваться в качестве следующей формы.
Развернуть узлы ... Изделие предварительного планирования |  Эти свойства имеют значение для отчетов "Спецификация изделий", "Групповая спецификация изделий", а также частично — для отчета "Список обозначений устройств". Для этих свойств используются настройки по умолчанию, заданные в меню ++Настройки++ > Изделие. Задать другие настройки для шаблонов отчетов можно во вкладке Отчеты. Эти настройки в дальнейшем будут отображаться во вкладке Отчеты.
Фильтр проекта |  Доступно только для шаблонов отчетов. Позволяет сгенерировать [условные отчеты](formgeneratorgui_k_auswertungsvorlagen.md). Если в данном поле указан фильтр проекта, то из выделенного шаблона отчета генерируется только один отчет, когда текущий проект удовлетворяет условиям фильтра.
Шаблон активен |  Доступно только для шаблонов отчетов. Если флажок установлен, то согласно инструкциям "Генерация отчета" и "Генерация отчета по проект" из выделенного шаблона отчета будет сгенерирован отчет. Если флажок не установлен, то отчет не генерируется. Данный флажок устанавливается EPLAN автоматически. Это происходит в том случае, когда для шаблона отчета не назначено ни одного фильтра проекта или если текущий проект удовлетворяет условиям выделенного фильтра проекта.
Действие |  Только для API-действий: если определенное пользователем расширение сконфигурировано посредством операции, оно отображается здесь.

### [Настройки]

Пункт меню |  Значение
---|---
Вывод в страницах |  Открывает диалоговое окно Настройки: Вывод в страницах. Позволяет установить настройки для вывода отчетов в страницах.
Изделие |  Открывает диалоговое окно Настройки: Изделие. Обеспечивает определение настроек для выполнения отчета по изделиям.
Отображение / вывод |  Открывает диалоговое окно Настройки: Отображение / вывод. Обеспечивает ограничение отображения и вывода данных проекта в страницах отчета.

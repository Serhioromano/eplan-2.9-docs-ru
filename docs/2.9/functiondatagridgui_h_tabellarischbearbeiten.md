# Обработать объекты в табличном виде

Условия:

* Вы открыли проект.
* Вы открыли навигатор (навигатор страниц, навигатор устройств и т. д.) в графическом редакторе.

1. Выделите требуемые объекты на странице проекта или в навигаторе и выберите пункты меню Обработать > В табличном виде.

!!! info "Для сведения:"

    В диалоговом окне Табличная обработка — <Имя проекта> данные выбранных объектов отображаются в форме таблицы. Если выбор содержит несколько типов объекта, для каждого типа объекта отображаются отдельные вкладки, например Функции, Соединения и т. д.

2. При необходимости выберите для функций и соединений другую схему, чтобы изменить отображение столбцов.
3. Измените требуемые свойства в соответствующих полях.
4. Закройте диалоговое окно с помощью кнопки ![](images/all_shut_as.png){: .ui-icon } или щелкните мышью в другом (открытом) диалоговом окне.

!!! info "Для сведения:"

    Измененные данные будут приняты.

!!! note "Замечание:"

    * При смене выделения в открытом диалоговом окне, необходимо обновить отображение в диалоговом окне с помощью пункта всплывающего меню Табличная обработка.
    * Если для ***соединений*** изменить значение свойства соединения, это измененное значение переносится как на соединение, так и на все связанные точки определения соединения, на которых заполнено свойство. Пустые свойства не будут перезаписаны.
    * Если у ***соединения*** нет точек определения соединения, при изменении значения свойства точка определения соединения размещается автоматически, и значение свойства записывается на нее.

!!! example "Пример:"

    Вам нужно изменить технические характеристики всех катушек контакторов на страницесхемы соединений"=AP+ST1/3". Для этого вы выделяете катушку контактора и выбираете пункт всплывающего меню Выделить объекты одного типа. Все катушки контакторов на этой странице выделяются. Далее выберите пункт всплывающего меню Табличная обработка. Данные катушек контакторов отображаются в диалоговом окне Табличная обработка — <Имя проекта>.Для первой катушки контактора в поле Технические параметрывведите значение "24-240V". Затем скопируйте значение через всплывающее меню, выделите ячейки в столбце Технические параметрыдля остальных катушек и вставьте значение через всплывающее меню. После внесения изменений щелкните один раз мышкой на графическом редакторе. Возле катушек сразу же появятся измененные технические характеристики.

# Вкладка Функциональные элементы (автоматическое проставление размеров)

* Вы открыли проект.В Графическом редакторе открыта страница проекта. Вставлен обзор модели (Вставить > Графика > Обзор модели ( ++...++)) и размещен. В диалоговом окне Свойства в свойстве Обзор модели: Схема для автоматического проставления размеров щелкните по кнопке ++...++.
* Вы открыли проект. Параметры > Настройки > Проекты > Графическая обработка > Автоматическое проставление размеров. Выберите вкладку Функциональные элементы.

В этой вкладке необходимо выбрать функциональные элементы, для которых следует автоматически проставить размеры. Для каждого типа функционального элемента можно задать пороговую ступень иерархии, после которой процесс автоматического проставления размеров для него будет прекращен. Кроме того, для каждого типа функционального элемента можно установить позицию на элементе, в которой должна находиться точка измерения во время проставления размеров.

Обзор основных элементов диалогового окна:

### Панель инструментов

Кнопка |  Значение
---|---
![](images/all_new_as.png){: .ui-icon } (Создать) |  Создает новую строку в таблице. Для выбора или записи значений функциональных элементов дважды щелкните по ячейкам.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет выделенные строки.

### Таблица

Таблица содержит следующие столбцы:

* Функциональные элементы, для которых требуется проставить размеры: Выбор типа функционального элемента на основе определения его функции.
* Проставить размеры до степени: Выбор иерархической ступени, до которой следует проставлять размеры функционального элемента.
* Точка измерения на функциональном элементе: Выбор позиции на функциональном элементе, в которой должна находиться точка измерения.

# Указания по ознакомлению с руководством

Перед началом чтения обратите внимание на следующие указания:

!!! warning "Предупреждение:"

    Фрагменты текста, сопровождаемые этим знаком, являются предупреждениями. Прочтите их, прежде чем продолжить обработку!

Данным изображением предваряются примеры, которые можно развернуть. Для просмотра текста примера щелкните по стрелке.

!!! example "Пример:"

    Текст примера выводится на отображение. Щелкните по стрелке, чтобы скрыть текст примера.

!!! note "Замечание:"

    Рядом с таким изображением находятся указания, которые должны облегчить работу с программой.

!!! tip "Совет:"

    Рядом с этим изображением находятся полезные советы, например, по альтернативным методам работы.

* Элементы пользовательского интерфейса выделены шрифтом синего цвета, что позволяет быстро найти их в тексте.
* ***Полужирный*** шрифтом выделен текст, имеющий особую важность, на который следует обязательно обратить внимание.
* Примеры кодов, имена каталогов, команды прямого ввода и т.п. выделяются непропорциональным шрифтом.
* Для облегчения поиска определенных опций программы в некоторых темах справки имеется специальный путь меню. Выберите выделенные синим цветом меню и опции меню, чтобы перейти к соответствующей опции из главного меню. С помощью следующего меню можно, например, вызвать справочную систему EPLAN: Справка > Содержание.
* В указаниях по действиям, ведущих шаг за шагом к определенной цели, во многих местах после показывающего вправо символа стрелки ![](images/arrow.png){: .ui-icon } описывается реакция программы на ваши действия.
* Под указаниями ***См. также*** находятся гиперссылки на страницы, также относящиеся к данной теме. Например, с помощью нижеследующей гиперссылки можно перейти к разделу "Использование справочной системы", где представлена подробная информация о работе со справочной системой.

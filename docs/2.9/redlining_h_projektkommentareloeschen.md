# Удалить импортированные PDF-комментарии из EPLAN

PDF-комментарии, появившиеся в проекте путем импортирования PDF-комментариев, можно удалить вручную по отдельности или все вместе.

Условие:

* Откройте проект, в котором содержатся PDF-комментарии.

1. Проект > Организовать > Сжать
2. Щелкните в диалоговом окне Сжать проект рядом с полем Настройки по ++...++.
3. В диалоговом окне Настройки: Сжатие нажмите кнопку ![](images/all_new_as.png){: .ui-icon } (Создать), введите в поле Имя информативное имя и нажмите кнопку ++OK++.
4. Установите флажок Комментарии, нажмите кнопки ![](images/all_save_as.png){: .ui-icon } (Сохранить) и [ОК].
5. В диалоговом окне Сжать проект нажмите кнопку ++ОК++.

!!! info "Для сведения:"

    Комментарии будут удалены из проекта.

# Диалоговое окно Выбрать индекс ревизии

Вы создали ревизию и открыли проект ревизии (расширение имени файла *.ell). Выполнены изменения на странице и открыто диалоговое окно Описание изменения страниц. Рядом с полем Индекс ревизии щелкните ++...++.

В этом диалоговом окне выберите имеющийся индекс ревизии для описания страниц.

Доступны следующие элементы диалогового окна:

В таблице отображаются уже содержащиеся в проекте индексы ревизий.

Выберите индекс ревизии и закройте диалоговое окно, нажав ++OK++. Имя ревизии и соответствующая запись для описания и причины изменения копируются затем в диалоговое окно Описание изменения страниц, и вы сможете изменить эти данные там.

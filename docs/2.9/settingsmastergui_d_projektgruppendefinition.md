# Диалоговое окно Настройки: Группы проектов (определение)

Параметры > Настройки > Пользователь > Отображение > Группы проектов (определение)

В этом диалоговом окне можно создать дополнительные группы проектов и задать для имеющихся групп проектов цвет для отображения в навигаторах данных проекта, а также дополнительные навигаторы данных проекта.

Обзор основных элементов диалогового окна:

### Группы проектов

Таблица содержит следующие столбцы:

* Строка: Показывает непрерывный номер строки.
* Имя: Здесь можно ввести имена максимум для трех групп проектов. Имена первых двух групп проектов "Проекты схем соединений" и "Проекты макросов" предварительно определены и не могут быть изменены.
* Цвет: Здесь отображается выбранный для группы проектов цвет, используемый в навигаторах данных проекта в качестве цвета контура для проектов этой группы. Для перехода в диалоговое окно [Выбор цвета](mf_d_farbauswahl.md) и выбора другого цвета щелкните ячейку, а затем нажмите кнопку ++...++.
* Цвет активирован: Установите соответствующий флажок, если для группы проекта следует отобразить выбранный цвет.

Если для проекта активирован цвет посредством присвоения групп проектов, заданный здесь цвет отображается в навигаторах данных проекта и в настройках проекта в виде цветной границы для открытого проекта. Если вы открываете страницу / пространство листа и т. д. для такого проекта, цвет также используется для выделения вкладки соответствующего редактора.

### Дополнительные навигаторы

Допустимо использование до десяти дополнительных навигаторов. Эти навигаторы можно отображать и скрывать в заданной здесь последовательности с помощью меню Данные проекта > Дополнительные навигаторы > ….

Дополнительные навигаторы можно использовать, например, чтобы для определенных данных проекта (страниц, устройств и т. д.) одновременно отображать несколько представлений, отфильтрованных разными способами.

Таблица содержит следующие столбцы:

* Строка: Показывает непрерывный номер строки.
* Тип: Выберите здесь тип навигатора данных проекта, (например, страницы, устройства и т. д.), который должен быть доступен дополнительно.
* Группа проектов: Выберите здесь имя группы проектов с соответствующим дополнительным навигатором.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

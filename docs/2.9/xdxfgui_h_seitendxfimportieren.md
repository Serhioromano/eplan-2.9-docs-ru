# Импортировать страницы в формате DXF/DWG

При импорте файлов DXF / DWG на уровне проекта, для каждого выбранного файла DXF / DWG создается графическая страница.

Условие:

Вы открыли проект.

1. Страница > Импортировать > DXF/DWG
2. Определите в диалоговом окне Выбор файла DXF/DWG в поле Тип файла, будут ли отображаться только файлы DXF или DWG.
3. При необходимости перейдите к каталогу, в котором находятся файлы DXF или DWG. Выберите необходимые файлы и щелкните ++Открыть++.
4. В поле Схема диалогового окна Импорт DXF/DWG выберите из раскрывающегося списка подходящую схему. Или щелкните ++...++, чтобы вызвать диалоговое окно Настройки: Экспорт и импорт DXF / DWG. Здесь можно посмотреть настройки стандартной схемы, создать и обработать собственные схемы.
5. Щелкните по кнопке ++OK++.
6. В диалоговом окне Присвоить страницы определите порядок обработки новых страниц проекта.
7. Щелкните по кнопке ++OK++.
8. Если в меню Параметры > Настройки > Пользователь > Интерфейсы > Экспорт и импорт DXF / DWG на вкладке Импорт выбран параметр Корректировка вручную, откроется диалоговое окно Форматир. импорта, и вы сможете дополнительно обработать автоматически полученные границы чертежа.
9. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Файлы будут перенесены в текущий проект в качестве вновь созданных графических страниц.

!!! tip "Совет:"

    Вы можете импортировать файлы DXF / DWG с помощью функции перетаскивания мышью, перетащив выделенный файл DXF / DWG из Windows Explorer в навигатор страниц вашего проекта.

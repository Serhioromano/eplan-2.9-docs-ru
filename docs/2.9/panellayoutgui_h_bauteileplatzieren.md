# Разместить функциональные элементы на монтажной плате

При размещении функциональных элементов на монтажной плате различают отдельное размещение, многократное размещение и размещение заблокированных областей. При этом необходимо сохранить интервалы установки для каждого функционального элемента / устройства, чтоб стенки изделий находились не ***друг на друге***, а ***рядом друг с другом***. В этом случае при многократном размещении интервалы учитываются автоматически, с других точек зрения отдельное и многократное размещение идентичны.

Условия:

* Вы открыли проект.
* Вы открыли диалоговое окно Компоновка электрошкафа 2D - <Имя проекта>.
* Вы находитесь в графическом редакторе на странице типа "компоновка электрошкафа".
* Вы вставили монтажную плату.
* В виде представления Список имеются изделия со статусом "Не размещен" (свойство Число штук / количество (не размещен)).
* В диалоговом окне Настройки: Компоновка электрошкафа 2D (Всплывающее меню > Настройки) ***не*** выбрана ни одна опция Ввод вручную ++...++.

1. Выделите еще неразмещенные функциональные элементы во вкладке Список. (Возможен многократный выбор.)
2. Выберите пункты Всплывающее меню > Установка на монтажную плату.

!!! info "Для сведения:"

    Первое размещение изделия присоединяется к курсору.

3. Поместите размещение изделия на монтажной плате, щелкнув клавишей мыши.
4. Выполните аналогичные действия для всех размещений изделий, которые были выбраны в дереве.

!!! info "Для сведения:"

    Свойство Число штук / количество (не размещено) вновь размещенных изделий сократится соответствующим образом. В ***представлении в виде дерева*** изделия определяются на иерархическом уровне монтажной платы, и при помощи флажка отображается, что данное изделие размещено.

!!! note "Замечание:"

    * При размещении неразмещенного изделия ***клеммника*** будет отображен запрос (если ни одно изделие данного клеммника не размещено), с помощью которого можно решить, необходимо ли разместить клеммник полностью. Если ответить [Да]на данный запрос, то можно разместить клеммник полностью, и все клеммы клеммника в дереве навигатора компоновки электрошкафа будут помечены зеленой галочкой, как размещенные. При этом размеры клеммника на монтажной плате складываются из суммы сохраненных в базе данных изделий данных ширины клемм. Высота нового размещения изделия использует наибольшую высоту, сохраненную для изделия клеммы.
    Если выбрать [Нет]в запросе, клеммы будут размещены по отдельности.
    * Даже если не перемещать курсор во время размещения, символы размещаются слева направо. Если поле выходит за край монтажной платы, выводится соответствующее сообщение.
    * Если в диалоговом окне Настройки: Компоновка электрошкафа 2D (Всплывающее меню > Настройки) выбрать опцию Ввод вручную ++...++, перед размещением устройства необходимо также указать его размеры в диалоговом окне Удлинение и интервал.

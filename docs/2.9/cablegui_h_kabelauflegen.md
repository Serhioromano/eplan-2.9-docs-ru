# Присвоить кабельные соединения автоматически

С помощью пункта меню Присвоить кабельные соединения вы можете автоматически присвоить кабельным соединениям соединения изделия кабеля двумя разными способами:

* Параметр Все присвоить заново перезаписывает все свойства соединений свойствами из изделия. Сравнивается только тип потенциала.
* Опция Оставить имеющиеся свойства без изменения оставляет имеющиеся свойства соединения без изменений. Присвоение происходит только в том случае, если все идентифицирующие свойства кабельного соединения соответствуют свойствам соединения (изделия).

При этом для выделенных кабелей повторно сортируется последовательность кабельных соединений. Порядок соединений определяется по правилам определения источника и цели кабеля или, в случае клемм, по порядку клемм в клеммнике.

Условия:

* Вы выделили линию определения кабеля в Графическом редакторе или определение кабеля в диалоговом окне навигатора Кабели — <Имя проекта>.
* Кабелю присвоено изделие.

### Присвоить все кабельные соединения заново

1. Выберите пункты меню Данные проекта > Кабели > Присвоить кабельные соединения > Все присвоить заново.

!!! info "Для сведения:"

    Если тип потенциала совпадает, соединения изделия кабеля присваиваются кабельным соединениям. Существующие свойства кабельных соединений перезаписываются со свойств изделия.

!!! info "Для сведения:"

    Данные точек определения соединений соответственно обновляются. Имеющиеся данные перезаписываются. Новые данные записываются на точку определения соединения, только если они не записаны на соответствующей линии определения кабеля / экранировании.

!!! info "Для сведения:"

    Порядок экранирований сохраняется, но обозначения экранирований могут изменяться.

!!! tip "Совет:"

    Пункт меню Присвоить кабельные соединения можно вызвать через всплывающее меню в диалоговом окне навигатора Кабель — <Имя проекта> и в диалоговом окне Обработать кабель.

### Присвоить кабельные соединения с подходящими свойствами

Идентифицирующие свойства присвоения:

* Определение функции
* Обозначение экранирования (Экранировано от)
* Парный индекс
* Тип потенциала
* Цвет / номер
* Поперечное сечение / диаметр.

Правила присвоения:

* Если свойство соединения изделия заполнено, кабельное соединение должно иметь аналогичное значение.
* Пустые свойства соединения изделия подходят к любым значениям кабельного соединения.
* Экранированные жилы подходят и к неэкранированным соединениям; но если подходящее экранированное соединение уже найдено, то его присвоение осуществляется в первую очередь.

1. Выберите пункты меню Данные проекта > Кабели > Присвоить кабельные соединения > Оставить имеющиеся свойства без изменения.

!!! info "Для сведения:"

    Соединения изделия кабеля присваиваются кабельным соединениям, если их идентифицирующие свойства совпадают.

!!! info "Для сведения:"

    Имеющиеся свойства кабельных соединений остаются при этом без изменений, дополнительные свойства изделия кабеля дополняются.

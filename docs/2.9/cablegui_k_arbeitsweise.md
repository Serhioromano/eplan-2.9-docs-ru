# Определение кабелей

Кабели представлены в EPLAN посредством определений кабеля. Они могут быть отображены графически посредством линии определения кабеля или экранирования. После этого в диалоговом окне свойств вы можете задать свойства кабеля и определить таким образом кабель.

В навигаторе кабелей или навигаторе устройств предусмотрена возможность предварительного определения кабеля с определенными свойствами без графического представления в проекте. Такой кабель без размещения позже можно будет проложить на вывод устройства или на клемму / клеммник (включая резервную клемму / резервный клеммник).

Предварительно определенный кабель идентифицируется на схеме соединений посредством обозначения устройства. При этом предварительное определение должно однозначно содержать тип кабеля и его свойства (Кабель / Группа соединений: Число соединений, Кабель / Группа соединений: Поперечное сечение / диаметр и т. д.). Дополнительно существует возможность предварительного определения длины для управления предварительно изготовленными (резервными) кабелями, которые будут использоваться позже.

### Определение кабельных соединений

Есть несколько возможностей определить кабельные соединения (т. е. жилы или — в кабелях-гибридах — другие входящие в кабель соединения) вручную:

* Путем черчения ***Линии определения кабеля*** при помощи соединений, которые должны входить в кабель. Определение функции этих соединений автоматически устанавливается на значение "Жила / провод", а для свойства Соединение: Принадлежность задается значение "Кабель".
* ***Точка определения соединения*** размещается на каждом соединении, которое должно входить в кабель. На точках определения соединения для свойства Соединение: Принадлежность выберите значение "Кабель".
* ***Выводу устройства*** функции (в логической схеме выводов устройства) присваивается свойство Вывод кабеля.

!!! note "Замечание:"

    Однополюсное представление кабелей:
    Если в однополюсном представлении протянуть линию определения кабеля через соединение типа 'Кабели', там будет размещена точка определения соединения, и на ней автоматически будет записан вид представления 'Однополюс.'. При смешанном отображении (однополюсные символы на многополюсных страницах) Вам необходимо ввести вид представления самостоятельно на точке обозначения соединения.

После определения кабельных соединений можно выбрать к ним соответствующий кабель из базы данных изделия. Для этого используется или выбор изделий, или автоматический выбор кабелей. При автоматическом выборе кабелей EPLAN согласно установленным настройкам автоматически генерирует определение кабеля и задает тип кабеля с определенными свойствами. Дополнительно изменить свойства кабеля вручную можно и после автоматического присвоения кабеля.

### Присвоение кабельных соединений кабелю

Кабельные соединения присваиваются кабелю, при этом им назначаются одинаковые ОУ.

Если соединения отрезаются от ***Линии определения кабеля***, там автоматически размещаются точки определения соединений, а соединения сохраняют ОУ линии определения кабеля. Точки определения соединений в этом случае невидимы, просмотреть точки вставки и таким образом найти точки определения соединений можно через Вид > Точки вставки.

Значения в полях Видимое ОУ, Длина, Поперечное сечение / диаметр и Единица измерения переносятся с линии определения кабеля на отсеченные вами соединения, если соответствующие поля в диалоговом окне свойств автоматически размещенной точки определения соединения не имеют записи.

Кабельные соединения сохраняют ОУ ***Экранирования***, если точки определения соединений расположены внутри символа экранирования.

Соединение можно также присвоить кабелю, для этого ***Точка определения соединения*** размещается на соединении, после чего там записывается ОУ кабеля. Кроме того, необходимо установить для свойства Соединение: Принадлежность значение "Кабель".

Точки определения соединений, определяющие кабельное соединение, могут принимать ОУ от других точек определения соединений, определяющих кабельное соединение, которые находятся слева (или сверху, в зависимости от рамки). Здесь имеется в виду обычное сохранение ОУ (слева от черного ящика, ...). Таким образом, ОУ не переносится в поле свойства Видимое ОУ точек определения соединений.

### Кабель-гибрид

Кроме жил кабель может включать световоды или шланги (кабель-гибрид).

На точках определения соединений, которые вставляются вместе с линией определения кабеля или экранированием, тип вывода устройства автоматически устанавливается на 'Жила / Провод'. Все остальные типы выводов устройств необходимо настраивать вручную в точке определения соединения через Выбор определения функции. Кроме того, если при обновлении соединений типы вывода устройства не соответствуют данным на точках определения соединений, будет отображаться сообщение об ошибке.

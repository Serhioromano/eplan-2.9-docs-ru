# P028028: Указанное вещество '\<x\>' содержится в проекте не в качестве шаблона

### Причина

На объекте планирования (трубопроводе) или определении трубопровода / соединения найдено вещество, не определенное в качестве шаблона в проекте.

### Решение

1. Установите местоположение объекта планирования (трубопровода) в навигаторе предварительного планирования с помощью функции Синхронизировать выбор во всплывающем меню диалогового окна Управление сообщениями. Или воспользуйтесь функцией Перейти к (графика) всплывающего меню, чтобы перейти к точке определения трубопровода / соединения на функциональной схеме автоматизации.
2. Откройте диалоговое окно "Свойства" объекта планирования (трубопровода) или точки определения трубопровода / соединения и переместите первую вкладку на передний план.
3. В таблице свойств объекта планирования (трубопровода) для свойства Вещество в поле значений щелкните ++...++. Или, в случае с точкой определения трубопровода / соединения, щелкните ++...++ рядом с полем Вещество.
4. В диалоговом окне Выбрать вещество выделите имя из предлагаемого списка.
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Если требуемое вещество отсутствует в диалоговом окне Выбрать вещество, необходимо определить его в ***навигаторе шаблонов сегментов*** в качестве нового шаблона.

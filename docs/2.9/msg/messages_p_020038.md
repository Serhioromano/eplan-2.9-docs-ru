# P020038: Для свойства 'Обрабатывать ОУ в отдельных полях' значение '\<x\>' было изменено на '\<y\>'

### Описание

Выполнено сравнение проектов, при котором ***Структура*** проверяемого проекта (проекта с замечаниями) была сравнена со свойствами проекта для сравнения и были определены различия.

Это сообщение указывает, изменилась ли в проверяемом проекте настройка Обрабатывать ОУ в отдельных полях (диалоговое окно Управление проектами, вкладка Структура ОУ). \<x\> выводит состояние настройки перед изменением, а \<y\> — состояние после изменения.

!!! note "Замечание:"

* Это сообщение выводится только с учетом определенной, специфической для клиента версии EPLAN.
* Это относящееся к модулю сообщение можно удалить из диалогового окна управления сообщениями посредством нового контрольного прогона (по любой настроенной схеме).

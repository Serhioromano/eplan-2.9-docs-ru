# P501020: Внесенная в изделие принадлежность \<x\>, вариант \<y\>, \<z\>, отсутствует

### Причина

В проверяемой базе данных изделий на главном изделии было обнаружено изделие, которое не внесено в список принадлежностей и не существует как запись данных в базе данных изделий. EPLAN выводит имя изделия в заполнителе \<x\>, вариант изделия в заполнителе \<y\> и имя списка принадлежностей в заполнителе \<z\>.

### Решение

1. Откройте базу данных изделий и создайте требуемое изделие. Или импортируйте изделие.
2. Обозначьте изделие на вкладке Принадлежности как предмет оборудования, установив флажок Принадлежности.
3. Щелкните по кнопке ++OK++.
4. Выделите в дереве базы данных изделий требуемый ***список принадлежностей***.
5. Выведите на передний план вкладку Изделие и выберите в поле Номер изделия предмет оборудования для списка принадлежностей.
6. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
7. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется база данных изделий, нужное главное изделие в ней уже выделено.

8. Выведите на передний план вкладку Принадлежности и выберите в поле Номер изделия / Имя список принадлежностей.
9. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
10. Затем запустите новый контрольный прогон.

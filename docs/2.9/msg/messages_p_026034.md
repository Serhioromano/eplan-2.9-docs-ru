# P026034: Монтажная поверхность ЧУ не имеет действительного размера поля

### Причина

В пространстве листа находится размещение изделия, включающее в себя релевантные для ЧУ монтажные поверхности без действительного размера поля. Скорее всего, для изделия был составлен трехмерный макрос с релевантной для ЧУ монтажной поверхностью, в котором отсутствуют данные о размере поля.

### Решение

1. Установите местоположение размещения изделия в пространстве листа с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте навигатор пространства листа и перейдите в дереве к релевантной для ЧУ монтажной поверхности, указанной в пространстве листа.
3. В дереве выделите монтажную поверхность, а во всплывающем меню выберите пункт Размер поля.

!!! info "Для сведения:"

    В пространстве листа по краям монтажной поверхности появятся 2 вертикальные и 2 горизонтальные линии, задающие размер поля.

4. Перейдите на пространство листа и выделите одну из линий на монтажной поверхности, которая затем изменит свой цвет.
5. Щелкните по линии левой кнопкой мыши и переместите ее на нужную позицию.
6. Завершите режим перемещения, для чего еще раз щелкните левой кнопкой мыши.
7. При необходимости повторите эти рабочие шаги для остальных линий релевантной для ЧУ монтажной поверхности.
8. Затем запустите новый контрольный прогон.

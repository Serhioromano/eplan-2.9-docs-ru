# P005048: Сеть содержит символы соединения со свойством 'Определение целей'

### Причина

Сеть содержит один или несколько символов соединения, на которых установлено свойство Указывать цели. Тогда вместо точки соединения отображаются цели разводки.

!!! example "Пример:"

    Сеть на схеме соединений содержит тройник, на котором установлено свойство Указывать цели(см. кружок на следующем рисунке).

### Решение

1. Чтобы определить, о какой сети из схемы соединений идет речь, щелкните двойным щелчком в диалоговом окне Управление сообщениями по сообщению, которое содержит символы соединения с указанием целей.
2. Проследите тройники, перекрестные соединения и перемычки, имеющиеся в сети, и проверьте отображение этих символов.
3. Обнаружив один или несколько символов соединения с указанием цели, вызовите для него диалоговое окно "Свойства".
4. Снимите в диалоговом окне "Свойства" флажок Указывать цели.

!!! info "Для сведения:"

    Все варианты целей символа соединения станут неактивными.

5. Щелкните по кнопке ++ОК++.

!!! info "Для сведения:"

    Соответствующий символ соединения будет теперь отображаться в качестве точки.

!!! tip "Совет:"

    Если вы проектируете сети, у вас есть возможность деактивировать указание цели уже при создании символов соединения при помощи настройки Деактивировать свойство "Указывать цели" при вставке точки определения сети (Параметры > Настройки > Пользователи > Графическая обработка > Символы соединения), при этом символы будут отображаться как точки соединения.

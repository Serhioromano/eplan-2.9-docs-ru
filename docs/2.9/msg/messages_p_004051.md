# P004051: Соединение отсутствует! '\<x\>' не соединено с физической сетью '\<y\>'

### Причина

При импорте данных ПЛК было определено соединение между устройством \<x\> и физической сетью \<y\>, которое на данный момент не найдено в проекте.

### Решение

Установите соединение устройства с физической сетью посредством вывода шины.

!!! note "Замечание:"

    Это относящееся к модулю сообщение выводится при импорте данных ПЛК. При выполнении нового контрольного прогона (любой настроенной схемы) его можно удалить из диалогового окна управления сообщениями.

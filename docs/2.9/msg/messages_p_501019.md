# P501019: Библиотека символов \<x\> в проекте не найдена (\<y\>)

### Причина

В проверяемой базе данных изделий было найдено изделие, на котором сохранен макрос символа, состоящий из символов библиотеки символов, которая отсутствует в контрольном проекте. Вероятно, речь идет о таком типе библиотеки символов, который не является стандартным для EPLAN. EPLAN отображает имя библиотеки символов в заполнителе \<x\> и имя макроса в заполнителе \<y\>.

### Решение

1. Создайте макрос с помощью символов, которые являются составной частью стандартной библиотеки символов EPLAN. Или скопируйте использованную для создания макроса библиотеку символов (если это возможно) в требуемый каталог основных данных системы символов и сохраните ее в контрольный проект.
2. Затем запустите новый контрольный прогон.

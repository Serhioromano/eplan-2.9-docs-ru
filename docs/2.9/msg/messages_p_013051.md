# P013051: Устройству можно присвоить не более 50 изделий

### Причина

При импорте спецификации выполнена попытка присвоения обозначению устройства более 50 изделий.

!!! note "Замечание:"

    В EPLAN устройству может быть присвоено максимум ***50*** изделий. Индекс изделия задает последовательность внесения изделий в диалоговом окне Свойства ++...++ на вкладке Изделие.

### Решение

Откройте файл импорта с помощью текстового или XML-редактора и удалите лишние изделия. Сохраните файл и заново импортируйте спецификацию.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

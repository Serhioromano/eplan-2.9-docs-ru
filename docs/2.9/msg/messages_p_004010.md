# P004010: Неизвестный каркас '\<x\>'

### Причина

На разъеме указано несуществующее обозначение каркаса.

### Решение

#### Возможность 1

Создайте новый каркас с соответствующим обозначением.

#### Возможность 2

1. Укажите для разъема уже существующий каркас.
2. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Этот контрольный прогон служит только для проверки блоков ПЛК, на которых указан проект конфигураций.

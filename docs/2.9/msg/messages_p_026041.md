# P026041: Источник/цель, размещенные в различных пространствах листа

### Причина

Источник и цель маршрутизируемого соединения находятся в различных пространствах листа.

### Решение

1. Установите местоположение источника маршрутизируемого соединения в пространстве листа при помощи функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями. Также перейдите к цели маршрутизируемого соединения в пространстве листа при помощи функции Перейти ко 2-й координате из всплывающего меню.
2. Убедитесь, что источник и цель маршрутизируемого соединения размещены в одном и том же пространстве листа.
3. Еще раз выполните маршрутизацию соединений.
4. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

# P501009: Записанная на изделии схема сверления \<x\> отсутствует

### Причина

В проверяемой базе данных изделий было найдено изделие, на котором сохранена схема сверления, которую EPLAN не может найти.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Выведите на передний план вкладку Изготовление.
3. Щелкните в столбце Схема сверления по записи в таблице и нажмите кнопку ++...++.
4. Выберите в открывшемся диалоговом окне выбора нужную схему сверления.
5. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
6. Затем запустите новый контрольный прогон.

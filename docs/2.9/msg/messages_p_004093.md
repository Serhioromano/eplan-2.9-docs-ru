# P004093: Вывод устройства ПЛК с помощью свойства 'Подустройство ПЛК: Индекс' указывает на несуществующее подустройство

### Причина

На выводе устройства ПЛК или выводе шины была обнаружена недействительная ссылка на подустройство ПЛК. Ссылка на подустройство ПЛК недействительна, если в соответствующем блоке ПЛК для свойства Подустройство ПЛК x: Имя и / или Подустройство ПЛК x: На позиции / разъем и / или для свойства Подчиненное устройство ПЛК x: начальные адреса (выходы) или Подустройство ПЛК x: Начальный адрес (входы) не задано никакое значение.

### Решение

#### Возможность 1

1. Установите местоположение вывода устройства ПЛК / вывода шины на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно 'Свойства' вывода устройства ПЛК или вывода шины.
3. Переместите первую вкладку на передний план.
4. Определите в списке свойств заданное значение для номера подустройства в поле Подустройство ПЛК: Индекс.
5. Перейдите от вывода устройства ПЛК / вывода шины, например с помощью пункта всплывающего меню Перейти к (обратный эквивалент), к соответствующему блоку ПЛК.
6. Также переместите первую вкладку на передний план.
7. В списке свойств проверьте, приведены ли для соответствующего подустройства ПЛК значения свойств, указанных в разделе "Причина".
8. При необходимости запишите соответствующие значения в пустые поля свойств.
9. Щелкните по кнопке ++OK++.
10. Затем запустите новый контрольный прогон.

#### Возможность 2

1. Повторите шаги с 1-го по 4-й, описанные в разделе "Возможность 1".
2. Измените значение в поле Подустройство ПЛК: Индекс, чтобы вывод устройства ПЛК / вывод шины соответствовал правильному подустройству ПЛК.
3. Затем запустите новый контрольный прогон.

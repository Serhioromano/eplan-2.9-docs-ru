# P008001: Текст '\<x\>' не может быть переведен на язык \<y\>

### Причина

Отсутствуют необходимые записи (ключевые слова) в соответствующем языке словаря.

### Решение

Выберите пункты меню Сервисные программы > Перевод > Обработать словарь. Проверьте в диалоговом окне Словарь используемые языки словаря. Если требуемый язык словаря отсутствует в списке на вкладке Управление, вставьте его с помощью кнопки ![](../images/all_new_as.png){: .ui-icon } (Создать). Затем перейдите на вкладку Обработать слова и выберите из раскрывающегося списка Исходный язык нужный язык. Введите в поле Поиск соответствующие ключевые слова и проверьте, имеются ли необходимые тексты перевода на данном языке словаря. Если нет, создайте отсутствующие ключевые слова на исходном языке и соответствующие тексты перевода на требуемом языке или дополните их, использовав кнопку ![](../images/all_new_as.png){: .ui-icon } (Создать). Закройте диалоговое окно Словарь и переведите необходимые тексты.

Затем запустите новый контрольный прогон.

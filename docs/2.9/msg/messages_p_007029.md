# P007029: Идентификатор ОУ '\<x\>' вспомогательной функции отличается от идентификатора ОУ главной функции ('\<y\>')

### Причина

На вспомогательной функции, являющейся составной частью макроса для модуля, внесенный идентификатор ОУ отличается от соответствующей главной функции в макросе.

### Решение

1. Установите местоположение вспомогательной функции на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно "Свойства" вспомогательной функции.
3. Введите отображаемый в сообщении идентификатор ОУ главной функции в свойство Идентификатор ОУ вспомогательной функции, чтобы оба значения свойства совпадали.
4. Щелкните по кнопке ++OK++.
5. Затем запустите новый контрольный прогон.

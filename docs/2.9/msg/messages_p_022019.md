# P022019: У условного обозначения или символа '\<x\>' отсутствует действительное определении функции '\<y\>'

### Причина

Вы используете в своем проекте условное обозначение, определения функции которого нет в сохраненной библиотеке определений функций.

### Решение

Обновите сохраненную библиотеку определений функций проекта, выбрав пункт меню Сервисные программы > Основные данные > Обновить текущий проект.

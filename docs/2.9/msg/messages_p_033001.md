# P033001: Недействительное имя проводного монтажа функции

### Причина

На схеме соединений определена функция, которой было присвоено имя проводного монтажа, но которая не имеет определения проводного монтажа с идентичным именем проводного монтажа. Вероятно, функция была скопирована из схемы соединений другого проекта и вставлена в открытый на данный момент проект, или имя проводного монтажа определений проводного монтажа в дальнейшем было переименовано.

### Решение

Если функции не присвоено определение проводного монтажа, удалите имя проводного монтажа с функции. Или присвойте функции существующее имя проводного монтажа, которое указано на определении проводного монтажа.

1. Установите местоположение данной функции на схеме соединений, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно свойств функции.
3. В списке свойств первой вкладки удалите значение свойства Имя проводного монтажа. Или при помощи кнопки ++...++ в поле значений свойства выберите существующее имя проводного монтажа определения проводного монтажа.
4. Щелкните по кнопке ++OK++.
5. Затем запустите новый контрольный прогон.

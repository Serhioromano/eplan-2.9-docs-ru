# P020043: Найден новый слой: \<x\>, \<y\>

### Описание

Выполнено сравнение проектов, при котором ***Слои*** проверяемого проекта (проекта с замечаниями) были сравнены со слоями проекта для сравнения и были определены различия.

Это сообщение появляется в том случае, если в проверяемом проекте был создан новый слой. В заполнителе \<x\> отображается ***имя***, а в заполнителе \<y\> — ***описание*** созданного слоя.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

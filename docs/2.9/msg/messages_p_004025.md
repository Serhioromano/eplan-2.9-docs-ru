# P004025: Отсутствующий разъем

### Причина

Для блока ПЛК в поле Карта ПЛК вставлена в каркас введено обозначение каркаса, но не указана позиция разъема.

### Решение

Откройте диалоговое окно Свойства ++...++ для соответствующего блока ПЛК. Введите отсутствующую позицию разъема на вкладке Данные структуры ПЛК в поле На позиции / разъем.

Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Этот контрольный прогон служит только для проверки блоков ПЛК, на которых указан проект конфигураций.

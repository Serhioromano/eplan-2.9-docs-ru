# P005075: Разные вещества внутри трубопровода

### Причина

На функциональной схеме автоматизации на соединении содержатся несколько определений трубопровода (выводы трубопровода или точки определения трубопровода), на которых сохранены противоречивые значения относительно свойства Вещество.

### Решение

1. Установите местоположение соответствующих определений трубопровода с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Для каждого определения трубопровода откройте диалоговое окно "Свойства" и введите одинаковое значение в поле Вещество.
3. Обновите соединения и затем повторно запустите контрольный прогон.

# P020010: \<x\> различается в настройке представления

### Описание

При восстановлении выгруженного проекта его настройки сравнивались с настройками исходного проекта. В отношении настройки представления имеется различие.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

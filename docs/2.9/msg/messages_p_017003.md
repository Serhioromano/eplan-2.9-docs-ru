# P017003: Неверное буквенное обозначение

### Причина

При вводе обозначения устройства использовано неразрешенное буквенное обозначение.

### Решение

#### Возможность 1

Выберите пункты меню Сервисные программы > Основные данные > Буквенное обозначение и просмотрите в диалоговом окне Предложения буквенных обозначений, какое буквенное обозначение сохранено для выбранного определения функции. При вводе ОУ используйте это буквенное обозначение.

#### Возможность 2

Если несмотря на это будет выводится неверное буквенное обозначение, необходимо создать соответствующее присвоение буквенного обозначения. Выберите пункты меню Сервисные программы > Основные данные > Буквенное обозначение и в диалоговом окне Предложения буквенных обозначений выберите во всплывающем меню пункт Создать. Сохраните обозначение для нового набора буквенных обозначений и введите условное обозначение для соответствующей функции.

После этого настоятельно рекомендуется выполнить синхронизацию основных данных проекта (Сервисные программы > Основные данные > Синхронизировать текущий проект).

Затем запустите новый контрольный прогон.

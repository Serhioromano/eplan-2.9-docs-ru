# P022003: Двойной номер страницы: \<x\>

### Причина

С помощью интерфейсов данных, например, при копировании данных из EPLAN 21, возможно возникновение страниц с двойными номерами.

### Решение

Переименуйте одну из страниц, использовав идентичный номер страницы в навигаторе страниц.

Затем запустите новый контрольный прогон.

# P501022: Размеры присоединений шаблонов функций не совпадают с размерами присоединений на схеме соединений

### Причина

В проверяемой базе данных изделий были определены шаблоны функций с размерами присоединения на изделии, которые не совпадали с размерами присоединения выбранной на изделии схемы соединений.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий; нужное изделие уже выделено.

2. Выведите на передний план вкладку Шаблоны функций.
3. Запишите размеры присоединения, внесенные в таблицу для выводов устройства.
4. Перейдите на вкладку Технические данные.
5. Запомните имя, указанное в групповом поле Схема соединений.
6. Закройте базу данных изделий.
7. Повторно откройте базу данных изделий, на этот раз через меню Сервисные программы > Изделие > Управление.
8. В виде дерева выделите схему соединений, имя которой вы запомнили.
9. Перейдите на вкладку Выводы устройства.
10. Скопируйте в таблицу предварительно записанные размеры присоединения выводов устройства из шаблонов функций, чтобы значения на схеме соединений и на шаблонах функций полностью совпадали.
11. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
12. Затем запустите новый контрольный прогон.

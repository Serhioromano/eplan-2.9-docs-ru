# P028003: Структурный идентификатор в сегменте структуры встречается несколько раз (\<x\> = структурный идентификатор)

### Причина

Структурный идентификатор, выводимый посредством \<x\> в диалоговом окне Управление сообщениями, встречается более одного раза в нескольких сегментах структуры проекта.

### Решение

1. Установите местоположение многократно встречающегося структурного идентификатора при помощи функций Перейти к (графика), Перейти ко 2-й координате или Синхронизировать выбор, которые можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно Свойства ++...++ сегмента структуры и переместите вкладку <Отображаемое имя сегмента структуры> на передний план.
3. Измените соответствующие структурные идентификаторы в блоке идентификаторов сегмента структуры, чтобы сохранить в проекте однозначный структурный идентификатор.
4. Закройте диалоговое окно "Свойства", нажав ++OK++, и повторно запустите контрольный прогон.

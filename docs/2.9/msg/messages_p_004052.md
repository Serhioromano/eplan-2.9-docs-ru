# P004052: Соединение с неверной физической сетью! '\<x\>' должно быть соединено с физической сетью '\<y\>'

### Причина

При последнем импорте данных ПЛК было найдено соединение, противоречащее существующему сейчас в проекте соединению. Могло быть создано неверное соединение.

### Решение

Замените соединение в проекте указанным в сообщении соединением \<x\> и подключите его к физической сети \<y\>.

!!! note "Замечание:"

    Это относящееся к модулю сообщение выводится при импорте данных ПЛК. При выполнении нового контрольного прогона (любой настроенной схемы) его можно удалить из диалогового окна управления сообщениями.

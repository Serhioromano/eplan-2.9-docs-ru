# P024010: Недействительная длина группы соединений

### Причина

На определении группы соединений введено отрицательное значение для длины группы соединений.

### Решение

1. Установите линию определения группы соединений на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно свойств линии определения группы соединений.
3. В поле Длина введите положительное значение.
4. Щелкните по кнопке ++OK++.
5. Обновите соединения и затем повторно запустите контрольный прогон.

!!! note "Замечание:"

    Другие недопустимые значения, например буквы, не принимаются. Недопустимые значения, содержащие интерпретируемые единицы, корректируются автоматически. Например, введенное значение 2М интерпретируется как длина 2 м.

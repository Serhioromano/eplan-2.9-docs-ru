# P007026: Логическая модель функции отличается от определения функции

### Описание

В диалоговом окне Логическая схема выводов устройства функции было задано отличающееся от определения функции значение. Например, были изменены тип вывода устройства, число целей и / или вывод устройства символов. Поля, значения в которых отличаются от стандартной настройки, выделяются в этом диалоговом окне желтым фоном.

Диалоговое окно Логическая схема выводов устройства вызывается в диалоговом окне свойств функции щелчком кнопки ++Логика++ на вкладке Данные символа / функции.

Местоположение функции на схеме соединений можно установить с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.

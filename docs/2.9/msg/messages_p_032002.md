# P032002: Имя схемы сверления \<x\> представлено многократно

### Причина

Для нескольких схем сверления было задано одинаковое имя. Заполнитель \<x\> в сообщении отображает многократно встречающееся имя схемы сверления.

### Решение

1. Установите местоположение многократно встречающихся схем сверления в проекте, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями .
2. По очереди измените в диалоговом окне Свойства ++...++ соответствующих границ схем сверления имена, так чтобы каждое имя было представлено в проекте только один раз.
3. Подтвердите изменения, нажав кнопку ++OK++.
4. Запустите любой контрольный прогон, чтобы удалить сообщение.
5. Затем заново сгенерируйте схему сверления, выбрав пункты меню Сервисные программы > Генерировать схемы сверления / контуры > Автоматически из проекта макросов.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

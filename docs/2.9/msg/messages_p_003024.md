# P003024: Кабель и его кабельные соединения имеют разный статус искробезопаности

### Причина

На кабеле в качестве главной функции активировано свойство Искробезопасный (вкладка Данные символа / функции). Минимум одно кабельное соединение кабеля не обладает этим свойством. И наоборот, может получиться, что все кабельные соединения обладают свойством Искробезопасный, а кабель в качестве главной функции — нет.

### Решение

#### Возможность 1

Двойным щелчком мыши последовательно выделите точки пересечения линии определения кабеля с кабельными соединениями. Проверьте для каждого кабельного соединения, установлен ли в диалоговом окне Свойства (усл. обозначение): Точка определения соединения флажок Искробезопасный на вкладке Данные символа / функции. Если флажок не установлен, установите его.

#### Возможность 2

Если кабель и относящиеся к нему кабельные соединения не должны быть искробезопасными, снимите флажок Искробезопасн. для кабеля и всех кабельных соединений.

Обновите соединения и затем повторно запустите контрольный прогон.

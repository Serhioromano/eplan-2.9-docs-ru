# P004038: Для подчиненной шины '\<x\>' в выводе шины в шинной системе '\<y\>' определена скорость передачи данных

### Причина

Для подчиненной шины указанного вывода шины в шинной системе в качестве свойства была определена скорость передачи данных, и за ней было закреплено значение.

Обратите внимание, что скорость передачи данных можно определить только для главного в шинной системе, а не для подчиненной шины! Главный определяется с помощью свойства Логическая сеть: Вывод шины является главным.

Этот контрольный прогон служит только для проверки выводов шины, на которых указан проект конфигураций.

### Решение

1. Установите местоположение подчиненной шины на схеме соединений, пользуясь функцией Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно Свойства ++...++ для вывода подчиненной шины и переместите вкладку Данные шины на передний план.
3. Удалите из таблицы в групповом поле Свойства сетевого интерфейса свойство Скорость передачи данных и закройте диалоговое окно Свойства ++...++, нажав кнопку ++OK++.
4. Затем запустите новый контрольный прогон.

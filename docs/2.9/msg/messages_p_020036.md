# P020036: Для элемента '\<x\>' структуры проекта значение '\<y\>' было заменено на '\<z\>'

### Описание

Выполнено сравнение проектов, при котором ***Структура*** проверяемого проекта (проекта с замечаниями) была сравнена со свойствами проекта для сравнения и были определены различия.

Это сообщение указывает, что в проверяемом проекте отображено изменение элемента структуры проекта, расширенного относительно структуры проекта. К расширенным элементам структуры проекта относятся все элементы, которые можно настроить в диалоговом окне Расширенные структуры проекта, например разделители. (Выделить проект в навигаторе страниц, выбрать пункты всплывающего меню Проект > Свойства, открыть вкладку Структура и нажать кнопку ++Другие++.) \<x\> обозначает расширенный элемент структуры проекта, \<y\> — старую настройку, а \<z\> — новую настройку.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

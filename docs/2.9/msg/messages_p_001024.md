# P001024: Соединить различные клеммные выводы (внутри/снаружи)

### Причина

Две или более связанных друг с другом клемм имеют различные настройки в отношении свойства вывода устройства Внутр. / внешн. Это происходит, например, если вывод устройства функции 1 одной клеммы определен как внутренний и связан с выводом устройства функции 1 соседней клеммы, которая определена свойством вывода устройства 'Внешн.'.

### Решение

1. Установите местоположение данных клемм на схеме соединений, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Для этого запустите диалоговое окно Свойства для клемм и щелкните вкладку Данные символа / функции.
3. Посредством кнопки ++Логика++ перейдите в диалоговое окно Логическая схема выводов устройства.
4. Проверьте настройку свойства выводов устройства Внутр. / внешн. у связанных друг с другом выводов устройства функции клемм.
5. Измените свойство выводов устройства Внутр. / внешн. у связанных друг с другом выводов устройства функции клемм таким образом, чтобы настройка имела единое значение 'Внутр.' или 'Внешн.'.
6. Завершите изменения нажатием кнопки ++OK++.
7. Обновите соединения и затем повторно запустите контрольный прогон.

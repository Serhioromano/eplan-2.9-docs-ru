# P030002: Частичный проект '\<x\>' содержит изменения за пределами определенной области обработки

### Причина

В частичном проекте были созданы или переименованы, к примеру, страницы, пространства листа или сегменты предварительного планирования, которые находятся за пределами определенной области обработки.

!!! note "Замечание:"

    Это сообщение выводится только в частичном проекте.

### Решение

1. Удалите в частичном проекте страницы, пространства листа или сегменты предварительного планирования, которые находятся за пределами определенной области обработки, или переместите их в определенную область обработки.
2. Запустите любой контрольный прогон, чтобы удалить сообщение.
3. Заново сохраните частичный проект в исходном проекте.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

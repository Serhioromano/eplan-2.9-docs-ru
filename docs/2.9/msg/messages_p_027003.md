# P027003: Функция топологии не связана с соединением

### Причина

#### Возможность 1

В сети соединенных сегментов была обнаружена функция топологии, которая не связана с целью посредством маршрутизируемого соединения. То есть отсутствует маршрутизируемое соединение на выводе (схема соединений) функции топологии.

#### Возможность 2

В сети соединенных сегментов один из клеммников с видом представления "Топология" был представлен в распределенном виде, однако выводы устройства не были определены.

### Решение

#### Для возможности 1

1. Установите местоположение функции топологии в сети соединенных сегментов, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вставьте в сеть соединенных сегментов дополнительную функцию топологии в качестве цели маршрутизируемого соединения.
3. Добавьте в сеть соединенных сегментов сегменты и точки маршрутизации при помощи пунктов меню Вставить > Топология > Сегмент маршрутизации и Вставить > Топология > Точка маршрутизации.
4. Соедините обе функции топологии и точки маршрутизации с сегментами маршрутизации, и задайте таким образом трассу маршрутизации через сеть соединенных сегментов.
5. Выделите всю сеть соединенных сегментов и выберите Данные проекта > Топология > Маршрутизировать.

!!! info "Для сведения:"

    Будет сгенерировано маршрутизируемое соединение через сеть соединенных сегментов между обоими функциями топологии.

6. Затем запустите новый контрольный прогон.

#### Для возможности 2

1. Установите местоположение клеммника, представленного в распределенном виде, в сети соединенных сегментов, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите для каждого клеммника, представленного в распределенном виде, диалоговое окно Свойства ++...++ и переместите вкладку Схема соединений на передний план.
3. Выберите пункт всплывающего меню Обновить схему соединений.
4. Удалите из схемы соединений все выводы, которые не должны быть соединены с представлением клеммника. (Например, это могут быть выводы устройства, которые соединены с целями в другой сети соединенных сегментов.)
5. Выделите всю сеть соединенных сегментов и заново маршрутизируйте соединения.
6. Затем запустите новый контрольный прогон.

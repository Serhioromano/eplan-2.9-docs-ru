# P028001: Сегмент структуры встречается несколько раз

### Причина

Сегмент структуры встречается в проекте более одного раза с одним и тем же обозначением.

### Решение

1. Установите местоположение многократно встречающегося сегмента структуры с помощью функциональностей Перейти к (графика), Перейти ко 2-ой координате или Синхронизировать выбор, которые можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно Свойства ++...++ сегмента структуры и переместите вкладку <Отображаемое имя сегмента структуры> на передний план.
3. Измените в поле Обозначение последовательность знаков, так чтобы в проекте был сохранен однозначный сегмент структуры.
4. Закройте диалоговое окно "Свойства", нажав ++OK++, и повторно запустите контрольный прогон.

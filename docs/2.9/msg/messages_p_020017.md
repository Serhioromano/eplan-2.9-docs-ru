# P020017: \<x\> (значение \<y\>) \<z\> -> \<u\>

### Описание

Выполнено сравнение проектов, при котором ***Настройки*** проверяемого проекта (проект с замечаниями) сравнивались с настройками проекта для сравнения, а также была установлена разница. В этом сообщении \<x\> обозначает путь файла к настройке проекта, в которой были обнаружены различия между проверяемым проектом и проектом для сравнения. \<y\> показывает при этом номер и позицию значения соответствующей настройки. \<z\> показывает значение настройки в проекте для сравнения, \<u\> — значение в проверяемом проекте.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

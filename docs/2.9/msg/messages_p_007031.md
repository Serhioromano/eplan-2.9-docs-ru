# P007031: Указанное на функции изделие '\<x\>' не содержится в классе трубы '\<y\>'

### Причина

На функциональной схеме автоматизации на технологической функции указан номер изделия, не совпадающий с номером изделия из выбранного класса трубы.

### Решение

#### Возможность 1

1. Откройте ***навигатор шаблонов сегментов*** и вызовите диалоговое окно "Свойства" класса трубы, выведенное в заполнителе \<y\> сообщения.
2. Переместите вкладку Изделие класса трубы на передний план.
3. Добавьте в левую таблицу выведенный в тексте сообщения номер изделия \<x\> технологической функции.
4. Затем запустите новый контрольный прогон.

#### Возможность 2

1. Установите местоположение технологической функции на функциональной схеме автоматизации с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно свойств функции.
3. На первой вкладке выберите класс трубы, на котором указан номер изделия, выведенный в тексте сообщения \<x\>.
4. Затем запустите новый контрольный прогон.

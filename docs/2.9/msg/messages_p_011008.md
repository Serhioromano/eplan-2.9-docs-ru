# P011008: Несколько источников типа “звезда” с одинаковым обознач. устройства

### Причина

В проекте существует несколько точек разрыва, которые являются источниками типа "звезда" и имеют одинаковое обозначение устройства.

### Решение

#### Возможность 1

Удалите лишние точки разрыва, являющиеся источниками типа "звезда".

#### Возможность 2

Введите отличающиеся обозначения устройства для многократно использованных точек разрыва, являющихся источниками типа "звезда".

#### Возможность 3

Дважды щелкните по лишним точкам разрыва, являющимся источниками типа "звезда", и снимите в соответствующем диалоговом окне Свойства ++...++ флаг Источник типа "звезда".

Обновите соединения и затем повторно запустите контрольный прогон.

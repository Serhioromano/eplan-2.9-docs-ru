# P501016: Свойство изделия 'Подгруппа продуктов' не определено

### Причина

В проверяемой базе данных изделий было найдено изделие, на котором в свойстве Подгруппа продуктов указано значение "Не определено".

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Переместите вкладку Общее на передний план.
3. Измените в раскрывающемся списке Главная группа продуктов установленную ранее запись "Не определено". Выберите другую запись из предлагаемых в списке вариантов.
4. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
5. Затем запустите новый контрольный прогон.

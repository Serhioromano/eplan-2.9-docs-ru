# P004086: Отсутствует обозначение типа ПЛК или файл описания устройства

### Причина

На блоке ПЛК, присвоенном проекту конфигураций, для свойства Обозначение типа ПЛК или свойства Описание устройства: Имя файла не указано значение. Или значение не указано для обоих свойств.

Этот контрольный прогон служит только для проверки блоков ПЛК, на которых указан проект конфигураций.

### Решение

1. Установите местоположение блока ПЛК на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно свойств блока ПЛК.
3. Переместите первую вкладку на передний план.
4. В таблице свойств введите в поле значений свойства Обозначение типа ПЛК или свойства Описание устройства: Имя файла требуемую информацию.
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Если активировано свойство Интегрированный узел на вкладке Данные структуры ПЛК, свойства Обозначение типа ПЛК и Описание устройства: Имя файла на первой вкладке могут быть пустыми.

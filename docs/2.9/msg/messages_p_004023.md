# P004023: Отсутствующий адрес

### Причина

На выводе устройства ПЛК не указан адрес.

Выводы устройства ПЛК с определением функции "Вывод устройства ПЛК, внутр., общ." не проверяются этим контрольным прогоном, поскольку они служат только для отображения свободных символических адресов.

### Решение

1. В диалоговом окне Свойства ++...++ вывода устройства ПЛК введите отсутствующий адрес в одноименное поле.
2. Затем запустите новый контрольный прогон.

# P501026: Вывод шины '\<x\>', внесенный в шаблоны функций изделия, не найден в однополюсном представлении в макросе '\<y\>'

### Причина

В проверяемой базе данных изделий было найдено изделие, на котором определен шаблон функции вывода шины. В соответствующем макросе, сохраненном на вкладке Технические данные базы данных изделий, введены разные значения для одного или нескольких свойств, перечисленных ниже:

* Интерфейс шины: Имя
* Система шин
* Определение функции
* Обозначение штекера.

### Решение

#### Возможность 1

1. Перенесите значение найденного свойства из шаблона функции изделия в одноименное свойство указанного макроса.
2. Сохраните изменения в макросе.
3. Затем запустите новый контрольный прогон.

#### Возможность 2

1. Перенесите значение найденного свойства из макроса на изделие.
2. Сохраните изменения в базе данных изделий.
3. Затем запустите новый контрольный прогон.

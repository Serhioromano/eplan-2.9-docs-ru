# P004018: Нет исп. элемента на выходе

### Причина

При отслеживании цели не удалось определить исполнительный элемент на выходе ПЛК. Причиной может быть неверное настроенный ***вид сигнала*** на выходе ПЛК и / или на исполнительном элементе. Кроме того, может отсутствовать ***указание отслеживания цели*** на исполнительном элементе / предвключенной клемме / штекере исполнительного элемента.

### Решение

1. Сначала дважды щелкните выход ПЛК и в диалоговом окне Свойства ++...++ выберите вкладку Данные символа / функции.
2. Щелкните по кнопке ++Логика++.
3. В диалоговом окне Логическая схема выводов устройства проверьте свойство Вид сигнала на выходе ПЛК. Это свойство ***должно*** совпадать с определением функции! Например, если вставлен вывод устройства ПЛК с определением функции "Вывод устройства ПЛК, дискр. выход" или "Вывод устройства ПЛК (Fluid-техника), дискретный выход", то в поле Вид сигнала необходимо установить "Цифровой выход"!
4. После этого проверьте, настроено ли для исполнительного элемента свойство Вид сигнала = "Исп. эл.".
5. Если между исполнительным элементом и выходом ПЛК находится клемма или штекер, дополнительно проверьте для этого объекта настройки в поле Отслеживание цели (ПЛК) в. На выводе устройства функции 1 ***необходимо*** ввести значение "2", а на выводе устройства функции 2 — значение "1".
6. Обновите соединения и затем повторно запустите контрольный прогон.

# P032001: Не удалось сгенерировать схему сверления, или она была сгенерирована не полностью. \<x\>

### Причина

Возможно несколько причин, выведенных в заполнителе \<x\> сообщения:

* Отсутствует имя схемы сверления.
* Блок не разбит.
* Контур не замкнут.
* Не идентифицировано ни одно монтажное отверстие.

### Решение

Для разных причин соответственно имеются следующие возможности решения для вывода сообщения:

1. Установите местоположение данной границы схемы сверления в проекте, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.

\- Внесите в диалоговое окно Свойства ++...++ границы схемы сверления имя.

\- Или разделите блок на отдельные элементы.

\- Или замкните контур.

\- Или проверьте присвоение уровней объектам, относящимся к границам схемы сверления.
2. Затем проведите любой контрольный прогон, чтобы удалить относящееся к модулю сообщение.
3. Затем заново сгенерируйте схему сверления, выбрав пункты меню Сервисные программы > Генерировать схемы сверления / контуры > Автоматически из проекта макросов.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

# P014001: Имя макроса '\<x\>' имеется многократно

### Причина

При генерации макросов из проекта макросов на рамке макроса было определено имя макроса, уже присвоенное другой рамке макроса. Или для одной и той же рамки макроса в определении макроса были сохранены несколько идентичных имен макроса.

### Решение

1. Установите местоположение данной рамки макроса в проекте макросов, пользуясь функцией Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.

!!! info "Для сведения:"

    Рамка макроса будет выделена на странице проекта.

2. Откройте с помощью всплывающего меню диалоговое окно свойств рамки макроса.
3. Переместите вкладку Рамка макроса на передний план.
4. В поле Имя измените имя макроса. Выберите имя, которое еще не задавалось другим рамкам макросов.
5. Перейдите на вкладку Определение макроса и проверьте, нет ли под 1-й строкой в столбце таблицы Макрос: Имя других строк с идентичным именем макроса.
6. Если есть, удалите повторяющиеся имена макроса или замените их на уникальные имена.
7. Закройте диалоговое окно Свойства, нажав кнопку ++ОК++.
8. Запустите новый контрольный прогон.
9. Повторите шаги 1–9 для других отображенных в диалоговом окне Управление сообщениями рамок макросов.

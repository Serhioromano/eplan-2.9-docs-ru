# P028002: Объект планирования встречается несколько раз

### Причина

Объект планирования встречается в проекте более одного раза с одним и тем же обозначением.

### Решение

1. Установите местоположение многократно встречающегося объекта планирования с помощью функциональностей Перейти к (графика), Перейти ко 2-ой координате или Синхронизировать выбор, которые можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно Свойства ++...++ объекта планирования и переместите вкладку <Отображаемое имя объекта планирования> на передний план.
3. Измените в поле Обозначение последовательность знаков, так чтобы в проекте был сохранен однозначный объект планирования.
4. Закройте диалоговое окно "Свойства", нажав ++OK++, и повторно запустите контрольный прогон.

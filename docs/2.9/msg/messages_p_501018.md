# P501018: Сохраненные в изделии размеры 'Ширина' и 'Высота' не совпадают с размерами макроса \<x\> (\<y\> / \<z\>) (допуск \<u\> проц.)

### Причина

В проверяемой базе данных изделий было найдено изделие, на котором сохранен двухмерный макрос окна, размеры которого отличаются от записанных на изделии значений свойств изделия Высота и Ширина на предварительно заданный максимальный допуск. EPLAN отображает имя макроса в заполнителе \<x\>, вид представления в заполнителе \<y\>, вариант в заполнителе \<z\> и применяемый допуск в заполнителе \<u\>.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Переместите вкладку Монтажные данные на передний план.
3. Запишите значения в полях Ширина и Высота.
4. Перейдите на вкладку Технические данные.
5. Запомните введенное в поле Макрос имя макроса.
6. Закройте базу данных изделий.
7. Теперь разместите макрос на странице схемы соединений вашего контрольного проекта и вызовите свойства макроса.
8. Запишите присвоенные значения свойств Ширина и Высота на вкладке Формат.
9. Сравните эти значения с уже записанными значениями из базы данных изделий.
10. Закройте диалоговое окно "Свойства".
11. Снова выделите в диалоговом окне управления сообщениями строки с сообщениями и откройте с помощью пункта всплывающего меню Свойства базу данных изделий.
12. Перейдите на вкладку Монтажные данные для выделенных изделий и введите в поля Высота и Ширина размеры из макроса.
13. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
14. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Обратите внимание, что размеры в двухмерном макросе проверяются только для варианта "А".

# P022050: GUID (AutomationML) '\<x\>' неоднозначен

### Причина

В проекте несколько главных функций имеют идентификатор GUID с одинаковым значением \<x\>. Для экспорта проекта в AutomationML каждая главная функция должна иметь однозначный GUID.

### Решение

1. Установите местоположение главных функций с многократно заданным в проекте идентификатором GUID с помощью функции Перейти к (графика) или Синхронизировать выбор во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно Свойства для каждой главной функции.
3. В таблице свойств в поле GUID (AutomationML) задайте однозначное значение.
4. Щелкните по кнопке ++OK++.
5. Затем запустите новый контрольный прогон.

# P005057: Точка определения сети с неверным потенциалом

### Причина

На одном соединении была размещена точка определения сети и снабжена именем потенциала. Однако этот потенциал не определен ни в одном месте схемы соединений.

### Решение

#### Возможность 1

Введите в точке определения сети имя уже существующего определения потенциала.

#### Возможность 2

Определите новый потенциал в схеме соединений, например, с помощью вывода потенциала. Присвойте этому выводу устройства имя потенциала точки определения сети.

Обновите соединения и затем повторно запустите контрольный прогон.

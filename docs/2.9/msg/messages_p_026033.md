# P026033: Объединенные провода, распределенные на несколько групп

### Причина

Соединение присвоено объединенным проводам, которые уже были присвоены другой группе объединенных проводов.

### Решение

1. Установите местоположение соединения в пространстве листа, используя функцию Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Дважды щелкните отображаемое соединение, чтобы открыть диалоговое окно Свойства.
3. Присвойте соединение правильным объединенным проводам и правильной группе объединенных проводов, для чего введите в групповом поле Свойства действительные обозначения в поля Объединенные провода и Группа объединенных проводов. При присвоении обозначений учитывайте, что хотя одна и та же группа объединенных проводов может содержаться в разных объединенных проводах, одни и те же объединенные провода ***не являются*** составной частью нескольких групп объединенных проводов.
4. При необходимости повторите обозначение объединенных проводов и групп объединенных проводов для других соединений.
5. Затем запустите новый контрольный прогон.

# P005065: Соединения имеют противоречивые данные в свойстве '\<x\>'

### Причина

Соединение, присутствующее в проекте в нескольких типах представления, содержит противоречивые данные в отношении свойства \<x\>. Например, соединению в многополюсном представлении был присвоен цвет соединения 'RD', т. е. 'красный', в однополюсном представлении, напротив, был присвоен цвет соединения 'BU', т. е. 'синий'.

### Решение

1. Определите местоположение соединения в соответствующем виде представления с помощью функциональности Перейти к (графика) и Перейти ко 2-й координате во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно Свойства ++...++ соединения в каждом виде представления и синхронизируйте соответствующим образом значение для свойства \<x\>, указанного в сообщении.
3. Затем запустите новый контрольный прогон.

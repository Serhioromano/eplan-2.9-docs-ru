# P005055: Несколько определений сигнала на один сигнал

### Причина

Вы разместили на одном сигнале несколько определений сигнала или точек определения сети с различными именами сигнала.

### Решение

#### Возможность 1

Сократите число определений сигнала и/или точек определения сети с записанным именем сигнала, оставив только одно действующее определение сигнала.

#### Возможность 2

Присвойте всем определениям сигнала одно и то же имя сигнала.

Затем запустите новый контрольный прогон.

!!! tip "Совет:"

    Через Вид > Отслеживание сигнала можно выделить путь сигнала цветом.

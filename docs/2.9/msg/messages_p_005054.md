# P005054: Разные сети с одинаковым именем сети (\<x\>)

### Причина

Одно и то же имя было обнаружено у нескольких сетей.

#### Возможность 1

Сети не связаны между собой, но имеют точки определения сети с одним и тем же именем сети.

!!! example "Пример:"

    Две сети не связаны между собой, но носят одно и то же имяN1(обозначены кружками на следующем рисунке).

#### Возможность 2

Каждая из двух одноименных сетей имеет вывод потенциала. Выводы потенциала имеют различные имена потенциалов.

!!! example "Пример:"

    Две сети имеют одно и то же имяN1, и каждая из них связана с одним выводом потенциала. Вывод потенциала на первой сети (вверху) имеет имя потенциалаL1, вывод потенциала на второй сети (внизу) —L2(обозначение кружками на следующем рисунке).

### Решение

#### Для возможности 1

1. Соедините обе сети при помощи соединительных символов, например тройников.
2. Удалите одну из точек определения сети.
3. Расположите выводы устройства в диалоговом окне Свойства ++...++ оставшейся точки определения сети (Вкладка Соединения сети) в желаемом порядке в шлейфе.
4. Закройте диалоговое окно "Свойства", нажав ++OK++, и повторно запустите контрольный прогон.

#### Для возможности 2

1. Присвойте обоим выводам потенциала одно и то же имя потенциала.
2. Удалите одну из точек определения сети.
3. Расположите выводы устройства в диалоговом окне Свойства ++...++ оставшейся точки определения сети (Вкладка Соединения сети) в желаемом порядке в шлейфе.
4. Закройте диалоговое окно "Свойства", нажав ++OK++, и повторно запустите контрольный прогон.

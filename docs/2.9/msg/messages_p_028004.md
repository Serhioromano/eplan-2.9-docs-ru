# P028004: Объект планирования имеет несколько изделий с потенциальными главными функциями

### Причина

Одному объекту планирования присвоено несколько изделий. Из-за этого среди шаблонов функций находится несколько потенциальных главных функций. Потенциальные главные функции — это функции, в определениях которых установлен атрибут "Главная функция". Это назначение нельзя изменить.

У одного объекта планирования может быть только одна потенциальная главная функция.

### Решение

1. Установите местоположение соответствующего объекта планирования в проекте посредством функциональности Синхронизировать выбор, которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите в ***навигаторе предварительного планирования*** диалоговое окно Свойства ++...++ объекта планирования и выведите вкладку Изделие на передний план.
3. Удалите все лишние изделия, чтобы у объекта планирования осталось только одно изделие с однозначной потенциальной главной функцией.
4. Закройте диалоговое окно "Свойства", нажав ++OK++, и повторно запустите контрольный прогон.

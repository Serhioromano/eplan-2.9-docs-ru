# P004094: Многократно заданное имя ЦПУ '\<x\>'

### Причина

В одном проекте конфигураций и одной рабочей станции несколько раз используется одно и то же имя ЦПУ.

### Решение

1. Установите местоположение блока ПЛК ЦПУ на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно свойств блока ПЛК.
3. Переместите вкладку Данные структуры ПЛК на передний план.
4. Измените в списке свойств значение в поле ЦПУ: Имя [1].
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

# P004050: Соединение от '\<x\>' до '\<y\>' отсутствует

### Причина

При импорте данных ПЛК было найдено отдельное соединение от устройства \<x\> к устройству \<y\>, которое в данный момент не найдено в проекте.

### Решение

Установите соединение между устройствами с помощью выводов шины.

!!! note "Замечание:"

    Это относящееся к модулю сообщение выводится при импорте данных ПЛК. При выполнении нового контрольного прогона (любой настроенной схемы) его можно удалить из диалогового окна управления сообщениями.

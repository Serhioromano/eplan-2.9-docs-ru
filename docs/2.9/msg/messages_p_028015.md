# P028015: Сегмент с определением сегмента '\<x\>', которое не должно больше использоваться

### Причина

В проекте существует определение сегмента \<x\>, которому была присвоена настройка "Больше не использовать". Либо вы как авторизованный пользователь сами установили эту настройку, либо этот сегмент вставлен через макрос.

### Решение

1. Установите местоположение определения сегмента в навигаторе предварительного планирования с помощью функции Синхронизировать выбор, которая находится во всплывающем меню диалогового окна Управление сообщениями.
2. Как авторизованный пользователь откройте определение сегмента для использования или выберите для сегмента другое определение.
3. Затем запустите новый контрольный прогон.

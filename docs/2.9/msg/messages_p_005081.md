# P005081: Нереалистичное соединение трассы на распределителе

### Причина

Вы разместили внутри соединения распределитель и на нем настроили для определенных выводов устройства свойство "Сплошное соединение между выводами устройства". В отношении определения функции соединения для указанных выводов устройства сплошное соединение нереалистично. Это касается определений функции "Шланг", "Световод" и определений функции для всех видов перемычек.

### Решение

1. Установите местоположение распределителя в проекте, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно Свойства ++...++ распределителя и измените обозначения выводов устройства в поле Сплошное соединение между выводами устройства.

Или удалите размещенный распределитель, или измените определение функции соединения, на котором был размещен распределитель.
3. Обновите соединения и затем повторно запустите контрольный прогон.

# P005012: Несколько точек определения соединения на одном соединении

### Причина

На одном соединении находится несколько точек определения соединения.

### Решение

Удалите лишние точки определения соединения, чтобы для соответствующего соединения осталась только одна точка определения соединения.

Обновите соединения и затем повторно запустите контрольный прогон.

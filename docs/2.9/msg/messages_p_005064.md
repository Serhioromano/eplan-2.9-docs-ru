# P005064: Недопустимое соединение (вывод устройства источника/цели отсутствует)

### Причина

Источник или цель соединения, для которых в проекте существует несколько видов представления, не представлены в типе представления. Например, цель маршрутизируемого соединения типа 'Трехмерный чертеж монтажных поверхностей', существующего на схеме соединений в качестве многополюсного соединения, удалена с пространства листа.

### Решение

1. Установите местоположение данного соединения, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Произведите поиск недостающего источника или цели через существующее соединение.
3. Разместите недостающий источник или недостающую цель.
4. Затем запустите новый контрольный прогон.

# P004027: Обозначение канала отсутствует или указано неверно

### Причина

На выводе устройства ПЛК, который является дискретным или аналоговым входом или выходом, обнаружено отсутствующее или неправильное обозначение канала.

Сообщение выводится, если для свойства Обозначение канала вывода устройства ПЛК выполнено одно из следующих условий:

* Обозначение канала не задано.
* Обозначение канала состоит из буквенно-цифровых последовательностей.
* В качестве обозначения канала введена цифровая последовательность, значение которой меньше нуля.

Выводы устройства ПЛК с определением функции "Вывод устройства ПЛК, внутр., общ." не проверяются этим контрольным прогоном, поскольку они служат только для отображения свободных символических адресов.

### Решение

1. Установите местоположение соответствующего вывода устройства ПЛК на схеме соединений или в обзоре с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно "Свойства" вывода устройства ПЛК.
3. В поле Обозначение канала вкладки Вывод устройства ПЛК задайте значение, которое будет состоять из цифровой последовательности и превышать ноль.
4. Затем запустите новый контрольный прогон.

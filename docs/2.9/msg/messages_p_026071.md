# P026071: Отсутствует инструмент для экспорта (ЧУ, медь). \<x\>

### Причина

Нужный инструмент отсутствует в текущем перечне инструментов машины для экспорта ЧУ, медь. Поэтому монтажное отверстие не удается экспортировать.

### Решение

1. Установите местоположение данного монтажного отверстия в пространстве листа, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Дополните перечень инструментов или скорректируйте монтажное отверстие.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

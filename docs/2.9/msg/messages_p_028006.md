# P028006: Полное ОУ функции не соответствует предварительному планированию

### Причина

Полное ОУ функции не соответствует определенной структуре идентификаторов объекта планирования. Выполняется проверка, соответствуют ли части полного обозначения устройства, которые указаны через объект планирования (и структуру, в которую загружен объект планирования) частям функции. Таким образом, если, например, в предварительном планировании определено только обозначение установки, то сравниваются также только обозначения установки данной функции.

### Решение

1. Установите местоположение функции в навигаторе предварительного планирования с помощью функции Синхронизировать выбор, которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Переименуйте ОУ функции, снова присвойте его функции или повторно пронумеруйте ОУ с учетом заданной структуры предварительного планирования.
3. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Если в предварительном планировании используется блок идентификаторов "Продукт", а используемый в нем структурный идентификатор содержит вопросительный знак, то с помощью этого контрольного прогона будет проверено также буквенное обозначение.

# P026053: Вывод устройства '\<x\>' отсутствует и не может быть создан

### Причина

Несколько размещений изделий имеют одинаковое ОУ, но на схемах соединений, присвоенных соответствующим изделиям, требуемое обозначение вывода устройства \<x\> отсутствует.

### Решение

1. Добавьте недостающее обозначение вывода устройства на одну из присвоенных схем соединений.
2. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

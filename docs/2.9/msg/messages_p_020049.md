# P020049: Сегмент предварительного планирования \<x\>: \<y\>

### Описание

Выполнено сравнение проектов, при котором ***Свойства*** проверяемого проекта (проекта с замечаниями) были сравнены со свойствами проекта для сравнения и были определены различия.

Это сообщение указывает, что в отношении сегмента предварительного планирования отображается различие. \<x\> указывает обозначение сегмента предварительного планирования, \<y\> говорит о том, было ли обозначение удалено, добавлено или было изменено одно из свойств сегмента предварительного планирования.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

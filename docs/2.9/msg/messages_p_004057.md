# P004057: Найдены несколько изделий с одинаковым или подходящим обозначением типа ПЛК '\<x\>'! Если это возможно, функции '\<y\>' следует присвоить другое изделие

### Причина

При импорте данных ПЛК выполняется поиск подходящего изделия в базе данных изделий по обозначению типа ПЛК. В базе данных изделий могут быть найдены два или несколько изделий с идентичным обозначением типа. При этом обозначение типа ПЛК также может содержать заполнители. При обнаружении нескольких изделий будет использоваться то, которое было найдено первым.

В заполнителе \<x\> сообщения отображается многократно найденное изделие, а в заполнителе \<y\> — ОУ соответствующего изделия.

### Решение

1. Если было найдено несколько изделий с идентичным обозначением типа ПЛК, следует откорректировать изделия в базе данных изделий. Если было найдено изделие с заполнителями, следует проверить, указано ли правильное изделие на устройстве. При необходимости выберите другое изделие.
2. Запустите контрольный прогон (по произвольной схеме), чтобы удалить относящееся к модулю сообщение из диалогового окна управления сообщениями.

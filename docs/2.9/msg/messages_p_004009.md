# P004009: Многократно заданный разъем '\<x\>' каркаса '\<y\>'

### Причина

В нескольких блоках ПЛК, содержащих данные шины, при ***одинаковых*** введенных данных каркаса и проекта конфигураций дополнительно указаны ***одинаковые*** позиции разъема.

### Решение

1. Откройте диалоговое окно Свойства ++...++ для соответствующих блоков ПЛК.
2. Введите разные данные на вкладке Данные структуры ПЛК в соответствующих полях На позиции / разъем.
3. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

* Этот контрольный прогон служит только для проверки блоков ПЛК, на которых указан проект конфигураций.
* Во время этой проверки учитывается также свойство Карта ПЛК вставлена в первичную станцию. Сообщение не выводится, если из двух карт ПЛК с одинаковым номером разъема одна вставлена в каркас, а другая — в первичную станцию. Свойство Карта ПЛК вставлена в каркас означает боковой монтаж, свойство Карта ПЛК вставлена в первичную станцию означает монтаж сверху.

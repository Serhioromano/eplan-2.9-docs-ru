# P013054: Изделие с индексом '\<x\>': Перед импортом спецификации нельзя модифицировать свойство '\<y\>'

### Причина

При импорте спецификации была попытка изменения свойства ссылки изделия в файле импорта, например свойства Номер для заказа. Это свойство невозможно изменить.

### Решение

Отмените изменение и заново импортируйте спецификацию.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

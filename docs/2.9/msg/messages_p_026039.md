# P026039: Соединение проходит через недействительные сегменты маршрутизации

### Причина

После маршрутизации соединения был удален объект сети соединенных сегментов (например кабельный канал, сегмент маршрутизации или кривая), который перед этим проходил через соединение.

### Решение

1. Выберите пункты меню Пространство листа > Маршрутизировать соединения > Генерировать сеть соединенных сегментов, чтобы обновить сеть соединенных сегментов.
2. Еще раз выполните маршрутизацию соединений.
3. Затем запустите новый контрольный прогон.

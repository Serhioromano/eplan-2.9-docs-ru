# P022001: Библиотека символов '\<x\>' на позиции \<y\> не найдена

### Причина

В исходном проекте был создан макрос, после чего он был вставлен в целевой проект (текущий проект). Макрос содержит информацию о сохраненной в исходном проекте библиотеке символов \<x\>, из которой были выбраны символы для макроса. Кроме того, в нем содержатся данные о сохраненной в настройках исходного проекта позиции \<y\> сохраненного проекта. Эта информация сравнивается при вставке макроса с соответствующими настройками целевого проекта. В данном конкретном случае в целевом проекте эта библиотека символов \<x\> не определяется в позиции \<y\>, так как она там не сохранена.

### Решение

Сохраните искомую библиотеку символов в требуемой позиции в текущем проекте. Выберите пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Управление > Библиотеки символов. Выделите в диалоговом окне Настройки: Библиотеки символов в поле Библиотеки символов позицию (строка) \<y\>. Щелкните в поле по ++...++. Выберите в следующем диалоговом окне отсутствующую библиотеку символов. Щелкните по кнопке ++Открыть++.

!!! info "Для сведения:"

    Библиотека символов сохраняется на требуемой позиции в целевом проекте.

Удалите макрос в целевом проекте и вставьте его снова.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

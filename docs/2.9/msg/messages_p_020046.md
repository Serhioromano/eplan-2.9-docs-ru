# P020046: Сравнение структуры проектов: Текущая дата и время: \<x\>

### Описание

Выполнено сравнение проектов, при котором ***Структура проекта*** проверяемого проекта (проект с замечаниями) сравнивалась со структурой проекта для сравнения.

Это сообщение информирует при помощи \<x\> о текущей дате и времени выполнения сравнения.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

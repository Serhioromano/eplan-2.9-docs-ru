# P027007: Больше одной функции топологии в устройстве \<x\>

### Причина

В сети соединенных сегментов устройству \<x\> присвоено более одной функции топологии с одинаковым обозначением устройства. Каждая функция топологии может существовать только один раз в сети соединенных сегментов с видом представления "Топология".

!!! note "Замечание:"

1. Сообщение выдается также, если многократно встречающиеся точки маршрутизации или сегменты маршрутизации размещены с одинаковым идентификатором устройства.
2. Определения клеммников составляют исключение. Допускается их многократное существование и они не проверяются при многократном размещении.

### Решение

1. Установите местоположение многократно возникающей функции топологии устройства в Вашей сети соединенных сегментов, пользуясь функций Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Переименуйте многократно возникающие функции топологии устройства или удалите их, так что останется только одна функция топологии с однозначным обозначением устройства.
3. Затем запустите новый контрольный прогон.

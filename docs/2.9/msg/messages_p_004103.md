# P004103: Подчиненный и главный шины отличаются в свойстве 'Физическая сеть: Имя'

### Причина

В проекте конфигураций подчиненная и главная шины, которые относятся к одной и той же логической сети, отличаются по свойству Физическая сеть: Имя.

### Решение

1. Установите местоположение выводов подчиненной и главной шин на схеме соединений с помощью функции Перейти к (графика) и Перейти ко 2-й координате во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно "Свойства" выводов шин и проверьте на вкладке Данные шины значение, внесенное в поле Физическая сеть: Имя.
3. Скопируйте значение из поля Физическая сеть: Имя на выводе главной шины, вставьте его в одноименное поле на выводах подчиненной шины и подтвердите ввод.
Вместо изменения вручную можно выполнить корректировку проекта с помощью настройки Синхронизировать имена физических сетей.
4. Затем запустите новый контрольный прогон.

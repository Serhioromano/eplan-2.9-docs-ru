# P008004: Невозможно перевести текст '\<x\>'

### Причина

Ключевые слова, необходимые для перевода текста '\<x\>', отсутствуют в словаре.

### Решение

1. Дополните отсутствующие тексты в словаре как новые [ключевые слова](../translatedbgui_h_stichwortanlegen.md). При переводе обратите внимание на [специфические для проекта настройки](../translategui_r_einstellungprojekt.md), относящиеся к сегментированию.
2. Затем запустите новый контрольный прогон.

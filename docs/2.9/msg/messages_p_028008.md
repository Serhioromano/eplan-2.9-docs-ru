# P028008: Объект планирования имеет шаблон функции, к которому отсутствуют функции

### Причина

Объект планирования имеет как минимум один свободный шаблон функции, которому не присвоено подробное планирование. Т. е. шаблон функции, присвоенный объекту планирования, еще не размещен в схеме соединений в качестве функции.

### Решение

Создайте подробное планирование.

1. Установите местоположение шаблона функции и соответствующего объекта планирования в навигаторе предварительного планирования с помощью функции Синхронизировать выбор, которая находится во всплывающем меню в диалоговом окне Управление сообщениями.
2. Отметьте шаблон функции в навигаторе предварительного планирования и разместите его с помощью всплывающего меню Разместить на странице схемы соединений.
3. Затем запустите новый контрольный прогон.

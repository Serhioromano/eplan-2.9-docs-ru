# P003007: Несколько целей кабеля

### Причина

Выполнена нумерация кабеля, у которого более, чем одна цель. При этом выбран формат нумерации "По источнику и цели", который выводит обозначение устройства цели как счетчика в обозначение устройства кабеля, например -W(=ANL+EBO-X5/=ANL+EBO-H3). Так как имеется более, чем одна цель кабеля, однако можно вывести только обозначение цели, произошел конфликт.

!!! note "Замечание:"

    К настройкам нумерации "По источнику и цели" перейдите следующим образом.

    Выделите кабель в схеме соединений. Выберите пункты меню Данные проекта > Кабели > Нумерация. Щелкните в диалоговом окне Нумеровать кабели рядом с полем Настройки по кнопке ++...++. Откройте в диалоговом окне Настройки: Нумерация кабелей раскрывающийся список Формат.

### Решение

Присвойте кабелю только одну цель кабеля. После этого пронумеруйте кабель заново с помощью формата "По цели и источнику".

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

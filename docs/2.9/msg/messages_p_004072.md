# P004072: Многократно заданный адрес шины '\<x\>' физической сети '\<y\>'

### Причина

На нескольких выводах шины системы шин в ***физической сети*** проекта конфигураций заданы ***одинаковые*** адреса шин. Использование этого контрольного прогона целесообразно, если адреса шин должны быть уникальными во всей физической сети.

Этот контрольный прогон служит только для проверки выводов шины, на которых указан проект конфигураций.

### Решение

1. Установите местоположение выводов шины на схеме соединений с помощью функции Перейти к (графика) и Перейти ко 2-й координате во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно Свойства ++...++ для соответствующих выводов шины. Введите разные данные на вкладке Данные шины в соответствующих полях Физическая сеть: Адрес шины / номер позиции или Физическая сеть: Адрес шины / номер позиции 2 с учетом физической сети, выведенной в заполнителе \<y\>.
3. Либо измените запись в поле Физическая сеть: Имя на одном из выводов шины, чтобы присвоить этот вывод шины другой физической сети.
4. Затем запустите новый контрольный прогон.

# P026021: Разные места монтажа (описывающие) монтажной поверхности и главной функции / главной клеммы

### Причина

В полях "Свойства" Место монтажа (описыв.) главной функции/главной клеммы размещения изделия на схеме соединений и соответствующей монтажной платы в пространстве листа были введены различные значения.

### Решение

1. Укажите как для главной функции/главной клеммы размещения изделия на схеме соединений, так и для соответствующей монтажной поверхности в пространстве листа в диалоговом окне "Свойства" Место монтажа (описывающее) одинаковые данные.
2. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Контрольный прогон выводится также, если в обоих вышеназванных полях свойств присутствуют разные статусы перевода, т. е. если свойство главной функции / главной клеммы было задано на текущем языке интерфейса, свойство монтажной поверхности также имеется на другом языке.

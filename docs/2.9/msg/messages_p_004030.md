# P004030: Недостающий символический адрес (автоматич.)

### Причина

Не указан символический адрес для одного из выводов устройства ПЛК. У вывода устройства нет подсоединенной цели, которая вместе с ОУ может автоматически образовать символический адрес.

### Решение

#### Возможность 1

Откройте диалоговое окно Свойства ++...++ для соответствующего вывода устройства ПЛК. Внесите на вкладку Вывод устройства ПЛК, в поле Символический адрес, символический адрес.

Затем запустите новый контрольный прогон.

#### Возможность 2

Соедините вывод устройства ПЛК с датчиком или исполнительным элементом, имеющим обозначение устройства.

С помощью отслеживания цели EPLAN автоматически определяет подсоединенное ОУ и генерирует символический адрес.

Затем запустите новый контрольный прогон.

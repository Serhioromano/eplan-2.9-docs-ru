# P024001: Свойство 'Напорная гидролиния / блок управления' не подходит к свойству 'Подвод давления / управляющий вывод устройства' на одном из выводов устройства

### Причина

Целевые функции, имеющиеся на соединении, противоречат друг другу на выводах устройства функций с точки зрения настроенного свойства Рабочая линия/блок управления. Например, для источника соединения установлена "Рабочая линия", а для цели — "Трубопровод для слива масла".

### Решение

Присвойте источнику и цели соединения на выводах устройства функций совпадающие данные с точки зрения свойства Рабочая линия/блок управления. Это свойство настраивается в диалоговом окне Логическая схема выводов устройства соответствующей функции (запустите диалоговое окно Свойства ++...++, выберите вкладку Данные символа / функции и нажмите ++Логическая схема++).

Обновите соединения и затем повторно запустите контрольный прогон.

# P028012: Имеется более чем одно графическое размещение сегмента структуры

### Причина

На одной или нескольких страницах предварительного планирования вашего проекта имеется более чем одно графическое размещение сегмента структуры.

### Решение

1. Установите местоположение графически многократно размещенного сегмента структуры, пользуясь функцией Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Удалите избыточное графическое размещение.
3. Повторите операцию удаления для других избыточных размещений.
4. Затем запустите новый контрольный прогон.

# P007015: Находящийся при первом изделии макрос не соответствует макросу, которому принадлежит главная функция

### Причина

Это сообщение выводится в связи с рамками макроса, если выполнены следующие условия:

* Макрос, ограниченный с помощью рамки макроса, содержит несколько функций, присвоенных рамке макроса.
* В рамке макроса как минимум для одного свойства имеется присвоение данных изделия.
* Для главной функции на первой позиции изделия было выбрано изделие, которому присвоен макрос. Название макроса изделия и рамки макроса различаются.

### Решение

Присвойте изделие главной функции в рамке макроса на первой позиции изделия с тем же именем макроса, что и в рамке макроса. Оба имени макроса должны совпадать.

Затем запустите новый контрольный прогон.

# P012002: Размещение изделия накладывается на заблокированную область

### Причина

Одно из размещений изделий на монтажной плате накладывается минимум на одну заблокированную область.

### Решение

Переместите размещение изделия таким образом, чтобы оно не находилось на заблокированной области (областях).

Затем запустите новый контрольный прогон.

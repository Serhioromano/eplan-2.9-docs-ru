# P501033: Идентифицированная информация вывода устройства, относящаяся к шаблонам функций, не совпадает с данными вывода устройства на схеме соединений

### Причина

В проверяемой базе данных изделий обнаружено изделие, идентифицирующая информация вывода устройства шаблонов функции которого не соответствует данным вывода устройства на схеме соединений.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Выведите на передний план вкладку Шаблоны функции.
3. Запомните введенные в столбце Обозначения выводов устройства значения.
4. Переместите вкладку Технические данные на передний план.
5. Щелкните в групповом поле Схема соединений рядом с полем Имя на ++...++.
6. Перейдите на вкладку Выводы устройства и сравните сохраненные там обозначения выводов устройства с обозначениями выводов устройства на шаблонах функции изделия.
7. Исправьте обозначения выводов устройства на схеме соединений. Обратите внимание, что, в отличие от шаблонов функций, каждое значение вывода на схеме соединений должно стоять в отдельной строке.
8. Сохраните измененные данные и выполните синхронизацию данных изделий.
9. Затем запустите новый контрольный прогон.

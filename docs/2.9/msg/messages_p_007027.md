# P007027: Логическая модель функции содержит недействительные ссылки выводов устройства на обозначения выводов устройства: \<x\>

### Причина

В случае с выводами устройства и выводами устройства ПЛК возможна передача логической информации другим функциям с одинаковым ОУ. Это касается свойств Передать потенциал в, Потребитель / источник в и Отслеживание цели (ПЛК) в. В конкретном случае в логической модели указанной функции на выводе устройства функции в качестве ссылки на вывод устройства было внесено недействительное обозначение вывода устройства.

!!! example "Пример:"

    В черном ящике потенциал должен передаваться от вывода устройства с обозначением "1" на вывод устройства с обозначением "4". Поэтому в логической модели функции для вывода устройства функции "1" на свойстве Передать потенциал ввнесено значение"4". Однако вывод устройства с таким обозначением отсутствует на схеме соединений.

### Решение

1. Установите местоположение устройства или вывода устройства ПЛК на схеме соединений при помощи функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно свойств функции.
3. Выведите на передний план вкладку Данные символа / функции.
4. Щелкните кнопку ++Логика++.
5. В диалоговом окне Логическая схема выводов устройства в столбце для соответствующего вывода устройства функции исправьте ссылку на вывод устройства для свойств Передать потенциал в, Потребитель / источник в и / или Отслеживание цели (ПЛК) в.
6. Щелкните по кнопке ++OK++.
7. Затем запустите новый контрольный прогон.

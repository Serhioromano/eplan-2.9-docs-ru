# P002006: Имеется только сторона гнезд штекеров

!!! note "Замечание:"

    Ответным элементом штыря штекера является либо гнездо розетки с таким же ОУ и обозначением контакта штекера, либо гнездо розетки, соединенное со штырем штекера через прямые выводы устройства.

### Причина

В схеме соединений размещено определение штекера, для которого настроено определение функции "Определение штекера для гнезд". Согласно этой настройке программа выполняет поиск соответствующего ответного элемента, то есть стороны штырей штекеров. Однако этот ответный элемент отсутствует в схеме соединений.

### Решение

Вставьте в схеме соединений другое определение штекера и настройте в качестве определения функции "Определение штекера для штырей". Кроме того, введите для данного определения штекера то же ОУ, что и имеющееся для определения штекера для гнезд.

Обновите соединения и затем повторно запустите контрольный прогон.

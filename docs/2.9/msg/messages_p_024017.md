# P024017: У функции вида представления 'Обзор функций (Fluid-Техника)' нет соответствия на схеме соединений

### Причина

На странице типа "Обзор функций (Fluid-Техника)" размещена функция с видом представления "Обзор функций (Fluid-Техника)". Для функции отсутствует эквивалент вида представления "Многополюсный" на соответствующей многополюсной схеме соединений.

### Решение

1. Определите из диалогового окна Управление сообщениями функцию на странице типа "Обзор функций (Fluid-Техника)" с помощью функции Перейти к (графика).
2. Вызовите диалоговое окно Свойства ++...++ функции и убедитесь, что функция не является главной функцией.
3. Скопируйте функцию и перейдите на соответствующую страницу типа "Схема соединений (Fluid-Техника)".
4. Внесите функцию на эту страницу в качестве главной функции, задайте такое же ОУ и присвойте функции вид представления "Многополюсный" на вкладке Данные символа / функции.
5. Подтвердите ввод.
6. Затем запустите новый контрольный прогон.

# P012001: Размещению изделия не присвоено изделие

### Причина

Отсутствует номер изделия у размещения изделия на монтажной плате.

### Решение

Дважды щелкните по соответствующему размещению изделия на монтажной плате. Выберите в диалоговом окне Свойства ++...++ размещения изделия отсутствующий номер в поле Изделие.

Затем запустите новый контрольный прогон.

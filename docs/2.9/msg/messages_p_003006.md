# P003006: Кабель между одним и тем же функциональным элементом

### Причина

Исходная и целевая функция кабеля имеют идентичные обозначения устройства и связаны перекрестной ссылкой.

### Решение

Введите разные обозначения устройства для функции источника и функции цели подсоединенного кабеля.

Обновите соединения и затем повторно запустите контрольный прогон.

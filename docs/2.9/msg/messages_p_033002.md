# P033002: Соединение содержит недействительное имя проводного монтажа

### Причина

На схеме соединений было определено соединение, которому было присвоено имя проводного монтажа, не имеющее определения проводного монтажа с идентичным именем проводного монтажа. Вероятно, соединение было скопировано из схемы соединений другого проекта и вставлено в открытый на данный момент проект, или имя проводного монтажа определений проводного монтажа было переименовано в дальнейшем.

### Решение

Если соединению не присвоено определение проводного монтажа, удалите имя проводного монтажа на соединении. Или присвойте соединению существующее имя проводного монтажа, указанное на определении проводного монтажа.

1. Установите местоположение точки определения группы соединений соединения на схеме соединений при помощи функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно свойств точки определения соединения.
3. В списке свойств первой вкладки удалите значение свойства Имя проводного монтажа. Или при помощи кнопки ++...++ в поле значений свойства выберите существующее имя проводного монтажа определения проводного монтажа.
4. Щелкните по кнопке ++OK++.
5. Затем запустите новый контрольный прогон.

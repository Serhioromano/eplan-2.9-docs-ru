# P501017: В изделии не выбран раздел

### Причина

В проверяемой базе данных изделий было найдено изделие, на котором не указан раздел.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Переместите вкладку Общее на передний план.
3. Щелкните рядом с полем Раздел / Подраздел по ++...++.
4. Установите в диалоговом окне Раздел / Подраздел для нужного раздела и / или подраздела соответствующий флажок или несколько флажков в столбце Использовать.
5. Щелкните по кнопке ++OK++.
6. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
7. Затем запустите новый контрольный прогон.

# P012009: Изделие размещения изделия присутствует на главной функции в недостаточном количестве

### Причина

#### Возможность 1

Размещение изделия на странице типа "Компоновка электрошкафа" имеет определенный номер изделия. Соответствующей соединяемой с размещением изделия главной функции на странице схемы соединений присвоен другой номер изделия или этот номер отсутствует.

#### Возможность 2

Размещение изделия с определенным номером изделия было несколько раз вставлено на странице типа "Компоновка электрошкафа". Соответствующая соединяемая с размещением изделия главная функция на странице схемы соединений имеет тот же номер изделия, но на изделии в свойстве Число штук / колич. указано значение, которое меньше вставленного на странице типа "Компоновка электрошкафа" числа размещения изделия.

### Решение

#### Для возможности 1

Введите для размещения изделия на странице типа "Компоновка электрошкафа" и главной функции на схеме соединений одинаковый номер изделия.

Затем запустите новый контрольный прогон.

#### Для возможности 2

Приведите введенное значение для числа штук / количества в свойствах изделия главной функции на схеме соединений в соответствие с числом вставленного на странице типа "Компоновка электрошкафа" размещения изделия. Т. е. либо увеличьте значение числа штук / количества для главной функции, либо удалите избыточное число размещения изделия.

Затем запустите новый контрольный прогон.

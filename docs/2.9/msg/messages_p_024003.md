# P024003: Электротехническая функция характера Fluid как представление парной перекрестной ссылки отсутствует

### Причина

Данное сообщение выводится, если электротехническая функция характера Fluid-техники ***не*** имеет вида представления "Парная перекрестная ссылка". Например, катушка электромагнита вставлена в качестве органа управления клапана на странице схемы соединений Fluid-техники, а затем вручную был изменен автоматически заданный вид представления "Парная перекрестная ссылка".

К электротехническим функциям характера Fluid, которые включены в проверку, относятся все электротехнические функции категории "Клапаны" и все электротехнические функции категории "Аналоговый датчик", присвоенные физическим величинам "Давление", "Расход", "Уровень заполнения" и "Температура".

### Решение

Присвойте функции в поле Вид представления запись "Парная перекрестная ссылка". Вид представления настраивается на вкладке Данные символа / функции диалогового окна Свойства ++...++.

Обновите соединения и затем повторно запустите контрольный прогон.

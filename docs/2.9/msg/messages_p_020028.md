# P020028: Число сравненных свойств: \<x\>

### Описание

Выполнено сравнение проектов, при котором ***Свойства*** проверяемого проекта (проект с замечаниями) сравнивались со свойствами проекта для сравнения. В этом сообщении при помощи \<x\> указано число свойств, для которых выполнено сравнение.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

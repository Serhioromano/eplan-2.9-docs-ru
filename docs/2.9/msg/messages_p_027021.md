# P027021: Нет маршрутизации всеполярного соединения от \<x\> до \<y\>

### Причина

В проекте существует многополюсное соединение, проходящее между многополюсными функциями с обозначениями устройств \<x\> и \<y\>. Кроме того, в проекте находятся две функции топологии с обозначениями устройств \<x\> и \<y\>, которые, однако, не связаны друг с другом при помощи маршрутизируемого соединения.

### Решение

1. Выберите пункты меню Поиск > Поиск, и ищите в одноименном диалоговом окне по всему проекту оба обозначения устройств \<x\> и \<y\>. Пусть результат поиска будет показан в списке результатов.
2. В диалоговом окне Результаты поиска — <Имя проекта определите местоположение обеих функций топологии \<x\> и \<y\>, выбрав из всплывающего меню функциональность Перейти к (графика).
3. Добавьте к обеим функциям топологии сегменты и точки маршрутизации при помощи пунктов меню Вставить > Топология > Сегмент маршрутизации и / или Вставить > Топология > Точка маршрутизации.
4. Соедините обе функции топологии и точки маршрутизации с сегментами маршрутизации, и задайте таким образом трассу маршрутизации через сеть соединенных сегментов.
5. Выделите всю сеть соединенных сегментов и выберите Данные проекта > Топология > Маршрутизировать.

!!! info "Для сведения:"

    Будет сгенерировано маршрутизируемое соединение через сеть соединенных сегментов между обоими функциями топологии.

6. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Если между обеими функциями топологии уже существует сеть соединенных сегментов и ***еще не проложенное маршрутизированное соединение***, проверьте в диалоговом окне Свойства ++...++ для каждого сегмента маршрутизации соединения значение свойства Топология: Фильтр соединений. При обнаружении различий унифицируйте значение для всех сегментов маршрутизации. Затем маршрутизируйте соединения заново и запустите новый контрольный прогон.

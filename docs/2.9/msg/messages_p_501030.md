# P501030: Внесенная на изделии схема сверления '\<x\>' является неполной. Контур '\<y\>' отсутствует

### Причина

В проверяемой базе данных изделий обнаружено изделие со схемой сверления, для который указан контур, не найденный EPLAN.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Выведите на передний план вкладку Изготовление.
3. Щелкните в столбце Схема сверления по записи в таблице и нажмите кнопку ++...++.
4. В открывшемся меню выбора изделий выведите на передний план вкладку Монтажные отверстия.
5. В столбце Имя контура таблицы введите существующий контур или нажмите кнопку ++...++ и выберите контур в диалоговом окне.
6. Щелкните по кнопке ++OK++.
7. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
8. Затем запустите новый контрольный прогон.

# P005087: Прямое соединение: Несоответствующий размер присоединения '\<x\>' / '\<y\>'

### Причина

Размеры присоединения функций, подключенных к прямому соединению, не совпадают. В заполнителе \<x\> отображается размер присоединения подключенной функции, которая является источником прямого соединения, а в заполнителе \<y\> — размер присоединения подключенной функции, которая является целью прямого соединения. Размеры присоединения обеих функций непосредственно сравниваются между собой во время проверки.

### Решение

1. Определите местоположение функций, подключенных к прямому соединению, на схеме соединений с помощью функции Перейти к (графика) и Перейти ко 2-й координате во всплывающем меню диалогового окна Управление сообщениями.
2. По очереди откройте диалоговое окно свойств обеих функций.
3. Унифицируйте значения свойства Размер присоединения (все) на обеих функциях. При этом обращайте внимание на единообразие в написании данных с прописной / строчной буквы.
4. Если размеры присоединения были перенесены на функцию с помощью выбора изделия, выберите соответствующие изделия с подходящими значениями.
5. Затем запустите новый контрольный прогон.

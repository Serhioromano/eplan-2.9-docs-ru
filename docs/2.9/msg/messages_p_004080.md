# P004080: Отсутствует проект конфигураций

### Причина

На блоке ПЛК или выводе шины не указан проект конфигураций. Эта информация требуется для корректного экспорта данных ПЛК из EPLAN.

### Решение

1. Установите местоположение блока ПЛК или вывода шины на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно свойств блока ПЛК или вывода шины.
3. В случае с блоком ПЛК переместите на передний план вкладку Данные структуры ПЛК, а в случае с выводом шины — вкладку Данные шины.
4. В поле Проект конфигураций введите соответствующее имя или выберите запись из раскрывающегося списка.
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

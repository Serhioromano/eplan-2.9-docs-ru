# P005021: Определение потенциала с неверным потенциалом

### Причина

На соединении была размещена точка определения потенциала, и снабжена именем потенциала и именем сигнала. Однако этот потенциал не определен ни в одном месте схемы соединений.

### Решение

#### Возможность 1

Введите в точке определения потенциала имя уже существующего определения потенциала.

#### Возможность 2

Определите новый потенциал в схеме соединений, например, с помощью вывода потенциала. Введите для этого вывода устройства имя точки определения потенциала, для которой было определено имя сигнала.

Обновите соединения и затем повторно запустите контрольный прогон.

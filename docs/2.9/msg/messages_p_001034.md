# P001034: Старая настройка созданной вручную мостовой перемычки: '\<x\>'

### Причина

В проект на странице схемы соединений был добавлен (например, через макрос) клеммник с созданной вручную мостовой перемычкой, спроектированной с помощью версии EPLAN 2.4 или более ранней версии. EPLAN выводит в диалоговом окне управления сообщениями список всех клемм, задействованных в создании этой мостовой перемычки. В заполнителе \<x\> сообщения доступна информация о том, какая настройка созданной вручную мостовой перемычки использовалась до этого, например "Начало перемычки".

!!! note "Замечание:"

    Так как начиная с версии 2.5 работа с созданными вручную мостовыми перемычками была изменена, этот контрольный прогон позволяет установить наличие в текущем проекте созданных вручную мостовых перемычек, спроектированных с помощью более ранних версий EPLAN.

### Решение

1. Установите сначала местоположение соответствующей клеммы на схеме соединений, пользуясь функцией Перейти к (графика), которую вы можете найти во всплывающем меню диалогового окна Управление сообщениями.
2. Выберите пункт всплывающего меню Свойства.
3. В диалоговом окне свойств клеммы вынесите на передний план первую вкладку.
4. Измените настройку "[значение не установлено]" в поле Мостовая перемычка, которое еще может использоваться для автоматического создания мостовых перемычек в версии 2.5 и более поздних версиях. Выберите, например, настройку "Автоматич.".
5. Щелкните по кнопке ++OK++.
6. Повторите изменение этой настройки для всех остальных клемм клеммника, соединенных перемычками.
7. Сгенерируйте заново в диалоге Обработать клеммник (пункты меню Данные проекта > Клеммники > Обработать) требуемую мостовую перемычку, созданную вручную.
8. Затем запустите новый контрольный прогон.

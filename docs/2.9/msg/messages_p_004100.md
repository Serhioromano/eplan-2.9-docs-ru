# P004100: В ЦПУ отсутствует присвоение к разъему и / или каркасу

### Причина

На карте ПЛК, определенной как ***ЦПУ***, отсутствует структурная информация, указанная в тексте сообщения.

### Решение

1. Установите местоположение карты ПЛК на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Переместите вкладку Данные структуры ПЛК на передний план.
3. Затем проверьте, чтобы структурная информация в полях Проект конфигураций и Имя станции была введена полностью.
4. Дополнительно введите значение в поле На позиции / разъем ***или*** введите соответствующие значения в полях Карта ПЛК вставлена в каркас ***и*** Каркас.
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

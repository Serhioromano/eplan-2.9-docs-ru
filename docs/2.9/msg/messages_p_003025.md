# P003025: Число соединений не указано

### Причина

На главной функции кабеля отсутствует указание числа соединений.

### Решение

Определите из диалогового окна Управление сообщениями главную функцию кабеля в схеме соединений с помощью функциональности Перейти к (графика). Щелкните там двойным щелчком по выделенной линии определения кабеля и вызовите диалоговое окно Свойства ++...++ кабеля. Перейдите, если это не настроено предварительно, во вкладку Кабель, и введите значение в поле Число соединений. У вас есть и другая возможность: выбрать изделия и присвоить кабелю соответствующий тип. Для этого щелкните по ++...++ у поля Тип и выберите в диалоговом окне выбора изделий кабель с нужными свойствами.

Щелкните на ++OK++ и повторно запустите контрольный прогон.

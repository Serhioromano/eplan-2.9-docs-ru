# P007028: Идентификатор ОУ '\<x\>' отсутствует на вспомогательной функции

### Причина

На вспомогательной функции, являющейся составной частью макроса для модуля, не указан идентификатор ОУ. При этом на соответствующей главной функции в макросе эта информация внесена.

### Решение

1. Установите местоположение вспомогательной функции на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно "Свойства" вспомогательной функции.
3. Введите отображаемый в сообщении идентификатор ОУ \<x\> главной функции в свойство Идентификатор ОУ вспомогательной функции.
4. Щелкните по кнопке ++OK++.
5. Затем запустите новый контрольный прогон.

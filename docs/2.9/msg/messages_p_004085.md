# P004085: Отсутствует тип данных

### Причина

На выводе устройства ПЛК — аналоговом или дискретном входе или выходе, значение для свойства Тип данных не указано.

### Решение

1. Установите местоположение вывода устройства ПЛК на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно "Свойства" вывода устройства ПЛК.
3. Переместите первую вкладку на передний план.
4. В таблице свойств щелкните в поле значений свойства Тип данных.
5. С помощью кнопки ++...++ выберите требуемый тип данных.
6. Щелкните по кнопке ++OK++.
7. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

* Выводы устройства ПЛК с определением функции ***Вывод устройства ПЛК, многофункциональный*** также учитываются при этом контрольном прогоне, если вид сигнала на логической схеме выводов устройства настроен на ***дискретный вход*** / ***дискретный выход*** или ***аналоговый вход*** / ***аналоговый выход*** соответственно.
* Выводы устройства ПЛК с определением функции ***Вывод устройства ПЛК, общий*** и ***Вывод устройства ПЛК, перемен.*** при этом контрольном прогоне не учитываются.

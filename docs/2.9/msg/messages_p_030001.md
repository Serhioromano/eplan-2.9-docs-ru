# P030001: Оригинальный проект содержит изменения в области обработки частичного проекта '\<x\>'

### Причина

В исходном проекте, в области обработки, в которую вы хотите сохранить частичный проект, были созданы, например, страницы, пространства листа или сегменты предварительного планирования.

!!! note "Замечание:"

    Это сообщение выводится только в исходном проекте.

### Решение

1. Удалите страницы, пространства листа или сегменты предварительного планирования из соответствующей области обработки исходного проекта или переместите их в другие области обработки исходного проекта.
2. Запустите любой контрольный прогон, чтобы удалить сообщение.
3. После этого сохраните частичный проект в нужную область обработки исходного проекта.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

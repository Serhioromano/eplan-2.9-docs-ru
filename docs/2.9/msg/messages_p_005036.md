# P005036: Превышено минимальное число разрядов для формата (\<x\>)

### Причина

Минимальное число разрядов для элемента формата нумерации соединений больше, чем указано в настройках.

### Решение

Чтобы это сообщение не выводилось, увеличьте минимальное число разрядов для соответствующего элемента формата.

Настройка выполняется следующим образом:

1. Выберите пункты меню Данные проекта > Соединения > Нумерация > Настройки.
2. В диалоговом окне Настройки: Нумерация соединений — <Имя проекта> переместите вкладку Обозначение на передний план.
3. Выделите в списке Группы форматов необходимую группу соединений и щелкните ![](../images/all_edit_as.png){: .ui-icon } (Обработать).
4. Выделите в диалоговом окне Нумерация соединений: Форматы необходимый элемент формата в списке Выбранные элементы формата и щелкните ![](../images/all_edit_as.png){: .ui-icon } (Обработать).

!!! info "Для сведения:"

    Откроется диалоговое окно Формат: <Элемент формата, содержащий поле настройки Мин. число разрядов.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

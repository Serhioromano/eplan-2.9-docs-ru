# P026093: Недостающие условия окружающей среды для расчета кондиционирования в электрошкафу

### Причина

На электрошкафу отсутствуют свойства, которые нужны для расчета кондиционирования.

Для расчета кондиционирования должны быть определены и заполнены значениями следующие свойства на электрошкафу:

* Мощность потерь: Ситуация установки
* Мощность потерь: Максимальная температура снаружи
* Мощность потерь: Минимальная температура снаружи
* Мощность потерь: Максимальная температура внутри
* Мощность потерь: Минимальная температура внутри

### Решение

1. Установите местонахождение электрошкафа в пространстве листа посредством функциональности  Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно "Свойства" электрошкафа и проверьте на первой вкладке, все ли перечисленные свойства внесены в таблицу свойств и/или заполнены значениями.
3. Если нет, дополните свойства и/или введите недостающие значения.
4. Затем запустите новый контрольный прогон.

# P001015: Слишком много уровней проволочной перемычки

### Причина

При создании отчета для спецификации клеммника невозможно вывести в выбранной форме это число находящихся друг над другом проволочных перемычек. Причиной является слишком малое настроенное значение для слоев проволочных перемычек в форме спецификации клеммника. Обратите внимание на то, что данный контрольный прогон выполняется только тогда, когда в соответствующей форме число уровней превышает "0".

### Решение

1. Выберите пункты меню Сервисные программы > Основные данные > Форма > Открыть.
2. В диалоговом окне Открыть форму в поле Тип файла выберите тип файла "Спецификация клеммника (*.f13)".
3. Щелкните по нужной форме и откройте ее.

!!! info "Для сведения:"

    Форма отобразится в навигаторе страниц.

4. Выделите форму в навигаторе страниц и выберите пункт всплывающего меню Свойства. Введите в таблицу свойств формы в свойство Перемычки: число слоев большее значение.
5. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

* При контрольном прогоне учитываются как внешние, так и внутренние проволочные перемычки.
* Проверяются только те клеммники, для которых существует определение клеммника.

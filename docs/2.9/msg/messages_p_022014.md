# P022014: Свойство '\<x\>' для определения 'Число / имена страниц на свойство' на странице '\<y\>' является пустым

### Причина

Свойству, выбранному в настройке Число / имена страниц на свойство не присваивается на странице проекта никакого значения.

### Решение

1. Дважды щелкните "Cообщение" в диалоговом окне Управление сообщениями и определите место соответствующей страницы в древовидной структуре навигатора страниц.
2. Откройте через всплывающее меню диалоговое окно Свойства страницы и задайте значение для указанного в тексте сообщения свойства, которое выбрано в настройках для страниц (через Настройки > Проекты > "Имя проекта" > Управление > Страницы > Число / имена страниц на свойство).
3. Выберите пункты меню Страница > Число / имена страниц на свойство.
4. Затем запустите новый контрольный прогон.

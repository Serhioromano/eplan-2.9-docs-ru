# P022015: Последовательность страниц в разделе 'Количество страниц / имена страниц в свойстве' нарушена. Значение настройки: '\<x\>' Страницы: '\<y\>'

### Причина

При расчете таких же значений для свойств страниц Количество страниц в свойстве и Имена станиц в свойстве были обнаружены нарушения последовательности страниц.

### Решение

1. Рассмотрите в диалоговом окне Управление сообщениями распечатку области страницы в этом сообщении.
2. Откройте навигатор страниц и разверните все раскрывающиеся списки структуры дерева.
3. Отметьте в структуре дерева всю область тех страниц, которые выпадают из общей последовательности.
4. Выберите пункт всплывающего меню Нумерация и пронумеруйте страницы, начиная с первой, с шагом "1". (Если используется структура страниц "по установке" и "по месту установки", добавьте нумерацию, ориентированную на структуру.)
5. После нумерации выберите пункты меню Страница > Число / имена страниц на свойство.
6. Затем запустите новый контрольный прогон.

# P007001: Устройство использует больше функций, чем предоставлено соответствующим изделием

### Причина

К устройству, которое размещено как изделие с шаблонами функций (устройство) добавлена еще одна функция, и ей присвоено ОУ устройства. Изделие не предоставляет возможность такого дополнения функции.

### Решение

#### Возможность 1

Выберите изделие с соответствующими шаблонами функций.

#### Возможность 2

Расширьте в базе данных изделий число шаблонов функций для выбранного изделия. Выберите для этого пункты меню Сервисные программы > Изделие > Управление и выберите в диалоговом окне База данных изделий вкладку Шаблоны функций. Щелкните по кнопке ![](../images/all_new_as.png){: .ui-icon } (Создать), расположенной над таблицей. После этого вставляется новая строка. Щелкните по ячейке в первом столбце, а затем нажмите ++...++. Выберите требуемую функцию в диалоговом окне Определения функций, на которую требуется расширить изделие. Подтвердите выбор, нажав кнопку ++OK++, и примените измененную настройку в базе данных изделий. Затем еще раз разместите символ устройства в схеме соединений.

Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Сообщение выводится также в следующей ситуации: Одна из функций в одном из устройств была обозначена как ***Защитная функция***. Однако в выбранном изделии соответствующий шаблон функции ***не*** определен как защитная функция. Чтобы избежать появления сообщения, можно в диалоговом окне ***База данных изделий*** во вкладке Шаблоны функций установить для выбранного изделия флажок на соответствующем шаблоне функции.

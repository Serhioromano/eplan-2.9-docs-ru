# P021002: Объект-заполнитель не содержит ссылки объектов

### Причина

При расширении объекта-заполнителя были случайно удалены прежние ссылки объекта. Из-за этого объект-заполнитель содержит переменные и наборы значений, но не имеет ссылки объекта в таблице на вкладке Присвоение.

### Решение

Чтобы расширить объекты-заполнители, выполните следующие действия: Нажмите клавишу ++Ctrl++ и удерживайте ее нажатой. Щелкните друг за другом по тем объектам в схеме соединений, которые должны стать частями объекта-заполнителя. Отпустите клавишу ++Ctrl++ и щелкните по объекту-заполнителю правой клавишей мыши. Выберите из открывшегося всплывающего меню пункт Присвоить объекты объекту-заполнителю и дважды щелкните по объекту-заполнителю.

!!! info "Для сведения:"

    На вкладке Присвоение диалогового окна Объект-заполнитель в таблице теперь выведены свойства всех выделенных объектов.

Присвойте требуемому свойству объекта переменную с соответствующим значением.

Затем запустите новый контрольный прогон.

# P026078: Диаметр жгута слишком мал, невозможно объединить соединения с выбранным изделием

### Причина

Суммарный диаметр всех маршрутизируемых соединений, которые необходимо объединить при помощи размещенной защиты кабеля / шланга или хомута для кабеля / шланга, меньше, чем минимальный диаметр жгута используемой защиты кабеля / шланга или хомута для кабеля / шланга.

### Решение

1. Установите местонахождение защиты кабеля/шланга или хомута для кабеля/шланга в пространстве листа при помощи функциональности Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно "Свойства" защиты кабеля/шланга или хомута для кабеля/шланга и переместите вкладку Изделие на передний план.
3. Перейдите через используемое изделие в окно "Выбор изделия" и переместите вкладку Данные маршрутизируемой принадлежности на передний план.
4. Проверьте, какое значение настроено на используемом изделии в поле Минимальный диаметр жгута.
5. Выберите защиту кабеля / шланга или хомут для кабеля / шланга, минимальный диаметр жгута которых больше, чем суммарный диаметр всех маршрутизируемых соединений в связке.
6. Закройте выбор изделия и диалоговое окно свойств, нажав ++OK++, и повторно запустите контрольный прогон.

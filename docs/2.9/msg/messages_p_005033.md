# P005033: Прогон обозначения группы форматов №. \<x\>: Первое присвоенное обозначение соединений '\<y\>', последнее присвоенное обозначение соединений '\<z\>' внутри группы форматов

### Описание

Это сообщение выводится для подтверждения после каждого прогона обозначения точек определения соединения. Информация представлена с помощью номера (позиции) в таблице диалогового окна Настройки: Нумерация соединений указанной группы форматов, а также с помощью первого присвоенного и последнего присвоенного обозначения соединения или первого присвоенного и последнего присвоенного имени в рамках группы форматов.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

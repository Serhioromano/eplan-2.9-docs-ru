# P005084: Свойство Вещество '\<x\>' соединения противоречит определению трубопровода

### Причина

На функциональной схеме автоматизации содержится соединение, на котором размещено определение трубопровода (вывод или точка определения трубопровода). Соединение и определение трубопровода имеют разные значения относительно свойства Вещество. Например, для точки определения соединения указано вещество "Вода", а для определения трубопровода — "Воздух".

### Решение

1. Установите местоположение точки определения соединения на функциональной схеме автоматизации с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. С помощью функции Перейти ко 2-й координате всплывающего меню перейдите к определению трубопровода.
3. Найдите значение, внесенное в соответствующем диалоговом окне "Свойства", в поле Вещество.
4. Унифицируйте оба значения или удалите значение на точке определения соединения.
5. Обновите соединения и затем повторно запустите контрольный прогон.

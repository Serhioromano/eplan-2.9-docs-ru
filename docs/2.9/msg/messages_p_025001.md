# P025001: Значение свойства страницы 'Сетка' страницы \<x\> отличается от настройки проекта 'Сетка для типа страницы'

### Причина

Значение свойства страницы Сетка для помеченной страницы не соответствует настройке проекта Сетка для типа страницы. К примеру, сетка в 4 мм используется по умолчанию для всех страниц схемы соединений электротехники в вашем проекте ("Многополюсная схема соединений" и "Однополюсная схема соединений"). Для помеченной страницы схемы соединений значение свойства страницы Сетка впоследствии будет заменено на 2 мм.

### Решение

Настройте значение свойства страницы Сетка для помеченной страницы в соответствии с настройкой проекта Сетка для типа страницы или наоборот.

(Обратитесь к свойству страницы Сетка после того, как вы выделите страницу в навигаторе страниц и выберете пункт всплывающего меню  Свойства. Найдите настройку проекта Сетка для типа страницSeitentyp после того, как вы выберете пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Управление > Страницы.

Затем запустите новый контрольный прогон.

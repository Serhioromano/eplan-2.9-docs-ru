# P026030: Отсутствует запись данных ЧУ для контура, определенного пользователем

### Причина

На монтажной поверхности было размещено монтажное отверстие в качестве определенного пользователем контура без записи данных ЧУ.

### Решение

Определите запись данных ЧУ для определенного пользователем контура. При этом рекомендуется действовать следующим образом.

1. Установите местоположение контура в пространстве листа, пользуясь функцией Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно Свойства ++...++ определенного пользователем контура и запомните имя указанного в поле Контур файла с типом *.fc1.
3. Закройте диалоговое окно 'Свойства', нажав ++Отменить++.
4. Выберите пункты меню Сервисные программы > Основные данные > Контур (данные ЧУ) > Открыть.
5. В диалоговом окне Открыть контур выделите имя файла, которое вы запомнили, и щелкните ++Открыть++.

!!! info "Для сведения:"

    В редакторе контура откроется определенный пользователем контур.

6. Выберите пункты меню Сервисные программы > Основные данные > Контур (данные ЧУ) > Открыть > Генерировать запись данных > ЧУ Steinhauer или Rittal - Perforex / Secarex.
7. В диалоговом окне Запись данных контура ЧУ ++...++ выберите машину, фрезер, а также способ пуска и подтвердите эти данные, нажав ++OK++.
8. С помощью курсора установите ***начальную точку*** фрезера для определенного пользователем контура.

!!! info "Для сведения:"

    Пути фрезерования и точки сверления отмечены цветом.

9. Выберите пункты меню Сервисные программы > Основные данные > Контур (данные ЧУ) > Закрыть и подтвердите последующее системное сообщение о синхронизации основных данных, нажав ++Да++.

!!! info "Для сведения:"

    В файле контура *.fc1 будет создана запись данных ЧУ.

10. Снова разместите определенный пользователем контур на монтажной поверхности.

11. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Обратите внимание, что контрольный прогон учитывает записи, заданные для машины в меню Параметры > Настройки > Фирма > Механическая обработка. Если там определены области настройки типа машин Экспорт ЧУ Rittal - Perforex / Secarex и Экспорт ЧУ Steinhauer или их варианты, необходимо генерировать недостающие записи данных ЧУ для всех типов машин или их вариантов, чтобы заблокировать сообщение контрольного прогона 026030.

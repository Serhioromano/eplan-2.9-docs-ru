# P005076: Разные свойства '\<x\>' внутри трубопровода

### Причина

На функциональной схеме автоматизации на соединении содержатся несколько определений трубопровода (выводы трубопровода или точки определения трубопровода), имеющие одинаковые имя трубопровода, вещество и состояние, но разные значения относительно определенного свойства \<x\>. При контрольном прогоне учитываются следующие свойства:

* Обозначение соединения
* Поперечное сечение / диаметр соединения
* Цвет / номер соединения
* Описание соединения
* Соединение: Длина с единицей измерения
* Обозначение типа
* Примечание
* Линия автом. соединения: Толщина линии
* Линия автом. соединения: Цвет
* Линия автом. соединения: Тип линии
* Линия автом. соединения: Масштаб линии
* Линия автом. соединения: Слой.

### Решение

1. Установите местоположение определений трубопровода на функциональной схеме автоматизации с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно "Свойства" для определения трубопровода и внесите одинаковое значение для свойства \<x\>, указанного в сообщении.
3. Обновите соединения и затем повторно запустите контрольный прогон.

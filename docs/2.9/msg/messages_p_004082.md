# P004082: Проект конфигураций '\<x\>' не содержит ЦПУ

### Причина

На блоке ПЛК не указано, что этот блок ПЛК является ЦПУ. Эта информация требуется для корректного экспорта данных ПЛК из EPLAN.

Этот контрольный прогон служит только для проверки блоков ПЛК, которые являются главными функциями и на которых указан проект конфигураций.

### Решение

1. Установите местоположение блока ПЛК на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно свойств блока ПЛК.
3. Переместите вкладку Данные структуры ПЛК на передний план.
4. Установите в списке свойств флажок ЦПУ.
5. Убедитесь, что на блоке ПЛК также указаны ***проект конфигураций*** и ***имя станции***.
6. Щелкните по кнопке ++OK++.
7. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Обратите внимание, что для каждой рабочей станции не требуется отдельное ЦПУ. Достаточно как минимум одного ЦПУ для всех рабочих станций, подчиненных одному проекту конфигураций.

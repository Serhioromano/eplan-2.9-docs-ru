# P022061: Определенное пользователем свойство '\<x\>' (помощь ввода '\<z\>') содержит неопределенное значение '\<y\>'

### Причина

В проекте на объектах (страницах, макросах, функциях и т. д.) используется определенное пользователем свойство '\<x\>', для которого в диалоговом окне Конфигурировать свойства посредством использования помощи ввода '\<z\>' заданы определенные значения по умолчанию '\<y\>' в списке выбора.

В данном случае использовалось значение, которое не является составной частью списка выбора, указанного в диалоговом окне конфигурации, и поэтому считается недействительным.

### Решение

#### Возможность 1

1. Определите местоположения объекта в проекте с помощью пункта меню Поиск. Или используйте функцию Перейти к (графика) из всплывающего меню диалогового окна Управление сообщениями для поиска объекта на странице схемы соединений.
2. Вызовите диалоговое окно Свойства объекта.
3. В списке свойств для определенного пользователем свойства '\<x\>' с помощью кнопки ++...++ выберите действительное значение по умолчанию '\<y\>'.
4. Щелкните по кнопке ++OK++.
5. Затем запустите новый контрольный прогон.

#### Возможность 2

1. Выберите пункты меню Данные проекта > Конфигурировать свойства.
2. В диалоговом окне Конфигурировать свойства добавьте вручную значение '\<y\>' в список выбора определенного пользователем свойства '\<x\>', используя помощь ввода '\<z\>'.
3. Щелкните по кнопке ++OK++.
4. Затем запустите новый контрольный прогон.

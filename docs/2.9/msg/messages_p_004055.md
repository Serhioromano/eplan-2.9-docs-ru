# P004055: При импорте было найдено более одного каркаса. Убедитесь, что для соединения используется надлежащее аппаратное обеспечение

### Причина

Используется интерфейс импорта ПЛК ***Schneider Unity Pro***: На линии продукции M340 в файле импорта отсутствуют блоки ПЛК для соединения каркасов между собой.

### Решение

Убедитесь, что имеющиеся расширения каркасов спроектированы с использованием надлежащего аппаратного обеспечения.

!!! note "Замечание:"

    Это относящееся к модулю сообщение выводится при импорте данных ПЛК. При выполнении нового контрольного прогона (любой настроенной схемы) его можно удалить из диалогового окна управления сообщениями.

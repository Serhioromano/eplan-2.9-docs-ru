# P021005: Имена объектов-заполнителей и различных переменных совпадают

### Причина

В проекте макросов содержатся макросы с объектами-заполнителями, имеющими одинаковое имя, для которых при этом были определены разные переменные. Если открыть проект в EPLAN Engineering Configuration One (EEC One), появятся сообщения об ошибках.

### Решение

1. Определите местоположение объектов-заполнителей при помощи функции Перейти к (графика) или Перейти ко 2-й координате во всплывающем меню диалогового окна Управление сообщениями.
2. Переименуйте многократно встречающиеся объекты-заполнители или используйте для них одинаковые переменные.
3. Затем запустите новый контрольный прогон.

# P001028: Распределенные клеммы имеют другое определение функции

### Причина

Проект содержит распределенные клеммы с разными определениями функций.

### Решение

1. Установите местоположение данных клемм на схеме соединений или в навигаторе устройств, пользуясь функциональностью Перейти к (...), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Запустите диалоговое окно Свойства для каждой распределенной клеммы и измените в нем вкладку Данные символа / функции.
3. В групповом поле Функциональные данные (логика) нажмите кнопку ++...++ рядом с полем Определение и выберите необходимое определение функции для распределенной клеммы.
4. Повторите данную операцию для всех остальных распределенных клемм. При этом обратите внимание на то, чтобы все определения функций были единообразны.
5. Обновите соединения и затем повторно запустите контрольный прогон.

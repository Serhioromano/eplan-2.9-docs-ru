# P004029: Объект отмечен для удаления

### Описание

При импорте файлов конфигурации ПЛК объекты ПЛК (блоки ПЛК, выводы устройства ПЛК или адреса) выделяются программно, если в проекте EPLAN они содержатся в соответствующем проекте конфигураций, а не в файле импорта.

После импорта данных ПЛК выделение соответствующих объектов в диалоговом окне Адреса / Списки присвоения (путь меню: Данные проекта > ПЛК > Адреса / Списки присвоения) в столбце таблицы Выделено для удаления отображается в виде установленного флажка.

При импорте в ***формате AutomationML*** выводы устройства ПЛК дополнительно выделяются для удаления, если символический адрес в файле импорта пуст, а свойство Символический адрес (автоматич.) на соответствующем выводе устройства ПЛК в EPLAN заполнено. Имеющийся символический адрес остается.

Во время этого контрольного прогона выполняется поиск всех объектов, выделенных таким образом. Для каждого найденного объекта ПЛК выводится сообщение в Управлении сообщениями.

!!! note "Замечание:"

    Выделение сохраняется после вывода сообщения.

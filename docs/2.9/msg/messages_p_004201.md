# P004201: Siemens: Неверное присвоение типа рабочей станции ПЛК: \<x\>

### Причина

EPLAN обнаружил неверное присвоение карты ПЛК в отношении используемого типа рабочей станции ПЛК. В заполнителе \<x\> сообщения отображается неверно присвоенная карта ПЛК и способ корректировки присвоения.

### Решение

1. Установите местоположение блока ПЛК, представляющего карту ПЛК, на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно свойств блока ПЛК.
3. Переместите первую вкладку на передний план.
4. Откорректируйте значение свойства типа рабочей станции ПЛК, указанного в тексте сообщения, в соответствии со значением по умолчанию.
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

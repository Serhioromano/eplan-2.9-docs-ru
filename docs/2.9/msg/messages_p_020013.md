# P020013: \<x\> различается в станд. значениях

### Описание

При восстановлении выгруженного проекта его настройки сравнивались с настройками исходного проекта. В отношении стандартных значений имеется различие.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

# P012008: Структурный идентификатор не соответствует структурному идентификатору монтажной платы

### Причина

Вы создали структуру проекта и определили для устройств этого проекта блоки идентификаторов, которым были присвоены структурные идентификаторы. Однако теперь с точки зрения блока идентификаторов имеются разные структурные идентификаторы на размещении изделия и на соответствующей монтажной плате.

### Решение

Введите (в диалоговом окне Свойства ++...++ в поле Видимое ОУ) для размещения изделия и соответствующей монтажной платы одинаковые структурные идентификаторы для требуемых блоков идентификаторов (например, для ид. "Установка" и/или "Место установки").

Затем запустите новый контрольный прогон.

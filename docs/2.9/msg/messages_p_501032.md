# P501032: Неверное обозначение канала, указанного в шаблонах функций изделия

### Причина

В проверяемой базе данных изделий обнаружено изделие ПЛК, на котором сохранено неправильное обозначение канала.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Выведите на передний план вкладку Шаблоны функции.
3. В столбце Обозначение канала измените одно или несколько значений в определениях функций, отображающих выводы устройств для сигналов, сетей, шинных кабелей и / или точек подвода питания, на исключительно числовые значения. Следите, чтобы перед значением и после него не было пробела. Пробела также не должно быть между двумя числовыми значениями.
4. Щелкните по кнопке ++OK++.
5. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
6. Затем запустите новый контрольный прогон.

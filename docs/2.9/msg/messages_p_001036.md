# P001036: Локальные шаблоны функций несмотря на изделия с шаблонами функций

### Причина

В EPLAN шаблоны функций предоставляются изделиями. Такие функции, как клеммники, могут предоставляться также "локальными" шаблонами функций. Такие локальные шаблоны функций на клеммнике можно создать, например, в диалоговом окне Обработать клеммник с помощью всплывающего меню. В случае присвоения клеммнику с локальными шаблонами функций изделия с шаблонами функций локальные шаблоны функций, как правило, удаляются.

Поведение программы может быть другим, если применять в текущем проекте клеммники из более ранних версий EPLAN (до версии 2.4): Если такой клеммник имеет локальные шаблоны функций и ему присваивается изделие с шаблонами функций, то удаления локальных шаблонов функций может и не произойти. Также возможно, что EPLAN допускал создание шаблонов локальных функций, несмотря на то, что клеммнику из более ранних версий уже было присвоено изделие с шаблонами функций. Возникла одна из этих ситуаций, что привело к выводу сообщения.

### Решение

Локальные шаблоны функций на клеммнике невозможно удалить напрямую.

Поэтому рекомендуется следующий порядок действий:

1. Установите сначала местоположение соответствующего клеммника на схеме соединений, пользуясь функцией Перейти к (графика), которую вы можете найти во всплывающем меню диалогового окна Управление сообщениями.
2. Выберите пункт всплывающего меню Свойства.
3. В диалоговом окне свойств клеммника вынесите на передний план вкладку Изделие.
4. В поле Число штук / количество увеличьте введенное значение для одного из применяемых изделий. При этом не играет роли, на какое значение увеличивается число штук или количество. Важно лишь отображение изменения.
5. Закройте диалоговое окно Свойства, щелкнув по кнопке ++ОК++.

!!! info "Для сведения:"

    Локальные шаблоны функций удаляются.

6. Снова откройте диалоговое окно Свойства и откорректируйте увеличенное перед этим значение в поле Число штук / количество во вкладке Изделие.
7. Затем запустите новый контрольный прогон.

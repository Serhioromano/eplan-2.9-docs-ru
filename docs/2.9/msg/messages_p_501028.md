# P501028: Свойство 'Обозначение штекера' на выводе шины пусто

### Причина

В проверяемой базе данных изделий было найдено изделие, на котором определен шаблон функции с определением функции "Кабельный вывод сети / шины". На этом шаблоне функции отсутствует значение для свойства Обозначение штекера.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Выведите на передний план вкладку Шаблоны функции.
3. Внесите значение на шаблоне функции в поле Обозначение штекера.
4. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
5. Затем запустите новый контрольный прогон.

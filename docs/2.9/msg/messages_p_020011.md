# P020011: \<x\> различается в описании

### Описание

При восстановлении выгруженного проекта настройки защищенного от записи исходного проекта сравнивались с настройками выгруженного проекта. В отношении описания имеется различие.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

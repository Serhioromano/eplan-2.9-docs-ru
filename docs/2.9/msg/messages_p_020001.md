# P020001: Сравнение настроек проекта: Текущая дата и время: \<x\>

### Описание

Выполнено сравнение проектов, при котором ***Настройки*** проверяемого проекта (проект с замечаниями) сравнивались с настройками проекта для сравнения. Это сообщение информирует при помощи \<x\> о текущей дате и времени выполнения сравнения.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

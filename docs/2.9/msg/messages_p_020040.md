# P020040: Последовательность структур проекта для страниц заменена с '\<x\>' на '\<y\>'

### Описание

Выполнено сравнение проектов, при котором ***Структура*** проверяемого проекта (проекта с замечаниями) была сравнена со свойствами проекта для сравнения и были определены различия.

Это сообщение указывает, что в проверяемом проекте логическая последовательность блоков идентификаторов изменена в диалоговом окне Структура страниц. В сообщении в заполнителе \<x\> выводится последовательность блоков идентификаторов проекта для сравнения, а в заполнителе \<y\> — последовательность блоков идентификаторов проекта с замечаниями.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

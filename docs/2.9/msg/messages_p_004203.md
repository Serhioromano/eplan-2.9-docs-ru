# P004203: Siemens: карта ПЛК с типом рабочей станции ПЛК '\<x\>' является недействительным каркасом

### Причины

Карта ПЛК типа рабочей станции ПЛК Siemens ***ET200xxx*** идентифицирована как недействительный каркас, поскольку свойства Шинный интерфейс / первичная станция и ЦПУ заданы неправильно.

### Решение

1. Установите местоположение карты ПЛК на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно "Свойства" карты ПЛК.
3. Переместите вкладку Данные структуры ПЛК на передний план.
4. Если блок ПЛК представляет шинный интерфейс и / или ЦПУ, проследите, чтобы свойства Шинный интерфейс / первичная станция и / или ЦПУ были активированы. Кроме того, введите в поле Каркас требуемое значение.
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

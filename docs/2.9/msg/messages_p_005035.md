# P005035: Точка определения соединения (\<x\>) на многократном соединении заново выровнена и отформатирована

### Описание

Это сообщение выводится в том случае, если при выравнивании и форматировании точек определения соединений одна из этих точек находится на многократном соединении.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

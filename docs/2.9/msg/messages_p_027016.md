# P027016: Топология - Не найдена трасса маршрутизации к цели

### Причина

Для данного соединения не найдена трасса маршрутизации. Вы определили в настройках фильтры для маршрутизируемых соединений. Эти настройки фильтра также можно присваивать в качестве свойств сегментам маршрутизации внутри сети соединенных сегментов. Как минимум один сегмент маршрутизации имеет значение, отличающееся от соответствующих значений у других сегментов в сети соединенных сегментов, относительно выбранного фильтра соединений.

Например, два или три сегмента маршрутизации, ведущие от источника к цели, имеют в качестве цвета соединения значение "синий", а один - значение "красный". Поэтому при попытке сгенерировать маршрутизируемое соединение системе не удалось найти трассу маршрутизации, и отображается только одно немаршрутизируемое соединение между источником и целью.

### Решение

Создайте действительную трассу маршрутизации от источника к цели.

1. Установите местоположение цели недействительного маршрутизируемого соединения, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Проследите сеть соединенных сегментов обратно к началу и проверьте в диалоговом окне Свойства ++...++ для каждого сегмента маршрутизации значение свойства Топология: Фильтр соединений.
3. Присвойте всем сегментам маршрутизации одинаковые значения.
4. Маршрутизируйте соединение заново и запустите новый контрольный прогон.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

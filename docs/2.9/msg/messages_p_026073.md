# P026073: Маршрутизируемое соединение без номера изделия, цвета / номера и (или) поперечного сечения / диаметра

### Причина

При маршрутизации маршрутизируемых соединений было обнаружено маршрутизируемое соединение, для которого не было определено как минимум одно из следующих свойств.

* Номер изделия
* Цвет / номер
* Поперечное сечение / диаметр.

### Решение

1. Установите местоположение маршрутизируемого соединения в ***пространстве листа*** с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Дважды щелкните в пространстве листа выделенное цветом маршрутизируемое соединение.

!!! info "Для сведения:"

    Откроется диалоговое окно Свойства ++...++ для этого маршрутизируемого соединения.

3. Перейдите в диалоговом окне на вкладку Изделие и проверьте, указан ли в таблице номер изделия.
4. Если нет, выделите первую строку и откройте диалоговое окно "Выбор изделия" с помощью кнопки ++...++.
5. Перейдите в древовидной структуре к нужному изделию и выберите его, нажав ++ОК++.
6. Перейдите на вкладку Соединение.
7. Проверьте, указаны ли значения в полях Цвет/Номер и/или Поперечное сечение / Диаметр.
8. Если нет, введите недостающие значения.
9. Закройте диалоговое окно Свойства ++...++, для чего нажмите на кнопку ++OK++.
10. Запустите новый контрольный прогон и снова маршрутизируйте соединения.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

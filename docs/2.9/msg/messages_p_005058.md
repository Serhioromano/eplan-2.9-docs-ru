# P005058: Точка определения сети с неверным сигналом

### Причина

На одном соединении была размещена точка определения сети и снабжена именем сигнала. Однако этот сигнал не определен ни в одном месте схемы соединений.

### Решение

#### Возможность 1

Введите в точке определения сети имя уже существующего определения сигнала.

#### Возможность 2

Определите новый потенциал в схеме соединений, например с помощью вывода сигнала. Присвойте этому выводу устройства имя потенциала и имя сигнала точки определения сети.

Обновите соединения и затем повторно запустите контрольный прогон.

# P007005: Устройство без главной функции

### Причина

Вы вставили устройство с распределенными функциями в схему соединений. При этом ни одна из функций не обозначена как главная функция.

!!! note "Замечание:"

    Это сообщение также выводится для отдельно размещенной функции, которая не обозначена как главная функция.

### Решение

Сделайте одну из размещенных функций устройства главной функцией. Дважды щелкните по соответствующему условному обозначению и выберите в диалоговом окне Свойства ++...++ вкладку Данные символа / функции. В групповом поле Данные функции (логика) установите флажок Главная функция.

Затем запустите новый контрольный прогон.

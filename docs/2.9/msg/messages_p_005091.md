# P005091: Разный класс трубы соединения и функции

### Причина

Значение, внесенное для соединения на функциональной схеме автоматизации для свойства Класс трубы не совпадает со значением, внесенным для одноименного свойства функции, которая является источником или целью этого соединения.

### Решение

1. Установите местоположение функции с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. С помощью функции Перейти ко 2-й координате всплывающего меню перейдите к точке определения соединения.
3. Найдите значение, внесенное в соответствующем диалоговом окне "Свойства", в поле Класс трубы.
4. Унифицируйте оба значения.
5. Обновите соединения и затем повторно запустите контрольный прогон.

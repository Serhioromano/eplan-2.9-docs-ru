# P026098: Размещение изделия совпадает с зарезервированной зоной для воздушного потока

### Причина

Размещение изделия вдается в зарезервированную зону для воздушного потока компонента системы кондиционирования. В результате нарушается циркуляция воздуха в электрошкафу.

### Решение

1. Установите местоположение соответствующего размещения изделия в пространстве листа с помощью функции Перейти к (графика) во всплывающем меню диалогового окна Управление сообщениями. С помощью пункта всплывающего меню Перейти к 2-й координате перейдите к размещению изделия компонента системы кондиционирования.
2. Переместите размещение изделия таким образом, чтобы исключить его наложение на зарезервированную зону компонента системы кондиционирования.
3. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    При соприкосновении размещения изделия и зарезервированной зоны для воздушного потока сообщение о конфликте выводиться не будет.

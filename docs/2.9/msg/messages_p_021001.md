# P021001: Объект-заполнитель не содержит переменные

### Причина

В ходе присвоения значений не были определены переменные для объекта-заполнителя.

### Решение

Дважды щелкните по объекту-заполнителю. Перейдите в диалоговом окне Объект-заполнитель на вкладку Значения. Определите здесь как минимум одну переменную. Выберите для этого пункт всплывающего меню  Создать переменную. Введите в диалоговом окне Назвать новую переменную требуемое имя и подтвердите ввод, нажав ++OK++.

!!! info "Для сведения:"

    Новая переменная вставляется в виде новой строки под заголовком столбца Переменная.

Затем выберите пункт всплывающего меню Новый набор значений.

!!! info "Для сведения:"

    К таблице добавляется пустой столбец.

Введите в верхнюю строку столбца обозначение для набора значений. Введите во второй ячейке значение для переменной. Перейдите на вкладку Присвоение. Выделите в таблице присвоений в столбце Переменная ячейку с требуемым свойством объекта. Выберите пункт всплывающего меню Выбрать переменную. Присвойте в следующем диалоговом окне новую переменную требуемому свойству объекта.

Затем запустите новый контрольный прогон.

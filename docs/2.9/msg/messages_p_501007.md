# P501007: На изделии не указана высота, ширина и / или глубина: Высота = \<x\>, Ширина = \<y\>, Глубина = \<z\>

### Причина

В проверяемой базе данных изделий было найдено изделие, на котором отсутствует монтажная информация о ***высоте***, ***ширине*** и / или ***глубине***.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Переместите вкладку Монтажные данные на передний план.
3. Занесите в поле Ширина, Высота и / или Глубина недостающие значения.
4. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
5. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

* Чтобы сообщение больше не выводилось, не менее двух из приведенных выше параметров монтажных данных на изделии должны иметь значения. Исключения составляют изделия, на которых сохранен 3D-макрос. В этом случае должны присутствовать значения всех трех параметров монтажных данных.

# P004092: Отсутствует имя интерфейса шины

### Причина

Выводу шины ***на базе Ethernet*** не присвоено имя интерфейса шины.

Этот контрольный прогон служит только для проверки выводов шины, на которых указан проект конфигураций.

### Решение

1. Установите местоположение вывода шины на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно "Свойства" вывода шины.
3. Переместите вкладку Данные шины на передний план.
4. В списке свойств в поле Интерфейс шины: Имя введите недостающее значение.
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

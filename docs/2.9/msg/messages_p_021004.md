# P021004: Объект-заполнитель содержит недействительное присвоение переменных

### Причина

Программа проверяет, подходит ли присвоение переменной в столбце Переменная таблицы присвоений к соответствующему свойству выделенного объекта. В этом случае присвоение переменной и свойство объекта присвоены неверно. Например, свойству, которое может иметь состояния "Да/""Нет", присвоена переменная, которая в качестве набора значений содержит текст.

### Решение

Присвойте соответствующему свойству объекта переменную с соответствующим значением. Затем запустите новый контрольный прогон.

# P004045: Связующее устройство на блоке ПЛК: '\<x\>'

...Жгут / Сеть = \<x\>, Главная шина / Шинный интерфейс = \<y\>, Подключен к главной шине / шинному интерфейсу = \<z\>, Шинная система = \<u\>, К позиции / адресу шины = \<v\>, Скорость передачи данных = \<w\>, Подразъем = \<r\>, Имя устройства в DNS/PROFINET = \<s\>

### Причина

Возможно несколько причин:

* В проекте содержатся блоки ПЛК со связующими устройствами.
* Или вы импортировали старый проект с блоками ПЛК, которые содержат связующие устройства.
* Или вы импортировали старые макросы, которые содержат блоки ПЛК со связующими устройствами.

### Решение

Вам предоставляется право выбора по обращению с данными:

1. Например, создать новый вывод шины или использовать имеющийся вывод и внести туда соответствующие данные, отображенные в заполнителях сообщения.
2. Затем удалить имеющиеся связующие устройства с блоков ПЛК с помощью функции Проект > Организовать > Сжать.

!!! note "Замечание:"

    Это относящееся к модулю сообщение выводится при импорте данных ПЛК. При выполнении нового контрольного прогона (любой настроенной схемы) его можно удалить из диалогового окна управления сообщениями.

# P003005: Цель кабеля без полного ОУ

### Причина

У подсоединенной к кабелю исходной или целевой функции отсутствует полное обозначение устройства.

### Решение

Введите полное обозначение устройства для функции, являющейся источником или целью кабеля.

Обновите соединения и затем повторно запустите контрольный прогон.

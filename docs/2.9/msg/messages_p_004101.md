# P004101: Каркас '\<x\>' не соединен через удлинительный кабель шины со всеми остальными каркасами рабочей станции '\<y\>'

### Причины

#### Возможность 1

В проекте конфигураций один каркас не соединен с другими каркасами той же рабочей станции.

#### Возможность 2

В проекте конфигураций один каркас соединен с другими каркасами той же рабочей станции не с помощью удлинительного кабеля шины. Это значит, что как минимум одному из соединенных между собой выводов шины ***не*** присвоена настройка системы шин "Local-Bus: Расширение". Эта настройка необходима, если каркасы присвоены одной и той же рабочей станции.

### Решение

#### Возможность 1

1. Установите местоположение блока ПЛК каркаса '\<x\>' на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Соедините вывод шины этого каркаса с выводом шины другого каркаса в той же рабочей станции, чтобы образовались линии автоматического соединения.
3. Вызовите диалоговое окно 'Свойства' первого вывода шины.
4. Переместите вкладку Данные шины на передний план.
5. В раскрывающемся списке Система шин выберите значение "Local-Bus: Расширение".
6. Щелкните по кнопке ++OK++.
7. Выполните настройки, описанные в пунктах 3–6, для второго вывода шины.
8. Затем запустите новый контрольный прогон.

#### Возможность 2

1. Установите местоположение блока ПЛК каркаса '\<x\>' на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно 'Свойства' вывода шины, соединенного с выводом шины другого каркаса в одной рабочей станции.
3. Переместите вкладку Данные шины на передний план.
4. В раскрывающемся списке Система шин выберите значение "Local-Bus: Расширение".
5. Щелкните по кнопке ++OK++.
6. Установите местоположение на схеме соединений вывода шины, соединенного с первым выводом шины,, выбрав во всплывающем меню управления сообщениями функцию Перейти ко 2-й координате.
7. Выполните настройки, описанные в пунктах 2–5, для этого вывода шины.
8. Затем запустите новый контрольный прогон.

#### Возможность 3

1. Откройте диалоговое окно 'Свойства' блока ПЛК для одного из двух выводов шины.
2. Переместите вкладку Данные структуры ПЛК на передний план.
3. Измените имя рабочей станции.
4. Щелкните по кнопке ++OK++.
5. Затем запустите новый контрольный прогон.

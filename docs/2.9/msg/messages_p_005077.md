# P005077: Соединение трубопровода или точка определения трубопровода без указания вещества

### Причина

На функциональной схеме автоматизации содержится соединение трубопровода или точка определения трубопровода без указания вещества.

### Решение

1. Установите местоположение вывода или точки определения трубопровода на функциональной схеме автоматизации с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Для каждого вывода или точки определения трубопровода откройте диалоговое окно "Свойства" и введите требуемое значение в поле Вещество.
3. Щелкните по кнопке ++OK++.
4. Затем запустите новый контрольный прогон.

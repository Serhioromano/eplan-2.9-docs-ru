# P004039: Соединение между разными системами шин ('\<x\>', '\<y\>')

### Причина

Между двумя разными системами шин обнаружено соединение. Например, вывод шины шинной системы "Profibus" был соединен с выводом шины системы "Ethernet". В заполнителях \<x\> и \<y\> отображается соответствующая система шин.

### Решение

1. Определите местоположение соединенных между собой выводов шины на схеме соединений с помощью функции Перейти к (графика) и Перейти к 2-й координате во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите на обоих выводах шины диалоговое окно Свойства ++...++.
3. Переместите вкладку Данные шины на передний план.
4. Проверьте в поле Система шин настройку для обоих выводов шины.
5. Выберите на обоих выводах шины одинаковую систему шин и закройте диалоговое окно Свойства ++...++, нажав ++OK++.
6. Затем запустите новый контрольный прогон.

# P027019: Соединения кабеля определены неоднозначно

### Причина

Связи кабеля, спроектированного в многополюсном представлении, определены не уникально, то есть не менее чем для двух кабельных соединений свойства "Цвет/номер" и/или "Индекс пары" имеют идентичные значения. Поэтому кабель не мог маршрутизироваться в топологии.

### Решение

1. Установите местоположение многополюсного кабеля в проекте, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно Свойства ++...++ для всех соединений кабеля и сравните значения в полях Цвет / номер и/или Парный индекс.
3. Если два или несколько соединений относительно указанных выше свойств имеют идентичные значения, то измените их. Для каждого соединения обязательно назначать свойство с уникальным значением:
4. Снова маршрутизируйте кабель через сеть соединенных сегментов и заново запустите контрольный прогон.

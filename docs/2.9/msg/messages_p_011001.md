# P011001: Точка разрыва без обозначения устройства

### Причина

На точке разрыва отсутствует структурный идентификатор в полном обозначении устройства. (Отображаются только предшествующие знаки выбранных блоков обозначения.)

### Решение

Откройте диалоговое окно Свойства ++...++ точки разрыва, щелкните рядом с полем Полное ОУ по кнопке ++...++. Укажите в изменяемых полях для структуры обозначений, использующейся в проекте для точек разрыва, выбранные вами обозначения структуры или выберите их через [...]. Укажите в полях, начиная с поля Предш. цифра и до поля Подсчетчик, обозначение ОУ. Подтвердите ввод, нажав кнопку ++OK++.

Обновите соединения и затем повторно запустите контрольный прогон.

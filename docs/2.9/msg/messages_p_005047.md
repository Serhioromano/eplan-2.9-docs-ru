# P005047: Точка определения сети содержит неполные соединения

### Причина

На схеме соединений было удалено одно или несколько соединений в одной из сетей. Из-за этого между шлейфами возникает зазор. Зазор между соединениями в диалоговом окне "Свойства" точки определения сети (Вкладка Соединения сети, Таблица Соединения) помечается маленьким предупреждающим треугольником в столбце Выводы устройства. Кроме того, в этом случае появляется сообщение [P005046: Неполная сеть](messages_p_005046.md).

!!! example "Пример:"

    СетьN1в схеме соединений соединяет 5 резервирований (начиная с-F1и до-F5) с потенциаломL1. Выводы устройства образуют следующий шлейф:Вывод устройства Источник ЦельL1L1-F1:1-F1:1-F1:1-F2:1-F2:1-F2:1-F3:1-F3:1-F3:1-F4:1-F4:1-F4:1-F5:1-F5:1В схеме соединений было удалено резервирование-F4и присвоенный ему тройник.В результате удаления-F4и тройника в шлейфе возникает зазор:Вывод устройства Источник ЦельL1L1-F1:1-F1:1-F1:1-F2:1-F2:1-F2:1-F3:1-F3:1-F3:1-F5:1-F5:1

### Решение

Дополните сеть, вновь вставив в сеть удаленное соединение и тем самым ликвидировав зазор.

!!! example "Пример:"

    Пользуясь приведенным выше примером, действуйте следующим образом:Установите местоположение соответствующей точки определения сети на схеме соединений, пользуясь функцией Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.Вызовите диалоговое окно Свойства ++...++ точки определения сети.Выведите на передний план вкладку Соединения сети.Найдите в таблице Соединениянедостающий вывод устройства, который помечен предупреждающим треугольником. В нашем примере это было резервирование-F4:1.Закройте диалоговое окно "Свойства" и вновь вставьте резервирование с ОУ-F4.Соедините вывод устройства резервирования-F4:1с сетью при помощи тройника.Вызовите диалоговое окно Свойства <... точки определения сетиN1и переместите вкладку Соединения сетина передний план.Щелкните кнопку(Создать шлейф).Выделите в таблице Выводы устройствавывод устройства-F4:1с числом целей "0".Вывод устройства-F4:1будет скопирован в таблицу Соединенияи записан под горизонтальной линией шлейфа. Генерируется новый шлейф.Щелкните по(Завершить шлейф), чтобы закрыть шлейф.Появится горизонтальная линия под выводом устройства-F4:1в таблице Соединения.Выделите вывод устройства-F4:1в новые шлейфе и переместите его, щелкнув по кнопке(Переместить выше), в первый шлейф.Второй шлейф будет при этом расформирован.Расположите вывод устройства-F5:1прямо под предупреждающим треугольником.Выделите запись с предупреждающим треугольником и удалите его, нажав(Удалить).Закройте диалоговое окно Свойства ++...++ точки определения сети, нажав ++OK++, и повторно запустите контрольный прогон.

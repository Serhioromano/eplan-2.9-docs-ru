# P024012: Слишком соединений группы

### Причина

Значение, внесенное в поле Число соединений диалогового окна свойств группы соединений, меньше, чем фактически имеющееся количество соединений группы в группе соединений.

!!! note "Замечание:"

    Свободные шаблоны ***не*** учитываются!

### Решение

1. Установите линию определения группы соединений на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно "Свойства" группы соединений.
3. Увеличьте значение в поле Число соединений так, чтобы оно совпадало с фактически имеющимся количеством соединений группы соединений.

Или нажмите рядом с полем Тип кнопку ++...++ и задайте через выбор изделий подходящую группу соединений с соответствующим количеством соединений.
4. Щелкните по кнопке ++OK++.
5. Обновите соединения и затем повторно запустите контрольный прогон.

# P026019: Изделие размещения изделия присутствует на главной функции / главной клемме в недостаточном количестве

### Причина

#### Возможность 1

Размещение изделия в пространстве листа имеет определенный номер изделия. Соответствующей соединяемой с размещением изделия главной функции / клемме на странице схемы соединений присвоен другой номер изделия или этот номер отсутствует.

#### Возможность 2

Размещение изделия с определенным номером изделия было несколько раз вставлено в пространстве листа. Соответствующая соединяемая с размещением изделия главная функция / клемма на странице схемы соединений имеет тот же номер изделия, но на изделии в свойстве Число штук / колич. указано значение, которое меньше вставленного в пространстве листа числа размещения изделия.

### Решение

#### Для возможности 1

* Введите для размещения изделия в пространстве листа и для главной функции на схеме соединений одинаковый номер изделия.
* Затем запустите новый контрольный прогон.

#### Для возможности 2

* Приведите введенное значение для числа штук / количества в свойствах изделия главной функции /клеммы на схеме соединений в соответствие с числом вставленного в пространстве листа размещений изделия. Т. е. либо увеличьте значение числа штук / количества для главной функции / клеммы, либо удалите избыточное число размещения изделия.
* Затем запустите новый контрольный прогон.

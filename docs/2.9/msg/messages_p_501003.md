# P501003: Записанный на изделии документ \<x\> отсутствует

### Причина

В проверяемой базе данных изделий было найдено изделие, на котором сохранен документ, который EPLAN не может найти.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Переместите вкладку Документы на передний план.
3. Щелкните в столбце таблицы Файл / гиперссылка по записи, содержащей путь к ненайденному документу и нажмите на кнопку ++...++.
4. Выберите в открывшемся диалоговом окне выбора нужный документ.

Или удалите в качестве альтернативы запись таблицы с указанием пути отсутствующего документа.

Или сохраните ***новый*** документ в нужном каталоге документов и выберите его в столбце Файл / гиперссылка.
5. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
6. Затем запустите новый контрольный прогон.

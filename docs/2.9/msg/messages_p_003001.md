# P003001: Автоматический выбор кабеля прерывается, так как кабель не мог быть выбран

### Причина

Вы создали кабель при помощи ***автоматического выбора кабеля***. EPLAN не обнаружил изделия кабеля в диалоговом окне Предв. выбор кабеля. Поэтому автоматический выбор прекращается.

### Решение

Выберите пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Устройствo > Кабель (автоматически). Нажмите на ++...++ рядом с полем Автоматический выбор кабеля. Щелкните в диалоговом окне Настройки: Автоматический выбор кабеля справа над таблицей Предв. выбор кабеля по ![](../images/all_new_as.png){: .ui-icon } (Создать). Перейдите в древовидной структуре открывшегося диалогового окна Выбор изделия к группе продуктов Кабели / соединения и выберите подходящее для соединения изделие кабеля. При помощи кнопки ++OK++ изделие будет перенесено в таблицу предварительного выбора кабеля. Повторите выбор для остальных кабелей. Закройте диалоговое окно и создайте кабель снова автоматически.

!!! note "Замечание:"

    Это относящееся к модулю сообщение выводится при автоматическом выборе кабеля. При выполнении нового контрольного прогона (любой настроенной схемы) его можно удалить из диалогового окна управления сообщениями.

# P007007: Отсутствует место монтажа (описывающее)

### Причина

Не указано описывающее место монтажа для функции.

### Решение

Дважды щелкните по соответствующему условному обозначению и введите в диалоговом окне Свойства ++...++ отсутствующую информацию в поле Место монтажа (описывающее).

Затем запустите новый контрольный прогон.

# P007010: Функции имеют противоречивые данные в свойстве '\<x\>'

### Причина

В случае связанных друг с другом перекрестными ссылками функций данные определенного свойства главной и вспомогательной функции противоречат друг другу.

### Решение

Введите одинаковые данные соответствующего свойства для главной и вспомогательной функции.

Затем запустите новый контрольный прогон.

!!! tip "Совет:"

    Используйте групповую или табличную обработку. Благодаря этим двум возможностям обработки можно за один раз изменить свойства нескольких функций.

# P027009: Топология - У сегмента маршрутизации имеется как минимум один незамкнутый конец

### Причина

Находящиеся в сети сегменты маршрутизации соединены только одним концом или ни одним из своих концов с функцией топологии, точкой разрыва топологии или точкой маршрутизации.

### Решение

1. Установите местоположение сегмента маршрутизации внутри сети соединенных сегментов, пользуясь функцией Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Расположите в соответствии с числом незамкнутых концов сегментов маршрутизации функцию топологии, точку разрыва топологии или точку маршрутизации в сети соединенных сегментов.
3. Установите позиции объектов сети на сегментах маршрутизации таким образом, чтобы каждый вывод устройства или точка ввода состыковались с сегментом маршрутизации. Проверьте это, передвигая объекты сети соединенных сегментов на конце сегмента маршрутизации. При этом сегмент маршрутизации должен передвигаться вместе с ними.
4. Затем запустите новый контрольный прогон.

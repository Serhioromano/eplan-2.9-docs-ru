# P026029: Размещение изделия не имеет действительной схемы сверления

### Причина

#### Возможность 1

Размещение изделия было отменено посредством ***активированного*** свойства Функц. элементу требуется отверстие на монтажной поверхности на монтажной поверхности. Однако размещение изделия не располагает названием и данными схемы сверления.

#### Возможность 2

Размещение изделия было выполнено посредством присвоения схемы сверления на монтажной поверхности. Название схемы сверления, где дается определение схемы сверления, больше не существует в базе данных изделий.

### Решение

1. Установите местоположение размещения изделия в пространстве листа с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Выберите для размещения изделия действующую схему сверления (см. ссылку внизу).
3. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    ***Действующая*** схема сверления состоит из ***Набора данных схемы сверления*** и ***должна быть задана*** в ***Базе данных изделий***. Данные схемы сверления содержат данные о названии, а также данные о положении и размерах. Схема сверления ***должна быть присвоена*** размещению изделия в базе данных изделий. Присвоение выполните во вкладке Изготовление, где можно выбрать схему сверления по ***названию схемы сверления***.

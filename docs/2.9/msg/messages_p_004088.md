# P004088: Имя карты ПЛК '\<x\>' уже используется в каркасе или ЦПУ

### Причина

В проекте конфигураций на карте ПЛК, представляющей каркас или ЦПУ, указано имя карты ПЛК, уже существующее на другой карте ПЛК, которая также является каркасом или ЦПУ.

Этот контрольный прогон служит только для проверки блоков ПЛК, на которых указан проект конфигураций.

### Решение

1. Используйте функцию поиска, чтобы установить местоположение блоков ПЛК на схеме соединений. Для этого в качестве искомого текста введите имя карты ПЛК, указанное в заполнителе '\<x\>'.
2. С помощью функции Перейти к (графика), предоставляющей доступ к всплывающему меню диалогового окна Управление сообщениями, в отобразившемся результате поиска перейдите к соответствующему блоку ПЛК и откройте его.
3. Переместите первую вкладку на передний план.
4. Измените значение свойства Имя карты ПЛК в таблице свойств. При этом проследите, чтобы каждый каркас или каждое ЦПУ содержали уникальное имя карты ПЛК.

5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

!!! tip "Совет:"

    Для удобного поиска ошибок в проекте также можно использовать [табличную обработку](../functiondatagridgui_k_start.md). Просто создайте схему для отображения имени карты ПЛК и требуемых данных структуры ПЛК в удобной форме.

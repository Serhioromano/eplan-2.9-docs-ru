# P003027: Кабели: 'Функциональный текст (автоматич.)' различается (в языке '\<x\>')

### Причина

На кабеле, распределенно размещенном на схеме соединений, отличается свойство Функциональный текст (автоматич.) в одном или нескольких размещениях кабеля на выбранном языке отображения.

### Решение

1. Унифицируйте для всех размещений кабеля свойство Функциональный текст (автоматич.) на выбранном языке отображения.
2. Затем запустите новый контрольный прогон.

# P007006: Функция без обозначения вывода устройства

### Причина

Отсутствует как минимум одно обозначение вывода устройства для устройства, размещенного в схеме соединений.

### Решение

Дважды щелкните по соответствующему условному обозначению и введите в диалоговом окне Свойства ++...++ недостающее обозначение(я) вывода устройства.

Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Оповещения о выводах устройства без обозначения не отображаются, если им присвоен тип вывода устройства "Внутренний", "Прямой вывод устройства" или "Мостовая перемычка".

# P022016: Имя страницы '\<x\>' содержит буквы

### Причина

Имя страницы на одной странице содержит одну или несколько букв.

### Решение

1. Дважды щелкните "Cообщение" в диалоговом окне Управление сообщениями и определите место соответствующей страницы в древовидной структуре навигатора страниц.
2. Откройте через всплывающее меню диалоговое окно Свойства страницы и измените в поле Полное имя страницы букву или буквы, которые стоят после разделительного знака "/".
3. Затем запустите новый контрольный прогон.

# P501025: Свойства изделия 'Обозначение типа ПЛК' и 'Описание устройства: Имя файла' не заполнены

### Причина

В проверяемой базе данных изделий определено изделие ПЛК, на котором для свойства Обозначение типа ПЛК или свойства Описание устройства: Имя файла не задано значение.

Сообщение контрольного прогона также выводится, если изделие не является устройством ПЛК и имеет шаблон функции с определением функции "Кабельный вывод сети / шины".

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Переместите вкладку Свойства на передний план.
3. Введите значение в поле Обозначение типа ПЛК или поле Описание устройства: Имя файла.
4. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
5. Затем запустите новый контрольный прогон.

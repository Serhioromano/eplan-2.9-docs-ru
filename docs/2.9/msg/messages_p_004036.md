# P004036: Незнакомая логическая сеть '\<x\>' (Вывод шины в системе шин: '\<y\>')

### Причина

На выводе шины, являющемся подчиненным в системе шин, введено имя логической сети, для которой не найден вывод шины, обозначенный как главный.

Этот контрольный прогон служит только для проверки выводов шины, на которых указан проект конфигураций.

### Решение

1. Создайте вывод шины в качестве главного с соответствующим именем логической сети или задайте на выводе шины имя логической сети, которое уже существует.
2. Затем запустите новый контрольный прогон.

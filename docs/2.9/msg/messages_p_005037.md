# P005037: Противоречие в искробезопасности выводов устройства

### Причина

Имеющиеся на соединении целевые функции противоречат друг другу с точки зрения установленного свойства Искробезопасн.. Так, например, в настройках данных функции одной цели был установлен флаг Искробезопасн. и, кроме того, на выводах устройства функции была указана настройка Искробезопасность возможна. Однако другая цель соединения не получила настройки искробезопасности.

### Решение

#### Возможность 1

Если соединения должно быть искробезопасным, необходимо сделать искробезопасными все цели соединения. Для этого установите в диалоговом окне Свойства ++...++ в групповом поле Данные функции (логика) соответствующий флаг Искробезопасн.. Кроме того, в диалоговом окне Логическая схема выводов устройства на соответствующих выводах устройства функции следует установить флаг Искробезопасность возможна.

#### Возможность 2

Если соединение ***не*** должно быть искробезопасным, для всех целей в диалоговом окне Свойства ++...++ в групповом поле Данные функции следует снять флаг Искробезопасн..

Обновите соединения и затем повторно запустите контрольный прогон.

# P020041: Различие слоев: символ \<x\> : размещение свойства \<y\> : слой \<z\> \- \<u\>

### Причина

Выполнено сравнение проектов, при котором ***Слои*** проверяемого проекта (проекта с замечаниями) были сравнены со слоями проекта для сравнения и были определены различия.

Это сообщение появляется, если размещение свойства размещенного на схеме соединений символа имеет другую настройку уровней, чем указано в библиотеке символов для аналогичного символа. При этом \<x\> указывает имя символа, \<y\> — имя размещение свойства, \<z\> — уровень размещения свойства в библиотеке символов,а \<u\> — уровень размещения свойства символа на схеме соединений.

### Решение

Измените уровень размещения свойства для символа находящегося в схеме соединений и синхронизируйте его с уровнем размещения свойства в библиотеке символов.

Затем запустите сравнение проектов или новый контрольный прогон.

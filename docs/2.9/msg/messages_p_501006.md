# P501006: Элементы макроса \<x\> (\<y\> / \<z\>) не сгруппированы

!!! note "Замечание:"

* Этот контрольный прогон ***не учитывает 3D-макросы***.
* Это сообщение не выводится, если в макросе есть неразмещенные функции.

### Причина

В проверяемой базе данных изделий было найдено изделие, на котором сохранен макрос, составные части которого не сгруппированы.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Переместите вкладку Технические данные на передний план.
3. Щелкните рядом с полем Макрос по кнопке ++...++ и выберите в открывшемся диалоговом окне выбора файл макроса, содержащий макрос с несгруппированными составными частями.

Или в качестве альтернативы удалите запись в поле Макрос.

Или создайте ***новый*** макрос со сгруппированными составными частями, сохраните его в нужном каталоге макросов и выберите его в поле Макрос.
4. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
5. Затем запустите новый контрольный прогон.

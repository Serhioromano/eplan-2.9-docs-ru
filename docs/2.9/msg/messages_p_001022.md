# P001022: Расположенные одна над другой мостов. перемычки

### Причина

Для клеммника в схеме соединений, который состоит из простых клемм, определено более одной мостовой перемычки. По причине выбранной последовательности клемм образовать мостовую перемычку все же невозможно, так как мостовые перемычки - с точки зрения технологии проводного монтажа - будут находиться друг над другом.

### Решение

#### Возможность 1

Выделите определение клеммника в навигаторе клеммников или в схеме соединений. Выберите пункт меню Данные проекта > Клеммники > Обработать.

В диалоговом окне Обработать клеммник рассмотрите графическое представление соединений в столбце Мостовые перемычки. Отображаются параллельные соединения.

С помощью кнопок со стрелками измените на панели инструментов последовательность клемм. Следите за тем, чтобы графики мостовых перемычек были расположены не параллельно, а друг за другом.

Примените измененную последовательность клемм в диалоговом окне Обработать клеммник, нажав ++OK++.

#### Возможность 2

Измените определение функции точек определения соединения, ответственных за образование мостовой перемычки. Замените мостовые перемычки на проволочные перемычки.

Для этого запустите диалоговое окно Свойства ++...++ для соответствующих точек определения соединения и щелкните по вкладке Данные символа / функции.

В групповом поле Данные функции (логика) рядом с полем Определение щелкните по ++...++ и выберите в структуре дерева запись "Проволочная перемычка".

Примените изменение и закройте диалоговое окно Свойства ++...++, нажав ++OK++.

Обновите соединения и затем повторно запустите контрольный прогон.

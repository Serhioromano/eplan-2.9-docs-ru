# P028007: Изделия или число / кол-во изделий на главной функции и соответствующем объекте планирования различаются

### Причина

На вкладке Изделие, принадлежащей объекту планирования главной функции, и на одноименной вкладке объекта планирования (диалоговое окно Свойства ++...++) были сделаны разные записи изделий и/или указано разное число штук / количество изделий.

### Решение

1. Установите местоположение главной функции и соответствующего объекта планирования в навигаторе предварительного планирования с помощью функции Синхронизировать выбор, которая находится во всплывающем меню в диалоговом окне Управление сообщениями.
2. Последовательно откройте диалоговые окна Свойства ++...++ главной функции и соответствующего объекта планирования и выведите для этих двух объектов вкладку Изделие на передний план.
3. Сравните записи в столбце Номер изделия и / или Число штук / Количество на главной функции и соответствующем объекте планирования.
4. Если имеются различия по свойствам Номер изделия и/или Число штук / Количество, приведите в соответствие записи для обоих объектов и сохраните изменения.
5. Затем запустите новый контрольный прогон.

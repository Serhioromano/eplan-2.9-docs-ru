# P005074: Сечение/диаметр соединения отличается от сечения/диаметра точки вывода

### Причина

Один или несколько выводов функций подключены к соединению. Значение сечения/диаметра вывода или по крайней мере одного из нескольких выводов отличается от подсоединенного соединения.

### Решение

1. Вставьте одинаковое значение сечения/диаметра вывода для выводов всех функций и для подсоединенного соединения.
2. Обновите соединения и затем повторно запустите контрольный прогон.

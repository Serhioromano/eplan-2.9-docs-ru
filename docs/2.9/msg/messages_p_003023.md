# P003023: Противоречие между количеством изделий и настройкой для суммирования длины кабелей

### Причина

Для кабеля активировано свойство Не учитывать кабель при суммировании длины кабелей, кроме того, на закладке Изделие диалогового окна свойств кабеля в поле Число штук / Количество указано значение "0".

### Решение

#### Возможность 1

Дважды щелкните по линии определения кабеля и выберите в диалоговом окне Свойства ++...++ вкладку Кабель. В групповом поле Свойства снимите флаг Не учитывать кабель при суммировании длины кабелей.

#### Возможность 2

Дважды щелкните по линии определения кабеля и выберите в диалоговом окне Свойства ++...++ вкладку Изделие. Введите для кабеля в поле Число штук / Количество значение, которое больше "0".

Обновите соединения и затем повторно запустите контрольный прогон.

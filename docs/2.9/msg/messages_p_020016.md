# P020016: \<x\> (Количество значений) \<y\> -> \<z\>

### Описание

Выполнено сравнение проектов, при котором ***Настройки*** проверяемого проекта (проект с замечаниями) сравнивались с настройками проекта для сравнения, а также была установлена разница. В этом сообщении \<x\> обозначает путь файла к настройке проекта, в которой были обнаружены различия между проверяемым проектом и проектом для сравнения. \<y\> и \<z\> указывают, на сколько значений изменилась настройка в проверяемом проекте.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

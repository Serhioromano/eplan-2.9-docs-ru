# P020037: Структура проекта '\<x\>: \<y\>' была изменена с '\<z\>' на '\<u\>'

### Описание

Выполнено сравнение проектов, при котором ***Структура*** проверяемого проекта (проекта с замечаниями) была сравнена со свойствами проекта для сравнения и были определены различия.

Это сообщение указывает, что в проверяемом проекте отображено изменение структуры страниц или устройств. \<x\> обозначает измененный элемент структуры, \<y\> — измененный блок идентификаторов, \<z\> — старую настройку, а \<u\> — новую настройку. Структуру страниц и устройств можно проследить или настроить на вкладке Структура. (Выделите проект в навигаторе страниц, выбрате пункты всплывающего меню Проект > Свойства, откройте вкладку Структура.)

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

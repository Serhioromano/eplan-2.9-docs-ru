# P501027: Данные вывода устройства на выводе шины заполнены

### Причина

В проверяемой базе данных изделий было найдено изделие, на котором определен шаблон функции с определением функции "Кабельный вывод сети / шины". На шаблоне функции данные выводов устройства (обозначения выводов устройства, описания выводов устройства, сечение вывода устройства и / или размер присоединения) заполнены значениями.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Выведите на передний план вкладку Шаблоны функции.
3. Удалите значения на шаблоне функции в полях Обозначения выводов устройства, Описания выводов устройств, Сечение вывода устройства и Размер присоединения.
4. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
5. Затем запустите новый контрольный прогон.

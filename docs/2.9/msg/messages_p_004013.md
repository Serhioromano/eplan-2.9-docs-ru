# P004013: Каркас '\<x\>': кольцевое соединение

### Причина

Обозначение каркаса и указанные данные обозначения каркаса для разъема карты ПЛК ссылаются на два различных блока ПЛК.

!!! example "Пример:"

    На первом блоке ПЛК в поле Каркасвведено обозначениеBGT1, а в поле Карта ПЛК вставлена вкаркас— обозначениеBGT2. На втором блоке ПЛК в поле Каркасвведено обозначениеBGT2, а в поле Карта ПЛК вставлена в каркас— обозначениеBGT1.

### Решение

1. Измените данные структуры ПЛК в диалоговом окне Свойства ++...++ для обоих блоков ПЛК, чтобы обозначения каркасов больше не ссылались друг на друга.
2. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Этот контрольный прогон служит только для проверки блоков ПЛК, на которых указан проект конфигураций.

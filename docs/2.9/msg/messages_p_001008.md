# P001008: Вывод клеммы имеет слишком много целей: \<x\>. Макс. число целей: \<y\>

### Причина

Как минимум одно из двух выводов устройства функции клеммы соединено с большим числом условных обозначений, чем указано.

### Решение

#### Возможность 1

Щелкните дважды в схеме соединений по клемме и выберите в диалоговом окне Свойства ++...++ вкладку Данные символа / функции. Щелкните в групповом поле Данные функции по кнопке ++Логика++. В диалоговом окне Логическая схема выводов устройства следует просмотреть, какие значения в строке Число целей настроены для обоих выводов устройства функции клеммы. Увеличьте до требуемого уровня настроенное значение для одного или обоих выводов устройства функции.

#### Возможность 2

Щелкните дважды в схеме соединений по клемме и выберите в диалоговом окне Dialog Свойства ++...++ вкладку Данные символа / функции. Щелкните в групповом поле Данные функции по кнопке ++Логика++. В диалоговом окне Логическая схема выводов устройства следует просмотреть, какие значения в строке Число целей настроены для обоих выводов устройства функции клеммы. Удалите соответствующие условные обозначения, которых размещено слишком много, на соответствующем выводе устройства функции клеммы в схеме соединений.

Обновите соединения и затем повторно запустите контрольный прогон.

# P005049: Несколько точек определения сети в одной сети

### Причина

В сети присутствует более чем одна точка определения сети.

!!! example "Пример:"

    В одной сети на схеме соединений содержится дветочки определения сети(см. кружок на следующем рисунке).

### Решение

В одной сети должна быть только ***одна*** точка определения сети. Лишние точки определения сети следует удалить. При этом рекомендуется действовать следующим образом:

1. Установите местоположение первой точки определения сети на схеме соединений, пользуясь функциональностью перейти к (графика), которую вы можете найти во всплывающем меню диалогового окна Управление сообщениями.
2. Найдите другие многократно размещенные точки определения сети, пользуясь функциональностью всплывающего меню Перейти ко 2-й координате.
3. Удалите многократно размещенную точку определения сети из схемы соединений.
4. Затем запустите новый контрольный прогон.

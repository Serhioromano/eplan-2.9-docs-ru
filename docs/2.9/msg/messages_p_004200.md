# P004200: Siemens: Неверное присвоение карты ПЛК: \<x\>

### Причина

EPLAN обнаружил неверное присвоение карты ПЛК в отношении используемого каркаса. В заполнителе \<x\> сообщения отображается неверно присвоенная карта ПЛК и способ корректировки присвоения.

### Решение

1. Установите местоположение блока ПЛК, представляющего карту ПЛК, на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно свойств блока ПЛК.
3. Переместите вкладку Данные структуры ПЛК на передний план.
4. Скорректируйте значение или настройку свойства, указанного в тексте сообщения, согласно значению по умолчанию (напр., для свойства Каркас, Карта ПЛК вставлена в каркас, На позиции / разъем или ЦПУ).
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

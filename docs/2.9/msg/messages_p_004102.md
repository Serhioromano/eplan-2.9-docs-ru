# P004102: Вывод шины в станции '\<x\>' не соединен через удлинительный кабель шины с выводом шины в рабочей станции '\<y\>'

### Причина

В проекте конфигураций два вывода шины, находящиеся на каркасах ***разных*** рабочих станций, соединены с помощью удлинительного кабеля шины. Это значит, что соединенным между собой выводам шины присвоена настройка системы шин "Local-Bus: Расширение". Эта настройка недопустима, если каркасы присвоены разным рабочим станциям.

### Решение

#### Возможность 1

1. Установите местоположение вывода шины рабочей станции '\<x\>' на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно 'Свойства' вывода шины.
3. Переместите вкладку Данные шины на передний план.
4. В раскрывающемся списке Система шин измените установленное значение "Local-Bus: Расширение" и выберите вместо него требуемую систему шин.
5. Щелкните по кнопке ++OK++.
6. Установите местоположение вывода шины рабочей станции '\<y\>' на схеме соединений, выбрав во всплывающем меню управления сообщениями функцию Перейти ко 2-й координате.
7. Выполните настройки, описанные в пунктах 2–5, для этого вывода шины.
8. Затем запустите новый контрольный прогон.

#### Возможность 2

1. Откройте диалоговое окно 'Свойства' соответствующих блоков ПЛК для обоих выводов шины и запомните имена рабочих станций, введенные на вкладке Данные структуры ПЛК.
2. Затем унифицируйте имя рабочей станции обоих блоков ПЛК.
3. Щелкните по кнопке ++OK++.
4. Затем запустите новый контрольный прогон.

# P004019: Нет датчика на входе

### Причина

При отслеживании цели не удалось определить датчик на входе ПЛК. Причиной может быть неправильно настроенный ***вид сигнала*** на входе ПЛК и / или на датчике. Кроме того, ***информация об отслеживании цели*** может отсутствовать на предвключенной для датчика клемме / предвключенном штекере.

### Решение

1. Сначала дважды щелкните вход ПЛК и в диалоговом окне Свойства ++...++ выберите вкладку Данные символа / функции.
2. Щелкните по кнопке ++Логика++.
3. В диалоговом окне Логическая схема выводов устройства проверьте свойство Вид сигнала на входе ПЛК. Это свойство ***должно*** совпадать с определением функции! Например, если вставлен вывод устройства ПЛК с определением функции "Вывод устройства ПЛК (дискр. вход)" или "Вывод устройства ПЛК (Fluid-техника), дискретный вход", то в поле Вид сигнала необходимо установить "Дискретный вход".
4. После этого проверьте, настроено ли для датчика свойство Вид сигнала = "Датчик".
5. Если между датчиком и входом ПЛК находится клемма или штекер, дополнительно проверьте для этого объекта настройку в поле Отслеживание цели (ПЛК) в. На выводе устройства функции 1 необходимо ввести значение "2", а на выводе устройства функции 2 — значение "1".
6. Обновите соединения и затем повторно запустите контрольный прогон.

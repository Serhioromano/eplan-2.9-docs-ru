# P011004: Соединение через точки разрыва на той же странице

### Причина

На одной и той же странице схемы соединений находится две точки разрыва, связанных перекрестной ссылкой.

### Решение

разместите одну из точек разрыва на другой странице схемы соединений.

Обновите соединения и затем повторно запустите контрольный прогон.

# P013050: Невозможно добавить покупное изделие

### Причина

При импорте спецификации невозможно было добавить покупное изделие, так как максимально возможное число было перезаписано.

!!! note "Замечание:"

    В EPLAN число импортируемых покупных изделий составляет максимум 1 000.

### Решение

Откройте файл импорта с помощью текстового или XML-редактора и удалите лишние покупные изделия. Сохраните файл и заново импортируйте спецификацию.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

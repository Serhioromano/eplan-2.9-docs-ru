# P020018: \<x\> различается в \<y\> Настройки

### Описание

Выполнено сравнение проектов, при котором ***Настройки*** проверяемого проекта (проект с замечаниями) сравнивались с настройками проекта для сравнения, а также была установлена разница. Это сообщение \<x\> для структуры папки области настройки в проверяемом проекте выводится на просмотр так же, как и отображается в структуре дерева диалогового окна Настройки (Пункты меню Параметры > Настройки). \<y\> указывает число настроек внутри области настройки, которые в проверяемом проекте выявляются на этапе сравнения проектов.

!!! example "Пример:"

    В области настройки Нумерация (автономно)проверяемого проекта схема нумерации изменилась. При этом были добавлены дополнительныеэлементы формата. При сравнении проектов сообщение могло бы выглядеть следующим образом:"Устройство+нумерация (автономно)+элементыформыразличаются в 5 настройках".

!!! note "Замечание:"

* Это относящееся к модулю сообщение ограничено областями настроек Нумерация с данными ПЛК и Нумерация (автономно).
* При выполнении нового контрольного прогона (любой настроенной схемы) его можно удалить из диалогового окна управления сообщениями.

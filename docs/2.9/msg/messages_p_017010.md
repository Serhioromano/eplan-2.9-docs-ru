# P017010: Накладывающиеся обозначения местоположения содержат противоречащие друг другу структурные идентификаторы

### Причина

В проекте, где для структурирования используется маркировка продукции, существуют два накладывающихся обозначения местоположения, которые в отношении одного или нескольких блоков идентификаторов содержат разные структурные идентификаторы.

!!! example "Пример:"

    Обозначение местоположения с идентификатором+O1накладывается на обозначение местоположения с идентификатором=A1+E1. По отношению к устройству-F1структурные идентификаторы для блока идентификаторов "Место установки" с идентификаторамиO1иE1противоречат друг другу.

### Решение

#### Возможность 1

1. Установите местоположение первого из накладывающихся обозначений местоположения на схеме соединений, пользуясь функцией Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно Свойства ++...++ обозначения местоположения и переместите вкладку Идентификатор на передний план.
3. Приведите структурный идентификатор для соответствующего блока идентификаторов в соответствие со структурой идентификаторов ***второго*** обозначения местоположения и подтвердите изменение, нажав ++OK++.
4. Затем запустите новый контрольный прогон.

#### Возможность 2

1. Установите местоположение второго из накладывающихся обозначений местоположения на схеме соединений, пользуясь функцией Перейти ко 2-й координате, которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно Свойства ++...++ обозначения местоположения и переместите вкладку Идентификатор на передний план.
3. Приведите структурный идентификатор для соответствующего блока идентификаторов в соответствие со структурой идентификаторов ***первого*** обозначения местоположения и подтвердите изменение, нажав ++OK++.
4. Затем запустите новый контрольный прогон.

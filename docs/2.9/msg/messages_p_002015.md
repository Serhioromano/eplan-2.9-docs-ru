# P002015: Локальные шаблоны функций несмотря на изделия с шаблонами функций

### Причина

В EPLAN шаблоны функций предоставляются изделиями. Такие функции, как определения штекеров, также доступны в качестве "локальных" шаблонов функций. Такие локальные шаблоны функций на определении штекера можно создать, например, в диалоговом окне Обработать штекер с помощью всплывающего меню. Локальные шаблоны функций, как правило, удаляются в случае присвоения определению штекера с локальными шаблонами функций изделия с шаблонами функций.

Поведение программы может быть другим, если применять в текущем проекте определения штекеров из более ранних версий EPLAN (до версии 2.4): Если такое определение штекера имеет локальные шаблоны функций и ему присваивается изделие с шаблонами функций, то удаление локальных шаблонов функций может не произойти. Также возможно, что EPLAN допускал создание шаблонов локальных функций, несмотря на то, что определению штекера из более ранних версий уже было присвоено изделие с шаблонами функций. Возникла одна из этих ситуаций, что привело к выводу сообщения.

### Решение

Локальные шаблоны функций на определении штекера невозможно удалить напрямую.

Поэтому рекомендуется следующий порядок действий:

1. Вначале установите местоположение соответствующего определения штекера на схеме соединений, пользуясь функцией Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Выберите пункт всплывающего меню Свойства.
3. В диалоговом окне свойств определения штекера переместите на передний план вкладку Изделие.
4. В поле Число штук / количество увеличьте введенное значение для одного из применяемых изделий. При этом не играет роли, на какое значение увеличивается число штук или количество. Важно лишь отображение изменения.
5. Закройте диалоговое окно Свойства, щелкнув по кнопке ++ОК++.

!!! info "Для сведения:"

    Локальные шаблоны функций удаляются.

6. Снова откройте диалоговое окно Свойства и откорректируйте увеличенное перед этим значение в поле Число штук / количество во вкладке Изделие.
7. Затем запустите новый контрольный прогон.

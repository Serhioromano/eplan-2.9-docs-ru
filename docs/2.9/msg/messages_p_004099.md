# P004099: Карта ПЛК с обозначением типа ПЛК '\<x\>' не вставлена в первичную станцию

### Причина

На карте ПЛК с обозначением типа ПЛК \<x\>, которая вставлена в определенный каркас, с помощью активированного свойства Карта ПЛК вставлена в первичную станцию документируется, что каркас является первичной станцией. Карта ПЛК, которая представляет каркас, ***не является*** первичной станцией, так как не хватает определенных данных. Также возможно, что первичная станция не определена.

### Решение

1. Установите местоположение вставленной в каркас карты ПЛК на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Выберите пункт Перейти ко 2-й координате из всплывающего меню, чтобы сразу перейти к блоку ПЛК, который представляет каркас.
3. Откройте диалоговое окно свойств блока ПЛК.
4. Переместите вкладку Данные структуры ПЛК на передний план.
5. Установите флажок для свойства Шинный интерфейс / первичная станция и / или ЦПУ в списке свойств.
6. Щелкните по кнопке ++OK++.
7. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Если перейти к каркасу невозможно, значит, он не определен. В таком случае выберите или создайте сначала карту ПЛК, которая будет служить в качестве каркаса. Присвойте этому каркасу (вторая вкладка, поле Каркас) значение, которое вы задали в карте ПЛК, которая должна быть вставлена в каркас (вторая вкладка, поле Карта ПЛК вставлена в каркас). Не забудьте задекларировать определенный таким образом каркас в качестве ***шинного интерфейса / первичной станции*** и / или ***ЦПУ***.

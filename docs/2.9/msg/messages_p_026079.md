# P026079: Экспорт Rittal - Athex: \<x\>

### Причина

При экспорте данных несущих шин для машины по оснащению клеммников "Rittal - Athex" была установлена недопустимая настройка и / или недопустимое состояние.

На то, о какой настройке и / или каком состоянии шла речь, указывает заполнитель \<x\> в сообщении. Выводится следующий заполнитель:

* Несущая шина не экспортирована из-за избыточной длины.

В этом случае была предпринята попытка экспортировать несущую шину, длина которой больше, чем максимальная длина шины, определенная в фирменных настройках для машины по оснащению клеммников Rittal - Athex.

### Решение

укоротить несущую шину

1. Выберите пункты меню Параметры > Настройки > Фирма > Автоматическое оснащение > Rittal - Athex и сначала определите, какая максимальная длина несущей шины задана для машины по оснащению клеммников.
2. Для этого в диалоговом окне Настройки: Rittal - Athex переместите вкладку Машина на передний план и запомните значение, сохраненное в поле Максимальная длина шины.
3. Перейдите в область настроек Пользователь > Графическая обработка > Область ввода данных и в диалоговом окне Настройки: Строка ввода установите флажок Всегда отображать область ввода данных (3D), чтобы иметь возможность изменять длину несущей шины с помощью области ввода данных.
4. Закройте диалоговое окно Настройки: Строка ввода нажатием кнопки ++OK++.
5. Установите местоположение несущей шины в пространстве листа, пользуясь функцией Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
6. Вызовите двойным щелчком диалоговое окно Настройки ++...++ несущей шины, переместите вкладку Изделие на передний план и запомните значение, записанное в поле Подмножество / длина данных ссылки изделия.
7. Закройте диалоговое окно Свойства ++...++ несущей шины, нажав на кнопку ++Отменить++.
8. Выберите пункты меню Обработать > Графика > Изменить длину и кратко щелкните по нужному концу той несущей шины в пространстве листа, которую вы хотите укоротить.

!!! info "Для сведения:"

    Рядом с курсором на выделенном конце несущей шины появится точка захвата красного цвета.

!!! info "Для сведения:"

    Под точкой захвата отображается активированная прежде область ввода данных.

9. Укоротите несущую шину, передвинув точку захвата при помощи мышки в направлении другого конца несущей шины. Проследите при этом за значением, отображаемым в области ввода данных.
10. Щелкните левой кнопкой мыши, если значение в области ввода данных совпадает со значением фирменной настройки для машины по оснащению клеммников Athex (или меньше него).
11. Завершите изменение длины несущей шины с помощью пункта всплывающего меню Прервать операцию.
12. Затем запустите контрольный прогон и снова экспортируйте данные несущей шины.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

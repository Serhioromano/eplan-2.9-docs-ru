# P004096: Многократно заданный адрес (отказоуст.) '\<x\>'

### Причина

Как минимум в двух релевантных для безопасности блоках ПЛК введено одинаковое значение свойства Адрес (отказоуст.).

### Решение

1. Определите местоположение блоков ПЛК на схеме соединений с помощью функции Перейти к (графика) и Перейти ко 2-й координате во всплывающем меню диалогового окна Управление сообщениями.
2. Поочередно откройте диалоговое окно 'Свойства' каждого блока ПЛК.
3. Переместите первую вкладку на передний план.
4. Введите в списке свойств в поле Адрес (отказоуст.) уникальное значение.
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

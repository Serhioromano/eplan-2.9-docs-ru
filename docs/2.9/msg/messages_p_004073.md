# P004073: Многократно заданный 'Функциональный текст (автоматич.)' (в языке '\<x\>') в одном ЦПУ

### Причина

На выводах устройства ПЛК ЦПУ на свойстве Функциональный текст (автоматически) одна и та же запись многократно введена на языке \<x\>.

### Решение

1. Установите местоположение выводов устройства ПЛК на схеме соединений, с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Измените функциональный текст или функциональный текст зоны для соответствующих выводов устройства ПЛК на указанном языке таким образом, чтобы каждая запись была однозначной.
3. Переведите измененные функциональные тексты или функциональные тексты зон на все остальные используемые вами языки.
4. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

* Сравнение функциональных текстов выполняется отдельно для каждого языка.
* Независимые от языка (= непереведенные) функциональные тексты сравниваются только с непереведенными текстами. Таким образом функциональные тексты должны быть переведены (или повторно переведены после изменения) для выполнения проверки.
* Если функциональный текст пустой, сообщение не появится.

# P028013: Имеется более чем одно графическое размещение для объекта планирования в виде представления '\<x\>'

### Причина

На одной или нескольких страницах предварительного планирования вашего проекта имеется более одного графического размещения для какого-либо объекта планирования в виде представления \<x\>.

### Решение

1. Установите местоположение графически многократно размещенного объекта планирования, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Удалите избыточное графическое размещение.
3. Повторите операцию удаления для других избыточных размещений.
4. Затем запустите новый контрольный прогон.

# P004202: Siemens: Неверный вывод шины: \<x\>

### Причина

EPLAN обнаружил неверный вывод шины. В заполнителе \<x\> сообщения отображается неверно присвоенная карта ПЛК и свойства вывода шины, которые следует откорректировать.

### Решение

1. Установите местоположение вывода шины на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно "Свойства" вывода шины.
3. Переместите вкладку Вывод устройства ПЛК на передний план.
4. Проверьте значение свойства Обозначение штекера и откорректируйте его в случае необходимости.
5. Переместите вкладку Данные шины на передний план, если отображается сообщение об ошибках в других свойствах, кроме обозначения штекера.
6. Проверьте значения свойств, указанных в тексте сообщения, и откорректируйте их при необходимости (например, для свойства Система шин или Интерфейс шины: Имя).
7. Щелкните по кнопке ++OK++.
8. Затем запустите новый контрольный прогон.

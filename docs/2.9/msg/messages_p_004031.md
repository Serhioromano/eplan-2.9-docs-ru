# P004031: Многократно заданный символический адрес в одном ЦПУ

### Причина

Внутри карты ЦПУ существует вывод устройства ПЛК с символическим адресом, который предоставляется другим выводом ПЛК.

### Решение

Откройте диалоговое окно Свойства ++...++ для соответствующего вывода устройства ПЛК и измените запись в поле Символический адрес.

Затем запустите новый контрольный прогон.

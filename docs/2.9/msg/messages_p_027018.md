# P027018: Вывод устройства \<x\> отсутствует и не может быть создан

### Причина

На функции топологии отсутствует указанный вывод соединения \<x\> на схеме соединений, и его невозможно сгенерировать при маршрутизации.

Сообщение появляется на многократно отображенных клеммниках.

### Решение

1. Установите местоположение функции топологии в своей сети соединенных сегментов при помощи функциональности Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно Свойства ++...++ функции топологии и выполните замену во вкладке Схема соединений.
3. Выделите таблицу и выберите в раскрывающемся меню пункт Обновить.
4. Удалите из таблицы все выводы, которые не должны быть соединены на данном представлении клеммника. (Например, это могут быть выводы устройства, которые соединены с целями в другой сети соединенных сегментов.)
5. Закройте диалоговое окно Свойства ++...++, для чего нажмите на кнопку ++OK++, и еще раз маршрутизируйте соединения.
6. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

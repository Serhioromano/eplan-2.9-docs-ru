# P026096: Дополнительная длина арматуры равна 0

### Причина

В пространстве листа размещен 3D-макрос арматуры. На схеме соединений графического макроса дополнительной длине соответствует значение "0".

### Решение

1. Установите местонахождение 3D-макроса в пространстве листа с помощью функции Перейти к (графика) во всплывающем меню диалогового окна Управление сообщениями.
2. Внесите в схему соединений 3D-макроса значение больше 0.
3. Затем запустите новый контрольный прогон.

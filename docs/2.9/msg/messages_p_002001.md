# P002001: Вывод контакта штекера имеет слишком много целей: \<x\>. Макс. число целей: \<y\>

### Причина

Как минимум одно из двух выводов устройства функции контакта штекера соединено с большим числом условных обозначений, чем указано.

### Решение

#### Возможность 1

Щелкните дважды в схеме соединений на контакт штекера и выберите в диалоговом окне Свойства ++...++ вкладку Данные символа / функции. Щелкните в групповом поле Данные функции (Логика) по кнопке ++Логика++. В диалоговом окне Логическая схема выводов устройства следует проверить, какие значения введены в строке Число целей для обоих выводов устройства функции контакта штекера Увеличьте до требуемого уровня настроенное значение для одного или обоих выводов устройства функции.

#### Возможность 2

Щелкните дважды в схеме соединений на контакт штекера и выберите в диалоговом окне Свойства ++...++ вкладку Данные символа / функции. Щелкните в групповом поле Данные функции (Логика) по кнопке ++Логика++. В диалоговом окне Логическая схема выводов устройства следует проверить, какие значения введены в строке Число целей для обоих выводов устройства функции контакта штекера Удалите соответствующие условные обозначения, которых размещено слишком много, на соответствующем выводе устройства функции контакта штекера в схеме соединений.

Обновите соединения и затем повторно запустите контрольный прогон.

# P007030: Идентификатор ОУ '\<x\>' отличается от идентификаторов ОУ на шаблонах изделия '\<y\>' ('\<z\>')

### Причина

На функции (главной или вспомогательной), являющейся составной частью макроса изделия для модуля, внесенный идентификатор ОУ '\<x\>' отличается от идентификатора ОУ '\<z\>', указанного на соответствующем шаблоне функции на изделии модуля '\<y\>'.

### Решение

1. Определите местоположение функции макроса изделия, размещенного на схеме соединений, присвоенного модулю. Для этого используйте функцию Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно 'Свойства' функции и найдите внесенный идентификатор ОУ в списке свойств.
3. Откройте базу данных изделий и вызовите свойства изделия модуля.
4. Переместите вкладку Шаблоны функций на передний план и проверьте идентификатор, внесенный для соответствующей функции в столбце Нижестоящее ОУ / идентификатор ОУ.
5. Перейдите на вкладку Модуль и на изделии функции внесите идентификатор ОУ, который был найден на функции размещенного на схеме соединений макроса изделия.
6. Выберите ++Дополнительно++ > Суммировать шаблоны функций.

!!! info "Для сведения:"

    Внесенный идентификатор ОУ функции изменяется на шаблоне функции изделия модуля.

7. Закройте базу данных изделий и сохраните измененные данные изделия.
8. Затем запустите новый контрольный прогон.

# P004053: Между структурой сети в файле импорта и в проекте найдено слишком много различий! Пожалуйста, проверьте все структуры сети

### Причина

При последнем импорте данных ПЛК было найдено соединение, противоречащее существующему сейчас в проекте соединению. Неверное соединение не могло быть создано по причине сложного противоречия.

### Решение

Проверьте, все ли выводы шины соединены с правильной физической сетью.

!!! note "Замечание:"

    Это относящееся к модулю сообщение выводится при импорте данных ПЛК. При выполнении нового контрольного прогона (любой настроенной схемы) его можно удалить из диалогового окна управления сообщениями.

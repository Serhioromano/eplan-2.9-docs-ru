# P004097: Больше одного вывода устройства В/В на один канал в виде сигнала в подустройстве

### Причина

Определено больше одного вывода устройства В/В на один канал в виде сигнала для подустройства ПЛК.

Этот контрольный прогон служит для проверки карт ПЛК, имеющих входы и выходы, которыми можно управлять как подустройствами. Для таких карт допускается, например, следующее присвоение каналов:

E1 => канал 1, E2 => канал 2 и т. д.

A1 => канал 1, A2 => канал 2 и т. д.

Критерием проверки является вид сигнала (цифровой / аналоговый вход / выход ПЛК ) и индекс на подустройстве.

Для многофункциональных выводов в логической схеме выводов устройства учитывается сохраненный вид сигнала.

Выводы устройства ПЛК с определением функции "Вывод устройства ПЛК, внутр., общ." не проверяются этим контрольным прогоном, поскольку они служат только для отображения свободных символических адресов.

### Решение

Присвойте каждому каналу только один вывод устройства В/В для каждого подустройства. Затем введите в диалоговом окне Свойства ++...++ однозначное обозначение канала для каждого вывода устройства В/В.

Затем запустите новый контрольный прогон.

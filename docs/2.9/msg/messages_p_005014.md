# P005014: Разный тип вывода соединения и вывода устройства

### Причина

На выводах устройства функций целей соединения настроен одинаковый тип вывода устройства. Кроме того, соединение снабжено точкой определения соединения, и ей присвоено определение функции, которое не совпадает с типом вывода устройства целей.

!!! note "Замечание:"

    Это сообщение не выводится при следующих условиях:

* соединение типа "Перемычка"
* соединение типа "Мостовая перемычка"
* соединение типа "Проволочная перемычка"
* соединение типа "переключаемая перемычка";
* соединение типа "внутренняя перемычка";
* соединение типа "вставная перемычка".
* соединение типа "Прямое соединение"
* соединение типа "Не определено"
* вывод устройства типа "Не определен"

### Решение

Присвойте одинаковый тип вывода устройства точке определения соединения и целям соединения.

Обновите соединения и затем повторно запустите контрольный прогон.

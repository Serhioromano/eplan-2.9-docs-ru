# P027028: Длина сегмента маршрутизации больше, чем поставляемая длина изделия

### Причина

Одному из размещенных сегментов маршрутизации в сети соединенных элементов было присвоено изделие с определенной поставляемой длиной. Однако длина сегмента маршрутизации больше, чем поставляемая длина, указанная на изделии.

### Решение

1. Установите местоположение сегмента маршрутизации внутри сети соединенных сегментов, пользуясь функцией Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Сократите длину сегмента маршрутизации минимум на размер поставляемой длины или уменьшите сам сегмент маршрутизации. Или в качестве альтернативы выберите для него изделие с поставляемой длиной, превышающей длину сегмента маршрутизации.
3. Затем запустите новый контрольный прогон.

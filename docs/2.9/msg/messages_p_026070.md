# P026070: Гнутая сборная шина с недействительной общей длиной

### Причина

Не удалось правильно вычислить общую длину. В текущей таблице изгибов на момент размещения функционального элемента не было действительных коэффициентов поправки.

### Решение

1. Установите местоположение гнутой сборной шины в пространстве листа с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Дополните таблицу изгибов.
3. После корректировки выберите пункт всплывающего меню Обновить размещение изделия, чтобы обновить уже размещенные функциональные элементы.

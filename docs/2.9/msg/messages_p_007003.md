# P007003: Не удалось присвоить все изделия. Достигнуто макс. число изделий

### Причина

При суммировании длины кабелей достигнуто максимально допустимое количество покупных изделий "1000".

### Решение

Сократите количество покупных изделий. Например, удалите лишние покупные изделия.

!!! note "Замечание:"

    Это сообщение, относящееся к модулю, выведено посредством автоматического суммирования длины кабеля. При выполнении нового контрольного прогона (любой настроенной схемы) его можно удалить из диалогового окна управления сообщениями.

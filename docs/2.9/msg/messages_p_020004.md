# P020004: Число различий: \<x\>

### Описание

Выполнено сравнение проектов, при котором ***Настройки*** проверяемого проекта (проект с замечаниями) сравнивались с настройками проекта для сравнения. В этом сообщении при помощи \<x\> указано число обнаруженных различий.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

# P020012: \<x\> различается в типе данных

### Описание

При восстановлении выгруженного проекта настройки защищенного от записи исходного проекта сравнивались с настройками выгруженного проекта. В отношении типа данных имеется различие.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

# P004087: Отсутствует начальный адрес

### Причина

На блоках ПЛК, имеющих входы и / или выходы, на свойстве Начальный адрес 1 карты ПЛК и / или свойстве Начальный адрес 2 карты ПЛК отсутствует требуемое значение.

### Решение

1. Установите местоположение блока ПЛК на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно свойств блока ПЛК.
3. Переместите первую вкладку на передний план.
4. В таблице свойств заполните поля значений свойств Начальный адрес 1 карты ПЛК и Начальный адрес 2 карты ПЛК в соответствии со следующими условиями:
Если блок ПЛК содержит ***только входы***, значение начального адреса необходимо указать для свойства Начальный адрес 1 карты ПЛК.
Если блок ПЛК содержит ***только выходы***, значение начального адреса необходимо указать для свойства Начальный адрес 1 карты ПЛК или Начальный адрес 2 карты ПЛК.
Если блок ПЛК содержит ***входы и выходы***, на ***обоих*** свойствах необходимо указать значение для соответствующего начального адреса.
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

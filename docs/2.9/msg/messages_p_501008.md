# P501008: Записанная на изделии схема соединений \<x\> отсутствует

### Причина

В проверяемой базе данных изделий было найдено изделие, на котором сохранена схема соединений, которую EPLAN не может найти.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Переместите вкладку Технические данные на передний план.
3. Щелкните рядом с полем Схема соединений по кнопке ++...++.
4. Выберите в открывшемся диалоговом окне выбора нужную схему соединений.
5. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
6. Затем запустите новый контрольный прогон.

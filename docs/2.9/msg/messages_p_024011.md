# P024011: Цель группы соединений без полного ОУ

### Причина

У одного или нескольких соединений группы соединений в подключенной исходной или целевой функции отсутствует полное обозначение устройства.

### Решение

1. Установите линию определения группы соединений на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Во всех точках определения соединения группы соединений отследите подключенные к соответствующему соединению исходные и целевые функции.
3. Откройте диалоговое окно "Свойства" для каждой исходной и целевой функции.
4. Добавьте полное обозначение устройства везде, где оно не введено.
5. Подтвердите ввод.
6. Затем запустите новый контрольный прогон.

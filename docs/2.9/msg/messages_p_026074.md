# P026074: Экспорт ЧУ: \<x\>

### Причина

При экспорте данных обработки ЧУ была обнаружена недостающая настройка.

О недостающей настройке в сообщении указывает заполнитель \<x\>. Доступны следующие заполнители:

* Определение контура не содержит записи данных контура ЧУ для машины
* Определение контура не содержит записей данных контура ЧУ
* Определение контура не было найдено
* Отсутствует инструмент для отверстия D<Значение a>
* Отсутствует инструмент для резьбы M<Значение a>
* Монтажное отверстие <Вариант формы> слишком мало для выбранного инструмента
* В качестве альтернативы для отверстия D<Значение a> выполняется отверстие D<Значение b>
* В качестве альтернативы фрезеруется отверстие D<Значение a>
* В качестве альтернативы для резьбы M<Значение a> выполняется резьба М<Значение b>
* В качестве альтернативы для резьбы M<Значение a> нарезается резьба М<Значение b>
* В качестве альтернативы для резьбы M<Значение a> фрезеруется отверстие под резьбу D<Значение b>
* Продольное отверстие "Высота = Длина" не экспортировано
* Не удалось экспортировать повернутую запретную зону для сверления.

Дополнительные сведения:

* Значение а = значение диаметра, который должен быть выполнен (из размещения)
* Значение b = значение диаметра, который выполняется в действительности
* Вариант формы = заполнитель для форм "Прямоугольник", "Продольное отверстие", "Восьмиугольник", "Шестиугольник"
* "Продольное отверстие "Высота = Длина" не экспортировано": схема сверления была неверно определена в базе данных изделий.

### Решение

1. Заполните и / или измените настройку экспорта данных обработки ЧУ в соответствии с заполнителем.

"Продольное отверстие "Высота = Длина" не экспортировано": измените тип сверления на "Сверлить".

2. Еще раз выполните экспорт данных обработки ЧУ.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

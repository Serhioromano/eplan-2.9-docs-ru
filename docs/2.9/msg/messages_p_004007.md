# P004007: Каркас '\<x\>' вставлен сам в себя

### Причина

В блоке ПЛК, содержащем данные шины, в полях Каркас и Карта ПЛК вставлена в каркас введены идентичные обозначения, т. е. согласно этим данным каркас вставлен сам в себя.

### Решение

1. Откройте диалоговое окно Свойства ++...++ соответствующего блока ПЛК.
2. На первой вкладке в полях Каркас и Карта ПЛК вставлена в каркас введите разные данные.
3. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Этот контрольный прогон служит только для проверки блоков ПЛК, на которых указан проект конфигураций.

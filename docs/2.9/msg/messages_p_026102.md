# P026102: Невозможно перенести изделие \<x\>.

!!! note "Замечание:"

    Это сообщение контрольного прогона относится к контексту переноса системы линейно присоединенных шкафов Rittal TS 8 на систему линейно присоединенных шкафов Rittal VX 25.

### Причина

Выведенное в заполнителе \<x\> изделие Rittal TS8 перенести невозможно, поскольку это деталь, снятая с производства.

### Решение

Разместите в этом месте другое изделие Rittal VX25.

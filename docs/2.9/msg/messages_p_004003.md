# P004003: Относящееся к указанному значению конфигурации '\<x\>' свойство '\<y\>' блока ПЛК не содержит число

### Причина

При адресации вывода устройства ПЛК используется схема адресации, для которой было указано определенное значение конфигурации в элементе формата "Счетчик" (например, "На позиции / разъем"). Однако в диалоговом окне Свойства ++...++ соответствующего блока ПЛК на вкладке Данные структуры ПЛК в соответствующем поле введено нечисловое значение (например, в поле На позиции / разъем).

### Решение

Введите для блока ПЛК только числовое значение в поле, которое было указано при адресации как требуемое значение конфигурации для элемента формата "Счетчик".

!!! note "Замечание:"

    Это относящееся к модулю сообщение выводится при прогоне адресации ПЛК. При выполнении нового контрольного прогона (любой настроенной схемы) его можно удалить из диалогового окна управления сообщениями.

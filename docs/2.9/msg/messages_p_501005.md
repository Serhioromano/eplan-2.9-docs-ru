# P501005: Присвоенный макросу \<x\> (\<y\> / \<z\>) слой \<u\> определен пользователем

### Причина

В проверяемой базе данных изделий было найдено изделие, на котором сохранен макрос с определенным пользователем слоем.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Переместите вкладку Технические данные на передний план.
3. Щелкните рядом с полем Макрос по кнопке ++...++ и выберите в открывшемся диалоговом окне выбора файл макроса, который содержит макрос с присвоенным специфическим для EPLAN слоем.

Или в качестве альтернативы удалите запись в поле Макрос.

Или создайте ***новый*** макрос со специфическим для EPLAN слоем, сохраните его в нужном каталоге макросов и выберите его в поле Макрос.
4. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
5. Затем запустите новый контрольный прогон.

# P011003: Точка разрыва без цели

### Причина

Точка разрыва не имеет цель.

### Решение

Соедините как минимум один вывод устройства точки разрыва с другим условным обозначением, чтобы образовалась линия автоматического соединения.

Обновите соединения и затем повторно запустите контрольный прогон.

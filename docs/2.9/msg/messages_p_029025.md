# P029025: \<x\>: Обязательное свойство '\<y\>' пусто

### Причина

В специфических для проекта настройках вы указали для определения сегмента \<x\> обязательное свойство \<y\>. Однако вы забыли сохранить в предварительном планировании значение определенного обязательного свойства для данного определения сегмента.

### Решение

1. Установите местоположение отображенного определения сегмента в проекте с помощью функции Перейти к (Графика) или Синхронизировать выбор, которые находятся во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно Свойства ++...++ определения сегмента и переместите соответствующую вкладку со свойствами на передний план.
3. Присвойте значение обязательному свойству определения сегмента, выведенному в диалоговом окне Управление сообщениями, и закройте диалоговое окно Свойства ++...++, нажав кнопку ++OK++.
4. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Этот контрольный прогон можно свободно конфигурировать с помощью диалогового окна Настройки: Глубина предварительного планирования (вызов диалогового окна: Параметры> Настройки> Проекты > "Имя проекта" > Предварительное планирование > Глубина предварительного планирования).

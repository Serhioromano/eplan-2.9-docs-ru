# P026040: Соединение/кабель без действительного диаметра (\<x\>)

### Причина

При расчете степени заполнения кабельных каналов программа EPLAN обнаружила, что на используемом отдельном соединении или на используемом кабеле не был задан действительный диаметр.

Диаметр является недействительным, если для него либо вообще не указано значение, либо указанное значение лежит за пределами диапазона от 0 до 500 мм.

### Решение

1. Укажите для отдельного соединения или для кабеля действительный диаметр.
2. Затем запустите новый контрольный прогон и еще раз обновите степень заполнения (пункты меню Данные проекта > Соединения > Обновить степень заполнения.)

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

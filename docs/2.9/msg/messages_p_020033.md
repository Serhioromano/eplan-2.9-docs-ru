# P020033: Потенциал \<x\>: \<y\>

### Описание

Выполнено сравнение проектов, при котором ***Свойства*** проверяемого проекта (проекта с замечаниями) были сравнены со свойствами проекта для сравнения и были определены различия.

Это сообщение указывает, что при сравнении свойств потенциала (определения потенциала или вывода устройства потенциала) было обнаружено различие. \<x\> указывает имя потенциала, \<y\> говорит о том, был ли потенциал удален, добавлен или было изменено одно из его свойств.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

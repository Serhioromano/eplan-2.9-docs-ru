# P004024: Отсутствующее описание вывода устройства

### Причина

На выводе устройства ПЛК отсутствует описание вывода устройства.

Выводы устройства ПЛК с определением функции "Вывод устройства ПЛК, внутр., общ." не проверяются этим контрольным прогоном, поскольку они служат только для отображения свободных символических адресов.

### Решение

1. В диалоговом окне Свойства ++...++ вывода устройства ПЛК введите отсутствующее описание вывода устройства в одноименное поле.
2. Затем запустите новый контрольный прогон.

# P010002: Многократное главное размещение одноименных обозначений местоположений

### Причина

Два или более, так как два обозначения местоположения с одинаковым ОУ обладают свойством Главное размещение.

### Решение

Выделите в диалоговом окне Управление сообщениями присвоенное сообщению обозначение устройства. С помощью пунктов меню Перейти к 1-й координате и Перейти ко 2-й координате всплывающего меню перейдите последовательно к соответствующим обозначениям местоположения на схеме соединений. Для этого двойным щелчком мыши по точке вставки вызовите свойства соответствующего обозначения местоположения. Снимите на всех, кроме одного, обозначениях местоположения флажок Главное размещение, чтобы только ***одно*** обозначение местоположения обладало этим свойством. При необходимости повторите этот процесс для других обозначений местоположения.

Затем запустите новый контрольный прогон.

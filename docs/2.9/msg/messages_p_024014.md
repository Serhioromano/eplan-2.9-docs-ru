# P024014: Длина группы соединений не указана

### Причина

На главной функции группы соединений отсутствует значение для длины группы соединений.

### Решение

1. Определите из диалогового окна Управление сообщениями главную функцию группы соединений в схеме соединений с помощью функции Перейти к (графика).
2. Щелкните там двойным щелчком по выделенной линии определения кабеля и вызовите диалоговое окно Свойства ++...++ группы соединений.
3. Перейдите — если не установлено по умолчанию — на вкладку Группа соединений / провод и введите значение в поле Длина.
4. Щелкните по кнопке ++OK++.
5. Затем запустите новый контрольный прогон.

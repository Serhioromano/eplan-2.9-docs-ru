# P007020: Данные защищенной главной функции или главной клеммы не совпадают с данными изделия: \<x\>

### Причина

Данные защищенной главной функции / главной клеммы, которую вы разместили как изделие с заданными шаблонами функции (устройство), дополнительно были изменены в отношении как минимум одного свойства (напр., свойства Технические параметры). Такое изменение защищенной главной функции / главной клеммы с уже выбранными данными изделия не предусмотрено.

Для главной клеммы, которая представлена в распределенном виде, проверяется вся функция клеммы. В этом случае данные, введенные на отдельных распределенных клеммах одной и той же функции клеммы, объединяются.

### Решение

1. Запомните свойство, выведенное в сообщении.
2. С помощью функций Перейти к установите местоположение главной функции / главной клеммы в проекте.
3. Вызовите свойства защищенной главной функции / главной клеммы и сравните данные обозначенного свойства с данными соответствующего свойства выбранного изделия.
4. Если данные отличаются, отмените дополнительное изменение защищенной главной функции / главной клеммы или выберите другое изделие, имеющее соответствующие данные.
5. Затем запустите новый контрольный прогон.

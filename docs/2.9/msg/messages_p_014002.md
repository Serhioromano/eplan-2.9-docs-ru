# P014002: Имя макроса, определенного на пространстве листа, '\<x\>' существует многократно

### Причина

При генерации макросов из проекта макросов в пространстве листа было определено имя макроса, уже присвоенное другому пространству листа. Или для одного и того же пространства листа в определении макроса были сохранены несколько идентичных имен макроса. Соответствующее пространство листа и одинаковые имена макросов \<x\> отображаются в столбцах Пространство листа и Текст сообщения в диалоговом окне Управление сообщениями.

### Решение

1. Откройте навигатор пространств листа.
2. Установите местоположение пространства листа в проекте с помощью функции Синхронизировать выбор, которая находится во всплывающем меню диалогового окна Управление сообщениями.

!!! info "Для сведения:"

    Пространство листа будет выделено в навигаторе пространств листа.

3. Откройте с помощью всплывающего меню навигатора пространства листа диалоговое окно свойств пространства листа.
4. Переместите вкладку Пространство листа на передний план.
5. В поле Макрос: Имя таблицы свойств измените имя макроса. Выберите имя, которое еще не задавалось другому пространству листа.
6. Перейдите на вкладку Определение макроса и проверьте, нет ли под 1-й строкой в столбце таблицы Макрос: Имя других строк с идентичным именем макроса.
7. Если есть, удалите повторяющиеся имена макроса или замените их на уникальные имена.
8. Закройте диалоговое окно Свойства, нажав кнопку ++ОК++.
9. Запустите новый контрольный прогон.
10. Повторите шаги 2–9 для других отображенных в диалоговом окне Управление сообщениями пространств листа.

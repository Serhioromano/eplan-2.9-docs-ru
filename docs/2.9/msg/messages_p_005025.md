# P005025: Слишком много соединений к одному выводу устройства

### Причина

Как минимум один из выводов устройства функции условного обозначения соединен с большим число условных обозначений, чем указано.

### Решение

#### Возможность 1

Щелкните дважды в схеме соединений на условное обозначение и выберите в диалоговом окне Свойства ++...++ вкладку Данные символа / функции. Щелкните в групповом поле Данные функции (Логика) по кнопке ++Логика++. В диалоговом окне Логическая схема выводов устройства следует просмотреть, какие значения в строке Число целей настроены для обоих выводов устройства функции условного обозначения. Увеличьте до требуемого уровня настроенное значение для одного или обоих выводов устройства функции.

#### Возможность 2

Щелкните дважды в схеме соединений на условное обозначение и выберите в диалоговом окне Свойства ++...++ вкладку Данные символа / функции. Щелкните в групповом поле Данные функции (Логика) по кнопке ++Логика++. В диалоговом окне Логическая схема выводов устройства следует просмотреть, какие значения в строке Число целей настроены для обоих выводов устройства функции клеммы. Удалите соответствующие условные обозначения, которых размещено слишком много, на соответствующем выводе устройства функции условного обозначения.

Обновите соединения и затем повторно запустите контрольный прогон.

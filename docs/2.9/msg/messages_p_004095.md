# P004095: Недостающий адрес (отказоуст.)

### Причина

На блоке ПЛК отсутствует релевантное для безопасности значение свойства Адрес (отказоуст.)

### Решение

1. Установите местоположение блока ПЛК на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно свойств блока ПЛК.
3. Переместите первую вкладку на передний план.
4. Введите в списке свойств в поле Адрес (отказоуст.) недостающее значение.
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

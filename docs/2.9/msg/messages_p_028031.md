# P028031: Указанное на сегменте изделие '\<x\>' не содержится в классе трубы '\<y\>'

### Причина

В предварительном планировании на сегменте указан номер изделия, не совпадающий с номером изделия из выбранного класса трубы.

### Решение

#### Возможность 1

1. Откройте ***навигатор шаблонов сегментов*** и вызовите диалоговое окно "Свойства" класса трубы, выведенное в заполнителе \<y\> сообщения.
2. Переместите вкладку Изделие класса трубы на передний план.
3. Добавьте в таблицу выведенный в тексте сообщения номер изделия \<x\> сегмента.
4. Затем запустите новый контрольный прогон.

#### Возможность 2

1. Установите местоположение сегмента на странице предварительного планирования с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями. Или с помощью функции Синхронизировать выбор всплывающего меню перейдите к сегменту в навигаторе предварительного планирования.
2. Вызовите диалоговое окно "Свойства" сегмента.
3. На первой вкладке выберите класс трубы, на котором указан номер изделия, выведенный в тексте сообщения \<x\>.
4. Затем запустите новый контрольный прогон.

# P001001: Клемма без обозначения

### Причина

Вы создали клемму без обозначения в навигаторе клеммников или разместили ее в схеме соединений.

### Решение

Дважды щелкните в навигаторе клеммников или схеме соединений по соответствующей клемме и введите в диалоговом окне Свойства ++...++ отсутствующее обозначение.

Затем запустите новый контрольный прогон.

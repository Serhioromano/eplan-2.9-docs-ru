# P024013: Группа соединений: 'Функциональный текст (автоматически)' различается (в языке '\<x\>')

### Причина

На группе соединений, распределенно размещенной на схеме соединений, отличается свойство Функциональный текст (автоматически) в одном или нескольких размещениях группы соединений на выбранном языке отображения.

### Решение

1. Установите местоположение главной функции группы соединений на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте в главной функции всплывающее меню и с помощью пункта меню Перейти к (обратный эквивалент) перейдите к вспомогательным функциям группы соединений.
3. Откройте диалоговое окно "Свойства" в основной и дополнительных функциях группы соединений и унифицируйте для всех размещений свойство Функциональный текст (автоматически) на выбранном языке отображения.
4. Подтвердите ввод.
5. Затем запустите новый контрольный прогон.

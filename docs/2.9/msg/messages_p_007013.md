# P007013: Изделие \<x\> сохранено, но не вариант \<y\>. Имеется вариант \<z\>

### Причина

Вы разместили устройство и вручную выбрали вариант изделия, отсутствующий в основных данных изделий. Однако в этом случае этот вариант изделия не сохранен в проекте.

### Решение

Создайте в базе данных изделий новый вариант изделия для выбранного устройства. Разместите устройство с этим вариантом

Затем запустите новый контрольный прогон.

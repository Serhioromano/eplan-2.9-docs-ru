# P022027: Текст слишком мал

### Причина

На странице проекта был найден текст, размер шрифта которого меньше значения, определенного в специфических для проекта настройках в качестве минимального размера шрифта.

### Решение

1. Прежде всего установите местоположение текста в проекте с помощью функциональности Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно Свойства ++...++ текста, переместите вкладку Формат на передний план и найдите значение, записанное в поле Размер шрифта.
3. Выберите пункты меню Настройки > Параметры > "Имя проекта" > Графическая обработка > Общее и найдите настроенное значение в поле Минимальный размер шрифта (проверка на наличие слишком мелких текстов).
4. Либо увеличьте значение размера шрифта текста так, чтобы оно было больше настроенного минимального размера шрифта. Либо уменьшите значение специфической для проекта настройки в поле Минимальный размер шрифта (проверка на наличие слишком мелких текстов), так чтобы размер шрифта, используемый в данный момент в тексте, оказался в диапазоне допустимых и сообщение о слишком мелком тексте не выводилось.
5. Затем запустите новый контрольный прогон.

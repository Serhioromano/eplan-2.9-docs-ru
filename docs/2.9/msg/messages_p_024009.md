# P024009: Соединение группы без группы соединений / провода

### Причина

Точка определения соединения группы больше не находится на точке пересечения с линией определения группы соединений. Поэтому EPLAN не может присвоить группе соединений точку определения соединения.

### Решение

1. Установите местоположение данной точки определения соединения на схеме соединений, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Переместите точку определения соединения на точку пересечения линии определения группы соединений с соединением группы или заново начертите линию определения группы соединений.
3. Обновите соединения и затем повторно запустите контрольный прогон.

!!! note "Замечание:"

    Если перемещенная точка определения соединения группы имеет такое же ОУ, как и определение группы соединений, соединение идентифицируется группой соединений. Поэтому сообщение не выводится.

# P501031: Внесенное на изделии общее определение функции '\<x\>' не должно использоваться на изделии

### Причина

В проверяемой базе данных изделий обнаружено изделие, на котором сохранено "общее" определение функции. Общие определения функций предназначены для использования в проекте, если при проектировании еще не установлено точно, о каком изделии идет речь. В базе данных изделий необходимо вводить только точные определения функций.

!!! note "Замечание:"

    Это сообщение выводится только для клемм.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Выведите на передний план вкладку Шаблоны функции.
3. Щелкните в таблице Выбор устройства (шаблоны функций) по записи в столбце Определение функции, который содержит общее определение функции, и нажмите кнопку ++...++.
4. Выберите в открывшемся диалоговом окне Определения функции точное определение функции.
5. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
6. Затем запустите новый контрольный прогон.

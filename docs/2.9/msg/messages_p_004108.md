# P004108: Определенное пользователем свойство '\<x\>' на карте ПЛК с обозначением типа ПЛК '\<y\>' неизвестно

### Причина

Определенное пользователем свойство '\<x\>' на карте ПЛК с обозначением типа '\<y\>' для импорта неизвестно, поскольку данные изделий для карты ПЛК в базе данных изделий не обновлены.

### Решение

#### Возможность 1

1. Узнайте у производителя, доступно ли обновленное изделие с заявленным свойством.
2. Если оно доступно, импортируйте изделие и назначьте его карте ПЛК.

#### Возможность 2

1. В базе данных изделий добавьте отсутствующее определенное пользователем свойство для изделия.
2. Затем назначьте изделие карте ПЛК.
3. Еще раз выполните импорт данных ПЛК.

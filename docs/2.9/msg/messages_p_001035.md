# P001035: Несколько выводов распределенных клемм с одинаковым обозначением: \<x\>

### Причина

Несколько распределенных клемм в проекте получили одно и то же обозначение вывода устройства.

### Решение

1. Установите местоположение соответствующих распределенных клемм в проекте с помощью функции Перейти к (перекрестная ссылка) или Перейти к (все виды представлений), которые находятся во всплывающем меню диалогового окна Управление сообщениями.

!!! info "Для сведения:"

    Распределенные клеммы перечисляются в диалоговом окне Результаты поиска - <Имя проекта> на вкладке Перейти к.

2. С помощью всплывающего меню вызовите диалоговое окно Свойства для каждой распределенной клеммы и присвойте каждой клемме в поле Обозначение вывода устройства (вкладка Клемма) однозначное значение.
3. Затем запустите новый контрольный прогон.

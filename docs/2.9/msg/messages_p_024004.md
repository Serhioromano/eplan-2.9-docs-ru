# P024004: Соединение между разными подразделами

### Причина

Существующие на соединении целевые функции противоречат друг другу в отношении заданного свойства Подраздел. Это происходит, например, если имеется соединение между функцией, относящейся к подразделу "Водяное охлаждение" и подразделу "Масляное охлаждение".

### Решение

Присвойте целям соединения согласованные данные в отношении свойства Подраздел. Подраздел для соответствующей функции можно установить в диалоговом окне Свойства ++...++ на первой вкладке.

Обновите соединения и затем повторно запустите контрольный прогон.

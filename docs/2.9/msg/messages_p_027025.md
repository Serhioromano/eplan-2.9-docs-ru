# P027025: Определение клеммника многократно представлено в той же сети соединенных сегментов

### Причина

Внутри сети соединенных сегментов было многократно размещено определение клеммника с одним и тем же ОУ.

!!! note "Замечание:"

    Определение клеммника с видом представления "Топология" может многократно встречаться в проекте, но только один раз в сети соединенных сегментов.

### Решение

1. Установите местоположение многократно встречающегося в сети соединенных сегментов определений клеммника в сети соединенных сегментов, пользуясь функцией Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно Свойства ++...++ этих определений клеммника и присвойте в поле Видимое ОУ однозначное обозначение устройства.
3. После изменения обозначения устройства запустите новый контрольный прогон.

# P004021: Многократно заданный адрес в одном ЦПУ

### Причина

В ПЛК многократно использован какой-либо адрес.

### Решение

1. Измените адреса соответствующих выводов устройства ПЛК с помощью диалогового окна Свойства ++...++, чтобы каждый адрес в ЦПУ был уникальным. (Назначьте ЦПУ блоку ПЛК, задав свойства ЦПУ и ЦПУ: Имя.)
2. Затем запустите новый контрольный прогон.

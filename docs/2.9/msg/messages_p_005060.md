# P005060: Вывод устройства уже имеет одно многополюсное соединение в одно- и многополюсном представлении

### Причина

Вывод устройства, который был подключен на вкладке Подключить устройства, уже имеет одно многополюсное соединение.

### Решение

1. Удалите одно из двух соединений.
2. Затем запустите новый контрольный прогон.

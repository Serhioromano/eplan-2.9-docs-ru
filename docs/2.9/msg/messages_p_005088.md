# P005088: Отдельное соединение с ОУ

### Причина

На многополюсном или однополюсном соединении свойству Соединение: Принадлежность присвоено значение "Отд. соединение". Кроме того, это соединение имеет обозначение устройства.

### Решение

1. Установите местоположение точки определения группы соединений соединения на схеме соединений при помощи функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно свойств точки определения соединения.
3. Удалите обозначение устройства соединения или присвойте свойству Соединение: Принадлежность другое значение.
4. Щелкните по кнопке ++OK++.
5. Затем запустите новый контрольный прогон.

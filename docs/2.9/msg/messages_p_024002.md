# P024002: Соединение между разными разделами

### Причина

Имеющиеся на соединении целевые функции противоречат друг другу с точки зрения установленного свойства Раздел. Например, это тот случай, когда имеется соединение между функцией, относящейся к разделу "Пневматика" и функцией, относящейся к разделу "Гидравлика".

### Решение

Присвойте целям соединения совпадающие данные с точки зрения свойства Раздел. Раздел для соответствующей функции можно установить в диалоговом окне Свойства ++...++ на первой вкладке.

Обновите соединения и затем повторно запустите контрольный прогон.

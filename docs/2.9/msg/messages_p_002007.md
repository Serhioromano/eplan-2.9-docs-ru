# P002007: Имеются несколько определений штекеров, не связанных между собой

### Причина

Вы осуществляете проектирование со штекерами и работаете с гнездами и штекерами раздельно. Теперь определения функций двух связанных друг с другом определений штекеров с одинаковым ОУ не сходятся. Вследствие этого образование пары между стороной штекера и гнезда отсутствует.

### Решение

Проверьте на определении штекера для стороны гнезд и штырей правильность установки определения функции. Для этого откройте диалоговое окно Свойства ++...++ и просмотрите на вкладке Определение штекера настройки в поле Определение функции.

Для определения штекера со стороны гнезд определение функции должно стоять на "Определение штекера для гнезд". Для определения штекера стороны штырей определение функции должно стоять на "Определение штекера для штырей".

Измените отличающиеся настройки с учетом этих значений, после чего запустите новый контрольный прогон.

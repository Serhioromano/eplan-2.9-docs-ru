# P004016: Многократное вхождение каркаса '\<x\>'

### Причина

Обозначение каркаса использовано многократно.

### Решение

1. В диалоговом окне Свойства ++...++ для соответствующих блоков ПЛК измените обозначения каркасов на вкладке Данные структуры ПЛК, чтобы введенные значения были разными.
2. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Этот контрольный прогон служит только для проверки блоков ПЛК, на которых указан проект конфигураций.

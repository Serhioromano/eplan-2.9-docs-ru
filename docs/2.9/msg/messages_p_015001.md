# P015001: Отчет '\<x\>' на странице '\<y\>' имеет неизвестный тип отчета и не обновляется

### Причина

EPLAN обнаружил в проекте новый тип отчета, который был создан посредством новой версии EPLAN. Обновить этот тип отчета посредством имеющейся у вас данный момент версии EPLAN невозможно.

### Решение

Обновить отчет можно только после обновления программы. Выполните резервирование данных проекта и установите последнюю версию EPLAN.

!!! note "Замечание:"

    Если вы хотите удалить из проекта не обновляемый отчет, выделите его в навигаторе страниц и выберите пункт всплывающего меню Удалить.

# P004111: Соединение между выводами шины с другим свойством 'Физическая сеть: имя' ('\<x\>', '\<y\>')

### Причина

Несколько взаимосвязанных выводов шин объединены в одну физическую сеть. В данном случае одному или нескольким взаимосвязанным выводам шины, которые являются частью физической сети, присвоены разные значения свойства Физическая сеть: Имя. Различные значения выводятся в заполнителях \<x\> и \<y\> сообщения.

### Решение

1. С помощью функций Перейти к (графика) и Перейти ко 2-й координате, которые доступны во всплывающем меню диалогового окна Управление сообщениями, перейдите к взаимосвязанным кабельным выводам сети / шины физической сети.
2. Вызовите диалоговое окно "Свойства" кабельных выводов сети / шины и на вкладке Данные шины в поле Физическая сеть: Имя введите одинаковое значение. Или введите имя физической сети только на одном из взаимосвязанных кабельных выводов сети / шины и удалите его на других кабельных выводах сети / шины.
3. Затем запустите новый контрольный прогон.

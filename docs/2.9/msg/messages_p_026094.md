# P026094: Номера климатической области в проекте должны быть однозначными

### Причина

В пространствах листа проекта в одном из электрошкафов обнаружена температурная область, уже присвоенная ранее.

### Решение

1. Установите местоположение электрошкафа, температурная область которого определена неоднозначно, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно "Свойства" электрошкафа и измените значение в поле Мощность потерь: Температурная область. Следите за тем, чтобы выбранный номер не был присвоен никакому другому электрошкафу в пространствах листа проекта.
3. Затем запустите новый контрольный прогон.

# P001016: Клемма без целей

### Причина

В схеме соединений размещена клемма, не соединенная с условным обозначением.

### Решение

Соедините с условным обозначением минимум один из двух выводов устройства символов размещенной клеммы.

Обновите соединения и затем повторно запустите контрольный прогон.

!!! note "Замечание:"

    Учтите, что ***неразмещенные клеммы*** не учитываются при данном контрольном прогоне!

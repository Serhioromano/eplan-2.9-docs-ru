# P026017: Структурный идентификатор не соответствует структурному идентификатору монтажной поверхности

### Причина

Вы создали структуру проекта и определили для устройств этого проекта блоки идентификаторов, которым были присвоены структурные идентификаторы. Однако теперь с точки зрения блока идентификаторов имеются разные структурные идентификаторы на размещении изделия и на соответствующей монтажной поверхности.

### Решение

1. Введите (в диалоговом окне Свойства ++...++ в поле Видимое ОУ) для размещения изделия и соответствующей монтажной поверхности одинаковые структурные идентификаторы для требуемых блоков идентификаторов (например, для ид. "Установка" и/или "Место установки").
2. Затем запустите новый контрольный прогон.

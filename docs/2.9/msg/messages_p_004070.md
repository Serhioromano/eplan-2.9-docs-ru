# P004070: Для вывода шины (вход) на '\<x\>' не найден подходящий вывод шины (выход) для адреса процессов '\<y\>'

### Причина

Используется интерфейс импорта ПЛК ***Beckhoff TwinCat***: При импорте для вывода шины (вход) на указанном в сообщении устройстве \<x\> не был найден вывод шины (выход). Поиск подходящего вывода шины (выход) всегда ведется по адресу процессов \<y\>.

### Решение

Проверьте, имеется ли для вывода шины (вход) указанного в сообщении устройства \<x\> по адресу процессов \<y\> вывод шины (выход).

!!! note "Замечание:"

    Это относящееся к модулю сообщение выводится при импорте данных ПЛК. При выполнении нового контрольного прогона (любой настроенной схемы) его можно удалить из диалогового окна управления сообщениями.

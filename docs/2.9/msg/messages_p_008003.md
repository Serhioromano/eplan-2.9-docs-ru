# P008003: Язык словаря '\<x\>' в словаре отсутствует

### Причина

Данный язык словаря был убран из словаря.

### Решение

Добавьте отсутствующий словарь в язык. Для этого выберите Сервисные программы > Перевод > Обработать словарь. Щелкните в диалоговом окне Словарь на вкладке Управление по кнопке ![](../images/all_new_as.png){: .ui-icon } (Создать). Выделите требуемый язык словаря, закройте диалоговое окно и переведите тексты.

Затем запустите новый контрольный прогон.

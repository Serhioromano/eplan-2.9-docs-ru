# P005073: Различные сечения/диаметры выводов целей

### Причина

При помощи индексированного свойства Сечение/диаметр вывода (все, в зависимости от копирования ОУ) на функции можно указать поперечное сечение или диаметр для каждого вывода устройства.

В этом случае у выводов устройств целей, связанных с функцией (целевых функций), будут разные поперечные сечения или диаметры.

### Решение

1. Укажите для целей (целевых функций) соответствующие одинаковые значения для поперечного сечения или диаметра вывода устройства.
2. Обновите соединения и затем повторно запустите контрольный прогон.

# P022026: Текст представлен фрагментарно

### Причина

На странице проекта был найден фрагментарно представленный текст с активированным блоком выравнивания.

### Решение

Прежде всего установите местоположение текста в проекте с помощью функциональности Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.

У вас имеется несколько возможностей решения, которые описаны ниже:

#### Возможность 1

Увеличить блок выравнивания.

1. Выделите текст на странице проекта, так чтобы точки элементов блока выравнивания были видны.
2. Разместите указатель мыши на одной из точек элементов, нажмите левую клавишу мыши и перетащите блок выравнивания в нужную сторону, пока текст не окажется целиком в пределах блока выравнивания.
3. Затем запустите новый контрольный прогон.

В качестве альтернативы можно изменить размер блока выравнивания в диалоговом окне Свойства ++...++ текста с помощью свойств формата Ширина и Высота.

#### Возможность 2

Уменьшить размер шрифта текста.

1. Откройте диалоговое окно Свойства ++...++ текста и выведите вкладку Формат на передний план.
2. Уменьшайте введенное значение в поле Размер шрифта, пока текст не войдет в блок выравнивания.
3. Закройте диалоговое окно Свойства ++...++, для чего нажмите на кнопку ++OK++.
4. Затем запустите новый контрольный прогон.

#### Возможность 3

Подогнать текст под блок выравнивания.

1. Откройте диалоговое окно Свойства ++...++ текста и выведите вкладку Формат на передний план.
2. Установите флажок Разрешен подгон текста.
3. Закройте диалоговое окно Свойства ++...++, для чего нажмите на кнопку ++OK++.
4. Если текст находится на странице схемы соединений, выберите пункты меню Обработать > Текст > Подогнать тексты в блоке выравнивания.
Если текст находится на странице отчета, сгенерируйте отчет заново.

!!! info "Для сведения:"

    Текст будет автоматически подогнан по размеру под размер блока выравнивания.

5. Затем запустите новый контрольный прогон.

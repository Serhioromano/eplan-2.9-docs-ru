# P501004: Записанное на изделии определение функции \<x\> устарело

### Причина

В проверяемой базе данных изделий было найдено изделие, на котором сохранено определение функции, которое EPLAN не может найти.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Выведите на передний план вкладку Шаблоны функции.
3. Щелкните в таблице Выбор устройства (шаблоны функций) по записи в столбце Определение функции, который содержит устаревшее и более недопустимое определение функции, и щелкните по кнопке ++...++.
4. Выберите в открывшемся диалоговом окне Определение функции текущее определение функции.
5. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
6. Затем запустите новый контрольный прогон.

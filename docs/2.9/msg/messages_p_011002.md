# P011002: Точка разрыва без обратного эквивалента

### Причина

В каждой точке разрыва должен быть как минимум один обратный эквивалент в качестве дополнительной точки разрыва, который связан с первой точкой разрыва по перекрестной ссылке. Эта перекрестная ссылка формируется, если обе точки разрыва имеют ***одинаковое обозначение устройства***. В данном случае для точки разрыва, используемой в проекте, нет соответствующего обратного эквивалента, поскольку перекрестная ссылка не может быть сформирована. Предположительно, обозначения устройств обеих точек разрыва вводились по-разному. Обозначение устройства первой точки разрыва при вставке в схему соединений было введено "обычным" путем через поле Видимое ОУ в диалоговом окне 'Свойства'. Однако одноименное обозначение устройства на второй точке разрыва было введено через диалоговое окно Полное ОУ, при этом в поле Буквенное обозначение занесены как буквенное обозначение, так и счетчик, которые ***взаимосвязаны***.

### Решение

1. Установите местоположение точек разрыва на схеме соединений с помощью функции Перейти к графике, которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно 'Свойства' обеих точек разрыва.
3. В диалоговом окне 'Свойства' рядом с полем Полное ОУ щелкните по кнопке ++...++.
4. В последующем диалоговом окне таблицы ОУ ***отдельно*** введите буквенное обозначение и счетчик в соответствующие поля.
5. Подтвердите ввод.
6. Затем запустите новый контрольный прогон.

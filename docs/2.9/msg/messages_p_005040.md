# P005040: Вывод жгута не встречается попарно

### Причина

Вывод устройства жгута размещен в схеме соединений только один раз.

### Решение

Разместите вывод устройства жгута в схеме соединений в пределах жгута два раза, чтобы была образована пара выводов устройств жгута. Введите одинаковое обозначение на соответствующих выводах устройства жгута.

Обновите соединения и затем повторно запустите контрольный прогон.

# P026044: Схема соединений не определена.

### Причина

На размещении изделия в пространстве листа не определена схема соединений.

### Решение

Укажите на размещении изделия схему соединений.

#### Возможность 1

1. Установите местоположение размещения изделия в пространстве листа с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно Свойства ++...++ размещения изделия.
3. Выведите на передний план вкладку Изделие, запомните указанный на ней номер изделия и закройте диалоговое окно 'Cвойства'.
4. Выберите пункты меню Сервисные программы > Изделие > Управление и перейдите в дереве диалогового окна База данных изделий к изделию, номер которого вы запомнили.
5. Выведите на передний план вкладку Технические данные и щелкните рядом с полем Схема соединений по кнопке ++...++.
6. В открывающемся диалоговом окне Выбор схемы соединений выделите имя необходимой схемы соединений и щелкните кнопку ++OK++.

!!! info "Для сведения:"

    Имя скопируется в поле Схема соединений на вкладке Технические данные, а схема соединений будет присвоена размещению изделия в пространстве листа.

7. Закройте базу данных изделий, сохраните измененные данные и обновите базу данных изделий.
8. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Если схема соединений в базе данных изделий отсутствует, ее нужно будет создать позднее. Выполните для этого следующие действия:

1. Выделите уровень иерархии Схема соединений в дереве базы данных изделий и выберите пункт всплывающего меню Создать.
2. Определите необходимые данные выводов на вкладках Схема соединений, Выводы устройства и / или Атрибуты, введите имя схемы соединений и щелкните по кнопке ++Применить++.
3. Выделите в дереве изделие и присвойте ему схему соединений, как описано выше.

    ![](../images/arrow.png){: .ui-icon } Присвоение выполняется также для размещения изделия в пространстве листа.

#### Возможность 2

1. Вызовите диалоговое окно Свойства ++...++ размещения изделия.
2. Выведите на передний план вкладку Схема соединений.
3. Установите флажок Локальная схема соединений.
4. Внесите в таблицу необходимые обозначения выводов устройства вместе с другими данными выводов.
5. Закройте диалоговое окно Свойства ++...++, для чего нажмите на кнопку ++OK++.
6. Затем запустите новый контрольный прогон.

# P014003: Версия, указанная на ссылающейся рамке макроса, не совпадает с версией (\<x\>) в файле макроса

### Причина

Версия, сохраненная на ссылающейся рамке макроса, не совпадает с версией (\<x\>) используемого файла макроса.

### Решение

1. Установите местоположение данной рамки макроса в проекте с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Выделите точку ввода рамки макроса.
3. Выберите пункт всплывающего меню Обновить макрос.
4. В открывшемся диалоговом окне выберите нужную схему, чтобы задать способ обновления макроса.
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

!!! info "Для сведения:"

    Рамка макроса и файл макроса имеют теперь одинаковую версию.

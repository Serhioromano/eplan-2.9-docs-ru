# P003013: PE-жила указана многократно

### Причина

В диалоговом окне свойств кабеля указано, что в кабеле допускается только одна PE- / PEN-жила. Однако фактически кабель имеет две PE- / PEN-жилы.

### Решение

#### Возможность 1

Измените тип потенциала на одной из указанных многократно жил.
Для этого дважды щелкните по условному обозначению источника кабеля и выберите в диалоговом окне Свойства ++...++ вкладку Данные символа / функции. Щелкните по кнопке ++Логика++. В диалоговом окне Логическая схема выводов устройства присвойте выводу устройства функции 2 в поле Тип потенциала другое значение, например "Не определено".

#### Возможность 2

Если требуется разрешить несколько PE- / PEN-жил в кабеле, дважды щелкните по линии определения кабеля на схеме соединений и установите в диалоговом окне Свойства ++...++ кабеля флажок Разрешить несколько жил PE.

Обновите соединения и затем повторно запустите контрольный прогон.

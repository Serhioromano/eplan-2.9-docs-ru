# P007011: Однополюсная функция не может быть однозначно присвоена однополюсной функции

### Причина

В однополюсном представлении у одной из функций устройства отсутствуют данные для определенного свойства. В многополюсном представлении это свойство для отдельных функций устройства было заполнено данными. Например, в многополюсном представлении для клеммника были введены обозначения каждой клеммы, а в однополюсном, наоборот, — нет.

### Решение

При необходимости скопируйте отсутствующие данные соответствующего свойства из одной из функций данного устройства в многополюсном представлении и введите их для функции в однополюсном представлении.

Затем запустите новый контрольный прогон.

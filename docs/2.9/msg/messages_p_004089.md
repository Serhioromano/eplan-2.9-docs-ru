# P004089: Многократно заданное имя карты ПЛК '\<x\>' в одном каркасе

### Причина

Как минимум два блока ПЛК, являющиеся картами ПЛК одного и того же каркаса в проекте конфигураций, имеют одинаковое имя карты ПЛК.

Этот контрольный прогон служит только для проверки блоков ПЛК, на которых указан проект конфигураций.

### Решение

1. Определите местоположение блоков ПЛК на схеме соединений с помощью функции Перейти к (графика) и Перейти ко 2-й координате во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно "Свойства" для каждого блока ПЛК каркаса.
3. Переместите первую вкладку на передний план.
4. Измените в таблице свойств значение свойства Имя карты ПЛК. При этом проследите, чтобы каждый блок ПЛК содержал уникальное имя карты ПЛК.
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

!!! tip "Совет:"

    Для удобного поиска ошибок в проекте также можно использовать [табличную обработку](../functiondatagridgui_k_start.md). Просто создайте схему для отображения имени карты ПЛК и требуемых данных структуры ПЛК в удобной форме.

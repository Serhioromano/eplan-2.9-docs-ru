# P005013: Разный тип вывода целей

### Причина

Подсоединенные к соединению цели имеют разные типы выводов устройства функций. Например, одна цель имеет тип вывода устройства "Жила / Проволока", в то время как другая цель имеет тип вывода устройства "Световод".

### Решение

Присвойте единообразные типы выводов устройства всем выводам устройства функций, являющихся целями соединения.

Обновите соединения и затем повторно запустите контрольный прогон.

!!! note "Замечание:"

    Тип вывода устройства "Не определен" может использоваться для всех выводов устройства.

# P004037: Отсутствует адрес шины (вывод шины '\<x\>' в шинной системе: '\<y\>')

### Причина

На выводе шины \<x\>, присвоенном системе шин \<y\>, отсутствует адрес шины.

### Решение

1. Установите местоположение вывода шины на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно "Свойства" вывода шины.
3. Переместите вкладку Данные шины на передний план.
4. Введите недостающий адрес в поле Физическая сеть: Адрес шины / номер позиции.
5. Либо активируйте свойство Игнорировать недостающий адрес шины, если для вывода шины не требуется адрес шины.
6. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

* Этот контрольный прогон сообщит об отсутствующем адресе шины, только если на выводе шины задано одно из следующих определений функций:

    \- ***Кабельный вывод сети / шины, общее***

    Проверяются только главные выводы шины. Оповещения о выводах шины, у которых для свойства Интерфейс шины: Имя введено значение и снят флажок свойства Интерфейс шины: Главный вывод шины, не выводятся.
* Учтите, пожалуйста, следующее:
    Для выводов шины помимо обозначения вывода устройства для идентификации также используются ***обозначение штекера*** и ***имя интерфейса шины***. Эта комбинация проверяется на идентичность (обозначение вывода устройства + обозначение штекера + имя интерфейса шины). Если как минимум одно из этих частичных обозначений отличается, сообщение ***не*** выводится.

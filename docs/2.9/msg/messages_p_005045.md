# P005045: Одинаковые обозначения соединений на разных сигналах или состояниях трубопроводов

### Причина

На схеме соединений или на функциональной схеме автоматизации на соединениях находятся как минимум два обозначения соединения, присвоенные разным сигналам или состояниям трубопроводов.

Появление такого сообщения для разделов "Электротехника" и "Технология производственных процессов" может быть вызвано несколькими причинами. Рассмотрим их на примерах.

#### Возможность 1 (Электротехника)

В результате нумерации соединений по потенциалу часть соединения отделилась от первоначального сигнала. Поэтому отделившееся соединение имеет другой сигнал, сохраняя прежнее обозначение соединения.

!!! example "Пример:"

    Два выводаустройства,-U5:1и-U6:2, связаны между собой при помощи клеммы с предохранителем-X2:1. Между каждым из выводов устройства и клеммой с предохранителем существует два отрезка соединения, которые имеют одинаковое обозначением002.На вкладке Данные символа /функциидиалогового окна "Свойства" клеммы с предохранителем установлен флажок С разделением сигнала. Из-за этойнастройкифункция прерывает передачу сигнала. Прерванная передача является причиной того, что участки соединения, имеющие одинаковые обозначения, присвоены разным сигналам.

#### Возможность 2 (Технология производственных процессов)

Нарушена передача сигнала на функции в трубопроводе.

!!! example "Пример:"

    Два резервуара,-CM01и-CM02, соединены между собой трубопроводом, обозначенным точкой определения трубопровода-WP01. В трубопровод встраивается запорная арматура-FL01. По трубе поступает веществоH2Oс состояниемA. На трубопроводе между выводами устройства резервуаров и запорной арматурой размещались точки определения соединений, имеющие одинаковое обозначение010.На вкладке Данные символа / функциидиалогового окна "Свойства" запорной арматуры установлен флажок С разделением сигнала. Из-за этой настройки функция прерывает передачу состояния трубопровода. Из-за прерванной передачи соединения, имеющие одинаковые обозначения, присвоены разным состояниям трубопровода.

### Решение

#### Для возможностей 1 и 2

1. Определите местоположение соединений, имеющих одинаковые обозначения, на схеме соединений или на функциональной схеме автоматизации с помощью функции Перейти к (графика) и Перейти ко 2-й координате во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно "Свойства" функции, размещенной между соединениями с одинаковыми обозначениями, и переместите вкладку Данные символа / функции на передний план.
3. Проверьте, установлен ли флажок С разделением сигнала. Если на функции между соответствующими соединениями передача сигнала прервана, снимите флажок С разделением сигнала.
4. Обновите соединения и повторно запустите контрольный прогон.

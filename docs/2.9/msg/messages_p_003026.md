# P003026: Не задана длина кабеля

### Причина

На главной функции кабеля не указана длина кабеля.

### Решение

Определите из диалогового окна Управление сообщениями главную функцию кабеля в схеме соединений с помощью функциональности Перейти к (графика). Щелкните там двойным щелчком по выделенной линии определения кабеля и вызовите диалоговое окно Свойства ++...++ кабеля. Перейдите — если не установлено по умолчанию — на вкладку Кабель и укажите в поле Длина количество жил.

Щелкните на ++OK++ и повторно запустите контрольный прогон.

# P003011: Слишком много кабельных соединений

### Причина

Значение, внесенное в поле Число соединений диалогового окна характеристик кабеля, меньше, чем фактически имеющееся количество кабельных соединений кабеля.

!!! note "Замечание:"

    Экранирования и свободные шаблоны ***не*** учитываются!

### Решение

#### Возможность 1

Увеличьте значение в поле Число соединений диалогового окна характеристик кабеля, так чтобы оно совпадало с фактически имеющимся числом соединений кабеля.

#### Возможность 2

Щелкните в диалоговом окне свойств кабеля рядом с полем Тип на ++...++ и задайте через выбор изделий подходящий кабель с соответствующим количеством соединений.

Обновите соединения и затем повторно запустите контрольный прогон.

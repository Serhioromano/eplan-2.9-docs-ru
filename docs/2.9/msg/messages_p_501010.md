# P501010: Внесенное на узле / модуле изделие \<x\>, вариант \<y\>, отсутствует

### Причина

В проверяемой базе данных изделий было найдено внесенное на узле или модуле изделие, которое не существует как запись данных в базе данных изделий. EPLAN отображает имя изделия в заполнителе \<x\> и вариант изделия в заполнителе \<y\>.

### Решение

1. Откройте базу данных изделий и создайте сначала требуемое изделие в базе данных изделий. Или импортируйте изделие.
2. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, в котором соответствующий узел или модуль уже выделен.

3. Переместите вкладку Узел или Модуль на передний план.
4. Выберите в поле Номер изделия новое созданное изделие.
5. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
6. Затем запустите новый контрольный прогон.

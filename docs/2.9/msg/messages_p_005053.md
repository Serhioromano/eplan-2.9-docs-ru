# P005053: Неприсоединенный вывод устройства на клемме

### Причина

В схеме соединений был обнаружен несвязанный вывод устройства символов на клемме. Это сообщение появляется также в том случае, если у клеммы есть несколько выводов устройства, из которых только один задействован в схеме соединений.

### Решение

1. Установите местоположение данной клеммы на схеме соединений, пользуясь функциональностью Перейти к (графика), которую вы можете найти во всплывающем меню диалогового окна Управление сообщениями.
2. Соедините все свободные выводы символа с условным обозначением так, чтобы образовались линии автоматического содинения.
3. Или, если клемма больше не нужна, удалите ее из схемы соединений.
4. Затем запустите новый контрольный прогон.

# P026097: Размещение изделия выходит за пределы области с оптимальным кондиционированием

### Причина

Центр размещения изделия находится за пределами области, которая может оптимально охлаждаться компонентом системы кондиционирования.

### Решение

1. Установите местоположение соответствующего размещения изделия в пространстве листа с помощью функции Перейти к (графика) во всплывающем меню диалогового окна Управление сообщениями.
2. Переместите центр размещения изделия в область оптимального кондиционирования или разместите соответствующие принадлежности компонента системы кондиционирования для подачи воздуха.
3. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Непосредственно компонент системы кондиционирования и его принадлежности для подачи воздуха, все механические изделия и все изделия, полностью размещенные вне электрошкафа, исключены из этой проверки.

    Компонент системы кондиционирования и размещения изделий должны быть присвоены идентичной температурной области (свойство ***Мощность потерь: Температурная область (автоматически)*** (ид. 36046)).

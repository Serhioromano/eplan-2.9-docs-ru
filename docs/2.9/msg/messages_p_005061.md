# P005061: Больше одного определения многополюсного соединения для однополюсного соединения

### Причина

Однополюсное соединение имеет несколько точек определения соединений, в которые входят многополюсные определения соединений. То есть соединения определены на вкладке Подключить устройства в диалоговом окне "Свойства" обоих точек определения соединения.

### Решение

1. Занесите многополюсные определения соединений только на одну точку определения соединения.
2. Удалите все многополюсные определения соединений на других точках определения соединения.
3. Затем запустите новый контрольный прогон.

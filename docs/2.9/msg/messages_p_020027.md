# P020027: Первый проект:\<x\> \- Второй проект:\<y\>

### Описание

Выполнено сравнение проектов, при котором ***Настройки***, ***Свойства*** или ***Слои*** проверяемого проекта (проект с замечаниями) сравнивались с настройками проекта для сравнения.

В этом сообщении выводятся путь файла и имя сравниваемых проектов. Первый проект \<x\> следует проверить, второй проект \<y\> — проект для сравнения.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

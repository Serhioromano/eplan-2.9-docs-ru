# P024006: Ключ среды не подходит к разделу

### Причина

Использованный в обозначении устройства Fluid ключ среды не подходит к выбранному разделу и не соответствует действующей редакции стандарта DIN ISO 1219-2. Например, управляющий клапан был обозначен в логической ячейке пневматики ключом среды "Н" для гидравлики, а не правильным ключом "Р" для пневматики.

### Решение

1. Установите местоположение устройства Fluid на схеме соединений с помощью функции Перейти к (графика), которая имеется в диалоговом окне всплывающего меню Управление сообщениями.
2. Адаптируйте ключ среды в обозначении устройства Fluid в соответствии со стандартом к разделу, используя правильную букву.

По стандарту требуется следующий ключ среды для разных разделов:

\- H для гидравлики
\- P для пневматики;
\- C для охлаждения;
\- K для смазочно-охлаждающей жидкости;
\- L для смазки;
\- G для газовой техники.

3. Затем запустите новый контрольный прогон.

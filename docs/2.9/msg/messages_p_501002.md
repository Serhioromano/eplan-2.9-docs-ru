# P501002: Записанный на изделии графический файл \<x\> отсутствует

### Причина

В проверяемой базе данных изделий было найдено изделие, на котором сохранен графический файл, который EPLAN не может найти.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Переместите вкладку Монтажные данные на передний план.
3. Щелкните рядом с полем Графический файл по кнопке ++...++ и выберите в открывшемся диалоговом окне выбора нужный графический файл из одного из предлагаемых каталогов картинок.

Или в качестве альтернативы удалите запись в поле Графический файл.

Или создайте ***новую*** картинку, сохраните ее в нужном каталоге картинок и выберите ее в поле Графический файл.
4. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
5. Затем запустите новый контрольный прогон.

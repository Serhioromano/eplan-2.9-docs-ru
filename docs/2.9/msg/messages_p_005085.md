# P005085: Соединение выделено для удаления

### Причина

В проекте была установлена точка определения соединения, в которой активировано свойство Выделено для удаления. В дальнейшем это свойство всегда будет автоматически устанавливаться программой, если точка определения соединения, импортированная вами при помощи сторонней программы, будет удалена из этой программы.

### Решение

1. Установите точку определения соединения в проекте при помощи функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно свойств точки определения соединения и снимите флажок Соединение выделено для удаления. Или удалите точку определения соединения.
3. Обновите соединения и затем повторно запустите контрольный прогон.

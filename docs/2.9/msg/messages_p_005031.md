# P005031: На части соединения (\<x\>) имеющаяся точка определения соединения (\<y\>) снабжена обозначением соединения

### Описание

Точка определения соединения, которая с помощью соединения Без обозначения соединения содержит (раскрывающийся список Разрешенные данные в диалоговом окне Свойства ++...++), центрирована на части соединения. При размещении других точек определения соединения вставленная ранее по центру точка определения соединения подготавливается к обозначению соединения. При этом в раскрывающемся списке Разрешенные данные автоматически изменяется на "Все данные".

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

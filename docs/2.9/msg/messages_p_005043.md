# P005043: Поперечное сечение / диаметр не содержится в разделе 'Поперечные сечения / диаметры' в настройках проекта!

### Причина

Вы используете поперечное сечение/диаметр, которого нет в настройках проекта. В диалоговом окне Настройки: Поперечные сечения / диаметры (Пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Поперечные сечения/диаметры) могут быть указаны поперечные сечения или диаметры, которые присваиваются любому соединению проекта.

Выбранное вами значение отсутствует в таблице допустимых значений.

### Решение

#### Возможность 1

Выберите значение из имеющихся в таблице Поперечные сечения / диаметры. Для этого откройте диалоговое окно Свойства ++...++ для соответствующего соединения или соответствующей точки определения соединения. Рядом с полем Поперечное сечение/диаметр щелкните ++...++. Выделите в открывшемся диалоговом окне Поперечные сечения / диаметры одно из возможных значений и нажмите ++OK++.

#### Возможность 2

Добавьте в таблицу допустимых значений Поперечные сечения / диаметры то значение, которое вы хотели бы использовать.

Выберите пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Поперечные сечения / диаметры. Щелкните в диалоговом окне Настройки: Поперечные сечения / диаметры по ![](../images/all_new_as.png){: .ui-icon } (Создать) и занесите в новую строку таблицы нужное значение. Затем щелкните по кнопке ++ОК++.

Обновите соединения и затем повторно запустите контрольный прогон.

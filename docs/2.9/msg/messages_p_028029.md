# P028029: Указанный класс трубы '\<x\>' содержится в проекте не в качестве шаблона

### Причина

На объекте планирования (трубопроводе), определении трубопровода / соединения или технологической функции найден класс трубы, который не был определен в проекте в качестве шаблона.

### Решение

1. Установите местоположение объекта планирования (трубопровода) в навигаторе предварительного планирования с помощью функции Синхронизировать выбор во всплывающем меню диалогового окна Управление сообщениями. Или воспользуйтесь функцией Перейти к (графика) всплывающего меню, чтобы перейти к точке определения трубопровода / соединения или технологической функции на функциональной схеме автоматизации.
2. Откройте диалоговое окно "Свойства" соответствующего объекта и переместите вкладку "Схема соединений" на передний план.
3. В таблице свойств объекта планирования (трубопровода) для свойства Класс трубы в поле значений щелкните ++...++. Или, в случае с точкой определения трубопровода / соединения или технологической функцией, щелкните ++...++ рядом с полем Класс трубы.
4. В диалоговом окне Выбрать класс трубы выделите имя из предлагаемого списка.
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Если требуемый класс трубы отсутствует в диалоговом окне Выбрать класс трубы, необходимо определить его в ***навигаторе шаблонов сегментов*** в качестве нового шаблона.

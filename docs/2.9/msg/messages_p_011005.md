# P011005: Острия стрел перекрестных точек разрыва не подходят друг к другу

### Причина

Стрелки двух точек разрыва, связанных перекрестной ссылкой, направлены друг на друга.

### Решение

Поверните одну и точек разрыва, связанных друг с другом перекрестной ссылкой, таким образом, чтобы острие стрелки одной точки разрыва указывало на конец стрелки другой точки разрыва. Для этого дважды щелкните по точке разрыва, которую надо повернуть. Выберите в диалоговом окне Свойства ++...++ вкладку Данные символа / функции. С помощью раскрывающегося списка Вариант поверните точку разрыва на 180°.

Обновите соединения и затем повторно запустите контрольный прогон.

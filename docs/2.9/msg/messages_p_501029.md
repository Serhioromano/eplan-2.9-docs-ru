# P501029: Внесенное на модуле \<x\>, вариант \<y\>, изделие \<z\>, вариант \<u\>, не имеет ни ОУ, ни идентификатора ОУ

### Причина

В проверяемой базе данных изделий определен модуль, на котором указано подизделие (т. н. позиция модуля) без ОУ и идентификатора ОУ. В заполнителе \<x\> выводится номер изделия модуля, в заполнителе \<y\> — вариант модуля, в заполнителе \<z\> — номер изделия, а в заполнителе \<u\> — вариант указанного подизделия.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, в котором соответствующий вариант указанного на модуле подизделия уже выделен.

2. Переместите вкладку Модуль на передний план.
3. В поле ОУ и / или Идентификатор ОУ задайте соответствующее значение.
4. Закройте базу данных изделий, сохраните измененные данные и выполните синхронизацию данных изделий.
5. Затем запустите новый контрольный прогон.

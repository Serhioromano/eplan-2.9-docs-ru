# P001023: Главная клемма не будет присвоена первому шаблону функции

### Причина

Главная клемма присвоена не первому, а одному из следующих шаблонов функции выбранного изделия.

### Решение

#### Возможность 1

Сделайте другую клемму главной клеммой, если присвоение выполнено к первому шаблону функции.

#### Возможность 2

Выберите другое изделие-клеммы, гарантирующее, что главная клемма может быть присвоена первому шаблону функции.

Обновите соединения и затем повторно запустите контрольный прогон.

# P007025: Идентификатор ОУ '\<x\>' задан многократно

### Причина

На функции, являющейся составной частью макроса для модуля, был определен уже заданный идентификатор ОУ.

### Решение

1. Установите местоположение данной функции на схеме соединений, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно свойств функции.
3. Измените в списке свойств значение свойства Идентификатор ОУ. Введите значение, которое еще не задавалось.
4. Затем запустите новый контрольный прогон.

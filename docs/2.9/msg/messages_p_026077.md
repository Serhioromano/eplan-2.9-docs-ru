# P026077: Диаметр жгута слишком велик, невозможно объединить соединения с выбранным изделием

### Причина

Общий диаметр всех маршрутизируемых соединений, которые необходимо объединить с помощью размещенной защиты кабеля/шланга или хомута для кабеля/шланга больше, чем максимальный диаметр жгута для используемых изделий защиты кабеля/шланга или хомута для кабеля/шланга.

### Решение

1. Установите местонахождение защиты кабеля/шланга или хомута для кабеля/шланга в пространстве листа при помощи функциональности Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно "Свойства" защиты кабеля/шланга или хомута для кабеля/шланга и переместите вкладку Изделие на передний план.
3. Перейдите через используемое изделие в окно "Выбор изделия" и переместите вкладку Данные маршрутизируемой принадлежности на передний план.
4. Проверьте, какое значение установлено в поле Максимальный диаметр жгута для используемого изделия.
5. Выберите защиту кабеля/шланга или хомут для кабеля/шланга с максимальным диаметром, превышающим общий диаметр жгута все маршрутизируемых соединений в данном жгуте.
6. Закройте диалоговое окно "Выбор изделия" и диалоговое окно "Свойства", нажав ++ОК++, и повторно запустите контрольный прогон.

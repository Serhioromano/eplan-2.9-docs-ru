# P004020: Больше одного вывода устройства В/В на один канал

### Причина

Определено больше одного вывода устройства В/В в канале. Соответственно, эти выводы устройства В/В имеют одинаковое обозначение канала.

### Решение

1. Присвойте каждому каналу только один вывод устройства В/В.
2. Затем введите в диалоговом окне Свойства ++...++ однозначное обозначение канала для каждого вывода устройства В/В.
3. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

* Использование нескольких выводов устройства В/В в канале ***допускается***, если только один из них определен как ***активированный*** вывод устройства. Все остальные выводы устройства В/В в канале необходимо обозначить как ***деактивированные***, назначив им свойство Деактивированный вывод устройства В/В.
* Если вы используете карты ПЛК, у которых вход и выход присвоены одному и тому же каналу, выполните контрольный прогон [P004097](messages_p_004097.md).

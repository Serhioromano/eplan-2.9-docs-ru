# P013047: Изделие с индексом '\<x\>' уже импортировано

### Причина

При импорте спецификации был обнаружен индекс изделия, использующийся два или более раз. Возможно, в импортируемом файле для нескольких изделий были вручную изменены значения в имени поля "REFERENCE_POS" , и при этом произошло присвоение нескольких идентичных индексов.

!!! note "Замечание:"

    В EPLAN устройству может быть присвоено максимум ***50*** изделий. Индекс изделия задает последовательность внесения изделий в диалоговом окне Свойства ++...++ на вкладке Изделие.

### Решение

Откройте импортируемый файл при помощи текстового или XML-редактора и найдите индекс, указанный в диалоговом окне управления сообщениями в качестве сообщения "013047", например, организуя поиск по слову REFERENCE_POS="<Индекс>".

Обнаружив повторно присвоенный индекс, скорректируйте его. Сохраните файл и заново импортируйте спецификацию.

Более подробную информацию и рекомендации вы найдете в разделе [Импорт спецификаций](../partslistgui_h_stuecklistenimportieren.md).

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

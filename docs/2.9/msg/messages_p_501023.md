# P501023: Шаблоны функций узла / модуля \<x\>, вариант \<y\>, неактуальны

### Причина

В проверяемой базе данных изделий в варианте \<y\> был определен узел / модуль \<x\>, шаблоны функций которого уже неактуальны.

### Решение

1. Выделите в диалоговом окне управления сообщениями строку с сообщением и выберите пункт всплывающего меню Свойства.

!!! info "Для сведения:"

    Откроется диалоговое окно База данных изделий, нужное изделие в ней уже выделено.

2. Выберите пункт меню ++Дополнительно++ > Суммировать шаблоны функций.

!!! info "Для сведения:"

    Находящиеся на подизделиях узлов / модулей шаблоны функций суммируются и обновляются.

3. Затем запустите новый контрольный прогон.

# P001026: Несоединенные распределенные клеммы имеют одинаковые обозначения

### Причина

В схеме соединений находятся распределенные клеммы, имеющие одинаковое обозначение, но не связанные друг с другом. Возможно, не активирована специфическая для проекта настройка Автоматически связать родственные распределенные клеммы.

### Решение

1. Выберите пункты меню Параметры > Настройки и отследите в дереве диалогового окна Настройки структуру папок Проекты > "Имя проекта" > Устройство.
2. Щелкните область настройки Клеммы (распределенные клеммы) и установите флажок Автоматически связать родственные распределенные клеммы.
3. В диалоговом окне нажмите кнопку ++OK++.
4. Обновите соединения и затем повторно запустите контрольный прогон.

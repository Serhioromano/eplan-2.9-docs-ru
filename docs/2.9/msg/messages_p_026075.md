# P026075: Радиус изгиба маршрутизируемого соединения слишком мал

### Причина

Радиус изгиба маршрутизируемого соединения между двумя точками опоры меньше минимально допустимого радиуса изгиба, сохраненного в базе данных изделий для маршрутизируемого соединения (шланга / трубы).

### Решение

1. Установите местоположение маршрутизируемого соединения в пространстве листа, пользуясь функцией Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Измените радиус изгиба маршрутизируемого соединения в пространстве листа так, чтобы он как минимум соответствовал значению минимально допустимого радиуса изгиба, определенному в базе данных изделий для маршрутизируемого соединения, или был больше него.
3. Повторно проложите соединения и запустите новый контрольный прогон.

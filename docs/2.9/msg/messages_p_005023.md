# P005023: Точка определения соединения находится не на соединении

### Причина

Найдена точка определения соединения, которая не находится на линии автоматического соединения.

### Решение

Переместите точку определения соединения на линию автоматического соединения.

Обновите соединения и затем повторно запустите контрольный прогон.

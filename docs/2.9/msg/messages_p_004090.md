# P004090: Тип рабочей станции ПЛК в проекте и в изделии отличается

### Причина

На блоке ПЛК, представляющем ЦПУ или рабочую станцию, значение, указанное для свойства Тип рабочей станции ПЛК, отличается от значения, указанного для одноименного свойства на выбранном изделии.

### Решение

#### Возможность 1

1. Установите местоположение блока ПЛК на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно свойств блока ПЛК.
3. Переместите вкладку Изделие блока ПЛК на передний план.
4. В списке слева вызовите диалоговое окно выбора изделий с помощью указанного номера изделия.
5. Выведите на передний план вкладку Свойства для изделия ПЛК, выделенного в выборе изделий.
6. Определите значение, указанное для свойства Имя рабочей станции ПЛК.
7. Покиньте меню выбора изделий и перейдите к первой вкладке блока ПЛК в диалоговом окне "Свойства".
8. В таблице свойств введите в поле значений свойства Имя рабочей станции ПЛК значение, определенное на одноименном свойстве выбранного изделия.
9. Щелкните по кнопке ++OK++.
10. Затем запустите новый контрольный прогон.

#### Возможность 2

1. Повторите шаги с 1-го по 2-й, описанные в разделе "Возможность 1".
2. Выберите для ЦПУ изделие, значение которого на свойстве изделия Имя рабочей станции ПЛК совпадает со значением одноименного свойства на блоке ПЛК.
3. Затем запустите новый контрольный прогон.

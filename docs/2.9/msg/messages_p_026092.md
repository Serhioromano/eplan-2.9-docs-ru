# P026092: Требуется новый расчет кондиционирования для климатической области \<x\>

### Причина

Значение мощности потерь в температурной области \<x\> изменилось, поэтому для нее требуется новый расчет кондиционирования.

!!! note "Замечание:"

    Критерием сравнения этого контрольного прогона является имеющаяся в данной температурной области мощность потерь. Сравнивается текущая мощность потерь с мощностью потерь последнего расчета кондиционирования. Эта проверка может быть проведена максимум для 100 температурных областей.

    В зависимости от того, какая температурная область выбрана, все мощности потерь суммируются с учетом коэффициента одновременности для всех функциональных элементов выбранной температурной области по следующей формуле:

    ***Общая мощность потерь на температурную область = сумма (мощность потерь x коэффициент одновременности) + сумма (сборные шины)***

    Это рассчитанное значение сравнивается со значениями свойств проекта от Тепловой расчет: Общая мощность потерь для климатической области [1] до Тепловой расчет: Общая мощность потерь для климатической области [100].

    Если текущее рассчитанное значение не совпадает со значением, сохраненным в свойствах проекта, выводится сообщение контрольного прогона.

    Помимо этого, сообщение контрольного прогона выводится также, когда в свойствах проекта указана общая мощность потерь для температурной области, которая (больше) не существует.

### Решение

1. Запомните номер выведенной в сообщении температурной области.
2. Выберите пункт меню Сервисные программы > Отчеты > Рассчитать общую мощность потерь.
3. В диалоговом окне Рассчитать мощность потерь установите флажок Отобразить все имеющиеся в проекте температурные области.
4. Установите соответствующий флажок упоминающейся в сообщении температурной области и запустите расчет кнопкой [OK].
5. Проверьте также, не записана ли в свойствах проекта температурная область, (более) не существующая в электрошкафу, и удалите запись.
6. Затем запустите новый контрольный прогон.

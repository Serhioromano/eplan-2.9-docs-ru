# P005015: Разные типы потенциала целей

### Причина

Подсоединенные к соединению цели имеют разные типы потенциалов с точки зрения выводов устройства функций. Например, одна цель имеет тип потенциала "L", в то время как другая цель имеет тип потенциала "PE / PEN".

### Решение

Присвойте единообразные типы потенциалов всем выводам устройства функций, являющихся целями соединения.

Обновите соединения и затем повторно запустите контрольный прогон.

!!! note "Замечание:"

    Тип потенциала "Не определен" может использоваться для всех выводов устройства.

# P004035: Многократно заданный адрес шины '\<x\>' логической сети '\<y\>' (Вывод шины в системе шин: '\<z\>')

### Причина

На одном или нескольких выводах шины в ***логической сети*** системы шин заданы ***одинаковые*** адреса шины. При проверке учитываются как внесенное значение свойства Физическая сеть: Адрес шины / номер позиции, так и значение свойства Физическая сеть: Адрес шины / номер позиции 2 на вкладке Данные шины.

Использование этого контрольного прогона целесообразно, если в пределах физической сети допускаются несколько адресов шины, а однозначность требуется только в логической сети.

Этот контрольный прогон служит только для проверки выводов шины, на которых указан проект конфигураций.

### Решение

1. Откройте диалоговое окно Свойства ++...++ для соответствующего вывода шины / соответствующих выводов шины. На вкладке Данные шины в полях Физическая сеть: Адрес шины / номер позиции и / или Физическая сеть: Адрес шины / номер позиции 2 введите разные значения.
2. Либо на одном из выводов шины измените запись в поле Логическая сеть: Имя, чтобы присвоить этот вывод шины другой логической сети.
3. Затем запустите новый контрольный прогон.

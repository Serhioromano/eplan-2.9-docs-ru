# P004110: Физическая сеть без имени

### Причина

Несколько взаимосвязанных выводов шин объединяются в физическую сеть. В данном случае физической сети не было назначено имя.

### Решение

1. Установите местоположение сети с помощью функции Перейти к (графика), которую вы можете найти во всплывающем меню диалогового окна Управление сообщениями, чтобы перейти к первому кабельному выводу сети / шины.
2. Вызовите диалоговое окно "Свойства" кабельного вывода сети / шины и на вкладке Данные шины введите значение в поле Физическая сеть: Имя. Или введите имя на другом кабельном выводе сети / шины, который подключен к физической сети. Достаточно, чтобы имя физической сети было введено хотя бы на одном из взаимосвязанных кабельных выводов сети / шины.
3. Затем запустите новый контрольный прогон.

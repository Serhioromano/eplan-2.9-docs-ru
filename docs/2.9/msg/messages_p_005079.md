# P005079: Разные состояния внутри трубопровода

### Причина

На функциональной схеме автоматизации на соединении содержатся несколько определений трубопровода (выводы трубопровода или точки определения трубопровода), на которых сохранены противоречивые значения относительно свойства Состояние (вещество).

### Решение

1. Установите местоположение определений трубопровода на функциональной схеме автоматизации с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно "Свойства" для определения трубопровода и введите одинаковое значение в поле Состояние (вещество).
3. Обновите соединения и затем повторно запустите контрольный прогон.

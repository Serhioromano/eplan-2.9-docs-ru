# P014004: Имя макроса не определено

### Причина

Для автоматической генерации макросов из проекта макросов на рамке макроса должно быть указано как минимум одно имя макроса. Оно должно быть определено в качестве свойства как на вкладке Рамка макроса, так и на вкладке Определение макроса. В этом случае запись отсутствует на обеих вкладках.

### Решение

1. Установите местоположение данной рамки макроса в проекте макросов, пользуясь функцией Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.

!!! info "Для сведения:"

    Рамка макроса будет выделена на странице проекта.

2. Откройте с помощью всплывающего меню диалоговое окно свойств рамки макроса.

3. Переместите вкладку Определение макроса на передний план.
4. В ***первую строку*** таблицы, в столбце Макрос: Имя, введите требуемое имя макроса или выберите его в интерактивном режиме с помощью [...]. (Для определения других имен макросов используйте остальные строки таблицы.)
5. Введите при необходимости еще одно описание макроса в столбце Макрос: Описание.
6. Закройте диалоговое окно Свойства, нажав кнопку ++ОК++.
7. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Если в первую строку вкладки Определение макроса ввести имя макроса, оно автоматически копируется в поле Имя вкладки Рамка макроса. И наоборот: имя макроса, определенное в поле Имя вкладки Рамка макроса, автоматически копируется в первую строку вкладки Определение макроса и отображается там в столбце Макрос: Имя.

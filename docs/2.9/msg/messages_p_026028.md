# P026028: Размещение изделия находится полностью внутри монтажного отверстия.

### Причина

Размещение изделия было полностью выполнено внутри монтажного отверстия и более не контактирует с монтажной поверхностью.

### Решение

1. Установите местоположение размещения изделия в пространстве листа с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Выполните перемещение размещения изделия таким образом, чтобы линия его контура еще контактировала с монтажной поверхностью.
3. Затем запустите новый контрольный прогон.

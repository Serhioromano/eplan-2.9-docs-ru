# P022025: Угол текста не равен 0° или 90°

### Причина

На схеме соединений был найден текст, угол которого не равен 0° или 90°.

### Решение

1. Установите местоположение текста на схеме соединений с помощью функциональности Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно Свойства ++...++ текста и выведите вкладку Формат на передний план.
3. Настройте для свойства Угол присвоение на 0,00° или 90,00°.
4. Закройте диалоговое окно Свойства ++...++, для чего нажмите на кнопку ++OK++.
5. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

* Проверяются тексты на страницах следующих типов:

    \- Многопол. схема соедин.
    \- Однопол. схема соедин.
    \- Схема соединений, Fluid-Техника
    \- Обзор.

* При присоединенных текстах свойств проверяется только текст основного свойства.
* Из проверки исключены тексты указания размеров, тексты внутри блоков и тексты внутри встроенных отчетов.

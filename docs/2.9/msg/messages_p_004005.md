# P004005: Недостающий вывод устройства ПЛК на обзоре

### Описание

Невозможно создать перекрестную ссылку между выводом устройства ПЛК в схеме соединений и выводом устройства ПЛК на странице обзора.

### Причина

Вывод устройства ПЛК в схеме соединений не может быть соединен перекрестной ссылкой с соответствующим выводом устройства ПЛК в обзоре из-за отсутствующего обозначения устройства блока ПЛК.

### Решение

Укажите одинаковое ОУ для блока ПЛК в схеме соединений и блока ПЛК в обзоре на выводе устройства ПЛК, после чего запустите контрольный прогон.

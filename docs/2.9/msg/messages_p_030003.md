# P030003: Из частичного проекта '\<x\>' были выгружены другие частичные проекты

### Причина

Часть проекта была выгружена как частичный проект \<x\>. Затем из этого выгруженного частичного проекта был выгружен другой частичный проект. При загрузке частичного проекта \<x\> обратно в первоначальный проект он удаляется. При этом проект, выгруженный из частичного проекта \<x\>, сохраняется, но больше не подлежит загрузке.

### Решение

1. Запустите любой контрольный прогон, чтобы удалить сообщение.
2. Загрузите частичные проекты в обратном порядке (как они были выгружены).

# P028022: Объект планирования не помещен в вид представления 'Предварительное планирование'

### Причина

Созданный в навигаторе предварительного планирования объект планирования еще не был помещен в вид представления "Предварительное планирование".

### Решение

1. Откройте страницу типа "Предварительное планирование", на которой должен размещаться объект планирования.
2. Откройте навигатор предварительного планирования (пункты меню Данные проекта > Предварительное планирование > Навигатор).
3. Выделите сообщение в диалоговом окне Управление сообщениями и выберите пункт всплывающего меню Синхронизировать выбор.

!!! info "Для сведения:"

    Соответствующий объект планирования выделяется в навигаторе предварительного планирования.

4. В навигаторе предварительного планирования выберите пункт всплывающего меню Разместить.

!!! info "Для сведения:"

    Рядом с курсором появится символ черного ящика.

5. Щелкните левой кнопкой мыши.
6. С помощью курсора растяните прямоугольник, представляющий блок в качестве объекта планирования.
7. Щелкните еще раз левой кнопкой мыши.

!!! info "Для сведения:"

    Объект планирования размещен.

8. Затем запустите новый контрольный прогон.

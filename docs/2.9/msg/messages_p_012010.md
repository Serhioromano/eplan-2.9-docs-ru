# P012010: Функциональный элемент на монтажной плате без представления в схеме соединений

### Причина

Главная функция функционального элемента, которая находится на монтажной плате, отсутствует в схеме соединений.

### Решение

Разместите в схеме соединений отсутствующую главную функцию функционального элемента, которая находится на монтажной плате.

Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Проверяются типы страниц "Многополюсная схема соединений", "Однополюсная схема соединений" и "Обзор".

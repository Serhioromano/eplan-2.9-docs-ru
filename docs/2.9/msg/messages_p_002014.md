# P002014: Многократно использован контакт штекера с одинаковым обозначением

### Причина

Контакт штекера используется неоднократно на страницах схемы соединений проекта с тем же полным ОУ.

### Решение

Определите местоположение многократно используемых контактов штекера в схеме соединений при помощи пункта меню Перейти к, или дважды щелкните по выданным сообщениям в диалоговом окне Управление сообщениями. Откройте диалоговое окно Свойства ++...++ для каждого из найденных контактов штекера и сравните полное ОУ. Выделите для каждого из многократно используемых контактов штекера другое полное ОУ, задав уникальные обозначения контактов.

!!! note "Замечание:"

    EPLAN позволяет раздельно отображать гнезда розетки и штыри штекера штекерных соединений. Для этого имеются специальные символы контактов штекера для гнезда розетки и штырей штекера. Штыри штекера и гнезда розетки (если они относятся к одному и тому же штекерному соединению, т. е. являются обратными эквивалентами) могут содержать одинаковые обозначения контакта штекера. Однотипные контакты штекера на стороне гнезд или штырей штекера должны, напротив, иметь разные обозначения.

Обновите соединения и затем повторно запустите контрольный прогон.

# P024016: Расхождение между ОУ соединения и ОУ группы соединений

### Причина

Точка определения соединения, которая находится в зоне воздействия линии определения группы соединений, имеет собственное обозначение устройства. ОУ группы соединений и ОУ точки определения соединения противоречат друг другу.

### Решение

Дважды щелкните по точке определения соединения в схеме соединений и удалите в диалоговом окне Свойства ++...++ видимое обозначение устройства точки определения соединения.

Обновите соединения и затем повторно запустите контрольный прогон.

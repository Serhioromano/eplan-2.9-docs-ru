# P026012: Размещение изделия накладывается на заблокированную для размещения зону

### Причина

Находящееся на монтажной поверхности размещение изделия накладывается на запретную зону для размещения или наоборот.

Возможные случаи наложения:

* Размещение изделия заходит на запретную зону для размещения.
* Запретная зона для размещения заходит на размещение изделия.
* Размещение изделия было расположено на запретной зоне для размещения.
* Запретная зона для размещения была расположена на размещении изделия.

### Решение

1. Установите местоположение размещения изделия или запретной зоны для размещения в пространстве листа с помощью функциональности "Перейти к", которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Переместите размещение изделия или запретную зону для размещения на монтажной поверхности таким образом, чтобы оба объекта не накладывались друг на друга.
3. Затем запустите новый контрольный прогон.

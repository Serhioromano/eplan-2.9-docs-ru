# P004084: MasterSystemID содержит ошибку

### Причина

На выводе шины, присвоенном системе шин на базе Ethernet или Profibus, указан недействительный MasterSystemID.

Этот контрольный прогон служит только для проверки выводов шины, на которых указан проект конфигураций.

### Решение

1. Установите местоположение вывода шины на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно "Свойства" вывода шины.
3. Переместите вкладку Данные шины на передний план.
4. Если установлена система шин "Profibus", в групповом поле Свойства сетевого интерфейса внесите в свойство MasterSystemID значение от 1 до 99. Если установлена система шин на базе Ethernet, в поле значений этого свойства введите значение, которое больше или равно 100.
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    На данный момент этот контрольный прогон действителен только для блоков управления Siemens.

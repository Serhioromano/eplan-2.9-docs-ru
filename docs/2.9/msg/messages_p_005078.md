# P005078: Соединение с указанием вещества без определения трубопровода

### Причина

На функциональной схеме автоматизации для соединения определено вещество. При этом на соединении отсутствует определение трубопровода (вывод трубопровода или точка определения трубопровода).

### Решение

1. Установите местоположение соединения на функциональной схеме автоматизации с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно свойств точки определения соединения.
3. Скопируйте в буфер обмена значение, внесенное в таблицу свойств для свойства Вещество, а затем удалите его.
4. Разместите вывод или точку определения трубопровода на соединении.
5. В диалоговом окне "Свойства" определения трубопровода в поле Вещество вставьте значение из буфера обмена.
6. Задайте имя трубопровода и определите состояние.
7. Обновите соединения и затем повторно запустите контрольный прогон.

# P003028: Кабель использует не все кабельные соединения изделия

### Причина

Вы разместили кабель в схеме соединений, которой присвоено изделие с установленными шаблонами функции (устройство). При размещении кабеля использовано меньше функций (кабельных соединений), чем предоставлено в соответствующем изделии. Например, для размещенного кабеля вы выбрали изделие, содержащее 4 кабельных соединения, из которых только 3 использованы в схеме соединений.

### Решение

#### Возможность 1

Выберите кабельное изделие с соответствующим числом шаблонов функций (количеством кабельных соединений).

#### Возможность 2

Уменьшите в базе данных изделий число шаблонов функций для выбранного изделия (количество кабельных соединений). Выберите для этого пункты меню Сервисные программы > Изделие > Управление и выберите в диалоговом окне База данных изделий вкладку Шаблоны функций. Выделите в таблице убираемые шаблоны функций и щелкните по кнопке ![](../images/all_delete_as.png){: .ui-icon } (Удалить). Примените измененную настройку в базе данных изделий. Затем еще раз разместите кабель как устройство в схеме соединений.

Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Вывод этого сообщения зависит от настроек Свернуть свободные функции и Число свободных функций. Перейти к настройкам можно, сначала открыв диалоговое окно Свойства устройства, переместив на передний план вкладку Изделие и нажав кнопку ++Выбор устройства++. В диалоговом окне Выбор устройства нажмите ++Настройки++ и откройте диалоговое окно Настройки: Выбор устройства с перечисленными ранее возможностями настройки. С помощью настройки Число свободных функций определите число неиспользуемых в устройстве функций (резервных функций), которое должно предоставлять изделие. Количество можно указать, только если установлен флажок в настройке Свернуть свободные функции.
    Если в настройке Число свободных функций установлено значение '0', то любой свободный шаблон воспринимается как ошибка.

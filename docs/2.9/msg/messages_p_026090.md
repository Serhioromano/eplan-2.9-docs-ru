# P026090: Не удалось рассчитать фактическую мощность потерь из-за отсутствующих значений

### Причина

Как минимум для одного из ниже перечисленных свойств на размещении изделия, релевантных для расчета мощности потерь, не записано никакого значения:

* Коэффициент одновременности (для устройств),
* Мощность потерь (для устройств),
* Поперечное сечение шины, Материал шины, Проводимость (при +20 °C) и / или Температурный коэффициент (для сборных шин).

### Решение

1. Установите местоположение размещения изделия в пространстве листа с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно свойств размещения изделия и проверьте на первой вкладке, выбрано ли одно из свойств Мощность потерь: Коэффициент одновременности (автоматически) или Мощность потерь: Коэффициент одновременности (устройство) с заданным значением больше 0 и меньше / равно 1.

!!! note "Замечание:"

    Значение свойства Мощность потерь: Коэффициент одновременности (автоматически) при вставке размещения изделия в пространство листа автоматически копируется из свойства проекта Мощность потерь: Коэффициент одновременности. Для размещений изделия можно определить коэффициент одновременности, отличающийся от проекта, в свойстве Мощность потерь: Коэффициент одновременности (устройство). При записи значения в это поле оно копируется также в свойство Мощность потерь: Коэффициент одновременности (автоматически). Значение, идущее из проекта, перезаписывается. Без указания коэффициента одновременности выполнить расчет мощности потерь невозможно.

1. Если значение для обоих свойств равно 0, запишите допустимое значение в свойство проекта Мощность потерь: Коэффициент одновременности или сохраните значение в свойстве Мощность потерь: Коэффициент одновременности (устройство) размещения изделия.
2. Проверьте, записано ли на размещении изделия изделие, имеющее указание мощности потерь.
3. Для этой цели переместите в диалоговом окне "Свойства" размещения изделия вкладку Изделие на передний план и вызовите на записанном изделии ***Выбор изделия***.
4. Переместите в выборе изделия вкладку Свойства на передний план и проверьте, записано ли в поле Макс. мощность потерь значение выше 0.
5. Если значение отсутствует, выберите другое изделие или добавьте изделию в ***базе данных изделий*** мощность потерь и выберите изделие еще раз.
6. Если в случае размещения изделия речь идет о ***сборной шине***, проверьте, записаны ли на изделии значения для следующих свойств:

\- Поперечное сечение шины;
\- Материал шины;
\- Проводимость (при +20 °C)
\- Температурный коэффициент.
7. Для этой цели откройте диалоговое окно ***Выбор изделия***, перейдите на вкладку Свойства и проверьте, присвоены ли вышеперечисленным свойствам значения больше 0.
8. Если нет, добавьте недостающее значение для сборной шины в ***базе данных изделий*** и выберите изделие сборной шины еще раз.
9. Запустите новый контрольный прогон и снова рассчитайте общую мощность потерь.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

# P027027: Распределитель проводов \<x\> находится не на трассе маршрутизации соединения

### Причина

На схеме соединений существует многополюсный распределитель, имеющий в топологии соответствующий размещенный обратный эквивалент. При маршрутизации соединений через сеть соединенных сегментов не был найден распределитель, так как точка маршрутизации, через которую к трассе маршрутизации изначально был подключен распределитель, удалена.

### Решение

1. Установите местоположение размещенного распределителя с видом представления "Топология", пользуясь функциональностью Перейти к (графика), которую вы можете найти во всплывающем меню диалогового окна Управление сообщениями.
2. Перемаршрутизируйте соединение так, чтобы оно проходило рядом с точкой маршрутизации, к которой подключен распределитель.

Или подсоедините распределитель к точке маршрутизации, расположенной на трассе маршрутизации.
3. После изменения обозначения устройства запустите любой контрольный прогон, чтобы удалить сообщение.
4. Маршрутизируйте соединения через сеть соединенных сегментов заново.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

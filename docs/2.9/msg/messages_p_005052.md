# P005052: Неприсоединенный вывод устройства на контакте штекера

### Причина

В схеме соединений был обнаружен несвязанный вывод устройства символов на контакте штекера. Это сообщение появляется также в том случае, если у контакта штекера есть несколько выводов устройства, из которых только один задействован в схеме соединений.

### Решение

1. Установите местоположение данного контакта штекера на схеме соединений, пользуясь функциональностью Перейти к (графика), которую вы можете найти во всплывающем меню диалогового окна Управление сообщениями.
2. Соедините все свободные выводы символа контакта штекера с условным обозначением так, чтобы образовались линии автоматического соединения.
3. Или, если контакт штекера больше не нужен, удалите его из схемы соединений.
4. Затем запустите новый контрольный прогон.

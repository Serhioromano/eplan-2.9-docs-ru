# P028030: Указанный в точке определения трубопровода класс трубы не соответствует классу трубы '\<x\>' связанного объекта планирования (трубопровод)

### Причина

В точке определения трубопровода, размещенной на функциональной схеме автоматизации, связанной с объектом планирования (трубопроводом) в предварительном планировании, указан класс трубы, не совпадающий с классом трубы на связанном объекте планирования (трубопроводе).

### Решение

1. Установите местоположение объекта планирования (трубопровода) в навигаторе предварительного планирования с помощью функции Синхронизировать выбор во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно "Свойства" объекта планирования (трубопровода) и переместите первую вкладку на передний план.
3. В поле Класс трубы выберите обозначение класса трубы \<x\>, выведенное в тексте сообщения, или введите его вручную.
4. Щелкните по кнопке ++OK++.
5. Затем запустите новый контрольный прогон.

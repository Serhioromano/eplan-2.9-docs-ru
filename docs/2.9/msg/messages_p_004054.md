# P004054: Соединение только в проекте! Соединение на '\<x\>' имеется только в проекте

### Причина

В проекте найдено сетевое соединение, которого не было при последнем импорте данных ПЛК.

### Решение

Уберите сетевое соединение на выводе шины указанного в сообщении устройства \<x\>.

!!! note "Замечание:"

    Это относящееся к модулю сообщение выводится при импорте данных ПЛК. При выполнении нового контрольного прогона (любой настроенной схемы) его можно удалить из диалогового окна управления сообщениями.

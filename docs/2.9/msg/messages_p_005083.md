# P005083: Соединение группы без цвета / номера

### Причина

Вы не сохранили цвет или номер на соединении группы.

### Решение

1. Установите местоположение точки определения группы соединений на схеме соединений, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно Свойства ++...++ точки определения соединения и запишите значение в поле Цвет / номер.
3. Обновите соединения и затем повторно запустите контрольный прогон.

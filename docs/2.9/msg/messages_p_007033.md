# P007033: Ручной функциональный текст на функции, у которой есть формат блока для вычисления автоматического функционального текста

### Причина

Вы определили на функции свойство блока для автоматического функционального текста. Однако автоматический функциональный текст, определенный как свойство блока, стал недействительным вследствие импорта, общей обработки или ручного ввода функционального текста.

### Решение

1. Установите местоположение этой функции на схеме соединений с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно свойств функции.
3. Удалите функциональный текст, вручную введенный на функции, если определенное свойство блока для автоматического функционального текста должно быть действительным. Если у введенного вручную функционального текста должен быть приоритет, не вносите изменений или удалите свойство блока для автоматического функционального текста из списка свойств.

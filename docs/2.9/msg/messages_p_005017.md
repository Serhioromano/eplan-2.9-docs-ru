# P005017: Цель отсутствует

### Причина

На неразмещенном соединении не указана функция, которая является целью и / или источником соединения.

### Решение

#### Возможность 1

1. Откройте навигатор соединений, выбрав пункты меню Данные проекта > Соединения > Навигатор.
2. Нажмите на сообщение в диалоговом окне управления сообщениями и выберите из всплывающего меню пункт Синхронизировать выбор, чтобы определить недостающее соединение.

3. Соединение будет выделено в навигаторе соединений.
4. Откройте диалоговое окно "Свойства" соединения и добавьте в списке свойств для свойства Источник и / или свойства Цель обозначение устройства существующей в проекте функции.
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

#### Возможность 2

1. Выберите пункты меню Проект > Организовать > Сжать, чтобы удалить ненужные данные из проекта.
2. Установите флажок Убрать неразмещенное соединение в настройках сжатия.
3. Затем запустите новый контрольный прогон.

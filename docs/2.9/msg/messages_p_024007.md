# P024007: Главная функция с неопределенным разделом

### Причина

В главной функции устройства Fluid для свойства Раздел было выбрано значение "Свойство Fluid (не определено)".

### Решение

1. Установите местоположение устройства Fluid на схеме соединений с помощью функции Перейти к (графика), которая имеется в диалоговом окне всплывающего меню Управление сообщениями.
2. Присвойте одной из главных функций обозначенный раздел.
3. При обозначении устройства главной функции используйте в отношении выбранного раздела правильный ключ среды по стандарту DIN ISO 1219-2.
4. Затем запустите новый контрольный прогон.

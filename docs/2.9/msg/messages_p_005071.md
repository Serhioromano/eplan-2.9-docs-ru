# P005071: Различные обозначения соединений в одном сигнале

### Причина

Внутри сигнала есть точки определения соединения с различными обозначениями соединений.

### Решение

1. Установите местоположение данной точки определения соединения на схеме соединений, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Удалите обозначения соединений или переименуйте их так, чтобы все обозначения соединений внутри сигнала выглядели одинаково.
3. Обновите соединения и затем повторно запустите контрольный прогон.

!!! tip "Совет:"

    Воспользуйтесь функцией нумерации соединений, чтобы автоматически задать обозначения соединений.

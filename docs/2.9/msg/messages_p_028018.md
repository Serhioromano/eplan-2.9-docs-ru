# P028018: Разница между сегментом и шаблоном сегмента (свойство '\<x\>')

### Причина

Сегменту был присвоен шаблон сегмента с определенными свойствами. После присвоения на сегменте было изменено значение для свойства \<x\>.

### Решение

Присвойте свойству \<x\> на сегменте то же значение, что было определено для свойства \<x\> на шаблоне сегмента:

1. Откройте навигатор предварительного планирования и навигатор шаблонов сегментов (пункты меню Данные проекта > Предварительное планирование > Навигатор и Данные проекта > Предварительное планирование > Навигатор шаблонов сегментов).
2. Дважды щелкните по сообщению в диалоговом окне Управление сообщениями.

!!! info "Для сведения:"

    Сегмент выделяется в навигаторе предварительного планирования.

!!! info "Для сведения:"

    Используемый шаблон сегмента выделяется в навигаторе шаблонов сегментов.

3. Вызовите диалоговое окно Свойства ++...++ сегмента.
4. Определите в групповом поле Свойства значение для отображенного в тексте сообщения свойства \<x\> и запишите его.
5. Перейдите в навигатор шаблонов сегментов и вызовите для выделенного шаблона сегмента диалоговое окно Свойства ++...++.
6. Переместите вкладку Сегмент структуры на передний план.
7. Найдите в групповом поле Свойства отображаемое в тексте сообщения свойство \<x\> и скопируйте значение в буфер обмена.
8. Закройте диалоговое окно свойств шаблона сегмента и снова откройте с помощью навигатора предварительного планирования диалоговое окно свойств сегмента.
9. Введите в поле для свойства \<x\> скопированное значение из буфера обмена и закройте диалоговое окно свойств сегмента, нажав ++OK++.
10. Затем запустите новый контрольный прогон.

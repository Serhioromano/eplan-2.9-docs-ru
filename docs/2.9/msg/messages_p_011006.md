# P011006: Кольцевое соединение для точек разрыва

### Причина

Острие стрелки обратного эквивалента двух точек разрыва, связанных перекрестной ссылкой, соединено с концом стрелки первой точки разрыва. Таким образом, возникает кольцевое соединение.

### Решение

Разорвите соединение между точками разрыва, связанными перекрестной ссылкой.

Обновите соединения и затем повторно запустите контрольный прогон.

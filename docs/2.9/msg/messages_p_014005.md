# P014005: В пространстве листа не определено имя макроса

### Причина

Для автоматической генерации макросов из проекта макросов в пространстве листа должно быть указано как минимум одно имя макроса. Оно должно быть определено в качестве свойства как на вкладке Пространство листа, так и на вкладке Определение макроса. В этом случае запись отсутствует на обеих вкладках.

### Решение

1. Откройте навигатор пространств листа.
2. Установите местоположение пространства листа в проекте макросов с помощью функции Синхронизировать выбор, которая находится во всплывающем меню диалогового окна Управление сообщениями.

!!! info "Для сведения:"

    Пространство листа будет выделено в навигаторе пространств листа.

3. Откройте с помощью всплывающего меню навигатора пространства листа диалоговое окно свойств пространства листа.

4. Переместите вкладку Определение макроса на передний план.
5. В ***первую строку*** таблицы, в столбце Макрос: Имя, введите требуемое имя макроса или выберите его в интерактивном режиме с помощью [...]. (Для определения других имен макросов используйте остальные строки таблицы.)
6. Введите при необходимости еще одно описание макроса в столбце Макрос: Описание.
7. Закройте диалоговое окно Свойства, нажав кнопку ++ОК++.
8. Затем запустите новый контрольный прогон.

!!! note "Замечание:"

    Если в первую строку вкладки Определение макроса ввести имя макроса, оно автоматически копируется в таблицу свойств вкладки Пространство листа и отображается там в виде свойства Макрос: Имя. И наоборот: имя макроса, определенное для свойства Макрос: Имя на вкладке Пространство листа, автоматически копируется в первую строку вкладки Определение макроса и отображается там в столбце Макрос: Имя.

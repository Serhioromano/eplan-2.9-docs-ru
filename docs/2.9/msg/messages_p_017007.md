# P017007: ОУ не имеет перекрестной ссылки

### Причина

Предположительно, обозначения устройств обеих функций вводились по-разному:

Заявленное обозначение устройства при вставке в схему соединений было введено "обычным" путем через поле Видимое ОУ в диалоговом окне 'Свойства'. Однако одноименное обозначение устройства другой функции было введено через диалоговое окно Полное ОУ, при этом в поле Буквенное обозначение занесены как буквенное обозначение, так и счетчик, которые взаимосвязаны.

!!! note "Замечание:"

    Диалоговое окно Полное ОУ можно открыть, щелкнув в диалоговом окне Свойства ++...++ рядом с полем Полное ОУ по кнопке ++...++.

### Решение

Измените обозначение устройства, которое было введено в диалоговом окне Полное ОУ. Отдельно введите буквенное обозначение и счетчик в соответствующие поля.

Затем запустите новый контрольный прогон.

# P022060: Определенное пользователем свойство '\<x\>' (помощь ввода '\<z\>') содержит недействительное значение '\<y\>'

### Причина

В проекте на объектах (страницах, макросах, функциях и т. д.) используются определенные пользователем свойства, а для присвоения значений по умолчанию — список выбора. При копировании из другого проекта или при вставке макроса определенному пользователем свойству '\<x\>' с использованием помощи ввода '\<z\>' было присвоено значение по умолчанию '\<y\>', которое не сконфигурировано в списке выбора.

### Решение

1. Установите местонахождение объекта на схеме соединений с помощью функции Перейти к (графика) во всплывающем меню диалогового окна Управление сообщениями или выделите отображенный в диалоговом окне номер страницы в навигаторе страниц, если объект представляет собой страницу.
2. Вызовите диалоговое окно Свойства объекта.
3. Выберите для свойства '\<x\>' действительное значение по умолчанию '\<y\>' из списка выбора, используя тип ввода '\<z\>'.
4. Щелкните по кнопке ++OK++.
5. Затем запустите новый контрольный прогон.

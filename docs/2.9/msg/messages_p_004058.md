# P004058: Предмет оборудования '\<x\>' в функции '\<y\>' имеется только в проекте

### Причина

При импорте файла AML установлено, что в проекте на функции '\<y\>' существует предмет оборудования '\<x\>', не содержащийся в файле импорта.

### Решение

1. Установите местоположение данной функции на схеме соединений, пользуясь функциональностью Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Проверьте, требуется ли предмет оборудования.
3. Если нет, удалите его на функции.
4. Затем повторите импорт ПЛК.

!!! note "Замечание:"

    Это относящееся к модулю сообщение выводится при импорте данных ПЛК. При выполнении нового контрольного прогона (любой настроенной схемы) его можно удалить из диалогового окна управления сообщениями.

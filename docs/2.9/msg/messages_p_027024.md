# P027024: Определение функции топологии не подходит для главной функции

### Причина

Внутри сети соединенных сегментов найдена функция топологии, не являющаяся главной функций, определение функции которой не совпадает с соответствующей многополюсной главной функцией на схеме соединений.

### Решение

1. Установите местоположение функции топологии в своей сети соединенных сегментов при помощи функциональности Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Из всплывающего меню выберите пункт Перейти ко 2-й координате.

!!! info "Для сведения:"

    Выполняется обращение к многополюсному обратному эквиваленту — главной функции.

3. Вызовите диалоговое окно Свойства ++...++ главной функции и перейдите на вкладку Данные символа / функции.
4. Запомните запись в поле Определение группового поля Данные функции (логика).
5. Закройте диалоговое окно Свойства ++...++ главной функции, нажав на кнопку ++Отменить++.
6. Откройте всплывающее меню на точке вставки главной функции и выберите пункт Перейти к (обратный эквивалент).

!!! info "Для сведения:"

    Осуществляется переход к функции топологии.

7. Вызовите диалоговое окно Свойства ++...++ функции топологии и перейдите на вкладку Данные символа / функции.

8. Щелкните мышью в групповом поле Данные функции (логика) в поле Определение по ++...++, и установите на функции топологии такое же определение функции, как и у главной функции.
9. Закройте диалоговое окно Свойства ++...++, нажав ++OK++, а затем запустите новый контрольный прогон.

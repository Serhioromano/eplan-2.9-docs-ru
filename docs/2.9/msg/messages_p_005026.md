# P005026: Не присоединенный вывод устройства

### Причина

Как минимум один вывод устройства условного обозначения не соединен с другим условным обозначением. Таким образом, отсутствует структура соединения на выводе устройства.

### Решение

Соедините соответствующий вывод устройства условного обозначения с выводом устройства другого условного обозначения таким образом, чтобы образовалась линия автоматического соединения, имеющая цели.

Обновите соединения и затем повторно запустите контрольный прогон.

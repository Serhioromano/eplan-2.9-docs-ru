# P022020: Символ '\<x\>/\<y\>' из библиотеки символов '\<z\>' возник во время копирования данных проекта EPLAN 5 или EPLAN 21

### Описание

В проекте был обнаружен ***устаревший*** символ под номером > 3000, который возник при копировании данных проекта EPLAN 5 или EPLAN 21. Символ не следует более использовать.

Значение заполнителей в тексте сообщения:

* \<x\> указывает номер символа.
* \<y\> указывает имя символа.
* \<z\> указывает имя библиотеки символов.

С помощью функции Перейти к (графика), которая находится во всплывающем меню диалогового окна Управление сообщениями, можно определить местоположение соответствующего символа на схеме соединений, не выходя из управления сообщениями.

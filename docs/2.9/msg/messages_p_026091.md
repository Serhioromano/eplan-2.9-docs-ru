# P026091: Неполные свойства проекта для расчета кондиционирования

### Причина

Минимум для одного из перечисленных далее свойств проекта, нужных для расчета кондиционирования в электрошкафу, не указано значение:

* Тепловой расчет: Коэффициент одновременности
* Тепловой расчет: Напряжение
* Тепловой расчет: Частота.

### Решение

1. Выберите пункты меню Проект > Свойства и проверьте на вкладке Свойства, для какого из вышеперечисленных свойств не указано значение.
2. Введите недостающие значения.
3. Закройте диалоговое окно свойств проекта, нажав ++OK++, а затем запустите новый контрольный прогон.

# P003002: Невозможно выполнить автоматический выбор кабеля для этого кабеля, так как соответствующий критерий фильтра не найден

### Причина

Вы создали кабель при помощи ***автоматического выбора кабеля***. Однако у кабеля уже есть созданное вручную определение кабеля, которое включает определенные значения, например, для свойств Число соединений, Напряжение, Поперечное сечение и т. д. И одно или несколько из этих значений не подходит для кабелей, которые вы сохранили в таблице Предв. выбор кабеля для автоматического выбора кабеля. То есть отсутствует подходящее изделие кабеля для определенного вручную кабеля.

!!! example "Пример:"

    Вы определили кабель со значением напряжения 1000 В. Однако в таблице для предв. выбора кабеля имеются только изделия кабеля с напряжением 230 В и 400 В.

### Решение

Приведите предварительный выбор кабеля в соответствие с определенным вручную значением напряжения.

Выберите пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Устройствo > Кабель (автоматически). Нажмите на ++...++ рядом с полем Автоматический выбор кабеля. Щелкните в диалоговом окне Настройки: Автоматический выбор кабеля справа над таблицей Предв. выбор кабеля по ![](../images/all_new_as.png){: .ui-icon } (Создать). Перейдите в древовидной структуре открывшегося диалогового окна Выбор изделия к группе продуктов Кабели/соединения и выберите изделие кабеля с соответствующими значениями свойств. При помощи кнопки ++OK++ изделие будет перенесено в таблицу предварительного выбора кабеля. При необходимости повторите выбор для других кабелей. Закройте диалоговое окно и создайте кабель снова автоматически.

!!! note "Замечание:"

    Это относящееся к модулю сообщение выводится при автоматическом выборе кабеля. При выполнении нового контрольного прогона (любой настроенной схемы) его можно удалить из диалогового окна управления сообщениями.

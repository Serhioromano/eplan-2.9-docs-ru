# P004071: Интерфейс шины '\<x\>': Свойство 'Интерфейс шины: Главный вывод шины' задано многократно или не активировано

### Причина

В карте ПЛК один или несколько выводов шины были присвоены определенному интерфейсу шины путем ввода единого обозначения на свойстве Интерфейс шины: Имя. При этом вы либо забыли активировать свойство Интерфейс шины: Главный вывод шины, либо это свойство было активировано многократно.

Этот контрольный прогон служит только для проверки выводов шины, на которых указан проект конфигураций.

### Решение

1. Определите местоположение выводов шины на схеме соединений с помощью функций Перейти к (графика) и Перейти ко 2-й координате во всплывающем меню диалогового окна Управление сообщениями.
2. Откройте диалоговое окно Свойства ++...++ для соответствующего вывода шины / соответствующих выводов шины.
3. Переместите вкладку Данные шины на передний план.
4. Если существует только ***один*** вывод шины с именем интерфейса шины, активируйте для него свойство Интерфейс шины: Главный вывод шины. Если существуют ***несколько*** выводов шины с ***одинаковыми*** именами интерфейсов шины, активируйте свойство Интерфейс шины: Главный вывод шины только для ***одного*** из этих выводов шины.
5. Щелкните по кнопке ++OK++.
6. Затем запустите новый контрольный прогон.

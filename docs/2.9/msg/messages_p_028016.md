# P028016: Сегмент подчинен неразрешенному определению сегмента '\<x\>'

### Причина

Сегмент был подчинен недопустимому определению сегмента \<x\>, которому он не должен быть подчинен.

### Решение

1. Установите местоположение определения сегмента в навигаторе предварительного планирования с помощью функции Синхронизировать выбор, которая находится во всплывающем меню диалогового окна Управление сообщениями.
2. Выберите для сегмента другое определение сегмента или адаптируйте данное определение сегмента.
3. Затем запустите новый контрольный прогон.

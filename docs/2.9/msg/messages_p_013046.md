# P013046: Незнакомое ОУ '\<x\>'

### Причина

Вы импортировали в проект спецификацию. Одно обозначение устройства, указанное в импортированной спецификации, не было найдено в проекте.

### Решение

Откройте используемый импортируемый файл в текстовом редакторе или редакторе файлов XML. Сравните обозначения устройства, указанные в импортированном файле в тэге XML "<device_P_ARTICLEREF_IDENTNAME>", c обозначениями устройства, встречающимися в схеме соединений вашего проекта. Обнаружив несовпадение, измените обозначение устройства на схеме соединений так, чтобы оно совпало с обозначением устройства импортированного файла. Затем сохраните файл и заново импортируйте его в спецификацию.

Более подробную информацию и рекомендации вы найдете в разделе [Импорт спецификаций](../partslistgui_h_stuecklistenimportieren.md).

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

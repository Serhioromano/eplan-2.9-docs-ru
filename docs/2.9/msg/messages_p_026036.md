# P026036: Размещение изделия находится в области маршрутизации

### Причина

Размещение изделия было выполнено внутри области маршрутизации на монтажной поверхности.

### Решение

1. Установите местоположение размещения изделия в пространстве листа с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Переместите размещение изделия из области маршрутизации или удалите его. Либо измените область маршрутизации на монтажной поверхности.
3. Затем запустите новый контрольный прогон.

# P013055: На защищенном устройстве изменить изделие нельзя

### Причина

При импорте спецификации была предпринята попытка изменить изделие у защищенного устройства. Но изделие невозможно изменить.

### Решение

Отмените изменение и заново импортируйте спецификацию.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

# P005034: Обозначенная точка определения соединения находится на многократном соединении

### Описание

Это сообщение выводится в том случае, если при обозначении точек определения соединений одна из этих точек находится на многократном соединении.

!!! note "Замечание:"

    Это относящееся к модулю сообщение можно удалить при выполнении нового контрольного прогона (любой настроенной схемы) из диалогового окна управления сообщениями.

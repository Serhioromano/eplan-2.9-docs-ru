# P003019: Подкабель сгенерирован

### Причина

Автоматически сгенерирован кабель с подкабелями.

!!! note "Замечание:"

    Это относящееся к модулю сообщение выводится при автоматическом генерировании кабеля. При выполнении нового контрольного прогона (любой настроенной схемы) его можно удалить из диалогового окна управления сообщениями.

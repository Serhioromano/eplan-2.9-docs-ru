# P005042: Значение свойства 'Цвет/номер' является цветовым кодом или именем цвета, которое отсутствует в меню 'Цвета соединений' в настройках проекта

### Причина

В диалоговом окне Настройки: Цвета соединений (Пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Цвета соединений) могут быть определены отдельно для каждого проекта через коды цветов или имена цветов. Выбранное вами значение для свойства Цвет / номер является кодом или именем цвета, не перечисленного в таблице Цвета соединений.

### Решение

#### Возможность 1

Выберите код или имя цвета, определенного в таблице цветов соединений:

1. Откройте диалоговое окно Свойства ++...++ для соответствующего соединения или соответствующей точки определения соединения.
2. Щелкните рядом с полем Цвет/номер по ++...++.
3. Выделите в открывшемся диалоговом окне Цвета соединений один из возможных кодов цвета или имя цвета и нажмите ++OK++.

#### Возможность 2

Добавьте код или имя желаемого цвет в перечень цветов соединений, заложенных в проекте:

1. Выберите пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Цвета соединений.
2. Щелкните в диалоговом окне Настройки: Цвета соединений по ![](../images/all_new_as.png){: .ui-icon } (Создать) и занесите нужное значение в новую строку таблицы в разделе Цвет. код или Цвет. имя.

Обновите соединения и затем повторно запустите контрольный прогон.

# P026013: Размещение изделия накладывается на заблокированную для сверления зону

### Причина

Находящееся на монтажной поверхности размещение изделия накладывается на запретную зону для сверления или наоборот.

Возможные случаи наложения:

* Размещение изделия заходит на запретную зону для сверления.
* Запретная зона для сверления заходит на размещение изделия.
* Размещение изделия было расположено на запретной зоне для сверления.
* Запретная зона для сверления была расположена на размещении изделия.

### Решение

1. Установите местоположение размещения изделия или запретной зоны для сверления в пространстве листа с помощью функциональности Перейти к, которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. Переместите размещение изделия или запретную зону для сверления на монтажной поверхности таким образом, чтобы оба объекта не накладывались друг на друга.
3. Затем запустите новый контрольный прогон.

# P028010: Данные вывода устройства ПЛК отличаются от данных адреса ПЛК в предварительном планировании

### Причина

Идентифицируется по имени ЦПУ и символическому адресу. Адрес, текст функции, тип данных и направление могут отличаться. Направление выводов устройства передается из определения функции.

### Решение

1. Синхронизируйте данные выводов устройства ПЛК с данными адреса ПЛК в предварительном планировании или присвойте данные адреса ПЛК выводу устройства ПЛК в подробном планировании.
2. Затем запустите новый контрольный прогон.

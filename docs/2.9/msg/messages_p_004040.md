# P004040: Для вывода шины на '\<x\>' найдены противоречивые данные на соответствующем выводе шины

### Причина

Кабельные выводы сети / шины типа "Кабельный вывод сети / шины, общее" рассматриваются как взаимосвязанные и автоматически связываются между собой посредством внутреннего соединения, если имеют одинаковое ОУ, принадлежат к одной и той же системе шин и имеют одинаковое имя интерфейса шины.

В этом случае были заданы противоречивые данные на одном из выводов шин, внутренне связанных между собой, поскольку одно или несколько значений, указанных после этого свойства, отличаются друг от друга:

* Физическая сеть: Имя
* Логическая сеть: Имя
* Логическая сеть: Вывод шины является главным
* Физическая сеть: Адрес шины / номер позиции
* Физическая сеть: Адрес шины / номер позиции 2
* Имя устройства в DNS / PROFINET
* Скорость передачи данных
* Интегрированный узел
* MasterSystemID
* Маска подсети
* Подразъем
* Выделено для удаления.

В заполнителе \<x\> сообщения выводится полное ОУ соответствующего вывода шины.

### Решение

1. Установите местоположение выводов шины на схеме соединений с помощью функции Перейти к (графика) и Перейти ко 2-й координате во всплывающем меню диалогового окна Управление сообщениями.
2. Вызовите диалоговое окно 'Свойства' вывода шины и сравните внесенные свойства на вкладке Данные шины со списком, приведенным выше.
3. Внесите в соответствующие свойства однозначные значения.

4. Щелкните по кнопке ++OK++.
5. Запустите новый контрольный прогон.

# P026100: Невозможно обновить размещение изделия: \<x\>

### Причина

Возможно несколько причин. Причина, по которой появилось сообщение, выводится в заполнителе \<x\> сообщения.

### Решение

1. Установите местоположение размещения изделия в пространстве листа с помощью функции Перейти к (графика), которую можно найти во всплывающем меню диалогового окна Управление сообщениями.
2. В зависимости от информации, отображенной в заполнителе, выполните одно из следующих корректировочных действий:

Заполнитель |  Решение
---|---
* Электрошкаф не может быть обновлен
|  Отсутствует (электрошкафы невозможно обновить).
* Не удалось найти изделие
|  Создайте требуемое изделие в базе данных изделий.
* Определение функции изделия не соответствует его размещению
|  Проверьте определение функции изделия и приведите его в соответствие с размещением.
* Группа определений функций макроса изделия не соответствует размещению изделия
* Макрос изделия содержит больше одного логического функционального элемента
* Макрос содержит больше одного варианта вида представления "Трехмерный чертеж монтажных поверхностей"
|  Обработайте макрос таким образом, чтобы выполнялось соответствующее условие.
3. Запустите контрольный прогон (по произвольной схеме), чтобы удалить относящееся к модулю сообщение из диалогового окна управления сообщениями.
4. Обновите размещение изделия.

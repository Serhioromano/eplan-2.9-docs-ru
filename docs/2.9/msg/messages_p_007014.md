# P007014: 'Функциональный текст (автоматич.)' в разных видах представлений различается (в языке '\<x\>')

### Причина

На размещенной в схеме соединений функции, которая существует в различных видах представлений (например, как многополюсная, однополюсная функция или функция обзора), запись в поле Функциональный текст (автоматически) отличается при одном или нескольких представлениях в одном из выбранных языков отображения.

### Решение

Унифицируйте запись в поле Функциональный текст (автоматически) в выбранном языке для всех видов представления функции.

1. Дважды щелкните в диалоговом окне Управление сообщениями строку с сообщением 007014.

!!! info "Для сведения:"

    В графическом редакторе выделяется главная функция.

2. Щелкните правой клавишей мыши, выберите из открывшегося всплывающего меню пункт Свойства (общие).
3. Щелкните в диалоговом окне Свойства (общие): ++...++ вкладку <Категория функции> (Устройство) в поле Функциональный текст и вызовите правой кнопкой мыши всплывающее меню.
4. Выберите пункт меню Перевести.

!!! info "Для сведения:"

    Открывается диалоговое окно Перевести, которое показывает существующие в словаре переводы текста на все языки.

5. Нажмите ++OK++ и закройте диалоговое окно Свойства (общие): ++...++, нажав ++OK++.

!!! info "Для сведения:"

    Текст будет отображаться целиком на выбранном вами языке в схеме соединений для всех видов представления функции.

6. Затем запустите новый контрольный прогон.

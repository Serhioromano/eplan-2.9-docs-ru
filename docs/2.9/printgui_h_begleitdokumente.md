# Печать присоединенных документов

В EPLAN присоединенные документы могут быть распечатаны по отдельности. Присоединенными называются внешние документы, вставленные на страницы схемы соединений при помощи гиперссылок или внесенные в используемые изделия. Чтобы присоединенные документы могли быть распечатаны, файлы должны находиться на локальном или на сетевом диске, к которому у вас есть доступ.

Условия:

* Вы выделили проект или страницы в навигаторе страниц.
* Вы вставили внешние документы при помощи гиперссылок или внесли в изделие.

1. Выберите следующие пункты меню: Проект > Печать присоединенных документов

!!! info "Для сведения:"

    В диалоговом окне Печать присоединенных документов перечислены все присоединенные документы, связанные с проектом. По умолчанию в столбце Печать для вывода на печать выделены все документы.

2. В диалоговом окне Печать присоединенных документов снимите флажок в столбце Печать для тех документов, которые ***не*** нужно печатать.
3. Щелкните по кнопке ++OK++.
4. В диалоговом окне Печать в групповом поле Принтер из раскрывающегося списка выберите необходимый принтер.
5. В поле Число задайте количество экземпляров, которое необходимо распечатать.

!!! info "Для сведения:"

    Кнопки ++Настройки++ и [Предв. просм. перед печатью], а также поля для выбора диапазона страниц и порядка печати недоступны, поскольку данные настройки невозможно применить к присоединенным документам.

6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Процесс печати запущен.

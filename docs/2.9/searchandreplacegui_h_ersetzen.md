# Заменить данные проекта

После поиска можно выбрать данные проекта и заменить их значения другими.

1. Выберите следующие пункты меню: Поиск > отобразить результаты
2. В диалоговом окне Результаты поиска — <Имя проекта> в списке результатов выделите запись, значение которой необходимо изменить. Возможен многократный выбор.
3. Выберите пункт всплывающего меню Заменить.

!!! info "Для сведения:"

    В поле Поиск диалогового окна Заменить отображается значение (первой) выделенной записи.

4. Введите в поле Заменить на значение, которым вы хотите заменить значение в поле Поиск.
5. Определите с помощью флажка, должен ли при поиске учитываться Регистр или только весь текст (то есть в точности записанный в поле ввода текст).
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В выделенных записях меняется значение.

# Диалоговое окно Изменение свойства

Вы открыли проект.

* Открыта динамическая форма (например, через меню Сервисные программы > Основные данные > Форма > Открыть > [Открыть]). Вставить > Динамическая область > Область условий. Область условий вытянута, как прямоугольник. Дважды щелкните по прямоугольнику и выберите в диалоговом окне свойств вкладку Условия. В поле При изменении свойств щелкните ++...++.
* Сервисные программы > Основные данные > Форма > Открыть > [Открыть]. Вы открыли диалоговое окно Свойства формы — <Имя формы> (Всплывающее меню > Свойства на форме в диалоговом окне навигатора Страницы — <Имя проекта>) и в поле Значение свойства При изменении заново генерировать заголовок щелкнули кнопку ++...++.

В данном диалоговом окне выберите свойства, при которых используется изменение области условий / заголовка.

Обзор основных элементов диалогового окна:

### Свойство

Щелкните ![](images/all_new_as.png){: .ui-icon } (Создать) на панели инструментов вверху листа, чтобы открыть диалоговое окно Выбор свойств и там выбрать свойства, при которых используется изменение области / заголовка. Для списка обозначений устройств можно, например, задать, что при каждом изменении свойства ОУ (видимое) будет начерчена толстая линия для следующего ОУ или сгенерирован новый основной заголовок.

### Всплывающее меню

Пункт меню |  Значение
---|---
Выделить все |  Выделяет все свойства в таблице.
Подобрать ширину столбца |  Все столбцы так подогнаны к таблице по ширине, что и заголовок, и содержимое столбцов полностью читаемы.

# Закрыть проекты / области обработки

EPLAN позволяет закрывать проекты в любой момент. Затем проект защищается от записи и не может быть изменен. Закрытый проект сохраняется в каталоге проектов с расширением имени файлов *.elr.

Области обработки также можно закрывать в любой момент. В этом случае от записи будут защищены только те страницы или пространства листа, которые относятся к выбранной области обработки.

Условие:

Открыт стандартный проект (Расширение имени файла *.elk) или проект ревизии (Расширение имени файла *.ell).

1. Выберите пункты меню Сервисные программы > Управление ревизиями > Закрыть проект / область.

!!! info "Для сведения:"

    Если закрыть ***стандартный проект***, проект защищается от записи и сохраняется с расширением имени файла *.elr.

!!! info "Для сведения:"

    При закрытии ***проекта ревизии*** открывается диалоговое окно Закрыть проект.

!!! info "Для сведения:"

    При закрытии ***области обработки*** открывается диалоговое окно Закрыть область обработки.

2. В диалоговом окне [Закрыть проект](revisionlogmgtgui_d_projektabschliessen.md) (или Закрыть область обработки) введите данные ревизии (индекс ревизии, описание и причину изменения).
3. Установите флажок Сгенерировать отчет при закрытии проекта, если при закрытии проекта все страницы отчета проекта должны обновляться.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Сначала закрываются все страницы проекта / области обработки и данные ревизии присваиваются всем измененным страницам проекта ревизии.

!!! info "Для сведения:"

    Если закрыть проект, он защищается от записи и сохраняется с расширением имени файла *.elr.

!!! info "Для сведения:"

    Если закрыть область обработки, страницы области обработки защищаются от записи.

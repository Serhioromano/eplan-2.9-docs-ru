# EPLAN Fluid: Предварительное назначение свойств для соединений Fluid-техники

На соединения Fluid-техники можно предварительно накладывать свойства соединения со значениями. Это предварительное задание значений для разделов "Гидравлика", "Пневматика", "Охлаждение", "Смазка", "Смазочно-охлаждающая жидкость" и "Газовая техника" можно выполнить в меню Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Свойства.

Значения данных свойств по умолчанию присваиваются всем соединениям проекта. Если вы устанавливаете точку определения соединения для какого-нибудь соединения, значения его свойств перезаписывают стандартно заданные для этого соединения.

# Пронумеровать Fluid-ОУ согласно стандарту DIN ISO 1219-2 в автономном режиме

Используйте для Fluid-устройств идентификатор, соответствующий стандарту DIN ISO 1219-2, тогда с помощью справки нумерации в автономном режиме вы сможете увеличивать пошагово номер функционального элемента в ОУ.

Условия:

* Вы открыли проект.

* Проект создан с использованием шаблона `FL_1219-2_tpl001` . Благодаря этому шаблону проекта обозначения устройств Fluid-техники структурируются согласно стандарту DIN ISO 1219-2 по зависимым от раздела ***ключу среды***, ***номеру логической ячейки*** и ***номеру функционального элемента***.
* Проект содержит одну или несколько страниц схем соединений типа "Схема соединений Fluid-техники" с устройствами Fluid-техники, обозначенными в соответствии со стандартом.

1. Выделите несколько устройств Fluid-техники на одной странице схемы соединений или выделите в навигаторе страниц несколько страниц схемы соединений или проект.
2. Выберите следующие пункты меню: Данные проекта > Устройство > Нумеровать.

3. В диалоговом окне [Нумеровать (автономно)](offlinenumerationgui_d_nummerieren.md) в столбцах Начальное значение и Величина шага задайте значение, с которого начинаются номера функциональных элементов соответствующего ОУ Fluid-техники при нумерации в автономном режиме, и величину шага (разницу) между двумя номерами.
4. Щелкните по кнопке ++OK++.

!!! example "Пример:"

    ТриустройстваFluid-техники на странице схемы соединений имеют следующие обозначения устройства:P1.1,P1.2иP1.3В качестве начального значения вы выбрали5, а в качестве величины шага —3.После нумерации в автономном режиме новые ОУ выглядят следующим образом:P1.5,P1.8иP1.11

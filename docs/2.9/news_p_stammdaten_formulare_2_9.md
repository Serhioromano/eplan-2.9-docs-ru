# Основные данные: Формы

В этом разделе содержится информация о наиболее важных изменениях в формах.

Создана новая форма для ***схемы подключения устройства***:

* F05_004.f05 (тип файла "Схема подключения устройства"; для встроенных отчетов)

В перечисленных ниже формах для ***спецификации клеммника*** обновлены тексты-заполнители в верхнем колонтитуле и области данных:

* F13_008.f13
* F13_008_en_US.f13

Устаревшие заполнители с обозначением "через позицию в форме" были заменены заполнителями с обозначением "внешн." или "внутр." (например, "Таблицы кабелей — верхний колонтитул над позицией в форме [A] / ОУ (полное)" на "Таблицы кабелей — верхний колонтитул, внешн. / ОУ (полное)").

Две формы для вывода ***обзора ревизий*** были исправлены.

* F17_002.f17
* F17_002_en_US.f17

Свойство При изменении заново генерировать заголовок было добавлено в свойства этих форм, и ему присвоено определенное значение. При изменении свойства Имя ревизии (из отслеживания изменений) [1] во время генерации отчета генерируется новый заголовок. Кроме того, в формах текст-заполнитель "Страница / имя ревизии (из отслеживания изменений) [1]" был заменен текстом-заполнителем "Проект / имя ревизии (из отслеживания изменений) [1]".

В перечисленных ниже формах для ***обзора трубопровода*** в заголовках столбцов оптимизированы блоки выравнивания:

* F37_003.f37 (тип файла "Функциональная схема автоматизации: Обзор трубопровода")
* F37_003_en_US.f37

Это означает, что границы столбцов больше не выходят за пределы, даже для языков с длинными словами.

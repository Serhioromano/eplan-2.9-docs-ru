# Диалоговое окно Настройки: База данных изделий

* Вы открыли базу данных изделий. Сервисные программы > Изделие > Управление > [Дополнительно] > Настройки.
* Вы открыли проект и выделили главную функцию в графическом редакторе или навигаторе. Обработать > Свойства. Выберите вкладку Изделие и щелкните рядом с полем Источн. данных по ++...++.
* Параметры > Настройки > Пользователь > Управление > Изделия.

В диалоговом окне Настройки: Изделие (пользователь) щелкните по кнопке ++Другие++.

В этом диалоговом окне устанавливаются определенные предварительные настройки для работы с базой данных изделий.

Обзор основных элементов диалогового окна:

### Библиотека определений функций для выбора шаблона

В данном поле укажите каталог, в котором содержится библиотека определений функций для выбора шаблонов функций: Щелкните по ++...++ и выберите в диалоговом окне, например, каталог идентификатора клиента.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Сортировка списков / деревьев

Выберите из этого раскрывающегося списка, должна ли выполняться сортировка записей данных в представлении в виде списка и дерева в алфавитном или алфавитно-цифровом порядке.

### Импорт / экспорт единицы измерения

В базе данных изделий единицы измерения всегда сохраняются в "мм / кг". При импорте или экспорте данных EPLAN с помощью этих опций можно установить, какая система единиц измерения должна использоваться.

### Валюта 1 / валюта 2

В этих двух полях определите обе валютные единицы и коэффициент пересчета, которые необходимо использовать для выбранной базы данных изделий.

# Экспортировать данные изготовления для автоматических систем оснащения

Следующий формат экспорта из EPLAN Pro Panel — вывод данных изготовления для автоматических систем оснащения. Для этого предусмотрен интерфейс Rittal — Athex. Данная машина автоматически оснащает несущие шины клеммниками и последовательными принадлежностями клемм в соответствии с планированием в проекте EPLAN.

Экспортированные файлы содержат номера изделий, ОУ и длину оснащаемых несущих шин, а также позиционные данные расположенных на них клемм.

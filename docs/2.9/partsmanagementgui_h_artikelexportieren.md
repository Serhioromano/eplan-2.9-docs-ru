# Экспортировать данные изделий

При экспорте данных элемента различают между:

* экспортом части базы данных
* экспортом выбранных записей данных.

Условие:

Через меню ++Дополнительно++ > Настройки в диалоговом окне База данных изделий была открыта база данных изделий.

### Экспорт части базы данных

1. Сервисные программы > Изделие > Управление > [Дополнительно] > Экспортировать
2. Выберите требуемый тип файла из раскрывающегося списка Тип файла диалогового окна Экспортировать записи данных.
3. Щелкните по кнопке ++...++, чтобы для ***Текстового формата*** и ***Формата CSV*** указать путь к соответствующему файлу конфигурации; аналогичным образом можно для ***Формата обмена данными EPLAN Data Portal (EDZ)*** задать тип экспорта данных: Установите флажок Полностью экспортировать справочные данные, если в случае с узлами, модулями и т. д. при экспорте также должны учитываться подузлы, подмодули и т. д.

!!! note "Замечание:"

    При экспорте данных в файл типа "Текст" следует обязательно помнить о том, что файл конфигурации textimportexport.cfg, расположенный в каталоге \Изделие\<Ид. клиента> текущей версии программы, при необходимости должен быть скорректирован для учета определенных полей (таких, например, как у групп ***Схема соединений*** и ***Схема сверления***).
    Подробные сведения см. в теме [База данных изделий: Файлы конфигурации для импорта и экспорта данных](partsmanagementgui_k_steuerdateiimportexport.md).

4. Выберите, должен ли экспорт осуществляться в общий файл (параметр Общий файл) или в отдельный файл для каждой записи данных (параметр Отдельные файлы); затем, в зависимости от вашего выбора, введите имя файла в поле Имя файла для файла экспорта или укажите каталог экспорта.
5. Установите в групповом поле Тип записи данных флажки для тех данных, которые вы хотите экспортировать.
6. Установите в групповом поле Раздел флажки для тех разделов, которые должны быть экспортированы.
7. Измените объем данных Fluid-Техники, которые должны быть экспортированы, установив / сняв соответствующие флажки.
8. Щелкните по кнопке ++OK++.

### Экспорт выбранных записей данных

Условие:

В дереве или списке выделены экспортируемые записи данных.

1. Сервисные программы > Изделие > Управление > Всплывающее меню > Экспортировать
2. Продолжайте, как описано выше.

!!! note "Замечание:"

    Если текущее выделение содержит узлы (в представлении в виде дерева), будет выполнен экспорт находящихся под ними записей данных.

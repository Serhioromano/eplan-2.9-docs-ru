# Диалоговое окно Настройки: Область ввода данных

Параметры > Настройки > Пользователь > Графическая обработка > Область ввода данных.

В данном диалоговом окне задаются настройки для области ввода данных с клавиатуры.

Обзор основных элементов диалогового окна:

### Всегда отображать область ввода данных (2D) / Всегда отображать область ввода данных (3D)

Данный флажок следует установить, если в 2D или 3D-редакторе область ввода данных должна отображаться всегда.

!!! note "Замечание:"

    Включение и отключение области ввода данных можно выполнить как здесь в настройках, так и в графическом редакторе (через меню или через кнопку 'Вид' на панели инструментов). Данная настройка при этом соответствующим образом синхронизируется, т.е., если область ввода данных активировать в настройках, а затем отключить в редакторе, она будет отключена в настройках.

### Степень прозрачности (без фокуса) / Степень прозрачности (с фокусом)

Это свойство позволяет задать в процентах (%) степень прозрачности с фокусом или без него; максимальное значение для степени прозрачности — "85 %".

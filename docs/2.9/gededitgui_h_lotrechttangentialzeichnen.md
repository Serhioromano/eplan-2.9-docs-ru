# Начертить перпендикулярные или касательные линии

С помощью обоих режимов, Перпендикуляр и Касательная, можно начертить перпендикуляр или касательную к объекту. Например, для определения начальной точки линии или многоугольника можно использовать режим Перпендикуляр. Режим Касательная можно дополнительно установить при черчении, чтобы определить радиус кругов, дуг и секторов.

Условие:

Вы открыли страницу, форму, рамку или символ.

### Начертить вертикальные линии к объекту

1. Выберите пункты меню Вставить > Графика > Линия (или ... > Ломаная линия, или ... > Многоугольник).
2. Выберите пункт всплывающего меню Перпендикуляр.

!!! info "Для сведения:"

    Пиктограмма ![](images/gededitgui_vertically_as.png){: .ui-icon } у курсора отображает, что режим Перпендикуляр включен.

3. Щелкните соответствующий объект (линию, прямоугольник, круг и т. д.), к которому необходимо начертить вертикальную линию. До тех пор, пока не определена конечная точка, начальную точку можно изменять и линию из выбранного сегмента линии объекта можно передвигать по вертикали.
4. Щелкните еще раз, чтобы определить конечную точку линии (следующую вершину угла ломаной линии).

### Начертить касательные линии к объекту

1. Выберите пункты меню Вставить > Графика > Линия (или ... > Ломаная линия, или ... > Многоугольник).
2. Выберите пункт всплывающего меню Касательная.

!!! info "Для сведения:"

    Пиктограмма ![](images/gededitgui_tangential_as.png){: .ui-icon } у курсора отображает, что режим Касательная включен.

3. Щелкните соответствующий объект (линию, дугу, круг), к которому необходимо начертить касательную линию. До тех пор, пока не определена конечная точка, начальную точку можно изменять и линию из выбранного сегмента линии объекта можно передвигать.
4. Щелкните еще раз, чтобы определить конечную точку линии (следующую вершину угла ломаной линии).

### Определить радиус посредством точки касательной

1. Выберите пункты меню Вставить > Графика > Круг (или ... > Дуга через центр, или ... > Сектор).
2. Укажите центральную точку, щелкнув левой клавишей мыши.
3. Выберите пункт всплывающего меню Касательная.

!!! info "Для сведения:"

    Пиктограмма ![](images/gededitgui_tangential_as.png){: .ui-icon } у курсора отображает, что режим Касательная включен.

4. Затем передвиньте курсор на объект, к которому необходимо провести касательную.

!!! info "Для сведения:"

    Как только будет достигнута точка касательной, отобразится дуга окружности.

5. Щелкните еще раз, чтобы начертить круг/дугу/сектор.

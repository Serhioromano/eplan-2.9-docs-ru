# Передача данных конфигурации на EPLAN Fluid

Чтобы иметь возможность вывести документацию по установкам и/или машинам в форме спецификаций изделий и/или схем групп соединений/проводов, созданные в конфигураторе шлангопроводов данные конфигурации должны быть записаны обратно на размещенные в EPLAN Fluid шлангопроводы.

Предпосылкой для правильной передачи данных является то, что данные конфигурации в конфигураторе шлангопроводов и размещенные на схеме соединений в EPLAN Fluid шлангопроводы имеют ***одинаковое*** (полное) ОУ.

Если вы создали данные конфигурации с помощью конфигуратора шлангопроводов в качестве дополнительного модуля, данные передаются в текущий открытый в EPLAN Fluid проект простым щелчком мыши или нажатием клавиши, а там записываются и сохраняются как свойства на выделенных в схеме соединений шлангопроводах.

Если вы создали данные конфигурации с помощью конфигуратора шлангопроводов в качестве самостоятельной программы, сначала следует сохранить данные в файле обмена. Затем отправьте файл обмена получателю, напр. конструктору, который продолжит обработку данных шлангопроводов в EPLAN Fluid и у которого установлен для этой цели конфигуратор шлангопроводов в качестве дополнительного модуля к EPLAN Fluid. Получатель импортирует файл обмена и передает данные конфигурации, как описано выше, на находящиеся в открытом проекте шлангопроводы.

При передаче данных в EPLAN Fluid на выделенных в проекте шлангопроводах на вкладке Группа соединений / Провод отображаются следующие свойства устройств, которые заполняются следующими значениями конфигурации:

* Тип (здесь задается соответствующий тип шланга);
* Поперечное сечение / диаметр (здесь вводится соответствующий номинальный диаметр);
* Длина шлангопровода
* Обозначение шлангопровода
* Дата производства
* Угол кручения
* Рабочее давление.

Кроме того, в свойствах изделия шлангопроводов указывается следующая информация:

* Тип конструкции арматуры, слева (в положении изделия 1);
* Тип конструкции арматуры, справа (в положении изделия 2);
* Тип шланга (в положении изделия 3);
* Код типа (в положении изделия 4).

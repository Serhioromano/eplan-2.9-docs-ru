# Символы: Порядок действий

Использование символов при создании и обработке схемы соединений состоит из следующих этапов:

* Сохранить библиотеку символов в проекте: если вы хотите использовать символы из библиотеки символов в проекте, необходимо сохранить их в проекте. Таким образом соответствующая библиотека символов связывается в этом проекте с основными данными: Параметры > Настройки > Проекты > "Имя проекта" > Управление > Библиотеки символов.
* Вставить символы: Вставить > Символ открывает диалоговое окно Выбор символа, в котором вы можете выбрать символ, используя различные возможности поиска.
* Разместить символы: после выбора необходимого символа в диалоговом окне Выбор символа вы размещаете символ на странице в графическом редакторе. Связь нового размещенного символа с другими символами в схеме соединений создается автоматически при помощи автоматического соединения.
* Задать свойства условного обозначения: после размещения символа на странице в диалоговом окне Свойства ++...++ вы определяете специфические для условного обозначения свойства (к ОУ, функциональному тексту и т. д.).

# Использование справочной системы

Справка > Содержание

Если вы открыли диалоговое окно, можно, кроме того, активировать контекстную справку ([F1]).

Тему справки можно выбрать непосредственно через содержание. Но в справочной системе в вашем распоряжении имеются и различные возможности поиска. Так, например, можно ориентироваться в справке через указатель или полнотекстовый поиск. Для определения понятий предусмотрен глоссарий.

Синтаксис, используемый для поиска, ориентируется на синтаксис широко используемых поисковых систем в интернете.

!!! note "Замечание:"

    Текстовый поиск обеспечивает поиск по синонимам: Например, при вводе схожих, более не используемых или неправильных искомых понятий в большинстве случаев будет отображаться подходящий результат поиска.

Поиск текста в справке можно ограничить специфическими категориями с помощью фильтра поиска. Возможные категории: варианты продукта EPLAN (напр., "EPLAN Electric P8") и типы страниц справочной системы (напр., "Указания по действиям", "Сообщения контрольного прогона"). Фильтр поиска можно настроить перед началом поиска или изменить в результате поиска.

1. Щелкните кнопку ***[Поиск в]*** справа от поля ввода для поиска текста.
2. Из раскрывающегося списка выберите категорию.
3. Введите искомое понятие и запустите поиск с помощью кнопки ![](images/helpsystem_startsearch_as.png){: .ui-icon }.

!!! info "Для сведения:"

    В результате поиска будут отображаться только записи из выбранной категории.

4. Из раскрывающегося списка выберите другую категорию.

!!! info "Для сведения:"

    В результате поиска будут отображаться только записи из новой категории.

Искомые понятия будут автоматически дополняться при вводе в поле для поиска текста. Предлагаемые понятия содержатся в индексе справочной системы.

Кроме того, автозаполнение можно комбинировать с фильтрами поиска.

В справочной системе можно переключаться между всеми доступными языками.

1. На панели инструментов справки щелкните кнопку, на которой отображается текущий язык.
2. Из раскрывающегося списка выберите язык, на который нужно переключиться.

!!! info "Для сведения:"

    Текущая страница справочной системы отобразится на выбранном языке.

Нажмите кнопку ![](images/CopyLink.png){: .ui-icon } (Копировать URL в буфер обмена) на панели инструментов справочной системы, чтобы скопировать URL открытой страницы справки в буфер обмена, а затем, при необходимости, вставить адрес в другие приложения. Это может пригодиться, например, при отправке запросов в службу поддержки.

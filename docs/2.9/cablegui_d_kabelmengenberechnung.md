# Диалоговое окно Суммировать длину кабелей

В диалоговом окне навигатора выделен проект Страницы — <Имя проекта>. В проекте внесена длина кабелей. Данные проекта > Кабели > Суммировать длину кабелей.

В этом диалоговом окне рассчитывается число требуемых кабельных барабанов. Аналогичные типы кабеля обобщаются в изделия проекта кабеля, а длина отдельных кабелей суммируется.

Обзор основных элементов диалогового окна:

### Длина в

В этом поле отображается единица измерения, в которой выдается длина кабеля. Единица измерения устанавливается в диалоговом окне Настройки: Кабель в поле Единица измерения длины кабеля.

* * *

Таблица

В таблице отображается следующая информация:

### Полный тип кабеля

Здесь можно просмотреть полный тип кабеля. Несколько кабелей имеют одинаковый полный тип кабеля, если следующие свойства кабелей идентичны:

* Кабель / Группа соединений: Тип (напр., NYM)
* Кабель / Группа соединений: Количество соединений (напр., 3)
* Кабель / Группа соединений: Поперечное сечение/диаметр (напр., 1,5)
* Кабель / Группа соединений: Единица поперечного сечения / диаметра (напр., мм² или см).

### Рассчитанная длина

Здесь выводится сумма длины отдельных кабелей. Длина задается в стандартной единице для длины кабеля.

### Сам. большая отд. длина

Здесь отображается длина макс. отдельного кабеля.

### Номер изделия

Здесь отображаются номера изделий, присвоенные макс. отдельному кабелю. Посредством всплывающего меню можно запустить выбор изделия для текущего типа кабеля и выбрать другое изделия (даже более короткое, чем максимальная отдельная длина). После выбора автоматически рассчитывается требуемое количество, а строка обновляется.

### Вариант изделия

Здесь выводится вариант изделия. Посредством всплывающего меню можно запустить выбор изделия для текущего типа кабеля и выбрать другого изделия / другой вариант.

### Длина изделия

Здесь отображается длина изделия кабеля, которой присвоен макс. отдельный кабель.

### Требуемое количество

Требуемое количество выводится из рассчитанной длины, разделенной на длину изделия кабеля (в поле Длина изделия). Отображаемое значение можно изменить вручную.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Выбор изделия |  Открывает диалоговое окно выбора изделия.
Удалить |  Удаляет выделенные строки. Соответствующие изделия кабеля удаляются.

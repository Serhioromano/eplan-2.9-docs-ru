# Генерация перечня отчетов без шаблона

В EPLAN есть типы отчетов для [перечней отчетов](formgeneratorgui_k_auswertungsuebersicht.md) и для отчетов, относящихся к функции. Перечни отчетов по умолчанию предоставляют обзор нескольких устройств, как, например, "Перечень клеммников". То, как сортируются страницы отчетов в имеющуюся структуру страниц, зависит от того, какой тип [Сортировка страниц отчетов](formgeneratorgui_k_seitensortierung.md) установлен в настройках отчета для типа отчета.

Условия:

* Вы открыли проект.
* Требуемому типу отчета присвоена форма.
* В диалоговом окне Настройки: Вывод в страницах заданы все необходимые настройки.

1. Сервисные программы > Отчеты > Генерировать
2. Щелкните в диалоговом окне Отчеты — <Имя проекта> по ++Настройки.++ > Вывод в страницах.
3. Выберите тип отчетов для перечней отчетов.
4. В столбце Сортировка страниц щелкните кнопку ++...++ и в выведенном диалоговом окне задайте сортировку страниц.

!!! info "Для сведения:"

    Чтобы сгенерировать общий отчет для всего проекта, выберите запись Общее из раскрывающегося списка Группировать по / в.

!!! info "Для сведения:"

    Чтобы сгенерировать отчеты, относящиеся к структурному идентификатору, например, если меняется структурный идентификатор "Установка" или "Место установки", установите флажки Установка или Место установки.

5. Щелкните по кнопке ++OK++.
6. В диалоговом окне Настройки: Вывод в страницах щелкните ++OK++.
7. Выберите в диалоговом окне Отчеты — <Имя проекта> вкладку Отчеты и щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
8. Выберите из раскрывающегося списка Форма вывода диалогового окна Определить отчет форму вывода"Страница".
9. Проверьте, внесен ли в поле Исходный проект проект, являющийся источником данных для отчетов и шаблонов.
10. Для выбора другого проекта нажмите кнопку ++...++, выберите в диалоговом окне Исходный проект требуемый проект и нажмите кнопку ++OK++.
11. В поле Выбрать тип отчета выделите обзор отчетов.

!!! info "Для сведения:"

    Если обзор отчетов выделен, флажок Индивидуальный выбор недоступен.

12. Щелкните по кнопке ++OK++.
13. Выберите в диалоговом окне Настройки — <Тип отчета> дополнительные [схемы](schemeconfig_k_start.md) для фильтрации, сортировки и разбивки на страницы в отчетах и нажмите кнопку ++OK++.
14. Введите в диалоговом окне <Тип отчета> имя начальной страницы в первом структурном идентификаторе.
15. При необходимости определите описание страницы и номер листа и щелкните по ++OK++.
16. Таким же образом определите свойства начальной страницы для всех других структурных идентификаторов. Доступность определенного структурного идентификатора зависит от сортировки страниц, которая присвоена типу отчета в диалоговом окне Настройки: Вывод в страницах.
17. Чтобы скопировать свойства начальной страницы для всех структурных идентификаторов, установите флажок Применить начало для всех структурных идентификаторов.
18. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Создаются страницы отчета.

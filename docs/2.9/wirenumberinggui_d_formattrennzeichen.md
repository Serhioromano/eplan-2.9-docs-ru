# Диалоговое окно Формат: Разделитель (нумерация соединений)

Вы открыли проект. Вы открыли диалоговое окно [Настройки: Нумерация соединений](wirenumberinggui_d_verbnumeinstellungen.md) (например, через пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Нумерация соединений). В диалоговом окне Настройки: Нумерация соединений вы выбрали определенную пользователем схему и перешли во вкладку Обозначение. Вы выделили запись в поле Группы форматов и щелкнули по ![](images/all_edit_as.png){: .ui-icon } (Обработать) или по ![](images/all_new_as.png){: .ui-icon } (Создать).

В диалоговом окне Нумерация соединений: Форматы в списке Доступные элементы формата выберите элемент формата типа "Разделитель" и перенесите его при помощи кнопки со стрелкой в список Выбранные элементы формата.

В этом диалоговом окне указывается форматирование разделителя для группы форматов.

Обзор основных элементов диалогового окна:

В качестве разделителя можно использовать следующие символы:

* Пробел
* Новая строка
* Другие символы: После выбора этой опции введите в поле ввода требуемую комбинацию символов. Разрешается использовать любые цифры, буквы и специальные символы за исключением символов #,<, >. Однако эти символы следует вводить после косой черты (\\).

### Число символов

Если в качестве разделителя вы выбрали пробел или разрыв строки, здесь можно определить число символов. Таким образом, разделитель может состоять из нескольких пробелов или разрывов строк. В случае выбора опции Другие символы это поле становится недоступным и в нем выводится число введенных символов.

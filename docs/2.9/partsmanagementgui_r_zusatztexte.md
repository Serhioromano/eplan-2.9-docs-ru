# Вкладка Произвольн. свойства

Сервисные программы > Изделие > Управление. В дереве или списке выделено одно изделие.

На этой вкладке может указываться дополнительная текстовая информация, которая не зависит от выбранного типа изделия (отдельная деталь, кабели и т. д.). Эти свойства идентичны затем для всех вариантов изделия.

!!! note "Замечание:"

    В базе данных изделий для ввода и отображения дополнительной информации мы рекомендуем использовать ***определенные пользователем свойства***. При конфигурации они предоставляют больше возможностей по настройке, чем произвольные свойства, и их характеристики аналогичны свойствам, определенным пользователем для проекта. Произвольные свойства доступны только для совместимости.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать. Или щелкните по ++...++, чтобы в диалоговом окне Конфигурировать произвольные свойства обработать схему или создать новую.

### [Присвоить схему]

Щелкните по кнопке, чтобы присвоить выбранную схему в раскрывающемся списке.

!!! note "Замечание:"

    Учтите, что перезаписываются только заполненные ячейки с данными из схемы, пустое содержимое игнорируется.

### Таблица

В ячейках "Описание", "Значение" и "Единица измерения" введите дополнительную текстовую информацию, которую вы хотите использовать для отдельных изделий, независимо от типа изделия.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

!!! note "Замечание:"

    Любое поле каждого произвольного свойства может использоваться для фильтрации с помощью фильтра по полю.

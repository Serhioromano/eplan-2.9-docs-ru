# Вставить определения проводного монтажа

Определения проводного монтажа можно задать непосредственно при вставке соответствующего символа на схему соединений. В навигаторе проводного монтажа или в навигаторе устройств вы можете предварительно задавать определения проводного монтажа с определенными свойствами. Созданные таким образом функции вы можете в дальнейшем разместить в схеме соединений. Определения проводного монтажа должны быть размещены, если их требуется скопировать.

1. Выберите следующие пункты меню: Вставить > Символ.
2. В диалоговом окне Выбор символа выберите вкладку Список.
3. В поле Прямой ввод введите символьную строку "HDEF".
4. Вставьте определение проводного монтажа на странице схемы соединений.
5. В диалоговом окне Свойства ++...++ на вкладке Проводной монтаж введите данные для определения проводного монтажа.
6. Щелкните по кнопке ++OK++.

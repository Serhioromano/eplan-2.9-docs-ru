# Диалоговое окно Сборка проводов: <машина>

* Параметры > Настройки > Фирма > Сборка проводов > "Машина"
* Сервисные программы > Данные изготовления > Сборка проводов > "Машина"

В этом диалоговом окне устанавливаются настройки для экспорта данных сборки проводов.

Обзор основных элементов диалогового окна:

### Схема

В объеме поставок содержится определение машины с именем производителя. Если в качестве схемы сгенерирована другая машина, можно выбрать из раскрывающегося списка другие определения машины и работать с ними. С помощью кнопки ++...++ откройте диалоговое окно Настройки: Сборка проводов, в котором вы можете изменить имеющуюся схему или создать новую.

### Язык

Если пользователь не ввел текст для направления подсоединения, выводится внутренний текст на языке диалогового окна. Если установлено несколько языков диалогового окна, существует возможность выбрать язык из раскрывающегося списка.

### Путь файла

Выберите каталог, в котором следует разместить экспортируемые файлы.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Имя файла

Затем можно присвоить имя сгенерированному файлу или, нажав ++...++, выбрать его в диалоговом окне Сохранить как. Расширение имени файла зависит от выбранного формата машины. Если в результате экспорта генерируются несколько файлов с различными расширениями имени файла, введенное здесь имя будет применяться ко всем файлам.

Некоторые имена файлов и / или расширения имен файлов устанавливаются в зависимости от производителя. В этом случае настроить можно только путь файла.

### Вновь экспортировать готовые разводки

При помощи этого флажка можно выбрать, следует ли выводить только новые соединения или также уже готовые. Удаленные соединения никогда не экспортируются через это диалоговое окно.

### Применить ко всему проекту

Если установлен этот флажок, экспортироваться будут все соединения в проекте, при этом учитывается настройка флажка Вновь экспортировать готовые разводки.

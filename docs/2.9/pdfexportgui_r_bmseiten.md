# Вкладка Страницы устройства

* Страница > Экспортировать > PDF. Щелкните в поле Схема диалогового окна Экспорт PDF[...]. Выберите вкладку Страницы устройства.
* Параметры > Настройки > Пользователь > Интерфейсы > Экспорт PDF. Выберите вкладку Страницы устройства.

На этой вкладке указывается, должны ли отображаться в PDF страницы с данными устройств. Для этих "страниц устройства" вы можете указать, на какие области подразделяются данные устройства.

Обзор основных элементов диалогового окна:

### Генерировать страницы устройства

Установите этот флажок, если для устройства необходимо генерировать страницы в PDF с данными устройства. Затем для каждого устройства в PDF под заголовком — для полного ОУ соответствующего устройства — перечисляется информация из диалогового окна "Свойства", а также информация к ссылкам (видам представлений, отчетам и т.д.).

Если этот флажок не установлен, то устройство на дереве устройств в PDF только перечисляется, но страницы с данными устройства не генерируются.

### Фильтр устройств

В этой таблице определяются критерии фильтра для свойств объекта. Отдельные строки стандартно связаны логическим оператором И. Так вы можете, например, ограничить экспорт PDF по определенному устройству (например, кабелю) или сократить экспорт PDF, исключив устройство из экспорта.

### Области

Выдаваемая информация для устройства по умолчанию подразделяется на три области Свойства устройства, Свойства изделия и Ссылки.

Если определенная область не должна появляться в PDF, то снимите соответствующий флажок в таблице Области. Над ![](images/all_arrowup_as.png){: .ui-icon } (Переместить вверх) / ![](images/all_arrowdown_as.png){: .ui-icon } (Переместить вниз) можно изменить последовательность областей в PDF.

### Выводятся ссылки на Представления функции/ Отчеты

Если этот флажок не установлен, то для устройства в области Ссылки не перечисленно ни одной ссылки для представления вида функции или/и ни одной ссылки для отчетов страниц.

# Сгенерировать схему кабельных соединений

[Схемы кабельных соединений](cablegui_k_kabeldarstellungsplan.md) генерируются в сочетании со следующей формой для групповой спецификации изделий. [Формы для схем кабельных соединений](formeditorgui_k_kdpformular.md) позволяют вывести данные для представления кабеля и данные для групповой спецификации изделий. Для каждого типа кабеля генерируется схема кабельных соединений. Таким образом, схема кабельных соединений какого-либо типа кабеля содержит все кабели этого типа. Каждому кабелю, для которого требуется выполнить анализ, должна быть присвоена одна форма.

Условие:

У вас имеется доступ к форме схемы кабельных соединений *.f08 и форме групповой спецификации изделий *.f02.

### Размещение точки следующей формы

Чтобы для формы можно было задать [Следующую форму](formeditorgui_k_folgeformulare.md), на этой форме должна быть размещена точка следующей формы.

1. Выберите пункты меню Сервисные программы > Основные данные > Форма > Открыть.
2. Выберите в поле Тип файла запись Схема кабельных соединений (*.f08), выделите форму и нажмите кнопку ++Открыть++.
3. Выберите пункты меню Вставить > Точка вставки следующей формы.
4. В диалоговом окне Точка вставки следующей формы в поле Тип отчета выберите запись "Групповая спецификация изделий".
5. Чтобы вывести групповую спецификацию изделий на новой странице отчета, установите флажок Новая страница.
6. Чтобы групповую спецификацию изделий вывести на странице отчета со схемой кабельных соединений, снимите флажок Новая страница.
7. Щелкните по кнопке ++OK++.
8. Разместите точку следующей формы на странице формы.
9. Закройте форму.

### Определение следующей формы

1. Выберите пункты меню Сервисные программы > Отчеты > Генерировать.
2. В диалоговом окне Отчеты выберите ++Настройки++ > Вывод в страницах.
3. В строке "Схема кабельных соединений" щелкните в столбце Следующая форма, а затем нажмите кнопку "Поиск".
4. В следующем диалоговом окне выберите тип данных Групповая спецификация изделий (*.f02) и выделите форму.
5. Подтвердите ввод.

### Присвоение формы всем кабелям одного типа

1. Выберите пункты меню Данные проекта > Кабель > Навигатор.
2. Выберите на вкладке Список в поле Фильтр запись "Главная функция".
3. Выберите пункт всплывающего меню Конфигурировать представление, включите свойство Кабель / Группа соединений: Тип и щелкните по кнопке ++OK++.
4. На вкладке Список выделите все определения кабелей одного типа кабеля.
5. Выберите пункт всплывающего меню Свойства.
6. Перейдите на вкладку Кабели.
7. В групповом поле Свойства щелкните ![](images/all_new_as.png){: .ui-icon } (Создать).
8. Отметьте свойство Форма схемы кабельных соединений и щелкните ++ОК++.
9. В строке свойства Форма схемы кабельных соединений щелкните в столбце Значение, а затем нажмите кнопку "Поиск".
10. В следующем диалоговом окне выберите форму для схемы кабельных соединений.
11. Подтвердите ввод.
12. Таким же образом присвойте формы другим типам кабелей.

### Генерация отчетов

1. Выберите пункты меню Сервисные программы > Отчеты > Генерировать.
2. Выберите вкладку Отчеты. Щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
3. В следующем диалоговом окне выделите в поле Выбрать тип отчета запись "Схема кабельных соединений". Щелкните по кнопке ++OK++.
4. Щелкните в диалоговом окне Настройки — <Тип отчета> по кнопке ++OK++.
5. Чтобы задать начальную страницу, выберите в диалоговом окне Схема кабельных соединений структурный идентификатор структуры страниц или введите новый структурный идентификатор.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Все кабели, для которых записана одинаковая форма схемы кабельных соединений, выдаются вместе в ***одном*** блоке отчета. Если вы используете для каждого типа кабеля отдельную форму схемы кабельных соединений, генерируется один блок отчета для каждого типа кабеля. В блоке отчетов содержатся страницы отчета для схемы кабельных соединений и групповая спецификация изделий. Возможная сортировка учитывается только внутри блока отчета.

# Диалоговое окно Настройки: Экспорт Excel

* Вы открыли проект. Сервисные программы > Внешняя обработка свойств > Экспортировать данные > Настройки ([...]). Выберите вкладку Общ.. Выберите из раскрывающегося списка Тип файла запись "Файл Excel (*.xls, *.xlsx, *.xlsm, *.xlsb)" и щелкните по кнопке ++Параметры++.
* Параметры > Настройки > Пользователь > Интерфейсы > Внешняя обработка ( ++...++). Выберите вкладку Общ.. Выберите из раскрывающегося списка Тип файла запись "Файл Excel (*.xls, *.xlsx, *.xlsm, *.xlsb)" и щелкните по кнопке ++Параметры++.
* Вы открыли проект. Сервисные программы > Данные изготовления > Экспорт / маркировка. В диалоговом окне Экспортировать данные изготовления / Вывести маркировку рядом с полем Настройки щелкните по кнопке ++...++. В диалоговом окне Настройки: Экспорт данных изготовления / маркировка выберите вкладку Файл. Выберите из раскрывающегося списка Тип файла запись "Файл Excel (*.xls, *.xlsx, *.xlsm, *.xlsb)" и щелкните по кнопке ++Параметры++.
* Параметры > Настройки > Пользователь > Интерфейсы > Экспорт данных изготовления / маркировка. В диалоговом окне Настройки: Экспорт данных изготовления / маркировка выберите вкладку Файл. Выберите из раскрывающегося списка Тип файла запись "Файл Excel (*.xls, *.xlsx, *.xlsm, *.xlsb)" и щелкните по кнопке ++Параметры++.

В этом диалоговом окне указываются настройки для внешней обработки страниц, функций или соединений или для вывода маркировки в программе Excel.

### Также вывести обозначение столбца

В этом раскрывающемся списке определите, какие элементы формата должны выводиться при экспорте как заголовки столбцов. Предусмотрен выбор следующих записей:

* Нет: Элементы формата не выводятся.
* Только имя свойства: Выводится только имя свойства, напр. "ОУ (полное)".
* Полный: Перед именем свойства указывается относящееся к нему определение, например "Кабель / ОУ (полное)".

# Новый интерфейс для экспорта данных изготовления в "Rittal — Wire Terminal WT"

Для экспорта данных изготовления теперь имеется интерфейс для полностью автоматической машины по сборке проводов ***Rittal — Wire Terminal WT***. Для этого под путем меню Сервисные программы > Данные изготовления > Сборка проводов доступен новый пункт меню Rittal — Wire Terminal WT. После выбора пункта меню открывается диалоговое окно экспорта, в котором можно выполнить настройки для экспорта данных сборки проводов и сохранить их в виде схемы.

![](images/benefit.png)Эффект:

С помощью нового интерфейса теперь также можно экспортировать данные изготовления для полностью автоматической машины по сборке проводов "Rittal — Wire Terminal WT".

!!! note "Замечание:"

    Лицензия на этот новый интерфейс предоставляется через дополнительный модуль расширения, за который взимается плата. Информацию об этом можно найти в текущей спецификации договорных работ и услуг для EPLAN Pro Panel.

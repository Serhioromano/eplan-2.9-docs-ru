# Замена символов

При изменении стандарта символы одного стандарта заменяются подходящими символами другого стандарта. Символы можно заменять только такими символами, структура и размер которых соответствуют исходным. Поэтому определения функции символов должны совпадать. Целевой символ может иметь ***столько же или больше*** выводов устройств, как исходный символ. Направление выводов устройств должно соответствовать повороту проекта.

!!! note "Замечание:"

    * символы с номерами символов > 3000 в проектах EPLAN больше не используются. Поэтому изменение стандарта для этих символов не предусмотрено в принципе. Тем не менее, такие символы можно адаптировать или заменить вручную.
    * Кроме того, обратите внимание, что символы библиотеки символов IEC_single_Symbol можно заменить только символами библиотеки символов NFPA_single_Symbol. Поэтому после изменения стандарта требуется ручная корректировка замещенных символов.

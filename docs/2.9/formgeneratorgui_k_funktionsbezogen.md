# Отчеты, относящиеся к функции

У отчетов, относящихся к функции, в отличие от [перечней отчетов](formgeneratorgui_k_auswertungsuebersicht.md), по умолчанию в верхнем колонтитуле содержится устройство, например, клеммник. В области данных находятся соответствующие ***функции***, как, например, клеммы клеммника.

Отчеты, относящиеся к функции, в проекте могут встречаться только один раз.

!!! example "Пример:"

    Спецификация клеммника Х1в проекте может встречаться только один раз.

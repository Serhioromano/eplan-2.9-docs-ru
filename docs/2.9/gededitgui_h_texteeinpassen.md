# Подогнать тексты в блоке выравнивания

Тексты или тексты-заполнители можно подогнать в блоке выравнивания и таким образом согласовать с определенным местом, имеющимся на странице или в форме. Черчение блока выравнивания не требуется. Для масштабирования текста таким образом, чтобы он вмещался в блок выравнивания без обрезки, можно воспользоваться свойством отображения Разрешен подгон текста. Этот флажок ***позволяет выполнять*** масштабирование текстов. Сам подгон выполняется в ходе последующего действия.

### Подготовка текстов для подгонки

Условие:

Вы открыли страницу или форму.

1. Дважды щелкните мышкой по тексту / тексту-заполнителю.
2. В диалоговом окне Свойства переместите на передний план вкладку [Формат](gededitgui_r_format.md).
3. Откройте уровень иерархии Блок выравнивания и поставьте флажок для свойств Активировать блок выравнивания, Ширина текста фиксирована, Высота текста фиксирована и Разрешен подгон текста.
4. Щелкните по кнопке ++OK++.

!!! tip "Совет:"

    Эти четыре настройки, как и все другие настройки формата и отображения, можно применить к другим текстам / текстам заполнителям посредством действий Копировать формат и Присвоить формат.

### Автоматический подгон текстов

При генерировании ***отчетов*** выведенные тексты-заполнители подгоняются автоматически.

### Ручной подгон текстов

Условие:

Вы открыли страницу, форму или навигатор страниц.

1. Выделите один или несколько текстов / текстов-заполнителей. Или выделите выбранные страницы или проект в навигаторе страниц.
2. Выберите пункты меню Обработать > Текст > Подогнать тексты в блоке выравнивания.

!!! info "Для сведения:"

    Размер шрифта в соответствующих текстах подгоняется таким образом, чтобы текст вписался в блок выравнивания. Рассчитанный размер шрифта автоматически заносится в диалоговом окне "Свойства" в настройку Размер шрифта.

!!! info "Для сведения:"

    Если в навигаторе страниц выделены страницы отчета, выведенные тексты с соответствующей настройкой подгоняются вручную.

!!! info "Для сведения:"

    Если вы выделили в графическом редакторе или навигаторе страниц невидимые тексты, подгон будет выполнен только для таких текстов.

!!! note "Замечание:"

    * При подгоне текста размер шрифта только уменьшается. При последующем увеличении блока выравнивания размер шрифта текста при подгоне в блоке выравнивания увеличиваться не будет.
    * Если пункт меню Подогнать тексты в блоке выравнивания выполнен для условного обозначения в схеме соединений, масштабирование будет проведено только для текстов свойства, для которых соблюдены требуемые условия для подгона (активирован блок выравнивания и т. д.).

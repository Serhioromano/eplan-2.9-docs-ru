# Добавить покупные изделия

Изделиями проекта являются изделия, которые ***не*** присвоены специальному устройству, как, напр., комплектующие детали.

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Данные проекта > Устройство / изделие > Навигатор спецификаций
2. В диалоговом окне Спецификация — <Имя проекта> выберите пункт всплывающего меню Добавить покупные изделия.
3. В диалоговом окне Выбор изделия выберите добавляемое изделие(я). Возможен многократный выбор.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Новые покупные изделия вводятся в дереве с сортировкой по алфавиту.

!!! note "Замечание:"

    Могут быть использованы все функции диалогового окна Выбор изделия, при этом учитываются предварительные настройки, специфические для базы данных изделий.

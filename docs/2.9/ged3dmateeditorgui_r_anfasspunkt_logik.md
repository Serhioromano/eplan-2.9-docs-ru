# Вкладка Логика (точка захвата)

Вы открыли проект макросов. Проект содержит как минимум одно пространство листа. Пространство листа содержит трехмерный объект. На трехмерном объекте определена точка захвата. Выделите точку захвата и выберите пункты меню Всплывающее меню > Свойства > Логика.

На этой вкладке задайте значения для размещения с этой точкой захвата. Здесь следует указать, к каким точкам монтажа и исходным точкам подходит данная точка захвата.

Обзор основных элементов диалогового окна:

### Разрешить все точки монтажа

Все определенные в проекте точки монтажа будут отображаться и использоваться при размещении.

Если установлен флажок (Стандарт), то перечисляются все точки монтажа в поле Точки монтажа, имеющиеся в проекте. Точки монтажа, встречающиеся в проекте несколько раз, приводятся в списке только один раз.

Если флажок снят, клавиши со стрелками между обоими нижними полями становятся активными, при помощи них можно перемещать объекты из одного списка в другой.

### Точки монтажа, имеющиеся в проекте

В этом поле перечисляются определенные в проекте точки монтажа. При помощи клавиш со стрелками между обоими полями можно переместить одну или несколько точек монтажа в список в поле Присвоенные точки монтажа.

### Присвоенные точки монтажа

В этом поле перечисляются все присвоенные точке захвата точки монтажа. Эти точки монтажа при размещении отображаются и используются вместе с этой точкой захвата. При помощи клавиш со стрелками между обоими полями можно переместить одну или несколько точек монтажа в список в поле Точки монтажа, имеющиеся в проекте.

При помощи кнопки ![](images/all_new_as.png){: .ui-icon } (Создать) можно также генерировать записи в поле Присвоенные точки монтажа. Таким образом можно присваивать точки монтажа, которых еще нет в проекте. Если повторно установить флажок Разрешить все точки монтажа, все записи в поле Присвоенные точки монтажа переместятся обратно в список в поле Точки монтажа, имеющиеся в проекте. Вновь сгенерированные точки монтажа будут удалены.

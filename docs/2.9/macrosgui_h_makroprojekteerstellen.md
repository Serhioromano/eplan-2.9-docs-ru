# Создать проекты макросов

С помощью проектов макросов можно управлять макросами и автоматически генерировать макросы.

Условие:

Вы создали базовый проект *.zw9 или шаблон проекта *.ept.

1. Выберите пункты меню Проект > Создать.
2. В диалоговом окне Создать проект создайте новый проект, как описано в разделе [Создать проект](projects_h_projekteerstellen.md).

!!! info "Для сведения:"

    EPLAN открывает диалоговое окно Свойства проекта: <Имя проекта>.

3. Выберите на вкладке Свойства категорию "Настройки".

!!! info "Для сведения:"

    Выводится только свойство Вид проекта.

4. Щелкните в столбце Значение по соответствующему полю и выберите из раскрывающегося списка настройку "Проект макросов".
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Создается новый проект макросов, который можно обрабатывать как "стандартный" проект.

!!! tip "Совет:"

    Проекты макросов можно создать только с помощью создания нового проекта. Также сначала можно создать проект схемы соединений, подготовить соответствующие фрагменты схемы соединений для генерирования макросов, а затем получить проект макросов, изменив свойства проекта. Для этого нужно всего лишь изменить свойство проекта Вид проекта на "Проект макросов", как это описано в шаге 3.

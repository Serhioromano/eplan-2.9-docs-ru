# Операция: synchronize

**Параметр** |  **Описание**
---|---
TYPE  |  Вид выполняемой задачи:
MULTILINE: синхр. 'многополюсных' данных.
SINGLELINE: синхр. 'однополюсных' данных.
OVERVIEW: синхр. данных 'обзора'.
SYSTEMPARTSTOPROJECT: добавление в проект изделий системы. PARTSTOSYSTEM: добавление к основным данным системы изделий из проекта.

PROJECTNAME  |  Имя проекта с полным путем файла (является опцией).
Если не задано, то выбранный проект используется, когда операция вызывается через интерфейс пользователя (например, через сценарий или панель инструментов). При вызове из командной строки Windows следует определить PROJECTNAME или сначала следует использовать ProjectAction. В противном случае отобразится системное сообщение.
STOREMODE  |  Указывает, следует перезаписать имеющиеся изделия или только добавить новые (не обязательно):

* 0: только добавление новых изделий (значение по умолчанию)
* 1: только перезапись имеющихся изделий
* 2: перезапись имеющихся изделий и добавление новых

Этот параметр оказывает влияние, только когда параметр TYPE имеет следующее значение: SYSTEMPARTSTOPROJECT.

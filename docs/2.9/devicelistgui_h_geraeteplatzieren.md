# Разместить функции устройств

Из навигаторов, отображающих функции, можно размещать функции / устройства в графическом редакторе. При этом в зависимости от типа открытой страницы вид представления функции определяется автоматически. Но при необходимости можно задать другой вид представления во время размещения. При размещении всегда используется первый подходящий символ из библиотеки символов или тот символ, который был присвоен функции.

Ниже описывается размещение функций / шаблонов функций на многополюсной схеме соединений.

Условия:

* Открыт навигатор, отображающий функции, например Данные проекта > Кабели > Навигатор или Данные проекта > Клеммники > Навигатор.
* В Графическом редакторе открыта страница проекта.

1. В навигаторе выделите одну или несколько функций / шаблонов функций, не размещенных на многополюсной схеме соединений. Или просто выделите узел устройств в виде дерева.

!!! info "Для сведения:"

    Система будет пытаться разместить сначала те функции / шаблоны функций этого устройства, которые еще не были размещены на многополюсной схеме соединений.

2. Выберите пункт всплывающего меню Разместить.

!!! info "Для сведения:"

    Присвоенный изделию макрос или отдельная функция появится рядом с курсором.

3. Чтобы использовать другой вид представления, макрос или символ, непосредственно перед размещением нажмите клавишу ++Backspace++.

!!! info "Для сведения:"

    Откроется диалоговое окно [Разместить устройство](egedgui_d_platzieren.md).

4. В этом диалоговом окне задайте параметры вставки. Если, например, на изделии определен макрос, необходимо выбрать параметр Отдельные функции.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В зависимости от выбранного параметра при определенных обстоятельствах откроется следующее диалоговое окно. Например, при выборе опции Выбор макросов откроется диалоговое окно Выбрать макросы.

!!! info "Для сведения:"

    Присвоенный или выбранный макрос / символ прикреплен к курсору.

6. Разместите макрос / символ на схеме соединений.
7. Разместите остальные функции выбора или прервите процесс, используя пункты меню Обработать > Прервать операцию.

!!! tip "Совет:"

    * При размещении нажмите клавишу ++N++, чтобы "перелистать" функции и выбрать нужную. Или используйте во время размещения вспомогательной функции клавишу ++Пробел++, чтобы прервать размещение вспомогательной функции и заново начать размещение главной функции.
    При выполнении макроса "пролистывание" по видам представления возможно при помощи клавиш [Shift] \+ [Tab]. Клавиша [Tab] позволяет "пролистать" варианты макроса определенного вида представления.
    * Устройства также можно вставлять, перетаскивая выделенное устройство мышью из навигаторов на открытую страницу графического редактора. Порядок действий аналогичен вышеописанному.

### Удалить размещение

Условие:

* Вы находитесь в проекте схемы соединений.
* Вы открыли страницу в графическом редакторе или навигатор данных проекта (например, навигатор устройств).

1. Выделите условное обозначение в графическом редакторе или размещенное устройство в навигаторе и воспользуйтесь пунктами меню Обработать > Удалить размещение.

!!! info "Для сведения:"

    Графическое представление функции в схеме соединений удаляется. Функция остается как неразмещенная функция с первоначальным видом представления и может быть снова размещена, например, из навигатора устройств.

!!! note "Замечание:"

    Для функций со следующими видами представлений размещение ***не ***может быть удалено:

* Внешн.
* Графика
* Компоновка электрошкафа, компоновка электрошкафа подробно (виды представления размещения изделий для 2D-компоновки электрошкафа).

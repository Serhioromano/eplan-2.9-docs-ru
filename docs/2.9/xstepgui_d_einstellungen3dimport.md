# Диалоговое окно Настройки: 3D-импорт

Параметры> Настройки > Проекты > 'Имя проекта' > Управление > 3D-импорт.

В этом диалоговом окне установите настройки для импорта файлов STEP.

!!! note "Замечание:"

    EPLAN поддерживает исключительно импорт файлов STEP, сгенерированных с помощью прикладного протокола AP 203 или AP 214.

Обзор основных элементов диалогового окна:

### Автоматически указанное определение функции после импорта

Определение функций, содержащих импортированное трехмерное тело из файла STEP; настройка 'Размещение изделия, стандартный функциональный элемент'. Используя кнопку ++...++, можно выбрать следующие определения функций.

### Степень детализации

Для согласования объема данных при импорте файлов STEP с помощью бегунка выбирается степень детализации. Возможны пять вариантов настроек между следующими значениями:

* низкая
* средняя
* высокая

Низкая степень детализации ускоряет импорт больших файлов STEP, но ведет к более грубому представлению закруглений. Высокая степень детализации улучшает представление закруглений, но увеличивает количество данных и время импорта.

### Отделить функциональные элементы

Поведение при импорте зависит от действий системы CAD по отношению к файлу STEP. Файл STEP перед импортом подвергается анализу, в ходе которого устанавливается существующая структура организации.

* Возможность 1: Файл STEP содержит несколько тел, которые при генерировании ***не*** были объединены в узел. В этом случае флажок указывает, генерируется ли из него несколько или только один логический функциональный элемент.
* Возможность 2: Файл STEP содержит несколько тел, которые при генерировании были объединены в узел. В этом случае флажок игнорируется, и генерируется несколько логических функциональных элементов.

Если этот флажок установлен, при импорте STEP генерируется ***несколько*** логических функциональных элементов, если в импортированном файле STEP обнаружено несколько ***отдельных*** тел / поверхностей, не объединенных генерирующей системой CAD в одну единицу.

Если флажок снят, то ***отдельные*** тела / поверхности в файле STEP не разделяются на несколько логических функциональных элементов, а генерируется ***один*** логический функциональный элемент.

Если файл STEP содержит узлы, тела которых были объединены генерирующей системой CAD в одну единицу, то все объединенные тела — независимо от настройки этого флажка — всегда разделяются.

!!! tip "Совет:"

    Для последующего объединения нескольких отдельных объектов можно использовать функцию Обработать > Графика > Объединить.

### Корректировать векторы нормали и выравнивание

Вектор нормали поверхности определяет, выровнена ли поверхность по направлению к наблюдателю или от него. В файлах STEP могут содержаться поверхности, вектор нормали которых обращен внутрь, за счет чего такие поверхности интерпретируются как прозрачные.

Если этот флажок установлен, соответствующие поверхности при импорте генерируются таким образом, что они оказываются непрозрачными с обеих сторон.

!!! note "Замечание:"

    Корректировка векторов нормали может производиться только на полных замкнутых телах. Поэтому для включения корректировки необходимо снять флажок Разделить функциональные элементы. Обе настройки не могут использоваться вместе.

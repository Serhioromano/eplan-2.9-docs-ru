# Диалоговое окно Настройки: Внешняя обработка

* Вы открыли проект. Сервисные программы > Внешняя обработка свойств > Экспортировать данные. Нажмите в диалоговом окне Внешняя обработка рядом с полем Настройки на ++...++.

* Параметры > Настройки > Пользователь > Интерфейсы > Внешняя обработка.

В этом диалоговом окне можно управлять схемами и обрабатывать их для внешней обработки. В схеме указываются настройки для внешней обработки.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

!!! note "Замечание:"

    Чтобы старые схемы были доступны, их необходимо сначала экспортировать в одной из предыдущих версий программы. Это можно сделать либо для всех схем при помощи кнопки ![](images/all_export_as.png){: .ui-icon } (Экспортировать) в левой части диалогового окна настроек (панель инструментов над структурой дерева областей настроек), либо для отдельной схемы с помощью кнопки ![](images/all_export_as.png){: .ui-icon } (Экспортировать) в правой части диалогового окна настроек (панель инструментов рядом с полем Схема).

    Затем после импорта с помощью соответствующей кнопки ![](images/all_import_as.png){: .ui-icon } (Импортировать) эти схемы конвертируются в новый формат.

### Тип объекта

Схема для внешней обработки всегда создается вместе с типом объекта. В этом поле отображается тип объекта, выбранный при создании текущей схемы с помощью кнопки ![](images/all_new_as.png){: .ui-icon } (Создать) в диалоговом окне Тип объекта. Тип объекта затем уже ***нельзя*** изменить!

В диалоговом окне Тип объекта на выбор предлагаются следующие типы объектов:

* Точки разрыва
* Экранирования
* Все функции
* Общие функции
* Размещение изделия (3D) (в зависимости от лицензии)
* Кабели
* Клеммы
* Клеммники
* Функциональные тексты зон
* Определение потенциала
* Определение трубопровода
* Выводы устройства ПЛК
* Блоки ПЛК
* Сегмент: Резервуары
* Сегмент: Технологические контуры
* Сегмент: Функции ТК
* Сегмент: Объекты планирования
* Сегмент: Сегменты структуры
* Шаблон сегмента: Резервуары
* Шаблон сегмента: Технологические контуры
* Шаблон сегмента: Функции ТК
* Шаблон сегмента: Объекты планирования
* Шаблон сегмента: Сегменты структуры
* Страницы
* Штекеры
* Контакты штекеров
* Обозначения местоположения
* Соединения
* Точки определения соединения с ОУ.

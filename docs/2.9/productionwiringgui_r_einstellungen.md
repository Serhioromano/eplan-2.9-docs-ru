# Вкладка Настройки

* Параметры > Настройки > Фирма > Сборка проводов > "Машина" > Настройки
* Сервисные программы > Данные изготовления > Сборка проводов > "Машина". Щелкните по кнопке ++...++ рядом с полем Схема. Выберите вкладку Настройки.

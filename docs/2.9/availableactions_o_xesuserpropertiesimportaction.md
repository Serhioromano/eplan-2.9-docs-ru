# Операция: XEsUserPropertiesImportAction

**Параметр** |  **Описание**
---|---
XMLFile |  Имя файла XML, из которого следует экспортировать, вкл. полный путь (является опцией). Если данные отсутствуют, откроется диалоговое окно выбора файла.
Project  |  Полное имя проекта, в который следует импортировать определенные пользователем свойства (является опцией). Если данные отсутствуют, будет использован текущий проект или откроется диалоговое окно выбора файла.
Overwrite  |  Если значение равно 0, определенные пользователем свойства с одинаковым именем перезаписываются при импорте (является опцией). Значение по умолчанию: 1 (Откроется диалоговое окно с запросом).

!!! example "Пример:"

    XEsUserPropertiesImportAction
    /XMLFile:c:\my_user.xml
    /Project:c:\...\ESS_Sample_Project.elk

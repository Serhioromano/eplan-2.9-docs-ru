# Считать данные клиента

Вы можете считать из базы данных изделий в свойства текущего проекта такие адресные данные клиента и производителя/поставщика, как, например, адрес, название фирмы или номер телефона. Таким образом, не требуется повторный ввод данных. Возможно считывание адресных данных производителя/поставщика в группу свойств "Автор" и считывание адресных данных клиента в группу свойств "Клиент" и "Заказчик". Имеющиеся свойства затираются.

Условия:

* Вы считали каталоги и проекты в управление проектами.
* Вы задали в глобальных настройках базу данных изделий, содержащую необходимые адресные данные клиента или производителя/поставщика.

1. Выберите следующие пункты меню: Проект > Управление
2. Выделите проект.
3. Чтобы считать данные производителя / поставщика в группу свойств "Автор", выберите ++Дополнительно++ > Считать данные клиента > Автор.
4. Чтобы считать данные клиента в группу свойств "Клиент", выберите ++Дополнительно++ > Считать данные клиента > Клиент.
5. Чтобы считать данные клиента в группу свойств "Заказчик", выберите ++Дополнительно++ > Считать данные клиента > Заказчик.
6. Выделите в диалоговом окне Выбрать адрес одну запись.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Данные клиента принимаются в свойства проекта и отображаются на вкладке Свойства.

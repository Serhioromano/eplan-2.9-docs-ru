# Диалоговое окно Свойства контура — <Название контура> / диалоговое окно Свойства профиля (медь) — <Имя профиля>

Вы открыли проект.

* Сервисные программы > Основные данные > Контур (...) > Создать > [Сохранить].
* Сервисные программы > Основные данные > Контур (...) > Открыть. Выберите пункт всплывающего меню Свойства на контуре в диалоговом окне навигатора Страницы - <Имя проекта>.
* Сервисные программы > Основные данные > Профиль (медь) > Создать > [Сохранить]
* Сервисные программы > Основные данные > Профиль (медь) > Открыть. Выберите пункт всплывающего меню Свойства на профиле в диалоговом окне навигатора Страницы - <Имя проекта>.

В этом диалоговом окне установите свойства сгенерированного и открытого контура или профиля (медь).

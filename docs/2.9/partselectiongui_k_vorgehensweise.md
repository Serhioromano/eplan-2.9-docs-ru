# Выбор устройства: Способ действий

Выбор устройства можно проводить для всего проекта или выбора устройств. Выбор устройства можно вызывать из различных мест:

* из диалогового окна Свойства ++...++ * из графического редактора
* из навигатора устройств
* из навигатора спецификаций.

В зависимости от того, откуда был вызван выбор устройства, возможны следующие различные количества выбора:

* текущий ***проект***
* выделенные ***страницы***
* выделенные ***элементы*** в графическом редакторе, навигаторе устройств или спецификаций.

При выборе устройства вначале для соответствующего устройства проверяется, имеются ли в базе данных изделий идентичные функции. Если это не так, проверяется, имеется ли общая функция (напр., замыкающий контакт, общий). При ее наличии предлагается устройство. Наоборот, для устройства, имеющего общую функцию, при выборе устройства предлагаются все устройства, которые имеют шаблон функции из той же группы.

Можно выбрать устройства на основе номера типа, номера заказа или номера изделия. Для каждого выбранного изделия копируются (электрические) параметры, группа функций, тип изделия, вид изделия, вариант контактора, узел и группа продуктов из определения изделия в условное обозначение. При этом также отображаются возможные принадлежности изделия.

Затем показывается табличный предварительный просмотр определенных шаблонов функции. Показываются необходимые (т.е. определенные через его функции для устройства) и установленные в определении устройства шаблоны функции.

В зависимости от настроек для выбора устройства проверяется, изменяются ли вследствие выбора изделия имеющиеся данные функции (напр., обозначения вывода устройства). Пиктограмма показывает, подходят ли определения функции устройства к определениям изделия.

!!! note "Замечание:"

    * Для устройств с определениями функций группы "Графика" при выборе устройства отображаются все изделия, имеющие шаблоны функций с определением функции "Графика". Дополнительно отображаются все изделия без шаблонов функций и изделия принадлежностей. Таким способом выбираются, например, изделия главной группы продуктов "Механика".
    * Чтобы при добавлении принадлежностей через выбор устройства учитывались имеющиеся выводы устройства, необходимо установить в настройках выбора устройства флажок Обозначение вывода устройства.

### Выбор устройства для специальных функций

Функции, необходимые только для определения устройства, при выборе устройства обрабатываются отдельно. Сюда относятся черные ящики, блоки ПЛК, определения клеммников, определения штекеров, определения кабеля и экранирования.

Если выполнен выбор устройства для устройства, который состоит ***только*** из таких функций и в противном случае пуст, при выборе устройств отображаются все изделия обладающие как минимум шаблоном функции для данной функции.

!!! example "Пример:"

    Например, при выполнении выбора устройства на***пустом***блоке ПЛК отображаются только изделия, обладающие шаблоном функции с определением функции "Блок ПЛК" (и любым количеством шаблонов функций для выводов устройств ПЛК).Но изделия не отображаются, если они обладают только шаблонами функций для выводов устройства ПЛК, но не для блоков ПЛК.

### Многократный выбор (групповая обработка)

При выборе нескольких элементов поведение выбора устройства или изделия изменяется в зависимости от того, запускается эта операция из диалогового окна Свойства ++...++ или из навигатора:

* При выборе устройства или изделия из диалогового окна Свойства ++...++ выбранные данные изделий "прописываются" на все выделенные условные обозначения (т. е. выбранное изделие присваивается всем выделенным элементам). В качестве критерия выбора, впрочем, применяется только первое условное обозначение. Если выбранное изделие не подходит другим элементам, после контрольного прогона показываются соответствующие сообщения.
* При выборе устройства или изделия из навигатора спецификаций диалоговое окно выбора вызывается последовательно для каждого выделенного элемента.

Выбор устройства выполняется последовательно для всех выбранных главных функций.

### Присвоение изделий устройствам

Устройства могут представляться при помощи одного или нескольких условных обозначений:

* Отдельный символ, напр., двигатель
* Распределенное устройство (несколько символов, одинаковое ОУ), напр., контактор, кабель с оболочкой
* Модуль (несколько символов, различные ОУ), напр., схема соединения "звезда-треугольник" как 19" / вставной блок МСС.

Этим устройствам можно присвоить различные изделия:

* Отдельное изделие
* Узел
* Вложенный узел
* Модуль.

При этом возможна любая комбинация устройства и изделия. Информация о модуле сохраняется в главной функции.

### Присвоение модулей

Помимо отдельных изделий или узлов, при выборе устройства можно присваивать модули.

Если модуль присваивается "обычной" главной функции, то там сохраняется номер изделия модуля.

Если, напротив, модуль присваивается черному ящику, то EPLAN учитывает все основные и вспомогательные функции, которые содержатся в выделенном черном ящике, а также во всех перекрестных с ним черных ящиках, и присваивает изделие модуля соответствующим главным функциям. Кроме того, данные из шаблонов функций переносятся в функции, содержащиеся в черном ящике.

Если черному ящику присваивается не модуль, а узел, то изделия узла сохраняются в черном ящике и не распределяются на содержащиеся в нем функции.

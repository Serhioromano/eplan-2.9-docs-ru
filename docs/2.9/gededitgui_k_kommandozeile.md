# Область ввода данных

Область ввода данных предоставляет возможность ввода значений с клавиатуры при использовании различных функций в редакторах (графическом редакторе, 3D-редакторе и т. д.). Если область ввода данных включена, то при использовании таких функций (например, при вставке линии) у курсора выводится небольшое текстовое окно для ввода чисел. Степень прозрачности данного окна задана предварительно, и ее можно изменить в [настройках](gedviewer_d_einstellkommandozeile.md).

### Ввод в области ввода данных

Текстовое окно области ввода данных состоит из двух строк заголовков и собственно области для ввода данных. В первой строке заголовков отображается соответствующая операция (например, начальная точка для линии), а во второй строке — требуемый ввод как текст комментария. В области ввода данных отображаются числа, введенные при помощи клавиатуры.

Последовательность вводимых значений для каждой функции задается в системе. Например, для линии возможны два ввода в последовательности "X- и Y-координаты точки" или "Угол и длина".
Во время ввода данных о точке два введенных значения разделяются пробелом и интерпретируется как значения координат X и Y (начальной или конечной точки). Введенные значения при этом относятся к графической системе координат, которая начинается в левой нижней точке, и представляются в соответствии с настроенными единицами измерения в "мм" или "дюймах".

!!! note "Замечание:"

    Обратите внимание на то, что только при первом вводе данных о точке используются абсолютные значения, а во всех последующих операциях ввода данных о точках используются относительные координаты.

Десятичные дроби записываются с используемым в операционной среде десятичным разделителем "." или ",".

Нажатие клавиши ++Ввод++ приводит к немедленной передаче введенного значения рабочей функции; после этого область ввода данных очищается. После последнего ввода или завершения операции область ввода данных закрывается.

Во время ввода данных в открытой области ввода вызов других функций посредством соответствующих комбинаций клавиш ***не***возможен. Нажатие клавиши ++Esc++ позволяет отменить ввод в область ввода данных.

### Функции с поддержкой области ввода данных

Ввод в области ввода данных возможен при работе со следующими функциями платформы EPLAN:

* при вставке графических элементов: кривых, ломаных линий, прямоугольников, окружностей, дуг, секторов, текста;
* при вставке указаний размеров;
* при обработке графических элементов: повороте, копировании, перемещении, зеркальном отображении, масштабировании, растягивании, обрезке, изменении длины, закруглении углов, фасках;
* при вставке трехмерных функциональных элементов: несущей шины, кабельного канала, электрошкафа, монтажной платы, заблокированной области, логического функционального элемента;
* при обработке трехмерных функциональных элементов: размещении с копированием длины, изменении длины, повороте вокруг оси, определении точек сопряжения и точек захвата.

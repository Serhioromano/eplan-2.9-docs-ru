# Создать базовые проекты

[Базовые проекты](projects_k_basisprojekteerweitert.md) можно создавать с помощью меню проектов или управления проектами. Можно использовать базовый проект для создания проекта. При создании базового проекта из имеющегося проекта, весь проект сохраняется в базовом проекте.

Условие:

Вы выделили проект.

Навигатор страниц: Проект > Организовать > Создать базовый проект.

Управление проектами: [Дополнительно] > Создать базовый проект.

1. В диалоговом окне Создать базовый проект укажите место хранения и имя для базового проекта.

!!! info "Для сведения:"

    В качестве места хранения EPLAN предлагает каталог шаблонов, указанный в настройках пользователя для основных данных в диалоговом окне Настройки: Каталоги.

2. Щелкните по кнопке ++Сохранить++.

!!! info "Для сведения:"

    Создается базовый проект *.zw9.

# Работа с вложенными обозначениями местоположения

Если в диалоговом окне Расширенные структуры проекта на вкладке Наследование установлен флажок Обозначение местоположения в групповом поле Вложение, то обозначения местоположения будут вложенными. В этом случае нет необходимости в отображении (вводе) точки для находящегося внутри идентификатора в схеме соединений.

!!! example "Пример:"

    Если для многократно вложенных обозначений местоположения=A2+O2,=A3+O3и=A4+O4установлен флажок Обозначение местоположения(левая часть рисунка), то предохранитель получает ОУ=A2.A3.A4+O2.O3.O4-F3; если флажок Обозначение местоположенияне установлен (правая часть), то предохранитель имеет обозначение=A4+O4-F3.

Таким образом, в случае изменения этой настройки требуется доработать все условные обозначения, находящиеся в обозначениях местоположения.

При этом всегда используется направление снаружи внутрь, т.е. в наружном обозначении местоположения отсутствующие части дополняются из страницы. Обозначение местоположения, которое находится в наружном обозначении, дополняет свои отсутствующие части из окружающего его местоположения и т.д. И в конце уже условное обозначение дополняет отсутствующие части из самого внутреннего обозначения местоположения. Это означает, что отсутствующие ли неполные идентификаторы записи Видимое ОУ дополняются из вышестоящей единицы, причем вышестоящей единицей может быть страница, обозначение местоположения или черный ящик.

Условие:

Вы открыли проект.

1. В представлении в виде дерева навигатора страниц выберите пункт всплывающего меню Проект > Свойства.
2. В диалоговом окне Свойства проекта выберите вкладку Структура.
3. Нажмите кнопку ++Другие++.
4. В диалоговом окне Расширенные структуры проекта на вкладке Наследование установите флажок Обозначения местоположения в групповом поле Вложение.
5. Подтвердите ввод.

!!! info "Для сведения:"

    Структуры проекта меняются, и обозначения устройств адаптируются соответствующим образом.

6. Перейдите на страницу схемы соединений.
7. Вставьте несколько вложенных обозначений местоположения.

!!! info "Для сведения:"

    В поле Полное ОУ с помощью вложенного обозначения устройства для каждого вложенного обозначения местоположения указывается вложение.

!!! note "Замечание:"

    В случае вложенных обозначений местоположения идентификаторы в ***видимом*** ОУ обозначения местоположения всегда интерпретируются как подидентификаторы независимо от того, был ли введен разделитель для подидентификаторов (точка) или нет.

!!! tip "Совет:"

    При перемещении элемента из блока удерживайте нажатой клавишу ++Shift++, чтобы сохранить изначально вложенное ОУ.

# Диалоговое окно Настройки: Драйвер для сервера SQL

Параметры > Настройки > Рабочая станция > Интерфейсы > Драйвер для сервера SQL

При использовании баз данных SQL для базы данных изделий, управления проектами и словаря в этом диалоговом окне можно установить альтернативный драйвер для сервера SQL. Это также позволяет использовать базы данных на серверах SQL с более высокими требованиями безопасности (например протокол шифрования с TLS 1.1 или TLS 1.2).

Обзор основных элементов диалогового окна:

### Драйвер для сервера SQL

Выберите параметр Альтернативный драйвер, если требуется выбрать альтернативный драйвер для сервера SQL из раскрывающегося списка.

!!! note "Замечание:"

    Выбор в раскрывающемся списке доступен, только если на компьютере установлены соответствующие драйверы. При возникновении вопросов о возможных драйверах обратитесь к системному администратору SQL.

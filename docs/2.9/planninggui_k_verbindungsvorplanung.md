# Объекты планирования, соединения

Объекты планирования, соединения (объекты планирования, трубопроводы; объекты планирования, кабели) управляются параллельно с общими объектами планирования и ведут себя аналогично. На объекте планирования (соединении) можно указывать внешние документы / страницы, изделия и шаблоны функций, но не адреса ПЛК или макросы.

Объект планирования (соединение) должен иметь источник и цель по аналогии с соединением. Источник и цель можно указать в диалоговом окно "Свойства" объекта планирования (трубопровода) с помощью двух полей (Исходный сегмент и Целевой сегмент), а в диалоговом окне "Свойства" объекта планирования (кабеля) — с помощью двух одноименных свойств.

Объекты планирования (соединения) можно обозначать и размещать в [графическом предварительном планировании](planninggui_k_grafischevorplanung.md) в виде открытых ломаных линий на страницах предварительного планирования. При этом исходный и целевой сегмент начальной и конечной точек ломаной линии переносятся из графического редактора. Если объект планирования (соединение) размещен на странице предварительного планирования в виде ломаной линии, исходный и целевой сегменты будут недоступны для обработки в диалоговом окне "Свойства". При этом источник и / или цель объекта планирования (соединения) можно редактировать путем графического изменения соответствующей ломаной линии.

С помощью пунктов меню Данные проекта > Соединения > Заменить источник и цель вы можете поменять источник и цель на объектах планирования соединения, выбранных в навигаторе предварительного планирования. Эта операция меняет в диалоговом окне "Свойства" данные для исходных и целевых сегментов и соответствующим образом адаптирует графику на страницах предварительного планирования.

При помощи контрольного прогона "028026" можно проверить наличие источника или цели объекта планирования (соединения).

### Предварительное планирование кабелей с объектами планирования (кабелями)

С помощью объектов планирования (кабелей), размещенных в графическом предварительном планировании, в ходе проектирования можно заранее указать первые данные кабеля без имеющегося детального планирования. Такое предварительное планирование кабелей может быть полезно, например, в том случае, если разводка кабелей всех компонентов электротехники и системы управления и защиты выполнена сеткообразно через всю установку.

Чтобы определить общую длину кабелей на более позднем этапе, укажите на объекте планирования (кабеле) неопределенное изделие (напр., с помощью записи "not yet known" для номера изделия в первой строке на вкладке Изделие) и введите соответствующую длину кабеля в свойство Подмножество / длина данных ссылки изделия.

С помощью отчета "Предварительное планирование: Обзор объектов планирования" можно определить и проанализировать исходные и целевые сегменты, затраты времени (планирование / строительство), первые расчетные данные (цены), а также значения общей длины кабелей.

На последующем этапе детального планирования можно присвоить объекты планирования (кабели) кабелям на схеме соединений. Таким образом значения, введенные в предварительном планировании, можно копировать в детальное планирование, отображать и анализировать.

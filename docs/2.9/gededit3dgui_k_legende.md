# Легенды электрошкафа обзоров модели

Легенды электрошкафа генерируются не пространством листа, а полученными из пространства листа обзорами модели. Таким образом, страница проекта может содержать несколько обзоров модели, которые, соответственно, показывают одну определенную область пространства листа, ограниченную в диалоговом окне Обзор модели посредством выбора в поле Основной функциональный элемент. Это может быть, к примеру, монтажная плата, оснащенная несущая шина или целый электрошкаф. Количество объектов, содержащихся в обзоре модели, можно дополнительно изменять кнопкой [Отобразить / Скрыть].

При генерировании легенды электрошкафа на странице проекта, содержащей несколько обзоров модели, для каждого обзора модели создается отдельная легенда электрошкафа. Каждая отдельная легенда электрошкафа содержит компоненты, которые были предварительно выбраны для соответствующего обзора модели.

Чтобы сгенерировать легенду электрошкафа для всего содержимого пространства листа, необходимо сначала вставить обзор модели без ограничения основных функциональных элементов.

### Подавить генерирование легенды электрошкафа

Чтобы исключить обзор модели из генерирования легенды электрошкафа, откройте диалоговое окно Обзор модели. В групповом поле Свойства (категория: "Настройки") вставьте свойство Подавить генерирование легенды электрошкафа и установите соответствующий флажок. Для этого обзора модели легенда электрошкафа создаваться не будет.

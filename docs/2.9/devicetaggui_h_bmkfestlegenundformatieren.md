# Определить и форматировать свойства устройства

Обозначение устройства идентифицирует устройство, и его структуру можно определять согласно различным стандартам. При этом можно использовать структурные идентификаторы согласно DIN или применять определенные пользователем структуры.

Условия:

* Вы открыли проект.
* В диалоговом окне Настройки: Общее (путь: Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Общее), указывается, как должны обрабатываться прописные и строчные буквы и указание предшествующих знаков для определенных устройств.

1. Откройте диалоговое окно "Свойства" для функции, например, через меню Обработать > Свойства.
2. Выберите в диалоговом окне Свойства ++...++ вкладку <Категория функции>.
3. В поле Видимое ОУ введите устройство, которое должно отображаться в графическом редакторе.

!!! info "Для сведения:"

    Запись в поле Полное ОУ обновляется автоматически.

4. Щелкните в поле Полное ОУ по ++...++.
5. Обработайте в диалоговом окне Полное ОУ элементы ОУ и щелкните по кнопке ++OK++.
6. Выберите вкладку Отображение.
7. Выделите свойство ОУ (видимое) в списке свойств в левой области диалогового окна.
8. Измените настройки отображения и присоединения, скопированные из "Стандартного значения".

!!! info "Для сведения:"

    Раскрывающийся список Порядок свойств будет автоматически изменен на "определяемый пользователем".

9. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Обозначение устройства отображается согласно вашим настройкам в графическом редакторе.

!!! tip "Совет:"

    В поле Видимое ОУ с помощью пункта всплывающего меню Определить новое ОУ или при помощи комбинации клавиш [Ctrl] \+ [N] можно генерировать новое ОУ со следующим свободным счетчиком.

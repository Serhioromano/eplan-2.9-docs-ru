# Диалоговое окно Настройки: Группа соединений

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Группа соединений.

В этом диалоговом окне вы задаете предварительные настройки для символа, который будет отображаться в точках определения соединения при определении группы соединений. По умолчанию настраивается символ без графики, благодаря чему точки определения соединений, автоматически размещаемые при определении группы соединений, видны только при помощи точек вставки в графическом редакторе.

Обзор основных элементов диалогового окна:

### Номер / Имя

В этом поле выводятся имя и номер текущего символа; щелкните по кнопке ++...++, чтобы открыть диалоговое окно [Выбор символа](xsbgui_d_symbolauswahl.md) и при необходимости выбрать другой символ.

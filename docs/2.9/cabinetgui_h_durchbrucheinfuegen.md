# Вставить монтажные отверстия

Монтажные отверстия представляют собой размещенные вручную отверстия в конструктивных функциональных элементах, монтажных платах и панелях, просверленные или выполненные автоматами ЧУ. Монтажные отверстия моделируют трехмерные тела, в которые они вставляются не буквально, а визуально посредством заполнителя, который проходит через функциональный элемент и получает все данные о координатах и толщине материала. Машины ЧУ могут интерпретировать эти данные и преобразовывать в этапы изготовления.

Если на изделии сохранена соответствующая схема сверления, при размещении функциональных элементов с переменной длиной автоматически генерируются монтажные отверстия. Если из-за настроек схемы сверления одно из этих монтажных отверстий полностью или частично расположено в запретной зоне для сверления, автоматически генерируется альтернативное монтажное отверстие вне запретной зоны для сверления.

Монтажные отверстия следующих типов можно вставлять отдельно или многократно:

* Отверстие
* Отверстие с резьбой
* Прямоугольник
* Продольн. отверстие
* Шестиугольник
* Восьмиугольник
* Определенный пользователем контур
* Из схемы сверления.

У каждого типа монтажного отверстия есть диалоговое окно, в котором настраиваются необходимые для определения значения. При размещении всех типов можно воспользоваться справкой по размещению:

* Использование [автоматического активирования](cabinetgui_h_aktivierenautomatisch.md)
* Переключение между девятью точками захвата.

Условия:

* Вы открыли проект.
* Навигатор пространства листа открыт, и открыто пространство листа.
* Пространство листа содержит монтажную плату и / или конструктивные функциональные элементы.

### Вставка монтажного отверстия

Здесь приводится пример вставки отверстия для всех восьми типов монтажных отверстий. Остальные типы размещаются аналогично, отличается только вид вводимого значения в диалоговом окне <Тип монтажного отверстия>.

1. Выберите пункты меню Вставить > Монтажное отверстие > Отверстие.
2. В диалоговом окне Отверстие введите значение диаметра или выберите значение из раскрывающегося списка.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Рядом с курсором появится прозрачное изображение предварительного просмотра отверстия в заданном размере. Выбранная в текущий момент точка захвата выделена красным цветом и дополнительно отмечена красным квадратом.

4. С помощью клавиши ++A++ можно переключать точку захвата.

!!! info "Для сведения:"

    При нажатии клавиши ++A++ точка захвата перемещается от позиции 'центр' к одной из восьми окружающих позиций.

5. Выберите пункт всплывающего меню Опции размещения, чтобы открыть диалоговое окно [Опции размещения](cabinetgui_d_platzieroptionen.md).
6. С помощью щелчка разместите монтажное отверстие на требуемом месте.

!!! tip "Совет:"

    При размещении монтажного отверстия может возникнуть ситуация, когда точки монтажа, исходные точки, монтажная сетка и монтажная поверхность накладываются друг на друга на одном месте и при этом могут в равной степени использоваться для размещения. В подобной ситуации найденные интерактивные точки и поверхности после щелчка по ним предлагаются на выбор в виде списка. Выбрать отдельные записи в списке можно щелчком мыши или при помощи клавиш курсора. При этом соответствующая точка или поверхность зрительно выделяется, и отображается предварительный просмотр ожидаемого в этом месте размещения. Выбрать нужную запись для размещения можно двойным щелчком мыши или клавишей [Ввод].

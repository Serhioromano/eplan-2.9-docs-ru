# Произвести настройки для отображения перекрестных ссылок

Перекрестная ссылка важна для того, чтобы с уверенностью находить условное обозначение на большом количестве страниц. По этой причине перекрестная ссылка должна однозначно обозначать, как минимум, искомую страницу. Дополнительно необходима оптимизация в пределах страницы, столбца. Так как и одна страница может быть относительно большой, имеется еще один ориентир в пределах страницы, строка.

Условие:

Вы открыли проект.

Параметры > Настройки > Проекты > "Имя проекта" > Перекр. ссылки / образы контактов

1. Выберите область настройки Точки разрыва.
2. Укажите в групповом поле Отображение, каким образом должны отображаться перекрестные ссылки на точках разрыва.
3. Выберите область настройки Общее.
4. Укажите в групповом поле Отображение, каким образом должны отображаться перекрестные ссылки на функциональном элементе.
5. Укажите в групповом поле Разделитель, какие символы должны использоваться как разделители ***перед страницей***, а также между ***страницей и столбцом***, и ***строкой и столбцом***.
6. Укажите в групповом поле Выделение для, как символ должен использоваться как выделение, чтобы различать разные виды перекрестных ссылок.
7. Укажите здесь для различных типов страниц, к каким другим типам страниц должны отображаться перекрестные ссылки и в какой последовательности.
8. Укажите в групповом поле Отображать перекрестные ссылки обзора действие обзорных перекрестных ссылок.
9. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Настройки применяются.

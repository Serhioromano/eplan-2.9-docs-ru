# Подготовка таблицы Excel для импорта (предварительное планирование)

Чтобы подготовить таблицу Excel для импорта данных, необходимо определить в таблице Excel область, содержащую импортируемые данные. Таблица Excel может содержать имена столбцов (но не обязательно).

Условие:

У вас есть доступ к таблице Excel с соответствующими данными.

1. Откройте таблицу в Excel.
2. Выберите в Excel пункты меню Формулы > Диспетчер имен.
3. Нажмите кнопку ++Создать++.
4. В диалоговом окне Новое имя в поле Имя введите информативное имя для импортируемой области.
5. Справа внизу щелкните по символу возле поля Относится к.

!!! info "Для сведения:"

    Диалоговое окно сворачивается.

6. Выделите в таблице Excel требуемую область данных, включая заголовки столбцов.

!!! info "Для сведения:"

    Выделенная область отмечается штриховой линией.

7. Щелкните снова по символу возле поля Относится к.

!!! info "Для сведения:"

    Диалоговое окно раскрывается.

8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Новое имя вносится в поле списка диспетчера имен.

9. Щелкните по кнопке ++Закрыть++.

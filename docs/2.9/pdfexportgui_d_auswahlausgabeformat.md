# Диалоговое окно Выбрать формат бумаги

* Выделен проект или, как минимум, одна страница проекта. Страница > Экспортировать > PDF > [Настройки] > Выходная величина. Вы выбрали опцию Масштабировать на страницу. Щелкните мышкой рядом с полем Ширина по кнопке ++...++.
* Параметры > Настройки > Рабочая станция > Графическая обработка > PDF-выходная величина. Вы выбрали опцию Масштабировать на страницу. Щелкните мышкой рядом с полем Ширина по кнопке ++...++.

В этом диалоговом окне можно выбрать формат бумаги, который будет использоваться для вывода PDF.

Обзор основных элементов диалогового окна:

### Формат

Выберите из раскрывающегося списка формат бумаги, который вы хотите использовать. Доступны следующие форматы:

Формат бумаги |  Размеры (мм x мм) |  Размеры (дюйм x дюйм)
---|---|---
A0 |  841 x 1189 |  33.1 x 46.8
A1 |  594 x 841 |  23.4 x 33.1
A2 |  420 x 594 |  16.5 x 23.4
A3 |  297 x 420 |  11.7 x 16.5
A4 |  210 x 297 |  8.3 x 11.7
A5 |  148 x 210 |  5.8 x 8.3
ANSI A (Letter) |  216 x 279 |  8.5 x 11
ANSI B (Ledger, Tabloid) |  279 x 432 |  11 x 17
ANSI C (Broadsheet) |  432 x 559 |  17 x 22
ANSI D |  559 x 864 |  22 x 34
ANSI E |  864 x 1118 |  34 x 44

!!! note "Замечание:"

    Обратите внимание, что страницы проекта при выводе в формате PDF ***нельзя*** повернуть. Если вы выбрали для выходной величины формат бумаги DIN A4 в ***книжном формате***, а страница проекта имеет формат бумаги DIN A4 в ***альбомном формате***, страница проекта масштабируется до размера DIN A5 и выводится на странице PDF.

### Ширина / высота

В этих полях отображаются размеры выбранного формата бумаги в настроенных единицах измерения. Размеры переносятся в диалоговое окно Настройки: PDF-выходная величина.

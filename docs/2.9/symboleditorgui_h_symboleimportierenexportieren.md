# Импортировать / экспортировать символы

Вы можете экспортировать символы, чтобы импортировать их в другие библиотеки символов.

Условие:

Вы открыли проект и как минимум одну библиотеку символов.

1. Выберите следующие пункты меню: Сервисные программы > Основные данные > Символ > Импорт
2. если открыто несколько библиотек символов, то в диалоговом окне Выбор библиотеки символов выберите библиотеку символов, в которую должен быть импортирован символ.
3. Щелкните по кнопке ++OK++.
4. В диалоговом окне Импортировать символа выберите из списка требуемый символ (*.esy). Многократный выбор ***невозможен***.
5. Щелкните по кнопке ++Открыть++.
6. В диалоговом окне Свойства символа - <Имя библиотеки> задайте данные импортируемого символа; при этом будут показаны следующие предварительные настройки:
Номер символа: Следующий свободный номер
Имя символа: "<Имя оригинала>", если он еще не существует в текущей библиотеке символов, иначе "<Имя оригинала> (копия)"
Определение функции: Определение функции оригинала.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Символ импортируется и доступен для дальнейшей обработки в текущем проекте.

1. Выберите следующие пункты меню: Сервисные программы > Основные данные > Символ > Экспорт
2. Выберите в диалоговом окне Выбор символа экспортируемые символы. Возможен многократный выбор.
3. Щелкните по кнопке ++OK++.
4. Выберите в диалоговом окне Экспортировать символы целевой каталог, в который следует экспортировать символы, для этого щелкните по ++...++ и перейдите в нужный каталог.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Файлы экспорта генерируются на указанном дисководе и в указанном каталоге.

!!! note "Замечание:"

    При экспорте учитываются все имеющиеся варианты символа.

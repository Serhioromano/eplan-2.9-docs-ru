# Вкладка Кабель / экранирование

Вы открыли проект.

* Вы выделили в Графическом редакторе линию определения кабеля или экранирование. Обработать > Свойства. Выберите в диалоговом окне Свойства ++...++ вкладку Кабель / Экранирование.
* Вы выделили определение кабеля или экранирование в Навигаторе кабелей или Навигаторе устройств. Пункт всплывающего меню Свойства или Свойства (общие).Выберите в диалоговом окне Свойства ++...++ вкладку Кабель / Экранирование.

На этой вкладке можно обрабатывать свойства кабеля или экранирования.

В режиме обработки ***Свойства (общие)*** вы изменяете свойства сразу для всех представлений функции, в этом случае во вкладке дополнительно отображается "(Устройство)".

Обзор основных элементов диалогового окна:

### Видимое ОУ

Видимое ОУ будет показано в графическом редакторе рядом с условным обозначением. Если вы определили в настройках проекта, что устройство должно автоматически нумероваться при вставке символов, здесь вам будет автоматически предложено ОУ. Вы можете сохранить или переписать вручную это ОУ.

Кроме того, можно использовать [всплывающее меню или кнопку ++...++](devicetaggui_r_feldsichtbaresbmk.md), чтобы cгенерировать или выбрать ОУ.

При вставке ***Экранирований*** видимое ОУ остается пустым. Но при необходимости вы можете ввести ОУ вручную или задать его через нумерацию устройств.

### Полное ОУ

В этом поле отображается полное ОУ (т. е. структурный идентификатор текущего окружения + видимое ОУ) видимого ОУ / функционального элемента. С помощью кнопки ++...++ откройте диалоговое окно Полное ОУ. Здесь можно обработать структурный идентификатор и, например, выбрать другое место установки.

### Тип

В этом поле можно ввести тип кабеля. С помощью кнопки ++...++ откройте диалоговое окно Выбор изделия. В нем вы можете выбрать изделие кабеля и присвоить его технические данные текущему кабелю.

Данные об изделиях будут обновлены (как при выборе нового изделия). В отличие от обычного выбора изделий диалоговое окно конфликта, в котором отображаются старые и новые данные, здесь не появляется, а вместо этого старые данные перезаписываются.

### Функциональный текст

Укажите здесь функциональный текст для функции / устройства.

Независимо от введенного здесь вручную функционального текста существуют другие (определяемые автоматически) функциональные тексты, которые находятся в таблице:

* Функциональный текст (автоматически): Отображает содержание введенного вручную функционального текста или, при его отсутствии, функциональный текст зоны. Если вы изменяете функциональный текст в режиме обработки ***Свойства (общие)***, его содержание переносится обратно в ту функцию, в которой функциональный текст был первоначально задан. При отображении этого свойства на странице проекта разрывы строк удаляются.
* Функциональный текст (общий): отображает (автоматический) функциональный текст функции или, при его отсутствии, (автоматический) функциональный текст соответствующей главной функции.

### Число соединений

Число соединений копируется из базы данных изделий и может быть использовано в качестве критерия фильтрации при выборе устройства.

Вы можете ввести значение вручную; при этом возможен ввод данных типа "9PE" или "3x3". При автоматическом выборе кабеля число соединений определяется на основании значения:

* В случае комбинации числа и букв "PE", "PEN" или "N" EPLAN распознает это как наличие PE-, PEN- или N-жилы в общем числе жил (например, 4PE => всего четыре кабельных соединения, одно из которых - PE-жила).
* При комбинации числа и букв определение производится только по числу (напр., "4G" или "4X" => четыре кабельных соединения).
* При комбинации <Число> символ <число> (за исключением символа "+") числа перемножаются (напр., 3x3 или 3a3= > девять кабельных соединений).
* При комбинации <Число> \+ <число> числа суммируются (напр., 3+3 => шесть кабельных соединений).

### Длина

Укажите здесь длину кабеля (включая единицу). Если вы не указали единицу, будет использована единица, определенная в настройках проекта. Обзор возможных единиц измерения приводится в соответствующем разделе "[Соединения: Единицы измерения длины и поперечного сечения](connectionbrowsergui_k_einheitenverbindungen.md)".

### Поперечное сечение / диаметр соединения:Единица измерения

Поперечное сечение или диаметр соединения копируется из базы данных изделий и могут быть использованы в качестве критерия отбора при выборе устройств. Щелкните по ++...++, чтобы открыть диалоговое окно Поперечные сечения / диаметры и выбрать сечение / диаметр, включая соответствующую единицу, из предложенного списка.

Если в поле Поперечное сечение / диаметр соединения вы выбрали предварительно определенное поперечное сечение / диаметр, то в поле Единица измерения автоматически появится соответствующая единица. С помощью раскрывающегося списка возможен выбор другой единицы измерения. Обзор возможных единиц измерения приводится в соответствующем разделе "[Соединения: Единицы измерения длины и поперечного сечения](connectionbrowsergui_k_einheitenverbindungen.md)".

### Напряжение

Укажите здесь напряжение кабеля (включая единицу). При выборе изделия это поле заполняется автоматически.

### Имя экранирования

Это имя доступно только для экранирований. Введите здесь имя экранирования. При использовании имеющегося ОУ это поле заполняется автоматически, если используется имеющаяся функция типа "Экранирование" (не соединение, а символ экранирования).

### Главная функция

установите этот флажок, если текущая функция является главной.
При использовании имеющегося ОУ, для которого уже задана главная функция, при использовании уже размещенной главной функции, а также при размещении уже размещенной главной функции данное свойство будет автоматически отключено на целевой функции, то есть данный флажок будет снят.

!!! note "Замечание:"

    эта настройка автоматически синхронизируется с одноименным флажком на вкладке Данные символа / функции.

[Групповое поле "Свойства"](devicetaggui_r_gruppenfeldeigenschaften.md)

Для использования в ***Отчетах*** (напр., в спецификации кабеля) предусмотрены ***предварительно форматированные свойства***, которые составляют определенные комбинации введенных выше свойств:

* Кабель / Группа соединений: Поперечное сечение / диаметр с единицей измерения (напр., 1,5 мм²)
* Кабель / Группа соединений: Число соединений и поперечное сечение / диаметр (напр., 4x1,5)
* Кабель / Группа соединений: Число соединений, поперечное сечение / диаметр с единицей измерения (напр., 4x1,5 мм²)
* Кабель / Группа соединений: Число соединений, поперечное сечение / диаметр с единицей измерения, длина с единицей измерения (напр., 4x1,5 мм² 50 м)
* Кабель / Группа соединений: Тип, Число соединений, поперечное сечение / диаметр с единицей измерения (напр., NYY-J 4x1,5 мм²)
* Кабель / Группа соединений: Тип, Число соединений, поперечное сечение / диаметр с единицей измерения, длина с единицей измерения (напр., NYY-J 4x1,5 мм² 20 м).

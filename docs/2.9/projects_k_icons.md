# Расширения имен файлов и пиктограммы для проектов

В дереве ***Навигатор страниц*** и Управление проектами следующие пиктограммы и расширения имен файлов наглядно демонстрируют типе проекта:

Пиктограмма |  Тип файла |  Значение |  Расширение имени файлов
---|---|---|---
![](images/all_project_as.png){: .ui-icon } ![](images/space_50x5.png) |  Проект EPLAN |  Стандартный проект EPLAN, который можно открыть и обработать. Если вы работаете в многопользовательском режиме, открытый таким образом проект могут обрабатывать сразу несколько пользователей. |  *.elk
![](images/all_projectexclusive_as.png){: .ui-icon } |  Эксклюзивно открытый проект EPLAN |  Проект EPLAN, который был открыт ***эксклюзивно***. Проект можете обрабатывать только вы. Если вы работаете в многопользовательском режиме, другие пользователи не имеют доступа к открытому таким образом проекту. |  *.elk
![](images/all_projectlocked_as.png){: .ui-icon } |  Проект EPLAN с защитой от записи |  Проект EPLAN, открытый ***с защитой от записи***. Вы можете просматривать проект, но не можете его обрабатывать. Когда вы работаете в многопользовательском режиме, другие пользователи не могут обрабатывать данный проект. |  *.elk
![](images/all_packing_as.png){: .ui-icon } ![](images/all_project_as.png){: .ui-icon } |  Запакованный проект EPLAN |  Запакованный проект. Проект открыть ***невозможно***. Пиктограмма отображается только в Управление проектами. Создается с помощью выбора в меню: Проект > Запаковать. Из проекта *.elk получается проект *.elp. |  *.elp
![](images/all_filedoff_as.png){: .ui-icon } ![](images/all_project_as.png){: .ui-icon } |  Выгруженный проект EPLAN |  Для внешней обработки выгруженного проекта. Проект можно открыть в EPLAN ***с защитой от записи***, не вносить изменений и не удалять. Создается с помощью выбора в меню: Проект > Резервировать > Проект > Метод "Выгрузить для внешней обработки". Из проекта *.elk в том же каталоге получается проект *.els. Дополнительно в выбранном каталоге резервирования генерируется файл резервирования *.zw1. |  *.els
\--- |  Сохранение данных EPLAN: Проект |  1) Файл резервирования выгруженного проекта EPLAN или 2) Файл резервирования выгруженного проекта EPLAN или 3) Файл резервирования дополнительно сохраненного проекта. Создается с помощью выбора в меню: Проект > Резервировать > Проект > Метод "Дополнительно сохранить". Проект *.elk сохраняется в том же каталоге. Дополнительно в выбранном каталоге резервирования генерируется файл резервирования *.zw1. |  *.zw1
![](images/all_filedoffcompress_as.png){: .ui-icon } ![](images/all_project_as.png){: .ui-icon } |  Выгруженный запакованный проект EPLAN |  Выгруженный и запакованный проект. Такой проект следует распаковать до того, как его можно будет открыть ***с защитой от записи***. Пиктограмма отображается только в Управление проектами. Создается с помощью выбора в меню: Проект > Резервировать > Проект > Метод "Выгрузить для внешней обработки". Выгруженный проект упаковывается в управлении проектами с помощью Всплывающее меню > Запаковать. Из проекта *.elk получается проект *.els. Из проекта *.elk получается проект *.elx. |  *.elx
![](images/all_archived_as.png){: .ui-icon } ![](images/all_project_as.png){: .ui-icon } |  Архивированный проект EPLAN |  Проект открыть ***невозможно***. Пиктограмма отображается только в Управление проектами.
Создается с помощью выбора в меню: Проект > Резервировать > Проект > Метод "Архивировать". Из проекта *.elk в том же каталоге получается проект *.ela. Дополнительно в выбранном каталоге резервирования генерируется файл резервирования *.zw1. |  *.ela
![](images/revisionmgtgui_redstamp_as.png){: .ui-icon } ![](images/all_project_as.png){: .ui-icon } |  Закрытый проект EPLAN |  Закрытый проект. Проект можно открыть в EPLAN ***с защитой от записи***, не вносить изменений и не удалять. Создается с помощью выбора в меню: Сервисные программы > Управление ревизиями > Закрыть проект. Из проекта *.elk получается проект *.elr. |  *.elr
![](images/revisionmgtgui_redstamp_as.png){: .ui-icon } ![](images/all_project_as.png){: .ui-icon } |  Исходный проект EPLAN |  Исходный проект. Проект можно открыть в EPLAN ***с защитой от записи***, не вносить изменений и не удалять. Создается с помощью выбора в меню: Сервисные программы > Управление ревизиями > Сравнение свойств проектов > Сгенерировать исходный проект. Проект *.elk сохраняется. Дополнительно создается копия проекта в качестве закрытого проекта с расширением *.elr. |  REV*.elr
![](images/revisionmgtgui_graystamp_as.png){: .ui-icon } ![](images/all_project_as.png){: .ui-icon } |  Проект ревизии EPLAN |  Проект с отслеживанием изменений. Проект можно открыть и обработать. Создается с помощью выбора в меню: Сервисные программы > Управление ревизиями > Отслеживание изменений > Генерировать ревизию. Из проекта *.elk получается проект *.ell. |  *.ell

!!! note "Замечание:"

    Учтите, пожалуйста, что в диалоговом окне Открыть проект используется другая пиктограмма. Здесь проекты EPLAN обозначены с помощью символа приложения EPLAN.

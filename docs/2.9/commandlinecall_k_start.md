# Параметры командной строки EPLAN

Как и многие другие программы, EPLAN можно запустить через неграфический интерфейс пользователя. При этом мы предоставляем вам соответствующие параметры командной строки, которые можно ввести в поле Требование ввода Windows, через другую консоль или посредством команды Windows Выполнить.

Сам вызов EPLAN осуществляется при указании исполняемого файла Eplan.exe. При вводе этой команды следует соблюдать путь, по которому файл EXE был сохранен во время инсталляции.

#### Синтаксис команд

Eplan.exe Операция /Параметр1:Значение1 /Параметр2:Значение2 /Параметр3:Значение3

При вводе команды, действия и параметра они разделяются пробелами. Для каждого вызова командной строки всегда возможно только одно исполняемое действие. Все параметры в командной строке, которые следуют за именем действия и начинаются с "/" или "-", передаются действию как параметры.

EPLAN предоставляет вам несколько ***автоматических действий***, с помощью которых Вы можете выполнять полные задачи через командную строку без необходимости просмотра диалогового окна.

#### Общие параметры командной строки

В командной строке кроме параметров действия можно также вводить несколько общих параметров командной строки. Без этих общих параметров, как, напр., /Auto вы не сможете автоматически закончить работу EPLAN или запустить его в невидимом режиме. Общие параметры следует указывать перед именем действия.

Eplan.exe /Общ Параметры Операция /Параметр1:Значение1

!!! note "Замечание:"

    При вводе параметров командной строки учитывайте регистр. Написание имени операции и параметров должно соответствовать приведенному здесь написанию.

!!! example "Пример:"

    При помощи автоматического действия по имениprintВы можете распечатать страницы проекта или весь проект. Возможные параметры данного действия — это, напр.:/TYPE:PAGES(= тип команды печати; здесь: печатать страницы)/PROJECTNAME(= имя проекта с полным путем файла)/PAGENAME(= распечатываемая страница).При вводе следующих данных посредством команды WindowsВыполнитьдля приложения EPLAN "EPLAN Electric P8" печатается страница (=GB1+A1&EFS1/1) проектаESS_Sample_Project:"C:\Программы\EPLAN\Electric P8\2.7.0\BIN\Eplan.exe" /Variant:"Electric P8" /NoSplash /Frame:0 /Auto /Quiet print /TYPE:PAGES/PROJECTNAME: "C:\Документы и настройки\Все пользователи\Данные приложения\EPLAN\Electric P8\Проекты\Мой идентификатор фирмы\ESS_Sample_Project.elk" /PAGENAME:=GB1+A1&EFS1/1При этом вследствие использования общих параметров командной строки/NoSplash,/Frame:0,/Autoи/QuietEPLAN запускается в невидимом режиме и после выполнения команды печати снова завершает работу.

Если при выполнении командной строки возникает ошибка, она регистрируется в системных сообщениях (в меню Сервисные программы > Системные сообщения).

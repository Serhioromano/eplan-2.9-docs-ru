# Диалоговое окно Настройки: Каталоги

* Параметры > Настройки > Пользователь > Управление > Каталоги.
* Сервисные программы > Основные данные > Синхронизировать основные данные системы. В поле Каталоги щелкните по ++...++.
* Сервисные программы > Основные данные > Синхронизировать текущий проект. В поле Каталоги щелкните по ++...++.

В этом диалоговом окне задаются или обрабатываются предварительные настройки для каталогов при выборе и обработке основных данных и пути к необходимым базам данных и файлам. Эта информация учитывается частями программы и функциями, когда подается запрос на определенные данные. Настройки относятся к пользователю, так что вы можете управлять и организовывать свой собственный пул основных данных.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

В полях ввода Проекты, Шаблоны, Символы и т. д. указываются дисковод и каталог, в котором EPLAN по умолчанию должен управлять своими проектами, шаблонами, символами и т. д. Щелкните кнопку ++...++, чтобы выполнить интерактивный выбор места хранения в открывшемся диалоговом окне. Пример предварительной настройки после установки EPLAN: <Дисковод_EPLAN>:\<Каталог программы>\Проекты\<Идентификатор фирмы>.

!!! note "Замечание:"

    * Во всех полях ввода, в которых можно определять каталоги, действительно следующее:
    С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.
    Следите за тем, чтобы в этом месте предлагалось меньше переменных пути, так как в диалоговом окне Настройки: Каталоги вы определяете значения переменных пути MD_PROJECTS, MD_MACROS и т. д.!
    * Во всех диалоговых окнах выбора файлов, в которых вы выбираете эти данные (например, в диалоговом окне Выбрать макрос), вы можете путем выбора всплывающего меню Сбросить на стандарт в поле Путь настроенные здесь ранее каталоги.

# Диалоговое окно Экспортировать данные изготовления / вывести маркировку

Вы открыли проект. Сервисные программы > Данные изготовления > Экспорт / маркировка.

В этом диалоговом окне определяется, в каком объеме и в какой последовательности должны выводиться данные изготовления и маркировки в файле.

Обзор основных элементов диалогового окна:

### Настройки

В данном раскрывающемся списке отображаются все пользовательские и предварительно определенные схемы экспорта / маркировки. Для обработки или создания схемы щелкните по кнопке ++...++.

### Тип отчета

Схема для экспортированных данных изготовления / маркировки всегда создается вместе с типом отчета. В этом поле выводится тип отчета, который был выбран для текущей схемы.

Обратите внимание на то, что значения изделия (например, тип отчета "Групповая спецификация изделий") зависят от текущего выбора. Это означает следующее:

* Если выбраны клеммы, то выводятся изделия клемм.
* Если выбраны клеммники, то выводятся все изделия клемм и клеммников.
* Если выбраны клеммники и отдельные клеммы, то выводятся все изделия клемм и клеммников, причем изделия клемм анализируются только один раз.

То же самое действует для кабеля и соединений.

Все изделия выводятся в том случае, когда установлена соответствующая настройка (например, настройки Изделие клеммы, Изделие, жила кабеля и т. д.).

### Язык

Выберите из раскрывающегося списка язык, на котором должен быть выведен файл экспорта / маркировки. Доступны все языки, которые определены как языки перевода в настройках перевода соответствующего проекта.

* Все языки отображения: Обеспечивает вывод данных EPLAN на нескольких языках. Данные выводятся на всех языках, которые определены как языки отображения в настройках перевода соответствующего проекта.
* <Язык>: Обеспечивает вывод данных EPLAN на выбранном языке. Если вы хотите вывести данные во внешний файл Excel, то можете направить названия таблиц в файл Excel. Название таблицы должно быть аналогично коду языка. Если файл не содержит таблицу с подходящими кодом языка, то выполняется последовательный поиск таблицы, подходящей для языка диалоговых окон или альтернативного языка. Если для этого также нет никаких подходящих таблиц, то используется первая таблица из шаблона.

!!! example "Пример:"

    В качестве шаблона вы выбрали файл Excel, 5 таблиц с названиями таблиц, содержащими en_US, de_DE, fr_FR, ru_RU и zh_CN.В поле Языквы выбрали языкen_US.Данные EPLAN будут выведены в таблице Excelen_US.

### Целевой файл

В этом поле выводится путь к файлу экспорта / маркировки.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Повторений на этикетку

Число в этом поле указывает на то, сколько раз должна повторяться каждая этикетка в файле экспорта / маркировки.

### Повторений на полный вывод

Число в этом поле указывает на то, сколько раз должны повторяться все выведенные данные в файле экспорта / маркировки.

### Вид вывода

В этом групповом поле указывается, должно ли запускаться приложение после вывода данных.

* Экспортировать: Если выбран этот параметр, выполняется вывод данных, но внешнее приложение не запускается.
* Экспортировать и запустить приложение: Если выбран этот параметр, выполняется вывод данных и запускается внешнее приложение, например Notepad.

# Произвести настройки для просмотра образов контактов

Настройки образа контакта могут задаваться как для всего проекта, так и устанавливаться для каждого устройства индивидуально. При этом данная настройка условного обозначения имеет ***преимущество*** над настройкой проекта.

### Настроить специфический для проекта образ контакта

Условие:

Вы открыли проект.

1. Выберите пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Перекр. ссылки / образы контактов.
2. Выберите область настройки Общее.
3. В групповом поле Таблица образов контактов введите символы для маркировки таблицы образов контактов.
4. Кроме того, в этом групповом поле введите значения для Ширины и Высоты таблицы образов контактов.
5. Выберите область настройки Образ контакта у усл. обозначения.
6. Определите здесь предварительные настройки, учитываемые при отображении образов контакта на условных обозначениях.
7. Установите флажок Отображать в форме таблицы, если число страниц и обозначение столбцов в перекрестной ссылке нужно ввести в виде таблицы; заголовки таблиц для размыкающих и замыкающих контактов при этом извлекаются из настройки Таблица образов контактов: Маркировка (область настройки Общее).
8. Установите флажок Табличный крест с переменной длиной, если табличный крест должен автоматически быть адаптирован к максимальной длине записей образа контакта; в таком случае настройка Таблица образов контактов: Ширина / высота (область настройки Общее) ***не*** учитывается.
9. Выберите в поле Поворот требуемое значение, если образ контакта должен отображаться под углом, отличающимся от установленного по умолчанию.
10. Если номер изделия и / или номер типа в образе контакта условного обозначения должен отображаться на разных позициях, выберите из раскрывающихся списков Отображение 1-го / 2-го номера изделия и Отображение 1-го /2-го типа изделия соответствующие записи.
11. Выберите область настройки Образ контакта в зоне.
12. Определите здесь предварительные настройки, учитываемые при отображении образов контакта в зоне условного обозначения. (Возможности ввода - те же самые, что и для образа контакта на условном обозначении.)
13. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Настройки будут скопированы.

### Настроить образ контакта у условного обозначения

Условия:

* Вы открыли проект и выбрали страницу схемы соединений для обработки.
* Двойным щелчком вы открыли диалоговое окно свойств условного обозначения, для которого вы можете определить образ контакта (например, общее устройства, кабель или экранирование).

1. На вкладке Отображение диалогового окна 'Свойства' под порядком свойств выберите вкладку Образ контакта.
2. Щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
3. В диалоговом окне Генерировать образ контакта выберите выравнивание образа контакта "В зоне" или "На условном обозначении".
4. На вкладке Образ контакта выделите выбранное выравнивание образа контакта.
5. В таблице Свойство / присвоение на уровне иерархии Позиция установите флажок Выровнять автоматически, чтобы EPLAN автоматически позиционировал образ контакта вертикально под соответствующей главной функцией.
6. Снимите флажок, если вы хотите самостоятельно задать позицию образа контакта, и введите в полях Координата X и Координата Y значения, на которые должен переместиться образ контакта.
7. На уровне иерархии Настройки установите флажок Определено пользователем, если настройки ***не*** требуется копировать из настроек проекта.
8. В разных настройках этого уровня иерархии укажите, как должен отображаться образ контакта для текущего выбранного устройства.
9. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Образ контакта размещается согласно настройкам на условном обозначении.

!!! note "Замечание:"

    Настройку, связанную с условными обозначениями, можно изменить в любое время, дважды щелкнув по устройству, а затем с помощью ![](images/all_edit_as.png){: .ui-icon } (Обработать) в диалоговом окне Генерировать образ контакта выбрав другое выравнивание образа контакта, изменив настройки для вкладки Образ контакта в таблице Свойство / присвоение или убрав образ контакта с помощью ![](images/all_delete_as.png){: .ui-icon } (Удалить).

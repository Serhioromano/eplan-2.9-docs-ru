# Создание перемычек при распределенных клеммах

Из всех соединений при обновлении соединений между распределенными клеммами генерируются соответствующие внутренние, мостовые и проволочные перемычки. Между родственными распределенными клеммами (одинаковое определение функции, одинаковое ОУ, одинаковое обозначение клемм, одинаковый номер уровня) генерируются внутренние перемычки, между неродственными распределенными клеммами — мостовые или проволочные перемычки.

### Автоматически генерируемые внутренние перемычки

Если установлена настройка проекта Автоматически связать родственные распределенные клеммы, то распределенные клеммы функции клеммы автоматически связываются, даже если между ними нет обозначенных соединений. Для этого при обновлении соединений между родственными распределенными клеммами генерируются внутренние перемычки. Данные автоматически сгенерированные внутренние перемычки соответственно располагаются на последнем выводе устройства функции клеммы. Для генерирования внутренних перемычек решающее значение имеет последовательность распределенных клемм в навигаторе клеммников.

Все распределенные клеммы с одинаковым ОУ, одинаковым (не пустым) обозначением клемм и отличающимся обозначением вывода устройства логически рассматриваются как клемма, все выводы устройства которой соединены друг под другом.

### Оптимизация перемычек

Графически обозначенные соединения между распределенными клеммами не всегда соответствуют логически необходимым соединениям для создания перемычек. Например, распределенные клеммы могут многократно соединяться друг с другом графически. Если настройка проекта Оптимизация перемычек активна, то независимо от обозначенных соединений на месте проволочных перемычек по возможности генерируются мостовые перемычки. Также заново генерируются внутренние перемычки. Из графических соединений отображаются только распределенные клеммы, которые соединены друг под другом, но не видно, между какими отдельными клеммами имеются перемычки.

Мостовые перемычки, возникающие при оптимизации перемычек, всегда располагаются на первой распределенной клемме (с наименьшим обозначением вывода устройства) от последнего к предпоследнему выводу устройства клеммы. Тип вывода устройства, настроенный для последнего вывода устройства клеммы, определяет вид возникающей перемычки / соединения. Обычно последний вывод устройства должен иметь тип вывода устройства 'Мостовая перемычка'.

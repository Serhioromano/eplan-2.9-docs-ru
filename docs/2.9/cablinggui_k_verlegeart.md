# Вид прокладки

Для сегментов маршрутизации топологии можно присвоить определенный тип сегмента маршрутизации. Тип сегмента маршрутизации указывает на то, каким образом должна быть выполнена маршрутизация соединений или кабелей в данном сегменте маршрутизации (например, труба, кабельный канал, кабельный лоток и т. д.). При маршрутизации соединений типы сегментов маршрутизации, которые предварительно определены на сегментах маршрутизации, назначаются для проходящих кабелей и соединений и переносятся на них.

При выводе групповой спецификации изделий может выполняться анализ функций по типам сегментов маршрутизации, чтобы рассчитать время и затраты на маршрутизацию.

### Ввод типа сегмента маршрутизации

Для ввода типа сегмента маршрутизации на сегментах маршрутизации топологии доступно свойство Топология: Тип сегмента маршрутизации (сегмент маршрутизации) (Ид. 20345). Это свойство указывает на то, каким образом должна быть выполнена маршрутизация соединений или кабелей в данном сегменте маршрутизации (например, труба, кабельный канал, кабельный лоток и т. д.). Из этого следует тип сегмента маршрутизации соединений / кабелей, которые маршрутизированы в данном сегменте маршрутизации.

* В диалоговом окне "Свойства" сегмента маршрутизации свойство можно обработать в форме списка выбора — при этом доступен многоязычный ввод. Если на сегменте маршрутизации введен определенный тип сегмента маршрутизации, его затем можно будет выбрать на других сегментах маршрутизации.
* На сегмент маршрутизации топологии допускается только один тип сегмента маршрутизации.
* Это свойство доступно в навигаторе топологии в представлении в виде списка, а также в качестве критерия фильтра.
* Данное свойство можно выводить в отчетах топологии.

### Вывод типа сегмента маршрутизации

Для кабелей и соединений доступно свойство Топология: Типы сегмента маршрутизации (Ид. 20344). В этом свойстве выводятся типы сегментов маршрутизации, через которые проходит соединение, маршрутизированное в сети соединенных сегментов, или маршрутизированный кабель. Типы сегментов маршрутизации перечислены согласно последовательности трасс маршрутизации и разделяются точкой с запятой.

* Свойство может отображаться в графическом редакторе на функции.
* Это свойство доступно в навигаторе кабелей и соединений в представлении в виде списка.
* Данное свойство можно выводить в отчетах по кабелям и в таблицах соединений.

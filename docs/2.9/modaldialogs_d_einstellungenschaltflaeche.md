# Диалоговое окно Конфигурировать кнопку

Вы щелкнули правой клавишей мыши на свободной области строки меню или панели инструментов и выбрали пункт всплывающего меню Настроить.

* В панель инструментов добавлена кнопка, речь идет об ***операции*** или о ***меню*** без фиксированного присвоения графического файла.
* Вы выбрали на панели инструментов пункт всплывающего меню Обработать для кнопки, которая связана с ***операцией*** или ***меню*** без фиксированного присвоения графического файла, или дважды щелкнули на кнопку.

В этом диалоговом окне определяются настройки кнопки, которая связана с операцией или пунктом меню без фиксированного присвоения графического файла.

Обзор основных элементов диалогового окна:

### Всплывающая подсказка

В этом поле можно ввести описание кнопки для выбранной ***операции***. Это описание позже будет выводиться как небольшое текстовое окно, если переместить курсор на кнопку.

Это поле невозможно обработать для ***меню*** без фиксированного присвоенного графического файла.

### Командная строка

Операции управляются с помощью командной строки. В этом поле выводится пример записи для выбранной ***операции***. Настройте его в соответствии с собственными желаниями. Знак вопроса "?" в командной строке является заполнителем, который можно заменить.

Это поле невозможно обработать для ***меню*** без фиксированного присвоенного графического файла.

!!! example "Пример:"

    Текст в командной строке для операции "Вставить символ":XEGActionInsertSymRef /SymbolLibName:? /SymbolId:? /VariantId:0Вы открыли страницу. При щелчке по соответствующей кнопке вызывается диалоговое окно Выбор символа.Заполнитель "?" в командной строке заменяется конкретными значениями:XEGActionInsertSymRef /SymbolLibName:IEC_Symbol /SymbolId:30 /VariantId:0При щелчке по соответствующей кнопке символ "клемма" избиблиотеки символовIEC_Symbolдобавляется к курсору и его можно разместить на странице.

### Рисунок

В этом поле необходимо выбрать графический файл, который нужно присвоить кнопке. С помощью кнопки ++Добавить++ / [Заменить] можно перейти в диалоговое окно выбора и выбрать там требуемый графический файл / заменить текущий выбранный графический файл. При этом в качестве формата графических файлов могут использоваться только JPG / JPEG и BMP. Чтобы графические файлы точно соответствовали кнопкам, их размер должен составлять 16x16 пикселов. Если указывается недействительный графический файл, для кнопки будет использован стандартный графический файл.

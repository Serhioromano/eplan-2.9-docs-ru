# Вставить прерывания для трубопроводов

Условия:

* Вы открыли проект, который содержит данные предварительного планирования.
* Открыта страница типа "Функциональная схема автоматизации".
* Вы определили трубопровод на функциональной схеме автоматизации с помощью точки определения трубопровода.

1. Выберите следующие пункты меню: Вставить > Символ
2. В библиотеке символов SPECIAL выберите символ PBR для прерывания трубопровода.
3. В предварительном просмотре выделите вариант символа прерывания, который подходит для вертикального или горизонтального участка трубопровода.

4. Для требуемого трубопровода щелкните по позициям, между которыми нужно вставить прерывание.

!!! info "Для сведения:"

    Трубопровод между этими двумя позициями не удалится, а будет только прерван графически.

5. Выполните аналогичную операцию для всех остальных трубопроводов, которые нужно прервать.
6. Завершите операцию, выбрав пункт всплывающего меню Прервать операцию или нажав клавишу ++Esc++.

# Диалоговое окно "Настройки: Перенос изделия"

Параметры > Настройки > Пользователь > Устройство > Перенос изделия

В этом диалоговом окне можно указать, какие вспомогательные элементы следует перенести при переносе изделия:

* Сегмент маршрутизации
* Кривая
* Область маршрутизации
* Соединительное отверстие для проводов
* Запретная зона для размещения
* Запретная зона для сверления
* Линия.

!!! note "Замечание:"

    Следующие типы объектов, как правило, не переносятся:

* Монтажная поверхность
* Точка захвата
* Точка монтажа
* Линия монтажа
* Монтажная сетка
* Поверхность установки
* Исходная точка.

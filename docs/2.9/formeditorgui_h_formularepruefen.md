# Проверить формы

Чтобы во время обработки форм можно было в любое время проверить, верна ли Ваша форма, или содержит ошибки, существует специальная функция.

Условия:

* Вы открыли проект.
* В редакторе форм открыта та форма, которую требуется обработать (Сервисные программы > Основные данные > Форма > Открыть > [Открыть]).

1. Сервисные программы > Проверить форму
2. Подтвердите сообщение в диалоговом окне Проверить форму.
3. В диалоговом окне Системные сообщения щелкните кнопку ++Сохранить++.
4. Установите в диалоговом окне Открыть дисковод и каталог, в котором должен быть сохранен файл журнала.
5. Введите в поле Имя файла информативное имя для файла журнала.
6. Щелкните по кнопке ++Открыть++.
7. В диалоговом окне Системные сообщения щелкните кнопку ++Закрыть++.

!!! info "Для сведения:"

    Системные сообщения, которые относятся к текущей форме, сохраняются в соответствующем файле и при необходимости могут быть распечатаны.

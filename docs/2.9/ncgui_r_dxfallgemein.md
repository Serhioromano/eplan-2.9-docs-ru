# Вкладка DXF общ. (экспорт ЧУ в формате DXF / экспорт меди в формате DXF)

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт ЧУ DXF. Выберите вкладку Общ..
* Сервисные программы > Данные изготовления > Механическая обработка > ЧУ DXF. Рядом с полем Машина нажмите ++...++. Выберите вкладку Общ..

На этой вкладке установите общие настройки для экспорта ЧУ в используемом на многих машинах формате ***DXF***.

Обзор основных элементов диалогового окна:

Групповое поле 'Формат'

### Версия AutoCAD

Здесь при помощи версии AutoCAD необходимо выбрать формат, в котором должны экспортироваться данные DXF:

* AutoCAD 2013/2014/2015/2016/2017
* AutoCAD 2018/2019.

### Код. страница

Настройка используемого набора символов. По умолчанию настроен набор символов CP_ANSI_1252.

* * *

### Каталог вывода

В этом поле укажите место хранения для генерируемого файла DXF. В качестве целевых для экспорта данных можно указать любой дисковод и подкаталог.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Генерировать имена файлов

В этом раскрывающемся списке определяется стандартный порядок присвоения имен файлам экспорта DXF.

* Из непрерывного номера: При выборе этого параметра EPLAN по умолчанию присваивает файлам DXF имена по образцу <непрерывный_номер>.DXF. Если уже имеются файлы DXF, EPLAN начнет нумерацию со следующего доступного числа.
* Из обозначения функц. элемента: При выборе этого параметра EPLAN по умолчанию присваивает файлам DXF имена по образцу <Обозначение функц. элемента>.DXF.
* Из свойств: Если выбран этот параметр, имя файла составляется из свойств пространства листа, проекта и устройства, а также свободно определяемого разделителя. Используя кнопку ++...++, откройте диалоговое окно Формат: Свойство для конфигурации имени файла.

### Генерировать подкаталоги

В этом раскрывающемся списке определяется стандартный порядок создания подкаталогов при экспорте файлов DXF.

* Нет: Если выбран данный параметр, при экспорте подкаталоги не генерируются.
* Из структуры проекта: Если выбран данный параметр, то при экспорте автоматически генерируются подкаталоги, отражающие структуру проекта.
* Из свойств: Если выбран этот параметр, имя подкаталога составляется из свойств пространства листа, проекта и устройства, а также свободно определяемого разделителя. Используя кнопку ++...++, откройте диалоговое окно Формат: Свойство для конфигурации имени файла. Используйте здесь, например, элемент формата "Разделитель каталога", чтобы сгенерировать структуру каталога с несколькими расположенными друг под другом подкаталогами.

# Создать и удалить формат нумерации

Если Вы не хотите изменять имеющийся формат нумерации, можно настроить новый формат нумерации.

Условия:

* Выберите устройства для нумерации в навигаторе страниц, навигаторе устройств или в графическом редакторе.
* Вы открыли диалоговое окно Нумеровать (Данные проекта > Устройство > Нумеровать).

### Создание формата нумерации

1. Выберите следующие пункты меню: [Настройки] > Нумерация
2. В диалоговом окне Настройки: Нумеровать (автономно) нажмите кнопку ![](images/all_new_as.png){: .ui-icon } (Создать).
3. Введите в диалоговом окне Новая схема в поле Имя требуемое имя, а в поле Описание \- описание формата нумерации.
4. Щелкните по кнопке ++OK++.
5. Выделите в поле Доступные элементы формата элемент формата для "Страница", "Столбец" или "Строка" и щелкните по кнопке ![](images/all_arrowright_as.png){: .ui-icon } (переместить вправо).
6. Укажите в диалоговом окне <Формат> в поле Мин. число разрядов, сколько разрядов должен иметь выбранный элемент формата, и нажмите кнопку ++OK++.
7. Для определения элементов формата выберите в поле Выбранные элементы формата элемент и нажимайте на ![](images/all_arrowup_as.png){: .ui-icon } (Переместить вверх) или ![](images/all_arrowdown_as.png){: .ui-icon } (Переместить вниз) до тех пор, пока элемент не будет установлен в нужную позицию.

### Удаление формата нумерации

1. Выберите в диалоговом окне Настройки: Нумеровать (автономно) в поле Схема формат, который требуется удалить.
2. Щелкните по кнопке ![](images/all_delete_as.png){: .ui-icon } (Удалить).

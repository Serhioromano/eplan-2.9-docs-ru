# Присвоить данные функции условному обозначению

При присвоении из навигатора устройств можно переместить функцию или шаблон функций на условное обозначение и перенести данные функции в условное обозначение. Присвоенная функция должна содержать такое же количество выводов, что и условное обозначение (или больше). Данные действия можно также выполнить в другом навигаторе, например навигаторе клеммника или навигаторе точки разрыва. При присвоении проверяется, принадлежит ли исходная функция в навигаторе и целевая функция в схеме соединений одной группе функций. В ином случае отобразится соответствующее сообщение.

Условия:

* Вы открыли проект. В Графическом редакторе открыта страница проекта.
* Вы разместили условное обозначение на странице проекта.

1. Выберите следующие пункты меню: Данные проекта > Устройство > Навигатор
2. В диалоговом окне Устройство <Имя проекта> выделите одну или несколько функций.
3. Выберите пункт всплывающего меню Присвоить.

!!! info "Для сведения:"

    Описание (первой) выделенной функции появится рядом с курсором.

4. Наведите курсор на условное обозначение, чтобы изменился цвет условного обозначения.
5. Дважды щелкните по условному обозначению.

!!! info "Для сведения:"

    Данные функции будут перенесены в условное обозначение. Соответствующая функция условного обозначения заново сортируется навигатором устройств.

!!! info "Для сведения:"

    Описание функции следующей функции появится рядом с курсором.

!!! tip "Совет:"

    * При помощи комбинации клавиш [Ctrl]+[Shift] можно также перетащить из навигатора устройств и присвоить функцию. Если во время перетаскивания отпустить левую кнопку мыши, описание выделенной функции прикрепляется к курсору. Если передвинуть курсор на подходящее условное обозначение, данные функции можно будет перенести в условное обозначение щелчком мыши.
    * Одновременное присвоение нескольких функций / шаблонов функций, выделенных в навигаторе, возможно через многократный выбор целевых функций. Для этого сначала выберите пункт всплывающего меню Присвоить, далее щелкните по схеме соединений и растяните рамку вокруг требуемого условного обозначения. Функции / шаблоны функций присваиваются в том порядке, в котором они отображены в навигаторе. На схеме соединений выполняется поиск подходящих целевых функций согласно их графической последовательности.

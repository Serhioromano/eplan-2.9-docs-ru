# Убрать перевод из проекта

При удалении из проекта перевода удаляются тексты перевода на один или несколько языков перевода. При этом удаляются также тексты перевода, которые не отображаются в проекте.

Условия:

* Вы открыли проект.
* Проект содержит тексты перевода.

1. Выберите следующие пункты меню: Страница > Навигатор
2. В диалоговом окне навигатора Страницы — <Имя проекта> выделите проект, из которого необходимо убрать тексты перевода.
3. Выберите пункты меню Сервисные программы > Перевод > Убрать перевод из проекта.
4. Нажмите клавишу ++Ctrl++ и выделите в диалоговом окне Выбрать языки все языки перевода, тексты перевода на которые необходимо убрать.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Тексты перевода удаляются. А выбранные языки больше не являются языками перевода.

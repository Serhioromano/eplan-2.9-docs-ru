# Диалоговое окно Спецификация — <Имя проекта>

Вы открыли проект. Данные проекта > Устройство / изделие > Навигатор спецификаций.

В этом диалоговом окне перечисляются все данные изделия открытого проекта, и их можно обрабатывать.

Обзор основных элементов диалогового окна:

### Список

Представление спецификации в виде списка ориентировано на устройство. Обратите внимание, что, используя пункты меню Всплывающее меню > Конфигурировать представление, в любой момент можно выбрать, какая информация должна быть выведена (например, номер изделия, ОУ, число штук и т. д.).

### Дерево

В отличие от представления в виде списка представление в виде дерева ориентировано на изделия, т. е. здесь по умолчанию выводятся только такие ОУ, которые уже имеют изделие. С помощью пункта всплывающего меню Вид можно выбрать различные виды для отображения.

Если активировано диалоговое окно Предв. просмотр графики и изделию присвоено фото продукта, то оно отображается.

### Фильтр

В этом раскрывающемся списке отображаются все доступные фильтры. Выбранный фильтр активируется автоматически и применяется как к дереву, так и к списку. Запись "- Не активировано -" отключает фильтр и приводит к тому, что данные отображаются в неотфильтрованном виде. С помощью кнопки ++...++ откройте диалоговое окно [Фильтр](modaldialogsdb_d_filternnach.md). Здесь можно создать, обработать, удалить, скопировать, экспортировать, импортировать фильтр и управлять им.

Во всплывающем меню раскрывающегося списка Фильтр содержатся следующие записи:

* Выключить: Этот пункт меню доступен, если фильтр установлен: Сбрасывает настройку фильтра до записи "- Не активировано -".
* Активировать <Имя фильтра>: Этот пункт меню доступен, если для настройки фильтра установлено значение "- Не активировано -": Повторно активирует последний активный фильтр.

Таким образом можно быстро переключаться между неотфильтрованным и отфильтрованным в соответствии с требованиями пользователя представлениями.

### Значение: <Свойство>

При помощи [быстрого ввода](modaldialogsdb_k_filter.md) в данном поле для определенного и активированного фильтра можно быстро изменить значение его критерия.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Добавить покупные изделия |  Добавляет к текущему проекту изделия, не присвоенные специальному ОУ, напр. комплектующие детали. Для этого EPLAN откроет диалоговое окно Выбор изделия, в котором можно выбрать требуемые изделия, причем новые изделия проекта всегда сортируются в дереве в алфавитном порядке. Возможен многократный выбор.
Добавить изделие |  Присваивает текущему устройству одно (или несколько) другое(ие) изделие(я). Новые изделия в данных устройства добавляются к уже имеющимся изделиям. Для этого EPLAN откроет диалоговое окно Выбор изделия, в котором можно выбрать изделия.
Удалить |  Удаляет выбранные изделия после запроса. Изделия, для которых существует размещение, удалить нельзя.
Вставить устройство |  Вставляет выделенное устройство. Непосредственно перед размещением нажмите клавишу ++Backspace++, чтобы открыть диалоговое окно Вставить устройство и, например, Выбор макросов (при необходимости) или чтобы изменить вид представления.
Заменить изделие |  Позволяет заменить для выбранных устройств присвоенные изделия на новое изделие. В открывшемся диалоговом окне Выбор изделия множественный выбор ***невозможен***. При замене изделий, в отличие от выбора изделия, количество изделий сохраняется. При этом узлы сохраняются.
Обработать изделие |  Открывает диалоговое окно Обработать изделие, в котором можно обрабатывать выбранные изделия, т. е. присваивать номер изделия, число штук и свойства или переписывать уже присвоенные данные.
Выбор устройства |  Открывает диалоговое окно Выбор устройства для выделенных изделий. При этом всегда учитываются ***все*** изделия устройства, то есть даже невыбранные.
Многократный выбор изделий возможен, однако для точек определения изделия и покупных изделий этот пункт меню ***не***доступен.
Обновить данные устройства |  Переносит данные устройства выбранных изделий на функции в проекте, использующие эти изделия. Это представляет интерес, если вы выбрали изделие в списке для выбора изделий или устройств, имеющее более старый статус в проекте, чем в базе данных изделий. Дополнительные сведения по этой теме см. в разделе "[Обновление данных устройства](partselectiongui_k_prinzip.md)".
Задать номер позиций |  Открывает диалоговое окно Нумеровать позиции, в котором можно указывать значения новой нумерации выделенных текущих или всех позиций проекта.
Синхронизировать данные изделия |  Выполняет синхронизацию изделий основных данных с изделиями проекта. При этом использованные в проекте изделия, отсутствующие в базе данных изделий, генерируются в базе данных изделий.
Обработать маркеры ревизий |  Посредством данного пункта меню Вы можете обрабатывать тексты маркировки ревизий и их формат. Пункт меню доступен только в том случае, если в ревизии выделен измененный объект.
Удалить маркер ревизии |  Через этот пункт меню можно удалить тексты маркеров ревизий. Пункт меню доступен только в том случае, если в ревизии выделен измененный объект.
Экспортировать |  Экспорт и импорт можно осуществлять только в проекте! Открывает диалоговое окно Экспортировать спецификацию, в котором устанавливается, в какие файлы и в каком формате должен осуществляться экспорт.
Импортировать |  Экспорт и импорт можно осуществлять только в проекте! Открывает диалоговое окно Импортировать спецификацию, в котором устанавливается, какой файл должен импортироваться.
Перейти к (графика) |  Открывает предварительный просмотр графики страницы и показывает текущее ОУ выделением; двойным щелчком в пределах этого предварительного просмотра может открыться диалоговое окно Размещение изделия или диалоговое окно Свойства, специфическое для функционального элемента.
Вставить в список результатов поиска |  Заносит все объекты проекта, имеющие ссылку на выделенный элемент, в список результатов поиска.
Настройки |  Открывает диалоговое окно [Настройки: Изделие](partslistgui_d_paramartikelauswertung.md), в котором можно установить несколько настроек относительно обработки определенных типов изделий. Эти настройки влияют на данные, отображенные в навигаторе спецификаций, и частично на отчеты "Список обозначений устройств", "Спецификация изделий", "Групповая спецификация изделий" и "Список производителей / поставщиков".
Список с предварительным выбором (только дерево) |  Уменьшает число отображаемых элементов, чтобы ускорить ориентирование в представлении в виде списка. Если этот параметр активирован, представление в виде списка вызывается с автоматическим фильтром (предварительный выбор), причем этот фильтр содержит только что выбранные элементы.
Вид (только дерево) |  Переключает отображение в представление структуры дерева и таким образом задает последовательность уровней иерархии:

* Номер изделия: Отображаются номер изделия, буквенное обозначение и полное ОУ.
* Номер изделия / структурный идентификатор: Отображаются номер изделия, структурный идентификатор и краткое обозначение ОУ.
* Производитель: Отображаются производитель, номер изделия, буквенное обозначение и полное ОУ.
* Поставщик: Отображаются поставщик, номер изделия, буквенное обозначение и полное ОУ.
* Группирование спецификаций: Отображаются группа спецификаций, номер изделия, структурный идентификатор и краткое обозначение ОУ.

Выбрать в дереве (только список) |  Показывает выделенный объект во вкладке Дерево.
Свойства |  Открывает специфическое для функциональных элементов диалоговое окно свойств данных устройства, в котором можно обработать выделенные элементы или — для размещенных функциональных элементов, которым присвоены элементы — диалоговое окно Размещение изделия.
Свойства (общие) |  Открывает диалоговое окно Свойства (общие): ++...++. Позволяет обрабатывать свойства устройства.

# Фильтровать обновленные измененные страницы отчета

При обновлении дата изменения отчетов меняется, только если изменился отчет. Отчет меняется, например, при изменении данных проекта или формы. При обновлении всех отчетов посредством функции Генерация отчета по проекту с помощью свойства Дата изменения можно отфильтровать фактически изменившиеся страницы отчетов.

Условия:

* Вы открыли проект.
* Отчеты обновлены.

1. Выберите следующие пункты меню: Страница > Навигатор
2. В навигаторе страниц рядом с полем Фильтр нажмите ++...++.
3. В диалоговом окне Фильтр рядом с полем Схема нажмите кнопку ![](images/all_new_as.png){: .ui-icon } (Создать) и в открывшемся диалоговом окне создайте схему.
4. Нажмите кнопку ![](images/all_new_as.png){: .ui-icon } (Создать) на панели инструментов для критериев фильтра.
5. В диалоговом окне Выбор критериев под уровнем иерархии "Свойства" выберите свойство Дата изменения (автоматич.).
6. В диалоговом окне Фильтр в столбце Оператор выберите запись >=.
7. Дважды щелкните в столбце Значение и выберите дату в качестве даты изменения.
8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Имя новой схемы фильтра автоматически вносится в список Фильтр и активируется.

!!! info "Для сведения:"

    EPLAN отображает все страницы отчетов, измененные к выбранной дате и после нее.

!!! info "Для сведения:"

    Скрыты все страницы отчетов, которые с даты изменения и точно до момента выбранной даты изменения ***не*** изменялись.

# Изменить свойства проекта посредством объектов-заполнителей

Свойства проекта теперь можно изменить также посредством объектов-заполнителей. Ввиду этого теперь объектом, относящимся к объекту-заполнителю. является не только страница, на которой находится объект-заполнитель, но и сам проект.

![](images/benefit.png)Эффект:

Функцию изменения свойств проекта с помощью объектов-заполнителей можно, например, использовать в EPLAN Cogineer, чтобы перезаписать данные лежащего в основе базового проекта в возникающем целевом проекте (например, комиссионный номер).

Поэтому в диалоговом окне 'Свойства' на вкладке Присвоение проект доступен как самый верхний узел в таблице свойств. При выборе свойств для этого объекта отображаются только изменяемые свойства проекта, а после выбора они отображаются в форме структуры дерева.

В диалоговом окне Присвоить набор значений в рамках этого расширения настройка для отображения и присвоения свойств страниц переименована:

Старое обозначение |  Новое обозначение
---|---
Отображать переменные для свойств страниц |  Отобразить переменные для свойств проекта и страницы

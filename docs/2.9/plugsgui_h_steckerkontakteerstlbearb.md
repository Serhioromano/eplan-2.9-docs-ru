# Создать и обработать контакты штекера

Контакты штекера можно создавать в графическом редакторе, где вы его размещаете и задаете свойства в соответствующем диалоговом окне. У вас есть возможность в навигаторе штекеров или в навигаторе устройств предопределять контакты штекера с заданными свойствами, которые вы можете использовать позже.

Если вы создаете контакт штекера, определите, какой это будет контакт: штырь, гнездо, или комбинация штырей и гнезд, а затем отредактируйте свойства. Кроме того, с помощью определений функций вы можете создать контакт штекера с предопределенными свойствами.

Во вкладке [Контакт штекера](devicetaggui_r_steckerkontakt.md) в диалоговом окне, описывающем свойства, отредактируйте свойства штекера.

### Создать контакт штекера

1. Выберите пункты меню Данные проекта > Штекеры > Навигатор.
2. Выделите определение штекера или контакт штекера под определением штекера.
3. Выберите пункт всплывающего меню Генерировать контакт штекера и затем выберите одно из значений: Штырь и гнездо, Штырь или Гнездо.
4. Введите в диалоговом окне Свойства ++...++ на вкладке Контакт штекера нужные данные для нового контакта штекера.
5. Щелкните по кнопке ++OK++.

!!! tip "Совет:"

    Используя пункт всплывающего меню Новые функции, вы можете одновременно создавать несколько контактов штекера.

### Создать контакты штекера с предопределенными свойствами.

1. Выберите пункты меню Данные проекта > Штекеры > Навигатор.
2. Выделите ОУ или определение штекера, затем выберите пункт всплывающего меню Создать.
3. В диалоговом окне Определения функций в поле Выбор в дереве найдите и выделите контакт штекера.
4. Щелкните по кнопке ++OK++.
5. Введите в диалоговом окне Свойства ++...++ на вкладке Контакт штекера (или Штырь, или Гнездо) нужные данные для нового контакта штекера.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Под выделенным ОУ будет создан контакт штекера с соответствующими свойствами.

!!! tip "Совет:"

    Контакты штекеров можно копировать и размещать при помощи функции перетаскивания: перетаскивая внутри проекта контакт штекера, выбранный в навигаторе, на страницу, открытую в графическом редакторе.

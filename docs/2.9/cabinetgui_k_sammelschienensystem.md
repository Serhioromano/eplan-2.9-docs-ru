# Системы сборных шин

Системы сборных шин можно размещать в пространстве листа из следующих источников:

* Пункты меню Вставить > Система сборных шин
* Навигатор основных данных изделий
* Навигатор спецификаций
* Пункты меню Вставить > Устройство
* Список устройств
* Макросы окон
* Data Portal.

Системы сборных шин обычно размещаются на монтажных платах, но их также можно размещать на любых других монтажных поверхностях. Выбор и способ размещения такой же, как для [функциональных элементов с переменной длиной](cabinetgui_h_varbtplatzieren.md).

Системы сборных шин, в отличие от отдельных шин, имеют дополнительные признаки.

* В системе сборных шин размещаются собственно шины и дополнительно необходимые кронштейны. Число кронштейнов можно настраивать и изменять позднее. Также возможно размещение без кронштейна.
* Уже размещенную систему сборных шин можно изменить позднее, не потеряв уже размещенные функциональные элементы.
* Кронштейнам системы сборных шин через базу данных можно присваивать схему сверления.
* При размещении системы сборных шин автоматически учитываются поставляемые длины.
* На полюсах системы сборных шин возможно прямое размещение устройств.
* Отдельные полюса системы сборных шин имеют собственный номер изделия, и их длина может изменяться по отдельности. Отдельные полюса на практике часто имеют разную длину. Это также учитывается в отчете в спецификации.
* Полюса изготавливаются отдельно. Чтобы подсоединить медные детали и кабели, на полюсах сверлятся отверстия.
* Перед размещением выполняется проверка, можно ли выполнить расчет системы сборных шин из определенных записей, имеющихся в базе данных изделий. Если необходимые для размещения свойства изделия в базе данных неполны, появляется сообщение об этом и размещение отменяется.

!!! note "Замечание:"

    Если в данных изделия системы сборных шин на вкладке Система сборных шин записан кронштейн с графическим макросом, при размещении используется сначала точка захвата, которая определена в этом макросе. Таким образом можно расположить точку захвата по центру отверстия в кронштейне. Стандартные точки захвата, которые не определены в макросе, можно выбрать в диалоговом окне Опции размещения или при помощи кнопки ++A++.

    При размещении интегрированной системы сборных шин точка захвата при вводе начальной точки располагается на левой стороне кронштейна. Для ввода конечной точки точка захвата переключается на "вправо". При использовании макроса для кронштейнов, напротив, используется точка захвата из макроса для начальной и конечной точек.

# Продолжить нумерацию после функции

Если нумерация должна быть продолжена после функций, на которых она заканчивается (в соответствии с настроенным объемом, настройте эту опцию для соответствующих функций.

Условие:

Выделите функцию в графическом редакторе.

1. Обработать > Свойства
2. В диалоговом окне Свойства ++...++ активируйте во вкладке <Категория функции> свойство "Продолжить нумерацию с данными ПЛК далее после функции" (групповое поле Свойства).
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    При нумерации с данными ПЛК также будет выполнена нумерация устройств, которые подсоединены к этой функции.

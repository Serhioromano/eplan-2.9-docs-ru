# Определить графику в зависимости от значений свойств в формах и рамках

Свойство формы/рамки Присвоение свойства/значения графике необходимо, чтобы для свойства, имеющего определенное значение, выводилась определенная графика в области данных.

Условия:

* Вы открыли проект.
* В редакторе форм открыта та форма, свойства которой требуется обработать (Сервисные программы > Основные данные > Форма > Открыть > [Открыть]). Или в редакторе рамок открыта та рамка, свойства которой необходимо обработать (Сервисные программы > Основные данные > Рамка > Открыть > [Открыть]).
* Открыто диалоговое окно Свойства формы — <Имя формы> (Всплывающее меню > Свойства на форме в диалоговом окне навигатора Страницы — <Имя проекта>). Или открыто диалоговое окно Свойства рамки — <Имя рамки> (Всплывающее меню > Свойства на рамке в диалоговом окне навигатора Страницы — <Имя проекта>).

1. В диалоговом окне Свойства формы — <Имя формы> / Свойства рамки — <Имя рамки> щелкните кнопку ++...++ в столбце Значение свойства Присвоение свойства / значения графике.
2. В диалоговом окне "Присвоение свойства/значения графике" щелкните по ![](images/all_new_as.png){: .ui-icon } (Создать), чтобы создать новую строку в таблице присвоений.
3. В поле Категория / свойство щелкните по кнопке ++...++.
4. В диалоговом окне Выбор свойств выберите свойство, которому Вы хотите присвоить графику.
5. Щелкните по кнопке ++OK++.
6. В поле Значение задайте значение, которым должно обладать свойство.
7. В поле Имя символа щелкните по кнопке ++...++.
8. Выберите в диалоговом окне Выбор символов символ, графику которого вы хотите присвоить свойству.
9. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Соответствующие данные будут применены в полях Библиотека символов и Описание.

10. В поле Вариант символа выберите требуемый вариант из раскрывающегося списка всех имеющихся вариантов текущего символа.
11. Выполните аналогичные действия для всех других присвоений.
12. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Присвоения сохраняются и могут быть вставлены с помощью заполнителей в форме/рамке.

В диалоговом окне Свойства формы — <Имя формы> / Свойства рамки — <Имя рамки> присвоения выводятся в столбце Значение следующим образом:

<Номер свойства>;<Категория>;<Значение>; <Библиотека символов>;<Номер символа>;<Вариант>|
<Номер свойства>;<Категория>;<Значение>; <Библиотека символов>;<Номер символа>;<Вариант>|...

!!! example "Пример:"

    20400;0;SIE;SPECIAL;310;0|1120;0;A*;IEC_EPLAN_D;7;1

!!! note "Замечание:"

    Обратите внимание на то, что на просмотр выводятся только символы ***сохраненных*** библиотек символов.

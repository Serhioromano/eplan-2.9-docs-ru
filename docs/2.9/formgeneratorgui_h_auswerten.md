# Присвоение формы типу отчета

Прежде чем использовать [Типы отчетов](formgeneratorgui_k_auswertungstypen.md) для генерации отчетов, типу отчета необходимо присвоить форму. Формы для отчетов по умолчанию содержатся в комплекте поставки EPLAN.

Условия:

* Вы открыли проект.
* У вас есть форма, подходящая для типа отчетов.

1. Сервисные программы > Отчеты > Генерировать
2. Выберите ++Настройки++ > Вывод в страницах.
3. Выберите в столбце Тип отчета один тип отчета.
4. Чтобы присвоить типу отчета форму, в столбце Форма щелкните по соответствующей строке, а затем в раскрывающемся списке — по записи "Поиск"

!!! info "Для сведения:"

    EPLAN открывает диалоговое окно Выбрать форму и выводит каталог, который установлен в настройках как каталог форм.

5. Чтобы копировать выделенную форму, щелкните по ++Открыть++.

!!! info "Для сведения:"

    Сейчас форме присвоен тип отчета.

6. Щелкните по кнопке ++OK++.

# Оптимизировать отображение перекрестных ссылок

Перекрестная ссылка, представленная в проекте, состоит в основном из полных обозначений устройств, страницы (включая определенный разделитель ***до*** страницы) и позиции в используемой рамке, которая выводится как обозначения столбцов и строк. Перекрестные ссылки помогают Вам быстро находить функции устройства, распределено представленного в проекте. Посредством информации о странице, столбце и / или строке можно быстро определить соответствующую функцию. Страница, столбец и строка отображаются только тогда, когда какой-либо идентифицирующийся идентификатор (например, для блока идентификаторов "Установка") устройства ***не равен*** вышестоящему уровню (странице). Для этой оптимизации отображения действуют следующие правила:

* Как вышестоящий уровень можно просмотреть исключительно актуальную страницу, здесь также ***не учитывается*** обозначение местоположения и т.д.
* [Разделитель]Страница[Разделитель]Столбец[Разделитель]Строка отображается, если какой-либо идентифицирующий идентификатор (например, для блока идентификаторов "Установка") устройства не равен вышестоящему уровню (странице).
* если идентифицирующие блоки идентификаторов идентичны вышестоящему уровню, они могут быть подавлены.
* если все идентифицирующие свойства идентичны, номер страницы тоже может быть подавлен.
* [Разделитель]Страница[Разделитель]Строка никогда не скрываются.
Если перекрестная ссылка отображает только столбец (строку), тогда элемент перекрестной ссылки находится на той же странице.

Для автоматической оптимизации дополнительно можно произвести следующие настройки:

Условие:

Вы открыли проект.

1. Параметры > Настройки > Проекты > "Имя проекта" > Перекр. ссылки / образы контактов > Общ.
2. Установите флажок Отнести правила сокр. перекр. ссылок на ОУ, если приведенные выше правила сокращения должны применяться не к информации для страницы, а к полному ОУ.
3. Активируйте флаг Подавить одинаковые страницы, если номера страниц должны быть подавлены; это возможно только для перекрестных ссылок, для которых идентичны все идентифицирующие свойства.
4. Щелкните по кнопке ++ОК++.

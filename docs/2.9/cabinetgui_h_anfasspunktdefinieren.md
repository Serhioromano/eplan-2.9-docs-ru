# Определить точку захвата

Точка захвата служит для размещения 3D-макроса. Кроме точек захвата, которые автоматически генерируются у трехмерных тел, в макросе есть точка захвата, определенная пользователем. При размещении функционального элемента в проекте данную точку захвата можно выбрать в опциях размещения в поле Точка захвата под названием "Графический макрос".

Условия:

* Вы открыли проект.
* Навигатор пространства листов открыт, и одно пространство листов открыто.
* Пространство листа содержит импортированные трехмерные тела, которые необходимо сохранить в виде макроса.
* Включен захват объекта.

1. Выберите пункты меню Обработать > Логика устройства > Точка захвата.
2. Переместите курсор над трехмерной геометрической фигурой.

!!! info "Для сведения:"

    На объекте будут показаны точки захвата 3D. Это конечные или серединные точки ребер или угловые точки параллелепипеда, обрамляющего объект.

3. Щелкните по требуемой точке.

!!! info "Для сведения:"

    Точка захвата будет выделена оранжевым параллелепипедом.

4. Чтобы переместить точку захвата, повторите шаги с первого по третий.

!!! tip "Совет:"

    В функции Обработать > Создать макрос окна / символа измените точку захвата в диалоговом окне Сохранить как, кнопка ++Дополнительно++, с помощью пункта меню Определить точку захвата.

5. Чтобы настроить расширенную логику точки захвата, измените свойства точки захвата.

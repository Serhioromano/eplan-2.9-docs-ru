# Диалоговое окно Фрагмент

Вы открыли проект.

* Данные проекта > Опции > Генерировать фрагмент.
* Данные проекта > Опции > Навигатор. Выберите пункты всплывающего меню Создать > Фрагмент.
* Данные проекта > Опции > Навигатор. Выделен фрагмент. Обработать > Свойства.

В этом диалоговом окне создается новый фрагмент или выполняется обработка уже имеющегося фрагмента.

Обзор основных элементов диалогового окна:

### Опциональная группа

В этом групповом поле выбирается опциональная группа, которой присваивается опция.

* Имя: Выберите имеющуюся опциональную группу из раскрывающегося списка или введите новое имя. Если оставить это поле пустым (или удалить имеющуюся запись) опция не присваивается никакой опциональной группе.
* Описание: В этом поле отображается текст описания выбранной опциональной группы. При вводе новой опциональной группы это поле доступно для записи, и в него можно ввести текст описания.

### Опция

В этом групповом поле выбирается опция, которой присваивается фрагмент.

* Имя: Выберите имеющуюся опцию из раскрывающегося списка или введите новое имя.
* Описание: В этом поле отображается текст описания выбранной опции. Этот текст нельзя изменить. При вводе новой опции это поле доступно для записи, в него можно ввести текст описания.

### Имя

Введите здесь имя фрагмента или измените имеющееся имя.

### Описание

Введите здесь описывающий текст для фрагмента. Под описанием понимается многоязычный текст, отображаемый по умолчанию в представлении структуры дерева рядом с именем (отделяется пробелом и заключается в круглые скобки).

# Определить медное изделие

Условие:

Вы открыли проект.

1. Сервисные программы > Изделие > Управление
2. В диалоговом окне База данных изделий перейдите в структуре дерева изделий в слое иерархии "Механика > Отдельная часть > Сборные шины" на узел "Шина".
3. Сгенерируйте новую запись через пункт всплывающего меню Новый.
4. Заполните все релевантные поля на вкладке Общее, особенно поля Номер изделия и Обозначение 1.
5. На вкладке Монтажные данные определите значения для полей Ширина и Глубина, чтобы задать размеры прямоугольного контура шины.

!!! info "Для сведения:"

    Эти записи используются при размещении, чтобы определить прямоугольный контур изогнутой шины.

6. На вкладке Определение функции выберите определение функции "Сборная шина" и обозначение функционального элемента "Сборная шина — общее".
7. Сохраните новое изделие, нажав кнопку ++Применить++, если Вы хотите определить еще какие-то изделия, либо кнопку ++Закрыть++, если Вы хотите определить только это изделие и закрыть базу данных изделий.

# Всплывающие подсказки для пиктограмм в навигаторах

Если в представлениях в виде дерева навигаторов данных проекта навести курсор на пиктограмму, теперь можно будет увидеть краткий текст с информацией про эту пиктограмму (всплывающая подсказка). Если пиктограммы объединены (например, ![](images/navigatorgui_mainfunctionpart_as.png){: .ui-icon }![](images/navigatorgui_function_as.png){: .ui-icon } для размещенной многополюсной главной функции), то для каждой пиктограммы будет отображаться отдельная всплывающая подсказка. Для пиктограмм в других представлениях в виде дерева дополнительная информация не будет отображаться (например, в управлении проектами, навигаторе основных данных изделия и т. д.).

![](images/benefit.png)Эффект:

С помощью всплывающих подсказок можно быстрее понять значение пиктограмм в навигаторах данных проекта.

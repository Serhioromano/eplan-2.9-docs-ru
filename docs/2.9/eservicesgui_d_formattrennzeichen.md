# Диалоговое окно Формат: Разделитель

Вы открыли проект. Открыто диалоговое окно свойств для функции, символа или проекта. В поле Свойства выбрано свойство Свойство блока: Формат [n], а в столбце Значение нажата кнопка ++...++.

* Выберите в диалоговом окне Формат в списке Доступные элементы формата элемент формата Разделитель и щелкните по ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо).
* Выделите в диалоговом окне Формат в списке Выбранные элементы формата имеющийся элемент формата типа Разделитель и щелкните по ![](images/all_edit_as.png){: .ui-icon } (Обработать).

Вы открыли проект. Вы открыли диалоговое окно [Настройки: ПЛК](plcgui_d_allgemeinespseinstellungen.md) (напр., через пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Устройство > ПЛК). В диалоговом окне Настройки: ПЛК рядом с полем Специфические для ПЛК настройки вы щелкнули кнопку ++...++. В диалоговом окне [Настройки: Специфич. для ПЛК](plcprocessorgui_d_spsspezifischeeinstellungen.md) вы выбрали вкладку Формат списка присвоений и рядом с полями Формат строки списка присвоений или Формат адреса списка присвоений щелкнули кнопку ++...++.

В следующем диалоговом окне выберите из списка Доступные элементы формата элемент формата типа Разделитель и щелкните по ![](images/all_arrowright_as.png){: .ui-icon } (переместить направо) (или дважды щелкните по списку Выбранные элементы формата на уже определенном разделителе).

В этом диалоговом окне указывается форматирование разделителя для группы форматов.

Обзор основных элементов диалогового окна:

В качестве разделителя можно использовать следующие символы:

* Пробел
* Табулятор
* Новая строка
* Другие символы: После выбора этого параметра введите в поле ввода под ним требуемую комбинацию символов. Допускаются любые цифры, буквы и специальные символы.

!!! note "Замечание:"

    * Для всех ***Форматов ПЛК*** новая строка в качестве разделителя ***не*** доступна. Для ***Форматов адресов*** дискретных и аналоговых входов и выходов табулятор в качестве разделителя также ***не*** доступен.
    * Для ***Свойствa блокa*** табулятор в качестве разделителя ***не*** доступен.

### Число символов

Если вы выбрали в качестве разделителя пробел, табулятор или новую строку, то здесь Вы можете установить число строк. Разделитель может также состоять из нескольких пробелов, табуляторов или новых строк. В случае выбора параметра Другие символы это поле становится недоступным и в нем выводится число введенных символов.

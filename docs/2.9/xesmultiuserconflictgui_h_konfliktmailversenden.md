# Отправить эл. почту в случае конфликта

Условия:

* Для отправки электронных сообщений на компьютере должен быть установлен и настроен клиент электронной почты MAPI. В связи с этим не может применяться почтовое приложение Windows 8®.
* Для многопользовательского режима Вы открыли проект в режиме обработки "Стандарт". Кроме того, пользователи, участвующие в многопользовательском режиме, указали соответствующие данные пользователя в настройках (Параметры > Настройки > Пользователь > Отображение > Ид. пользователя / адрес).

В случае конфликта в многопользовательском режиме вызывается диалоговое окно Многопользовательский конфликт. При помощи диалогового окна Вы можете отправить электронное сообщение пользователю или пользователям, которые в это же время обрабатывают те же данные проекта.

### Отправить пользователю электронное сообщение

1. Выделите в диалоговом окне Многопользовательский конфликт строку желаемого пользователя.
2. Выберите пункт всплывающего меню Отправить эл. почту.
3. если вы еще не запустили свой редактор электронной почты, в следующем диалогом окне сначала выберите профиль для информационной службы. Лишь после этого запускается редактор электронной почты.

!!! info "Для сведения:"

    Вызывается программа электронной почты, установленная на вашем компьютере. Поле для ввода адреса эл. почты уже автоматически заполнено и строка "Тема" заполнена записью "Многопользовательский конфликт".

4. Добавьте Ваш текст и отправьте эл. сообщение, нажав на кнопку ++Отправить++, другому пользователю.

### Отправить электронное сообщение всем пользователям одновременно

В диалоговом окне Многопользовательский конфликт приведено несколько пользователей, и у Вас есть возможность проинформировать по эл. почте сразу всех пользователей.

1. Выберите для этого строку Электронная почта.
2. Выберите пункт всплывающего меню Отправить эл. почту.
3. если вы еще не запустили свой редактор электронной почты, в следующем диалогом окне сначала выберите профиль для информационной службы. Лишь после этого запускается редактор электронной почты

!!! info "Для сведения:"

    Вызывается программа электронной почты, установленная на вашем компьютере. Все выбранные адреса эл. почты уже введены в поле "Адрес" и строка "Тема" заполнена записью "Многопользовательский конфликт".

4. Добавьте Ваш текст и отправьте эл. сообщение, нажав на кнопку ++Отправить++ всем остальным пользователям.

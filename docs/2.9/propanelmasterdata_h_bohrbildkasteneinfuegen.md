# Вставить границу схемы сверления

Чертеж DXF / DWG может содержать несколько схем сверления. Графические элементы чертежа, которые должны принадлежать к схеме сверления, могут быть определены пользователем через границы схем сверления. Граница схемы сверления рассматривается по аналогии с рамкой макроса. Условием для генерирования схем сверления является присвоение объектов определенным слоям EPLAN.

Вставка границ схем сверления возможна только в проекте макросов.

Условия:

* Вы открыли проект макросов.
* Вы импортировали файлы DXF / DWG в качестве страниц, сгенерированных при помощи внешнего машинного ПО (напр., Steinhauer eCAB) из данных ЧУ.
* Блоки, если таковые содержались в импортированных файлах DXF / DWG, были разбиты через меню Обработать > Другое > Разбить блок.

1. Выберите пункты меню Вставить > Блок / Вывод устройства / Монтажная плата > Граница схемы сверления.
2. Очертите область выбора с определенными для схемы сверления отверстиями прямоугольником или ломаной линией.

!!! info "Для сведения:"

    Все объекты, описанный прямоугольник которых находится внутри прямоугольной границы схемы сверления, относятся к границе схемы сверления.

!!! info "Для сведения:"

    Нулевая точка генерируемой схемы сверления лежит слева внизу.

!!! tip "Совет:"

    При генерировании схемы сверления все расчеты позиций и интервалов между участвующими объектами базируются на нулевой точке границы схемы сверления. Позиция нулевой точки на границе схемы сверления определяет позицию нулевой точки на генерируемой схеме сверления. С помощью пункта всплывающего меню Переместить нулевую точку можно изменить позицию нулевой точки и таким образом задать требуемые интервалы для первого и всех последующих отверстий на схеме сверления.

1. Введите на вкладке Граница схемы сверления диалогового окна 'Свойства' имя и описание генерируемой схемы сверления.
2. Щелкните по ++...++ в поле Подкаталог для контура (данные ЧУ) и выберите каталог, в котором должны сохраняться файлы *.fc1, автоматически сгенерированные при генерировании схемы сверления.
3. Щелкните по кнопке ++OK++.
4. При необходимости измените позицию нулевой точки, пользуясь пунктом всплывающего меню Переместить нулевую точку.
5. Присвойте границе схемы сверления другие отверстия, пользуясь пунктом всплывающего меню Присвоить объекты границе схемы сверления.

!!! tip "Совет:"

    Схемы сверления также можно чертить в виде ломаных линий с помощью символа BB2. Для этого в диалоговом окне Выбор символа выберите символ и задайте начальную точку схемы, щелкнув левой клавишей мыши в схеме соединений. После задания всех остальных пунктов завершите действие через Всплывающее меню Закрыть ломаную линию.

# Диалоговое окно Объекты-заполнители - <Имя проекта>

Открыт проект и этом проекте вставлены объекты-заполнители. Данные проекта > Опции > Объекты-заполнители.

В этом диалоговом окне перечислены все объекты-заполнители, имеющиеся в данном проекте. С помощью всплывающего меню можно обработать свойства объектов-заполнителей и присвоить им наборы значений.

Обзор основных элементов диалогового окна:

На вкладке Дерево выводятся объекты-заполнители с иерархической сортировкой по "<Имя>" и "Размещению (<Описание>)". (Это случай для стандартной конфигурации.) При этом объекты-заполнители с идентичными именами объединяются.

!!! note "Замечание:"

    В отличие от ***имени***, которое является идентифицирующим и поэтому не многоязычным, ***описание*** можно использовать для дополнительной информации, которую можно отобразить на иностранном языке. Соответствующий многоязычный текст вводится на вкладке Объект-заполнитель.

На вкладке Список по умолчанию выводятся имя, описание, размещение и последний выбранный набор значений. С помощью всплывающего меню выполняется конфигурирование выведенных свойств.

### Фильтр

В этом раскрывающемся списке отображаются все доступные фильтры. Выбранный фильтр активируется автоматически и применяется как к дереву, так и к списку. Запись "- Не активировано -" отключает фильтр и приводит к тому, что данные отображаются в неотфильтрованном виде. С помощью кнопки ++...++ откройте диалоговое окно [Фильтр](modaldialogsdb_d_filternnach.md). Здесь можно создать, обработать, удалить, скопировать, экспортировать, импортировать фильтр и управлять им.

Во всплывающем меню раскрывающегося списка Фильтр содержатся следующие записи:

* Выключить: Этот пункт меню доступен, если фильтр установлен: Сбрасывает настройку фильтра до записи "- Не активировано -".
* Активировать <Имя фильтра>: Этот пункт меню доступен, если для настройки фильтра установлено значение "- Не активировано -": Повторно активирует последний активный фильтр.

Таким образом можно быстро переключаться между неотфильтрованным и отфильтрованным в соответствии с требованиями пользователя представлениями.

### Значение: <Свойство>

При помощи [быстрого ввода](modaldialogsdb_k_filter.md) в данном поле для определенного и активированного фильтра можно быстро изменить значение его критерия.

### Всплывающее меню

Пункт меню |  Значение
---|---
Выделить все (только список) |  Выделяет все записи в списке.
Подобрать ширину столбца (только список) |  Все столбцы так подогнаны к таблице по ширине, что и заголовок, и содержимое столбцов полностью читаемы.
Присвоить набор значений |  Открывает диалоговое окно Выбрать набор значений – <Имя заполнителя>, где выделенным объектам-заполнителям можно присвоить набор значений.
Выделить соответствующие объекты |  Выделяет в графическом редакторе все объекты, относящиеся к текущему объекту-заполнителю.
Обработать маркеры ревизий |  Посредством данного пункта меню Вы можете обрабатывать тексты маркировки ревизий и их формат. Пункт меню доступен только в том случае, если в ревизии выделен измененный объект.
Удалить маркер ревизии |  Через этот пункт меню можно удалить тексты маркеров ревизий. Пункт меню доступен только в том случае, если в ревизии выделен измененный объект.
Перейти к (графика) |  Отображает выделенный объект-заполнитель в графическом редакторе.
Конфигурировать представление |  Открывает диалоговое окно Конфигурировать представление, в котором можно задать, какие свойства должны отображаться в представлении в виде списка и в представлении структуры дерева.
Свойства |  Открывает диалоговое окно Объект-заполнитель. Позволяет обрабатывать свойства объекта-заполнителя.

# Диалоговые окна Удалить сохраненные свойства изделия

Сервисные программы > Изделие > Удалить сохраненные свойства.

В этих диалоговых окнах можно удалить свойства сохраненных в проекте изделий. Возможен вызов для всего проекта, выделенных страниц или выделенных элементов в графическом редакторе.

Обзор основных элементов диалогового окна:

### Настройки

В данном раскрывающемся списке отображаются все пользовательские и предварительно определенные схемы. Выберите схему, чтобы задать настройки для удаления сохраненных свойств. С помощью кнопки ++...++ откройте следующее одноименное диалоговое окно, в котором можно обработать схемы и управлять ими.

### Следующее диалоговое окно: Удалить сохраненные свойства изделия

В следующем диалоговом окне задайте настройки для удаления и сохраните их как схему.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Свойство изделия

В этом столбце перечисляются свойства изделия, которые следует удалить.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

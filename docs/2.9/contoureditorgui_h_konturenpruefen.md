# Проверка контуров

Чтобы во время обработки контура можно было в любое время проверить, верен ли контур, или он содержит ошибки, существует проверка геометрии. В ходе данной процедуры — в зависимости от используемого типа файла (тип контура) — выполняются следующие виды проверок:

Она выполняет следующие виды проверки:

* Использование неразрешенных элементов
* Наличие как минимум одного закрытого контура
* Все существующие контуры закрыты и не содержат удвоенных элементов
* Отсутствуют вложенные внутренние контуры.

Проверка геометрии для типа ***Генерирование записи данных контура ЧУ***:

* Использование неразрешенных элементов
* Наличие ровно одного замкнутого контура, не содержащего двойных элементов
* Отсутствуют вложенные внутренние контуры.

Условия:

* Вы открыли проект.
* Контур, который необходимо изменить, открыт в редакторе контуров (Сервисные программы > Основные данные > Контур (...) > Открыть > [Открыть]).

1. Сервисные программы > Проверить контур
2. Подтвердите сообщение в диалоговом окне Проверить контур нажатием по ++OK++.
3. Выберите пункты меню Сервисные программы > Системные сообщения.
4. В диалоговом окне Системные сообщения щелкните по кнопке ++Сохранить++.

!!! info "Для сведения:"

    Системные сообщения, которые относятся к текущему контуру, сохраняются в соответствующем файле и при необходимости могут быть напечатаны.

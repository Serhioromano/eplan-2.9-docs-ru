# Генерировать схемы соединений ПЛК

В EPLAN неразмещенные (или неразмещенные в определенных видах представления) функции ПЛК можно разместить автоматически и таким образом сгенерировать схемы соединений ПЛК, включая различные страницы обзора. Неразмещенные функции ПЛК можно предварительно создать в навигаторе ПЛК или импортировать из файла конфигурации ПЛК. Можно также импортировать адреса из списка присвоений и при помощи выбора устройств сгенерировать для них функции ПЛК. Если схема соединений ПЛК уже сгенерирована, автоматически размещенные функции ПЛК можно удалить и разместить заново.

Условия:

* Вы открыли проект.
* Вы создали неразмещенные функции ПЛК или импортировали их из файла конфигурации ПЛК.
* Либо вы импортировали адреса ПЛК из списка присвоений и при помощи команды всплывающего меню Генерировать с устройством сгенерировали для этих адресов неразмещенные функции ПЛК.
* Либо вы сгенерировали схему соединений ПЛК, вследствие чего в проекте содержатся функции ПЛК, размещенные автоматически.
* Вы выделили проект или страницы проекта в навигаторе страниц. Либо вы выделили функции ПЛК в одном из навигаторов. Либо вы выделили условные обозначения ПЛК в графическом редакторе.

1. Выберите следующие пункты меню: Данные проекта > ПЛК > Генерировать схему соединений

!!! info "Для сведения:"

    В диалоговом окне Генерировать схему соединений ПЛК в столбце Объекты отображаются устройства ПЛК, полностью или частично включенные в выборку.

!!! info "Для сведения:"

    Если функции уже размещены в определенных видах представления, то в столбцах Страница Однополюс., Страница Многополюс., Страница Обзор и Страница Создание каркаса будут отображены соответствующие имена страниц.

2. В столбце Файл макроса присвойте блокам ПЛК соответствующий файл макроса, если в изделии никакие макросы не введены, либо при необходимости измените макрос, присвоенный автоматически (через изделие). Для этого щелкните в строке по кнопке ++...++ и в диалоговом окне Выбрать макрос выберите необходимый макрос.
3. В столбце Файл макроса присвойте выводам устройства ПЛК подходящий макрос проводки. Для этого щелкните в строке по кнопке ++...++ и в диалоговом окне Выбрать макрос выберите необходимый макрос.
4. Если присвоенные макросы содержат переменные, в столбцах с именами переменных при необходимости можно выбрать другой набор значений. Кроме того, ввести значение можно вручную.
5. Установите флажки для представлений, которые необходимо сгенерировать: Генерировать однополюсное представление, Генерировать многополюсное представление, Генерировать обзорные представления или Генерировать обзорные представления каркаса.
6. Выделите строки с теми устройствами, для которых необходимо сгенерировать страницы схемы переключений.
7. Выберите пункт всплывающего меню Вычислить страницы.
8. В диалоговом окне Нач. страницы для каждого необходимого вида представления введите имя первой страницы, либо нажатием кнопки ++...++ откройте диалоговое окно Полное имя страницы, чтобы обработать структурный идентификатор и имена страниц. Нажав кнопку ++...++, там можно выбрать уже имеющиеся структурные идентификаторы (например, установку и имя установки). Идентификаторы можно ввести и вручную, при этом имеется возможность создания новых идентификаторов.
9. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    На основе введенных для ***первых*** начальных страниц имен в EPLAN рассчитываются все остальные начальные страницы для других устройств, которые затем вносятся в таблицу.

10. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Страницы генерируются в соответствии с настройками, и на них размещаются указанные макросы. Макросы при этом вставляются на свои исходные позиции. Если на одну страницу помещаются несколько макросов, они располагаются (в зависимости от рамки) друг рядом с другом или один под другим.

!!! info "Для сведения:"

    Автоматически сгенерированным страницам присваивается свойство Сгенерировано автоматически, при этом значение в раскрывающемся списке обозначает вид представления.

!!! tip "Совет:"

    * В диалоговом окне Генерировать схему соединений при помощи кнопки ![](images/all_new_as.png){: .ui-icon } (Создать) в таблицу можно вставить другие устройства ПЛК из проекта. При этом можно, например, выбрать функции, не размещенные только в некоторых определенных видах представлений. Таким образом сначала можно сгенерировать все многополюсные страницы схемы соединений, а на втором этапе – страницы обзора.
    * При помощи кнопок ![](images/all_export_as.png){: .ui-icon } (Экспорт) и ![](images/all_import_as.png){: .ui-icon } (Импорт) можно сохранить конфигурацию (то есть настройки, установленные в диалоговом окне Генерировать схему соединений ПЛК) и обратиться к ней позднее.

!!! note "Замечание:"

    автоматически сгенерированные страницы можно впоследствии обработать вручную. В любом случае при повторной генерации эти страницы будут перезаписаны, а внесенные вручную изменения – утеряны. Если такой результат нежелателен, значение свойства страницы Сгенерировано автоматически нужно изменить на "Нет".

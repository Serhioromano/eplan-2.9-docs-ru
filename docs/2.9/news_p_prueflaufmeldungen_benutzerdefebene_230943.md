# Улучшенное распознавание определенных пользователем слоев

Сообщение контрольного прогона ***501005*** из класса сообщения 501 "Основные данные изделий" было изменено. Это улучшает поиск определенных пользователем слоев в макросах, которые были сохранены на изделиях. В результате для определенных пользователем слоев теперь генерируется значительно больше сообщений контрольного прогона, чем до изменений.

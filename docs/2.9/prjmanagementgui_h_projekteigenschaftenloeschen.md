# Обработать свойства проекта (Управление проектами)

Если вы обрабатываете проект без использования базы данных управления проектами, вы можете добавлять и удалять свойства проекта в навигаторе страниц через пункт всплывающего меню Свойства.

Если вы работаете с базой данных управления проектами, вы можете также добавлять и удалять свойства проекта в навигаторе страниц. Изменения применяются в базе данных управления проектами, и могут в дальнейшем обрабатываться в управлении проектами.

Условия:

* Вы выбрали базу данных управления проектами.
* Вы считали каталоги и проекты.

### Установка режима обработки

1. Выберите следующие пункты меню: Проект > Управление
2. В диалоговом окне Управление проектами выделите в структуре дерева необходимый проект и выберите вкладку Свойства.

!!! info "Для сведения:"

    Содержимое отображается в режиме Вид.

3. Для изменения режима обработки нажмите кнопку ++Обработать++.

### Добавить свойства проекта

1. Для уменьшения количества отображаемых и выбираемых свойств выберите запись на вкладке Свойства в раскрывающемся списке Категория.
2. Нажмите кнопку ![](images/all_new_as.png){: .ui-icon } (Создать).
3. В диалоговом окне Выбор свойств выделите одно или несколько свойств.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Свойства переносятся во вкладку Свойства. Свойства, значения которых содержат автоматически генерируемые записи, представлены серым цветом, и в этом месте не могут быть изменены.

5. Для обработки значения свойства щелкните столбец Значение и введите содержимое.
6. Щелкните по кнопке ++Сохранить++.

### Удалить свойства проекта

1. Выберите вкладку Свойства.
2. Выделите в столбце Имя свойства одно или несколько свойств.
3. Щелкните по кнопке ![](images/all_delete_as.png){: .ui-icon } (Удалить).
4. Щелкните по кнопке ++Сохранить++.

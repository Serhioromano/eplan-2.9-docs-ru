# Вкладка <Отображаемое имя объекта планирования (трубопровода)>

Вы открыли проект.

* Вы сгенерировали объект планирования (трубопровод) в навигаторе предварительного планирования или вставили его в графическом редакторе. В диалоговом окне Свойства ++...++ выберите вкладку <Отображаемое имя объекта планирования (трубопровода)>.
* Вы выделили объект планирования (трубопровод) в навигаторе предварительного планирования или в графическом редакторе. Обработать > Свойства. Выберите вкладку Отображаемое имя объекта планирования (трубопровода).

На этой вкладке задаются свойства выделенного объекта планирования (трубопровода).

Если в диалоговом окне свойств записи в полях отличаются от стандартных значений присвоенного шаблона сегмента, они выделяются [желтым цветом фона](planninggui_k_segmentvorlagen.md).

Обзор основных элементов диалогового окна:

### Обозначение

В этом поле введите имя объекта планирования (трубопровода). Указанное здесь обозначение по умолчанию отображается в навигаторе предварительного планирования на объекте планирования (трубопроводе).

### Полное обозначение

В этом поле отображается обозначение объекта планирования (трубопровода), включая обозначения вышестоящих объектов планирования и сегментов структуры.

### Описание

Введите здесь описательный текст для объекта планирования (трубопровода). Этот текст по умолчанию отобразится в навигаторе предварительного планирования вместе с обозначением объекта планирования (трубопровода).

### Шаблон сегмента

В этом поле отображается шаблон сегмента, присвоенный сегменту. С помощью кнопки ++...++ откройте диалоговое окно Выбрать шаблон сегмента, чтобы выбрать шаблон сегмента. Отображаются только шаблоны сегментов, подходящие к определению текущего сегмента. При замене шаблона сегмента значения свойств нового шаблона сегмента переносятся на сегмент и отображаются в диалоговом окне Свойства. Значения, измененные вручную, сохраняются.

С помощью кнопки ![](images/all_delete_as.png){: .ui-icon } (Удалить) из сегмента можно удалить ссылку на шаблон сегмента. Определенные в шаблоне сегмента значения свойств удаляются из сегмента. Значения, измененные вручную, сохраняются.

### Исходный сегмент:Целевой сегмент

Введите здесь обозначение сегмента, которое должно быть источником / целью объекта планирования (трубопровода). С помощью кнопки ++...++ откройте диалоговое окно Выбрать источник / цель. Здесь предлагаются для выбора все сегменты проекта.

Если объект планирования (трубопровод) размещен на странице предварительного планирования в виде ломаной линии, записи в этих полях будут недоступны для обработки. При этом источник и / или цель объекта планирования (трубопровода) можно адаптировать путем графического изменения ломаной линии.

### Техническое описание

Введите здесь обзор технических характеристик для объекта планирования (трубопровода).

### Класс трубы

Укажите здесь имя для класса трубы. С помощью кнопки ++...++ откройте диалоговое окно Выбрать класс трубы. Здесь предлагаются для выбора классы трубы, управляемые в диалоговом окне Шаблоны сегментов - <Имя проекта>.

# Диалоговое окно Формат: свойство

Вы открыли проект. Сервисные программы > Основные данные > Форма > Открыть > [Открыть].

* Вставить > Текст заполнитель. В диалоговом окне перейдите на вкладку Размещение и рядом с полем Свойство отформатировано/вычислено щелкните по кнопке ++...++.
* Дважды щелкните по тексту-заполнителю (или выберите во всплывающем меню пункт Свойства; в этом случае можно выделить несколько текстов). В диалоговом окне перейдите на вкладку Размещение и рядом с полем Свойство отформатировано/вычислено щелкните по кнопке ++...++.

Параметры > Настройки > Пользователь > Интерфейсы > Экспорт и импорт DXF / DWG. Перейдите на вкладку Экспорт и выберите в поле Генерировать имена файлов запись "Из свойств", а затем нажмите кнопку ++...++.

В этом диалоговом окне для текста-заполнителя можно задавать элементы формата и / или формулу расчета, или задавать свойства, которые должны использоваться для генерирования имени файла DXF / DWG.

Обзор основных элементов диалогового окна:

### Доступные элементы формата

В этом поле отображаются все элементы формата, которые можно выбрать и отформатировать для текста-заполнителя. Доступность определенного элемента формата зависит от выбранного типа формы.

### Выбранные элементы формата

В этом поле отображаются все выбранные Вами элементы формата. Чтобы выбрать элемент формата, выделите в поле Доступные элементы формата этот элемент формата и щелкните по кнопке ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо).

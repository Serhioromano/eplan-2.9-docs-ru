# Элементы-заполнители EPLAN: Основные положения

Символы-заполнители служат при обработки формулы для выбора свойств, вводимых в форму как тексты-заполнители. Набор имеющихся в распоряжении символов-заполнителей зависит от типа формулы. Они представляют возможные свойства, сгруппированные по объектам, используемым в форме.

Текст-заполнитель состоит из элемента-заполнителя и свойства. При этом выбор элемента-заполнителя определяет, свойства какого объекта устанавливаются. Это разделение видно также в диалоговом окне Тексты заполнителей — <Тип формы>: В окне слева показываются элементы-заполнители, а справа — свойства, имеющиеся в распоряжении для отмеченного элемента-заполнителя.

В обзоре элементов-заполнителей EPLAN отображается следующая информация:

Запись |  Значение
---|---
Ид. |  Расширение имени файла типа формы.
Тип формы |  Название свойства, отображаемое в интерфейсе программы.
Элементы |  Название элемента-заполнителя, показываемого в оболочке программы.
Описание |  Текст комментария, описывающий вид свойств, доступный для соответствующего элемента-заполнителя.
Индексирование |  Укажите, возможен ли индекс для элемента-заполнителя. В диалоговом окне выбора рядом с элементом-заполнителем показывается значок плюса. Если щелкнуть по этому значку, структура дерева списка доступных для индексации элементов-заполнителей раскроется, и вы сможете выбрать нужный элемент. Индекс может использоваться, например, для того, чтобы выбирать определенную цель из нескольких доступных. Если вы выберете вышестоящий элемента-заполнитель (или значение индекса), это означает, что будут выданы свойства всех целей (по возможности).

!!! example "Пример:"

    В списке обозначений устройств индексируется элемент-заполнитель "Данные изделия". Через запись "Данные изделия [3]" вы получаете доступ к свойствам третьего внесенного в устройство изделия; выберите, например, свойство "Номер изделия". В форме затем выдается номер изделия третьего внесенного в устройство изделия.Есл вы выберите вышестоящий элемент-заполнитель "Данные изделия" без индекса, а затем - свойство "Номер изделия", будут выданы номера изделий всех изделий, внесенных в устройство.

Данные заголовка |  Укажите, предусмотрено ли использование данного элемента-заполнитель в данных заголовка формы. Для спецификации клеммника, к примеру, данные клеммника являются данными заголовка.

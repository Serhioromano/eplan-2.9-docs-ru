# Переименовать, удалить проект (Управление проектами)

Условие:

В структуре дерева управления проектами находится считанный каталог с проектом.

### Переименовать проект

!!! note "Замечание:"

    Выгруженные и заархивированные проекты переименовывать нельзя.

1. Выберите следующие пункты меню: Проект > Управление
2. Выделите в структуре дерева проект.
3. Выберите пункт всплывающего меню Переименовать.
4. Введите новое имя проекта.
5. Нажмите клавишу ++Ввод++.

!!! info "Для сведения:"

    Если вводится имя уже имеющегося проекта, после запроса на подтверждение оно перезаписывается и перемещается в корзину, аналогичную корзине Windows.

### Удалить проект

1. Выберите следующие пункты меню: Проект > Управление
2. Выделите в структуре дерева проект.
3. Выберите пункт всплывающего меню Удалить.

!!! info "Для сведения:"

    Файл проекта и каталог проекта со всеми содержащимися данными удаляются и копируются в корзину, аналогичную корзине Windows. При необходимости из нее можно восстановить проект.

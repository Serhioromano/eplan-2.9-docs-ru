# Медное изделие: Принцип

Изделия для изогнутых сборных шин можно найти в базе данных изделий в главной группе Механика, подгруппе Отдельная деталь, Сборные шины, Шина. При размещении "Сборной шины (изогнутой)" используется фильтр для выбора номера изделия, и сразу открывается нужное место в базе данных изделий.

На вкладке Монтажные данные изделия-шины в полях Ширина и Глубина записаны значения. Эти значения образуют прямоугольный контур поперечного сечения медной шины. Они считываются при размещении шины и присваиваются ***Профилю***. Профиль — это путь, по которому проходит прямоугольный основной контур, формирующий изогнутую медную шину.

Если изделию вместо значений в полях Ширина и Глубина присвоен ***Контур (Выдавливание)*** с расширением файла *.fp1 (для создания пользовательской шины), это изделие нельзя использовать для изогнутой медной шины. При размещении появится сообщение об ошибке.

На вкладке Свойства изделия-шины у купленных изделий-шин для свойства Поставляемая длина записано значение. Это значение учитывается при размещении шины, так что шина не может превосходить его по длине профиля. Как только будет достигнута поставляемая длина, появится сообщение; в этом месте необходимо подсоединить в конструкции другую шину.

### Медные функциональные элементы, не подлежащие изменению

Длину, угол и места изгиба размещенного медного функционального элемента, как правило, можно изменить. Однако на практике иногда требуется стандартизировать медные функциональные элементы или закупить их в строго заданной форме. Такие постоянные детали не подлежат изменению после размещения.

Управлять этим поведением можно при помощи свойства Предварительно собранный (ид. 36011), которое устанавливается на функциональном элементе, например, в макросе. Если это свойство имеется и установлено, то после размещения медный функциональный элемент уже нельзя изменить. Если это свойство снято, можно использовать функции для обработки медных функциональных элементов.

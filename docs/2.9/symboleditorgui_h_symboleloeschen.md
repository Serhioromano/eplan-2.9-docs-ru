# Удалить символы

Условие:

Вы открыли проект и как минимум одну библиотеку символов.

1. Сервисные программы > Основные данные > Символ > Удалить
2. Отметьте стираемый символ в диалоговом окне Выбор символа.
3. Щелкните по кнопке ++OK++.
4. Выберите необходимый ответ в запросе.

* [Диалоговое окно Выбор символа](xsbgui_d_symbolauswahl.md)

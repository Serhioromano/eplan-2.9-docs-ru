# Синхронизация данных проекта

Проект включает в себя не только страницы и основные данные, но и настройки, шаблоны отчетов и другие данные, такие как слои, определенные пользователем свойства, порядок свойств и т. д. С помощью нового диалогового окна синхронизации теперь можно избирательно переносить настройки и данные из исходного проекта в другой проект. При этом данные копируются в целевой проект.

![](images/benefit.png)Эффект:

С помощью нового диалогового окна синхронизации можно быстро и легко перенести настройки и данные проекта из другого проекта (например, из базового проекта). Больше не требуется ручная подстройка проектов и обмен данных проекта с помощью различных операций экспорта и импорта XML-файлов.

Чтобы открыть новое диалоговое окно Синхронизировать данные проекта, выберите путь меню Проект > Организовать > Синхронизировать. Данные целевого проекта отображаются в левой части диалогового окна, которое построено так же, как и существующее диалоговое окно для синхронизации основных данных.

Исходный проект, данные которого отображаются в правой части окна, можно выбрать с помощью кнопки ++...++ рядом с полем Исходный проект. При следующем открытии диалогового окна синхронизации предварительно устанавливается последний выбранный исходный проект.

Обратите внимание на такие особенности:

* С помощью раскрывающегося списка Тип диалоговое окно синхронизации данных проекта можно настроить так, чтобы здесь отображались только данные определенного типа.
* Тип данных "Настройки" включает в себя не только настройки проекта, но и связанные с ними схемы (например, схемы контрольных прогонов) и шаблоны отчетов.
* В отличие от синхронизации основных данных, данные некоторых типов не имеют даты изменения. Это касается данных типа "Слой", "Порядок свойств", "Настройка" и "Структурный идентификатор". Связанные элементы данных имеют состояния "Идентичные", "Не совпадает" и "Только в проекте".
* Чтобы скопировать отдельные данные проекта в целевой проект, выделите их в списке Исходный проект и нажмите ![](images/all_arrowleft_as.png){: .ui-icon } (копировать налево).
* Для обновления и пополнения всех данных целевого проекта используйте оба пункта меню под кнопкой [Обновить].

!!! note "Замечание:"

    Если во время синхронизации выбрать данные типа "Настройка" и перенести их в целевой проект, действие Отменить будет недоступно. Но если выбраны другие типы данных, настройку можно будет отменить.

# Выделить и обработать трассу трубопровода

Чтобы быстро оценить удлинение трубопровода на функциональной схеме автоматизации, трубопровод можно временно выделить цветом. Добавляя / убирая линии автоматического соединения, можно обрабатывать трассу трубопровода.

При этом текущий выбранный трубопровод выделяется заданным в настройках пользователя цветом для отслеживания потенциалов. Линии автоматического соединения, которые можно добавить к трубопроводу, выделяются заданным для внешних символов цветом.

Условия:

* Вы открыли проект, который содержит данные предварительного планирования.
* Открыта страница типа "Функциональная схема автоматизации".
* Вы определили трубопровод на функциональной схеме автоматизации с помощью точки определения трубопровода.

1. Выберите следующие пункты меню: Данные проекта > Соединения > Трасса трубопровода
2. Выберите точку определения трубопровода.

!!! info "Для сведения:"

    Относящиеся к трубопроводу линии автоматического соединения выделяются цветом. Линии автоматического соединения, которые можно добавить к трубопроводу, выделяются другим цветом.

!!! info "Для сведения:"

    На линиях автоматического соединения трубопровода направление потока отображается с помощью острия стрел.

!!! info "Для сведения:"

    Пиктограммы на курсоре указывают, когда можно добавить линию автоматического соединения трубопроводу или убрать ее с него с помощью щелчка мыши:

Курсор |  Значение
---|---
![](images/planningri_cursoradddelete_as.png){: .ui-icon } |  Трасса трубопровода активирована
|  Следующие три пиктограммы отображаются, если курсор находится точно над линией автоматического соединения:
![](images/planningri_cursoraddconnection_as.png){: .ui-icon } |  Добавить трубопроводу линию автоматического соединения
![](images/planningri_cursornotallowed_as.png){: .ui-icon } |  Запрещено добавлять / убирать линию автоматического соединения
![](images/planningri_cursordeleteconnection_as.png){: .ui-icon } |  Убрать линию автоматического соединения из трубопровода
3. Щелкните линию автоматического соединения, чтобы добавить это соединение трубопроводу или убрать с него соединение (в зависимости от отображаемого курсора).

!!! info "Для сведения:"

    Логическая схема соответствующего вывода устройства адаптируется автоматически.

4. Если для трассы трубопровода необходимо вручную добавить условные обозначения, на которых на логической схеме выводов устройства по умолчанию не задана подходящие передача потенциала, нажмите и удерживайте клавишу ++Shift++.

!!! info "Для сведения:"

    На соответствующих условных обозначениях, ограничивающих трубопровод, другие линии автоматического соединения выделяются цветом.

5. Выберите необходимые линии автоматического соединения. Путем добавления других линий автоматического соединения можно постепенно растянуть трассу трубопровода на все доступные выводы устройства на функциональной схеме автоматизации.

!!! info "Для сведения:"

    При этом логическая схема соответствующих выводов устройства адаптируется автоматически.

6. Завершите операцию с помощью пункта всплывающего меню Завершить действие / Закрыть трассу трубопровода или клавиши ++Пробел++.

!!! tip "Совет:"

    * Сначала можно выбрать точку определения трубопровода (или объект, присвоенный определению трубопровода) в графическом редакторе или в навигаторе (навигатор предварительного планирования, навигатор трубопроводов и т. д.), а затем — пункты меню Данные проекта > Соединения > Трасса трубопровода. Трасса трубопровода выделяется, и при необходимости открывается соответствующая страница.
    * С помощью свойства Трубопровод: Заменить источник и цель или пункта меню Заменить источник и цель вы можете также изменить направление потока в трубопроводе на функциональной схеме автоматизации. Подробную информацию можно найти в разделе "[Трасса и источник / цель трубопровода](planningri_k_rdp.md)".
    * Используйте специальные символы AR1 или AR2 для постоянного отображения направления трассы трубопровода. Для этого выберите один из этих символов в диалоговом окне Выбор символа в библиотеке символов SPECIAL и поместите его на линию автоматического соединения нужного трубопровода. Чтобы указатель в виде стрелки показывал в правильном направлении, выбранный вариант символа при необходимости нужно привести в соответствие с трассой трубопровода при помощи функций Обновить детальное планирование или Трасса трубопровода.

# Работа с вариантами символов

Один символ может иметь до восьми вариантов с именами от A до H, зеркальный вариант расположения контактов, а также вариант для представления точки соединения, причем все варианты одного символа имеют общие свойства. То есть все варианты ведут себя одинаково и они отражают одинаковую в каждом случае функцию. То есть у них всегда одинаковое количество выводов устройства и они отличаются только графическим расположением выводов устройства.

Условие:

Вы открыли проект и обрабатываете символ, для которого существуют еще ***не*** все варианты.

1. Сервисные программы > Основные данные > Символ > Создать вариант
2. В диалоговом окне Генерировать варианты выберите создаваемый вариант из списка еще не созданных вариантов.
3. В следующем диалоговом окне Генерировать варианты выберите вариант, который должен использоваться в качестве Исходного варианта, причем в этом диалоговом окне перечислены в качестве возможных исходных вариантов только уже имеющиеся варианты.
4. В поле Поворачивать на выберите, насколько градусов должен быть повернут исходный вариант, чтобы создать новый вариант.
5. Активируйте флажок Отобразить зеркально графику у оси Y, если графика символа должна быть зеркально повернута вокруг оси Y (через нулевую точку).
6. Активируйте флажок Поворачивать обозначение вывода устройства, если текст обозначения вывода устройства должен быть повернут.
7. Активируйте флажок Поворачивать размещенное свойство, если размещенные свойства должны быть повернуты.
8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Создается вариант и отображается на новой странице <Библиотека символов>;<Имя символа>:<Номер символа>;<Обозначение варианта> (например новый.тест:1;A для варианта A символа тест с внутренним номером символа 1 библиотеки символов новый.slk). Кроме того, на панели инструментов редактора символов становится активной кнопка для нового варианта так, что можно легко переключаться между вариантами одного символа.

!!! note "Замечание:"

    если созданы все варианты, то функция Создать вариант становится недоступной.

Условие:

Вы обрабатываете символ, имеющий, как минимум, ***один*** вариант, и активировали графический образ удаляемого варианта.

1. Выберите пункты меню Сервисные программы > Основные данные > Символ > Удалить вариант.
2. Выберите необходимый ответ в запросе.

Для быстрого переключения между графическими образами вариантов символа выполните следующие действия:

Условие:

Вы обрабатываете символ, для которого существует, как минимум, ***один*** вариант.

1. Активируйте панель инструментов Редактор символов (Всплывающее меню > Редактор символов в строке меню EPLAN или в любой другой панели инструментов), если она еще не отображается.
2. Щелкните по кнопке варианта, графический образ которого Вы хотите увидеть.
3. Ответьте на предупреждение о том, что изменения, сделанные на текущей странице, еще не сохранены.

!!! note "Замечание:"

    При просмотре символов в редакторе символов всегда отображаются аналогичные варианты символов. Если для определенного символа вариант не существует, в редакторе отображается пустая страница с замечанием "Вариант не существует".

!!! example "Пример:"

    Символ1имеет вариантыA,B,CиD, символ2имеет вариантыA,BиD, а символ3имеет вариантыA,CиD.Если у символа1настроен вариантCи в редакторе символов через него просмотрено 3 символа, то при символе2появится информация, что вариант не существует. При символе3, как настроено, отобразится вариантC.

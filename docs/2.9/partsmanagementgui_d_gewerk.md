# Диалоговое окно Раздел/подраздел

Сервисные программы > Изделие > Управление. В дереве или списке выделено одно изделие. Выберите вкладку Общ.  и в поле Раздел / подраздел щелкните по кнопке ++...++.

В этом диалоговом окне определяются разделы, которым присвоено изделие.

Обзор основных элементов диалогового окна:

### Панель инструментов

Для раздела (напр., электротехника, гидравлика) с помощью кнопки ![](images/all_new_as.png){: .ui-icon } (Создать) можно открыть диалоговое окно Создать подраздел и создать новый подраздел. Если был создан подраздел, дополнительно доступна кнопка ![](images/all_edit_as.png){: .ui-icon } (Обработать), с помощью которой можно переименовать подраздел.

### Таблица

В таблице перечисляются имеющиеся разделы и подразделы. Установив флажок в столбце Использовать, задайте дополнительные разделы, которым следует присвоить изделие.

!!! note "Замечание:"

    Если изделию присвоено больше одного раздела, при вставке изделия в качестве устройства с функциями Fluid-техники (в присвоенном макросе или в качестве шаблона функций) на схему соединений откроется диалоговое окно Выбрать раздел. В этом диалоговом окне перечисляются все разделы Fluid-техники; в этом случае выберите подходящий для устройства раздел.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

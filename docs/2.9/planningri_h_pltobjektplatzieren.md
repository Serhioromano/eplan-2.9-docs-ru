# Разместить технологический контур / функцию ТК

Технологическому контуру / функции ТК при размещении на функциональной схеме автоматизации автоматически присваивается функциональный символ. Возможно многократное размещение технологического контура / функции ТК, например на разных страницах. Из навигатора предварительного планирования через пункт всплывающего меню Перейти к (графика) можно перейти к размещенному технологическому контуру / функции ТК.

Условия:

* Вы открыли проект, который содержит данные предварительного планирования.
* Технологическому контуру / функции ТК был присвоен тип ТК (уст./реал.).
* Открыта страница типа "Функциональная схема автоматизации" в графическом редакторе.

1. Выберите следующие пункты меню: Проект > Предварительное планирование > Навигатор
2. Выделите технологический контур / функцию ТК.
3. Перетащите мышью технологический контур / функцию ТК на страницу.
4. Щелкните на странице по требуемой позиции, установите необходимый размер символа.
5. Для размещения символа щелкните по странице.
6. Для выбора другого символа нажмите перед отпусканием символа клавишу Backspace и выберите другой символ в диалоговом окне Выбор символа.

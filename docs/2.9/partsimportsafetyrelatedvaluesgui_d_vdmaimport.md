# Диалоговое окно "Присвоение VDMA 66413"

Сервисные программы > Изделие > Управление. Через меню ++Дополнительно++ > Настройки была открыта база данных изделий. [Дополнительно] > Импортировать. Выберите в поле Тип файла запись "VDMA 66413" и щелкните по кнопке ++...++ рядом с ней.

В этом диалоговом окне указывается, какое идентифицирующее свойство в файле XML соответствует какому однозначному свойству изделия в базе данных изделий.

Доступны следующие элементы диалогового окна:

### VDMA-свойство

Выберите из раскрывающегося списка нужное идентифицирующее свойство:

* Identifier: специфический для производителя уникальный идентификатор устройства
* PartNumber: специфический для производителя номер изделия или номер для заказа.

### Свойство изделия

Выберите из раскрывающегося списка свойство EPLAN (Номер изделия, Номер ERP, Номер для заказа и т. д.), соответствующее этому VDMA-свойству. Если присвоение уникально, то данные переносятся с параметрами, важными с точки зрения безопасности.

!!! note "Замечание:"

    Если при импорте данных изделия с типом файла 'VDMA 66413' происходит присвоение свойству EPLAN Номер изделия, при таком импорте можно также создавать новые изделия. В этом случае перед импортом необходимо выбрать опцию Обновить имеющиеся записи данных и добавить новые. Напротив, если необходимо дополнить уже имеющиеся изделия, выберите в диалоговом окне импорта опцию Обновить только имеющиеся записи данных.

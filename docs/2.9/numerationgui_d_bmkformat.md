# Диалоговое окно Формат ОУ

* Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация (online). Щелкните в диалоговом окне по кнопке Настройки: Нумерация (online) рядом с полем Формат нумерации по ++...++. Щелкните на вкладке ОУ по кнопке ++...++.
* Сервисные программы > Изменение стандарта. Введите необходимые данные во вкладках Проект, Поворот / инверсия, Основные данные, Символы и Структура, нажимая каждый раз [Далее]. Щелкните на вкладке Нумерация рядом с полем Нумерация устройств по кнопке ++...++. Щелкните на вкладке ОУ по кнопке ++...++.

В этом диалоговом окне можно выбрать и обработать элементы для формата нумерации в режиме подключения для общих устройств, клеммников, штекеров, черных ящиков, ПЛК, кабелей, устройств Fluid или распределителей и соединителей Fluid-техники.

Доступны следующие элементы диалогового окна:

### Доступные элементы формата

В этом поле представлены все доступные элементы формата. Эти элементы формата либо являются частью формата или указывают, каким образом должна выполняться нумерация. Чтобы использовать элемент формата для формата, его необходимо скопировать в поле Выбранные элементы формата, использовав кнопку ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо).

Элемент формата |  Предварительный просмотр |  Значение
---|---|---
Буквенное обозначение |  ***I***  |  Выводит буквенное обозначение устройства.
Счетчик |  ***C***  |  Выводит счетчик устройства.
Страница |  ***S***  |  Здесь указано имя страницы, на которой находится устройство.
Столбец |  ***P***  |  Здесь указан столбец, в котором находится устройство.
Строка |  ***R***  |  Здесь указана строка, в которой находится устройство.
Разделитель |  |  Разделяет элементы формата друг от друга. Разделитель можно вставлять только ***после*** буквенного обозначения и ***перед*** счетчиком. Если в качестве разделителя используется точка, знак 'минус' или предшествующий знак структурного идентификатора, вокруг составного элемента формата, между буквенным обозначением и счетчиком, автоматически вставляются круглые скобки. Отображение скобок можно включить или выключить при помощи флажка Скрыть скобки в ОУ (в пунктах меню Параметры > Настройки > Проекты > 'Имя проекта' > Устройство > ОУ).

!!! example "Пример:"

    Вы выбрали следующие элементы формата online-нумерации:Страница Буквенное обозначение Разделитель Столбец Разделитель Счетчик Для каждого из этих разделителей выбирается точка. В ОУ составной элемент формата 'Разделитель / Столбец / Разделитель' заключается в круглые скобки. Пример: предохранитель-2F(.4.)3.

### Выбранные элементы формата

В этом поле выводятся все элементы формата, из которых состоит текущий формат. Текущий формат выводится в поле Предварительный просмотр.

Кнопка |  Значение
---|---
![](images/all_edit_as.png){: .ui-icon } (Обработать) |  Обеспечивает обработку выделенного элемента формата.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет выделенный элемент формата.
![](images/all_arrowup_as.png){: .ui-icon } (На одну строку вверх) |  Позиционирует выделенный элемент формата на одну строку вверх. Тем самым можно задать последовательность элементов формата.
![](images/all_arrowdown_as.png){: .ui-icon } (На одну строку вниз) |  Позиционирует выделенный элемент формата на одну строку вниз. Тем самым можно задать последовательность элементов формата.

### Предв. просмотр

Это поле используется для предварительного просмотра формата. Формат состоит из элементов формата, указанных в поле Выбранные элементы формата.

### Предш. цифру не изменять

Эта функция относится к тем ОУ, которые имеют предшествующую цифру. Если этот флажок установлен, предшествующая цифра в ОУ не убирается. Если этот флажок установлен, предшествующая цифра в ОУ убирается. С помощью этой функциональности можно, например, определить, оставить ли номер схемы в ОУ для устройств Fluid.

### Буквенные элементы счетчика не изменять

Эта функциональность используется для счетчиков и обозначений, имеющих буквенные элементы. Если этот флажок установлен, цифровой элемент считает по нарастающей, а буквенный элемент остается без изменений.

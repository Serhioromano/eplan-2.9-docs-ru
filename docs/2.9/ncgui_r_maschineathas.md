# Вкладка Машина (Автоматическое оснащение Rittal — Athex)

* Параметры > Настройки > Фирма > Автоматическое оснащение > Rittal — Athex. Выберите вкладку Машина.
* Сервисные программы > Данные изготовления > Автоматическое оснащение Rittal — Athex. Рядом с полем Машина нажмите ++...++. Выберите вкладку Машина.

На этой вкладке определяются машинные настройки для системы автоматического оснащения Rittal — Athex.

Обзор основных элементов диалогового окна:

### Максимальная длина шины

Введите в этом поле максимальную длину несущих шин, которыми должна оснащаться машина. На машине можно обрабатывать шины длиной от 150 до 1200 мм. Несущие шины, длина которых превышает указанные здесь значения, не экспортируются.

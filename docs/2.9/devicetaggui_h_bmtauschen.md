# Заменить устройства

При замене двух устройств ***все*** функции соответствующих устройств изменяются. ОУ заменяется.

Условия:

* Вы открыли проект.
* Вы открыли страницу проекта в графическом редакторе или навигатор данных проекта.

1. Выделите любое условное обозначение / любую функцию обоих устройств, которые вы хотите заменить.
2. Выберите пункты меню Данные проекта > Устройствo > Заменить.

!!! info "Для сведения:"

    Все функции обоих выбранных устройств получают ОУ другого устройства.

# Вкладка Макрос

Вы открыли проект.

* Вы сгенерировали объект планирования в навигаторе предварительного планирования или вставили его в Графическом редакторе. Выберите в диалоговом окне Свойства ++...++ вкладку Макрос.
* Вы выделили объект планирования в навигаторе предварительного планирования или в Графическом редакторе. Обработать > Свойства (или Всплывающее меню > Свойства). Выберите вкладку Макрос.

На этой вкладке объекту планирования присваивается файл макроса с макросами окна или страницы. Если макрос содержит объекты-заполнители, здесь отображаются объекты-заполнители и переменные и вы можете обрабатывать значения объектов-заполнителей. Эти значения используются при размещении макроса.

Обзор основных элементов диалогового окна:

### Макрос

Здесь отображается имя макроса. Нажав на ++...++, вы откроете выбор макроса и сможете выбрать макрос.

Если отображаемое здесь имя макроса нельзя изменить, значит макрос происходит из присвоенного [шаблона сегмента](planninggui_r_planungsobjekt.md).

В ***информационной строке*** под этим полем в определенных случаях предоставляется информация по присвоенному макросу. Это происходит, например, если уже присвоенный макрос впоследствии был изменен и наборы значений на объекте планирования не соответствуют наборам значений в присвоенном макросе:

* Если с помощью макроса уже сгенерирована ***страница схемы соединений***, вы можете обновить данные на объекте планирования посредством кнопки ![](images/all_update_as.png){: .ui-icon }.
* Если ***страница схемы соединений еще не сгенерирована***, данные из макроса переносятся автоматически при первом открытии диалогового окна "Свойства" на объект планирования.

С помощью контрольного прогона 028005 можно найти отличающиеся данные на объекте планирования и макросе.

!!! tip "Совет:"

    В навигаторе предварительного планирования с помощью пункта всплывающего меню Обновить данные заполнителя можно обновить данные объектов-заполнителей на нескольких объектах планирования одновременно. Для обновления данных заполнителей на нескольких шаблонах сегментов одновременно, этот пункт всплывающего меню также доступен в навигаторе шаблонов сегментов.

* * *

Групповое поле "Свойства"

В таблице отображаются определенные в макросе объекты-заполнители и переменные. Значения переменных можно выбрать, используя выбор набора значений, или обработать вручную.

Выберите в строке Набор значений набор значений, после этого весь столбец заполняется значениями из этого набора значений.

Для выполнения обработки щелкните двойным щелчком по ячейке, а затем по ++...++, откроется диалоговое окно Выбор свойств, где вы сможете выбрать свойство объекта планирования. Тогда в столбце Значение вместо конкретного значения будет отображаться номер выбранного свойства. Используя свойства блока, можно получить доступ также к свойствам других объектов, связанных с объектом планирования, например, к свойствам изделия, главной функции или данным ПЛК.

При размещении макроса значения однократно определяются из свойств объекта планирования или связанных с ним объектов и переносятся на размещенные функции.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

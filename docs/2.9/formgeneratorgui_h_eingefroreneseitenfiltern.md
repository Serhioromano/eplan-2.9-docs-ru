# Отобразить / фильтровать необновляемые страницы отчета

Эта функция дает возможность фильтровать [необновляемые страницы отчета](formgeneratorgui_k_eingefrorseiten.md). Также можно отобразить число необновляемых страниц отчета в статистике проекта.

Условия:

* Вы открыли проект.
* Вы зафиксировали страницы отчета.

### Показать число необновляемых страниц отчета

1. Выберите следующие пункты меню: Страница > Навигатор
2. Выделите уровень структурного идентификатора.
3. Выберите пункты меню Проект > Свойства.
4. В диалоговом окне Свойства проекта — <Имя проекта> выберите вкладку Статистика.

!!! info "Для сведения:"

    В столбце Необновляемые страницы отображается число необновляемых страниц отчета для каждого типа страниц.

### Фильтровать необновляемые страницы отчета

1. Выберите следующие пункты меню: Страница > Навигатор
2. В навигаторе страниц рядом с полем Фильтр нажмите ++...++.
3. В диалоговом окне Фильтр рядом с полем Схема нажмите кнопку ![](images/all_new_as.png){: .ui-icon } (Создать) и в открывшемся диалоговом окне создайте схему.
4. Нажмите кнопку ![](images/all_new_as.png){: .ui-icon } (Создать) на панели инструментов для критериев фильтра.
5. В диалоговом окне Выбор критериев под уровнем иерархии "Свойства" выберите свойство Необновляемая страница отчета.
6. В диалоговом окне Фильтр установите для этого критерия соответствующий флажок в столбце Значение.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Таким образом сгенерированный фильтр автоматически вносится в поле Фильтр навигатора страниц и активируется.

!!! info "Для сведения:"

    Все обновляемые страницы отчетов скрыты. Отображаются только необновляемые страницы отчетов.

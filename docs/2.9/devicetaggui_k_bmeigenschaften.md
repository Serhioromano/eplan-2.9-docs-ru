# Свойства устройства

Если в схему соединений вставляется условное обозначение, то в нем заложена определенная функция. Например, если в схеме соединений размещается переключатель, то генерируется новое устройство, которому присваивается определение функции "Переключатель". Определение функции несет с собой логическую модель; тем не менее, у функции можно изменить определенные свойства логической схемы выводов устройства.

Каждой функции присвоены определенные группы свойств. Свойства подразделяются на идентифицирующие и описывающие свойства.

***Идентифицирующие*** свойства:

* Обозначение устройства
* Обозначение вывода устройства
* Определение функции
* Вид представления.

***Описывающие*** свойства функции можно настроить в диалоговом окне Свойства ++...++. Описывающие свойства — это, например:

* Технические параметры
* Функциональный текст
* Текст гравировки
* Место монтажа (описывающее)
* Дополнительные поля
* Примечание
* Перекрестная ссылка вручную

### Отображение свойств устройства

Диалоговое окно Свойства ++...++ отображает все свойства выделенных устройств в тематических вкладках. При этом любые свойства можно размещать и, следовательно, отображать по отдельности, причем учитываются и многострочные тексты. Такие отдельные размещенные свойства могут быть присоединены друг к другу и образовывать, таким образом, общий блок; это позволяет Вам перемещать все (требуемые) тексты одновременно и предотвращает перекрытие многострочных текстов другими текстами.

Символам в общем присваиваются ***все*** свойства, имеющие значение для символа (и возникающего позже условного обозначения). При этом уже предварительно задается позиция и последовательность свойств, а также свойства их представления (напр., слой). В этом случае на вкладке Отображение можно решать, соответствуют ли эти заданные значения вашим требованиям или же определенные свойства должны представляться в другом порядке.

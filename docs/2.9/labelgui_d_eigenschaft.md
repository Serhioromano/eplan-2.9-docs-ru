# Диалоговое окно Свойство — <Тип свойства>

Вы открыли проект. Сервисные программы > Данные изготовления > Экспорт / маркировка. В диалоговом окне Экспортировать данные изготовления / Вывести маркировку рядом с полем Настройки щелкните по кнопке ++...++. В диалоговом окне Настройки: Экспорт данных изготовления / маркировка выберите одну из вкладок Верхний колонтитул, Нижний колонтитул или Этикетка, выделите элемент формата для свойства или для данных проекта и нажмите ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо).

В этом диалоговом окне выберите свойство для текста экспорта данных изготовления / маркировки и укажите способ форматирования текста.

Обзор основных элементов диалогового окна:

### Категория

Выберите в этом поле категорию, чтобы ограничить просмотр свойств в находящемся ниже поле.

### Фильтр

Введите в это поле текст, по которому должна выполняться фильтрация свойств.

Если необходимо вернуться к прежнему виду свойств, удалите текст из поля ввода. Для этого нажмите кнопку ![](images/all_delete_as.png){: .ui-icon } (Удалить).

### Свойство/Присвоение

Как только в поле Категория будет выделено какое-либо свойство, вы сможете обрабатывать свойства его отображения в таблице, расположенной в правой области диалогового окна. Для этого существуют уровни иерархии Формат и Значение/единица измерения, которые открываются после щелчка по символу ![](images/all_plussign_as.png){: .ui-icon }.

* * *

На уровне иерархии Формат имеются следующие настройки:

### Длина

С помощью этого раскрывающегося списка определяется длина соответствующих текстов свойств:

* Неограниченно перемен.: Если выбран этот параметр, длина текста не ограничивается.
* Ограничено перемен.: Если выбран этот параметр, длина текста ограничивается. Если текст выделенного свойства не входит в поле, то он укорачивается до длины, указанной в поле Число символов.
* Фиксированная: Если выбран этот параметр, длина текста фиксирована. Эта длина вводится в поле Число символов.

### Число символов

Это поле доступно в том случае, если выбрана опция Фиксированная или опция Ограниченно перемен. Число в этом поле указывает на длину текста.

### Вывести по правому краю

Это поле доступно, если выбран параметр Фиксированная. Если установлен этот флажок, текст выделенного свойства выводится по правому краю.

### Печать разрывов

Тексты свойств могут содержать знаки абзаца в качестве заполнителя для разрыва строки. Если этот флажок установлен, то в файле экспорта / маркировки в тексте свойств выделенного свойства для каждого разрыва строки-заполнителя будет вставлена новая строка.

### Разделитель для индекса 0

Эта функция относится только к элементам формата, являющимися вышестоящими элементами-заполнителями. Вышестоящий элемент-заполнитель обозначается как индекс 0, как, например, "Изделие клеммы". Он содержит нижестоящие элементы-заполнители с индекса [1] до индекса [10], как, например, "Изделие клеммы [1]" до "Изделие клеммы [10]". Вышестоящий элемент-заполнитель "Изделие клеммы" выводит изделия-клеммы всего клеммника. В этом поле отображается разделитель, заданный при помощи ++...++ в диалоговом окне Формат: Разделитель. Этот разделитель вставляется между данными проекта.

### Суммировать в одной строке

Этот флажок доступен только для групповых спецификаций изделий!
Установите этот флажок, чтобы вывести сумму отдельных значений соответствующего свойства. Таким образом, при помощи этой настройки можно вывести как общее количество (число штук) отдельных изделий, так и общую длину, например, изделия кабеля.

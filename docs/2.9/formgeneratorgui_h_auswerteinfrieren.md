# Закрепить страницы отчета

Вы можете закрепить страницы отчета для исключения обновления. Тогда данные проекта в текстах заполнителя на [закрепленных страницах отчета](formgeneratorgui_k_eingefrorseiten.md) больше не будут обновляться. Обновляемые и необновляемые страницы отчета в навигаторе страниц обозначены разными пиктограммами.

Условия:

* Вы открыли проект.
* Вы сгенерировали страницы отчета.

1. Сервисные программы > Отчеты > Генерировать
2. Выберите вкладку Отчеты.
3. Выделите один блок отчетов.
4. Выберите пункт всплывающего меню Закрепить страницы отчета.

!!! info "Для сведения:"

    Блок отчетов удаляется. Сохранятся страницы отчета.

!!! tip "Совет:"

    Чтобы генерировать новые страницы отчета вам сначала нужно удалить вручную существующие страницы и затем создать новые. Если вы не удалите страницы отчета, то будут созданы дополнительные страницы отчета.

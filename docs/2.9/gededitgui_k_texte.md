# Тексты

В EPLAN имеются следующие виды текстов:

* ***Текст*** — любые текст, которые можно произвольно разместить на странице.
* ***Тексты свойств*** используются с целью представления свойств условного обозначения для соответствующего символа в схеме соединений.
* С помощью ***специальных текстов*** выводится значение свойства; эти тексты автоматически обновляются при изменении данного свойства. Это могут быть свойства страницы или проекта; функциональный текст зоны также является специальным текстом.
* ***Тексты заполнителей*** имеют много общего со специальными текстами, однако они используются в формах, чтобы определить позиции для размещения полей в записи данных. Во время выполнения отчета EPLAN автоматически заменяет эти заполнители на значения соответствующих объектов проекта, для которого выполняется отчет, например функциональных элементов, страниц, символов и т. д.

С помощью буфера обмена можно обмениваться текстами между двумя разными приложениями. При этом сохраняются имеющиеся переходы на новую строку.

С помощью непосредственной обработки (Параметры > Прямая обработка) вы можете включить режим обработки, который позволяет непосредственно изменять отдельные графические тексты. Если включен режим обработки, то это выводится на панель состояния с помощью рамки обработки и над пиктограммой ![](images/inplaceeditinggui_icon_as.png){: .ui-icon }.

### Форматирование текстов

Все тексты можно форматировать, т.е. можно произвольно изменять их размер, цвет, вид шрифта, стиль шрифта и т.д.

Возможные виды шрифтов определяются в настройках проекта и фирмы. При форматировании текста в проекте можно использовать эти предварительно настроенные виды шрифтов. Виды шрифтов не сохраняются в проекте.

Кроме того, определенные специальные символы можно вставить с помощью диалогового окна выбора, которое вызывается при обработке текстов через всплывающее меню.

### Отображение вертикальных текстов

Во многих азиатских языках, например китайском, японском или корейском, текст набирается сверху вниз. Такие тексты можно отображать и печатать с помощью так называемых "@-Fonts" (типов шрифтов, которым предшествует знак "@").

Используя @-Fonts, можно набирать текст в диалоговом окне в горизонтальном направлении, при этом символы будут правильно выровнены и доступны для чтения. Однако при выводе текста на страницу проекта азиатские печатные символы отображаются в перевернутом виде.

Выберите угол текста -90°, и азиатские символы будут отображаться на странице проекта в правильном направлении (сверху вниз), оставаясь доступными для чтения. Однако латинские буквы и цифры будут отображаться в перевернутом виде. После экспорта PDF или печати страницы эти символы также будут отображаться сверху вниз.

!!! example "Пример:"

    Текст вводится на китайском. Выбранный тип шрифта — "@SimSun". При угле текста 0° китайские печатные символы отображаются на странице проекта в перевернутом виде: они как бы лежат на боку.При повороте текста на -90° они будут отображаться в правильном направлении: сверху вниз. Однако латинские буквы и цифры будут отображаться в перевернутом виде.После экспорта PDF или печати страницы все символы будут отображаться сверху вниз.

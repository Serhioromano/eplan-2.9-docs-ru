# Режимы обработки при открытии проектов

Проекты можно открывать в различных режимах обработки:

* ***Стандарт:*** Возможна обработка проекта. Если вы работаете в многопользовательском режиме, открытый таким образом проект могут обрабатывать сразу несколько пользователей.
* ***Эксклюзивный:*** Проект можете обрабатывать только вы. Если вы работаете в многопользовательском режиме, другие пользователи не имеют доступа к открытому таким образом проекту.
* ***С защитой от записи:*** Проект доступен только для просмотра, но не для обработки. Когда вы работаете в многопользовательском режиме, другие пользователи не могут обрабатывать данный проект.
Для обозначения защищенных от записи проектов в представлениях навигаторов в виде дерева и в виде списка используется серый цвет фона. Дополнительно доступны [отдельные цветовые настройки](settingsmastergui_d_oberflaecheneinstellungen.md) для представления открытых защищенных от записи страниц и пространств листа.

!!! note "Замечание:"

    * Режим обработки также отображается в списке последних проектов. Если вы открываете проект путем выбора значений из этого списка, проект будет открыт в указанном там режиме обработки, который использовался последним.
    * Если в настройках проекта (Параметры > Настройки > Проекты > "Имя проекта" > Управление > Общее) установлен флажок Синхронизировать основные данные проекта при открытии, то при открытии "устаревшего" проекта появится запрос Основные данные проекта. При помощи кнопки ++Определено пользователем++ можно перейти в диалоговое окно [Синхронизация основных данных — <Имя проекта>](prjmanagementgui_d_stammdatenabgleich.md) и скопировать специально отобранные основные данные. После нажатия кнопки ++Да++ основные данные проекта будут автоматически обновлены. Если вы нажмете [Нет], основные данные не будут обновлены, и вы сможете провести синхронизацию основных данных проекта позже.

!!! tip "Совет:"

    Проекты можно открывать путем перетаскивания, переместив с помощью мыши выделенный файл проекта *.elk из Windows Explorer в навигатор страниц.

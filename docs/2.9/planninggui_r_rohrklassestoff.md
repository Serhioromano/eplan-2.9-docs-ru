# Вкладка <Отображаемое имя класса трубы> / вкладка <Отображаемое имя вещества>

Вы открыли проект.

* Вы сгенерировали класс трубы или вещество в навигаторе шаблонов сегментов. В диалоговом окне Свойства ++...++ выберите вкладку <Отображаемое имя класса трубы> или <Отображаемое имя вещества>.
* Вы выделили класс трубы или вещество в навигаторе шаблонов сегментов. Обработать > Свойства. Выберите вкладку <Отображаемое имя класса трубы> или <Отображаемое имя вещества>.

На этой вкладке указываются свойства выделенного класса трубы или выделенного вещества.

Обзор основных элементов диалогового окна:

### Идентифицирующее имя

Введите здесь имя класса трубы / вещества. Вносимое здесь имя по умолчанию отображается в навигаторе шаблонов сегментов на классе трубы / веществе.

### Описание

Введите здесь описывающий текст для класса трубы / вещества. Этот текст по умолчанию отображается в навигаторе шаблонов сегментов вместе с именем на классе трубы / веществе.

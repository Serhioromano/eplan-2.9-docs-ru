# Вкладка Точка разрыва

Вы открыли проект и находитесь на странице схемы соединений в Графическом редакторе.

* Вы разместили новую точку разрыва или дважды щелкнули по точке разрыва (или выбрали из всплывающего меню команду Свойства; в этом случае может быть также выделено несколько точек разрыва). В диалоговом окне Свойства ++...++ выберите вкладку Точка разрыва.
* Вы выделили, как минимум, ***одну*** точку разрыва. Данные проекта > Соединения > Сортировать точки разрыва. В диалоговом окне Сортировать точки разрыва выберите пункт всплывающего меню Свойства. В диалоговом окне Свойства ++...++ выберите вкладку Точка разрыва.
* Данные проекта > Соединения > Навигатор точек разрыва. Выделите в диалоговом окне Точки разрыва — <Имя проекта> как минимум ***одну*** точку разрыва и выберите из всплывающего меню пункт Свойства. В диалоговом окне Свойства ++...++ выберите вкладку Точка разрыва.

На этой вкладке укажите свойства выделенной точки (выделенных точек) разрыва. При этом доступно диалоговое окно Свойства ++...++ для ***размещенных*** точек разрыва.

Обзор основных элементов диалогового окна:

### Видимое ОУ

Введите здесь обозначение устройства, которое должно отображаться в Графическом редакторе у точки разрыва; или нажмите на ++...++, чтобы открыть диалоговое окно Использ. имеющ. точку разрыва и выбрать в нем точку разрыва, которой нужно присвоить запись данных.

### Полное ОУ

В этом поле отображается полное ОУ (т. е. структурный идентификатор текущего окружения + видимое ОУ) видимого ОУ / функционального элемента. С помощью кнопки ++...++ откройте диалоговое окно Полное ОУ. Здесь можно обработать структурный идентификатор и, например, выбрать другое место установки.

### Идентификатор сортировки

Ид. сортировки определяет последовательность, в которой выполняется сортировка точек разрыва. EPLAN задает его автоматически, но в этом поле он может быть изменен. (Стандартный идентификатор сортировки — 100, т.е. сортировка производится только по странице, строке и столбцу.)

### Описание

Здесь можно присвоить точке разрыва описательный текст. При использовании имеющихся точек разрыва этот текст отображается для первой точки разрыва (источника).

### Источник типа "звезда"

Установите этот флажок, если текущая точка разрыва должна быть связана со всеми другими точками разрыва как источник звездообразного расположения.

#  Графический выбор макроса

При вставке макросов у вас есть возможность использовать графическое диалоговое окно выбора. Вы словно "пролистаете" в этом диалоговом окне выбора всю структуру каталогов на основе сохраненных пиктограмм, символов или других графических файлов и легко найдете нужный макрос. Графический выбор макроса облегчает поиск и выбор макросов в сложной структуре каталогов. Используя специальную конфигурацию в настройках фирмы, можно адаптировать графический выбор макроса к особенностям предприятия.

Включить или выключить графический выбор макроса можно при помощи специфической для пользователя настройки Использовать графический выбор макроса. Эту настройку можно найти в пунктах меню Параметры > Настройки > Пользователь > Графическая обработка > Макросы.

Если флажок активирован, во всех местах платформы EPLAN, где выбраны макросы, используется графический выбор макросов. Это происходит при следующих действиях:

* Вставить макрос (Вставить > Макрос окна / символа и Страница > Макрос страницы > Вставить)
* Разместить макрос в навигаторах данных проекта (с помощью пункта всплывающего меню Разместить)
* Выбрать макрос в базе данных изделия.

# Диалоговое окно "Выбрать источник данных" (предварительное планирование, импорт Excel)

Вы открыли проект и навигатор предварительного планирования. Данные проекта > Предварительное планирование > Импортировать. В диалоговом окне Импортировать данные предварительного планирования — <Имя проекта> выберите в поле Тип источника данных запись "Excel", а затем щелкните в поле Источник данных на ++...++.

В этом диалоговом окне выбирается источник данных для импорта файла Excel.

Обзор основных элементов диалогового окна:

### Файл

В этом поле выберите файл Excel: щелкните по ++...++, чтобы в открывшемся диалоговом окне перемещаться по структуре каталогов для выбора требуемого файла.

### Таблица

В этом раскрывающемся списке показаны все рабочие листы и обозначенные области данных, находящиеся в выбранном файле Excel.

Для файла Excel с настроенным фильтром здесь отображается специальная запись, в имени которой встречается понятие "Фильтр". После выбора такой записи в диалоговом окне Синхронизировать данные предварительного планирования для импорта будут предложены только отфильтрованные значения файла Excel.

### Имя столбца в заголовке таблицы

Если установлен этот флажок, имена столбцов соответствующей таблицы данных выводятся в столбце Внешнее поле таблицы присвоений. Если он снят, имена столбцов игнорируются при импорте.

# Вкладка Вид (Отобразить отверстия)

* Вы открыли проект, который содержит пространства листа. В Графическом редакторе открыта страница проекта. Вы вставили 2D-отображение отверстий (Вставить > Графика > 2D-отображение отверстий) и разместили его. В диалоговом окне 2D-отображение отверстий выберите вкладку Вид.
* Вы открыли проект, который содержит пространства листа. В Графическом редакторе открыта страница проекта. Вы дважды щелкнули по двухмерному отображению отверстия. В диалоговом окне 2D-отображение отверстий выберите вкладку Вид.

На этой вкладке обрабатываются свойства двухмерных отображений отверстий.

Обзор основных элементов диалогового окна:

### Имя вида

Каждый вид идентифицируется через уникальное имя, сохраненное в соответствующем настроенном языке. Наличие нескольких обзоров с одинаковыми именами невозможно.

### Описание

Описание представляет собой любой многоязычный текст, подробно описывающий вид.

### Пространство листа

Двухмерное отображение отверстий может быть создано только для одного пространства листа. В этом раскрывающемся списке перечислены все созданные в проекте пространства листов. При первом открытии диалогового окна в нем отображается имя первого пространства листа проекта.

### Основные функциональные элементы

Здесь определите, что должно быть представлено в двухмерном отображении отверстия. Таким образом можно дополнительно ограничить отображаемое содержимое пространства листа. Щелкните мышкой по ++...++ для открытия диалогового окна [Выбор 3D-объекта](gededit3dgui_d_3dobjektauswahl.md) и выбора объектов, отображаемых в двухмерном отображении отверстия.

В двухмерных отображениях отверстий и в обзорах моделей вы можете также выбрать в качестве базисного функционального элемента компоненты, расположенные глубже в иерархии функциональных элементов, например монтажные поверхности. В случае с развертками шины это невозможно.

Если выбранные базисные функциональные элементы не содержат поверхности, релевантные для ЧУ, автоматически добавляется следующий функциональный элемент с поверхностью, релевантной для ЧУ.

### Настройка масштаба

Следующий способ можно настроить для вычисления масштаба.

* Автоматически: Масштаб вычисляется таким образом, что базисный функциональный элемент оптимально подходит к выделенной области. При этом используются только масштабы, рекомендуемые стандартом DIN ISO 5455.
* Подогнать: Масштаб вычисляется таким образом, что базисный функциональный элемент оптимально подходит к выделенной области. Тем не менее можно использовать любой возможный масштаб, например "1:12", но допускаются только целые числа.
* Задано вручную: При выборе данной настройки разблокируется поле Масштаб, и можно установить любой масштаб.

### Масштаб

Данное поле активно, если в поле Настройка масштаба выбран параметр "Задано вручную"; введите здесь масштаб в формате "1:x".

[Групповое поле "Свойства"](devicetaggui_r_gruppenfeldeigenschaften.md)

Для двухмерных отображений отверстий предлагаются следующие свойства:

* Маркировка функционального элемента: Здесь определяется схема, согласно которой маркируются функциональные элементы в двухмерном отображении отверстия. Щелкните по кнопке ++...++, чтобы открыть диалоговое окно [Настройки: Маркировка](gededit3dgui_d_einstellbeschriftung.md) и выбрать имеющуюся схему или создать новую.
* Поворот: При выборе значения из этого раскрывающегося списка содержание двухмерного отображения отверстий поворачивается на 90°, 180° или 270° против часовой стрелки. Это может быть полезно для оптимального использования вертикальных двухмерных отображений отверстий на страницах альбомного формата. При этом рамка двухмерного отображения отверстия не поворачивается. Тексты двухмерного отображения отверстия, повернутого на 90°, выводятся повернутыми под таким же углом. Исключение составляют тексты свойств, читаемые справа. Чтобы они оставались читабельными, поворот выполняется на -90°.
* Автоматическое проставление размеров: Здесь определяется схема, согласно которой в двухмерном отображении отверстий автоматически проставляются размеры функциональных элементов. Щелкните по кнопке ++...++, чтобы открыть диалоговое окно [Настройки: Автоматическое проставление размеров](gededit3dgui_d_einstellautobemassung.md) и выбрать имеющуюся схему или создать новую.
* Обзор модели: Занять точку наблюдения за монтажной поверхностью: Если флажок этого свойства установлен, [вид функционального элемента в опрокинутом положении](gededit3dgui_k_start.md) выравнивается по его монтажной поверхности. За счет этого осуществляется правильное ортогональное отображение в двухмерном отображении отверстий даже выровненного по косой функционального элемента. Сгенерированные на основе этого отображения размеры соответствуют реальным значениям.
* Отображать силуэты функциональных элементов: Если поставлен этот флажок, на экране отображаются базисные контуры экструдированных функциональных элементов (например, несущих шин и кабельных каналов) и боковые ограничительные линии 3D-объекта, сгенерированного из базисного контура. Отключение этой настройки ускорит вывод сложных двухмерных отображений отверстий, но скажется на качестве отображения функциональных элементов.

# Оценка и вывод списков труб / шлангов

По спискам трубных и / или шланговых соединений можно составлять отчеты и выводить эти списки посредством стандартных установок для свойств соединений или посредством определения функции размещенных точек определений соединений. Ниже описывается, как составить отчет по такому списку при помощи стандартных настроек и вывести его в качестве страницы проекта. В этом списке сводятся вместе все трубные и все шланговые соединения проекта Fluid-техники.

Условия:

* Вы открыли проект.
* Проект содержит одну или несколько страниц схем соединений Fluid-техники с устройствами разделов "Гидравлика" и "Пневматика".
* Между устройствами раздела есть соединения.
* Вы имеете доступ к формам таблицы соединений. (Выберите Сервисные программы > Отчеты > Генерировать, а затем [Настройки] > Вывод в страницах. Должна быть запись формы для типа отчета "Таблица соединений", например F27_FT004.)

* На функциональных подключениях всех условных обозначений Fluid-техники установлен ***тип вывода устройства*** Fluid. (Данный тип вывода устройства присваивается автоматически при вставке функций Fluid. Для проверки вызовите диалоговое окно Свойства ++...++ для соответствующей функции, перейдите на вкладку Данные символа функции и щелкните по кнопке ++Логика++.)
* В качестве стандартной настройки для всех соединений раздела "Гидравлика" установлено определение функции "Труба", а для всех соединений раздела "Пневматика" — "Шланг". (Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Свойства > вкладки Гидравлика, Пневматика.)
* В качестве стандартной настройки для всех Fluid-технических соединений разделов "Гидравлика" и "Пневматика" установлено поперечное сечение или диаметр. Кроме того, для всех гидравлических и пневматических соединений заданы длины. (Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Свойства > вкладки Гидравлика, Пневматика.)

1. Пометьте проект в навигаторе страниц.
2. Выберите пункт меню Сервисные программы > Отчеты > Генерировать.
3. В диалоговом окне Отчеты — <Имя проекта> переместите вкладку Отчеты на передний план.
4. В виде дерева выберите пункт всплывающего меню ++Создать++.
5. В диалоговом окне Определить отчет в поле Исходный проект выберите (если не было установлено предварительно) свой проект, а из списка Выбрать тип отчета выберите тип "Таблица соединений".
6. Щелкните по кнопке ++OK++.
7. В диалоговом окне Настройки — Таблица соединений возле раскрывающегося списка Фильтр щелкните по кнопке ++...++, чтобы создать новую схему фильтрации.
8. Щелкните в диалоговом окне Фильтр по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
9. В диалоговом окне Новая схема введите имя схемы, например "Трубы + шланги", и соответствующее описание.
10. Щелкните в диалоговом окне Фильтр справа над таблицей на ![](images/all_new_as.png){: .ui-icon } (Создать), чтобы установить критерии фильтрации.
11. В диалоговом окне Выбор критериев выделите запись "Соединение: Определение функции".
12. В первой строке таблицы диалогового окна щелкните под столбцом Значение по кнопке ++...++, чтобы выбрать значение для критерия "Определение функции".

13. В обзоре дерева диалогового окна Определения функций пройдите по ветке "Общая информация > Общая специальная функция > Соединение > Определение соединения" и выберите определение соединения "Труба".
14. В диалоговом окне Фильтр задайте следующий критерий фильтра во второй строке таблицы. Выберите запись "ИЛИ".
15. В третьей строке таблицы создайте последний критерий фильтра "Определение функции" со значением "Шланг".
16. Щелкните по кнопке ++OK++.

17. В диалоговом окне Настройки — Таблица соединений выберите в качестве критерия сортировки запись "Раздел".
18. Щелкните по кнопке ++OK++.

19. Установите в этом диалоговом окне, как должны вводиться таблицы соединений в имеющуюся структуру страниц вашего проекта. Укажите в полях Номер установки, Имя страницы и Дополнительное поле, номера листа соответствующие значения.
20. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Начнется создание отчета. После этого производится вывод страниц в проекте. Сгенерированная таблица соединений содержит трубные и шланговые соединения вашего проекта, при этом соединения перечислены отдельно по трубам и шлангам.

!!! note "Замечание:"

    Для анализа и вывода списка трубных / шланговых соединений посредством определения функции размещенных точек определения соединений необходимо выполнить следующие условия:

* На Fluid-технических соединениях должны располагаться точки определения соединения.
* Для каждой точки определения соединения необходимо определить следующие свойства:

    \- Поперечное сечение / диаметр
    \- Обозначение соединения
    \- Определение функции ("Труба" для раздела "Гидравлика" и "Шланг" для раздела "Пневматика")
    \- Соединение: Длина с единицей измерения.

# Генерировать данные изготовления для механической обработки

EPLAN Pro Panel позволяет выводить схемы сверления, относящиеся к функциональным элементам, в отношении 1:1 в виде PDF-файлов. Эти шаблоны, распечатанные в том же размере на плоттере или крупноформатном принтере, служат в качестве шаблона при сверлильных работах на монтажной плате.

Модули интерфейса ЧУ передают координаты и размеры отверстий, монтажных отверстий, а также остальные данные по изготовлению для обработки монтажных плат, дверей или боковых частей прямо на фрезерные и сверлильные станки с ЧУ. Запретные зоны для сверления экспортируются только в машины Perforex. Для изготовления пользовательских контуров ЧУ можно генерировать записи данных контуров специально для используемой машины.

Интерфейсы для экспорта данных изготовления доступны для таких машин ЧУ:

* Rittal - Perforex BC / Secarex
* Rittal - Perforex LC / Secarex (центр лазерной обработки)
* Steinhauer
* DXF.

Элементы электрошкафа автоматически подготавливаются к монтажу. Время изготовления сокращается за счет высокой степени автоматизации.

!!! note "Замечание:"

    Чтобы настроить генерацию монтажных отверстий посредством схем сверления на функциональных элементах с переменной длиной (несущих шинах, кабельных каналах и т. д.) в соответствии с принципом работы используемой машины, в диалоговом окне [Настройки: Исходная точка](cabinetgui_d_einstellungennullschnitt.md) можно задать исходную точку. Таким образом задается страница функционального элемента, на основе которой рассчитывается схема его сверления.

Во всех случаях экспорта машин из EPLAN Pro Panel в качестве заключительного сообщения отображается диалоговое окно, в котором перечисляются конкретные данные об экспортируемых компонентах или указываются причины, почему компоненты не могут быть экспортированы. В диалоговом окне отображается следующая информация:

* Имя экспортированного формата
* Имя использованной схемы и настроенного пути вывода
* Количество экспортированных компонентов
* Альтернативные варианты экспорта
* Перечень неэкспортированных компонентов с указанием причины, по которой они не были экспортированы
* Указание на возможно выводившиеся системные сообщения.

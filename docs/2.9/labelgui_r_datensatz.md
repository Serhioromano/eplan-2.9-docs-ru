# Вкладка Этикетка

* Вы открыли проект. Сервисные программы > Данные изготовления > Экспорт / маркировка. В диалоговом окне Экспортировать данные изготовления / Вывести маркировку рядом с полем Настройки щелкните по кнопке ++...++. В диалоговом окне Настройки: Экспорт данных изготовления / маркировка выберите вкладку Этикетка.
* Параметры > Настройки > Пользователь > Интерфейсы > Экспорт данных изготовления / маркировка. В диалоговом окне Настройки: Экспорт данных изготовления / маркировка выберите вкладку Этикетка.

На этой вкладке указывается формат для данных, которые выводятся / экспортируются на этикетках.

Обзор основных элементов диалогового окна:

### Доступные элементы формата

В этом поле представлены доступные элементы формата. Доступные элементы формата зависят от выбранного типа отчета.

### Выбранные элементы формата

В этом поле находятся элементы формата, из которых состоит формат данных этикетки.

### Этикетки рядом друг с другом

Цель в этом поле показывает, сколько этикеток должно быть выведено рядом друг с другом.

Если на вкладке Файл выбран тип файла "Файл Excel (*.xls, *.xlsx, *.xlsm, *.xlsb)", то при настройке ***без шаблона*** свойства выводятся в соответствующих нескольких столбцах, причем оцениваются сначала все свойства объекта 1, затем все свойства объекта 2 и т. д. Если выбрана настройка ***с шаблоном***, вывод осуществляется аналогичным образом, но для каждого столбца необходимо определить заполнитель.

!!! example "Пример:"

    Если вы хотите вывести три свойства и три этикетки рядом друг с другом, то в шаблоне необходимо определить заполнители для девяти столбцов.

### Разделитель между находящимися рядом друг с другом этикетками

Введите в этом поле разделитель, который должен быть установлен между находящимися рядом друг с другом этикетками.

!!! example "Пример:"

    Ввод Результат;K1 aNL1;K2 aNL2,K1 aNL1,K2 aNL2

### Ряды этикеток на странице

Число в этом поле указывает на то, сколько рядов этикеток должно быть выведено на странице. Если данное значение равно "1", то выводится 1 страница на изделие.

При достижении установленного числа со следующей строки начинается новый табличный лист. (Максимально возможное значение — "1 048 500 строк").

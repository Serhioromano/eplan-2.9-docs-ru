# Разместить точки определения соединения

Условие:

Вы выделили соединения в графическом редакторе или в диалоговом окне навигатора Соединения — <Имя проекта>. Или вы выделили одну или несколько страниц либо проект в диалоговом окне навигатора Страницы — <Имя проекта>.

1. Выберите следующие пункты меню: Данные проекта > Соединения > Нумерация > Разместить

!!! info "Для сведения:"

    После вызова пункта меню сначала автоматически генерируются новые соединения.

2. В диалоговом окне Разместить точки определения соединенийв поле Настройки выберите из раскрывающегося списка схему, которая будет определять настройки нумерации соединений. С помощью кнопки ++...++ откройте диалоговое окно Настройки: Нумерация соединений. Здесь вы можете определять настройки для нумерации и сохранять их в схеме.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Точки определения соединений будут размещены на страницах схемы соединений. Им будет присвоено обозначение "????", если относящееся к ним соединение еще не имеет никакого обозначения.

!!! note "Замечание:"

    Для точек определения соединений, размещенных при помощи нумерации соединений, все свойства будут пусты и сброшены на неопределенные значения, поэтому эти точки определения соединений ведут себя нейтрально.

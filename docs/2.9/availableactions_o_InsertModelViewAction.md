# Операция: InsertModelViewAction

**Параметр** |  **Описание**
---|---
PROJECTNAME  |  Имя проекта с полным путем файла (является опцией).
Если не задано, то выбранный проект используется, когда операция вызывается через интерфейс пользователя (напр., через сценарий или панель инструментов). При вызове из командной строки Windows следует определить PROJECTNAME или сначала следует использовать ProjectAction. В противном случае отобразится системное сообщение.
LAYOUTSPACE  |  Имя пространства листа, для которого необходимо сгенерировать обзор модели. Должно быть указано.
STRUCTURE  |  Структурный идентификатор, для которого необходимо сгенерировать обзор модели. Должен быть указан, если имя пространства листа в проекте неоднозначно.
PAGENAME  |  Полное имя страницы, на которую надо вставить обзор модели. Должно быть указано.
X  |  Координата X левого нижнего угла обзора модели. Должна быть указана.
Y  |  Координата Y левого нижнего угла обзора модели. Должна быть указана.
DX  |  Ширина обзора модели. Должно быть указано.
DY  |  Высота обзора модели. Должна быть указана.
VIEWNAME  |  Имя обзора модели. Должна быть указана.
DESCRIPTION  |  Описание обзора модели (многоязычная символьная строка).
ANGLE  |  Угол поворота содержания обзора модели. Возможные значения: 1: Содержание повернуто на 90° против часовой стрелки; 2: Содержание повернуто на 90° по часовой стрелке.
SELECTIONSCHEME  |  Имя схемы выбора.
STYLE  |  Стиль, в котором представляется обзор модели. Возможные значения: 0: Проволочная модель; 1: Скрытые ребра; 2: Оттенение; 3: Скрытые ребра / упрощенное представление; 4: Оттенение / упрощенное представление.
ITEMLABELING  |  Имя схемы, которая применяется для маркировки функционального элемента в обзоре модели.
VIEWPOINT  |  Направление просмотра для объектов в обзоре модели (точка наблюдения). Возможные значения: 0: Стандарт; 1: Внизу; 2: Вверху; 3: Слева; 4: Справа; 5: Спереди; 6: Сзади; 7: ЮВ изометрия; 8: ЮЗ изометрия; 9: СВ изометрия; 10: СЗ изометрия.
ROOTELEMENTS  |  Основной функциональный элемент для обзора модели.
SCALESETTING  |  Настройка масштаба для отображения объектов в обзоре модели. Возможные значения:
0: Автоматич.
1: Подогнать
2: Задано вручную
SCALE  |  Масштаб. Ввод необходим, если для SCALESETTING была выбрана настройка 2.
OBJECTID  |  Идентифицирующее имя сгенерированного обзора модели.

!!! example "Пример:"

    InsertModelViewAction
    /PROJECTNAME:C:\Projects\EPLAN\ESS_Sample_Project.elk
    /LAYOUTSPACE:L_Space1
    /PAGENAME:=EB3+ET1/1
    /X:0
    /Y:0
    /DX:40
    /DY:60
    /ANGLE:1
    /SELECTIONSCHEME:Scheme1
    /STYLE:3
    /ITEMLABELING:Label_Scheme_1
    /VIEWPOINT:5
    /SCALESETTING:0
    /OBJECTID:NewModelView

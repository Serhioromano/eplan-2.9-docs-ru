# Диалоговое окно Управление правами: Рабочий режим

Сервисные программы > Управление правами > Рабочий режим

В этом диалоговом окне определяется рабочий режим.

Обзор основных элементов диалогового окна:

Если выделить в левом поле папку Рабочий режим, справа в диалоговом окне можно будет настроить необходимый рабочий режим:

### Регистрация

EPLAN используется с включенной функцией управления правами. Каждый пользователь должен входить в систему под своим именем. Пользователь, вошедший в Windows, по умолчанию указывается в диалоговом окне регистрации EPLAN. Пользователю остается только ввести свой (заданный в управлении правами) пароль.

Регистрация импортированных из Windows пользователей в этом рабочем режиме не поддерживается, т. к. в EPLAN не выполняется проверка пароля в Active Directory.

### Регистрация с именем пользователя Windows

EPLAN используется с включенной функцией управления правами. Пользователь может войти в EPLAN напрямую, используя свое имя пользователя и пароль для Windows. Если вошедший в Windows (локально) пользователь зарегистрирован как пользователь EPLAN, вход в систему выполняется автоматически. Пользователю не нужно вводить свое имя и пароль в EPLAN.

В этом рабочем режиме также возможна автоматическая регистрация пользователей, импортированных из Windows (в качестве пользователей или групп Active Directory).

### Без управления правами

EPLAN используется без управления правами, т. е., EPLAN использует имя пользователя Windows без пароля и наделяет всех пользователей правами администратора EPLAN.

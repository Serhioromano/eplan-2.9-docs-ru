# Диалоговое окно Порядок свойств (редактор символов)

Вы открыли проект и открыли или создали символ в редакторе символов (Сервисные программы > Основные данные > Символ > ...).

* Вид > Порядок свойств.
* Выберите пункт всплывающего меню Свойства на имени символа в навигаторе страниц. В диалоговом окне Свойства символа — <Имя символа> выберите из раскрывающегося списка Категория пункт "Настройки". Нажмите ![](images/all_new_as.png){: .ui-icon } (Создать) и выберите из диалогового окна Выбор свойств одну из записей "Стандартный порядок свойств для варианта x". [OK]. Нажмите ++...++ в ячейке Значение свойства "Стандартный порядок свойств для варианта x".

В данном диалоговом окне выберите запись данных порядка для свойств, отображаемую сейчас в редакторе символов. Установленная в данный момент запись данных порядка отображается в строке заголовка после информации о символе, и для варианта ее можно определять индивидуально.

### Порядок свойств

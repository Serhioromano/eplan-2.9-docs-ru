# Печать страниц

Вы можете печатать отдельные страницы или все страницы проекта. При этом имеется возможность регулирования вывода страниц проекта в зависимости от настроек печати.

Условия:

* В навигаторе страниц вы выделили необходимые страницы (или проект) в представлении дерева или списка.
* Открыта страница в графическом редакторе.

1. Проект > Печать
2. В диалоговом окне Печать в групповом поле Принтер из раскрывающегося списка выберите необходимый принтер.
3. В групповом поле Область страницы установите, нужно ли распечатать выделенные страницы или все страницы текущего проекта.
Если диалоговое окно Печать вызвано из графического редактора, можно распечатать либо находящуюся в настоящее время на переднем плане страницу, либо все страницы проекта.
4. Установите также количество экземпляров и параметры сортировки.
5. Откройте с помощью [Настройки] диалоговое окно Настройки: Печать, чтобы выполнить там дальнейшие настройки.
6. Благодаря [Предв. просм. перед печатью] у вас имеется возможность перед распечаткой проверить влияние настроек. При помощи клавиш [Назад] и [Далее] пролистайте выбранные к печати страницы страницы.
7. Запустите процесс печати нажатием клавиши ++OK++ в диалоговом окне Печать или нажав ++Печать++ в диалоговом окне Предв. просм. перед печатью.

!!! note "Замечание:"

    Можно продолжать работать с другим проектом, а печать будет выполняться в фоновом режиме. Для этого нужно несколько раз запустить EPLAN.

!!! tip "Совет:"

    Если при печати страниц необходимо вывести только текущие страницы активного фильтра страниц, выделите в навигаторе страниц проект и в диалоговом окне Печать включите параметр Выделение.

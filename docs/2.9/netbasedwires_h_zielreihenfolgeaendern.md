# Изменить последовательность целей сетевых соединений

Последовательность целей сетевых соединений можно впоследствии изменять в диалоговом окне "Свойства" соответствующей точки определения сети.

Условия:

* Вы открыли проект и находитесь на странице схемы соединений.
* Вставлена точка определения сети.

1. Выделите в графическом редакторе точку определения сети и выберите пункты меню Обработать > Свойства.
2. Выберите в диалоговом окне Свойства ++...++ вкладку Сетевые соединения.
3. Выделите вывод устройства в таблице Выводы устройства.
4. Выберите Всплывающее меню > Изменить последовательность целей.

!!! info "Для сведения:"

    В диалоговом окне Последовательность целей отображаются все цели вывода устройства, которые связаны посредством сетевых соединений.

5. Переместите цели в нужную позицию при помощи кнопки со стрелками.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Измененная последовательность целей будет сохранена.

# Функции формы: Блок выравнивания для заполнителя

В форму могут вноситься заполнители, заменяемые текстом или графикой (например, перемычками) при генерировании списков отчетов.

В частности, на схемах подключения клемм может в графическом виде выдаваться различная информация:

* любая, но фиксированная графика, например, для "винтиковых символов"
* переменная графика на каждую запись данных, например, для особых типов клемм (PE / PEN)
* сгенерированная графика для линий кабелей, блоков устройств и вилок соединений
* Символьная графика.

При этом, по крайней мере, частично требуется, чтобы графика была масштабированной, и ее растяжение было уже установлено в форме. Кроме того, графика с учетом ее блока выравнивания располагается с выравниванием. Эти настройки выполняются на вкладке Формат на уровне иерархии Блок выравнивания.

Для выравнивания текста вместе с этим выравниванием блока выравнивания доступны еще две другие настройки, которые можно активировать как на вкладке Формат, так и на уровне иерархии Рамка текста.

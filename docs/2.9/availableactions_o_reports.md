# Операция: reports

**Параметр** |  **описание**
---|---
TYPE  |  Вид выполняемой задачи:
PROJECT: Обновление всех отчетов проектов.
PAGES: Обновление всех отчетов, имеющих тот же тип, что и указанные страницы. Обновляется только тот тип отчетов, к которому принадлежит страница.
UPDATEMODELVIEWPAGES: Обновление обзоров модели на указанных страницах.
CREATEMODELVIEWS: Генерирование обзоров модели.
CREATECOPPERUNFOLDS: Генерирование разверток шины.
CREATEDRILLINGVIEWS: Генерирование отображения отверстий.
PROJECTNAME |  Имя проекта с полным путем файла (является опцией).
Если не задано, то выбранный проект используется, когда операция вызывается через интерфейс пользователя (напр., через сценарий или панель инструментов). При вызове из командной строки Windows следует определить PROJECTNAME или сначала следует использовать ProjectAction.
PAGENAME |  Имя обновляемой страницы (необязательно)
Если не указано, будет обновлена только данная страница отчета. Этот параметр оказывает влияние, только когда параметр TYPE имеет одно из следующих значений: PAGES, UPDATEMODELVIEWPAGES.
USEPAGEFILTER |  Определяет, должны ли использоваться только отфильтрованные страницы или все страницы проекта (необязательно). Соответствует активации фильтра в интерфейсе пользователя. Этот параметр оказывает влияние, только когда параметр TYPE имеет одно из следующих значений: PAGES, UPDATEMODELVIEWPAGES.
Значение по умолчанию: 0
PAGEFILTERNAME |  Имя фильтра страницы. На страницах обновляются только те отчеты, которые прошли через фильтр с именем PAGEFILTERNAME. Этот параметр оказывает влияние, только когда параметр TYPE имеет одно из следующих значений: PAGES, UPDATEMODELVIEWPAGES.
PAGENAMEn |  Имена страниц, для которых необходимо выполнить обновление отчетов (не обязательно). При этом n — это номер, например /PAGENAME1:=EB3+ET1/2 /PAGENAME2:=EB3+ET1/4 /PAGENAME3:=EB3+ET1/7 и т. д. Эти параметры оказывают влияние, только когда параметр TYPE имеет одно из следующих значений: PAGES, UPDATEMODELVIEWPAGES.
SELn |  Ид. объекта страниц, для которых необходимо выполнить обновление отчетов (необязательно). При этом n — это номер, например /SEL1:38/4/12/0. В качестве альтернативы для PAGENAMEn. Эти параметры оказывают влияние, только когда параметр TYPE имеет одно из следующих значений: PAGES, UPDATEMODELVIEWPAGES.

!!! example "Пример:"

    reports
    /TYPE:PROJECT
    /PROJECTNAME:C:\Projects\EPLAN\ESS_Sample_Project.elk

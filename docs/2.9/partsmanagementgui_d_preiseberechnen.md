# Диалоговое окно Рассчитать цены

Сервисные программы > Изделие > Управление. Открыта база данных элементов через [Дополнительно]> Настройки в диалоговом окне База данных изделий, под [Дополнительно]> Валюты установлены единицы валюты и коэффициенты пересчета для обеих единиц цены и выделены изделия, для которых должны рассчитываться цены. [Дополнительно] > Цены.

В этом диалоговом окне можно пересчитывать цены для выбранных изделий.

Доступны следующие элементы диалогового окна:

### Групповое поле Валюта 1 (<имя>) / групповое поле Валюта 2 (<имя>)

Укажите для обеих валют, какую закупочную цену должен рассчитать EPLAN. Для одной валюты всегда можно выбрать только ***одну*** опцию, т. е. можно выполнить расчет либо для опции "Закупочная цена/Единица цены" либо для "Закупочная цена/Упаковка".

Соответствующие цены после этого снова рассчитываются для выбранных изделий.

!!! note "Замечание:"

    Согласно бухгалтерским принципам при расчете производится округление до 2 десятичных разрядов.

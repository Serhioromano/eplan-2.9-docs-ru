# Диалоговое окно Импортировать настройки

Параметры > Настройки. Выделите область настройки "Пользователь" или "Рабочая станция" и щелкните по кнопке ![](images/all_import_as.png){: .ui-icon } (Импортировать).

В этом диалоговом окне определяется, из какого исходного файла будет выполнен импорт настроек.

Обзор основных элементов диалогового окна:

### <Путь иерархии>

Текст рядом с флажком указывает на то, какая область настройки выделена в диалоговом окне Настройки в представлении в виде дерева. Если этот флажок установлен, настройки исходного файла импортируются в выделенную область настройки и все нижестоящие области настройки.

### Другие области настройки

Здесь указывается, какие другие настройки программы и конфигурации также следует импортировать. Если этот влаг установлен, то выполняется их импорт. Доступность флажков зависит от того, какая область настройки выделена в диалоговом окне Настройки в представлении в виде дерева.

* Конфигурация свойств: Импортирует конфигурации свойств устройств, страниц и т. д., заданные соответствующим образом в диалоговом окне Свойства ++...++ через пункт всплывающего меню Конфигурировать.
* Диалоговая память: Импортирует из диалоговых окон, например, размер, положение на экране, разрешение экрана, статус "Свернуто" / "Развернуто" и настройки ширины столбцов.

!!! note "Замечание:"

    При импорте / экспорте настроек для диалоговой памяти рекомендуется установить одинаковое разрешение экрана на разных рабочих станциях. В случае разного разрешения экрана диалоговые окна могут выйти за пределы видимой области экрана.

* Конфигурации дерева и списка: Импортирует все конфигурации для представлений навигаторов в виде дерева и в виде списка, которые были заданы через пункт всплывающего меню Конфигурировать представление.
* Внешние программы: Импортирует список для быстрого доступа к внешним программам, которые были заданы в пунктах меню Параметры > Внешние программы.
* Фильтр: Импортирует все ***определенные пользователем*** настройки фильтра.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

### Исходный файл

В этом поле выводится путь и имя ранее экспортированного файла. По умолчанию файл экспорта получает имя, которое соответствует выделенной области в дереве диалогового окна Настройки. Имя файла состоит из следующих элементов: <Основная категория>+<Подкатегория>+<Область настройки>.xml. Нажав кнопку ++...++, можно выбрать другой файл импорта.

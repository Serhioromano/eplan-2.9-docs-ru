# Диалоговое окно Разместить точки определения соединения

Вы выделили соединения в графическом редакторе или в диалоговом окне навигатора Соединения — <Имя проекта>. Или вы выделили одну или несколько страниц либо проект в диалоговом окне навигатора Страницы — <Имя проекта>. Данные проекта > Соединения > Нумерация > Разместить.

В этом диалоговом окне запускается нумерация соединений для выбранных соединений, страниц или всего проекта.

После вызова пункта меню сначала автоматически генерируются новые соединения.

### Настройки

Выберите из раскрывающегося списка схему, которая определяет настройки для нумерации соединений. С помощью кнопки ++...++ откройте диалоговое окно Настройки: Нумерация соединений. Здесь можно определить настройки для нумерации и сохранить их в схеме.

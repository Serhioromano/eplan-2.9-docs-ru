# Вставить и обработать выводы устройства

Условие:

Вы открыли проект и находитесь на странице схемы соединений в Графическом редакторе.

1. Выберите пункты меню Вставить > Блок / вывод устройства / монтажная плата > Вывод устройства или Вставить > Блок / вывод устройства / монтажная плата > Вывод устройства (двустороннее).

!!! info "Для сведения:"

    Символ вывода устройства появится рядом с курсором.

2. Для пролистывания имеющихся вариантов вывода устройства при необходимости можно использовать клавишу ++Ctrl++.
3. Щелкните левой кнопкой мыши, чтобы разместить символ на странице.
4. Укажите данные вывода устройства в диалоговом окне Свойства ++...++.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выводы устройств будут вставлены на странице вместе с ОУ и обозначениями вывода устройства.

!!! note "Замечание:"

    В двухсторонних выводах устройств есть только одно общее обозначение вывода устройства (как при обозначении клемм). Двухсторонние символы устройств представлены одним символом, где отличаются внутренняя и внешняя сторона. Следовательно, такой символ существует в четырех вариантах. Соответствующее определение функции обладает следующими свойствами (диалоговое окно Логическая схема выводов устройства):
    Передать потенциал в: Ввод
    Отслеживание цели (ПЛК) в: Без ввода
    По одному прямому выводу устройства для внутренней стороны и одному общему выводу устройства — для внешней.

При ***обработке*** выводов устройства учитывайте следующее:

* Для выводов устройства только с ***одним*** выводом устройства невозможно указать связи с другими выводами устройства; поэтому поля для указания свойства Логическая схема выводов устройства недоступны.

* Выводы устройств, находящиеся в одном блоке и не имеющие собственного ОУ, всегда являютсся вспомогательными функциями. Для них никогда нельзя просмотреть перекрестные ссылки, так как достаточно перекрестной ссылки блока.

* Подсоединения устройств, находящиеся не в одном блоке, имеют перекрестные ссылки. При этом главной функцией может быть как вывод устройства, так и блок.

# Диалоговое окно Свойства страницы / Новая страница

Вы открыли навигатор страниц и открыли (как минимум) один проект.

* В навигаторе страниц выделена одна или несколько страниц. Обработать > Свойства.
* Страница > Создать.

В диалоговом окне Свойства страницы можно просмотреть или изменить свойства страниц.

При создании страниц в диалоговом окне Новая страница указываются свойства для новой страницы. Если какая-либо страница была выделена в навигаторе страниц, EPLAN применяет ее свойства, в ином случае значения свойств остаются пустыми.

Обзор основных элементов диалогового окна:

### Полное имя страницы

При создании страницы EPLAN автоматически предлагает для нее имя. Вы можете его скопировать или изменить вручную. Нажав кнопку ++...++, можно открыть диалоговое окно [Полное имя страницы](pagebrowsergui_d_seitenname.md) и обработать в нем структурные идентификаторы и имя страницы.

### Тип страницы

Присвойте странице интерактивный тип страницы из раскрывающегося списка или откройте с помощью кнопки ++...++ диалоговое окно Добавить тип страницы для выбора автоматического типа страницы. При выборе типа страницы все другие свойства страницы сохраняются.

### Описание страницы

Введите здесь текст описания для страницы. Этот текст также выводится в представлении в виде списки или дерева в навигаторе страниц.

[Групповое поле Свойства](devicetaggui_r_gruppenfeldeigenschaften.md)

В этой таблице выводятся свойства текущей страницы, которые можно обработать.

### [Очистить поля]

Эта кнопка доступна только при генерировании страниц. С ее помощью можно удалить содержимое полей данных текущей выделенной страницы. Только "тип страницы" занят типом выделенной страницы.

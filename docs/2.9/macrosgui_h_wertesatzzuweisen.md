# Присвоить набор значений объекту-заполнителю

Если объект-заполнитель имеет различные наборы значений, то позже при обработке объекта-заполнителя можно будет выбрать между различными наборами значений. При этом свойства, которым была присвоена переменная, содержат значения нового набора значений.

Условие:

Объект-заполнитель с несколькими наборами значений вставлен на страницу или в пространство листа.

1. Поместите курсор на символ "Якорь" объекта-заполнителя.
2. Выберите пункт всплывающего меню Присвоить набор значений.
3. В поле Набор значений диалогового окна Выбрать набор значений выберите имя требуемого набора значений.
4. При необходимости измените в таблице значения в выбранном наборе значений. Для этого переменная в диалоговом окне свойств объекта-заполнителя должна быть настроена как изменяемая.

!!! info "Для сведения:"

    В результате создается дополнительный определенный пользователем набор значений, а перед именем первоначально выбранного набора значений устанавливается префикс "Определено пользователем". При повторном выборе исходного набора значений определенный пользователем набор значений снова удаляется.

5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Свойства, которым на вкладке Присвоение была присвоена переменная, содержат значения, заданные для выбранного набора значений.

!!! tip "Совет:"

    Набор значений также можно присвоить из диалогового окна Свойства ++...++: Объект-заполнитель. Для этого выделите на вкладке Значения столбец нужного набора значений и выберите пункт всплывающего меню Присвоить набор значений. Объекты, свойствам которых на вкладке Присвоение присвоена переменная, получат соответствующие значения.

# Диалоговое окно Настройки: Генерация кабеля

Вы открыли проект.

* Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Кабель (автоматически). Щелкните рядом с полем Генерация кабеля по ++...++.
* Данные проекта > Кабели > Автоматически генерировать кабели. Щелкните в диалоговом окне Автоматически генерировать кабели в групповом поле Генерация кабеля рядом с полем Настройки на ++...++.

В этом диалоговом окне устанавливается, на каких объектах будут размещаться свойства кабелей и какие объекты будут учитываться при генерации кабелей.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

* * *

Групповое поле Разместить свойства

### На линии определения кабеля

Если этот флажок установлен, при генерировании кабеля свойства кабеля будут размещаться на линиях определения кабеля.

Если флажок снят, чертятся только линии определения кабеля, но не свойства на них.

### На точке определения соединения

Если этот флажок установлен, при генерировании кабеля свойства кабеля будут размещаться на точках определения соединения кабеля.

Если флажок снят, на точках определения соединений свойства кабеля не отображаются.

* * *

### Заполнить только имеющиеся кабели

Если этот флажок установлен, заполняться будут имеющиеся (размещенные или неразмещенные) кабели. Для выделенных кабелей будут собираться кабельные соединения, проходящие между одинаковыми целями. Если выделена только часть проекта (фрагмент схемы соединений или отдельные страницы), учитываться будут только кабельные соединения, содержащиеся в этой конкретной части. Новые кабели не генерируются.

Если флажок снят, будут заполняться существующие кабели и при необходимости генерироваться новые. Для этого будут собираться все (выделенные) кабельные соединения, проходящие между двумя одинаковыми целями.

### Объединить главные функции между одинаковыми целями

Если этот флажок установлен, между двумя одинаковыми целями несколько главных функций объединяются в одну главную функцию. (Это настройка по умолчанию.)

Если флажок снят, для каждой главной функции генерируется собственный кабель.

!!! example "Пример:"

    Для двух разных соединений (между одинаковыми целями) было начерчено по одной линии определения кабеля. Каждая из этихлиний определения кабеляявляется главной функций.Если флажок установлен, графически первая линия определения кабеля становится главной функцией, а вторая - вспомогательной функцией. В данном случае EPLAN генерирует только один кабель с данными из первой линии определения кабеля. В отчетах учитываются только данные главной функции.Если флажок снят, генерируются два кабеля.

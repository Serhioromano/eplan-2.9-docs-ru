# Диалоговое окно Настройки: Экспорт изгибания труб

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт изгибания труб.
* Сервисные программы > Данные изготовления > Механическая обработка > Изгибание труб. Рядом с полем Машина нажмите ++...++.

В этом диалоговом окне задаются общие настройки экспорта данных для системы управления с целью управления машинами по изгибанию.

Обзор основных элементов диалогового окна:

### Каталог вывода

Экспорт изгибания труб будет сгенерирован в виде XML-файла. В этом поле укажите место хранения. В качестве целевых для экспорта данных можно указать любой дисковод и подкаталог. Предварительно установлен каталог $(EPLAN_DATA)\Экспорт\XML.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Генерировать имена файлов

В этом раскрывающемся списке определяется стандартный порядок присвоения имен файлам экспорта:

* Из непрерывного номера: при выборе этого параметра EPLAN по умолчанию присваивает файлам XML имена по образцу <непрерывный_номер>.XML. Если уже имеются файлы XML, EPLAN начнет нумерацию со следующего доступного числа.
* Из обозначения функц. элемента: При выборе этого параметра EPLAN по умолчанию присваивает файлам XML имена по образцу <Обозначение функц. элемента>.XML.
* Из свойств: если выбран этот параметр, имя файла составляется из свойств проекта, а также свободно определяемого разделителя. Используя кнопку ++...++, откройте диалоговое окно Формат: Свойство для конфигурации имени файла.

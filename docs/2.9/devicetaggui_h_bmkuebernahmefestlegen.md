# Определить направление поиска для переноса ОУ

По умолчанию ОУ переносятся в соответствии с выравниванием рамки. Тем не менее, Вы можете изменить направление поиска у любой функции, у которой возможен перенос ОУ.

Условия:

* Вы открыли проект.
* Вы определили в настройках проекта, что устройство и макросы должны нумероваться при вставке в режиме online (Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация (online)).

1. Выделите в графическом редакторе или в одном из навигаторов функцию и выберите пункт меню Обработать > Свойства.
2. Выберите в диалоговом окне Свойства ++...++ вкладку <Категория функции>.
3. В таблице Свойства выберите свойство Перенос ОУ: Направление поиска, а в столбце Значение — возможные настройки из раскрывающегося списка.

!!! info "Для сведения:"

    Если направление поиска — "Согласно выравниванию рамки", то в случае с проектами по стандарту IEC ОУ переносится слева, в случае с проектами по стандарту NFPA — сверху, в случае с проектами по стандарту ГОСТ — справа.

!!! info "Для сведения:"

    Если направление поиска — "Альтернативно выравниванию рамки", то в случае с проектами по стандарту IEC ОУ переносится сверху, в случае с проектами по стандарту NFPA — слева, в случае с проектами по стандарту ГОСТ — снизу.

!!! info "Для сведения:"

    Если направление поиска - "Только блоков", переносится ОУ только блока (черный ящик, блоки ПЛК), вмещающего функцию. Если функция находится не в блоке, перенос ОУ не осуществляется.

4. Щелкните по кнопке ++OK++.

!!! note "Замечание:"

    * Чтобы предотвратить перенос ОУ, деактивируйте онлайн-режим нумерации.
    * Чтобы предотвратить копирование ОУ для отдельных страниц, активируйте в выбранных для страниц рамках свойство Копирование ОУ: Только блоков.
    * Чтобы предотвратить перенос ОУ для отдельных функций, установите у соответствующих функций направление поиска "Только блоки".

# Диалоговое окно Add-ons

Сервисные программы > Add-ons

В этом диалоговом окне можно вручную зарегистрировать Add-ons или удалить их регистрацию.

Обзор основных элементов диалогового окна:

В таблице отображаются все установленные Add-ons. Имеются в виду Add-ons как предоставленные EPLAN, так и созданные пользователем самостоятельно. Add-on добавляет в стандартный комплект поставки варианта продукта EPLAN дополнительные функции или предоставляет дополнительные данные. Add-ons программы EPLAN обычно регистрируются автоматически. Собственные Add-ons вы регистрируете вручную, если они сохранены в локальном каталоге. Для этого установите флажок Зарегистрировано после того, как с помощью кнопки ![](images/all_new_as.png){: .ui-icon } (Создать) был выбран Add-on.

!!! note "Замечание:"

    Add-ons регистрируются автоматически, если они находятся в каталоге на сервере. Для этого соответствующий файл install.xml должен находиться в пути к каталогу, который вы задали в настройках фирмы в диалоговом окне Настройки: Add-ons.

### Панель инструментов

Кнопка |  Значение
---|---
![](images/all_new_as.png){: .ui-icon } (Создать) |  Открывает диалоговое окно Выбрать Add-on, в котором можно выбрать файл install.xml соответствующего дополнительного модуля.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет Add-on из таблицы. Удаление возможно при условии, что Add-on не зарегистрирован. Как правило, Add-ons программы EPLAN удалять нельзя.

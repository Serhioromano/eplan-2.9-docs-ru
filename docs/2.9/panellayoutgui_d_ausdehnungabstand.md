# Диалоговое окно Удлинение и интервал

Вы открыли проект. Данные проекта > Устройства / Изделия > Компоновка электрошкафа 2D. В диалоговом окне Компоновка электрошкафа 2D - <Имя проекта> во всплывающем меню > Настройки выбраны параметры Ввод вручную ++...++ группового поля Копирование. Вы находитесь в графическом редакторе на странице типа "компоновка электрошкафа". Выберите Всплывающее меню > Установка на монтажную плату на ***не*** размещенном изделии.

В этом диалоговом окне при установке неразмещенных изделий на монтажную плату можно вручную ввести размеры изделий.

Доступны следующие элементы диалогового окна:

### Удлинение

Введите в этом групповом поле значения для ширины и высоты вставляемого изделия.

### Интервал

Введите в этом групповом поле значения для горизонтального и вертикального интервала вставляемого изделия по отношению к соседним изделиям.

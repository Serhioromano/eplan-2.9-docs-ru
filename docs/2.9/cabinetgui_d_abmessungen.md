# Диалоговое окно Размеры

Вы открыли проект. Навигатор пространства листов открыт, и одно пространство листов открыто. Вставить > Устройство.

В этом диалоговом окне перед размещением электротехнических изделий, изделий Fluid-техники, а также определенных механических изделий можно указать значения ширины, высоты и глубины размещаемых изделий. Диалоговое окно отображается на экране автоматически, если для изделия, выбранного для размещения, не указаны габариты и ему не присвоен графический макрос. Такие изделия можно указать в базе данных изделий, например, как переменные прототипы для размещения маленькими сериями или для штучного производства, при этом нет необходимости в составлении полных основных данных в форме комплексного определения изделий.

!!! note "Замечание:"

    Для изделий из главной группы продуктов "Механика" допускается ручной ввод размеров, если они относятся к одной из следующих групп продуктов и дополнительно обладают одним из следующих определений функций:

    Группа продуктов |  Подгруппа продуктов |  Определение функции
    ---|---|---
    Сборная шина |  Адаптер |  Адаптер Адаптер вывода устройства Адаптер устройства
    Принадлежности системы |  Общее |  Резьбовое соединение кабелей Функциональный элемент электрошкафа

    Для всех остальных механических изделий из главной группы продуктов "Механика" или изделий из группы продуктов "Технология производственных процессов" ручной ввод размеров изделий невозможен.

Обзор основных элементов диалогового окна:

### Ширина, Высота, Глубина

Эти поля служат для ввода или изменения размеров, специфичных для каждого устройства, в отображении единиц измерения, заданных как пользовательская установка. Если для соответствующего изделия частично уже указаны габариты изделия, в соответствующие поля необходимо внести данные значения.

### Записать размеры обратно на изделия

Если этот флажок установлен, значения Ширина, Высота, Глубина, указанные в этом диалоговом окне, сохранятся в монтажных данных размещаемого изделия. Таким образом, при последующем размещении данного изделия его размеры будут полностью определены, а диалоговое окно Размеры больше отображаться не будет.

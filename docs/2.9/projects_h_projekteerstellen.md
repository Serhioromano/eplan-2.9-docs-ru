# Создать проекты

Проекты можно создавать с помощью меню проектов или через управление проектами. В обоих случаях проект создается на основе шаблона проекта или базового проекта. Ниже описывается, как создать проект с помощью меню проектов. Если вы сами еще не создали шаблон проекта или базовый проект, можно использовать [Примерный шаблон](projects_k_basisprojekt.md) EPLAN.

Условие:

Вы имеете доступ к [Шаблоны проекта](projects_k_projektvorlagen.md) или [Базовые проекты](projects_k_basisprojekteerweitert.md).

1. Выберите пункты меню Проект > Создать.
2. В поле Имя проекта диалогового окна Создать проект введите имя проекта без расширения файла.
3. Щелкните у поля Шаблон [...] и замените в диалоговом окне Выбрать базовый проект/шаблон проекта каталог, в котором находится проект.
4. Выберите в поле Тип файла запись Шаблон проекта EPLAN (*.ept) или Базовый проект EPLAN (*.zw9).
5. Выделите нужный шаблон и нажмите ++Открыть++.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В целевом каталоге создается новый проект *.elk.

!!! info "Для сведения:"

    Если вы работаете с управлением проектами, новый проект будет автоматически добавлен в управление проектами.

!!! tip "Совет:"

    Можно указать в свойствах проекта, из какого шаблона проекта / базового проекта проект был сгенерирован:
    Выделите проект в навигаторе страниц и выберите пункт всплывающего меню Свойства. Добавьте во вкладку Свойства свойств проекта Шаблон проекта . Отображается применяемый шаблон проекта / базовый проект.

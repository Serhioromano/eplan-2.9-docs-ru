# Объединение объектов в пространстве листа

3D-макрос (т. е. пространство листа в проекте макросов) может содержать несколько отдельных элементов, потому что несколько элементов было в импортируемом исходном файле. Хотя элементы нельзя просуммировать геометрически, имеется функциональность, позволяющая охватить все элементы пространства листа таким образом, что 3D-макрос в процессе работы рассматривается как ***один*** функциональный элемент. Эта функциональность имеется только в пространствах листа, и ее не следует смешивать с группировкой в двухмерной монтажной конструкции.

!!! note "Замечание:"

    Объединение трехмерных тел не подходит для уменьшения объема данных в генерируемом макросе. Если импортированный файл STEP содержит несколько одинаковых или очень подробных частей, это может привести к тому, что объем данных после объединения существенно возрастет. Это имеет непосредственное влияние на размер файлов макросов, поскольку особенности системы не допускают их сжатия.

    Для достижения приемлемого размера файла макроса в таком случае следует отказаться от объединения импортированных одинаковых частей и генерировать макрос с импортированными отдельными частями.

Условия:

* Вы открыли проект.
* Навигатор пространства листов открыт, и одно пространство листов открыто.
* В пространстве листа содержится импортированный трехмерный элемент или находящийся в обработке 3D-макрос.

1. Выберите пункты меню Обработать > Графика > Объединить.
2. Выделите выбранные функциональные элементы с помощью прямоугольника. Если объекты уже выделены до вызова этой функции, перемещать рамку не нужно.
3. Переместите курсор над выделенными объектами и щелкните нужную точку захвата, которая станет точкой захвата нового объединенного функционального элемента.

!!! info "Для сведения:"

    Выбранная точка захвата будет сохранена на новом объекте. Если из этого объекта будет создан 3D-макрос, при размещении на этой точке он будет перемещаться и размещаться при помощи курсора.

!!! note "Замечание:"

    По техническим причинам объединенные объекты можно снова разделить только сразу после выполнения операции при помощи функции Отменить. В дальнейшем разделить объекты будет невозможно.

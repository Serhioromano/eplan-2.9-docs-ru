# Диалоговое окно Свойства (усл. обозначение): Пространство листа

Вы открыли проект. Открыт навигатор пространства листа. Проект содержит как минимум одно пространство листа. В навигаторе пространства листа выбрано пространство листа. Обработать > Свойства.

В этом диалоговом окне можно задать и обработать свойства пространств листов.

Обзор основных элементов диалогового окна:

### Структурный идентификатор

Нажав кнопку ++...++, можно открыть диалоговое окно Структурный идентификатор пространства листа и обработать в нем структурные идентификаторы.

### Имя

При создании пространства листа EPLAN автоматически предлагает для него имя, представляющее собой какое-либо число. Для каждого нового пространства листа число в предлагаемом имени увеличивается на одну позицию. Можно использовать предложенное имя или вручную дополнить или изменить его.

### Описание

Введите здесь информативный текст для пространства листа. Этот текст также отображается в представлении в виде дерева навигатора пространства листа.

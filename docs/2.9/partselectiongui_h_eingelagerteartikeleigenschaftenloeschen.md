# Удалить свойства сохраненных изделий

Вы можете стереть выбранные свойства сохраненных в проекте изделий. Таким образом, напр., можно удалить информацию о цене перед передачей проекта заказчикам.

Условие:

Вы выделили проект или отдельные страницы в навигаторе страниц или вы выделили условное обозначение в графическом редакторе.

1. Выберите следующие пункты меню: Сервисные программы > Изделие > Удалить сохраненные свойства
2. В диалоговом окне Удалить сохраненные свойства изделия из раскрывающегося списка выберите схему Настройки или нажмите кнопку ++...++, чтобы открыть одноименное диалоговое окно, в котором можно обработать схемы и управлять ими.
3. Нажмите кнопку ![](images/all_new_as.png){: .ui-icon } (Создать) и выберите в диалоговом окне Выбор свойств свойства изделия для удаления. Возможен многократный выбор.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выбранные свойства изделия удаляются из проекта.

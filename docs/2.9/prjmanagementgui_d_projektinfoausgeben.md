# Диалоговое окно Вывести информацию о проекте

Вы выбрали базу данных управления проектами. Вы считали каталоги и проекты. Проект > Управление. Выделите проект. Выберите ++Дополнительно++ > Вывести информацию о проекте.

В этом диалоговом окне установите, в каком объеме выводится специфическая для проекта информация. Затем информация отображается в Internet Explorer и для этой цели сохраняется в локальном каталоге \tmp в файле HTML PrjInfo.htm.

Обзор основных элементов диалогового окна:

### Краткий обзор

При выборе этого параметра выводится следующая информация проекта: место хранения проекта, имя проекта, описание проекта, номер проекта, дополнительные поля.

### Общий обзор

При выборе этого параметра выводится вся связанная с проектом информация.

### Общий обзор с обновлением

При выборе этого параметра специфическая для проекта информация обновляется, а затем генерируется общий обзор. В зависимости от размера базы данных проекта эта операция может продлиться некоторое время.

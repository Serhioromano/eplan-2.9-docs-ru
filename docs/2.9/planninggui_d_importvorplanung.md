# Диалоговое окно Импортировать данные предварительного планирования — <Имя проекта>

Вы открыли проект и навигатор предварительного планирования. Данные проекта > Предварительное планирование > Импортировать.

В этом диалоговом окне у вас есть возможность импортировать списки с данными предварительного планирования, созданные во внешних приложениях.

Обзор основных элементов диалогового окна:

### Тип источника данных

В этом поле выбирается вид источника данных:

* Excel: Позволяет выбрать файл Excel.
* Текст: Позволяет выбрать текстовый файл в качестве источника данных.

### Источник данных

В этом поле отображается файл, из которого следует импортировать данные предварительного планирования. Щелкните ++...++, чтобы в зависимости от поля Тип источника данных выбранного типа данных в последующем диалоговом окне выбрать источник данных.

### Сегмент структуры цели

Укажите здесь, под каким уровнем иерархии в представлении в виде дерева вставляются импортированные данные. Предварительно настроен выделенный сегмент структуры. Раскрывающийся список содержит все определенные в проекте сегменты структуры.

### Присвоения полей

В этом поле отображается схема, задающая присвоение внешних полей данных свойствам EPLAN. С помощью кнопки ++...++ откройте диалоговое окно [Присвоения полей](planninggui_d_feldzuordnung.md). Здесь можно создать или изменить схему.

### Игнорировать ошибку

Если этот флажок установлен, импорт не будет прерываться из-за появляющихся ошибок и сообщений.

Если флажок снят, при возникновении ошибки импорт будет прерван.

### Перезаписать имеющиеся объекты планирования

Если этот флажок установлен, уже имеющиеся объекты планирования перезаписываются данными одноименных объектов планирования из файла импорта.

Если флажок снят, уже имеющиеся объекты планирования остаются без изменений.

### Не генерировать новые сегменты структуры и объекты планирования

Если этот флажок установлен, происходит только обновление данных уже имеющихся сегментов структуры и объектов планирования. Новые сегменты структуры и объекты планирования не генерируются, соответствующие записи в файле импорта игнорируются.

Если флажок снят, импортируются все данные из файла импорта и при необходимости генерируются новые сегменты структуры и объекты планирования.

### Удалить недостающие объекты в проекте при повторном импорте

Если этот флажок установлен, сегменты, отсутствующие при повторном импорте в источнике данных (которые были убраны), отображаются в следующем диалоговом окне Синхронизировать данные предварительного планирования и обозначаются как удаленные. Эти сегменты удаляются из проекта, если в настройках не выбрано действие "Не удалять".

Если флажок снят, сегменты, которые были убраны в источнике данных, сохраняются в проекте.

# Проверить обозначение устройства

Устройство содержит обычно всю идентифицируемую информацию (такую как: установка, место установки, буквенное обозначение) в своем обозначении. Чтобы проверить введенные для ОУ данные на длину и допустимые символы, Вы можете использовать проверку обозначений.

Условие:

Вы открыли проект.

1. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Синтаксическая проверка ОУ
2. Выберите в диалоговом окне Настройки: Синтаксическая проверка ОУ на вкладке Электротехника и/или Fluid-Техника флажок Активировать проверку.
3. В групповых полях Обозначение устройства и Счетн. номер определите объем проверки для элементов обозначения устройств полного ОУ.
4. На вкладке Структурный идентификатор в групповом поле Блоки идентификаторов определите объем проверки для блоков идентификаторов в рамках полного ОУ.
5. Щелкните по кнопке ++ОК++.

EPLAN после ввода данных устройства в диалоговом окне Свойства ++...++ проверяет правильность синтаксиса введенного обозначения устройства. При этом всегда проверяется вся символьная строка обозначения устройства (т. е. полного ОУ). Для блоков идентификаторов используются настройки на вкладке Структурные идентификаторы, даже если флажок не установлен.

!!! note "Замечание:"

    На вкладке Структурный идентификатор флажок Активировать проверку должен быть установлен только тогда, когда требуется проверка электротехнического ОУ или ОУ Fluid-Техники, а также структурных идентификаторов (например, страниц).

Если синтаксис не совпадает с указанными критериями, EPLAN откроет окно сообщения, в котором будут выделены неверные элементы. Посредством [Настройки] Вы можете из этого окна сообщений перейти к настройкам для синтаксической проверки ОУ. Здесь Вы можете проверить Ваши настройки синтаксиса и, при необходимости, откорректировать их. И если "ошибка" - это преднамеренное отклонение от синтаксиса, то Вы можете выйти из окна сообщения, нажав ++Игнорировать++, и скопировать введенные Вами данные, не прибегая к изменению настроек.

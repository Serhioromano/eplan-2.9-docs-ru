# Копирование данных проектов EPLAN PPE: Монтажные чертежи

В EPLAN PPE монтажный чертеж служит для графического представления элемента ТК и необходимых для его монтажа принадлежностей. Кроме графического представления, в качестве табличного вида для этого монтажного чертежа можно создать список принадлежностей для монтажа, в котором будут перечислены используемые материалы. Они могут браться из базы данных изделий, однако это не обязательно.

При копировании данных проекта PPE монтажные чертежи изображаются в качестве ***узлов***, т. е. для каждого монтажного чертежа в проекте генерируется узел с изделиями. При этом номер и обозначение монтажного чертежа превращаются в "Номер изделия" и "Обозначение 1" узла. Если изделие присутствует в базе данных изделий, оно сохраняется. Если изделия отсутствуют, изделие с данными из монтажного чертежа создается только в проекте. При необходимости впоследствии можно выполнить синхронизацию с базой данных изделий.

При синхронизации с базой данных изделий происходит следующее:

* Узлы, сгенерированные во время передачи данных, размещаются в подгруппе продуктов "Механика > Монтажный чертеж > Общее".
Возможные номера позиций и исполнения принадлежностей для монтажа из монтажных чертежей автоматически вводятся на вкладке Узел базы данных изделий в оба столбца Номер позиции и Дополнительный текст.
* Отдельные изделия узла размещаются как изделия типа "Отдельная часть" в подгруппе продуктов "Механика > Монтажный чертеж > Принадлежности".

После копирования данных проекта PPE преобразованные монтажные чертежи сохраняются на объектах планирования (датчиках, исполнительных элементах и т. д.) в качестве изделий узла на вкладке Изделие. При этом изделия узлов автоматически помещаются в десятую строку изделия.

Монтажные чертежи из проектов PPE далее существуют в качестве графических страниц. Размещения свойств заменяются фиксированными текстами. Тем самым монтажный чертеж "замораживается" в состоянии на момент копирования данных. Чтобы снова соединить данные с монтажным чертежом, необходимо либо заново сгенерировать отчет типа "Обзор групп изделий" в качестве новой страницы отчета, либо разместить его на существующей странице.

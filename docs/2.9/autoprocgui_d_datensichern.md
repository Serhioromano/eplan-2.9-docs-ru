# Диалоговое окно Резервировать данные

* Вы открыли проект. Сервисные программы > Автоматизированная обработка. В диалоговом окне Выполнить: Автоматизированная обработка щелкните по кнопке ++...++ рядом с полем Схема. В диалоговом окне Настройки: Автоматизированная обработка в поле Доступные операции выделите операцию "Резервировать данные", после чего щелкните по кнопке ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо).
* Проект > Управление. Вы считали в диалоговом окне Управление проектами каталоги и выделили, как минимум, один проект. [Организовать] > Автоматизированная обработка. В диалоговом окне Выполнить: Автоматизированная обработка щелкните по кнопке ++...++ рядом с полем Схема. Выберите параметр Схема и нажмите кнопку ++...++ рядом с этим полем. В диалоговом окне Настройки: Автоматизированная обработка в поле Доступные операции выделите операцию "Резервировать данные", после чего щелкните по кнопке ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо).

В этом диалоговом окне задаются настройки для автоматизированного резервирования данных.

Обзор основных элементов диалогового окна:

### Метод

В этом окне представлен метод резервирования данных EPLAN. Для автоматизированного резервирования данных имеется метод "Дополнительное резервирование". В данном случае данные сжимаются как файл *.zw1 и хранятся в каталоге, выбранном в поле Каталог для резервирования. Исходный проект сохраняется без изменений.

### Каталог для резервирования

В этом поле расположен каталог для резервирования данных. С помощью кнопки ++...++ откройте диалоговое окно выбора каталога. В этом диалоговом окне в качестве места хранения данных можно выбрать другой каталог. Вы можете использовать любой каталог на любом доступном дисководе.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

* * *

Групповое поле 'Параметры'

### Резервировать внешние документы

Если этот флажок установлен, то вместе с проектом будут резервироваться также все внешние документы, находящиеся в каталоге $(EPLAN)\Проекты\ ++...++ \<Имя Проекта>\DOC. $(EPLAN)  – главный каталог, созданный во время установки.

### Резервировать графические файлы

Если этот флажок установлен, то вместе с проектом будут резервироваться также все графические файлы, находящиеся в каталоге $(EPLAN)\Проекты\ ++...++ \<Имя Проекта\Images. $(EPLAN)  – главный каталог, созданный во время установки.

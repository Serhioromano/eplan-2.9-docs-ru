# Вставить функции

Функции добавляются в проект таким образом:

* Вы работаете с графикой. В Графическом редакторе вызовите выбор символов и выберете ***символ***. В символе указывается определение функции, которое добавляется одновременно с символом. (символ и функция группируются с условным обозначением.)
* Добавьте ***макрос*** в проект, в котором содержатся функции (то есть условные обозначения). Условному обозначению присваивается определение функции или в нем сохраняется специальное логическое определение.
* Вы работаете с логическим применением инжиниринга и хотели бы работать с предварительно определенными устройствами. Выберите ***Устройство*** (а также Изделие); это устройство имеет данные своей функции. Выберите одну из этих функций. Предлагаются все символы, которые предназначены для представления этой функции. Выберите символ и разместите его. Одновременно создается устройство со всеми его функциями. Информация попадает к функциям через изделие.
* Выберите (в навигаторах) непосредственно ***(определение)***функции и добавьте его (размещенное или неразмещенное) в проект.

### Вставить функции при помощи символов

Просто вставьте функции, разместив символ на странице схемы соединений.

Условие:

В графическом редакторе открыта страница схемы соединений.

1. Вставить > символ
2. Выберите в диалоговом окне "Выбор символа" желаемый символ, и щелкните по ++OK++.
3. Разместите символ на схеме соединений.

!!! info "Для сведения:"

    В символе указывается определение функции; оно размещается с символом как функция.

4. В диалоговом окне Свойства ++...++ обработайте свойства функции.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Новые и измененные свойства сохраняются на схеме соединений и отображаются там (если они выбраны для отображения).

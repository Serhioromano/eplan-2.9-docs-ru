# Перекрестные ссылки между графическим предварительным планированием / функциональными схемами автоматизации и отчетами предварительного планирования

Для размещенных сегментов в графическом предварительном планировании или для размещенных технологических контуров на функциональной схеме автоматизации до сих пор нельзя было отобразить перекрестные ссылки на отчеты предварительного планирования. В новой версии настройки перекрестных ссылок были дополнены, чтобы отображались перекрестные ссылки между сегментом, размещенным на соответствующей логической странице, и отчетами этого сегмента на страницах отчетов.

Для этого настройки отображения перекрестных ссылок были дополнены такими типами страниц:

* Предварительное планирование: Обзор объектов планирования
* Предварительное планирование: План объекта планирования
* Предварительное планирование: Обзор сегментов структуры
* Предварительное планирование: План сегмента структуры

С одной стороны, это затрагивает таблицу Отображать перекрестные ссылки устройств в диалоговом окне Настройки: Отображение, а с другой — таблицу Тип страницы / префикс в диалоговом окне Настройки: Общее. Путь меню к этим настройкам: Параметры > Настройки > Проекты > "Имя проекта" > Перекр. ссылки / образы контактов.

Если в таблице Отображать перекрестные ссылки устройств для столбца Предварительное планирование и / или для столбца Функциональная схема автоматизации выбран такой отчет предварительного планирования, то на страницах предварительного планирования и / или функциональных схемах автоматизации на первом размещении сегмента отображаются перекрестные ссылки на страницы отчета, если для сегмента существует данный отчет предварительного планирования. Если дополнительно активирована настройка Вспомогательные функции также указывают на список обозначений устройств, на других размещениях сегмента также будут отображаться соответствующие перекрестные ссылки.

### Улучшенные переходы в экспортированном PDF-документе

При экспорте PDF-документов теперь для перекрестных ссылок технологических контуров выводятся переходы на соответствующие отчеты предварительного планирования. Таким образом, в PDF-документе можно перейти, например, от технологических контуров на функциональной схеме автоматизации к соответствующим спецификациям технологических контуров.

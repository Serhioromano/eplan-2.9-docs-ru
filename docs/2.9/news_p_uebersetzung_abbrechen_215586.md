# Улучшение перевода с индивидуальным выбором при многозначных значениях

Если в настройках проекта или пользователя для перевода активирована настройка Индивидуальный выбор при многозначных значениях, для слов, которые многократно занесены в словарь с различными значениями, будет отображаться диалоговое окно выбора. Кнопка ++Отменить++ в этом диалоговом окне ранее отменяла только перевод текущего текста. Теперь эта кнопка полностью отменяет процесс перевода. Чтобы отменить только перевод текущего текста, теперь в диалоговом окне доступна новая кнопка ++Пропустить++.

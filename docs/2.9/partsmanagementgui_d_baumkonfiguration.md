# Диалоговое окно Конфигурация дерева

* Сервисные программы > Изделие > Управление > [Дополнительно] > Настройки.
* Параметры > Настройки > Пользователь > Управление > Изделия.

Щелкните в поле Конфигурация дерева по ++...++.

В этом диалоговом окне укажите, как конфигурируется представление в виде дерева базы данных изделий.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Главные узлы

Этот список содержит возможные главные узлы для конфигурации. Щелкните внутри имени главного узла по ++...++ по правому краю строки, чтобы в диалоговом окне Конфигурация дерева — Главные узлы задать подробные настройки для соответствующей конфигурации.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

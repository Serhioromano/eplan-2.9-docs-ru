# Операция: XMDeleteReprTypeAction

**Параметр** |  **Описание**
---|---
RepresentationType  |  Вид представления, который необходимо убрать из выбранного макроса. Разрешены следующие значения:

* 0 = нейтральный
* 1 = многополюсный
* 2 = однополюсный
* 3 = парная перекрестная ссылка
* 4 = обзор
* 5 = графика
* 6 = компоновка электрошкафа
* 7 = функциональная схема автоматизации
* 8 = многополюсный Fluid
* 9 = топология
* 10 = трехмерный чертеж монтажных поверхностей
* 11 = функциональный.

Source  |  Задает файлы, которые необходимо изменить.
Имя файла должно быть указано с полным путем файла или без него. При вводе расширения имени файла можно использовать заполнители.
Destination  |  Целевой каталог

!!! note "Замечание:"

    Если при вызове операции указан только параметр RepresentationType, то сначала открывается диалоговое окно для выбора макроса, а затем диалоговое окно, в котором выбирается целевой каталог для сохраняемых макросов.

!!! example "Пример:"

    XMDeleteReprTypeAction
    /RepresentationType:0
    /Source:C:\macros\*.ema
    /Destination:C:\dest

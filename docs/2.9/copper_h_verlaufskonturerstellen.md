# Создать профиль для изогнутых медных шин

Профиль для изогнутых медных шин — это путь, по которому проходит прямоугольный основной контур, формирующий изогнутую медную шину. Этот путь создается в редакторе контура в виде двухмерной линии.

В нашем примере точка захвата шины совпадает с нулевой точкой контура.

Условие:

Вы открыли проект.

1. Выберите пункты меню Сервисные программы > Основные данные > Профиль (медь) > Создать.
2. Выберите в диалоговом окне Создать профиль в поле Сохранить в тот каталог, в который вы хотите сохранить новый чертеж профиля.
3. Введите в поле Имя файла имя для нового чертежа профиля.
4. В поле Тип файла внесена запись Профиль, медь (*.fp1), эту запись изменить нельзя.
5. Щелкните по ++Сохранить++.

!!! info "Для сведения:"

    Откроется диалоговое окно Свойства профиля — <Имя файла>. Свойство Сетка уже заполнено, здесь по умолчанию установлено значение 4 мм.

6. Добавьте другие свойства или оставьте предварительную настройку без изменений.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Профиль откроется в редакторе контура. Нулевая точка чертежа отмечена красным кружком и находится слева внизу. Здесь должна находиться начальная точка первой линии, поскольку нулевая точка при размещении превращается в точку захвата медной шины.

8. Выберите один из графических элементов "линия" или "ломаная линия".
9. Разместите начальную точку линии в красном кружке нулевой точки.
10. Начертите профиль изогнутой медной шины, как вы считаете нужным.

11. Выберите пункты меню Сервисные программы > Проверить профиль, чтобы убедиться, что профиль замкнут.
12. Выберите пункты меню Сервисные программы > Основные данные > Профиль (медь) > Закрыть.

!!! info "Для сведения:"

    Чертеж профиля будет сохранен и закрыт. Его можно сразу использовать при размещении медной шины.

# Вкладка Обозначение

Вы открыли проект.

* Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Нумерация соединений. В диалоговом окне Настройки: Нумерация соединений выберите вкладку Обозначение.
* Вы выделили соединения в графическом редакторе или в диалоговом окне навигатора Соединения — <Имя проекта>. Или вы выделили одну или несколько страниц либо проект в диалоговом окне навигатора Страницы — <Имя проекта>. Данные проекта > Соединения > Нумерация > Разместить > [...]. В диалоговом окне Настройки: Нумерация соединений выберите вкладку Обозначение.

В этой вкладке можно задать не более 20 групп форматов. В группе форматов можно объединить несколько определений функций, чтобы выполнить их совместное форматирование. При этом также можно использовать Wildcards (*), что позволит объединить группы функций, для которых впоследствии будет определен общий формат.

Обзор основных элементов диалогового окна:

### Группы форматов

В этом списке отображаются выбранные для нумерации группы соединений, их объем и предварительный просмотр установленного формата.

!!! note "Замечание:"

    Соединения нумеруются согласно последовательности групп соединений. Поэтому группа "Общие соединения" должна стоять на последнем месте.

!!! example "Пример:"

    Вы выбрали в поле Группы форматовследующие группы соединений:Соединения, подсоединенные к выводамустройства ПЛКСоединения, подсоединенные к клеммам Общие соединения.В этом случае сначала нумеруются соединения, которые подсоединены к выводам устройств ПЛК. При этом используется установленный для этой группы формат нумерации. Затем нумеруются соединения, которые подсоединены к клеммам (с форматом нумерации для соответствующей группы). В заключение нумеруются оставшиеся соединения с форматом нумерации для группы "Общие соединения".Если бы общие соединения стояли на первом месте в поле Группы форматов, в процесс нумерации были бы включены***все***соединения с форматом нумерации "Общие соединения". В этом случаеформатыне были бы применены для соединений, подсоединенных к выводам устройств ПЛК и клеммам, так как соответствующие соединения были бы уже пронумерованы.

### [Сетевое расширение]

Открывает диалоговое окно [Сетевое расширение](wirenumberinggui_d_netzerweiterung.md). В этом диалоговом окне можно расширить определение сети. Эта настройка влияет на обозначение, если в качестве объема выбрана "сеть".

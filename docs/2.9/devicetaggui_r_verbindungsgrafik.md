# Вкладка Графика соединения

Вы открыли проект.

* В графическом редакторе выделена точка определения соединения, точка определения потенциала, вывод потенциала или точка определения сети. Обработать > Свойства. В диалоговом окне Свойства ++...++ перейдите на вкладку Графика соединения.
* Данные проекта > Соединения > Навигатор. Выделите соединение и выберите во всплывающем меню пункт Свойства. В диалоговом окне Свойства ++...++ перейдите на вкладку Графика соединения.

На этой вкладке обрабатываются графические свойства соединения.

Обзор основных элементов диалогового окна:

### Свойство/Присвоение

В таблице доступен уровень иерархии Формат, который можно открыть щелчком по знаку ![](images/all_plussign_as.png){: .ui-icon }.

### Слой

Слой соединения будет автоматически передан из определения функции соединения и настройки Подвод давления / управляющий вывод устройства (в [Логической схеме выводов устройства](xfctdefbrowsergui_d_anschlusslogik.md) \- функции, подключенной к соединению).

Для точек определения соединения, точек определения потенциала, выводов потенциала и точек определения сети слой по умолчанию не указывается, то есть используется автоматически передаваемый слой соединения.

При необходимости выберите из раскрывающегося списка другой слой. Из выбранного здесь слоя будут скопированы настройки толщины линии, типа линии и т. д.

!!! example "Пример:"

    Для электротехнического соединения в логической схеме выводовустройствапредусмотрены следующие свойства:Тип вывода устройства:Жила / провод Подвод давления / управляющий вывод устройства:Не определено Соединение находится на слоеEPLAN311, Символьная графика.Символы соединения.Автосоединение.Для соединения Fluid-техники в логической схеме выводов устройства предусмотрены следующие свойства:Тип вывода устройства:FluidПодвод давления / управляющий вывод устройства:Рабочая линия При этом соединение находится на слоеEPLAN507, Символьная графика.Символы соединения.Автосоединение.Рабочие линии.

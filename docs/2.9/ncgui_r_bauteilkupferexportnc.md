# Вкладка Функциональный элемент (экспорт ЧУ, медь)

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт ЧУ, медь. Выберите вкладку Функц. элемент.
* Сервисные программы > Данные изготовления > Механическая обработка > ЧУ, медь. Рядом с полем Машина нажмите ++...++. Выберите вкладку Функц. элемент.

На этой вкладке содержатся данные для машины ЧУ, на которой будет производиться дальнейшая обработка.

Обзор основных элементов диалогового окна:

### Материал

Данные материала для обработки на машине. Здесь можно ввести любой короткий текст; поле может также остаться пустым. Стандартное значение: "Cu".

### Генерировать имена функциональный элементов для экспорта

Эта настройка определяет, из чего складывается уникальное имя медной шины. Под этим именем будут сохраняться шаги обработки медного функц. элемента на машине ЧУ, которые можно воспроизводить в любом количестве.

Существует следующие способы:

* Из непрерывного номера: при выборе этого параметра EPLAN по умолчанию присваивает файлам XML имена по образцу <непрерывный_номер>.XML. Если уже имеются файлы XML, EPLAN начнет нумерацию со следующего доступного числа.
* Из обозначения функц. элемента: При выборе этого параметра EPLAN по умолчанию присваивает файлам XML имена по образцу <Обозначение функц. элемента>.XML.
* Из свойств: Если выбран этот параметр, имя файла составляется из свойств пространства листа, проекта и устройства, а также свободно определяемого разделителя. Используя кнопку ++...++, откройте диалоговое окно Формат: Свойство для конфигурации имени файла.

### Список инструментов

Список инструментов — это файл в формате XML. Он содержит данные о том, какие штамповочные инструменты имеются на машине, и при помощи какого инструмента проводятся соответствующие шаги обработки (штамповка с различным диаметром и различной формы). Список инструментов поставляется производителем машины и корректируется обслуживающим персоналом машины.

Механическая обработка, для которой в списке инструментов отсутствует подходящий инструмент, не подлежит экспорту.

# Версия AutoCAD при экспорте DXF / DWG

Для экспорта страниц DXF / DWG из настроек удалены старые и уже неподдерживаемые версии AutoCAD. В раскрывающемся списке поля Версия формата теперь доступны следующие две записи:

* AutoCAD 2013/2014/2015/2016/2017
* AutoCAD 2018/2019.

(Путь меню для диалогового окна 'Настройки': Параметры > Настройки > Пользователь > Интерфейсы > Экспорт и импорт DXF / DWG; вкладка Экспорт.)

# Вставить графические файлы

В EPLAN можно вставить графические файлы форматов GIF, JPG, BMP, ICO, PNG или TIFF.

Условие:

Вы открыли страницу, форму, рамку или символ.

1. Выберите следующие пункты меню: Вставить > Графика > Графический файл
2. В диалоговом окне Выбрать графический файл выберите файл через меню выбора файлов.
3. В диалоговом окне [Копировать графический файл](gededitgui_d_bildkopieren.md) следует решить, следует ли скопировать этот файл в каталог проекта (т.е. в подкаталог проекта Images) или оставить его в исходном каталоге.
4. Щелкните по кнопке ++OK++.
5. Начертите рамку, в которой должен быть масштабирован рисунок.

!!! info "Для сведения:"

    Оригинальные соотношения страниц при размещении сохраняются.

6. Измените свойства отображения графического файла в диалоговом окне Свойства (графический файл).
7. Щелкните по кнопке ++OK++.

!!! note "Замечание:"

    * При резервировании проекта резервируются только те графические файлы, которые находятся в каталоге проекта.
    * При удалении страницы графические файлы, скопированные в каталог проекта, не удаляются.
    * Не вставленные в каталог проекта графические файлы можно скопировать в каталог проекта через меню Обработать > Другое > Связи и изменить связи.

Если графический файл удален из его каталога и поэтому больше не выводится, вместо него выводится прямоугольник с путем файла.

!!! tip "Совет:"

    Графический файл можно вставить с помощью перетаскивания, выделив графический файл в Windows Explorer и перетащив его в графический редактор проекта с помощью мыши.

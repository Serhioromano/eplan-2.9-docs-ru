# Диалоговое окно Настройки: Общие (Пользователь, Устройство)

Параметры > Настройки > Пользователь > Устройство > Общее

В этом диалоговом окне можно задать специфические для пользователя общие настройки устройств.

Обзор основных элементов диалогового окна:

### Учитывать обозначения выводов устройства при присвоении устройств

!!! note "Замечание:"

    Эта настройка действует только при условии, что при подготовке макросов в проекте макросов на соответствующей рамке макроса для настройки [Учитывать обозначения выводов устройства](macrosgui_r_makrokasteneinstellungen.md) выбран параметр "Из настроек пользователя".

При присвоении устройств выполняется поиск наиболее подходящих функций; поиск, как правило, выполняется на основе определений функций и свойств шаблонов.

Установите этот флажок, если обозначения выводов устройства также должны быть идентичными, чтобы во время присвоения функций или размещения сложных устройств можно было назначить подходящие функции или присвоить их из указанного макроса.

Эта настройка также используется при ***обновлении макросов***. Если этот флажок установлен, обозначения выводов устройства при обновлении переносятся из файлов макроса на вставленные макросы в схеме соединений.

### Сохранить вариант символа / макроса при размещении устройства

При вставке или размещении устройства можно с помощью клавиши ++Tab++ "перелистывать" варианты символа / макроса в символе / макросе символа, который введен в соответствующий шаблон функции.

Если этот флажок установлен, выбранный вариант сохраняется.

* При ***размещении*** устройства из навигатора устройств вариант символа функции сохраняется, если следующие функции относятся к той же категории функции.
* При повторной ***вставке*** устройства для первого символа устройства применяется вариант, который использовался для этого устройства при последней вставке.
Во время вставки для следующих символов устройства используется такая же процедура, как и при размещении — выбранный вариант сохраняется для этой вставки, пока не изменится категория функции.

!!! note "Замечание:"

    Эта настройка действует ***только*** для устройств, определяемых с помощью шаблонов функций. При вставке устройств, которые определяются с помощью макроса, сохраняется только вариант первого символа.

### Удалить ручную мостовую перемычку при копировании

Эта настройка регулирует управление вручную созданных мостовых перемычек при копировании и вставке клемм в ***графическом редакторе***.

Если этот флажок установлен, созданные вручную мостовые перемычки удаляются при копировании и вставке. Значения свойств Вручную созданные мостовые перемычки (внутр.) (ид. 20350) и Вручную созданные мостовые перемычки (внешн.) (ид. 20351) после этого будут удалены.

Эта настройка не действует для копирования в навигаторе страниц или вставки макросов, и в этом случае информация о перемычке всегда сохраняется.

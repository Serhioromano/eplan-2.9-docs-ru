# Схемы кабельных соединений

При использовании стандартных кабелей они часто изображаются в виде однополюсных. В схеме кабельных соединений в этом случае документируется тип строения используемого стандартного кабеля.

* Схема кабельных соединений отображает строение кабеля, включая монтаж по концам (напр., штекер) и информацию по использованию кабеля в проекте.
* Изделия (кабель) можно вывести в схему кабельных соединений.

### Отчет

Для отчета рассматриваются только те кабели, которым присвоена форма в свойстве Форма схемы кабельных соединений. Все другие кабели не учитываются.

Устанавливаемый вручную фильтр позволяет ввести дополнительные ограничения для блоков отчетов, число которых не ограничено. Для сортировки и фильтрации можно дополнительно использовать свойство Форма схемы кабельных соединений. Таким образом на одно имя формы приходится один блок отчетов с числом страниц х. В качестве формы используется форма, указанная на кабеле.

Для отчета 'Спецификация кабеля' через один фильтр можно вывести только те кабели, свойство Форма схемы кабельных соединений у которых не заполнено.

Кабели, для которых существует схема кабельных соединений, подавляются при помощи фильтра в списках обозначений устройств, спецификаций и т. д., но выводятся по умолчанию.

!!! note "Замечание:"

    При создании схем кабельных соединений все кабели, для которых записана одинаковая форма схемы кабельных соединений, выдаются вместе в ***одном*** блоке отчета. Возможная сортировка учитывается только внутри блока отчета.

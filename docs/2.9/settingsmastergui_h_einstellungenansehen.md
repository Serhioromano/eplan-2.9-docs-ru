# Просмотр и изменение настроек

В диалоговом окне Настройки можно увидеть и изменить все настройки для категорий Проекты, Пользователь, Рабочая станция или Фирма.

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Параметры > Настройки
2. В диалоговом окне Настройки просмотрите структуру дерева до необходимой области настроек.
3. Измените требуемые настройки в диалоговом окне настроек справа.
4. Щелкните по кнопке ++OK++.

!!! tip "Совет:"

    Используйте поле ввода Поиск для быстрого поиска определенной настройки. С вводом первых символов в списке под полем ввода отображаются релевантные совпадения.

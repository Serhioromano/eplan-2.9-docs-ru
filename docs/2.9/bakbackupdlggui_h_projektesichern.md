# Резервировать проекты

Проекты можно резервировать из Навигатора страниц и из Управления проектами.

Условие:

Вы выделили проект в Навигаторе страниц.

1. Выберите следующие пункты меню: Проект > Резервировать > Проект
2. В диалоговом окне [Резервировать проекты](bakbackupdlggui_d_projektesichern.md) задайте настройки резервирования данных.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Проект будет резервирован и отправлен по электронной почте в соответствии с настройками в заданном каталоге.

!!! note "Замечание:"

    EPLAN запоминает настройки для соответствующего проекта и снова предлагает их при повторном процессе резервирования.

Условия:

* Вы настроили управление проектами.
* В представлении структуры дерева управления проектами находится считанный каталог, как минимум, с одним проектом.

1. Выберите следующие пункты меню: Проект > Управление
2. Выделите в представлении структуры дерева диалогового окна Управление проектами требуемые проекты.
3. Выберите пункт всплывающего меню Резервировать.
4. В диалоговом окне [Резервировать проекты](bakbackupdlggui_d_projektesichern.md) задайте настройки резервирования данных.
5. Подтвердите ввод.

!!! info "Для сведения:"

    Проекты будут последовательно сжаты и сохранены в заданном каталоге (или отправлены по электронной почте).

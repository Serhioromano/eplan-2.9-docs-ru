# Исключить выводы устройства ПЛК из нумерации

В случае нумерации с данными ПЛК по умолчанию выполняется нумерация всех устройств, содержащихся в объеме и подсоединенных к выводам устройства ПЛК. Однако определенные выводы устройства ПЛК можно исключить из нумерации.

Условие:

Выделите вывод устройства ПЛК в графическом редакторе или навигаторе.

1. Обработать > Свойства
2. В диалоговом окне Свойства ++...++ активируйте во вкладке Вывод устройства ПЛК в таблице свойство "Предотвратить нумерацию с помощью данных ПЛК".
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    При нумерации с данными ПЛК не будет выполнена нумерация устройств, которые подсоединены к этому выводу устройства ПЛК.

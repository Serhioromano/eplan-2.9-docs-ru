# Синхронизировать основные данные

На основные данные (напр. на библиотеку символов) не идет ссылка, а они сохраняются в проекте, то есть копируются. Сохранение используемых основных данных происходит также в меню ***Проект***. В зависимости от настроек проекта в меню Параметры > Настройки > Проекты > "Имя проекта" > Управление > Общее, Параметры > Настройки > Проекты > "Имя проекта" > Управление > Библиотеки символов и Параметры > Настройки > Проекты > "Имя проекта" > Отчеты > Отображение / вывод вы можете при необходимости вручную запустить синхронизацию специфических для проекта основных данных с относящимися к системе данными.

!!! note "Замечание:"

    При сравнении основных данных допускаются имена файлов с прописными и строчными буквами, но обрабатываются они одинаково. То есть имя файла ABC.XYZ идентично с abc.xyz.

Условия:

* Вы открыли проект, основные данные которого должны синхронизироваться.
* Вы ***не открыли*** редактор для обработки основных данных (редактор рамок, редактор форм ...).

### Синхронизировать отдельные объекты основных данных

1. Сервисные программы > Основные данные > Синхронизировать текущий проект
2. В диалоговом окне Синхронизация основных данных — <Имя проекта> щелкните по надписи над столбцами таблиц, чтобы сортировать данные в алфавитно-цифровом порядке по возрастанию или по убыванию в выбранном столбце. Щелкните, например, на столбец Тип, чтобы все записи были показаны отсортированными по ***Типу основных данных***.
3. В списке Основные данные проекта выберите объекты основных данных, которые вы хотите перенести в основные данные системы. (Возможен многократный выбор.)
4. Щелкните по ![](images/all_arrowright_as.png){: .ui-icon }.
5. В списке Основные данные системы выберите объекты основных данных, которые вы хотите перенести в основные данные проекта. (Возможен многократный выбор.)
6. Щелкните по ![](images/all_arrowleft_as.png){: .ui-icon }.
7. Щелкните по кнопке ++Закрыть++.

!!! note "Замечание:"

    При обеих операциях EPLAN указывает вам в окне с информацией на то, сколько данных проектов / системных основных данных были обновлены; если данные проектов и системные основныек данные идентичны, об этом также дается информация.

### Синхронизировать основные данные глобально

1. Сервисные программы > Основные данные > Синхронизировать текущий проект
2. В диалоговом окне Синхронизация основных данных — <Имя проекта> выберите пункт [Обновить] > Проект, чтобы полностью заменить все устаревшие основные данные проекта на более новые основные данные системы.
3. Выберите пункт меню ++Обновить++ > Система, чтобы полностью заменить все устаревшие основные данные системы на более новые основные данные проекта.
4. Щелкните по кнопке ++Закрыть++.

!!! note "Замечание:"

    * При обеих операциях EPLAN указывает вам в окне с информацией на то, сколько данных проектов / системных основных данных были обновлены; если данные проектов и системные основныек данные идентичны, об этом также дается информация.
    * При обновлении ***библиотек символов*** среди прочего проверяется, совместимы ли файлы (например относительно обозначений выводов устройства); если это не так, процесс прерывается с соответствующим сообщением.

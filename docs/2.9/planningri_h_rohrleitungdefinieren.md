# Определить трубопроводы на функциональной схеме автоматизации

В навигаторе предварительного планирования можно предварительно задавать трубопроводы с определенными свойствами. Созданные таким образом объекты планирования (трубопроводы) в дальнейшем можно разместить на функциональной схеме автоматизации. При этом появится точка определения трубопровода со ссылкой на объект планирования (трубопровод). При этом точки определения трубопровода также можно напрямую (без присвоения объекту планирования) вставлять на функциональной схеме автоматизации с помощью символа.

Условия:

* Вы открыли проект, который содержит данные предварительного планирования.
* Открыта страница типа "Функциональная схема автоматизации".
* На этой странице размещены технологические символы, связанные с линиями автоматического соединения.

### Разместить объекты планирования (трубопроводы) в качестве точек определения трубопровода

1. Выберите следующие пункты меню: Данные проекта > Предварительное планирование > Навигатор
2. Выделите объект планирования (трубопровод) для размещения.
3. Выберите пункт всплывающего меню Разместить.
4. Разместите точку определения трубопровода на линии автоматического соединения.

!!! info "Для сведения:"

    Точке определения трубопровода присваиваются данные объекта планирования (трубопровода).

Если необходимо повторно внести изменения в присвоенные данные, откройте диалоговое окно "Свойства" для точки определения трубопровода (напр., двойным щелчком мыши) и на вкладке Определение трубопровода введите другие свойства для трубопровода.

!!! tip "Совет:"

    Вы также можете размещать точки определения трубопроводов, просто перетаскивая выделенный объект планирования (трубопровод) из навигатора предварительного планирования на линию автоматического соединения открытой страницы функциональной схемы автоматизации в графическом редакторе.

### Начертить точки определения трубопровода

1. Выберите следующие пункты меню: Вставить > Точка определения трубопровода
2. Разместите точку определения трубопровода на линии автоматического соединения.
3. В диалоговом окне Свойства ++...++ на вкладке Определение трубопровода введите свойства трубопровода (а при необходимости и другие свойства в других вкладках).
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Свойства присваиваются трубопроводу, проходящему через точку определения трубопровода.

!!! tip "Совет:"

    Вы можете добавлять одновременно несколько точек определения трубопровода, удерживая нажатой левую клавишу мыши и протягивая курсор через несколько линий автоматического соединения. В этом случае на каждой пересеченной линии автоматического соединения будет размещена точка определения трубопровода.

Вставленной точке определения трубопровода можно присвоить объект планирования (трубопровод) из навигатора предварительного планирования с помощью пункта всплывающего меню Присвоить. Таким образом точка определения трубопровода связывается с объектом планирования (трубопроводом). Данные объекта планирования (трубопровода) переносятся на точку определения трубопровода.

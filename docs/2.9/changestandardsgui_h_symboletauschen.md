# Заменить библиотеки символов и символы (Изменение стандарта)

Можно заменить библиотеки символов и отдельные символы.

Условия:

* Вы открыли проект.
* При нескольких открытых проектах: В навигаторе страниц выделен проект.
* Вы настроили целевой проект.
* Вы определили шаблон и поворот / инверсию.

### Присвоить целевую библиотеку символов

1. Сервисные программы > Изменение стандарта > Вкладка Проект > [Далее] > Вкладка Поворот / инверсия > [Далее]
2. На вкладке Основные данные в столбце Имя выберите библиотеку символов, которую необходимо заменить.
3. Щелкните внутри той же строки в столбце Создать имя по ++...++.
4. В диалоговом окне Выбрать библиотеку символов выберите библиотеку символов, которая будет использоваться для целевого проекта, и щелкните ++Открыть++.

!!! info "Для сведения:"

    Библиотека символов будет занесена в столбец Создать имя.

5. Чтобы присвоить всю библиотеку символов, щелкните на вкладке Основные данные по ++Далее++, и щелкните на вкладке Символы по ++Далее++.

!!! info "Для сведения:"

    [Определить Структура проекта](changestandardsgui_h_projektstrukturanpassen.md).

6. Чтобы по отдельности присвоить символы исходной библиотеки символов символам целевой библиотеки символов, на вкладке Основные данные рядом с целевой библиотекой символов установите флажок Присвоить отдельно и щелкните по ++Далее++.

!!! info "Для сведения:"

    Исходная и целевая библиотека символов будут внесены на вкладку Символы.

7. На вкладке Символы в столбце Схема выберите схему присвоения.

!!! info "Для сведения:"

    Если вы хотите использовать существующую схему присвоения, щелкните ++Схема++ и составьте схему присвоения.

!!! info "Для сведения:"

    [Создать схему присвоения для символов](changestandardsgui_h_stammdatenersetzen.md)

8. Щелкните по ++Далее++.

!!! info "Для сведения:"

    [Определить Структура проекта](changestandardsgui_h_projektstrukturanpassen.md).

# Импортировать тексты изделий в словарь

Тексты изделий можно импортировать из базы данных изделий в словарь. При этом все тексты, которые доступны для перевода и присутствуют в базе данных изделий на выбранном языке, будут включены в словарь в качестве ключевых слов.

Условия:

* Вы можете выбрать базу данных изделий в меню Параметры > Настройки > Пользователь > Управление > База данных изделий.
* В базе данных изделий находятся тексты на одном или нескольких языках.

1. Выберите следующие пункты меню: Сервисные программы > Перевод > Обработать словарь
2. Выберите вкладку Управление.
3. Рядом с полем Словарь щелкните ++...++, в выведенном диалоговом окне выберите словарь системы EPLAN, в который вы собираетесь импортировать.
4. Подтвердите ввод.
5. Выберите пункты меню Сервисные программы > Изделие > Управление.
6. Выберите пункты меню ++Дополнительно++ > Перевод > Настройки.
7. Щелкните в поле Перевод рядом с полем Языки по ![](images/all_new_as.png){: .ui-icon } (Создать), выберите один за другим те языки перевода, тексты изделий на которых хотите импортировать в словарь, и щелкните по ++OK++.
8. Выберите ++Дополнительно++ > Перевод > Импортировать тексты изделий.

!!! info "Для сведения:"

    В диалоговом окне Выбрать языки доступны все выбранные языки перевода.

9. Выделите языки, которые Вы хотите импортировать.

!!! info "Для сведения:"

    Если выбранные языки не являются языками словаря, появится сообщение "Язык отсутствует".

!!! info "Для сведения:"

    Тексты изделий импортируются в словарь.

10. Щелкните по кнопке ++Закрыть++.
11. Чтобы проверить импорт, выберите пункт меню Сервисные программы > Перевод > Обработать словарь.
12. Выберите на вкладке Обработать словарь в поле Исходный язык один из языков, которые вы импортировали.

!!! info "Для сведения:"

    В поле Поиск будут показаны все импортированные тексты изделий на выбранном языке.

# Диалоговое окно Настройки: Синтаксическая проверка ОУ

Вы открыли проект.

* Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Синтаксическая проверка ОУ.
* В настройках для проверки синтаксиса ОУ проверка активирована. Вы ввели в диалоговом окне Свойства ++...++ недопустимый символ в качестве элемента обозначения. Щелкните в диалоговом окне Синтаксическая проверка ОУ кнопку ++Настройки++.

В этом диалоговом окне укажите в связи с проектом, какие символы допустимы для блоков идентификаторов ОУ, обозначений устройств и счетных номеров. Эти настройки можно определять отдельно для структурных идентификаторов (например, страниц) и для ОУ электротехники и Fluid-Техники.

Доступны следующие элементы диалогового окна:

Вкладка Структурные идентификаторы

Активировать проверку

Установите этот флажок, если необходимо проверять блоки идентификаторов. После ввода свойств страницы или свойств устройства EPLAN автоматически проверит правильность синтаксиса введенного обозначения устройства. Если синтаксис неправильный, то появляется окно сообщения, в котором отображаются и выделяются неверные элементы.

!!! note "Замечание:"

    Блоки идентификаторов для ОУ электротехники или Fluid-техники ***не*** проверяются, если на вкладке Электротехника или Fluid-Техника не установлен флажок Активировать проверку.

### Блоки идентификаторов

В этом групповом поле задайте объем проверки блоков идентификаторов (например, Установка и Место установки в случае со структурой идентификаторов по DIN), причем можно указать следующее:

* Цифры: Выберите данный параметр, если при проверке должны учитываться только числа.
* Все прописные буквы и числа: Выберите этот параметр, если при проверке должны учитываться все прописные буквы и числа.
* Все прописные и строчные буквы и числа: Выберите этот параметр, если при проверке кроме прописных букв и чисел должны также учитываться строчные буквы.
* Допустимые спец. символы: Установите этот флажок, если спецсимволы, приведенные рядом с полем, должны быть разрешены при вводе ОУ.
* Текстовые символы: Установите этот флажок, если должны допускаться дополнительные спецсимволы (напр., умлауты); соответствующие спецсимволы Вы введете позже в поле ввода, находящемся рядом.
* Для номеров установок только числа: Установите этот флажок, если при проверке номеров установок должны допускаться только числа.

* * *

Вкладки Электротехника / Fluid-Техника

Выполненные здесь настройки действительны только для элементов обозначения устройств полного ОУ (например, Буквенное обозначение и Счетчик), но не для блоков идентификаторов.

Активировать проверку

Установите этот флажок, если обозначения устройств должны проверяться. Затем EPLAN после ввода свойств устройства автоматически проверяет правильность синтаксиса введенного обозначения устройства. Если синтаксис неправильный, то появляется окно сообщения, в котором отображаются и выделяются неверные элементы.

Если флажок не установлен, то ОУ электротехники/Fluid-Техники не проверяется.

!!! note "Замечание:"

    Если на вкладке Электротехника или Fluid-Техника установлен флажок Активировать проверку, то проверяется вся символьная строка обозначения устройства, т. е. полное ОУ, включая блоки идентификаторов. Для блоков идентификаторов используются настройки на вкладке Структурные идентификаторы, даже если флажок не установлен.

### Обозначение устройства

В данном групповом поле укажите объем проверки обозначений устройств:

* Все прописные буквы и числа: Выберите этот параметр, если при проверке должны учитываться все прописные буквы и числа.
* Все прописные и строчные буквы и числа: Выберите этот параметр, если при проверке кроме прописных букв и чисел должны также учитываться строчные буквы.
* Допустимые спец. символы: Установите этот флажок, если спецсимволы, приведенные рядом с полем, должны быть разрешены при вводе ОУ.
* Текстовые символы: Установите этот флажок, если должны допускаться дополнительные спецсимволы (напр., умлауты); соответствующие спецсимволы Вы введете позже в поле ввода, находящемся рядом.
* Предш. цифра: Выберите из раскрывающегося списка, должны ли предшествующие цифры допускаться, запрещаться или принудительно предписываться.

### Счетный номер

В данном групповом поле укажите объем проверки счетных номеров; до указания Текстовых спецсимволов, которые ***недоступны*** для счетных номеров, возможные данные также соответствуют данным в Блоках идентификаторов (на вкладке Структурные идентификаторы).

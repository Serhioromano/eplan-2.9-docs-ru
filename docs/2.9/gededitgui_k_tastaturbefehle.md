# Обзор комбинаций клавиш

С помощью комбинаций клавиш в EPLAN можно очень быстро вызвать команды. В диалоговом окне Настройки: Комбинации клавиш можно изменить комбинации клавиш и установить собственные, а при необходимости — сбросить комбинации клавиш до настроек по умолчанию.

!!! note "Замечание:"

    Обратите внимание, что некоторые комбинации клавиш используются в EPLAN или в Windows. При попытке присвоить такую внутреннюю комбинацию клавиш команде меню будет выведено соответствующее сообщение.

    Например, в EPLAN для функции Область ввода данных необходимы клавиши +++++, [-] и клавиши с цифрами. По этой причине эти клавиши больше нельзя использовать как самостоятельную комбинацию клавиш.
    В Windows, к примеру, используется комбинация клавиш [Ctrl] \+ [F6] для перехода в следующее окно документа или комбинация клавиш [Shift] \+ стрелка [Вправо] для выделения символа справа от индикатора вставки.
    Комбинации клавиш [F1], [F4], [F6] с [Ctrl], [Shift], [Alt] также нельзя назначать.

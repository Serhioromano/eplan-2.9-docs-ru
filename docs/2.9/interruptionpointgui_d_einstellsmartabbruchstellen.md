# Диалоговое окно Настройки: Точки разрыва для интеллектуального соединения

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Интеллектуальное соединение. Щелкните рядом с полем Символ точек разрыва по кнопке ++...++.

В этом диалоговом окне можно управлять схемами для точек разрыва, используемыми при интеллектуальном соединении, и обрабатывать их.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Выбор символа

В этой таблице обрабатывается схема точек разрыва. В такой схеме точек разрыва для интеллектуального соединения указываются данные для каждого направления вывода устройства и для страниц источника и целевых страниц.

Таблица содержит следующие столбцы:

* Направление вывода устройства: Здесь отображаются возможные направления вывода устройства. Для каждого направления вывода устройства можно выбрать отдельные настройки для символа.
* Символ: Для перехода в диалоговое окно Выбор символа и выбора требуемого символа точек разрыва щелкните по этой ячейке, а затем — по кнопке ++...++.
* Вариант: Выберите требуемый вариант символа из раскрывающегося списка.
* Порядок свойств: Выберите один из возможных порядков свойств из раскрывающегося списка.
* Интервал: Введите здесь интервал между выводом устройства и точкой вставки точки разрыва. Стандартное значение: 4 мм.

### Вставить прерывание для автоматического соединения

Эта настройка имеет смысл в случае использования для интеллектуального соединения точек разрыва с двумя выводами устройства. Если этот флажок установлен, на лишнем выводе устройства точек разрыва автоматически вставляется прерывание. Это прерывание предотвращает появление еще одного соединения.

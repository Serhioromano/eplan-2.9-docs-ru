# Автоматически оптимизировать сети

Во время автоматической оптимизации соединения сети / шлейфы преобразуются так, чтобы следующие условия были выполнены:

* Все остальные выводы устройства подключены.
* Минимально возможная сумма прокладываемых проводов.
* Максимум два соединения оканчиваются у одного вывода устройства.
* Всегда генерируется один шлейф.

При автоматической оптимизации все соединения сети получают свойства (в том числе определение функции) первого соединения. Фиксированные соединения (перемычки, прямые соединения, внутренние соединения) остаются; их нельзя оптимизировать.

Условие:

Сгенерированы маршрутизируемые соединения. В навигаторе пространства листа или в навигаторе соединений выделен компонент, содержащий маршрутизируемые соединения (пространства листа, электрошкафы, монтажные поверхности, трехмерные размещения изделий, соединения).

1. Выберите пункты меню Данные проекта > Соединения > Автоматически оптимизировать сети.

!!! info "Для сведения:"

    Если не выбрана ни одна сеть или выбрана неполная сеть, появится сообщение об ошибке. В этом случае повторите и измените выбор.

2. В диалоговом окне Автоматически оптимизировать сети выберите настройки для оптимизации сетей.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Проводится анализ сетей.

!!! info "Для сведения:"

    Отобразится диалоговое окно Автоматически оптимизировать сети: Предв. просмотр результата. В диалоговом окне отображаются результаты анализа. Соединения, найденные с помощью оптимизации, автоматически маршрутизируются заново с помощью клавиши ++OK++.

4. В диалоговом окне Автоматически оптимизировать сети: Предв. просмотр результата щелкните по ++OK++.

!!! info "Для сведения:"

    Соединения со статусом 'новые' будут проложены по новым трассам маршрутизации.

# Вкладка Монтажные данные

Сервисные программы > Изделие > Управление. В дереве или списке выделено изделие, которое ***не*** относится к Группе продуктов 'Кабели'.

На этой вкладке можно установить монтажные данные для изделий, которые ***не*** относятся к Группе продуктов "Кабели".

Обзор основных элементов диалогового окна:

### Ширина / Высота / Глубина

Эти поля используются для ввода размеров изделия в мм. В случае использования систем разводки данные о размерах следует вводить в соответствии с монтажной шиной, кабельной развилкой или перфорированной планкой.

!!! note "Замечание:"

    Если необходимо разместить цоколь электрошкафа из диалогового окна Размещение принадлежностей, который определен свойствами Ширина, Высота и Глубина, следует поменять значения свойств Глубина и Высота изделия, чтобы принадлежность разместилась в правильном положении.

### Занимаемая площадь

Это значение рассчитывается в том случае, если сделан соответствующий выбор в меню ++Дополнительно++ > Занимаемая площадь. При этом используются значения из полей Ширина и Высота, а для расчета используется следующая формула:
(b*h) где b = ширина и h = высота.

### Внешнее размещение

Установите этот флажок, если изделие обозначается как "внешне размещенный" и тем самым должен подсоединяться от вида использования в компоновке элетрошкафа (напр., двигатели или генераторы).

### Графический макрос

Графический макрос исполняет отношение между изделием и макросов EPLAN для графического представления этого изделия. С помощью флажка Предв. просмотр в диалоговом окне Выбрать графич. макрос можно вывести предварительный просмотр графики для текущего выделенного файла макроса. Предварительный просмотр показывается правильно лишь в том случае, когда проект открыт; если открытых проектов нет, специфическая для проекта информация отсутствует. В этом случае отсутствующие символы представляются красными крестиками.

В этом поле можно также задать контуры типа "Выдавить контур". Эти контурные чертежи можно использовать для размещения определенных пользователем шин в пространстве листа.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

!!! note "Замечание:"

    Соблюдайте наши [рекомендации по работы с 2D- и 3D-макросами](cabinetgui_k_3dmakros.md).

### Графический файл

!!! note "Замечание:"

    Это присвоение важно прежде всего для выбора устройства и получают из выбора штекера с предварительным просмотром символа штекера.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Несовпадение центров

Если в виде спереди изделие размещается не по центру, здесь следует ввести расстояние для смещения относительно середины несущей шины. Функциональный элемент будет автоматически перемещен на заданное расстояние.

!!! example "Пример:"

    (1) несовпадение центров = 0; (2) положительное значение для смещения центра; (3) отрицательное значение для смещения центра

!!! warning "Предупреждение:"

    Эта функциональность предназначена ***исключительно*** для приложения "Pro Panel"; то есть свойство ***не*** влияет на двухмерную компоновку электрошкафа.

### Высота крепления зажимом

Высота крепления зажимом описывает позицию в виде сбоку или сверху/снизу. При размещении на несущей шине функциональный элемент автоматически размещается на шине на нужной глубине установки. При замене шины функциональный элемент соответствующим образом перемещается по высоте. Таким образом, выравнивание функционального элемента осуществляется не по нижнему краю шины, а по верхней кромке с учетом введенного здесь значения.

!!! example "Пример:"

    (1) высота крепления зажимом

!!! warning "Предупреждение:"

    Эта функциональность предназначена ***исключительно*** для приложения "Pro Panel"; то есть свойство ***не*** влияет на двухмерную компоновку электрошкафа.

### Глубина установки

Введите глубину установки в мм (предварительная настройка — 0 мм). Это свойство изделия учитывается при размещении изделия 3D только в том случае, если для размещения изделия не используется графический макрос 3D. Значение автоматически указывает, насколько функциональный элемент после размещения изделия входит в другой функциональный элемент по оси Z и относительно монтажной поверхности.

!!! example "Пример:"

    (1) Функциональный элемент (дверь), (2) монтажная поверхность, (3) размещение изделия, (4) глубина установки

### Текстура

Для выбора требуемого графического файла для текстуры в следующем диалоговом окне выбора файла щелкните по ++...++ в этом поле. Обозначенный в этом поле графический файл при размещении изделия в пространстве листа (при наличии размеров изделия) будет отображен на передней поверхности размещенного 3D-объекта. При применении графического файла в качестве текстуры поверхность 3D-модели будет детализирована и представлена более реалистично, либо будет отображена фотография оригинального функционального элемента.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Интервалы установки

Эти поля используются для ввода интервала установки в мм. В случае использования систем разводки данные о размерах следует вводить в соответствии с монтажной шиной, кабельной развилкой или перфорированной планкой.
Если в этих поля введены значения, расчет занимаемой площади ([Дополнительно] > Занимаемая площадь) производится по следующей формуле:
((b+ab) * (h+ah)), где b = ширина, ab = интервал установки по ширине, h = высота и ah = интервал установки по высоте.

# Вкладка Настройки

Сервисные программы > Изделие > Управление > [Дополнительно] > Импортировать. В поле Тип файла выбрана запись для версии eCl@ss. Щелкните кнопку ++...++ рядом с полем Тип файла и выберите вкладку Настройки.

На этой вкладке можно задать определенные настройки для присвоения признаков eCl@ss свойствам изделий базы данных изделий.

Обзор основных элементов диалогового окна:

### Производитель (краткое имя)

Для однозначного присвоения новых номеров изделий EPLAN следует, используя это поле ввода, указать краткое имя производителя перед самим номером изделия. Например, введите "ABC": такое название получит номер изделия "4711" после импорта "ABC.4711".
Эта символьная строка используется также при генерировании имен схем соединений и сверления, а также при генерировании определенных пользователем каталогов.

!!! note "Замечание:"

    Обратите внимание, что для последующего присвоения изделиям в EPLAN Data Portal необходимо использовать ***трехзначное*** краткое имя производителя.

### Генерировать схемы соединений / Генерировать схемы сверления

Установите эти флажки, если во время импорта данных eCl@ss необходимо выполнить автоматическое генерирование схем соединений / сверления. При этом присвоение соответствующих свойств выполняется автоматически с помощью схемы импорта.

!!! note "Замечание:"

    Обратите внимание, что при генерировании краткое имя производителя, указанное в поле Производитель (краткое имя), стоит перед именем схемы соединений / сверления.

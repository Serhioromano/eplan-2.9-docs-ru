# Создать и удалить страницы

### Создание страниц

Условие:

Вы открыли проект. Открыто диалоговое окно навигатора Страницы — <Имя проекта>.

1. Выберите пункты меню Страница > Создать.

!!! info "Для сведения:"

    Откроется диалоговое окно Новая страница. Если в диалоговом окне навигатора Страницы — <Имя проекта> была выделена какая-либо страница, ее свойства копируются в это диалоговое окно. Если был выделен уровень структуры, копируются свойства первой страницы, находящейся под этим уровнем. Настроенная сетка всегда копируется из рамки.

2. Щелкните по кнопке ++Очистить поля++, если необходимо очистить поля данных текущей выделенной страницы, то есть удалить их содержимое.

!!! info "Для сведения:"

    Содержимое полей данных удаляется. Только "тип страницы" занят типом выделенной страницы. Кроме того, остается автоматически назначенное имя страницы. Масштаб и сетка копируются из рамки и также остаются в свойствах.

3. Измените при необходимости в поле Полное имя страницы предложенные имена страниц вручную или откройте с помощью кнопки ++...++ диалоговое окно Полное имя страницы, чтобы обработать структурные идентификаторы и имена страницы. Нажав кнопку ++...++, там можно выбрать уже имеющиеся структурные идентификаторы (например, установку и имя установки). Идентификаторы можно ввести и вручную, при этом имеется возможность создания новых идентификаторов.
4. Присвойте странице в поле Тип страницы интерактивный тип страницы из раскрывающегося списка или откройте диалоговое окно Добавить тип страницы, нажав кнопку ++...++, и выберите тип страницы из списка.
5. Введите в поле Описание страницы информативный описательный текст; он также отображается в представлении в виде дерева или списка навигатора страниц.
6. При необходимости введите свойства для новой страницы.
7. Если необходимо создать несколько страниц, щелкните несколько раз по кнопке ++Применить++.

!!! info "Для сведения:"

    С каждым щелчком по кнопке ++Применить++ генерируется новая страница. В поле Автор автоматически вносится идентификатор пользователя.

8. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Если для нумерации зон (Параметры > Настройки > Проекты > "Имя проекта" > Управление > Страницы) выбрана одна из настроек — По проектам или На каждый структурный идентификатор, то выдается запрос о необходимости автоматического обновления обозначений зон.

!!! info "Для сведения:"

    EPLAN проверяет, не существует ли уже введенное вручную имя страницы, и если этот так, выводится соответствующе сообщение, после чего следует ввести другое имя.

!!! info "Для сведения:"

    (Последняя) новая страница открывается в графическом редакторе.

!!! tip "Совет:"

    * Для создания новых страниц можно воспользоваться кнопкой ![](images/all_newpageactive_as.png), расположенной на стандартной панели инструментов.
    * В навигаторе страниц с помощью пункта всплывающего меню Создать можно создавать новые страницы. При этом можно выделить любую страницу или уровень структуры.

### Удалить страницы

1. Выделите требуемые страницы в диалоговом окне навигатора Страницы — <Имя проекта>.
2. Выберите пункт всплывающего меню Удалить.

!!! info "Для сведения:"

    После подтверждения запроса выделенные страницы будут удалены из проекта.

!!! note "Замечание:"

    Размещенные функциональные элементы при удалении страницы полностью убираются.
    Неразмещенные функциональные элементы не убираются, так как они не выделены.

!!! tip "Совет:"

    Используя пункты меню Обработать > Выделить > Страница, можно выделить страницу, открытую в графическом редакторе, и удалить ее, нажав клавишу ++Delete++. При этом необязательно открывать навигатор страниц.

### Удалить все страницы проекта

1. Выделите имя проекта, из которого требуется удалить все страницы, в диалоговом окне навигатора Страницы — <Имя проекта>.
2. Выберите пункт всплывающего меню Удалить.

!!! info "Для сведения:"

    После подтверждения запроса все страницы проекта будут удалены.

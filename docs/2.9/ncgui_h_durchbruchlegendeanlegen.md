# Генерировать легенду монтажного отверстия для 2D-отображения отверстий

Для монтажных отверстий, представленных в 2D-отображении отверстий, можно генерировать отчеты. Самые важные данные этих вставленных вручную или скопированных из схем сверления монтажных отверстий можно наглядно отобразить в таблице.

Для этого в диалоговом окне Определить отчет предусмотрен тип отчета "Легенда монтажного отверстия". По аналогии с легендой электрошкафа можно выводить легенду монтажного отверстия в виде новой страницы отчета или размещать вручную в виде встроенного отчета на имеющейся странице проекта.

В отчетах этого типа в табличном виде представлены самые важные данные монтажных отверстий (напр., такие обозначения функциональных элементов, как тип сверления "Отверстие", "Прямоугольник", "Продольное отверстие" и т. д., координаты X и Y точек захвата, размер монтажного отверстия и т. д.).

Условия:

* Вы открыли проект.
* Проект содержит важные для ЧУ монтажные отверстия.
* В Графическом редакторе открыта страница проекта. Страница содержит 2D-отображение отверстий.
* Свойство Подавить генерирование легенды электрошкафа ***не*** активировано в диалоговом окне свойств 2D-отображения отверстий.
* В диалоговом окне Настройки: Вывод в страницах задана форма для типа отчета Легенда монтажного отверстия с типом файла "Легенда монтажного отверстия (*.f47)".

1. Выберите следующие пункты меню: Сервисные программы > Отчеты > Генерировать
2. Щелкните в диалоговом окне Отчеты — <Имя проекта> по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
3. В диалоговом окне Определить отчет выберите из раскрывающегося списка Форма вывода запись Размещение вручную.
4. Выберите тип отчета Легенда электрошкафа из списка доступных типов отчетов. Щелкните по кнопке ++OK++.
5. При необходимости задайте другие настройки фильтра в диалоговом окне Настройки — Легенда монтажного отверстия. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Легенда прикреплена к курсору.

6. Разместите легенду на странице в графическом редакторе.

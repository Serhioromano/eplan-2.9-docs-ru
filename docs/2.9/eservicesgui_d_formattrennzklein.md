# Диалоговое окно Формат: Разделитель

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация (online). В диалоговом окне Настройки: Нумерация (online) рядом с полем Формат нумерации щелкните по кнопке ++...++. Выберите в диалоговом окне Форматы нумерации определенную пользователем схему. Щелкните на вкладке ОУ или Обозначения по кнопке ++...++. Дважды щелкните по элементу формата 'Разделитель' в диалоговом окне Формат ОУ.

Вы открыли проект. Открыто диалоговое окно [Настройки: ПЛК](plcgui_d_allgemeinespseinstellungen.md) (напр., через меню Данные проекта > ПЛК > Настройки). В диалоговом окне Настройки: ПЛК рядом с полем Специфические для ПЛК настройки вы щелкнули кнопку ++...++. В диалоговом окне [Настройки: Специфич. для ПЛК](plcprocessorgui_d_spsspezifischeeinstellungen.md) выбрана вкладка Форматы адресов, а рядом с полем для требуемого формата адреса нажата кнопка ++...++.

В диалоговом окне Формат адреса ПЛК выберите в списке Доступные элементы формата элемент формата типа Разделитель и щелкните по ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо) (или щелкните два раза в списке Выбранные элементы формата по уже определенному разделителю).

В этом диалоговом окне указывается форматирование разделителя для группы форматов.

Обзор основных элементов диалогового окна:

### Пробел

Если активирована эта опция, в качестве разделителя используется пробел. Число пробелов вводится в поле Число символов.

### Другие символы

Если этот параметр активирован, можно ввести в нижележащее поле нужную комбинацию символов, которая будет использоваться в качестве разделителя. Допускаются любые цифры, буквы и специальные символы.

### Число символов

Это поле доступно в том случае, если активирована опция Пробел. Число в этом поле указывает на то, сколько пробелов должно быть выведено в формате нумерации в качестве разделителей. В случае выбора параметра Другие символы это поле становится недоступным и в нем выводится число введенных символов.

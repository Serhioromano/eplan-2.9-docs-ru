# Вкладка Система сборных шин

Сервисные программы > Изделие > Управление. В дереве или списке выделено одно изделие Главной группы продуктов 'Механика', Группы продуктов 'Сборные шины', Подгруппы продуктов 'Система'.

На этой вкладке установите специфические данные для систем сборных шин, т. е. для изделий Главной группы продуктов 'Механика', Группы продуктов 'Сборные шины', Подгруппа продуктов 'Система'. При этом данные для каждого изделия могут указываться ***многократно***, а именно один раз для каждого варианта. Кронштейны сборной шины служат при этом для надежного и рационального вывода устройства встроенных устройств-распределителей, таких как силовые выключатели, модульные монтажные устройства и т.п.

Обзор основных элементов диалогового окна:

### Геометрия профиля Г x В

Введите здесь соответствующие значения, например, "35x12x20x5".

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

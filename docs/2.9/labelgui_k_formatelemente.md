# Элементы формата для файла экспорта / маркировки / Внешний файл отчетов

С помощью элементов формата указывается, какие данные экспорта / маркировки / проекта выводятся в файлах экспорта / маркировки / внешних файлах отчетов.

Также с помощью элементов формата задаются свойства форматирования, например новые строки и разбивка на страницы. Следующая таблица содержит некоторые доступные элементы формата:

Элемент формата |  Верхний колонтитул |  Этикетка |  Нижний колонтитул
---|---|---|---
***Свойство:*** Указывает, какие данные проекта должны выводиться в файле экспорта / маркировки / файле отчетов.  |  x |  x |  x
***Разделитель:*** Вставляет разделитель после текста экспорта / маркировки / текста отчетов. |  x |  x |  x
***Новая строка:*** Вставляет новую строку после текста экспорта / маркировки / текста отчетов. Позволяет создать многострочный текст. |  x |  x |  x
***Разбивка на страницы:*** Создает разбивку на страницы |  |  |  x
***Разбивка на странице при смене ОУ:*** Создает разбивку на страницы при изменении обозначения устройства. |  |  |  x
***Вычисление:*** позволяет создать расчетную формулу. |  |  x |
***Операция:*** Позволяет указать операцию. |  |  x |

# Диалоговое окно Настройки: Выбор изделия API

Вы открыли проект.

* Параметры > Настройки > Пользователь > Управление > Выбор изделия. Выберите в диалоговом окне "Настройки": Выбор изделия определенную пользователем схему из раскрывающегося списка. Выберите опцию API и щелкните по кнопке ++...++.
* Вы выделили главную функцию в графическом редакторе или навигаторе. "Обработать" > "Свойства". Вы щелкнули на вкладке Изделие рядом с полем Источн. данных по кнопке ++...++. Выберите в диалоговом окне "Настройки": Выбор изделия (пользователь) определенную пользователем схему из раскрывающегося списка. Выберите опцию API и щелкните по кнопке ++...++.

В этом диалоговом окне можно установить, какие свойства изделия загружаются через интерфейс API.

Выбор изделия через API является чисто функциональным интерфейсом. Как настроена возможность выбора и выглядит диалоговое окно выбора, зависит от внедрения соответствующего сервера API.

Доступны следующие элементы диалогового окна:

### Имя операции

В этом поле вводится операция, которая начинается при выборе изделий через интерфейс API. Эта операция готовится через программу API.

# Операция: ProjectAction

**Параметр** |  **Описание**
---|---
Project  |  Используемый проект.
Action  |  Используемая операция, за ней следует параметр операции.
NOCLOSE  |  Если "1", то проект впоследствии не закрывается (необязательный). По умолчанию проект впоследствии закрывается.
OpenMode |  Задает режим обработки при открытии проектов (необязательно). Возможные значения:
STANDARD: Открытый таким образом проект допускает любую обработку. В многопользовательском режиме проект могут обрабатывать сразу несколько пользователей.
READONLY: Проект открывается с защитой от записи.
EXCLUSIVE: Проект открывается только для текущего пользователя. В многопользовательском режиме другие пользователи не имеют доступа к открытому таким образом проекту.

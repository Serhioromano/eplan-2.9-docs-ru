# Схемы

Схема — это группа настроек, которые сохраняются под определенным именем. Для большинства настроек существуют стандартные схемы EPLAN, которые нельзя обрабатывать. Они создаются во время установки в собственном каталоге (по умолчанию в меню \Схемы\<Идентификатор фирмы> на дисководе и в каталоге, в котором произведена установка EPLAN). Так, например, существуют схемы для экспорта данных изготовления, настроек нумерации для устройства, кабелей и т. д., рабочих областей и т. п.

Чтобы задать собственные настройки, можно создать определенные пользователем схемы: С помощью [панели инструментов](schemeconfig_m_schemaschaltflaechen.md), расположенной рядом с полем Схема, можно создавать новые схемы, сохранять изменения в них, копировать или удалять. Кроме того, схемы можно импортировать и экспортировать. При импорте возможен многократный выбор схем, а при экспорте — нет.

# Переместить тексты свойств в обзоры модели

С помощью команды Переместить тексты свойств активируются точки включения текстов свойств функциональных элементов в обзор модели, которые становятся видимыми, если щелкнуть по отдельному условному обозначению. Их можно переместить с помощью перетаскивания мышью.

Условия:

* Вы открыли страницу, которая содержит обзор модели.
* Отображение маркировки функционального элемента включено.

1. Выделите обзор модели.
2. Выберите пункт всплывающего меню Текст > Переместить текст свойства.
3. Выделите текст свойства, который хотите переместить.

!!! info "Для сведения:"

    Точки элементов текста свойств отображаются как небольшие незаполненные квадраты.

4. Переместите текст свойства.

!!! tip "Совет:"

    Если удерживать нажатой клавишу ++Shift++ при перемещении, текущая сетка будет разделена на "четверти". При этом точку вставки можно перемещать в шагах, которые имеют значение четверти шага сетки.

### Сбросить перемещенные тексты свойств

С помощью параметра Сбросить перемещенный текст свойства все перемещенные тексты свойств выделенных функциональных элементов удаляются. И снова отображаются тексты свойств, сохраненные в функциональном элементе. (Каждый функциональный элемент содержит информацию о том, какие тексты свойств по умолчанию отображаются при его вставке.)

1. Выделите нужные функциональные элементы и выберите пункт всплывающего меню Текст > Сбросить перемещенный текст свойства.

!!! info "Для сведения:"

    У всех выделенных функциональных элементов будут удалены скопированные тексты свойств и восстановлены тексты свойств функционального элемента, которые отображаются по умолчанию.

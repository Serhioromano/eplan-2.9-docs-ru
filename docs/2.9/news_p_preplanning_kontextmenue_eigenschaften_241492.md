# Пункт всплывающего меню для свойств присвоенного сегмента

Для функций на схеме соединений или для точек определения трубопровода на функциональной схеме автоматизации, которые присвоены сегменту в предварительном планировании, во всплывающем меню ***графического редактора*** теперь доступен пункт Свойства (предварительное планирование). С помощью этого пункта меню открывается диалоговое окно "Свойства" присвоенного сегмента.

![](images/benefit.png)Эффект:

С помощью пункта всплывающего меню Свойства (предварительное планирование) вы можете сразу обрабатывать свойства присвоенных сегментов из графического редактора, минуя навигатор предварительного планирования.

Если этот пункт всплывающего меню выбрать, например, на странице схемы соединений для функции, связанной с предварительным планированием, откроется диалоговое окно "Свойства" присвоенного объекта планирования.

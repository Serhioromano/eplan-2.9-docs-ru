# Отображение инструмента для монтажных работ

Инструменты для монтажных работ, например точки захвата, точки монтажа, монтажные поверхности и т. д., являются элементами логики устройств. Данные элементы видны только в проекте схемы соединения при их использовании, т. е. во время размещения в пространстве листа. В некоторых ситуациях необходимо сделать инструменты для монтажных работ видимыми вне процессов размещения, чтобы их обработать. Через пункты меню Вид > Инструмент для монтажных работ в проекте схемы соединений при необходимости выполняется управление видимостью инструмента для монтажных работ.

!!! tip "Совет:"

    При размещении устройства может возникнуть ситуация, когда разные инструменты для монтажных работ лежат друг на друге на одном месте и при этом могут в равной степени использоваться для размещения. В подобной ситуации найденные интерактивные точки и поверхности после щелчка по ним предлагаются на выбор в виде списка. Выбрать отдельные записи в списке можно щелчком мыши или при помощи клавиш курсора. При этом соответствующая точка или поверхность зрительно выделяется, и отображается предварительный просмотр ожидаемого в этом месте размещения.

Условия:

* Вы открыли проект.
* Навигатор пространства листов открыт, и одно пространство листов открыто.
* В пространстве листа находятся трехмерные размещения изделия.

1. Выберите пункты меню Вид > Инструмент для монтажных работ.

!!! info "Для сведения:"

    Отображаются все существующие инструменты для монтажных работ, которые можно выбирать и обрабатывать.

2. Выберите заново пункты меню Вид > Инструмент для монтажных работ.

!!! info "Для сведения:"

    Все отображаемые инструменты для монтажных работ скрываются.

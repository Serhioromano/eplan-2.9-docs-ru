# Изменить пароль диалогового окна

* Сервисные программы > Управление правами > Пользователь. В диалоговом окне Управление правами: вы выделили пользователя в левом поле папки Пользователь и имя пользователя в правом списке. Щелкните по ++Изменить пароль++.
* Вы работаете в режиме "Регистрация". Вы запускаете EPLAN. Щелкните в диалоговом окне Регистрация в EPLAN по ++Изменить пароль++.

В этом диалоговом окне вы можете изменить пароль регистрации в системе EPLAN. При вызове диалогового окна из "Управление правами" у администраторов есть возможность присвоить пользователю новый пароль, например, если он забыл свой пароль.

Доступны следующие элементы диалогового окна:

### Имя

В этом поле выводится имя пользователя.

### Старый пароль

Введите здесь прежний пароль. Для защиты введенных данных пароль отображается не открытым текстом, а в виде символов *.

При вызове диалогового окна из "Управление правами" это поле недоступно и не может обрабатываться.

### Новый пароль

Введите здесь новый пароль. Для защиты введенных данных пароль отображается не открытым текстом, а в виде символов *.

### Подтвердить новый пароль

Во избежание опечаток при вводе новый пароль необходимо повторно ввести в это поле. Только после этого EPLAN сохранит пароль.

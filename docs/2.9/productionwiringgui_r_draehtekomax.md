# Вкладка Провода (Komax)

* Параметры > Настройки > Фирма > Сборка проводов > Komax > Провода
* Сервисные программы > Данные изготовления > Сборка проводов > Komax. Щелкните по кнопке ++...++ рядом с полем Схема. Выберите вкладку Провода.

В этой вкладке определяются основные данные по проводам для обработки проводов на машине 'Komax'.

Обзор основных элементов диалогового окна:

### Панель инструментов

Кнопка |  Значение
---|---
![](images/all_update_as.png){: .ui-icon } (Обновить) |  Заново считывает провода и изделия проводов из проекта.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет выделенные строки.

### Цвет

Идентификатор для цвета провода, например SW = черный.

### Поперечное сечение

Идентификатор для поперечного сечения провода, например 2.5 = 2,5 мм.

### Номер изделия

Номер изделия для изделия проводов.

### Komax WireKey

Код провода; идентификатор поперечного сечения и цвета провода, например 2.5 SW = 2,5 мм черный.

### Komax FontKey

Идентификатор для передачи в определение вида шрифта в машине. Стандарт — это значение кода провода (Komax WireKey). Вид шрифта, размер шрифта и цвет шрифта, которые должны будут фактически использоваться согласно данному идентификатору при нанесении маркировки на провод, определяются при помощи той же маркировки в машине 'Komax' в качестве набора параметров.

### Интервал стороны 1 / стороны 2

Интервал, через который наносится маркировка от обоих концов провода.

### Интервал непрерывного текста

Если маркировка на провод наносится как непрерывный текст, здесь задается интервал между отдельными блоками текста.

### Длина зачистки

Длина изоляции (в миллиметрах), которую машина должна обрезать.

![](images/productionwiringgui_cutofflength_as.png)

### Длина сдвига

Длина (в миллиметрах), на которую машина должна стянуть обрезанную изоляцию с провода.

Если полностью снять изоляцию конца провода, существует риск, особенно при использовании многожильных проводов, что открытые концы расплетутся во время дальнейшей обработки в машине, и что провода больше не смогут проходить через машину.

![](images/productionwiringgui_splicedends_as.png)

Чтобы предотвратить это, обрезанный конец лишь слегка оттягивается от проволоки. Отрезанная часть остается на проводе для защиты конца провода и будет убрана при разводке. В зависимости от машины это значение действительно для обоих концов провода.

![](images/productionwiringgui_stripofflength_as.png)

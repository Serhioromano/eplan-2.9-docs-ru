# Работать с рабочими областями

Рабочая область EPLAN охватывает размер и позицию главного окна EPLAN, а также позиции, размеры и настройки "присоединяемых" диалоговых окон, строк меню и панелей инструментов. Кроме того, в рабочей области можно сохранить панели инструментов, определенные пользователем.

Это может быть полезным в том случае, если требуется настроить EPLAN для выполнения разных задач по проектированию. Так, например, "нажатием кнопки" можно переключиться в обработку ПЛК и автоматически запустить все требуемые для этого диалоговые окна, причем позиции и настройки уже являются фиксированными. Все непредусмотренные в данном случае диалоговые окна автоматически закрываются.

Рабочие области можно в любой момент сохранить в виде ***схем***, вновь выбрать их позже или даже удалить. Помимо этого, в диалоговом окне Рабочая область EPLAN имеется несколько предварительно определенных рабочих областей, которые, например, можно использовать в качестве шаблонов для других рабочих областей. Так же, как и для схем, здесь с помощью [панели инструментов](schemeconfig_m_schemaschaltflaechen.md) можно создавать, копировать, удалять, импортировать и экспортировать схемы для рабочей области.

!!! note "Замечание:"

    Размер, положение или другие настройки ***редакторов*** (графический редактор, редактор символов, редактор форм и т.д.) невозможно сохранить в одной рабочей области.

Условие:

Интерфейс пользователя EPLAN настроен для определенной задачи проектирования.

1. Вид > Рабочая область
2. В диалоговом окне Рабочая область щелкните ![](images/all_new_as.png){: .ui-icon } (Создать) на панели инструментов рядом с раскрывающимся списком Схема.
3. Введите в диалоговом окне Новая схема в поле Имя имя для новой схемы, то есть перезапишите имя текущей выделенной.
4. В поле Описание введите краткое информативное описание для новой схемы.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Диалоговое окно Новая схема закрывается, а новая схема становится в диалоговом окне текущей.

6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Новая рабочая область создается под введенным именем. Текущие настройки интерфейса пользователя, относящиеся к открытым "присоединяемым" диалоговым окнам, положению диалоговых окон, панелям инструментов и т.д., сохраняются.

Конфигурацию интерфейса пользователя этой рабочей области при необходимости можно изменить (например, изменить размер и положение "присоединяемых" диалоговых окон, импортировать определенные пользователем панели инструментов и т.д.) и сохранить заново (см. "Обработка рабочих областей").

1. Вид > Рабочая область
2. В диалоговом окне Рабочая область из раскрывающегося списка Схема выберите рабочую область, которая будет использоваться как текущая.

!!! info "Для сведения:"

    Информация в поле Описание обновляется.

3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Настройки выбранной рабочей области, относящиеся к открытым "присоединяемым" диалоговым окнам, положению диалоговых окон, панелям инструментов и т.д. применяются, а интерфейс пользователя программы соответствующим образом обновляется.

Конфигурацию интерфейса пользователя рабочей области при необходимости сначала можно настроить, а затем в рамках следующего шага заново сохранить путем обработки рабочей области. Кроме того, при обработке рабочей области существует возможность перезаписи выбранной рабочей области с помощью содержимого другой рабочей области. Это имеет значение, например, в том случае, если уже существующая схема должна послужить образцом для новой рабочей области.

1. Вид > Рабочая область
2. В диалоговом окне Рабочая область щелкните ![](images/all_edit_as.png){: .ui-icon } (Обработать).

!!! info "Для сведения:"

    Открывается диалоговое окно Обработать рабочую область.

3. В нем выберите запись Текущая конфигурация, если выбранную рабочую область необходимо сохранить заново. Если в качестве шаблона для текущей рабочей области должна использоваться другая схема, выберите соответствующее имя. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Диалоговое окно Обработать рабочую область закрывается.

4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Измененные настройки сохраняются в текущей рабочей области.

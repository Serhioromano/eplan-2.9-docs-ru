# Чертить линии определения кабелей или экранирования

Кабели можно определить по линиям определения кабелей или экранированиям. Для этого линии определения кабелей или символ экранирования должны пересекать соединение, которое должно быть принято в кабель или экранирование.

### Чертить линий определения кабеля

1. Выберите пункты меню Вставить > Определение кабеля.
2. Протяните линию через соединения, которые должны войти в кабель.

!!! info "Для сведения:"

    На каждой точке пересечения соединений с линией определения кабеля автоматически размещается точка определения соединения.

3. В диалоговом окне Свойства ++...++ можно изменить ОУ и установить другие настройки для кабеля.
4. Щелкните по кнопке ++OK++.

!!! note "Замечание:"

    * При повторном уменьшении линии определения кабеля точки определения соединений на уже более не пересекаемых соединениях будут убраны. При увеличении линии определения кабеля для новых пересекаемых соединений будут размещены новые точки определения соединений.
    * Однополюсное представление кабелей:
    Если в однополюсном представлении протянуть линию определения кабеля через соединение типа 'Кабели', там будет размещена точка определения соединения, и на ней автоматически будет записан вид представления 'Однополюс.'. При смешанном отображении (однополюсные символы на многополюсных страницах) Вам необходимо ввести вид представления самостоятельно на точке обозначения соединения.

### Чертить экранирования

1. Выберите пункты меню Вставить > Экранирование.
2. Протяните экранирование (как прямоугольник) через соединения, которые нужно экранировать.

!!! info "Для сведения:"

    На всех соединениях, находящихся внутри экранирования, будет автоматически размещена точка определения соединения.

!!! info "Для сведения:"

    Если в пользовательских настройках (Параметры > Настройки > Пользователь > Графическая обработка > Диалоговые окна свойств) установлен флажок При вставке экранирования, на экран будет выводиться диалоговое окно Свойства ++...++.

3. В диалоговом окне Свойства ++...++ можно изменить ОУ и установить другие настройки для экранирования.
4. Если вывод свойств отображения нужно подавить, на вкладке Отображение в поле Порядок свойств выберите запись "Не отображать".
5. Щелкните по кнопке ++OK++.

!!! note "Замечание:"

    При повторном уменьшении экранирования точки определения соединений на расположенных теперь уже снаружи соединениях будут убраны. При увеличении экранирования для новых включенных соединений будут размещены новые точки определения соединений.

# Копировать, вырезать и вставить элементы схемы соединений

Условия:

* Вы открыли проект.
* Проект содержит страницы, на которых размещены функции.
* Вы открыли навигатор страниц: Страница > Навигатор

### Копирование или вырезание

1. Дважды щелкните страницу схемы соединений в диалоговом окне Страницы - <Имя проекта>.
2. Выберите пункты меню Обработать > Выделить > Область.
3. Удерживая нажатой левую клавишу мыши, начертите прямоугольник вокруг всех элементов схемы соединений, которые требуется скопировать или вырезать.
4. Выберите пункты меню Обработать > Копировать или Обработать > Вырезать.

### Вставка в страницу

1. Дважды щелкните в диалоговом окне Страницы - <Имя проекта> по странице схемы соединений, в которую следует вставить элементы схемы соединений.
2. Выберите пункты меню Обработать > Вставить.
3. Щелкните в странице схемы соединений по той позиции, куда требуется вставить элементы из буфера обмена EPLAN.

!!! info "Для сведения:"

    Если в диалоговом окне Настройки: Нумерация (online) выбрана опция C запросом, открывается диалоговое окно Режим вставки.

4. Укажите, каким образом требуется пронумеровать вставленные элементы схемы соединений, и щелкните по кнопке ++OK++.

### Вставка в несколько страниц

1. Если элементы схемы соединений также требуется вставить в других позициях, снова щелкните по схеме соединений.

!!! info "Для сведения:"

    Выбранный режим вставки не изменяется.

2. Для изменения выбранного режима вставки нажмите клавишу ++Esc++.

!!! info "Для сведения:"

    Операция прервана.

3. Выберите пункты меню Обработать > Вставить и таким же образом вставьте элементы схемы соединений.

!!! info "Для сведения:"

    При использовании этого метода в диалоговом окне Режим вставки можно указать, каким образом следует пронумеровать эти элементы схемы соединений.

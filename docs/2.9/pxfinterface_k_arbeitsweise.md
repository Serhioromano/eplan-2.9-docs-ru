# Внешняя обработка: Принцип работы

При внешней обработке всегда выводятся только данные определенного объекта (например, устройства, кабели, страницы и т. д.). Объект для экспорта задается в настройках при создании схемы с помощью так называемого [типа объекта](macrosgui_d_pxfeinstellungenexport.md).

!!! note "Замечание:"

    Обязательным условием для создания в EPLAN файлов в формате Microsoft Office является наличие установленных на компьютере следующих разрешенных EPLAN программных продуктов.

Данные, в основном, выводятся в неформатированном виде. Можно изменять только последовательность столбцов изменяемых данных.

Данные сортируются таким образом, что перекрестные ссылки выводятся связанными, а страницы находятся в одной и той же последовательности, как и в проекте. Точки определения соединения без ОУ вывести невозможно.

Импорт и экспорт можно также реализовать через так называемые [Операции EPLAN](availableactions_k_start.md) (действия), четко распознать операцию и таким образом задать автоматическое осуществление функциональностей программы.

Существуют данные, которые выводятся для информирования, но при импорте игнорируются, при этом речь идет о следующих косвенных данных:

У ***функций***:

* Полное ОУ
* Функции
* Позиция в схеме соединений (страница, зона, ячейка)
* Определение функции (символьная строка)
* Символ (символьная строка)
* Функциональный текст (автоматически)
* Символический адрес (автоматич.)
* Символический адрес (определен).

У ***страниц***:

* Полное имя страницы
* Тип страниц.

При внешней обработке использование внешнего приложения осуществляется через поле Вид вывода диалогового окна Внешняя обработка. При работе с опциями этого группового поля следует учитывать следующее:

* Настройку Обработать во внешнем приложении и считать обратно можно выбрать только тогда, когда в схеме настроено целевое приложение или выбран Excel в качества типа файла.
* При настройке Обработать во внешнем приложении и считать обратно EPLAN остается открытым на заднем плане. Если внешнее приложение закрывается, после изменения данных, EPLAN откроет диалоговое окно, в котором можно решить, следует ли импортировать измененные данные обратно.
* При настройке Обработать во внешнем приложении и считать обратно поле Целевой файл становится недоступным и пустым. Если снова выбирается другая настройка, она опять становится активной.

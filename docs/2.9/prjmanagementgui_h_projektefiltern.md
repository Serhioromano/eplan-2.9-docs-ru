# Искать / фильтровать проект

Вы можете искать проект при помощи фильтра. В определении фильтра вы можете задать свойства в качестве критериев фильтрования. Вы также можете задать каталог поиска, в котором EPLAN должен искать проекты. Другие каталоги при поиске не учитываются.

Условие:

Вы сосчитали каталоги и проекты в управление проектами.

1. Выберите следующие пункты меню: Проект > Управление
2. В диалоговом окне Управление проектами щелкните рядом с полем Фильтр кнопку ++...++.
3. В диалоговом окне [Фильтр](modaldialogsdb_d_filternnach.md) в поле Схема выберите схему или создайте новую схему при помощи кнопки ![](images/all_new_as.png){: .ui-icon } (Создать).
4. Щелкните ![](images/all_new_as.png){: .ui-icon } (Создать) в диалоговом окне Фильтр справа над таблицей.
5. Для установки ***каталога поиска*** в диалоговом окне Выбор критериев выделите запись "Каталог" и нажмите кнопку ++OK++.
6. В столбце Значение нажмите ++...++.
7. Выберите в диалоговом окне Выбрать каталог каталог и щелкните по кнопке ++OK++.
8. При необходимости задайте дополнительные свойства в качестве критериев.
9. Щелкните по кнопке ++OK++.
10. В диалоговом окне Управление проектами в поле Фильтр выберите необходимый фильтр.

!!! info "Для сведения:"

    EPLAN отображает в структуре дерева все проекты, соответствующие критериям фильтрации.

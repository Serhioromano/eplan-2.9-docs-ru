# Основные данные: Основные положения

В проекте нет прямой ссылки на основные данные системы, а основные данные сохранены в меню ***Проект***. При этом в проекте создается копия (например в библиотеке символов), которая используется во время работы. Во время проектирования всегда обрабатывается специфическая для проекта копия. На нее уже, впрочем, сослались; так что, например, символ, используемый в проекте многократно, появляется только один раз в (специфической для проекта) библиотеке символов.

Основные данные проекта можно синхронизировать при открытии проекта, а также при резервировании / восстановлении с помощью пула основных данных (т.е. основных данных системы). Для этого существуют следующие настройки проекта:

* Синхронизировать основные данные проекта при открытии: меню Параметры > Настройки > Проекты > "Имя проекта" > Управление > Общее: установите этот флажок, если вы хотите синхронизировать сохраненные в текущем проекте основные данные при каждом открытии проекта с пулом основных данных.
* Синхронизировать рамки: меню Параметры > Настройки > Проекты > "Имя проекта" > Управление > Общее: установите этот флажок, если вы хотите синхронизировать сохраненные в текущем проекте рамки с пулом основных данных при каждом открытии проекта.
* Стандартная рамка: меню Параметры > Настройки > Проекты > "Имя проекта" > Управление > Страницы: из раскрывающегося списка или диалогового окна Выбрать рамку выберите рамку, используемую по умолчанию.

Сами основные данные могут, в основном, обрабатываться ***только*** в пуле основных данных (как напр., в редакторе символов).

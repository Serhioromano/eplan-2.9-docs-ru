# Открыть перекрестные ссылки в новом окне

В графическом редакторе можно переходить от отображенных перекрестных ссылок к обратным эквивалентам с перекрестными ссылками (точки разрыва, главные / вспомогательные функции и т. д.). С помощью этого действия теперь можно также открыть новое окно графического редактора при условии, что соответствующий объект с перекрестными ссылками находится на другой странице проекта. Щелчок мыши по отображенной перекрестной ссылке и одновременное нажатие комбинации клавиш [Ctrl] \+ [Shift] открывает новое окно.

![](images/benefit.png)Эффект:

При открытии перекрестной ссылки в новом окне страница проекта с первой отображенной перекрестной ссылкой остается открытой. Неудобная обратная навигация на исходную страницу проекта исключается.

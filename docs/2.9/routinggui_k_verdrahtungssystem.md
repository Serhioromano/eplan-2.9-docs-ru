# Маршрутизируемые соединения в системе направляющих для проводов

Соединения можно прокладывать не только в кабельных каналах или свободных сегментах маршрутизации, но и в системе направляющих для проводов. Для этого рекомендуется следующий принцип работы:

### Подготовительные действия в редакторе контура для выдавливаний

Одним из элементов системы направляющих для проводов является гребенка для проводов, используемая для фиксации проводов без монтажной платы и кабельных каналов.

* Гребенки для проводов реализованы в EPLAN как пользовательская шина на основе контура выдавливания.
* При создании контура выдавливания для гребенки для проводов в редакторе контура можно определить монтажные поверхности "Гребенка для проводов, передняя сторона" и "Гребенка для проводов, задняя сторона". Для этого в меню редактора контура Вставить > Логика предусмотрены одноименные пункты.

Монтажные поверхности "Гребенка для проводов, передняя сторона" или "Гребенка для проводов, задняя сторона" используются для размещения других компонентов системы направляющих для проводов.

На монтажной поверхности с обозначением функционального элемента "Гребенка для проводов, передняя сторона" одновременно с гребенкой для проводов разместите автоматически сгенерированное ***соединительное отверстие для проводов***. Соединительное отверстие для проводов иерархически расположено в навигаторе пространства листа под монтажной поверхностью "Гребенка для проводов, передняя сторона" и получает свое ОУ.

### Работа в чертеже монтажных поверхностей

* Разместите на горизонтальном профиле устройства, например автоматы защиты двигателя, контакторы и клеммы.
* Затем разместите определенную пользователем шину "Гребенка для проводов" на верхней и нижней части горизонтального профиля вместе с автоматически сгенерированным соединительным отверстием для проводов.
* Потом разместите область маршрутизации на верхней и нижней гребенках для проводов с обратной стороны, чтобы соединения можно было маршрутизировать с передней стороны через соединительные отверстия для проводов на обратную сторону.
* Чтобы маршрутизировать соединения из системы направляющих для проводов (например, на дверь), разместите сегмент маршрутизации или кривую. Этот сегмент маршрутизации или кривая соприкасается с другим сегментом маршрутизации или кривой на двери и, с другой стороны, с областью маршрутизации в системе направляющих для проводов.

# Диалоговое окно Предварительный выбор изделий

Вы открыли проект.

* Параметры > Настройки > Проекты > "Имя проекта" > Управление > Выбор изделия. Рядом с полем Предварительный выбор изделий нажмите ++...++.
* Вы выделили главную функцию в графическом редакторе или навигаторе. Обработать > Свойства. Выберите вкладку Изделие. [Настройки] > Выбор изделия (проект). В диалоговом окне Настройки: Выбор изделия (проект) рядом с полем Предварительный выбор изделий нажмите ++...++.

В этом диалоговом окне можно обрабатывать списки для предварительного выбора изделий или создавать новые.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую вы хотите обработать. Вы можете обрабатывать только пользовательские схемы.

Через расположенную рядом [панель инструментов](schemeconfig_m_schemaschaltflaechen.md) можно создавать новые схемы, сохранять изменения в схеме, копировать или удалить схему. Кроме того, схемы можно импортировать и экспортировать.

### Описание

Данное поле отображает текст описания, который вы ввели при создании схемы в диалоговом окне Новая схема в поле Описание.

### Панель инструментов

На панели инструментов доступны такие кнопки:

Кнопка |  Значение
---|---
![](images/all_new_as.png){: .ui-icon } (Создать) |  С помощью этой кнопки можно открыть диалоговое окно Выбор изделия и выбрать там другие изделия, которые переносятся в список предварительного выбора изделия.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  С помощью этой кнопки можно удалить выделенные изделия из текущего списка.

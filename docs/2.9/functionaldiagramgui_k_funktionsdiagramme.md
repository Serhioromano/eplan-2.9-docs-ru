# Порядок создания блок-схем рабочего процесса

Блок-схему рабочего процесса можно создать на уже существующей или новой странице. При создании новой страницы для блок-схемы рабочего процесса рекомендуется использовать тип страницы 'Обзор'. Для ***Функциональной диаграммы ***укажите в свойствах новой страницы используемую форму в свойстве Имя формы. Для ***Диаграммы GRAFCET*** не обязательно указывать форму.

Вместе с функциональной диаграммой выдаются следующие две формы типа 'Графика (*.f28)'. Формы основываются на сетке 3 мм.

* F28_001.f28: для функциональной диаграммы с ориентацией на текст.
* F28_002.f28: для функциональной диаграммы с ориентацией на символ.

Центральным диалоговым окном для вставки функций EPLAN в функциональной диаграмме или диаграмме GRAFCET является навигатор устройств. При этом не имеет значения, размещена функция в схеме соединений или сгенерирована с помощью всплывающего меню Новое устройство как новая, еще не размещенная функция.

Чтобы разместить функцию на открытой странице, выделите функцию в навигаторе устройств, а затем выберите пункт всплывающего меню Функциональное размещение > С графикой символов. После этого тексты свойств (видимое ОУ, функциональный текст и т. д.) и, в зависимости от настройки проекта Функциональное размещение с графикой символов, графика символа функции повисают на курсоре и их можно разместить в диаграмме.

При размещении различные тексты свойств и графика символа функции (если размещена) объединяются в одну группу. Поэтому данные свойства можно обрабатывать, перемещать или удалять как группу.

По умолчанию для функции выводятся значения свойств Видимое ОУ, Функциональный текст и Состояние. Отображаемые в функциональной диаграмме свойства функции зависят от настроенного специального символа.

Для функциональной диаграммы имеет большое значение возможность задания статуса установки или устройства. Для этого используется индексированное свойство Статус [1–10] (ID 20285), которое можно выбрать в диалоговом окне 'Свойства' устройства через диалоговое окно Выбор свойств.

При создании функциональных диаграмм с ориентацией на символ можно также размещать графику макроса изделия в функциональной диаграмме. Для этого в навигаторе устройств выделите главную функцию, у которой над изделием имеется макрос, и во всплывающем меню выберите пункты Функциональное размещение > С графикой макросов.

При выполнении данного действия функции, лежащие в основе макроса, внутренне преобразуются в графику. При этом используются первый доступный вид представления и первый доступный вариант макроса. Сгруппированная графика макроса и тексты свойств главной функции повисают на курсоре, и их можно разместить в диаграмме.

!!! note "Замечание:"

    Настройка проекта Функциональное размещение с графикой символов никак не влияет на функциональное размещение макросов. Графика макроса вставляется всякий раз заново.

* При размещении функции используется символ уже выделенной функции. Если в дереве навигатора выбран уровень иерархии устройства, то используется символ первой главной функции. При размещении макросов независимо от выделенной функции всегда используется макрос изделия главной функции.
* Если для выбранной функции отсутствует символ или макрос изделия, у вас есть возможность выбрать символ / макрос в диалоговом окне выбора.
* Если графика символа / макроса больше, чем указано в настройке проекта Целевой размер для графики символа / макроса, то графика масштабируется (уменьшается). Увеличения не происходит.
* С помощью настройки проекта Позиция для графики символа / макроса можно задать перемещение для позиции размещаемой графики символа / макроса. Исходной точкой для перемещения служит точка вставки специального символа для размещения свойства.

После того как в левой описывающей области функциональной диаграммы размещены функции EPLAN, в диаграмму необходимо вставить шкалу времени / шагов, а также порядок рабочего процесса.

В уже имеющихся формах шкала времени / шагов уже существует. Значения для шкалы можно вставить с помощью макроса. Для этого в основных данных доступен макрос Numbers.ema с примерными значениями.

Порядки рабочих процессов необходимо обозначить с помощью графических элементов (например, ломаной линии). EPLAN поддерживает ускоренную вставку типичных элементов функциональной диаграммы. Стрелки, сигнальные точки, условия И и ИЛИ можно обозначить с помощью соответствующих пунктов в меню Вставить > Блок-схемы рабочего процесса в соответствующей функциональной диаграмме.

Для диаграмм GRAFCET в меню Вставить > Блок-схема рабочего процесса > GRAFCET имеются пункты Шаг, Шаг с прохождением, Действие, Действие с привязкой и Прохождение. Если при вставке элементов GRAFCET нажать клавишу ++Табуляция++, можно "пролистать" различные готовые варианты. Таким образом, посредством выбора соответствующего прямоугольника можно вставить, например, начальный шаг, завершающий шаг, шаг макроса и т. д.

Звездочка (*) во вставленных элементах GRAFCET показывает, что здесь можно вводить тексты, например имена шагов, имена или условия прохождений или идентификатор действия. Различные объекты элементов GRAFCET объединяются в группу. Для непосредственного ввода отдельного текста без применения группировки удерживайте клавишу ++Shift++ и щелчком по соответствующей звездочке откройте диалоговое окно Свойства — текст.

Остальные данные (например, комментарии, которые должны находиться в кавычках) можно вставить в диаграмму GRAFCET как графический текст.

При создании текстов в диаграммах GRAFCET можно вставлять в том числе и следующие специальные символы:

* Стрелка вверх (↑).
* Стрелка вниз (↓)
* Неравенство (≠).

!!! note "Замечание:"

    Обратите внимание, что некоторые специальные символы доступны не во всех видах шрифта. Чтобы иметь возможность использовать эти специальные символы, выберите в диалоговом окне Настройки: Виды шрифтов соответствующие виды шрифтов.

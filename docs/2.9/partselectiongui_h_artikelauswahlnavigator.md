# Выбрать изделие в Навигаторах или из списка устройств

В навигаторе устройств, навигаторе спецификаций и в списке устройств выбор изделия можно вызывать через всплывающее меню.

### Генерирование устройств в навигаторе устройств

В навигаторе устройств изделия можно генерировать через выбор устройства.

1. Данные проекта > Устройство > Навигатор
2. Выделите любой элемент и выберите пункт всплывающего меню Новое устройство.
3. В диалоговом окне Выбор изделия можно просмотреть структуру дерева и выбрать требуемое изделие(я).
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Генерируется новое устройство. Устройство сортируется в структуре проекта в зависимости от выделенного элемента.

!!! info "Для сведения:"

    Данные изделия сохраняются в устройстве. Таким образом новое изделие генерируется через выбор устройств.

!!! example "Пример:"

    Допустим, выделено какое-то устройство под местом установкиPT1. Теперь открывайте через функцию Новое устройствовыбор изделий и выберите контактор. ПодPT1генерируется устройствоK1со всеми функциями контактора.

### Выбор изделий из навигатора спецификаций

1. Данные проекта > Устройства / Изделия > Навигатор спецификаций
2. Выделите запись и выберите пункт всплывающего меню Добавить изделие или Заменить изделие.
3. В диалоговом окне Выбор изделия можно просмотреть структуру дерева и выбрать требуемое изделие(я).
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выделенному устройству присваивается новое изделие. При нажатии Заменить изделие заменяется прежнее изделие, при нажатии обавить изделие сохраняются имеющиеся ссылки изделия.

### Выбор изделия из списка устройств.

1. Данные проекта > Устройство / изделие > Список устройств
2. Выберите пункт всплывающего меню Добавить изделие.
3. В диалоговом окне Выбор изделия можно просмотреть структуру дерева и выбрать требуемое изделие(я).
4. Щелкните по кнопке ++OK++.
5. В диалоговом окне Свойства сохраните специфические для изделия данные.

!!! info "Для сведения:"

    Данные изделия переносятся в Список устройств. Генерируется новая запись для контролируемого изделия.

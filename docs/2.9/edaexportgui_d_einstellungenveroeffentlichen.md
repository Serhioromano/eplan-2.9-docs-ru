# Диалоговое окно Настройки: Опубликовать

* Проект > Опубликовать. Щелкните в поле Схема диалогового окна Опубликовать на ++...++.
* Параметры > Настройки > Пользователь > Интерфейсы > Опубликовать.

В этом диалоговом окне устанавливаются настройки для экспорта проекта в формат EPDZ. Формат EPDZ можно прочитать при помощи приложения ***EPLAN View*** через Dropbox на мобильных устройствах (например, iPad).

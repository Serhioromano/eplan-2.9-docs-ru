# Вкладка Условия

Открыты проект и динамическая форма (например, через меню Сервисные программы > Основные данные > Форма > Открыть > [Открыть]). Вставить > Динамическая область > Область условий. Область условий вытянута, как прямоугольник. Дважды щелкните по прямоугольнику и выберите в диалоговом окне свойств вкладку Условия.

Через данную вкладку укажите, при каких условиях в отчетах выводятся определенные области. При этом в подчиненных областях могут использоваться графические элементы и заполнители.

Обзор основных элементов диалогового окна:

### При изменении свойств

Щелкните в поле кнопку ++...++, чтобы перейти в диалоговое окно Изменение свойства и там выбрать свойство, при изменении которого будет анализироваться область. Для списка обозначений устройств можно, например, задать, что при каждом изменении свойства ОУ (видимое) будет начерчена толстая линия для следующего ОУ.

В динамических формах для ***таблицы соединений*** при выборе свойств в диалоговом окне Изменение свойства на выбор предлагаются также свойства блока. Это позволяет, например, выводить пустую строку в таблице соединений, если изменяется значение определенного свойства блока.

### При смене объекта

Установите флажок, чтобы задать анализ подчиненной области при каждой смене объекта.

!!! example "Пример:"

    Вспецификацииклеммника цели приведены на клемму. Перед каждым анализируемым объектом (в данном примере перед каждой клеммой) должна быть начерчена серая полоса. Начертите полосу в подчиненной области и установите данный флажок.

### В начале первого столбца/В начале каждого столбца/В конце каждого столбца/В конце последнего столбца

Установите флажок, чтобы задать вставку определенных графических элементов в подчиненной области только в начале первого столбца или в начале каждого столбца/в конце каждого столбца, или только в конце последнего столбца. В объединенных отчетах начало первого столбца и конец последнего столбца всегда относятся к анализируемому объекту, например к клеммнику.

### После каждой строки x

Введите в данное поле значение строки, после которой в условной области должны отображаться определенные графические элементы.

### Адаптировать высоту

Установите флажок, если подчиненная область должна масштабироваться на основе высоты предыдущей строки данных.

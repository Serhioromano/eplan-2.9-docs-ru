# EPLAN Fluid: Маршрутизация соединений Fluid-техники

Для соединений Fluid-техники маршрутизация в сети соединенных сегментов неприменима. Поэтому EPLAN поддерживает возможность свободной маршрутизации шланговых соединений (также вне сети соединенных сегментов) в пространстве листа. Маршрутизация шлангов в трехмерном чертеже монтажных поверхностей позволяет на очень ранней стадии создавать виртуальные прототипы, например, для пневматических комплексных решений, которые включают систему шлангов.

Свободно маршрутизируемое соединение генерируется вдоль определенной пространственной кривой между источником и целью. Сеть соединенных сегментов не нужна, ее использование не предусмотрено. Направление кривой свободно маршрутизируемого соединения задается опорными точками, впоследствии его можно изменить и адаптировать к пространственным условиям в трехмерном чертеже монтажных поверхностей.

    На следующем рисунке показано несколько шланговых соединений, маршрутизированных в трехмерном чертеже монтажных поверхностей.

Маршрутизация гидротехнических соединений выполняется при помощи следующей последовательности действий.

### Подготовка основных данных

Успешная разработка виртуального прототипа для Fluid-техники в трехмерном чертеже монтажных поверхностей и последующая маршрутизация шлангов требует, чтобы предварительно вы соответствующим образом подготовили основные данные в базе данных изделий.

* ***Записать графический 3D-макрос:***
Для того чтобы сохранить реалистичное и подробное отображение прототипа на трехмерном чертеже монтажных поверхностей, необходимо создать для функциональных элементов Fluid-техники (клапанов, насосов и т.п.) соответствующие графические 3D-макросы и сохранить их на изделиях. Макросы с трехмерными данными и типом отображения "Трехмерный чертеж монтажных поверхностей" заносятся в базе данных изделий на вкладку Монтажные данные в поле Графический макрос.
* ***Определить схемы соединений:***
Изделия для размещения изделия в пространстве листа требуют определенных схем соединений. Из схемы соединений, сохраненной на изделии, размещение изделия чаще всего получает информацию о выводах устройства, имеющихся в размещении изделия. Так, например, в схеме соединений устанавливается направление вывода устройства. Направление вывода устройства задает направление, в котором маршрутизируется соединение на выводе устройства.
Для определения схемы соединений с соответствующими данными о выводах устройств в базе данных изделий предусмотрен собственный главный узел. На изделии на вкладке Технические данные при помощи поля Схема соединений можно выбрать необходимую схему соединений и, таким образом, сохранить на изделии.
* ***Создать изделия для резьбовых шланговых соединений:***
Для усовершенствования отображения на трехмерном чертеже монтажной поверхности и для определения правильной длины шланга в базе данных изделий дополнительно необходимы изделия для резьбовых шланговых соединений. Для этих гидротехнических изделий, которые в группе продуктов "Принадлежности" были отнесены к группе продуктов "Резьбовые соединения", также необходимо сохранить соответствующие трехмерные графические макросы и схемы соединений. При создании проекта макросов этим трехмерным графическим макросам необходимо присвоить определение функции 'Размещение изделия, распределитель Fluid-техники'.
* ***Создать изделия для принадлежностей для маршрутизации:***
Аналогично другим трехмерным компонентам, которые вы хотите разместить в пространстве листа, для размещения принадлежностей для маршрутизации необходимо сохранить соответствующие изделия в базе данных изделий.
* ***Внести данные соединений для шлангов:***
Для правильного изображения проложенных шлангов в пространстве листа нужно определить необходимые данные соединений (цвет, внешний диаметр). Эти настройки можно взять из диалогового окна Свойства соединения или в записи об изделии в базе данных изделий.

!!! example "Пример:"

    Вы открыли диалоговое окно "Свойства" для соединения и перешли на вкладку Соединение. В поле Цвет/номер соединенияукажите цвет шланга (например: BU = синий). Размеры шланга укажите в поле Поперечное сечение / диаметр(например:Ø10илиØ10x1,5для шланга с внешним диаметром 10 мм и толщиной стенки 1,5 мм).

### Свободная маршрутизация соединений

В Fluid-технике шланги, как правило, присоединяются к гидротехническим функциональным элементам (клапанам, насосам и т. п.) с помощью резьбовых соединений. На схемах соединений Fluid-техники эти принадлежности часто не обозначаются. Чтобы создать детализированное представление на трехмерном чертеже монтажных поверхностей и правильно определить длину шлангов, необходимо вставить соответствующие размещения изделий для шланговых резьбовых соединений. При дальнейшей маршрутизации шлангов в пространстве листа эти принадлежности будут учитываться. Если на гидротехническом функциональном элементе размещено такое резьбовое соединение, то при маршрутизации шланга вместо вывода устройства функционального элемента используется следующий свободный вывод устройства шлангового резьбового соединения.

При свободной маршрутизации соединений с помощью операции Свободная маршрутизация (пространство листа) генерируются гидротехнические маршрутизируемые соединения в форме кривых за пределами сети соединенных сегментов. При этом источник и цель автоматически соединяются между собой по кратчайшему пути.

Операция Маршрутизировать (пространство листа) предназначена для электротехнических соединений, которые маршрутизируются в пределах сети соединенных сегментов.

### Адаптация профилей соединений

Чтобы адаптировать направление кривой свободно маршрутизируемого соединения к пространственным условиям в трехмерном чертеже монтажных поверхностей, присвойте профиль соединения через кривые и сегменты маршрутизации. С помощью действия Изменить маршрутизацию измените трассу маршрутизации автоматически маршрутизированных соединений на желаемую трассу маршрутизации.

В особых ситуациях встраивания устройств может потребоваться изменить направление подсоединения всех выводов устройств, указанных на схеме соединений, чтобы тем самым избавиться от неудобных трасс маршрутизации. Это можно сделать при помощи действия Изменить направление подсоединения.

### Вставка принадлежностей для маршрутизации

Свободно маршрутизированные соединения часто укрепляются или объединяются при помощи принадлежностей для маршрутизации (крепление кабеля / шланга, хомут для кабеля / шланга, защита кабеля / шланга). При этом можно объединить несколько шлангов, маршрутизированных по общей трассе маршрутизации, при помощи хомута для кабеля/шланга или защитить их при помощи защиты кабеля/шланга. Принадлежности для маршрутизации, например защита шланга, присваиваются сегментам маршрутизации / кривым, через которые маршрутизируется соединение. Пустому сегменту маршрутизации / кривой нельзя присвоить принадлежность для маршрутизации.

# Автоматизированная обработка проектов

Функциональность "Автоматизированная обработка" обеспечивает автоматизированное выполнение отдельных областей обработки проекта при помощи пользовательской схемы.

* При этом EPLAN поддерживает создание схем с предварительно заданными операциями. Знание программирования не требуется.
* EPLAN автоматически создает файл сценария из сохраненной в схеме последовательности операций. По умолчанию сценарии сохраняются в каталог <Дисковод программы>:\<Каталог программы>\Сценарии\<Идентификатор фирмы>.
* Возможна автоматизированная обработка одного или нескольких проектов.

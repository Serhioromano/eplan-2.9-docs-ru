# Отобразить гиперссылки на Web-узел Festo

Гиперссылки в данных изделий позволяют обращаться к Web-узлу Festo, на котором содержатся текущая документация и технические данные изделия.

Условия:

* Вы открыли проект.
* Установлен каталог продуктов Festo.
* В файле конфигурации festo.cfg для импорта определены необходимые настройки.
* Подходящий макрос окна, или макрос символа, или подходящий символ размещены в графическом редакторе, и ему присвоено изделие Festo.

1. Выделите условное обозначение в графическом редакторе или функцию — в одном из навигаторов и выберите пункт меню Обработать > Свойства.
2. Выберите в диалоговом окне Свойства ++...++ вкладку Отображение.
3. Щелкните по ![](images/all_new_as.png){: .ui-icon } (Создать) в панели инструментов сверху списка свойств.
4. В диалоговом окне Выбор свойств выберите свойство Изделие: Внешний документ 1 [n]. При помощи индекса [n] укажите, свойство какого из присвоенных изделий необходимо отобразить. (Возможен многократный выбор.)
5. Щелкните по кнопке ++OK++.
6. Щелкните кнопку ![](images/all_save_as.png){: .ui-icon } (Сохранить) на панели инструментов над раскрывающимся списком Порядок свойств и в диалоговом окне Сохранить порядок свойств введите описательное имя для порядка свойств.
7. Если только что созданный порядок свойств при всех дальнейших случаях применения варианта символа должен использоваться по умолчанию, установите флажок Использовать как стандарт.
8. Подтвердите ввод.

!!! info "Для сведения:"

    Отображение свойства будет актуализировано, и в графическом редакторе будут отображены гиперссылки. При этом отображается текст на текущем языке диалогового окна интерфейса и номер детали. (Если текст на данном языке недоступен, текстовое поле остается пустым.)

9. Нажмите клавишу ++Ctrl++ и щелкните по гиперссылке.

!!! info "Для сведения:"

    Откроется веб-узел Festo для соответствующего изделия. При этом Web-узел Festo будет отображен на языке, выбранном во время импорта в качестве языка диалогового окна. Если этот язык недоступен, будет использован альтернативный язык отображения, заданный в файле конфигурации.

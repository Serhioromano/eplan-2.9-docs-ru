# Использование графического выбора макросов

Условия:

* Вы открыли проект.
* Вы установили флажок Использовать графический выбор макроса (Параметры > Настройки > Пользователь > Графическая обработка > Макросы).

1. Выберите один из пунктов меню Вставить > Макрос окна / символа или Страница > Макрос страницы > Вставить.

!!! info "Для сведения:"

    Откроется диалоговое окно Графический выбор макроса.

2. Выберите путь файла, для которого необходимо отобразить каталоги макросов.
3. Выберите пункт всплывающего меню Рисунки, чтобы отобразить рисунки каталогов в другом размере.

!!! info "Для сведения:"

    Выполняется согласование размера рисунков.

4. Выберите пункт всплывающего меню Фильтр: <Тип файла макроса>, чтобы выполнить фильтрацию по определенному типу макроса.

!!! info "Для сведения:"

    Выполняется корректировка отображения файлов в области Выбор файлов.

5. Щелкните мышью по значку "Плюс" перед именем макроса в области Выбор файлов, чтобы узнать, какой вид представления имеет макрос и существует ли несколько вариантов.
6. Щелкните мышью по нужному варианту макроса, а затем нажмите ++OK++, чтобы выбрать данный макрос. Или дважды щелкните мышью по имени макроса, чтобы выбрать для вставки первый имеющийся вариант макроса.

!!! info "Для сведения:"

    Диалоговое окно закрывается, и макрос вставляется.

!!! note "Замечание:"

    В отличие от обычно используемого выбора макросов в диалоговом окне Графический выбор макроса нельзя выполнять функции, касающиеся каталогов или файлов. Так, например, нельзя создавать, переименовывать или удалять каталоги или файлы.

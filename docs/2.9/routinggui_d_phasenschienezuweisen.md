# Диалоговое окно Присвоить фазную шину

Открыто одно пространство листа. Открыт навигатор соединений. Генерировать Данные проекта > Соединения > Соединения фазовых шин. Выберите пункт всплывающего меню Присвоить фазную шину.

В этом диалоговом окне можно присвоить изделию соединение фазных шин. Кроме того, можно ввести или выбрать ОУ для соединения фазных шин.

Доступны следующие элементы диалогового окна:

### ОУ

Задайте требуемое ОУ фазной шины. Кнопка ++...++ открывает диалоговое окно [Выбор ОУ](modaldialogsdb_d_bmkauswahl.md), в котором можно выбрать доступное в проекте ОУ.

### Изделие

Показывает изделие, присвоенное фазной шине. Кнопка ++...++ открывает выбор изделия, в котором можно выбрать требуемое изделие фазных шин. На присвоенном изделии можно записать 3D-макрос, который используется при размещении фазной шины.

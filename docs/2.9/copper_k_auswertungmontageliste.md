# Медные жгуты в таблицах для сборочного чертежа

Для трехмерных размещений изделий пространства листа в EPLAN Pro Panel можно выводить отчеты типа ***Таблица для сборочного чертежа***. Таблицы для сборочного чертежа используются для воспроизведения в отчете иерархической структуры чертежа монтажных поверхностей в пространстве листа. Такой отчет можно также произвести для медных жгутов и медных функц. элементов.

В данном относящемся к функции отчете анализируются медные жгуты, у которых в диалоговом окне "Cвойства" активировано свойство Создать таблицу для сборочного чертежа (ид. 36032). Данное свойство выбирается у всех трехмерных объектов (кроме монтажных поверхностей) через выбор свойств. Для медного жгута с активным свойством в выведенной таблице для сборочного чертежа перечисляются все содержащиеся в нем медные функц. элементы. При этом по умолчанию перечисленные объекты сортируются по позиции легенды.

### Формы таблиц для сборочного чертежа

При генерировании таблицы для сборочного чертежа используется форма, заданная в относящемся к проекту диалоговом окне Настройки: Вывод в страницах для типа формы "Таблица для сборочного чертежа (*.f32)". Для форм данного типа в диалоговом окне Тексты заполнителей — Таблица для сборочного чертежа можно выбрать тексты заполнители. В верхнем колонтитуле, как и в области данных, на выбор предлагаются идентичные тексты заполнители.

### Фильтрация в навигаторе пространства листа

Если необходимо узнать, какие устройства предусмотрены для генерирования таблицы для сборочного чертежа, создайте для навигатора пространства листа новый фильтр и для данного фильтра выберите свойство Создать таблицу для сборочного чертежа в качестве критерия фильтрации.

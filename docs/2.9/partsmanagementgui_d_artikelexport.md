# Диалоговое окно Экспортировать записи данных

Сервисные программы > Изделие > Управление. Через меню ++Дополнительно++ > Настройки в диалоговом окне База данных изделий была открыта база данных изделий.

* [Дополнительно] > Экспортировать.
* В дереве или списке выделены экспортируемые записи данных. Всплывающее меню > Экспортировать.

В этом диалоговом окне устанавливаются настройки экспорта.

Обзор основных элементов диалогового окна:

### Тип файла

Выберите требуемый тип файла из раскрывающегося списка. Возможные настройки — "CSV" (при этом адреса, выводы устройства, изделия, схемы сверления, списки принадлежностей и размещения принадлежностей могут экспортироваться раздельно), "Текст", "XML" и "Формат обмена данными EPLAN Data Portal (EDZ)".
Щелкните по кнопке ++...++, чтобы для ***Текстового формата*** и ***Формата CSV*** указать путь к соответствующему файлу конфигурации; аналогичным образом можно для ***Формата обмена данными EPLAN Data Portal (EDZ)*** задать тип экспорта данных: Установите флажок Полностью экспортировать справочные данные, если в случае с узлами, модулями и т. д. при экспорте также должны учитываться подузлы, подмодули и т. д.

!!! note "Замечание:"

    При экспорте данных в файл типа "Текст" следует обязательно помнить о том, что файл конфигурации textimportexport.cfg, расположенный в каталоге \Изделие\<Ид. клиента> текущей версии программы, должен быть скорректирован для учета определенных полей (например, таких, как у групп ***Схема соединений*** и ***Схема сверления***).
    Подробные сведения см. в теме [База данных изделий: Файлы конфигурации для импорта и экспорта данных](partsmanagementgui_k_steuerdateiimportexport.md).

### Общий файл

Выберите этот параметр, если вы хотите экспортировать выбранные данные в ***один*** файл. В этом случае будет активировано поле Имя файла, укажите в нем имя файла экспорта или щелкните по ++...++, чтобы выбрать его в интерактивном режиме. При этом расширение имени файла добавляется автоматически в зависимости от выбранного в поле Тип файла типа файла (например, .csv для всех типов файла 'CSV для ++...++ ').

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Отдельные файлы

Выберите этот параметр, если вы хотите экспортировать выбранные данные в виде отдельных файлов. В этом случае активируется поле Каталог, и в нем указывается каталог, в котором должны выводиться экспортируемые данные в виде отдельных файлов. Имя каждого файла экспорта соответствует номеру изделия, имени схемы сверления, имени производителя и т. д.

### Тип записи данных

Эта настройка доступна ***только*** в том случае, если вы экспортируете всю базу данных изделий или ее подмножество, т. е. вы открыли диалоговое окно через пункты [Дополнительно] > Экспортировать. Выберите с помощью флажков в этой части, какие данные требуется экспортировать.

С помощью типа записи данных "Конфигурация свойств" вы можете экспортировать определенные пользователем свойства, созданные для изделий, в диалоговом окне [Конфигурировать свойства](eservicesgui_d_konfigeigenschaften.md). Однако это возможно только для типов файлов EDZ и XML.

### Раздел

Эта настройка доступна ***только*** в том случае, если вы экспортируете всю базу данных изделий или ее подмножество, т. е. вы открыли диалоговое окно через пункты [Дополнительно] > Экспортировать. Выберите с помощью флажков в этой части, какие разделы требуется экспортировать; если раздел выбран, экспортируются все относящиеся к этому разделу типы изделий. Для раздела Fluid-Техника здесь можно далее ограничить или расширить объем экспорта.

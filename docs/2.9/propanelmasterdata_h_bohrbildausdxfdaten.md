# Генерировать схемы сверления из файлов DXF / DWG

Импорт информации об отверстиях из файлов ЧУ Steinhauer с расширением имен файла *.CNC напрямую невозможен, поскольку в файле ЧПУ всегда собрана вся информация об отверстиях для всех компонентов монтажной платы. Чтобы сделать эту информацию доступной для генерирования схем сверления, файл ЧПУ в машинном ПО Steinhauer необходимо экспортировать в формат DXF / DWG. Экспортирванные файлы DXF- / DWG затем можно импортировать в EPLAN и с помощью границ схемы сверления использовать для генерирования схем сверления.

Кроме файлов DXF- / DWG из машинного ПО Steinhauer, эта функция позволяет также импортировать данные сверления, которые были экспортированы из других источников DXF- / DWG, например систем CAD.

!!! note "Замечание:"

    Для идентификации объектов в качестве функциональных элементов, монтажных отверстий или контуров при импорте требуется специальная схема импорта. Выберите пункты меню Параметры > Настройки > Пользователь > Интерфейсы > Экспорт и импорт DXF- / DWG и создайте новую схему.

    Условием для генерирования схем сверления является присвоение объектов определенным слоям EPLAN. На вкладке [Слои](xdxfgui_r_ebenen.md) присвойте четырем слоям EPLAN для генерирования схем сверления (802 - 805) соответствующее специфическое для CAD имя слоя для функциональных элементов, отверстий, резьбы и контуров. Эти имена слоя определяются через пункты меню Параметры > Настройки > Фирма > Механическая обработка > Экспорт ЧУ DXF.

    ***Пример:***

    Имя (слой) |  Описание |  Имя CAD
    ---|---|---
    EPLAN802 |  Графика.Генерирование схемы сверления.Функциональные элементы |  Component
    EPLAN803 |  Графика.Генерирование схемы сверления.Отверстия |  Drilling
    EPLAN804 |  Графика.Генерирование схемы сверления.Резьбовые отверстия |  Thread
    EPLAN805 |  Графика.Генерирование схемы сверления.Контуры |  Contour

    Чтобы убедиться, что элементы DXF- / DWG импортируются в масштабе 1:1, установите на вкладке Экспорт для настройки Коэффициент масштабирования значение "1".

Условие:

Вы открыли проект макросов.

1. Импортируйте файл DXF / DWG: Для вставки графики DXF / DWG в уже открытую страницу выберите пункты меню Страница > Импортировать > DXF / DWG или Вставить > Графика > DXF / DWG. Используйте при этом предварительно созданную схему импорта с присвоением имен слоя CAD слоям EPLAN для генерирования схем сверления.

!!! info "Для сведения:"

    Графика DXF- / DWG будет добавлена на новую или на уже открытую страницу.

2. При необходимости удалите содержащиеся в графике DXF / DWG блоки. Выберите пункты меню Обработать > Другое > Разбить блок.
3. Проверьте содержащиеся в графике DXF- / DWG контуры на замкнутость.
4. Вставьте границу схемы сверления, которая охватывала бы объекты, предназначенные для схемы сверления: Выберите пункты меню Вставить > Блок / Вывод устройства / Монтажная плата > Граница схемы сверления.

!!! tip "Совет:"

    При генерировании схемы сверления все расчеты позиций и интервалов между участвующими объектами базируются на нулевой точке границы схемы сверления. Позиция нулевой точки на границе схемы сверления определяет позицию нулевой точки на генерируемой схеме сверления. С помощью пункта всплывающего меню Переместить нулевую точку можно изменить позицию нулевой точки и таким образом задать требуемые интервалы для первого и всех последующих отверстий на схеме сверления.

1. В диалоговом окне "Свойства" границы схемы сверления введите имя, описание и каталог для генерируемых определенных пользователем контуров.
2. Сгенерируйте схемы сверления: Выберите пункты меню Сервисные программы > Генерировать схемы сверления / контуры > Автоматически из проекта макросов.
3. Выберите, должен ли процесс применяться ко всему проекту или только к выделенной границе схемы сверления.

!!! info "Для сведения:"

    Схемы сверления генерируются в базе данных изделий из импортированных объектов DXF / DWG, которые присвоены границам схем сверления, в виде сверху.

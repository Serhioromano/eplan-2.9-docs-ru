# Создать библиотеки символов

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Сервисные программы > Основные данные > Библиотека символов > Создать
2. В диалоговом окне Создать библиотеку символов из раскрывающегося списка Сохранить в выберите дисковод и каталог, в котором должна быть сохранена новая библиотека символов.
3. Замените предлагаемое имя файла <Новая библиотека символов.slk> на информативное имя.
4. Щелкните по кнопке ++Сохранить++.

!!! info "Для сведения:"

    если в выбранном дисководе и каталоге уже существует библиотека символов с заданным именем, то ответьте на предупреждение о перезаписи файла.

5. В диалоговом окне Свойства библиотеки символов – <Имя библиотеки> присвойте значения нужным свойствам.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Новая библиотека символов создается, включается в список открытых библиотек символов и может обрабатываться. При этом EPLAN берет системные и пользовательские настройки, сокращенное название фирмы, номер рабочей станции, сокращенное название работника и сохраняет эти данные в библиотеке символов.

!!! note "Замечание:"

    Через настройку проекта (Параметры > Настройки > Проекты > "Имя проекта" > Управление > Библиотеки символов) проекту можно присвоить любое количество библиотек символов. Затем выбранные библиотеки символов сохраняются в проекте.

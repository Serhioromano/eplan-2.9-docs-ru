# Диалоговое окно Формат: Счетчик (нумерация индекса ревизии)

Параметры > Настройки > Фирма > Управление > Отслеживание изменений (нумерация индекса ревизии). В диалоговом окне Настройки: Отслеживание изменений (нумерация индекса ревизии) выберите в списке Доступные элементы формата элемент формата типа "Счетчик" и нажмите на ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо) (или дважды щелкните в списке Выбранные элементы формата на уже определенном счетчике).

В этом диалоговом окне установите форматирование для счетчика в группе форматов.

Обзор основных элементов диалогового окна:

### Минимальное число разрядов

Здесь указывается минимальное число разрядов для значения свойства. Более длинные значения выводятся полностью. Разрешено использовать только положительные числовые значения. При вводе нуля число разрядов является переменным.

### Заполнитель

Здесь можно указать символ заполнителя. Значение заполняется с помощью этого символа, если минимальное число разрядов больше, чем ширина значения. При переменном числе разрядов (Запись "0" в поле Мин. число разрядов) ввод заполнителя невозможен.

### Использовать алфавитно-числов. счетчик

Если установлен этот флажок, применяется алфавитно-числовой счетчик. Если флажок снят, применяется числовой счетчик.

### Алфавитно-числовой счетчик

При использовании алфавитно-числового счетчика здесь необходимо определить символы, которые будут использоваться для счетчика. Разрешается использовать только буквы и цифры.

Алфавитно-числовые счетчики сортируются по символам слева направа (как в столбцах в Excel). Например, если предварительно заданы символы "Тех. марк.", то счет осуществляется следующим образом: A, K, S, AA, AK, AS, KA, ..., AAA, AAK, AAS, AKA ..., AAAA, AAAK, ...

# Вкладка Соединения

Вы открыли проект. В настройках проекта задано, что вкладка Соединения должна использоваться для клемм и контактов штекера (путь меню: Параметры > Настройки > Проекты > "Имя проекта" > Управление > Совместимость).

* В графическом редакторе уже выделена клемма. Обработать > Свойства. Выберите в диалоговом окне Свойства ++...++ вкладку Соединения.
* Вы выделили клемму в навигаторе клеммников или в навигаторе устройств. Обработать > Свойства. Выберите в диалоговом окне Свойства ++...++ вкладку Соединения.

На этой вкладке вы можете изменить определенные свойства клеммных подключений и их соединений, как, напр., тип потенциала или свойство Вывод кабеля. Эти настройки влияют на вывод клемм и их цели в отчетах (например, спецификации клеммников и кабелей). Кроме того, вы можете менять внешние и внутренние цели клеммы.

Обзор основных элементов диалогового окна:

Вкладка Вывод устройства [n]

Для каждого вывода устройства функции клеммы с типом вывода "Жила / провод" отображается отдельная вкладка Вывод устройства [n], в которой перечисляются все цели данного вывода. Функциональные выводы других типов (например, выводы устройства с перемычкой) не отображаются.

!!! example "Пример:"

    Для каждой клеммы с шестью выводами, первые четыре из которых имеют тип вывода "Жила / провод", а остальные две — "Мостовая перемычка", отображаются четыре вкладки:Вывод устройства 1,Вывод устройства 2,Вывод устройства 3и Вывод устройства 4.При изменении типа***второго***вывода устройства в диалоговом окне Логическая схема выводов устройствана тип "Перемычка" будут отображаться только три вкладки:Вывод устройства 1,Вывод устройства 3и Вывод устройства 4.

### Соединения

В данной таблице представлены цели каждого из выводов устройства. Как предварительно установленные отображаются свойства выводов устройства, заданные в диалоговом окне Логическая схема выводов устройства (на вкладке Данные символа / функции). Вы можете изменить свойства выводов устройства; в этом случае измененные значения учитываются в отчетах и при отслеживании целей. При изменениях в диалоговом окне Логическая схема выводов устройства здесь обновляются соответствующие значения.

* Цель: Это поле служит для отображения целей выводов устройства; здесь их нельзя изменить.
* Вывод устройства: С помощью этого раскрывающегося списка вы можете выбирать между параметрами Внутр. и Внешн..
* Подключение кабеля: Если установлен этот флажок, то соответствующее соединение рассматривается как кабельное соединение и учитывается при автоматическом генерировании кабелей. Посредством этой настройки вы можете указать, соединение каких целей будет происходить через кабельное соединение.
* Поз. в ф. (позиция в форме): Позиция в форме указывает, какой по счету целью отображается цель клеммы в отчетах (A, B, C, D).

!!! note "Замечание:"

    Если в диалоговом окне Обработать клеммник с помощью пункта всплывающего меню Клемма: Поменять внешн. / внутр. поменять местами внутреннюю и внешнюю стороны клеммных целей, это изменение также будет отображено здесь. Для первого и второго выводов устройств символов соответственно меняются местами записи в полях Вывод устройства и Поз. форм..

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Сбросить свойства соединения |  Возвращает свойства соединения к значениям, заданным в диалоговом окне Логическая схема выводов устройства. Эта настройка влияет на ***все*** выводы клеммы.
Скрыть цели перемычки |  Если выбран этот параметр, то цели, являющиеся перемычками, не отображаются в столбце Цель. Эта настройка влияет на ***все*** выводы клеммы.

### Число целей

В данном поле отображается число целей для вывода устройства.

### Тип потенциала

В данных полях вы можете задать тип потенциала для вывода устройства согласно присвоению кабельных соединений. На выбор есть следующие типы потенциалов: -, +, L, M, N, неопределенный, PE, PEN и SH. Как предварительно установленные отображаются значения из диалогового окна Логическая схема выводов устройства.

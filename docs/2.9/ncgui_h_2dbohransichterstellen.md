# Вставить двухмерное отображение отверстий

Отображение отверстия (2D) — форма обзора модели, специализированная на отображении данных, релевантных для ЧУ. Она отображает данные, релевантные для ЧУ, отверстия, монтажные отверстия, а также геометрические контуры базисных функциональных элементов и изделий. В отображение отверстия может быть вставлено масштабирование.

Условия:

* Вы открыли проект и находитесь на странице в графическом редакторе.
* Проект содержит пространство листа с базисными функциональными элементами, релевантными для ЧУ (монтажные платы, двери, щитки, профили), на которых размещаются функциональные элементы со схемами сверления или монтажными отверстиями, или в которые помещены стандартные контуры ЧУ (отверстия, продольные отверстия, фрагменты) или свободные контуры ЧУ.

1. Выберите пункты меню Вставить > Графика > 2D-отображение отверстий.
2. Задайте углы области для двухмерного отображения отверстия.
3. Задайте во вкладке Вид диалогового окна Отобразить отверстия специфические данные для двухмерного отображения отверстия.
4. В поле Основной функциональный элемент при помощи кнопки ++...++ выберите функциональный элемент или поверхность, релевантную для ЧУ, которые необходимо отобразить в двухмерном отображении отверстия.
5. На вкладке Отображение определите настройки отображения для описания двухмерного отображения отверстия, например размер шрифта, выравнивание и ширину / высоту блока выравнивания текста.
6. На вкладке Прямоугольник определите графические свойства рамки двухмерного отображения отверстия, например начальную точку, конечную точку и ширину.
7. На вкладке Формат укажите свойства формата для рамки двухмерного отображения отверстия.
8. Щелкните по кнопке ++OK++.

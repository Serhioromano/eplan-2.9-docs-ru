# Трубопроводы в пространстве листа

Соединения из гидротехнического планирования можно свободно маршрутизировать в качестве трубопроводов в пространстве листа. Полный порядок действий для свободной маршрутизации и последующей обработки трубопровода включает следующие шаги:

* Вставить функциональные элементы Fluid-техники на многополюсной странице схемы соединений Fluid-техники
* Вставить резьбовые соединения на многополюсной странице схемы соединений Fluid-техники
* [Спроектировать многополюсные трубопроводы](ftechnic_h_paketedefinieren.md) между функциональными элементами Fluid-техники или резьбовыми соединениями: Проектирование выполняется для трубопровода с определением функции Труба при помощи точки определения соединения. Такой трубопровод не будет иметь собственное ОУ. Проектирование также может выполняться для группы соединений / линии с определением функции Определение трубопровода. В этом случае трубопровод имеет собственное ОУ.
* Вставить функциональные элементы Fluid-техники в пространство листа
* Вставить резьбовые соединения на функциональном элементе Fluid-техники
* Свободно маршрутизировать трубопроводы между резьбовыми соединениями: Для этого на минимальном расстоянии между источником и целью трубопровода автоматически генерируется сегмент маршрутизации, на котором автоматически маршрутизируется трубопровод. Сгенерированные при этом колена труб будут иметь угол изгиба 90°. Наложения свободно маршрутизируемого трубопровода на другие функциональные элементы при свободной маршрутизации автоматически не контролируются.
* После маршрутизации автоматически сгенерированные трубопроводы и колена труб вручную изменяются в соответствии с установочным положением:

***Прямые сегменты трубы***: Перемещение ортогонально в пределах плоскости или за ее пределами, удлинение, укорачивание

***Колена труб / Места изгибов***: Перемещение, изменение радиуса, вставка дополнительных мест изгибов, удаление.

При обработке трубопровода режим конструкции также можно использовать вместе с включенным захватом объекта.
* Сгенерировать обзор модели и таким образом представить свободно маршрутизированные трубопроводы вместе с функциональными элементами Fluid-техники и резьбовыми соединениями
* Экспортировать свободно маршрутизируемые трубопроводы в формате XML
* Генерировать данные изготовления для изгибания трубопроводов с помощью программы трубогибочной машины
* Изогнуть трубопроводы с помощью трубогибочной машины.

# Диалоговое окно Подчиненные формы

Открыты проект и динамическая форма (например, через меню Сервисные программы > Основные данные > Форма > Открыть > [Открыть]). Вставить > Подчиненные формы. Размещена подчиненная форма. Дважды щелкните по символу формы.

В данном диалоговом окне задаются формы, использующиеся при определенных условиях фильтрации для отчетов. При этом формы находятся в списке, сгруппированными по условиям фильтрации. Если несколько форм имеют одинаковые условия фильтрации, они объединяются в одном разделе между двумя линиями.

Обзор основных элементов диалогового окна:

### Имя формы

Если формы уже заданы, список отображает их имена. Щелкните кнопку ![](images/all_new_as.png){: .ui-icon } (Создать), чтобы в диалоговом окне Выбрать форму выбрать подходящую форму. Подходящими считаются только те формы, которые имеют такой же тип файла, как "Главная форма".

!!! example "Пример:"

    В редакторе форм открыта форма дляспецификацииклеммника. Диалоговое окно Выбрать формупредлагает только формы типа "Спецификация клеммника".

Если список содержит несколько форм, имеющих различные условия, то данные формы отображаются сгруппированно и при смене условий разделяются линией.

Таблица

В таблице определяются критерии фильтра для форм. Отдельные строки стандартно связаны логическим оператором И.

### Значение

Здесь занесите значение для сравнения, с которым должно сравниваться значение выбранного свойства. Возможно использование [заполнителей](searchandreplacegui_k_platzhalter.md) типа '*' и '?'. Ввод нескольких значений в одну ячейку (с разделением точкой с запятой) расценивается как операция ИЛИ.

Для некоторых свойств имеется возможность открыть диалоговое окно выбора с помощью кнопки ++...++. Если в качестве критерия выбран, например, структурный идентификатор, диалоговое окно выбора содержит имеющиеся в проекте значения этого структурного идентификатора.

В случае с полями, которые представляют каталоги, можно вручную ввести замещающий символ "*"; в этом случае в выбор войдут все подкаталоги.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

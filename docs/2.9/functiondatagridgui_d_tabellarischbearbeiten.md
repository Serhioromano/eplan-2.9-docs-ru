# Диалоговое окно Табличная обработка — <Имя проекта>

Вы открыли проект.

* Вы выделили в графическом редакторе объекты или фрагмент схемы соединений. Обработать > В табличном виде.
* Вы выделили объекты в одном из навигаторов. Всплывающее меню > Табличная обработка.

В этом диалоговом окне данные выделенных объектов отображаются в виде таблицы и доступны для обработки. Если выбор содержит несколько типов объекта (функции, соединения, макросы, точки разрыва, потенциалы, трубопроводы, сегменты), для каждого типа объекта отображается отдельная вкладка.

!!! note "Замечание:"

    При смене выделения в открытом диалоговом окне, необходимо обновить отображение в диалоговом окне с помощью пункта всплывающего меню Табличная обработка.

Обзор основных элементов диалогового окна:

### Схема

Доступно только для функций и соединений. Для каждой функции и соединения выберите из раскрывающегося списка отдельную доступную схему, определяющую конфигурацию столбцов. С помощью кнопки ++...++ откройте диалоговое окно Настройки: Табличная обработка, в котором можно создать или обработать схему.

### Таблица

В таблице отображаются столбцы, указанные в выбранной схеме или диалоговом окне Конфигурировать представление. Здесь показываются лишь определенные данные (поля заливаются серым), остальные могут обрабатываться. Каждый выделенный объект отображается в отдельной строке.

Обработка функций

Записи можно обрабатывать в столбце Имя (видимое), чтобы, например, изменять буквенные обозначения и счетчик устройств. При изменении записи выполняется автоматическая синтаксическая проверка ОУ, как в диалоговом окне Свойства ++...++.

Обработка соединений

* В диалоговом окне табличной обработки или на вкладке Соединения отображаются свойства выделенных соединений. При этом учитываются также все точки определения соединения, размещенные на соединении. Если для одного свойства на соединении и на соответствующих точках обозначения соединений записаны разные значения, отображаться будет значение с ***графически первой*** точки обозначения соединения, для которой заполнено это свойство.
* Определить, взяты ли данные, отображаемые в какой-либо строке, из соединения или из соответствующих точек обозначения соединения, можно с помощью столбца Имеется точка определения соединения. Если на соединении размещена как минимум одна точка определения соединения, то флажок установлен.
* При изменении значения свойства соединения на вкладке Соединения измененное значение сразу же переносится как на соединение, так и на все соответствующие точки определения соединения, на которых заполнено это свойство. Пустые свойства не будут перезаписаны.
* Если у соединения нет точек обозначения соединения, при изменении значения одного из свойств, точка обозначения соединения размещается автоматически, и значение свойства записывается на нее.

Обработка макросов

В диалоговом окне табличной обработки или на вкладке Макросы отображаются свойства макросов, содержащихся в выделении. Эти свойства доступны для обработки в ***проекте макросов***. В ***проекте схем соединений*** данные свойств только отображаются и не доступны для обработки.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

Дополнительно здесь представлены следующие пункты всплывающего меню, специфические для данного диалогового окна:

Пункт меню |  Значение
---|---
Перейти к (графика) |  Отображает выделенный объект в Графическом редакторе.
Вставить в список результатов поиска |  Заносит все объекты проекта, имеющие ссылку на выделенный элемент, в список результатов поиска.
Свойства |  Открывает диалоговое окно Свойства (усл. обозначение): ++...++. Обеспечивает обработку функции.
Свойства (общие) |  Открывает диалоговое окно Свойства (общие): ++...++. Позволяет обрабатывать свойства устройства.

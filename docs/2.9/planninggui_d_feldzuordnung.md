# Диалоговое окно Присвоение полей (предварительное планирование)

Вы открыли проект и навигатор предварительного планирования. Данные проекта > Предварительное планирование > Импортировать. В диалоговом окне Импортировать данные предварительного планирования — <Имя проекта> выберите в поле Тип источника данных записи "Excel" и "Текст" и щелкните в поле Присвоения полей по кнопке ++...++.

В этом диалоговом окне создаются и изменяются схемы, задающие присвоение внешних полей данных свойствам EPLAN.

Доступны следующие элементы диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую вы хотите обработать. Вы можете обрабатывать только пользовательские схемы.

Через расположенную рядом [панель инструментов](schemeconfig_m_schemaschaltflaechen.md) можно создавать новые схемы, сохранять изменения в схеме, копировать или удалить схему. Кроме того, схемы можно импортировать и экспортировать.

### Описание

Данное поле отображает тексты описаний, которые вы ввели при создании схемы в диалоговом окне Новая схема в поле Описание.

* * *

Таблица

В таблице имеются следующие столбцы:

### Статус

Зеленая пиктограмма ![](images/all_statusok_as.png){: .ui-icon } показывает, что это поле данных имеется в источнике данных и его удалось идентифицировать. Желтая пиктограмма ![](images/all_statusnotok_as.png){: .ui-icon } означает, что соответствующие поля данных отсутствуют. Эти строки не учитываются при импорте, и при необходимости вы можете выделить их и удалить.

### Внешнее поле

В этом столбце отображаются поля данных / свойства выбранной таблицы внешнего источника данных.

### Уровень

Здесь отображается уровень иерархии представления в виде дерева. Уровень "0" означает, что данная строка не импортируется.

### Определение сегмента

Укажите здесь, какой тип сегмента из записи данных необходимо создать, например объект планирования или сегмент структуры.

### Свойство

В этом столбце отображается присвоенное свойство EPLAN. Для присвоения свойства щелкните столбец, а затем [...] и выберите в открывшемся диалоговом окне свойство.

### Идентифицирующ.

Установите данный флажок, если соответствующее свойство является идентифицирующим при присвоении. На основании свойств, отмеченных как идентифицирующие, идентифицируются и синхронизируются объекты при импорте. Для каждого типа объектов по крайней мере одно внешнее поле должно быть идентифицирующим!

### Фильтр

Здесь можно задать фильтр для соответствующего свойства. При внесении значения выполняется фильтрация по совпадению. Например, можно ввести "> 100".

### Панель инструментов

На панели инструментов сверху от таблицы доступны следующие кнопки:

Кнопка |  Значение
---|---
![](images/all_new_as.png){: .ui-icon } (Создать) |  Создает новую строку в таблице. При этом новая строка добавляется в конце таблицы.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет со всем содержимым ту строку, в одной из ячеек которой находится курсор. Возможен многократный выбор.

# Диалоговое окно Формат: Комментарий

Вы открыли проект. Открыто диалоговое окно свойств для функции, символа или проекта. В поле Свойства выбрано свойство Свойство блока: Формат [n], а в столбце Значение нажата кнопка ++...++.

* Выберите в диалоговом окне Формат в списке Доступные элементы формата элемент формата Комментарий и щелкните по ![](images/all_arrowright_as.png){: .ui-icon } (Переместить направо).
* Выделите в диалоговом окне Формат в списке Выбранные элементы формата имеющийся элемент формата типа Комментарий и щелкните по ![](images/all_edit_as.png){: .ui-icon } (Обработать).

В этом диалоговом окне задайте в группе форматов комментарий, при помощи которого Вы можете документировать Ваши настройки формата. Комментарий может вводиться только один раз на каждое свойство и сортируется всегда в конце. Он не выводится в результате.

Обзор основных элементов диалогового окна:

### Комментарий

Введите здесь любой текст. Комментарий, как и вся символьная строка формата, независим от языка.

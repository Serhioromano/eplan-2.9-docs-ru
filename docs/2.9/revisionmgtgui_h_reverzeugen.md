# Генерировать ревизию

Условия:

* Открыт стандартный проект (Расширение имени файла *.elk) или проект ревизии (Расширение имени файла *.ell).
* Вы закрыли все страницы проекта ревизии.

1. Выберите следующие пункты меню: Сервисные программы > Управление ревизиями > Отслеживание изменений > Генерировать ревизию
2. В диалоговом окне [Генерировать ревизию](revisionlogmgtgui_d_revisionerzeugen.md) введите в соответствующих полях имя и краткое описание (необязательно) ревизии.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    EPLAN генерирует ревизию под указанным именем.

!!! info "Для сведения:"

    Если вы генерируете ревизию для стандартного проекта, проект изменяется на проект ревизии, т. е. расширение имени файла меняется с *.elk на *.ell.

!!! info "Для сведения:"

    При изменениях на странице она обозначается как "План".

!!! tip "Совет:"

    Если в настройках проекта (Параметры > Настройки > Проекты > "Имя проекта" > Управление > Ревизия (отслеживание изменений)) установлен флажок При новой ревизии закрытого проекта сгенерировать исходный проект, дополнительно генерируется исходный проект (т. е. копия проекта с защитой от записи), когда генерируется новая ревизия ***закрытого*** проекта.

### Генерировать ревизию для области обработки

!!! note "Замечание:"

    Генерирование ревизии для области обработки возможно, только если сам проект не был подвергнут ревизии ранее. Но при этом можно сгенерировать ревизию для всего проекта, если он содержит уже подвергнутые ревизии области обработки.

Условия:

* Вы открыли стандартный проект (расширение имени файла *.elk).
* Вы выбрали и активировали область обработки, которая не накладывается на другие области обработки.
* Вы закрыли область обработки.

1. Выберите следующие пункты меню: Сервисные программы > Управление ревизиями > Отслеживание изменений > Генерировать ревизию
2. В диалоговом окне [Генерировать ревизию](revisionlogmgtgui_d_revisionerzeugen.md) введите в соответствующих полях имя и краткое описание (необязательно) ревизии.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    EPLAN генерирует ревизию под указанным именем.

!!! info "Для сведения:"

    Ревизия присвоена текущей области обработки.

!!! info "Для сведения:"

    Проект становится проектом ревизии, т. е. расширение имени файла изменяется с *.elk на *.ell.

!!! info "Для сведения:"

    При изменениях на странице она обозначается как "План".

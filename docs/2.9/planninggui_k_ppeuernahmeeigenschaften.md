# Копирование данных проектов EPLAN PPE: Свойства

При копировании данных проектов с базой данных PPE свойства из EPLAN PPE преобразуются в определенные пользователем свойства для предварительного планирования. В дереве диалогового окна конфигурации Конфигурировать свойства эти свойства отображаются под узлом Eplan и PPE.

В EPLAN PPE одно свойство можно присвоить различным группам свойств. Поскольку на платформе EPLAN свойство должно быть однозначным, группы свойств при копировании данных изображаются в качестве дополнительных узлов во внутреннем идентифицирующем имени свойства (GROUP1–GROUP8) и копируются как часть отображаемого имени. Отображаемое имя впоследствии можно изменить.

!!! example "Пример:"

    Из свойства PPEАгрегатное состояние, которое использовалось в группах свойств "Данные измеряемого материала" и "Данные регулируемого материала", после копирования данных получаются следующие два определенные пользователем свойства:Идентифицирующее имя:Eplan.PPE.GROUP2.АГРЕГАТНОЕ_СОСТОЯНИЕОтображаемое имя:Данные измеряемого материала: Агрегатное состояние.Идентифицирующее имя:Eplan.PPE.GROUP5.АГРЕГАТНОЕ_СОСТОЯНИЕОтображаемое имя:Данные регулируемого материала: Агрегатное состояние.

Для отображения в программе эти свойства, которые прежде использовались в одной группе свойств, по умолчанию присваиваются категориям от "Предварительное планирование 1" до "Предварительное планирование 8". В помощью этих категорий свойства можно наглядно представить, например в диалоговом окне "Свойства" сегментов. Однако группировка свойств по категориям не является обязательной.

Все другие скопированные свойства PPE по умолчанию получают категории "Данные".

Группа свойств (PPE) |  Узел |  Категория
---|---|---
Измерительные контуры (общее) / потребители (общее) |  GROUP1 |  Предварительное планирование 1
Данные измеряемого материала / вид потребителя |  GROUP2 |  Предварительное планирование 2
Данные места измерения / электр. данные потребителя |  GROUP3 |  Предварительное планирование 3
Данные измерительного устройства / мех. данные потребителя |  GROUP4 |  Предварительное планирование 4
Данные регулируемого материала / управление |  GROUP5 |  Предварительное планирование 5
Данные по местоположению / характ. пуска двигателя |  GROUP6 |  Предварительное планирование 6
Данные сервомеханизмов / конструкция |  GROUP7 |  Предварительное планирование 7
Прочие измерительные контуры / прочие потребители |  GROUP8 |  Предварительное планирование 8

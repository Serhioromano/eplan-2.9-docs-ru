# Диалоговое окно Нумеровать клеммы/контакты штекеров

Вы открыли проект.

* Вы выделили несколько клемм в графическом редакторе, навигаторе клеммников или в навигаторе устройств. Или вы выделили страницы или проект в навигаторе страниц. Данные проекта > Клеммники > Нумеровать клеммы.
* Вы выделили несколько клемм в графическом редакторе или навигаторе клеммников. Данные проекта > Клеммники > Обработать. Выделите одну или несколько клемм. Выберите пункт всплывающего меню Нумеровать клеммы.
* Вы выделили несколько контактов штекеров в графическом редакторе, навигаторе штекеров или навигаторе устройств. Или вы выделили страницы или проект в навигаторе страниц. Данные проекта > Штекеры > Нумерация контактов штекеров.
* Вы выделили несколько контактов штекеров в графическом редакторе или навигаторе штекеров. Данные проекта > Штекеры > Обработать. Выделите один или несколько контактов штекеров. Выберите пункт всплывающего меню Нумерация контактов штекеров.

В этом диалоговом окне устанавливаются настройки обозначения клемм или обозначения контактов штекеров. При нумерации клемм откроется диалоговое окно Нумеровать клеммы, а при нумерации контактов штекеров откроется диалоговое окно Нумерация контактов штекеров. Все клеммы и контакты штекеров имеют четкие обозначения, которые могут быть как алфавитно-цифровыми, так и просто цифровыми.

Обзор основных элементов диалогового окна:

### Схема

Из этого раскрывающегося списка можно выбрать схему нумерации.

С помощью кнопки ++...++ откройте диалоговое окно Настройки: Нумерация. В этом диалоговом окне можно обработать существующую схему или создать новую.

### Последовательность

Через раскрывающийся список указывается последовательность нумерации:

* Как сортировка: в диалоговом окне Клеммники — <Имя проекта> или Штекеры — <Имя проекта> используется отображенная последовательность сортировки клемм / контактов штекера. Выберите этот параметр, если вам необходимо пронумеровать имеющиеся клеммы / контакты штекеров в текущей последовательности.
* C ориент. на страницы: Клеммы / контакты штекеров нумеруются постранично в той последовательности, в которой они были проанализированы. Последовательность анализа задана в рамке. Например, при работе с проектами по стандарту IEC нумерация осуществляется по столбцам, а с проектами по стандарту NFPA — по строкам.
* С ориент. на кабели: Клеммы / контакты штекеров нумеруются в той последовательности, в которой проложены кабельные соединения.
* C ориент. на уровни: Клеммы нумеруются последовательно по уровням. При этом присвоение многоуровневых клемм сохраняется. Настройка доступна только для клемм.
* С ориентацией на страницы (альтернативно выравниванию рамки): клеммы / контакты штекера нумеруются по страницам, при этом последовательность является альтернативной направлению создания отчета для рамки. Например, при работе с проектами по стандарту IEC нумерация осуществляется по строкам, а с проектами по стандарту NFPA — по столбцам.

### Объем

Через раскрывающийся список указывается объем нумерации:

* Выделенные: Нумеруются только выделенные клеммы или контакты штекеров.
* Все из выбранных клеммников / штекеров: Нумеруются все клеммы выделенных клеммников или все контакты выделенных штекеров.
* Все из всех клеммников / штекеров: Нумеруются клеммы всех клеммников в проекте или контакты всех штекеров в проекте.

* * *

Групповое поле "Цифровые настройки"

### Начальное значение

Здесь указывается начальное значение нумерации клемм/контактов штекеров.

### Величина шага

Здесь задается интервал между номерами двух клемм/контактов штекеров. Так, например, если вы задали величину шага "5", клеммам/контактам штекеров будут автоматически присвоены номера 1, 6, 11 и т. д.

Количество позиций определяется из формата нумерации в режиме подключения (Параметры > Настройки > Проекты > "Имя проекта" > Устройства > Нумеровать (online)).

### Префикс / Суффикс

Префикс ставится перед нумерацией, суффикс добавляется к нумерации. Введите здесь соответствующий знак.

* * *

Групповое поле Клеммы / контакты штекера, связанные с потенциалом

### PE-клеммы / -контакты штекера... SH-клеммы / -контакты штекера

В данных полях определяются настройки нумерации PE- / PEN-, N- и SH-клемм или контактов штекеров. Настройки для типов потенциалов PE / PEN, N и SH можно копировать раздельно. Предусмотрен выбор следующих опций:

* Не изменять: Все клеммы или контакты штекеров не менее чем с одним выводом устройства, которое настроено на тип потенциала "PE", "PEN", "N" или "SH", при нумерации игнорируются.
* Нумеровать: Все PE-, PEN-, N- или SH-клеммы или контакты штекеров нумеруются.
* Не изменять, соблюдать последовательность: Настройка доступна только для клемм. PE-, PEN-, N- или SH-клеммы при нумерации не изменяются, но учитываются при нумерации остальных клемм. Это означает, что обозначения для остальных клемм распределяются так, что PE-, PEN-, N- или SH-клеммы нумеруются вместе.

### Многоуровневые клеммы

Через раскрывающийся список определите, как нумеруются многоуровневые клеммы:

* Не изменять: Клеммы со свойством Допускается одинаковое обозначение при нумерации игнорируются.
* Нумеровать одинаково: Одинаково обозначенные клеммы со свойством Допускается одинаковое обозначение получают одинаковый номер.
* Нумеровать по отдельности: Клеммы со свойством Допускается одинаковое обозначение получают собственный номер. Если многоуровневые клеммы перед нумерацией имели одинаковый номер, то после они имеют различные номера.

!!! tip "Совет:"

    Многоуровневые клеммы различаются по предшествующему знаку (например, 1, +1, -1), и, если их необходимо пронумеровать одинаково, в поле Обозначенные в алфавитном порядке выберите параметр 'Алфавитные элементы не изменять'.

### Обозначенные в алфавитном порядке

С помощью раскрывающегося списка можно определить, каким образом буду нумероваться клеммы или контакты штекера с буквенными компонентами в обозначении. Если обозначение состоит не только из цифр, оно состоит из буквенных компонентов.

* Не изменять: Клеммы или контакты штекеров с символами алфавита в обозначении при нумерации игнорируются.
* Алфавитные элементы не изменять: Символы алфавита в обозначении клеммы или контакта штекера остаются. Если обозначение включает только алфавитные элементы, старое обозначение будет добавлено к новой нумерации. Следующие одна за другой клеммы с неодинаковыми счетчиками, но с одинаковым числовым элементом после нумерации получают одинаковый числовой элемент в счетчике.
* Нумеровать: Все клеммы / контакты штекера нумеруются заново. При этом старое обозначение перезаписывается.

### Разрешить изменение сортировки

Если этот флажок установлен, сортировка, а следовательно, последовательность клемм / контактов штекера внутри клеммника / штекера при нумерации могут изменяться, если идентификатор сортировки клемм / контактов штекера ***пуст***. Идентификаторы сортировки, занесенные вручную, остаются неизменными и учитываются в последовательности нумерации.

Если флажок снят, при нумерации сохраняется сортировка, а следовательно, и последовательность клемм / контактов штекеров внутри клеммника / штекера. С этой целью клеммы / контакты штекера с ***пустым*** идентификатором сортировки перед нумерацией автоматически получают идентификатор сортировки. Идентификаторы сортировки, занесенные вручную, остаются неизменными и учитываются в последовательности нумерации. В качестве автоматически определяемого идентификатора сортировки используется имеющееся обозначение клемм / контактов штекера.

### Пронумеровать только обозначенные

Если этот флажок установлен, нумероваться будут только те клеммы / контакты штекеров, обозначение которых начинается с "?" или не заполнено.

### Заполнить пробелы

Этот флажок доступен, если активирован параметр Пронумеровать только обозначенные.

Если этот флажок установлен, при нумерации учитываются имеющиеся обозначения внутри клеммников / штекеров. Первая клемма, считая от начального значения, обозначенная "?", (или первый контакт штекера) получает первое свободное обозначение. Затем нумерация продолжается с указанной величиной шага, причем уже имеющиеся обозначения пропускаются.

Если флажок снят, имеющиеся обозначения при нумерации не учитываются. В этом случае возможно появление многократных обозначений внутри клеммников / штекеров.

!!! note "Замечание:"

    Если вы нумеруете клеммы по схеме "Определение устройства" или "Инициаторы", может случиться, что все обозначения клемм устройства или инициатора уже используются. В этом случае клеммы, для которых не найдено свободное обозначение, не нумеруются.

### Также нумеровать с учетом влияния ПЛК

Если установить этот флажок, в нумерацию будут также включены клеммы/контакты штекеров с влиянием ПЛК.

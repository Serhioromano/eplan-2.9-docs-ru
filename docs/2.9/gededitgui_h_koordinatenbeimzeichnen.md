# Ввести координаты при черчении

Во время черчения графически элементов, а также при вставке символов можно также ввести координаты начальной и конченой точки или точки вставки. Для этого используется абсолютный или относительный ввод координат.

Начальную и конечную точку можно указать с помощью ввода координат, или же можно указать начальную точку вручную, а конечную точку — с помощью ввода координат. Ввод координат можно комбинировать со вводом через область ввода данных. Однако необходимо учитывать, что область ввода данных всегда использует ***графическую*** систему координат независимо от системы координат соответствующей страницы проекта.

В примере описывается ввод при черчении линии, таким же образом можно начертить ломаные линии, прямоугольники, круги и т.д.

При вставке символов для точки вставки требуется только ввод координат.

Условие:

Вы открыли страницу, форму, рамку или символ.

### Ввести абсолютные координаты

1. Выберите пункты меню Вставить > Графика > Линия.
2. Выберите пункты меню Параметры > Ввод координат.
3. В диалоговом окне Ввод координат выводится текущая позиция курсора. Введите координаты начальной точки.
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Курсор перемещается на указанную позицию координат.

5. Снова выберите пункты меню Параметры > Ввод координат.
6. Введите координаты конечной точки в диалоговом окне Ввод координат.
7. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Линия чертится до указанной точки.

### Ввести относительные координаты

1. Выберите пункты меню Вставить > Графика > Линия.
2. Укажите вручную начальную точку и нажмите клавишу ++Ввод++. Или укажите начальную точку с помощью абсолютного ввода координат (Параметры > Ввод координат).
3. Выберите пункты меню Параметры > Относительный ввод координат.
4. Укажите позицию конечной точки в диалоговом окне Относительный ввод координат. Для этого предусмотрены две возможности: Во-первых, в поле Интервал можно указать, на какое значение в направлении X и Y относительно координат начальной точки должна быть перемещена конечная точка. Во-вторых, можно указать угол и длину линии. (Для прямоугольника вводится длина его диагоналей)
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Линия чертится до указанной точки.

!!! note "Замечание:"

    Если на ***Граф. страницы*** с относительным вводом координат необходимо задать точку относительно другой в миллиметрах, при активной сетке учитываются не введенные миллиметры, а выполняется приблизительно подходящий шаг вдоль сетки.

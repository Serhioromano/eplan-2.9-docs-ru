# Диалоговое окно Копировать графический файл

Вы открыли страницу, форму или рамку. Вставить > Графика > Графический файл. В диалоговом окне Выбрать графический файл выберите требуемый графический файл и нажмите кнопку ++Открыть++.

В EPLAN можно вставить графические файлы форматов GIF, JPG, BMP, ICO, PNG или TIFF. В этом диалоговом окне указывается, следует ли скопировать графический файл в каталог проекта или должна быть создана ссылка на исходный каталог графического файла.

Доступны следующие элементы диалогового окна:

### Исходный каталог не изменять

В проект вставляется ссылка на графический файл. В случае изменений этот графический файл будет обновлен при открытии данной страницы. Однако файлы, на которые указывают ссылки, не учитываются при резервировании данных.

### Копировать

Файл копируется в каталог проекта (т.е. в подкаталог Images проекта). Графический файлы, которые находятся в этом каталоге, учитываются при резервировании данных.

### Перезаписать с запросом

Установите этот флаг, чтобы в случае использования опции Копировать выводился запрос, если данный рисунок уже имеется в каталоге проекта. При этом можно выбрать, перезаписывать этот рисунок или нет.

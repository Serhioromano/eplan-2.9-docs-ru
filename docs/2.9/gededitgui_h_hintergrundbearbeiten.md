# Выбор и обработка фоновых изображений

Такие элементы, как графические файлы, графические элементы или импортированные файлы DXF / DWG, можно использовать в качестве фонового изображения. Для этой цели в управлении слоями служит настройка Фон. Элементы в настроенном таким образом слое всегда лежат в самом низу, то есть даже позади линий автоматического соединения.

Условия:

* Вы открыли страницу, форму, рамку или символ.
* Вы вставили графический файл, файл DXF / DWG, графические элементы и т.д.
* В управлении слоями для определенных слоев установлен флажок Фон.

### Указать элементы в качестве фонового изображения

1. Щелкните дважды на выбранном элементе, чтобы открыть диалоговое окно Свойства.
2. Выберите в диалоговом окне Свойства вкладку Формат.
3. Выберите в поле Слой слой, для которого в управлении слоями установлен флажок Фон (например, слой EPLAN705, Графика.Фоновая картинка).
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Элемент будет отодвинут на задний план, и его нельзя будет больше выделить или обработать.

### Обработать фоновое изображение

1. Выберите пункты меню Параметры > Обработать фон.

!!! info "Для сведения:"

    Элементы, лежащие на слое, для которого установлена настройка Фон, можно выделять и обрабатывать.

2. Щелкните дважды по выбранному элементу, чтобы открыть диалоговое окно Свойства, и обработайте в нем свойства элемента.
3. Чтобы отключить эту опцию, выберите пункты меню Параметры > Обработать фон.

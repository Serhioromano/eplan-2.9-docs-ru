# Вставить новые устройства без ОУ

Новые устройства/функции можно разместить через "Вставить > Символ" без автоматического присвоения ОУ в схеме соединений. При этом записи видимого и полного ОУ остаются пустыми.

Условие:

Вы открыли проект.

1. Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация (online)
2. Выберите в групповом поле При вставке символов опцию Пустое поле.
3. Щелкните по кнопке ++ОК++.

### Добавить устройства/функции через символ без ОУ

1. Выберите пункт меню Вставить > Символ.
2. Выберите символ в диалоговом окне Выбор символа и нажмите ++OK++.

!!! info "Для сведения:"

    Символ появится рядом с курсором.

3. Переместите курсор в нужное место на странице схемы.
4. Для размещения символа щелкните по странице.

!!! info "Для сведения:"

    Диалоговое окно Свойства (усл. обозначение): ++...++ будет открыто. В поле Видимое ОУ отсутствует ОУ.

5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Символ будет размещен без ОУ.

### Создавать устройства/функции без размещения и без ОУ

1. Выберите пункты меню Данные проекта > Устройство > Навигатор.
2. Выберите вкладку Дерево.
3. Выделите слой проекта.
4. Выберите во всплывающем меню > "Создать".
5. Отметьте в диалоговом окне Определения функций в поле Выбор определение функции и щелкните ++OK++.

!!! info "Для сведения:"

    Диалоговое окно Свойства (усл. обозначение): ++...++ будет открыто. В поле Видимое ОУ отсутствует ОУ.

6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Устройство/функция будут отнесены к уровню структуры дерева ***без ОУ***.

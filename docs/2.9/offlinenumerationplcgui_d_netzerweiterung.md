# Диалоговое окно Сетевое расширение (нумерация с данными ПЛК)

Вы открыли проект. Вы открыли диалоговое окно [Настройки: Нумерация с данными ПЛК](offlinenumerationplcgui_d_einstellungen.md) (например, через Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Нумерация с данными ПЛК). Выберите в диалоговом окне Настройки: Нумерация с данными ПЛК определенную пользователем схему и нажмите кнопку ++Сетевое расширение++.

В этом диалоговом окне можно расширить определение сети. Эта настройка влияет на обозначение, если в качестве объема выбрана "сеть". Сетевое расширение для нумерации с данными ПЛК не зависит от свойства Соединенный с сетью соответствующей функции.

Обзор основных элементов диалогового окна:

### Выбор

В дереве выводятся все функции. С помощью кнопки со стрелкой можно перенести выделенную запись из левого списка в правый. Кроме того, можно выделить категорию и перенести все находящиеся под ней функции.

### Сетевое расширение через следующие объекты

В этом списке выводятся все функции, которые были перенесены из списка выбора. В этом случае сеть не заканчивается на выбранных функциях, а продолжается за ними.

С помощью команды ![](images/all_delete_as.png){: .ui-icon } (Удалить) можно убрать выбранную функцию списка.

### Всплывающее меню

Пункт меню |  Значение
---|---
Выделить все (только список) |  Выделяет все записи в списке.

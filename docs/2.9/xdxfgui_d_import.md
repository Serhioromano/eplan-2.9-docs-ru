# Диалоговое окно Импорт DXF/DWG

Вы открыли проект.

* Страница > Импортировать > DXF / DWG. Выберите в диалоговом окне Выбор файла DXF/DWG (как минимум) один файл и щелкните по ++Открыть++.
* В Графическом редакторе открыта страница проекта. Вставить > Графика > DXF / DWG. Выберите в диалоговом окне Выбор файла DXF/DWG файл и щелкните по ++Открыть++.
* Сервисные программы > Генерировать макросы > Из файлов DXF / DWG. Выберите в диалоговом окне Выбор файла DXF/DWG (как минимум) один файл и щелкните по ++Открыть++.

В этом диалоговом окне запускается непосредственный процесс импорта.

Обзор основных элементов диалогового окна:

### Источник

Здесь отображается имя файла, в котором изначально содержались данные.

### Схема

Выберите из раскрывающегося списка подходящую схему или щелкните ++...++, чтобы вызвать диалоговое окно Настройки: Экспорт и импорт DXF / DWG и открыть или обработать схему или создать новую схему.

### Цель

Только для создания макросов! Введите здесь целевой каталог или выберите его в интерактивном режиме с помощью кнопки ++...++. С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.
Выбранные файлы сохраняются в указанном каталоге макросов как макросы окон <исходное_имя>.ema.

После подтверждения нажатием ++OK++ определите при необходимости Форматир. импорта и в следующем диалоговом окне (Присвоить страницы) порядок обработки страниц (при импортировании).

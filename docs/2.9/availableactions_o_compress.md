# Операция: compress

**Параметр** |  **Описание**
---|---
PROJECTNAME  |  Имя проекта с полным путем файла (является опцией).
Если не задано, то выбранный проект используется, когда операция вызывается через интерфейс пользователя (напр., через сценарий или панель инструментов). При вызове из командной строки Windows следует определить PROJECTNAME или сначала следует использовать ProjectAction. В противном случае отобразится системное сообщение.
CONFIGSCHEME  |  Схема конфигурации для сжатия проекта (необязательно).
Значение по умолчанию: последняя схема конфигурации. При передаче пустой символьной строки используется последняя использованная схема.
FILTERSCHEME  |  Схема фильтра для сжатия проекта (необязательно).
Значение по умолчанию: последняя использованная схема фильтра. При передаче пустой символьной строки используется последняя использованная схема.
USEFILTER  |  Используется схема фильтра, заданная в параметре FILTERSCHEME.
Значение по умолчанию: 1

!!! example "Пример:"

    Сжать проект:compress
    /PROJECTNAME:C:\Projects\EPLAN\ESS_Sample_Project.elk
    /CONFIGSCHEME:config_scheme
    /FILTERSCHEME:filter_schemeИспользование последней схемы конфигурации: Параметр CONFIGSCHEME не указывается.compress
    /PROJECTNAME:C:\Projects\EPLAN\ESS_Sample_Project.elk
    /FILTERSCHEME:filter_scheme

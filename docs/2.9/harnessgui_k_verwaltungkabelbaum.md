# Управление проводным монтажом

Управление проводным монтажом осуществляется через определение проводного монтажа. Определение проводного монтажа идентифицирует проводной монтаж, а также содержит все его важные данные и изделия.

### Обработка данных проводного монтажа

Вы можете обрабатывать свойства проводного монтажа в различных местах:

* В диалоговом окне Свойства ++...++ можно обрабатывать данные для выделенного проводного монтажа. Самые важные свойства вводятся в полях в верхней области вкладки Проводной монтаж (напр., имя проводного монтажа или ОУ). Проводной монтаж дополнительно идентифицируется для обозначения устройства с помощью имени.
* В навигаторе проводного монтажа (диалоговое окно Проводной монтаж — <Имя проекта>) представлен обзор всех вариантов проводного монтажа открытого проекта. Здесь можно выбрать один или несколько вариантов проводного монтажа и его компонентов. С помощью всплывающего меню можно генерировать проводной монтаж или вызывать соответствующие диалоговые окна для обработки.

### Присвоение функций и соединений проводному монтажу

Присвоение функций и соединений определенному проводному монтажу осуществляется с помощью свойства Имя проводного монтажа (ид. 31143). В таблице свойств объектов для этого свойства указывается соответствующее имя проводного монтажа. В навигаторах, отображающих данные проводного монтажа, свойство Имя проводного монтажа можно применять как критерий фильтра и отображать в виде столбца в представлении в виде списка.

Посредством табличной обработки выбранным функциям и / или соединениям можно быстро и просто присваивать имена проводного монтажа или изменять их.

### Изделие

Для определения проводного монтажа в первой строке вкладки Изделие можно внести изделие узла проводного монтажа. Остальные строки этой вкладки предусмотрены для изделий (максимум 999), для которых по умолчанию отсутствуют соответствующие функции EPLAN.

Создание узла проводного монтажа в базе данных изделий необязательно. Во время проектирования также можно указывать изделия только на отдельных компонентах проводного монтажа (соединениях, кабелях, контактах штекера и т. д.).

### Отчеты

Содержащиеся в проекте данные изделий проводного монтажа можно перечислить в спецификации и вывести в соответствующих отчетах (напр., спецификации изделий, групповой спецификации изделий и т. д.). В диалоговом окне Настройки: Изделие можно определить, должны ли при этом учитываться только изделия в узле проводного монтажа или / и изделия компонентов проводного монтажа.

При этом необходимо учитывать следующее:

* Настройка Изделие в узле проводного монтажа учитывает изделия, внесенные в первую строку вкладки Изделие диалогового окна свойств определения проводного монтажа. Этим изделием может (но не должно) быть изделие узла в базе данных изделий.
* Настройка Изделие компонентов проводного монтажа учитывает изделия компонентов проводного монтажа. С одной стороны, этими изделиями могут быть изделия на главных функциях, для которых в диалоговом окне свойств в свойстве Имя проводного монтажа (ид. 31143) указано имеющееся имя проводного монтажа. С другой, ими могут быть остальные изделия на определении проводного монтажа (в диалоговом окне свойств в строках 2–1000).
* В диалогом окне Настройки: Изделие настройки для проводного монтажа связаны логическим оператором И с остальными настройками. Например, чтобы отобразить кабель в качестве компонента проводного монтажа, необходимо установить флажок Изделие компонентов проводного монтажа, а также флажок Изделие кабеля.

Данные изделий проводного монтажа можно учитывать также при выводе данных изготовления / маркировки. Аналогичные настройки (как для значений изделия) доступны в диалоговом окне Настройки: Экспорт данных изготовления / маркировка на вкладке Настройки.

# Режим-Свойства (общие)

***Режим-Свойства (общие)*** позволяет обрабатывать общие свойства распределенных функций одновременно в одной операции. При этом можно обрабатывать или общие свойства всех представлений функций вспомогательной функции, или общие свойства всех представлений функций главной функции. Также есть возможность изменять ОУ для всех представлений функций устройства общ. назначения.

!!! example "Пример:"

    Ваш проект содержит устройство***K1***. Устройство состоит из многополюсной главной функции***Катушка***и нескольких вспомогательных функций категорий***Замыкающий контакт***и***Размыкающий контакт***. Многополюсная главная функция существует дополнительно в однополюсном представлении и представлении функции обзора. С помощью режима "Свойства (общие)" можно одновременно обрабатывать функциональный текст для всех представлений функций главной функции.

За несколькими исключениями можно обрабатывать все свойства, доступные в диалоговом окне Свойства (общие) ++...++ в следующих вкладках.

* [Вкладка <Категория функции>](devicetaggui_r_bmdaten.md)
* [Вкладка <Категория функции> (Устройство)](devicetaggui_r_bmdaten.md)

Следующая таблица показывает, какие свойства на вкладках в режиме "Свойства (общие)" ***не*** могут обрабатываться.

Свойство |  Значение
---|---
Отображение перекрестных ссылок: Представление |  Представление отображения перекрестных ссылок.
Отображение перекрестных ссылок: количество строчек / колонок |  Количество строчек или столбцов для отображения перекрестных ссылок по строчкам или столбцам.
Маркер ревизии (из сравнения проектов) |  Выделение изменений ревизии
Свойство блока |  Свойства, установленные в соответствующем формате Свойство блока: Формат.
Свойство блока: Формат |  Определяет, какие свойства объединяются в соответствующем свойстве блока.
Сгенерировано автоматически |  Показывает, сгенерирована ли функция с помощью автоматического прогона (например, автоматически сгенерированный кабель).
Макрос |  Макрос-родитель функции.

Также в диалоговом окне Свойства (общие) ++...++ на следующих вкладках свойства в режиме "Свойства (общие)" ***не*** могут обрабатываться:

* Вкладка Отображение
* Вкладка Данные символа/функции
* Вкладка Формат

На этих вкладках можно обрабатывать только выбранное представление функции главной или вспомогательной функции. Все другие соответствующие представления функции не изменяются.

# Диалоговое окно XML-проект

Имеется (как минимум) один проект с расширением имени файла *.epj, который был ранее экспортирован.

* Проект > Организовать > Импортировать. Выберите в диалоговом окне Импортировать проект файл и щелкните по кнопке ++Открыть++.
* Вы открыли окно управления проектами. [Организовать] > Импортировать. Выберите в диалоговом окне Импортировать проект файл и щелкните по ++Открыть++.

В этом диалоговом окне определяются настройки для XML-импорта проекта.

Доступны следующие элементы диалогового окна:

### XML-файл

Введите здесь дисковод, каталоги и имя импортируемого XML-файла или щелкните по ++...++, чтобы выбрать файл.

Перейдите через Всплывающее меню > Вставить переменную пути в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно скопировать одну из доступных переменных пути, специфическую для EPLAN.

### Выполнить синхронизацию изделий

Установите этот флажок, если во время импорта необходимо выполнить автоматическую синхронизацию изделий. В этом случае указатель процесса показывает ход синхронизации.

### Целевой проект

Введите здесь дисковод, каталоги и имя проекта, в который требуется импортировать данные или щелкните по ++...++, чтобы выбрать файл.

Перейдите через Всплывающее меню > Вставить переменную пути в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно скопировать одну из доступных переменных пути, специфическую для EPLAN.

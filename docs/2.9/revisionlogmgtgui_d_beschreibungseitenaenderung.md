# Диалоговое окно Описание изменения страниц

Вы создали ревизию и открыли проект ревизии (расширение имени файла *.ell).

* В настройках проекта в диалоговом окне Настройки: Ревизия (отслеживание изменений) установлен флажок Всегда запрашивать описание изменения страниц. Вы выполнили изменения на странице, например, создание, удаление, переименнование (также с помощью нумерации), импортирование, формирование отчетов и т.д. В диалоговом окне Маркер ревизии обработайте или переименуйте текст маркировки и щелкните по ++OK++.
* Вы выполнили изменения на одной или нескольких страницах. В навигаторе страниц выделены соответствующие страницы. Сервисные программы > Управление ревизиями > Отслеживание изменений > Закрыть страницы.

В этом диалоговом окне можно описать изменения, выполненные в проекте ревизии на странице. Введенные тексты анализируются для вывода на печать и/или графическую страницу обзора ревизий.

Если вы хотите затем обработать введенные здесь тексты, можно изменить их в диалоговом окне [Дополнительно обработать данные ревизии](revisionlogmgtgui_d_revisionsdatennachbearbeiten.md).

Доступны следующие элементы диалогового окна:

### Индекс ревизии

Это поле предусмотрено для текста, с чьей помощью можно пометить изменение страницы. Индекс ревизии может состоять из любых символов. С помощью кнопки ++...++ откройте диалоговое окно [Выбрать индекс ревизии](revisionlogmgtgui_d_indexauswaehlen.md). Там можно выбрать уже введенный индекс ревизии или внести изменения при необходимости.

Если в диалоговом окне [Настройки: Отслеживание изменений (нумерация индекса ревизии)](revisionlogmgtgui_d_einstellungenseitenindex.md) задана схема нумерации, то новое значение индекса ревизии автоматически определяется и предлагается из схемы нумерации.

### Описание

В этом поле можно прокомментировать выполненные изменения страницы.

### Причина изменения

Это поле предусмотрено для ввода причин изменений.

### Пользователь

В этом поле автоматически вводится пользователь, который описал изменение страниц. Его нельзя изменить.

### Область обработки

В этом поле отображается текущая активная область обработки. Запись изменить невозможно.

### Дата

В этом поле автоматически вводится дата создания описания. Ее нельзя изменить.

### Сгенерировать отчет при закрытии страниц

Если этот флажок установлен, то при закрытии страниц выбранные страницы отчета (например, содержание) обновляются. Если при создании отчета на странице отчета производится изменение (например, в Содержании, в Обзоре ревизии и т. д.), то эта страница закрывается с тем же индексом ревизии, а пометка страницы "План" удаляется. Выбранные страницы отчета также обновляются, если они не помечены как "План".

Если этот флажок не установлен, то страницы отчета не обновляются.

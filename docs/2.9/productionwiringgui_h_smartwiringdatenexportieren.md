# Экспорт данных изготовления для EPLAN Smart Wiring

Данные изготовления для EPLAN Smart Wiring экспортируются в файл EPDZ через отдельный интерфейс. Файл EPDZ содержит свойства проекта, свойства страницы и свойства пространства листа, а следовательно, полную структуру проекта, а также данные о соединениях и свойствах кабелей для маршрутизируемых соединений. Файл EPDZ также содержит файл E3D с трехмерными графическими данными для каждого пространства листа.

Условия:

* Вы открыли проект.
* Навигатор пространства листа открыт, и открыто пространство листа.
* Сгенерированы маршрутизируемые соединения.

1. Выберите пункты меню Сервисные программы > Данные изготовления > EPLAN Smart Wiring.
2. В диалоговом окне EPLAN Smart Wiring выберите имя файла и каталог вывода для файла EPDZ.
3. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    В настроенном каталоге вывода файл EPDZ сгенерируется с заранее выбранным именем. Этот файл будет введен в EPLAN Smart Wiring.

# Вкладка Присвоение класса

Сервисные программы > Изделие > Управление > [Дополнительно] > Импортировать. В поле Тип файла выбрана запись для версии eCl@ss, а в поле Имя файла выбран XML-файл с наборами данных изделий и нажата кнопка ++OK++. В диалоговом окне Импорт eCl@ss нажмите кнопку ++...++ рядом с полем Настройки и выберите вкладку Присвоение класса.

На этой вкладке можно присваивать сегменты eCl@ss группам продуктов базы данных изделий EPLAN.

Обзор основных элементов диалогового окна:

### Сегмент eCl@ss / Главная группа eCl@ss / Группа eCl@ss / Подгруппа eCl@ss

В этих полях укажите свойства изделия, которым присваиваются иерархические уровни классов BMEcat, щелкнув по ++...++ и выбрав в диалоговом окне Выбор свойств соответствующее свойство. (Присвоение выполняется по умолчанию между пунктами "Атрибуты [1]" и "Атрибуты [4]").

### Таблица

Расположенная в нижней части таблица содержит следующие столбцы:

* Ид. класса: Содержит 8-значный Ид. eCl@ss класса.
* Имя класса: Содержит имя класса на языке (по умолчанию) файла xml. Язык файла xml указан в верхнем колонтитуле этого файла. Если eCl@ss не представлен на этом языке, используется исходный язык.
* Главная группа продуктов: Раскрывающийся список для выбора главной группы продуктов базы данных изделий EPLAN. По умолчанию установлен параметр "Электротехника".
* Группа продуктов: Раскрывающийся список для выбора группы продуктов базы данных изделий EPLAN. По умолчанию установлен параметр "Общее".
* Подгруппа продуктов: Раскрывающийся список для выбора подгруппы продуктов базы данных изделий EPLAN. По умолчанию установлен параметр "Не определено".

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

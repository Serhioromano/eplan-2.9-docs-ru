#  Компоновка электрошкафа 2D

Наряду с созданием схем соединений с соответствующими схемами и списками, программа также позволяет проектировать устройства электрошкафов и монтажные платы. При этом монтажные платы можно создавать только на страницах проекта типа "Компоновка электрошкафа".

Для обозначения функциональных элементов монтажной платы используются графические объекты или так называемые "объединенные макросы". которые надо создать и сохранить в основных данных базы данных изделий.

В дополнение к монтажной плате возможно создание легенд функциональных элементов, которые различным образом встраиваются в проект.

# Создать схему экспорта / маркировки

Схемы экспорта / маркировки / схемы для внешних отчетов позволяют сохранить и повторно использовать в схеме элементы формата и настройки для экспорта данных изготовления / маркировки или для внешних отчетов. Благодаря этому можно постоянно выводить данные с аналогичными настройками.

Условие:

Вы открыли проект.

1. Сервисные программы > Данные изготовления > Экспорт / маркировка
2. В диалоговом окне Экспортировать данные изготовления / Вывести маркировку рядом с полем Настройки щелкните по кнопке ++...++.
3. В диалоговом окне Настройки: Экспорт данных изготовления / маркировка щелкните по кнопке ![](images/all_new_as.png){: .ui-icon } (Создать).
4. Выделите в диалоговом окне Тип отчета требуемый тип отчета и щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Тип отчета указывает, какая группа одинаковой информации анализируется для файла экспорта / маркировки / внешнего файла отчетов.

5. Введите в диалоговом окне Новая схема имя и описание и щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Тип отчета выводится в диалоговом окне Настройки: Экспорт данных изготовления / маркировка в поле Тип отчета.

!!! info "Для сведения:"

    Теперь можно указать [настройки вывода *.txt](labelgui_h_textausgabe.md) или [настройки вывода Excel](labelgui_h_exceldatei.md).

# Чертить точки обозначения соединения

Соединения получают дополнительные свойства посредством точек определения соединений.

1. Выберите пункты меню Вставить > Точка определения соединения.
2. Разместите точку определения соединения на линии автоматического соединения.
3. В диалоговом окне Свойства ++...++ во вкладке [Точка определения соединения](devicetaggui_r_vdp.md) введите свойства соединения (а при необходимости и другие свойства в других вкладках).
4. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Свойства присваиваются соединению, проходящему через точку определения соединения.

!!! tip "Совет:"

    * Вы можете добавлять одновременно несколько точек определения соединения, удерживая нажатой левую клавишу мыши и протягивая курсор через несколько линий автоматического соединения. В этом случае на каждой пересеченной линии автоматического соединения будет размещена точка определения соединения.
    * Посредством линии определения кабеля можно одновременно установить несколько точек определения соединений. На всех перекрещивающихся соединениях устанавливаются точки определения. Затем можно будет удалить отдельные точки определения из их числа.

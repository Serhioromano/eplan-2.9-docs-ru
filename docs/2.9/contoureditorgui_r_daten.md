# Вкладка Данные (Точка определения контура)

Вы открыли проект. Открыт контур типа "Выдавить контур".

* В графическом редакторе выделена точка определения контура. Обработать > Свойства. В диалоговом окне Точка определения контура: ++...++ выберите вкладку Данные.
* Вставить > Логика > Направляющая рейка / Направляющая рейка с выравниванием / Монтажная поверхность / Монтажная поверхность ЧУ / Размер поля. Разместите точку определения контура, и затем дважды щелкните по ней. В диалоговом окне Точка определения контура: ++...++ выберите вкладку Данные.

На этой вкладке задаются свойства точки определения контура.

Обзор основных элементов диалогового окна:

### Счетчик

Здесь отражается счетчик текущего логического элемента. Первый счетчик предварительно задан как "1", но может быть изменен на любое значение.

### Подсчетчик

Здесь отражается подсчетчик текущего логического элемента. Если присутствует только один элемент выбранного типа (направляющая рейка, монтажная поверхность и т. д.), то поле недоступно и не может быть обработано. Если на странице несколько элементов одного типа, то счетчик, начинающийся с "1", считает по нарастающей.

# Спецификации

Спецификация в EPLAN ***не*** является частью базы данных изделий, а относится к специфическим для проекта данным; поэтому навигатор спецификаций вызывается из меню Данные проекта и имеет представления в виде списка и структуры дерева.

При открытии навигатора спецификаций всегда считываются текущие данные из текущего проекта. В представлении структуры списка / дерева отображаются все ОУ с изделиями, ОУ без изделий, изделия по точкам определения и покупные изделия.

При помощи этого навигатора можно централизованно обрабатывать в ***одном*** месте системы данные изделий проекта. Возможно следующее:

* Представление в виде дерева ориентировано на изделия, т.е. здесь по умолчанию отображаются только такие ОУ, которые уже имеют изделие.
* Представление в виде списка ориентировано на устройства, и по отдельным ОУ можно выводить различную информацию. Настроить отображение информации можно через Всплывающее меню > Конфигурировать представление.
* В обоих представлениях можно использовать пункты меню Всплывающее меню > Настройки для определения (дополнительных) изделий, которые должны учитываться в окне отображения.
* В этом диалоговом окне навигатора можно вручную вводить или изменять номер изделия. Также можно изменять количество и выполнять выбор изделий.
* Кроме того, возможен выбор устройства в навигаторе спецификаций.

# Операция: XPamSelectPart

**Параметр** |  **Описание**
---|---
DatabaseId |  Идентификатор базы данных.
SelectedPartsCount |  Выводит количество выбранных изделий.
PartNr(nr) |  Выводит номер выбранных изделий, при этом nr = 1, 2, ...
Variant(nr) |  Выводит варианты выбранных изделий, при этом nr = 1, 2, ...
PartType(nr) |  Выводит тип выбранных изделий, при этом nr = 1, 2, ...

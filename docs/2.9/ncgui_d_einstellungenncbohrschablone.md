# Диалоговое окно Настройки: Экспорт шаблона сверления

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт шаблона сверления
* Сервисные программы > Данные изготовления > Механическая обработка > Шаблон сверления. Рядом с полем Машина нажмите ++...++.

В этом диалоговом окне установите настройки для экспорта шаблонов сверления.

Обзор основных элементов диалогового окна:

### Машина

В комплекте поставки содержится определение машины с названием "Стандарт". Если в качестве схемы сгенерирована другая машина, можно выбрать из раскрывающегося списка другие определения машины и работать с ними.

Через расположенную рядом [панель инструментов](schemeconfig_m_schemaschaltflaechen.md) можно создавать новые схемы, сохранять изменения в схеме, копировать или удалить схему. Кроме того, схемы можно импортировать и экспортировать.

### Описание

В этом поле дается описание, которое было введено в диалоговом окне Новая схема в поле Описание для машины.

### Вкладки

Для конфигурации параметров экспорта шаблонов сверления предусмотрены следующие вкладки:

* [Вкладка Общее](ncgui_r_allgemeinbohrschablone.md)
* [Вкладка Маркировка](ncgui_r_beschriftungbohrschablone.md)
* [Вкладка Исключения при сверлении](ncgui_r_bohrausnahmen.md)

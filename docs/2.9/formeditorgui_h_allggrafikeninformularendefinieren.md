# Определить общую графику в формах

Свойство формы Присвоение индекса графике необходимо, чтобы, например, в спецификации клеммника могла быть выведена (фиксированная) графика в области данных.

Условия:

* Вы открыли проект.
* В редакторе форм открыта та форма, свойства которой требуется обработать (Сервисные программы > Основные данные > Форма > Открыть > [Открыть]).
* Открыто диалоговое окно Свойства формы — <Имя формы> (Всплывающее меню > Свойства на форме в диалоговом окне навигатора Страницы — <Имя проекта>).

1. В диалоговом окне Свойства формы — <Имя формы> щелкните кнопку ++...++ в столбце Значение свойства Присвоение индекса графике.
2. В диалоговом окне Присвоение индекса графике щелкните ![](images/all_new_as.png){: .ui-icon } (Создать), чтобы генерировать новую строку в таблице присвоений.
3. При необходимости задайте в поле Индекс другое значение. Предварительная настройка всегда является <следующим номером>.
4. Щелкните в поле Имя элемента по ++...++.
5. Выберите в диалоговом окне Выбор символов символ, графику которого вы хотите присвоить индексу.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Соответствующие данные будут применены в полях Библиотека символов и Описание.

7. В поле Вариант символа выберите требуемый вариант из раскрывающегося списка всех имеющихся вариантов текущего символа.
8. Выполните аналогичные действия для всех других присвоений.
9. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Присвоения сохраняются и могут быть вставлены с помощью заполнителей в форме.

В диалоговом окне Свойства формы — <Имя формы> присвоения выводятся в столбце Значение следующим образом:

<Номер индекса>;<Библиотека символов>;<Номер символа>; <Вариант>|<Номер индекса>;<Библиотека символов>; <Номер символа>;<Вариант>|...

!!! example "Пример:"

    1;SPECIAL;307;0|2;IEC_EPLAN_D;12;1

!!! note "Замечание:"

    Обратите внимание на то, что на просмотр выводятся только символы ***сохраненных*** библиотек символов.

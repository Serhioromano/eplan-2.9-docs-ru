# Диалоговое окно Управление слоями — <Имя проекта>

Вы открыли проект. Если открыто несколько проектов, выделите в навигаторе страниц проект, слоями которого нужно управлять. Данные проекта > Управление слоями.

В этом диалоговом окне можно просмотреть и изменить свойства стандартных слоев EPLAN для проекта. Кроме того, можно создавать собственные слои или удалять их, а также экспортировать или импортировать конфигурацию управления слоями. В строке заголовка диалогового окна выводится имя текущего выбранного проекта.

Диалоговое окно можно прикрепить к любым другим элементам интерфейса как навигатор. Изменения в диалоговом окне сохраняются сразу.

!!! note "Замечание:"

    Перед изменением настройки управления слоями для проекта следует сначала экспортировать файл конфигурации. Благодаря этому выполняется резервирование стандартных настроек, и стандартные значения для слоев можно будет позднее восстановить импортированием.

Обзор основных элементов диалогового окна:

### Дерево

В представлении в виде дерева слои выводятся в виде иерархической структуры. Эта структура создается на основе описания слоя.

Если вы выделите один или несколько слоев или узлов в дереве, отображение слоев в таблице отфильтруется соответственно вашему выбору.

### Таблица

В таблице выводятся все слои, выделенные в дереве. Дважды щелкните по соответствующему заголовку столбца, чтобы отсортировать слои в восходящем или нисходящем порядке.

Как в дереве, так и в таблице возможен многократный выбор слоев. В зависимости от вида слоя таблица содержит следующие свойства:

### Слой

Здесь выводится имя слоя EPLAN. Имена стандартных слоев изменить невозможно.

При создании собственных слоев EPLAN автоматически вводит здесь имя (НОВЫЙ_СЛОЙ_1, НОВЫЙ_СЛОЙ_2 и т. д.). Эти имена можно изменить. Имя слоя должно быть однозначным для каждого слоя. Имена слоев могут иметь длину до 255 символов. Благодаря этому они совместимы с программами CAD.

### Описание

Здесь выводится описание слоя EPLAN. Описание слоев не обязательно должны быть однозначными. На базе описания слоя в левой части диалогового окна генерируется представление в виде дерева. Текст, находящийся в описании уровня после точки, считается подструктурой. Описания стандартных слоев EPLAN изменить невозможно.

При создании собственных слоев описание перенимается от выделенного слоя или – если выделена узловая точка – от первого слоя под узловой точкой. Эти описания слоев можно изменить. Кроме того, можно перевести описания ***определенных пользователем*** слоев. Чтобы описание слоя учитывалось при автоматическом переводе, в [настройках проекта для перевода](translategui_r_einstellungumfang.md) в разделе "Проект" необходимо установить флажок для свойства Слой: описание.

### Тип линии

Здесь выводится тип линии, с помощью которого представлены элементы соответствующего слоя. Выберите требуемый тип линии из раскрывающегося списка.

### Масштаб линии

Для прерванных линий здесь указывается длина, после которой линия повторяется по образцу. Выберите значение из раскрывающегося списка или введите десятичное значение вручную.

### Толщина линии

Здесь указывается толщина линии при черчении элементов на соответствующем слое. Выберите значение из раскрывающегося списка или введите десятичное значение вручную.

### Цвет

Здесь выводится цвет, который используется для вывода элементов соответствующего слоя. Нажав кнопку ++...++, можно открыть диалоговое окно [Выбор цвета](mf_d_farbauswahl.md) и выбрать в нем другой цвет.

### Прозрачность

Для трехмерных слоев введите здесь степень прозрачности. Значение 100% делает объект полностью прозрачным.

### Размер шрифта

Здесь указывается размер текстов свойств соответствующего слоя. Выберите значение из раскрывающегося списка или введите десятичное значение вручную.

### Выравнивание

Здесь указывается выравнивание, с которым отображаются тексты свойств соответствующего слоя. Выберите значение из раскрывающегося списка.

### Угол

Здесь указывается в каком углу отображаются тексты свойств соответствующего слоя. Выберите значение из раскрывающегося списка или введите десятичное значение вручную.

### Межстрочный интервал

Данное поле позволяет задать вертикальный интервал между строками текста, т. е. расстояние от нижнего края одной строки до нижнего края другой. Интервал между строками зависит от выбранного вида шрифта. Выберите запись из раскрывающегося списка или введите значение вручную. При этом значение '1' соответствует одинарному интервалу между строками, значение '2' —двойному интервалу между строками и т. д. Одинарный интервал между строками составляет 1,9-кратный размер шрифта (высота прописных букв 'M').

### Интервал между абзацами

Здесь задается вертикальный интервал между двумя присоединенными свойствами, расположенными при отображении в одной общей группе. Выберите значение из раскрывающегося списка или введите десятичное значение вручную.

!!! note "Замечание:"

    Учтите, пожалуйста, что при присоединенных текстах настроенный интервал между строками добавляется к интервалу между абзацами.

### Рамка текста

Выберите из раскрывающегося списка, должны ли текст или обозначения устройства, находящиеся на соответствующем уровне, помещаться в рамку и какой формы должна быть рамка: прямоугольной, эллиптической или овальной. При этом размер рамки зависит от текста. Выберите настройку Нет, чтобы для текста или обозначений устройств в данном слое не отображались рамки текста.

### Видимый

Установите этот флаг, если требуется вывести в графическом редакторе элементы, которые находятся на соответствующем слое. Если флаг снят, эти объекты не выводятся.

!!! note "Замечание:"

    В EPLAN имеется возможность просмотра невидимых элементов на экране. Для этого в настройках следует установить флажок Отображать невидимые элементы (Параметры > Настройки > Пользователь > Графическая обработка > 2D). Кроме того, с помощью этих пользовательских настроек можно настроить особый цвет для невидимых элементов.

### Печать

Установите этот флаг, если при печати требуется показать элементы, которые находятся на соответствующем слое. Если флаг снят, они не печатаются.

### Заблокировано

Этот флажок релевантен только для импорта и экспорта DXF/ DWG. В AutoCAD можно блокировать определенные слои для обработки. При импорте заблокированного слоя этот флажок автоматически снимается, тем не менее слой можно обрабатывать в EPLAN. При экспорте слоев с помощью данного флажка устанавливается, заблокированы ли присоединенные слои для обработки в AutoCAD.

### Фон

Установите этот флажок, если требуется переместить в фон элементы, которые находятся на соответствующем слое. Элементы в настроенном таким образом слое всегда лежат в самом низу, то есть даже позади линий автоматического соединения. Таким образом можно определить, например, вставленный графический файл или импортированный файл DXF / DWG как фоновое изображение. Элементы можно обрабатывать, только если активен параметр Обработать фон.

Для слоя EPLAN705, Графика.Фоновая картинка флажок установлен по умолчанию.

### Масштабируемый

Установите этот флажок, если хотите, чтобы элементы в этом слое масштабировались вместе с другими при масштабировании. Такое масштабирование может происходить, например, если вы изменяете масштаб страницы или вставляете макрос на страницу, которая имеет другой масштаб, чем тот, какой использовался при создании макроса. При масштабировании толщина линий, масштаб линий и размеры шрифта для линий, текстов и размеров (тексты и стрелки) будут адаптироваться.

### Трехмерный слой

Установите этот флажок, если хотите выбрать этот слой в пространствах листа и использовать его. Для слоев, расположенных ниже узла 'Трехмерная графика', флажок установлен по умолчанию.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

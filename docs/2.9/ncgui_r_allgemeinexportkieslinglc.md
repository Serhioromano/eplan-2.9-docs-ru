# Вкладка Общ. (экспорт Rittal - Perforex LC / Secarex)

* Параметры > Настройки > Фирма > Механическая обработка > Экспорт Rittal - Perforex LC / Secarex. Выберите вкладку Общ.
* Сервисные программы > Данные изготовления > Механическая обработка > Rittal - Perforex LC / Secarex. Рядом с полем Машина нажмите ++...++. Выберите вкладку Общ.

На этой вкладке установите общие настройки для экспорта ЧУ на центр лазерной обработки Perforex LC / Secarex производителя Rittal.

Обзор основных элементов диалогового окна:

### Расстояние начального отверстия

Расстояние от расточенного отверстия лазерного пучка внутрь до обрабатываемого края контура монтажного отверстия. Лазер начинает процесс фрезерования в начальном отверстии, а затем входит в контур. После этого он покидает контур и возвращается к начальному отверстию, чтобы завершить процесс.

Стандартное значение: 1,5 мм.

### Префикс

Здесь вы можете указать символ или строку символов, которые в экспортируемом файле ЧУ ставятся перед записями для генерирования имени контура Perforex. По умолчанию используется настройка "LC_".

Префикс для контуров действует также для идентификатора поля Функциональный элемент / Изменяемый функциональный элемент - Имя на вкладке [Идентификатор поля](ncgui_r_feldbezeichner.md), если соответствующая настройка для генерирования установлена на "Автоматически".

### Каталог вывода

Выберите каталог, в котором следует разместить экспортируемые файлы. Рабочее место в мастерской возле машины должно иметь доступ к этому каталогу.

С помощью пункта всплывающего меню Вставить переменную пути перейдите в диалоговое окно [Выбрать переменную пути](modaldialogsdb_d_pfadvariablen.md), из которого можно будет скопировать одну из доступных переменных пути.

### Генерировать имена файлов

С помощью записи "Из свойств" при экспорте ЧУ имя файла составляется из свойств пространства листа и проекта, а также свободно определяемого разделителя. Используя кнопку ++...++, откройте диалоговое окно Формат: Свойство для конфигурации имени файла.

### Генерировать подкаталоги

В этом раскрывающемся списке определяется стандартный порядок создания подкаталогов при экспорте файлов ЧПУ.

* Нет: Если выбран данный параметр, при экспорте подкаталоги не генерируются.
* Из свойств: Если выбран этот параметр, имя подкаталога составляется из свойств пространства листа и проекта, а также свободно определяемого разделителя. При нажатии кнопки ++...++ открывается диалоговое окно Формат: Свойство для конфигурации имени каталога. Воспользуйтесь, например, элементом формата "Разделитель каталога", чтобы создать структуру каталога с несколькими подкаталогами, расположенными друг под другом.

Таблица номеров материала

При экспорте Rittal Perforex LC / Secarex доступны до 10 различных номеров материала, которые можно присвоить обрабатываемым материалам. Все присвоенные материалы на вкладке Инструменты можно также назначить для отдельных инструментов. Вводится соответствующий номер материала.

При экспорте этот номер будет выбран и перенесен в машину. Эти настройки необходимо синхронизировать с настройками на машине. По умолчанию первые три номера материала заданы следующим образом:

* Номер материала 1: таль
* Номер материала 2: высококач. сталь
* Номер материала 3: алюминий.

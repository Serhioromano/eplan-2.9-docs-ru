# Начертить точку определения изделия

С помощью точки определения изделия можно сохранить изделия, не присвоенные никакому устройству, на графике в схеме соединений. Каждой точке определения изделия (как и любой главной функции) могут быть присвоены максимум 50 изделий.

Чтобы передать изделия, указанные на точке определения изделия, в спецификацию изделий или в групповую спецификацию изделий с сохранением структуры, можно указать на точке определения изделия свойства Видимое ОУ и Полное ОУ. Таким образом изделия присваиваются определенному структурному идентификатору. Тем не менее точки определения изделия не являются устройствами и не перечисляются в навигаторе устройств и в списке обозначений устройств.

Условие:

Вы открыли проект и находитесь на странице схемы соединений.

1. Выберите пункты меню Вставить > Точка определения изделия.

!!! info "Для сведения:"

    Символ точки определения изделия появится рядом с курсором.

2. Щелкните по схеме соединений, чтобы разместить точку определения изделия.
3. В диалоговом окне "Свойства" на вкладке Изделие выберите нужное изделие (и при необходимости введите данные для ссылки на изделие).
4. На вкладке Точка определения изделия укажите структурный идентификатор.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выбранное изделие будет присвоено точке определения изделия.

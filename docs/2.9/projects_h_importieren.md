# Импортировать проекты

Можно импортировать проекты, предварительно экспортированные в формате XML с расширением типа имени файла epj.

Навигатор страниц: Проект > Организовать > Импортировать.

Управление проектами: [Организовать] > Импортировать.

1. В диалоговом окне Импортировать проект выберите требуемый проект.
2. Щелкните по кнопке ++Открыть++.
3. В поле XML-файл диалогового окна XML-проект проверьте правильность выбора файла проекта и щелкните, если необходимо, по кнопке ++...++, чтобы открыть другой проект для импорта.
4. Активируйте флажок Выполнить синхронизацию изделий, если при импорте необходимо автоматически выполнить синхронизацию изделий.
5. При необходимости введите в поле Целевой проект другое имя для импортируемого проекта.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выполняется импорт проекта, указатель процесса показывает ход импорта. Если вы активировали флажок Выполнить синхронизацию изделий, после импорта появится дополнительный указатель процесса, отражающий ход выполнения.

# Импортировать свойства с внешней обработкой

Условие:

Вы открыли проект и экспортировали данные через меню Сервисные программы > Внешняя обработка свойств > Экспортировать данные и для поля Вид вывода ***не*** выбрали параметр Обработать во внешнем приложении и считать обратно. Вы обработали данные в выбранном внешнем приложении и сохранили файл. Откройте в EPLAN проект, в который следует импортировать данные.

1. Выберите пункты меню Сервисные программы > Внешняя обработка свойств > Импортировать данные.
2. В диалоговом окне Открыть выберите каталог, в котором находится импортируемый файл, выделите его и нажмите ++Открыть++.

!!! info "Для сведения:"

    Данные импортируются.

!!! note "Замечание:"

    При импорте обратите внимание на следующее: если в экспортируемый файл перед импортом добавляется новый объект (новая функция, новая страница и т. д.), эту новую строку в файле Excel также необходимо указать в определении имени "EPL_VALUES". Для этого в Excel откройте диалоговое окно Определить имя, например комбинацией клавиш [Ctrl] \+ [F3]. Здесь необходимо настроить соответствующим образом область данных переменной "EPL_VALUES" в поле Относится к, чтобы при импорте учитывать новый объект.

### Создать новые функции при импорте из Excel

Условие:

Вы открыли проект.

1. Выберите следующие пункты меню: Сервисные программы > Внешняя обработка свойств > Экспортировать данные
2. В поле Настройки диалогового окна Внешняя обработка щелкните ++...++.
3. В диалоговом окне Настройки: Внешняя обработка при необходимости выберите другую схему или создайте схему. Независимо от выбранной схемы необходимо убедиться, что для типа объекта задана настройка Все функции.
4. Щелкните по кнопке ++OK++.
5. В диалоговом окне Внешняя обработка выберите опцию Обработать во внешнем приложении и считать обратно.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Данные экспортируются в Excel.

!!! info "Для сведения:"

    Excel открывается вместе с экспортированными свойствами функций.

!!! example "Пример:"

    7. Чтобы создать новые функции, скопируйте, например, ячейки существующих функций и измените записи для ***ОУ (полное)*** и ***ОУ (видимое)***. (Эти записи выделены на нижнем рисунке в правой области с помощью рамки.)
    8. В заключение измените запись в каждом первом столбце функций, обнулив число после косой черты ("/").

!!! example "Пример:"

    9. Чтобы новые функции сделать доступными для проекта EPLAN, необходимо расширить область значений EPLAN в Excel. Нажмите ++Ctrl++ \+ [F3].

!!! info "Для сведения:"

    Откроется диалоговое окно Определить имя.

10. Щелкните "EPL_VALUES" и измените запись в поле Относится к на текущую нижнюю правую строку, например сверху на "=EplSheet!$A$3$L$7".
11. Щелкните по кнопке ++OK++.
12. Сохраните таблицу и закройте Excel.

!!! info "Для сведения:"

    Выполняется экспорт данных.

!!! info "Для сведения:"

    EPLAN открывает диалоговое окно запросом, должны ли импортироваться данные.

13. Щелкните кнопку ++Да++.

!!! info "Для сведения:"

    Созданные в Excel функции импортируются и как неразмещенные функции добавляются в навигатор устройств.

# Опубликовать проекты

Вы можете опубликовать проект, экспортировав его в файл типа EPDZ. При помощи предоставляемого EPLAN мобильного приложения ***EPLAN View App*** можно отображать содержания проектов через Dropbox на мобильных устройствах (например, iPad). Файл EPDZ содержит в Запакованном виде всю структуру проекта (свойства проекта, свойства страницы, свойства пространства листа, свойства функции). Для каждой страницы проекта имеется файл с двухмерной векторной графикой (*.svg), а для каждого пространства листа — файл с трехмерными графическими данными (*.e3d).

Условие:

Вы открыли проект.

1. Выберите пункты меню Проект > Опубликовать.
2. Выберите в диалоговом окне Опубликовать схему с настройками для экспорта. Если схема отсутствует, с помощью кнопки ++...++ можно вызвать диалоговое окно Настройки: Опубликовать и генерировать там новую схему.
3. Если требуется сохранить файл EPDZ под именем, отличающемся от имени проекта, в поле Файл EPDZ введите новое имя файла или при помощи кнопки ++...++ выберите имя существующего файла, который будет перезаписан.
4. Выберите в групповом поле Средство экспорта, следует ли сохранить файл EPDZ на носителе данных или отправить его по электронной почте.

!!! info "Для сведения:"

    При публикации на носителе данных файл EPDZ можно сохранить на устройстве USB или записать непосредственно на CD / DVD.

!!! info "Для сведения:"

    При публикации в виде сообщения электронной почты у вас есть возможность отправить файл EPDZ непосредственно на мобильное устройство.

5. Если вы хотите сохранить файл на носителе данных, укажите в диалоговом окне Опубликовать в поле Каталог вывода место хранения для экспортируемого файла EPDZ.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выполняется сохранение экспортированного проекта *.epdz в целевом каталоге или отправка в качестве электронного письма.

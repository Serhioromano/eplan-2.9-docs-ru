# Диалоговое окно Выбрать величину шага

Открыта страница в графическом редакторе. Параметры > Величина шага.

Величина шага определяет, насколько далеко перемещается курсор в требуемом направлении при нажатии клавиши со стрелкой. Она настраивается и активируется исходя из текущей позиции курсора.

Обзор основных элементов диалогового окна:

### Текущая величина шага

При вызове диалогового окна здесь выводится текущая настроенная величина шага. Занесите требуемую величину шага в поля X и Y.

### Использовать графические координаты

Установите этот флаг, чтобы в полях списков отображалась величина шага для ***графических координат***. Величина шага задается в чертежных единицах (мм или дюйм).
Снимите этот флажок, чтобы вывести величины шага для ***логических координат***. Величина шага задается в шагах сетки.

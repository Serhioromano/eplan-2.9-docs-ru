# Вкладки Многополюсный / Однополюсный (настройки: Кабели (кабельные соединения))

Вы открыли проект. Параметры > Настройки > Проекты > 'Имя проекта' > Устройство > Кабель (кабельные соединения). Выберите вкладку Многополюс или Однополюс.

На этих вкладках указываются предварительные настройки, которые отображаются на точках определения соединения при линиях определения кабеля и экранированиях. Эти настройки можно задать отдельно для многополюсного и однополюсного представлений.

Обзор основных элементов диалогового окна:

Групповые поля Символ (графика) точек определения соединения для линии определения кабеля / экранирования

Установленный в групповом поле Символ (графика) точек определения соединения для линии определения кабеля используется для точек определения соединений, которые размещаются при черчении ***линий определения кабелей*** или при автоматическом генерировании ***кабелей***.

Установленный в групповом поле Символ (графика) точек определения соединения для экранирования используется для точек определения соединений, которые размещаются при черчении ***экранирований***.

### Номер / Имя

В этом поле выводятся имя и номер текущего символа; щелкните по кнопке ++...++, чтобы открыть диалоговое окно [Выбор символа](xsbgui_d_symbolauswahl.md) и при необходимости выбрать другой символ.

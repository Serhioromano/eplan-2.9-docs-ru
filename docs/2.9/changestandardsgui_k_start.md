# Изменение стандарта

Часто необходимо составить проект и адаптировать его для международного рынка и для конкретной страны. В EPLAN существует возможность адаптировать проект к значениям другого стандарта:

* При создании нового проекта из существующего проекта, если при этом проект адаптируется к значениям другого стандарта, первоначальный проект не изменяется.
* Страницы проекта можно адаптировать к другой рамке. Данные проекта будут таким образом представлены в горизонтальном или вертикальном формате.
* Можно заменить библиотеки символов.
* Если вы не хотите вести несколько библиотек символов, можно поменять и отдельные символы.
* Для отчетов можно заменять формы.
* Чтобы адаптировать наименования и названия к целевому стандарту, выберите другой шаблон.
* Для изменения структуры проекта используйте другой шаблон.
* Обозначения вывода устройства можно отключить.

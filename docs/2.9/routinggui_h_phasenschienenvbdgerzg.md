# Генерировать соединения фазовых шин

По этому методу определенные в схеме соединений многополюсные соединения преобразовываются в соединения фазных шин. После этого данный вид соединений не будет учитываться при расчете разводки и при сборке проводов.

Чтобы спроектировать фазную шину на чертеже монтажных поверхностей, можно присвоить соответствующим соединениям фазной шины ОУ или изделие. После этого фазная шина становится устройством, и ее можно разместить из трехмерного навигатора монтажных поверхностей.

Условие:

Открыто одно пространство листа.

1. Выберите пункты меню Данные проекта > Соединения > Генерировать соединения фазовых шин.

!!! info "Для сведения:"

    Вам предлагается выбрать размещения изделий, которые необходимо преобразовать в соединения фазовых шин.

!!! note "Замечание:"

    При выборе функциональных элементов учитываются только размещения изделий со следующими определениями функции:

* Размещение изделия, стандартный функциональный элемент
* Размещение изделия, карта ПЛК
* Размещение изделия, черный ящик

2. Опишите вокруг выбранных функциональных элементов прямоугольник.

!!! info "Для сведения:"

    В диалоговом окне Генерировать соединения фазовых шин перечисляются все соединения с видом представления "Многополюсный" и "Трехмерный чертеж монтажных поверхностей", источником и целью которых является один из выбранных ранее функциональных элементов (например, соединение с определением функции "Жила / провод").

### Преобразовать соединения в соединения фазных шин

1. Выделите в диалоговом окне Генерировать соединение фазных шин все соединения, которые необходимо преобразовать в соединение фазных шин.
2. Выберите пункт всплывающего меню Генерировать соединение фазных шин.

!!! info "Для сведения:"

    Все соединения с установленными флажками будут преобразованы в соединения фазовых шин и получат новое определение функции "Соединение фазовых шин".

!!! info "Для сведения:"

    Все свойства этого соединения будут удалены.

!!! info "Для сведения:"

    Из соединений с видом представления "Трехмерный чертеж монтажных поверхностей" возникают немаршрутизированные соединения.

!!! info "Для сведения:"

    Возникшие таким образом соединения фазных шин исключаются из расчетов при разводке и при сборке проводов.

### Присвоить фазную шину

1. Для присвоения изделия определенному соединению фазной шины выберите во всплывающем меню пункт Присвоить фазную шину.
2. Введите в поле ОУ ОУ для соединения фазной шины или выберите его из раскрывающегося списка или при помощи кнопки ++...++ в диалоговом окне Выбор ОУ.
3. Откройте в поле Изделие с помощью кнопки ++...++ выбор изделия, чтобы присвоить соединению фазной шины изделие, и нажмите кнопку ++OK++.

!!! info "Для сведения:"

    ОУ выводится в диалоговом окне Генерировать соединения фазовых шин.

!!! info "Для сведения:"

    Фазная шина отображается в трехмерном навигаторе монтажных поверхностей в качестве неразмещенного устройства и может быть размещена там в пространстве листа.

### Удалить соединения фазных шин

1. Чтобы снова убрать определение соединений фазных шин, выделите в диалоговом окне Генерировать соединения фазовых шин нужные соединения с определением функции "Соединение фазных шин".
2. Выберите пункт всплывающего меню Удалить соединение фазных шин.

!!! info "Для сведения:"

    Все свойства, записанные на соединении фазовых шин, будут удалены.

!!! info "Для сведения:"

    Соединение получит определение функции "Жила/провод".

3. Заново маршрутизируйте возвращенные в свой прежний вид соединения в пространстве листа.

### Удалить немаршрутизируемые соединения

1. Выберите пункты меню Данные проекта > Соединения > Навигатор.
2. Определите в навигаторе соединений фильтр на определение функции "Соединение фазовых шин".

!!! info "Для сведения:"

    Отображаются только имеющиеся соединения фазовых шин и немаршрутизированные соединения, возникшие в результате преобразования в соединения фазовых шин.

3. Удалите возникшие немаршрутизируемые соединения в навигаторе соединений.

!!! tip "Совет:"

    Для удаления из проекта немаршрутизируемых соединений можно воспользоваться сжатием проектов. Для этого выберите пункты меню Проект > Организовать > Сжать. В диалоговом окне Настройки: Сжатие установите флажок для уровня иерархии Убрать неразмещенные соединения.

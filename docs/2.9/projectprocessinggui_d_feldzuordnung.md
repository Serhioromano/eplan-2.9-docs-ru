# Диалоговое окно Присвоение полей (импорт устройств / маршрутизируемых соединений) / Диалоговое окно Настройки: Импорт

Вы открыли проект.

* Данные проекта > Устройство > Импортировать. В диалоговом окне Импортировать данные устройств щелкните мышкой рядом с полем Присвоение полей по кнопке ++...++.
* Данные проекта > Соединения > Импортировать > Маршрутизируемые соединения. В диалоговом окне Импортировать данные соединений щелкните мышкой рядом с полем Присвоение полей по кнопке ++...++.

Вы открыли проект.

* Параметры > Настройки > Проекты > "Имя проекта" > Устройство > Импорт
* Параметры > Настройки > Проекты > "Имя проекта" > Маршрутизируемые соединения > Импорт

В диалоговом окне Настройки: Импорт можно создавать и обрабатывать схемы с настройками для присвоения полей при импорте устройств и маршрутизируемых соединений. Эти настройки выбираются и применяются в диалоговом окне Присвоение полей.

Диалоговое окно Присвоение полей делает возможным присвоение внешнего поля данных из источника данных свойству EPLAN. Присвоения можно выбирать в качестве схемы, сохранять и обрабатывать.

Обзор основных элементов диалогового окна:

### Схема

Выберите из раскрывающегося списка схему, которую требуется использовать или обработать. Для этого также воспользуйтесь соседней [панелью инструментов](schemeconfig_m_schemaschaltflaechen.md).

### Панель инструментов Таблица

Кнопка |  Значение
---|---
![](images/all_new_as.png){: .ui-icon } (Создать) |  Создает новую строку. Только в диалоговом окне Присвоения полей в строках со статусом ![](images/all_statusdelete_as.png){: .ui-icon }: разблокирует еще не присвоенное обозначение поля данных для обработки.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет выделенные строки.

### Таблица

В таблице имеются следующие столбцы:

* Статус: Пиктограмма ![](images/erroricon_error_systemmessages.png){: .ui-icon } показывает, что поле данных есть только в схеме. Эти строки не учитываются при импорте и могут быть удалены при помощи кнопки ![](images/all_delete_as.png){: .ui-icon } (Удалить).
Только в диалоговом окне Присвоение полей: пиктограмма ![](images/all_statusok_as.png){: .ui-icon } показывает, что поле данных есть в источнике данных и его удалось идентифицировать.
* Номер столбца: Возможность ввода для номера столбца, на основании которого выбираются поля данных во внешнем источнике данных, из которого должны импортироваться данные.
* Внешнее обозначение поля данных: В этом столбце отображаются обозначения поля данных внешнего источника данных. Если в источнике данных отсутствует строка с обозначениями поля данных, здесь ничего не отображается.
* Свойство: В этом столбце отображается свойство EPLAN, присвоенное внешнему полю данных. Для присвоения свойства щелкните в столбце Свойство по одной из ячеек. Щелкнув кнопку ++...++, можно открыть диалоговое окно Выбор свойства и выбрать другое свойство.

!!! tip "Совет:"

    В диалоговом окне Присвоение полей все ячейки строки выделены серым цветом, если обозначению поля данных, обнаруженному во внешнем источнике данных, еще не присвоено свойство EPLAN. В этом случае поле данных имеет статус ![](images/all_statusdelete_as.png){: .ui-icon } (Поле данных в схеме отсутствует).

    Для присвоения полей щелкните одну из ячеек в окрашенной серым цветом строке, а затем на панели инструментов таблицы щелкните кнопку ![](images/all_new_as.png){: .ui-icon } (Создать). После этого все другие ячейки этой строки станут доступны для ввода.

!!! note "Замечание:"

    Значение свойства Обозначения выводов устройства (все) (ид. 20038) вводится в файле источника с разделителем:

    Синтаксис, например для четырех обозначений выводов устройства, выглядит следующим образом: a¶b¶c¶b

    Требуемый разделитель "¶" можно ввести при помощи комбинации клавиш [Alt] + 0182.

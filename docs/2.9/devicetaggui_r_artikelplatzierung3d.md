# Вкладка Размещение изделия (3D)

Вы открыли проект. Навигатор пространства листов открыт, и одно пространство листов открыто. В пространстве листа находятся трехмерные размещения изделия. Вы дважды щелкнули по размещению изделия или выбрали пункты меню Обработать > Свойства. В диалоговом окне Свойства <Условное обозначение>: Размещение изделия (3D) выберите вкладку Размещение изделия.

На этой вкладке указываются свойства размещенного на монтажной плате функционального элемента.

Обзор основных элементов диалогового окна:

### Видимое ОУ

Видимое ОУ будет показано в графическом редакторе рядом с условным обозначением. Если вы определили в настройках проекта, что устройство должно автоматически нумероваться при вставке символов, здесь вам будет автоматически предложено ОУ. Вы можете сохранить или переписать вручную это ОУ.

Кроме того, можно использовать [всплывающее меню или кнопку ++...++](devicetaggui_r_feldsichtbaresbmk.md), чтобы cгенерировать или выбрать ОУ.

### Полное ОУ

В этом поле отображается полное ОУ (т. е. структурный идентификатор текущего окружения + видимое ОУ) видимого ОУ / функционального элемента. С помощью кнопки ++...++ откройте диалоговое окно Полное ОУ. Здесь можно обработать структурный идентификатор и, например, выбрать другое место установки.

### Обозначение

В этом поле отображается уже назначенное обозначение для ***Клемм***, в этом поле можно его изменить. Для остальных функций это поле не имеет значения.

### Определение функции

Указанное здесь определение функции для размещения изделия в зависимости от определения функции автоматически присваевается локальному функциональному элементу. Щелкните по ++...++, чтобы открыть диалоговое окно Определения функций и, если нужно, выберите другое определение функции.

### Функциональный элемент

Группа продуктов в главной группе продуктов Механика в базе данных изделий, которая специально определена для механических функциональных элементов определения функции изделия.

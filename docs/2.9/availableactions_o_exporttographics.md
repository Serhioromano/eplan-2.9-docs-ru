# Операция: exportToGraphics

**Параметр** |  **Описание**
---|---
TYPE  |  Вид задачи, которую должна выполнить операция:
GRAPHICPROJECT: Экспортировать проект в графическом формате (TIF, GIF, PNG, JPG).
GRAPHICPAGE: Экспортировать страницы в графическом формате (TIF, GIF, PNG, JPG).
PROJECTNAME  |  Имя проекта с полным путем файла (является опцией).
Если не задано, то выбранный проект используется, когда операция вызывается через интерфейс пользователя (напр., через сценарий или панель инструментов). При вызове из командной строки Windows следует определить PROJECTNAME или сначала следует использовать ProjectAction. В противном случае отобразится системное сообщение.
PAGENAME  |  Имя экспортируемой страницы (необязательно). Этот параметр оказывает влияние, только когда параметр TYPE имеет следующее значение: GRAPHICPAGE.
EXPORTSCHEME  |  Схема для графического экспорта (не обязательно). Задает значения по умолчанию для других необязательных параметров. Если данный параметр отсутствует или пуст (''), то применяется схема графического экспорта, использовавшаяся последней.
DESTINATIONPATH  |  Целевой каталог, в котором сохраняются графические файлы. Если этого каталога еще нет, то он создается. При экспорте проекта под этим каталогом создается новый каталог с именем проекта, в котором сохраняются отдельные графические файлы.
FORMAT  |  Формат вывода: Поддерживаемые форматы экспорта:

* BMP: доступная глубина цвета: 1, 4, 8, 16, 24 и 32
* TIF: доступная глубина цвета: 1, 4, 8, 24 и 32
* GIF: глубина цвета нерелевантна
* PNG: доступная глубина цвета: 1, 4, 8, 16, 24 и 32
* JPG: доступная глубина цвета: 1, 4, 8, 16, 24 и 32

Данный параметр необязателен. При отсутствии иных указаний используется значение по умолчанию из настроек схемы.
COLORDEPTH  |  Глубина цвета рисунка. Возможные значения — 1, 8, 16, 24 и 32. Данный параметр необязателен. При отсутствии иных указаний используется значение по умолчанию из настроек схемы.
IMAGEWIDTH  |  Ширина рисунка в пикселах. Высота рассчитывается автоматически исходя из габаритов страницы. Данный параметр необязателен. При отсутствии иных указаний используется значение по умолчанию из настроек схемы.
IMAGECOMPRESSION  |  Тип сжатия при выводе в формате Tiff. Возможные значения LZW, RLE, CCITT3, CCITT4, NONE. Данный параметр не влияет на другие форматы вывода. Для сжатия CCITT3-, CCITT4- и RLE- глубина цвета всегда составляет 1, т. е. создается двоичный рисунок. Данный параметр необязателен. При отсутствии иных указаний используется значение по умолчанию из настроек схемы.
BLACKWHITE  |  Вывод — черно-белый. Замечание: На формат и размер рисунка это не влияет. Данный параметр необязателен.
Значение по умолчанию: 1 (черно-белый).
USEPAGEFILTER  |  Определяет, должны ли использоваться только отфильтрованные страницы или все страницы проекта (необязательно). Значение по умолчанию: 0.

!!! note "Замечание:"

    * Если страница указана в параметре PAGENAME, то экспортируется только эта страница, а параметр USEPAGEFILTER игнорируется.
    * Если в параметре PAGENAME явно не указано страниц, и значение параметра USEPAGEFILTER установлено на 1, в навигаторе страниц экспортируются только отфильтрованные страницы.

!!! example "Пример:"

    Экспортировать проект:exportToGraphics
    /TYPE:GRAPHICPROJECT
    /PROJECTNAME:C:\Projects\EPLAN\ESS_Sample_Project.elk
    /DESTINATIONPATH:C:\temp
    /FORMAT:BMP
    /COLORDEPTH:24
    /IMAGEWIDTH:1024
    /BLACKWHITE:1Экспортировать страницу:exportToGraphics
    /TYPE:GRAPHICPAGE
    /PROJECTNAME:C:\Projects\EPLAN\ESS_Sample_Project.elk
    /PAGENAME:=EB3+ET1/2
    /DESTINATIONPATH:C:\temp
    /FORMAT:BMP
    /COLORDEPTH:24
    /IMAGEWIDTH:1024
    /BLACKWHITE:1Экспортировать несколько страниц:exportToGraphics
    /TYPE:GRAPHICPAGE
    /PROJECTNAME:C:\Projects\EPLAN\ESS_Sample_Project.elk
    /DESTINATIONPATH:C:\temp
    /FORMAT:BMP
    /COLORDEPTH:24
    /IMAGEWIDTH:1024
    /BLACKWHITE:1
    /USEPAGEFILTER:1Экспортировать страницу. При этом копируется несколько значений по умолчанию из схемы экспорта:exportToGraphics
    /TYPE:GRAPHICPAGE
    /PROJECTNAME:C:\Projects\EPLAN\ESS_Sample_Project.elk
    /PAGENAME:=EB3+ET1/2
    /EXPORTSCHEME:Bitmap

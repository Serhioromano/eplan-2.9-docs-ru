# Обработать позиции легенды

В случае изменения оснащения монтажной платы может потребоваться заново пронумеровать позиции легенды.

Условия:

* Вы открыли проект.
* Вы открыли диалоговое окно Компоновка электрошкафа 2D - <Имя проекта>.
* Вы выделили на вкладке Дерево монтажную плату, позиции легенды которой требуется обновить. (Многократный выбор здесь невозможен.)

1. Выберите команды Всплывающее меню > Обработать позицию легенды.
2. Используйте в диалоговом окне Обработать позицию легенды кнопки ![](images/all_arrowtop_as.png){: .ui-icon }, ![](images/all_arrowup_as.png){: .ui-icon }, ![](images/all_arrowdown_as.png){: .ui-icon } и ![](images/all_arrowbottom_as.png){: .ui-icon }, чтобы переместить позиции легенды вверх/вниз или в начало/конец списка.
3. Щелкните по кнопке ++Нумеровать++.
4. Определите настройки нумерации в диалоговом окне Нумеровать.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выполняется адаптация записей в столбце Позиция легенды.

6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Выполняется обновление позиций легенды.

Обновленные записи можно просмотреть на вкладке Список (Всплывающее меню > Конфигурировать представление в списке и выбор флажка Позиция легенды).

# Создать и обработать штекер

Управление штекерами осуществляется на основе определения штекеров, а создание штекеров возможно в навигаторе штекеров или навигаторе устройств, а также при вставке в схему соединений.

Чтобы создать штекер с предопределенными свойствами (с определенным количеством штырей и / или гнезд), сначала требуется создать определение штекера. При этом задается, содержит ли штекер только штыри, только гнезда или оба вида контактов.

Если вы хотите создать новый штекер (без дополнительных свойств), при добавлении контакта штекера укажите просто ОУ штекера в диалоговом окне Свойства ++...++. Тогда будет создан новый штекер, отличный от заданных ранее, однако без определения штекера. Такой штекер виден в отчетах, однако вы не можете ему назначить свойства (это можно сделать только через определение штекера).

Вы можете сначала создать штекеры, отличные от заданных до этого, и позже обработать их свойства Тем не менее, у вас есть возможность задать свойства штекера прямо при его создании. С помощью определений функций вы можете задать штекер с предопределенными свойствами.

Во вкладке [Определение штекера](devicetaggui_r_steckerdefinition.md) в диалоговом окне, описывающем свойства, отредактируйте свойства штекера.

### Создать штекер

1. Выберите пункты меню Данные проекта > Штекеры > Навигатор.
2. Выберите пункт всплывающего меню Генерировать определение штекера.
3. Выберите из списка подменю, из каких разъемов должен состоять штекер: из гнезд, штырей или из комбинации штырей и гнезд. При общей работе со штырями и гнездами, выберите строку Штыри и гнезда; при раздельной работе выберите Гнезда или Штыри, для того, чтобы задать сторону, на которой они будут находиться.
4. Введите в диалоговом окне Свойства ++...++ на вкладке Определение штекера нужные данные для нового штекера.
5. Щелкните по кнопке ++OK++.

### Создать штекер с предопределенными свойствами.

1. Выберите пункты меню Данные проекта > Штекеры > Навигатор.
2. Выделите ОУ и выберите пункт всплывающего меню Создать.
3. В диалоговом окне Определения функций в поле Выбор выделите определение штекера.
4. Щелкните по кнопке ++OK++.
5. Введите в диалоговом окне Свойства ++...++ на вкладке Определение штекера нужные данные для нового штекера.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Под выделенным ОУ будет создано определение штекера с соответствующими свойствами.

### Обработать штекер

1. Выделите определение штекера в графическом редакторе или навигаторе клеммников.
2. Выберите пункты меню Данные проекта > Штекеры > Обработать.
3. При необходимости контакты штекера можно пронумеровать в диалоговом окне Обработать штекеры или отсортировать их по определенным критериям. Для обработки используйте возможности всплывающего меню.
4. Щелкните по кнопке ++OK++.

# Диалоговое окно Настройки: Графический выбор макроса

Параметры > Настройки > Фирма > Представление > Графический выбор макроса.

В этом диалоговом окне задаются настройки для графического выбора макроса.

Обзор основных элементов диалогового окна:

### Панель инструментов

В панели инструментов, расположенной над списком, имеются следующие кнопки:

Кнопка |  Значение
---|---
![](images/all_new_as.png){: .ui-icon } (Создать) |  Генерирует новую пустую строку в таблице.
![](images/all_delete_as.png){: .ui-icon } (Удалить) |  Удаляет выделенные строки из таблицы.

### Каталог

В этом поле определяется имя каталога, который должен быть заменен при графическом выборе макроса текстом в расположенном поблизости поле. Щелкните по кнопке ++...++, чтобы пролистать в диалоговом окне Выбрать каталог дерево каталогов и выбрать каталог.

### Текст

Сохраните в этом поле текст, который должен заменить в графическом выборе макроса каталог, заданный в расположенном рядом поле Каталог.

### Рисунок

В этом поле определяется рисунок, который должен быть показан для каталога. Нажмите кнопку ++...++, чтобы открыть диалоговое окно Выбор рисунка и выбрать другой рисунок.

### Всплывающее меню

Всплывающее меню дает доступ, в зависимости от типа поля (например, дата, целое число, многоязычный), к пунктам меню, при помощи которых вы можете по необходимости, например, влиять на представление таблиц или обрабатывать значения в полях. Обзор пунктов этого всплывающего меню вы можете найти в разделе [Пункты всплывающего меню](userinterface_m_kontextmenu.md).

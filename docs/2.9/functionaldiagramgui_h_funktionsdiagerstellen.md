# Создание диаграмм функций

Функциональные диаграммы можно создавать на уже существующей или новой странице. Ниже описано создание функциональной диаграммы на новой странице. Порядок действий при создании функциональных диаграмм на уже существующих страницах аналогичен.

Условия:

* Вы открыли проект.
* Вы добавили в проект библиотеку символов OS_SYM_ESS.slk. (Необходимые настройки можно выполнить через пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Управление > Библиотеки символов).
* Вы указали в настройках проекта специальный символ, который будет использоваться для размещения текстов свойств в блок-схемах рабочего процесса (например, OS1 для функциональных диаграмм с ориентацией на текст или OS2 для функциональных диаграмм с ориентацией на символ).
* Вы вставили функции или устройства в проект и указали необходимые значения для свойств Функциональный текст, Статус [1] по Статус [10] и т. д.

!!! note "Замечание:"

    * Если вы создаете функциональную диаграмму с ориентацией на символ, настройка проекта Функциональное размещение с графикой символов должна быть активирована. Для функциональной диаграммы с ориентацией на текст эту настройку необходимо отключить.
    * Свойство Состояние [10] по умолчанию предназначено для ввода переменных для диаграмм GRAFCET.

1. Создайте с помощью пунктов меню Страница > Создать страницу проекта.
2. Выберите в диалоговом окне Новая страница в поле Тип страницы запись 'Обзор (I)' и наберите в поле Описание страниц текст описания для страницы проекта.
3. Выберите в таблице в поле Имя формы нужную форму, например F28_001.f28 для функциональной диаграммы с ориентацией на текст или F28_002.f28 для функциональной диаграммы с ориентацией на символ.
4. Введите в таблице в поле Сетка значение "3 мм".
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Будет создана страница проекта, содержащая выбранную форму для блок-схемы рабочего процесса.

!!! info "Для сведения:"

    Если настройки сетки и используемая форма подходят друг к другу, то точки сетки попадут ровно в середину определенных в форме ячеек таблицы. Это облегчает процесс размещения устройств и стадий рабочего процесса.

1. Откройте страницу проекта для блок-схемы рабочего процесса, например, с помощью двойного щелчка мыши на странице проекта в навигаторе страниц.
2. Выберите пункты меню Данные проекта > Устройство > Навигатор.
3. Выделите функцию в навигаторе устройств и выберите пункт всплывающего меню Функциональное размещение и в открывшемся меню — пункт С графикой символов.

!!! info "Для сведения:"

    Специальный символ, заданный в настройках проекта, появится рядом с курсором. При создании функциональной диаграммы с ориентацией на текст отображаются только тексты свойств функции, при создании функциональной диаграммы с ориентацией на символ отображается, помимо этого, также и символ функции.

4. Или выберите пункт в открывшемся меню С графикой макросов, если при создании функциональных диаграмм с ориентацией на символ нужно размещать графику макроса изделия в функциональной диаграмме. Если для выбранного устройства не указан макрос изделия, можно после соответствующего запроса открыть пункт "Выбор макроса" и выбрать макрос.
5. Разместите графику на странице проекта. При этом точка вставки этого символа должна находиться в левом верхнем углу ячейки таблицы в диаграмме.

!!! info "Для сведения:"

    Тексты свойств будут размещены в диаграмме. У функциональных диаграмм с ориентацией на символ будет дополнительно размещена графика символа.

6. Или разместите макрос в функциональной диаграмме.

!!! info "Для сведения:"

    Сгруппированная графика макроса и тексты свойств главной функции размещаются в выбранной позиции на странице.

7. Повторите вышеописанные шаги для других устройств, которые следует разместить в блок-схеме рабочего процесса.

!!! tip "Совет:"

    При размещении с помощью кнопки ++Табуляция++ можно "пролистывать" варианты символа / макроса. С помощью комбинации клавиш [Shift]+[Табуляция] при размещении макроса можно "пролистывать" виды представлений.

Порядок рабочего процесса чертится с помощью графических элементов (например, ломаных линий). EPLAN поддерживает ускоренную вставку типичных элементов функциональной диаграммы. Стрелки, сигнальные точки, условия И и ИЛИ можно обозначить с помощью соответствующих пунктов в меню Вставить > Блок-схема рабочего процесса в соответствующей функциональной диаграмме.

1. Если вам необходимо начертить ломаную линию, выберите пункты меню Вставить > Графика > Ломаная линия. (Пользуясь другими подпунктами меню Графика, вы можете чертить также прямоугольники, круги и т. д.)
2. Укажите по очереди точки ломаной линии и щелкните левой клавишей мыши, чтобы разместить каждую из этих точек.
3. Завершите операцию через [Пробел] или пункт всплывающего меню Закрыть ломаную линию.
4. Если вам необходимо вставить стрелку, выберите пункты меню Вставить > Блок-схема рабочего процесса > Стрелка (пользуясь подпунктами меню Сигнальная точка, Условие И и Условие ИЛИ, можно вставлять другие элементы блок-схемы рабочего процесса).

!!! info "Для сведения:"

    Рядом с курсором появится стрелка.

5. Нажимая клавишу ++Tab++ несколько раз, выберите нужный вариант стрелки.
6. Разместите стрелку в функциональной диаграмме.

!!! tip "Совет:"

    Специальные элементы блок-схемы рабочего процесса (например, косую черту, различные источники сигнала и т. д.) можно вставлять в качестве макроса окна из каталога макросов OS_ESS.

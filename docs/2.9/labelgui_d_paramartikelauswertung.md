# Вкладка Настройки

* Вы открыли проект. Сервисные программы > Данные изготовления > Экспорт / маркировка. В диалоговом окне Экспортировать данные изготовления / Вывести маркировку рядом с полем Настройки щелкните по кнопке ++...++. В диалоговом окне выберите Настройки: Экспорт данных изготовления / маркировка вкладку Настройки.
* Параметры > Настройки > Пользователь > Интерфейсы > Экспорт данных изготовления / маркировка. В диалоговом окне выберите Настройки: Экспорт данных изготовления / маркировка вкладку Настройки.

На этой вкладке указывается, какие данные проекта или изделий отображаются при выводе данных экспорта / маркировки.

Обзор основных элементов диалогового окна:

### Значение изделия

Настройки в этом групповом поле влияют только на спецификацию изделий и групповую спецификацию изделий:

* Развернуть узлы: Установите этот флажок, если необходимо развернуть узлы на их составные части, т. е. на отдельные изделия.
* Разъединить модули: Установите этот флажок, если необходимо разъединить модули на их составные части, то есть на отдельные изделия.
* До степени: Укажите в этом поле, до какого уровня иерархии следует развернуть узлы / разъединить модули. Ввод возможен, только если установлен флажок Развернуть узлы и / или Разъединить модули.
* Объединить изделия устройства: Этот флажок следует установить для схемы типа отчета "Спецификация изделий", если в спецификации изделий изделия на клеммах нужно объединить в клеммник, а изделия на кабельных соединениях — в кабели. При снятии флажка изделия выводятся отдельно.

!!! example "Пример:"

    Если флажок установлен, спецификация изделий содержит одну позицию с 25 клеммамиUK5и одну позицию с 10 втулками для жил; если флажок не установлен, клеммаUK5перечисляется 25 раз, а втулки для жил — 5 раз (по две связанные втулки дляжилы).

### Настройка / Значение

В этой таблице можно указать относящиеся к проекту и функции настройки фильтра и сортировки: щелкните в столбце Значение на ++...++, выберите необходимый проект, и укажите в диалоговом окне Фильтр / в диалоговом окне Сортировка соответствующие критерии.

В поле Значение для настройки "Операция" выводится определенное пользователем расширение, присвоенное этикетке в ходе выполнения операции; щелкните ++...++, чтобы открыть диалоговое окно Ввести имя операции и при необходимости заменить эту операцию на другую.

### Учитывать изделие

Установите флажок для изделий, данные которых должны выводиться при отображении для "спецификации изделий", "групповой спецификации изделий" и "списка производителей / поставщиков".

!!! note "Замечание:"

    * Настройки Устройство без номера изделия, Изделие клеммника, Изделие клеммы, Изделие кабеля, Изделие штекера, Изделие контакта штекера, Изделие сборных шин, Изделие выводов устройства сборной шины также предназначены для отчета ***Список обозначений устройств***.
    * Настройка Изделие предварительного планирования учитывает изделия, которые были записаны ***только*** в ходе предварительного планирования на объектах планирования. Если в объекте планирования уже имеется главная функция или размещение изделий, эта настройка не будет учитываться в значениях изделий.

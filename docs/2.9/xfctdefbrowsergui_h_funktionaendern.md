# Изменить функции

если функция используется в проекте, присвоенное определение функции уже несет с собой определенные свойства. Данные из определения функции являются всего лишь предварительным расположением и их можно перезаписать на каждом отдельном условном обозначении. Кроме того, можно добавить дополнительные свойства.

### Изменить свойства

Условие:

Вы разместили функцию (то есть условное обозначение) на странице схемы соединений. Или вы сгенерировали функцию в одном из навигаторов.

1. Выделите функцию.
2. Выберите пункт меню Обработать > Свойства.
3. В диалоговом окне Свойства ++...++ измените свойства функции. Используйте для этого разные вкладки и поля ввода.
4. Определите– если необходимо, – дальнейшие свойства, которые не были предварительно указаны в определении функции.
5. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Новые и измененные свойства сохраняются в функции.

!!! info "Для сведения:"

    На условном обозначении отображаются актуальные свойства (если они выбраны для отображения).

### Выбрать обозначения и описания вывода устройства.

Для каждого определения функции можно сохранить десять наборов обозначений и описаний выводов устройства. При создании функции в качестве предварительного назначения используется первое предложение обозначений выводов устройства.

Условие:

Вы разместили функцию (то есть условное обозначение) на странице схемы соединений. Или вы сгенерировали функцию в одном из навигаторов.

1. Выделите функцию.
2. Выберите пункт меню Обработать > Свойства.
3. Выберите в диалоговом окне Свойства ++...++ вкладку <Категория функции>.
4. Выберите в поле Обозначение вывода устройства запись из раскрывающегося списка или введите обозначение вывода устройства вручную.
5. Выберите в поле Описания выводов устройства запись из раскрывающегося списка или введите описание вывода устройства вручную.
6. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    Функции присваиваются данные вывода устройства.

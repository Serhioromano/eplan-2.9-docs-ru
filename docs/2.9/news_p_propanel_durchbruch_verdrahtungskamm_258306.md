# Автоматически сгенерированные соединительные отверстия гребенки для проводов

Гребенки для проводов, используемые в системах направляющих для проводов, реализованы в EPLAN как пользовательская шина на основе контура выдавливания. Чтобы после вставки такого контура в пространство листа можно было проложить маршрутизируемые соединения через гребенку для проводов на обратной стороне, соединительное отверстие для проводов на гребенке для проводов ранее необходимо было размещать вручную. Теперь при вставке гребенки для проводов соединительное отверстие для проводов добавляется ***автоматически***.

Условием для этого является то, что сначала на передней стороне гребенки для проводов в редакторе контура размещается монтажная поверхность "Гребенка для проводов, передняя сторона", а на обратной — монтажная поверхность "Гребенка для проводов, задняя сторона".

![](images/benefit.png)Эффект:

После размещения гребенки для проводов в пространстве листа отпадает необходимость в последующей ручной вставке соединительного отверстия для проводов при маршрутизации.

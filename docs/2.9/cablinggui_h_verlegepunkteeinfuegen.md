# Вставить точки маршрутизации (топология)

Точки маршрутизации необходимы для построения сети, например для входа соединения к цели, для разветвления на несколько сегментов или для соединения сегментов маршрутизации.

Точки маршрутизации структурного идентификатора представляют собой определенную структуру обозначений. С помощью этих специальных точек маршрутизации можно прокладывать соединения между различными структурами обозначений, не размещая подключенные функции топологии.

Условие:

В графическом редакторе уже открыта страница топологии.

1. Выберите пункты меню Вставить > Топология > Точка маршрутизации.
2. Разместите точку маршрутизации посредством щелчка мышью в графическом редакторе.

!!! info "Для сведения:"

    Откроется диалоговое окно Свойства ++...++ (если в настройках пользователя для графической обработки в категории Диалоговые окна свойств установлен флажок При вставке точек маршрутизации и сегментов маршрутизации (топология)).

3. Задайте во вкладке [Точка маршрутизации](devicetaggui_r_cablingstrecke.md) данные для точки маршрутизации.
4. Чтобы вставить ***точку маршрутизации структурного идентификатора***, щелкните по кнопке ++...++ рядом с полем Определение функции. В следующем диалоговом окне выберите определение функции "Точка маршрутизации структурного идентификатора топологии".

!!! info "Для сведения:"

    Для точек маршрутизации структурного идентификатора отображается дополнительная вкладка Подключенные структуры. В ней можно установить фильтр, настройки которого будут учитываться при автоматическом генерировании функций топологии.

5. Щелкните по кнопке ++OK++.
6. Завершите операцию, выбрав пункт всплывающего меню Прервать операцию или нажав клавишу ++Esc++.

!!! tip "Совет:"

    В навигаторе топологии можно генерировать неразмещенные точки маршрутизации через пункт всплывающего меню Генерировать точку маршрутизации.

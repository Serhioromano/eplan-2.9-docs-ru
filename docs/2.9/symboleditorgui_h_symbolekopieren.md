# Копировать символы

При копировании символов создается точная копия оригинального символа, причем возможен перенос символов из одной библиотеки символов в другую.

Условие:

Вы открыли проект и как минимум одну библиотеку символов.

1. Выберите следующие пункты меню: Сервисные программы > Основные данные > Символ > Копировать
2. В диалоговом окне Выбор символа выберите символ, который хотите скопировать. Здесь возможен многократный выбор.
3. Щелкните по кнопке ++OK++.
4. В диалоговом окне Копировать символы выберите в пункте Цель библиотеку символов, в которую необходимо скопировать символы: Откройте раскрывающийся список в поле Библиотека символов.
5. В поле Номер исправьте предлагаемый номер или откройте раскрывающийся список еще не распределенных номеров.
6. В поле Имя введите имя символа, для этого исправьте предлагаемое имя (<Имя оригинала (копия)> для копий в библиотеках символов, в которых это имя уже есть, или <Имя оригинала>, если отсутствуют символы с таким именем).
7. В поле Описание введите текст описания символа.
8. Активируйте флажок в поле С вариантами, если символ должен копироваться вместе с имеющимися своими вариантами. если флажок не установлен, копируется только первый существующий вариант.
9. Щелкните по кнопке ++OK++.

# Вкладка Определение блока

* Параметры > Настройки > Пользователь > Интерфейсы > Экспорт и импорт DXF / DWG. Выберите вкладку Определ. блока.
* Страница > Экспортировать > DXF/DWG. Щелкните рядом с полем Схема по кнопке ++...++. Выберите вкладку Определ. блока.

Определите на этой вкладке, как блоки должны учитываться при экспорте.

Обзор основных элементов диалогового окна:

Групповое поле 'Динамические отчеты'

При экспорте в формате DXF/DWG динамические и подчиненные области форм объединяются и копируются как блоки. Задайте здесь префиксы обозначений на область, например "ВЕРХ" для верхнего колонтитула.

* * *

Групповое поле 'Свойства блока'

В этом групповом поле можно выполнить следующие настройки:

* Экспортировать невидимые атрибуты: установите этот флажок, если необходимо экспортировать ***все*** специфические для AutoCAD атрибуты функционального элемента, в том числе невидимые.
* Генерировать один блок для символа: установите этот флажок, если для каждого размещенного символа EPLAN необходимо генерировать блок AutoCAD. если флажок снят, блоки создаваться не будут, а страницы будут экспортироваться как графика.
* Генерировать один блок для рамки: установите этот флажок, если необходимо генерировать блоки AutoCAD для всех рамок EPLAN. если флажок снят, блоки создаваться не будут, а рамки будут экспортироваться как графика.
* Генерировать один блок для станд. формы: установите этот флажок, если необходимо генерировать блоки AutoCAD для всех стандартных форм EPLAN. если флажок снят, блоки создаваться не будут, а формы будут экспортироваться как графика.

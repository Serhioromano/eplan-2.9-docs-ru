# Вкладка "Параметры шлангов"

У вас установлен конфигуратор шлангопроводов в качестве дополнительного модуля для EPLAN Fluid. Вы запустили EPLAN Fluid и конфигуратор шлангопроводов.

* Выберите в EPLAN Fluid пункты меню Параметры > Настройки > Пользователь > Add-ons > Конфигуратор шлангопроводов и щелкните в следующем диалоговом окне по вкладке Параметры шлангов.
* Или выберите в диалоговом окне Конфигурировать шлангопроводы конфигуратора шлангопроводов пункты меню Дополнительно > Настройки и щелкните в следующем диалоговом окне по вкладке Параметры шлангов.

Или у вас установлен конфигуратор шлангопроводов в качестве самостоятельной программы (Stand-alone), и программа запущена.

* Выберите в конфигураторе шлангопроводов пункты меню Параметры > Настройки > Пользователь > Add-ons > Конфигуратор шлангопроводов и щелкните в следующем диалоговом окне по вкладке Параметры шлангов.

На этой вкладке можно определить данные по умолчанию для параметров шланга. При этом вы присваиваете определенному рабочему давлению разные группы шлангов и сохраняете для выбранных групп шлангов требуемые максимально и минимально допустимые значения диаметра шланга. Сохраненные в таблице рабочие давления предлагаются на выбор при конфигурации шлангопроводов в диалоговом окне Обработать ++...++.

Используйте кнопки панели инструментов над списком, чтобы создавать новые записи, обрабатывать имеющиеся и/или удалять более ненужные записи.

Обзор основных элементов диалогового окна:

### Рабочее давление (список)

Определите здесь максимально допустимые значения динамического рабочего давления шлангопроводов. Значения указываются в "бар".

Используйте кнопки панели инструментов над списком, чтобы создавать новые записи, обрабатывать имеющиеся и/или удалять более ненужные записи.

!!! note "Замечание:"

    Разным рабочим давлениям могут быть присвоены, соответственно, разные группы шлангов.

### Таблица

В этой таблице рабочему давлению присваиваются требуемые группы шлангов, а также минимально и максимально допустимые номинальные диаметры шлангов.

Используйте кнопки панели инструментов над таблицей для создания новых записей и/или удаления ненужных записей.

!!! note "Замечание:"

    При выборе стандартной схемы DIN 20066_2012-01 настройки на этой вкладке и на вкладке [Группы шлангов](fluidhoseconfiggui_r_schlauchgruppen.md) влияют таким образом, что в диалоговом окне Обработать ++...++ для каждого выбранного рабочего давления и для каждого выбранного номинального диаметра шланга на выбор предлагаются типы шлангов, заданные в соответствии со стандартом DIN 20066.

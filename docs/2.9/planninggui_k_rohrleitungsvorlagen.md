# Шаблоны для классов трубы и веществ

К данным, описывающим трубопровод, относятся ***класс трубы*** и ***вещество***, транспортируемое по соответствующему трубопроводу (напр., "Вода", "Масло", "Кислород"). В навигаторе шаблонов сегментов можно предварительно определять классы трубы и вещества со всеми важными данными, необходимыми для проекта, в качестве шаблонов. Таким образом, не требуется постоянно отдельно вводить важные данные: шаблоны можно использовать каждый раз при создании новых сегментов в предварительном планировании или при определении трубопроводов на функциональной схеме автоматизации. С помощью пунктов всплывающего меню Экспортировать и Импортировать в навигаторе шаблонов сегментов можно предоставить доступ к шаблонам классов трубы и веществ другим пользователям.

### Создать классы трубы и вещества

Шаблоны классов трубы и веществ управляются в ***навигаторе шаблонов сегментов*** под уровнем иерархии "Другие шаблоны". В отличие от шаблонов сегментов шаблоны классов трубы / веществ создаются ***только*** с помощью соответствующих пунктов всплывающего меню (Новый класс трубы, Новое вещество).

На шаблоне классов трубы, кроме идентифицирующего имени и описания, дополнительно задаются такие свойства, как Ступень номинального давления, Материал и Стандарт. Сначала на шаблоне вещества будут отсутствовать другие свойства для ввода данных. Посредством конфигурации определенных пользователем свойств для классов трубы и веществ можно предоставить другие свойства для выбора и ввода данных.

### Выбор шаблонов

В предварительном планировании на сегментах для выбора доступны созданные классы трубы и вещества. На функциональной схеме автоматизации можно выбрать классы трубы и вещества на точках определения трубопровода и выводах трубопровода.

### Присвоенные изделия в качестве фильтров для выбора изделия и устройства

Для шаблонов классов трубы на вкладке Изделие можно указать дополнительное изделие. Затем при выборе изделия выбирать на сегментах можно только изделия, присвоенные классу трубы.

Если для устройства на функциональной схеме автоматизации или для соединения выбран класс трубы, в выборе устройства при соответствующей настройке критериев выбора учитываются только изделия, сохраненные на классе трубы.

### Контрольные прогоны

При проектировании классов трубы и веществ можно использовать различные контрольные прогоны. С помощью контрольных прогонов 028028 и 028029 можно проверить, существует ли в проекте внесенное на объекте вещество или класс трубы в качестве шаблона.

Чтобы определить, подходят ли изделия, внесенные на соединениях / функциях / сегментах, к выбранному классу трубы, можно использовать контрольные прогоны 005092, 007031 и 028031.

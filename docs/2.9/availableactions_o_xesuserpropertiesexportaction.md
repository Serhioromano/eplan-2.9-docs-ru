# Операция: XEsUserPropertiesExportAction

**Параметр** |  **Описание**
---|---
XMLFile |  Имя файла XML, в который следует экспортировать, вкл. полный путь
Project  |  Полное имя проекта, определенные пользователем свойства которого следует экспортировать

!!! example "Пример:"

    XEsUserPropertiesExportAction
    /XMLFile:c:\my_user.xml
    /Project:c:\...\ESS_Sample_Project.elk

# Дублировать объекты в пространстве листа

Команда Дублировать позволяет очень просто вставить несколько копий трехмерных объектов и 3D-макросов в пространство листа. Функция дублирования работает только в пределах пространства листа, она не может быть использована для всех пространств листа сразу.

Условие:

Вы открыли проект. Открыто одно пространство листа.

1. Выделите дублируемый объект, 3D-макрос или группу объектов, начертив рамку вокруг этих объектов и выберите пункты меню Обработать > Дублировать.

!!! info "Для сведения:"

    Объекты появятся рядом с точкой захвата слева над курсором. Точку захвата можно переключать при помощи клавиши ++A++.

2. Выберите пункт всплывающего меню Опции размещения, чтобы вызвать диалоговое окно [Опции размещения](cabinetgui_d_platzieroptionen.md) и задать угол поворота, точку захвата и значения смещения. Щелкните по кнопке ++OK++.
3. Разместите первую копию.
4. Укажите в диалоговом окне Дублировать общее число генерируемых элементов.
5. В следующем диалоговом окне Режим вставки укажите способ обработки структурных идентификаторов дублированных объектов. Щелкните по кнопке ++OK++.

!!! info "Для сведения:"

    EPLAN автоматически генерирует указанное число копий и размещает их, используя интервал, выбранный для первой копии. В навигаторе пространства листа копии отображаются параллельно исходным объектам.

!!! tip "Совет:"

    * Чтобы сгенерировать простую копию, можно также щелкнуть по объекту, удерживая нажатой клавишу ++Ctrl++, а затем перетащить копию на новое место.
    * Если в пространстве листа требуется совместно обработать несколько трехмерных объектов (дублировать, копировать, переместить, вырезать, удалить), выделите эти объекты и с помощью пунктов меню Обработать > Другое > Группировать объедините их в [группу](gededitgui_h_elementegruppieren.md). Затем группирование можно отменить с помощью пунктов меню Обработать > Другое > Отменить группирование.

# Вкладка Настройки (Rittal — Wire Terminal WT)

* Параметры > Настройки > Фирма > Сборка проводов > Rittal — Wire Terminal WT > Настройки
* Сервисные программы > Данные изготовления > Сборка проводов > Rittal — Wire Terminal WT. Щелкните по кнопке ++...++ рядом с полем Схема. Выберите вкладку Настройки.

В этой вкладке хранится информация о технических возможностях используемой машины. Отдельные настройки задаются заранее индивидуально для машины при помощи подходящих значений, которые можно здесь изменить.

Обзор основных элементов диалогового окна:

### Минимальная длина провода

Минимальная длина, необходимая для заданной обработки провода машиной (стандартное значение: 230 мм). Более короткие провода передаются в машину с данной минимальной длиной.

### Максимальная длина провода

Провода, длина которых превышает указанную здесь величину, машина изготовить не может (стандартное значение: 10 000 мм).

### Минимальное поперечное сечение жилы

Минимальное поперечное сечение провода, которое машина может обрабатывать (стандартное значение: 0,5 мм²).

### Максимальное поперечное сечение жилы

Максимальное поперечное сечение провода, которое машина может обработать (стандартное значение: 2,5 мм²).

### Направление подсоединения сверху / снизу / слева / справа

Тексты, которые необходимо напечатать на проводе для указания отдельных направлений подсоединения.

# Генерировать страницы с видами посредством функции "Генерация отчета по проекту"

При создании отчета по проекту с помощью пунктов меню Сервисные программы > Отчеты > Генерация отчета по проекту теперь также генерируются страницы с видами (обзор модели, 2D-отображение отверстий и / или развертка шины), если в диалоговых окнах управления видами доступны соответствующие шаблоны отчетов с макросами страниц. Кроме того, все уже созданные виды обновляются.

![](images/benefit.png)Эффект:

Так как функция "Генерация отчета по проекту" учитывает шаблоны для видов, страницы с видами можно генерировать быстрее. Больше не нужно по отдельности генерировать отчеты для различных видов с помощью различных пунктов меню.

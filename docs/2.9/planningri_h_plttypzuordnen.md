# Присвоение типа ТК (уст./реал.) технологическому контуру / функции ТК

С помощью типа ТК (уст./реал.) определяется, с каким символом технологический контур / функция ТК будут размещены на функциональной схеме автоматизации.

Условия:

* Вы открыли проект, который содержит данные предварительного планирования.
* Вы создали технологический контур / функцию ТК.

1. Выберите следующие пункты меню: Данные проекта > Предварительное планирование > Навигатор
2. Выделите технологический контур / функцию ТК.
3. Выберите пункты меню Обработать > Свойства.
4. Выберите в диалоговом окне свойств вкладку <Технологический контур> / <Функция ТК>.
5. Выберите в таблице Свойства для свойства Тип ТК (уст./реал.) нужный параметр из раскрывающегося списка.

!!! info "Для сведения:"

    При размещении технологического контура / функции ТК автоматически присваивается соответствующий символ из библиотеки символов.

!!! note "Замечание:"

    Для размещенного технологического контура / функции ТК больше нельзя изменить свойство Тип ТК (уст./реал.) через раскрывающийся список. Однако на вкладке Данные символа можно поменять параметры реализации и расположения и тем самым изменить функциональный символ и тип ТК (уст./реал.).

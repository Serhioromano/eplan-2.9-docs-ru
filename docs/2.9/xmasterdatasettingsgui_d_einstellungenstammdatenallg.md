# Диалоговое окно Настройки: Общие (проекты, управление)

Вы открыли проект. Параметры > Настройки > Проекты > "Имя проекта" > Управление > Общее.

В этом диалоговом окне задайте общие, специфические для основных данных, предварительные настройки.

Обзор основных элементов диалогового окна:

### Синхронизировать основные данные проекта при открытии

Если установлен этот флажок, используемые в проекте основные данные синхронизируются с пулом основных данных, как только соответствующий проект будет открыт или восстановлен.

Библиотеки символов и формы могут синхронизироваться только тогда, когда акивирован флажок для отдельных библиотек символов и форм Синхронизировать. Те библиотеки символов и формы, которые синхронизируются, устновите в следующих местах:

* ***Библиотека символов:*** диалоговое окно Настройки: Библиотека символов (Параметры > Настройки > Проекты > "Имя проекта" > Управление > Библиотеки символов)
* ***Формы:*** диалоговое окно Настройки: Вывод в страницах (Параметры > Настройки > Проекты > "Имя проекта" > Отчеты > Вывод в страницах)

Если флажок снят, основные данные ***не*** синхронизируются при открытии проекта, даже если для отдельных библиотек символов или форм установлен флажок Синхронизировать.

### Синхронизировать рамки

Если этот флажок установлен, используемые в проекте рамки синхронизируются с пулом основных данных, как только соответствующий проект будет открыт или восстановлен. Необходимо только, чтобы был активирован флажок Синхронизировать основные данные проекта при открытии.

### Обновить сохраненные изделия при открытии

Если установлен этот флажок, то имеющиеся в проекте изделия синхронизируются с изделиями базы данных изделий при открытии проекта. Если изделия проекта устарели, открывается запрос.

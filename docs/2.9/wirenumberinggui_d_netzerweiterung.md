# Диалоговое окно Сетевое расширение (Нумерация соединений)

Вы открыли проект. Вы открыли диалоговое окно [Настройки: Нумерация соединений](wirenumberinggui_d_verbnumeinstellungen.md) (например, через пункты меню Параметры > Настройки > Проекты > "Имя проекта" > Соединения > Нумерация соединений). В диалоговом окне Настройки: Нумерация соединений выберите определенную пользователем схему, а также вкладку Обозначение. Щелкните по кнопке ++Сетевое расширение++.

В этом диалоговом окне можно расширить определение сети. Эта настройка влияет на обозначение, если в качестве объема выбрана "сеть". Сетевое расширение для нумерации соединений не зависит от свойства Соединенный с сетью соответствующей функции.

Обзор основных элементов диалогового окна:

### Выбор

В дереве выводятся все функции. С помощью кнопки со стрелкой можно перенести выделенную запись из левого списка в правый. Кроме того, можно выделить категорию и перенести все находящиеся под ней функции.

### Сетевое расширение через следующие объекты

В списке приводятся все функции, которые вы перенесли из списка выбора. В этом случае сеть не заканчивается на выбранных функциях, а продолжается за ними.

С помощью команды ![](images/all_delete_as.png){: .ui-icon } (Удалить) можно убрать выбранную функцию списка.

### Всплывающее меню

Пункт меню |  Значение
---|---
Выделить все (только список) |  Выделяет все записи в списке.
